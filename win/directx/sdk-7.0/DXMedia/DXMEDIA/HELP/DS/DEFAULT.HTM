Microsoft DirectShow SDK

Welcome to the Microsoft® DirectShow™ SDK. Before jumping into any of the reference sections, consider reading the articles in the "Getting Started" and "DirectShow Basics" sections first. These articles provide important overview and conceptual information about DirectShow that you will need in other areas of the documentation. For the most recent updates to this documentation, consult the Microsoft DirectX® Web site at http://www.microsoft.com/DirectX/dxm/.
