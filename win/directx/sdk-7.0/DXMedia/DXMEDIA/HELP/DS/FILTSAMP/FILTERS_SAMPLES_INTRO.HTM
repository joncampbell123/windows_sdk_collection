Filters and Samples

This section contains a brief description of the filters and sample applications shipped with Microsoft® DirectShow™. The filters are supplied as binary code only and are available through the Filter Graph Editor. The samples include source code and demonstrate how to write DirectShow filters and applications. You can use these filters and samples as they are or modify them for your own applications.
