MPEG-2 Splitter

The MPEG-2 Splitter filter currently supports playback of MPEG-2 program streams that contain at least one of the following stream types:

  • MPEG-2 video
  • MPEG-2 audio
  • AC3 audio encoded as defined for DVD video
  • LPCM audio encoded as defined for DVD video

The MPEG-2 Splitter filter supports playback of these types when there are DirectShow-compatible decoders capable of handling the particular streams found in the program stream installed.

The MPEG-2 Splitter filter supports the following interfaces:

  • IAMParse—MPEG-2 program stream only
  • IAMStreamSelect—MPEG-2 program stream only, audio streams only
  • IMediaSeeking—includes BYTE mode seeking

For more information, see Using the MPEG-2 Splitter filter.
