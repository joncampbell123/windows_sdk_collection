IAMClockAdjust Interface

The IAMClockAdjust interface gives timing information from an external source to the filter graph. In the case of multimedia streams, if the application uses both audio and video streams outside the filter graph, it can communicate clock information to the filter graph by using this interface. This can be used to synchronize different streams.

When to Implement

Currently, the DirectShow system reference clock object implements this interface. See IReferenceClock for more information. It is also implemented by the IAMMultiMediaStream interface.

When to Use

Use this interface to adjust the default clock object in DirectShow.

IAMClockAdjust::SetClockDelta

IAMClockAdjust Interface

Adjusts the clock by passing the reference time to the filter graph. The incremental time change then gets added to the system reference clock object.

Syntax

HRESULT SetClockDelta(
    REFERENCE_TIME rtDelta
);

Parameters

rtDelta
[in] Value representing the time change, in milliseconds.

Return Value

Returns S_OK if successful or E_POINTER if rtDelta is invalid.
