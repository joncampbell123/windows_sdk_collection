IDirectSound3DListener Interface

The IDirectSound3DListener interface adds 3-D sound to your application.

An application can tell the DSound Audio renderer in a Microsoft® DirectShow™ filter graph to position the audio of any DirectShow-compatible media file anywhere in three dimensions, making your applications more interesting and lifelike. For example, imagine walking down a hallway with a video monitor on the right wall, where the soundtrack of the video goes by your right ear as you walk past it.

To position the audio in three dimensions, your application must query the DSound Audio Renderer filter for this interface (and IDirectSound3DBuffer) before streaming the filter graph. Then you can make calls to IDirectSound3DListener and IDirectSound3DBuffer to position the soundtrack as you like. These methods might not succeed unless the filter graph is streaming when you call them.

Do not query the filter for this interface, or for IDirectSound3DBuffer, unless you will use them to position the soundtrack in 3-D. Otherwise, you will only hear MONO playback.

When to Implement

Only the DSound Audio Renderer filter implements this interface.

When to Use

Use this interface (and IDirectSound3DBuffer) to make the soundtrack of any DirectShow-compatible media file sound like it is coming from a particular point in space.

For a description of IDirectSound3DListener methods, see the Microsoft DirectSound® documentation of the DirectX® Foundation SDK.
