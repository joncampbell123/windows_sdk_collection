Utility Functions

This section contains reference entries for the Microsoft® DirectShow™ utility functions and macros. DirectShow provides utilities for conversion, setup, timers, retrieving interfaces and declaring IUnknown, helper functions for math operations, property pages, BSTR functions, and strings, and stream integer functions. Most utilities are contained in Wxutil.h, but others are contained in Combase.h, Errors.h, Pstream.h, Refclock.h, Renbase.h, Videoctl.h, and Wtype.h.
