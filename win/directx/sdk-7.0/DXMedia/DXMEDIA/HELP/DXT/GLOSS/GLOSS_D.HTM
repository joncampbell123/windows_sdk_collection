DDSURFACEDESC
A data structure that contains a description of the properties of a DirectDrawSurface object. For more information, see the Microsoft® DirectDraw® documentation in the DirectX Foundation SDK.
DeleteObject
A Microsoft Win32® function used to delete a logical pen, brush, font, bitmap, region, or palette, freeing all system resources associated with an object. For more information, see the Platform SDK.
Device context
A data structure defining the graphic objects, their associated attributes, and the graphic modes affecting output on a device.
DIBSECTION
A structure that contains information about a device-independent bitmap created by calling the CreateDIBSection function. See the Platform SDK for more information.
Direct3DRMMeshBuilder3
An object used to interact with 3-D mesh objects that supports the IDirect3DRMMeshBuilder3 interface from Microsoft Direct3D® Retained Mode. The methods of this interface are commonly used to modify input meshes and produce 3-D transform output. For more information, see the Direct3D Retained Mode SDK documentation.
DirectDrawSurface
An object that contains an image's pixel data. The color and alpha of each pixel can be stored in many different pixel formats. For more information, see "Surfaces" in the "DirectDraw Essentials" section of the DirectX Foundation SDK.
Dithering
A way of displaying a wide range colors with a limited palette. Each pixel on the source image is represented by multiple pixels (usually a 2 × 2 square) on the destination image. From a distance, the eye blends the multiple pixels into one color that has more shades than the original palette. The technique results in a better visual appearance than the brute force removal of low precision bits. See a standard graphics reference for details.

Dithering is a supported option by DXSurfaces and DirectX Transforms.

DXSurface
A DirectX Transform data object used to store 2-D images. DXSurfaces usually wrap DirectDrawSurfaces to provide enhanced functionality, such as simplified pixel access.
DXTransform
A DirectX Transform object used to manipulate 2-D images and 3-D meshes. DXTransform objects are created with the IDXTransformFactory interface from a list of possible transform types. They can produce time-dependent effects on images, such as fade, melt, twist, or explode.
