S

Sample
A sample is an indivisible element of an image that is stored in computer memory. In DirectX Transform, a DXSurface is comprised of an array of samples.

The terms 'pixel' and 'sample' are often used interchangably.

Surface picking
The process of choosing which one of the input surfaces contributes most to the output surface at a certain position on the output. Surface picking is often used with mouse interactions to choose different actions in code that depend on which image you select. In cases where several images are alpha blended on the output point, the transform can use the alpha channel of each input sample to choose the input surface.
