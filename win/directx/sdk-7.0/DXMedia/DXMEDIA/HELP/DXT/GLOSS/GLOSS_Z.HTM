z-buffering
A technique for rendering 2-D surfaces or 3-D objects that overlap. For 2-D rendering, each surface must have an attached z-buffer that assigns a depth value for each pixel. When the two overlapping surfaces are rendered, the pixel with the lowest depth value gets drawn on top. The rendering process is similar for overlapping 3-D meshes.
