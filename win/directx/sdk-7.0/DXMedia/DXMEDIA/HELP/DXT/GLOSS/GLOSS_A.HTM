Alpha
The measure of a pixel's opacity. It is commonly represented as a number stored with the color information for a pixel. A maximum value for alpha indicates that a pixel is opaque, while a value of zero indicates it is clear. An intermediate value indicates that a pixel is translucent, and that a pixel rendered under it will partially show through.
Alpha blending
A method for combining two images that uses both pixel colors and alpha values to determine the color of the resulting pixel. This allows an image to be rendered on top of another image, with a blend of both images showing.

When blending two pixels, the color components of both pixels are first scaled by their alpha values. Then the bottom pixel is scaled by the inverse of the top pixel's alpha value and added to the top pixel to form the final blended color.

Alpha channel
The collection of alpha values defined for an image.
Alpha premultiplication
The technique of scaling the three color components of a sample by alpha before storing their values. This saves many mathematical steps when alpha blending two images. For rhe PMARGB32 pixel format, all color values are alpha premultiplied.
ARGB32
One of the two common pixel formats supported by DirectX Transform, consisting of uncompressed alpha, red, green, and blue. The color channel is not premultiplied by alpha in this format.
