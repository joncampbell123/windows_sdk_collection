Dissolve Transform

Animates two input images by texturing them onto two three-dimensional (3-D) rectangles with the first in front of the second. The first image appears to dissolve downward until the it is no longer displayed, providing a transition between the two images.


 

Transform Specifications

The following table contains the information you need in order to use this transform in C++ and script.

Class identifier (CLSID) CLSID_Dissolve
Globally unique identifier (GUID) 90B5E2E1-5050-11D1-B83E-00A0C933BE86
Programmatic identifier (ProgId) DX3DTransform.Microsoft.Dissolve
Category identifier (CATID) CATID_DX3DTransform
Custom interfaces IDissolve
Supported interfaces IDXEffect
Inputs One is required of type DXSurface (preferred) or DirectDrawSurface. The optimal pixel format is DDPF_PMARGB32. The second optional input is of the same type as first. If supplied, it results with a transition from the first to the second.
Output Direct3DRMMeshBuilder3
