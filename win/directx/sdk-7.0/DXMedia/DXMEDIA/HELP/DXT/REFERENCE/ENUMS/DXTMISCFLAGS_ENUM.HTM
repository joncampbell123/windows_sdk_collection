DXTMISCFLAGS Enumeration

Describes the transform flags that specify how to generate the transform output.

Syntax

typedef enum DXTMISCFLAGS
{
    DXTMF_BLEND_WITH_OUTPUT   = ( 1L << 0 ),
    DXTMF_DITHER_OUTPUT   = ( 1L << 1 ),
    DXTMF_OPTION_MASK         = (0x0000FFFF),
    DXTMF_VALID_OPTIONS       = (DXTMF_BLEND_WITH_OUTPUT | DXTMF_DITHER_OUTPUT),
	
    DXTMF_BLEND_SUPPORTED     = ( 1L << 16 ),
    DXTMF_DITHER_SUPPORTED    = ( 1L << 17 ),
    DXTMF_INPLACE_OPERATION   = ( 1L << 24 ),
    DXTMF_BOUNDS_SUPPORTED    = ( 1L << 25 ),
    DXTMF_PLACEMENT_SUPPORTED = ( 1L << 26 ),
    DXTMF_QUALITY_SUPPORTED   = ( 1L << 27 )
    DXTMF_OPAQUE_RESULT       = ( 1L << 28 )

} DXTMISCFLAGS;

Elements

DXTMF_BLEND_WITH_OUTPUT
Transform will perform a composite over the output surface when writing the result.
DXTMF_DITHER_OUTPUT
Transform will read data from the output surface to compute the result.
DXTMF_OPTION_MASK
Bitmask that defines transform options that can be set by the user.
DXTMF_VALID_OPTIONS
Bitmask used to test for valid option input.
DXTMF_BLEND_SUPPORTED
Transform supports color blending.
DXTMF_DITHER_SUPPORTED
Transform supports image dithering.
DXTMF_INPLACE_OPERATION
Transform will read data from the output surface to compute the result.
DXTMF_BOUNDS_SUPPORTED
Transform supports using theDXBNDS structure.
DXTMF_PLACEMENT_SUPPORTED
Transform supports using theDXVEC structure.
DXTMF_QUALITY_SUPPORTED
Transform has a quality parameter from 0 to 1 which specifies how well to render the result.
DXTMF_OPAQUE_RESULT
Transform will produce output will alpha of 255, making all samples opaque.

Remarks

The constructor of the transform sets the transform flags, depending on the kind of transform. You need to use the IDXTransform::GetMiscFlags method to find out what the transform's flags are, and then use the IDXTransform::SetMiscFlags method to change the flags. Note that you can only change the DXTMF_BLEND_WITH_OUTPUT and DXTMF_DITHER_OUTPUT flags. The other flags can only be read, as these are determined by the kind of transform.

IDXTransform::Execute, IDXTransform::SetQuality, IDXTransform::GetQuality
