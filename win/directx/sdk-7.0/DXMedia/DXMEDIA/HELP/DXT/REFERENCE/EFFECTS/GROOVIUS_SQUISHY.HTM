Groovius Transform

Procedurally animates the position and size of three-dimensional (3-D) particles using oscillation and dynamics. Parameters can be set as properties. By adjusting the position, size, and parameters of the 3-D particles, you can create a wide variety of effects.

For more information, see Copyright, Installation, and Redistribution Information.


 

Transform Specifications

The following table contains the information you need in order to use this transform in C++ and script.
