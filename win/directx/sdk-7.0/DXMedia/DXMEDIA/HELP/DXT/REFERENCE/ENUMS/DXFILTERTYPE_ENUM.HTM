DXFILTERTYPE Enumeration

Identifies a filtering algorithm to be used with the IDXTScale interface. When an image is scaled, each pixel in the destination image is mapped back to the source image to determine its color. The filter used in scaling sets the method used to determine that color. For DXFILTER_NEAREST, the color of the closest source pixel for the mapped pixel determines the color of the destination pixel. Other filters use pixels surrounding the mapped pixel to determine the destination color.

The type of filter is selected by changing the value of the Quality property with the IDXTransform::SetQuality method.

Syntax

typedef enum DXFILTERTYPE
{
    DXFILTER_NEAREST,
    DXFILTER_BILINEAR,
    DXFILTER_NUM_FILTERS
} DXFILTERTYPE;

Elements

DXFILTER_NEAREST
Color of the nearest mapped point should determine the destination color.
DXFILTER_BILINEAR
Bilinear filter should be used to determine the destination color.
DXFILTER_NUM_FILTERS
Number of filter types used for parameter validation.
