Summary of Structures

This section describes the Microsoft® DirectX® Transform structures.

DXBASESAMPLE
Describes a 32-bit pixel format, consisting of relative intensities of red, green, blue, and alpha.
DXBNDS
Describes either a continuous or a discrete boundary for a DXTransform object.
DXCBND
Describes a continuous, 32-bit boundary for a DXTransform object.
DXCBND64
Describes a continuous, 64-bit boundary for a DXTransform object.
DXDBND
Describes a discrete, 32-bit boundary for a DXTransform object.
DXDBND64
Describes a discrete, 64-bit boundary for a DXTransform object.
DXDITHERDESC
Describes the parameters used to dither a set of samples.
DXNATIVETYPEINFO
Describes the underlying native pixel format of a DXSurface object.
DXOVERSAMPLEDESC
Describes a pixel that is to be blended over another pixel.
DXPACKEDRECTDESC
Describes a rectangular array of pixels, possibly packed as a subset of a larger array.
DXPMSAMPLE
Describes a 32-bit ARGB pixel format, with all colors premultiplied by the alpha channel.
DXPtrFillInfo
describe a fill operation that should be performed on a procedural surface. It is used by the CDXBaseARGBPtr::FillSamples function..
DXRAWSURFACEINFO
Describes the properties of a user-defined raw surface image format.
DXRUNINFO
Describes the run information for a series of identical pixels.
DXSAMPLE
Describes a 32-bit pixel format, consisting of relative intensities of red, green, blue, and alpha.
DXTMTASKINFO
Describes a task to be performed by a task manager object.
DXVEC
Describes either a continuous or a discrete four-dimensional (4-D) vector.
