Transform Reference

You can use transforms included with this Software Development Kit (SDK) in your applications. For information on how these transforms are used in code, see Using Transforms in C++.

Note that to view three-dimensional (3-D) transforms, you must have Microsoft® Direct3D® Retained Mode version 6.0 (from DirectX foundation 6.0) installed on your computer. You can use the two-dimensional (2-D) transforms without Direct3D Retained Mode 6.0. However, to use some 2-D transforms, you must install the optional transform DLLs. You can do this when you install the DirectX Media SDK by choosing "Optional DirectX Transforms." These DLLs are installed by default.

For descriptions of the information contained in the Transform Reference, see the Transform Reference Guide.
