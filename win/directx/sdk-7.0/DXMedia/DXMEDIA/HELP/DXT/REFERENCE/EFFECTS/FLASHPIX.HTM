FlashPix Transform

Opens and renders (at any resolution) a local Kodak Flashpix™ file or a remote Flashpix file by using Internet Imaging Protocol (IIP).

For more information, see http://www.livepicture.com.

This transform is not part of the Microsoft® DirectX® Media Software Development Kit (SDK) core install. It is only installed if the "Optional DirectX Transforms" option is selected. This is selected by default. This particular transform is located in Dxtlipi.dll in the \Samples\Multimedia\Dtrans\Bin folder. This dynamic-link library (DLL) file can be redistributed according to the same terms as the DirectX Media 6.0 SDK redist. For information on the DirectX media redist, see the Redistribution Information.

Transform Specifications

The following table contains the information you need to use this transform in C++ and script.

Class identifier (CLSID) CLSID_LPsurface
Globally unique identifier (GUID) 201EA564-A6F6-11D1-811D-00C04FB6BD36
Programmatic identifier (ProgId) DXSurface.LivePicture.FlashPix
Category identifier (CATID) CATID_DXSurface
Custom interfaces ILPsurface
Supported interfaces IDXSurface
Inputs None.
Output None.

Custom Properties

The following table lists the custom properties that control the transform output.

Property Type Default
 Description
SourceUrl bstr "" Flashpix file URL.
TopLeft_RI_x float 0.0 A point in the Flashpix resolution-independent coordinate system that will be mapped to the pixel (0, 0) of the DXSurface object.
TopLeft_RI_y float 0.0 A point in the Flashpix resolution-independent coordinate system that will be mapped to the pixel (0, 0) of the DXSurface object.
BottomRight _RI_x float 1.0 A point in the Flashpix resolution-independent coordinate system that will be mapped to the pixel (width - 1, height - 1) of the DXSurface object.
BottomRight_RI_y float 1.0 A point in the Flashpix resolution-independent coordinate system that will be mapped to the pixel (width - 1, height - 1) of the DXSurface object.
Width long 256 Width corresponds to the width of the internal image buffer that the Flashpix image is rendered into. This value is updated to match the aspect ratio of the Flashpix image.
Height long 256 Height corresponds to the height of the internal image buffer that the Flashpix image is rendered into.
BackgroundColor BSTR "black" The background color of the Flashpix image.
