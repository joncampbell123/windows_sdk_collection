Quick Look at All Transforms

This section contains demonstrations of the transforms that ship with the Microsoft® DirectX® Transform Software Development Kit (SDK). Simply select the transform you want to see from the drop-down list, then click the Show Me button to see the effects of the various transforms. You will need to click the Stop button before selecting another transform. For information about creating these transforms, see the Transform Reference.

Note: To use three-dimensional (3-D) transforms, you must have Microsoft Direct3D® Retained Mode version 6.0 (from DirectX foundation 6.0) installed on your computer. To use two-dimensional (2-D) transforms, you must have optional DLLs installed. You can install these by choosing "Optional DirectX Transforms" when you install the DirectX Media SDK. See the Transform Reference for a complete list of transforms and what additional software, if any, is required to use each.
