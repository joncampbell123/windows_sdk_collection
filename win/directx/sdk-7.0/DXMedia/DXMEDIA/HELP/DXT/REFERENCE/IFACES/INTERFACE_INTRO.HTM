Interfaces

This section contains reference documentation for the Microsoft® DirectX® Transform interfaces and their methods. See Summary of Interfaces for a brief description of the interfaces.

The interfaces are grouped into the following categories.

DXTransform Interfaces

  • IDXBindHost
  • IDXEffect
  • IDXScaleOutput
  • IDXSurfacePick
  • IDXTransform
  • IDXTransformFactory

DXSurface Interfaces

  • IDXARGBReadPtr
  • IDXARGBReadWritePtr
  • IDXARGBSurfaceInit
  • IDXDCLock
  • IDXRawSurface
  • IDXSurface
  • IDXSurfaceFactory
  • IDXSurfaceInit
  • IDXSurfaceModifier

Utility Interfaces

  • IDXBaseObject
  • IDXLookupTable
  • IDXLUTBuilder
  • IDXTaskManager

Utility Transforms

  • IDXGradient
  • IDXTConvolution
  • IDXTScale
