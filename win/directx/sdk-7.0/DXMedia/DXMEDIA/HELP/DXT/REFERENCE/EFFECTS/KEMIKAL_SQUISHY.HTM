Kemikal Surface

Creates an image based on an interactive simulation of chemical reaction and diffusion. The image can be used as an animated procedural texture map for a three-dimensional (3-D) object, a background, or as a stand-alone moving image.

For more information, see Copyright, Installation, and Redistribution Information.


 

Transform Specifications

The following table contains the information you need in order to use this transform in C++ and script.

Class identifier (CLSID) CLSID_Kemikal
Globally unique identifier (GUID) 1A951643-0797-11D2-9F70-00A0C923140E
Programmatic identifier (ProgId) DXSurface.SquishyFX.Kemikal
Category identifier (CATID) CATID_DXImageTransform and CATID_DXSurface
Custom interfaces IKemikal
Supported interfaces None.
Inputs None; the output is determined by setting the custom properties.
Output IDXSurface

Custom Properties

The following table lists the custom properties that control the transform output.

Property Type Default Description
Height long 128.0 Height of the output image, in pixels.
Width long 128.0 Width of the output image, in pixels.
Da float 0.035125 Diffusion constant of chemical A (unclamped). Must not be negative.
Db float 0.38 Diffusion constant of chemical B (unclamped). Must not be negative.
dt float 0.57 Time delta of simulation between zero and one (clamped).
K float .12 Speed constant of simulation between zero and one (clamped).
redA float 2.5 The strength of red for chemical A. Can be negative.
greenA float 2.5 The strength of green for chemical A. Can be negative.
blueA float 10.0 The strength of blue for chemical A. Can be negative.
redB float 25.0 The strength of red for chemical B. Can be negative.
greenB float 11.0 The strength of green for chemical B. Can be negative.
blueB float 0.0 The strength of blue for chemical B. Can be negative.
command BSTR "" A string that can control the simulation. One of the following:

"reset" — Resets the simulation to equilibrium state. This is what you should call when the simulation explodes chaotically.

"addAatXY" — Adds a bit of chemical A at the given x- and y-location.

"addBatXY" — Adds a bit of chemical B at the given x- and y-location.
kemikalTime float 0.0 A number from 0.0 through 1.0 that controls the progress of the transform animation.
preset integer 1 Presets to demonstrate different custom property combinations and color balance settings. Valid values range from 1 through 6.
