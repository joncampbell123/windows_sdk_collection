DXBNDTYPE Enumeration

Identifies the data type within a DXBNDS union.

Syntax

typedef enum DXBNDTYPE
{
    DXBT_DISCRETE,
    DXBT_DISCRETE64,
    DXBT_CONTINUOUS,
    DXBT_CONTINUOUS64
} DXBNDTYPE;

Elements

DXBT_DISCRETE
Indicates the DXBNDS union is using the LONG data type.
DXBT_DISCRETE64
Indicates the DXBNDS union is using the LONGLONG data type.
DXBT_CONTINUOUS
Indicates the DXBNDS union is using the FLOAT data type.
DXBT_CONTINUOUS64
Indicates the DXBNDS union is using the DOUBLE data type.
