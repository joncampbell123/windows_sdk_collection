IDXTComposite Interface

The IDXTComposite interface is a utility that composites one image over another. It functions as a transform that requires two input images and one output image. When the IDXTransform::Execute method is called, the transform combines the two images based on one of the compositing methods described in the DXCOMPFUNC enumeration and writes the resulting image to the output. The default compositing method is DXCOMPFUNC_A_OVER_B, which simply places the first image over the second image.

This interface uses all alpha information that is defined for the input images to produce the output.

IDXTComposite Methods

get_Function Retrieves the method used for compositing the two input images.
put_Function Specifies the method used for compositing the two input images.

IDXTComposite::get_Function

Retrieves the method used for compositing the two input images.

Syntax

HRESULT get_Function(
	DXCOMPFUNC *peFunc
);

Parameters

peFunc
[out] Pointer to the currently selected compositing method as a member of the DXCOMPFUNC enumeration.

Return Value

Returns an HRESULT value that depends on the implementation of the interface.

IDXTComposite::put_Function

Specifies the method used for compositing the two input images.

Syntax

HRESULT put_Function(
	DXCOMPFUNC eFunc
);

Parameters

eFunc
[in] Compositing method to use for combining the two input images, as a member of the DXCOMPFUNC enumeration.

Return Value

Returns an HRESULT value that depends on the implementation of the interface.
