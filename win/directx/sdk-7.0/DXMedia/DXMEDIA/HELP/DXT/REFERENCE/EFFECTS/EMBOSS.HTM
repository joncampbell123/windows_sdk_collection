Emboss Transform

Produces an effect that makes the features in an image appear as an embossed texture by using grayscale values.


 

Transform Specifications

The following table contains the information you need in order to use this transform in C++ and script.

Class identifier (CLSID) CLSID_CrEmboss
Globally unique identifier (GUID) F515306D-0156-11d2-81EA-0000F87557DB
Programmatic identifier (ProgId) DXImageTransform.Microsoft.CrEmboss
Category identifier (CATID) CATID_DXImageTransform
Custom interfaces None.
Supported interfaces None.
Inputs One required of type IDXSurface.
Output IDXSurface
