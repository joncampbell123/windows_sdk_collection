IDXDCLock Interface

The IDXDCLock interface is used to lock a surface and to obtain a device context (DC) that can be used with Graphics Device Interface (GDI) calls. You can use IDXDCLock by calling the IDXSurface::LockSurface method and by specifying IID_IDXDCLock as the desired interface, or by calling the IDXSuface::LockSurfaceDC method. When the IDXDCLock interface is finally released, the underlying GDI DC will be released.

IDXDCLock can be used by any transform that allows the caller to specify the desired output bounds.

IDXDCLock Methods

GetDC Retrieves the current DC.

IDXDCLock::GetDC

Retrieves the current DC.

Syntax

HDC GetDC(void);

Return Value

Returns a handle to a device context.
