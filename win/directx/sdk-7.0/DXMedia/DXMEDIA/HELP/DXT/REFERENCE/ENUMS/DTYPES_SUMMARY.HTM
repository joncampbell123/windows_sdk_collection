Summary of Enumerated Types

This section describes the Microsoft® DirectX® Transform enumerated types.

DXBLTOPTIONS
Specifies the method used to blit an image onto a DXSurface.
DXBNDID
Identifies the axis for a specified surface boundary.
DXBNDTYPE
Specifies the data type of a surface boundary.
DXCOMPFUNC
Indicates the type of compositing operation used for the Compositor transform.
DXCONVFILTERTYPE
Indicates the type of filter used for the IDXTConvolution interface.
DXEFFECTTYPE
Describes the type of effect used for a particular DirectX Transform.
DXFILTERTYPE
Selects a filter for imgage resampling.
DXLOCKSURF
A set of flags that detemines how a surface should be locked by the IDXSurface::LockSuface method.
DXLUTCOLOR
A set of flags used to select a color for color balancing in the IDXLUTBuilder interface.
DXRUNTYPE
Describes the opacity of a run of pixels in the DXRUNINFO structure.
DXSAMPLEFORMATENUM
Describes the pixel format of a DXSurface.
DXSFCREATE
Enables the user to set special properties of DXSurfaces created with the IDXSurfaceFactory interface.
DXSURFMODCOMPOP
Describes the method used by the IDXSurfaceModifier interface for compositing operations.
DXSURFSTATUS
Describes the general properties of a DXSurface.
DXTMISCFLAGS
Describes the transform flags that specify how to generate the transform output.
