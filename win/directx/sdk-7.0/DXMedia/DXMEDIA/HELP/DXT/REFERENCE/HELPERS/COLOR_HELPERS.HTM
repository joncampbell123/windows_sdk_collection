Color Selection Helper Function

A number of transforms require you to set a color with a string of characters. This enables users to select "red" or "blue" instead of the more complicated 32-bit color value. You can use the DXColorFromBSTR helper function inside transforms to make the conversion to a 32-bit ARGB value, which can then be cast to DXSAMPLE format.

To use this function, you must include the file dxclrhlp.h from the DXMedia\include directory in your program.

Color Selection Function

DXColorFromBSTR Converts a BSTR enumerated color to a 32-bit ARGB value.

DXColorFromBSTR

Converts a BSTR enumerated color to a 32-bit ARGB value.

Syntax

HRESULT WINAPI DXColorFromBSTR(
	BSTR Color,
	DWORD *pdwColor
);

Parameters

Color
[in] String that contains either an enumerated color word, a 24-bit RGB color string of the form "#RRGGBB", or a 32-bit ARGB color string of the form "#AARRGGBB".
pdwColor
[out] Pointer to a 32-bit converted color value indicated by the Color string.

Return Value

Returns an HRESULT that indicates whether the color conversion was successful.
