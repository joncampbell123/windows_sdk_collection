Gradient Transform

Produces a horizontal or vertical linear gradient, which is a gradual transition from a StartColor to an EndColor. You can also specify the size of the output image.

Transform Specifications

The following table contains the information you need in order to use this transform in C++ and script.

Class identifier (CLSID) CLSID_DXTGradientD
Globally unique identifier (GUID) 623E2882-FC0E-11D1-9A77-0000F8756A10
Programmatic identifier (ProgId) DXImageTransform.Microsoft.Gradient
Category identifier (CATID) CATID_DXImageTransform
Custom interfaces IDXTGradientD
Supported interfaces None.
Inputs None.
Output IDXSurface

Custom Properties

The following table lists the custom properties that control the transform output.

Property Type Default Description
GradientType DXGRADIENTTYPE DXGRADIENT_VERTICAL The direction of the gradient; can be DXGRADIENT_VERTICAL or DXGRADIENT_HORIZONTAL.
StartColor OLE_COLOR 0xFF0000FF Starting color for the gradient in ARGB32 format. For horizontal gradients, the StartColor is on the left of the image. For vertical gradients, the StartColor is on the top. in ARGB32 format.
EndColor OLE_COLOR 0xFF000000 Ending color for the gradient in ARGB32 format. For horizontal gradients, the EndColor is on the right of the image. For vertical gradients, the EndColor is on the bottom.
GradientWidth long 1024 Width of the gradient, in pixels.
GradientHeight long 768 Height of the gradient, in pixels.
KeepAspectRatio Boolean FALSE Indicates whether the ratio between the height and width of the image should be preserved.

Remarks

There is also an IDXGradient interface that is exposed on the gradient transform which can only be used from C++.
