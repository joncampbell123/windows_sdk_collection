Using Transforms

This section discusses how to use a Microsoft® DirectX® Transform in a Microsoft Windows® application. This Software Development Kit (SDK) provides many prewritten images and three-dimensional (3-D) mesh transforms, such as Explode, Blur, Fade, and Ripple. You can use these prewritten transforms in your applications with only a few additional method calls. You can expand your library of transforms by obtaining them from other transform authors or by writing your own transforms with this SDK.
