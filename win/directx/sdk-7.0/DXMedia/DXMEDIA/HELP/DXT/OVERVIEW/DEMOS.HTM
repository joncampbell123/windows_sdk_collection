DirectX Transform Demos

These HTML files show you how to use transforms with Microsoft® DirectAnimation™. The simple demos show a single transform. The showcase demos show several transforms or enable you to define the parameters of the transform. You can launch a Window and display a demo by clicking on its name.

  • Simple DirectAnimation Transform Demos
  • Showcase DirectAnimation Transform Demos

Note: To use three-dimensional (3-D) transforms, you must have Microsoft Direct3D® Retained Mode version 6.0 (from DirectX foundation 6.0) installed on your computer. To use two-dimensional (2-D) transforms, you must have optional DLLs installed. You can install these by choosing "Optional DirectX Transforms" when you install the DirectX Media SDK. See the Transform Reference for a complete list of transforms and what additional software, if any, is required to use each.

Simple DirectAnimation Transform Demos

The following table describes the DirectAnimation transform demos that show a single transform. You can launch a window displaying the demo by clicking the name of the demo.
Demo NameDescription
Wipe Generates an output image that wipes from one input image to another. You can set the width of the transition region between the two inputs, which is a dual alpha ramp. In the transition region, the images are blended.
Ripple Creates a 2-D, distorted rippling effect (like waves from a pebble dropped into a pond), blending from one image to another image underneath.
ExplodingSailboat Generates a 3-D explosion effect using a sailboat model. The faces of the 3-D mesh rotate and tumble as they move outward in the direction of their normal.

Showcase DirectAnimation Transform Demos

The following table describes the DirectAnimation transform demos that display a whole range of transforms or enable you to choose transform features.
Demo NameDescription
MetaImage Transforms Displays many of the MetaCreations transforms and enables you to view a transform's performance as it plays.
MetaStream Demonstrates the progressive download of 3-D scenes in MetaStream format from a URL that you specify. The transform converts models from MetaStream format to Direct3D Retained Mode mesh format, as needed, enabling progressive download of data. You can choose different 3-D models and set the model quality.
FlashPix Demonstrates panning and zooming on a Kodak Flashpix™ file while maintaining the same resolution, demonstrates streaming data via NetShow™, and demonstrates opening and rendering a local Flashpix file or a remote Flashpix file by using Internet Imaging Protocol (IIP) at any resolution.
Squishy® GeomGroov Wizard Demonstrates how to procedurally animate a 3-D object's position using oscillation and dynamics. You can set different speeds, dimensions, and other parameters and see the effects.
Squishy® Groovius Wizard Demonstrates how to procedurally animate the position and size of 3-D particles using oscillation and dynamics. You can set different speeds, dimensions, and other parameters and see the effects.
Squishy® Kemikal Wizard Demonstrates how create an image based on an interactive simulation of chemical reaction and diffusion. You can set color balance, diffusion, and other parameters and see the effects.
