Writing Transforms

Microsoft® DirectX® Transform enables you to produce effects on images and meshes. The application programming interface (API) provides sophisticated graphics tools for modifying and combining images. These tools include scaling, dithering, compositing, alpha blending, image filtering, and many others. You can use transforms in your own image-based applications. In addition, transforms contained in dynamic-link libraries (.dll files) can be used by Microsoft DirectAnimation™ and Microsoft Chromeffects™ interactive media technology to produce image effects on Web pages. For more information about Chromeffects, see http://www.microsoft.com/windows/chromeffects. For more information about DirectAnimation, see the DirectAnimation SDK.
