Reference

The Microsoft® Direct3D® Retained Mode reference provides detailed information about the interfaces, methods (interface member functions), nonmember functions, and callback functions, with syntax, parameter descriptions, and flag descriptions. It also describes the members of Direct3D Retained Mode structures, enumerated types, other defined types, and the possible error values that can be returned.
