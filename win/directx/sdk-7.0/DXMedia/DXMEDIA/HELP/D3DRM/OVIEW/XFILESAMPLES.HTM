DirectX File Samples

The following samples demonstrate loading from, converting to, and saving to the Microsoft® DirectX® file format. These samples come with the DirectX Foundation 6.0 Software Development Kit (SDK) and can be found in folders beneath the Mssdk\Samples\Multimedia\D3drm\Src subdirectory.

  • CONVXFMT
  • XOFLOAD
  • XOFSAVE
  • XOFUSER

CONVXFMT

Path

Mssdk\Samples\Multimedia\D3drm\Src\Convxfmt

Description

Converts from one DirectX file format to another.

XOFLOAD

Path

Mssdk\Samples\Multimedia\D3drm\Src\Xofload

Description

Loads a file using the Direct3D Retained Mode templates.

XOFSAVE

Path

Mssdk\Samples\Multimedia\D3drm\Src\Xofsave

Description

Saves a file using the Direct3D Retained Mode templates.

XOFUSER

Path

Mssdk\Samples\Multimedia\D3drm\Src\Xofuser

Description

Saves a file using user-defined templates.
