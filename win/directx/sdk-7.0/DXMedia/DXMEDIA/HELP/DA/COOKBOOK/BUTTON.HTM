Button

This example displays a button that, when clicked, goes to the Microsoft home page. To make the button, we first draw a square and make it pickable, using the Pickable function. This means the image recognizes when the mouse is within its bounding box. The blue button turns red when this happens. On a left-button click, the button transfers you to the Microsoft home page. This is done with the ScriptCallback function.
