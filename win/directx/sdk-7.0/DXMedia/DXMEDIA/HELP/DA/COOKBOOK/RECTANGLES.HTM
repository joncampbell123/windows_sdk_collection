Rectangles

This example demonstrates how to create a rectangle and turn it into an image that can be rendered. An animated rectangle with a width that varies over time also is shown. Finally, a rectangle is used as a path rather than as an image. A line of text traces the path. (This also demonstrates the FollowPath function.)
