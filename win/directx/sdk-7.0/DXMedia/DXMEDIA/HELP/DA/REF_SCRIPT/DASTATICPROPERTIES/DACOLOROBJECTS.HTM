Aqua

DAStatics Class

A read-only property with a DAColor value that is always aqua. In a range of [0, 1], its red, green, and blue intensity values are 0.00, 1.00, 1.00. In a range of [0-255], its values are 0, 255, 255.

Syntax

lib.Aqua
