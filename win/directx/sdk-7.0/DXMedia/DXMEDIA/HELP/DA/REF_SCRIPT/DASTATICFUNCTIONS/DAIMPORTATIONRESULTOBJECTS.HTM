DAStatics Functions Relevant to DAImportationResult Objects

ImportMovieStream

DAStatics Class

Creates a DAImportationResult object from the movie file at the given URL. The movie will play (stream) as it downloads.

Syntax

lib.ImportMovieStream( url )

Parameters

url
String (BSTR) that contains the full path to the file; for example, http://business.com/media/movie.mp2.

Return Value

Returns the DAImportationResult object that is the video stream from the movie file. The DAImportationResult object has these relevant properties:

  • CompletionEvent, which is the event that triggers when the streaming is complete.
  • Duration, which is the length of the movie, in seconds.
  • Image, which is the DAImage value of the movie being streamed.
  • Progress, which is the percentage of the download that has completed.
  • Size, which is the size of the imported movie, in bytes.

Remarks

The movie can be one of the following formats: .avi, .mpg., .mpeg, .mpv, .mp2, .mpe, .mov, or .qt.

ImportSoundStream

DAStatics Class

Creates a DAImportationResult object from the sound file at the given URL. The sound will play (stream) as it downloads.

Syntax

lib.ImportSoundStream( url )

Parameters

url
String (BSTR) that contains the full path to the file; for example, http://business.com/media/sound.wav.

Return Value

Returns the DAImportationResult object that is the audio stream from the movie file. The DAImportationResult object has these relevant properties:

  • CompletionEvent, which is the event that triggers when the streaming is complete.
  • Duration, which is the length of the sound, in seconds.
  • Progress, which is the percentage of the download that has completed.
  • Size, which is the size of the imported sound, in bytes.
  • Sound, which is the DASound value of the sound being streamed.

Remarks

The sound can be .mid or .wav format.
