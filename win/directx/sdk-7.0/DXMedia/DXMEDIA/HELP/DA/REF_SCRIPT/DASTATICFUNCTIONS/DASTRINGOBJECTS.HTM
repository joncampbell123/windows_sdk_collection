DAStatics Functions Relevant to DAString Objects

ConcatString Creates a DAString object representing the concatenation of the given DAString objects.
DAString Converts a string to a DAString object.
ModifiableString Creates a DAString object whose value can be switched with the DABehavior function SwitchToString.

ConcatString

DAStatics Class

Creates a DAString object representing the concatenation of the given DAString objects. The new object's value is the character string consisting of the value of b appended to the end of the value of a.

Syntax

lib.ConcatString( a, b )

Parameters

a and b
DAString objects to concatenate.

Return Value

Returns the DAString object.

DAString

DAStatics Class

Converts a string to a DAString object. The only time DAString is used is when converting strings to their corresponding DAString objects.

Syntax

lib.DAString( string )

Parameters

string
String to be converted.

Return Value

Returns the DAString object.

ModifiableString

DAStatics Class

Creates a DAString object whose value can be switched with the DABehavior function SwitchToString.

Note this function can be used only with DirectAnimation™ version 6.x or later.

Syntax

lib.ModifiableString( initVal )

Parameters

initVal
The initial value of the new modifiable DAString object.

Return Value

Returns the new modifiable DAString object.

Remarks

Used with the DABehavior function SwitchToString, this function makes it easier to change the value of a DAString object.
