PlayState Property

Path Control

Retrieves the path's playback state: stopped, playing, or paused. Read-only at run time.

Syntax

Scriptingstate=pathObj.PlayState

Possible Values

state
Playback state of the Path object; 0 if stopped, 1 if playing, 2 if paused.
