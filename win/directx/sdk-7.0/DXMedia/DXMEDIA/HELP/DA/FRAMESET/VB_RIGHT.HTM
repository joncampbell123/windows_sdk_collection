Welcome to the DirectAnimation Visual Basic Samples

The DirectAnimation Visual Basic Samples were created with the DirectAnimation Windowed Control.

The source code for the DirectAnimation Visual Basic Samples is located in the samples/multimedia/danim/visualbasic/showcase directory.

Please Note: The Ripple sample uses a DirectX Transform effect, which requires Microsoft® DirectX® version 6.x or later. You must have DirectX 6.x or later installed for this sample to work.
