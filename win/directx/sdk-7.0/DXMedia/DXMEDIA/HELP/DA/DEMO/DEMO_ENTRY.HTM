Interactive Script Demos

This section contains interactive script demos for Microsoft® DirectAnimation™. With the demos, you make choices to animate images and geometries in different ways, to change their properties, or to apply Microsoft DirectX® transforms to them. As you make changes to the object shown, the corresponding Microsoft JScript® code appears with the code you have created or changed in bold. You can cut and paste the code into your own HTML pages.
