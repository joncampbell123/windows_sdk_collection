Select DXTransform:

DirectX Transform Effects

Select different DirectX® Transform effects, apply them to DirectAnimation™ objects, and see the result. If the effect is a 3-D transform, it is applied to a sphere. If the effect is a 2-D transform, it is applied to an image. Note: To use 3-D transforms, you must have Microsoft® DirectX® version 6.x or later installed on your computer. To use some 2-D DirectX Transform effects, you must have optional DLLs installed. You can install these by choosing "Optional DirectX Transforms" when you install the DirectX Media SDK. See the DirectX Transform documentation for a complete list of DirectX Transform effects and what additional software, if any, is required to use each. Show more information.

DirectX® Transform supplies many 2-D and 3-D effects that can be applied to DirectAnimation™ 2-D and 3-D objects. You access the effects through new ActiveXObject("type.creator.name");, where type is the type of transform, creator is the company that created it, and name is the name of the effect. If the effect is a 2-D transform the type is DXImageTransform, for example, xf=new ActiveXObject("DXImageTransform.Microsoft.Compositor");. If the effect is 3-D, the type is DX3DTransform, for example, xf=new ActiveXObject("DX3DTransform.Microsoft.Explode");
