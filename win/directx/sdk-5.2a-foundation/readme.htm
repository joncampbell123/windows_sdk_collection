Welcome to the DirectX Foundation and Media SDK! With this CD, you can install both the DirectX Foundation and DirectX Media software development kits. This CD also contains the latest version of Microsoft Internet Explorer. Please see the Installation section to install each component. You may also want to see the release note located in each SDK's directory on this CD.

Release notes

The release note for the DirectX Foundation SDK and DirectX Media SDK are available in each SDK directory.

To view the DirectX Foundation SDK release note, click here or do the following:

  • Open the "DirectX 5.2a SDK" folder, then open the "DirectX foundation" folder. Double click the icon labeled "readme".

To view the DirectX Media SDK release note, click here or do the following:

  • Open the "DirectX 5.2a SDK" folder, then open the "DirectX media" folder. Double click the icon labeled "readme".

License agreement

You should also review the licensing agreement for each SDK.

To view the DirectX Foundation SDK license agreement, click here or do the following:

  • Open the "DirectX 5.2a SDK" folder, then open the "DirectX foundation" folder, then open the "license" folder. Double click the icon labeled "eula".

To view the DirectX Media SDK license agreement, click here or do the following:

  • Open the "DirectX 5.2a SDK" folder, then open the "DirectX media" folder. Double click the icon labeled "license".

Installation

To install the DirectX Foundation SDK do the following:

  • Change directory to "D:\DirectX 5.2a SDK\DirectX foundation" (where D:\ is your CDROM drive) and run setup.

To install the DirectX Media SDK do the following:

  • On the X86 platform, change directory to "D:\DirectX 5.2a SDK\DirectX media\x86" (where D:\ is your CDROM drive) and run setup.exe.
  • On the Alpha platform, change directory to "D:\DirectX 5.2a SDK\DirectX media\Alpha" (where D:\ is your CDROM drive) and run setup.exe.

To install Microsoft Internet Explorer, do the following:

  • On the X86 platform, change directory to "D:\Internet Explorer\x86" (where D:\ is your CDROM drive) and run Ie4setup.exe.
  • The Alpha version of Internet Explorer is not included in this SDK. Please visit the Microsoft Internet Explorer site (http://www.microsoft.com/ie/default.htm) to obtain the Alpha version.
  • To install the Japanese language version of Internet Explorer, run Ie4setup.exe from "D:\Internet Explorer\x86jpn" (where D:\ is your CDROM drive).
