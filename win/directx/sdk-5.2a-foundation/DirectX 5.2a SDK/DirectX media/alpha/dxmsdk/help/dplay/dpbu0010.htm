DirectPlay Reference


The DirectPlay Reference provides detailed information about functions, callbacks, and the IDirectPlay3 and IDirectPlayLobby2 interface methods, with syntax, parameter descriptions, flag descriptions, return values, and comments about how they are used. It also describes the members of DirectPlay structures, system messages, lobby messages, and defined properties, and the possible error values that can be returned.
