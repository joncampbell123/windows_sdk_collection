========================
= DirectX 5 SDK Readme =
========================

==============================
= DirectX 5.2 SDK の追加事項 =
==============================
DirectX 5.2 SDK は、DirectX 5 SDK のアップデート版です。更新された内容について
は、このファイルの DirectX 5.2 リリースノートを参照してください。また、弊社
DirectXウェブサイト、http://www.microsoft.com/directx には常に最新の情報が掲載
されていますので、あわせてご覧ください。（日本語サイトの URL は、
http://www.microsoft.com/japan/directx です。）


DirectX SDK または DirectX をインストールするには、ルートディレクトリから
SETUP.EXE を実行してください。

概要
----
DirectX 5 SDK をフルインストールすると、以下のようなファイルがハード
ディスク上に作成されます。

DOCS     - 各 DirectX コンポーネントのヘルプと Readme ファイル
FOXBEAR  - Fox & Bear DirectDraw/DirectSound サンプルデモ
ROCKEM   - Direct3D サンプル デモ
SDK      - DirectX SDK.  サンプルコード、ライブラリ、インクルードファイル、および
           デバッグ版 DirectX のコンポーネントが含まれています。

上記以外にも、以下のようなディレクトリがあります。これらは、ハードディス
クにはインストールされません。

DEBUG    - デバッグ版の DirectX コアコンポーネント
EXTRAS   - サードパーティのドライバ、デバッグツールおよびその他の有用なファイル
LICENSE  - 使用許諾契約書
REDIST   - DirectX 再頒布可能コンポーネント
SAMPGAME - サンプルゲーム
 
各コンポーネント (DirectDraw, Direct3D, DirectSound, DirectPlay および
DirectInput) の情報は、DOCS\README の中のファイルを参照してください。

DirectX 5.2 リリースノート
--------------------------
1) DirectX 5.2 には、DirectX 5 SDK のリリース以降に発見され修正された問題や
   変更が含まれています。以下は、その一覧です。

   o DirectX を各国語の Windows 95 にインストールする際に発生する問題の修正。
       DirectX でサポートされていない言語の Windows 95 にインストールすることが
       できませんでした。また、国別設定を変更すると、正しい言語バージョンのドラ
       イバをインストールすることができませんでした。
   o NEC PC-98シリーズで、デジタルジョイスティックが使用できなかった問題の修正。
       特定の NEC PC-98コンピュータ上で、デジタルジョイスティックが動作しないこ
       とがありました。
   o サウンドドライバのアップデートの問題の修正。
       DirectSound をサポートしていない、古いドライバをアップグレードしないとい
       う障害がありました。
   o 中国語版（台湾版）の Windows 95 にインストールする際の問題の修正。
       特定のハードウェアで、DirectX 5 をインストールすると、ブート時に、GPF が
       発生することがありました。
   o 更新された Dxbug.exe および Dxinfo.exe. 
       これらの更新されたツールは、誤ってインストールされた DirectX のコンポー
       ネントを検出します。

2) フィンランド、デンマーク、および ハンガリー版のドライバを追加しました。これら
   の新しいドライバは、既存のものと同様、再頒布することができます。


リリースノート
--------------
1) EXTRAS ディレクトリには、有用なファイルが収められています。
  
2) この SDK は、x86 用 Windows NT 4.0 およびそれ以降のシステムにインストール
   することができますが、DirectX のラインタイム コンポーネントはインストール
   されません。
   Windows NT 4.0 には、DirectX 2 相当の機能があらかじめ組み込まれています。
   DirectX 3 相当の機能は、サービスパック 3 (SP3)により組み込まれます。

3) Watcom コンパイラを使用する場合、最新のものを使用してください。最新版は、
   DirectX のサンプルをコンパイルするために必要な、最新の Windows ヘッダファ
   イルが含まれています。以前含まれていた \sdk\samples\watinc ディレクトリ
   （これらのヘッダファイルが含まれていました）は、今回のリリースでは削除さ
   れていますので、常に最新の Watcom ヘッダファイルを入手するようにしてくだ
   さい。

4) Direct3D と DirectSound3D は、Pentium クラスの CPU で動作するように設計さ
   れています。これらのコンポーネントは、浮動小数点演算を使用するため、浮動
   小数点コプロセッサを持たないマシンでは性能が著しく低下します。これらのコ
   ンポーネントを使用する場合、そのアプリケーションが、486（特に 486SX）上で
   問題ない速度で動作するか確認してください。もし問題があるようならば、例え
   ば、セットアップ時に CPU を検出し、これらのマシン上では動作しないようにす
   るような処理を行うことをお勧めします。

5) ATI-TV チューナをご利用の場合、DirectX ドライバのインストールにより、TV 
   チューナと MPEG ビューワ機能が無効になります。この場合、DirectX のインス
   トール後、ディスプレイドライバの復元を行う必要があります。コントロールパ
   ネルから[アプリケーションの追加と削除]を選び、[DirectXドライバ]を選択し、
   [ディスプレイドライバの復元]を選択してください。

6) DirectX セットアップは、ATI 社の Rage 1/2 用 OEM ドライバを復元することが
   できないことがあります。このような場合、元の OEM ドライバを以下の手順で再
   度インストールしてください。
      コントロールパネルから[画面]を選択し、[画面のプロパティ]を表示します。
      次に、[ディスプレイの詳細]タブをクリックします。次に、[詳細プロパティ]
      をクリックし、[変更]をクリックし、元の OEM ドライバを再度インストールし
      ます。

7) NEC PC-9821Ct20をお使いのお客様は、DirectX 5 のインストール時に、ディスプ
   レイ ドライバのインストールを推奨するダイアログ ボックスが、複数回表示され
   ることがあります。このような場合、これらすべてのダイアログ ボックスで、
   [はい]をクリックしてください。[いいえ]をクリックすると、システム再起動時に、
   すべてのディスプレイ ドライバが使用不可になる場合があります。この場合、シス
   テム再起動時に、「ﾃﾞｨｽﾌﾟﾚｲｱﾀﾞﾌﾟﾀは使用不可にされています。[OK]を押してﾃﾞﾊﾞｲｽ
   ﾏﾈｰｼﾞｬを開始し、問題を解消してください。」というメッセージが表示されますので、
   以下の手順でディスプレイ ドライバを使用可能にします。
      [ﾃﾞﾊﾞｲｽ ﾏﾈｰｼﾞｬ]ダイアログ ボックスの[ﾃﾞｨｽﾌﾟﾚｲ ｱﾀﾞﾌﾟﾀ]の項目から使用可能
      に設定するディスプレイ ドライバを選択し、[ﾌﾟﾛﾊﾟﾃｨ]をクリックします。次に、
      [ﾃﾞﾊﾞｲｽを使用]チェックボックスをオンにし、[OK]をクリックします。次に、シ
      ステム再起動のメッセージで[はい]をクリックし、システムを再起動します。

8) NEC PC-9821Cu13, Cu16, Ct20をお使いのお客様は、DirectX 5 のディスプレイドライ
   バをインストール後に、98TV アプリケーションの画面が表示されなくなる場合があり
   ます。その場合は、お客様のパソコンに入っている98TV 関連モジュールのアップデー
   トが必要です。アップデートモジュールの入手方法については、PC-9821Cu13, Cu16, 
   Ct20に添付されている説明書「CanBeを使いこなそう」の「ソフトウェア最新情報のお
   知らせ」をご覧ください。
