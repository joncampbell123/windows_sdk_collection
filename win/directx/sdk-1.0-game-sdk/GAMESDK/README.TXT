===================
= Game SDK Readme =
===================

To install the Game SDK and/or DirectX, please run SETUP.EXE from the root 
directory.

Overview
--------
Welcome to the Windows 95 Game SDK.   If you did a full installation, you
will find a number of directories:

DOCS     - readme files for each of the DirectX components
DRIVERS  - audio & display drivers
FOXBEAR  - fox & bear sample demo of DirectDraw/DirectSound
HELP     - help files
IKLOWNS  - Immortal Klowns: demo of DirectDraw, DirectSound and DirectPlay.
REDIST   - the DirectX redistributables.
ROIDS    - another DirectDraw/DirectSound demo.
SDK      - Game SDK.  Contains sample code, libraries, include files,
           and debug versions of the DirectX Components
SYSTEM   - contains retail binaries of the DirectX Components
SAMPGAME - contains sample game including DirectX installation.

Please see the readme file for each component (DirectDraw, DirectSound, and 
DirectPlay) for more information.   These readme files can be found in
the DOCS directory.
