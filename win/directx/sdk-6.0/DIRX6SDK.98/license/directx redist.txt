DirectX Software Development Kit Redistributable Code
-------------------------------------------------------------
Redistributable Code is identified as the following files
and all of the files in the following directories on the
DirectX 6 SDK CD:

Core Directories:

All files in the following directories must be included
in any distribution of Redistributable Code:

redist/directx6/
redist/directx6/directx

The following directories can be optionally included in your
distribution of the Redistributable Code.  If any of these
directories are included, the English (eng) directory must
be included, or driver installation errors will result.

redist/directx6/directx/drivers
