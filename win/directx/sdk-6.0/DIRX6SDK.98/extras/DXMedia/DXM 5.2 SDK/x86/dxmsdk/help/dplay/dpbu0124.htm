DirectPlay address
A data format that DirectPlay uses to pass addressing information between lobby servers, applications, DirectPlay, and service providers. A DirectPlay address is a sequence of variable-length data chunks tagged with a globally unique identifier (GUID) that supplies all the address information needed by DirectPlay. Examples of address information are: the network address of a server, the network address of a player, the e-mail name of a player, and the network address of a session.
DirectPlay defined property
A property that can be set and retrieved by applications from a lobby. When a DPLMSG_GETPROPERTYRESPONSE message is received, its guidPropertyTag field identifies the property, and its dwPropertyData field contains the property information.
