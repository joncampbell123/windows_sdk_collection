Filter Categories


If you use the GraphEdt sample tool that comes with the DirectShow™ SDK, and choose Insert Filters from the Graph menu, you will notice that the DirectShow filters on the system are grouped into categories. The DirectShow Filters category is the generic category that most filters fall into. New filters will be registered in this category by default. Graphedt does not display empty categories.

However, if your filter is one of the following kinds of filters, you should make sure that your filter appears in the proper category:

  • Audio Capture Sources - generate audio data from an external device.
  • Audio Compressors - compress PCM audio.
  • Audio Renderers - play audio a data on a local set of speakers.
  • Midi Renderers - play MIDI data on a local set of speakers.
  • Video Capture Sources - generate video data from an external device.
  • Video Compressors - turn RGB video into compressed video.

How to register your filters by category is described in the AMovieDllRegisterServer2 topic, where sample code is shown for registering a Video Compressor filter. The AMCap sample shows how to enumerate a special category. The VidCap sample demonstrates registering a filter in a special category. See Register DirectShow Objects for additional information about registering.
