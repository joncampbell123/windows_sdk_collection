------------------------------
INSTALLATION AND CONFIGURATION
------------------------------
Install IPEAK using the following procedure:

 1. Boot Windows on your system.

 2a. If you have an IPEAK CD, insert media in your
    CD-ROM drive.  The setup.exe program will run
    automatically if Autoplay is enabled.

 2b. If you've downloaded IPEAK as a ZIP* file, 
    un-zip the file into a temporary directory and run
    the setup.exe program.

31 March 1998 - Known Issues / Errata
=====================================

* Microsoft HellBender and Microsoft Monster Truck Madness should be exited
  using Alt-F4, not the <escape> key.

* Microsoft Internet Explorer and Netscape Navigator do not cause a system
  hang when used with IPEAK GPT.  However, instrumentation of these products
  is not supported in the current release.  These products are expected to 
  be supported in an upcoming patch.

  ==================================

  Platform
  ========
  CPU Type:
  CPU Speed:
  RAM:
  Free hard disc space:
  Operating system:
  Operating system version:

  Graphics Controller
  ===================
  Type of graphics card:
  Using DirectDraw?
  Using Direct3D RM?
  Using Direct3D IM?
  Driver version:

and attempt re-production of the problem.  If you don't know, 
please specify that as well.  Please also add anything else you 
think would be useful for us to know about the system and/or 
application to reproduce this problem.
