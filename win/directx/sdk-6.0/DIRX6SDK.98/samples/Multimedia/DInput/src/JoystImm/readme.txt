Description
===========
  The JoysImm program obtains and displays joystick data.

Path
====
  Source: Mssdk\Samples\Multimedia\DInput\Src\JoystImm

  Executable: Mssdk\Samples\Multimedia\DInput\Bin

User's Guide
============
  Observe how the displayed data changes when you move and twist the stick, 
  rotate the throttle wheel, and press buttons in various combinations.

Programming Notes
=================
  The application polls the joystick for immediate data in response to a timer 
  set inside the dialog procedure.
