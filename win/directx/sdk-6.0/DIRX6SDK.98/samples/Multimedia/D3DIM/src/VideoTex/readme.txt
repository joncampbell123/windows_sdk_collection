  The VideoTex sample shows how to use an AVI file as a texture map.

Path
====
  Source: Mssdk\Samples\Multimedia\D3dim\Src\VideoTex

  Executable: Mssdk\Samples\Multimedia\D3dim\Bin

User's Guide
============
  Press F1 to see a list of the usual commands. 

Programming Notes
=================
  The program draws a cube with an AVI texture mapped to each of its faces. The 
  key is to have the texture's surface use the DDSCAPS2_HINTDYNAMIC flag.
