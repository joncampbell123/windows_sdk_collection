Description
===========
  The Memtime sample program is intended to be a demonstration of the 
  advantages and disadvantages of various drawing methods. It runs a series of 
  tests of memory bandwidth and displays a report.

Path
====
  Source: Mssdk\Samples\Multimedia\DDraw\Src\MemTime

  Executable: None.

User's Guide
============
  Choose Time All from the main menu to begin the tests. Note that your screen 
  will change resolution during the tests. It will also go blank for short 
  periods of time.

Programming Notes
=================
  The sample is not intended as an example of particular techniques in 
  DirectDraw programming. Of more interest are the results of the memory tests.
