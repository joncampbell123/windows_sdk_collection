Description
===========
  The DDEx2 program is an extension of DDEx1 that adds a bitmap.

Path
====
  Source: Mssdk\Samples\Multimedia\DDraw\Src\Ddex2

  Executable: Mssdk\Samples\Multimedia\DDraw\bin

User's Guide
============
  DDEx2 requires no user input. Press F12 or ESC to quit the program.

Programming Notes
=================
  DDEx2 shows how to set a palettized video mode. Routines in DDutil.cpp load a 
  bitmap file and copy it to a DirectDraw surface.
