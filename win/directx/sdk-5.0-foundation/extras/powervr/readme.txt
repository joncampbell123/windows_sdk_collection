NEC/VIDEOLOGIC POWERVR DRIVER
-----------------------------
INTRODUCTION
This directory contains a third-party Windows 9x display driver for the PowerVR PCX1
and PCX2 chips. It is provided as a convenience to software vendors only, for use during
product development. This driver has not been tested by Microsoft for compatibility. 

INSTALLATION INSTRUCTIONS
1) Right-click on the desktop and select "Properties".
2) In the "Display Properties" dialog, click on the "Settings" tab.
3) Click on "Change Display Type..." (Win95 Gold) or "Advanced Properties" (Win95 OSR2).
4) Under "Adapter", click "Change".
5) Select "Have Disk".
6) Enter the path of this directory and click "OK"

NOTE (Japan only)
These driver binaries will work on both PC-AT and PC98 systems.
