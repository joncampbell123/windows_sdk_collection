Uses various file I/O system calls to test the CD-ROM file
system.
Note: the HCT/DDK CD is necessary to run this test.

PARAMETERS

Test Drive
    Provide the drive letter of the CD-ROM you'd like to test,
    with no trailing '\'.
File Time/Date
    The second and third parameters are the file date/time of
    the files on the HCT CD.  Note that these will be picked up
    by the test manager when you start the test, aslong as the
    correct CD is in the drive.
