Uses multiple threads to read/write/verify a 1mb pattern to
stress simultaneous floppy, CD and hard disk access.

PARAMETERS

Partition
    Provide the logical drive letter and number of I/O threads
    in the format D:N.

CD-ROM
    Same as above.

Floppy Drive
    Same as above.  At most only four I/O threads can be used
    on floppy drives.
