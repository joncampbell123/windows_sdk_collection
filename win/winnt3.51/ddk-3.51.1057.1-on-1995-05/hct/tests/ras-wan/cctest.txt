Usage: cctest <machine ID> -tfsirbpcnNClLDvSPzo

              <Usage> must be a unique name or * (to use the computername)
              -t <destinations>
              -f <filename>
              -s <file size>
              -i <increment number>
              -r <file size range>
              -I increment buffer size by 1
              -D decrement buffer size by 1
              -b <transmit buffer size range>
              -p <pattern>
              -c [activate continue mode]
              -n <number of times>
              -N <number of cycles>
              -C <your comment>
              -l <log file>
              -L [no log output for compare PASS]
              -d [activate debugging mode]
              -v [disable VIO on screen]
              -S [stop options file upon error]
              -P [output error log to PRN:]
              -z <time in seconds to sleep>
              -V <variation number (0..9)>
              -o <options file>
