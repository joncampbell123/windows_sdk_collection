CPSTRESS.BAT copies a file to the given drive, copies 
it back to the local disk, and then compares the original
file with the new one.  Results are recorded in a user 
specified log file.

CPSTRESS.BAT expects the following parameters to run:

cpstress   name_of_the_source_file  destination_drive
	   name_of_destination_file  logfile

To run the Test:

(1) Make a RAS connection to the network
(2) NET USE to a server on the ras server 
    network segment and assign it a drive letter.
(3) Run CPSTRESS.BAT

Note:  Use file sizes ranging from 10K to 5M Bytes.
