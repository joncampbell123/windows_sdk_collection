FTPTEST.BAT does a file copy over TCP/IP. It uses the FTP 
command PUT to copy a file to the server and GET to download from the server.  The test than compares the uploaded and download file with the original source.

It uses anonymous account to log on to FTP server. The FTP
server used for this test should have anonymous accounts 
enabled before you can run this test.

To run the test: 

(1) Modify SCRIPTX file

	Change the IP address to the IP address 
	of the destination FTP server.

	Change the file names in the SCRIPTX file
	to the desired source and new file.

(2)	Change the file names in FTPTEST.BAT
	to the desired source and new file.


(3) Connect to the network via RAS (using graphical
    phone book or rasdial command)

(4) Run FTPTEST.BAT

Note:  Use file sizes ranging from 10K to 5M Bytes.
