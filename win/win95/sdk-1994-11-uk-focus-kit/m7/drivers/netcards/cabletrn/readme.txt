This directory contains 32-bit netcard drivers for the Cabletron E20XX
and E21XX Ethernet cards only.

To use, please copy the netcable.inf file to your windows\inf directory.
When setup prompts you for the location of the xxxx.sys file, point it to
this directory.  Or, copy the xxxx.sys files to your windows\system
directory yourself.

Enjoy.  These files should give you a moderate performance increase.
