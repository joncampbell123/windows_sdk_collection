The \VIPER directory contains updated versions
of the Windows 3.1 drivers for Diamond Viper
VLB and PCI display adapters.  This driver corrects
GP fault problems with this Chicago release.

To install this update, copy the driver files 
from the appropriate directory into your Chicago
\SYSTEM directory, over the versions already there.
Restart Chicago for the changes to take effect.

See the release notes for other known problems with
the Diamond Viper drivers under Chicago.
