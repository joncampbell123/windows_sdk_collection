The \Drivers\Display\Matrox directory contains
an alpha version Win95 driver for Matrox MGA 
display adapters.

To install:

Go to the Display control panel Settings page.
Click "Change Display Type".
Click "Change" in the Adapter Type section.
Click "Have Disk".
Enter the path to the MATROX.INF file in this directory.
Select "Matrox MGA", and click OK.

Click "OK" if an error message appears about a
"transient error".

After the files are copied, select desired display
settings, and restart your computer when prompted.
