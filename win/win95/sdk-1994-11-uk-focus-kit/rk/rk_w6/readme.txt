This is the readme to the Windows 95 Resource Kit.  The Resource Kit provides over 1200 pages of technical information for the IS manager and Network administrator installing and supporting Windows 95.  This is a BETA DRAFT based on October builds of the M7 product and you can expect the information to be updated over time.

You can use Wordpad or Microsoft Word 6, and the included template file (*.DOT) to read the Resource Kit.  A legend to the chapter names is below.
