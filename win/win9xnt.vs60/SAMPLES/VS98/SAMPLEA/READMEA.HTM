Installing and Configuring Island Hopper News Level A

This topic describes the prerequisites and installation instructions for level A of the Island Hopper News sample. You can install Level A on one machine.

Prerequisites

Level A of the Island Hopper News sample has no hardware requirements, beyond one machine that meets the minimum requirements for Visual Studio 6.0.

Software Requirements

Before installing Level A, you must have installed the following software:

  • Visual Studio 6.0, Professional Edition
  • NT Option Pack, version 4.0
  • Internet Explorer 4.0 or higher

Other Information

The Installation Wizard will prompt you for a location to install web content. You can accept the proposed location or specify your own server.

Installing Sample A and Sample C on the same machine is not supported. Make sure you completely uninstall one level of the sample before installing the other.

Installation Instructions

To install Island Hopper News Level A:
  1. Insert your MSDN CD into the drive.
  2. Navigate to samples\vs98\SampleA.
  3. Run SetupA.exe. This program is a self-extracting ZIP file that uncompresses and copies the required installation files to your local drive, then launches the Island Hopper A setup program.

Level A of the Island Hopper News is now installed on your system. The source code is located in Program Files\IslandHopper\ScenarioA. You can run either the Windows or Web client for the application.

To start the Windows client:

  1. Click Start.
  2. Point to Programs.
  3. Point to Island Hopper.
  4. Click Island Hopper A.

To start the Web client:
  1. Start Internet Explorer.
  2. In the Address box, type the name of the Web server where you installed the Island Hopper sample and the directory for the Web client (islandhoppera). For example, http://myserver/islandhoppera.

For more information, see the following topics online in MSDN Library Visual Studio 6.0.

  • The Island Hopper News Sample
  • Finding and Running Level A
  • Level A Specification

Uninstallation Instructions

To uninstall Level A:
  1. Insert your MSDN CD into the drive.
  2. Navigate to samples\vs98\SampleA.
  3. Run RemPackA.exe. This program removes the IsleHop package from Microsoft Transaction Server.
  4. Click Start.
  5. Point to Settings.
  6. Click Control Panel.
  7. Double-click Add/Remove Programs.
  8. Click the Install/Uninstall tab.
  9. Select Island Hopper Scenario A.
  10. Click Add/Remove.
  11. Click Yes when prompted to confirm the uninstall.
