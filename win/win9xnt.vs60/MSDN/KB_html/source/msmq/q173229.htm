PRB: Error Code = 0x00E0052A Installing Primary Site Controller

 Last reviewed: September 16, 1997
Article ID: Q173229
The information in this article applies to:
  • Microsoft Message Queue Server version 1.0

SYMPTOMS

Setup for a Primary Site Controller (PSC) may fail with the following error message: 

   Unable to create the site on the information server.

   Create site operation failed, because there is
   already a PSC with the same name. PSC name
   must be unique.
   Error code = 0x00E0052A.

CAUSE

This error occurs when site information already exists in the Message Queue Information Store (MQIS) that refers to the computer name as the Site Controller. That is, a Site viewable in MQExplorer identifies the current computer as its site controller.

WORKAROUND

To work around this problem, delete the site for which the computer appears as the PSC. Then restart setup.

MORE INFORMATION

This behavior is by design.
