Web Page Background .gif Image Appears Distorted

 Last reviewed: September 29, 1997
Article ID: Q166481

The information in this article applies to:

  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you view a Web page that uses a .gif image for the background, the image may appear distorted.

CAUSE

The Web page may have been created using an unusual color palette scheme.

RESOLUTION

To display the image correctly, follow these steps:

  1. Start Internet Explorer, and then click Options on the View menu.

  2. Click the Advanced tab.

  3. Scroll to the Multimedia category.

  4. Click the Smart Image Dithering check box to clear it.

  5. Click Apply.

  6. Click OK.
