Unable to Use Context Menu for Folders in My Computer

 Last reviewed: January 29, 1998
Article ID: Q179936
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer Plus 4.0

SYMPTOMS

When you right-click a folder after you install Internet Explorer with the Windows Desktop Update component, the context menu may appear, but you may be unable to click any of the commands on the menu.

CAUSE

This behavior can occur when both of the following conditions exist:

  • You have enabled the Web Style option on the General tab in the Folder Options dialog box.
  • You click a folder, and then immediately right-click it.

RESOLUTION

To work around this behavior, use either of the following methods:

Method 1

Close the folder window by clicking Close on the File menu. The next time you click the folder, do not immediately right-click it.

Method 2

Disable the Web Style option. To do so, follow these steps:

  1. Click My Computer.

  2. On the View menu, click Folder Options.

  3. On the General tab, click Classic View, and then click OK.
