Refresh Button Does Not Refresh Document in Internet Explorer

 Last reviewed: February 4, 1998
Article ID: Q180491
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you click Refresh on the Internet Explorer toolbar, the document you are currently viewing may not be refreshed.

CAUSE

This behavior can occur if you are viewing a Microsoft Office document (such as a Microsoft Word or Microsoft Excel document) in Internet Explorer.

STATUS

This behavior is by design.

MORE INFORMATION

You cannot refresh Office documents that you view in Internet Explorer.

For information about using Internet Explorer to host Office documents, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q162059
   TITLE     : Office Documents Open in Internet Explorer
