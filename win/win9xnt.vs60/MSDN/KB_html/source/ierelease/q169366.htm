Newsgroup Filter Is Not Applied to Current View

 Last reviewed: October 1, 1997
Article ID: Q169366
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you create newsgroup filters in Microsoft Outlook Express version 4.0, the filters are not applied to the current view.

CAUSE

Newsgroup filters are applied to message headers as they are downloaded. Because the current view has already been downloaded, the filters are not applied until you download the message headers again.

RESOLUTION

To download the message headers again, follow these steps:

  1. On the Tools menu, click Download This Newsgroup.

  2. Click the Get The Following Items box to select it, and then click New Headers.

  3. Click OK.
