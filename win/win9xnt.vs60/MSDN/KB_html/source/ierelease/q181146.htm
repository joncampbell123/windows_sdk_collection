Err Msg: You Need More Memory or System Resources. Close Some...

 Last reviewed: February 16, 1998
Article ID: Q181146
The information in this article applies to:
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows 95

SYMPTOMS

When you double-click Mail in Control Panel, or right-click the Inbox icon on the desktop and then click Properties, you may receive the following error message: 

   You need more memory or system resources. Close some windows and try
   again.

CAUSE

This behavior can occur if Outlook Express is your default Simple Messaging application programming interface (MAPI) client.

RESOLUTION

To work around this issue, do not use Outlook Express as your default Simple MAPI client. To configure Outlook Express so that it is not your default Simple MAPI client, follow these steps:

  1. Start Outlook Express.

  2. On the Tools menu, click Options.

  3. Click the "Make Outlook Express my default Simple MAPI client" check box to clear it, and then click OK.
