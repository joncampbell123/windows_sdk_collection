Graphic Image Does Not Appear in an Open Folder

 Last reviewed: October 20, 1997
Article ID: Q175369
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0
  • Microsoft Internet Explorer version 4.0 for Windows 95

SYMPTOMS

If you install Internet Explorer with the Windows Desktop Update and you then enable the Web Style option on the General tab in the Folder Options dialog box, the graphic image may not appear on the left side of the window when you open a folder.

This behavior may also occur if you enable the "For all folders with HTML content" option in the Custom Settings dialog box.

CAUSE

This behavior can occur if the open folder you are viewing is too small to properly display the graphic image.

RESOLUTION

To work around this issue, resize the open folder until it is large enough to display the graphic image. To resize an open folder, click the system icon on the title bar (the icon in the upper-left corner of the open folder), click Size, press the ARROW keys to resize the folder, and then press ENTER.
