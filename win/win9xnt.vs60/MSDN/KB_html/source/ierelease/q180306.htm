OK Button in Add Favorite Dialog Box May Be Unavailable

 Last reviewed: February 26, 1998
Article ID: Q180306
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95

SYMPTOMS

When you attempt to add a file to your Favorites folder, you may experience one of the following symptoms:

  • The OK button in the Add Favorite dialog box may be unavailable.
  • The Add To Favorites command on the Favorites menu may be unavailable.

CAUSE

This behavior can occur if the Favorites folder is damaged.

RESOLUTION

To work around this behavior, follow these steps:

  1. Use Windows Explorer or Windows NT Explorer to create a folder to store a copy of the files in your Favorites folder.

  2. Copy the files from your Favorites folder to the folder you created in step 1. The Favorites folder has the following default location for Windows NT: 

          %SystemRoot%\Profiles\%username%\Favorites

    The Favorites folder has the following default location for Windows 95: 

          C:\Windows\Favorites

  3. Start Internet Explorer, Windows Explorer or Windows NT Explorer.

  4. On the Favorites menu, click Organize Favorites.

  5. Delete any files in the Favorites folder. Do not delete the Channels, Links, or Software Updates folders.

  6. Close the Organize Favorites window.

  7. Confirm that you can now add to the Favorites folder in Internet Explorer, Windows Explorer or Windows NT Explorer.

  8. Use Windows Explorer or Windows NT Explorer to copy one of the files from the folder you created in step 1 back to the Favorites folder.

  9. Repeat steps 7-8 until you copy all the files back to the Favorites folder.

NOTE: If the OK button in the Add Favorite dialog box becomes unavailable after copying one of the files back to the Favorites folder, you may need to delete this file and add it to the Favorites folder from the original source.
