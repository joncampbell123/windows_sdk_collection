Selected Web Page Frame Is Not Saved

 Last reviewed: December 8, 1997
Article ID: Q177798
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

If you click a frame within a Web page, click Save As on the File menu, and then click Save in the Save As dialog box to save the file, the frame you selected is not saved. When you view the saved file with Notepad, the control frame (the frame that is first displayed by the Web site) is displayed instead.

RESOLUTION

To save the frame you want, follow these steps:

  1. Right-click the frame you want to save, and then click View Source.

  2. On the File menu, click Save As, and then click Save.

STATUS

Microsoft has confirmed this to be a problem in Internet Explorer versions 4.0 and 4.01. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
