Damaged or Missing .microsoft Folder Deletes User Settings

 Last reviewed: February 23, 1998
Article ID: Q176759
The information in this article applies to:
  • Microsoft Internet Explorer version 4.01 for UNIX on Sun Solaris

SYMPTOMS

When you start Internet Explorer, your settings, cookies, and all of the files in the Favorites, TempInternetfiles, and History folders may be missing.

CAUSE

This behavior can occur if the .microsoft folder in your home folder is damaged or missing.

MORE INFORMATION

The .microsoft folder contains the following folders and files:

  • Cookies/
  • Desktop/
  • Favorites/
  • Ieconfig/
  • Recent/
  • TempInternetfiles/
  • Registry
  • Mwfc-6dd9

NOTE: "Mwfc-" is the first five characters of the Mwfc-6dd9 file, the last four characters of the Mwfc-6dd9 file may vary.

The TempInternetfiles file is the cache file. The Mwfc-6dd9 file is the font cache file.
