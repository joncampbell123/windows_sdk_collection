Exporting Messages to an Exchange Server Resets Dates and Times

 Last reviewed: October 1, 1997
Article ID: Q174559
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you export messages from Outlook Express to a Microsoft Exchange Server mailbox, the date and time stamps on the messages change to the date and time the export takes place.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

This behavior occurs only when you export messages to a Microsoft Exchange Server mailbox. This behavior does not occur when you export messages to a local Personal Folders (.pst) file.
