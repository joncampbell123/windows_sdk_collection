Error Message: Internet Explorer 4.0 Active Setup Is Already...

 Last reviewed: December 30, 1997
Article ID: Q169340
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you are installing Internet Explorer version 4.0, you may experience any of the following symptoms:

  • Active Setup quits immediately after you choose the download location.
  • Active Setup quits after you accept the license agreement. When you attempt to run Setup again, the following error message occurs: 

          Internet Explorer 4.0 Active Setup is already running. You cannot
      run a second instance of this program.

CAUSE

This behavior occurs if the Inseng.dll file is loaded into memory and in use before you start Setup. When this occurs, Setup cannot replace the file.

RESOLUTION

To install Internet Explorer, restart the computer to unload the Inseng.dll file from memory, and then run Setup again.

NOTE: Some Web pages may load the Inseng.dll file into memory. You should avoid visiting other Web pages before running Setup for Internet Explorer.
