How to Integrate Internet Explorer Into the CDE

 Last reviewed: March 2, 1998
Article ID: Q181622
The information in this article applies to:
  • Microsoft Internet Explorer version 4.01 for UNIX on Sun Solaris

SUMMARY

This article describes how to integrate Internet Explorer version 4.01 for UNIX on Sun Solaris version 2.6 into the Common Desktop Environment (CDE).

MORE INFORMATION

If you start Internet Explorer within a CDE session, Internet Explorer adopts the CDE color scheme unless you specify custom color settings. Custom color settings are stored in the Ie4.dt file, which is located in the /usr/local/microsoft/ie4 folder. There is a set of Internet Explorer icons for use with CDE that are located in the ie4/ie4icons directory.

To integrate Internet Explorer 4.01 for UNIX on Sun Solaris into the CDE, use the appropriate method:

Single User Integration

  1. At the command prompt, type the following command exactly as it appears, and then press ENTER: 

          ln -s <install-dir>/ie4/IE4.dt $HOME/.dt/types

  2. Log off your computer, and then log on.

System Integration

  1. At the command prompt, type the following command exactly as it appears, and then press ENTER: 

          ln -s <install-dir>/ie4/IE4.dt /etc/dt/appconfig/types/C

  2. Log off your computer, and then log on.

NOTE: To remove CDE integration, delete the Ie4.dt file.
