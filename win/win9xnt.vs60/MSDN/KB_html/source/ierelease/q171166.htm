How to Restore the Default Internet Explorer 4.0 Wallpaper

 Last reviewed: November 26, 1997
Article ID: Q171166
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SUMMARY

This article describes how to restore the default Internet Explorer 4.0 desktop wallpaper.

MORE INFORMATION

To restore the default Internet Explorer 4.0 desktop wallpaper, follow these steps:

  1. Click Start, point to Settings, point to Control Panel, and then double-click Display.

  2. On the Background tab, click Browse.

  3. In the Files Of Type box, click Background Files.

  4. Click the Wallpapr.htm file in the Windows\Web\Wallpaper folder, and then click Open.

  5. Click OK.
