Internet Explorer Quits Unexpectedly When You Press CTRL+W

 Last reviewed: October 23, 1997
Article ID: Q161233
3.00 3.01 WINDOWS kbui

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows 95
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows NT 4.0

SYMPTOMS

When you press the CTRL+W key combination, Internet Explorer quits with no warning.

CAUSE

Microsoft Internet Explorer quits with both the ALT+F4 and CTRL+W key combinations.

MORE INFORMATION

The CTRL+W key combination is used to close some third-party Internet- based programs and is included in Internet Explorer for consistency.

Some Microsoft Office programs use this key combination to close the program. Others, such as Microsoft Internet News, use the CTRL+W key combination for opening a listing of available newsgroups.
