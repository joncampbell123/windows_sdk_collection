Internet Explorer Does Not Import Favorites During Upgrade

 Last reviewed: November 26, 1997
Article ID: Q175528
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you install Internet Explorer version 4.0 over a browser program other than Netscape Navigator or an earlier version of Internet Explorer, your favorites may not be imported into Internet Explorer's Favorites folder.

CAUSE

This behavior occurs because Internet Explorer only imports favorites (or Navigator bookmarks) from Netscape Navigator or earlier versions of Internet Explorer.

RESOLUTION

To resolve this issue, you must manually re-create your favorites list. To do so, start Internet Explorer, navigate to a Web site you want to add to your favorites list, and then click Add To Favorites on the Favorites menu.

STATUS

This behavior is by design.
