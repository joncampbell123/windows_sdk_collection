 Last reviewed: March 20, 1998
Article ID: Q181589
The information in this article applies to:
  • Microsoft Internet Explorer version 4.01 for Windows 95

SUMMARY

This article lists the files that are downloaded when you download Internet Explorer 4.01, and the contents of the cabinet files included in the download.

Note that the Ie4_1.cab file is in the Ie4_s1.cab file, the Ie4_2.cab file is in the Ie4_s2.cab file, the Ie4_3.cab file is in the Ie4_s3.cab file, the Ie4_4.cab file is in the Ie4_s4.cab file, the Ie4_5.cab file is in the Ie4_s5.cab file, and the Ie4_6.cab file is in the Ie4_s6.cab file. To extract any of these five cabinet files, type the following command at a command prompt, and then press ENTER 

   extract ie4_s<n>.cab /e
where <n> is the number of the cabinet file you want to extract.

For additional information about how to use the Extract tool included with Windows 95, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q129605
   TITLE     : How to Extract Original Compressed Windows Files
