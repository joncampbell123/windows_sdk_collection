Repeatedly Prompted to Install the Authenticode Update

 Last reviewed: October 28, 1997
Article ID: Q171580

The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02 for Windows 95

SYMPTOMS

After you install the Internet Explorer Authenticode Update, you may be prompted to install it again the next time you connect to the Internet.

CAUSE

This issue can occur if you choose to save the Authenticode Update to disk, and then install it after disconnecting from the Internet.

RESOLUTION

To resolve this issue, follow these steps:

  1. Connect to the Internet. Go to the Authenticode Update Web site when you are prompted.

  2. Install the Authenticode Update. When you are prompted to Open or Save To Disk, click Open.

  3. Finish installing the Authenticode Update, quit Internet Explorer, and then restart your computer.
