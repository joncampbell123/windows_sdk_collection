Cannot Connect to AS/400 with Internet Explorer 4.0 Installed

 Last reviewed: December 9, 1997
Article ID: Q177844
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95

SYMPTOMS

After you install Internet Explorer 4.0 on a Windows 95-based computer, you may be unable to connect to an AS/400 computer.

CAUSE

This behavior can occur if Internet Explorer 4.0 is installed on a computer running NetSoft Elite to connect to an AS/400 computer.

RESOLUTION

To resolve this issue, contact NetSoft to inquire about a resolution for this issue.
