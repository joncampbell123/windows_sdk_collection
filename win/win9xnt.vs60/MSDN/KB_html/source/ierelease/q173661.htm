"Setup Error 514" During Internet Explorer Active Setup

 Last reviewed: October 1, 1997
Article ID: Q173661
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you try to install Internet Explorer 4.0, you may receive the following error message: 

   Setup Error 514:
   The log file command line option (/G) must have a valid filename as
   an argument.
When you receive this error message, Active Setup stops.

CAUSE

This error message can occur if the Windows 3.1 file system is enabled.

RESOLUTION

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.

To resolve this issue, disable the Windows 3.1 file system by changing the Win31FileSystem DWORD value to "0" (without quotation marks) under the following registry key: 

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\FileSystem\
   Win31FileSystem
Then, restart Internet Explorer Setup.
