Documents in Web Pages with Frames May Not Be Printed Properly

 Last reviewed: October 1, 1997
Article ID: Q174562
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95

SYMPTOMS

Web pages containing frames may not be printed properly.

CAUSE

If a Web page contains a Microsoft Office 97 document in either a frame or a floating frame, a blank document is printed if you use the As Laid Out On Screen option when you print the page.

RESOLUTION

To print the document, use one of the other options under Print Frames in the Print dialog box. For Office 97 documents in floating frames, make sure to select the appropriate frame before you attempt to print it.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
