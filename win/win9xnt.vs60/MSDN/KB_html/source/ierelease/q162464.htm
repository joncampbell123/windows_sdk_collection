Internet Explorer 3.0x and MSN 2.0 Share Most Option Settings

 Last reviewed: September 29, 1997
Article ID: Q162464
2.00 3.00 3.01 3.02 WINDOWS kbui

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02 for Windows 95
  • The Microsoft Network version 2.0

SUMMARY

Internet Explorer and MSN, The Microsoft Network, share several option settings. Changes made to common settings affect both programs.

MORE INFORMATION

There are six tabs in the Internet Explorer Options dialog box. To access these tabs, click Options on the View menu in Internet Explorer.

  • General
  • Connection
  • Navigation
  • Programs
  • Security
  • Advanced

There are four tabs in the MSN Options dialog box. To access these tabs, click the MSN logo in the upper-left corner of Program Viewer, and then click Options.
  • General
  • Connection
  • Security
  • Advanced

All settings on the four common tabs are shared except the Toolbar settings on the General tab. The toolbar options apply to Internet Explorer only.
