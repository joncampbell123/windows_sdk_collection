Cannot Install the Internet Components in Microsoft Plus!

 Last reviewed: November 26, 1997
Article ID: Q137294
The information in this article applies to:
  • Microsoft Plus! for Windows 95
  • Microsoft Windows 95
  • Microsoft Internet Startup Kit version 3.0

IMPORTANT: This article contains information about editing the registry. Before you edit the registry, you should first make a backup copy of the registry files (System.dat and User.dat). Both are hidden files in the Windows folder.

SYMPTOMS

You may experience problems installing the Internet components (Internet JumpStart Kit) in Microsoft Plus! for Windows 95 or the Microsoft Internet Startup Kit, version 3.0, if one of the following conditions exist:

  • An OEM (original equipment manufacturer) version of Windows 95 is installed on your computer.
  • You installed Microsoft Internet Explorer (Msie10.exe) from an online service.

These Internet components include the Internet Explorer and Dial-Up Scripting. While you can select these components for installation, they are not installed.

If you attempt to install the Internet Startup Kit, version 3.0, you may receive the following error message: 

   Cannot find a device file c:\\iexplore.exe on Setup.

CAUSE

A registry key is set that prevents Microsoft Plus! Setup from installing the Internet components.

RESOLUTION

To install the Internet components, you must remove the IVer value from the following registry key: 

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer
For information about how to edit the registry, view the "Changing Keys And Values" online Help topic in Registry Editor (Regedit.exe). Note that you should make a backup copy of the registry files (System.dat and User.dat) before you edit the registry.

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows 95. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.
