Signature File Size Limitation in Outlook Express

 Last reviewed: December 17, 1997
Article ID: Q165806
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you are using a signature file with a mail or news message in Outlook Express, you may receive the following error message: 

   Warning: Your signature file is too large. It has been truncated,
   please make the file smaller.

CAUSE

Your signature file is larger than the 4K size limitation.

RESOLUTION

Make your signature file smaller or use the text option rather than an HTML file.

MORE INFORMATION

Before the option to use an HTML file for a signature was available, there was rarely a need for a signature file to be larger than 4K. With the addition of this ability, file sizes may exceed this 4k value.
