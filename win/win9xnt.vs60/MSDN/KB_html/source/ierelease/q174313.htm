Internet Explorer 4.0 Hangs Loading Web Page

 Last reviewed: September 29, 1997
Article ID: Q174313
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95
  • The Microsoft Network, versions 1.0, 1.05, 1.2, 1.3

SYMPTOMS

When you try to view a Web page, Internet Explorer 4.0 may seem to stop responding (hang).

CAUSE

This behavior can occur if you install MSN, The Microsoft Network, version 1.0, 1.05, 1.2, or 1.3 after you install Internet Explorer 4.0. These versions of MSN replace the Msnsspc.dll file with an older version of the file that does not work with Internet Explorer 4.0.

RESOLUTION

To resolve this issue, reinstall Internet Explorer 4.0 in the same folder.
