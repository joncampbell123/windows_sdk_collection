Unable to Browse When Exporting Windows Address Book

 Last reviewed: October 1, 1997
Article ID: Q169365
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

If you click Browse in the CSV Export dialog box when you are exporting a Windows Address Book to a comma-delimited text file, the browse function may not work correctly. For example, if you type "c:\windows\temp" in the Save Exported File As box and then click Browse, the Save In box may display "C:\Windows" and the File Name box may display "Temp.csv."

CAUSE

This behavior can occur if you enter a partial path in the Save Exported File As box before you click Browse.

RESOLUTION

Clear the Save Exported File As box before you click Browse, and then navigate through the folders in the Save In box to the folder to which you want to save the exported file.

STATUS
