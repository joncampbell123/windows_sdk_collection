Installed Components for Browser-Only, Standard, and Full Setup

 Last reviewed: October 7, 1997
Article ID: Q174828
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SUMMARY

This article describes the components that are installed when you choose the Browser-Only, Standard, or Full options in Internet Explorer Setup.

MORE INFORMATION

Note that the components listed in the "Components available on Internet Explorer Component Download page only" column must be installed after you install Internet Explorer.
