Cannot Use Internet Mail and News with MSN 2.0

 Last reviewed: October 29, 1997
Article ID: Q153843
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows 95
  • The Microsoft Network, version 2.0

SUMMARY

Microsoft Internet Mail and News and Microsoft Outlook Express require Simple Mail Transport Protocol (SMTP) and Post Office Protocol (POP3) mail servers for mail and a Network News Transport Protocol (NNTP) news server for news. MSN, The Microsoft Network, version 2.0 does not offer SMTP/POP3 mail services.

MSN 2.0 supports Microsoft Internet News and the Outlook Express news reader on the following news servers:

  • msnnews.msn.com
  • netnews.msn.com

NOTE: If you want to use Internet Mail and News or Outlook Express with MSN, you need to upgrade to MSN version 2.5. You can order a free copy of the MSN 2.5 CD-ROM by calling (800) FREE-MSN. If you are already an MSN member, the MSN 2.5 CD-ROM will be automatically sent to you.

For more information about this issue and about installing MSN 2.5, please see the following articles in the Microsoft Knowledge Base:

MORE INFORMATION

If you are using Microsoft Internet Explorer as your Web browser and MSN as your Internet service provider, follow these steps to use Windows Messaging when you click an e-mail shortcut in the browser:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Internet.

  3. Click the Programs tab.

  4. In the Mail box, click Windows Messaging.

  5. Click OK.
