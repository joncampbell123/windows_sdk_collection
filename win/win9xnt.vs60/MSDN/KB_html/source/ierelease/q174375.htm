Mail in Outbox Sent When You Send a New Message

 Last reviewed: September 29, 1997
Article ID: Q174375
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

If you click Send in a New Message window to send a new message, the message is sent using your default mail account. However, other messages placed in your Outbox using the Send Later Using command may also be sent on the specified accounts.

RESOLUTION

To prevent messages placed in the Outbox using the Send Later Using command and an account other than your default account from being sent when you click Send in a New Message window, move the messages from the Outbox before you send the new message.
