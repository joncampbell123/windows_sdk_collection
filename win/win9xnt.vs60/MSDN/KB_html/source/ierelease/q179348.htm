Status Bar in Internet Explorer May Not Remain Hidden

 Last reviewed: January 21, 1998
Article ID: Q179348
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you hide the status bar in Internet Explorer, the status bar may no longer be hidden the next time you start Internet Explorer.

CAUSE

This behavior can occur if you uninstall the Windows Desktop Update component.

RESOLUTION

To work around this behavior, reinstall the Windows Desktop Update component.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

For information about how to install or uninstall the Windows Desktop Update component, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q165695
   TITLE     : How to Add or Remove Windows Desktop Update
