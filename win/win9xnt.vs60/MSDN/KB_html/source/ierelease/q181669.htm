Error Message: Load Error #35: Cannot Access Bce Server...

 Last reviewed: March 19, 1998
Article ID: Q181669
The information in this article applies to:

  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you install Internet Explorer, you may receive either of the following error messages: 

   Load error #35: Cannot access bce server
   Load error #5: Initialize failed

   Load error #35: Couldn't register with bce server
   Load error #5: Initialize failed

CAUSE

This behavior can occur if you have Lexmark printer drivers installed.

RESOLUTION

To resolve this issue, obtain the Newbce.exe self-extracting executable file from Lexmark Technical Support. Once you obtain this file, follow these steps to install updated versions of the Lexbce.dll and Lexbces.exe files:

  1. Copy the Newbce.exe file to the Windows\System folder.

  2. In Windows Explorer, locate and double-click the Newbce.exe file.

  3. When you are prompted to replace the existing files, click Yes.
