How to Send Mail from an Account That Is Not the Default Account

 Last reviewed: March 6, 1998
Article ID: Q182134
The information in this article applies to:
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows 95
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows NT 4.0

SUMMARY

This article describes how to send e-mail from an account that is not your default account using Outlook Express.

MORE INFORMATION

You can use the following method to send a message, or reply to a message, so that your message is sent from an account that is not your default account.

After you compose your message (or reply), click Send Later Using or Send Message Using on the File menu. In the list of your e-mail accounts, click the account from which you want to send the message. The message is sent using the account you specify.

Note that if you simply click Send, the message is sent using your default mail account.
