Shell Integration Not Enabled after Installing Internet Explorer

 Last reviewed: November 26, 1997
Article ID: Q167318
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95

SYMPTOMS

After you install Internet Explorer with Shell Integration, Shell Integration mode may be disabled. As a result, you may notice that no Web sites or pictures are displayed on your desktop and that the Desktop tab does not appear in the Display Properties dialog box. In addition, "Microsoft Internet Explorer 4.0 - Shell Integration Mode" may not appear in the list of installed programs on the Install/Uninstall tab in the Add/Remove Programs Properties dialog box. Therefore, you cannot enable Shell Integration mode after Internet Explorer is installed.

CAUSE

This problem occurs when you install a different language version of Internet Explorer than the version of Windows 95 or Windows 95 OEM Service Release that is installed on your computer. For example, this behavior occurs if you install the English version of Internet Explorer on a computer that is running a non-English version of Windows 95. Shell Integration mode is not supported when you are running different language versions of Internet Explorer and Windows 95 or OEM Service Release.

RESOLUTION

To work around this problem, uninstall Internet Explorer 4.0, and then install the same language version of Internet Explorer 4.0 as the version of Windows 95 or OEM Service Release that is installed on your computer. For information about installing Internet Explorer, visit the following Microsoft Web site: 

   http://www.microsoft.com/ie/ie40/
For information about uninstalling Internet Explorer, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q170757
   TITLE     : Uninstalling Internet Explorer 4.0 Platform Preview 2.0
If the necessary language version of Internet Explorer is not available, you can use the Internet Explorer browser and Dynamic Hypertext Markup Language (HTML) and content delivery features, but you cannot use Shell Integration mode.
