Error Message: This Program Has Performed an Illegal Operation

 Last reviewed: November 26, 1997
Article ID: Q174714
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95

SYMPTOMS

When you try to log off from Outlook Express without choosing a user name, you may receive the following error message: 

   This program has performed an illegal operation.
If you then click Details, you see the following message: 

   Rundll32 caused an Invalid Page Fault in module Mslocusr.dll
Note that when you close the error message box, you remain logged on.

RESOLUTION

To resolve this issue, repeat the log-off process and click a user name when you are prompted.
