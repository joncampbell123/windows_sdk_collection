SUMMARY

This article lists the files contained in the Minnt30f.exe file and the cabinet files it contains. Internet Explorer version 3.03 for Microsoft Windows NT 3.51 is available as a self-extracting file named Minnt30f.exe. The Minnt30f.exe file contains the Microsoft Internet Explorer 3.03 Internet browser, and Virtual Machine for Java.
