How to Install Internet Assistants for Microsoft Office 95

 Last reviewed: February 17, 1998
Article ID: Q181164
The information in this article applies to:
  • Microsoft Internet Explorer Plus 4.0

SUMMARY

This article describes how to install and use the Internet Assistants for Microsoft Office version 7.0 for Windows 95.

MORE INFORMATION

To use the Internet Assistants in Microsoft Office version 7.0 in Windows 95, you must first install the appropriate Internet Assistant from the Internet Explorer Plus CD-ROM.

To install Internet Assistants for Microsoft Office version 7.0 for Windows 95, follow these steps:

  1. Insert the Internet Explorer Plus CD-ROM into the CD-ROM drive.

  2. If the Microsoft Internet Explorer Plus screen appears, click Add-Ons, and then skip to step 7.

  3. Double-click My Computer.

  4. Right-click the CD-ROM drive icon, and then click Explore.

  5. Double-click the Setup.exe file.

  6. When the Microsoft Internet Explorer Plus screen appears, click Add- Ons.

  7. Click Internet Assistants For Office 95.

  8. Click the Internet Assistant you want to install. Internet Assistants are available for the following Microsoft Office 95 programs: 

          Microsoft Word 7.0 for Windows 95
      Microsoft Excel 7.0 for Windows 95
      Microsoft PowerPoint 7.0 for Windows 95
      Microsoft Access 7.0 for Windows 95

To use the Internet Assistants, use the appropriate method:
  • In Word, click New on the File menu, and then double-click Html.dot.
  • In Microsoft Excel, click Internet Assistant Wizard on the Tools menu.
  • In PowerPoint, click Export As HTML on the File menu.
  • In Access, point to Add-Ins on the Tools menu, and then click Internet Assistant.

Note that the Internet Assistants are already installed in Microsoft Office 97 programs.
