Playing .avi or .mid Files in Temporary Internet Files Folder

 Last reviewed: September 29, 1997
Article ID: Q165413
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01 for Windows 95

SUMMARY

When you double-click a .mid or .avi file in the Temporary Internet Files folder, the appropriate player starts and Internet Explorer opens, with Internet Explorer displaying the Windows\System\Blank.htm file.

MORE INFORMATION

Earlier versions of Internet Explorer opened the associated Web site containing the file. Internet Explorer versions 3.0, 3.01, and 3.02 open the Windows\System\Blank.htm file and play the file directly.
