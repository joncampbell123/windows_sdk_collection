FrontPage Express Wizard Does Not Save File Name Correctly

 Last reviewed: October 27, 1997
Article ID: Q175686
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you save a Web page or form that was created with a wizard, the saved file name may not be the same as the name you chose.

RESOLUTION

To work around this problem, rename the file after you save it. To rename a file, right-click the file in Windows Explorer, click Rename, type the name you want, and then press ENTER.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FrontPage Express. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
