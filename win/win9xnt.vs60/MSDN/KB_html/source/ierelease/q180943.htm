MORE INFORMATION

Microsoft Outlook Express 4.0 for Macintosh

Welcome Thank you for downloading Microsoft Outlook Express 4.0 for Macintosh.

Microsoft Outlook Express 4.0 for Macintosh will replace Microsoft Internet Mail and News as the premier mail and news client for Macintosh.

Microsoft Outlook Express enables you to communicate with others on the Internet or corporate Intranet through electronic mail and newsgroups.

  • Electronic mail, or e-mail, lets you exchange messages and files with other people on the Internet.
  • Internet newsgroups consist of messages by individuals, organized by topic. You can take part by posting a message of your own, responding to a message and sending it to the whole newsgroup, or sending an e- mail message directly to the author of a message.

Requirements System Requirements
  • 68030 processor or better
  • System 7.1 and above
  • 3,600K of available RAM
  • 6.7 MB of hard disk space to install.
  • 1 MB of hard disk space to run, depending on the amount of e-mail you receive.

Access to the Internet You must have access to the Internet through an Internet service provider (ISP) or the network to which your computer is connected.

Internet Software

  • MacTCP (control panel) or Open Transport 1.08 or greater (extension) with TCP/IP (control panel)
  • If you are a dial-up user, Config PPP or similar PPP connection software (control panel) with PPP (extension)

New and Improved Features Standards Support
  • Enhanced support for SMTP, POP3, and NNTP makes it even easier to communicate over e-mail.
  • Internet Mail Access Protocol (IMAP4) support allows you to store mail messages on a server so you can access them from different locations.
  • Lightweight Directory Access Protocol (LDAP) server support lets you find people on the Internet or Intranet by accessing directory services such as Four11, Bigfoot, or a company's internal directory.
  • Support for HTML mail and MHTML allows you to read e-mail with HTML pages that have embedded images.
  • Outlook Express has rich international features including Unicode support.
  • Support for scripting following Apple's Mail Suite. Compiled scripts that are saved in the Script Menu Items folder are displayed in a script menu in the program. Outlook Express also supports Frontier's menu sharing protocol.

Contact and Information Management Features
  • The Contacts feature provides you an extended address book in which you can maintain extensive contact information.
  • You can easily group your contacts into mailing lists.
  • Inbox and Newsgroup rules allow you to filter and automate tasks for incoming messages.
  • Import messages and address book entries from Netscape Navigator 2.0 and 3.0, Eudora 3.0, and Netscape Communicator 4.0.

Simple, Integrated Interface:
  • Hierarchical folders help you organize your mail.
  • A folder list in the left hand pane lets you view your hierarchical mail folders, contacts, news servers, LDAP servers, and multiple e-mail accounts, all in the same easy-access window.
  • You can send and receive mail from all your mail accounts at once, or you can select individual accounts to send and receive mail.
  • A Drafts folder lets you keep track of the messages you create and aren't ready to send.
  • You can display images within messages (you must have QuickTime 2.5, or newer, installed to take advantage of this feature).

Extra Value with Office 98
  • With Office 98 installed, you can easily spell check your e-mail.
  • You can choose to spell check messages when you send them, or check spelling right as you type.
  • The Mail Merge feature in Word 98 lets you address letters and envelopes using your Outlook Express contacts.
  • The Office 98 programs make it easy to e-mail documents with Outlook Express by allowing you to click Send To from within any Office 98 program.

Security Issues

Outlook Express has several features that make it easier to keep your communications private. These features are intended to prevent other people from accessing your mail through normal use of Outlook Express. They are not designed to prevent malicious people from using alternate tactics to gain access to your private information. Below are some tips on using the security features in Outlook Express.
  • If other people have access to your computer and you are concerned about security, do not choose the Save Password option when creating e- mail accounts. This is especially important if you are using the multiple user feature.
  • If you do not want other users to access the mail in your local folders, choose the Require Password option in the Startup Settings section of the Startup & Quit panel in Preferences. Note that this will not prevent other users from using another program to directly access the mail files stored on your disk.
  • Some security options, including SSL, require support from the server that you are connecting to. If you are having trouble with these features, please contact your server administrator.

Known Limitations Upgrading from Internet Mail and News To import messages and addresses from Internet Mail and News, you must have Internet Mail and News 3.0 or later.

Outlook Express does not import filters or newsgroup favorites from Internet Mail and News. After upgrading, you will have to subscribe to your favorite newsgroups again and set up Inbox Rules and Newsgroup Rules to replace your IMN filters.

International Outlook Express does not support right to left languages.

Extended characters are not permitted in server passwords because of potential character set conflicts between different operating systems.

Inbox rules Inbox rules do not work with IMAP or online POP mail accounts.

Internet Config (IC) Preferences When you start Outlook Express for the first time, it will check for Internet Config settings. If you have IC settings, OE will prompt you to verify those settings, but will only display the New Account dialog box. When this occurs, you should type the name of your Internet Config mail account in the Account Name box, and click OK.

Technical Support Microsoft offers technical support and services ranging from self-help tools to direct assistance with a Microsoft technical engineer. For more information on options in your area, read Support.txt located in the Help folder (inside the Outlook Express folder).

Reporting Problems/Suggestions To report problems, send mail to msmacoe@microsoft.com.

Please include:

  • The version of Outlook Express you are using
  • What version you upgraded from
  • System configuration information
  • Version of operating system
  • Type of Macintosh
  • Type of connection to the Internet
  • Steps that lead to the bug. For example: 1. Start Outlook Express. 2. From the File menu, choose Open. 3. Click the Cancel button. 4. Result: Nothing happens.
  • Any other information that will help us reproduce the bug successfully.
