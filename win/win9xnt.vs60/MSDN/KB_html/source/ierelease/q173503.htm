Cannot Install Internet Explorer in Safe Mode

 Last reviewed: January 9, 1998
Article ID: Q173503
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95

SYMPTOMS

When you attempt to install Internet Explorer in Safe mode, or Safe mode with network support, one or more of the following symptoms may occur:

  • Windows may stop responding (hang).
  • You may receive error messages.

RESOLUTION

Install Internet Explorer running Windows 95 normally.
