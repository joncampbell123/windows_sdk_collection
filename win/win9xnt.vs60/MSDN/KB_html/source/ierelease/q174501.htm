How to Change Reply Font and Line Color in Outlook Express

 Last reviewed: September 30, 1997
Article ID: Q174501
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SUMMARY

This article describes how to change the color of line that appears to the left of the original text when you reply to a message using the HTML Mail Sending format.

MORE INFORMATION

When you reply to a mail message using the HTML Mail Sending format, a colored line appears to the left of the original text. By default, this line is the same color as the font you use to reply to the message. To change the color, follow these steps:

  1. In Outlook Express, click Stationery on the Tools menu.

  2. Click Font Settings.

  3. In the Color box, click the color you want.

  4. Click OK, and then click OK again.
