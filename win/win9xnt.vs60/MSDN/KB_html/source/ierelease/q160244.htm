Java Class Files Must Be Version 45.3 or Later

 Last reviewed: September 29, 1997
Article ID: Q160244
The information in this article applies to:
  • Microsoft Internet Explorer version 3.0 for Windows 95

SUMMARY

The Java implementation for Internet Explorer 3.0 does not support loading class files earlier than version 45.3.

MORE INFORMATION

When a Java tool is compiled, the class files are tagged with a version stamp. Internet Explorer interprets only files with a version of 45.3 or later because earlier version numbers are not completely compatible with version 45.3.

If earlier class files are encountered, the Java code must be recompiled with a newer compiler version.

Microsoft Technical Support Desktop Systems division does not have the tools to determine the version of a Java class file. Contact the compiler manufacturer to discuss this issue.
