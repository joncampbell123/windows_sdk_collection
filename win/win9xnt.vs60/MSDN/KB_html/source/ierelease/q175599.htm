Ordered List Font Size Is Larger than Default Text Font Size

 Last reviewed: November 26, 1997
Article ID: Q175599
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you apply a numbered list to text in a message, the font size of the ordinals (numbers) may be larger than the font size of the text in the body of the message.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook Express. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
