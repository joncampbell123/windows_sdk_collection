"Download Complete" Message After Downloading a File

 Last reviewed: February 16, 1998
Article ID: Q179621
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you finish downloading a file in Internet Explorer, you receive the following message: 

   Download Complete

STATUS

This behavior is by design. This is a new feature in Internet Explorer that cannot be disabled.
