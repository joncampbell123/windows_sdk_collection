Windows Messaging Missing from Mail and News Options

 Last reviewed: September 29, 1997
Article ID: Q158435
3.00 3.01 4.0 WINDOWS kbsetup kbenv

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you configure the Mail and News options on the Programs tab in Internet Explorer, Windows Messaging may not be available in the Mail list.

CAUSE

Microsoft Exchange or Windows Messaging was installed after Internet Explorer.

RESOLUTION

To add Windows Messaging as an option in the Mail list box, install and configure Microsoft Exchange or Windows Messaging before installing Internet Explorer, or reinstall Internet Explorer after installing and configuring Microsoft Exchange or Windows Messaging.
