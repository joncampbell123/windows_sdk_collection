System Properties Do Not Show That Microsoft Plus! Is Installed

 Last reviewed: November 26, 1997
Article ID: Q167598
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Plus! for Windows 95

SYMPTOMS

If you have installed Microsoft Plus!, System properties no longer shows that Microsoft Plus! is installed after you install Internet Explorer 4.0. The information line no longer shows: 

   Microsoft Plus! for Windows 95
Instead, this line reads: 

   IE 4.0 <7-digit build number>

STATUS

Microsoft has confirmed this to be a problem in Internet Explorer 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The opposite behavior also occurs. If you install Microsoft Plus! after installing Internet Explorer version 4.0, the System properties information line no longer shows: 

   IE 4.0 <7-digit build number>
Instead, this line reads: 

   Microsoft Plus! for Windows 95
This behavior is purely cosmetic; it does not affect your ability to run installed Microsoft Plus! or Internet Explorer components.
