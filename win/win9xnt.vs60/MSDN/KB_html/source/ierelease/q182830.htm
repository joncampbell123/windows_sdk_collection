Scrolling Problems When You Use Internet Relay Chat

 Last reviewed: March 20, 1998
Article ID: Q182830
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0, 4.01 for Windows 95

SYMPTOMS

When you participate in an Internet Relay Chat (IRC) session, the most recent entries to the chat session may not scroll normally.

CAUSE

This behavior can occur if experimental Hypertext Markup Language (HTML) tags are used in the source code of the Web page.

RESOLUTION

To work around this issue and to view the most recently typed entries to the chat session, manually scroll to the bottom of the chat frame.

STATUS

Experimental tags are not an approved standard by the W3 Consortium (W3C) and are not an approved MIME type. Due to their experimental nature, they are not supported by Internet Explorer.

MORE INFORMATION
