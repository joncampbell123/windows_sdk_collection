No New Posting Notification in Outlook Express

 Last reviewed: October 1, 1997
Article ID: Q169448
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you view a list of newsgroup messages in Microsoft Outlook Express version 4.0, you receive no indication that there are new, unread messages in a thread. You must manually expand the thread to see that there are new messages.

RESOLUTION

To work around this behavior, set Outlook Express News to automatically expand conversation threads. To do this, perform the following steps:

  1. On the Tools menu, click Options.

  2. Click the Read tab, and then click the Auto Expand Conversation Threads check box to select it.

  3. Click OK.

STATUS

This feature is under review and will be considered for inclusion in a future release.
