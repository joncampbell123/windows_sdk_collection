Additional Components Are Installed in the Program Files Folder

 Last reviewed: November 26, 1997
Article ID: Q173592
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you install additional components from the Internet Explorer 4.0 Components Download page, you are not prompted for a location in which to install the components.

MORE INFORMATION

Internet Explorer 4.0 additional components are installed by default into folders in the Program Files folder.
