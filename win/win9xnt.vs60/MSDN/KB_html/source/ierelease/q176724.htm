Shortcut Keys Do Not Function in Internet Options Dialog Box

 Last reviewed: February 23, 1998
Article ID: Q176724
The information in this article applies to:
  • Microsoft Internet Explorer version 4.01 for UNIX on Sun Solaris

SYMPTOMS

When you attempt to use shortcut keys in the Internet Options dialog box, the shortcut keys may not function.

RESOLUTION

To work around this issue, use the TAB key or pointing device to navigate in the Internet Options dialog box.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To open the Internet Options dialog box using a pointing device, click Internet Options on the View menu in Internet Explorer.

To open the Internet Options dialog box using shortcut keys, press ALT+V and then press O (the letter "o") in Internet Explorer.

For additional information about shortcut key issues in Internet Explorer Preview 1 for UNIX, please see the following article in the Microsoft Knowledge Base:
