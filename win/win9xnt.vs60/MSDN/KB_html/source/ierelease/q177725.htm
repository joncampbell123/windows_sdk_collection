Description of the Internet Explorer 4.0 PowerToys

 Last reviewed: December 5, 1997
Article ID: Q177725
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

NOTE: This article provides information about a configuration that is not supported by Microsoft Technical Support. This article is provided for informational purposes only; Microsoft makes no guarantee that this configuration will function properly.

SUMMARY

This article describes the Internet Explorer PowerToys.

MORE INFORMATION

The Internet Explorer PowerToys enhance your use of Internet Explorer. These tools are designed for Internet Explorer 4.0 and can be obtained from the following Microsoft Web site: 

   http://www.microsoft.com/ie/ie40/powertoys/
The following tools are added to Internet Explorer when you install the PowerToys.

Image Toggler

Using Image Toggler you can enable or disable displaying images in Internet Explorer. To toggle images off or on, click Toggle Images on the Links Bar.

Links List

Using Links List you can view all the links on a Web page. To view all the links on a Web page, right-click the Web page, and then click Link Lists. When you click Link Lists, a new window opens that lists every link on the page.

Open Frame in New Window

Using Open Frame In New Window you can view a Web page designed to use frames as a non-frame Web page. To view a Web page without frames, right- click the Web page, and then click Open Frame In New Window.

Quick Search

Using Quick Search you can access search engines using search engine abbreviations. The following is a list of the default Quick Search abbreviations. 

   Abbreviation   Search Engine
   -----------------------------
   av             AltaVista
   ex             Excite
   hb             HotBot
   is             InfoSeek
   isu            InfoSeek Ultra
   ly             Lycos
   mc             MetaCrawler
   mg             Magellan
   ot             OpenText
   wc             WebCrawler
   y              Yahoo!
For example, to search the AltaVista search engine on the word "ActiveX," type "av ActiveX" (without quotation marks) in the address bar in Internet Explorer.

Text Highlighter

Using Text Highlighter you can highlight text in a document. To highlight text in a document in Internet Explorer, right-click the selected text, and then click Hightlight.

Web Search

Using Web Search you can search the Internet for additional Web pages that contain selected words from a Web page. To start a Web search, drag the mouse pointer to select the text for which you want to search, right-click the selected text, and then click Web Search.

Zoom In/Zoom Out

Using Zoom In/Zoom Out you can zoom in and out of any image on a Web page. To zoom in or zoom out of an image in Internet Explorer, right-click the image, and then click Zoom In or Zoom Out.
