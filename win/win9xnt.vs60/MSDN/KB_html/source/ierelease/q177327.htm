Problems Printing or Saving Multiple E-mail Messages

 Last reviewed: November 25, 1997
Article ID: Q177327
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

If you select multiple e-mail messages, and then click the File menu, the Save As command is unavailable. Also, if you try to print these messages, only the last message is printed.

CAUSE

Outlook Express does not support saving or printing multiple e-mail messages at once.

MORE INFORMATION

This functionality is available in Microsoft Windows Messaging and Microsoft Outlook 97.
