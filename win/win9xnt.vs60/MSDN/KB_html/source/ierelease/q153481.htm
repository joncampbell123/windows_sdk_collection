Internet Explorer Cannot Sort Temporary Internet Files by Type

 Last reviewed: September 29, 1997
Article ID: Q153481

The information in this article applies to:

  • Microsoft Internet Explorer version 3.0, 3.01, 3.02, 4.0 for Windows 95

SYMPTOMS

When you are viewing the Temporary Internet Files folder in Internet Explorer, or viewing the Windows\Temporary Internet Files folder in Windows Explorer, you cannot sort the files by type (that is, by file name extension).

CAUSE

When files are downloaded using Internet Explorer, the files are saved with temporary file names on the hard disk. When you view these files with Internet Explorer or Windows Explorer, the temporary names are shown as the actual names of the files. The program that tracks the names and displays them in the Temporary Internet Files folder is not able to sort the file names by file name extension.
