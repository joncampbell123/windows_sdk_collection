PRB: No Find Tab in Help Under DBCS Windows 95

 Last reviewed: February 26, 1998
Article ID: Q181603
The information in this article applies to:
  • Microsoft Win32 Software Development Kit (SDK) on the following platform: 

        - Microsoft Windows 95

SYMPTOMS

You cannot see a Find tab in Help.

CAUSE

Chinese, Japanese, and Korean Windows 95 do not include a full text search DLL (Ftsrch.dll located in the \\windows\system directory). This is because these versions of Windows 95 use the double-byte character set (DBCS), and the full text search is bogus for DBCS.

RESOLUTION

To work around this, you can copy the US Ftsrch.dll into Chinese, Japanese, and Korean Windows 95, and then you can see the Find tab. However, the content of the find tab is invalid because full text search does not exist for DBCS.
