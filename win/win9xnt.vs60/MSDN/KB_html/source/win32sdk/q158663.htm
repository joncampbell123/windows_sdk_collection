SAMPLE: Textin.exe Reads Text Files into an ActiveMovie Graph

 Last reviewed: January 16, 1998
Article ID: Q158663

The information in this article applies to:
  • ActiveMovie Software Development Kit, version 1.0 for: - Microsoft Windows NT 3.51, 4.0 - Microsoft Windows 95

SUMMARY

Textin.exe is a sample source filter for the ActiveMovie SDK. It demonstrates the minimal implementation required to read text from a file, and to connect to the TextOut renderer filter. Developers can use Textin.exe as a starting point for more sophisticated source filters.
