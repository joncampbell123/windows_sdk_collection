Buglist for ActiveMovie SDK Version 1.0

 Last reviewed: October 28, 1996
Article ID: Q157060
The information in this article applies to:
  • ActiveMovie Software Development Kit version 1.0 for: 

        - Microsoft Windows 95
    - Microsoft Windows NT version 4.0

SUMMARY

The following information lists the known bugs and methods not fully implemented in the ActiveMovie 1.0 SDK.

  • IMediaPosition::put_Rate(negative values) is unsupported.
  • IMediaPosition::put_Rate(0 or 10E50) does not play back.
  • ActiveMovie OCX occasionally interferes with screen savers.
  • Rendering a file can cause asserts during playback in Visual Basic.
  • Filters and libraries will not build with UNICODE set.
  • CancelDefaultHandling() unavailable for EC_COMPLETE, EC_NEED_RESTART.
  • The Filename property cannot be set through VBScript.

MCI Commands Unsupported

  • CUE NOSHOW [MPEG device type]
  • SET VIDEO OFF [MPEG device type]
  • SET VIDEO STREAM TO NUMBER [MPEG device type]
  • LOAD

MCI Commands Not Fully Implemented

  • SET AUDIO STREAM TO NUMBER [MPEG device type]
  • PUT DESTINATION WINDOW
  • PUT WINDOW
  • CAPABILITY [WAVE device type]
  • STATUS POSITION [in byte format]
  • SETAUDIO (LEFT, RIGHT, STEREO)
  • PLAY FROM/TO [backwards]
