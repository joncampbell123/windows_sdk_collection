PRB: WS_EX_TOOLWINDOW Windows Don't Show System Menu Icon

 Last reviewed: January 15, 1998
Article ID: Q179376
The information in this article applies to:
  • Microsoft Win32 Application Programming Interface (API) included with: - Microsoft Windows NT 4.0 - Microsoft Windows 95

SYMPTOMS

Windows created with the WS_EX_TOOLWINDOW, WS_CAPTION, and WS_SYSMENU styles have system menus, but they do not display an icon for the system menu on the caption bar.

RESOLUTION

You can display the menu by right-clicking the title bar, or by typing Alt- Space.

STATUS

This behavior is by design.
