INFO: ADSI Version 2 Is Available on the Web

 Last reviewed: November 25, 1997
Article ID: Q177283
The information in this article applies to:
  • Microsoft Windows Software Development Kit (SDK) on the following platforms: - Alpha - Windows NT - Windows 95 - x86

SUMMARY

ADSI Version 2 is available on the Web.

MORE INFORMATION

The runtime and helpfiles can be downloaded from: 

   http://backoffice.microsoft.com/downtrial/moreinfo/adsi2.asp
ADSI version 2.0 client runtimes are available for Windows 95 and Windows NT on both Intel and Alpha platforms. The runtimes contain providers for Windows NT version 3 or 4, Novell Netware 3.12, Novell Netware 4 NDS, and LDAP.

The ADSI version 2.0 SDK is part of the Platform SDK build environment and can be downloaded from: 

   http://www.microsoft.com/msdownload/platformsdk/bldenv.htm
The ADSI version 2.0 samples are part of the Distributed Services Components and can be downloaded from: 

   http://www.microsoft.com/msdownload/platformsdk/ds.htm
