SAMPLE: Psycho.exe Demonstrates Simple Palette Animation

 Last reviewed: January 16, 1998
Article ID: Q158662

The information in this article applies to:
  • ActiveMovie Software Development Kit, version 1.0 for: - Microsoft Windows NT 3.51, 4.0 - Microsoft Windows 95

SUMMARY

Psycho.exe is a sample source filter for the ActiveMovie SDK. It demonstrates a simple palette-animation technique in ActiveMovie you can use by changing the entries in the palette portion of the media sample.

For information on ActiveMovie or the ActiveMovie architecture, check out http://www.microsoft.com/directx/default.asp.
