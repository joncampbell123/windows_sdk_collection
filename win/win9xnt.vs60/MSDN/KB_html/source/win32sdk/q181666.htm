PRB: Problems Installing the MSAA SDK

 Last reviewed: February 25, 1998
Article ID: Q181666
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows NT, version 4.0

SYMPTOMS

When you are installing the Microsoft Active Accessibility (MSAA) SDK, you may receive the following server error message: 

   Serial Number Validation Failed

CAUSE

This message is a result of rounding problems when the system performs the CRC (cyclic redundancy check) on the unique serial number. Because the installation program randomly generates this serial number, forcing it to generate a new serial number usually results in a successful install.

RESOLUTION

To work around this problem, reinstall the SDK. Because a different serialization number will be generated for each new installation, the problem is not likely to reoccur.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES
