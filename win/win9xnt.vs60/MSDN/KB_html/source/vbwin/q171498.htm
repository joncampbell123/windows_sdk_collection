FIX: Screen Turns Black When Compiling

 Last reviewed: December 18, 1997
Article ID: Q171498
The information in this article applies to:
  • Microsoft Visual Basic Control Creation, Learning, Professional, and Enterprise Editions for Windows, version 5.0 On the following platform: NT

SYMPTOMS

If the Console Window display option setting is set to "Full Screen," then the screen will go black while compiling the Visual Basic Application.

RESOLUTION

Install Service Pack 2 (SP2) or follow the steps below:

  1. Open the Control Panel.

  2. Double-click the Console Applet.

  3. Set the display option to Window.

  4. Click OK to accept the change.

STATUS

Microsoft has confirmed this to be a bug in the Microsoft products listed at the beginning of this article. This bug has been fixed in Visual Studio 97 Service Pack 2.
