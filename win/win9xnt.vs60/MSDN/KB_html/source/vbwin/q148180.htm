INFO: File Differences in Visual Basic 4.0a

 Last reviewed: February 11, 1997
Article ID: Q148180
The information in this article applies to:
  • Standard, Professional, and Enterprise Editions of Microsoft Visual Basic for Windows, 16-bit and 32-bit, version 4.0a

SUMMARY

This article contains a list of files that are updated after installing Visual Basic 4.0a and the version differences between version 4.0 and 4.0a.
