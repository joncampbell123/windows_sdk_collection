INFO: Application Default Location is hpc Folder

 Last reviewed: February 10, 1998
Article ID: Q180758
The information in this article applies to:
  • Windows CE Toolkit for Visual Basic 5.0, version 1.0

SUMMARY

When using the Application Install Wizard, you will be prompted for the location of the .pvb file. The .pvb file is placed in the following folder by default: 

   \Program Files\Windows CE Platform SDK\wce\emul\hpc
This is not stated in the documentation.

MORE INFORMATION

You can specify where the .pvb file is stored by setting the Local Path parameter in the Project Properties dialog.

However, the Application Install Wizard always looks first in the full default path listed above, regardless of project properties.
