This file lists all the files on the distribution disks provided with this product. NOTE: Most of the files on the distribution disks are compressed (indicated by an underscore character "_" in the file extension) and must be decompressed before they are used.

The Setup program on Disk 1 decompresses files as it installs them.

Files with a PA_ extension decompress to multiple .ICO or .BMP files.

If you need to decompress files manually, you can do so by running setup from the file manager or program manager in the following way:

  1. Choose the Run item from the File menu.
2. Type SETUP.EXE /Z [source file] [destination file]

The /Z switch activates an internal decompression system. SETUP.EXE must be run from within Windows. ***** DISK1 ****
