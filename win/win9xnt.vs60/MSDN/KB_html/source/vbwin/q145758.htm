INFO: Redistributable Files in the PSS Directory of VB 4.0 CD

 Last reviewed: February 26, 1998
Article ID: Q145758
The information in this article applies to:
  • Standard, Professional, and Enterprise Editions of Microsoft Visual Basic, 16-bit and 32-bit, for Windows, version 4.0

SUMMARY

The following files from the PSS directory of the CD-ROM version of Visual Basic 4.0 can be redistributed with your Visual Basic program. Redistributing these files with your Visual Basic program is within the terms of the End User Licensing Agreement.

  • REGC16.DLL
  • REGC32.DLL
  • REGCLEAN.EXE
  • REGCLN16.EXE
  • REGOCX16.EXE
  • REGOCX32.EXE
  • REGSVR.EXE
  • REGSVR32.EXE

NOTE: The RegClean utility is being updated. The new and improved utility will be available for download by Friday, March 13, 1998.
