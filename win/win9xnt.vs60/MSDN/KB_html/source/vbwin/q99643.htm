FILE: Source Code for the Visual Basic Data Manager

 Last reviewed: March 10, 1998
Article ID: Q99643
The information in this article applies to:

- Microsoft Visual Basic programming system for Windows, version 3.0

SUMMARY

Datamgr.exe is a self-extracting compressed file that contains the source code for the Visual Basic Data Manager. Most of the files are dated April 21, 1993 and were last updated on April 25, 1994.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Datamgr.exe (size: 38644 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from
               Online Services
The source code has been made available because of a press release announcement made by Microsoft that stated that the source code for the Visual Basic version 3.0 Data Manager would be available.
