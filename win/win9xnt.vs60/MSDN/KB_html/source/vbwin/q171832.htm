FIX: Statement Builder Can Cause the Object Browser to Crash

 Last reviewed: December 18, 1997
Article ID: Q171832
The information in this article applies to:
  • Microsoft Visual Basic Control Creation, Professional, and Enterprise Editions for Windows, version 5.0

SYMPTOMS

Using Ctrl-Shift-J to bring up the Statement Builder sometimes causes a crash under circumstances where the typelib being referenced has become disconnected. This is a sporadic bug, and not easily reproduced.

RESOLUTION

To avoid this problem, install Visual Studio 97 Service Pack 2 (SP2).

STATUS

Microsoft has confirmed this to be a bug in the Microsoft products listed at the beginning of this article. This bug has been fixed in Visual Studio 97 Service Pack 2.
