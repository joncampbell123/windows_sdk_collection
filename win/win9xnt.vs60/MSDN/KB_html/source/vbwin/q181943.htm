INFO: System Requirements for Windows CE Toolkit for VB 5.0

 Last reviewed: March 3, 1998
Article ID: Q181943
The information in this article applies to:
  • Windows CE Toolkit for Visual Basic 5.0, version 1.0

SUMMARY

In order to run the Windows CE Toolkit for Visual Basic 5.0 on a device or in emulation, you need the following items:

  1. PC with a 486/66 MHz or higher processor; Pentium or higher processor recommended.

  2. Microsoft Windows NT Workstation operating system version 4.0 (Service Pack 3 required); Windows NT Workstation 4.0 is required for emulation.

  3. Microsoft Visual Basic, Professional or Enterprise edition, version 5.0 (Visual Studio Service Pack 2 required, Service Pack 3 recommended).

  4. 24 MB of RAM; 32 MB recommended.

  5. Hard-disk space required: 

        - Minimum installation: 30 MB.
    - Complete installation: 115 MB;
      220 MB required if installing online documentation.

  6. CD-ROM drive.

  7. VGA or higher-resolution monitor; Super VGA recommended.

  8. Microsoft Mouse or compatible pointing device.

NOTE: You need Windows NT to run emulation because Windows CE is Unicode only and NT provides the necessary Unicode environment.

Please note that the Windows CE Toolkit for Visual Basic 5.0, version 1.0 is not currently compatible with Windows 95.

If you need to install Internet Explorer when you bring up Windows CE Books Online, you can do so by running the msie302.exe in the /msie302 directory.

MORE INFORMATION

Information on how to obtain the latest Visual Studio Service Pack can be found at: 

   http://www.microsoft.com/vstudio/sp/default.htm
Information on how to obtain the latest Windows NT Service Pack can be found at: 

   http://www.microsoft.com/ntworkstation/info/ntupdates.htm.
