HOWTO: Obtain/Distribute the Compatibility Layer in VB 3.0

 Last reviewed: March 10, 1998
Article ID: Q113951
The information in this article applies to:

- Microsoft Visual Basic programming system for Windows, version 3.0

SUMMARY

This article describes the Microsoft Jet 2.0/Visual Basic 3.0 Compatibility Layer that allows owners of Microsoft Access 2.0 and Visual Basic 3.0 to use and create Jet 2.0 databases in Visual Basic 3.0.

MORE INFORMATION

How to Obtain the Compatibility Layer

To obtain the Compatibility Layer, choose one of the following methods:

The following file is available for download from the Microsoft Software Library: 

 ~ Comlyr.exe (size: 741798 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from
               Online Services
  • If you are unable to access the source listed above, you can have COMLYR.EXE mailed or faxed to you by calling Microsoft Technical Support Monday through Friday, 6:00 a.m. to 6:00 p.m. Pacific time at (425) 646-5105. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, go to the Microsoft Web site: 

          http://www.microsoft.com/worldwide/default.htm
  • Purchase the Microsoft Access Developer's Toolkit (ADT) version 2.0 available through resellers or direct from Microsoft by calling the Microsoft Sales Information Center at 800-426-9400. The suggested retail price is $495 for new users. However, until August 31, 1994, owners of Microsoft Access Distribution Kit version 1.1 and Visual Basic version 3.0 will receive $100 off price via a rebate. All the files you need, including the Jet 2.0 database engine, are included in the ADT.
  • Purchase the Microsoft Office Developer's Kit (ODK) and Microsoft Access version 2.0. Both products are available through resellers or direct from Microsoft by calling the Microsoft Sales Information Center at 800-426-9400. The suggested retail price for the ODK is $99.95. However, if you own Visual Basic version 3.0 Professional Edition, you can get the ODK for only $49.95 by calling the Microsoft Sales Information Center at 800-426-9400.

    The ODK will also be available as a standard component in all new packages of Visual Basic version 3.0 Professional Edition. The ODK comes on CD-ROM only.

    The Compatibility Layer that you download from the MSL or that is included in the ODK does not include the Microsoft Access version 2.0 (Jet 2.0) database engine file. The 2.0 database engine comes with Microsoft Access or the ADT. Therefore, to use the Compatibility Layer, you need to purchase Microsoft Access version 2.0 or the ADT. The license in the ODK permits distribution of this file with Visual Basic version 3.0-based applications. Suggested retail price for Microsoft Access version 2.0 is $495. Until August 31, 1994, current Microsoft Access version 1.x users can upgrade for $99. And those users upgrading from competing products and from Microsoft Office can purchase Microsoft Access version 2.0 for $129.

    How to Distribute the Compatibility Layer

    After obtaining the Compatibility Layer from any of the options listed above, Microsoft allows propagation of the Compatibility Layer files free- of-charge to other eligible Visual Basic licenses within the same organization.

    Applications created with Visual Basic and the Microsoft Access version 2.0 (Jet 2.0) database engine are, as before with Microsoft Access version 1.1, fully distributable royalty-free to application end users. The following files may be distributed with your Visual Basic applications:

    MSAJT200.DLL -- The Microsoft Access 2.0 database engine MSAJT112.DLL -- The Microsoft Access 2.0 compatibility layer. 

    VBDB300.DLL   -- Microsoft Visual Basic to Access support DLL.

BTRV200.DLL   -- Btrieve ISAM driver library.
PDX200.DLL    -- Paradox ISAM driver library.
XBS200.DLL    -- Microsoft FoxPro ISAM driver library.

PDBJET.DLL    -- Crystal Reports file.
PDCTJET.DLL   -- Crystal Reports file.
PDIRJET.DLL   -- Crystal Reports file.
