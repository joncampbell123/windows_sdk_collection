FILE: WIN32API.EXE: Updated Version of Win32api32.txt File

 Last reviewed: December 12, 1997
Article ID: Q178020
The information in this article applies to:
  • Microsoft Visual Basic Learning, Professional, and Enterprise Editions for Windows, version 5.0

SUMMARY

WIN32API.EXE is a self-extracting program that installs an updated version of the Win32 API Declarations text file, Win32api.txt. The updated version corrects incomplete or wrong declare statements shown in the version that ships with Visual Basic.

The file can be downloaded from the Visual Basic Owner's site at http://premium.microsoft.com/download/vbasic/win32api.exe. Only registered users of Visual Basic can access this site.
