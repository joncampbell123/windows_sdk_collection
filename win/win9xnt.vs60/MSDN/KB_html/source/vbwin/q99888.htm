FILE: GENERIC Sample Not Provided with Visual Basic 3.0

 Last reviewed: March 9, 1998
Article ID: Q99888
The information in this article applies to:

- Professional Edition of Visual Basic for Windows, version 3.0

SUMMARY

Generic.exe is a self-extracting compressed file that contains source code to create a custom control wrapper. This sample was omitted from Visual Basic version 3.0.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~Generic.exe
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
After downloading the file, run it to obtain the GENERIC sample files.

STATUS

Microsoft has confirmed this to be a bug in the Microsoft Visual Basic programming system version 3.0 for Windows. This problem can be corrected by downloading the GENERIC sample files.
