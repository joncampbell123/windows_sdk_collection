INFO: VBCE Unable to Automate Pocket Word and Pocket Excel

 Last reviewed: February 10, 1998
Article ID: Q180848
The information in this article applies to:
  • Windows CE Toolkit for Visual Basic 5.0, version 1.0

SUMMARY

Windows CE versions of Microsoft Office products, such as Pocket Word and Pocket Excel, etc., cannot be automated because their object models are not exposed. Thus, Windows CE for Visual Basic 5.0 has no means by which to automate them.

Books Online for Microsoft Windows CE Toolkit for Visual Basic 5.0
