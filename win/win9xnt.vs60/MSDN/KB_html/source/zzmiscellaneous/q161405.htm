Black Screen or Green Vertical Lines with ATI Card

 Last reviewed: March 5, 1998
Article ID: Q161405
The information in this article applies to:
  • DreamWorks Interactive, The Neverhood for Windows, version 1.0
  • DreamWorks Interactive, Scholastic's Goosebumps Series: 

         Escape from Horrorland for Windows, version 1.0
  • Microsoft Deadly Tide, version 1.0
  • Microsoft Flight Simulator for Windows 95
  • Microsoft Monster Truck Madness, version 1.0
  • Microsoft Gex, version 1.0
  • Microsoft Golf, version 3.0
  • Microsoft Hellbender, version 1.0
  • Microsoft NBA Full Court Press, version 1.0
  • Microsoft Soccer, version 1.0

SYMPTOMS

After you run Setup for one of the programs listed above, your screen may be black, or you may see green lines running across the screen.

CAUSE

This problem can occur if you have an ATI Graphics Pro Mach 64 PCI video card.

RESOLUTION

To resolve this problem, you need to change your video settings. To change the settings, do the following:

  1. Boot Windows into Safe Mode:

    a. Restart the computer.

    b. Press the F5 key on the keyboard when 'Starting Windows '95...' 

          appears on the screen.

  2. Switch the display adapter to VGA:

    a. In Windows, click the Start button, point to Settings, and then click 

          Control Panel.

    b. Double-click Display.

    c. Click the Settings tab.

    d. Click the Change Display Type button.

    e. Click the Change button to change the Adapter Type.

    f. Select Show All Devices.

    g. For Manufacturer, click Standard Display Types.

    h. From the list of models, select VGA.

    i. Click OK twice, and click Yes to restart Windows.

  3. Switch the display adapter to ATI Mach 64 PCI (DirectDraw) or to ATI 3D Expression PCI (Mach 64):

    a. After the computer restarts, Click the Start button, point to 

          Settings, and Click Control Panel.

    b. Double-click the Display icon.

    c. Click the Settings tab.

    d. Click Change Display Type.

    e. Click Change to change the Adapter Type.

    f. Select Show All Devices.

    g. For Manufacturer, select ATI Tech Enhanced.

    h. From the list of models, select ATI Mach 64 PCI (DirectDraw) or 

          ATI 3D Expression PCI (Mach 64).

    i. Click OK twice, then click Yes to restart Windows.

  4. Start the program.
