WD6X: Supported Foreign File Formats in Word 6 for the Macintosh

 Last reviewed: February 3, 1998
Article ID: Q119461
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1

SUMMARY

Microsoft Word 6.x for the Macintosh supports conversion to and from a number of different file formats, including word processor document files, spreadsheets, and databases. Word interprets some file formats internally and others through use of external converters. A list of supported formats is included below. For additional details, see the "More Information" section following the table.

  • Converters for formats preceded by an asterisk (*) are available on a supplemental disk and are not supplied with Word. For information on how to obtain this disk, please see the following article in the Microsoft Knowledge Base: 

        ARTICLE-ID: Q119859

   converters installed by the "Typical" Setup option for Word.

MORE INFORMATION

 


  • Conversion of formats for which the converter location above is
   referenced as "Internal" is handled internally by Word without the use
   of an external converter.


  • For information about specific converters supplied with Word, double-
   click the Help button on the Standard toolbar in Word and then type
   "readme" (without the quotation marks). Press ENTER twice, choose the
   "File Conversion" topic, and then choose the "Conversion Limitations"
   topic.


  • For information about supplemental converters, see the Conversion
   Information file on the Supplemental File Conversion disk.


  • To obtain the Supplemental File Conversions disk free of charge, see the
   Microsoft Word for the Macintosh Supplemental Offers coupon included in
   your Word package. If you cannot locate this coupon, call the Microsoft
   Sales Information Center at (800) 426-9400. Deaf or hard-of-hearing
   customers can call Microsoft Text Telephone (TT/TDD) services at
   (800) 892-5234. Or see the following article in the Microsoft Knowledge
   Base:

    

        ARTICLE-ID: Q119859

        TITLE     : MacWord 6 Supplemental Files Conversion Disk Available


  • For information about third-party vendors that support conversion to and
   from different file formats not directly supported by Word, please see
   the following article in the Microsoft Knowledge Base.

    

        ARTICLE ID: Q174813

        TITLE     : WD: Third-Party Conversion Solutions


  • A converter is also installed with Word 6.0 that allows Word for the
   Macintosh 5.0 and 5.1 users to directly open Word 6.0 documents. During
   a complete Word Setup, this converter is installed in the Microsoft
   folder with the name "Word 6.0 for Windows&Macintosh."

    
   To use this converter with Word 5.x for the Macintosh, copy or move the
   converter file into the Word Commands folder in your Word 5.1 folder.
   The next time you start Word 5.1, you can open a Word 6.0 for the
   Macintosh or a Word 6.0 for Windows document.

    
   This converter is also available through a separate kit for Word 5.x
   users who don't have immediate access to Word 6 for the Macintosh.
   For additional information on how to obtain this kit, please see the
   following article in the Microsoft Knowledge Base:

    
   ARTICLE-ID: Q105319

       TITLE     : WD: How to Obtain a Word 6.x/7.0 Converter for MacWord
               5.x Users


  • The MacWrite converter built into Word 5.0 and 5.1 for the Macintosh is
   no longer supplied with Word 6.0. For information on how to convert to
   and from MacWrite format, please see the following article in the
   Microsoft Knowledge Base:

    
   ARTICLE-ID: Q133126

               Mac
