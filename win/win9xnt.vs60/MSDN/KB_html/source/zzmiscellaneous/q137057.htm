What's Installed with PowerPoint for Windows 95 Setup Options

 Last reviewed: January 23, 1998
Article ID: Q137057
The information in this article applies to:
  • Microsoft PowerPoint for Windows 95, version 7.0

SUMMARY

Below is a table with a summary of what components are installed with various PowerPoint setup options.

Setup Choices

Compact: Installs the minimum files necessary to run PowerPoint.

Typical: Installs the most commonly used features of PowerPoint.

Custom: Defaults to installing the same choices as Typical (above). You 

         can, however, override these defaults and choose which components
         you want to install.
Office: Lists the options that are available if you install PowerPoint 
         as part of Microsoft Office. To select these components, run
         Office Setup and choose the Custom option.
