PPT97: How to Obtain PowerPoint Viewer 97

 Last reviewed: March 9, 1998
Article ID: Q170801
The information in this article applies to:
  • Microsoft PowerPoint 97 for Windows

SUMMARY

The Microsoft PowerPoint Viewer 97 is not available on floppy disks. It is available as a self-extracting archive in the Microsoft Software Library.

To install the PowerPoint Viewer, download the following file from the Microsoft Software Library and then double-click the Ppview97.exe file to start the Setup program: 

 ~ Ppview97.exe (size: 2856336 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

MORE INFORMATION

The PowerPoint Viewer 97 program is a windows-based program that you can use to display presentations created in PowerPoint 97 for Windows (and earlier versions of PowerPoint for Windows) as well as PowerPoint for the Macintosh, versions 3.0 and later.

The Ppview97.exe is also available for download from the following Microsoft Web site: 

   http://www.microsoft.com/powerpoint/internet/viewer/viewer32.htm
Microsoft encourages users to copy the PowerPoint Viewer 97 file freely and distribute it to friends and co-workers along with presentations they have created in PowerPoint format.

NOTE: You may also be able to find this file on other Internet sites or electronic bulletin board services.
