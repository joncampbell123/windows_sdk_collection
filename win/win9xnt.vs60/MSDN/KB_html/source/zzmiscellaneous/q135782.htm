WD: Converters Installed with Each Setup Type in Word/Office 95

 Last reviewed: February 2, 1998
Article ID: Q135782
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Office for Windows 95
  • Microsoft Windows 95 and Windows NT operating systems

SUMMARY

This article lists converters and supporting files installed with each of Word's Typical, Compact, Custom, and Run From Network Server installation options.

MORE INFORMATION

Typical Setup

If you choose Typical during a stand-alone Word or Office for Windows 95 installation or if, during a workstation installation, you choose to write shared components to your local hard disk, the following converters and support files are written locally: 

   Converter      File size   Supported formats
   --------------------------------------------

   Mswrd632.cnv   143360      Word 6.0-7.0 for Windows/Macintosh
   Wpft532.cnv    395264      WordPerfect 5.x for DOS and Windows
   Wpft632.cnv    222720      WordPerfect 6.x import
   Wpequ532.dll   102912      WordPerfect equation converter
   Wpgimp32.flt   146944      WPG graphic import filter
   Wpgexp32.flt   68608       WPG graphic export filter
   Excel32.cnv    137728      Microsoft Excel 3.x-7.0
   Wwpab.cnv      34304       Exchange Personal Address Books and
                              Schedule+ Contact Lists
If you choose Typical during a workstation installation and choose to access shared components from the server, only the Wwpab.cnv converter supporting Exchange Personal Address Books and Schedule+ Contact Lists is written to your local hard disk. All other converters referenced above are registered to be accessed from the server.

Compact Setup

If you choose Compact during either a stand-alone or workstation installation, only the Wwpab.cnv converter is written to your local hard disk.

Custom Setup

If you choose Custom during a stand-alone Word or Office for Windows 95 installation or if, during a workstation installation, you choose to write shared components to your local hard disk and choose Continue without making any changes, the same converters are installed as for a stand-alone Typical setup referenced above.

If you choose Select All before choosing Continue, the converters referenced beneath the heading "'Run from NetWork Server' Workstation Setup" below are installed.

If you choose Custom during a workstation installation for which you choose to access shared components from the server, only the Wwpab.cnv converter is written to your local hard disk. If you choose Continue without making any changes, the following converters are registered to be accessed from the server: 

   Converter      File size   Supported formats
   --------------------------------------------

   Mswrd632.cnv   143360      Word 6.0-7.0 for Windows/Macintosh
   Wpft532.cnv    395264      WordPerfect 5.x for DOS and Windows
   Wpft632.cnv    222720      WordPerfect 6.x import
   Excel32.cnv    137728      Microsoft Excel 3.x-7.0
If you choose Select All before choosing Continue, the converters referenced beneath the heading "'Run from Network Server' Workstation Setup" below are registered to be accessed from the server.
