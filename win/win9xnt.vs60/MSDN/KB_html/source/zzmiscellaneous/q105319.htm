WD: How to Obtain a Word 6.x/7.0 Converter for MacWord 5.x Users

 Last reviewed: February 3, 1998
Article ID: Q105319
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a, 6.0, 6.0.1
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

If you are using Word 5.x for the Macintosh, you need a converter to read documents that were created in any of the following versions of Word:

  • Word 6.0 for Windows
  • Word 6.0 for Windows NT
  • Word for Windows 95 version 7.0
  • Word 6.0 for the Macintosh

Word 5.x users that don't have access to Word 6.0 for the Macintosh, can obtain this converter by ordering or downloading the Word 6.0 Converter Kit (for Word 5.x for the Macintosh users).

Alternatively, if a complete installation of Word 6.0 exists on the same Macintosh computer as Word 5.x, the converter is immediately available for installation and use with Word 5.x.

For information on obtaining and installing the converter, see the appropriate section below.

Installing the Converter as Supplied with Word 6.0 for the Macintosh

  1. Quit Microsoft Word if it is running on your Macintosh.

  2. Copy the file "Word 6.0 for Windows&Macintosh" from the Microsoft folder existing at the same level as your Word 6.0 folder to the Word Commands subfolder in your Word 5.x folder. If this file is not available, run Word 6.0 Setup again to install the converter and repeat step 2.

  3. Move the file "Word 6.0 for Windows/Macintosh" to the trash if it exists in the Word Commands folder. This similarly named file is an earlier version of the converter that can only interpret Word 6.0 for Windows documents.

  4. Restart Word 5.x. The converter should now be properly installed.

MORE INFORMATION

Although the converter is called "Word 6.0 for Windows&Macintosh" and the Word 7.0 file is referenced as Word 6.0 for Windows during conversion, the converter is the correct one to use when converting Word 7.0 for Windows files.

This kit is not required to open Word 5.x for the Macintosh files into Word 6.0 and 7.0 for Windows or Word 6.0 for the Macintosh. Both programs recognize and interpret Word 5.x for the Macintosh format automatically.

Those formatting features in Word 6.0 or 7.0 that do not exist in Word 5.0 or 5.1 for the Macintosh are not retained in conversion. A list of converter limitations is included in the file "Word 6.0 Conversion Information," which is installed with the converter.

If you have questions about installing or using the converter, in the United States call Product Support Services at (425) 635-7200 between 6:00 A.M. and 6:00 P.M. Pacific time. Outside the United States, contact your local Microsoft subsidiary.
