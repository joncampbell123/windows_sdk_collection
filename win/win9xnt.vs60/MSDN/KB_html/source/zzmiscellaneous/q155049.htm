International Extensions for Internet Explorer 3.0 Available

 Last reviewed: September 29, 1997
Article ID: Q155049
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02 for Windows 95
  • Microsoft Internet Explorer version 3.0, 3.01, 3.02 for Windows NT 4.0

SUMMARY

Microsoft has released International Extensions for Microsoft Internet Explorer 3.0. Internet Explorer 3.0 includes language support for some languages. International Extensions for Internet Explorer 3.0 lets you view Hypertext Markup Language (HTML) documents created in the following languages:

  • Japanese
  • Korean
  • Pan European
  • Traditional Chinese
  • Simplified Chinese

Note that if any international character set is installed, an extra icon appears in the Internet Explorer status bar. This icon appears as a globe with several flags in front of it. By clicking this icon, you can select the active character set from the currently installed sets.

MORE INFORMATION

To use International Extensions for Internet Explorer 3.0, you need any language version of Windows 95 or Windows NT 4.0 and any language version of Internet Explorer 3.0.

International Extensions for Internet Explorer 3.0 does not run with Beta versions of Internet Explorer 3.0. For information about the availability of Internet Explorer 3.02, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q164475
   TITLE     : Availability of Internet Explorer 3.02 for Windows 95 and
               NT 4.0

Instructions for Downloading

  1. Download the appropriate file from the Microsoft Software Library to an empty folder on your hard disk.

  2. Double-click the file you downloaded in step 1.

  3. Follow the instructions on the screen.
