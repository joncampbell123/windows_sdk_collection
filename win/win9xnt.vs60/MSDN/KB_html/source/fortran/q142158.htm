PRB: QuickWin App Causes Access Violation Under Windows 95

 Last reviewed: January 5, 1996
Article ID: Q142158
The information in this article applies to: 

  - Microsoft FORTRAN Compiler for MS-DOS, version 5.1

SYMPTOMS

A QuickWin application created with FORTRAN version 5.1 causes an access violation when run under Windows 95.

CAUSE

The Fl.def file makes reference to a program loader that is now obsolete under Windows 95.

RESOLUTION

Remove the following line from the Fl.def file located in the \Fortran\Bin directory: 

   APPLOADER   '_MSLANGLOAD'
