Internet Newsgroups Shown as Bulletin Board Type

 Last reviewed: July 14, 1997
Article ID: Q131810

The information in this article applies to:

  • The Microsoft Network versions 1.0, 1.05, 1.2, 1.3

SYMPTOMS

When you use the Find command to locate an Internet newsgroup, the Find window shows the newsgroup's type as a bulletin board folder or BBS folder.

CAUSE

The type shown is a limitation of a tool used by The Microsoft Network.
