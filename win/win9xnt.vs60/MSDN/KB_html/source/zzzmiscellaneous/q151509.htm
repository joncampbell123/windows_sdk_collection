How to Install Internet News

 Last reviewed: January 20, 1998
Article ID: Q151509

The information in this article applies to:

  • The Microsoft Network versions 1.00, 1.05, 1.20, 1.30

SUMMARY

MORE INFORMATION

Q. How do I install Internet News?

A. To install Internet News, follow these steps: 

   1. Start your Web browser.

   2. Connect to the following Microsoft Web page:

         http://www.microsoft.com/msdownload/ieplatform/imn/02.htm

   3. In the Please Select The Desired Language box, click the language
      version of Internet News you want to download, and then click Next.

   4. Click the Mailnews.exe link at the download site located nearest
      to you.

   5. If you are using Microsoft Internet Explorer, click Open. If you
      are using another browser, download the file and then open it from
      the location that it was downloaded to.

   6. Click Yes on the License Agreement to continue.

   7. Type your name and organization (optional), and then click Next.

   8. Verify the information, and then click Next.

   9. Click Internet News only, and then click Next. (Internet Mail is not
      supported by MSN at this time.)

   10. Select the location for installation (default is C:\Program
       Files\Internet Mail and News), and then click Next.

   11. Click Finish. If you are prompted to restart your computer, click
       Yes.

   12. Click Start, point to Programs, and then click Internet News.

   13. Click Next.

   14. Type your name and e-mail address in the appropriate box, and then
       click Next.

   15. In the News Server box type "msnnews.msn.com" (without the quotation
       marks).

   16. Click "My news server requires me to logon."

   17. Click Logon Using Secure Password Authentication, and then click
       Next.

   18. Click "I use a modem to access my newsgroups," and then click The
       Microsoft Network from the drop down list.

   19. Click Next, and then click Finish.

   20. In the connection dialog box on the screen, click OK to connect to
       The Microsoft Network.

   21. Upon connecting, Internet News downloads the list of available
       newsgroups from the server. Type "msn.memberservices.msn" (without
       the quotation marks) into the Newsgroups box, and it should appear
       in the list of available newsgroups.

   22. Click msn.memberservices.msn, click Subscribe, and then click Go To.

   23. If the MSN Sign In screen appears, click OK.

       NOTE: In order for this to work you must have your access number
       set to Internet And The Microsoft Network. To verify your settings,
       right-click the MSN icon on the desktop, and then click Connection
       Settings. In the Connection Settings dialog box, click Access
       Numbers, and then click Internet And The Microsoft Network in
       the Service Type box.
