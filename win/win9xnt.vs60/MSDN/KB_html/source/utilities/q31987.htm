Maximum Number of Object Modules LINK Supports

 Last reviewed: January 24, 1995
Article ID: Q31987
The information in this article applies to:
  • Microsoft LINK for MS-DOS, versions 3.x, 5.01.20, 5.01.21, 5.03, 5.05, 5.1, 5.13, 5.15, 5.3, 5.31.009, 5.5 and 5.6
  • Microsoft LINK for Windows NT, versions 1.0 and 2.5

The Microsoft LINK utility does not impose any limit on the number of object modules specified on the linker command line. However, LINK is constrained by the amount of available system resources.
