DOCERR: PWB Online Help Lists /D CodeView Option As /O

 Last reviewed: July 17, 1997
Article ID: Q67081
1.00 1.10 MS-DOS kbtool kbdocerr

The information in this article applies to:

  • Microsoft Programmer's Workbench for MS-DOS, versions 1.0 and 1.1
  • Microsoft Programmer's Workbench for OS/2, versions 1.0 and 1.1

The online help for the Microsoft Programmer's WorkBench (PWB) versions 1.0 and 1.1 contains a minor error in the information concerning the setting of the option to have CodeView use disk overlays (/D).

From the CodeView Options dialog box in PWB, press F1 for help, choose "memory configuration," and then choose "disk overlays." The help screen states that this menu item is comparable to the /O command line option. The help screen should say it is comparable to the /D command line option.
