Not Enough Core Message in PWB Caused By Lack of Memory

 Last reviewed: July 17, 1997
Article ID: Q68070
1.00 1.10 MS-DOS kbtool

The information in this article applies to:

  • Microsoft Programmer's Workbench for MS-DOS, versions 1.0 and 1.1

When loading the Programmer's WorkBench (PWB) version 1.0 or 1.1, a message box may appear on the screen containing the message "Cannot Autoload Extension, not enough core." The extension file that could not be loaded will be indicated. You need to free more memory so that the extensions can be loaded.
