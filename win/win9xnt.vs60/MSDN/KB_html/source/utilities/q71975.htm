DOCERR: QuickHelp Searches C:\QH, DPATH, PATH, BOOKSHELF

 Last reviewed: July 17, 1997
Article ID: Q71975
1.70 1.83 | 1.70 1.83 
MS-DOS    | OS/2
kbtool kbdocerr

The information in this article applies to:

  • Microsoft QuickHelp utility versions 1.7 and 1.83

When invoking QuickHelp with the HELPFILES and QH environment variables set to directories that do not contain help files, QuickHelp will search (in the order listed) the following places for help files:

  1. The directory C:\QH (if it exists).

  2. The directories listed in the DPATH environment variable.

  3. The directories listed in the PATH environment variable.

  4. The directories listed in the BOOKSHELF environment variable (this only applies to the OS/2 version of QuickHelp).

QuickHelp will stop searching as soon as it finds any help files.
