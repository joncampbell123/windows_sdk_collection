DOCERR: TRACE /MAP Causes "TR1007: Unrecognized Option" Error

 Last reviewed: July 17, 1997
Article ID: Q91119
1.00 1.10 MS-DOS kbtool kberrmsg kbdocerr

The information in this article applies to:

  • Microsoft Trace Utility for MS-DOS, versions 1.0, and 1.1

SYMPTOMS

An attempt to use the /MAP switch with versions 1.0 or 1.1 of the Microsoft Trace Utility for MS-DOS fails with the following error message: 

   TR1007: unrecognized option

CAUSE

The /MAP switch was not implemented.

MORE INFORMATION

The MOVEAPI.TXT file included with Microsoft C/C++ version 7.0 lists /MAP as one of the options for the TRACE utility. As indicated above, this is incorrect.
