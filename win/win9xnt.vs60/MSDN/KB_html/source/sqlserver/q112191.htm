INF: Maximum Size for DBTEXTLIMIT Option

 Last reviewed: April 29, 1997
Article ID: Q112191

The information in this article applies to:
  • Microsoft SQL Server Programmer's Toolkit, version 4.2

With version 4.21 of DB-Library (DB-Lib), the maximum size that can be specified for the DB-Lib option DBTEXTLIMIT has been increased from 65534 to 2147483647 for Windows NT applications. The maximum size that can be specified for this parameter on all other platforms is still 65534.
