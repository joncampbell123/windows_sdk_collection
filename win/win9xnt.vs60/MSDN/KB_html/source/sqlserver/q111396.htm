BUG: TEXT Columns Truncated to 127 Characters in Query Window

 Last reviewed: April 29, 1997
Article ID: Q111396

The information in this article applies to:
  • Microsoft SQL Server Programmer's Toolkit, version 4.2
BUG# 9801 (4.2)

SYMPTOMS

Results displayed in a Query Window using the QUERY button in ISQL/W, SQLAdmin, or Object Manager will be truncated to 127 characters for any column defined as a TEXT over this length.

This occurs on both 16- and 32-bit versions of these front-ends running against 4.2b for OS/2 and 4.2 for Windows NT.

WORKAROUND

Use ISQL.EXE, Select Data menu option under Tools in SQL Object Manager, or develop your own DB-Library application.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SQL Server version 4.2 and SQL Server version 4.2b for OS/2. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
