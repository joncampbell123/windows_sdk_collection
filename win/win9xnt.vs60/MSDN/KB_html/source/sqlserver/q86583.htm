PRB: "Network Not Started" Error in SQL Administrator

 Last reviewed: April 28, 1997
Article ID: Q86583

The information in this article applies to: 

  - Microsoft SQL Server version 4.2

SYMPTOMS

If you click the icon for starting SQL Administrator for Windows, you may receive the following error message: 

   The network has not been started, cannot continue.
This problem occurs more often on Novell networks.

CAUSE

SQL Administrator for Windows can not find the correct version of a file called NETAPI.DLL on the workstation.

WORKAROUND

Make sure that the correct NETAPI.DLL file is on your computer. For Windows clients on Novell networks, this file (with a size of 7168 bytes) can be found on the Novell NetWare OS/2 Requester disk under the WINDOWS.NP subdirectory. The latest version of this file is dated 6-24-91 and is part of the Novell OS/2 Requester.

If you find any other versions of this file (with different sizes and dates) on your computer, rename or remove them and also reboot your computer to ensure that the correct NETAPI.DLL is loaded in memory.
