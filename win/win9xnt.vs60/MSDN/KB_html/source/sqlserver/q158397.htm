BUG: Global WinNT Groups List Empty in French Security Manager

 Last reviewed: April 17, 1997
Article ID: Q158397
The information in this article applies to:
  • Microsoft SQL Server, version 6.5
BUG #: 16604 (NT: 6.50)

SYMPTOMS

The French Security Manager is unable to retrieve the default domain name, if you do the following:

  1. On the Securite menu, click Accorder Nouveau (on the Security menu, click Grant New).

  2. Select Groupes du Domaine par Défaut (Groups on Default Domain).

The list of domain groups is empty and the default domain name in the status bar is empty.

WORKAROUND

To work around this problem, set your security up with local Windows NT Server groups including global groups.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SQL Server version 6.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
