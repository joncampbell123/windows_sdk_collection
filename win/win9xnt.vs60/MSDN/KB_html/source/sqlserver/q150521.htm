INF: Priority Plus Database Server Support Offering

 Last reviewed: January 19, 1998
Article ID: Q150521

The information in this article applies to:
  • Microsoft SQL Server, versions 4.21a, 6.0, and 6.5

SUMMARY

This article describes the Priority Plus Database Server Support offering which is also known as the SQL Server Priority Plus offering.

The Microsoft Priority Plus Database Server Support plan provides a cost effective means of obtaining unlimited annual technical support for your Windows NT-based database system. Support is provided on an unlimited annual basis 24 hours per day, 365 days per year, and is invoked via electronic issue submission.

For additional information about this and other support offerings from Microsoft, please call Microsoft Support Sales at 800-936-3500.

MORE INFORMATION

Priority Plus Database Server Support provides the following benefits:

  • Assured response within 4 hours for customer identified urgent issues.
  • Unlimited annual support for one database server system composed of one or more Windows NT Workstations, Windows NT Servers, and attached clients running Windows NT (Workstation or Server), Windows for Workgroups on MS-DOS, or Windows 95. Also covered are Microsoft SQL Server and all client development and runtime tools and utilities, as well as the Microsoft programming development tools such as Visual C++ and Visual Basic. This offering covers Microsoft products only. Support for non-Microsoft components must be obtained from appropriate alternate support providers.
  • Software patches as they are made available via the support bulletin board and other electronic means.
  • Electronic issue submission.

In order to qualify for this most cost effective support, the following are required:
  • The supported system configuration must be reviewed and accepted by Microsoft Technical Support to be in accordance with the guidelines mentioned above. Changes to the configuration can lead to discontinuation of support by Microsoft, and the support price will be refunded on a prorated basis.
  • Support is provided to a designated contact. Backup support will be provided to other individuals during periods of illness or other normal absences of the designated contact.

Current pricing for Microsoft Priority Plus Database Server support is $2095.00 per year.
