PRB: SQL Admin: Small Unreadable Fonts in Query Window

 Last reviewed: April 29, 1997
Article ID: Q112055

The information in this article applies to: 

  - Microsoft SQL Server Programmer's Toolkit, version 4.2

SYMPTOMS

If SQL Admin query window is used after removing Courier 10,12,15 (VGA RES) from Windows or Windows NT screen fonts, the query window shows an unreadable (very small) font.

CAUSE

When Windows attempts to create a font, it will attempt to use a similar font as a substitute if the specified font does not exist.

WORKAROUND

Reinstall Courier 10,12,15 font.

MORE INFORMATION

Steps to Reproduce Problem:

  1. From Windows (or Windows NT), select the Control Panel.

  2. Choose fonts.

  3. Remove Courier 10,12,15 (VGA RES).

  4. Go to SQL Admin and run query screen.

    The font displays very small characters.
