INF: SQL Server Use of Math Coprocessor

 Last reviewed: April 25, 1997
Article ID: Q58087

The information in this article applies to: 

  - Microsoft SQL Server version 4.2 for OS/2

SQL Server is designed to take advantage of the math Intel 80287 or Intel 80387 coprocessor for floating-point math operations. However, there are performance differences between these two math coprocessors. If an 80287 math coprocessor is running on an 80286 machine, the performance enhancements are not as significant as they are for an 80387 math coprocessor running on an 80386 machine, which will demonstrate a significant increase in performance.

Descriptions of the math functions that perform calculations in floating-point math can be found in the "Mathematical Functions" section of the "Microsoft SQL Server Learning Transact-SQL" manual.
