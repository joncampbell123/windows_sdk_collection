PRB: DATALENGTH Function NULL Argument Documentation Error

 Last reviewed: April 25, 1997
Article ID: Q47050

The information in this article applies to: 

  - Microsoft SQL Server version 4.2 for OS/2
  - Microsoft SQL Server versions 4.2, 4.21, and 4.21a

SYMPTOMS

Version 4.2 of the "Microsoft SQL Server Language Reference" and version 4.2, 4.21, and 4.21a of the "Transact-SQL Reference" for Microsoft SQL Server (for Windows NT) inadvertently dropped the documentation for DATALENGTH. You can execute the 'sp_helpsql datalength' command to obtain the description and syntax of the DATALENGTH function.

STATUS

Microsoft has confirmed that these documentation errors exist in the 4.2 versions of the "Microsoft SQL Server Language Reference" and in version 4.2, 4.21, and 4.21a of Microsoft SQL Server "Transact-SQL Reference." We will post new information here when the documentation has been updated to correct this error.
