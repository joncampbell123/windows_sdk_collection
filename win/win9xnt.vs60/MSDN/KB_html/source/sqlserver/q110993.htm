BUG: Multiple RPCs From Sybase Through Bridge to ODS May Fail

 Last reviewed: April 29, 1997
Article ID: Q110993

The information in this article applies to:
  • Microsoft SQL Server Programmer's Toolkit, version 4.2
BUG# 9680 (4.2b)

SYMPTOMS

When calling multiple RPCs simultaneously from Sybase SQL Server through OS/2 SQL Bridge to OS/2 ODS, one of the RPC calls may fail with the error, 

   Receive bad token from site
and generate the following error in the ODS log: 

   FATAL CONNECT ERROR, Invalid event
Subsequent RPC calls will work fine and other connections will not be affected.

NOTE: Calling multiple RPCs directly from SQL Server to OS/2 ODS works fine.

WORKAROUND

Re-issue the failed RPC call.

STATUS

Microsoft has confirmed this to be a problem in SQL Bridge version 4.2b for OS/2. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
