INF: How to Obtain German SQL Server 6.5 Service Pack 1

 Last reviewed: April 9, 1997
Article ID: Q159673
The information in this article applies to:
  • Microsoft German SQL Server, version 6.5 Service Pack 1

This article describes how to obtain Service Pack 1 for the German version of Microsoft SQL Server 6.5. To see a complete list of bugs that are fixed by this update, see the Fixlist.txt file that accompanies the Service Pack.

NOTE: This Service Pack is for use only by holders of a current valid Microsoft SQL Server version 6.5 license. Most customers will not need to apply this Service Pack. This Service Pack addresses some specific problems that were discovered in SQL Server 6.5 after it began shipping. Microsoft recommends that most customers use the shipping version of 6.5 and only apply this Service Pack if you encounter one of the problems it corrects.

SQL Server German 6.5 Service Pack 1 is available for Intel-based (x86) computers.

FILES AVAILABLE FROM ONLINE SERVICES

The following file(s) are available for download from the Microsoft Software Library: 

 ~ Sq65dsp1.exe (size: 4940819 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
The files in the MSL are compressed, self-extracting executables. After you download the file(s) you need, simply run it to extract the files it contains. Note that the expanded file for each of the platforms contains both the Readme.txt file for installation instructions of the Service Pack, and the Fixlist.txt file describing all bugs corrected with this patch.

If you need more information or assistance regarding this service pack, please contact your primary support provider.
