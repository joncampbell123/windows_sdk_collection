BUG: SQL Administrator & Object Manager Fail with ISQL/w v.6.0

 Last reviewed: April 30, 1997
Article ID: Q134657

The information in this article applies to:
  • Microsoft SQL Server, version 6.0
BUG# NT 10975: (6.00)

SYMPTOMS

The 16-bit versions of SQL Administrator and Object Manager do not work when the version of ISQL/w that comes with Microsoft SQL Server version 6.0 is already running.

When you try to start these programs with the new ISQL/w already running, you will see a brief hourglass cursor, but then nothing else happens and the programs do not start.

WORKAROUND

If you shut down ISQl/w, SQL Administrator and Object Manager will start up normally.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SQL Server version 6.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
