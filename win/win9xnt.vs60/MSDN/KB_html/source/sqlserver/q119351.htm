INF: Error Message List for NIK and SQL Bridge for OS/2

 Last reviewed: April 30, 1997
Article ID: Q119351

The information in this article applies to: 

  - Microsoft SQL Server version 4.2 for OS/2

SUMMARY

Microsoft SQL Server Network Integration Kit (NIK) and SQL Bridge for OS/2 share most of the same error messages, which you typically find in the NIK or SQL Bridge error logs. Although most of these messages listed below are also included in the product manuals, the message lists in the product manuals are not complete and do not include the corresponding error numbers.

MORE INFORMATION

NIK and SQL Bridge error messages are listed below: 

 0      Unknown internal error value
 1     Thread creation failed
 2     Insufficient memory available
 3     Invalid 'nints' value
 4     Invalid buffer received from client
 5     Invalid event specification
 6     Invalid event
 7     Invalid starting position specified
 8     Unable to find free SRV_PROC slot
 9     Unable to set up named pipe
10 Requested data-conversion does not exist 11 Data conversion resulted in overflow 12 Attempt to convert data stopped by syntax error in source field 13 Invalid function parameter 14 Named pipe already exists 15 No longer waiting on client connections 16 No active RPC or parameter value out of range 17 No active RPC or no parameters 18 No active RPC or parameter name not found 19 No active RPC 20 Invalid datatype parameter 21 Unable to set up subchannel. 22 Unable to load server-side Net-Library 23 Unable to find server-side key name in OS2.INI 24 Unable to read from server-side connection 25 Unable to write to server-side connection 26 Unable to close server-side connection 27 Unable to set up server-side connection 28 Unable to load client-side Net-Library 29 Unable to read from client-side connection 30 Unable to write to client-side connection 31 Unable to close client-side connection 32 Network Manager: Unable to connect: SQL Server is unavailable 
       or does not exist.
32 Unable to connect: SQL Bridge is unable to establish a 
       connection with the server.
33 Login Failed. Network Manager only supports 4.x clients. 33 Login Failed. SQL Bridge only supports 4.x clients.
