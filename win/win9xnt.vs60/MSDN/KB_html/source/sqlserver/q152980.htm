BUG: AnsiToOem Check Box Does Not Work in 6.0 Transfer Manager

 Last reviewed: April 15, 1997
Article ID: Q152980
The information in this article applies to:
  • Microsoft SQL Server, version 6.0
BUG #: 9007 (SQL Server 6.00)

SYMPTOMS

The check box for AnsiToOem in SQL Transfer Manager version 6.0 is not set to the same value as the DB-Library option AutoAnsiToOem. Changing it does not affect the AutoAnsiToOem value.

WORKAROUND

To work around this problem, use the SQL Client Configuration Utility to change the AutoAnsiToOem DB-Library option.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SQL Server version 6.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
