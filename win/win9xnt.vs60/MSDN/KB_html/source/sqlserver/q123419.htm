BUG: SQL Server PTK Setup Fails with > 256 Color Video Modes

 Last reviewed: April 30, 1997
Article ID: Q123419

The information in this article applies to:
  • Microsoft SQL Server Programmer's Toolkit, version 4.21a
BUG# 10652 (4.21a)

SYMPTOMS

The SQL Server Programmer's Toolkit for Windows NT will not install on a Microsoft Windows NT version 3.5 system running in a video mode with greater than 256 colors. When the setup program starts to copy the files, setup will hang and never recover.

NOTE: This problem does not apply to Microsoft Windows NT version 3.1

WORKAROUND

A workaround for this is to run the setup program in a video mode with 256 colors or less.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft SQL Server Programmer's Toolkit version 4.21a. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
