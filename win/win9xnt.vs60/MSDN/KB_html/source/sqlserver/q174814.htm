BUG: Large SQC Files That Contain Structures Cause NSQLPREP AV

 Last reviewed: December 18, 1997
Article ID: Q174814
The information in this article applies to:
  • Microsoft Embedded SQL for C Programmer's Toolkit, version 6.5
BUG #: 17082 (Windows: 6.50)

SYMPTOMS

If a large .sqc text file contains many complex structures, NSQLPREP can end unexpectedly because of an access violation.

WORKAROUND

To avoid this problem, break the file into smaller modules.

STATUS

Microsoft has confirmed this to be a problem in the Embedded SQL for C Programmer's Toolkit version 6.5. A supported fix is now available, but has not been fully regression-tested and should be applied only to systems experiencing this specific problem. Unless you are severely impacted by this specific problem, Microsoft recommends that you wait for the next Service Pack that contains this fix. Contact Microsoft Technical Support for more information.
