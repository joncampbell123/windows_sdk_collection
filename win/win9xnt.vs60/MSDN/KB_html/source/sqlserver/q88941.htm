INF: SECURE Sample Application Documentation Incorrect

 Last reviewed: April 28, 1997
Article ID: Q88941

The information in this article applies to: 

  - Microsoft SQL Server version 4.2 for OS/2

Page 24 of the "Microsoft Open Data Services User's Guide" incorrectly states that in the SECURE sample application, LMSQL restricts access to the \PIPE\SQLHIDE\QUERY named pipe by any client other than the LAN Manager system administrator.

Instead, the LAN Manager system administrator should restrict access to this named pipe, if so desired. This is not done by LMSQL.
