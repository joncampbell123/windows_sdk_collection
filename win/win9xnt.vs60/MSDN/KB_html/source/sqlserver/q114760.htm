FIX: How to Obtain SQL Server 4.21a Service Pack 4

 Last reviewed: November 7, 1997
Article ID: Q114760
The information in this article applies to:
  • Microsoft SQL Server, version 4.21a Service Pack 4

This article describes how to obtain Service Pack 4 for Microsoft SQL Server version 4.21a. To see a complete list of bugs that are fixed by this update, see the FIXLIST.TXT file that accompanies the Service Pack, or query on article Q132495 in the Microsoft Knowledge Base.

NOTE: This Service Pack is for use only by holders of a current valid Microsoft SQL Server version 4.21a license. Most customers will not need to apply this Service Pack. This Service Pack addresses some specific problems that were discovered in SQL Server 4.21a after it began shipping. Microsoft recommends that most customers use the shipping version of 4.21a and only apply this Service Pack if you encounter one of the problems it corrects.

SQL Server 4.21a Service Pack 4 is available for Intel (x86), MIPS, and Alpha-based computers.

FILES AVAILABLE FROM ONLINE SERVICES

You can get the files from the Microsoft Software Library (MSL) in the following online services: 

File ID      Filename           Description
-------      --------           -----------
S15883       ALPSP4.EXE         executable for Alpha
S15885       I386SP4.EXE        executable for Intel
S15886       MIPSSP4.EXE        executable for MIPS
  • The Microsoft Download Service 

          Dial (425) 936-6735 to connect to MSDL
      Download <FILENAME.EXE>
  • Anonymous FTP via the Internet 

          ftp ftp.microsoft.com
      Change to the SOFTLIB\MSLFILES directory
      Get <FILENAME.EXE>
  • The Microsoft Network 

          On the Edit menu, click Go To, and then click Other Location.
      Type "mssupport" (without the quotation marks).
      Double-click the MS Software Library icon.
      Find the appropriate product area.
      Locate and Download <FILENAME.EXE>.
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select SQL Server.
      Enter kbfile <FILENAME>.EXE, and click GO!
      Open the article, and click the button to download the file.

    The files in the MSL are compressed, self-extracting executables. After you download the file(s) you need, simply run it to extract the files it contains. Note that the expanded file for each of the platforms contains both the README.TXT file for installation instructions of the Service Pack, and the FIXLIST.TXT file describing all bugs corrected with this patch.

    If you need more information or assistance regarding this Service Pack, please contact your primary support provider.
