INF: RSP and Alternate Pipes Documentation Incorrect

 Last reviewed: April 28, 1997
Article ID: Q88942

The information in this article applies to: 

  - Microsoft SQL Server version 4.2 for OS/2

Page 173 of the Microsoft Open Data Services "Programmer's Reference" shows an incorrect OS2.INI entry for remote stored procedures and alternate pipe names.

The following entry is incorrect: 

   SQLSERVER
   SERVNAME: DBNMPP,\pipe\odspipe1
The correct entry is: 

   SQLSERVER_SERVNAME
   QUERY: DBNMPP,\pipe\odspipe1

Additional query words: ODS
