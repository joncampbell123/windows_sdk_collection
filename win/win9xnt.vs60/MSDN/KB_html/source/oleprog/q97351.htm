OLE Client Implementation Guide Version 1.02

 Last reviewed: May 31, 1996
Article ID: Q97351
The information in this article applies to:
  • Microsoft OLE version 1.0

CLIENT is a file in the Microsoft Software Library that discusses implementing an OLE client. CLIENT contains the OLE Client Implementation Guide version 1.02. 

You can find CLIENT.EXE (size: 130904 bytes) 
                       , a self-extracting file, on these services:
  • Microsoft's World Wide Web site on the Internet 

          On the www.microsoft.com home page, click the Support icon
      Click Knowledge Base, and select the product

          Enter kbfile CLIENT.EXE (size: 130904 bytes) 
                             , and click GO!
      Open the article, and click the button to download the file


  • Internet (anonymous FTP)

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder
      Get CLIENT.EXE (size: 130904 bytes) 

    

  • The Microsoft Network

          On the Edit menu, click Go To, and then click Other Location
      Type "mssupport" (without the quotation marks)
      Double-click the MS Software Library icon
      Find the appropriate product area
      Locate and Download CLIENT.EXE

    

  • Microsoft Download Service (MSDL)

          Dial (206) 936-6735 to connect to MSDL
      Download CLIENT.EXE (size: 130904 bytes)
