ODE97: List of Files You Can Redistribute with ODE Run-Time App

 Last reviewed: February 4, 1998
Article ID: Q163535
The information in this article applies to:
  • Microsoft Office 97 Developer Edition Tools

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

This article contains a list of the files that you can redistribute with a run-time application which you create using the Microsoft Office 97 Developer Edition Tools. This list is an update to the list of files in the Odeeula.txt file, which is located in your \ODETOOLS subfolder.

NOTE: Not all Microsoft Office 97 files can be redistributed with a run- time application. For example, the spell checker files (Mssp_am.lex and Mssp32.dll) and product wizards (such as Wzcnf80.mda, Wzlib80.mde, Wzmain80.mde and Wztool80.mde) are not redistributable, and thus, are omitted from the list below. For more information about some of the files that may not be redistributed, please see the following article in the Microsoft Knowledge Base:
