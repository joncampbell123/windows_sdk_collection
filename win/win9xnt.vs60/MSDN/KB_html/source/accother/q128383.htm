ADT2: Updated ADT License Agreement Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q128383
The information in this article applies to:
  • Microsoft Access Developer's Toolkit version 2.0

SUMMARY

The ADTLIC.EXE file contains the updated license agreement for the Microsoft Access Developer's Toolkit (ADT) version 2.0 and an explanation of the distribution rights for the Data Outline control. 

 ~ ADTLIC.EXE (size: 24206 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following file:

   ADTLA.WRI       A text file containing the ADT License Agreement.
   ADTLETT.WRI     A text file explaining the changed distribution rights
                   for the Data Outline control.
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
