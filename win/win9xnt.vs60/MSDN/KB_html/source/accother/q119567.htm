ADT2: Cannot Distribute Access 2.0 ODBC Driver with ADT Apps

 Last reviewed: May 13, 1997
Article ID: Q119567
The information in this article applies to:
  • Microsoft Access Developer's Toolkit version 2.0

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

There are two Microsoft Access version 2.0 ODBC drivers available. One of the drivers is not licensed for distribution with applications created with the Microsoft Access Developer's Toolkit (ADT); the other is licensed for distribution: 

   Filename       Size (bytes)   Date       Version       Distributable
   --------------------------------------------------------------------
   ODBCJT16.DLL   138480         4/5/94     01.00.15.21   No
   ODBCJT16.DLL   246928         11/18/94   02.00.23.17   Yes

MORE INFORMATION

Version 01.00.15.21 of the ODBC driver is included with Microsoft Office Professional version 4.3 and Microsoft Access version 2.0. You cannot distribute this driver as part of an application even though this file is not listed as one of the files prohibited from distribution in the ADT documentation.

Version 02.00.23.17 of the ODBC driver is included with the ODBC Desktop Drivers version 2.0, the ODBC Driver Pack version 2.0 Fulfillment Kit, and the ODBC Software Development Kit (SDK). You can distribute this driver as part of an application.

If you need an ODBC driver to distribute with an application created with the ADT, you can purchase the ODBC Driver Pack version 2.0 Fulfillment Kit. For more information about the ODBC Driver Pack 2.0 Fulfillment Kit, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q125235
   TITLE     : ODBC Driver Pack 2.0 Fulfillment Kit Q&A

REFERENCES

For more information about the two available Microsoft Access ODBC drivers, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q124815
   TITLE     : Explanation of the Two Available MS Access Drivers
