XFOR: Cannot Enable (Appletalk) MTA Service NT SP3

 Last reviewed: October 22, 1997
Article ID: Q169839
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0
  • Microsoft Exchange Server, version 5.0

SYMTOMS =======

After you enable the Microsoft Mail Connector (Appletalk) MTA, the following error might be reported when you attempt to apply the settings on the Microsoft Mail Connector Properties page: 

   Microsoft Mail Connector (Appletalk) MTA service was not created because the
   counters for the Performance Monitor could not be installed.

STATUS

Microsoft has been confirmed this to be a problem in Microsoft Windows NT 4.0 Service Pack 3. We are currently researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.
