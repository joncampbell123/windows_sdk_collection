Cannot Find Home Directory Share on Windows NT Server

 Last reviewed: September 9, 1996
Article ID: Q131785
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

When you use Windows for Workgroups you cannot connect to your home directory share on a Windows NT Server. The following error message appears: 

   Error 53: The computer name specified in the network path cannot be
   located.

CAUSE

This problem occurs when the Windows NT server name is 13 or more characters long.

WORKAROUND

To work around this problem, use a server name that is 12 characters or fewer.
