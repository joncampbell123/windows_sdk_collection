TCP/IP-32 Version 3.11 Does Not Include Terminal Font

 Last reviewed: September 9, 1996
Article ID: Q121317
The information in this article applies to:
  • Microsoft TCP/IP-32 for Windows for Workgroups, version 3.11

SUMMARY

Beta versions of Microsoft TCP/IP-32 included a font named "Terminal" in the Telnet application. This font is not included in the released version 3.11 of the TCP/IP-32 Telnet application.

RESOLUTION

A new version of TELNET.EXE is available that includes the Terminal Font.

STATUS

Microsoft has confirmed this to be a problem in TCP/IP-32. This problem has been corrected in an updated TCP/IP-32 version 3.11a. For information on the availability of TCP/IP-32 3.11a, see article Q122544 in the Microsoft Knowledge Base.
