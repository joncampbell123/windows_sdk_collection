Doc for IBM Auto 16/4 Token Ring ISA Card List Wrong Disk Number

 Last reviewed: September 18, 1996
Article ID: Q137728
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51
  • Microsoft LAN Manager versions 2.0, 2.1, and 2.2
  • Microsoft Windows for Workgroups version 3.11

SUMMARY

The documentation for the IBM Auto 16/4 Token Ring ISA card lists the wrong disk number for the location of the network drivers.

MORE INFORMATION

In several places the IBM Auto 16/4 Token Ring ISA card documentation incorrectly states that the drivers can be found on Disk 2 under the appropriate directory. Foe example, DOS, NT, WFW, MSLANMAN.DOS, MSLANMAN.OS2. These drivers are actually found on Disk 1.

The text also states that the diagnostic utilities are on Disk 1, however, they are actually on Disk 2.

When you insert the incorrect disk, the install routine does not find the file, OEMSETUP.INF.

REFERENCES

IBM Auto 16/4 Token Ring ISA Adapter Installation and Testing Instructions, Part Number: 13H9525
