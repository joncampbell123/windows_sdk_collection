Updated AMI BIOS Needed for Large IDE Drives and 64+ MB Memory

 Last reviewed: September 9, 1996
Article ID: Q123739
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

AMI BIOSs dated prior to 7/25/94 do not provide Logical Block Addressing (LBA) and the necessary enhanced registers to work correctly with IDE hard drives larger than 540 megabytes.

AMI BIOSs dated prior to 10/10/94 do not support more than 64 MB of RAM on ISA systems. AMI BIOSs dated 10/10/94 and later support up to four gigabytes on ISA systems.
