How To Obtain TCP/IP-32 3.11b for Windows for Workgroups

 Last reviewed: October 14, 1997
Article ID: Q99891
3.11 WINDOWS kbnetwork kbfile kbfasttip

The information in this article applies to:

  • Microsoft Windows for Workgroups version 3.11
  • Microsoft TCP/IP-32 for Windows for Workgroups, version 3.11b

SUMMARY

Microsoft TCP/IP-32 version 3.11b is an update to Microsoft's TCP/IP-32, a 32-bit protected mode VxD TCP/IP network protocol stack for Windows for Workgroups version 3.11. This update is a complete refresh of TCP/IP-32 and contains all components of the product. For information on what is fixed in this TCP/IP-32 Update, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   T C P U P D A T E

MORE INFORMATION

TCP32B.EXE is a self-extracting file containing the TCP/IP-32 version 3.11b files. You can download it from the following electronic services:

- Microsoft's World Wide Web Site on the Internet 

      On the www.microsoft.com home page, click the Support icon
      Click Knowledge Base, and select the appropriate product area
      Enter kbfile TCP32B.EXE (size: 690031 bytes) 
                             , and click GO!
      Open the article, and click the button to download the file
- Microsoft Download Service (MSDL) 
      Dial (425) 936-6735 to connect to MSDL
      Download TCP32B.EXE (size: 690031 bytes)
  • Internet (anonymous FTP) 

          ftp://ftp.microsoft.com/softlib/mslfiles
      -or-
      ftp://ftp.microsoft.com/bussys/clients/wfw
      Get TCP32B.EXE (size: 690031 bytes)
  • The Microsoft Network 

          On the Edit menu, click Go To, and then click Other Location
      Type "mssupport" (without the quotation marks)
      Double-click the MS Software Library icon
      Find the appropriate product area
      Download TCP32B.EXE (size: 690031 bytes)
