How to Disable Persistent Connections for Network Client 3.0

 Last reviewed: September 9, 1996
Article ID: Q122550
The information in this article applies to:
  • Microsoft Network Client version 3.0 for MS-DOS

To disable persistent connections for Microsoft Network Client version 3.0 for MS-DOS, do the following before you log on:

  1. Using a text editor, change the RECONNECT entry in the [Network] section of SYSTEM.INI to the following: 

          RECONNECT=NO

  2. Rename or delete the CONNECT.DAT file.

NOTE: Both SYSTEM.INI and CONNECT.DAT are located in the Network Client directory, which is usually called C:\NET.
