How to Obtain the IEEE P1284 (ECP) Specification

 Last reviewed: December 12, 1995
Article ID: Q140581
The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows 95

SUMMARY

To get the full P1284 specification, contact:

Ask*IEEE PO Box 4327 Burlingame, CA 94011-4327 (800)949-4333

MORE INFORMATION

The IEEE P1284 specification contains information on how the new Enhanced Centronics Port (ECP) works. Microsoft used this specification in the design and implementation of Lpt.vxd, the port driver that supports ECP ports.

Additional reference words: 4.00 parallel LPT
