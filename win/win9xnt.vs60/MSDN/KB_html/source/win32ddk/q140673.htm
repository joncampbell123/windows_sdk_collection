DOC: EngCreatePalette pulColors Parameter Description Error

 Last reviewed: January 8, 1997
Article ID: Q140673
The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows NT, versions 3.5, 3.51

SUMMARY

There is an error in the online documentation for the EngCreatePalette function. This function is used for both Win32 display drivers and Win32 printer drivers.

The last line of the pulColors parameter is misleading and should be ignored. In other words, ignore the line that says, "These RGB colors are assumed to be relative to the CALDATA supplied in the DEVINFO structure." There is no CALDATA member in the DEVINFO structure.

This documentation error has been corrected in the Windows NT DDK version 4.0 online documentation.
