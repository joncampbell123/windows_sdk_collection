INFO: Windows Postscript Level Support

 Last reviewed: June 13, 1997
Article ID: Q140658
The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows 95

SUMMARY

Windows 95's Pscript.drv provides Postscript Level 2 support, but Windows 3.xx's Pscript.drv provides only Postscript Level 1 support.

MORE INFORMATION

If your Postscript printer supports Postscript Level 2 and you are running Windows 95, you can go to your printer's Postscript Advanced Property Sheet and set the Use Postscript Level 2 features button. This will enable Pscript.drv to send Postscript Level 2 commands.
