DOCFIX: Error in SURFOBJ Description

 Last reviewed: July 18, 1996
Article ID: Q131774
The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows NT, version 3.5, 3.51

There is an error in the online documentation for the SURFOBJ structure. The SURFOBJ structure is used for both Win32 display drivers and Win32 printer drivers. The last line of the description is misleading and should be ignored; in other words, ignore the line that says, "The driver has access to the following public members of the SURFOBJ structure:"

This documentation error has been corrected in the Windows NT DDK version 4.0 online documentation.
