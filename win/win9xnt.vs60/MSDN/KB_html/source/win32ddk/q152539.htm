Windows 95 Multimedia MCI Device Drivers Must Be 16-bit

 Last reviewed: June 18, 1996
Article ID: Q152539
The information in this article applies to:
  • Microsoft Win32 Device Driver Kit (DDK) for Windows 95, version 4.0

SUMMARY

Windows 95 multimedia MCI device drivers must be 16-bit drivers. The Windows 95 MCI architecture ensures compatibility with MCI drivers written for previous versions of the Windows operating system. 32-bit MCI drivers are not supported under Windows 95. A 16-bit MCI driver may, however, thunk to a 32-bit DLL.

REFERENCES

See the Windows 95 DDK for more information on Windows 95 multimedia MCI device drivers. See the Win32 SDK for more information on thunking.
