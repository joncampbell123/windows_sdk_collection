HOWTO: Search for Win32 DDK Articles Using KBKeywords

 Last reviewed: March 10, 1998
Article ID: Q148730
The information in this article applies to:
  • Microsoft Win32 Device Driver Kit (DDK) for Windows NT, versions 3.1, 3.5, 3.51, 4.0
  • Microsoft Win32 Device Development Kit (DDK) for Windows 95

SUMMARY

The Microsoft Knowledge Base (KB) is categorized by using keywords. This article lists the keywords specific to articles in the Microsoft Win32 Device Development Kit (DDK) for Windows NT/Windows 95 Collection.

MORE INFORMATION

Microsoft Win32 DDK KBKeywords Major Keywords

Each KB article in the Win32 DDK collection may contain one or more product- specific keywords (called KBKeywords) that places the article in an appropriate Win32 DDK category. Here are the topics and the corresponding KBKeywords. 

   Major Topic                KBKeyword
   ---------------------------------------

   Communications             NTDDKComm
   Debugging                  NTDDKDebug
   Installable File Systems   NTDDKIFS
   Input                      NTDDKInput
   General kernel mode        NTDDKKMode
   Multimedia                 NTDDKMMedia
   NIC Drivers                NTDDKNDIS
   Printing                   NTDDKPrnt
   Storage devices            NTDDKStorage
   TDI/Transports             NTDDKTDI
   Video                      NTDDKVideo
   Virtual device drivers     NTDDKVDD

Product-Specific Keywords

You can use the KBKeywords to organize Win32 DDK articles or to search for specific groups of Win32 DDK articles.

For information about KBKeywords for other Microsoft developer products, please query the Microsoft Knowledge Base using these keywords: 

   dskbguide and kbkeyword

KB-wide Keywords

Each article in the Win32 DDK collection also contains at least one generic, KB-wide keyword. The KB-wide keywords are standard throughout the Microsoft Knowledge Base, appearing in all KB articles, regardless of product. You can use the KB-wide keywords to organize all KB articles or to search for articles across several Microsoft products. For more information about these KB-wide keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
