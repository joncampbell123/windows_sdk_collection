INFO: Floating-Point Math Not Supported in a Kernel-Mode Driver

 Last reviewed: February 17, 1998
Article ID: Q126547

The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows NT, versions 3.5, 3.51

SUMMARY

Floating-point math is not supported in a Windows NT kernel-mode driver. Attempts to use floating-point math results in link errors or run-time failures. There are no plans to add floating-point math support to kernel- mode drivers.

Version           : WINNT:3.5,3.51;
Platform          : winnt
