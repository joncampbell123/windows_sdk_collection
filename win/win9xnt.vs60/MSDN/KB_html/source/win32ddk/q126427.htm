DOCFIX: IoAcquireCancelSpinLock Comments Are Incorrect

 Last reviewed: April 7, 1997
Article ID: Q126427
3.50 WINDOWS NT kbprg kbdocerr

The information in this article applies to:

  • Microsoft Win32 Device Development Kit (DDK) version 3.5

The Win32 DDK version 3.5 online documentation incorrectly references a function in the Comments section of IoAcquireCancelSpinLock. The documentation states: 

   The holder of the cancel spin lock should release it promptly by
   calling IoReleaseSpinLock.
The function IoReleaseSpinLock should be IoReleaseCancelSpinLock.

This documentation error is corrected in the Win32 DDK version 3.51 online documentation.
