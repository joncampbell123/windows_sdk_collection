FP1: "Msvcrt20.Dll Is an Out of Date Version" Starting FrontPage

 Last reviewed: March 17, 1998
Article ID: Q143091
The information in this article applies to:
  • Microsoft FrontPage for Windows, version 1.0

SYMPTOMS

When you start FrontPage Explorer or FrontPage Editor, you may receive the following error message: 

   Msvcrt20.dll is an out of date version.

CAUSE

FrontPage was unable to update a dynamic link library (.dll) file during setup. This problem occurs when another program is running during setup, and setup is not able to update the file.

RESOLUTION

Make sure all programs are closed, rename the Msvcrt20.dll file, and then reinstall FrontPage.
