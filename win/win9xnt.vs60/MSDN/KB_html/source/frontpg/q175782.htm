FP98: Can't Type, System Beeps Typing in Text Overlay on Image

 Last reviewed: March 18, 1998
Article ID: Q175782
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you insert a text overlay on a small image, you cannot move the cursor and you hear a system beep each time you type a character.

CAUSE

The problem occurs when you use a small image and you specify a large point size.

RESOLUTION

To work around this problem, use either of the following methods:

Method 1: Increase the pixel size of the image

Select the image and drag the sizing handle outward to enlarge the image.

Method 2: Decrease the font size

  1. On the Format menu, click Font.

  2. In the Font Size box, select a smaller point size.

  3. Click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
