FP97: FP Explorer Hangs When You Attempt to Save File

 Last reviewed: March 18, 1998
Article ID: Q161966
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SYMPTOMS

If you attempt to save a page that was modified by someone else since you opened it, you receive the following message from FrontPage Explorer: 

   A more recent version of <page> has been saved to the server by
   <username> on <date time>.

   Do you want to replace this with your file?
If you do not dismiss this message box by clicking Yes, No, or Cancel within two minutes, FrontPage Explorer stops responding.

This problem occurs when using the Windows 95 operating system; it does not occur when using the Windows NT operating system.

RESOLUTION

Click Yes, No, or Cancel in the message box before two minutes have expired.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
