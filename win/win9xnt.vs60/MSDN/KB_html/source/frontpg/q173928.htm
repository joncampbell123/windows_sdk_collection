FP98: Changing Page Title Won't Change Title in Navigation View

 Last reviewed: March 18, 1998
Article ID: Q173928
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you change the page title in the Page Properties dialog box in FrontPage Editor, the change to the title is not reflected in the folder pane in Navigation view in FrontPage Explorer.

CAUSE

By design, if you change the title in the Page Properties dialog box in FrontPage Editor, the title is not updated in the folder pane in Navigation view in FrontPage Explorer. The folder pane in Navigation view always retains the default label (title).

WORKAROUND

To work around this behavior, use either of the following methods.

Method 1: Change the title in the folder pane in Navigation view

Use this method if you have not changed the title in FrontPage Editor. The change will be reflected in both FrontPage Editor and FrontPage Explorer.

  1. In FrontPage Explorer, click Navigation on the View menu.

  2. Do either of the following: 

        - Click the page whose title you want to change and then press F2.


           -or-


        - Right-click the page whose title you want to change and then click
      Rename on the menu that appears.

  3. On the Tools menu, click Recalculate Hyperlinks.

Navigation view will reflect the new title.

Method 2. Re-synchronize the titles

Use this method if you have already changed the title in FrontPage Editor and you want to change the title in FrontPage Explorer.

  1. If the page is open in FrontPage Editor, close the file.

  2. In FrontPage Explorer, click Navigation on the View menu.

  3. Click the page whose label you want to change.

  4. On the Edit menu, click Rename.

  5. Type the label you want so that it matches the title you entered in the Page Properties dialog box in FrontPage Editor.

  6. Press ENTER.

The page title and the label you see in Navigation view will match.
