FP: Words That Are Ignored by the WebBot Search Component

 Last reviewed: March 18, 1998
Article ID: Q156861

The information in this article applies to:

  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage for Windows, version 1.0, 1.1

SUMMARY

This article lists all the words that will be ignored when you attempt to search a Web using the Search Form (FrontPage 98) or the WebBot Search component (FrontPage 1.0, 1.1 and 97).

NOTE: Both the singular and the plural versions of the words will be ignored.

MORE INFORMATION

You can not configure this word list in FrontPage versions 1.1 and 1.0.

If you have installed Microsoft Index Server on a Microsoft Internet Information Server, FrontPage versions 98 and 97 will use the Microsoft Index Server for all its search capabilities on the Internet Information Server only. The Microsoft Index Server has its own configuration parameters. For more information about Microsoft Index Server, see 

   http://www.microsoft.com/ntserver/search/index2/ixinstal.asp
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com
