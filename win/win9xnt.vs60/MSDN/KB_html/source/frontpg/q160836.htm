FP97: WebBot Looks Different in Previous Version Server

 Last reviewed: March 18, 1998
Article ID: Q160836
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SYMPTOMS

When you use FrontPage 97 to create web pages on a FrontPage 1.1 extended server, certain WebBot components may appear in bold text surrounded by brackets.

CAUSE

Some FrontPage functions are implemented only by the FrontPage 97 server extensions.

If the browser encounters a function that cannot be evaluated with FrontPage 1.1 extensions, the function is represented by text enclosed in brackets.

RESOLUTION

To avoid this text representation, use either of the following methods.

Method 1

Upgrade the server extensions to the FrontPage 97 version.

The most recent versions of the FrontPage Server Extensions are posted at the following Web site: 

   http://www.microsoft.com/frontpage.
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/

Method 2

Avoid using the features described in the "More Information" section of this article.

MORE INFORMATION

The following FrontPage 97 features and WebBot components may produce the symptoms described in this article.

Field Validation Forms

Field validation will not function if authored on or copied to a version 1.1 extended server. The only validation enforced in this configuration is validation of maximum length. If the validated form is authored on a 97 extended server and published to a version 1.1 extended server, it will display the text "[Generated Script Bot]," and next to each validated field you will see "[Validation Bot]." If the validated form is directly authored against the version 1.1 extended server, you will see "[Validation Bot]" next to each validated field.

WebBot Hit Counter

This is a custom WebBot component, the code for which is in the FrontPage 97 Software Development Kit (SDK). Note that similar symptoms will occur whenever a WebBot does not exist on the server. If you are using a custom WebBot component, you will see this symptom if the custom WebBot component is not installed on the destination server.

Discussion Web Wizard

A Discussion web created in FrontPage 97 against a version 1.1 extended server will display the text "[EvaluateFirst Bot]" on the contents page.
