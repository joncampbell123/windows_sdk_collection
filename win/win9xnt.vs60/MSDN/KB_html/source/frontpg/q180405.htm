FP98: Sample ASP Wizard Available for Download

 Last reviewed: March 18, 1998
Article ID: Q180405
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SUMMARY

Microsoft has released a sample Active Server Page Database Wizard that you can use with FrontPage 98. This file is available for download from the Microsoft Software Library: 

 ~ aspwiz.exe (size: 2088160 bytes)
For additional information about system requirements and installation instructions, please see the following Microsoft World Wide Web site: 

   http://support.microsoft.com/support/frontpage/fp98/aspwiz.asp

MORE INFORMATION

For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
