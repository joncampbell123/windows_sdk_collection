FP: FrontPage Client, Personal Web Server Run Only on i386

 Last reviewed: March 18, 1998
Article ID: Q153630

The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage for Windows, version 1.0

SUMMARY

Microsoft FrontPage client software and the Personal Web Server will function only on an Intel 386 (i386) platform running Windows 95 or Windows NT 3.51 or later. Any other configurations are unsupported.

MORE INFORMATION

The FrontPage client and the Personal Web Server were tested only on the i386 platform; therefore, it is not supported on RISC-based platforms such as Alpha and MIPS. The FrontPage Server Extensions are compatible with many UNIX based systems. For more information about FrontPage Server Extensions, see the following page on the World Wide Web: 

   http://www.microsoft.com/frontpage/Freestuff/fs_fp_extensions.htm
