FP98: Table Properties Not Available After Erasing Table Lines

 Last reviewed: March 18, 1998
Article ID: Q174013
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you use the Eraser tool on the Table toolbar to erase lines in a table, the commands on the Table menu become unavailable (dimmed) except for the Draw Table command.

CAUSE

This behavior only occurs when you use the Eraser tool on the Draw Table menu to erase lines between cells.

WORKAROUND

To work around this behavior and make the commands on the Table menu available, click in any table cell.

REFERENCES

For more information about tables, click the Index tab in FrontPage Help, type the following text: 

     Tables
and then double-click the selected text to go to the "About Tables" topic.
