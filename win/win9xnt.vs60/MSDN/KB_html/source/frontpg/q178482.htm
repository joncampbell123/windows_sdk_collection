FP98: Channel Definition File Does Not Recognize New File Names

 Last reviewed: March 18, 1998
Article ID: Q178482
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

Renaming a file that is included in your Channel breaks the link to the renamed page.

CAUSE

This problem occurs because the new file name does not appear in the Channel Definition Format file.

WORKAROUND

To make the link to the renamed file work, re-create your Channel Definition Format file. To do this, double-click the Channel Definition Format file in the FrontPage Explorer to reopen the Channel Definition Wizard.
