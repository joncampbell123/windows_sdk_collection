FP: Can't Resolve FTP or Gopher Links or Links Generate Errors

 Last reviewed: March 18, 1998
Article ID: Q151557
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage for Windows, version 1.1

SUMMARY

FrontPage does not include File Transfer Protocol (FTP), Gopher, Telnet, or Wide Area Information Service (WAIS) protocol services (although these protocols are included in the drop-down menus for creating links in the FrontPage Editor). When you select the link in a browser or follow it in the FrontPage Editor, it may not be resolved or it may generate an error message.

MORE INFORMATION

FTP, Gopher, Telnet and WAIS servers are available through several third- party vendors. FTP and Gopher services can also be implemented through Microsoft Internet Information Server (IIS). For implementation and configuration of IIS, please consult the Readme file included with IIS.

The Microsoft Internet Information Server can be obtained from the Microsoft World Wide Web site at 

   http://www.microsoft.com/infoserv
The Microsoft Internet Information Server is not supported by FrontPage at this time; FTP and Gopher services, however, can be used in conjunction with FrontPage.
