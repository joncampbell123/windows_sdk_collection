FP1: Second Page with Frameset Overwrites Previous Pages

 Last reviewed: March 17, 1998
Article ID: Q154679
The information in this article applies to:
  • Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

If you create a frameset from the Link To New Page tab with the Frames Wizard using the same template as an existing frameset, the existing frameset pages will be overwritten without warning.

WORKAROUND

In webs where you want to base multiple framesets on the same template, start the Frames Wizard by clicking New on the File menu in FrontPage Editor rather than clicking the New Page tab in the Link Properties dialog box.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
