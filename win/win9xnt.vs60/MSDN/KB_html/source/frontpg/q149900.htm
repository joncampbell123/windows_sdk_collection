FP1: How to Create Online Books w/ Chapters in Separate Folders

 Last reviewed: March 17, 1998
Article ID: Q149900
The information in this article applies to:
  • Microsoft FrontPage for Windows, versions 1.0, 1.0a

SUMMARY

This article describes how to create an online book such that the chapters are stored in subdirectories of the Content folder.

MORE INFORMATION

In a FrontPage Web, all documents are stored in a single directory, making file management more difficult for large documents such as online books. To create an online book with multiple subdirectories, use the following steps:

  1. Start FrontPage Explorer.

  2. On the File menu, click New Web.

  3. Select Empty Web from the list and insert the name of the book as the Web name.

  4. Click the icon to open FrontPage Editor.

  5. On the File menu, click New.

  6. Select Table Of Contents from the list and set the page title. On the File menu, click Save, and specify Index.htm as a file name.

  7. Use Windows Explorer or File Manager to create the subfolders in the Content folder where the chapters will be located.

  8. Create your content documents using FrontPage Editor or any other HTML editor and save the documents in the appropriate chapter folder.

  9. Open FrontPage Explorer. Click Recalculate Links on the Tools menu to refresh the table of contents page.
