FP: Can't Choose Collate or Selection in Print Dialog Box

 Last reviewed: March 18, 1998
Article ID: Q167860
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SYMPTOMS

The Collate Copies and Selection options are unavailable (dimmed) in the Print dialog box (click Print on the File menu) when you attempt to print a FrontPage file.

CAUSE

The Print dialog box in Microsoft FrontPage is a Windows 95 common dialog box that does not support the printing of either selected text or collating.

WORKAROUND

To work around this problem, copy your text to an application that supports collating or printing a selection of text.
