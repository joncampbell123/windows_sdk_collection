FP: Errors Attempting to Start FrontPage Personal Web Server

 Last reviewed: March 18, 1998
Article ID: Q154262

The information in this article applies to: 
  
  - Microsoft FrontPage 98 for Windows
  - Microsoft FrontPage 97 for Windows with Bonus Pack
  - Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

When you attempt to start the FrontPage Personal Web Server, you may receive one of the following error messages:

FrontPage 98

 
   VHTTPD32 Syntax error on line 70 of c:\FrontPageWebs\Server/conf/httpd.cnf.
-or- 
 
   FPEXPLOR caused an invalid page fault in module FP30OUTL.DLL.

FrontPage 97 and FrontPage 1.1

 
   VHTTPD32 caused an invalid page fault in module VHTTPD32.EXE.

CAUSE

This error message will be displayed if the system clock on your computer is set to year 2037 or later.

RESOLUTION

Set the system clock to the proper date.
