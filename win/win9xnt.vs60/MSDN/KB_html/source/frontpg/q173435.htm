FP98: What are Shared Borders and How to Enable, Disable Them

 Last reviewed: March 18, 1998
Article ID: Q173435
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SUMMARY

FrontPage 98 introduces Shared Borders. Shared Borders is a feature that allows you to automatically apply one or more common border elements to all files within a Web or to specific files through a single command.

MORE INFORMATION

Shared Borders are page regions--top, bottom, left and right--used to hold content that you want to appear consistently throughout all pages in the current Web. Shared Borders are stored in four files:

  • Top.htm
  • Bottom.htm
  • Left.htm
  • Right.htm

Shared borders are stored in the _borders folder in the current Web. You can edit the shared borders files in one of two ways:
  • You can open the file that contains the shared border you want to modify.

-or-
  • You can open a page in the Web that uses the shared border.

NOTE: In FrontPage Editor, when you open a page that contains shared borders, the Web page appears to use a table format. If you view the HyperText Markup Language (HTML) code by clicking the HTML tab, you will only see the HTML for the central page. You will not see any of the HTML code for the shared borders.

In FrontPage Explorer, you can enable or disable shared borders for the entire Web or you can enable or disable them on a page-by-page basis.

To enable or disable shared borders from an entire FrontPage Web, follow these steps:

  1. In FrontPage Explorer, click Shared Borders on the Tools menu.

  2. Under Borders To Include, click to select or clear the check box to the right of the preview pane.

  3. Click OK.

To enable or disable shared borders on a page-by-page basis, follow these steps:

  1. In FrontPage Editor, open the page that you want to modify.

  2. On the Tools menu, click Shared Borders.

  3. Under Borders To Include, click "Set for this page only."

  4. Click to select or clear the check box to the right of the preview pane.

  5. Click OK.

REFERENCES

For more information about using Shared Borders, click the Index tab in FrontPage Help, type the following text 

   Shared borders, procedures
and then double-click the selected text to go to the "Using Navigation Bars and Shared Borders" topic.
