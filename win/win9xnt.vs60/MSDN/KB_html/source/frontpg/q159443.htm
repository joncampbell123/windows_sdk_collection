FP97: Page Fault After Using Desktop Themes Control Panel

 Last reviewed: March 18, 1998
Article ID: Q159443
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SYMPTOMS

After you install Microsoft FrontPage 97 for Windows, the following message appears when you use the Desktop Themes Control Panel program: 

   THEMES caused an invalid page fault in module THEMES.EXE at
   014f:0040abc6.
NOTE: The address of the invalid page fault may vary.

CAUSE

This behavior occurs when the following conditions are met:

  • You install Microsoft Plus! For Windows 95.

    -and-

  • You Install FrontPage 97 for Windows.

    -and-

  • You use Desktop Themes in the Control Panel.

This behavior occurs because the Desktop Themes Control Panel program, Themes.exe, is incompatible with the JPEG (.jpg) file filter, Jpeg32.flt, that is installed with Microsoft FrontPage 97 for Windows.

RESOLUTION

Microsoft FrontPage 97 for Windows Bonus Pack CD includes an updated version of the Themes.exe program. To resolve the behavior, install the updated version of Themes.exe.

If you previously installed Themes from the Plus! Pack for Windows 95, and if you installed the JPEG graphics filter with FrontPage 97, you can upgrade your existing Themes program by getting the Microsoft Plus! Themes Patch file from one of the subfolders of <CD DRIVE>:\Patch\Themes on the FrontPage 97 with Bonus Pack CD (where <CD DRIVE> is the drive letter for your CD-ROM drive).

To install the updated version of Themes.exe, use the following steps:

  1. Locate the Themes.exe patch file by language. Each subfolder contains a Themes.exe file in a particular language. The language codes are: 

          BRZ   Portuguese (Brazilian)
      CHT   Chinese (Traditional)
      DAN   Danish
      DEU   German
      ENU   English
      ESP   Spanish
      FIN   Finnish
      FRA   French
      ITA   Italian
      JPN   Japanese
      NL   Dutch (Standard)
      NOR   Norwegian (BokMal)
      SV   Swedish

  2. Locate the folder on your hard disk that contains the old version of Themes.exe. If you installed Plus! in the default location, then the file Themes.exe is located in the \Program Files\Plus! folder.

    In Windows 95, click the Windows Start button, point to Find, and then click Files Or Folders. Search for the Themes.exe file on your hard disk.

    In Windows NT Workstation 3.51, click Search on the File menu in File Manager, and then search for the file Themes.exe.

  3. Delete the file Themes.exe from your hard disk.

  4. Copy the new Themes.exe patch file (which you obtained in step 1) to the folder on your hard disk that contained the earlier version of the Themes.exe file.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft FrontPage 98 for Windows.
