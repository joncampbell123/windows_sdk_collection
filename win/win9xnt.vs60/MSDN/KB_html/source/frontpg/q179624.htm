FP98: Error Browsing Page with Include Page Component

 Last reviewed: March 18, 1998
Article ID: Q179624
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you open or refresh a page that contains an Include Page component that points to an Active Server Page that contains a database region, you may receive the following error message: 

   The contents of a FrontPage component have been modified.
   These contents will be overwritten when you save this page.

CAUSE

FrontPage Editor recognizes the FrontPage component used in the Database Region Wizard. To preserve the content, FrontPage Editor rewrites the code so that the Database Region Wizard will function properly.
