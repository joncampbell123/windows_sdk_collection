FP1: Configuration and Error Log Files

 Last reviewed: March 17, 1998
Article ID: Q143088
The information in this article applies to:
  • Microsoft FrontPage for Windows, version 1.0

SUMMARY

This article lists the configuration and error log files installed with FrontPage 1.0.

MORE INFORMATION

FrontPage Log files

   Vinstall.log     Installed in the Windows folder.
                    Check this file if you have problems with installation
                    or out of memory errors.

   Error.log        Installed in the Vermeer\Httpd\Logs folder.
                    Check this file if you have server (HTTP) errors.

   Vermeer.log      Installed in the root directory. Check this log
                    file for server errors and installation errors.

   Vermeer.out      Installed in the root directory. Check this log file
                    for unusual errors

FrontPage Configuration Files

   Services.cnf     Installed in the Content\_vti_pvt folder.
                    This file lists the webs on your server.

   Service.cnf      Installed in the Vermeer\_vti_pvt folder.
                    This lists Web parameters.

   Vermeer.ini      Installed in the Windows folder. This file
                    lists the directories and aliases used by FrontPage.

   Fpreg.reg        Installed in the Vermeer folder. This is the
                    registry file for FrontPage Explorer and FrontPage
                    Editor.

FrontPage Personal Web Server Configuration Files

   Httpd.cnf        Installed in the Vermeer\Httpd\Conf folder.
                    This file identifies the server (ServerName), port
                    number, and other server configuration settings.

   Access.cnf       Installed in the Vermeer\Httpd\Conf subfolder.
                    This file contains the global access configuration.

   Srm.cnf          Installed in the Vermeer\Httpd\Conf subfolder.
                    This file contains server resource information, such
                    as DocumentRoot, mime content type controls,
                    aliasing and redirection, and automatic folder
                    indexing.
