FP98: Problems Creating FrontPage 98 Webs on FrontPage 97 Server

 Last reviewed: March 18, 1998
Article ID: Q174816
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

If you create a Web using Microsoft FrontPage 98 on a server on which the FrontPage 97 Server Extensions are installed, you will not receive any warnings that FrontPage 98 features, such as shared borders, page banners, and navigation bars will not work.

CAUSE

Shared borders, page banners, and navigation bars require the FrontPage 98 Server Extensions to function. The FrontPage 97 Server Extensions do not support the new FrontPage 98 features.

RESOLUTION

To use FrontPage 98 content and retain all of the new features and functionality, install the FrontPage 98 Server Extensions on the Web server.

To maintain compatibility with the FrontPage 97 Server Extensions, do not use FrontPage 98 features in your Web content.

For additional information about using FrontPage 98 with the FrontPage 97 Server Extensions, please see the following article in the Microsoft Knowledge Base:
