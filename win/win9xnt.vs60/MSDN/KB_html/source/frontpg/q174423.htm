FP98: Duplicate Files Listed in Create Hyperlink Dialog Box

 Last reviewed: March 18, 1998
Article ID: Q174423
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you edit or insert a hyperlink on a page in FrontPage Editor you will see the currently open pages listed more than once in the Edit Hyperlink or Create Hyperlink dialog boxes. An icon that has a quill identifies the currently opened pages. The currently opened pages appear at the top of the list of files in either dialog box. A normal document icon identifies all the pages in the current Web. These files are listed at the bottom of either dialog box.

CAUSE

Opened pages appear in the Edit Hyperlink or Create Hyperlink dialog box with a quill icon. Unopened pages in the current Web appear with a normal document icon.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
