FP1: Error Starting FrontPage if Open Transport Not Installed

 Last reviewed: March 17, 1998
Article ID: Q166358
The information in this article applies to:
  • Microsoft FrontPage for the Power Macintosh, version 1.0

SYMPTOMS

If you attempt to start Microsoft FrontPage for the Power Macintosh on a computer that does not have Apple Open Transport installed, you will receive the following error message: 

   The application "FrontPage Explorer" could not be opened because
   "OTInetClientLib" could not be found.

CAUSE

Apple Open Transport is not installed. Microsoft FrontPage is a client/server program that requires network components. Apple Open Transport provides this functionality.

RESOLUTION

Install Apple Open Transport, version 1.1, or later.

MORE INFORMATION

Open Transport is shipped with the Macintosh Operating System Software, version 7.6 and later. It is also available as a separate program if you are running an earlier version of the Macintosh Operating System Software.
