FP98: Page Banner Blank if it Contains Quotation Mark

 Last reviewed: March 18, 1998
Article ID: Q181136
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

The page banner appears blank after you enter a title in FrontPage Editor.

You see a blank Page Banner in FrontPage Editor when you use quotation marks in the title of the Page Banner.

CAUSE

The page banner appears blank if your title contains quotation marks.

RESOLUTION

Change the title of the page banner so that it does not contain quotation marks. To do this, follow these steps:

  1. Start FrontPage Explorer. Open your Web.

  2. On the View menu, click Navigation.

  3. Right-click the page that contains the banner you want to change, and then click Rename on the menu that appears.

  4. Type a title that does not contain quotation marks and press ENTER.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
