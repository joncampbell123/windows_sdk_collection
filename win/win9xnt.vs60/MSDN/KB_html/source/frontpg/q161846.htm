FP97: Some Features Don't Work on FrontPage 1.1 Server

 Last reviewed: March 18, 1998
Article ID: Q161846
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SUMMARY

FrontPage 97 allows you to author and administer files posted on a web server that is using the FrontPage version 1.1 Server Extensions. Certain FrontPage 97 features will not function on a server using the FrontPage version 1.1 Server Extensions. FrontPage 97 files that use features that are available on a FrontPage version 1.1 extended server, will function correctly on the version 1.1 extended server.

MORE INFORMATION

The following new features do not function on a web server that uses the FrontPage version 1.1 server extensions: 

   - Field Validation
   - Scripting
   - Folder Management
   - Authoring over a SSL connection
   - NT Challenge/Response Logon
   - Multiple Language Support
   - Internet Database Connector Wizard
