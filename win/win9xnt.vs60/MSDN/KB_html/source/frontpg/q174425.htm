FP98: No Icon for Script, Formatting, or Inserted HTML

 Last reviewed: March 18, 1998
Article ID: Q174425
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

In FrontPage Editor, when you insert an HTML, JavaScript, or VB Script component, or add page elements, such as paragraph or line breaks, you do not see the format marks or placeholder icons that indicate you have inserted these elements on your page.

CAUSE

This behavior occurs when the Format Marks command on the View menu is not selected (not checked).

RESOLUTION

In FrontPage Editor, click Format Marks on the View menu. A check mark appears to the left of the Format Marks command to indicate that it is turned on.

MORE INFORMATION

The Format Marks command in FrontPage 98 is designed to give you a visual representation of code or formatting that is not visible in FrontPage Editor.

The following items will appear as an icon in FrontPage Editor if Format Marks is turned on: 

   - Paragraphs
   - Line Breaks
   - Bookmarks
   - JavaScript
   - VB Script
   - Inserted HTML
