FP: MS Internet Information Server Not in Server Type List

 Last reviewed: March 18, 1998
Article ID: Q152389

The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

Microsoft Internet Information Server is not listed in the Server Type list in the Configure Server Type dialog box in the FrontPage Server Administrator.

CAUSE

The Microsoft Internet Information Server extensions have not been installed or have not been installed correctly.

The FrontPage Server Extensions for Microsoft Internet Information Server were not available when FrontPage version 1.1 was shipped. The FrontPage Server Extensions for Microsoft Internet Information Server are shipped with FrontPage 97 and FrontPage 98.

RESOLUTION

Install or re-install the FrontPage Server Extensions for Microsoft Internet Information Server.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q156842
   TITLE     : FP: How to Install FrontPage Server Extensions for IIS

MORE INFORMATION

For more information about the FrontPage Server Extensions, please go to the following Microsoft World Wide Web site: 

   http://www.microsoft.com/frontpage/
