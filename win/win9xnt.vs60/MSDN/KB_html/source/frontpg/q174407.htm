FP98: How to Create IDC/HTX Files in FrontPage 98

 Last reviewed: March 18, 1998
Article ID: Q174407
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SUMMARY

In Microsoft FrontPage 97, Internet Database Connector (IDC) and HyperText Markup Language extension (HTX) files are created by using templates in FrontPage Editor. These templates are not available in Microsoft FrontPage 98. This article describes how to create IDC and HTX pages in FrontPage 98.

MORE INFORMATION

Creating an HTX File

To create an HTX file, follow these steps:

  1. Start FrontPage Explorer.

  2. On the File menu, point to New, and then click Folder.

    A new folder called New_Folder is created.

  3. Right-click the New_Folder folder and click Rename on the menu that appears.

  4. Type a descriptive name such as "Scripts" (without the quotation marks).

  5. Right-click the Scripts folder and click Properties on the menu that appears.

  6. Click the General Tab.

  7. Click to select the "Allow scripts or programs to be run" check box. Click OK.

  8. Double-click the Scripts folder.

  9. On the File menu, point to New, and then click Page.

  10. Right-click the page and click Rename on the menu that appears.

  11. Change the file name extension from HTM to HTX.

  12. Double-click the page to open it in FrontPage Editor.

Creating the IDC File

To create an IDC file, follow these steps within the same directory you created for the HTX file above:

  1. Point to New on the File menu, and then click Page.

  2. Right-click the page and click Rename on the menu that appears.

  3. Change the file name extension from HTM to IDC.

  4. Double-click the page to open it in the Internet Database Connector Wizard.
