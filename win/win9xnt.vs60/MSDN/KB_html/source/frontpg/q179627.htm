FP98: Error, Problems Using GIF Image and Hover Button

 Last reviewed: March 18, 1998
Article ID: Q179627
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you use a GIF image with a hover button, you may experience one of the following problems in Microsoft Internet Explorer 3.0:

  • The image does not appear.
  • The image appears scrambled.
  • You receive an illegal operation error.

CAUSE

These problems appear because Internet Explorer 3.0 Java does not handle any GIF that has a local color table.

WORKAROUND

To work around this problem use either of the following workarounds:

  • Use an image editing tool, such as Microsoft Image Composer, to open the image and save the GIF file without a local color table.

-or-
  • Use an image editing tool, such as Microsoft Image Composer, to open the image and convert it to JPG format.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Explorer 3.0. This problem was corrected in Microsoft Internet Explorer 4.0.
