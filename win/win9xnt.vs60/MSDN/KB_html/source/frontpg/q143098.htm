FP: How to Add White Space to a Web Page

 Last reviewed: March 18, 1998
Article ID: Q143098

The information in this article applies to:

  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage for Windows, versions 1.0, 1.1
  • Microsoft FrontPage for the Power Macintosh, version 1.0

SUMMARY

This article describes how to add white space to a Web page in the versions of Microsoft FrontPage listed at the beginning of this article.

NOTE: In FrontPage 98, you can insert any number of spaces by pressing the spacebar.

MORE INFORMATION

Microsoft FrontPage 97, 1.1, 1.0

To add white space to your HTML document, press CTRL+SHIFT+SPACE.

Microsoft FrontPage for the Power Macintosh

To add white space to your HTML document, press OPTION+SPACE.
