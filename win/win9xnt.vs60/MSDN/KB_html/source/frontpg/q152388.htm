FP: Task Added from To Do List Not Linked to Page

 Last reviewed: March 18, 1998
Article ID: Q152388

The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

The FrontPage To Do List will not create a link to the current page in FrontPage Editor or in FrontPage Explorer when you use the Add button.

WORKAROUND

To add a Task to the To Do List with a link to a specific page, use the following steps:

  1. In FrontPage Explorer or FrontPage Editor, click Add To Do Task on the Edit menu.

  2. Type a Task Name, Description, select a priority, and then click OK.

In the FrontPage To Do List, the new task will be linked to a specific page.

MORE INFORMATION

This feature is desirable if you want to avoid a page link. For example, you might use this feature when you want to add a task that is not related to a specific page.
