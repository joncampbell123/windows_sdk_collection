Cannot Add Blank Favorites Folder

 Last reviewed: February 10, 1998
Article ID: Q172134
The information in this article applies to:
  • Microsoft Internet Explorer Administration Kit, version 3.2a

SYMPTOMS

The Internet Explorer Administration Kit (IEAK) does not allow users to create Favorites folders that are blank for a customized version of Internet Explorer. If there is a blank folder specified in the wizard, then it will be dropped during setup.

CAUSE

One feature of IEAK is the ability to add Favorites for new users of Internet Explorer. However, the dialog in the IEAK wizard states that blank folders will be not be created. All folders must contain a link to be added to the Favorites folder. 

Version           : WINDOWS:3.2a
Platform          : WINDOWS
Issue type        : kbbug
