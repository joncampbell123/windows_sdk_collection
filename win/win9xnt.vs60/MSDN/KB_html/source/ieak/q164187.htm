IEAK Wizard Creates More than One Destination Folder

 Last reviewed: December 1, 1997
Article ID: Q164187
The information in this article applies to:
  • Microsoft Internet Explorer Administration Kit version 3.2a

SYMPTOMS

When you run the Internet Explorer Administration Kit (IEAK) Setup Wizard, if you specify the name of the destination folder, continue to the next Wizard dialog box, and then go back and change the destination folder, the Setup Wizard creates multiple destination folders.

CAUSE

The IEAK Setup Wizard creates the folder when you click the Next button. However, when you change this folder, the folder with the original name is not deleted.

WORKAROUND

Manually delete any unused directories that were created by the IEAK Setup Wizard.
