No Disk Space Information for Typical/Laptop/Workstation

 Last reviewed: September 10, 1996
Article ID: Q124235
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.0, 4.2, 4.3
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft PowerPoint for Windows, versions 4.0, 4.0c
  • Microsoft Access, version 2.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Project for Windows, version 4.0
  • Microsoft Works for Windows, version 3.0

SUMMARY

When you run the Setup program for one of the applications listed above, you do not see any information about how much disk space is available or required on your hard drive unless you choose the Custom/Complete installation option.

When you choose either the Typical, Laptop (Minimum), or Workstation installation options, the Setup program installs the appropriate files without displaying information about how much hard disk space is available or required on the drive to which you are installing the application.

WORKAROUNDS

To display the amount of disk space available on the drive to which you are installing the application, do the following:

  1. Run the Setup program. Enter your name and company as prompted.

  2. In the dialog box that contains the installation options, choose the Complete/Custom button.

    The Complete/Custom dialog box displays the disk space available on the drive to which you are installing the application. This dialog box also displays the disk space required for each of the applications components.

  3. Choose Cancel to return to the dialog box that displays the installation options.
