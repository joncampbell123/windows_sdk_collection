Bookshelf Shortcut Does Not Appear in the Msoffice Folder

 Last reviewed: September 9, 1996
Article ID: Q142525
The information in this article applies to:
  • Microsoft Office Professional for Windows 95, version 7.0

SYMPTOM

After you install Microsoft Bookshelf for Windows 95, the Bookshelf shortcut icon does not appear in a subfolder in the Msoffice folder.

CAUSE

This behavior is by design of Microsoft Office Professional for Windows 95. Instead of placing the Microsoft Bookshelf '95 icon in a subfolder in the Msoffice folder, the Microsoft Office Setup program copies the icon to the Microsoft Reference folder. Because Bookshelf is a Microsoft program that falls into the reference category, the icon is placed in the Microsoft Reference folder.

The Microsoft Office Professional CD jewelcase booklet tells you to start Bookshelf from this location. The jewelcase booklet includes the following information about starting Bookshelf: 

   To start Bookshelf '95

   1. Insert disc 2, the Reference Disc, into your CD-ROM drive.

   2. From the Office Shortcut bar, click the Bookshelf icon, or click the
      Start button, point to Programs, point to Microsoft Reference, and
      then click Bookshelf '95.

MORE INFORMATION

You can find the Microsoft Reference folder in the following location: 

   \Windows\Start menu\Programs
