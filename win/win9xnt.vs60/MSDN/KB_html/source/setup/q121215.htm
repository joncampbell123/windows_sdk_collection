Setup Err Msg: There Are <x> Users Connected to the System...

 Last reviewed: September 10, 1996
Article ID: Q121215
The information in this article applies to:
  • Microsoft Office Setup for Windows, version 1.0
  • Microsoft Office for Windows, version 4.0, 4.2, 4.3
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Word for Windows, version 6.0
  • Microsoft PowerPoint for Windows, version 4.0

SUMMARY

At the end of the Microsoft Setup program, you must restart your computer in order to complete the Setup. However, if someone is connected to one of your shared drives or directories, you will receive the following error message 

   There are <x> users connected to the system,
   do you still want to quit windows?
where <x> represents the number of users maintaining a connection to your computer.

You will not be able to use ALT+TAB or CTRL+ESC to check who is connected to your computer.

MORE INFORMATION

You are not able to switch from the dialog box, because it is a modal dialog with options only to select <Yes> or <No> (in order to dismiss the dialog). Attempting to switch out of Setup while the message box is displayed is not allowed.
