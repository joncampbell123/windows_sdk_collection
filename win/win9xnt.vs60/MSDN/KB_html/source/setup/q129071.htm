Error Running Setup from DMF Disks on HP OmniBook

 Last reviewed: September 10, 1996
Article ID: Q129071
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Office for Windows 95, versions 7.0, 7.0b

SYMPTOMS

When you run the Word for Windows Setup program from an external floppy disk drive on a Hewlett-Packard OmniBook computer, after you remove Disk 1, insert Disk 2, and then press ENTER, you receive the "Insert Disk 2" message again.

CAUSE

The OmniBook floppy disk drive is unable to read the Disk 2 file format.

Disk 2 and all subsequent disks are formatted with Distribution Media Format (DMF). DMF is a technology that enables more contents to be stored on a high-density disk by removing normal sector boundaries. DMF is a new disk format that increases the data capacity of a 3.5-inch disk.

STATUS

Hewlett-Packard is aware of this problem.

RESOLUTION

To work around this problem, contact Microsoft to obtain a replacement set of floppy disks that are not DMF formatted. Non-DMF diskettes are not available for Microsoft Office for Windows 95. If your Omnibook supports a cd-rom drive then contact Microsoft for a replacement cd-rom disk.

Note that using DRIVPARM has no affect. In addition, files cannot be extracted from Disk 2 with the Extract program.
