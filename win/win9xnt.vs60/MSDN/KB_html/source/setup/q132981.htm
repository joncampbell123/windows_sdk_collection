Online Registration Button Doesn't Appear During Setup

 Last reviewed: January 16, 1997
Article ID: Q132981
The information in this article applies to:
  • Microsoft Access 97 for Windows
  • Microsoft Access for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Project for Windows 95, version 4.1
  • Microsoft Publisher for Windows 95, version 3.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you run the Setup program, the Online Registration button may not appear in the Setup maintenance mode dialog box or in the "Setup was completed successfully" dialog box.

CAUSE

This behavior occurs when you run setup again in maintenance mode after successfully completing Setup and registering the application. The first time you run Setup, you receive a dialog box with the following information after Setup is complete 

   <Microsoft Application> Setup was completed successfully.

   If you have a modem, you can now click the "Online Registration" button
   to register your new software with Microsoft via modem.
where <Microsoft Application> is the application that you are installing, such as Microsoft Office for Windows 95. If you click Online Registration, the Registration Wizard dialog box appears and allows you to register your application. If you do not register your application the first time you run Setup, the Online Registration button appears in the Setup maintenance mode dialog box (along with the maintenance mode installation options) when you run Setup again. Note that when you run Setup in maintenance mode, the Online Registration button does not appear in the "Setup was completed successfully" dialog box.

However, if you have successfully registered your application, this Online Registration button does not appear in any of the dialog boxes when you run Setup. The registration button may also not appear if the file (Regwiz.exe) supplied by Windows 95 is missing from the Windows\System folder.
