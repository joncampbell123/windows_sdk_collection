WD97: Compile Error If VB Code Doesn't Find Comments

 Last reviewed: March 5, 1998
Article ID: Q158854
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you run a macro to retrieve a property of the Comment object, and if no comments exist in the document, you get the following error message: 

   Compile Error:
   Method or data member not found.

CAUSE

Your document does not contain comments.

WORKAROUND

To avoid this error message, you can test for the presence of comments in your document using the following sample Microsoft Visual Basic for Applications macro: 

   Sub CommentExist()
     If ActiveDocument.Comments.Count > 0 Then
       MsgBox "Word found Comments in your document."
     Else
       MsgBox "Word found no Comments in your document."
     End If
   End Sub
