GRAPH97: Visual Basic Help for Microsoft Graph Not Installed

 Last reviewed: March 11, 1998
Article ID: Q161388
The information in this article applies to:
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Office 97 for Windows
  • Microsoft Graph 97 for Windows

SUMMARY

The complete versions of the Visual Basic Help files for Microsoft Graph 97 are not installed during Setup. The complete versions are available in the ValuPack folder on the Office 97 and PowerPoint 97 compact discs, but be installed manually. This article explains how to install the Visual Basic Help files for Graph 97.

MORE INFORMATION

To install the Visual Basic Help files for Microsoft Graph 97, use the following steps:

  1. Close Help if it is running.

  2. Rename the Vbagrp8.hlp and Vbagrp8.cnt files located in the Office folder. By default, the path to the Office folder is: 

          C:\Program Files\Microsoft Office\Office

  3. Open the following file on your PowerPoint 97 or Office 97 compact disc (assuming the drive letter for your CD-ROM drive is D:): 

          D:\ValuPack\MoreHelp

  4. Copy the files Vbagrp8.hlp and Vbagrp8.cnt from the MoreHelp folder into your C:\Program Files\Microsoft Office\Office folder.
