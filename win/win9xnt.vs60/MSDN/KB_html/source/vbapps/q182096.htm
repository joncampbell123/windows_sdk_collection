WD98: Closing Document Window Does Not Close VB Editor

 Last reviewed: March 5, 1998
Article ID: Q182096
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When there are no documents open, the Visual Basic Editor remains open.

CAUSE

All documents and templates contain at least one Visual Basic for Applications project including the global template, Normal. Because the global Normal template is open as long as Word is running, the Visual Basic Editor will remain open when there are no documents open.
