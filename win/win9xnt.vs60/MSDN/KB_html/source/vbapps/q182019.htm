WD98: Cannot Trap FormFieldOptions Command on Double-Click

 Last reviewed: March 4, 1998
Article ID: Q182019
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you create a macro named FormFieldOptions, this macro will run when you click Properties on the shortcut menu for a form field. However, when you double-click the form field to display the Options dialog box for form fields, the FormFieldOptions macro does not run as expected.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
