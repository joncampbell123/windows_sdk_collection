WD97: WordBasic.MailMergeInsertFillin Fails to Prompt

 Last reviewed: February 11, 1998
Article ID: Q165291
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

In previous versions of Word for Windows, the WordBasic "MailMergeInsertFillin" command causes the Fillin Field to prompt the user upon execution of the command.

This behavior has changed when you use this same command with the Visual Basic for Applications WordBasic object. The command "WordBasic.MailMergeInsertFillin" does not automatically update the field to prompt the user.

MORE INFORMATION

To cause the fillin field to prompt the user, use the following Visual Basic for Applications command: 

   ActiveDocument.Fields.Update
