XL7: Visual Basic Command to Show Set Background Dialog Fails

 Last reviewed: September 2, 1997
Article ID: Q134264
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0

SYMPTOMS

In Microsoft Excel version 7.0, when you attempt to show the Set Background Picture Dialog box in a Visual Basic, Applications Edition, macro, you will receive the following error message: 

   Run-time error '1004'
   Dialog method of Application class failed

CAUSE

This problem occurs because of the following line of code: 

   Application.Dialogs(xlDialogSetBackground).Show

WORKAROUND

To work around this problem, use the ExecuteExcel4Macro method as in the following example: 

   Application.ExecuteExcel4Macro(Sheets.BackGround'())
