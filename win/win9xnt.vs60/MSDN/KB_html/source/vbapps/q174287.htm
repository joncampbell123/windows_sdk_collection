XL97: Formula Returns Incorrect Value When Changed by a Macro

 Last reviewed: March 13, 1998
Article ID: Q174287
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you use a Visual Basic for Applications macro to manipulate a formula, the formula is truncated and the value that is returned by the formula is incorrect.

CAUSE

This problem occurs when the following conditions are true:

  • You use a macro to copy or change a formula. 

        -and-
  • The formula contains more than 1,016 characters.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
