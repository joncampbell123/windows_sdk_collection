WD98: "Help Topic Does Not Exist" for ActiveX Jump in Help

 Last reviewed: March 3, 1998
Article ID: Q177826
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

When you access the CreateObject Function topic in Microsoft Visual Basic for Applications Help, and you click the ActiveX Object jump in the following CreateObject definition: 

   Creates and returns a reference to an "ActiveX Object"
you receive the following error message: 

   Microsoft Word - Help topic does not exist.

MORE INFORMATION

See the following topic in the MS Word ReadMe document in the Microsoft Office 98:ReadMe folder: 

   Help: Definitions Not Found
