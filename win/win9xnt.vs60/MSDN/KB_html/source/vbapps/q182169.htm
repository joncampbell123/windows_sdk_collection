WD97: Macro to Determine If a Form Field Exists

 Last reviewed: March 9, 1998
Article ID: Q182169
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

This article describes how to use Microsoft Visual Basic for Applications to determine if a form field exists in a Microsoft Word document.

To determine if a form field exists in a document, you can use the Exists method. The following sample Visual Basic for Applications macro will determine if the form field named Text1 exists in the current or active document: 

   Sub DoesFormFieldExist()
      If ActiveDocument.Bookmarks.Exists("Text1") Then
         MsgBox "The form field Text1 does exist."
      Else
         MsgBox "The form field Text1 does not exist."
      End If
   End Sub
