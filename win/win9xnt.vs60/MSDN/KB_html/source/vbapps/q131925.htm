XL7: Debug Window Dialog Box Tabs Appear in Version 5.0 Style

 Last reviewed: September 3, 1997
Article ID: Q131925
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Windows NT, version 3.51

SYMPTOMS

In Microsoft Excel version 7.0, when you display the Debug window, the Watch and Immediate tabs may have a different appearance than the other dialog box tabs in the Microsoft Excel.

CAUSE

This behavior occurs when you run Microsoft Excel version 7.0 under Microsoft Windows NT version 3.51. The Debug window tabs, Watch and Immediate, have the appearance of the tabs in Microsoft Excel version 5.0, instead of the expected appearance of the tabs in Microsoft Excel version 7.0. The dialog tabs in Microsoft Excel version 7.0 have a more 3-D appearance than in previous versions of Microsoft Excel.

Note that this behavior only occurs with the Debug window; the other tabs in Microsoft Excel, such as the tabs in the Format Cells, Object, and Option dialog boxes have the expected version 7.0 appearance.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about how to use the Debug window in Microsoft Excel, click the Index tab in Microsoft Excel Help, type the following text 

   Debug w
double-click the selected text, and then double-click How To Use The Debug Window to go to the "How To Use The Debug Window" topic.
