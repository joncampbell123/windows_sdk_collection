 Last reviewed: September 2, 1997
Article ID: Q114756
The information in this article applies to:
  • Microsoft Visual Basic Programming System, Applications Edition, version 1.0

SUMMARY

The Microsoft PSS Knowledge Base group regularly extracts the most current articles from the Visual Basic for Applications database and creates a Windows Help file, which we then post to the Microsoft Software Library. We update these files as new information becomes available.

For Visual Basic for Applications issues, we have created the self- extracting file VBAPPS.EXE, which contains a collection of articles about using Visual Basic for Applications with other Windows-based programs, such as Microsoft Excel version 5.0 and Microsoft Visual Basic.
