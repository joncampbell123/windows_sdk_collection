XL7: Invalid Page Fault When OLE Automation Used to Copy Chart

 Last reviewed: September 2, 1997
Article ID: Q139577
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0

SYMPTOMS

When you use OLE Automation to copy a chart from Microsoft Excel for Windows 95, version 7.0, and paste it to another Office for Windows 95 program, you may receive the following error message: 

   EXCEL caused an invalid page fault in module EXCEL.EXE
   at 0137:5002b3fd

RESOLUTION

To resolve this problem, use the CopyPicture method rather than the Copy method. Note that when you use this method, the chart is not linked to the source data, so it is not updated when the source data is changed.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel for Windows 95, version 7.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
