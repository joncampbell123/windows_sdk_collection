XL7: AutoCalculate in Group Edit Mode Works Only on Active Sheet

 Last reviewed: September 2, 1997
Article ID: Q135102

The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

When you use group edit mode, the AutoCalculate feature applies only to the currently active sheet.

This behavior is by design.

MORE INFORMATION

When you select a range on a worksheet, AutoCalculate allows you to quickly find information about the selected range; the sum, average, or count of the entries appears in the AutoCalculate area of the status bar.

Group edit mode allows you to select multiple sheets and group them as a single object--use this feature to easily make global changes. A change made to any of the grouped sheets affects all other grouped sheets.
