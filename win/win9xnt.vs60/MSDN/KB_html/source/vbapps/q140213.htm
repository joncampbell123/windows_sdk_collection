ACC95: Constants in the Microsoft DAO 3.0 Object Library

 Last reviewed: August 28, 1997
Article ID: Q140213
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

This article lists the constant variables that are in the Microsoft Access for Windows 95 version 7.0 object library.

MORE INFORMATION

A listing of all the available constants in the Microsoft Access 7.0 object library is available by using the Object Browser. To use Object Browser, open a module in Design view, press F2, and select "DAO 3.0 Object Library" in the Libraries/Database box. Then, choose "constants" from the list box in the lower left corner of the dialog box.
