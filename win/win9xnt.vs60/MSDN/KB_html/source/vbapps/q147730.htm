XL: MsgBox Function has 255 Character Limit

 Last reviewed: February 3, 1998
Article ID: Q147730
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

In Microsoft Excel versions 5.x and 7.x, the "MsgBox Function" topic in online Help reads as follows: 

   The maximum length of prompt is approximately 1024 characters,
   depending on the width of the characters used.
This information is incorrect. The maximum number of characters that you can use in the prompt argument is 255.
