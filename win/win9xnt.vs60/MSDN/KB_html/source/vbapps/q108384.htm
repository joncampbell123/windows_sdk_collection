XL: No Visual Basic Method for SET.UPDATE.STATUS

 Last reviewed: February 3, 1998
Article ID: Q108384
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

In Microsoft Excel, to set the links update status, you must use the ExcecuteExcel4Macro method to run the SET.UPDATE.STATUS() macro command. This article contains sample macros that demonstrate using this method.

NOTE: The macros provided in this article will work only if the link was created by following these steps:

  1. Copy the information you want linked from your Word document (or other document).

  2. Switch to Microsoft Excel.

  3. On the Edit menu, click Paste Special.

  4. In the Paste Special dialog box, click Microsoft Word Document Object (or another document object type), and then click Paste Link.

  5. Click OK.

If you linked to your Word document by doing the following steps, the link is different and the macros shown below will not work.

  1. Click Object on the Insert menu in Microsoft Excel.

  2. Click the "Create from File" tab.

  3. On the "Create from File" tab, enter the name of the source Word document, click the "Link to File" check box, and then click OK.

Microsoft Excel 97 and Microsoft Excel 98 Macro Example

This macro sets the update status to manual in the Links dialog box which is available when you click Links on the Edit menu. This macro assumes that the link in Microsoft Excel is to a Microsoft Word document titled "Document1" (without the quotation marks). 

   Sub LinkStatus()

       ExecuteExcel4Macro _
         "SET.UPDATE.STATUS(""Word.Document.8|Document1!!OLE_LINK1"",2,2)"

   End Sub

Microsoft Excel version 7.0 Macro Example

This macro sets the update status to manual in the Links dialog box which is available when you click Links on the Edit menu. This macro assumes that the link in Microsoft Excel is to a Microsoft Word document titled "Document1" (without the quotation marks). 

   Sub LinkStatus()

       ExecuteExcel4Macro _
         "SET.UPDATE.STATUS(""Word.Document.6|Document1!!OLE_LINK1"",2,2)"

   End Sub

Microsoft Excel version 5.0 Macro Example

This macro sets the update status to manual in the Links dialog box which is available when you choose Links from the Edit menu and it assumes your link in Microsoft Excel is to a Microsoft Word document titled "Doc" (without the quotation marks). 

   Sub LinkStatus()

       ExecuteExcel4Macro _
         "SET.UPDATE.STATUS(""Word.Document.6|Doc!!DDE_LINK1"",2,2)"

   End Sub
