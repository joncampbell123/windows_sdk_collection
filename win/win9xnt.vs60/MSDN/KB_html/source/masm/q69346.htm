BUG: MASM 5.10 Installation Asks for Disk "OS/2 Program"

 Last reviewed: October 21, 1994
Article ID: Q69346
The information in this article applies to:
  • Microsoft Macro Assembler for MS-DOS, version 5.1
  • Microsoft Macro Assembler for OS/2, version 5.1

SYMPTOMS

The Microsoft Macro Assembler (MASM) version 5.1 installation program RUNME.EXE may prompt you to insert a disk named "OS/2 Program."

There is no disk labeled "OS/2 Program" in the MASM 5.1 distribution disks package; the actual disk the installation program is looking for is labeled "MS OS/2 Tools."

RESOLUTION

When the installation program asks for the disk named "OS/2 Program" install the disk labeled "MS OS/2 Tools."
