PATCH: MASM 6.11d Patch Available

 Last reviewed: March 11, 1997
Article ID: Q138983
The information in this article applies to:
  • Microsoft Macro Assembler for MS-DOS, versions 6.11 and 6.11a

SUMMARY

ML611D.EXE is a file in the Microsoft Software Library. This file provides the necessary files to update Microsoft Macro Assembler (MASM) from version 6.11 and 6.11a to version 6.11d. This update requires either MASM version 6.11 or 6.11a to be currently installed. One or more of the files ML.EXE, ML.ERR, H2INC.EXE and H2INC.ERR, README.TXT, ERRMSG.TXT, and SUPPORT.TXT are affected by this update depending on the version installed. See the file README.TXT contained in this update for other information and installation instructions.

The following file is available for download from the Microsoft Software Library: 

 ~ Ml611d.exe (size: 499935 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

MORE INFORMATION

This patch file will not patch MASM version 6.11c, which was only made available in the Windows NT version 3.5 DDK and Windows 95 DDK in the form of the files ML.EXE and ML.ERR. The versions of ML.EXE and ML.ERR from MASM version 6.11d can be found in the Windows NT version 3.51 DDK. The other files provided in ML611D.EXE can then be copied to their respective locations to effectively complete the patch. The alternative is to run setup for MASM 6.11 from the original diskettes, reinstall ML.EXE and ML.ERR by selecting 'Copy a file from the distribution disks' from the Main Menu of Setup, specifying ML.EXE and ML.ERR, and then perform the patch.
