DOCERR: Programmer's Guide FREE ST(num) Reference Is Incorrect

 Last reviewed: November 3, 1994
Article ID: Q33382
The information in this article applies to:
  • Microsoft Macro Assembler for MS-DOS, versions 5.0 and 5.1
  • Microsoft Macro Assembler for OS/2, version 5.1

On Page 406 of the "Microsoft Macro Assembler 5.1 Programmer's Guide," the FFREE instruction is incorrectly given as FREE. The instruction appears correctly in the "Microsoft Macro Assembler 5.1 Reference".

Additional reference words: 5.00 5.10
