FILE: Internet Explorer 4.01 Refresh of the Internet Client SDK

 Last reviewed: December 22, 1997
Article ID: Q177877
The information in this article applies to:
  • Internet Client SDK, version 4.0

SUMMARY

This article contains the release notes for the Internet Explorer 4.01 refresh of the Internet Client SDK (INetSDK - also known as Internet Explorer 4.01 Author's toolkit).

Because there are relatively few changes in the downloadable file, it is suggested that you do not download and install this refresh unless there is some specific feature of Internet Explorer 4.01 that you think is not exposed in the Internet Client SDK for Internet Explorer 4.0.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Iesdk401.exe (size: 1984960 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

Prerequisites of this Release of the SDK

Before you can download the Internet Client SDK 4.01 refresh, you must have the following installed on your computer:

  • Internet Explorer 4.01.
  • Internet Client SDK 4.0 installed.
  • If you are developing software, the latest Win32 development files. You might already have installed these from the latest Platform SDK at http://www.microsoft.com/msdn/sdk/. Otherwise, the subset you need is available as a download from the Internet Client SDK download page under "Win32 SDK Libraries & Headers."
  • Windows 95 or Windows NT 4.0 SUR (final version).
  • Development tools, if you are building software. This product has been tested primarily on Visual C++ 5.0. (See "Configuration Issues" below for more information.)
  • Web authors and ISVs, please be sure to read the IE401Readme.txt file after installing this SDK refresh. It has the latest-breaking information and last-minute problems.

Installation Issues - Internet Installs

  • Download the Internet Client SDK for Internet Explorer 4.01. When prompted, run the executable from the Internet site or save the file to an alternative location to be run later.
  • The self-extracting file will prompt you for the location of the Internet Client SDK 4.0 directory. Verify that the directory is correct and then click the "UnZip" button.
  • To use the Alpha Headers & Libraries, you need to download and install the Internet Client SDK 4.01 refresh using an Intel machine. Then copy the Headers & Libraries to the Alpha platform.
  • If, in the future, you return to the Internet Client SDK download site and add (download) either the components "ActiveX Component Technology" or "Other Web Technology," you will need to refresh the Headers & Libraries files for IE 4.01 by re-running the IESDK401.exe self- extracting zip file.
  • Note that the documentation in the Internet Client SDK 4.0 has not been updated in this download. To view the most recent documentation, please see the Internet Client SDK on SiteBuilder at: (http://www.microsoft.com/workshop/prog/inetsdk/)

New and Updated Files Included

  • Refresh Intel Libraries & Headers.
  • Added Alpha Libraries.
  • Added Blesslnk.exe to \bin directory. Blesslnk.exe is a tool that updates shortcuts to fully utilize the Open Software Description (OSD) functionality.
  • Added a Generic Open Software Description (OSD) Sample to \Samples\OSD\GenericSample directory.
  • Updated the Glass Sample header file in \sample\glass\glassctl.h.
