DOC: Window.navigator.connectionSpeed Is Not Supported

 Last reviewed: January 22, 1998
Article ID: Q179609
The information in this article applies to:
  • Internet Client SDK, version 4.0

SUMMARY

The ConnectionSpeed property of the navigator object is documented in the current version of the Internet Client SDK as an indicator of session connection speed.

The documentation is incorrect. The Connectionspeed property is not available in the current version (4.0x) of Internet Explorer and any attempt to refer to the Connectionspeed property of the navigator object will result in the following script error: 

   Object doesn't support this action.

REFERENCES

Internet Client SDK Help: 

   http://www.microsoft.com/msdn/sdk/inetsdk/help/
