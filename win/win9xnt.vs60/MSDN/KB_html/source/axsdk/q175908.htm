FIX: Script Error in Nmstart.Htm Sample in Internet Client SDK

 Last reviewed: January 19, 1998
Article ID: Q175908
The information in this article applies to:
  • Internet Client SDK, versions 4.0, 4.01
  • Microsoft Internet Explorer (Programming), versions 4.0, 4.01
  • Microsoft Chat SDK 1.0j

SYMPTOMS

When the "Start NetMeeting Conference with" button is clicked in the Nmstart.htm sample that comes with the Internet Client SDK, the following script error occurs: 

   Object doesn't support this property or method: 'document.frmNMStart'

CAUSE

This error occurs because there is not a matching </FORM> tag for the first <FORM> tag in the HTML document.

STATUS

This problem was corrected in the Microsoft Chat SDK 2.0 Beta.

MORE INFORMATION

The Nmstart.htm sample is part of the Microsoft Chat SDK. The latest version can be downloaded from the following Web site: 

   http://www.microsoft.com/msdn/sdk/chatsdk/chatcontrol.htm
