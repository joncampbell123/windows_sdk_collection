PRB: Access Violation in RPAWINET.DLL

 Last reviewed: January 19, 1998
Article ID: Q169373
The information in this article applies to:
  • Microsoft ActiveX SDK, version 1.0
  • Internet Client SDK, versions 4.0, 4.01
  • Internet Explorer (Programming), versions 4.0, 4.01

SYMPTOMS

WinInet programs experience an access violation in Rpawinet.dll. This will occur only on computers that use CompuServe as their Internet Service Provider, and only when accessing certain Web sites.

CAUSE

The computer has an older version of Rpawinet.dll, which is provided by CompuServe.

RESOLUTION

The current version of the Rpawinet.dll for Internet Explorer 3.0 on Windows 95 and NT 4.0 is located at: 

   http://www.compuserve.com/rpa/browser.htm
See this page for further information and instructions.
