BUG: Unable to Close Transaction Statistics Err Message Dialog

 Last reviewed: February 9, 1998
Article ID: Q180192
The information in this article applies to:
  • Microsoft Transaction Server 2.0

SYMPTOMS

When you try to open the Transaction Statistics window as a separate window using Microsoft Transaction Server (MTS) Explorer, a dialog box appears with the following message: 

   Transaction statistics for 'My Computer' are already displayed in
   another window. You can only open one transaction statistics window
   for a given server.
Clicking OK does not close the dialog box.

STATUS

Microsoft has confirmed this to be a bug in the Microsoft products listed at the beginning of this article. We are researching this bug and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Behavior

This error appears after you perform the following steps:

  1. Start the MTS Explorer.

  2. Expand Microsoft Transaction Server and Computers and My Computer.

  3. Click Transaction Statistics so that it is visible in the right pane.

  4. Right click Transaction Statistics and select "New Window From Here."

    Once the dialog appears, you have to shut down and restart MTS Explorer before being able to resume administration.
