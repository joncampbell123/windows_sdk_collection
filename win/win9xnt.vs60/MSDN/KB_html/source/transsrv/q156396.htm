FIX: Warning LNK4099 Message from Visual C Version 5.0

 Last reviewed: October 29, 1997
Article ID: Q156396
The information in this article applies to:
  • Microsoft Transaction Server, version 1.0

SYMPTOMS

When building a component in Visual C version 5.0 with a custom interface that is linked to Mtxih.lib, you may receive the following warning message: 

   mtxih.lib(ProxyGlue.obj) : warning LNK4099: PDB "vc40.pdb" was not
   found
Although you receive this error, the Microsoft Transaction Server custom interface component builds correctly.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Transaction Server version 1.0. This problem was corrected in Microsoft Transaction Server version 1.1. For information on obtaining version 1.1, please see the following article the Microsoft Knowledge Base: 

   ARTICLE-ID: Q168031
   TITLE     : How to Obtain Microsoft Transaction Server Version 1.1
