BUG: Splash Screen and Background Dialog Box Are Different Sizes

 Last reviewed: July 22, 1997
Article ID: Q156394
The information in this article applies to:
  • Microsoft Transaction Server, version 1.0

SYMPTOMS

The splash screen and the background dialog box it is displayed on are different sizes in the Japanese version of Microsoft Transaction Server.

CAUSE

The size of the splash screen is based on the currently installed system font. If you changed the system font to large font, the size of the splash screen dialog box becomes too wide.

WORKAROUND

To work around this problem, change the system font to a smaller font.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Transaction Server version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
