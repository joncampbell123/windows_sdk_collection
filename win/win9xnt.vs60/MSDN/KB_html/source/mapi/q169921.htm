INFO: What's New in Active Messaging Library Version 1.1

 Last reviewed: March 2, 1998
Article ID: Q169921
The information in this article applies to:
  • Collaboration Data Objects (CDO), version 1.1

SUMMARY

Below is a list of the new objects, properties and methods available in Active Messaging Library, version 1.1 (Olemsg32.dll).
