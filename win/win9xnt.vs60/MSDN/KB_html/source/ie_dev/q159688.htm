BUG: Unable to Resolve Gopher Site Via Proxy in IE 3.0

 Last reviewed: November 27, 1996
Article ID: Q159688
The information in this article applies to:
  • Microsoft Internet Explorer for Windows NT, versions 3.0, 3.01
  • Microsoft Internet Explorer for Windows 95, versions 3.0, 3.01
  • Microsoft Internet Explorer for Windows 3.1, version 3.0

SYMPTOMS

If you use Internet Explorer 3.0 through a proxy server to access a gopher document, the following error message appears: 

   Internet Explorer cannot open the Internet site gopher
   //dosfan.lib.uic.edu/0f-1%3A4654%3A01Fact%20Sheet%20DOSFAN%09%09%2B. The
   system cannot find the file specified.

CAUSE

The HTTP request that the Internet Explorer sends to a proxy server is not encoded properly.

RESOLUTION

Use a proxy server that encodes the URL on behalf of the browser.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
