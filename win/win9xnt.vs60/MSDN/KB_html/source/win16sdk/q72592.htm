SAMPLE: TOOLHELP.DLL Functionality Demonstration

 Last reviewed: February 15, 1996
Article ID: Q72592
The information in this article applies to:
  • Microsoft Windows Software Development Kit (SDK) for Windows versions 3.1 and 3.0

The Microsoft Software Library contains a sample program that demonstrates some of TOOLHELP.DLL's capabilities.

The TOOLHELP.EXE file contains: 

   Filename     Description
   --------     -----------

   TEST.*       Sample code that demonstrates some of TOOLHELP's
                capabilities.

   TOOLHELP.*   The DLL, LIB, and so on needed to use this version of
                the TOOLHELP DLL.
Download TOOLHELP.EXE, a self-extracting file, from the Microsoft Software Library (MSL) on the following services:
  • Microsoft Download Service (MSDL) 

          Dial (206) 936-6735 to connect to MSDL
      Download TOOLHELP.EXE (size: 38018 bytes)
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the \SOFTLIB\MSLFILES directory
      Get TOOLHELP.EXE (size: 38018 bytes)
