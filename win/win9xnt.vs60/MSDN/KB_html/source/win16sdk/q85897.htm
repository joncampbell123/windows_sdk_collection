WDEB386-Compatible Timing Card Available

 Last reviewed: July 23, 1997
Article ID: Q85897
3.10 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

The Microsoft Windows 80386 Debugger (WDEB386.EXE) is documented in Chapter 5 of the "Microsoft Windows Software Development Kit: Programming Tools" manual. The documentation for the gs and gh commands, on page 148, indicates that each of these commands requires a timing card.

One timing card compatible with WDEB386 is the STAT! System Timing Analysis Tool, manufactured by Alpha Logic Technologies, Inc. For more information regarding this product, contact Alpha Logic Technologies at (206) 644-3094.
