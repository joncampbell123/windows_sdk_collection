You Need Arabic Word to Read or Print Arabic SDK Documents

 Last reviewed: January 15, 1998
Article ID: Q129079
3.10 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Arabic Windows Software Development Kit (SDK) for Araibc Windows version 3.1

The 16-bit Arabic Windows SDK and SDK documents are currently available on the Microsoft Developer Network (MSDN) Development Platform (level II CDs).

The Arabic SDK documents stored in the ARA_SDK\DOCS\DOCS directory cannot be read or printed by the U.S. version of Microsoft Word. They can only be read or printed by the Arabic version of Microsoft Word. Also, the Arabic version of Microsoft Word can be run only under Microsoft Arabic Windows, not U.S. Windows. Microsoft Arabic Windows version 3.1 is available on MSDN Development Platform.
