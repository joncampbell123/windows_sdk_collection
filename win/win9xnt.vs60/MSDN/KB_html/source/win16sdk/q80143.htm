BUG: Text Formatting Lost When Edit Text Dialog Box Used

 Last reviewed: January 5, 1995
Article ID: Q80143
The information in this article applies to:
  • Microsoft Windows for Pen Computing version 1.0

SYMPTOMS

When the Edit Text dialog box is used to edit formatted text in an application developed for Microsoft Windows for Pen Computing, formatting is lost.

STATUS

Microsoft has confirmed this to be a bug in Microsoft Windows for Pen Computing version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
