SAMPLE: Troubleshooting Guide for the 16-bit Setup Toolkit

 Last reviewed: February 15, 1996
Article ID: Q114603
The information in this article applies to:
  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

STPHLP.EXE contains a Microsoft Windows 3.1 Help file containing the answers to the most frequently asked questions about the Microsoft Windows 3.1 SDK Setup Toolkit. The help file contains a list of bugs and workarounds, samples for implementing commonly requested features, and short tutorials for commonly confusing issues.

STPHLP.EXE can be downloaded as a self-extracting file from the Microsoft Software Library (MSL) on the following services: 

      -or-

      GO WINSDK and download STPHLP.ZIP
  • Microsoft Download Service (MSDL) 

          Dial (206) 936-6735 to connect to MSDL
      Download STPHLP.EXE (size: 81493 bytes)
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the \SOFTLIB\MSLFILES directory
      Get STPHLP.EXE (size: 81493 bytes)
