SAMPLE: Code Demonstrates SystemParametersInfo()

 Last reviewed: February 15, 1996
Article ID: Q81140
The information in this article applies to:
  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

Version 3.1 of the Windows SDK documents a new function that allows an application to change most of the system settings. An application can retrieve and change settings as varied as the wallpaper on the desktop to the width of window borders using the SystemParametersInfo() function.

SYSPARAM is a file in the Microsoft Software Library that demonstrates using the SystemParametersInfo() function to retrieve and to change various parameters.

Download SYSPARAM.EXE, a self-extracting file, from the Microsoft Software Library (MSL) on the following services:

  • Microsoft Download Service (MSDL) 

          Dial (206) 936-6735 to connect to MSDL
      Download SYSPARAM.EXE (size: 25342 bytes)
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the \SOFTLIB\MSLFILES directory
      Get SYSPARAM.EXE (size: 25342 bytes)

NOTE: SYSPARAM modifies the WIN.INI file. Make a backup copy of WIN.INI before using this sample.
