SAMPLE: Capturing and Reproducing Ink

 Last reviewed: February 15, 1996
Article ID: Q80360
The information in this article applies to:
  • Microsoft Windows for Pen Computing version 1.0

SUMMARY

REPAINT is a file in the Microsoft Software Library that demonstrates how an application can capture "ink" that a user writes in the application's window and reproduce the ink at a later time with the same appearance.

Download REPAINT.EXE from the Microsoft Software Library (MSL) on the following services:

  • Microsoft Download Service (MSDL) 

          Dial (206) 936-6735 to connect to MSDL
      Download REPAINT.EXE (size: 25099 bytes)
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the \SOFTLIB\MSLFILES directory
      Get REPAINT.EXE (size: 25099 bytes)

MORE INFORMATION

The REPAINT sample application uses the RedisplayPenData() function to draw the ink. REPAINT also demonstrates how the size of ink can change as the size of the application window changes.

REPAINT includes a README.TXT file that provides additional details about the ink storage and reproduction process.
