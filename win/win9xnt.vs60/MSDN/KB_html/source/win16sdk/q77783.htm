SAMPLE: Modal Dialog Child of Modeless Dialog Box Sample Code

 Last reviewed: August 13, 1997
Article ID: Q77783

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows versions 3.0 and 3.1

Mdlmdls.exe is a file in the Microsoft Software Library that demonstrates how to create a modal child of a modeless dialog box. The sample uses two modeless dialog boxes. However, when the second dialog is created, the first dialog and the main application window are disabled.

The following file is available for download from the Microsoft Software Library: 

 ~ Mdlmdls.exe (size: 20914 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
