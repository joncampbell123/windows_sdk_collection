PRB: SPY Updates Only One Line with Small Buffer

 Last reviewed: July 22, 1997
Article ID: Q70889
3.00 3.10 WINDOWS kbtool kbprg kbprb

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows versions 3.1 and 3.0

SYMPTOMS

When using SPY, only the bottom line of the SPY window is updated as messages are received.

CAUSE

In the SPY Options dialog, the number of lines to save in the buffer is smaller than the number of lines that will fit in the SPY main window.

RESOLUTION

Specify a buffer size larger than the size of the SPY main window.
