DOCERR: EMC, EML, and EMR RTF Tokens Documented Incorrectly

 Last reviewed: July 23, 1997
Article ID: Q86923
3.10 WINDOWS kbtool kbprg kbdocerr

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

The online "SDK Reference" help file (WIN31HW.HLP) provided with the Microsoft Windows Software Development Kit version 3.1 documents that the RTF (rich-text format) tokens EMC, EML, and EMR insert an embedded window into a Windows Help file. This information is incorrect; the correct tokens are EWC, EWL, and EWR, respectively.
