SAMPLE: Multimedia Registration Kit Revision 3.0

 Last reviewed: July 3, 1996
Article ID: Q120253
The information in this article applies to:
  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

SUMMARY

The Multimedia Registration Packet (MDRK.EXE) is a collection of updated documents that developers can use for help in writing multimedia applications for the Windows environment,

Download MDRK.EXE, a self-extracting file, from the Microsoft Software Library (MSL) on the following services:

  • Microsoft Download Service (MSDL) 

          Dial (206) 936-6735 to connect to MSDL
      Download MDRK.EXE (size: 233646 bytes)
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the \softlib\mslfiles directory
      Get MDRK.EXE (size: 233646 bytes)

MORE INFORMATION

The packet includes Microsoft Word and ASCII text versions of the following files:

MMREG Multimedia Registration Kit

This kit enables a developer to register new formats and product IDs with Microsoft for the purpose of providing support for the format by other developers.

MSFTMM Multimedia Document Overview

This document provides a list of the current documents available from the Multimedia Technologies group at Microsoft.

RIFFNEW Multimedia Data Types and Techniques

This document provides a comprehensive description of available multimedia data types including Resource Interchange File Format (RIFF) and wave types.

PROD_IDS New Multimedia IDs

This is a complete list of registered manufacturer's and their registered products.

MMREG.H

This header file contains multimedia data types and product IDs that you can include in the source code of a multimedia C program.

Additional reference words: REGISTRATION MULTIMEDIA RIFF RIFFNEW 3.10
