BUG: True Type Font Display Problem with WINHELP ver 3.10.151

 Last reviewed: January 15, 1998
Article ID: Q123332
3.10 WINDOWS kbtool kbbuglist

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

SYMPTOMS

WINHELP.EXE version 3.10.151 is included in Traditional Chinese Windows version 3.1. When displaying help files using WINHELP.EXE version 3.10.151, large English characters ( 24pts and above) in True Type Font have a distorted appearance. Large True Type Font Chinese characters in the same help file are not affected. Non-True-Type-Font characters are not affected.

STATUS

Microsoft Japan has confirmed this to be a bug in Japanese Windows version 3.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
