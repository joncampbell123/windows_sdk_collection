Menus Supported in Dialog Boxes w/o DS_MODALFRAME Style

 Last reviewed: July 22, 1997
Article ID: Q71499
3.00 3.10 WINDOWS kbprg

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows, versions 3.0 and 3.1

Dialog boxes in Windows version 3.0 and later suppport menus as long as the DS_MODALFRAME style is absent and the WS_SYSMENU style is present. This behavior will continue to be supported in future versions of Windows.

Dialogs with a modal frame and menu do not work and are not designed to do so.
