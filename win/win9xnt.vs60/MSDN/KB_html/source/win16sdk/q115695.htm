HOWTO: Search for Windows SDK Articles Using KBKeywords

 Last reviewed: March 10, 1998
Article ID: Q115695
The information in this article applies to:
  • Microsoft Windows Software Development Kit (SDK) for Windows, versions 3.0, 3.1

SUMMARY

The Microsoft Knowledge Base (KB) is categorized by using keywords. This article lists the keywords specific to articles in the Microsoft Windows Software Development Kit (SDK) for Windows collection.

MORE INFORMATION

Microsoft Windows SDK KBKeywords Major and Minor Keywords

Each KB article in the Windows SDK collection may contain one or more product-specific keywords (called KBKeywords) that place the article in an appropriate Windows SDK category. Each KBKeyword is composed of the concatenation of a major topic keyword and a minor topic keyword. For example, you can find all the GDI display issue articles by using GdiDisplay as the keyword when you query the Microsoft Knowledge Base.

Use the asterisk (*) wildcard to find articles that fall into the general categories or into an intermediate subcategory. For example, to find all the articles that apply to network issues, query on Ntwk*. An article usually has only one KBkeyword, but it may have more.

Product-Specific Keywords

You can use the KBKeywords to organize Windows SDK articles or to search for specific groups of Windows SDK articles. For information about KBKeywords for other Microsoft developer products, please query the Microsoft Knowledge Base using these keywords: 

   dskbguide and kbkeyword

KB-wide Keywords

Each article in the Windows SDK collection also contains at least one generic, KB-wide keyword. The KB-wide keywords are standard throughout the Microsoft Knowledge Base, appearing in all KB articles, regardless of product. You can use the KB-wide keywords to organize all KB articles or to search for articles across several Microsoft products. For more information about these KB-wide keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
