MHS: Err Msg: Inpost: General File I/O Error with MHS Swap

 Last reviewed: October 20, 1997
Article ID: Q101681
3.00 MS-DOS kb3rdparty kberrmsg

The information in this article applies to:

  • Microsoft Mail Gateway to MHS, version 3.0

SYMPTOMS

The following error message may appear when Microsoft Mail Gateway to MHS is run in conjunction with Infinite Technologies MHS Swap software: 

   Inpost: General file I/O error

CAUSE

This software and the MHS gateway are incompatible because the gateway cannot have its temporary files redirected.

MORE INFORMATION

MHS Swap is an MHS utility from Infinite Technologies that swaps the temporary files from the network MHS database to the local computer where the MHS Host software is being run. MHS would usually use the MHS\SW\SWAP.FLG file to direct temporary file creation to its directory.
