SMTP: Dir-Sync Effect on SMTP Addresses

 Last reviewed: October 20, 1997
Article ID: Q87038
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to SMTP, version 3.0

Dir-Sync can affect SMTP addresses if it is enabled. The address lists that can be affected that have SMTP addresses are the global address list (GAL) and the SMTP address list.

For more information on the use of DirSync, see the "Global Directory Synchronization" section of the "Microsoft Mail Administrator's Guide."
