FAX: How to Check Whether Mail Is Queued for the Gateway

 Last reviewed: October 20, 1997
Article ID: Q85357
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to FAX, version 3.0

To check whether Microsoft Mail is queued for the gateway, follow these steps:

  1. From the Admin program, choose Queue.

  2. Enter FAX at the network name prompt or select it from the list of available networks.

A list of messages queued for the fax gateway will be displayed.
