PC WFW: Features of Extensions for Windows for Workgroups

 Last reviewed: October 23, 1995
Article ID: Q91038
The information in this article applies to:
  • Microsoft Mail and Schedule+ Extensions for Windows for Workgroups, version 3.0

Version 3.0 of Microsoft Mail and Schedule+ Extensions for Windows for Workgroups contains EXTEND.EXE, which automatically upgrades an existing Windows version 3.1 for Workgroups postoffice to a version 3.0 Microsoft Mail for PC Networks postoffice.

Upgrading to Microsoft Mail allows you to use the following additional features:

  • Exchange mail and share schedules with users on other Microsoft Mail postoffices on the network.
  • Exchange mail and Schedule+ meeting requests with remote Microsoft Mail postoffices.
  • Exchange mail with other mail systems, such as MCI Mail, PROFS, MHS, SNADS, SMTP, and with fax systems through gateways.
  • Dial in from a Microsoft Mail Remote workstation to send and receive messages.
  • Exchange mail with users on Macintosh, MS-DOS, Windows, and OS/2 computers.
  • Exchange and automatically update user names in the Global Address List.
