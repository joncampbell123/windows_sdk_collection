SMTP: RFCs Supported by the Gateway

 Last reviewed: October 20, 1997
Article ID: Q88777
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to SMTP, version 3.0

These requests for comments (RFCs) are supported by the Microsoft Mail Gateway to SMTP:

  • RFC-821 (SMTP) for message exchange
  • RFC-822 for message format
  • RFC-1154 for attachment support

REFERENCES

Titled attachments on page 80 (in Appendix C, "Gateway Features that Affect Mail Users") of the "Microsoft Mail Gateway to SMTP Administrator's Guide"
