PC WRmt: How to Install Supplemental Modem Scripts

 Last reviewed: November 22, 1995
Article ID: Q140068
The information in this article applies to:
  • Microsoft Mail Remote for Windows, version 3.2

SUMMARY

The combined software package of version 3.0 of Microsoft Mail Remote for MS-DOS and version 3.2 of Microsoft Mail Remote for Windows includes a disk labeled Microsoft Mail Supplementary Modem Scripts for PC Networks Version 3.5.

You can use the following procedure to install the modem scripts:

  1. Install Mail Remote for Windows as recommended in the "User's Guide."

  2. Insert the Microsoft Mail Supplementary Modem Scripts for PC Networks Version 3.5 disk into the appropriate disk drive.

  3. From File Manager, double-click SETUP.EXE.

  4. At the Welcome screen, press ENTER to continue.

  5. At the External Destination Directory screen, type the following path to the postoffice directory: 

          <drive>:\WINDOWS\MSMAIL\MSRMT\

    NOTE: If the Microsoft Windows operating system was installed to a different directory than <drive>:\WINDOWS\, then substitute that directory in the path above.

  6. Press ENTER to continue.

  7. At the Script Install Selection screen, scroll with the cursor keys to highlight the appropriate script. Press the ENTER key to select the script.

  8. Repeat step 7 as many times as necessary. When you are done selecting scripts, highlight the Continue Installation with Selected Script(s) options. Press ENTER to continue.

  9. At the Script Installation Complete screen, press any key to exit.

How to select the scripts from the Windows Remote client

  1. Run Mail Remote for Windows.

  2. From the Mail menu, choose Communications.

  3. From the Communications dialog box, click the down arrow in the Script field and scroll to the appropriate script. Click the script to select it. Click OK to continue.

MORE INFORMATION

The disk labeled Microsoft Mail Supplementary Modem Scripts For PC Networks Version 3.5. is also included with version 3.5 of Microsoft Mail for PC Networks.
