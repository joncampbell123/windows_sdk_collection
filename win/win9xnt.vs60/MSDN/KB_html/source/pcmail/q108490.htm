PC Win: Creating Groups That Contain Other Groups

 Last reviewed: October 31, 1994
Article ID: Q108490
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0, 3.0b, and 3.2

In versions 3.0, 3.0b and 3.2 of Microsoft Mail for Windows, you can define groups that contain other groups in the Personal Address Book under two conditions:

  • The included groups must be defined in the Postoffice Address List or the Global Address List, and cannot themselves contain other groups
  • The included groups cannot be defined in the Personal Address Book
