PC Win: MMF Displays No Messages After Integrity Check

 Last reviewed: December 15, 1995
Article ID: Q141065
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.2 and 3.5

SYMPTOMS

If you do an integrity check on your Mail Message File (MMF), your folders and messages may disappear.

CAUSE

This may happen if the pointers stored in the MMF are corrupt.

RESOLUTION

  1. Create a new MMF with a different name.

  2. From the File menu, choose Import. From this option, the folders and messages will be viewable for import into the new MMF.

  3. Select the folders and messages you want to import to your new MMF.
