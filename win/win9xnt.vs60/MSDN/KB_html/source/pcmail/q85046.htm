FFAPI: Components of the Application FFAPI

 Last reviewed: October 20, 1997
Article ID: Q85046
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Software Development Kit: FFAPI for Gateways and Applications, version 3.0

Here is a list of the components of the Application File Format Application Programming Interface (FFAPI):

Application FFAPI

   Filename     Description
   --------     -----------

   APPDIR.EXE   Extracts address list from MS Mail postoffice.

   APPPUT.EXE   Inserts mail items into MS Mail database.

   APPGET.EXE   Extracts mail for specified user from MS Mail
                database.

Application Remote FFAPI

   Filename     Description
   --------     -----------

   RMTGET.EXE   Extracts mail items from remote user's outbox.

   RMTPUT.EXE   Inserts mail items into remote user's inbox.
More detailed information on Application FFAPI components can be found in the appropriate FFAPI documentation.
