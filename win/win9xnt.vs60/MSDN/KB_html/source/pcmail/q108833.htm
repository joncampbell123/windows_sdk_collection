PC Adm: Err Msg: Notice 170 The Folder Index Below...

 Last reviewed: October 26, 1994
Article ID: Q108833
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 3.0, 3.0a, and 3.2

SYMPTOMS

With versions 3.0 and 3.0a of Microsoft Mail for PC Networks, when the Mail administrator tries to compress folders by choosing Local_Admin, Folders, Compress, the following error message may appear: 

   Notice 170 The Folder Index below is missing

CAUSE

This error occurs if the folders have been moved to private storage or if they are on the server but the index file is missing.

RESOLUTION

If the administrator presses the ENTER key, compression continues.

Microsoft has confirmed this to be a problem in versions 3.0 and 3.0b of Microsoft Mail for PC Networks. This problem was corrected in version 3.2.
