PC DOS: Command-Line Parameters for MSCAPT.EXE

 Last reviewed: October 27, 1994
Article ID: Q112207
The information in this article applies to:
  • Microsoft Mail for PC Networks, MS-DOS workstation, versions 3.0 and 3.2

Command-line parameters for the Microsoft Mail Screen Capture program (MSCAPT.EXE) are not listed in the Microsoft Mail MS-DOS client "User's Guide."

You can display the command-line parameters for MSCAPT.EXE by using the -? parameter at an MS-DOS prompt, as follows: 

   mscapt -?
The following information is displayed: 

   Usage:
        -c:garp     To set directory to C:GARP
        -w          To disable the keyboard in Windows
        -u          To uninstall E:\MSCAPT.EXE
        -o          To install more than one version
        -g          To enable keyboard <regardless>

   Set SNAP=c:\garp to set environment directory
