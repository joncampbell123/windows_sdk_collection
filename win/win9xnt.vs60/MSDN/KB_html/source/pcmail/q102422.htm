PC OS2: Doc Err: Associating Applications and File Extensions

 Last reviewed: October 21, 1994
Article ID: Q102422
The information in this article applies to:
  • Microsoft Mail for PC Networks, OS/2 workstation, versions 3.0, 3.0b, and 3.2

SYMPTOMS

Section E.10, "Associating Applications and Files Extensions," in the README.TXT file for version 3.0 or 3.0b of Microsoft Mail for PC Networks, OS/2 workstation, contains an error.

The second sentence of the first paragraph in section E.10 incorrectly states: 

   To do this, add a section to the WIN.INI file in your OS2/SYSTEM
   directory.
The WIN.INI file is located in the OS2\DLL directory, not, as stated above, in the OS2\SYSTEM directory.

STATUS

Microsoft has confirmed this to be a problem in the README.TXT file for version 3.0 of Microsoft Mail for PC Networks. This problem was corrected in the README.TXT file for version 3.2 of Microsoft Mail for PC Networks (section F.9).
