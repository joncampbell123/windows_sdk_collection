PC Win: Row/Column Numbers Missing in Excel OLE Object

 Last reviewed: October 29, 1994
Article ID: Q81532
The information in this article applies to:
  • Microsoft Mail for Windows, version 3.0

SYMPTOMS

When you insert a spreadsheet Object Linking and Embedding (OLE) object from Microsoft Excel version 3.0 for Windows into version 3.0 of Microsoft Mail for Windows, the object will be displayed with no column or row numbers.

RESOLUTION

If you double-click the spreadsheet object, then choose Update from the Excel File menu, the spreadsheet object then contains the column and row numbers.
