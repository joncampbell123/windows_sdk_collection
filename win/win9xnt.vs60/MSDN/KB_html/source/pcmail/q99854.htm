SMTP: Err Msg: 500 Huh?

 Last reviewed: October 20, 1997
Article ID: Q99854
2.10 3.00 MS-DOS kb3rdparty kbtlc

The information in this article applies to:

  • Microsoft Mail Gateway to SMTP, versions 2.1 and 3.0

SYMPTOMS

Versions 2.1 and 3.0 of Microsoft Mail Gateway to SMTP may display this message: 

   500 Huh?

CAUSE

This error message indicates that an unknown SMTP command was received by the gateway. After sending this error message, the SMTP Gateway usually closes the connection.

RESOLUTION

Check the SMTPGATE.LOG file to find the command that the SMTP did not understand. Then, contact the sending host to ascertain why the command was given.
