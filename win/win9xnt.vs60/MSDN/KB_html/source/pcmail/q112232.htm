FaxDrv: Installing the Fax Driver for Windows

 Last reviewed: October 20, 1997
Article ID: Q112232
3.00 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to FAX, version 3.0

SUMMARY

The Microsoft Mail Gateway to FAX Windows printer driver is a Windows printer driver designed to function with Microsoft Mail for Windows and the Microsoft Mail Gateway to FAX. The driver enables a Windows document or image to be sent through Mail Windows for Windows to the Mail Gateway to FAX.

MORE INFORMATION

The windows driver is installed like any Windows printer driver.

  1. Open the Control Panel and select Printers.

  2. Select the Add button.

  3. In the List of Printers box, select Install Unlisted or Updated Printer.

  4. Choose the Install button.

  5. When the dialog box appears prompting for the diskette, insert the Microsoft FAX diskette and type A:\usa or B:\usa at the prompt.

  6. In the List of Printers dialog box, make sure Microsoft Fax is selected. Choose the OK button. The Microsoft Fax printer driver is now installed.

NOTE: The Send Fax command is added to the File menu of Microsoft Word for Windows and Microsoft Excel.
