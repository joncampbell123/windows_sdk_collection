PROFS: How to Check VM Host ROUTE Statements

 Last reviewed: October 20, 1997
Article ID: Q98341
3.10 3.20 3.30 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to IBM PROFS and OfficeVision, versions 3.1, 3.2 and 3.3

VM users can use the following statement used to check that the ROUTE statements for the PROFS gateway are correct and properly initialized. 

   SM rscs_nodename Q SYS RO
It shows all the ROUTE statements that were initialized the last time the given RSCS was rerouted.
