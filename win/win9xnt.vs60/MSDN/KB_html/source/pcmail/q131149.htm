PC Gen: Err Msg: Unknown User with MOVEFOLD.EXE

 Last reviewed: June 5, 1995
Article ID: Q131149
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 3.2 and 3.2a

SYMPTOMS

When you use the MOVEFOLD.EXE utility, included in the Database Utilities (Application Note WA0641) for Microsoft Mail for PC Networks, you may receive the error 

   Unknown User
after you enter the command to create the export archive.

CAUSE

This error will be displayed if the postoffice has Enhanced Security installed.

RESOLUTION

To determine if Enhanced Security is installed, run ESCNTLR.EXE, included with the Database Utilities. If the postoffice has Enhanced Security installed, you must uninstall Enhanced Security. Run the MOVEFOLD, and then reinstall Enhanced Security.

To remove Enhanced Security you will need to use UPDATE.EXE for version 3.0 of Microsoft Mail for PC Networks and SETUP.EXE for version 3.2 of Microsoft Mail for PC Networks. You may also use the ESCNTLR.EXE utility provided with the Database Utilities Application Note (WA0641).
