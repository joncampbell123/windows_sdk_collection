X400: Comma Before Final Term Omitted by Display Format

 Last reviewed: October 20, 1997
Article ID: Q115364
3.20 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to X.400, version 3.2

The -AR option in version 3.0 displays the address in the following format: 

   givenName initials surname, generationQualifier
but the current address display format omits the comma before the final term: 

   givenName initials surname generationQualifier
