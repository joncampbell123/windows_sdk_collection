PC DOS: Monitor Erases Message When Trying to Forward Message

 Last reviewed: October 17, 1995
Article ID: Q125251
The information in this article applies to:
  • Microsoft Mail for PC Networks, MS-DOS workstation, versions 3.0, 3.0a, and 3.2

SYMPTOMS

In Microsoft Mail 3.x, the MONITOR.EXE program will erase the message when you try to do more than one forward.

RESOLUTION

In order to forward a message more than once with MONITOR.EXE, you can use the following workaround:

  1. After forwarding the first time, press the ESCAPE key three times to go back to the main menu. If you press the key less than three, the original message will not be visible.

  2. At the main menu, an additional forward can be performed.
