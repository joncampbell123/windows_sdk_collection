PC Win: Search All Folders Does Not Search Deleted Mail Folder

 Last reviewed: January 25, 1995
Article ID: Q125365
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 3.2 and 3.2a

In Microsoft Mail for PC Networks windows client, you can use the Message Finder command to search for messages. You can choose to search a single folder or all folders. If you select Look In All Folders, the messages in the Deleted Mail folder will not be searched.
