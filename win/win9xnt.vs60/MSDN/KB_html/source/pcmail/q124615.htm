PC Gen: Post Microsoft Mail 3.2a File Updates

 Last reviewed: June 19, 1996
Article ID: Q124615
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.2a

SUMMARY

Because of constant improvements to our product, it is necessary for Microsoft Mail for PC Networks customers to upgrade individual files to assure them of product integrity and up-to-date functionality. Below is a concise listing of required files for customers to have the most current client/postoffice configurations.

MORE INFORMATION

As of January 1995, version 3.2a is the most recent revision of our PC Mail product. Executable (*.EXE) files that shipped with version 3.2a are version 3.2.9. .DLL files shipped in 3.2a are version 3.2.0.4074. Although version 3.2a is the latest full revision, we continually update individual files to include fixes and improve performance. The following is a list of updated files you should obtain in conjunction with version 3.2a to ensure that you are running the latest version.
