PC DB: Structure of the MAI Directory

 Last reviewed: May 16, 1995
Article ID: Q87409
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 2.1e, 3.0, and 3.2

The MAI directory contains 16 subdirectories. These directories are named as follows and contain the following types of filenames: 

   Subdirectory  Filename Type
   --------------------------------------------------------------------

   MA0           nnnnnnn0.MAI
   MA1           nnnnnnn1.MAI
   MA2           nnnnnnn2.MAI
   MA3           nnnnnnn3.MAI
   MA4           nnnnnnn4.MAI
   MA5           nnnnnnn5.MAI
   MA6           nnnnnnn6.MAI
   MA7           nnnnnnn7.MAI
   MA8           nnnnnnn8.MAI
   MA9           nnnnnnn9.MAI
   MAA           nnnnnnnA.MAI
   MAB           nnnnnnnB.MAI
   MAC           nnnnnnnC.MAI
   MAD           nnnnnnnD.MAI
   MAE           nnnnnnnE.MAI
   MAF           nnnnnnnF.MAI
