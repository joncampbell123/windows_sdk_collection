PC Win: Err Msg: ...Make Sure <Friendly Names> Are Enabled...

 Last reviewed: July 24, 1995
Article ID: Q132962
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0, 3.2, and 3.2a
  • Microsoft Project for Windows, version 4.0

SYMPTOMS

When you use Microsoft Project for Windows version 4.0, you may encounter the following error: 

   A mail error has occurred. Make sure 'friendly
   names' are enabled if your mail server supports them.
This usually occurs after you select Add Routing Slip from the File menu of Microsoft Project and signing into a Microsoft Mail session.

CAUSE

The error is caused when the user has set the Global Address List (GAL) as the default directory in Microsoft Mail for Windows.

NOTE: To see if the GAL is set as the default directory, look for the following entry in the MSMAIL.INI file: 

   [Address Book]
   Directory Name = Global Address List
