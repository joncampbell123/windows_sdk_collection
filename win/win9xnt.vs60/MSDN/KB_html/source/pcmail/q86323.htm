PC Win: How to Move Folder to Top Level

 Last reviewed: October 31, 1994
Article ID: Q86323
The information in this article applies to:
  • Microsoft Mail for Windows, version 3.0

In version 3.0 of Microsoft Mail for Windows, there are two ways to move or copy a folder to the top level.

  1. Select the folder, then choose Move or Copy from the File menu. When the dialog box comes up, hold down the HOME key while clicking OK.

  2. Select the folder, then choose Folder Properties from the File menu. Select Top Level and click OK.
