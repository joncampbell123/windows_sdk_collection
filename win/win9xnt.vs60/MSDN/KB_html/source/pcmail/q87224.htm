FFAPI: Memory and Disk Usage

 Last reviewed: October 20, 1997
Article ID: Q87224
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Software Development Kit (SDK) File Format Application Program Interface (FFAPI), version 3.0

The Microsoft Mail File Format APIs are available in both large and small memory model. The large memory model API will better handle large messages and address lists. 

   Memory Model     Disk Space      RAM
   ------------     ----------      ---

   Small             75K            75K
   Large            100K            95K
This applies to both gateway and application FFAPI.

REFERENCES
