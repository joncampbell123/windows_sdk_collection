PC DirSync: IMPORT.EXE Command-Line Options Used and Ignored

 Last reviewed: May 24, 1996
Article ID: Q119016
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 3.0, 3.2, and 3.5

The Microsoft Mail for PC Networks IMPORT.EXE utility will ignore certain command-line options when certain other options are used. Under normal circumstances, this behavior is valuable because it keeps the administrator from making mistakes. However, to clarify the expected behavior, this article describes some of the options available for use and the corresponding options that are automatically ignored. 

   Option(s) Used               Option(s) Ignored
   --------------------------------------------------------
       -e                       -a -c -g -l -q -r -t -v -x
       -s                       -a -c -e -g -q -r -t -x -f
       -st -f                   -a -c -e -g -q -r -t -x
       -q                       -a -c -f -g -r -t -x
       -x                       -a -c -r
