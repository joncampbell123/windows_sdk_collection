PC OS2: Group Boxes in Options Dialog Are Unavailable

 Last reviewed: October 12, 1994
Article ID: Q82911
The information in this article applies to:
  • Microsoft Mail for PC Networks, OS/2 workstation, version 3.0

SYMPTOMS

When you choose Options from the Mail menu, the text for the group boxes is unavailable (grayed) even though the options within each group box are available.

CAUSE

The text for group boxes uses the default window frame color. With Microsoft Windows 3.0 and 3.1 systems, the default color is usually black. With OS/2, the default window frame color is gray.

RESOLUTION

Use the OS/2 Control Panel to change the default window frame color to black.
