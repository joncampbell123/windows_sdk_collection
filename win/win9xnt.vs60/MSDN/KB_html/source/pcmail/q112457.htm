PC Win: Err Msg: A Disk Error Occurred While Performing...

 Last reviewed: October 24, 1995
Article ID: Q112457
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0, 3.0b, and 3.2

SYMPTOMS

If a user selects Mail, Address Book and tries to switch to the Postoffice Address List or the Global Address List, the following error message may be displayed: 

   A disk error occurred while performing the operation

CAUSE

This error is caused by a locked open address list file (ADMIN.NME or GAL.NME).

RESOLUTION

If the file is locked open, close the file.

On a Novell Server

  1. You can determine the status of locked files by starting the RCONSOLE program.

  2. Choose the FileOpen/Lock activity.

  3. Choose <volume>/<dir>/<subdir>/MASTER.GLB. This will yield a connection number.

  4. From Connection Activity, you can determine who has the file open.

On a LAN Manager Server

  1. Check for locked files by running the NET ADMIN command.

  2. Select Status, Opened Files.

For Other File Servers

Consult the administrator's guide for that server to determine if a file is locked open and by whom.
