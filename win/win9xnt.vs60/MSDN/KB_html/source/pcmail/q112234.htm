FaxDrv: Improper Scaling with Excel 4.0

 Last reviewed: October 20, 1997
Article ID: Q112234
3.00a WINDOWS kb3rdparty kbinterop

The information in this article applies to:

  • Microsoft Mail Gateway to FAX, version 3.0a
  • Microsoft Excel, version 4.0

SYMPTOMS

The Microsoft Mail Gateway to FAX, FAX Printer Driver (FPD) may exhibit scaling problems with Excel version 4.0.

CAUSE

Printer drivers do not control how the image is scaled. Scaling is left to the application.

RESOLUTION

Select the Print Preview command before printing the image. If it is not scaled the way you want it, correct it before printing.

This problem does not occur if you connect the Microsoft FAX driver to Port rather than LPT1.
