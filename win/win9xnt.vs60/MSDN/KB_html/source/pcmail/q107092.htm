PROFS: Manually Regenerating the PRX Files for Host Access

 Last reviewed: October 20, 1997
Article ID: Q107092
3.20 3.30 3.40 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to IBM PROFS and OfficeVision, versions 3.2, 3.3, and 3.4

When you install the host access component of Microsoft Mail Gateway to IBM PROFS and OfficeVision, several of the modules and files generated are associated with the PROREXX software from the System Center libraries. During some installations, the PRX files are not correctly generated, so here is a method for manually regenerating them: 

   At the CMS prompt, type:

   PRXNLS

   These files are generated:

   PRXUME
   PRXUPA
   PRXUSY
   PRXUME
NOTE: File type of the files depends on the version of CMS being used.
