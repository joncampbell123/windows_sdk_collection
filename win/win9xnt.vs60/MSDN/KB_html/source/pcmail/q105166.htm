PC Ext: MinKDiskFull & MinKDiskNotFull Not Written to Log File

 Last reviewed: October 29, 1994
Article ID: Q105166
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.2

SYMPTOMS

With version 3.2 of the Microsoft Mail for PC Networks External Mail program, if you include the MinKDiskFull and MinKDiskNotFull options in the EXTERNAL.INI file, the two lines are not written to the log file. If you include the two file options at the command line, they are written to the log file.

STATUS

Microsoft has confirmed this to be a problem in version 3.2 of Microsoft Mail for PC Networks. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
