PC Win: GP Fault in Module AB.EXE Using Mail w/ Above & Beyond

 Last reviewed: May 19, 1995
Article ID: Q95962
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0 and 3.0b
  • 1Soft Corporation Above and Beyond, version 2.0

SYMPTOMS

An attempt to use 1Soft Corporation Above and Beyond, version 2.0, with version 3.0 or 3.0b Microsoft Mail for Windows fails when a general protection (GP) fault occurs in module AB.EXE.

CAUSE

The fault occurs because each product has a module named "AB."

RESOLUTION

Above and Beyond version 2.1 corrects this problem. To obtain the latest version of Above and Beyond, contact 1Soft technical support at (707) 987-0256.
