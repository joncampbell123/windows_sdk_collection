Gtwy: Gateway Connectivity Information

 Last reviewed: October 20, 1997
Article ID: Q118674
3.20 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Mail for PC Networks, versions 2.1, 3.0, and 3.2

The following list identifies the Microsoft Mail gateways that can be used to integrate Mail in a heterogeneous mail environment. Additional gateways that use the Microsoft Mail File Format API (FFAPI) may also be available.

NOTE: All gateways in the following list are Microsoft gateways unless otherwise noted.

   ARCOM 400 Swiss PTT          X.400

   Atlas 400                    X.400 (NF mark from Afnor)
   Banyan Mail                  MHS or X.400
   Beyond Mail                  MHS

   CompuServe                   SMTP or MHS
   The Coordinator              MHS

   DEC VMS Mail                 X.400 or SNADS

   GEIS                         QuickComm gateway from GE
