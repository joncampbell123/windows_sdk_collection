PC Win: Err Msg: A Message Could Not Be Copied

 Last reviewed: October 29, 1994
Article ID: Q104297
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0, 3.0b, and 3.2

SYMPTOMS

When using versions 3.0, 3.0b, and 3.2 of Microsoft Mail for Windows to copy messages to different folders, the following error message may appear: 

   A message could not be copied

CAUSE

The hard disk may be full.

RESOLUTION

Make sure there is still some free space on the hard disk. If there is none, free up some disk space.
