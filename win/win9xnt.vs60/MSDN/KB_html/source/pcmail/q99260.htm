PC Forms: Err Msg: Invalid Message

 Last reviewed: November 17, 1994
Article ID: Q99260
The information in this article applies to:
  • Microsoft Electronic Forms Designer, version 1.0, running with Microsoft Mail for PC Networks, version 3.0

SYMPTOMS

When you attempt to delete a message from a shared folder that was created using version 1.0 of Microsoft Electronic Forms Designer, the following error message may be displayed: 

   Invalid Message

CAUSE

This error can occur when you use version 3.0 of Microsoft Mail for Windows. This is a known problem with custom message types and shared folders in the Windows client.

RESOLUTION

Upgrade to version 3.2 of Microsoft Mail for Windows.
