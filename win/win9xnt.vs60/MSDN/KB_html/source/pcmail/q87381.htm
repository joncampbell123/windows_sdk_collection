PC DB: Symptoms of Not Having the ATT Subdirectory

 Last reviewed: August 29, 1995
Article ID: Q87381
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 2.1e, 3.0, and 3.2

The ATT attachment file is stored within one of the 16 subdirectories of the ATT subdirectory. If the subdirectory is not there, or the user is unable to create a file there, the following message will appear: 

   Error in sending attachment(s).
To prevent this error from occurring, use the MS-DOS MKDIR command to create the missing subdirectory.

For more information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q87378
   TITLE     : PC DB: ATT Directory Structure
