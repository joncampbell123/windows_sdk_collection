PC Win: Message Finders Closing Automatically

 Last reviewed: October 31, 1994
Article ID: Q86320
The information in this article applies to:
  • Microsoft Mail for Windows, version 3.0

The Message Finder is closed automatically when you exit Mail for Windows if the search criteria are blank.

Mail works under the assumption that if there is no search criteria, there is no reason to bring up a blank Message Finder when Mail for Windows is restarted.
