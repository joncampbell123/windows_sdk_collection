PC Mac: Microsoft Word Attachment Loses Its File Association

 Last reviewed: August 26, 1996
Article ID: Q150563
The information in this article applies to:
  • Microsoft Mail for PC Networks, Macintosh workstation, versions 3.0.6 and 3.5.1

SYMPTOMS

When a Microsoft Windows user sends mail with Microsoft Word for Windows attachment from Microsoft Mail for Windows to the Macintosh workstation for Mail for PC Networks, the attachment will loose its file association.

WORKAROUND

  1. Login to Mail for PC Networks on the Macintosh machine.

  2. Open the message and save the attachment to the desktop.

  3. Run Microsoft Word, and open the attachment.

  4. Click Save on the File menu to save the file. Quit Microsoft The file is associated with the Microsoft Word icon.
