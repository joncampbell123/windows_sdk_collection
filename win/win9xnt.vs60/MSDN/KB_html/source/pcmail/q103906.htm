PC DOS: Err Msg: Notice 80: Failure to Create Folder

 Last reviewed: October 24, 1995
Article ID: Q103906
The information in this article applies to:
  • Microsoft Mail for PC Networks, MS-DOS workstation, versions 2.1, 3.0, and 3.2

SYMPTOMS

When you create a folder in version 2.1, 3.0, or 3.2 of Microsoft Mail for PC Networks, MS-DOS workstation, you may receive the following error: 

   Notice 80: Failure to create folder.

CAUSE

This error occurs if the SERIALNO.IDX file is not available or accessible.

RESOLUTION

To resolve this problem, either unlock the file using the appropriate network utility or restore the file from a backup copy. The most recent backup copy should be safe for restoration with the consideration that folders cannot be created without this file being available.
