PC Win: Err Msg: The Application Could Not Be Executed

 Last reviewed: October 31, 1994
Article ID: Q87867
The information in this article applies to:
  • Microsoft Mail for Windows, version 3.0

SYMPTOMS

Attempting to view a WordPerfect version 5.1 for Windows document as an attachment in version 3.0 of Microsoft Mail for Windows may generate the following error: 

   The application could not be executed.

CAUSE

WordPerfect for Windows allows only one session of the application to be running. Close the current session of WordPerfect and then view the attachment.
