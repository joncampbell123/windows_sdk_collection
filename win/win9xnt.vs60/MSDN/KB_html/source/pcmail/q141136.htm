PC WRmt: GP Fault with Mail Remote and Hebrew WFW

 Last reviewed: December 15, 1995
Article ID: Q141136
The information in this article applies to:
  • Microsoft Mail Remote for Windows, version 3.2

SYMPTOMS

When you run the English version of Remote Mail for Windows on the Hebrew version of Microsoft Windows for Workgroups (WFW) version 3.11, you will receive the following error message: 

   Application Error
   MSMAIL caused a General Protection Fault in module
   MSMAIL.EXE at 0020:0C1f.

WORKAROUND

The only available workaround to this problem is to run the English version of Mail Remote with the English version of Windows for Workgroups.
