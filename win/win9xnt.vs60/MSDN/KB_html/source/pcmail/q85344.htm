FAX: Questions That Need to Be Answered During Installation

 Last reviewed: October 20, 1997
Article ID: Q85344
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to FAX, version 3.0

To install the Fax Gateway access component on a gateway postoffice, you must know the location of the Microsoft Mail data files and executables. If you are installing on a downstream postoffice, you must know the network and postoffice name of the gateway postoffice.

REFERENCES

The "Installing the Gateway" chapter of the "Microsoft Mail Gateway to Fax Administrator's Guide"
