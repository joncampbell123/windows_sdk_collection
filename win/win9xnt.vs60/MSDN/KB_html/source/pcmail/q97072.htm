PC WRmt: Alphanumeric Characters Only in AT&T Driver Addresses

 Last reviewed: October 31, 1994
Article ID: Q97072
The information in this article applies to:
  • Microsoft Mail Remote, Windows Client, version 3.2

In Microsoft Mail Remote version 3.2 for Windows, if you use the AT&T driver, do not enter nonalphanumeric characters (for example, !, :, ~) into the address template for an AT&T user, a mailing list, or a shared folder.

The AT&T Mail driver automatically adds nonalphanumeric characters as required. If you enter them into the address template, Mail creates invalid addresses and does not correctly deliver your messages.
