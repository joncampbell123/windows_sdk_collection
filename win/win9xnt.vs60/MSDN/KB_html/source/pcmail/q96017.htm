PROFS: Receive Command Drops the "$" on Download

 Last reviewed: October 20, 1997
Article ID: Q96017
3.30 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.3

SYMPTOMS

When the latest release of the Novell 3270 emulation software is installed, the Microsoft Mail Gateway to PROFS fails to download the PROFSOUT $MESSAGE. Mail generates a message similar to "Host file not found."

CAUSE

This error is caused by a problem with the latest Novell 3270 emulation software, which was released between October and December 1992.

RESOLUTION

Contact Novell to receive a patch for the RECEIVE program.
