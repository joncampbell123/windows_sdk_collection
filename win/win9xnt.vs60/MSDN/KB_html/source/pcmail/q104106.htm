PC Win: GP Fault During Setup When Installing Only Spelling

 Last reviewed: November 2, 1994
Article ID: Q104106
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0 and 3.0b

SYMPTOMS

If you specify the Custom Install option during the Setup process of version 3.0 or 3.0b of Microsoft Mail for Windows, and you choose to install only the spelling drivers, the following error occurs: 

   Could not open the file named: C:\WINDOWS\SYSTEM\TRNSCHED.DLL

   Choose Abort and click Yes to exit. Re-run SETUP.EXE.

   Result: GP Fault in MSCOMSTF.DLL @002D:001A.

CAUSE

This error occurs only if your current transport is not the PC Mail Transport; for example, if you are running Microsoft Mail Remote for Windows or if you do not have an existing installation of Mail.

RESOLUTION

Microsoft has confirmed this to be a problem in versions 3.0 and 3.0b of Microsoft Mail for Windows. This problem was corrected in version 3.2 of Microsoft Mail for Windows.
