X400: Gateway Does Not Support the PROBE Message Type

 Last reviewed: October 20, 1997
Article ID: Q134954
3.20 MS-DOS kb3rdparty kbother

The information in this article applies to:

  • Microsoft Mail Gateway to X400 version 3.2

SUMMARY

Version 3.2 of the Microsoft Mail Gateway to X400 currently does not have the capability of supporting the PROBE message type.

MORE INFORMATION

The PROBE message is defined in Recommendation X.411. It is an envelope containing the recipient's address and bodypart information. If the PROBE is successful, then the recipient can receive similar messages.

The ability to support PROBE message types may be addressed in future versions of Microsoft's X400 connectivity.
