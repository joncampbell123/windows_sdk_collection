OFF97: Using Netscape Navigator with Office 97

 Last reviewed: February 5, 1998
Article ID: Q165262
The information in this article applies to:
  • Microsoft Office 97 for Windows

SUMMARY

To configure Netscape Navigator so that you can use a proxy server or an Internet service provider with Microsoft Office 97, you must install the Internet Control Panel icon. You can find this program on the Microsoft Office 97 compact disc. If you are using a stand-alone version of a Microsoft Office program, you can find the Control Panel program in the Microsoft Software Library.

The following file is available for download from the Microsoft Software Library: 

 ~ Inetcpl.exe (size: 62957 bytes) 
               (62 KB)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

MORE INFORMATION

NOTE: This Control Panel program is supported only under Microsoft Windows 95 and Microsoft Windows NT 4.0.

Installation Instructions

After you download and extract the Inetcpl.cpl file from the self- extracting executable file, use the following steps to install the program.

  1. On the Start menu, point to Settings, and then click Control Panel.
2. Double-click the Internet Control Panel icon to open it. 3. Click the Connection tab and specify your connection type.

Specifying a Dial-Up Networking Connection:

  1. Click to select the "Connect to the Internet as needed" check box.
2. Click the Dial-Up Networking connection in the list (For example, 
   your MSN account or PPP provider).
3. If no connection type is defined, click Add and create a new 
   connection.
Specifying a Proxy Server Connection:

  1. Click to select the "Connect through a proxy server" check box.
2. Click Settings. 3. Type the name and port number of your proxy server. 

   This information is the same information that is in Netscape
   Navigator. (On the Options menu, click Network Preferences,
   click the Proxies tab, click Manual Proxy Configuration, and
   then click View.)
NOTE: You must use Manual Proxy Configuration. The Automatic Proxy Configuration feature in Netscape Navigator is not supported.
