OFF97: Hang Viewing Properties of MapInfo MapX Control

 Last reviewed: May 21, 1997
Article ID: Q168756
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

In any of the programs listed above, if you right-click a MapInfo MapX control in a document, point to MapInfo MapX Object on the shortcut menu, and then click Properties, the program may stop responding (hang).

CAUSE

This problem may occur if version 1.0 of the MapInfo MapX control is installed on your computer.

RESOLUTION

To prevent this problem from occurring, you should obtain version 2.0 of the MapX control from MapInfo Corporation. For information on how to obtain the new MapX control, use your Web browser to go to the following address on the Internet: 

   http://www.mapx.com/
If the problem has already occurred, you may have to press CTRL+ALT+DELETE to quit the program. Or, you may have to turn your computer off and then back on again.

STATUS

This is a known problem with version 1.0 of the MapInfo MapX control. The problem described above is corrected in version 2.0 of the MapInfo MapX control.

MORE INFORMATION

The MapInfo MapX control is designed to allow you to incorporate mapping functionality into programs that support the use of embedded controls.

You can quickly determine what version of the MapX control is installed on your computer by performing either of the following steps:

  • If the control is embedded in your document, right-click it. On the shortcut menu, you will see one (but not both) of the following menu items: 

          Menu Item                Version of MapX Installed
      --------------------------------------------------
      MapInfo MapX Object      1.0
      MapInfo MapX V2 Object   2.0
  • If the control is not yet embedded in your document, click the More Controls tool on the Control Toolbox. One of the following items will appear in the list of controls: 

          Control Name      Version of MapX Installed
      -------------------------------------------
      MapInfo MapX      1.0
      MapInfo MapX V2   2.0
If version 1.0 is installed on your computer, you should obtain version 2.0 from MapInfo Corporation.

NOTE: The problem described above will not occur if you right-click the control and then click Properties. The Properties menu item is different from the Properties submenu item attached to the MapInfo MapX Object menu item.
