OFFICE: Only Four Book Buttons Displayed on QuickShelf Toolbar

 Last reviewed: September 16, 1996
Article ID: Q122351
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SYMPTOMS

When you install Microsoft Bookshelf using the Microsoft Office Setup, and you choose to display the QuickShelf toolbar, only four book buttons are displayed on the Microsoft QuickShelf Button.

The QuickShelf toolbar displayed in the Microsoft Bookshelf Demo contains eight book buttons.

CAUSE

This behavior is by design of Microsoft Office. This is consistent with the number of buttons in the picture of the Microsoft QuickShelf toolbar displayed on the Microsoft Office package.

Note that the Microsoft Bookshelf Demo runs the first time you start Microsoft Bookshelf after installing the application. After this, you can run the demo by choosing Bookshelf Demo from the Help menu in Microsoft Bookshelf.

WORKAROUND

You can add, remove, and customize the buttons on the Microsoft QuickShelf toolbar. To add more buttons to the Microsoft QuickShelf toolbar, do the following:

  1. From the Microsoft QuickShelf menu, choose Customize.

  2. Select the Toolbar tab.

  3. Select the Find or Contents check box for any of the listed books.

  4. Choose OK.

MORE INFORMATION

By default, the following buttons are displayed on the QuickShelf toolbar after installing Microsoft Bookshelf using the Microsoft Office Setup: 

   Dictionary Contents
   Thesaurus Contents
   Encyclopedia Contents
   All Books Find
   Microsoft QuickShelf
