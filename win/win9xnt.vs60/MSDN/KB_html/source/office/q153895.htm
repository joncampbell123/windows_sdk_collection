OFF95: No Microsoft Office Folder Added to Programs Menu

 Last reviewed: March 27, 1997
Article ID: Q153895
7.00 7.00a WINDOWS kbusage kbdocerr

The information in this article applies to:

  • Microsoft Office for Windows 95, versions 7.0, 7.0a

MORE INFORMATION

Page 29 of the "Getting Results with Microsoft Office for Windows 95" manual describes how to start and quit Office applications. Under the heading "Start and Quit Office," it reads: 

   To start an Office application on Windows 95, click the Start button,
   click Programs, click Microsoft Office, and then click the application
   you want.
This information is incorrect. It should read: 

   To start an Office application on Windows 95, click the Start button,
   click Programs, and then click the application you want.

REFERENCES

"Getting Results with Microsoft Office for Windows 95," version 7.0, Part 1, "Install and Start Microsoft Office," page 29

For more information about Starting Programs in Windows 95, click Start, and choose Help from the menu. Click the Index tab in Help and type: 

   Starting, Programs
