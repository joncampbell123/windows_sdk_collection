OFF98: How to Install Value Pack Components

 Last reviewed: March 23, 1998
Article ID: Q181766
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SUMMARY

The Microsoft Office 98 Macintosh Edition Value Pack contains components that complement Microsoft Office and the individual Office programs. This article contains brief descriptions for each item in the Value Pack and instructions for installing these items.

MORE INFORMATION

Installing Value Pack Components

Use the Value Pack Installer to install the Value Pack components. To do this, follow these steps:

  1. Insert the Microsoft Office 98 compact disc (CD) into the CD-ROM drive.

  2. Open the Value Pack folder on the Office 98 CD.

  3. Double-click Value Pack Installer.

  4. Click to select any items in the list that you want to install.

    To see a list of the available components for a Value Pack component, click the triangle next to the program.

    A dash (-) in the check box for a Value Pack component indicates that some, but not all, options are selected for that program.

    An "X" in the check box for a Value Pack component indicates that all components are selected for that program.

    An empty check box indicates that no components will be installed for that Value Pack component.

  5. Click Install.

The selected Value Pack components are added to the existing Microsoft Office installation.

NOTE: You cannot use the Value Pack Installer to install Value Pack components if you are running Office from a network disk drive (a shared disk).

For additional information about using the Value Pack Installer with a network installation of Office, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q180274
   TITLE     : OFF98: "Default Location" Error Running Value Pack Installer
You cannot use the Value Pack Installer to remove Value Pack components from your computer.

NOTE: If you use the Remove Office 98 program, which is located in the Administration Tools folder of the Value Pack, to remove Office 98 from your computer, some files that are installed by the Value Pack Installer are not removed from your computer.

For additional information about removing Office 98 from your computer, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q179861
   TITLE     : OFF98: How to Remove Microsoft Office 98 from Your Computer

Description of Value Pack Components

Administration Tools

The Administration Tools folder contains several tools that can help you make a smooth transition to Office 98.

NOTE: Administration Tools are not installed by the Value Pack Installer. These tools do not require installation; you can run them directly from your Microsoft Office CD.

Remove Old Office Versions

You can use the Remove Old Office Versions program to remove earlier versions of Microsoft Word, Microsoft Excel, and Microsoft PowerPoint from your computer.

For additional information about using the Remove Old Office Versions program, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q179862
   TITLE     : OFF98: How to Remove Earlier Versions of Office Programs

Default Save

Each Office 98 program contains a feature that allows you to set the default save format. The Default Save folder contains Microsoft Word, Microsoft Excel, and Microsoft PowerPoint files that you can use to set the default save file format in these programs.

For additional information about using the default save files, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q181571
   TITLE     : OFF98: Using the Default Save Files in the Value Pack

Assistants

The Office Assistant provides Help and tips in your Office programs. Office Assistant is shared by all the Office programs. You can select an Assistant character that matches your personality and the way you work. Microsoft Office 98 includes a few Assistants; and you can install and select any of the additional Assistants in the Value Pack.

For information about selecting a different Assistant in your Office programs, see online Help in any Office program.

Bookshelf Integration

Microsoft Bookshelf is a multimedia reference collection with online books and materials that you can use to look up and cite information. To find information, you can search for titles in a table of contents or you can search for contents in a single reference or in all references at once.

When you install Microsoft Bookshelf 98, Microsoft Office 98, and the Bookshelf integration file, special Bookshelf commands are added to PowerPoint and Word. These commands allow you to look up information in Bookshelf without leaving your document, or to quickly find a definition for a word in your document. You can copy entire articles or selected text and images from Bookshelf directly to a new or existing file. When you do this, Bookshelf automatically includes copyright information in your file.

When you install the Bookshelf integration file, the Microsoft Reference Library file is added to the System Folder:Extensions folder. To install the Bookshelf integration file, run the Value Pack Installer, and then select the Bookshelf Integration check box.

Business Tools and Templates

The Business Tools and Templates component includes the Business Guide, checklists that complement the Business Guide, and 43 small business templates for Microsoft Excel, Word, and PowerPoint.

For more information about the files that are installed with the Business Tools And Templates component, see the Read Me! file that is installed in the Microsoft Office 98:Templates:Business Tools folder when you install the component.

Clipart

You can install additional clipart files, presentation and World Wide Web textures, and QuickTime movies that you can add to your Microsoft Office documents and presentations. Clipart files that are installed from the Value Pack are located in the Microsoft Office 98:Clipart folder.

Data Access

Microsoft Query is a tool you can use to construct a query and retrieve data from an external data source for use in a Microsoft Excel worksheet or a Word document.

Equation Editor

Equation Editor is a program that allows you to insert mathematical equations in your Microsoft Excel workbooks, Word documents, and PowerPoint presentations. Equation Editor is included in the Microsoft Office Installer; however, it is not installed if you select the Easy Install option.

TIP: For information about inserting an Equation Editor object in an Office program, see online Help in any Office program.

Fonts

You can install additional TrueType fonts to help you create, view, and print great looking documents. These additional fonts are installed in the Fonts folder and are available in any Macintosh program.

Genigraphics Wizard and GraphicsLink

You can install the Genigraphics Wizard and GraphsLink programs for use with PowerPoint. These files are included in the Microsoft Office Installer; however, they are not installed if you select the Easy Install option.

TIP: For information using the Genigraphics Wizard, see online Help in PowerPoint.

Microsoft Excel Add-Ins

You can install Microsoft Excel add-in files. These files are included in the Microsoft Office Installer; however, they are not installed if you select the Easy Install option.

TIP: For information about using these add-ins, see online Help in Excel.

Microsoft Movie

Microsoft Movie is a program that allows you to insert a QuickTime movie into your PowerPoint presentation, Word document, or Microsoft Excel workbook as a Microsoft Movie object.

When you install Microsoft Movie, the file Microsoft Movie is added to the Microsoft Office 98:Shared Applications folder. To insert a Microsoft Movie object in PowerPoint, Word, or Excel, click Object on the Insert menu, and then click Microsoft Movie in the Object type list.

Microsoft Office Manager

Microsoft Office Manager appears as an icon on the menu bar. When you click the icon, a menu appears that allows you to start an Office program or switch from one program to another. Clicking a program on the Microsoft Office Manager menu switches you to the program if it is already running or starts the program if it is not running. Microsoft Office Manager also gives you an option to switch between programs with the keyboard. When you install the Microsoft Office Manager, the file Microsoft Office Manager is added to the System Folder:Control Panels folder.

More Help

You can install Visual Basic Help files for Microsoft Excel, Word, and PowerPoint. These files are included in the Microsoft Office Installer; however, they are not installed if you select the Easy Install option.

When you install the More Help component, the Visual Basic Help files are added to the Microsoft Office 98:Office:Help folder.

PowerPoint Translators

This converter allows PowerPoint version 4.0 to open files that are created in PowerPoint 98 Macintosh Edition or PowerPoint 97 for Windows. Note that this file is not installed by the Value Pack Installer.

For more information about this converter, see the PP8Trans Read Me file that is located in the Value Pack:PowerPoint Translators folder on your Office CD.

Programmability

You can install Microsoft Excel and Word sample macros and Excel type libraries. When you install Word sample macros, the Sample Macros file is added to the Microsoft Office 98:Templates folder. When you install the Microsoft Excel VBA & Analysis Toolpak, the file is added to the Microsoft Office 98:Office:Excel Add-ins folder.

If you have Visual Basic for Applications macros that were created in a non- English version of Microsoft Excel 5.0, you must install the localized type library to translate and open the macros in Excel 98. The Value Pack Installer includes Excel type libraries for the following languages: Danish, Dutch, French, German, Italian, Japanese, Norwegian, Portuguese, Spanish, Swedish. When you install Excel type libraries, the files are added to the System Folder:Extensions:Type Libraries folder.

Proofing Tools

You can install Microsoft Office 98 hyphenation, dictionary, and thesaurus files for the following languages: Danish, Dutch, English, French, German, Italian, Norwegian, Portuguese, Spanish, and Swedish.

When you install a proofing tool, its corresponding files are added to the Microsoft Office 98:Shared Applications:Proofing Tools folder.

Support Files

The Support folder contains sample graphic files that you can use for troubleshooting the graphic filters in your Office Programs. These files are not installed by the Value Pack Installer.

Templates

To help you create great looking Office documents, you can install and use Word templates; Microsoft Excel templates; and PowerPoint templates. You can also install the templates that are installed with Word for Windows 97 to use in Word 98.

Text Converters

Install the Text with Layout, FoxPro - dBase, Word 1.x for Windows, Word 2.x for Windows, Word 3.x for the Macintosh, Word 3.x-6.0 for MS-DOS, Word Perfect 5.x, Works 2.x-3.x for the Macintosh, and Works 4.0 for the Macintosh converters to open these additional file types in Word.

Batch Converter

Install the batch converter to quickly convert multiple files of one file type to a different file type.

Word 97-98 Import Converter

Install the Word 97-98 Import converter to convert documents from Microsoft Word 97-98 format to Word 5.0 and 6.0 format for the Macintosh.

Unbinder

Binders, which are files that are created in Microsoft Office Binder for Windows, may contain any of the following types of files: Microsoft Excel workbooks, Word documents, and PowerPoint presentations. On the Macintosh, you cannot work on files while they're in a binder file, but you can work on the component files separately in the Office program that created the file.

To separate the component files from a binder file, use the Microsoft Unbind utility. When you install the Unbind utility, the Unbind file is added to the Microsoft Office 98 folder.

TIP: For information about using the Microsoft Unbind utility, see online Help in any Office program.

Word Speak

Word Speak is a Word add-in that allows Word to speak text out loud.

After installing and loading Word Speak, select the text you want Word to speak out loud, and then click Speak Selection on the Tools menu.

TIP: To change the voice that is used to speak text, click Pick Voice on the Tools menu.

Word Wizards and Template

You can install wizards and templates for Microsoft Word. These files are included in the Microsoft Office Installer; however, they are not installed if you select the Easy Install option.
