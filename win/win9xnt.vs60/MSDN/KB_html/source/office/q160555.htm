OFF97: Error Adding Blank Web Page to Binder

 Last reviewed: March 12, 1998
Article ID: Q160555

The information in this article applies to:

  • Microsoft Office 97 Windows

SYMPTOMS

When you add a Blank Web Page to a Microsoft Binder file, you may receive the following error message: 

   Web Page authoring features are not available while Word is running
   inside of another application.
and a blank Microsoft Word document is added to the Binder.

CAUSE

Binder does not support Hypertext Markup Language (HTML) documents; you cannot add an HTML document as a section.

NOTE: Microsoft Word 97 is the editor for the Blank Web Page template. This is not supported when you run Word under another program, such as Binder.

STATUS

This is by design of Microsoft Office Binder 97.
