Office for NT Setup Prompts for Name and Company Again

 Last reviewed: September 16, 1996
Article ID: Q123736
The information in this article applies to:
  • Microsoft Office for Windows NT, version 4.2
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft PowerPoint for Windows, version 4.0c

SUMMARY

When you install Microsoft Office for Windows NT, you are prompted for your name and company information again when you run the Setup program for Microsoft PowerPoint, even though you entered it when you installed Microsoft Word.

CAUSE

This behavior is by design of Microsoft Office for Windows NT version 4.2. Your name and company information for the 32-bit Microsoft Office applications is stored in a different location than for the 16-bit applications.

The instructions for installing Microsoft Office indicate that you should install the Microsoft Office applications in the following order: Microsoft Word, Microsoft Excel, Microsoft PowerPoint, and then Microsoft Office Manager. Note that Microsoft Word and Microsoft Excel are 32-bit applications, while Microsoft PowerPoint and Microsoft Office Manager are 16-bit applications.

Because of this, and assuming you install all of the Microsoft Office applications, you are prompted for your name and company information when you install Microsoft Word, but you are not prompted again when you install Microsoft Excel. You are prompted again for your name and company information when you install Microsoft PowerPoint, but you are not prompted again when you install Microsoft Office Manager.
