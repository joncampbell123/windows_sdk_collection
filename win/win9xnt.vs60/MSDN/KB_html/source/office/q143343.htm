OFF95: Network.txt Documentation Error

 Last reviewed: March 10, 1997
Article ID: Q143343
7.00 WINDOWS kbsetup kbnetwork kbdocerr

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The "Run From Network Server Errors on Novell 3.11" section in the Network.txt file that comes with Office for Windows 95, version 7.0, is incorrect.

The following is the incorrect text: 

   On some network configurations, the Run From Network Server
   client installation option may fail when the Windows 95
   driver for the DEC EtherWorks Turbo/TP (DE201) network
   adapter is being used on a Netware 3.11 server.
The following is the correct text: 

   On some network configurations, the Run From Network Server
   client installation option may fail when the DEC EtherWorks
   Turbo/TP (DE201) network adapter is being used on a Netware 3.11
   server.

MORE INFORMATION

The text is incorrect because the problem is not caused by a Microsoft Windows 95 network driver. Rather, the problem is caused by Depca.lan, which Novell provides.
