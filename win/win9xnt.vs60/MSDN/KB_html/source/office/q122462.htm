Office: Error Double-Clicking Empowerment Pack Alias from CD

 Last reviewed: November 21, 1997
Article ID: Q122462
The information in this article applies to:
  • Microsoft Empowerment Pack for the Mac, version 1.0
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SYMPTOMS

When you double-click the Microsoft Empowerment Pack alias on the Microsoft Empowerment Pack for the Mac compact disc, you may receive either of the following error messages: 

   If you have a Power Macintosh
   -----------------------------

      The application "Internet Explorer 2.0" could not be opened
      because "ThreadsLib" could not be found.

   Your only option with this error message is to click OK.

   If you have a 68k Macintosh
   ---------------------------------

      This program cannot run on this computer.

   Your only option is to click Quit.

CAUSE

You may experience either of these problems if your system software is earlier than System 7.5.

RESOLUTION

To resolve this problem, use the method appropriate for your type of Macintosh computer. 

Method 1:   If you are using a Power Macintosh computer, upgrade to System
            7.5 or later.

            NOTE: Even though the Empowerment Pack contains the installer
            for System 7.5 upgrade 2.0, you cannot run the installer for
            this if you are running a version of the System software
            earlier than 7.5. Therefore, you must contact Apple Computer,
            Inc., in order to obtain System 7.5, or later.

            For information about how to contact Apple Computer, Inc.,
            query in the Knowledge Base for one of the following articles:

             ARTICLE-ID: Q65416
             TITLE     : Hardware and Software Third-Party Vendor
                         Contact List, A-K

             ARTICLE-ID: Q60781
             TITLE     : Hardware and Software Third-Party Vendor
                         Contact List, L-P

             ARTICLE-ID: Q60782
             TITLE     : Hardware and Software Third-Party Vendor
                         Contact, Q-Z

Method 2:   If you using a 68k Macintosh, use the following steps to
            install the Thread Manager extension on your computer:

            a. Open the Internet Explorer folder on the Empowerment Pack
               compact disc.

            b. Double-click the Internet Explorer Installer icon.

            c. Click Continue after you read the license agreement.

            d. Click the Easy Install drop-down and select Custom Install.

            e. Click Thread Manager and then click Install.

            f. When the installer finishes installing the Thread
               Manager, click Restart.

            After your computer has restarted, double-click the Microsoft
            Empowerment Pack alias from the Empowerment Pack compact disc.
            Internet Explorer will start and the Empowerment Pack home page
            will appear.

            NOTE: If you want to install Internet Explorer on your computer
            along with the Thread Manager and the other files you need to
            run Internet Explorer, you can leave the drop-down at "Easy
            Install" and then click Install.

MORE INFORMATION

The following information is from the "Read Me-- Important!" text file that is on the Empowerment Pack compact disc.

System Requirements

The guided tour is presented using the Microsoft Internet Explorer. To run the Internet Explorer, your system must have: 

   - a 68030 processor or better
   - System 7.0.1 or above*
   - 8 MB of RAM
   - 2 - 4 MB of hard disk space
*Note: if you have a 68k Macintosh and are running a System version earlier than 7.5, you will need to install Thread Manager 2.0. You can do this by running the Internet Explorer installation program, located in the "Internet Explorer" folder of this CD.
