OFF95: Location of Template and Wizard Files

 Last reviewed: March 10, 1997
Article ID: Q135731
7.00 WINDOWS kbref kbother

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The following information describes the layout for the Microsoft Office templates and wizards. Note that these files are installed by default to the Microsoft Office Template folder.

MORE INFORMATION

The following is a list of the templates and wizards installed with Microsoft Office to the Template folder in the Microsoft Office folder (for example C:\Microsoft Office\Template).

NOTE: The templates in the list below appear in the New dialog box when you click the Start A New Document button (on the Office toolbar) on the Office Shortcut Bar. Additionally, the New dialog box contains an icon for Blank Document, Blank Workbook, and Blank Binder. These three templates, unlike the templates in the list below, are not actual files copied to the Templates folder by the Setup program. The information for these templates is stored in the Windows registry file. If the icons for these templates are missing from the New dialog box, you must run Setup again and click the Reinstall button, or you can run Setup /y to restore the registry information.

The following list contains the long filename and the short filename for each template and Wizard and is based on a Typical Installation. Note that the short filenames will appear when you install Microsoft Office on a network server or on a local workstation if Windows 95 is not configured to handle long filenames.
