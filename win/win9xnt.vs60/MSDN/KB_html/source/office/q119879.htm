MSWORD6MAC Entry Not Added to [MS Text Converters] in WIN.INI

 Last reviewed: February 2, 1998
Article ID: Q119879
4.00 4.20 4.30 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows, version 4.0 4.2 4.3

SYMPTOMS

When you run the Microsoft Office Setup program, and you select the Word for Windows 6.0 Converter option, the following entry is not added to the [MS Text Converters] section of your WIN.INI file: 

   MSWord6Mac=Word for Macintosh 6.0, C:\<path>\MSWORD6.CNV, doc

WORKAROUND

To correct this problem, you can edit your WIN.INI file using a text editor such as Notepad to add this entry to the [MS Text Converters] section. The entry should appear as follows: 

    MSWord6Mac=Word for Macintosh 6.0, C:\<path>\MSWORD6.CNV, doc
where <path> is the location of this file, such as WINDOWS\MSAPPS\TEXTCONV.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you run the Microsoft Word for Windows version 6.0 Setup program, and you select the Word for Windows 6.0 Converter check box, the following lines are added to the [MSWORD Text Converters] section of your WINWORD6.INI file as well as the [MS Text Converters] section of your WIN.INI file: 

   MSWord6=Microsoft Word 6.0, C:\<path>\MSWORD6.CNV, doc
   MSWord6Mac=Word for Macintosh 6.0, C:\<path>\MSWORD6.CNV, doc
The MSWORD6 converter allows you to save document files to the Microsoft Word for Windows version 6.0 file format.

When you select the Word for Windows 6.0 Converter in the Microsoft Office Setup program, the MSWORD6.CNV file is added to the \MSAPPS\TEXTCONV subdirectory of your Microsoft Windows directory as a shared converter. However, only the following entry is added to the [MS Text Converters] section of your WIN.INI file: 

   MSWord6=Microsoft Word 6.0, C:\<path>\MSWORD6.CNV, doc
The MSWord6Mac entry is associated with the Word for Macintosh 6.0 file type option in the Save As dialog box. Note that this converter is not meant to be used from within Word 6.0 for Windows itself.
