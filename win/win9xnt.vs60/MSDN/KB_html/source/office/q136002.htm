XL7: Binder Customize Menu Does Not Show Customize Dialog Box

 Last reviewed: March 27, 1997
Article ID: Q136002
7.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you use an Office Binder containing a worksheet section, the Customize menu located on the toolbar shortcut menu does not display the Customize dialog box and the Customize menu is unavailable (dimmed).

WORKAROUND

To customize the toolbars in the worksheet, use the following steps to view the worksheet outside the binder.

  1. Activate the desired worksheet section.

  2. On the Section menu, click View Outside.

  3. Use the right mouse button to click the toolbar area, and then click Customize.

  4. Make your desired toolbar changes and click OK.

  5. Choose Exit on the File menu to return to Office Binder.

MORE INFORMATION

When you use Microsoft Excel for Windows 95, version 7.0, outside of an Office Binder, you are able to customize your toolbar by clicking the toolbar area with the right mouse button and clicking Customize; however, you cannot customize toolbars from within a Binder. The only available menu for customizing is the one available the shortcut menu. Because the Customize dialog box has been disabled in Office Binder, the Customize menu will not display the dialog box.
