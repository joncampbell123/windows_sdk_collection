Office Does Not Remove Previous Version of Word

 Last reviewed: November 17, 1997
Article ID: Q124770
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2

SYMPTOMS

Microsoft Office 4.2 for the Macintosh setup does not remove the previously installed version of Microsoft Word, even if you select the option to remove it.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been corrected in the combined Macintosh/Power Macintosh release of Microsoft Office 4.2.

MORE INFORMATION

During the first portion of Microsoft Office Setup, a search is performed for any previous versions of the Office applications. If any Office applications are found, you have an option to have Setup remove them. The choice to remove Microsoft PowerPoint and Microsoft Excel are executed properly, but Microsoft Word remains on the system and is fully operational after the setup process is complete. To remove a previous version of Microsoft Word, manually delete it by dragging it to the trash.
