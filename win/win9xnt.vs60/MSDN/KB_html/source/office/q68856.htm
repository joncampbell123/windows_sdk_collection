How to Install the Windows Office

 Last reviewed: September 9, 1996
Article ID: Q68856
The information in this article applies to:
  • Microsoft Office for Windows, version 1.0

SUMMARY

This article outlines the procedures for installing Microsoft Office for Windows. You can find the instructions listed below in the "Read Me First!" insert that is included in The "Microsoft Windows Office" package documentation.

MORE INFORMATION

Installation Procedure

  1. You must have Microsoft Windows version 3.0 installed before you can install Microsoft Office.

  2. Find the Microsoft Office Setup disk labeled "READ ME FIRST."

  3. If you are at the DOS prompt: a. Insert the Microsoft Office Setup disk into drive A. b. Type WIN A:SETUP.

    -or-

    If you are in Windows 3.0: a. Insert the Microsoft Office for Windows Setup disk into drive A.

  4. From the Program Manager, choose Run from the File menu.

  5. Type A:SETUP.

  6. Click OK or press ENTER.

  7. Follow the instructions in the displayed dialog box.
