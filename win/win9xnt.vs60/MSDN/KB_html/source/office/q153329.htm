OFF: How to Download and Install the MOM Updater 4.2.1c

 Last reviewed: February 2, 1998
Article ID: Q153329
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

The Microsoft Office Manager Updater will update all previous versions of the Microsoft Office Manager (MOM), including all localized versions, to incorporate code changes made in version 4.2.1c of the Microsoft Office Manager (MOM). This article details where to find the MOM Updater on the World Wide Web, how to download it, and finally how to run it to update your version(s) of the Microsoft Office Manager.

MORE INFORMATION

Where to Find the Updater

The MOM Updater can be found on the Microsoft World Wide Web (WWW) site at the following address: 

   http://www.microsoft.com/macoffice/freestuff/momupdate.htm
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/

How to Download the Updater

The name of the MOM Updater on the Microsoft World Wide Web site is Momupdate.hqx. Once you download this file, you must run it through the BinHex program before you can update the Microsoft Office Manager.

To download the MOM Updater from the Microsoft World Wide Web site, do the following according to the platform on which your browser is running:

Microsoft Windows

  1. Go to the address (URL) listed in the "Where to Find the Updater" section and scroll to the bottom of the page.

  2. Click the link to "Download MomUpdate.hqx."

  3. In the next dialog that appears, click Save As.

  4. Leave the default file name and click Save.

  5. Copy the file that was downloaded to a floppy disk.

For the next step, you must have PC Exchange loaded and running on your Macintosh. If it is not currently running, enable it through Extensions Manager, and then restart your computer.

  1. Insert the disk from step 5 into your Macintosh.

  2. Start BinHex version 4.0 or 5.0.

  3. If you are using BinHex version 4.0, click "Upload -> Application" on the File menu.

    If you are using BinHex version 5.0, click "Download -> Application " on the File menu.

  4. In the dialog box that appears, open the floppy disk, click the file Momupd~1.hqx that you downloaded from the Internet, and then click Open.

  5. In the next dialog box that appears, leave the default name ("Office Manager Updater") in the "Save document as:" box, click Desktop, and then click Save.

The "Office Manager Updater" file will be expanded to your Desktop.

  1. Click Quit on the File menu to close BinHex.

Macintosh

  1. Go to the address (URL) listed in the "Where to Find the Updater" section and scroll to the bottom of the page.

  2. Click the link to "Download MomUpdate.hqx."

  3. In the next dialog box that appears, click Desktop, and then click Save.

The file Momupdate.hqx will be saved to your desktop.

  1. Start BinHex version 4.0 or 5.0.

  2. If you are using BinHex version 4.0, click "Upload -> Application" on the File menu.

    If you are using BinHex version 5.0, click "Download -> Application" on the File menu.

  3. In the next dialog box that appears, click Desktop, click the file Momupdate.hqx that was downloaded from the Internet, and then click Open.

  4. In the next dialog box that appears, leave the default name ("Office Manager Updater") in the "Save document as:" box, click Desktop, and then click Save.

The "Office Manager Updater" file will be expanded to your Desktop.

  1. Click Quit on the File menu to close BinHex.

Running the Updater

  1. Locate the file "Office Manager Updater" on your desktop and double- click the icon for the file.

This will start the MOM updater, and the updater will search all mounted drives for the Microsoft Office Manager control panel. Each one that it finds will be listed and selected in the Updater dialog box. If there is a copy of MOM that you do not want updated, click once on that item in the list.

NOTE: If the Updater finds a copy of MOM that has already been updated to version 4.2.1c, you will see "Up-to-date" before the item in this dialog box. There is no need to update these copies of MOM.

  1. Click Update.

Each item that you select in the list will be updated, and each item will now have "Up-to-date" before it.

NOTE: If you don't see a copy of MOM that you want updated, click Locate, and then search for it. Once you have found it, select it, and then click Update. It, too, will now appear in the list and will have "Up-to-date" before it.

  1. Click Quit to quit the updater.
