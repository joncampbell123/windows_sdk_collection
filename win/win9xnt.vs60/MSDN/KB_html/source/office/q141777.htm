"Building Applications with Microsoft Access for Windows 95"

 Last reviewed: September 16, 1996
Article ID: Q141777
The information in this article applies to:
  • Microsoft Office Professional for Windows 95, version 7.0
  • Microsoft Access for Windows 95, version 7.0

SUMMARY

Help topics in Microsoft Access Help refer to a Microsoft Press book called "Building Applications with Microsoft Access for Windows 95." However, the Supplemental Offers coupon that is included with Microsoft Office Professional for Windows 95 does not catalog this book.

MORE INFORMATION

Microsoft is correcting the omission of this information in future Supplemental Offers coupons that you receive in the Microsoft Office Professional for Windows 95 packages.

If you would like to obtain a copy of the book, call the Microsoft Sales and Information Center (MSIC) at (800) 426-9400 and ask for part number 62409. The cost of the book is (U.S.)$25.00 plus taxes, shipping and handling.
