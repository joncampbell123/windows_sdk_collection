OFF97: Office Resource Kit is not Available for Download

 Last reviewed: February 16, 1998
Article ID: Q163028
The information in this article applies to:
  • Microsoft Access 97 for Windows
  • Microsoft Office 97 for Windows, all editions
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Word 97 for Windows

SUMMARY

In the Microsoft Office 97 programs listed at the beginning of this article, the "How to obtain the Microsoft Office 97 Resource Kit" Help topic states that the Microsoft Office 97 Resource Kit is available for download. This is incorrect. The Microsoft Office 97 Resource Kit is not available for download.

MORE INFORMATION

The Microsoft Office 97 Resource Kit is available for viewing on the Microsoft web site. It has not been made available for download because Microsoft will be making regular updates to it with current information. You can view the Microsoft Office 97 Resource Kit at the following Web address: 

   http://www.microsoft.com/office/ork/
The Microsoft Office 97 Resource Kit is also available on TechNet. The Microsoft Office 97 Resource Kit on TechNet will be updated as needed. For more information on TechNet, connect to the following Web address: 

   http://www.microsoft.com/technet/
The Microsoft Office 97 Resource Kit is also available from Microsoft Press. To order the Microsoft Office 97 Resource Kit (ISBN: 1-57231-329-3), call (800) MS-PRESS in the United States or (800) 667-1115 in Canada, or order through the CompuServe Electronic Mall (GO MSP). For more information, visit the Microsoft Press site at this Web address: 

   http://www.microsoft.com/mspress/
