OFF97: Files Installed with Microsoft Camcorder 97

 Last reviewed: December 18, 1996
Article ID: Q161091
The information in this article applies to:
  • Microsoft Office 97 for Windows

SUMMARY

This article lists the location, name, and size for each file that is installed with Microsoft Camcorder.

MORE INFORMATION

You can use this list to determine which files are installed by Microsoft Camcorder and where they are installed, so that you can remove them if necessary.
