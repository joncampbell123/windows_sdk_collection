OFF97: Hyperlink ToolTips in MS Excel Worksheet Stop Working

 Last reviewed: August 22, 1997
Article ID: Q172950
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Office 97 for Windows

SYMPTOMS

In a Microsoft Excel 97 worksheet, when you move the pointer over a hyperlink or an object to which a hyperlink is attached, the ToolTip that is attached to the hyperlink does not appear. This behavior occurs even if you position the pointer over the hyperlink for several seconds.

CAUSE

This problem may occur if the following conditions are true:

  • You create a Microsoft Word document or a Microsoft PowerPoint presentation and you create a Microsoft Excel workbook.

    -and-

  • The document or presentation and the workbook each contain a hyperlink to the other file.

    For example, create a Book1.xls workbook and a Doc1.doc document where the files contain hyperlinks that are similar to the following. 

          Document    Document                   Contains a
      Name        Type                       Hyperlink to
      ---------------------------------------------------

      Book1.xls   Microsoft Excel workbook   Doc1.doc
      Doc1.doc    Microsoft Word document    Book1.xls

   NOTE: You can substitute any document that supports hyperlinks (for
   example, a Microsoft PowerPoint presentation) for the Microsoft Word
   document.

   -and-
  • You insert the workbook into a Binder in Microsoft Office 97 by clicking Add From File on the Section menu.

    -and-

  • In the workbook, you click the hyperlink, which makes the linked document active. Then, you click the hyperlink in the linked document, which makes the workbook active again.

The hyperlinks in the workbook fail and ToolTips that are attached to hyperlinks in the workbook fail to appear when you move the pointer over the hyperlinks. This problem occurs even if you switch back to Binder and move the pointer over the hyperlinks in the Microsoft Excel 97 section.

NOTE: This problem does not occur if you do not use Binder in Microsoft Office 97. If you open the workbook in Microsoft Excel 97, and then use the hyperlinks to move back and forth between the workbook and the document, the problem does not occur.

WORKAROUND

To work around this problem, quit and restart Binder. After you restart Binder, the ToolTips appear correctly.

NOTE: It is not possible to force the ToolTips that are attached to hyperlinks in the Microsoft Excel workbook to appear without quitting and restarting Binder.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
