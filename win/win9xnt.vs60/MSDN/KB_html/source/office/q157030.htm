OFF97: Change Directory Unavailable for Office Tools

 Last reviewed: March 13, 1998
Article ID: Q157030
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you run the Microsoft Office setup program and you choose the Custom setup option, the "Change Folder" button is unavailable when you select the "Office Tools" component of Microsoft Office.

CAUSE

By default, you are unable to change the destination directory for the entire set of the Microsoft Office "Office Tools" component and the "Change Folder" button is unavailable in the "Microsoft Office 97 -Custom" dialog box.

RESOLUTION

It is possible to change the destination directory for some of the different programs that are listed under the "Office Tools" component. To do this, use the following steps:

  1. Perform a Custom installation of Microsoft Office 97.

  2. In the "Microsoft Office 97 -Custom" dialog box click "Office Tools".

Note that the "Change Folder" button is unavailable.

  1. Click "Change Option"

The "Microsoft Office 97 - Office Tools" dialog box appears.

  1. Click the "Office Tool" component for which you want to change the destination folder.

The "Change Folder" button is now available.

  1. Click "Change Folder" and in the "Change Folder" dialog box locate the desired destination folder, then click OK.

After you click OK, note the setting for "Folder for Currently Selected Option". It points to the same folder you specified in Step 5.

Repeat Steps 4 and 5 with each of the "Office Tools" components for which you want to change the destination folder.

STATUS

This is by design of Microsoft Office 97.

MORE INFORMATION

The "Office Tools" component of Microsoft Office 97 contains the shared programs for the main Microsoft Office programs. The following table lists the components and whether you are able to change the destination folder: 

                                       Able to change
   Office Tools Component              Destination Folder
   ------------------------------------------------------

   Microsoft Office Shortcut Bar             no
   Office Assistant                          no
   Spelling Checker                          yes
   Organizational Chart                      yes
   Microsoft Graph                           no
   Microsoft Graph Help                      no
   Equation Editor                           yes
   Microsoft Photo Editor                    yes
   MS Info                                   yes
   Popular Clipart                           no
   Clip Gallery                              yes
   Find Fast                                 no
   Microsoft True Type fonts                 no
   Lotus VIM Mail Support                    no
   Find All Word Forms                       yes
