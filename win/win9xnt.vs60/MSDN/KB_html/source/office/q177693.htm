OFF98: Errors Starting Program from Microsoft Office Manager

 Last reviewed: February 2, 1998
Article ID: Q177693
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

When you attempt to start an Office 98 Macintosh Edition program by clicking the program menu command on the Microsoft Office Manager menu, you may receive one of the following error messages: 

   Could not save changes, the Preferences file has been moved or
   deleted. Restart the computer to correct the problem.

   -or-

   Could not continue because a resource is missing.
And the program that is associated with the menu command you clicked is not started.

CAUSE

These problems occur if the Office Manager Preferences file is not present in the System Folder:Preferences folder or if the file contains incomplete information.

WORKAROUND

To work around these problems, restart your computer. After you restart the computer, the menu commands on the Microsoft Office Manager menu work correctly.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Microsoft Office Manager is included in the Microsoft Office 98 Macintosh Edition Value Pack. When you install the Microsoft Office Manager control panel and restart your computer, the Microsoft Office Manager menu appears to the left of the Application menu, which is in the upper-right corner of the screen.

You can use the Microsoft Office Manager menu to start the programs included with Microsoft Office 98 Macintosh Edition and to switch between running programs. You can also customize the menu by clicking Customize on the Microsoft Office Manager menu.
