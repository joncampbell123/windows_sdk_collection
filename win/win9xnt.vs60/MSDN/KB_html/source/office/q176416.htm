OFF97: Files Are Not Extracted by CABFILE Viewer

 Last reviewed: November 7, 1997
Article ID: Q176416
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you double-click a cabinet (.cab) file, and then attempt to extract a file from the cabinet file to a location on the hard disk on your computer, the file may not be extracted. When this problem occurs, you do not receive an error message.

CAUSE

This problem may occur if you install the CABFILE Viewer utility (CabView program) that is included on Disk 1 of Microsoft Office 97 that is on 3.5- inch DMF floppy disks.

RESOLUTION

To resolve this problem, obtain the CabView program that is included on Disk 1 of Microsoft Office 97 for Windows, Service Release 1 (SR-1) that is on 3.5-inch DMF floppy disks.

For information about obtaining Microsoft Office 97 SR-1 on floppy disks, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q172475
   TITLE     : OFF97: How to Obtain and Install MS Office 97 SR-1
After you obtain Microsoft Office 97 SR-1, install the updated version of the CabView program. To do this, follow these steps:

  1. Insert Disk 1 into the floppy disk drive on the computer.

  2. On the Start menu, click Run. In the Open box, type "a:\" (without the quotation marks) and click OK.

  3. Right-click the Cabview or Cabview.inf file. On the shortcut menu, click Install.

After the necessary files are copied from the floppy disk to the hard disk on your computer, you can both view and extract files from cabinet files.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in the CabView program included with Microsoft Office 97 SR-1 that is on 3.5-inch DMF floppy disks.

MORE INFORMATION

The CabView file allows you to view the contents of cabinet files and to extract files from cabinet files without having to use the Extract.exe program.

After you install CabView, you can view the contents of a cabinet file by double-clicking the file. A new window, which is very similar to a window in Microsoft Windows Explorer, appears and lists the contents of the cabinet.

You can extract a file from a cabinet by doing either of the following:

  • Right-click the file you want to extract and click Extract on the shortcut menu. In the Browse For Folder dialog box, click the folder in which you want to extract the file. Then, click OK.

    -or-

  • Drag the file from the cabinet window to the desktop or to a specific folder window.

These steps may fail to extract the file if an older version of CabView is installed on the computer. To determine which version of CabView is installed, follow these steps:

  1. On the Start menu, point to Find, and then click Files Or Folders.

  2. In the Named box, type "Cabview.dll" (without the quotation marks). Then, click Find Now.

  3. Right-click Cabview.dll. On the shortcut menu, click Properties. Click the Version tab.

The versions and file sizes for the Cabview.dll file are listed in the following table. 

   Version      File size      Included with
   -----------------------------------------------------------------

   0.1.0.0      61,440 bytes   Microsoft Office 97 for Windows

   1.00.601.1   44,032 bytes   Microsoft Office 97 for Windows, SR-1
