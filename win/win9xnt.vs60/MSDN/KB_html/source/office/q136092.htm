Office: No Network Button in Open and Save As Dialog Boxes

 Last reviewed: December 16, 1996
Article ID: Q136092
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Access 97 for Windows
  • Microsoft Access, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0

SUMMARY

The Open and Save As dialog boxes for Microsoft Office 7.0 programs and Microsoft Office 97 programs do not include a Network button.

MORE INFORMATION

Earlier versions of Microsoft Office programs include a Network button in the Open and Save As dialog boxes. This Network button allows you to map network drives and assign a specific drive letter (for example, F:) to a specific network path (for example, \\Server\Share).

Although the Network button is not present in programs included with Microsoft Office 7.0 and 97, you can map network drives from the Open and Save As dialog boxes. To do this mapping, use the following steps:

  1. On the File menu, click Open (or click Save As).

  2. Click the Commands And Settings button, and then click Map Network Drive.

In the Map Network Drive dialog box, type a specific drive letter and a network path (for example F: and \\Server\Share).
