Ordering Additional Maps to Use with Data Map

 Last reviewed: March 12, 1998
Article ID: Q135793

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows

SUMMARY

Microsoft Data Map version 1.0 is shipped with Microsoft Excel 7.0 and Microsoft Office 7.0. Data Map is an OLE server that allows you to perform geographic data analysis. A set of maps is provided with Data Map, and you can purchase additional maps and demographic data from MapInfo Corporation directly. The following information describes how to purchase additional maps and data products to use with Data Map.

NOTE: Microsoft Excel 97 and Office 97 ship with Microsoft Map 8.0. You can also purchase additional maps and demographic data for Microsoft Map from MapInfo Corporation.

MORE INFORMATION

An example of an additional data product that you might be interested in purchasing is a zip code boundary map, which would let a local business owner analyze their customer base. The Map Catalog Help file (Mapcat.hlp), installed with Data Map, describes other available products.

You can purchase and download additional map and demographic data products from the MapInfo Forum on the Microsoft Network. You must be running Windows 95 and have the Microsoft Network installed. You can click the MSN button in the Data Map Help file to go directly to the MapInfo Forum.

Alternately, you can call or fax MapInfo using one of the numbers below. To fax, simply click the Fax button in Data Map Help, print and fill out the form provided. 

   In the U.S.     : (800) 488-3552
   Outside the U.S.: (518) 285-7110
   Fax             : (518) 285-7440 (U.S. Only)

REFERENCES

Microsoft Data Map 8.0

For more information about ordering additional maps for Microsoft Map, click "How to Get More Data" on the Help menu in Microsoft Map.

Microsoft Data Map 1.0

For more information about ordering additional maps to use with Data Map, click the Index tab in Data Map Help, type the following text 

   ordering additional
and then double-click the selected text to go to the "Purchasing Additional Maps" topic.
