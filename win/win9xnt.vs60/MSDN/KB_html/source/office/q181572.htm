OFF98: Mac OS System Updates That Are Included in the Value Pack

 Last reviewed: February 27, 1998
Article ID: Q181572
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SUMMARY

The Microsoft Office 98 Macintosh Edition Value Pack includes updates to the Macintosh operating system (OS). This article explains the purpose of the operating system updates and how they are supported.

MORE INFORMATION

How to Install Mac OS System Updates

The Value Pack:Mac OS System Updates folder on the Microsoft Office 98 Macintosh Edition compact disc includes folders that are designed to be used with specific operating system versions. These folders and the system versions for which they are designed are listed in the following tables.

US System Updates

Updates for the US version of the Macintosh OS are located at the top level of the Value Pack:Mac OS System Updates folder. Use the following table to determine which update is appropriate for the version of the Macintosh OS that you are running. 

   Install the contents      Only if the following Macintosh
   of this folder            operating system version is installed
   ----------------------------------------------------------------------

   System 7.5 Update 2.0     US System 7.5, US System 7.5.1,
                             US System 7.5.2

   System 7.5.5 Update       US System 7.5.3, US System 7.5.3 Revision 2

   Mac OS 7.6.1 Update       US Mac OS 7.6

   Mac OS 8.1 Update         US Mac OS 8.0
To install a particular update, open the folder that contains it and double- click the Installer file (for version 7.x) or the "Update to Mac OS 8.1" file (for version 8.0).

British System Updates

Updates for the British version of the Macintosh OS are located in the Value Pack:Mac OS System Updates:British Updates folder. Use the following table to determine which update is appropriate for the version of the Macintosh OS that you are running. 

   Install the contents      Only if the following Macintosh
   of this folder            operating system version is installed
   ----------------------------------------------------------------------

   B-System 7.5 Update 2.0   British System 7.5, British System 7.5.1,
                             British System 7.5.2

   ZM-System 7.5.5 Update    Any International version of System 7.5.3

   B-Mac OS 7.6.1 Update     British System 7.6
NOTE: An update to the British version of Macintosh OS version 8.0 is not available on the Office 98 CD. Do not use the Mac OS 8.1 update if you are running a British version of the Macintosh OS.

NOTE: The ZM-System 7.5.5 Update can be uses on the following versions of the Macintosh OS: Spanish, Danish, Swedish, German (including Swiss- German), Dutch, English (British, US and International), Norwegian, Catalan, Polish, Italian, Croatian, Finnish, French (including Swiss-French and French-Canadian), Ukrainian, Hungarian, Turkish, Bulgarian, Korean, Portuguese, Romanian, and Chinese (Simplified and Traditional).

Updates to International English and International versions of the Macintosh OS are located in the Value Pack:Mac OS System Updates:International Updates folder. Use the following table to determine which update is appropriate for the version of the Macintosh OS that you are running. 

   Install the contents      Only if the following Macintosh

NOTE: An update to the international versions of Macintosh OS version 7.6 is not available on the Office 98 CD. Do not use the Do not use the Mac OS 7.6.1 Update or the B-Mac OS 7.6.1 Update unless you are running the US or British version of the Macintosh OS.

NOTE: The ZM-System 7.5.5 Update can be used on the following versions of the Macintosh OS: Spanish, Danish, Swedish, German (including Swiss- German), Dutch, English (British, US and International), Norwegian, Catalan, Polish, Italian, Croatian, Finnish, French (including Swiss-French and French-Canadian), Ukrainian, Hungarian, Turkish, Bulgarian, Korean, Portuguese, Romanian, and Chinese (Simplified and Traditional).

Why Are These Updates Included with Microsoft Office 98?

For your convenience, Microsoft has included these system updates with Microsoft Office 98. The advantages of installing the system updates are listed as follows:

  • The system updates include improvements in system stability and performance.
  • The programs included in Microsoft Office 98 generally run more quickly and efficiently when you are using the most current operating system.

Note that the updates included with Microsoft Office 98 are NOT full versions of the operating systems. For example, you cannot install the Mac OS 8.1 Update unless Mac OS 8.0 is already installed on your computer. If you want to upgrade your operating system from System 7.x to Mac OS 8.0, you must purchase the full version of Mac OS 8.0.

Who Supports the Installation and Operation of These System Updates?

Support of the system updates is provided by Apple Computer. For more information, please see the support page on the Apple Web site at the following address: 

   http://www.info.apple.com/supportoptions
Or, call Apple technical support at (800) SOS-APPL (767-2775). Microsoft does not provide any support for the installation of the system updates that are included with Microsoft Office 98.
