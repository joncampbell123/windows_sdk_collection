ALT+CLICK Does Not Close All MOM Applications

 Last reviewed: September 16, 1996
Article ID: Q120121
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.2, 4.3

SYMPTOMS

In Microsoft Office, you cannot close the Run or Find File applications by selecting their buttons on the MOM toolbar, pressing the ALT button, and clicking the mouse button.

When you follow this procedure with other applications, the application will be closed.

CAUSE

The ALT+click key combination closes only applications that have a distinct executable file. The Run and Find File are tools that are dependent on other applications and do not have distinct executable files. Therefore, these applications cannot be closed with the ALT+click key combination.
