Office: Some Installation Problems Are Caused by SHARE.EXE

 Last reviewed: September 9, 1996
Article ID: Q108003
The information in this article applies to:
  • Microsoft Office for Windows, version 4.0

SUMMARY

If you encounter problems or receive error messages either during or immediately after you install Microsoft Office version 4.0, the problem may be related to the SHARE.EXE program.

When you install Microsoft Office 4.0, the Microsoft Office Setup program attempts to modify the line that loads SHARE.EXE in the AUTOEXEC.BAT file to 

   <path>\SHARE.EXE /L:500 /F:5100
where <path> is the drive and directory where the SHARE.EXE program is located. If this line is not already present, the Setup program will attempt to reload SHARE.EXE.

MORE INFORMATION

For more information about problems with SHARE.EXE , query on the following words in the Microsoft Knowledge Base: 

   WORD6 6.0 SHARE 5100
