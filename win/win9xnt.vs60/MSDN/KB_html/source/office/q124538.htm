Mac Office 4.2 Err: Could Not Continue ... Resource Is Missing

 Last reviewed: November 17, 1997
Article ID: Q124538
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1

SUMMARY

When you try to start an application from Microsoft Office Manager (MOM) for the Macintosh, you may receive the following error message: 

   Could not continue because a resource is missing
Note that you should still be able to start the application from the application icon in the application's folder.

CAUSE

This error message is generated because of a corrupted Office Manager Preferences file, which is located in the System Folder:Preferences folder.

WORKAROUNDS

To get the Office Manager menu icon to work properly, use the first method listed below. If this method does not work, try the second method.

Method 1: To reset the application icon in MOM: 

          a. From the MOM menu, choose Customize.

          b. Select the name of the application that will not start, and
             choose Edit.

          c. In the next dialog box (with the Menu Item Name box), choose
             Change.

          d. In the next dialog box, locate the appropriate application,
             select it, and choose Add.

          e. In the dialog box that contains Menu Item Name, choose OK.

          f. In the Customize dialog box, choose Close.

          Now, when you select the desired application in the retrained
          Microsoft Office Manager, the application should start as
          expected.
Method 2: To create a new preferences file that will reset MOM: 

          a. Move the Office Manager Preferences file from the System
             Folder:Preferences folder to the desktop.

          b. Restart your computer.

          A new preferences file will be created and you should be able to
          start the application by choosing the appropriate MOM icon. If
          the new preferences file works, delete the old file from the
          desktop.
