OFFICE: Description of Office for Windows NT 4.2

 Last reviewed: September 16, 1996
Article ID: Q126684
The information in this article applies to:
  • Microsoft Office for Windows NT, version 4.2

SUMMARY

The following information describes Microsoft Office version 4.2 for Windows NT, the applications that the package contains, and a brief description of the installation process.

MORE INFORMATION

The Microsoft Office version 4.2 for Windows NT package contains the following applications:

  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft PowerPoint version 4.0c (16 bit)
  • Microsoft Office Manager (MOM) version 4.3 (16 bit)

There is no integrated Setup program for the Microsoft Office for Windows NT applications. However, you can use the instructions included in the Microsoft Office Readme Help (for an administrative installation) or in the Office package itself (for a local installation). These instructions describe how to install the applications and which order you should install them in.

There is only one Microsoft Office version 4.2 for Windows NT package. This package contains the following:

  • The standard packages, including disks, of Microsoft Word for Windows NT, version 6.0, Microsoft Excel for Windows NT, version 5.0, and Microsoft PowerPoint version 4.0c (16 bit).
  • The standalone MOM disk (version 4.3, 16 bit)
  • All of the applications on one CD

Note that the CD does not contain online documentation, but manuals are included in the package.

The requirements for Microsoft Office for Windows NT, version 4.2: 

   Microsoft Windows NT, version 3.5 or later
   486 or higher, Alpha AXP microprocessor
   16 MB RAM for an Intel machine, 20 MB RAM recommended for RISC machines
   Hard Drive Requirements:
      Intel machines:
         typical installation 59 MB
         laptop installtion 26 MB
         complete 84 MB
      RISC machines:
         typical installation 71 MB
         laptop installation 39 MB
         complete installation 95 MB
   One 3.5-inch high density disk drive or a CD-ROM drive
   VGA or higher resolution monitor compatible with Windows NT 3.5
   Microsoft Mouse or compatible pointing device
