Office Small Business Pack Setup Error "Could Not Find ..."

 Last reviewed: September 16, 1996
Article ID: Q139886
The information in this article applies to:
  • Microsoft Office Professional for Windows 95, version 7.0
  • Microsoft Small Business Pack for Windows 95, version 1.0

SYMPTOMS

When you run the Setup program for the Microsoft Office Small Business Pack (OSBP), you may receive either of the following error messages: 

   Setup could not find Microsoft Office installed on your
   system. Please install Microsoft Office before installing
   this application.

   -or-

   Setup could not find Microsoft Access installed on your system.
   Please install Microsoft Access before installing this application.
If you receive either of the above two error messages, your only option is to click the OK button, and you are not able to continue with the setup of OSBP.

CAUSE

You may receive either of these error messages if you do not have Microsoft Access version 7.0 or the Microsoft Office Shortcut Bar installed on your computer. You will receive only the first error message if you either do not have the Office Shortcut Bar installed or you do not have both the Office Shortcut Bar and Access installed. You will only receive the second error message if you do not have Microsoft Access installed.

WORKAROUND

Microsoft has created an application note, "Installing Small Business Pack Without Microsoft Access 7.0" (WC1270), which allows you to install the OSBP without having to install Microsoft Access 7.0. For information on how to obtain this application note, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140122
   TITLE:      "WC1270: File to Allow SBP Setup Without MS Access 7.0"

MORE INFORMATION

Because of the design and function of the OSBP templates and OSBP toolbar, you normally need to have both Microsoft Access version 7.0 and the Office Shortcut Bar on your computer in order to successfully install and use OSBP.
