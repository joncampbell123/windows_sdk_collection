OFF97: Artgalry.cag Points to Deleted PCS Files

 Last reviewed: July 11, 1997
Article ID: Q171090
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft Access 97
  • Microsoft Outlook 97

SYMPTOMS

When you insert a Clip Gallery object in any of the programs listed at the beginning of this article, you may receive the following error message 

   Cannot Open Preview File

   Clip gallery cannot open the preview file
   <drive>:\<MSOffice folder>\clipart\pcsfiles\popular.cag

   That file may have been deleted or moved. If you moved that file,
   choose Import Clips, then select the file at its new location.
   This will re-import the clips in that previews file.
where <drive> is the hard disk on which you previously installed Microsoft Office 7.0 for Windows 95 (usually drive C), and <MSOffice folder> is the folder in which you previously installed Microsoft Office 7.0.

CAUSE

This problem occurs if the following conditions are true:

  • You upgraded from Microsoft Office 7.0 to Microsoft Office 97.

    -and-

  • You deleted the folder for ClipArt from in the folder in which you installed Office 7.0.

RESOLUTION

To resolve this problem, follow these steps:

  1. On the Start menu, point to Find and click Files or Folders.

  2. In the Named box, type "Artgalry.cag" (without the quotation marks). In the Look In box, type "c:\" (without the quotation marks). Click to select the Include Subfolders check box and click Find Now.

    Note the complete path that the search results window displays.

  3. Using the right mouse button, click the Artgalry.cag file in the search results window. Click delete.

  4. Copy the following file from the OS folder on the Microsoft Office 97 compact disc: 

          Artgalry.cag

    On the Edit menu, click Paste to paste the file in the folder listed in the search results window in step 2.

  5. Switch to the Find program window. Click Find Now.

  6. Right-click the Artgalry.cag file in the search results window. Click Properties.

  7. Clear the Read-only check box under Attributes in the Properties dialog box. Click Apply, and then click OK.

  8. Close the Find program window.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
