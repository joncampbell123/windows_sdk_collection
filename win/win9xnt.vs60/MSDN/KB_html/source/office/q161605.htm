OFF97: OSB May Not Be Installed Upgrading with Typical Setup

 Last reviewed: January 2, 1997
Article ID: Q161605
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you upgrade from an earlier version of Microsoft Office to Office 97 by running the Office Setup program and selecting the Typical option, the Office Shortcut Bar may be installed by default. However, if you are not upgrading from a previous version of Microsoft Office, the Office Shortcut Bar is not installed when you perform a Typical installation.

CAUSE

This behavior occurs when both of the following conditions are true:

  • You upgrade from an earlier version of Microsoft Office

    -and-

  • You previously installed the Microsoft Office Manager (for Office 4.x) or the Office Shortcut Bar (for Office 7.0).

NOTE: If you do not upgrade from an earlier version of Microsoft Office, the Office Shortcut Bar is not installed by default when you select the Typical option.

RESOLUTION

If the Office Shortcut Bar is not installed when you perform a Typical installation of Microsoft Office, use the following steps:

  1. On the Start menu, point to Settings, and then click Control Panel.

  2. In the Control Panel window, double-click Add/Remove Programs.

  3. Click the Install/Uninstall tab, select Microsoft Office 97, and then click Add/Remove.

    This starts the Microsoft Office Setup program in maintenance mode.

  4. In the first Office Setup dialog box that appears, click Add/Remove.

  5. In the Microsoft Office 97 - Maintenance dialog box, select the option for Office Tools, and then click Change Option.

  6. Select the Microsoft Office Shortcut Bar check box, and then click OK.

  7. Click Continue.

  8. When the Setup program is finished, click OK.

STATUS

This behavior is by design of the Microsoft Office Setup program.

MORE INFORMATION

The following table lists the upgrade scenarios when the Office Shortcut Bar is installed when you perform a Typical installation. 

                                     Installed
Existing Office version              with Typical installation


Microsoft Office 4.2, 4.3                yes
(Microsoft Office Manager installed) 

Microsoft Office 7.0                     yes
(Office Shortcut Bar installed) 

None (new install of Office 97)          no
