OFF95: Network Install Wizard Executable File Sizes

 Last reviewed: March 27, 1997
Article ID: Q152583
7.00 7.00a 7.00b WINDOWS kbref kbsetup kbnetwork

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0, 7.0a, 7.0b

SUMMARY

The only way to determine the version of the Network Installation Wizard (NIW) you are using is to check the size of the Niw.exe file.

MORE INFORMATION

The following table lists the current versions of the Network Installation Wizard, the size and date of its associated executable (.exe) file: 

   Version    Size                     Date
   -------    ----------------------   -------
   1.0        306 KB (313,979 Bytes)   11/3/95
   1.1        311 KB (319,099 Bytes)   1/5/96
   1.1a       324 KB (332,155 Bytes)   4/10/96
   1.1b       329 KB (335,995 Bytes)   5/19/96
The Network Installation Wizard is an application component of the Microsoft Office Resource Kit (ORK). By default, the Network Installation Wizard is installed in a folder within the "Program Files" folder.
