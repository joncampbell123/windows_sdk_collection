Office: Err Msg: "Cannot Run MSInfo"

 Last reviewed: December 13, 1996
Article ID: Q121626
This information in this article applies to:
  • Microsoft Office for Windows, versions 4.2, 4.3
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SUMMARY

If you run a custom installation of Microsoft Office and choose not to install the Tools option, the MSInfo program is not installed. If you try to access System Info from any of the installed Microsoft Office programs, you may receive one of the following error messages, depending on the program from which you attempt to access System Info:

If you access 

System Info from       You will receive the following error message
----------------       --------------------------------------------------
Microsoft Excel 97     MSINFO32.EXE cannot be found.
Microsoft Excel 7.0    MSINFO32.EXE cannot be found.
Microsoft Excel 5.x    Cannot run 'C:\WINDOWS\MSAPPPS\MSINFO\MSINFO.EXE'.
                       The program or one of its components is damaged or
                       missing.
Word                   Word cannot find or run the application MSInfo.
PowerPoint             PowerPoint cannot find or run the application
                       MSInfo.
Microsoft Access       Can't display system information.

WORKAROUND

To install the MSInfo program, do the following:

  1. Run the Microsoft Office Setup program in maintenance mode.

  2. Click the Add/Remove button.

  3. Under Options, select the Tools check box.

  4. Click the Change Option button.

  5. Select the MSInfo check box.

  6. Click OK, and then click the Continue button.

After the installation completes, MSInfo should be available in all the installed Microsoft Office programs.

MORE INFORMATION

Selecting the Tools option in Microsoft Office Setup, will install the following Office files: 

   MSInfo
   Spelling Checker
   True Type Fonts
   Setup
