Office: Files Not Removed When Installation Is Canceled

 Last reviewed: March 27, 1997
Article ID: Q123847
4.20 4.30 7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows versions 4.2, 4.3
  • Microsoft Office for Windows 95, version 7.0

SUMMARY

When you install the versions Microsoft Office for Windows mentioned at the beginning of this article, if the Setup program is stopped before the installation is complete, any files that were written during installation will remain on the hard disk drive.

MORE INFORMATION

To recover the hard disk drive space being used by these files, delete the files.

For a complete list of the files that are installed by Microsoft Office for Windows, see the following files on the Setup disks:

Microsoft Office version 4.2

Microsoft Excel              Open FILELIST.TXT located on disk 2 (3.5-
                             inch disk) or disk 3 (5.25-inch disk).

Microsoft PowerPoint         Open PPREADME.HLP located on disk 2.
                             Choose "Custom and Network Installations,"
                             and choose "Files Installed by (type of
                             installation)."

Microsoft Word               Open WDREADME.HLP located on disk 2.
                             Choose "Custom and Network Installations,"
                             and choose "Files Installed by (type of
                             installation)."

Microsoft Office version 4.3

Microsoft Access             Open ACREADME.HLP located on Disk 2.
                             Choose "Files Included with Microsoft
                             Access," and choose "Microsoft Access version
                             2.0 files."

Microsoft Excel              Open FILELIST.TXT located on Disk 6.

Microsoft PowerPoint         Open PPREADME.HLP located on Disk 2.
                             Choose "Custom and Network Installations,"
                             and choose "Files Installed by (type of
                             installation)."

Microsoft Word               Open WDREADME.HLP located on Disk 2.
                             Choose "Custom and Network Installations,"
                             and choose "Files Installed by (type of
                             installation)."
