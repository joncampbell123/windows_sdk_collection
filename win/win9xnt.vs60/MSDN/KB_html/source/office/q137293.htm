OFF95: Limited Answers from Answer Wizard on Shortcut Bar

 Last reviewed: March 10, 1997
Article ID: Q137293
7.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Access, version 7.0

SYMPTOMS

When you click the Answer Wizard on the Office Shortcut Bar, you expect to find information on frequently asked questions about Office products. However, the Office Answer Wizard includes information about the Office Shortcut Bar and cross-application questions only.

RESOLUTION

In order to find application-specific information, click the Answer Wizard in the actual product--for example, use Microsoft Excel Answer Wizard for Microsoft Excel, Word Answer Wizard for Word, and so on.

STATUS

This behavior is by design. The Answer Wizard that you access from the Office Shortcut Bar Help menu is specific to information about the Office Shortcut Bar.
