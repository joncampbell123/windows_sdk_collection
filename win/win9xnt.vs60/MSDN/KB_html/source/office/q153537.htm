OFF95: Microsoft Solutions Development Kit (MSDK) 2.0 Available

 Last reviewed: April 2, 1997
Article ID: Q153537
7.00 7.00a WINDOWS kbusage kbref

The information in this article applies to:

  • Microsoft Office for Windows 95, versions 7.0, 7.0a

SUMMARY

The Microsoft Solutions Development Kit (MSDK) 2.0 provides information on how to use Microsoft visual development tools to integrate Microsoft Office, Microsoft BackOffice, and Object Linking and Embedding (OLE) components to build custom solutions.

HOW TO OBTAIN THE RESOURCE KIT

To obtain the Microsoft Solutions Development Kit, call the Microsoft Order Desk at (800) 360-7561. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm

MORE INFORMATION

The MSDK 2.0 provides, on a single compact disc, the new Microsoft Office for Windows 95 and Microsoft BackOffice development kits, along with information, documentation, integrated sample code, utilities, and other useful reference material that documents how to build 32-bit integrated solutions with Office and BackOffice.

MSDK 2.0 helps show you how technologies, such as OLE, Visual Basic for Applications, Messaging API, and Open Database Connectivity (ODBC) provide the foundation for a consistent method of building component-based solutions that reach from the desktop to the server.

Microsoft Developer Network Level 2 and Level 3 members will receive the MSDK 2.0 free as a premium release. In addition, MSDK 2.0 will available on the April 1996 edition of the Microsoft Development Platform.

Starting in mid-February 1996, developers who are not Level 2 or Level 3 members of the Developer Network can purchase MSDK 2.0 directly from Microsoft for (U.S.) $99.95.

Developers who have already purchased the following products 

   Visual Basic 4.0, Professional or Enterprise Editions
   Visual FoxPro version 3.0
   Visual C++ version 4.0
   Microsoft Access Developer's Toolkit for Windows 95
may purchase the MSDK 2.0 for (U.S.) $49.95.
