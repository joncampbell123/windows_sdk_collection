OFF95: File Custom Number Property Appears Incorrectly

 Last reviewed: March 10, 1997
Article ID: Q149833
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Access versions 7.0

SYMPTOMS

In any of the Office products listed at the beginning of this article, when you assign a digit to a custom number property of a file, and you then view the file's custom properties in the Open or the Save As dialog box, the wrong value appears. However, the correct custom number value appears when you open the file and click Properties on the File menu, or when you right-click the file in the Open dialog box and click Properties.

NOTE: The value that the file actually stores in the property is correct although the Open and Save As dialog boxes display the wrong value.

CAUSE

This problem occurs when the following conditions are met:

  • You click Properties on the File menu in an existing Microsoft Excel, Word, PowerPoint, or Microsoft Access file.

    -and-

  • You click the Custom tab, type a name in the Name box, and click Number in the Type box.

    -and-

  • You type a number greater than 255 in the Value box, click Add, and then click OK.

    -and-

  • On the File menu, you click Open or Save As, select the file, and click the Properties button.

MORE INFORMATION

When you perform an advanced content search to find files with a custom number property, files that contain a custom number property greater than 255 may fail to appear in the search results list. The content search engine finds files based on the custom number value that appears in the Open dialog box when you click the Properties button, rather than the actual value that appears when you open the file, click Properties on the File menu, and click the Custom tab.

For additional information about viewing custom property values in the Open dialog box, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q135661
   TITLE     : Decimal Numbers Truncated in Properties in Open Dialog Box

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about document properties, click the Index tab in Microsoft Word Help, type the following text 

   properties
and then double-click the selected text to go to the "Properties (document properties)" topic.
