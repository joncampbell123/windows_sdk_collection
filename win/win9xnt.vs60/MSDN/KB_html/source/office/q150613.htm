OFF97: Microsoft Office 97 Service Release 1 Patch

 Last reviewed: March 13, 1998
Article ID: Q150613
The information in this article applies to:
  • Microsoft Office 97 for Windows, Service Release 1 (SR-1)

SUMMARY

Microsoft Office 97 SR-1 includes a number of updates for each program that is included in Microsoft Office 97. This article contains a list of the key updates that are included for each program in the SR-1 Patch.

For additional information about SR-1, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q172475
   TITLE     : OFF97: How to Obtain and Install the MS Office 97 SR-1 Patch

MORE INFORMATION

Microsoft Office 97 SR-1

Microsoft Office 97 SR-1 includes the following updates for Office 97:

  • If a corrupted AutoCorrect (.acl) file exists on your computer, you may encounter problems when you start or use the programs included in Microsoft Office 97. This problem was corrected in Microsoft Office 97 SR-1.

    For more information, please see the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q160552

          TITLE     : OFF97: Errors Caused by Corrupted ACL Files


  • Microsoft Office 97 SR-1 includes updated Open Database Connectivity
   (ODBC) 3.0 files. These updated ODBC files include improved security
   features.

    

Microsoft Word 97 SR-1

 


Microsoft Office 97 SR-1 includes the following updates for Word 97:


  • Microsoft Word 97 SR-1 allows you to use permanently mapped network
   drives.


  • When you save a document in the Microsoft Word 6.0/95 format in
   Microsoft Word 97 SR-1, the file is saved in the native Microsoft Word
   6.0/95 format instead of the Rich Text Format (RTF) file format that is
   used by Microsoft Word 97.

    
   For more information about the RTF format, please see the following
   article in the Microsoft Knowledge Base:

    

          ARTICLE-ID: Q162602

          TITLE     : WD97: Questions About Saving in the Word 6.0/95 File
                  Format


  • When you open a Microsoft Word for the Macintosh file in Microsoft Word
   97 SR-1, encapsulated PostScript (EPS) graphics are preserved when you
   print them to a PostScript printer.


  • In Microsoft Word 97 SR-1, inserted graphics default to inline graphics
   instead of floating graphics when you save documents in HTML and
   Microsoft Word 6.0/95 file formats.


  • Microsoft Word 97 SR-1 addresses specific known causes in which graphics
   that are inserted into a new document appear in a blank box with a red
   "X."

    
   For more information, please see the following article in the Microsoft
   Knowledge Base:

    

          ARTICLE-ID: Q162349

          TITLE     : WD97: Picture Displayed as Red "X" in Document


  • In Microsoft Word 97 SR-1, linked, inserted pictures retain sizing
   information if they are resized.


  • In Microsoft Word 97 SR-1, a document that may become corrupted under
   certain circumstances (when you extensively edit text, graphics, and
   fields in a header or footer) no longer becomes corrupted.

    
   For more information, please see the following article in the Microsoft
   Knowledge Base:

    

          ARTICLE-ID: Q170887

          TITLE     : WD97: Field Code Becomes Damaged Under Certain
                  Circumstances


  • In Microsoft Word 97 SR-1, the language default for new documents is
   correct in different language versions of Microsoft Windows.


  • Microsoft Word 97 SR-1 prints graphics correctly to printers with
   nonsymmetrical resolutions (for example, 360 x 180 dpi).

    

Microsoft Excel 97 SR-1

 


Microsoft Office 97 SR-1 includes the following updates for Microsoft
Excel 97:


  • Views and reports created in earlier versions of Microsoft Excel work
   correctly in Microsoft Excel 97 SR-1.


  • In Microsoft Excel 97 SR-1, invalid characters in module names are
   corrected when you save a workbook in the Microsoft Excel 97 Workbook
   (*.xls) format.


  • Microsoft Excel 97 SR-1 allows you to cut and paste cells that contain
   data used by a PivotTable.


  • Print titles that use a medium width bottom border are printed correctly
   in Microsoft Excel 97 SR-1.


  • In Microsoft Excel 97 SR-1, worksheet database functions can refer to
   ranges that contain more than 32,768 rows.


  • In Microsoft Excel 97 SR-1, the ROW and COLUMN worksheet functions
   are recalculated properly when you insert new rows or columns into a
   worksheet.


  • The HTML Wizard in Microsoft Excel 97 SR-1 supports Automation, which
   means you can control it by using a Visual Basic for Applications macro.

    
   For more information, please see the following articles in the Microsoft
   Knowledge Base:

    

          ARTICLE-ID: Q168345

          TITLE     : XL97: Contents of Readme.txt for the Updated Html.xla
                  Add-in

      ARTICLE-ID: Q168561
      TITLE     : XL97: How to Programmatically Save a Worksheet as HTML


  • In Microsoft Excel 97 SR-1, sheet names may include characters outside
   the standard American National Standards Institute (ANSI) character set.

    

Microsoft PowerPoint 97 SR-1

 


Microsoft Office 97 SR-1 includes the following updates for Microsoft
PowerPoint 97:


  • When you insert a graphic into a presentation in Microsoft PowerPoint
   97, a blank box that contains a red "X" may appear. This problem was
   corrected in Microsoft PowerPoint 97 SR-1.


  • In Microsoft PowerPoint 97 SR-1, if you save a presentation in the
   Microsoft PowerPoint 7.0 format, Graph 97 objects that are embedded in
   the presentation are automatically converted to Graph 7.0 objects.


  • Microsoft PowerPoint 97 SR-1 includes a new Lotus Freelance Graphics
   2.1 converter. This converter improves the conversion of objects,
   graphics, and bullets to Microsoft PowerPoint 97.

    

Microsoft Access 97 SR-1

 


Microsoft Office 97 SR-1 includes the following updates for Microsoft
Access 97:


  • Microsoft Access 97 SR-1 includes a new Microsoft Access Snapshot
   Creator, which allows you to save a snapshot of data in tables for
   viewing by users who do not have Microsoft Access.


  • Microsoft Access 97 SR-1 correctly requeries the LinkChildFields and
   LinkMasterFields properties in chart objects in reports.

    

Microsoft Outlook 97 SR-1

 


Microsoft Office 97 SR-1 includes the following updates for Outlook 97:


  • Microsoft Outlook 97 SR-1 includes the Internet Mail Enhancement Patch
   (IMEP). IMEP improves sending and receiving messages through the
   Internet Mail Transport Service in Microsoft Exchange Server.


  • Microsoft Outlook 97 SR-1 includes improved MAPI spooler stability.
