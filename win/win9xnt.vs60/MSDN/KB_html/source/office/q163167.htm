OFF97: Organization Chart 2.0 Does Not Run After Upgrade

 Last reviewed: June 3, 1997
Article ID: Q163167
97 WINDOWS kbprb kberrmsg kbtool

The information in this article applies to:

  • Microsoft Office 97 for Windows
  • Microsoft Organization Chart for Windows, version 2.0
  • Microsoft Windows NT version 4.0

SUMMARY

When you attempt to run the Microsoft Organization Chart 2.0 after you upgrade to Microsoft Office 97, you may receive the following error message: 

   The server application, source file, or item can't be found. You
   should reinstall the server application.

CAUSE

The error message will occur if all of the following conditions are true:

  • You installed Windows NT 3.51 on your computer.

    -and-

  • You installed Microsoft Office 7.0 on your computer.

    -and-

  • You upgraded your computer from Microsoft Windows NT 3.51 to Microsoft Windows NT 4.0.

    -and-

  • When you installed Microsoft Office 97, you allowed the Setup program to remove Microsoft Office 7.0 from your computer.

RESOLUTION

To resolve this problem, you must remove and reinstall the Organization Chart. This will require the compact disc or floppy disks you used when you installed the program.

Steps to Remove and Reinstall the Organization Chart

  1. Quit all programs.

  2. On the Windows Start menu, point to Settings, and then click Control Panel.

  3. Double-click Add/Remove Programs.

  4. On the Install/Uninstall tab, click Microsoft Office 97 and then click Add/Remove.

  5. Microsoft Office 97 will start in maintenance mode. Click Add/Remove.

  6. Select Office Tools and click Change Option.

  7. Click to clear the Organization Chart check box and then click OK.

  8. Click Continue. The following message will then appear:

    You have chosen to remove 1 component. Are you sure you want to remove it?

    Click Yes. Setup will then remove the Organization Chart program and display a message when setup is done. Click OK.

  9. Repeat steps 4-6.

  10. Click to select the Organization Chart check box and then click OK.

  11. Click Continue to reinstall the Organization Chart.

For more information on how to add or remove Office components, query on the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q120802
   TITLE     : Office: How to Add/Remove a Single Office Program or
               Component
