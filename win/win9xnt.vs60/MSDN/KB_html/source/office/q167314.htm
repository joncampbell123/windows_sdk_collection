Off97: Errors Using Lotus Notes 4.5 and Microsoft Office 97

 Last reviewed: April 23, 1997
Article ID: Q167314
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you click Send To on the File menu in any of the Microsoft Office 97 programs, you may receive the following error message: 

   Mail is not installed on this system.

CAUSE

This problem occurs if Lotus Notes 4.5 is not configured correctly on the computer. For Lotus Notes 4.5 to be properly configured during installation, the Windows 95 messaging components must already be installed on the computer when you install Lotus Notes.

Note Lotus Notes 4.5 installs additional dynamic-link library (DLL) files when it detects the Windows 95 messaging components. Lotus Notes 4.5 must be able to find these DLLs to connect to the Windows 95 messaging system.

RESOLUTION

To use Lotus Notes 4.5 with Microsoft Office 97, the programs must be installed in a specific order. If you already installed Microsoft Office 97 and Lotus Notes 4.5, please use the following procedure to remove and reinstall the programs. If you have a clean installation of Windows 95 and have not installed Microsoft Office 97 or Lotus Notes 4.5, skip steps 1 and 2.

To resolve this problem, use the following steps:

  1. To remove Microsoft Office 97, use the following steps: 

          a. On the Start menu, point to Settings and click Control Panel.


          b. In the Control Panel window, double-click Add/Remove Programs.


          c. In the Add/Remove Programs Properties window, click Microsoft

             Office 97 and click Add/Remove.

      d. When you are prompted, click Remove All to completely uninstall
         Microsoft Office 97.


  2. To remove the Windows 95 Messaging components, use the following steps:

    

          a. On the Start menu, point to Settings and click Control Panel.

    

          b. In the Control Panel window, double-click Add/Remove Programs.

    

          c. In the Add/Remove Programs Properties window, click the Windows

             Setup tab, clear the Microsoft Exchange check box, and then click
         OK.


  3. To remove Lotus Notes 4.5 client software, see the documentation for
   Lotus Notes 4.5.


  4. Install the updated Windows 95 Messaging components with no services. To
   do this, use the following steps:

    

          a. Connect to the following Internet address:

    

                http://www.microsoft.com/windows95/default.asp

      b. Click Free Software and click Product Updates.

      c. Click System Updates.

      d. Follow the download and installation instructions for Exchange
         Update for Windows '95.


  5. To install Lotus Notes 4.5 client and create a profile for Lotus Notes
   4.5, see the documentation for Lotus Notes 4.5.


  6. Run the Setup program for Microsoft Office 97 and be sure to select
   "Converters for use with Lotus Notes" under the Converters component. To
   do this, use the following steps:

    

          a. On the Start menu, click Run.

    

          b. Type "<Drive>:\setup.exe" (without the quotation marks) and click

             OK.

         NOTE: <Drive> represents the drive letter from which you are
         installing Microsoft Office 97.

      c. When you are prompted, click Custom.

      d. Click to select Converters And Filters, and then click Change
         Option.

      e. Click to select Text Converters, and then click Change Option.

      f. Click to select the "Converters for use with Lotus Notes" check
         box.

      g. Click OK twice and add or remove any other components you want to
         install or remove.

      h. Click Continue.

         NOTE: Clicking "Lotus VIM Mail Support" during installation is not
         required for Lotus Notes 4.5 because it no longer uses vendor
         independent messaging (VIM). Lotus Notes 4.5 uses messaging
         application programming interface (MAPI).
