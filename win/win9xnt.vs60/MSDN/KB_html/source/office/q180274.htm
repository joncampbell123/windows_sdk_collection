OFF98: "Default Location" Error Running Value Pack Installer

 Last reviewed: February 26, 1998
Article ID: Q180274
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

When you double-click the Value Pack Installer, which is located in the Value Pack folder that is included in Microsoft Office 98 Macintosh Edition, you may receive the following error message 

   The default location required for this install could not be found.
   This install will be cancelled.
and no Value Pack components are installed.

CAUSE

This problem may occur if any of the following conditions are true:

  • You are running the Office 98 programs directly from a location on a network server.

    -or-

  • You performed a Run From CD installation of Office 98 and you are running Office from the compact disc (CD).

    -or-

  • You have removed Microsoft Office 98 Macintosh Edition from your computer.

WORKAROUND

To work around the first two problems, use one of the following methods.

Method 1: Office 98 Is Running from a Network Server

To install components from the Value Pack, follow these steps:

  1. Drag the Microsoft Office 98 folder from the network server to the hard disk on your computer.

  2. Double-click the Value Pack Installer in the Value Pack folder on your Office CD.

  3. After you have installed the Value Pack components, drag the Microsoft Office 98 folder from hard disk on your computer back to the network server that contains the network installation of Microsoft Office 98 Macintosh Edition.

    NOTE: When you perform this step, you must have full access to the Microsoft Office 98 folder on the network server. If you do not, you may receive an error message that is similar to the following: 

          You cannot copy "Microsoft Office 98" onto the folder "<Folder>",
      because you do not have the privilege to make changes.

After the files in the local Microsoft Office 98 folder are copied to the Microsoft Office 98 folder on the network server, you can use the installed Value Pack components on your computer.

Method 2: Office 98 Is Running from the CD

If you have performed a Run From CD installation of Office 98, you cannot install most of the components included in the Value Pack. This is true because most of the components are installed in the Microsoft Office 98 folder; because the folder is located on the Office 98 CD, the files cannot be installed.

However, it is still possible to install the fonts and Microsoft Office Manager (MOM), which is included in the Value Pack. To do this, follow these steps:

  1. In the Value Pack folder on the Office 98 CD, open the folder that contains the component you want to install.

  2. Drag the files from the folder into the System Folder on the hard disk on the computer.

    The System software automatically moves the files to the correct subfolder in the System Folder.

  3. Restart the computer.

    The fonts or Microsoft Office Manager are available after the computer is restarted.

To work around the third problem, use the following method:

Method 3: Office 98 Has Been Removed from Your Computer

  1. In the Finder, create a folder named "Microsoft Office 98" in the root folder of your hard drive.

  2. Then, create a folder named "Office" in the Microsoft Office 98 folder.

  3. Copy any file into the Microsoft Office 98:Office folder. Then, rename the file to "Microsoft Office 98", without quotation marks.

When you run the Value Pack Installer, you should not receive the error message. You can then use the Installer to remove Value Pack components from your computer. When you are done removing components, you should delete the Microsoft Office 98 folder.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
