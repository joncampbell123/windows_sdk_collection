 Last reviewed: March 10, 1997
Article ID: Q136714
7.00 WINDOWS kbsetup kbnetwork

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

The following is a partial list of the folders and files installed when you perform an administrative installation (Setup /a) of Microsoft Office to a network server. This list contains some of the files that are installed to the Microsoft Office folder (Msoffice by default) on the network server.

To view the additional files installed when you perform an administrative installation of Office, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q136713
   TITLE     : OFF95: Location of Files Installed on Server (Part 1 of 3)

   ARTICLE-ID: Q136716
   TITLE     : OFF95: Location of Files Installed on Server (Part 3 of 3)
