Requirements for The Microsoft Empowerment Pack for the Mac

 Last reviewed: November 17, 1997
Article ID: Q153391
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

This article describes the system requirements for installing The Microsoft Empowerment Pack for the Macintosh version 1.0 and any of its components.

MORE INFORMATION

Installing/Using the Empowerment Pack

The Empowerment Pack is presented through a guided tour using Microsoft Internet Explorer. To run the guided tour, you need, at a minimum:

  • A CD-ROM drive.
  • A 68030 processor.
  • System 7.0.1 operating system.
  • 8 megabytes (MB) of RAM.
  • 2 to 4 MB of hard disk space.

NOTE: The Empowerment Pack (and Microsoft Internet Explorer) will not start on a Power Mac running System 7.1.2. If you have a Power Mac running System 7.1.2, to run the Empowerment Pack or Microsoft Internet Explorer, you need to upgrade to System 7.5.

NOTE: If you have a 68K Macintosh and you are running a version of the System software earlier than System 7.5, you need to install Thread Manager version 2.0. To install Thread Manager version 2.0, run the Microsoft Internet Explorer installer in the Internet Explorer folder on the Empowerment Pack compact disc.

The guided tour will work, even if you are not connected to the Internet. But some features of Microsoft Internet Explorer will not be available to you without an Internet connection. The following list describes some of the features that may not be available without an Internet connection:

  • You will not be able to read newsgroups or send mail.
  • You may not be able to change the Options settings in Microsoft Internet Explorer.
  • The Home and Search buttons on the toolbar in Microsoft Internet Explorer may bring up an error message if you click these buttons.
