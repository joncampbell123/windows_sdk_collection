OFF98: Cannot Assign Option + Any Key Keystroke

 Last reviewed: February 2, 1998
Article ID: Q178638
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

When you try to assign a keystroke that uses the Option key either through the Symbol Command on the Insert Menu or the Customize command on the Tools menu, you may receive either an error beep or nothing happens.

CAUSE

These Option keys are already assigned by the system.

RESOLUTION

Use Option + a function key or any other key that is not currently assigned by the System.

STATUS

This behavior is by design.

MORE INFORMATION

When you click Symbol on the Insert menu and choose the Special Characters tab, it lists different symbols that are assigned to Option keys. These are assigned by the system and not Word. The Option key has reserved keystrokes that inserts real characters on the Macintosh. These keystrokes can be identified in Key Caps.
