Mac Office 4.2 HD Requirements and Disks Skipped During Setup

 Last reviewed: November 17, 1997
Article ID: Q121402
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2
  • Microsoft Office for the Power Macintosh, version 4.2

SUMMARY

When you install Microsoft Office, it is more than likely that some of the Setup disks will be skipped. Disks are skipped during Setup because the files located on the skipped disks are not needed for the installation type and platform that you selected during Setup. The number of disks used depends on the type of Setup you choose.

The table in the "Microsoft Office For The Power Macintosh" section of this article lists the disk space requirements for Microsoft Office version 4.2 for the Macintosh. The table in the "Microsoft Office For The Power Macintosh" section of this article lists the disk space requirements for Microsoft Office version 4.2 for the Power Macintosh. These tables also list the disks that will be skipped with each type of installation.

MORE INFORMATION

Note that the PowerPoint Viewer disk is NEVER installed. The PowerPoint Viewer is an application that you may distribute so that you can view PowerPoint slide shows on computers that do not have a full installation of Microsoft PowerPoint.
