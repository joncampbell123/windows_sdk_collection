OFF95: Binder 'Yes To All' Button Available Adding One Section

 Last reviewed: March 27, 1997
Article ID: Q131398
7.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

In Microsoft Binder, if you add a section from a file that is not compatible with Office Binders, you receive a dialog box asking if you want to continue. The available buttons are Yes, No, and Yes To All, even if you have only chosen to add one section.

CAUSE

When you add a section to a Binder from an existing file by using the Add From File dialog box or by dragging the file to the section pane in Binder, if the file is not compatible with Office Binders, the following error message appears 

   The document <FILENAME> cannot be added to the binder because the
   document type is not compatible with Office Binders. Would you like to
   continue?
where <FILENAME> is the name of the file that you added. The buttons that are available in the dialog box that contains this error message are Yes, No, and Yes To All. If you have only selected one document (or if each selected document is incompatible) when you receive this error message, all of the buttons perform the same action; that is, the section is not added to the Binder.

These buttons are available in the case where you have selected multiple files from which to add sections to the Binder. The buttons in this case have the following result: 

Button       Result


Yes          Continue by trying to insert the next section
Yes To All   Keeps choosing 'Yes' without prompting you for each section
No           Quit the operation
