Office: MSQuery Is Not Installed with Microsoft Excel

 Last reviewed: September 16, 1996
Article ID: Q117930
The information in this article applies to:
  • Microsoft Office for Windows, version 4.2
  • Microsoft Query for Windows, version 1.0

In Microsoft Office, Microsoft Query is not installed with Microsoft Excel. Instead, Microsoft Query is treated as a separate application and is grouped with "Converters, Filters, and Data Access".

To install Microsoft Query from Microsoft Office setup, do the following:

  1. Select the Converters, Filters, And Data Access option (in the Complete/Custom options section).

  2. Choose to install the entire module by selecting the check box, or choose the Change Options button, and select Data Access.

NOTE: Make certain you select the check box to install the entire Data Access module or choose the Change Options button again and select MSQuery only.
