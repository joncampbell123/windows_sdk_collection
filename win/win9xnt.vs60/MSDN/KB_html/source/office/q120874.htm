Office: Error Executing FoxPro From MOM Toolbar

 Last reviewed: September 16, 1996
Article ID: Q120874
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.0, 4.2

SYMPTOMS

When you select the Microsoft FoxPro icon from the Microsoft Office Manager(MOM) toolbar, the following error message may be displayed: 

   Cannot not find 'FOXPROW.EXE'. Please start the application another
   way. While the application is running, click on the toolbar button or
   choose the menu command again.
This error message may also occur when you choose the FoxPro item from the MOM menu.

CAUSE

Microsoft FoxPro versions 2.5a, 2.5b, or 2.6a, are not properly registered in the REG.DAT file. As a result, Microsoft Office Manager (MOM) is unable to locate the Microsoft FoxPro executable file.

WORKAROUND

Obtain Microsoft Office Manager 4.2b or higher. Beginning with MOM 4.2b, MOM searches both the REG.DAT and the [Extensions] section of the WIN.INI.

-or-

Edit the REG.DAT file. Note: Be sure to make a backup copy of the REG.DAT file before making changes to it. To properly register FoxPro in the REG.DAT, follow the steps below:

  1. In Program Manager, Choose Run from the File Menu.

  2. Enter "REGEDIT" (without quotation marks), in the Command Line and choose OK. This opens the Registration Editor.

  3. Under Registered File Types, choose Microsoft Foxpro for Windows, and from the Edit menu, choose Delete File Type.

  4. Double-click the Microsoft FoxPro icon in Program Manager. This opens Microsoft Foxpro.

  5. Select the Microsoft FoxPro button on the MOM toolbar. MOM will "learn" the location of FOXPROW.EXE.

  6. Exit Microsoft FoxPro.

  7. Exit MOM.

  8. Restart MOM.

  9. Click the FoxPro button from the MOM toolbar.

FoxPro should now start correctly.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
