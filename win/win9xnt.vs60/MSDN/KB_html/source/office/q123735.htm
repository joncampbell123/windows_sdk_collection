About Microsoft Office Dialog Box Displays Incorrect Version

 Last reviewed: September 16, 1996
Article ID: Q123735
The information in this article applies to:
  • Microsoft Office for Windows NT, version 4.2

SYMPTOMS

When you choose About Microsoft Office from the Microsoft Office menu after installing Microsoft Office for Windows NT, the About Microsoft Office dialog box displays Microsoft Office Version 4.3.

CAUSE

The About Microsoft Office dialog box displays version 4.3, even though you are running Microsoft Office version 4.2. However, the Microsoft Office Manager file, MSOFFICE.EXE that ships with Microsoft Office for Windows NT, version 4.2 is the correct file, even though the version that is displayed in the About Microsoft Office dialog box does not match the version of the application.
