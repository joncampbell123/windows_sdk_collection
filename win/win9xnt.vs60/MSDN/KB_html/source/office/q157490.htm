OFF97: Templates Missing from Binder Add Section Dialog Box

 Last reviewed: March 12, 1998
Article ID: Q157490
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you click Add on the Section menu in Microsoft Office Binder, some of the Office program templates (files with .wiz, .xlt, .pwz, and .pot extensions) may not appear on any of the tabs in the Add Section dialog box.

CAUSE

Many of the templates that are installed with Microsoft Office 97 include macros. Because it is better to run macros that are contained in a Binder section outside of the Binder, these templates are not included in the list of available files in the Add Section dialog box.

NOTE: The PowerPoint templates with a .pot extension should be included in the Add Section dialog box because no macros are attached to them.

RESOLUTION

To add one of the PowerPoint presentation templates to a Binder, do the following:

  1. On the Section menu in Microsoft Binder, click "Add from File."

  2. Locate the directory in which you installed the Microsoft Office PowerPoint templates. The following folders are the default locations:

    Program Files\Microsoft Office\Templates\Presentations

    -or-

    Program Files\Microsoft Office\Templates\Presentation Designs

  3. In the "Files of type" list, click All Files(*.*).

  4. Select the presentation template you want to add to the Binder.

  5. Click Add.

The template is added to the Binder file.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
