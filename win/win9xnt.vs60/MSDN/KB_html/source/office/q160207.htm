OFF97: Unable to Edit Images for Controls With a Drop-Down

 Last reviewed: December 2, 1996
Article ID: Q160207
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

You may be unable to edit or copy the button image of a toolbar control in any of the Microsoft Office programs.

CAUSE

The following menu items are not available when you work with a toolbar control that has a drop-down (for example, the Draw control on the Drawing toolbar):

  • Copy Button Image
  • Reset Button Image
  • Edit Button Image
  • Change Button Image

RESOLUTION

You are unable to use any of the listed menu items when you work with a control that has a drop-down.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In any of the Microsoft Office programs you can customize the images for most of the controls on your toolbars. For example, to modify and reset the image for the Bold control on the Formatting toolbar, in Microsoft Excel do the following:

Edit the Image of a Control

  1. On the Tools menu, click Customize.

  2. If the Formatting toolbar is currently not displayed, click the Toolbars tab of the Customize dialog box and select the Formatting check box.

  3. With the Customize dialog box displayed, right-click the Bold control on the Formatting toolbar and click Edit Button Image on the shortcut menu.

NOTE: If you right-click the Font Color control, for example, the menu items listed earlier in this article are not available on the shortcut menu.

  1. Make any changes you like to the image and click OK.

  2. Click Close in the Customize dialog box.

The changes you made in Step 4 are now shown on the Bold control.

Reset the Image of a Control

  1. On the Tools menu, click Customize.

  2. If the Formatting toolbar is currently not displayed, click the Toolbars tab of the Customize dialog box and select the Formatting check box.

  3. With the Customize dialog box displayed, right-click the Bold control on the Formatting toolbar and click Reset Button Image on the shortcut menu.

  4. Click Close in the Customize dialog box.

The Bold control will be reset to the default image.

REFERENCES

For more information about Customizing Toolbar Buttons, click the Index tab in Microsoft Excel Help, type the following text 

   toolbars, buttons
and then double-click the selected text to go to the "Change a toolbar button image or a menu command image" topic.
