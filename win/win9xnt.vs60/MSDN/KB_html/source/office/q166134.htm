OFF97: Error Inserting Object on Certain Hitachi Computers

 Last reviewed: April 3, 1997
Article ID: Q166134
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

When you attempt to insert an object in any of the products listed at the beginning of this article, you may receive the following error message 

   <Program> caused an invalid page fault in module OLE32.DLL at
   <Memory Offset>.
where <Program> is the name of the program that you are running and <Memory Offset> is the memory location where the problem occurred. Note that the memory offset may vary each time you receive the error message.

CAUSE

This problem occurs if you are running Microsoft Office 97 on a Hitachi Mx150 or Mx166 computer.

STATUS

Hitachi America has confirmed this to be a problem when you use the Hitachi Mx150 and Mx166 computer models. Contact Hitachi for possible fixes or workarounds.

MORE INFORMATION
