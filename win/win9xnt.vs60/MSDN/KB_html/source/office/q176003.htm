OFF98: Error Clicking Gallery Tab in Office Assistant Dialog Box

 Last reviewed: February 2, 1998
Article ID: Q176003
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SYMPTOMS

When you click the Gallery tab in the Office Assistant dialog box, you receive the following error message: 

   There is not enough memory available to complete this operation.

CAUSE

This problem may occur if you remove all of the Office Assistant files from the Mac Office 98:Office:Assistants folder.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about changing the Office Assistant, click the Index tab in Microsoft Excel Help, type the following text 

   office assistant, characters
and then double-click the selected text to go to the "Choose a different Office Assistant" topic.
