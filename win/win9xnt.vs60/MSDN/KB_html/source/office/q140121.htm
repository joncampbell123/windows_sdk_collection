OFF95: Setup Errors with Reinstallation of Small Business Pack

 Last reviewed: March 10, 1997
Article ID: Q140121
1.00 WINDOWS kbtool kberrmsg

The information in this article applies to:

  • Microsoft Small Business Pack for Windows 95, version 1.0
  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

If you have installed the Microsoft Small Business Pack, and you run Setup in maintenance mode and you select the Reinstall option, you may receive the following error message 

   Setup Error 998: Object <number>: Your Setup Files may be damaged. Try
   restarting the Setup Program.
where <number> is the number 293 or 294.

When you receive this error message, your only option is to click OK. When you click OK, you receive the following message: 

   Microsoft Small Business Pack Setup was completed Successfully
If you run Setup one more time, you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you then click the Details button, you get the message: 

   ACMESETUP caused an invalid page fault in module SBP.dll at
   <memory address>

CAUSE

This problem occurs if you have partitioned your hard disk or are using more than one hard disk and Windows is installed to one partition (or drive) while Microsoft Office and Small Business Pack are installed on another partition (or drive). For example, this problem will occur if Windows is installed on your C drive and Microsoft Office and Small Business Pack are installed on your D drive.

RESOLUTION

To resolve this problem, remove both Microsoft Office and Small Business Pack from the drive partition (or the alternate drive) and reinstall them both to the drive where Windows is installed. For example, remove Microsoft Office and Small Business Pack from the D drive, and reinstall them to the C drive.

If you receive the invalid page fault error message, you may be able to temporarily work around this situation by doing the following:

  1. Rename the Setup.stf file that is located in the in the MSOffice\Small Business Pack\Setup folder (the default location of this file) to "Setup.xxx" (without the quotation marks).

  2. Run the Small Business Pack Setup program from Disk 1 and install it to the same folder where you previously installed Small Business Pack.
