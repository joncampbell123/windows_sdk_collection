Office: Task List May Include Extra Program

 Last reviewed: September 9, 1996
Article ID: Q110982
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.0, 4.2, 4.2c

SYMPTOMS

In Microsoft Windows, when you bring up the task list by pressing CTRL+ESC, the list of programs reported as running may include an extra program that is not running.

CAUSE

This problem occurs when you bring up the task and all of the following conditions are true:

  • The mouse pointer is placed on an icon in the Microsoft Office Manager toolbar.
  • The toolbar size in Microsoft Office Manager is set to small buttons.
  • The Show Tooltips option in Microsoft Office Manager is selected.

Note that this is not a problem in Windows NT.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
