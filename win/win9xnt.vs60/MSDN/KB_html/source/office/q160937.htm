OFF97: Microsoft Camcorder Requires Microsoft Windows 95

 Last reviewed: March 12, 1998
Article ID: Q160937
The information in this article applies to:
  • Microsoft Office 97

SYMPTOMS

If you start Microsoft Camcorder 97 under Microsoft Windows NT, version 3.51 or 4.0, you receive the following error message: 

   Microsoft Camcorder requires Microsoft Windows 95 to record and play
   movies. However, you can play standalone movies on Microsoft Windows NT
   Workstations.

CAUSE

Microsoft Camcorder records and plays movies only under Windows 95. However, you can play stand-alone movies on a Windows NT 3.51 or 4.0 workstation if you create them in Microsoft Camcorder under Windows 95.

MORE INFORMATION

The Setup program does not install Microsoft Camcorder when you install Microsoft Office 97; it is part of the ValuPack, which is on the Microsoft Office 97 compact disc. Microsoft Camcorder allows you to capture and play back screen events. It allows you to create a movie and save it as an audio- video interleaved (*.avi) file, you can export an .avi file as a stand- alone executable (*.exe) file. If you want to play the movie under Microsoft Windows NT, play the movie using an .exe file that you create from Microsoft Camcorder on a computer running Microsoft Windows 95.
