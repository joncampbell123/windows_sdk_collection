Office for the Macintosh 4.2.1b CD System Components

 Last reviewed: November 17, 1997
Article ID: Q164376
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2.1b

SUMMARY

Microsoft Office for the Macintosh, version 4.2.1b on compact disc, is now bundled with Microsoft Bookshelf 1996-97, a collection of business tools, and a set of Internet-related programs. This enhanced version replaces the standard version on compact disc and is not available on floppy disks.

This enhanced version of Microsoft Office for the Macintosh, version 4.2.1b, contains two compact discs; disc 1 is the standard CD-ROM version of Microsoft Office for the Macintosh, and disc 2 contains Bookshelf, the business tools, and the Internet programs. You must run the Setup program on each disc to install this version on your computer.

When you run the Setup program on disc 2, several system components (extensions, control panels, fonts, and preferences) are installed into your System folder. The number of system components vary, depending on the version of the operating system and the operating system components you installed. Also, the Setup program does not install system components are over existing copies unless the existing copy is older than the file that is being installed.

NOTE: When you run the disc 2 Setup program and click Remove, none of the extensions, control panels, fonts, or preferences are removed. This is because the Setup program cannot determine whether it installed the components, or if they already existed on the system.

MORE INFORMATION

The following table lists all of the system components that may be installed by the disc 2 Setup program.

NOTE: It is unlikely that all the components are installed. This is true because the components were installed with the system, you don't need to install the components (the system does not require it), or you perform a custom installation.
