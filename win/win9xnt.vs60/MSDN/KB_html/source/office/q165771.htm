Form Fields Are Missing When Recipient Is Not Using WordMail

 Last reviewed: March 28, 1997
Article ID: Q165771
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Outlook 97
  • Microsoft Word 97 for Windows

SYMPTOMS

If you use WordMail as your e-mail editor and you send a message that contains a form field, such as a drop-down list form field, the form field and its result fail to appear if you send the message to a recipient who is not using WordMail.

WORKAROUND

To work around this behavior, use one of the following methods.

Method 1: Use Word as Your E-Mail Editor

To see form fields in a e-mail message that you receive, select the "Use Microsoft Word as the e-mail editor" option, and then reopen the mail message. To do this, the following steps:

  1. Start Microsoft Outlook.

  2. On the Tools menu, click Options.

  3. Click the E-mail tab, click to select the “Use Microsoft Word as the e-mail editor” check box, and then click OK.

You can open your e-mail message and see the form fields.

Method 2: Attach the Document to the Mail Message

To prevent the form fields and results from disappearing when you send the form to a user who is not using WordMail, send the form as an attached document.
