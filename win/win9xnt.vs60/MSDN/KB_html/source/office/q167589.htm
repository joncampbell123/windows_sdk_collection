OFF97: MS Access CCP Does Not Detect Small Business Edition

 Last reviewed: March 13, 1998
Article ID: Q167589
The information in this article applies to:
  • Microsoft Access 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

The Compliance Checking Program (CCP) in the upgrade version of Microsoft Access 97 fails to recognize Microsoft Office 97 Small Business Edition as an application that qualifies for the competitive upgrade program.

RESOLUTION

Call Microsoft Technical Support for assistance installing your upgrade version of Microsoft Access 97. Refer to your product literature or your support contract for the technical support number to call. Or, to obtain the support phone number for your product, please visit Microsoft's "Support Options & Phone Numbers" Web page at the following address: 

    http://www.microsoft.com/supportnet/
When you call Microsoft Technical Support and speak with a support engineer, please inform the engineer that you own a qualifying program and that you need assistance installing it.

STATUS

Microsoft has confirmed this to be a problem in the upgrade version of Microsoft Access 97. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

If you own one of a selected list of database applications, you qualify for the Microsoft Access competitive upgrade, which entitles you to buy an upgrade version of Microsoft Access at a reduced price. The Setup program checks for a qualifying competitive database program before it installs Microsoft Access.

Microsoft Office 97 Small Business Edition is one of the products that qualifies for the Microsoft Access competitive upgrade program; however, the Setup program fails to recognize it as such. For a list of other database programs that qualify for a competitive upgrade, refer to the Microsoft Access 97 upgrade version box, or see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q167237
   TITLE     : ACC97: Products That Qualify for Access 97 Competitive
               Upgrade
