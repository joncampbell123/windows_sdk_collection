OFF98: MS Preference Panels Folder Location and Contents

 Last reviewed: February 2, 1998
Article ID: Q179805
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Internet Explorer for the Macintosh, version 4.0
  • Microsoft Internet Explorer for the Macintosh, version 3.x

SUMMARY

When you install any of the programs listed at the beginning of this article on your Macintosh, a folder called MS Preference Panels is created on the hard disk. Depending the program you install, the folder location and contents may vary.

This article lists the MS Preference Panels folder location and contents for each of the programs listed at the beginning of this article.

MORE INFORMATION

Microsoft Internet Explorer for the Macintosh, Version 3.x

When you install Microsoft Internet Explorer 3.x, the MS Preference Panels folder is created in the System Folder:Extensions folder. It contains the following control panels: 

   Advanced Networking
   Configurations
   Cookies
   Downloads
   File Helpers
   General E-mail
   Home/Search Panel
   Java
   Language/Fonts
   Password Dialogs
   Protocol Helpers
   Proxies
   Ratings
   Security
   Security Zones
   Subscriptions
   User Passwords
   Utility Dialogs
   Web Browser UI
   Web Content

Microsoft Internet Explorer for the Macintosh, Version 4.0

When you install Microsoft Internet Explorer 4.0, the MS Preference Panels folder is created in the System Folder. It contains the following control panels: 

   Advanced Networking
   Configurations
   Cookies
   Downloads
   File Helpers
   General E-mail
   Home and Search
   Java
   Language and Fonts
   Password Dialogs
   Protocol Helpers
   Proxies
   Ratings
   Security
   Security Zones
   Subscriptions
   User Passwords
   Utility Dialogs
   Web Browser UI
   Web Content

Microsoft Office 98 Macintosh Edition

When you install Microsoft Office 98 Macintosh Edition, the MS Preference Panels folder is created in the System Folder. It contains the following control panels: 

   Advanced Networking
   Cookies
   File Helpers
   General E-mail
   Home and Search
   Password Dialogs
   Protocol Helpers
   Proxies
   Ratings
   Security
   User Passwords

How the MS Preference Panels Folders Are Managed

When you install Microsoft Internet Explorer 4.0, the MS Preference Panels folder used by Microsoft Internet Explorer 3.x is automatically removed from the Extensions folder.

If you have removed Microsoft Internet Explorer 3.x from your computer by using some other means, you can safely remove the MS Preference Panels folder from the Extensions folder if it still exists.

Note that if you install both Microsoft Office 98 Macintosh Edition and Microsoft Internet Explorer 4.0 on the same computer, the MS Preference Panels folder contains all of the preference panels used by both programs.
