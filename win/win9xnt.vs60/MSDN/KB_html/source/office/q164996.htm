OFF97: Office Assistant Displays Different Startup Options

 Last reviewed: March 15, 1997
Article ID: Q164996
The information in this article applies to:
  • Microsoft Office 97 for Windows

SUMMARY

When you start Microsoft Outlook 97, the Office Assistant displays different startup options from the options other Microsoft Office 97 programs display. Microsoft Outlook also includes a check box that you can select to continue displaying these options each time you start the Microsoft Outlook 97. Other Microsoft Office 97 programs display the startup options only when you first start the program.

STATUS

This behavior is by design of Microsoft Outlook 97.

MORE INFORMATION

All of the information that the Office Assistant displays is also available in Help. To find the information that is displayed the first time you start an Office program, use either of the following methods.

Method 1

  1. On the Help menu, click Contents And Index.

  2. click the Contents tab.

  3. Double-click Key Information.

Method 2

  1. On the Help menu, click Contents And Index.

  2. Click the Contents tab.

  3. Double-click Getting Help.

The Office Assistant startup options for Microsoft Outlook are the following: 

   See key information for upgraders and new users
   Find out about the Office Assistant (That's me!)
   Add holidays to your Calendar
   Important information from Schedule+ and other programs
   Set up an e-mail service
When you click to select the "Show these choices at startup" check box, the options appear every time you start Microsoft Outlook 97.

The Office Assistant startup options for all Microsoft Office 97 programs except for Microsoft Outlook 97, are the following: 

   See key information for upgraders and new users
   Find out about the Office Assistant (That's me!)
   Start using <program name>
NOTE: No option allows you to display these options every time you start the Office program.
