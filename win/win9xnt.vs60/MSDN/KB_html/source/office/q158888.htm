OFF97: Extra Buttons Appear on Office Shortcut Bar After Setup

 Last reviewed: March 28, 1997
Article ID: Q158888
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

After you install Microsoft Office 97 on your computer, the Microsoft Office Shortcut Bar may contain too many buttons; and some of the buttons appear to be duplicates of other buttons. (For example, there is a "Start a New Document" and "New Office Document" button.)

CAUSE

This behavior may occur when you install Office 97 on a computer on which you previously installed Office 7.0 and its Office Shortcut Bar.

RESOLUTION

To resolve this behavior, remove the extra buttons. To do this, use the following steps:

  1. Right-click a blank area on the Office Shortcut Bar. On the menu that appears, click Customize.

  2. Click the Buttons tab. In the Toolbar list, click Office.

  3. In the "Show these Files as Buttons" list, click the name of a button that you want to delete. Then, click Delete.

    Repeat this step for all buttons that you want to delete.

NOTE: After you delete the duplicate buttons, you can display more buttons by selecting the check boxes to the left of the button names. After you select the buttons that you want to display, click OK.

STATUS

This behavior is by design of the Office Shortcut Bar and the Microsoft Office 97 Setup program.

MORE INFORMATION

When you install Microsoft Office 97 on a computer on which you already installed Microsoft Office 7.0, the Setup program retains the Office Shortcut Bar settings. Therefor, the shortcut bar may contain a superset of the buttons provided by both versions of Microsoft Office.

The number of additional buttons that appear depends on several factors, including the following:

  • Microsoft Office 7.0 programs that you previously installed
  • Whether you allow the Setup program to remove earlier Office 4.2, 4.3, and 7.0 components
  • How you configured the Office Shortcut Bar prior to installing Office 97

When you delete older buttons from the Office Shortcut Bar, make sure that you delete the correct buttons.

The following are older buttons that you may delete: 

   Add a Contact
   Add a Task
   Make an Appointment
   Microsoft Bookshelf '95
   Microsoft Schedule+
   Office Compatible Answer Wizard
   Open a Document
   Start a New Document
The following are Office 97 buttons that you should not delete: 

   Internet Explorer
   Microsoft Bookshelf Basics
   Microsoft Outlook
   New Appointment
   New Contact
   New Journal Entry
   New Message
   New Note
   New Office Document
   New Task
   Open Office Document
