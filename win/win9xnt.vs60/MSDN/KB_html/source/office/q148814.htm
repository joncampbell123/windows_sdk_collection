Conflict Catcher May Disable Necessary Office Extensions

 Last reviewed: November 21, 1997
Article ID: Q148814
The information in this article applies to:
  • Microsoft Office for the Power Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

If you are using Conflict Catcher on your Power Macintosh, you may inadvertently disable many or all of the System Extensions that are required by Microsoft Office programs and you may not be able to start the Office programs.

MORE INFORMATION

The following is a list of the extensions and control panels that Microsoft Office installs, and that Conflict Catcher "exposes." Problems may occur if some or all of these files are disabled. 

   Name of file                     Kind of file        Location
   ---------------------------------------------------------------
   
   Apple Event Manager              system extension    Extensions
   Microsoft Dialog Library         shared library      Extensions
   Microsoft Find File Library      shared library      Extensions
   Microsoft Mail Library           shared library      Extensions
   Microsoft OLE Automation         shared library      Extensions
   Microsoft OLE Extension          system extension    Extensions
   Microsoft OLE Library            shared library      Extensions
   Microsoft Tool Editor Library    shared library      Extensions
   ObjectSupportLib                 shared library      Extensions
   Shared Code Manager              system extension    Extensions
   Visual Basic for Applications    shared library      Extensions
   Microsoft Office Manager         control panel       Control Panels
NOTE: The file ObjectSupportLib is installed by the System software for Macintosh System versions 7.5.2 and 7.5.3. The Setup program for Microsoft Office will only install this file if the ObjectSupportLib extension is not in the Extensions folder during the Setup process.

The following is a list of the extensions that are required to be in the Extensions folder in order for the Microsoft Office programs to start. Note that the Office programs may start, but they may not have full functionality if other extensions are missing from the Extensions folder. For example, PowerPoint will start with just the Microsoft OLE Library extension, but you may not be able to insert a ClipArt image into your PowerPoint presentation with just this one extension loaded. 

   Extensions                     PowerPoint   Word   Microsoft Excel
   ------------------------------------------------------------------
   Apple Event Manager
   Microsoft Dialog Library                      x           x
   Microsoft Find File Library
   Microsoft Mail Library
   Microsoft OLE Automation                                  x
   Microsoft OLE Extension
   Microsoft OLE Library              x          x           x
   Microsoft Tool Editor Library
   ObjectSupportLib                                          x
   Shared Code Manager
   Visual Basic for Applications                             x
   Microsoft Office Manager
