OFF95: "Cannot Create" Errors When You Set Up MS Office Pro

 Last reviewed: March 27, 1997
Article ID: Q140242
7.00 WINDOWS kbsetup kberrmsg

The information in this article applies to:

  • Microsoft Office Professional for Windows 95, version 7.0

SYMPTOMS

When you perform a post-administrative setup of Microsoft Office Professional for Windows 95, the following error messages may appear: 

   Cannot Create MS Access Workgroup Administrator.lnk
   Cannot Create Shortcut Bar\Quickshelf\Dictionary Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\Dictionary Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\Thesaurus Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\Thesaurus Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\Quotations Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\Quotations Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\Encyclopedia Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\Encyclopedia Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\Atlas Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\Atlas Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\Chronology Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\Chronology Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\1995 Almanac Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\1995 Almanac Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\All Books Contents.lnk
   Cannot Create Shortcut Bar\Quickshelf\All Books Find.lnk
   Cannot Create Shortcut Bar\Quickshelf\Address Builder.lnk
NOTE: You can click Ignore when you encounter one of the above messages and safely continue with the setup process as an interim solution.

CAUSE

This problem occurs if you perform a network setup when the following conditions are true:

  • Your network administrator performs an administrative setup of Microsoft Office Professional for Windows 95 to a network server to which you do not have write privileges.

    -and-

  • Your network administrator uses the Network Installation Wizard to modify the Microsoft Office Professional for Windows 95 Setup.stf file.

    -and-

  • When modifying the Setup.stf file with the Network Installation Wizard, your network administrator clicks the Run from Network Server installation option and enables a Minimize Footprint option in the Office Options window.

RESOLUTION

An updated version of the Network Installation Wizard corrects this problem. The updated Network Installation Wizard is available on the December 1995 release of the Microsoft TechNet compact disc, the World Wide Web, and The Microsoft Network.

To correct this problem, rerun the updated Network Installation Wizard to modify the Setup.stf file and then perform a network setup. 

   To access Resource Kit files on the Microsoft World Wide Web server,
   connect to the following address:

      Http://Www.microsoft.com/TechNet/Desk/Office/Orkf/Orkftoc.htm

   To access Resource Kit files on the Microsoft Network:

   1. On the Edit menu, click Go To.

   2. Click Other Location.

   3. When the prompt "Type a Go word for a particular service" appears,
      type "TechNet" (without the quotation marks).

   4. Click TechNet Download Area.
You can also work around this problem if your network administrator disables the Minimize Footprint options in the Office Options window of the Network Installation Wizard before you perform a network setup.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft Network Installation Wizard. This problem has been corrected in an updated version of the Network Installation Wizard.

MORE INFORMATION

The Network Installation Wizard is a tool that allows you to modify the Microsoft Office for Windows 95 Setup information files to create a customized network installation of Microsoft Office. The Wizard is included in the "Microsoft Office for Windows 95 Resource Kit." The Wizard is fully documented in Chapter 9, "Customizing Client Installations," of the "Microsoft Office for Windows 95 Resource Kit."

The Minimize Footprint option in the Network Installation Wizard conserves disk space by making use of shortcuts, or links to Office files without installing them locally.
