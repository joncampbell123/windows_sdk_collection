OFF97: Errors Quitting Office Apps with IntranetWare Network

 Last reviewed: December 30, 1997
Article ID: Q177637
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Outlook 97
  • Microsoft Access 97

SYMPTOMS

When you quit a Microsoft Office 97 for Windows program that runs on a Novell IntranetWare Network, you may receive the following message 

   This connection must be maintained for Novell Directory Services
   use on tree <tree name>. It can only be removed after logging out of
   Directory Services on the tree. Do you wish to log out of directory
   services on the tree <tree name> now?
where <tree name> is the name of the Novell directory tree or network database.

CAUSE

This error may occur if the following conditions are true:

  • You are using Novell IntraNetware version 4.11.

    -and-

  • You are using the Novell IntraNetware client version 2.2 for Microsoft Windows 95.

    -and-

  • You open a file on the network with one of the products mentioned at the beginning of this article.

RESOLUTION

To correct this problem, contact Novell to obtain an updated copy of Novellnp.dll.

For information about how to contact Novell, query in the Knowledge Base for one of the following articles: 

   ARTICLE-ID: Q65416
   TITLE     : Hardware and Software Third-Party Vendor Contact List, A-K

   ARTICLE-ID: Q60781
   TITLE     : Hardware and Software Third-Party Vendor Contact List, L-P

   ARTICLE-ID: Q60782
   TITLE     : Hardware and Software Third-Party Vendor Contact List, Q-Z

MORE INFORMATION

This network has not been tested for use with Microsoft Office 97 for Windows and is not supported.
