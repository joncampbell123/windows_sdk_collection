OFF: All Sheets Print From Excel Section in Binder

 Last reviewed: February 20, 1998
Article ID: Q135656
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Binder for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Office Binder 97 for Windows
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you print a Microsoft Excel section in Binder 7.0 or 97, all sheets in the section that contain data (worksheets, macro sheets, dialog sheets, and modules) are printed. Dialog sheets are always printed whether they have been changed or not. Hidden sheets in the Microsoft Excel section are also printed.

WORKAROUND

To work around this behavior, do the following to print the Microsoft Excel section separately from the other sections in the binder:

  1. Activate the Microsoft Excel section by clicking the Microsoft Excel icon in the left pane.

  2. To select the sheets that need to be printed, hold down the CTRL key and select the sheets.

  3. Click Print on the Section menu, and then choose the Selected Sheets option in the Print dialog box. Click OK.

  4. In the left pane of the binder, click one of the other sections.

  5. Hold down the CTRL key and click on the other sections that need to be printed. On the File menu, click Print Binder and choose Selected Sections in the Print dialog box.

  6. Click OK to print the selected sections.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.

MORE INFORMATION

Note that when you use this workaround, the page numbers may not be printed continuously. After the Microsoft Excel section is printed, click Print Binder on the File menu. In the numbering section of the Print Binder dialog box, choose Consecutive in the Numbering section and change the starting page number to be one greater than the number of sheets that were printed from the Microsoft Excel section. For example, if three pages were printed from the Microsoft Excel section, change the Start Numbering box to 4 in the Print Binder dialog box.
