OFF: Error Trying to Quick View Microsoft Excel Workbook

 Last reviewed: March 25, 1997
Article ID: Q150366
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you use the Quick View command in Microsoft Windows 95 to view a Microsoft Excel workbook file, you may receive the following error message 

   There is no viewer capable of viewing Microsoft Excel Worksheet files.
and you are unable to view your file.

NOTE: In Microsoft Office 97, you do not receive the error message and you cannot view the file.

CAUSE

You may experience this problem if both of the following conditions are true:

  • Your Microsoft Excel file was unbound from a Microsoft Office Binder file.

    -and-

  • Your Microsoft Excel file was a Microsoft Excel Chart section when it was in the Binder.

WORKAROUND

To work around this problem, use one of the following appropriate methods:

Microsoft Excel 97

  1. Using Windows Explorer, right-click the file and click Quick View on the shortcut menu.

    The file is opened in Quick View, but you cannot view the worksheet.

  2. On the File menu, click "Open File for Editing."

    Microsoft Excel is started, and you can view and edit the workbook.

NOTE: If you did not install Microsoft Excel 97, you cannot use Quick View to view your workbook.

Microsoft Excel for Windows 95, version 7.0

Method 1:

Open your file in Microsoft Excel and on the File menu, click Save. Then, close the file and quit Microsoft Excel. You should now be able to Quick View your file.

Method 2:

If you do not have Microsoft Excel installed on your computer, use the Microsoft Excel Viewer for Windows 95 to view your file.

MORE INFORMATION

Quick View, which is installed by Microsoft Windows 95, allows you to view documents that are created by Windows-based programs without starting the program. If Quick View is not available on the shortcut menu, either the program that created your file does not support Quick View or Quick View is not installed on your computer. Please see your Windows 95 documentation for information about how to install Quick View on your computer.
