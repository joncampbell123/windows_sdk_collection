Can't Run Money Demo While Excel Is Running

 Last reviewed: September 16, 1996
Article ID: Q121805
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SYMPTOMS

When you run the Money demo that is included with Microsoft Office, you receive the following error message if Microsoft Excel is running: 

   An application named EXCEL is presently running. It is incompatible with
   this application. Please click "Ok" to exit. You must first quit out of
   EXCEL before running this application.

MORE INFORMATION

To run the demo of Microsoft Money that is included with Microsoft Office, you must close Microsoft Excel if it is running.

Note that this does not mean that the application Microsoft Money is incompatible with Microsoft Excel. It is only the demonstration application that you cannot run while Microsoft Excel is running.
