OFF97: Run From CD Reinstall Removes Shortcuts from Program Menu

 Last reviewed: June 3, 1997
Article ID: Q168103
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

After you reinstall Microsoft Office 97, Office icons are missing from the Windows 95 Start menu.

CAUSE

This problem occurs when you do both of the following:

  • You run the Office 97 Setup program and click Run from CD. 

        -and-
  • You run the Office 97 Setup program again and click Reinstall.

WORKAROUND

To restore missing icons, use either of the following methods.

Method 1: Remove and Reinstall Office

  1. Run the Office 97 Setup program. When you are prompted to select a type of installation, click Remove All.

  2. After Setup is finished, run the Setup program again and click Run from CD. This step installs Office again and all the program icons.

Method 2: Manually Create the Icons

  1. Right-click the Windows 95 Start menu and click Open. Double-click the Programs folder.

  2. On the File menu, point to New and click Shortcut. Click Browse, and then open the Office folder on the compact disc.

  3. Click the Office program for which you want to create a shortcut and click Open.

  4. Click Next and type a name for the shortcut (for example, Microsoft Access), and then click Finish.

    NOTE: By default, the programs for Office 97 are installed in the Program Files\Microsoft Office\Office folder.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Performing a Run From CD installation of Office 97 creates shortcuts to the Office compact disc to run the Office programs. This minimizes the amount of space required on the local hard disk. However, when you run the Office Setup program and click Reinstall, the Setup program removes the following shortcuts (icons) from the Programs group on the Windows 95 Start menu: 

   Access
   Powerpoint
Setup also removes the following shortcuts from the Microsoft Office folder on the local hard disk: 

   Microsoft Access.lnk
   Microsoft Office Shortcut Bar.lnk
   MS Access Workgroup Administrator.lnk
For more information about hard disk requirement for Office 97, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q158704
   TITLE     : OFF97: Disk Space Requirements for Microsoft Office 97
