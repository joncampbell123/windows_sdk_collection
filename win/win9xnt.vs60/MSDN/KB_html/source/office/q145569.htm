ORK: Network Install Wizard May Delete Existing Newsetup.stf

 Last reviewed: March 27, 1997
Article ID: Q145569
7.00 WINDOWS kbsetup kbnetwork

The information in this article applies to:

  • Microsoft Office Resource Kit for Windows 95, version 7.0

SYMPTOMS

If you run the Network Installation Wizard included in the Office Resource Kit (ORK), and you then quit the wizard, this action may delete the Newsetup.stf file. This problem occurs if the Newsetup.stf file is located in the same folder as the administrative Setup.stf file that the wizard modifies.

CAUSE

When you run the Network Installation Wizard, it checks to see if you already have a file called Newsetup.stf in the same folder as the Setup.stf file that the wizard modifies. If you quit the wizard, this deletes the Newsetup.stf file, if it exists.

RESOLUTION

If you run the Network Installation Wizard, cancel out of the wizard, and then notice that your Newsetup.stf file has been deleted, you will have to re-create a new one using the Network Installation Wizard.

If you have a modified .stf file called Newsetup.stf in the same folder as the admin Setup.stf, and you have not yet run the Network Installation Wizard again (meaning after the initial installation), make sure you create a backup of the file Newsetup.stf. Then, if you accidentally delete the file, you can always make a copy of this backup, and rename it Newsetup.stf.

You can also work around this problem by obtaining the Network Installation Wizard, version 1.1 or later. Netinst.exe, a self-extracting file, contains version 1.1 of the wizard. You can download the file from the following services:

  • The Microsoft World Wide Web Site:

http://www.microsoft.com/TechNet/desk/office/reskit/orkf/apadf.HTM
  • Internet (anonymous FTP)

ftp://ftp.microsoft.com/services/technet/office95/netinst.exe

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Microsoft Office for Windows 95 Resource Kit contains one Microsoft Press book, "Microsoft Office for Windows 95 Resource Kit," and one compact disc. The suggested retail price for the kit is (U.S.)$49.95. The kit is available free-of-charge on The Microsoft Network and the World Wide Web. The kit is also included on the Microsoft TechNet compact disc.

To obtain the Microsoft Press book, "Microsoft Office for Windows 95 Resource Kit," contact Microsoft Press at (800) MSPRESS or (800) 677-7377. Do not call the Microsoft Sales Information Center (MSIC) to order the kit.

You can obtain this Microsoft Press title from any local computer bookstore, or you can send in the Microsoft Office Supplemental Offers coupon, which is included in your Microsoft Office retail package.

You can also view the Microsoft Office for Windows 95 Resource Kit from the TechNet location on The Microsoft Network or the World Wide Web.

In the ORK, the Netinst.wiz folder contains the Network Installation Wizard. This tool allows you to modify the Microsoft Office for Windows 95 Setup information files to create a customized network installation of Microsoft Office. The wizard is fully documented in Chapter 9, "Customizing Client Installations," in the "Microsoft Office for Windows 95 Resource Kit." Please review the information in this chapter before you attempt to use the Network Installation Wizard.

For additional details about using the Wizard, see Readme.doc in the Netinst.wiz folder. This file supplements the information in Chapter 9 of the Office Resource Kit.

To set up the Network Installation Wizard, double-click the Setup.exe file in the Netinst.wiz folder.
