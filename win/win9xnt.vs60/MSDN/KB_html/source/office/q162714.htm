OFF97: Files Installed with the Data Access Pack

 Last reviewed: January 31, 1997
Article ID: Q162714
The information in this article applies to:
  • Microsoft Office 97 for Windows

SUMMARY

This article contains a list of files that are installed with the Data Access Pack from the Value Pack of the Microsoft Office 97 compact disc. The Data Pack setup file (Dataacc.exe) is located in the Valupack\Dataacc folder.

The Data Access Pack is a setup program that makes it possible for you to install any or all of the ODBC drivers and built-in drivers. The built-in drivers are also referred to as IISAMs (Installable Indexed Sequential Access Method). You can use these drivers to access data outside your Microsoft Access or Microsoft Excel program (for example, to import Paradox data into a Microsoft Access database).

For additional information about why you might want to install the Data Access Pack, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q156835
   TITLE     : OFF97: Complete Setup Doesn't Install All ODBC Drivers

MORE INFORMATION

The Data Access Pack contains the following ODBC and built-in drivers (IISAMs): 

   SQL Server Driver               (version 2.65.021300)
   Microsoft Access Driver         (version 3.50.342800)
   Microsoft Excel Driver          (version 3.50.342800)
   Microsoft Text and HTML Driver  (version 3.50.342800)
   Microsoft xBase Driver          (version 3.50.342800)
   Microsoft FoxPro Driver         (version 5.00.003480)
   Paradox Driver                  (version 3.50.342800)
   Lotus 1-2-3 Driver              (version 3.50.342800)
   Microsoft Messaging Driver      (version 3.50.342800)
