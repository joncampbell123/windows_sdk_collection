OFF: Binder Section Not Selected With Right Mouse Click

 Last reviewed: December 3, 1996
Article ID: Q132183
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

In Microsoft Office Binder, when you click the right mouse button on a section in the section pane in your Binder, the section may not be selected, although the shortcut menu that appears applies to this section.

CAUSE

This behavior occurs because, in Office Binder, the right mouse button does not select a section in the section pane. If you hold the mouse pointer over the section, a border appears around the section name after a brief period of time.

When you click the right mouse button on a section in the section pane, a shortcut menu appears with the following commands available: 

   Add
   Delete
   Duplicate
   Rename
However, there is no indication as to which section the shortcut menu applies to.

Although this behavior may be confusing, it also provides a benefit in performance. When you click a section to activate the section, it takes time to switch to this section in your Binder. When you click the right mouse button on a section, the shortcut menu appears almost instantaneously, and you can perform the desired action very quickly without displaying the section.

Note that when you click Duplicate on the section shortcut menu, the Duplicate Section dialog box appears with a list of all of the available sections. Although the section on which you clicked the right mouse button appears selected in the list, you can click any section in the list to duplicate it.

REFERENCES

For more information about selecting sections in a Binder, click the Index tab in Binder Help, type the following text 

   selecting
and then double-click the selected text to go to the "Select a Document in a Binder" topic.
