OFFICE: Using Office Professional with Bookshelf on a Network

 Last reviewed: September 16, 1996
Article ID: Q124308
The information in this article applies to:
  • Microsoft Office for Windows, version 4.30c-CD

SUMMARY

The Microsoft Office 4.3c-CD jewelcase booklet includes the following information about installing Bookshelf: 

   To install Bookshelf in a shared location see, "For Network
   Administrators: Before You Install," in the Office OFREADME.HLP file.
However, this section in the Microsoft Office Readme Help file does not contain specific information about installing Bookshelf in a shared location. The following information discusses how to install Bookshelf in a shared location.

MORE INFORMATION

For Network Administrators: Before You Install Office with Bookshelf '94

This document provides information on using Microsoft Office Professional with Bookshelf '94 on a network. It includes information to help a network administrator set up Office with Bookshelf '94 on a network and information for people to set up Office with Bookshelf '94 on their workstation over the network.

IMPORTANT: To comply with the license agreement for Microsoft Office Professional with Bookshelf '94, you must have a license for each workstation that has Office with Bookshelf installed. If you are running Office with Bookshelf '94 over the network, then the total number of users accessing the software simultaneously over the network must be equal to the total number of licenses you have. There must also be a reasonable mechanism or process in place to restrict the number of users accessing the software simultaneously over the network to the number of licenses in place. Please note that if you have multiple copies of Office with Bookshelf '94 in CD-ROM form and you are counting these copies as licenses for your use of Office with Bookshelf '94 over the network, then you must archive these CD-ROM copies and not make any other use of them for as long as you are counting them as network copies. For more information on your licensing rights for using Office with Bookshelf '94, please refer to the end user license agreement which accompanies the software.

How to Use Office with Bookshelf '94 on a Network

WARNING: This installation procedure will allow you to have Bookshelf integration, but requires approximately 73 MB of space on the hard drive of each workstation if you choose the "Run From CD" option. Other installation options can reduce the hard drive space required, for example if you select the Custom/Complete option, and only select a few of the components. A normal Workstation installation of Microsoft Office does not install Bookshelf, but will require minimal space on the hard drive. Additionally, video and sound will be slower running Bookshelf from a network drive since it must travel over the network.

  1. Start Windows on the workstation.

  2. Switch to File Manager.

  3. Connect to the shared CD ROM drive.

  4. Copy all CD #1 and CD #2 files to the server. This will require 335 MB of disk space for CD #1, and 662 MB of disk space for CD #2.

    a. Create directories on your server called CD1 and CD2.

    b. Copy the contents of CD #1 to the CD1 directory.

    c. Copy the contents of CD #2 to the CD2 directory.

    d. Set the attributes of the files in the CD1 and CD2 directories to 

          Read Only.

  5. Follow the following directions to run setup to the local machine.

    a. Start Windows on the workstation.

    b. Switch to File Manager.

    c. Connect to the server directory that contains the files from CD #1.

    d. Run SETUP.EXE located in the CD #1 server directory. Follow the 

          directions to set up Microsoft Office Professional with Bookshelf 94.

  6. After Setup is completed, modify the BS94.INI file to point to the location on the server where the CD #2 files are located.

    a. Open the workstation's BS94.INI in notepad or some other text 

          editor.

    b. In the [Options] section, change DRIVE to equal the server directory 

          where the CD #2 files are located.

    c. Save the file.

  7. Modify the Office Compatible Demos icon in Program Manager to point to the demo location on the server.

    a. Highlight Office Compatible Demos icon in Program Manager.

    b. Choose File, Properties.

    c. Change the command line to point to the server directory where the 

          CD #2 files are located.

NOTE: You must modify the .INI for Office Professional with Bookshelf to correctly locate all of the files.
