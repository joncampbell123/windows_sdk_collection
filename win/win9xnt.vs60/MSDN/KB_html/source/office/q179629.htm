OFF98: How to Remove Selected Office Components (Part 1 of 2)

 Last reviewed: February 2, 1998
Article ID: Q179629
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SUMMARY

Unlike the installation program for earlier versions of Microsoft Office, the installation program for Microsoft Office 98 Macintosh Edition does not allow you to add or remove selected Office components. To do this in Microsoft Office 98 Macintosh Edition, move the files for the component to the Trash. This article lists the files that are installed with each Office component.

MORE INFORMATION

The following tables list the files that are installed with each Office component. Each table lists the file names, the folder locations in which the files are installed, and whether the files are included in an Easy Install. To remove a particular Office component, move the files for that component to the Trash.

NOTE: Files marked with an asterisk (*) are installed by more than one Office component. Deleting these files affects the operation of other Office programs.

For additional information on removing selected Office components, please see the following article in the Microsoft Knowledge Base:
