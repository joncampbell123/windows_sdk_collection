Can't Run Quick Preview or Examples And Demos After Setup

 Last reviewed: November 17, 1997
Article ID: Q125431
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2
  • Microsoft Excel for the Macintosh, version 5.0
  • Microsoft Word for the Macintosh, version 6.0
  • Microsoft PowerPoint for the Macintosh, version 4.0

SYMPTOMS

After you install one of the applications listed above, one of the following symptoms may occur:

Microsoft Excel

In Microsoft Excel, when you choose Examples And Demos or Quick Preview from the Help menu, a dialog box appears with the message "Cannot find Examples And Demos" or "Cannot find Quick Preview," and you are requested to select the file from a folder on your hard disk drive.

Microsoft Word

In Microsoft Word, when you choose Examples And Demos or Quick Preview from the Help menu, a dialog box appears with the message "Please locate Examples and Demos" or "Please locate Quick Preview," and a list of folders on you hard drive is displayed. If you choose Cancel, one of the following error messages appears:

  • Word cannot start Examples and Demos

    -or-

  • Word cannot start the Quick Preview

PowerPoint

In Microsoft PowerPoint, when you choose Quick Preview from the Help menu, you receive the following error message: 

   PowerPoint could not locate or start the Quick Preview. You may not have
   enough memory, or you may need to install it using PowerPoint Setup.

CAUSE

This behavior occurs if you perform a Typical or Minimum installation of Microsoft Office, or of one of the Microsoft Office applications.

Examples And Demos is included with Microsoft Excel and Word as part of the Help system. Quick Preview is included with Microsoft Excel, Word, and PowerPoint. However, neither Quick Preview nor Examples And Demos is installed if you choose the Typical or Minimum installation options.

Note that the Examples And Demos command (in Microsoft Excel and Word) and the Quick Preview command are available on the Help menu, even if the corresponding files are not installed.

WORKAROUNDS

To install Examples And Demos or Quick Preview after you have installed Microsoft Office or one of the Microsoft Office applications, do the following:

  1. Run the Microsoft Office or Microsoft Office application Setup program again in maintenance mode.

  2. In the Components Not Yet Installed list, click the arrow next to Microsoft Excel, Microsoft Word or Microsoft PowerPoint to display the installation options.

  3. Clear any items that you do NOT want to install. Click the arrow next to Online Help and Lessons (Microsoft Excel), Online Help (Microsoft Word) or Help, Quick Preview and... (Microsoft PowerPoint)

  4. Clear any items that you do NOT want to install. Make sure that the appropriate Quick Preview or Examples And Demos item is selected.

    Note that when you select Examples and Demos under Microsoft Word, Quick Preview is also installed. There is no separate Quick Preview option when you install Microsoft Word.

  5. Choose Continue.
