OFF95: Clicking CD Player Button on Shortcut Bar Restarts CD

 Last reviewed: March 27, 1997
Article ID: Q131564
7.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

If you add a button to the Office Shortcut Bar for the Windows CD Player application, CDPLAYER.EXE, and you click this button to restore the CD Player application while an audio CD is in the CD player, the audio CD is restarted.

CAUSE

This behavior occurs if the CD Player is running when you click the CD Player button on the Office Shortcut Bar. This behavior occurs because the shortcut created for CDPLAYER.EXE does not activate the CD Player application if it is already running. Instead, the CD Player starts again, causing the audio CD to start playing again from the beginning.

RESOLUTION

To avoid restarting your audio CD when you are running the Windows CD Player, use the Windows taskbar to switch to the CD Player instead of the Office Shortcut Bar.
