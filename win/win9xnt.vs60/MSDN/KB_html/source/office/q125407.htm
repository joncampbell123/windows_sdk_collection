Methods for Starting Examples And Demos or Quick Preview

 Last reviewed: November 17, 1997
Article ID: Q125407
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2
  • Microsoft Excel for the Macintosh, version 5.0
  • Microsoft Word for the Macintosh, version 6.0
  • Microsoft PowerPoint for the Macintosh, version 4.0

SUMMARY

The following information discusses the different ways that you can start the Examples And Demos for Microsoft Excel or Microsoft Word or the Quick Preview for Microsoft Excel, or Microsoft Word, or Microsoft Project.

To Start Examples and Demos or Quick Preview From Within the Application

To start Examples And Demos or Quick Preview from within Microsoft Excel, Word, or PowerPoint, choose the option from the Help menu in the specific application. Note that PowerPoint does not have Examples and Demos.

To Start Examples And Demos Directly from Help

To start Examples and Demos for Microsoft Excel or Word directly from Help, do the following:

  1. In the Finder, open the Microsoft Excel or Microsoft Word folder. Note that these folders are located inside the Microsoft Office folder if you have installed Microsoft Office.

  2. Double-click the MS Excel Help or Word Help icon.

    Note that when you double-click the MS Excel Help icon, Microsoft Excel starts, and the MS Excel Help file opens in Microsoft Help. When you double-click the Word Help icon, Microsoft Help starts, and the Word Help file opens.

  3. Choose Examples And Demos, and then complete the lesson that you want.

Note that you can also start Examples and Demos using Help from within Microsoft Excel or Word. To do this, use the following steps:

  1. In Microsoft Excel, choose Microsoft Excel Help from the Help menu, or in Word, choose Microsoft Word Help from the Help menu.

  2. In the Microsoft Excel Help Contents window, or the Word Help Contents window, choose Examples and Demos, and then complete the lesson that you want.

To Start Examples And Demos or Quick Preview Directly from the Finder

To start Examples and Demos or Quick Preview for Microsoft Excel, Word, or PowerPoint from the Finder, do the following:

  1. In the Finder, open the Microsoft Excel, Word, or PowerPoint folder. Note that these folders are located inside the Microsoft Office folder if you have installed Microsoft Office.

  2. Double-click the Tutorial folder.

  3. Double-click the Examples and Demos icon, or the Quick Preview icon.

    When you double-click the Examples And Demos icon or the Quick Preview icon, HyperCard starts and opens Examples and Demos or Quick Preview.

MORE INFORMATION

Examples And Demos is included with Microsoft Excel and Word as part of the online help system. Quick Preview is included with Microsoft Excel, Word, and PowerPoint.

Note that these files are not installed when you perform a Typical or Minimum installation of Microsoft Office or the Office applications.
