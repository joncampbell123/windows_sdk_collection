CG3: Sounds and Videos not Listed Under Clip Type

 Last reviewed: February 25, 1998
Article ID: Q159581
The information in this article applies to:
  • Microsoft Clip Gallery, version 3.0
  • Microsoft Office 97 for Windows

SYMPTOMS

If you want to use the Find feature in the Microsoft Clip Gallery version 3.0, only available Picture types will be displayed in the Clip Type list in the Find Clip dialog box. If you want to be able to use the Find feature for locating other clip types (Sounds and Videos), you will have to use the File Name Containing list in the Find Clip dialog box along with the appropriate file type extensions.

WORKAROUND

To use the Microsoft Clip Gallery version 3.0 Find feature to locate clips other than pictures, follow these steps.

How to Find Sounds in the Clip Gallery

  1. In Microsoft Clip Gallery, click the Sounds tab.

  2. Click Find.

  3. In the Find Clip dialog box, type "*.wav", (without the quotation marks) in the File Name Containing box.

  4. Click Find Now.

All clips with a .wav file extension in the Clip Gallery will be displayed in the Sounds tab.

To insert one of these clips into your document, click the desired sound, and then click Insert.

NOTE: You can also use the .mid file extension in Step 3 if you have imported sound files with this extension into the Clip Gallery.

How to Find Videos in the Clip Gallery

  1. In Microsoft Clip Gallery, click the Videos tab.

  2. Click Find.

  3. In the Find Clip dialog box type "*.avi" (without the quotation marks) in the File Name Containing box.

  4. Click Find Now.

All of the clips with a .avi file extension in the Clip Gallery will be displayed in the Videos tab.

To insert one of these clips into your document, click the desired video, and then click Insert.

MORE INFORMATION

In the Microsoft Office 97 programs, you can insert clip art, pictures, sounds, and video clips into your documents. Many clips are installed by Microsoft Office. You can also add your own clips from other programs or from Microsoft sites on the World Wide Web.
