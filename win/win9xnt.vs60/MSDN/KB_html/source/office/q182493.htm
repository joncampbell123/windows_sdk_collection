OFF98: After Dark Screen Saver Is Not Activated If MOM Is Active

 Last reviewed: March 16, 1998
Article ID: Q182493
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

Your After Dark screen saver does not appear after your computer has been idle for the period of time you have set in the After Dark control panel. Also, the After Dark screen saver does not appear when you use the Sleep Hot Key or when you use the sleep corners that you have set in the After Dark control panel.

CAUSE

This problem occurs if the following conditions are true:

  • You are running After Dark version 4.0.3 

        -and-
  • You are running the Microsoft Office Manager. 

        -and-
  • You are running Mac OS 8.0 or 8.1.

WORKAROUND

To work around this behavior, turn off the Microsoft Office Manager (MOM) menu. To do this, follow these steps:

  1. Point to Control Panels on the Apple menu, and then click Microsoft Office Manager.

  2. Click to clear the Enable Office Manager Menu check box. Click OK when the alert message appears.

  3. Close the Microsoft Office Manager Control Panel.

  4. Click Restart on the Special menu to restart your computer.

After your computer is restarted, the Microsoft Office Manager menu no longer appears. However, you can still use the QuickSwitch key combination (usually COMMAND+TAB) to switch between running programs.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.

MORE INFORMATION

There are several alternatives to using the Microsoft Office Manager that make it easy to start the Office programs. The alternatives include the following:

  • Place aliases for the Office programs on your Apple menu.

    You can use the aliases to start your Office programs. For example, to place an alias for Excel on your Apple menu, follow these steps: 

        1. Open the Microsoft Office 98 Folder on your hard disk.


        2. Click to select Microsoft Excel.


        3. On the File menu, click Make Alias.


           A new file called "Microsoft Excel Alias" is created.


        4. Drag this file to your System Folder:Apple Menu Items folder.
  • Use the Recent Applications menu to start the Office programs.

    The Recent Applications menu appears as a submenu on the Apple menu. The Recent Applications menu is useful if you use the Office programs frequently relative to the other programs on your computer.
