Incorrect Name for Microsoft Query in Online Documentation

 Last reviewed: March 13, 1998
Article ID: Q130284
4.2.1-CD MACINTOSH kbother kbdocerr kbtool

The information in this article applies to:

  • Microsoft Office for the Macintosh, version 4.2.1-CD

SYMPTOMS

In the Microsoft Office online documentation, when you use the Index, the Which Index list contains the incorrect index "Microsoft Excel Query User's Guide."

Additionally, when you view a Microsoft Query topic in the Microsoft Office Documentation file, the window title displays Microsoft Excel Query.

This name is incorrect. The proper name for this application is Microsoft Query. Microsoft Query is a standalone application that ships with Microsoft Excel; that is, you do not need to run Microsoft Excel in order to use Microsoft Query.

Note that the information in this file correctly refers to Microsoft Query.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
