OFF: Look In List in Advanced Find Doesn't Display Folder

 Last reviewed: July 14, 1997
Article ID: Q135790
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Project 97 for Windows
  • Microsoft Access 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Project for Windows 95, version 4.1
  • Microsoft Access, version 7.0

SYMPTOMS

In the Advanced Find dialog box, the default file location folder (or the current folder), may not be displayed in the Look In drop-down list.

CAUSE

The Advanced Find dialog box is displayed when you choose the Advanced button in the Open dialog box. The Look In list, by default, displays the default file location for your program (or the current folder if you have opened a file in a folder other than the default file location). However, when you click another choice in the Look In list, the default file folder (or current folder) no longer appears in the Look In list. Additionally, you cannot "drill down" or select a subfolder using the Look In list in the Advanced Find dialog box.

WORKAROUND

To display the default file location or the current folder again in the Look In list in the Advanced Find dialog box, click Cancel in the Advanced Find dialog box, and click Advanced again in the Open dialog box.

MORE INFORMATION

You can use the Advanced Find dialog box to specify more detailed criteria when you search for files. For example, you can use Advanced Find to locate files by title, name of author, subject, and keywords. You can also create your own custom properties using the Advanced Find dialog box.

REFERENCES

For more information about Advanced Find, choose Microsoft Excel Help Topics from the Help menu, select the Find tab and search on: 

   Advanced Find
