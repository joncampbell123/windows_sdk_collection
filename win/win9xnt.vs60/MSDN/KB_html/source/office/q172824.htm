OFF95: MS Office Setup Cannot Restart Microsoft Windows under NT

 Last reviewed: February 16, 1998
Article ID: Q172824
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

After you finish running Setup for Microsoft Office 7.0 and click Restart Windows in Setup dialog box, nothing happens. When you click Cancel, the following message appears: 

   Setup was not completed successfully

CAUSE

This problem occurs if the following conditions are true:

  • You are running the Microsoft Office 7.0 Setup program.

    -and-

  • You are running Microsoft Windows NT 4.0.

    -and-

  • You click Restart Windows when the Microsoft Office 7.0 Setup dialog box is displayed.

WORKAROUND

To work around this problem, manually restart Windows NT. To manually restart Windows, follow these steps:

  1. In the Microsoft Office 7.0 Setup dialog box that prompts you to restart Windows, click Cancel.

    You receive the following error message: 

          Setup was not completed successfully

  2. After the Office 7.0 Setup closes, click Shut Down on the Start menu.

  3. In the Shut Down Windows dialog box, click Restart Windows, and then click Yes.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Office for Windows 95, version 7.0. This problem was corrected in Microsoft Office 97 for Windows.
