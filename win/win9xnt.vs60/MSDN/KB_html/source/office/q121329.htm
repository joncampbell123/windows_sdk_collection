AppleShare 4.0 Error After Excel 5.0 or Office 4.2 Installed

 Last reviewed: November 17, 1997
Article ID: Q121329
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2
  • Microsoft Excel for the Macintosh, version 5.0
  • Microsoft Excel for the Power Macintosh, version 5.0

SUMMARY

After you install Microsoft Excel 5.0 or Microsoft Office 4.2 on a computer that has AppleShare 4.0, you may receive the following error message: 

   AppleShare 4.0 is incompatible with the installed version of the Shared
   Library Manager

CAUSE

Microsoft Excel 5.0 and Microsoft Office 4.2 ship with version 1.1.2 of the Shared Library Manager. You receive the error message above because AppleShare 4.0 comes with a version of the Shared Library Manager that is earlier than version 1.1.2.

NOTE: This version of the Shared Library Manager, from Apple, was the current version available at the time that Microsoft Excel and Microsoft Office were released.

WORKAROUNDS

To work around this problem, do either of the following:

  • Upgrade to AppleShare 4.01 or later

    -or-

  • Install the earlier version of the Shared Library Manager.

NOTE: If you upgrade AppleShare to version 4.01, you maintain compatibility with both AppleShare and Microsoft Excel. If you install the earlier version of the Shared Library Manager, AppleShare 4.0 will work correctly, but Microsoft Excel and Microsoft Query may produce unexpected results.

Microsoft Excel and Microsoft Query have not been tested with versions of the Shared Library Manager earlier than version 1.1.2.
