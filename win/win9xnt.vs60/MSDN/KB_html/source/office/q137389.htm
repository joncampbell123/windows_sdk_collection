OFF95: Template Files Installed on Incorrect Drive

 Last reviewed: January 23, 1998
Article ID: Q137389
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you install Microsoft Office or Microsoft Excel, if you change the drive to which you are installing the program during Setup, the template files may be installed to the incorrect drive.

CAUSE

This problem occurs under the following conditions:

Using Change Folder After Selecting Destination

During the Microsoft Excel Setup program, if you use the Change Folder button in the dialog box that contains the installation options (such as Typical, Compact, Custom) to change the drive to which you are installing the program, the template files are installed to the original drive. For example, if you run the Microsoft Excel Setup program, and you accept the default location, C:\Msoffice, and you then use the Change Folder button in the installation options dialog box to change the destination to a different drive, the spreadsheet template files are installed to the original location (C:\Msoffice) while the rest of the Microsoft Excel files are installed to the new location.

During the Microsoft Office Setup program, if you use the Change Folder button in the Custom Setup dialog box to change the location of one of the Office components to a different drive, the template files for that component are installed to the original location.

Not Enough Disk Space

If you attempt to install either Microsoft Excel or Office to a drive that does not have enough available disk space, you may receive a dialog box informing you of this fact. If you click the Change Options button in this dialog box, a different dialog box appears, allowing you to change your Setup options, and/or to install the program to a different drive. However, if you change the destination drive, the template files for the program may be installed to the original drive instead of the new drive location.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
