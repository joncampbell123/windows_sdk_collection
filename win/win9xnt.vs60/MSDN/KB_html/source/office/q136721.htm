OFF95: Files Installed with Typical (w/Data Map) (1 of 2)

 Last reviewed: March 10, 1997
Article ID: Q136721
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The following is a partial list of the folders and files installed when you perform a Typical installation of Microsoft Office and you choose to install Data Map. This list contains the files installed to the Microsoft Office folder (Msoffice by default), the Clipart, Excel, Office, PowerPnt, Schedule, and Winword folders inside the Microsoft Office folder, and the files installed to the Windows folder.

To view the additional files installed when you perform a Typical installation of Microsoft Office (with Data Map), please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q136719
   TITLE     : OFF95: Files Installed with Typical (w/Data Map) (2 of 2)

MORE INFORMATION

You can use this list to determine which files are installed by Microsoft Office and where they are installed, so that you can remove them if necessary.
