OFF95: Can't find Qoffice.hlp After Removing Other Program

 Last reviewed: March 27, 1997
Article ID: Q135477
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 4.1
  • Microsoft Access, version 7.0

SYMPTOMS

When you use Help in a dialog box in one of your Microsoft applications, you may receive the following error message: 

   Cannot find the QOFFICE.HLP file. Do you want to try to find this file
   yourself?
If you click No, you receive the following error message: 

   Cannot find the QOFFICE.HLP file. Check to see that the file exists on
   your disk. If it doesn't, you need to reinstall it.

CAUSE

This problem occurs when you run the Setup program for one of the applications listed above in maintenance mode and you click Remove All.

For example, if you install Microsoft PowerPoint, and then install Microsoft Excel, if you run the Microsoft PowerPoint Setup program in maintenance mode, and you click Remove All, you can no longer use Help in a dialog box in Microsoft Excel.

This problem occurs because when you run the Setup program for these Microsoft programs in maintenance mode and you remove the entire application, the Setup program removes the Qoffice.hlp file, even though this Help file is an Office shared application, and is used in other programs.

The Qoffice.hlp file is used to display Help in the following dialog boxes in the Microsoft applications listed above: 

   Open
   Save As
   Properties
   About
To use Help in one of these dialog boxes, you can press F1, or you can click the Help button (the question mark button that appears in the right hand corner of the dialog box) and then click the item that you want help on.

The Qoffice.hlp file is not the Office Help file that is available on the Office Shortcut Bar.

RESOLUTION

To reinstall the Qoffice.hlp file, run the Setup program for your application, and click the Reinstall button.
