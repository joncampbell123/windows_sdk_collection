 Last reviewed: March 10, 1997
Article ID: Q136716
7.00 WINDOWS kbsetup kbnetwork

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

The following is a partial list of the folders and files installed when you perform an administrative installation (Setup /a) of Microsoft Office to a network server. This list contains the files that are installed to the Os folder in the Microsoft Office folder (Msoffice\Os by default) on the network server.

To view the additional files installed when you perform an administrative installation of Office, please see the following article(s) in the Microsoft Knowledge Base:
