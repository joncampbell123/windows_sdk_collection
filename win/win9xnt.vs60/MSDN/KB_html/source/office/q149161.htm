"Invalid Page Fault" with Small Business Pack Database

 Last reviewed: March 27, 1997
Article ID: Q149161
7.00 WINDOWS kbusage kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Access versions 7.0

SYMPTOMS

When you select a record on the Invoice form of the Business Transactions Database included with the Office Small Business Pack, you may receive the following error message: 

   MSACCESS caused an invalid page fault in module VBA232.DLL at
   0137:0fa942fa
This error occurs on some computers using any of the following buttons (available on the Small Business Pack custom toolbar): 

   Next Record
   Last Record
   Help (the error occurs when help is closed)

CAUSE

This error occurs on some computers if the Print button on the Small Business Pack custom toolbar is used before the Next Record, Last Record, or if Help buttons are selected.

RESOLUTION

To work around the error, use one of the following methods:

Method 1: After you use the Print button, close and re-open the form 

          before you use the Next, Last, or Help buttons on the custom
          toolbar.
Method 2: Reinstall the applications: 

          a. Uninstall both Microsoft Office for Windows 95 (Standard or
             Professional) and Microsoft Access 7.0.

          b. Rename or Delete VBA232.DLL located in the \Windows\System
             folder.

          c. Re-install Microsoft Access 7.0 and Microsoft Office for
             Windows 95.

             IMPORTANT: If you have Access as a separate package, install
             Access first, then install Office.

MORE INFORMATION

This error occurs on some computers with the following versions of VBA232.DLL: 

   Version          Date
   ---------------------

   2.0.5524      9/28/95
   2.0.6108       3/8/96
The Microsoft Office Small Business Pack for Office 95 is an add-on product for Microsoft Office Professional for Windows 95. The Small Business Pack includes a Business Database for use with Microsoft Access for Windows 95.

For more information about the Office Small Business Pack, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q139721
   TITLE     : What Is the Small Business Pack for Microsoft Office 95

REFERENCES

For more information about the Microsoft Office Small Business Pack, click the Help icon on the Small Business Pack toolbar which is available from the Office Shortcut Bar.
