Office: Find File May Not Run From Command Line

 Last reviewed: September 16, 1996
Article ID: Q117555
The information in this article applies to:
  • Microsoft Office for Windows, version 4.0, 4.2, and 4.3

SYMPTOMS

When you attempt to use the Find File program by typing the command line parameter for Microsoft Office Manager (MOM), as in the following example 

   c:\msoffice\msoffice.exe findfile
the Find File program will not run if MOM is currently running.

You will not receive an alert message warning you of this problem.

RESOLUTION

To run Find File by typing in the command at the command line, you must exit MOM first.
