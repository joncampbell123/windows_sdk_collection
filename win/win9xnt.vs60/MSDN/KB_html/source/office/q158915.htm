OFF97: Remove All Doesn't Remove Shortcut Bar .Tmp Files

 Last reviewed: March 12, 1998
Article ID: Q158915
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

If you run the Microsoft Office 97 Setup program in maintenance mode and click Remove All, the Setup program does not remove a number of .tmp files in the Program Files\Microsoft Office\Office\Shortcut Bar folder.

CAUSE

These files are temporary files that are created by the Microsoft Office 97 Shortcut Bar. The Setup program does not remove the files because their names vary.

RESOLUTION

After you remove Microsoft Office 97, you can safely delete .tmp files in the Program Files\Microsoft Office\Office\Shortcut Bar folder.

STATUS

This behavior is by design of the Microsoft Office 97 Setup program.

MORE INFORMATION

When you use the Remove All option in the Microsoft Office 97 Setup program, the program attempts to remove all files that are originally installed with Microsoft Office 97. You can use the Microsoft Office Upgrade Wizard to remove files that are not removed by the Setup program.

When you run the Microsoft Office 97 Shortcut Bar, a number of .tmp files are created in the Program Files\Microsoft Office\Office\Shortcut Bar folder on your hard drive. The names of the files resemble the following: 

   Acc1130.tmp
   Des1130.tmp
   Fav1130.tmp
   OffA1D3.tmp
   Pro6053.tmp
   Qui3344.tmp
These files are not removed when you click Remove All, and they are not removed if you use the Microsoft Office Upgrade Wizard to remove unnecessary files from your computer. If you want to remove these files, you must delete them manually.
