OFF97: Blank.pot Appears in New Office Document Dialog Box

 Last reviewed: March 12, 1998
Article ID: Q161350

The information in this article applies to:

  • Microsoft Office 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

In the New Office Document dialog box, you may see a Microsoft PowerPoint presentation template named "Blank.pot" in addition to the template named "Blank Presentation."

NOTE: If you are hiding MS-DOS file name extensions, the name of the additional template appears as "Blank."

CAUSE

This problem occurs after you install Microsoft Office using the "Run from Network Server" installation option.

RESOLUTION

To resolve this problem, delete one of the template files.

NOTE: The Blank.pot template file is installed when you perform an administrative installation of Microsoft Office on a network server. You must check with the network administrator before you delete this file (assuming you have permission to delete files on the server).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The contents of the template files are the same but the files are installed in different locations. Blank.pot is installed in the Template folder in the folder that contains the administrative installation of Microsoft Office. The Blank Presentation template is installed, by default, to the local Program Files\Microsoft Office\Templates folder.

REFERENCES

For more information about creating a new Microsoft Office document, click the Index tab in Microsoft Office Help, type the following text 

   creating, files
and then double-click the selected text to go to the "Start a new file from the Office Shortcut Bar" topic.
