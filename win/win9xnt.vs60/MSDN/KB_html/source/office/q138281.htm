Office Setup Remove All Does Not Remove Registry Tree

 Last reviewed: March 12, 1998
Article ID: Q138281
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

When you run Setup and select the Remove All option in Microsoft Office 97 or Microsoft Office version 7.0, the entries may not be removed from the Microsoft Windows 95 registry. Instead, the data values are changed to Value not set.

CAUSE

This behavior is by design of the Microsoft Office Setup program. The Microsoft Office components are unregistered; however, note that the registry tree entries remain intact.
