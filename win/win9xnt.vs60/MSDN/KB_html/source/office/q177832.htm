OFF97: Personal Web Server Is Not Included in the Value Pack

 Last reviewed: December 9, 1997
Article ID: Q177832
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

The "More Cool Stuff" template that is included in Microsoft Office 97 for Windows states that the Personal Web Server for Windows is included in the Microsoft Office 97 Value Pack. This information is incorrect.

RESOLUTION

Although the Personal Web Server for Windows is not included in the Value Pack, you can download it from the Microsoft Web site. To do this, use your Web browser to go to the following address: 

   http://www.microsoft.com/ie/download/
The link to download the Personal Web Server is located near the bottom of the page.

NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To view the incorrect information, follow these steps:

  1. On the Start menu, click New Office Document.

  2. Click the Web Pages tab. Click More Cool Stuff, and then click OK.

The More Cool Stuff template is opened in Microsoft Word 97 for Windows. It contains the following text: 

   More Cool Stuff

   Microsoft Word provides additional Web related tools and documents. You
   can download free additional Web Page Wizard content documents and
   visual styles from the Microsoft Word Web Site.

   You may also find the following files useful. If you installed
   Microsoft Word from a CD, insert the disk, open the ValuPack folder and
   double click on the appropriate file.

   AXPlugin   A plugin for Netscape that makes it possible to open
              Microsoft Office documents inside a Netscape browser.

   WebPost    A wizard that takes you through the steps of posting a
              document to the Web.

   PwebSrv    A program that runs a web server on your Windows 95 machine
              or Windows NT Workstation.

   IExplore   Internet Explorer 3.0
The PwebSrv file is not included in the ValuPack folder. The other three files (AXPlugin, WebPost, and IExplore) are included in the Value Pack.
