Office 4.2.1-CD Setup Hangs Copying Excel Resources

 Last reviewed: March 13, 1998
Article ID: Q79045
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2.1-CD

SUMMARY

When you run the Microsoft Office version 4.2.1 Setup from CD, the Setup program may appear to hang while Setup is copying Microsoft Excel resources.

CAUSE

This problem occurs if you have a previous installation of Microsoft Office on your Macintosh (specifically, the Microsoft Office Manager application) and you choose to install both Microsoft Excel and Microsoft Office Manager from the Microsoft Office 4.2.1-CD. This problem occurs, for example, if you choose to perform a Typical, Complete, or Minimum installation of Microsoft Office.

Note that this problem does not occur if you choose the Microsoft Office for the Power Macintosh installation option; that is, the problem only occurs if you choose the Microsoft Office for the Macintosh, or Microsoft Office for the Macintosh or Power Macintosh option.

WORKAROUNDS

To work around this problem, use any of the following methods.

Method 1

If you have a previous version of Microsoft Office installed on your Macintosh, do the following:

  1. In Finder, locate the Office Manager Preferences file (in the Preferences folder in your System Folder).

  2. Drag the Office Manager Preferences file to the Trash.

  3. Run the Setup program again from the Microsoft Office 4.2.1 CD.

The Office Manager Preferences file is not removed when you run Microsoft Office Setup in maintenance mode and choose the Remove All button.

NOTE: If the above procedure does not correct this problem, use the following method.

Method 2

Install Microsoft Office without the Microsoft Excel application, and then install Microsoft Excel separately by doing the following:

  1. Run Setup from the Microsoft Office 4.2.1 CD.

  2. Choose the Custom/Complete button.

  3. In the Options list, clear the Microsoft Excel check box.

  4. Clear any other components that you do not want to install, and click Continue to continue with the Setup.

To add Microsoft Excel to your Office installation, do the following:

  1. Run Setup again from the Microsoft Office 4.2.1 CD.

  2. In the Setup dialog box (in maintenance mode), choose the Add/Remove button.

  3. In the Components Not Yet Installed list, select the Microsoft Excel check box, and choose Setup.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Office for the Macintosh, version 4.2.1a-CD.
