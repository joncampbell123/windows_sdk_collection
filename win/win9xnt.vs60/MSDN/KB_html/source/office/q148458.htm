How to Share AutoCorrect Entries with Other Users

 Last reviewed: March 25, 1997
Article ID: Q148458

The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0

IMPORTANT: This article contains information about editing the registry. Before you edit the registry, you should first make a backup copy of the registry files (System.dat and User.dat).

SUMMARY

This article outlines the steps to use when you want to share the Office AutoCorrect list with other users. You can do this by modifying the path to the AutoCorrect list in the Windows registry.

MORE INFORMATION

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows 95. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.

For information about how to edit the registry, view the Changing Keys And Values online Help topic in Registry Editor (Regedit.exe). Note that you should make a backup copy of the registry files (System.dat and User.dat) before you edit the registry.

Microsoft Office installs a file that contains a default AutoCorrect list, Msoffice.acl, in the Windows folder. When the Office program saves the personal list for a user, it is stored as <Username>xxx.acl. To share your .acl file, use the following steps:

  1. Copy the .acl file from your folder to the Windows folder on the other user's computer.

  2. Run the Registry Editor and modify the Windows registry entry to point to this file. The value is located in the following key: 

          HKEY_CURRENT_USER\Software\Microsoft\Microsoft
      Office\95\AutoCorrect\List
