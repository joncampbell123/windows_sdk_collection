INTRODUCTION

The table in this Application Note lists the files that are installed with each installation option for Microsoft Office version 4.2, Microsoft Word version 6.0, Microsoft Excel version 5.0, and Microsoft PowerPoint(R) version 4.0. In the table, the columns of information are organized under the following headings:

  • File--Lists each file installed by the Microsoft Office Setup program or by the Setup program for the individual Microsoft Office applications
  • Size--Shows the size of the file
  • Location--Shows the location of the file when installed
  • Office Install--Lists the installation options with which the particular Microsoft Office 4.2 file is installed
  • Word Install--Lists the installation options with which the particular Microsoft Word 6.0 file is installed
  • Microsoft Excel Install--Lists the installation options with which the particular Microsoft Excel 5.0 file is installed
  • PowerPoint Install--Lists the installation options with which the particular Microsoft PowerPoint 4.0 file is installed

    NOTE: The Install columns contain abbreviations for the various installation options. In these columns, C means complete, T means typical, and M means minimum.

If an Install column is blank, that particular file is not installed for that application.
