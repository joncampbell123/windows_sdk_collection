OFF97: List of Files Installed with Network Installation Wizard

 Last reviewed: November 5, 1997
Article ID: Q176261
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office 97 Network Installation Wizard, version 2.1

SUMMARY

This article contains a list of the files that are installed when you install the Microsoft Office 97 Network Installation Wizard, version 2.1.

MORE INFORMATION

Files Installed by the Network Installation Wizard

The files that are installed by Network Installation Wizard are listed in the following tables.
