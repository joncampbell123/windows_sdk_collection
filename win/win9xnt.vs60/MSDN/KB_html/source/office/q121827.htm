OFFICE: Error Running the Office Compatible Demos

 Last reviewed: September 16, 1996
Article ID: Q121827
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SYMPTOMS

If you run the Office Compatible Demos from the icon in Program Manager, and you have Disk 1 of the Microsoft Office CD set in your CD ROM drive, the only item displayed in the Office Compatible Demos list is README, and the Brief Information displayed for the demo is the following: 

                                  README:

   Error: You have inserted Microsoft Office CD Disk #1. To run the
   Microsoft Office Compatible demos, you must insert Microsoft Office CD
   Disk #2.
Additionally, if you choose the Start Demo button while README is selected in the Office Compatible Demos list, you receive the following error message: 

   Error executing product demonstration for:
   "README:"

CAUSE

Microsoft Office 4.3c-CD ships with two CDs. The Disk 1 CD is the disk that you use to install Microsoft Office. The Disk 2 CD contains a collection of application demos. The Disk 2 CD must be in the CD-ROM drive when you run the Office Compatible Demos.

The icon for the Office Compatible Demos is added to the Microsoft Office group in Program Manager, and it contains the path information to the BONUS\DEMO.OC subdirectory. The Disk 1 CD also contains a BONUS\DEMO.OC subdirectory; however, the subdirectory on this disk does not contain the individual demo files.

WORKAROUNDS

To run the Office Compatible demos, you must insert the Microsoft Office Disk 2 CD in the CD-ROM drive before you double-click the Office Compatible demos icon in Program Manager.

If you run the demos application, and you receive the message above, you must close the demo application, insert the Disk 2 CD and then start the demos application again to update the list of the demos on the Disk 2 CD.
