How to Insert a Separator Line on the MOM Menu

 Last reviewed: November 17, 1997
Article ID: Q164552
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

By default, the Microsoft Office Manager (MOM) menu has three separator lines. There is no built-in command that allows you to add a new separator line. This article explains how to add additional separator lines to the MOM menu.

MORE INFORMATION

To add a separator line to the MOM menu, use the following steps:

  1. Create and save a text document on the hard disk. Save the file with the following name

    ----(Separator)---

    and close the document.

    NOTE: You can use any text editor program, for example, SimpleText, to create the text document.

  2. On the MOM menu, click Customize.

  3. Click New.

  4. In the dialog box that appears, locate and click the text document you created in step 1.

  5. Click Add.

  6. Click the UP ARROW or the DOWN ARROW keys in the Microsoft Office Manager dialog box to move the separator line.

  7. Click to clear the Include Menu Icons check box.

  8. Close the Microsoft Office Manager dialog box.

The separator line you created appears on the MOM menu.

NOTE: Step 7 is required for the procedure to work. If you do not clear the Include Menu Icons check box, the text file appears as an ordinary file and is opened if you click it.

REFERENCES

For more information about Customizing the Microsoft Office Manager menu, click Office Help on the MOM menu, click Search, type the following text 

   adding files to the menu
and then double-click the selected text to go to the "Customize Microsoft Office Manager" topic.
