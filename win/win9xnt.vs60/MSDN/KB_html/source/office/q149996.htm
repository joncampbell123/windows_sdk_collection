OFF95: Default Configurations for the Office Shortcut Bar

 Last reviewed: March 27, 1997
Article ID: Q149996
7.00 7.00a 7.00b WINDOWS kbsetup kbtool kbenv

The information in this article applies to:

  • Microsoft Office for Windows 95, versions 7.0, 7.0a, 7.0b
  • Microsoft Office Professional for Windows 95, versions 7.0, 7.0a, 7.0b

SUMMARY

How the Microsoft Office Setup program configures the Office Shortcut Bar depends on the platform you use (Windows 95 or Windows NT) and the computer hardware you have. This article provides information about the different default settings for the Office Shortcut Bar.

MORE INFORMATION

The following table shows the default settings for the Office Shortcut Bar based on the platform and screen resolution of the computer you use. 

Office Shortcut Bar   Windows 95    Windows 95    Windows NT    Windows NT
setting               640x480       800x600+      640x480       800x600+


Large Buttons         off           off           off           off
Show ToolTips         on            on            on            on
Always On Top         on            on            on            on
Auto Hide             off           off           off           off
Auto Fit (title bar)  on            off           on            on
Docked Location       top           right         top           top
Smooth Fill           off           off           off           off
Show Title Screen     off           off           off           off
Standard Color        off           off           off           off
Note that the Show ToolTips settings for Microsoft Windows NT is correct for Microsoft Office version 7.0. If you install either Microsoft Office version 7.0a or 7.0b under Microsoft Windows NT, this setting will default to off.

Also, when you restart your computer for the first time after you install Microsoft Office (and the Office Shortcut Bar), the title screen for Office is displayed. However, the Show Title Screen setting for the Office Shortcut Bar is set to off, and you should not see the title screen again, unless you set this option to on.

The following table shows more default settings for the Office Shortcut Bar based on the platform and the relative speed of the computer you use.

Office Shortcut Bar 

setting               "Fast" computer   "Slow" computer   16 colors


Animate Toolbar       on                 off              n/a
Use Gradient Fill     on                 off              off
Sound                 on                 off              with sound card
Note that a "slow" computer under Windows 95 or Windows NT will have a 386 CPU, low memory, or a slow display card.
