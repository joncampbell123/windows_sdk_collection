OFF97: What Is Included in the Product Box?

 Last reviewed: March 13, 1998
Article ID: Q159778
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Home Essentials 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft Access 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Outlook 97 for Windows
  • Microsoft Small Business Financial Manager 97 for Windows

SUMMARY

This article presents an overview of what is included in the North American product boxes for Microsoft Office 97 and standalone Microsoft Office programs, such as Microsoft Excel 97.

MORE INFORMATION

The overview is separated into three sections:

  • Product Information (what is contained in the product box)
  • Supplemental Offers (extra offers designed to enhance your programs)
  • ValuPack Content (what is contained in the ValuPack)

SUPPLEMENTAL OFFERS

Microsoft Word:

  • Word 97 Viewer for Windows 95 - freely downloadable or order for shipping and handling charge
  • Word Viewer for Windows 3.1 - freely downloadable or order for shipping and handling charge
  • Word Legal Resource Kit for Office 97 - freely downloadable or order for shipping and handling charge
  • Word for Windows 95 and 6.0 Converter Kit - freely downloadable or order for shipping and handling charge
  • Word 6.0 for Macintosh Converter Kit - freely downloadable or order for shipping and handling charge
  • 3.5-inch Disk Set - free of charge

Microsoft Home Essentials:
  • The same offers that apply to Microsoft Word also apply to Microsoft Home Essentials.

Microsoft Access:
  • Microsoft Access Upsizing Tools
  • Microsoft Access Partial Replica Wizard
  • 3.5-inch Disk Set - free of charge

Microsoft Outlook:
  • Cross-sell for Back Office and Exchange Server
  • DataViz third-party offer
  • Timex DataLink Watch third-party offer
  • 3.5-inch Disk Set - free of charge

Microsoft Office, Standard Edition:
  • All offers that apply to Microsoft Word and Microsoft Outlook also apply to Microsoft Office, Standard Edition.
  • "In & Out of the Classroom with Microsoft Office 97" training guide - text version is freely downloadable or order for $14.95 + shipping and handling
  • 3.5-inch Disk Set - Free of charge
  • Information on additional Microsoft products (Microsoft Publisher, Microsoft FrontPage, Microsoft Team Manager, Microsoft IntelliMouse, Microsoft Exchange Server and Microsoft Project for Windows 95, Microsoft BackOffice and Microsoft Office Compatible)

Microsoft Office, Professional Edition:
  • All offers that apply to Microsoft Office, Standard Edition, also apply to Microsoft Office, Professional Edition.
  • "Building Applications with Microsoft Access" book - shipping and handling charge
  • 3.5-inch Disk Set - Free of charge
  • Information on additional Microsoft products (Microsoft Publisher, Microsoft FrontPage, Microsoft Team Manager, Microsoft IntelliMouse, Developer Edition, Microsoft Exchange Server and Microsoft Project for Windows 95)
  • Bookshelf Upsell Offer

Microsoft Office, Small Business Edition:
  • All offers that apply to Microsoft Word and Microsoft Outlook also apply to Microsoft Office, Standard Edition.
  • Microsoft Trip Planner and Microsoft Bookshelf cross-promotional offers

VALUPACK CONTENT

The following additional features are included in the ValuPack:

  • Additional Help files
  • Animated cursors
  • Animated Graphics Interchange Format (GIF) files
  • Avery Wizard
  • Building Applications with Microsoft Access 97
  • Data Access Pack
  • Internet Help for Microsoft Access
  • Lotus cc:Mail information service
  • Microsoft ActiveMovie
  • Microsoft Camcorder
  • Microsoft Internet Explorer 3.0
  • Microsoft Office 97 sounds
  • Microsoft Office 97 Upgrade Wizard
  • Microsoft Office Binder templates
  • Microsoft Office Far East support
  • Microsoft Office templates, forms, and wizards
  • Microsoft Outlook import and export converters
  • Microsoft PowerPoint Custom Soundtrack
  • Microsoft PowerPoint Viewer
  • Microsoft Publish to ActiveMovie Stream Format
  • Microsoft Publisher trial version
  • Microsoft TrueType fonts
  • Microsoft Web Publishing Wizard
  • Microsoft Word 97 converter
  • Microsoft Word Viewer
  • Presentation and World Wide Web page textures
  • Software patches (Microsoft Plus! Themes patch, Microsoft Works 4.0a patch, Microsoft Outlook Inbox Icon patch)
  • Timex Data Link Watch Wizard

For more information on specific features included in the ValuPack, double- click the Valupk8.hlp file, located in the ValuPack folder on your CD-ROM, and then click the >> button next to the particular feature.
