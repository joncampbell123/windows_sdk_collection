No Accelerator Key for Continue Button in Setup

 Last reviewed: March 27, 1997
Article ID: Q148909
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office Professional for Windows 95, version 7.0
  • Microsoft Office Standard for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Access for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0

SUMMARY

The first dialog box (the welcome dialog) during Setup for the programs listed at the beginning of this article have two buttons: Continue and Exit Setup. The Exit Setup button uses the letter X for an accelerator key, but the Continue button has no accelerator key. All other buttons during Setup have accelerator keys assigned to them.

MORE INFORMATION

Accelerator keys are useful for moving quickly through dialog boxes and menus. You can identify an accelerator key by the underline of a character on a button or other object in a dialog box or on a menu item. Press the key that corresponds to the letter that is underlined to automatically activate the control.
