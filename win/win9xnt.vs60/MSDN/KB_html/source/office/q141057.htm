Inserting a Bullet in Marlett Font Also Inserts a Hollow Box

 Last reviewed: July 8, 1997
Article ID: Q141057
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0, 7.0a
  • Microsoft Publisher for Windows 95, version 3.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you create a bulleted list using the Marlett font, Word and Publisher insert one or more boxes, or a bullet and a box character instead of a single bullet.

NOTE: The box character is also printed.

CAUSE

This problem occurs when you modify one of the built-in bullets and numbering formats, and then select a bullet character in the Marlett font.

RESOLUTION

To resolve this problem, use one of the following methods.

Method 1

Use another font to insert the bullet character.

Method 2

To use a bullet from the Marlett typeface, click Symbol on the Insert menu. This method does not automatically indent the text.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
