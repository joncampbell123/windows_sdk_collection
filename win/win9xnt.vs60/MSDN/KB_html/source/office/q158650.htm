OFF97: Error When Using Online Registration

 Last reviewed: March 13, 1998
Article ID: Q158650
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you click Online Registration in the Microsoft Office 97 Setup program, you may receive the following error message: 

   The Registration Wizard was unable to locate the SignUp application,
   which is used to transmit your registration information to Microsoft.

   Please make sure that SignUp is properly installed on your computer,
   then try the Registration Wizard again.

CAUSE

This behavior occurs when the wizard cannot find the SignUp program, Signup.exe, in the "Program Files\The Microsoft Network" folder on your computer.

RESOLUTION

To resolve this behavior, verify that files for The Microsoft Network are properly installed on your computer. To do this, use the following steps:

  1. On the Start menu, point to Settings, and then click Control Panel.

  2. Double-click Add/Remove Programs and Click the Windows Setup tab.

  3. In the list of components, if the check box to the left of The Microsoft Network is not selected, select it, and click OK. If you are prompted to insert the Windows 95 disks or compact discs, do so.

    If the check box to the left of The Microsoft Network is already selected, remove and then reinstall the files. To remove the files, clear the check box, and click OK. Insert the Windows 95 disks or compact discs as requested by the Setup program. To reinstall the files, repeat steps 1 - 3.

After you install (or reinstall) the files, the wizard can find the SignUp program in the "Program Files\The Microsoft Network" folder when you click Online Registration.

STATUS

This behavior is by design of Signup.exe and the Microsoft Office 97 Registration Wizard.

MORE INFORMATION

When you run the Microsoft Office 97 Setup program, if you click Online Registration, you can register your software online. To use this feature, you must install a modem and the files for The Microsoft Network on your computer. Note that an account with The Microsoft Network is not required.

If these files, specifically Signup.exe, are not present on your computer, you cannot use the Registration Wizard, and you will receive the error message.
