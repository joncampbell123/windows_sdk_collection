OFF97: User Information Does not Display in Help About Window

 Last reviewed: December 6, 1996
Article ID: Q160025
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you click About <Office program name> on the Help menu in any of the Microsoft Office 97 programs, you see the following in the "This product is licensed to:" box: 

   Unknown User
   Uninstalled Version
   Product ID:
NOTE: The same information displays in the program splash screen you see when you start the program.

CAUSE

The User name, Company name, and Product ID fields are empty if all of the following are true:

  • You chose to have the Shared files installed on the server.

    -and-

  • You chose "Run from Network Server" as your installation type.

    -and-

  • You ran the Office setup program in maintenance mode and chose the Reinstall option.

RESOLUTION

To display the correct information in the "This product is licensed to:" box, do the following:

Remove Microsoft Office

  1. Quit all Office programs, including the Office Shortcut Bar.

  2. On the Windows Start menu, point to Settings and then click Control Panel.

  3. In the Control Panel window, double-click Add/Remove Programs.

  4. Click the Install/Uninstall tab, select Microsoft Office 97, and then click Add/Remove.

  5. In the first Microsoft Office Setup dialog box that appears, click Remove All.

  6. Click Yes when you are prompted whether or not you want to remove Microsoft Office.

  7. Click Restart Windows when the setup program finishes.

Rerun the Setup for Microsoft Office

Rerun the Office setup program from your administrative installation and choose the same options you selected the first time you ran the Office setup program.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
