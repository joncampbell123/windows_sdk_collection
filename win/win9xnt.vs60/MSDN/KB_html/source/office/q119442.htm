PPT4: PowerPoint Cannot Locate Cue Cards Error Message

 Last reviewed: March 12, 1998
Article ID: Q119442
The information in this article applies to:
  • Microsoft Office for Windows, version 4.2
  • Microsoft PowerPoint for Windows, version 4.0a

SYMPTOMS

Some copies of Microsoft Office 4.2 shipped with an incorrect version of the POWERPNT.DLL. This version of POWERPNT.DLL does not access the PowerPoint Cue Cards correctly. If you have this version of POWERPNT.DLL you will receive the following error message when you run the AutoContent Wizard: 

     PowerPoint could not locate or start the Cue Cards.
     It can be installed using PowerPoint setup.

WORKAROUNDS

Microsoft has made the correct version of POWERPNT.DLL available as an Application Note called "Correction to the POWERPNT.DLL File Included with Office 4.2" (WT1017). If you obtain the correct version and use it to replace the older POWERPNT.DLL the problem will be resolved.

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q113472
   TITLE     : PPT4: PowerPoint Cannot Start Cue Cards After Office 4.2 Setup
Another workaround is to do the following to use the AutoContent Wizards in PowerPoint 4.0a without accessing the Cue Cards:

  1. Open the POWERPNT.INI in a text editor such as Notepad. (POWERPNT.INI is located in the WINDOWS directory.)

  2. Locate the following line in the [Microsoft PowerPoint] section: 

          AutoContentCueCard=1

  3. Change this line to read: 

          AutoContentCueCard=0

  4. Save the file and restart PowerPoint.
