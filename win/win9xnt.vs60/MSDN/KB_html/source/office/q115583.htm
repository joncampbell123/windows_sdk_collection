OFFICE: SETUP /Q May Not Install All Applications

 Last reviewed: September 16, 1996
Article ID: Q115583
The information in this article applies to:
  • Microsoft Office for Windows, version 4.0, 4.2, 4.3

SUMMARY

If you use the "SETUP /Q" command to run a quiet mode installation, setup may only install Microsoft Excel 5.0 and its components. Microsoft Word, Microsoft PowerPoint, and their components will be missing. This is true regardless of whether it is a typical, laptop, workstation, or Complete/Custom installation.

CAUSE

There is a problem with the SETUP.STF file that was shipped with Microsoft Office. Certain lines have the wrong settings in the Install During BatchMode column (column B, when you open the file in Microsoft Excel 4.0).

WORKAROUND

To work around this problem, do the following:

  1. Open SETUP.STF in Microsoft Excel (or another suitable text editor).

    NOTE: If you open SETUP.STF in Microsoft Excel 5.0, complete the following additional steps before continuing: 

          a. In the Text Import Wizard dialog box, choose Next until Step 3

             Of 3 appears.

      b. In the Text Import Wizard - Step 3 Of 3 dialog box, under Data
         Preview, click the second column to select it. Under Column Data
         Format, select the Text option, and choose Finish.


  2. Make the changes outlined below (that is, find each object ID listed
   below in the Object ID column and change the corresponding entry in the
   "Install During Batch Mode" column).

    
   NOTE: If the entry in the table is (blank), clear whatever text is in
   the cell and leave it blank.
