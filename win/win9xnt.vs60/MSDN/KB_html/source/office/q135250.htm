OFF95: Components Installed with Each Type of Installation

 Last reviewed: March 10, 1997
Article ID: Q135250
7.00 WINDOWS kbsetup kbref

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

There are various installation types for Microsoft Office for Windows 95 version 7.0. Each type installs a different set of components. Below is a list of components that are installed with each of these types.

NOTE: If you are installing Microsoft Office from CD, you will have an additional folder named Valupack. The Valupack folder contains components that complement Microsoft Office. The components are listed below; each has its own Setup procedure.

Valupack: 

   Microsoft Imager
   Microsoft Word Viewer
   Options for WordPerfect users
   Microsoft PowerPoint multimedia
   Microsoft Office-compatible demos
If you would like more information on these components, read the Valupack.hlp file in the Valupack folder.
