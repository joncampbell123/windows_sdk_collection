Office or Word Setup: Setup Was Not Completed Successfully

 Last reviewed: November 17, 1997
Article ID: Q139914
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1
  • Microsoft Word for the Macintosh, versions 6.0.1

SYMPTOMS

Immediately after Microsoft Office Setup for the Macintosh is started, you may receive an error message similar to the following: 

   Microsoft Office Setup was not completed successfully.

   -Or-
Immediately after Microsoft Word Setup for the Macintosh is started, you may receive an error message similar to the following: 

   Microsoft Word Setup was not completed successfully.

CAUSE

This error occurs when the Microsoft Office Setup Data file has become corrupted. This also happens in when trying to install Word for the Mac with a damaged or old setup table file on the HD.

RESOLUTION

Replace the Microsoft Office Setup Data file using your original Setup disks. The following steps illustrate how to do this:

  1. Open the Setup Folder in the Microsoft Office folder on your hard disk.

  2. Drag the Microsoft Office Setup Data file to the Trash.

  3. Empty the Trash.

  4. Insert the Microsoft Office compact disc or Install Disk 1.

  5. Open the Setup folder. This folder is in the Install Disk 1 folder on the Office compact disc.

  6. Copy Microsoft Office Setup Data from the Setup folder on the Install Disk 1 to the Setup folder on your hard disk.

Microsoft Office Setup should now work as expected.

MORE INFORMATION

The Microsoft Office Setup Data file contains a list of installable files and their locations on the setup disks. If this file is corrupted, Microsoft Office Setup cannot function properly. When you install Microsoft Office, the Microsoft Office Setup Data file is copied from the Setup disks to the hard disk. If this file is damaged, you can manually replace it.
