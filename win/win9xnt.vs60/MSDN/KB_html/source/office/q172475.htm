OFF97: How to Obtain and Install MS Office 97 SR-1

 Last reviewed: March 9, 1998
Article ID: Q172475
The information in this article applies to:
  • Microsoft Office 97 for Windows, Service Release 1 (SR-1)

SUMMARY

Microsoft Office 97 SR-1 includes a number of updates for each program in Microsoft Office 97. There are two versions of Microsoft Office 97 SR-1: the SR-1 Patch and the SR-1 Enterprise Update. This article contains information about obtaining and installing these versions of Office 97 SR-1.

MORE INFORMATION

SR-1 Is Available in Two Forms

The Microsoft Office 97 SR-1 Enterprise Update is available on compact disc and on floppy disks. The SR-1 Enterprise Update includes updated Office 97 files and does not require that you install Microsoft Office 97 before you upgrade.

The SR-1 Patch is available on the Microsoft World Wide Web site, on compact disc, and on floppy disks. When you run the patch, you update an existing installation of Office 97. Note that you cannot use the patch to update some installations of Office 97.

Important Note to OEM Customers

If you have an OEM (Original Equipment Manufacturer) copy of Microsoft Office 97, you cannot order disks from the Microsoft Order Desk or from the Microsoft web site. You can only download the patch from the Microsoft web site, or contact your OEM to see if they have Microsoft Office 97 SR-1 or the Enterprise Update available.

To determine if you have an OEM copy of Microsoft Office 97, start any of the programs included with Microsoft Office 97 and click About on the Help menu in the program. If the Product ID (PID) includes the letters "OEM", you cannot order Microsoft Office 97 SR-1 floppy disks or compact discs from Microsoft.

Items shown below that are NOT avilable to customers with OEM copies of Microsoft Office 97 are marked with an asterisk in parentheses (*).

Limitations of Updating Office 97 with the Patch

You can use the downloadable patch to update configurations installed from the following sources:

  • Microsoft Office 97 Professional Edition for Windows (compact disc and floppy disks)
  • Microsoft Office 97 Standard Edition for Windows (compact disc and floppy disks)
  • Microsoft Office 97 Small Business Edition for Windows (compact disc and floppy disks)
  • Microsoft Excel 97 for Windows (compact disc and floppy disks)
  • Microsoft Word 97 for Windows (compact disc and floppy disks)
  • Microsoft PowerPoint 97 for Windows (compact disc and floppy disks)
  • Microsoft Access 97 (compact disc and floppy disks)
  • Microsoft Outlook (compact disc and floppy disks)

You cannot use the patch to update the following installations of Office 97:
  • Administrative installations

    To update an administrative installation, you must create a new administrative installation point by using the SR-1 Enterprise Edition. When you run Setup from this version of Office, use the /a switch with the Setup command again, for example, type the following: 

          setup /a
  • Run From CD-ROM installations

    To prevent problems that may occur because of mismatched dynamic link library (DLL) files, you cannot use the patch to update a Run From CD installation. If you attempt to update this kind of installation by using the patch, the following error message appears: 

          Microsoft Office 97 SR-1 has detected that office is running from a
      CDROM drive...cannot continue.
  • Run From Network Server installations

    Because you cannot use the patch to update an administrative installation, you cannot update a Run From Network Server installation. If you attempt to update this kind of installation by using the patch, the following error message appears: 

          You are running from a network drive. SR-1 Patch does not support
      this... aborting.
  • Installations in which shared programs are stored on a network server

    You cannot use the patch to update a post-administrative installation in which you installed Office 97 from a network server, and the shared programs run from the network server. However, you can use the patch to update this kind of installation if the shared programs run on the local computer.

NOTE: When you use the patch, the updating process may not be completed on computers running some configurations of Microsoft Windows NT 4.0. For the patch to work successfully, Service Pack 2 or later must be installed on a computer running Windows NT 4.0. In addition, if NT File System (NTFS) is being used, the user must have administrative privileges to the computer.

Obtaining Office 97 SR-1

To obtain the SR-1 Patch, use any of the following methods:

  • Download the SR-1 Patch from the Microsoft Web site

    To download the patch, follow these steps: 

        1. Use a Web browser to go to the following Microsoft Web site:


             http://www.microsoft.com/office/office/enhancements/sr1off97.asp

    2. Click Download Now.

       Locations from which you can download the patch are listed on the
       page that appears.

    3. Select a location by clicking the Sr1off97.exe link next to the
       location.

    4. When you are prompted to open or save the file you are downloading,
       click "Save it to disk", and then save the file in a folder on the
       hard disk.
  • Order the SR-1 Patch from the Microsoft Web Site (*)

    You can order the SR-1 Patch on compact disc or on 3.5-inch disks on the Microsoft Web site. Note that you will be required to provide your Product ID (PID) number, which can be found in the About box (click About on the Help menu) in your current Office 97 program.

    To order the SR-1 Patch on the Web, use a Web browser to go to the following Microsoft Web site: 

          http://www.microsoft.com/office/office/enhancements/sr1off97.asp

    At the bottom of the page, click the link for the media you want to receive.

  • Order the SR-1 Patch on compact disc or on floppy disk by calling the Supplemental Part Order Center at (800)-370-9272. (*)

To obtain the SR-1 Enterprise Update, use the following method:
  • Order the SR-1 Enterprise Update by calling the Microsoft Order Desk at (800)-360-7561. (*)

    If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the following Microsoft Web site: 

          http://www.microsoft.com/worldwide/default.htm

To obtain the SR-1 Enterprise Update or to order the SR-1 Patch, use the following method:
  • Order either version of Office 97 SR-1 by calling Microsoft Technical Support at (425)-635-7056. (*)

Installing the SR-1 Enterprise Edition

To install the SR-1 Enterprise Edition, insert the compact disc or Disk 1, and then run the Setup program. For more information, see the Ofread8.txt file on the compact disc (in the Office folder) or on Disk 1.

Installing the SR-1 Patch

To install the patch, follow these steps:

  1. Start the patch by double-clicking Sr1off97.exe in Windows Explorer or File Manager.

    After you start the patch, the License Agreement dialog box appears.

  2. Read the agreement and click Yes to start the next stage of the patching process. Clicking No ends the patching process.

    After the patch files are copied, a dialog box appears with a welcome message and an outline of the system requirements for running the patch.

  3. Click Continue to allow the patching process to proceed.

    A dialog box indicates that Setup is preparing to begin the patching process.

    After a few moments, a list of the installed components that can be updated appears in a dialog box. The dialog box also lists items that are already updated or items that the patch cannot update.

  4. Click Continue to begin the updating process.

    A status dialog box keeps you updated on the progress of the patch.

    At the conclusion of the patching process, a dialog box appears and prompts you to restart the computer.

  5. Click Yes to restart the computer and complete the patching process.
