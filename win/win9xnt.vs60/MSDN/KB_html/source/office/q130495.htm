OFF95: No Scroll Bar for Font Size List in MS Excel or Word

 Last reviewed: March 10, 1997
Article ID: Q130495
7.00 WINDOWS kbenv

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel or Microsoft Word, when you click the drop-down arrow in the Font Size box on the Formatting toolbar, the scroll bar may not appear next to the list of fonts as expected.

CAUSE

This behavior occurs if the Scrollbar item size in Windows is greater than approximately 45. The Scrollbar size is an option you can modify in Windows by double-clicking the Display icon in the Control Panel, and then by clicking the Appearance tab.

If you change the size of the Scrollbar item in Windows to a value greater than approximately 30, the scroll bar may appear on top of the values in the Font Size list in Microsoft Excel and Word, preventing you from viewing the values. If you change the Scrollbar size in Windows to a value greater than approximately 45, the scroll bar in the Font Size list does not appear at all in Microsoft Excel and Word.

Note that this problem does not occur in Microsoft PowerPoint.

RESOLUTION

To work around this behavior, avoid setting the Scrollbar size value in Windows to a value greater than 30.

Note that you can still select a font size from the Font Size box on the Formatting toolbar in Microsoft Excel and Word by doing the following:

  1. On the Formatting toolbar, click the drop-down arrow in the Font Size box.

  2. Press the UP ARROW or DOWN ARROW key to select a font size.

  3. Press ENTER to apply the selected size.
