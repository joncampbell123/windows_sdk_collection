Items Not Installed with "Run From CD" Installation Option

 Last reviewed: September 16, 1996
Article ID: Q121954
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.2-CD, 4.3-CD, 4.3c-CD

SYMPTOMS

When you perform a Run From CD installation of Microsoft Office, some of the Microsoft Office files are not installed. You can see this when you run Setup again in Maintenance mode: the check box next to some of the items is cleared. The following is a list of the items that are not installed.

Microsoft Office versions 4.2-CD and 4.3-CD

   Microsoft Word
      Wizards, Templates & Letters
         *Macro Templates

   Microsoft Excel
      Online Help & Lessons
         *Help for Excel 4.0 Macros

         *Tools
            Macro Translation Assistant
            1-2-3 CNV Converter

   Tools
      Truetype Fonts
         *Arial Narrow
         *Bookman Old Style
         *Century Gothic
         *Monotype Corsiva
         *Century Schoolbook

Microsoft Office version 4.3c-CD

   Microsoft Word
      Wizards, Templates and Letters
         *Templates & Wizards
         *Macro Templates

   Microsoft Excel
      Tools
         *1-2-3 CNF Converter

   Microsoft Bookshelf
     *Integration with Microsoft Word

   Tools
      *Setup Files

CAUSE

Some of these files are not installed by design. For example, because there is no *.INI setting for the Microsoft Word Macro Templates, and because they are not copied to the hard drive during a Run From CD installation, they are not selected as an installed item. The Macro Template files are simply located on the CD in this case, and you can open them in Microsoft Word from the CD-ROM drive if you want to use them.

WORKAROUND

To install any of the above items to your local hard drive, you can run the Setup program again in maintenance mode, and select the item that you want to install.
