Office Err Msg: Cannot Find TOOLHELP.DLL

 Last reviewed: September 16, 1996
Article ID: Q120110
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3

SYMPTOMS

When you install Microsoft Office for Windows version 4.3, you may receive the following error message 

   Cannot find TOOLHELP.DLL
followed by: 

   Setup Parse Error in Initialize Path
   Object ID 4924: Custom Action Object: Bad DLL data value.

CAUSE

This error message may be caused by a corrupt or missing TOOLHELP.DLL file which should be located in the WINDOWS\SYSTEM directory.

RESOLUTION

To correct this problem, do the following:

  1. If the file TOOLHELP.DLL is not missing, rename or delete it.

  2. Do the following to expand a new copy of the file from your original Windows disks to your WINDOWS\SYSTEM subdirectory of Windows:

    CAUTION: Make sure you exit Windows before expanding the file. 

        - At the MS-DOS prompt type


          C:\<MSDOS>\EXPAND <drive>\TOOLHELP.DL_ C:\WINDOWS\SYSTEM\TOOLHELP.DLL


          where <MSDOS> is the directory where MS-DOS is installed on your
      computer, <drive> is the drive where you disk is located, and
      <WINDOWS> is the directory where Windows is installed on you
      computer.

The TOOLHELP.DLL file is located on the following disks: 

   For Windows                                      Location of
   Version                  Size of disks           TOOLHELP.DLL
   -------------------------------------------------------------

   3.1                       3.5-inch                Disk 5
   3.1                       5.25-inch               Disk 8
   Workgroups version 3.11   3.5-inch                Disk 4
   Workgroups version 3.11   5.25-inch               Disk 8
