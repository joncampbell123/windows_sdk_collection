OFF97: MSGraph Installed if Powerpoint Is Selected

 Last reviewed: December 4, 1996
Article ID: Q160221
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

When you install Microsoft Office, Microsoft Graph, which is a shared program, is automatically installed.

CAUSE

This behavior occurs when you install Microsoft Office 97 and Microsoft PowerPoint is already installed.

STATUS

This is by design of Microsoft Office 97 for Windows.

MORE INFORMATION

Microsoft Graph is a charting program that can you use in Microsoft programs, such as Word, PowerPoint, Microsoft Access, Microsoft Excel, Profit, and FoxPro. You can use Graph to display numeric data in graphical form, making the data clearer, more interesting, and easier to read.

REFERENCES

For more information about Microsoft Graph, click the Index tab in Microsoft Word Help, type the following text 

   Graph
and then double-click the selected text to go to the "Graph (Microsoft Graph)" topic.
