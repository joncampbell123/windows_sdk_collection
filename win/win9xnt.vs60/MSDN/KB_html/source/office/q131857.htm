Top of Embedded Microsoft Excel Object Truncated in Word

 Last reviewed: March 27, 1997
Article ID: Q131857
7.00 WINDOWS kbinterop

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Windows 95

SYMPTOMS

When you edit a Microsoft Excel object in a Word document in Normal view, the top of the Microsoft Excel object is cut off. The column headings are only partially visible.

CAUSE

In Normal view, Word has less space available to display the column headings in Microsoft Excel.

WORKAROUND

To work around this problem:

  1. Before you activate the Microsoft Excel object, switch to page layout view.

  2. From the View menu, choose Page Layout.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95 version 7.0. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
