Contents of the Empowerment Pack Disks

 Last reviewed: November 17, 1997
Article ID: Q153541
The information in this article applies to:
  • Microsoft Empowerment Pack for the Mac, version 1.0
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

This article contains the disk contents of the four disks that ship with the Microsoft Empowerment Pack for the Mac.

MORE INFORMATION

The following is a listing of the contents of the four disks you receive when you order the Empowerment Pack on floppy disks:

Microsoft Internet Explorer (PPC)

Label: Setup for Power Mac

Contents: Internet Explorer Installer

The disk contains the installer for installing Microsoft Internet Explorer for the Power Macintosh. To install Microsoft Internet Explorer, double- click the icon on the disk and follow the directions supplied by the installer.

Microsoft Internet Explorer (68K)

Label: Setup for 68K Mac

Contents: Internet Explorer Installer

The disk contains the installer for installing Microsoft Internet Explorer for the Macintosh. To install Microsoft Internet Explorer, double-click the icon on the disk and follow the directions supplied by the installer.

For additional information about Microsoft Internet Explorer, please see the following articles in the Microsoft Knowledge Base:

   ARTICLE-ID: Q150591
   TITLE     : Troubleshooting Internet Explorer 2.0 for Macintosh Setup

   ARTICLE-ID: Q150539
   TITLE     : Support for Microsoft Internet Explorer

   ARTICLE-ID: Q150749
   TITLE     : Release Notes File from Internet Explorer 2.0 for Macintosh

Microsoft Internet Assistant for Microsoft Word 6.0.1

Label: Disk 1 - Setup

Contents: WordIA.sea 

           Readme
This disk contains the Internet Assistant 2.0 for Microsoft Word for the Macintosh. To install the Internet Assistant, double-click the file WordIA.sea on the disk to uncompress the Internet Assistant.

NOTE: You probably will want to uncompress the Internet Assistant to your Desktop.

Once it is uncompressed, open the Internet Assistant folder, double-click the Internet Assistant Setup icon in this folder, and follow the directions supplied by the setup program.

For more information about Microsoft Word Internet Assistant, please read the Readme text file contained on the Microsoft Word Internet Assistant disk.

Microsoft Internet Assistant for Microsoft Excel 5.0

Label: Disk 1 - Setup

Contents: Html.xla 

          Readme
To install the Microsoft Excel Internet Assistant for the Macintosh, drag the file Html.xla from the floppy disk to the Macro Library folder, which is located in your Microsoft Excel 5 folder. If Microsoft Excel is currently running, quit the program, and then restart it. Internet Assistant will now be available.

For more information about Microsoft Excel Internet Assistant, please read the Readme text file contained on the Microsoft Excel Internet Assistant disk.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q147273
   TITLE     : The Internet Assistant Wizard for Microsoft Excel (html.xla)
The Microsoft Empowerment Pack on compact disc is being mailed to all registered users of Microsoft Office for the Macintosh, version 4.2x, Microsoft Excel for the Macintosh, version 5.0, Microsoft Word for the Macintosh, version 6.0, and Microsoft PowerPoint for the Macintosh, version 4.0. If you do not have a CD-ROM drive, you can order The Microsoft Empowerment Pack on floppy disk by calling, in the United States, (800) 469- 6520. Because of the large differences in media type, the floppy disk version of the Empowerment Pack does not offer all of the same products and updates that is contained on the Empowerment Pack compact disc.

Below is a listing of the contents available on the Empowerment Pack compact disc.

  • Microsoft Internet Explorer 2.0 for the Macintosh.
  • Internet Assistant 2.0 for Microsoft Word for the Macintosh.
  • Internet Assistant for Microsoft Excel for the Macintosh.
  • Microsoft Word 6.0.1a Update for Macintosh.
  • System 7.5 Update 2.0 and System 7.5.3 Revision 2 from Apple Computer.

If you do not have a CD-ROM drive and you want the components of the Empowerment Pack contained on the compact disc but not on the floppy disks, do the following:
  • Microsoft Word 6.0.1a Update for Macintosh.

    Licensed users of Microsoft Word for the Macintosh version 6.0 can call (800) 360-7561 to order the free update.

    To determine which version of Word you have installed on your computer, do the following:

    1. Locate the Microsoft Word icon on your hard drive.

    2. Click the icon to select it.

    3. Press Command + I to bring up the Info window on the icon.

    The "Version:" line in the window will tell you the version of Word that is currently installed on your computer.

  • System 7.5 Update 2.0 and System 7.5.3 Revision 2 from Apple Computer.

    You can get either update by calling Apple Computer, Inc., or by downloading it from their World Wide Web Internet sites.
