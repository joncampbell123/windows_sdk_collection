PowerPoint Viewer and Associated Files on One Disk

 Last reviewed: September 16, 1996
Article ID: Q117554
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.2, 4.3
  • Microsoft PowerPoint for Windows, version 4.0

SUMMARY

You can use the Microsoft PowerPoint Viewer to view a slide show on a computer even when that computer does not have a full installation of PowerPoint. The viewer and all of its components fit on one disk; these files are located on the last disk of the Microsoft Office 4.2 Setup disk set.

You can copy and distribute this disk so that you can use the PowerPoint Viewer on a computer that does not have a full installation of PowerPoint.

MORE INFORMATION

When you install the PowerPoint Viewer on your computer, you can view a PowerPoint presentation even if you do not have Microsoft PowerPoint installed on your computer. You can install the PowerPoint Viewer version 4.0 either by running the Microsoft Office Setup program, and selecting the PowerPoint Viewer installation option (one of the PowerPoint options), or by running the PowerPoint Viewer Setup program (VSETUP.EXE) located on the last disk of your Microsoft Office disk set.

To install just the PowerPoint Viewer, use the following steps: 

   1. Insert the PowerPoint Viewer disk in your floppy disk drive. Use the
      following table to determine the number of the PowerPoint Viewer disk
      in your Microsoft Office disk set:

      Microsoft Office version   Disk
      -------------------------------
      4.2 (3.5 inch)              25
      4.2 (5.25 inch)             31
      4.3 (3.5 inch)              31

   2. In File Manager, choose Run from the File menu.

   3. In the Command Line box, type the following

         <drive>:vsetup

      where <drive> is the letter of the floppy disk drive that contains
      the PowerPoint Viewer disk, such as a:vsetup.
