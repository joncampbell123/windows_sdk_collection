OFF98: How to Use Menus and Menu Commands with the Keyboard

 Last reviewed: February 3, 1998
Article ID: Q176405
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SUMMARY

In the programs included in Microsoft Office 98 Macintosh Edition, you can use menus in each program by using the keyboard. This article explains how to use the menus in each program by pressing a key or key combination.

MORE INFORMATION

To use the keyboard to use menu commands in Microsoft Office 98 Macintosh Edition programs, use the menu bar, and then use the access key for the command you want to use. To use the menu bar, press the appropriate key or key combination that is listed in the following table. 

   Program                                     Key or key combination
   ------------------------------------------------------------------

   Microsoft Excel 98 Macintosh Edition        F10
   Microsoft PowerPoint 98 Macintosh Edition   F10
   Microsoft Word 98 Macintosh Edition         F10 or COMMAND+TAB
NOTE: You cannot press COMMAND+TAB to use menus in Microsoft Excel or Microsoft PowerPoint.

When you use the keyboard to activate the menu bar, access keys (underlined characters) appear on each menu and for each menu item. You can use the arrow keys to move between menus and menu items, or you can use the access keys to quickly select and use specific menus and menu items.

To use a particular menu command, press the access key for the command, or select the menu command and press RETURN. To quit the menu, press ESC.
