Write-Up Command Unavailable in Office Binder

 Last reviewed: September 16, 1996
Article ID: Q135561
The information in this article applies to:
  • Microsoft Office for Windows 95, versions 7.0

SYMPTOMS

When you try to use PowerPoint for the Write-Up feature while you are working in an Office Binder and both a PowerPoint and a Word section are open, you will receive the following error message: 

   This command is unavailable while there is an active Word Section in
   the Binder.

CAUSE

Office for Windows 95 binders are OLE clients designed to provide a common interface and storage area for project-specific data. Binders are not designed to provide full functionality in each of the server applications.

This behavior is by design.

RESOLUTION

You can successfully invoke the Write-Up feature under the above conditions by following these steps:

  1. Switch to the Word section. On the Section menu, click View Outside .

  2. Switch to the PowerPoint section. On the Tools menu, click Write-Up.

REFERENCE

For more information about the Write-Up command, click Answer Wizard on the Help menu in Microsoft PowerPoint, and type the following 

   write up
