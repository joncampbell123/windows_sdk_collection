OFF7: Cannot Delete Files in the Open Dialog Box in Windows NT

 Last reviewed: June 23, 1997
Article ID: Q145872
7.00 WINDOWS kbui kbdocerr xlwin

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Windows NT, version 3.51

SYMPTOMS

If you are using one of the products listed above with Microsoft Windows NT version 3.51, you will not be able to delete documents from within either the Open or Save dialog boxes. Also, the Help file entry erroneously states that you can delete documents in this manner.

CAUSE

You do not have any menu options or buttons for deleting files in either dialog box because these features were designed for running under the Microsoft Windows 95 operating system and are not features of Microsoft Windows NT.

RESOLUTION

If you want to delete files under the Windows NT operating system, use File Manager.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
