OFF95: Files Installed with Compact Installation

 Last reviewed: March 10, 1997
Article ID: Q136720
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The following is a list of the folders and files installed when you perform a Compact installation of Microsoft Office.

MORE INFORMATION

You can use this list to determine which files are installed by Microsoft Office and where they are installed, so that you can remove them if necessary.
