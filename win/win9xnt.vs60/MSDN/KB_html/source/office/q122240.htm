Office Help Files Available

 Last reviewed: February 13, 1998
Article ID: Q122240
The information in this article applies to:

SUMMARY

The Knowledge Base group extracts current articles from the Knowledge Base and creates Windows Help files for the Microsoft Office products. These files are posted to the Microsoft Software Library and are updated as new information becomes available.

The files listed below are available to download from the Microsoft Software Library (MSL). For information about downloading these files, please see the "Instructions for Downloading" section at the end of this article.
