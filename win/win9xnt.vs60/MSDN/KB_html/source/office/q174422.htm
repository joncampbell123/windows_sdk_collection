OFF97: Microsoft Office 97 Service Release 1 Patch Update

 Last reviewed: March 13, 1998
Article ID: Q174422
The information in this article applies to:
  • Microsoft Office 97 for Windows, Service Release 1 (SR-1)

SUMMARY

This article contains information about the Microsoft Office 97 SR-1 Patch Update.

MORE INFORMATION

Microsoft has released the Office 97 SR-1 Patch Update to address an isolated problem that may occur when you update Microsoft Word 97 by using the original Office 97 SR-1 Patch, and then save documents in Word SR-1. Current versions of the patch do not cause this problem. For more information about this problem, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE ID: Q173001
   TITLE     : WD97: Word SR-1 Hangs During Save Operation
If you have not yet installed any version of Office 97 SR-1 or the Office 97 SR-1 Patch, you will not be affected by this problem. You should either download the newest SR-1 Patch or order the Enterprise Update CD. For information about obtaining the newest version of the Office 97 SR-1 Patch or the Enterprise Update, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE ID: Q172475
   TITLE     : OFF97: How to Obtain and Install MS Office 97 SR-1

How Do I Determine Which Version of the Patch I Am Using?

To determine which version of the SR-1 Patch you are running, use either of the following methods.

Method 1:

Microsoft has released a utility file called the Office 97 SR-1 Patch Version Checker (Patchchk.exe). This file checks your computer to determine which version of the SR-1 Patch you should install: the Patch Update or the full U.S. English patch. You can run the Version Checker only if the U.S. English version of Word 97 is already installed on your computer. After you run the version checker to determine the version of the patch you are running, you can download and install the correct version of the SR-1 Patch if necessary. You can download and install the Office 97 SR-1 Patch Version Checker from the following Web site: 

   http://www.microsoft.com/office/office/enhancements/PatchChk.asp
NOTE: Because the Microsoft Web sites are constantly updated, the site addresses may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
Method 2:

If you downloaded the SR-1 Patch to the hard disk on your computer, you can determine the version of the SR-1 Patch you are running by viewing the date of the file. If you ran Setup from the Web site, you cannot correctly determine the version. Note that downloading, and then running the patch is the recommended method of using the patch.

To determine which version of the patch is installed on your computer, use Windows Explorer to look at the date of the Sr1off97.exe patch file. Earlier versions of the patch have a file date of 7/11/97. The current patch has a file date of 8/21/97. If you saved the SR-1 Patch from the Web to your hard disk, this file is located where you saved it. If you ran the patch from the SR-1 Patch CD, locate the file on the CD.

How To Install the Office 97 SR-1 Patch Update

The Office 97 SR-1 Patch Update updates the earlier patch that was available on the Web before 8/16/97 to the current, corrected version. You can download and install the Office 97 SR-1 Patch Update from the following web site: 

   http://microsoft.com/office/office97/servicerelease/
NOTE: Because the Microsoft Web sites are constantly updated, the site addresses may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
