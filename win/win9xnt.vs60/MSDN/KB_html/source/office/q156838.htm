OFF97: Button Images do not Appear on New Toolbar

 Last reviewed: March 13, 1998
Article ID: Q156838

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office 97 for Windows

SYMPTOMS

When you add a button to a toolbar on the Microsoft Office Shortcut Bar (OSB), its image may not display on the toolbar. A blank button appears on the toolbar instead. If you add several buttons to a toolbar, the only one that does not display correctly is the last one you added.

CAUSE

You may experience this behavior if the following are true:

  • You created a custom OSB toolbar.

    -and-

  • This custom toolbar has never had any buttons added to it.

    -and-

  • You add one or more buttons to this custom toolbar.

WORKAROUND

To refresh the Microsoft Office Shortcut Bar so that all button images display correctly: 

   1. Right-click on the background of your custom toolbar.

   2. Click "Refresh Icons" on the shortcut menu.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
