OFF97: Format Callout Dialog Box Is Available Only in Word

 Last reviewed: December 6, 1996
Article ID: Q159253
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

In the Format AutoShape dialog box, in Microsoft Excel and Microsoft PowerPoint, you cannot display the Format Callout dialog box that is available in Microsoft Word.

CAUSE

The Format Callout command button is available only on the Text Box tab of the Format AutoShape dialog box in Microsoft Word.

WORKAROUND

To change the format of a callout in Microsoft Excel or Microsoft PowerPoint, do the following:

  1. Select the callout.

  2. On the Drawing toolbar, click Draw, point to Change AutoShape, and then click Callouts.

  3. Click the type of callout that you want.

NOTE: When you move the pointer over each of the displayed example callouts, the type of callout is described in the ToolTip text.

MORE INFORMATION

In Microsoft Word, the Format Callout dialog box is also unavailable (dimmed) for callouts that are based on the first row of sample callouts.

The first row of sample callouts includes the following callout types: 

   Rectangular Callout
   Rounded Rectangular Callout
   Oval Callout
   Cloud Callout

REFERENCES

For more information about AutoShapes, click the Index tab in Microsoft Excel Help, type the following text 

   autoshapes
and then double-click the selected text to go to the "Change the AutoShape type" topic.
