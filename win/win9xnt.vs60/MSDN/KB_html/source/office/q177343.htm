MOFF: Cannot Use the Keyboard to Display the Help Menu

 Last reviewed: February 2, 1998
Article ID: Q177343
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft PowerPoint for Macintosh, version 4.0
  • Mac OS 8.0

SYMPTOMS

When you press F10 to activate the menu bar, no shortcut key is available for the Help menu.

CAUSE

This problem may occur if you are running Mac OS 8.0.

STATUS

This behavior is by design of the products listed at the beginning of this article.

MORE INFORMATION

The Help menu is drawn by the system software; it is not drawn by the program that you are using. In earlier versions of the Macintosh system software, the Help menu appears as a balloon icon on the right side of the menu bar. In Mac OS 8.0, the Help menu appears to be integrated with the menus of the program that you are using.
