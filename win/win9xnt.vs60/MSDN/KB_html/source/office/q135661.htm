Decimal Numbers Truncated in Properties in Open Dialog Box

 Last reviewed: September 16, 1996
Article ID: Q135661
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint Windows 95, version 7.0

SYMPTOMS

When you set a custom value containing a decimal on the custom tab in the OLE properties dialog box and you attempt to view the properties from the Open dialog box, the decimals are truncated or may be displayed as zero. For example, the number 12.345 may be displayed as 12 or 0. If you open the file and view the properties by clicking Properties on the File menu, the decimals are displayed correctly.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
