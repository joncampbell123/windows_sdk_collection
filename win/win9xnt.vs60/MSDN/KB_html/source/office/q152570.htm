OFF95: Office Setup 4.3c Icon Not Removed Under Windows NT

 Last reviewed: March 10, 1997
Article ID: Q152570
7.00 7.00a WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, versions 7.0, 7.0a

SYMPTOMS

After you upgrade from Microsoft Office version 4.3c to Microsoft Office for Windows 95 on a Windows NT Workstation, version 3.51, two Microsoft Office Setup icons appear in the Microsoft Office group of Program Manager.

CAUSE

One of the Setup icons is labeled Office Setup, and the other is labeled Microsoft Office Setup. The Office Setup icon does not work; however, the other icon does start Microsoft Office for Windows 95 Setup. The old Microsoft Office version 4.3c Setup icon is not removed during the setup of Microsoft Office for Windows 95.

WORKAROUND

To work around this behavior, delete the Office Setup icon manually. To do so, follow these steps:

  1. Open the Microsoft Office group in Program Manager by double-clicking it.

  2. Click the Office Setup icon.

  3. Press the DELETE key.
