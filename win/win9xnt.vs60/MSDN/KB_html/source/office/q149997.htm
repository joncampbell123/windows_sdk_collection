ORK: Doc Error with Shortcut Bar Default Settings

 Last reviewed: September 16, 1996
Article ID: Q149997
The information in this article applies to:
  • Microsoft Office Resource Kit for Windows 95, version 7.0

SUMMARY

Page 218 of the Office Resource Kit (ORK) includes a table that contains the default configuration values for the Office Shortcut Bar. Some of the values listed in the table are incorrect.

MORE INFORMATION

The following is the table listed on page 218 of the ORK: 

Office Shortcut Bar   Windows 95    Windows 95    Windows NT    Windows NT
Setting               640x480       800x600+      640x480       800x600+


Large Buttons         off           off           off           off
Show ToolTips         on            on            ON            ON
Always on Top         on            on            on            on
Auto Hide             off           off           off           off
Auto Fit (title bar)  on            off           on            on
Docked Location       top           right         top           top
Smooth Fill           off           off           off           off
Show title screen     ON            ON            ON            ON
Standard color        off           off           off           off
Note that if you use Microsoft Office version 7.0a or 7.0b and you are running Microsoft Windows NT, the information on the line referring to "Show ToolTips" is incorrect. (Note that these two incorrect values are in capital letters.) The correct line should read as follows: 

   Show ToolTips      on            on            OFF            OFF
The "Show title screen" line also contains misleading information. The line implies that every time you start the Office Shortcut Bar, the Office title screen will be displayed. In actuality, the Office title screen, by default, is displayed only the first time you bring up the Office Shortcut Bar after you install Microsoft Office. The correct line should read as follows: 

   Show title screen  OFF           OFF           OFF            OFF
NOTE: The compact disc that is shipped with the ORK also contains the table with the incorrect values.

For additional information about Default Office Shortcut Bar settings, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q149996
   TITLE     : OFF7: Default Configurations for the Office Shortcut Bar
