MSInfo Incorrectly Reports PowerPC (603, 604) Processor

 Last reviewed: November 17, 1997
Article ID: Q141951
The information in this article applies to:
  • Microsoft Office for Macintosh, versions 4.2, 4.2.1

SUMMARY

When you use Microsoft System Information (MSInfo) for the Macintosh, a utility that is installed with Microsoft Office, you may notice that the Processor Type in the System category is listed as the following: 

   Unknown PowerPC
The processor type may be listed this way when you are running the utility on a Power Macintosh because MSInfo is unable to detect 603 and 604 processors.

MORE INFORMATION

MSInfo is a utility that is installed into the Microsoft folder, and is a component shared by all Office programs. You can run MSInfo from within an Office program, or you can run it directly by double-clicking the MSInfo program icon.

MSInfo displays detailed information about your computer in the following categories:

  • System
  • Disk Storage
  • Fonts
  • System Extensions
