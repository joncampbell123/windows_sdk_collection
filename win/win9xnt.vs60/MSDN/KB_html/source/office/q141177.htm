Word List Dialog Box Is Hidden When You Click Find Tab

 Last reviewed: December 16, 1996
Article ID: Q141177
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Access 97 for Windows
  • Microsoft Access for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Outlook 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Windows 95
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

In some cases, when you create or load the word list, the Help Topics dialog box may appear to be disabled or may appear to hang (stop responding).

CAUSE

When you select the Find tab, the Creating Word List or the Loading Word List dialog boxes are displayed behind the Help Topics dialog box.

WORKAROUND

To ensure that the Word List is being created or loaded, move the Help Topics dialog box so that it is not in the middle of the screen. Moving the dialog box will allow the appropriate Word List dialog box to be displayed.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft Windows 95 Help Engine (Winhlp32.exe). We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you choose the Help menu in any of the products listed at the beginning of this article, and you build an index using the Find tab, either the Creating Word List or the Loading Word List dialog box will appear. The Creating Word List dialog box will appear if it is your first time using the Find tab. The Loading Word List dialog box will appear if you have used Find previously and an index was created.
