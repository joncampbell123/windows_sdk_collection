Office Err Msg: Sharing Violation on Drive...

 Last reviewed: September 9, 1996
Article ID: Q67908
The information in this article applies to:
  • Microsoft Office for Windows, version 1.0

SUMMARY

It has been reported that upon completion of the PowerPoint installation phase of the Microsoft Office, an error message indicating a "Sharing violation on drive x:" may occur if the DOS SHARE.EXE file is active. Installation has been successfully completed at that point and the error dialog can be canceled without causing harm to the system.

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
