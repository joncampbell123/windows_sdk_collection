OFF: Add/Remove Does Not Remove Shared Applications

 Last reviewed: December 13, 1996
Article ID: Q134745
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office 97 for Windows

SYMPTOMS

When you run Microsoft Office Setup in maintenance mode and use the Add/Remove option, Setup does not remove Shared Applications (including graphics filters) from the registry or hard disk. However, it does remove Shared Applications from the list of installed items.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.

For additional information, please see the following article(s) in the Microsoft Knowledge Base:
