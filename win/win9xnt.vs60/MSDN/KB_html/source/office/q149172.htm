ORK: Components of the ORK That Relate to the Macintosh

 Last reviewed: November 21, 1997
Article ID: Q149172
The information in this article applies to:
  • Microsoft Office Resource Kit for Windows 95, version 7.0
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

The Microsoft Office for Windows 95 Resource Kit (ORK) is an add-on product for Microsoft Office for Windows 95. It provides the information that a Management Information Systems (MIS) professional or administrator needs to roll out and support Microsoft Office version 7.0.

The ORK also contains some utilities for allowing users of Microsoft Office for the Macintosh versions 4.2 and later to open files created by the Microsoft Office for Windows version 7.0 programs. This article outlines all of the different programs on the ORK that allow this cross-platform functionality.

DESCRIPTIONS OF UTILITIES INCLUDED IN THE KIT

Coexist\mac\unbind

This folder contains a Macintosh utility that separates a Microsoft Office for Windows 95 binder file into its component files. The utility allows you to open Microsoft Excel, Word, and PowerPoint for Windows 95 component files in Microsoft Office for the Macintosh, versions 4.2 or later, or in standalone versions of the Microsoft Office for the Macintosh programs. Note, once your PowerPoint file has been separated from the Binder file, you still may not be able to open it on your Macintosh. In order to open a PowerPoint for Windows 95 file on your Macintosh, you first need to have the appropriate translator installed on your Macintosh.

To set up the Unbind utility, copy Unbind.hqx to a Macintosh computer and decompress the file using the BinHex version 4.0 or 5.0 utility.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q139888
   TITLE     : How to Use the ORK Unbind Utility on a Macintosh

Convert\Mac\Powerpnt

This folder contains a converter that allows Microsoft PowerPoint for the Macintosh, version 4.0, to read files created by Microsoft PowerPoint for Windows 95, version 7.0.

To set up the converter, copy Pp7trans.hqx to a Macintosh computer and decompress the file using either the BinHex version 4.0 or 5.0 utility. Note, the translator contained in the ORK is only for 68K Macintosh computers. If you have a Power Macintosh computer and you need to open PowerPoint version 7.0 files, you need to get the Power Macintosh version of the translator.

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q133803
   TITLE     : PowerPoint for Windows 95 Translators for PowerPoint for Mac

   ARTICLE-ID: Q139915
   TITLE     : Contents of PowerPoint 7.0 Translators Disk (for Macintosh)

Convert\Mac\Word\6&7Conv

This folder contains a converter that allows Microsoft Word 5.x for the Macintosh to read files created by Microsoft Word 6.x for the Macintosh, Windows, and Windows NT, and Microsoft Word for Windows 95 version 7.0.

To set up the converter, copy Mswrd6.hqx to a Macintosh computer and decompress the file using either the BinHex version 4.0 or 5.0 utility. Double-click the resulting Mswrd6.sea to extract the files it contains, and view the "ReadMe Before Installation" file for installation instructions. "ReadMe Before Installation" is a Word 5.1 for the Macintosh document.

Obtaining the ORK

The Microsoft Office for Windows 95 Resource Kit contains one Microsoft Press book, "Microsoft Office for Windows 95 Resource Kit," and one compact disc. The suggested retail price for the kit is (U.S.) $49.95. The kit is available free-of-charge on The Microsoft Network and the World Wide Web. The kit is also included on the Microsoft TechNet compact disc.

To obtain the Microsoft Press book, "Microsoft Office for Windows 95 Resource Kit," contact Microsoft Press at (800) MSPRESS or (800) 677-7377. Do not call the Microsoft Sales Information Center (MSIC) to order the kit.

You can obtain this Microsoft Press book from any local computer bookstore, or you can send in the Microsoft Office Supplemental Offers coupon, which is included in your Microsoft Office retail package.

You can also obtain the Microsoft Office for Windows 95 Resource Kit from the TechNet location on The Microsoft Network or the World Wide Web.
