OFF97: Timex Wizard Setup Fails Under Windows NT

 Last reviewed: March 12, 1998
Article ID: Q157764
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Windows NT, versions 3.51, 4.0

SYMPTOMS

When you install the Timex Data Link Watch Wizard, you may receive the following error messages: 

   You do not have administrator privileges on this machine. Some
   installations cannot be completed correctly unless they are run by an
   administrator.

   Do you want to continue?
If you continue the installation, you eventually receive the error message: 

   INF Install failure. Reason: Unknown
and the Timex Data Link Watch Wizard is not successfully installed.

CAUSE

This will occur if the following conditions are all true:

  • You attempt to install one of the Timex Data Link Watch Wizard files included in the Microsoft Office 97 Value Pack.

    -and-

  • You are using a computer running Windows NT, version 3.51 or 4.0.

    -and-

  • You are not logged on as an administrator.

RESOLUTION

To prevent this problem from occurring, log on to the computer as an administrator before installing the Timex Data Link Watch Wizard. Once installation is complete, you can log back on as a normal user.

STATUS

This behavior is by design of the Timex Data Link Watch Wizard included with Microsoft Office 97.

MORE INFORMATION

The Timex Data Link Watch, produced by Timex Corporation, receives information exported from Microsoft Outlook.

You can install the Timex Data Link Watch Wizard so that Microsoft Outlook can export any of the following to the watch: appointments, tasks, phone numbers, anniversaries, birthdays, reminders, and current time and time zone information.

The Timex Data Link Watch Wizard is installed by running one of two separate files: 

   File           Operating System
   --------------------------------------------

   Timexnt.exe    Windows NT 3.51
   Timexwiz.exe   Windows 95 and Windows NT 4.0
These files are located in the \ValuPack\Timex folder on your Microsoft Office 97 CD-ROM.

NOTE: You must have Microsoft Outlook installed on your computer in order to install the Timex Data Link Watch Wizard.
