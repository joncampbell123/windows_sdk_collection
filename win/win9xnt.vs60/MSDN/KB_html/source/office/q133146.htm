OFF95: Additional Tabs Appear in New Dialog Box

 Last reviewed: March 27, 1997
Article ID: Q133146
7.00-CD WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0-CD
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you perform a Run From CD or Run From Network Server installation of Microsoft Office, the New dialog box in Microsoft Word and the New dialog box that appears when you click Start A New Document on the Office toolbar contains the following additional tabs: 

   A4Letter
   A4Memos
   A4Other
   A4Pub
   A4Report
   AUSOther
   UKOther

CAUSE

This behavior occurs because Microsoft Office ships with templates for multiple international versions of Office, namely US, UK and AUS. Because each version of the individual templates has the same name, the files are located in different folders on the CD (in the Template folder) and on the network server when you perform an administrative installation (setup /a). When you perform a Run From CD or Run From Network Server install of Microsoft Office, the template files are not copied to your local hard disk. Instead, the template path for both Word and Office points to the Template directory on the CD or network server. Because the different international folders are located in the Template folder, each folder appears as a tab in the New dialog box in Microsoft Word, and the New dialog box that appears when you click Start A New Document on the Office toolbar.

When you perform a Run From CD or Run From Network Server installation of Office, you can use any of these template sets located on the CD or server; the version of the templates available in the New dialog box does not depend on the international version option that you select during Setup.

This behavior only occurs with the Run From CD and Run From Network Server installations. If you perform another installation, such as Custom, you only receive the templates for the international version that you are installing.
