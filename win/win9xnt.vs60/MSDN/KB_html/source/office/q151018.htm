Page Fault When You Insert a Spreadsheet Solutions Template

 Last reviewed: September 16, 1996
Article ID: Q151018
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you insert a Spreadsheet Solutions template as an object in Microsoft Excel, Word, or PowerPoint, you may receive the following error message: 

   <Template file name> is being modified by <user name>.
   Open as read-only?
If you click OK, the following message appears: 

   Cannot Open AutoTemplate Wizard.
If you click OK, the following message appears: 

   The numbering template add-in must be loaded for optimal numbering and
   toolbar behavior. Please locate this add-in in your Library directory.
If you click OK, the following message appears: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you click Details, a message similar to the following appears 

   EXCEL caused an invalid page fault in module EXCEL.EXE at
   0137:501548c5.

CAUSE

This problem occurs when the following conditions are true:

  • In a Microsoft Excel, Word, or PowerPoint file, you click Object on the Insert menu.

    -and-

  • You click Create From File, click Browse, and open the Templates folder.

    -and-

  • You open the Spreadsheet Solutions folder, select a template, click OK, and click OK again.

This problem occurs because the template contains a macro that cannot be run. The problem also occurs when you insert a Spreadsheet Solutions template in a binder.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q137674
   TITLE     : OFF7: Errors Working with a Binder That Contains a Worksheet

WORKAROUND

You may be able to insert the object if you continue to click OK, Close, or Retry on the error message windows that appear.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Office for Windows 95 includes a set of workbook templates for Microsoft Excel. The Village Software company designed these templates to address particular tasks for home and business. The following Village Software templates are installed in the Msoffice\Templates\Spreadsheet Solutions folder:

   Change Request.xlt
