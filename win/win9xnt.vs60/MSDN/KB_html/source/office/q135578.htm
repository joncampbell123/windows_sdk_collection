OFF95: Registration Button Appears After Setup From Server

 Last reviewed: March 27, 1997
Article ID: Q135578
7.00 WINDOWS kbsetup kbnetwork

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

When you run Microsoft Office Setup from a network server (post administrative installation), the Online Registration button is available after Setup is complete. This button appears even though the copy of Microsoft Office that you are running has probably been registered through a License Pak by your network administrator.

MORE INFORMATION

Because your network administrator has purchased a multiple License Pak and registered the copy of Office with Microsoft, you do not need to register the copy of Office that is installed on your workstation after you run Setup from the network server.
