Introduction

This updated Offpro97.inf file addresses a problem that occurs after you install the Service Release 1 (SR-1) Patch for the International English version of Microsoft(R) Office 97 Professional Edition with Microsoft Bookshelf(R) Basics. When this problem occurs, you can no longer run the Setup program in maintenance mode. When you attempt to do so, the following error message appears: 

   Setup Error 825 Object 9807. Your setup files may be damaged.
   Try restarting the setup program from where you originally ran
   it.
This message is followed by similar messages for Object 10004, Object 9905, and Object 9906.

Then, the following message appears: 

   Microsoft Office 97 setup was not completed successfully.
The problem occurs because the patch modifies the Off97Pro.inf file, which causes the maintenance mode installation to fail.

NOTE: This problem affects only the International English version of Microsoft Office 97 Professional with Bookshelf. All other versions of Microsoft Office 97 and individual programs are not affected by this issue.

Instructions

If you have already installed the SR-1 Patch and you are experiencing this problem, download this modified version of the Offpro97.inf file and copy it to the Office Setup folder on your computer to correct the problem.

NOTE: These instructions assume that you have already downloaded and extracted the contents of the compressed Offpro97.exe file.

To install the Offpro97.inf file, follow these steps:

  1. Start Microsoft Windows Explorer and open the Office Setup folder. By default, this folder is the following: 

          C:\Program Files\Microsoft Office\Office\Setup.

    If you cannot locate this folder, click Start on the Windows taskbar, point to Find, click Files Or Folders, and then search for the Off97pro.inf file.

    NOTE: After you run the Office SR-1 Patch, this file is located in the Office Setup folder. (If you cannot find this file, see the note in step 2.)

  2. Copy the updated Off97pro.inf file into the Office Setup folder. If you are prompted to overwrite the existing file, click Yes.

    NOTE: If you cannot find a copy of Off97Pro.inf on your hard disk, the following may be true: 

        - The Microsoft Office 97 SR-1 Patch was not successfully
      installed on your computer.


          -or-


        - Your original Office installation was not Microsoft Office 97
      Professional Edition with Bookshelf.

If you experience any problems, please contact Microsoft Technical Support at your local subsidiary.

To locate your subsidiary, see the Microsoft World Wide Office Web site at: 

 http://www.microsoft.com/worldwide/default.htm

More Information

The International English version of the SR-1 Patch has been updated. This problem will not occur after you use the updated patch to update future installations of Office.

If you have not installed the original patch and you have installed the International English version of Office 97 by using a compact disc or an administrative installation point, use the updated version of the patch that is currently available on the following Microsoft Web site: 

  http://www.microsoft.com/officefreestuff/office/dlpages/sr1of97z.htm
