Can't Restore Documentation Item to Original Location on MOM

 Last reviewed: March 13, 1998
Article ID: Q130283
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2.1-CD

SYMPTOMS

If you remove the Microsoft Office Documentation item from the Microsoft Office Manager (MOM) menu, you cannot add the item again to the same location on the menu.

CAUSE

When you install Microsoft Office from a CD-ROM drive, a Microsoft Office Documentation item is added to the bottom of the Microsoft Office Manager menu. If you remove this item from the Office Manager menu using the Remove button, you cannot add the item to the bottom of the menu again.

When you add the Microsoft Office Documentation item back to the Office menu, it is added above the last separator item in the Office Manager Menu list. You can move the item down just below the last separator item in the list. When you move the documentation item to this position, the down arrow (under Move) becomes unavailable, and you cannot move the item below this position.

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q129714
   TITLE     : Can't Move Documentation Item Down in MOM Customize List

WORKAROUNDS

To remove the Microsoft Office Documentation item from the Microsoft Office menu, without losing the location if you want to add the item again, clear the Microsoft Office Documentation check box in the Office Menu list instead of using the Remove button.

To add the Microsoft Office Documentation item to the bottom of the Microsoft Office Manager menu again, you must reinstall Microsoft Office.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Note that the Microsoft Office Documentation item on the Microsoft Office menu refers to the Microsoft Office Manuals file, located on your Microsoft Office CD in the following location: 

   Microsoft Office 4.2.1:Install Disk 1:Microsoft Office Manuals
When you add this item to the Office menu, the name that appears on the menu by default is Microsoft Office Manuals instead of Microsoft Office Documentation. You can change the name on the menu by selecting the item in the Microsoft Office Manager and by using the Edit button.

Note that although you can clear the Office Help check box or the Office Cue Cards check box in the Microsoft Office Manager dialog box, you cannot remove either of these items from the Office Manager Menu list.
