OFFICE: RAM Doubler Update Available for Use with Power Mac

 Last reviewed: November 21, 1997
Article ID: Q121363
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2

Connectix RAM Doubler version 1.0.4, which ships with Microsoft Office version 4.2, cannot be installed on a Power Macintosh. The Application Note, "Connectix RAM Doubler 1.5.1 Updater for Use with the Power Macintosh" (MC1063) is now available from Microsoft Product Support Services (PSS). The disk included with this Application Note contains the Connectix RAM Doubler 1.5.1 updater. You can use the 1.5.1 updater to update the version of Connectix RAM Doubler that ships with Microsoft Office version 4.2 for the Macintosh.

You can obtain this Application Note from the following sources:

  • Microsoft's World Wide Web Site on the Internet
  • The Internet (Microsoft anonymous ftp server)
  • The Microsoft Network (MSN)
  • Microsoft Download Service (MSDL)
  • Microsoft Product Support Services

For complete information, see the "To Obtain This Application Note" section at the end of this article.

INTRODUCTION

Connectix RAM Doubler version 1.0.4, which ships with Microsoft Office 4.2, cannot be installed on a Power Macintosh.

The MC1063 disk provided with this Application Note contains the Connectix RAM Doubler 1.5.1 Updater file, which you can use to install Connectix RAM Doubler version 1.5.1 on a Power Macintosh. Note that if you can install Connectix RAM Doubler version 1.0.4 on your Macintosh (that is, if you do not have a Power Macintosh), you can use this same file to update your current version of Connectix RAM Doubler to version 1.5.1. Connectix RAM Doubler version 1.5.1 provides the following enhancements to version 1.0.4:

  • Compatibility with the Power Macintosh. Connectix RAM Doubler 1.5.1 supports Macintosh computers equipped with a PowerPC(TM), 68040, 68030, or 68020 (with Paged Memory Management Unit [PMMU]) processor.
  • Compatibility with SCSI Manager 4.3.
  • Improved functionality with GeoPort on the Power Macintosh.
  • Compatibility with CD AllCache.
  • Increased reliability with file sharing on the Power Macintosh.
  • File mapping for the Power Macintosh. File mapping is a technology that allows for a reduction in memory requirements by using the memory management unit to manage the loading and unloading of native code segments.
  • Improved reliability with 4- and 5-megabyte (MB) Macintosh computers running in 24-bit addressing mode.
  • Better compatibility with Apple(R) System 7.5.

UPDATING TO CONNECTIX RAM DOUBLER 1.5.1

To use the Connectix RAM Doubler 1.5.1 Updater file on the enclosed MC1063 disk, you must have one of the following:

  • A master disk containing version 1.0.4 or earlier of Connectix RAM Doubler Installer.

    -or-

  • Version 1.5 or earlier of Connectix RAM Doubler installed on your computer.

    NOTE: To find out about known incompatibilities with version 1.5.1 of Connectix RAM Doubler and to find out how to contact Connectix Corporation, read the Connectix RAM Doubler Read Me file on the enclosed MC1063 disk.

Updating to Version 1.5.1 If You Are Using a Power Macintosh

If you are using a Power Macintosh, you need to create a master disk containing version 1.0.4 of Connectix RAM Doubler Installer, and then you need to update the installer on the master disk to version 1.5.1. You can then use the updated installer on this master disk to install Connectix RAM Doubler 1.5.1 on your computer.

To create a master disk that contains version 1.0.4 of the Connectix RAM Doubler Installer

  1. Insert a blank floppy disk in your Macintosh floppy disk drive and change the name of the disk icon to RAM DOUBLER. (For information about naming icons and copying files on your Macintosh, see the "Macintosh User's Guide".)

  2. Select the RAM Doubler disk icon, and choose Eject Disk from the Special menu. Insert Microsoft Office version 4.2 Install Disk 1, double-click the Install Disk 1 icon, and then double-click the Special Offer--RAM Doubler folder.

  3. From the Special Offer--RAM Doubler folder, copy the RAM Doubler Installer and RAM Doubler Read Me files to the RAM Doubler disk.

  4. Drag the RAM Doubler disk and Install Disk 1 icons to the Trash to remove them from the desktop.

To update the installer on the master disk

  1. Create a new folder on your hard disk called 1.5.1 Update.

  2. Insert the MC1063 disk in your floppy disk drive and copy the RAM Doubler 1.5.1 Updater and RAM Doubler 1.5.1 Updater Read Me files to the 1.5.1 Update folder on your hard disk. Drag the MC1063 disk icon to the Trash to remove it from the Macintosh desktop.

  3. Insert the RAM Doubler master disk in the floppy disk drive, and then double-click the RAM Doubler 1.5.1 Updater icon in the 1.5.1 Update folder on your hard disk.

  4. In the Connectix RAM Doubler 1.5.1 Updater dialog box, choose the Update Master Disk button.

When the Connectix RAM Doubler Updater has successfully updated the Installer, a dialog box will appear confirming the update.

To install Connectix RAM Doubler 1.5.1 on your computer

  1. With the RAM Doubler master disk in your floppy disk drive, double-click the updated Connectix RAM Doubler Installer icon.

    NOTE: The OLD folder on the RAM Doubler master disk contains the old version of the Connectix RAM Doubler Installer and Connectix RAM Doubler Read Me files. Do not double-click the Connectix RAM Doubler icon in this folder.

  2. In the Connectix RAM Doubler dialog box, choose Install.

  3. Enter the appropriate information in the Your Name and Company boxes, and choose OK.

  4. Choose the Restart button in the dialog box that appears with the message that Connectix RAM Doubler has been installed.

UPDATING TO 1.5.1 IF VERSION 1.5 OR EARLIER CAN BE INSTALLED ON YOUR COMPUTER

  1. If you have not already installed a version of Connectix RAM Doubler, run the Connectix RAM Doubler Installer that ships with Microsoft Office version 4.2.

  2. Insert the enclosed MC1063 disk in your floppy disk drive and double-click the disk icon to open it.

  3. Double-click the Connectix RAM Doubler 1.5.1 Updater icon on the MC1063 disk.

  4. In the Connectix RAM Doubler 1.5.1 Updater dialog box, choose the Update button to update the version of Connectix RAM Doubler that is installed on your hard disk.

  5. Choose OK when the dialog box appears with the message that your Connectix RAM Doubler extension has been updated to 1.5.1.
