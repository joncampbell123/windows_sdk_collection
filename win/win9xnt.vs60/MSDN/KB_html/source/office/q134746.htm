OFF: Customizing Microsoft Office Manager on the Macintosh

 Last reviewed: March 13, 1998
Article ID: Q134746
The information in this article applies to:
  • Microsoft Office for Macintosh versions 4.2, 4.2.1

SUMMARY

The Microsoft Office Manager (MOM) that is installed with the versions of Microsoft Office for the Macintosh and Power Macintosh listed above, can be customized from the default settings. However, the extent of customization varies for many of the items on the Office Manager menu.

MORE INFORMATION

By default, the Microsoft Office Manager control panel window (click Customize on the Office Manager menu), has the following items selected:

  • Quick Start
  • All installed Office applications (such as Microsoft Excel, Microsoft Word and Microsoft PowerPoint)
  • All of the Office components (Find File, Setup and Uninstall, Office Cue Cards, and Office Help)

If you installed Microsoft Office from CD, then there is also a listing for Online Documentation and this is also selected. If there are other Microsoft applications installed on your computer, they too will be selected in the Microsoft Office control panel window. This list includes:
  • Microsoft Mail
  • Microsoft FoxPro
  • Microsoft Works
  • Microsoft Project
  • Microsoft Schedule+

NOTE: If you did a custom setup of Microsoft Office and chose not to install the Office Manager Optional Files, the Setup And Uninstall, Office Cue Cards, and Office Help items will still be selected in the Office Manager Control Panel window. However, if you try to start one these items from the Office Manager menu, you will get an alert message asking you to run setup and reinstall the item(s).

The following table shows the default items listed in the Office Manager control panel window (click Customize on the Office Manager menu) and the extent to which you can customize the items. 

   Item                                  Edit    Remove    Move
   ------------------------------------------------------------

   Quick Start                                               x
   Microsoft Applications                  x*                x
   Finder                                                    x
   Find File
   Setup and Uninstall
   Office Cue Cards
   Office Help
   Online Documentation                    x        x        x
   Separators                                                x
   Applications/files added by you         x*       x        x
Note that for the default Microsoft applications listed in the Office Manager control panel window, when you Edit the item, you are able to change where Office Manager looks for these applications. However, when you Edit applications or files that you have added to Office Manager, you are only able to change the name that is displayed on the Office Manager menu. If you want to specify a different location for your application or file, you must first Remove the original item from the Office Manager and then create a new item that points to the new location.
