OL97: Outlook Hangs Printing Detailed Address Card View

 Last reviewed: January 19, 1998
Article ID: Q169054
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you attempt to print your Microsoft Outlook 97 Contacts from the Detailed Address Card view, to a Canon BJC-210 printer using Card Print Style, the print process may appear to stop responding (hang) with no error message.

CAUSE

You may have an older printer driver with a large nonprintable region near the edge of the page.

RESOLUTION

Try the following methods to work around this problem:

Method 1

You may be able to print Detailed Address Cards by resetting the default 0.5 inch margins to a larger setting, such as 1.0 inches. To change the margins follow these steps:

  1. On the File menu, click Print.

  2. Ensure you have the correct printer selected and the Print style is set to Card Style.

  3. Click Page Setup and then click the Paper tab.

  4. Under Margins, change all settings to 1.00 inches.

  5. Click OK and OK to print.

Method 2

You may be able to print a Phone List view without changing the margin settings. On the Outlook Standard toolbar's Current View list, click to select Phone List and then print.

Method 3

Contact Canon for an updated BJC-210 printer driver. This problem occurs with the following newer Canon printer driver that ships with the printer:

Use the Canon Bubble-Jet BJ-200 printer driver, version 3.2, which ships with Microsoft Windows 95.
