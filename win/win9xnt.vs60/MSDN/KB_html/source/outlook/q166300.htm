How to Check Your Information Service Files

You can determine the following information about files installed on your computer by a particular information service:

  • File names
  • Description
  • Company
  • Version
  • Language
  • Creation date
  • Size

Follow these steps to display the above information on your computer:

  1. On the Windows Taskbar, click Start, point to Settings and click Control Panel.

  2. Double-click either the Mail or the Mail and Fax icon to open your default mail profile properties.

  3. If you have more than one profile, click Show Profiles and select the profile which contains the information service you want to check. If you only have one profile, you may skip this step.

  4. On the Services tab, click to select the information service.

  5. Click About to open the About Information Service dialog.

The About Information Service dialog provides all the information about files included with the selected service. By clicking each file in the File name list, you can see the description, company, version, language, creation date, and file size information.
