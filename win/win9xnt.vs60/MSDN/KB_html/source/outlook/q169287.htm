OL97: Import Wizard Procedure Incomplete

 Last reviewed: January 19, 1998
Article ID: Q169287
The information in this article applies to:
  • Microsoft Outlook 97

WARNING: The information in this article has not been confirmed or tested by Microsoft. Some or all of the information in this article has been taken from unconfirmed customer reports. SYMPTOMS

In Microsoft Outlook 97, the Import Wizard may display the following behavior:

  • On the last pane, the Finish button is not available. 

        -and-
  • You cannot check any of the items beneath "The following actions will be performed:" 

        -and-
  • The Change Destination button results in a dialog box with no folders from which to select.

CAUSE

The user profile is corrupted.

RESOLUTION

Create a new user profile and run the Import Wizard again. For more information on setting up a user profile, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID:  Q162203
   TITLE     :  OL97: User Profiles and Information Services
