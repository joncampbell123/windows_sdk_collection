OL97: Notes Are Maximized When Opening or Creating New Notes

 Last reviewed: January 19, 1998
Article ID: Q167993
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

When you create a new note or open an existing note in Outlook, it may display in a maximized window that fills the entire screen.

MORE INFORMATION

If the last note you had open in Outlook was displayed in a Maximized window, all new and existing notes will also display in a Maximized window.

To make Outlook open all notes in a restored window, use the following steps:

  1. Open an existing note or create a new note in Outlook.

  2. Right-click the Control-menu box (the button at the far upper-left corner of the note).

  3. Click Restore.

Existing notes will open up in the same size window they were opened in last. The size of the window for new notes is determined by the size specified in the options for Outlook. To change this window size, use the following steps:

  1. On the Tools menu, click Options.

  2. Click the Tasks/Notes tab.

  3. In the Size field, select Small, Medium, or Large.

  4. Click OK.

Now all new notes will open in a window the size you selected in step 3.
