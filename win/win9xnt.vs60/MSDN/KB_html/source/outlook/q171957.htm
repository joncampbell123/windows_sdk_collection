OL97 ErrMsg: Outlook.stf Is Not from a Post Admin Network Image

 Last reviewed: January 19, 1998
Article ID: Q171957
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you run the Network Installation Wizard from the Microsoft Office 97 Resource Kit in order to modify the Outlook.stf file, you may receive the following error message: 

   STF File- Outlook.stf is not from a post admin network image.
   Please check file and try again.

RESOLUTION

To correct this problem run setup.exe /a to do an administrative installation to an empty directory on the network server. Then, run the Network Installation Wizard to customize the Outlook setup files in the new server directory.

MORE INFORMATION

The Network Installation Wizard version 2.0, is a tool that allows you to modify the Office 97 Setup information files to create a customized network installation of Office or Office programs. The wizard is included in the Microsoft Office 97 Resource Kit.

REFERENCES

For more information about using the Network Installation Wizard, see Chapter 6, "Customizing Client Installations," of the Microsoft Office 97 Resource Kit.
