OL97 DocErr: Microsoft Network Uses Internet Mail Service

 Last reviewed: January 19, 1998
Article ID: Q165484
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Office Resource Kit 97

SUMMARY

The following documentation errors pertain to using the Microsoft Network (MSN) online service with Microsoft Outlook 97:

The Outlook Help topics,"E-mail information services provided with Outlook" and "Install an information service" state: 

     The Microsoft Network online service (Use the Internet Mail service
     provider to use this service.)
This is incorrect, The help topic should say: 

     The Microsoft Network online service (Use the Microsoft Network
     service, which is available after you install the Microsoft Network
     online service on your computer).
Outlook 97 does not provide the MSN online transport service. MSN online e-mail does not work with the Internet Mail transport service.

The topic, "Using E-mail Information Services Provided with Outlook," in Chapter 19, "Switching to Microsoft Outlook," of the Microsoft Office 97 Resource Kit, states the following: 

     MSN, The Microsoft Network

     You can use the Internet Mail service provider to use this online
     service.
This is incorrect, The topic should say: 

     MSN, The Microsoft Network

     You can use the Microsoft Network service, which is available after
     you install the Microsoft Network online service on your computer.
Microsoft Network online e-mail does not work with the Internet Mail transport service.

MORE INFORMATION

The reference to "Internet Mail service provider" should say the Microsoft Network online service transport, provided by MSN Setup. If you have properly installed the Microsoft Network online service on your computer, you can add the Microsoft Network service to Outlook. MSN provides the transport service; MSN will not work using the Internet Mail service.
