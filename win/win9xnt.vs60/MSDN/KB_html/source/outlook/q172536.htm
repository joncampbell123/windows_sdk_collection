OL97: Synchronize Option Not Listed on the Tools Menu

 Last reviewed: January 19, 1998
Article ID: Q172536
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When using Microsoft Outlook 97 with Microsoft Exchange Server, the Tools menu may not list Synchronize as a command.

CAUSE

The Microsoft Exchange Extensions add-in is missing or not active.

RESOLUTION

To activate or install the Microsoft Exchange Extensions add-in, follow these steps:

  1. On the Tools menu, click Options, and then click the General tab.

  2. Click Add-In Manager.

  3. If Exchange Extensions is listed, but is not selected, click to select Exchange Extensions and then click OK.

  4. If Exchange Extensions in not listed, follow these steps: 

        a. Click Install to open your \Office\Addins folder.


        b. Select Emsuix.ecf, and click Open.


        c. You should see Exchange Extensions listed in the Add-Ins list.
       Click OK and OK again to close all dialog boxes.

Synchronize should now be a command on the Tools menu.

MORE INFORMATION

When you use Outlook with the Microsoft Exchange Server Information Service, you have the ability to work with folders offline. That is, you can use the contents of a folder without a network connection. This feature is beneficial for remote users. The Synchronize option is used to synchronize your offline folders with the folders in your mailbox on the Microsoft Exchange Server.

For more information on synchronizing offline folders, please see the following article in the Microsoft Knowledge Base: 

   Article-ID:   Q161725
   Title     :   OL97: What Are Offline Folders and How Do You Use Them?
