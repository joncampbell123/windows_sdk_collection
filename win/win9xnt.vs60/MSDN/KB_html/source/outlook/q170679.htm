OL97: Must Close All Outlook Windows to Log On as Different User

 Last reviewed: January 19, 1998
Article ID: Q170679
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you try to shut down Windows 95 by selecting "Close all programs and log on as a different user", Outlook displays the following message: 

   Please close all Outlook windows before shutting down the system. The
   easiest way to do this is to choose Exit and Log Off from the File menu.

CAUSE

This behavior is by design. Windows 95 cannot correctly close Outlook when you try to log off, and log on as a different user.

RESOLUTION

To prevent this message, close all Outlook windows before attempting to close all programs and log on as a different user.
