OL97: Internet E-mail Service Displays Unnecessary Message

 Last reviewed: January 19, 1998
Article ID: Q168647
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

Clicking OK in the Internet E-mail Service Properties dialog box displays the following message even if no changes where made to any setting: 

   Configuration changes will not take effect until the next time
   you log on to Internet Mail.

MORE INFORMATION

The Internet Mail Enhancement Patch for Microsoft Outlook 97 does not check to see if any changes were made to the settings and always displays the message.

Steps to Reproduce Behavior

  1. On the Tools menu, click Services.

  2. Click the Internet E-mail service.

  3. Click Properties and without making any changes, click OK.

  4. The message referring to configuration changes is displayed even though no changes were made.
