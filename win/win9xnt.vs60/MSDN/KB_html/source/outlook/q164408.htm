OL97: Imported Fax Numbers May not Work in Outlook

 Last reviewed: January 19, 1998
Article ID: Q164408

The information in this article applies to:

  • Microsoft Outlook 97

SYMPTOMS

When you import data containing Fax numbers into Outlook 97 Contacts, you may experience one of the following situations when you try to use the Fax number:

  • If the number is in your area code, Outlook may attempt to dial the area code even though it is not needed.
  • If the number is in a different area code, Outlook may fail to dial 1 preceding the area code.
  • If you require a preceding digit, such as a 9, to access an outside line, Outlook may fail to dial the preceding digit.

In all cases, the connection will be unsuccessful and the Fax will not be sent.

CAUSE

Outlook does not store the incoming Fax number in the proper format in the Contact record. The format may look correct such as, (123)456-7890, but still not work correctly.

RESOLUTION

To correct this problem, you must force Outlook to reformat and store the Fax number correctly by editing each Fax number and resaving the Contact. This will cause Outlook to reformat and store the Fax number correctly. See Steps to Correct the Problem, below.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97 for Windows. This problem was corrected in Microsoft Outlook 97 version 8.02 for Windows.

MORE INFORMATION

Steps to Correct the Problem

To reformat and store the Fax number correctly, do the following:

  1. Open the Contact containing the Fax number.

  2. Retype the Fax number or press SPACEBAR + BACKSPACE at the end of the Fax number.

  3. Click Save and Close.

This will cause Outlook to reformat and store the Fax number correctly.

Steps to Reproduce the Problem

  1. On the File menu, click Import and Export.

  2. Click "Import from Schedule+ or another program or file" and click Next.

  3. Click the file type to be imported and click Next.

  4. Type the name of a file containing Fax numbers and click Next.

  5. Click Import "<filename>" into the "Contacts" folder then click Map Custom Fields.

  6. Make sure the incoming Fax number is mapped to the proper Contact Fax number field and click OK and then click Finish

Send a Fax using the number imported above. You may experience one of the problems described under SYMPTOMS above.
