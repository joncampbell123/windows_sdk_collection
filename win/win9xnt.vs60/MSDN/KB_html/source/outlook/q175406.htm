OL97: Allow Upgrades to Large Tables Extends Limit of PST

 Last reviewed: January 19, 1998
Article ID: Q175406
The information in this article applies to:
  • Microsoft Outlook, version 8.03

SUMMARY

In Microsoft Outlook version 8.03, a new feature, "Allow upgrade to large tables," is available for Personal Folder (.pst) files. This feature increases the limit on the number of folders per file and the number of messages per folder in a .pst file. The limit has been increased from 16 KB folders per file and messages per folder to approximately 64 KB.

MORE INFORMATION

Making this Personal Folders option available causes the headers of the .pst file to be permanently updated. Once this feature has been made available for a specific file, the option will be unavailable in the dialog box.

NOTE: Previous versions of Microsoft Exchange and Outlook may be able to access a .pst file that has been updated to support the large table format. However, this is not supported and may cause instability and corruption in the .pst file.

Allow Upgrade to Large Tables

To make this feature available in Outlook, follow these steps:

  1. On the Tools menu, click Services.

  2. On the Services tab click to select the Personal Folders service and click Properties.

  3. On the General tab click to select "Allow upgrade to large tables."

  4. Click OK.
