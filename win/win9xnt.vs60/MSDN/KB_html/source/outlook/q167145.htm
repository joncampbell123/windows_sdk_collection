OL97: Headers Not Visible on Options Tab of Message

 Last reviewed: January 19, 1998
Article ID: Q167145
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you use the Internet Mail service in Microsoft Outlook 97 and open an e- mail message, you are unable to view the Internet Headers on the Options tab of the message.

RESOLUTION

To view the Internet headers on the Options Tab in an Internet Mail message, install the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP). The IMEP is available on the Microsoft Internet site at: 

   http://www.microsoft.com/OfficeFreeStuff/Outlook/
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. The problem is corrected in the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP).

MORE INFORMATION

The Internet Headers include information about the sender's Internet location, the date and time the message was sent, the message-ID, the subject of the message, and may also have other details about the path the message has taken en route to your inbox.

To view the Internet headers without the IMEP:

  1. Click to select the message in your Inbox.

  2. On the File menu, click Properties.

  3. Click the Internet tab. The details are displayed on this tab.
