OL97: Help Conflict between Rules Wizard and Outlook 8.02

 Last reviewed: January 19, 1998
Article ID: Q175516
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you install the Microsoft Outlook 97 Rules Wizard add-in, new Outlook help files installed with the Microsoft Office 97 SR-1 patch are no longer available.

CAUSE

The Office SR-1 patch installs a new Outlhlp.cnt file that contains links to new help files designed for Outlook 8.02. When you install the Rules Wizard, the Outlhlp.cnt file is overwritten with the old file designed for Outlook 8.0, and the links to the new files are lost.

RESOLUTION

Copy the newer Outlhlp.cnt file from the Office directory of the SR-1 CD over the older one, usually found in the Program Files\Microsoft Office\Office, directory.

MORE INFORMATION

If you install the Rules Wizard from either the ValuPack or from the Microsoft Web site, after installing the SR-1 patch, the new Outlhlp.cnt file for Outlook 8.02 will be overwritten with the old file designed for Outlook 8.0.
