OL97: Error Adding Lotus cc:Mail to Profile on Windows NT

 Last reviewed: January 19, 1998
Article ID: Q161987

The information in this article applies to:

  • Microsoft Outlook 97

SYMPTOMS

If you use Microsoft Outlook 97 on Windows NT and try to add the cc:Mail service to a profile, you may receive the following error message: 

   VIM32.DLL could not be found and is required to access the cc:Mail
   post office when using NT. This DLL can be obtained from Lotus at:

      http://www.support.lotus.com
For additional information contact your system administrator or visit Transend's web site at: 

      http://www.transend.com

   Or send e-mail to:

      support@transend.com

CAUSE

You are missing the required Lotus 32-bit VIM (vendor-independent messaging) files. Microsoft does not provide these files. If you use Windows NT and have not previously installed a 32-bit version of Lotus cc:Mail, these files are not present in your \WINNT\SYSTEM32 directory.

The cc:Mail service from the Office 97 ValuPack requires the presence of these files on Windows NT.

WORKAROUND

You should follow the instructions in the error message to download and install the 32-bit VIM files to your \WINNT\SYSTEM32 directory. Both Web sites link to the Lotus file, Vdlw32.zip, which contains the following 32- bit VIM files: 

  Cdatew32.dll   Cfilew32.dll   Chrset32.dll   Cintlw32.dll   Cmdtew32.dll
  Cmemew32.dll   Meapi32.dll    Medb632.dll    Vim32.dll      Vmefnw.dll
Unzip Vdlw32.zip and copy these files to your \WINNT\SYSTEM32 directory. You can then add the cc:Mail service to your profile.

REFERENCES

cc:Mail is manufactured by Lotus Development Corporation, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

For more information about installing the Lotus cc:Mail service, double- click to open the file Valupk8.hlp in the Microsoft Office 97 ValuPack. Under the topic "What do you want to know about?" click "Lotus cc:Mail information service."
