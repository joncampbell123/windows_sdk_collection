OL97: Information on Messages with a Message.txt Attachment

 Last reviewed: January 19, 1998
Article ID: Q167266

The information in this article applies to:

  • Microsoft Outlook 97

SUMMARY

When you use the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP), you may receive a message in your Inbox that contains the following text: 

   This message could not be converted. To report this problem to
   Microsoft, please forward this message to imepbugs@microsoft.com.
The message will also have an attachment file named, Message.txt

NOTE: This problem is usually caused by a message with a blank "Reply To" line.

MORE INFORMATION

If Outlook Internet Mail cannot fully convert an incoming message when downloading from your Internet Service Provider (ISP), it generates the message above and attaches the original message as a text file. You can read the original text of the incoming message by double-clicking the Message.txt file. If Message.txt is not legible, save the attachment as an .eml file then open it in the Microsoft Internet Explorer Mail and News reader.

To read the file with Microsoft Internet Explorer, follow these steps:

  1. With the message open, on the File menu, click Save Attachments.

  2. In the File name box, change the file extension from .txt to .eml and click Save.

  3. Double-click the file in Windows Explorer.

The file will open as a mail message in the Microsoft Internet Explorer Mail and News reader.

REFERENCES

NOTE: Earlier versions of the Outlook Internet Mail service would cease to download mail from the ISP when encountering a message with this problem. For more information on this issue, please see the following article in the Microsoft Knowledge Base:
