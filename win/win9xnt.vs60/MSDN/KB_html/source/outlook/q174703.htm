OL97: Printing Select Appointments in Day/Week/Month Styles

 Last reviewed: January 19, 1998
Article ID: Q174703
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

When you print a date-range of appointments using any of the Microsoft Outlook 97 Day/Week/Month styles, all appointments within the date-range are printed by default. This article describes how to selectively print appointments within a date-range.

MORE INFORMATION

To selectively print appointments in the Day/Week/Month styles, first assign a category code either to those appointments you want to print or to those appointments you want to exclude. Then define a filtered view to either include or exclude that category code.

To Assign a Category Code to an Appointment

  1. Right-click on the appointment and click Categories on the shortcut menu.

  2. Click to select a Category code from the list of "Available categories:" and click OK.

To Define a Filtered View of the Appointments

  1. On the View menu, click Filter.

  2. If you want to print only appointments with a specified Category code:

    a. On the More Choices tab, click Categories.

    b. In the Available Categories list, click to select the category 

            you want to display on the printout.

  3. If you want to exclude those appointments with a specified category code from printing, follow these steps:

    a. On the Advanced tab, click Field, point to All Appointment fields, 

            and click Categories.

   b. In the Condition list, click Doesn't Contain.

   c. In the Value box, type the category code you want to exclude
      from the printout.

To Remove the Filter

  1. On the View menu, click Filter.

  2. In the Filter dialog box, click Clear All, and then click OK.
