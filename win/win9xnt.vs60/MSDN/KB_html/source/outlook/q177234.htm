OL97: How to Import Calendar Information From Calendar Creator

 Last reviewed: January 19, 1998
Article ID: Q177234
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

Microsoft Outlook 97 does not have a converter for Calendar Creator Plus, by Spinnaker Software. The files must be exported as Comma Separated Value (CSV) files, and then imported into Outlook.

MORE INFORMATION

To import a file from Calendar Creator Plus export as a CSV file and then import the CSV file into Outlook.

Save the Calendar Creator Plus File as a CSV File

  1. On the File menu, point to Save As, then click to select the Event list you want to export.

  2. In Save Event List type a file name.

  3. In the list, Save File As Type click to select Comma Delimited (.csv). Make a note of the folder where the file will be saved then click OK.

  4. In the "Specify date range" dialog type the Starting and Ending dates you want to export then click OK.

  5. On the File menu, click Exit.

Import the CSV File into Outlook

  1. On the File menu, click Import and Export.

  2. Click to select "Import from a Schedule+ or another program or file", and click Next.

  3. Click to select "Comma Separated Values (Windows)" then click Next.

  4. Click Browse and locate the folder where the CSV file was saved.(Step 3 in the "Save the Calendar Creator Plus File as a CSV File" section.) Click to select the file, and click Open. Click Next.

  5. Click to select your Calendar as the import location.

  6. Click Map Custom Fields. Click to clear "First record contains field names." Drag the fields on the right to the appropriate field on the left. Use the table below as a guide for field mapping. 

       Calendar Creator Plus field    Outlook field
   ---------------------------    -------------

   Field 1                         Start Date
   Field 2                         Start Time
   Field 3                         End Time
   Field 4                         Subject
   Field 5                         Priority
   Field 6                         Category
   Field 7                         <none, contains Picture files>
   Field 8                         <none, note information>

   There is no Note field in Outlook; you may use it for another
   field, such as Location.

  7. Click OK, and click Finish.

You should see your information in the Outlook Calendar.
