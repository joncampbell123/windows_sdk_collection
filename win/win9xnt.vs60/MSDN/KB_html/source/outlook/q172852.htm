OL97: Configuring AT&T Worldnet in Outlook IMEP

 Last reviewed: January 19, 1998
Article ID: Q172852
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes how to configure the AT&T Worldnet Internet Service Provider (ISP) for use with the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP).

MORE INFORMATION

To configure the AT&T Worldnet ISP, follow these steps:

  1. Setup the Worldnet E-mail service:

    a. On the Outlook Tools menu, click Services.

    b. On the Services tab, click Add.

    c. Click Internet E-mail and click OK.

  2. Complete the General tab:

    a. Type a name to identify this service.

    b. Type a Name to identify this user.

    c. Type the user's E-mail address. This is the user's alias followed by 

          @worldnet.att.net.


             Example: johndoe@worldnet.att.net

  3. Complete the Servers tab:

    a. In the Outgoing Mail window, type mailhost.worldnet.att.net, in 

          lowercase.

    b. In the Incoming Mail window, type postoffice.worldnet.att.net, in 

          lowercase.

    c. Click to select Logon using.

    d. In the Account Name window, type the portion of the e-mail name 

          (chosen by the user) preceding the @ sign.


             Example: if the e-mail name is, johndoe@worldnet.att.net, use
         johndoe

   e. In the Password window, type the password chosen by the user. The
      password is case sensitive.

  4. Complete the Connection tab:

    a. Click to select, "I use a modem to access my e-mail."

    b. Click Add.

    c. Type a name to identify this connection, such as, WORLDNET.

    d. Select your modem from the list and click OK.

    e. Type in the Area code and Telephone number of your Worldnet 

          connection, and then click Next and Finish.

  5. Define the Server type:

    a. Click Properties and Server Type.

    b. In the Type of Dial-Up Server window, select "PPP:Windows 95, 

          Windows NT 3.5, Internet."

    c. Click TCP/IP Settings.

    d. Click to select Server assigned IP address.

    e. Click to select "Specify Name server addresses" and enter the 

          Primary Domain Name Service (DNS) and Secondary DNS. Worldnet
      supplies these addresses.

    f. Click to select "Use IP header compression" and "Use default 

          gateway on remote network."

  6. Finish the Worldnet Service definition:

    a. Click OK and OK and OK.

    b. Restart Outlook.

  7. Complete the "Connect To..." information:

    a. On the Tools menu click, "Check for new mail."

    b. Enter the network access ID supplied by Worldnet, this will be a 

          nine digit number followed by @worldnet.att.net.


             Example: 123456789@worldnet.att.net

   c. Enter the Network access password assigned by Worldnet. This is
      usually in the form of two words separated by a hyphen.

   d. Click to select, Save Password.
NOTE: It is important to remember that the Account/E-mail Name and password, referred to in step 3 above, are chosen by the user, but the Network Access ID and password, referenced in step 7, are assigned by Worldnet.
