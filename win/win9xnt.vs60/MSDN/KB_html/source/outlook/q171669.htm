OL97: Options Available in Outlook for Deleting Items

 Last reviewed: January 19, 1998
Article ID: Q171669
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes several options for deleting items within Microsoft Outlook 97.

MORE INFORMATION

When you click to select an item in the Information Viewer area of Outlook, you have several methods available for deleting the item. The behavior of Outlook varies, depending on which method you use to delete the item.

Moving the Item to the Deleted Items Folder from Another Folder

  • On the Standard Toolbar, click the Delete button.
  • On the Edit menu, click Delete.
  • Press CTL+D.
  • Press DELETE.
  • Hold the SHIFT key and on the Standard Toolbar, click the Delete button.
  • Drag the item to the Deleted Items on the Outlook Bar or in the Folder List.
  • Drag the item to the Windows Recycle Bin.

Permanently Deleting the Item Without Moving it to the Deleted Items Folder

  • Press SHIFT+DELETE.
  • Once an item is in the Deleted Items folder, using any of the methods in the previous section permanently deletes the item.

Automatically Emptying the Deleted Items Folder Upon Quitting Outlook

If you set an option to empty the Deleted Items Folder when quitting Outlook, all items in the folder are permanently deleted whenever you quit Outlook. To set this option, follow these steps:

  1. On the Tools menu, click Options to open the Options dialog box.

  2. Click the General tab.

  3. Click to select "Empty the Deleted Items folder upon exiting," and then click OK.

Using AutoArchive Settings to Delete Items in Deleted Items Folder

You can set AutoArchive to delete items from the Deleted Items folder after a set number of days. To accomplish this, you must not turn on the option to empty the Deleted Items folder when quitting Outlook. Because AutoArchive acts on the last modified date, re-opening and modifying the items in the Deleted items folder delays the permanent deletion. The following steps set up the AutoArchive to delete items from the Deleted Items folder after seven days:

  1. On the Standard Toolbar, click the Folder List button to turn on the Folder List.

  2. Using your right mouse button, click the Deleted Items folder, and on the shortcut menu, click Properties.

  3. Click the AutoArchive tab.

  4. Click to select "Clean out items older than 7 days," and then click "Permanently delete old items."

  5. Click OK.
