OL97: Mailto Function Includes Only E-mail Address

 Last reviewed: January 19, 1998
Article ID: Q169076
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you click a mailto Universal Resource Locator (URL) in the body of an e- mail message in Microsoft Outlook 97, a new, untitled e-mail message starts. The new message is automatically addressed to the recipient indicated in the mailto, but none of the other fields in the message header are completed.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97 for Windows. This problem was corrected in Microsoft Outlook 97 version 8.02 for Windows.

RESOLUTION

Outlook supports the basic mailto function having the following format: 

   someone@microsoft.com
Outlook does not support other formats, including: 

   mailto: someone@microsoft.com?subject=Subject of Message

   mailto: someone@microsoft.com?subject=Subject of Message&cc=others@
   microsoft.com

   mailto: someone@microsoft.com;someone1@microsoft.com?subject=Subject of
   Message
