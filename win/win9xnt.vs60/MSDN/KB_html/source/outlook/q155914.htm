OL97: How to Change Colors of Outlook Bar and Folders Banner

 Last reviewed: January 19, 1998
Article ID: Q155914
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

From within Microsoft Outlook 97, you cannot change the colors of the Major components such as the Outlook bar, Folders banner, background, and so on.

CAUSE

The Windows display appearance properties define the Outlook color scheme.

WORKAROUND

To change Outlook component colors, change the global Windows color scheme by following these steps:

  1. Using the right mouse button, click the Windows Desktop, and then click Properties on the menu that appears.

  2. In the Display Properties dialog box, click the Appearance tab.

  3. From the Scheme drop-down box, select a different Windows color scheme, or from the Item drop-down box, change specific Windows element colors.

  4. Click OK.

REFERENCES

For more information about changing the Windows color scheme, search on "Colors," using the Windows 95 Help Index.
