OL97: Problem Printing Multiple Excel Attachments

 Last reviewed: January 19, 1998
Article ID: Q172994
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

Microsoft Outlook 97 can print multiple e-mail attachment files with the original message. However, when you attempt to print multiple Microsoft Excel attachments, some of the attachments may not be printed.

The original e-mail message is printed followed by the first Excel attachment, and only some of the additional attachments if there are more than two.

This problem does not occur with Microsoft Word attachments.

CAUSE

This problem occurs when you are not using Microsoft Word (WordMail) as the e-mail editor in Outlook, and you select the "Print attached files with item(s)" check box in the Print dialog box.

If you are using WordMail, this check box does not appear in the Print dialog box. You must always print each attachment individually.

WORKAROUND

To work around this problem, print each attachment individually. To do this, follow these steps:

  1. Right-click the attachment.

  2. On the shortcut menu, click Print.

NOTE: If you select multiple attachments by pressing SHIFT, only the first attachment is printed. Other selected attachments are not printed. This behavior is the same regardless of the type of attachment.

MORE INFORMATION

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
