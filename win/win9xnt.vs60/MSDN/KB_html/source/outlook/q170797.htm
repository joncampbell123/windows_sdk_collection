OL97: Manual Journal Entry Does not Default to Current Time

 Last reviewed: January 19, 1998
Article ID: Q170797
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

In Microsoft Outlook 97, a manual Journal entry may open with a default Start time other than the current time.

CAUSE

You set the current Journal folder view to one of the grouped options such as By Type, By Contact, or By Category. In these time-line views, a new manual Journal entry has a default Start time of 12:00 AM, or any hour selected on the time-line bar.

RESOLUTION

Change your Journal folder view to one of the list views such as Entry List, Last Seven Days, or Phone Calls before opening a manual Journal entry.

-or-

Manually change the new Journal entry's Start time to the current time.

MORE INFORMATION

If you set the Journal folder view to By Type, the time-line bar displays hours. In this view, the current hour is highlighted by default and any new manual Journal entry has a default Start time on the even half-hour, such as 9:00 AM.

If you set the Journal folder view to By Contact, or By Category, the time- line bar displays dates. In this view, the current date is highlighted by default and any new manual Journal entry has a default Start time of 12:00 AM.
