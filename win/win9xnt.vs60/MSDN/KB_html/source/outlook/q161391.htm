OL97: Page Fault Setting Recurrence on Custom Task Form

 Last reviewed: March 17, 1998
Article ID: Q161391
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

In Microsoft Outlook 97, if you set a recurrence for an item in a custom Task form, you may receive a page fault error. When you click the Details button, you will receive the following message: 

   OUTLOOK caused an invalid page fault in
   Module OUTLLIB.DLL at 0137:6e2a5b1c.

CAUSE

This error occurs if the first tab of the custom Task form is hidden.

WORKAROUND

To display the first tab of your custom form, modify the custom form using the following steps:

  1. Create a new task based on the form.

  2. On the Tools menu, click Design Outlook Form.

  3. Click the first tab (task) of the form.

  4. On the Form menu, click to select Display This Page.

  5. Click Publish Form As on the File menu and click Publish.

STATUS

Microsoft has confirmed this to be a problem in the products listed at The beginning of this article. This problem was corrected in Microsoft Outlook 98.
