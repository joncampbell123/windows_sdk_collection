OL97: Translator Error Exporting to a File

 Last reviewed: February 20, 1998
Article ID: Q170786
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you attempt to use the Import and Export Wizard of Microsoft Outlook 97 to export a Journal, Notes, or Mail folder to a file, you may receive the following error message: 

    An error has occurred in the Microsoft Outlook translator while
    initializing a translator to build a field map.

    Internal error:  Unable to load File Access Engine (.FAE)DLL,
    additional Outlook translators can be installed from the ValuPack.

CAUSE

The only folders you can export to a file are, Calendar, Contacts, and Tasks folders.

RESOLUTION

In order to export the Journal, Notes, or Mail folders, you must install the converters from the ValuPack.

You can find the converters at the following locations:

  • The ValuPack on the Microsoft Office 97 or Outlook CD-ROM.
  • The ValuPack folder on the network server, Office or Outlook was installed from. (See your network administrator.)
  • The Microsoft Web site

If You Run Outlook or Office From Your Hard Drive

The extra converters are on the Office 97 or Outlook 97 CD-ROM in the following directory: 

   <CD-ROM Drive letter>:\ValuPack\Convert\Outlook
Make sure you run the appropriate file for your operating system. 

   Outcvt.exe     (Windows 95 and Windows NT 4.0)
   Outcvtnt.exe   (Windows NT 3.51)

If You Run Outlook From a CD-ROM or Networked Computer

If you run Outlook from a networked computer, follow the steps in the section of the Olreadme.txt file titled, "Using Import and Export Converters from the Microsoft Office 97 ValuPack When You Run Outlook From a CD-ROM or a Networked Computer."

Obtaining the Extra Converters From the Internet

The extra converters are available on the Microsoft Internet site at: 

   http://www.microsoft.com/OfficeFreeStuff/Outlook
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: http://www.microsoft.com/

REFERENCES

For more information about additional Outlook import and export converters, type "export converters" in the Office Assistant, click Search, and then click to view "File types you can import to or export from Outlook."
