OL97: Offline Folder Synchronization Error

 Last reviewed: January 19, 1998
Article ID: Q170632
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When synchronizing a Microsoft Outlook 97 offline folder, you may get the following error message: 

   Synchronization with the Offline folder file could not be
   completed. The attempt to logon to the Microsoft Exchange
   Server computer has failed.

RESOLUTION

Create a new user profile and offline store (OST) file.

MORE INFORMATION

To Create a New User Profile

  1. Quit Outlook.

  2. Click Start on the Windows Taskbar, point to Settings and click Control Panel.

  3. Double-click the Mail and Fax icon.

  4. Click Show Profiles then click Add.

  5. Follow the Setup Wizard to create a new profile with the required services.

To Create a New OST File

  1. Start Outlook with the user profile created above.

  2. On the Tools menu, click Services.

  3. Click the Microsoft Exchange Server service and then click Properties.

  4. On the Advanced tab, click "Offline Folder File Settings."

  5. Enter a path and name for a new OST file. Do not use the old name.

  6. Click Yes to create the new OST file.

  7. Click OK and OK, then quit and restart Outlook.
