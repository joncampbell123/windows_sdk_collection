OL97: Internet Tab Missing from Message Properties Dialog Box

 Last reviewed: January 19, 1998
Article ID: Q162149

The information in this article applies to:

  • Microsoft Outlook 97

SYMPTOM

In Microsoft Outlook 97 with Internet service installed, an e-mail message's Properties dialog box may not contain an Internet tab.

CAUSE

The Internet tab functionality comes from an installable add-in named Internet Mail. Internet Mail is stored in the extension configuration file, Minet.ecf, which you can install using the Add-in Manager. You can have the Internet Mail service installed and not have the Internet Mail add-in installed.

WORKAROUND

To install the Internet Mail add-in, follow these steps:

  1. On the Tools menu, click Options, and then click the General tab.

  2. Click the Add-In Manager button, and then the Install button to open your \Office\Addins folder.

  3. Select Minet.ecf, and click Open.

  4. You should see Internet Mail listed in the Add-Ins dialog box. Click OK and OK again to close all dialog boxes.

MORE INFORMATION

The Internet tab of a e-mail message's Properties dialog box specifies character set format and message format. With Internet services installed, the tab contains the following options:

Character Set Selection

  • Use Setting In Profile: Click this option if you want to use the profile settings.
  • Override The Profile Setting And Use: Click this option if you want to override the settings in the profile for this outgoing message. In the Character Set box, click the character set to be used for text messages (ISO 8859-1, Norwegian, Swedish, or US-ASCII). Only this message is set to a specific character set format.

Message Format Selection

  • Use Settings In Profile: Click this option if you want to use the profile settings.
  • Override The Profile Setting And Use: Click this option if you want to override the settings in the profile for this outgoing message. You can override a UUENCODE profile setting by changing this message option to MIME, or you can override a MIME setting by changing this message option to UUENCODE. Only this message is set to a specific format.

REFERENCES

For more information about installing add-ins, type "add-in" in the Office Assistant, click Search, and then click to view "Load, install, or remove an add-in."
