OL97: View Filter Fields Are Limited to 255 Characters

 Last reviewed: January 19, 1998
Article ID: Q170654
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

In Microsoft Outlook 97, to filter the view of Inbox messages, Calendar entries, Contacts, and so forth, click Filter on the View menu.

NOTE: You can type no more than 255 characters in the following three boxes in the Filter dialog box:

  • "Search for the word(s)" (on the Messages tab)
  • Categories (on the More Choices tab)
  • Value (on the Advanced tab)

MORE INFORMATION

Search for the Words

On the Messages tab, type any words in the "Search for the word(s)" box that you want to find in the field specified by the In drop-down list.

More Choices

On the More Choices tab, type any categories in the Categories box that you want to find in the selected records.

Value

On the Advanced tab, choose a Field and a Condition, and then type the value in the Value box that you want to compare the selected field to.
