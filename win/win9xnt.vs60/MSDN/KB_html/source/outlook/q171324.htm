OL97: Unable to Paste Text from Outlook 3-Pane Extension

 Last reviewed: January 19, 1998
Article ID: Q171324
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you preview a Microsoft Outlook 97 e-mail message using the Outlook 3-Pane Extension and copy text from the preview area, you cannot paste the same text into another e-mail message. Instead of pasting your selected text, you will paste an embedded copy of the entire previewed message.

CAUSE

The Outlook 3-Pane Extension provides a preview of the contents of an e-mail message below the viewer's splitter bar. Although you can see and select the previewed text, the message is not open and any attempt to copy will copy the entire message format. This behavior is due to the design of the Outlook 3-Pane Extension.

MORE INFORMATION

You can reproduce this behavior by following these steps:

  1. If it is not yet installed, install the Outlook 3-Pane Extension. For a location from where to install it, please see the "References" section later in this article.

  2. In the Information Viewer of Outlook, click to select an e-mail message.

  3. On the View menu, click Preview to turn on the 3-Pane Extension view.

  4. Using your pointer, select some message text below the viewer's splitter bar, and then on the Edit menu, click Copy.

  5. On the File menu, click New and then Mail Message to open a new untitled message.

  6. Click to place an insertion point into the body of the untitled message, and then on the Edit menu, click Paste.

Note that instead of pasting the text that you copied, the paste command inserts an embedded copy of the entire previewed e-mail message into the body of the untitled message.
