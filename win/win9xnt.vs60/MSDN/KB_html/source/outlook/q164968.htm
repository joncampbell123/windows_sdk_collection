OL97 ErrMsg: The Octal Was Not Found in Mapi32.dll

 Last reviewed: January 19, 1998
Article ID: Q164968
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Windows NT, version 4.0 and Service Pack 2.0

SYMPTOMS

When you attempt to start Outlook 97 on a computer with Windows NT, version 4.0 and Service Pack 2.0 installed, Outlook may not start and you receive will the following error message: 

   The octal was not found in Mapi32.dll.

CAUSE

Your computer has an incompatible version of Messaging Application Interface (MAPI) files.

RESOLUTION

You can resolve this problem by copying the correct MAPI files from the Microsoft Office 97 Compact Disc to your hard drive. Follow these steps to make sure you are using the correct MAPI files:

  1. On the Windows Taskbar, click Start point to Find and click Files or Folders.

  2. In the Named box, type Mapi.dll and click Find Now.

  3. With your right-click the Mapi.dll file and from the context-sensitive menu, click Rename.

  4. Rename the file so that the file extension is not dll. For example, you might rename the file Mapi.old .

  5. Repeat these steps for the Mapi32.dll file.

  6. Copy the Mapi.dll and Mapi32.dll files from the Office 97 Compact Disc \Office\WMS\Winnt folder to your hard drive's \Winnt\System32 folder.

  7. Using Windows Explorer, right-click the new Mapi.dll in your hard drive's \Winnt\System32 folder and from the context-sensitive menu, click Properties.

  8. Click to uncheck the Read-only attribute and click OK.

  9. Repeat steps 7 and 8, above for the Mapi32.dll file.

REFERENCES

For more information about changing file attributes, and how to perform this task in Windows, see your Windows printed documentation or online Help.
