OL97: Files Installed by the Timex Datalink Watch Wizard

 Last reviewed: January 19, 1998
Article ID: Q171629
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes the files copied to your computer when you run setup of the Timex Datalink Watch Wizard for Microsoft Outlook 97.

MORE INFORMATION

The Timex Datalink Watch Wizard setup file is located in the Microsoft Office 97 or Microsoft Outlook 97 ValuPack CD-ROM folder. The setup program copies the following files to your computer: 

   C:\Program Files\Microsoft Office\Office\Convert\Outltimx.dll
   C:\Windows\Inf\Timex.inf
   C:\Windows\System\Datcrt.exe
   C:\Windows\System\Datcrt.pif
   C:\Windows\System\Datzap32.dll
   C:\Windows\System\Protll32.dll
   C:\Windows\System\Tdlir32.dll
