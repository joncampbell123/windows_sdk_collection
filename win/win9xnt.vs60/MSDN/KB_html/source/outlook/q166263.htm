OL97: Zoom Percentage Too Small

 Last reviewed: January 19, 1998
Article ID: Q166263
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you start a new message using Microsoft Word 97 as the e-mail editor, the text appears smaller than expected. If you change the percentage of zoom in the new message, the change is not retained the next time you start Outlook 97.

RESOLUTION

You can change the percentage of zoom, font characteristics and other formatting in the Word template you are using. To determine which template to modify:

  1. On the Tools menu, click Options.

  2. Click the E-mail tab.

  3. Under the option "Use Microsoft Word as the e-mail editor" is the name of the default template. Make a note of it, then click Cancel.

To make the changes to the template:

  1. Quit Outlook. Make sure you close any e-mail messages, contacts, appointments, or any other Outlook related windows you may have open.

  2. Start Word if it is not already open.

  3. On the File menu, click Open.

  4. Change the Files of type to Document Templates (*.dot).

  5. Select the template and click Open.

    NOTE: The default template, Email.dot, is located in c:\Program Files\Microsoft Office\Office.

  6. On the View menu, click Zoom.

  7. Set the desired percentage, then click OK.

  8. To make other formatting changes, click Style on the Format menu, then click Modify. Click to check Add to template, then click Format and make the desired changes. Click OK, then Close when done.

  9. On the File menu, click Close. When prompted to save changes, click Yes.

  10. Quit Word.

Now when you restart Outlook the zoom percentage should be as you set it in the template.

REFERENCES

For more information about modifying your WordMail messages, click the Office Assistant, type "Change the look of your WordMail messages," and click Search.
