OL97: No Option to Import/Export dBase, Access, Excel or FoxPro

 Last reviewed: January 26, 1998
Article ID: Q168783
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you installed a standalone version of Microsoft Outlook 97, the Import and Export Wizard may not include options to import the following data types: 

   dBase
   Microsoft Access
   Microsoft Excel
   Microsoft FoxPro

CAUSE

The standalone version of Outlook does not install the Office Database Connectivity (ODBC) drivers required to import and export the above file formats. This is also true of the small business addition of Office97 when you install each program separately.

RESOLUTION

To import from these sources, you must install the ODBC drivers provided in the Data Access Pack of the Microsoft Outlook 97 CD-ROM ValuPack folder.

Follow these steps to install the appropriate converter files for dBase, Access, Excel and FoxPro:

  1. Quit all Office programs.

  2. Using Windows Explorer, double-click Valupk8.hlp in the ValuPack folder of the Outlook CD-ROM. This opens the ValuPack help.

  3. In the topic "What do you want to know about?" click "Data Access Pack."

  4. Follow the instruction to install the Data Access Pack.

MORE INFORMATION

If you do a complete installation of the standalone version of Outlook, by default the Import and Export Wizard provides support for importing the following file types: 

 Comma Separated Values (DOS)
 Comma Separated Values (Windows)
 Lotus Organizer 1.0, 1.1
 Lotus Organizer 2.1
 Schedule+ 1.0
 Schedule+ 7.0
 Tab Separated Values (DOS)
 Tab Separated Values (Windows)
If you do a complete installation of Microsoft Office 97 Professional, by default the Outlook Import and Export Wizard provides support for importing the following file types: 

 Comma Separated Values (DOS)
 Comma Separated Values (Windows)dBase
 Lotus Organizer 1.0, 1.1
 Lotus Organizer 2.1
 Microsoft Access
 Microsoft Excel
 Microsoft FoxPro
 Personal Address Book
 Schedule+ 1.0
 Schedule+ 7.0
 Tab Separated Values (DOS)
 Tab Separated Values (Windows)
Note: The Typical installation of Outlook and Office do not include the Lotus Organizer converter files.
