OL97: No Option for Printing All Text in Microsoft Outlook 97

 Last reviewed: January 19, 1998
Article ID: Q160290
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

Microsoft Outlook 97 does not have a printing option comparable to the All Text option in Microsoft Schedule+ 7.0. In Schedule+, this option is for printing extra long appointment descriptions.

WORKAROUND

To work around this use one of the methods described below.

Method 1

In Outlook, you can store your appointment details in the large note Window and print them using either the Memo Style or the Table style with the AutoPreview option.

Example 1

To print appointment details of less than 120 characters in length:

  1. On the View menu, point to Current View, and then click Active Appointments.

  2. On the View menu, click AutoPreview.

  3. Select the appointments that you want to print, and click the Print button.

  4. In the Print dialog box, under Print Style, select Table Style.

  5. Under Print Range, click to select Only Selected Rows.

  6. Click OK.

NOTE: This will print about the first 120 characters of the appointment, note window field.

Example 2

To print appointment details of more than 120 characters in length:

  1. On the View menu, point to Current View, and then click Active Appointments.

  2. Select the appointments that you want to print, and click the Print button.

  3. In the Print dialog box, under Print Style, select Memo Style.

  4. Under Print Options, click to clear "Start each item on a new page."

  5. Click OK.

NOTE: This will print all of the details from the appointment, note window field.

Method 2

Create a new view using the Card style:

  1. On the View menu, click Define Views, then click New.

  2. Type a name such as Card in the "Name of New View" field, and in Type of View, click Card, and then click OK.

  3. Click Fields in View Settings for Card.

  4. In "Show the fields in this order," click Start, then click Move Up until it appears at the top of the list, and then click OK and OK.

  5. Click Apply View.

  6. On the View menu, click Filter.

  7. On the Appointments and Meetings tab, select "ends" in the Time list, select "in the next 7 days" in the list to the right, then click OK.

  8. On the File menu, click Print, select Card Style, then click OK.

NOTE: This will print the details from the appointment up to about 255 characters, but not the text in the note window field.

STATUS

This behavior is by design in Microsoft Outlook.
