OL97: Unable to Configure More than One Internet Mail Service

 Last reviewed: January 19, 1998
Article ID: Q166647
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

In Microsoft Outlook 97, you are unable to configure more than one Internet Mail account in the same profile. If you have two different Internet Mail accounts, each service must reside in a different profile.

CAUSE

The MAPI 1.0 Service Provider for Internet Mail file (Minet32.dll, version 4.40.8371374, size 224,528 bytes), which ships with Microsoft Outlook 97 can not configure multiple Internet Mail accounts in the same profile. If you attempt to add a second Internet Mail service to a profile, you receive the following message: 

   The action could not be completed. This information service has
   already been added to your profile and cannot be specified twice.
Furthermore, the Internet Mail service does not provide a method of using multiple Internet Mail accounts in the same service.

Follow these steps to determine the version of your Internet Mail service file:

  1. On the Windows Taskbar, click Start, point to Settings and click Control Panel.

  2. Double-click either the Mail or the Mail and Fax icon to open your default mail profile properties.

  3. If you have more than one profile, click Show Profiles and select the profile that contains the information service you want to check. If you only have one profile, you may skip this step.

  4. On the Services tab, click Internet Mail service to select it.

  5. Click About to open the "About Information Service" and check the version.

RESOLUTION

If you are using the MAPI 1.0 Service Provider for Internet Mail, in order to access multiple Internet Mail accounts with Outlook, you must configure each account in a separate profile.

If you install the Internet Mail Enhancement Patch for Microsoft Outlook 97, you can configure more than one mail account in the same Internet Mail service. You can use multiple e-mail accounts on the same or different Internet e-mail servers. The Internet Mail Enhancement Patch uses Minet32.dll, version 8.0.

MORE INFORMATION

Microsoft is aware of the limitations of the MAPI 1.0 Service Provider for Internet Mail and provides a free Internet Mail Enhancement Patch for Microsoft Outlook 97, which will allow configuration of multiple accounts in the same user profile. More information on the Internet Mail Enhancement Patch for Microsoft Outlook is available on the Microsoft Internet site at: 

   http://www.microsoft.com/outlook/documents/imep.htm
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
