OL97: Stand-Alone Outlook 97 Setup Options

 Last reviewed: January 19, 1998
Article ID: Q171628
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes the stand-alone version of Microsoft Outlook 97 components that install with each setup option.

MORE INFORMATION

The stand-alone version of Outlook 97 provides two types of local setup options: Typical and Custom. The Typical setup requires approximately 32 megabytes (MB) of hard disk space, and the Custom setup can take up to a maximum of 39 MB of hard disk space.

You can also set up Outlook to run from a network server. The "Run from Network Server" option requires approximately 17 MB of hard disk space. To use this option, your network administrator must complete an administrative setup of Outlook on a network server by using the "setup /a" switch. Once the administrative setup is complete, you can run setup from the server location to receive a "Run from Network Server" option.

NOTE: The stand-alone version of Outlook does not provide a "Run from CD- ROM" setup option.

The Custom setup option provides you with the ability to choose which components you want to install. By default, the Custom option selects the same components as the Typical setup; however, you can modify this selection. The Typical setup limits the components to those that most people use. If you choose the Typical setup option, setup only installs the typical components. The following table lists the major setup components of a Typical setup and those components available with the Custom setup: 

   Component                           Typical    Custom             Size
   -----------------------------------------------------------------------

   Microsoft Outlook Program Files     Yes        Yes            12,600 KB
   Office Assistant                    Yes        Yes               817 KB
   Microsoft Exchange Server Support   No         Yes             5,488 KB
   Visuals for Forms Design            No         Yes               520 KB
   Find Fast                           Yes        Yes               266 KB
   Holiday and Forms                   Yes        Yes               744 KB
   Lotus Organizer Converters          No         Yes               744 KB
   Microsoft Outlook Help              Yes        Yes             2,370 KB
   Schedule+ Support Files             Yes        Yes             2,209 KB
   MS Info                             Yes        Yes               537 KB
   Spelling Checker                    Yes        Yes             1,274 KB
The "Run from Network Server" option of setup installs the Typical components; however, the network administrator may modify this behavior by editing the Outlook.stf file on the network server. More information about network administrator options to customize setup is available in the Microsoft Office 97 Resource Kit (ORK). The ORK is available on the Microsoft Web site at the following address: 

   http://www.microsoft.com/office/ork
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
