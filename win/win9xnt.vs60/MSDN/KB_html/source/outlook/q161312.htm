OL97: Cannot Browse File System Folders in Outlook on NT 3.51

 Last reviewed: January 19, 1998
Article ID: Q161312
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Windows NT 3.51

SYMPTOMS

You cannot browse file system folders from Microsoft Outlook 97 running on Microsoft Windows NT 3.51.

CAUSE

Some operating system components needed to allow browsing of file system folders by Outlook are not available when running on Windows NT 3.51.

MORE INFORMATION

To have access to file system folders in Outlook, you must run Outlook on Windows 95 or Windows NT 4.0.

You will see the following characteristics when running Outlook on Windows NT 3.51:

  • The My Computer option at the bottom of the Go menu will be unavailable (dimmed).
  • On the Go To Folder command on the Go menu, the dialog box will not list File System as a choice in the Look In box.
  • The Outlook Setup program will not add any file system shortcuts to the Outlook Bar.
