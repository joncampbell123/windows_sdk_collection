OL97: "Fatal Exception Error 06" With Cirrus Logic Driver

 Last reviewed: January 19, 1998
Article ID: Q166670
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When opening a message in Microsoft Outlook 97, you may get the following error message: 

   A Fatal Exception Error 06 has occurred

CAUSE

You are using the Cirrus Logic video driver version 4.0, with Windows Graphics acceleration turned on.

RESOLUTION

Turn off Windows Graphics acceleration.

MORE INFORMATION

Steps to Turn Off Windows Graphics Acceleration

  1. Quit Outlook.

  2. Right-click My Computer and click Properties.

  3. On the Performance tab, click Graphics.

  4. In the Advanced Graphics Setting, drag the Hardware acceleration slider to None.

  5. Click OK and Close.

  6. Click Yes to restart your computer.
