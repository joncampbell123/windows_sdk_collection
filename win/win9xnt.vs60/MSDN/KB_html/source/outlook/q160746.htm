OL97: How to Get Windows 95 Fax Service to Work with Outlook

 Last reviewed: February 5, 1998
Article ID: Q160746
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

Microsoft Outlook 97 works with the Windows 95 Fax service. However, to do so, the Windows Fax service must first be installed and working properly, so that the setup program of Outlook can update certain files correctly.

MORE INFORMATION

To ensure the proper installation order:

  1. Remove Outlook 97. To do so, follow these steps. 

        a. In Control Panel, double-click the Add/Remove Programs icon.


        b. On the Install/Uninstall tab, click the Microsoft Office 97 group
       and Click the Add/Remove button.


        c. Click the Add/Remove button on the Setup Wizard.


        d. Click to clear the check mark from Microsoft Outlook.


        e. Click Continue.

  2. Remove Microsoft Exchange and the Microsoft Fax service. To do so, follow these steps. 

        a. In Control Panel, double-click the Add/Remove Programs icon.


        b. Select the Windows Setup tab.


        c. Click to clear the check mark from Microsoft Exchange.


           NOTE: This may appear as Windows Messaging on your computer.


        d. Click to clear the check mark from Microsoft Fax.


        e. Click OK.

  3. Reinstall Exchange and the Fax service. To do so, follow these steps: 

        a. In Control Panel, double-click the Add/Remove Programs icon.


        b. Select the Windows Setup tab.


        c. Click to add a check mark to Microsoft Exchange.


        d. Click to add a check mark to Microsoft Fax.


        e. Click OK.


        f. Test Microsoft Fax to ascertain that it is working properly. If it
       is not, it will not behave properly within Outlook. Repeat steps 2
       and 3 above until Microsoft Fax is working properly.

  4. Reinstall Outlook. To do so, follow these steps. 

        a. In Control Panel, double-click the Add/Remove Programs icon.


        b. On the Install/Uninstall tab, click the Microsoft Office 97 group.


        c. Click the Add/Remove button.


        d. Click to add a check mark to Microsoft Outlook.


        e. Click Continue.
