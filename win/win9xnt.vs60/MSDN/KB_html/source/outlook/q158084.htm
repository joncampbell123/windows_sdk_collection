OL97: Converters Included with Outlook

 Last reviewed: January 19, 1998
Article ID: Q158084
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

Outlook includes converters for common time and information management programs. Converters for PDA (Personal Digital Assistants) hardware are not available from Microsoft but may be available from other companies.

MORE INFORMATION

Here is a list of the converters included with Microsoft Outlook and their functionality: 

Product                                File Format Functions


Microsoft Schedule+ 1.0, 7.0           Import
ACT! 2.0                               Import
Ecco Pro 3.0                           Import
Lotus Organizer(tm) 1.0, 1.1, 2.1      Import
SideKick(r) 1.0, 95                    Import
Comma-separated values (CSV)           Import and export
Tab-delimited text                     Import and export
dBASE(r), FoxPro (DBF)                 Import and export
Microsoft Access (MDB)                 Import and export
Microsoft Excel (XLS)                  Import and export
Timex(r) Data Link watch               Export
Exchange personal address book (PAB)   Import and export
