OL97: E-Mail Priority not Recognized by Outlook

 Last reviewed: January 19, 1998
Article ID: Q170138
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you receive an e-mail message in Microsoft Outlook 97 that was created by Eudora Pro or Netscape Communicator, Outlook does not recognize the priority settings.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. The problem is corrected in the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP).
