OL97 ErrMsg: Condition is not Valid Using Timex Wizard

 Last reviewed: January 19, 1998
Article ID: Q168619
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you use the Timex Data Link Watch Wizard in Microsoft Outlook 97 to export an exact number of days of tasks, you may receive the following error message: 

   Condition is not valid.
When you click OK, the above error displays a second time and then the wizard proceeds without exporting your tasks.

RESOLUTION

You can work around this problem by choosing to export all active tasks instead of an exact number of days of tasks.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Make sure you have the Timex Data Link Watch Wizard installed. For more information about how to install the wizard, type "timex" in the Office Assistant, click Search, and then click to view "The Timex Data Link Wizard."

  2. Start Outlook and create one or two tasks. The type of task information is not important.

  3. On the File menu, click Import and Export.

  4. In the "Choose an action to perform" list, click "Export to the Timex Data Link watch" and then click Next.

  5. Click to clear all items except Tasks and then click Next.

  6. Under Export click "Exactly xx day(s) of active tasks." Leave all other default settings and click Next.

You should receive the error message; "Condition is not valid."
