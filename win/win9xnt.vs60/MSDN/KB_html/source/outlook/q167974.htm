OL97: How to Remove Microsoft Sample Forms From Your Computer

 Last reviewed: January 19, 1998
Article ID: Q167974
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes how you can remove sample Microsoft Outlook 97 forms from your computer.

MORE INFORMATION

Microsoft provides Sample Groupware Applications on the Microsoft Web site that demonstrate how to use Outlook to solve business and personal needs. These samples contain customized forms that reside in a Personal Folders file, which you add to your Outlook profile.

The ValuPack also contains sample forms. The Valupk8.hlp help file contains a topic, "Microsoft Office templates, forms, and wizards," which allows you to install sample Outlook forms. These sample forms install as a Personal Folders file (Forms.pst) in your profile.

You can remove these custom forms from your computer by:

  • Identifying the name and path of the sample forms Personal Folders file
  • Removing the sample forms service from your profile
  • Deleting the sample forms Personal Folders file from your computer's hard drive

Identify the Sample Form's Personal Folder File Name

  1. Start Outlook with the profile that contains the sample forms.

  2. On the Tools menu, click Services.

  3. On the Services tab, click to select the Sample Forms folder and click Properties.

  4. In the Path box, note the path and name of the file that contains your sample forms.

    By default, the sample forms from the ValuPack install to the following location: 

          C:\Program Files\Microsoft Office\Office\Forms.pst

    By default, the sample forms from the Microsoft web site install to the following location: 

          C: Program Files\Microsoft Office\Office\<Month> Outlook Samples.pst

Remove the Sample Forms Service From Your Profile

  1. Start Outlook with the profile that contains the sample forms.

  2. On the Tools menu, click Services.

  3. On the Services tab, click to select the Sample Forms service and click Remove.

    NOTE: The name of the service depends on what samples you installed. The ValuPack forms are in a "Sample Forms" service. The Internet forms are in a service named, "<Month> Outlook Samples."

  4. When prompted, "Are you sure you want to remove this information service from the profile?" click Yes and then click OK.

Delete the Sample Forms Personal Folders File From Your Hard Drive

  1. On the Windows Taskbar, click Start point to Find and click Files or Folders.

  2. In the Named box, type the name of the .pst file that contains your sample forms.

  3. Click to select, Include subfolders and click Find Now.

  4. In the Name column, right-click the .pst file and on the shortcut menu, click Delete.
