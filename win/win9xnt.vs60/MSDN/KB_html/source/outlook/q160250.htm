OL97: Printout of Item Contains Header but no Body Text

 Last reviewed: February 11, 1998
Article ID: Q160250
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you print an item in Microsoft Outlook 97 to a Hewlett Packard LaserJet 5L printer, the header of the message is printed, but the body text is missing, or you may receive the following error message: 

     OUTLOOK caused a general protection fault
     in module HPPCL5G2.DLL at 0004:00000a55.
NOTE: Printing Contacts in Address Card view with this printer can also generate this error message.

CAUSE

This behavior may occur with LaserJet 5L printer drivers earlier than version 1.20.

WORKAROUND

There are three possible workarounds:

Method 1

Change the print quality in the LaserJet 5L driver. To do so, follow these steps:

  1. In the Print dialog box, click Properties.

  2. Click the Print Quality tab.

  3. Set the Print Optimization Setting to Primarily Text and click OK.

Method 2

Install the driver for the HP LaserJet 4L and print with this driver.

Method 3

Install the Host Based HP LaserJet 5L Printing System driver for Windows95/3.1x, version B. This version has a copyright date of March 26, 1996, and shipped with some printers. It is also available for download at the Hewlett Packard Web site at the following address: 

   http://www.hp.com/cposupport/indexes1/lj5ls.html

MORE INFORMATION

With Method 1, the quality of graphics or borders may be affected. With Method 2, any options that are new to the LaserJet 5 will not be available.
