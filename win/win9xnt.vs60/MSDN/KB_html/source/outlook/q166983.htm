OL97 ErrMsg: Unable to Load Fae.dll Exporting to Access

The information in this article applies to:

SYMPTOMS

When you use the Microsoft Outlook 97 Import and Export Wizard to export a file in a Microsoft Access file format, you may receive the following error message: 

   Unable to load fae.dll. Additional translators are available
   in the ValuPack.

RESOLUTION

To resolve this problem reinstall the Data Access Pack from the Microsoft Office 97 Compact Disc ValuPack.

MORE INFORMATION

To install the Data Access Pack, follow these steps:

  1. On the Office 97 Compact Disc open the ValuPack directory and double- click the Valupk8.hlp file.

  2. In the "Welcome to the Microsoft Office 97 ValuPack" help topic, click Data Access Pack.

  3. In the "Data Access Pack" help topic, click "To install the Data Access Pack, click."
