OL97: Error in PRT File When Printing Schedule+ Calendar

 Last reviewed: January 19, 1998
Article ID: Q174705
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When using Microsoft Schedule+ as your default calendar in Microsoft Outlook 97, you may receive the following error message when trying to print your calendar: 

   An unknown or incorrectly defined property was found.
   The error occurred in the .PRT file on line <line number>.

CAUSE

You are using the PRT files found in the file, Splprint.exe, downloaded from the Outlook Free Stuff on the Web.

RESOLUTION

Reinstall Schedule+ from the original medium and install, Additional Layouts and Paper Formats.

MORE INFORMATION

Schedule+ uses a combination of PRT and FMT files to control printing of the calendar. These files are not installed by default on a Compact installation of Schedule+. If you downloaded the files by the same name that are found on the Outlook Free Stuff Web page (Splprint.exe) they will not work. It is necessary to reinstall Schedule+ with a Custom installation and choose to install the, Additional Layouts and Paper Formats, to load the proper PRT and FMT files.

REFERENCES

For more information on this subject, please see the following article in the Microsoft Knowledge Base:

   TITLE     : "No Valid .FMT Files Available" Printing Schedule+ File
