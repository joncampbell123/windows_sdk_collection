OL97: How to Determine the Model of theTimex Data Link Watch

 Last reviewed: March 12, 1998
Article ID: Q173114
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

The Timex Data Link Watch produced by Timex Corporation receives information exported from Microsoft Outlook 97. The Timex Data Link Watch Wizard in Outlook makes it easy to export appointments, tasks, phone numbers, anniversaries, reminders, and current time and time zone information to the watch.

To export information to your Data Link Watch, you must know which model you have. There may be no model number on the watch itself. The simple visual difference is in the material used for the case. The Timex Data Link Watch Model 100 case is made of lightweight resin. This is shown as Timex Data Link Watch in the Wizard. The cases for Models 150 and 150s are made of metal.

Another way to determine the model you have is by identifying a number imprinted on the watch caseback. The code is located near the bottom of the caseback. The codes for the various models are: 

   786     Model 70    Timex Data Link Watch (supported in Outlook only
                       with version 2.1d of the Datalink software,
                       available from the Timex Web site, below)
   2025NA  Model 100   Timex Data Link Watch
   802     Model 150   Timex Data Link 150 Watch
   763     Model 150s  Timex Data Link 150s Watch
   816     No Model #  Ironman Triathlon with Data Link Watch
                       (not supported in Outlook)
An additional method is to put the watch in COMM MODE. To do this press the MODE BUTTON (middle button on the right side of the watch) repeatedly, until the watch displays "COMM MODE". The watch display will change momentarily and show one of the following examples: 

   Model 100    --------
               |  comm  |
               | ------ |
               | READY  |
               |        |
                --------

   Model 150    --------
               |  comm  |
               | ------ |
               | READY  |
               |  v2.0  |
                --------

   Model 150s   --------
               |  comm  |
               | ------ |
               | READY  |
               |  v2.1  |
                --------
NOTE: If you install version 2.1c of the Timex Data Link software, you will have 3 additional options in the Export Wizard:
  • Data Link Device Type 2
  • Royal FL90/95 Organizers
  • Tiger PDA2000 Organizer

Data Link Device Type 2 will work for the Tiger PDA2000 Organizer and future devices that use this same transfer mode. Royal FL90/95 Organizer users can choose either Databank Type or Royal FL90/95 Organizers.

You can download the latest version of the datalink software from the Timex Web site at: 

   http://www.timex.com/datalink/

MORE INFORMATION

The files you need to export information to the watch are included in the Microsoft Office 97 ValuPack. To install these files follow these steps:

  1. Insert the Office 97 (or Outlook 97) CD-ROM into the drive.

  2. Click Start, click Run, click Browse, then click the CD-ROM drive.

  3. Double-click the Valupack folder, click to select Valupk8.hlp, click Open, then click OK.

    NOTE: You may need to change the Files of Type in the Browse dialog from Programs to All Files to see the Valupk8.hlp file.

  4. Scroll to the last item in the Help window and click the button beside Timex Data Link Watch Wizard.

  5. To install the correct Wizard, select the version of Windows you use.

NOTE: If you did not install Outlook 97 from a CD-ROM and you have access to the World Wide Web, you can download the Wizard from the Microsoft Web site. On the Help menu in Outlook, point to "Microsoft on the Web," then click Free Stuff.
