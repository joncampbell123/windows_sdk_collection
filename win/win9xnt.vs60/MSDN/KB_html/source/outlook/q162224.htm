OL97: Err in"About add-ins supplied with Outlook" Help Topic

 Last reviewed: January 19, 1998
Article ID: Q162224

The information in this article applies to:

  • Microsoft Outlook 97

SUMMARY

In the "About add-ins supplied with Outlook" Help topic, the description about the Internet Mail add-in is not correct. The Internet Mail Minet.ecf file does not add Internet Mail to the list of available information services you can use. The Internet service is available without Minet.ecf installed. If you have a profile with Internet services, the Minet.ecf add- in adds an Internet tab in the message's Properties dialog box.

MORE INFORMATION

To locate this documentation error using the Office Assistant, type "add- in" in the Office Assistant, click Search, and then click to view "About add-ins supplied with Outlook."
