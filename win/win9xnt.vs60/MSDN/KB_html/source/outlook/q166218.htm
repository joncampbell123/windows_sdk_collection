OL97: Localized Versions of Outlook 97 and Exchange 5.0

 Last reviewed: January 19, 1998
Article ID: Q166218
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Exchange Server 5.0
  • Microsoft Exchange Client 5.0

SUMMARY

This article lists available localized versions for Microsoft Outlook 97, Microsoft Exchange Server 5.0, and Microsoft Exchange Client 5.0.

NOTES

  1. Microsoft Exchange Server localizes only four languages.

  2. Outlook ships a U.S. version in additional locales.

  3. Help files for Greek, Turkish, and Slovenian versions of Outlook are in English.
