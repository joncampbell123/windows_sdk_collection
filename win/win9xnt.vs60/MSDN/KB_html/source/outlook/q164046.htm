OL97: Converting ECCO Files to Outlook

 Last reviewed: February 5, 1998
Article ID: Q164046
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article provides information on known issues concerning the conversion of files from NetManage ECCO Pro to Microsoft Outlook 97.

ECCO Pro is manufactured by NetManage, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

MORE INFORMATION

To import an ECCO Pro file into Outlook, you must have ECCO Pro installed on your computer. The import and export converter for ECCO Pro in the Microsoft Office 97 ValuPack, supports ECCO Pro versions 3.0, 3.01, and 3.02.

To import an ECCO file from a version other than those above, do the following:

  1. In ECCO, save the file in .csv format. Under File, Database, Export, check the option for "Mail Merge Header".

  2. In Outlook, click Import and Export on the File menu.

  3. In the Import and Export Wizard, click "Import from Schedule+ or another program or file" and click Next.

  4. Click Comma Separated Values, and click Next.

  5. Select the .csv file to be imported and click Next.

  6. Select the destination folder and click Next.

  7. Click Map Custom Fields and map the .csv fields to the appropriate Outlook fields.

  8. Click OK and Finish.

Cannot Convert When Running Outlook 97 from CD-ROM

If you run Outlook from a CD-ROM, you cannot import from ECCO Pro even if you install the converter from the Microsoft Office 97 ValuPack. To use the ECCO Pro converter, you must run Outlook from your hard drive or a network.

Converting When Running Outlook 97 from a Networked Computer

If you run Outlook from a networked computer, your network administrator must complete the following steps so that you can import information from ECCO Pro:

  1. Install Outlook import and export converters from the Microsoft Office 97 ValuPack. For more information, see Outlook Help.

  2. Copy the Ecco.sam file to the Office\Convert folder where Outlook 97 or Office 97 is installed.

Steps to Import an ECCO Pro File

  1. Click Import and Export, on the File menu.

  2. Click "Import from Schedule+ or another program or file", and click Next.

  3. Click ECCO and click Next.

  4. Type the ECCO file name in the "File to import" window and click Next.

  5. Click Import "<filename>" into "<foldername>" and click Map Custom Fields (where the name in brackets is the name of the file or folder.)

  6. Map "<ECCO fieldname>" to "<Outlook fieldname>" and click OK.

Manually Map ECCO Fields to Outlook Fields

Some ECCO fields are built-in and others may be user-defined. The built-in fields may automatically map to the proper Outlook fields but the user-defined ones will not. It is always necessary to verify the mapping to assure each ECCO field is going to the appropriate Outlook field (step 6, above).

Undated Tasks

ECCO Tasks that have no To-Do date will not be imported into Outlook.

Percent Complete and Status Fields

If the ECCO file has a numeric, user-defined field that contains the percent complete, that field will import properly into Outlook's % Complete field.

The Outlook Status field will be translated based upon the following ECCO information:

ECCO To-Do Date ECCO % Complete Outlook % Complete Outlook Status --------------- --------------- ------------------ -------------- 

  In Future       none or < 100%      Copied           In Progress
  In Past            any value        Copied           Late (Red)
  Any date             100 %          Copied           Complete
  No Date            any value    ------- Record not imported ------
