OL97: Sample Groupware Applications Available

 Last reviewed: March 2, 1998
Article ID: Q173102
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

Microsoft provides sample applications for Microsoft Outlook 97 that demonstrate how to use Outlook to solve business and personal needs. These samples contain customized forms that reside in a Personal Folders file that you add to your Outlook profile.
