OL97: Cancel Button Does Not Work in Delivering Messages Dialog

 Last reviewed: January 19, 1998
Article ID: Q167089
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you check for new mail using the Internet Mail service and click Cancel in the Delivering Messages dialog box, the operation does not end.

RESOLUTION

To cancel the operation do the following:

  1. Press the CTRL+ALT+DELETE keystroke combination.

  2. In the Close Program list, click to select "Connected to <Your Connection>," then click End Task. (Where <Your Connection> is the name Of your Dial-Up Networking connection.)

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. The problem is corrected in the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP).
