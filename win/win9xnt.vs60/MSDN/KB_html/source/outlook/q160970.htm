OL97: How to Get Latest Version of CompuServe Mail Service

 Last reviewed: January 19, 1998
Article ID: Q160970
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

You can download the latest version of the CompuServe Mail Service driver from CompuServe's WinCim forum. The file name is CSMAIL.EXE.

MORE INFORMATION

To download and install the latest CompuServe Mail Service driver, use the following steps:

  1. Connect to CompuServe.

  2. Browse to the WinCim forum (GO WinCim).

  3. Select the option, Support and Other Software.

  4. Select the item, CompuServe Mail for Microsoft Exchange (Windows 95).

  5. Select the item, Download CompuServe Mail for Microsoft Exchange.

  6. Follow the screen instructions to download the file CSMAIL.EXE to your computer.

  7. When the download is complete, copy the CSMAIL.EXE file to an empty folder.

  8. To extract the files needed for setup, double-click the CSMAIL.EXE file.

  9. From the resulting file set, double-click the SETUP.EXE file.

  10. Follow the installation instructions on the screen. At the end of installation, you will receive a prompt to add the CompuServe mail service to your default profile.

REFERENCES

For general information about adding the CompuServe mail service to a profile, see the following Microsoft Knowledge Base article:
