OL97: Cannot Edit Received Fax Message

 Last reviewed: January 19, 1998
Article ID: Q177356
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

You can read but not edit a fax message received in Microsoft Outlook 97.

CAUSE

The correct imaging software is either not installed or not working properly.

RESOLUTION

Download and install, Imaging for Window 95.

MORE INFORMATION

Imaging for Windows 95 is also known as ImageVue and Wang Imaging Software. It is available for download from the Microsoft web site at: 

   http://www.microsoft.com/Windows95/info/Wang.htm
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
