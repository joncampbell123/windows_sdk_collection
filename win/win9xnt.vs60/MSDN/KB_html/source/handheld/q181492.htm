New Features in Windows CE 2.0

 Last reviewed: February 20, 1998
Article ID: Q181492
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SUMMARY

This article lists the new features available in Windows CE 2.0. Windows CE 2.0 includes new features in the following categories:

  • Connectivity
  • Microsoft Pocket Excel
  • Microsoft Pocket Outlook
  • Microsoft Pocket PowerPoint
  • Microsoft Pocket Word
  • Printing
  • Programming languages
  • System

MORE INFORMATION

Connectivity

The new connectivity features in Windows CE include:

  • Continuous synchronization.
  • Improved wireless communication.
  • Infrared, Dial-Up Networking, or Ethernet network connections.
  • Secure network connections.
  • The ability to synchronize one Handheld PC (H/PC) with two desktop computers.
  • The ability to synchronize one desktop computer with multiple H/PCs.

Pocket Excel

The new Pocket Excel features include:

  • Database features.
  • Full Screen view.
  • Password protection for spreadsheets.
  • Split and Freeze Panes views.
  • Spreadsheet templates.
  • Zoom view.

Pocket Outlook

The new Pocket Outlook features include:

  • Appointment categories.
  • Appointment filters.
  • Group scheduling in Calendar.
  • Number Of Days view in Calendar.
  • The ability to receive e-mail messages with binary attachments.
  • The ability to send an e-mail message or open a Uniform Resource Locator (URL) in Contacts.
  • The ability to synchronize e-mail messages in the Outbox and Sent Items folders.
  • The Find command on the Tools menu in Calendar, Contacts, and Tasks.
  • The Go menu in Calendar, Contacts, and Tasks.
  • The Tap Here To Add Notes box in Calendar and Tasks.
  • Different formatting for unread e-mail messages and e-mail messages with
header information only.

Pocket PowerPoint

The new Pocket PowerPoint features include:

  • The ability to create and edit slide presentations.
  • The ability to display Video Graphics Adapter (VGA) graphics with 800 X 600 or 640 X 480 screen resolution using an external monitor.
  • The ability to edit, show, or hide slides in a presentation.

Pocket Word

The new Pocket Word features include:

  • Custom dictionaries.
  • Document templates.
  • Full Screen view.
  • Improved formatting features including tabs, indentation, and numbered lists.
  • Outline view.
  • Password protection for documents.
  • Spelling checker.
  • Zoom view.

Printing

The new printing features in Windows CE include:

  • A Printer Control Language (PCL-3) printer driver.
  • The ability to use an infrared or serial port to print.

Programming Languages

The new supported programming languages in Windows CE include:

  • A subset of ActiveX.
  • DirectX.
  • Hypertext Markup Language (HTML).
  • Java Virtual Machines and Java Libraries.
  • Microsoft Foundation Classes.
  • Visual Basic Script.

System

The new system features in Windows CE include:

  • A cascading Start menu.
  • Color screens and improved grayscale screens.
  • Customization of command bars.
  • Integrated desktop.
  • International character sets.
  • The ability to use a variety of processors, including PowerPC, Arm, and StrongArm processors.
  • The ability to display VGA graphics using an external monitor.
  • The Auto Hide option for the taskbar.
  • TrueType fonts.
