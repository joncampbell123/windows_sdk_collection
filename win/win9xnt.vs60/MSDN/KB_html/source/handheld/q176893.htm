Pocket Word Does Not Prompt You for a Password

 Last reviewed: November 18, 1997
Article ID: Q176893
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you copy a Microsoft Word 97 file from your desktop computer to your Handheld PC (H/PC) and then open the file using Microsoft Pocket Word, you are not prompted for the password you specified in the Password To Modify box in Word 97.

CAUSE

When you copy a Word 97 file from your desktop computer to your H/PC, the conversion process generates a Rich Text Format (RTF) file type. The RTF file type does not store the "modify" password type.

STATUS

This behavior is by design.

MORE INFORMATION

For information about RTF and password protection, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q110109
   TITLE     : WD: Password Protection Lost When Saving
