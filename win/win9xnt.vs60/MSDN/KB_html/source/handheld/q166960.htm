How to Display a Context Menu in Windows CE

 Last reviewed: April 15, 1997
Article ID: Q166960
The information in this article applies to:
  • Microsoft Windows CE version 1.0

To display a context menu in Windows CE, press and hold down the ALT key while you tap the object for which you want to display a context menu.

This information is taken from the Welcome program in Windows CE. To review the Welcome program, tap the Start button, tap Programs, double- tap Accessories, and then double-tap Welcome.
