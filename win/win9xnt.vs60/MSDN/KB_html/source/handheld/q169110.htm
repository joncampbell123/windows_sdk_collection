Cannot Use the Keyboard to Navigate Date Picker Tool

 Last reviewed: May 28, 1997
Article ID: Q169110
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SUMMARY

You cannot use the Handheld PC (H/PC) keyboard to navigate the Date Picker tool in the Calendar, Contacts, or Tasks programs.

MORE INFORMATION

You can use the Date Picker tool to select a specific date from a calendar. To activate the Date Picker tool in the following programs, follow the appropriate steps:

Calendar

  1. Run the Calendar program.

  2. Press CTRL+G to display the Go To dialog box, and then tap the Date Picker icon.

Contacts

  1. Run the Contacts program.

  2. Double-tap a contact, tap the Personal tab, and then tap the Birthday field.

  3. Tap the Date Picker icon.

Tasks

  1. Run the Tasks program.

  2. Double-tap a task, and then tap the Date Picker icon in the Duration Of This Single Task area of the task.
