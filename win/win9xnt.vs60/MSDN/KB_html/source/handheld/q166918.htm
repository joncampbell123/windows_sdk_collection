How to Move or Copy Information from One Window to Another

 Last reviewed: April 15, 1997
Article ID: Q166918
The information in this article applies to:
  • Microsoft Windows CE version 1.0

In Windows CE, you cannot drag information from one window to another because only one window is displayed at a time. To move or copy files or information from one window to another in Windows CE, use the Cut, Copy, and Paste commands on the Edit menu or the context menu.

This information is taken from the Welcome program in Windows CE. To review the Welcome program, tap the Start button, tap Programs, double-tap Accessories, and then double-tap Welcome.
