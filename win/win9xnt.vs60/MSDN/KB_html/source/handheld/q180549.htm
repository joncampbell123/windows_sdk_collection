Windows CE Services Is Not Refreshed When You Connect Your H/PC

 Last reviewed: February 5, 1998
Article ID: Q180549
The information in this article applies to:
  • Microsoft Windows CE, versions 1.0, 2.0

SYMPTOMS

When you connect your Handheld PC (H/PC) to your desktop computer after you start Microsoft Windows CE Services (or Microsoft H/PC Explorer for Windows CE version 1.0), you may see the following message on the status bar: 

   Waiting for a mobile device to connect...

CAUSE

This behavior can occur if you do not refresh the screen on your desktop computer.

RESOLUTION

To resolve this behavior, refresh the screen on your desktop computer. To do so, click Refresh on the View menu in Windows CE Services (or H/PC Explorer for Windows CE version 1.0).
