Cell Border Formatting Is Missing in Pocket Excel Files

 Last reviewed: November 19, 1997
Article ID: Q176956
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you open a file in Microsoft Pocket Excel, the cells in the spreadsheet may be missing border formatting.

CAUSE

This problem can occur if you convert a Microsoft Excel 97 file that includes an empty cell with border formatting.

RESOLUTION

To resolve this problem, format the cell borders in Pocket Excel. To do so, use the following steps:

  1. Double-tap Pocket Excel.

  2. Open the file to which you want to add cell border formatting.

  3. On the Format menu, tap Cells.

  4. On the Borders & Fills tab, tap any of the Outline, Left, Right, Top, or Bottom check boxes to select them.

  5. In the Borders box, tap the color for your border, and then tap OK.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows CE Services version 2.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
