Err Msg: Cannot Start Communications with the Desktop...

 Last reviewed: April 14, 1997
Article ID: Q161446
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you try to connect your Handheld PC (H/PC) to your desktop computer using H/PC Explorer, you may receive the following error message: 

   Cannot start communications with the desktop computer. Remote
   Networking could not start due to a critical error (Error 754).
   Please reconnect the H/PC. If the problem persists, restart the
   H/PC using the instructions provided by the manufacturer.

CAUSE

This behavior occurs when your H/PC has low program memory.

RESOLUTION

Adjust the memory settings on your H/PC. To do so, follow these steps:

  1. Tap the Start button, tap Settings, and then double-tap System.

  2. Tap the Memory tab, move the slider to the left to increase available program memory, and then tap OK.

  3. If there is a limited amount of memory available, you may need to reset your H/PC to free additional memory. When you reset your H/PC, any unsaved data is lost.
