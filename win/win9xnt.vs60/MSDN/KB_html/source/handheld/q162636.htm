Characters Not Displayed at Specified Font Size

 Last reviewed: April 14, 1997
Article ID: Q162636
The information in this article applies to:
  • Microsoft Pocket Word version 1.0
  • Microsoft Pocket Excel version 1.0

SYMPTOMS

In Pocket Word or Pocket Excel, if you change the font size to a number that is not available in the Font Size box and then press ENTER, the number you type is displayed in the Font Size box. However, the font displayed on the screen is the next smaller font size available in the Font Size box.

CAUSE

Windows CE displays text using raster fonts, which cannot be scaled or rotated. Only the sizes listed in the Font Size box can be displayed in Windows CE-based programs, such as Pocket Word and Pocket Excel.

MORE INFORMATION

Although characters may not be displayed at the font size you specify, the text is formatted to the font size you type in the Font Size box. If you transfer a Pocket Word or Pocket Excel file to your desktop computer, and then open it in Microsoft Word or Microsoft Excel, the text is displayed at the font size you specified.
