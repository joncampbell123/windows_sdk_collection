Cannot Move Icons Within a Folder in H/PC Explorer

 Last reviewed: April 14, 1997
Article ID: Q162314
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

If you click an icon in H/PC Explorer with the right mouse button and then drag the icon to another location within the same folder, a context menu with the following options appears:

  • Copy Here
  • Move Here
  • Create Shortcut(s) Here
  • Cancel

Move Here is the default action. If you click Move Here, the icon is not moved.

Also, if you drag an icon within a folder in H/PC Explorer, the icon is not moved.

CAUSE

H/PC Explorer does not support moving icons within a folder.

STATUS

Microsoft has confirmed this to be a problem in Windows CE version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
