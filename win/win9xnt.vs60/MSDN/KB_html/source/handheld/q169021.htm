Err Msg: Could Not Transfer One or More of the Selected...

 Last reviewed: May 23, 1997
Article ID: Q169021
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you attempt to transfer a file from your Handheld PC (H/PC) to your desktop computer using H/PC Explorer, you may receive the following error message: 

   Could not transfer one or more of the selected file(s) to the specified
   destination.

   A temporary folder will be created containing temporary copies of these
   files. A shortcut called "Handheld PC Recovered Files" will be placed 
   on the Windows Desktop.

   Warning: The temporary files will be deleted upon the next error of 
   this type. To avoid having these files permanently deleted, you should
   immediately copy these temporary files to another folder. The folder 
   will now be opened, please copy the files to another folder at this 
   time.

CAUSE

This behavior can occur if you attempt to copy a file to any of the following target locations:

  • A CD-ROM drive
  • A read-only network share
  • A floppy disk drive with no disk inserted
  • A floppy disk drive with a floppy disk that is full

RESOLUTION

To resolve this problem, make sure that all of the following conditions are true before you attempt to copy a file:

  • You have write access to the target location.
  • There is enough available storage space on the target location to store the file that you are attempting to copy.
  • If the target location is a floppy disk drive, there is a disk inserted in the drive.
