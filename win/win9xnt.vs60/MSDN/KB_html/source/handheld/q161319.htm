Unsupported Internet Technology with Pocket Internet Explorer

 Last reviewed: April 30, 1997
Article ID: Q161319
The information in this article applies to:
  • Microsoft Pocket Internet Explorer versions 1.0, 1.1

SUMMARY

This article lists common Internet technologies that are not supported in Pocket Internet Explorer.

MORE INFORMATION

Internet Techologies Not Supported In Pocket Internet Explorer 1.0 and 1.1

Pocket Internet Explorer versions 1.0 and 1.1 do not support the following Internet technologies:

  • Scripting languages such as JavaScript, JScript, and Visual Basic Scripting
  • Java programs
  • Animated objects such as GIF, AVI, MPEG, and MOV (QuickTime) objects
  • MIDI, AU, and AIFF sound files
  • Black and white XBM image format
  • TIFF image format
  • ActiveX controls
  • Netscape plug-ins
  • Newsgroups
  • Frames
  • Style sheets

Other Internet Techologies Not Supported In Pocket Internet Explorer 1.0

  • Security features such as the SSL and PCT protocols
  • Cookies
