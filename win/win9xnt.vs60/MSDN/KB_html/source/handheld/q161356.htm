Err Msg: Windows Cannot Display the Image Currently Selected

 Last reviewed: April 14, 1997
Article ID: Q161356
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you select a background image for your Handheld PC (H/PC) in Display properties, you may receive the following error message: 

   Windows cannot display the image currently selected. It may be damaged
   or an invalid image. Please choose another image.

CAUSE

This problem can occur if there is not enough available program memory to display the image you selected.

RESOLUTION

To resolve this issue, follow these steps:

  1. Tap the Start button, tap Settings, and then double-tap System.

  2. Tap the Memory tab, and then move the slider to the left to free at least 135K of program memory.

If you continue to experience the problem, save all your data and then reset your H/PC. Resetting the H/PC frees as much program memory as possible.
