Key Combinations Perform Extra Keyboard Functions

 Last reviewed: April 14, 1997
Article ID: Q159732
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SUMMARY

This article lists key combinations for keyboard functions on the Handheld PC (H/PC).

MORE INFORMATION

The following table lists key combinations to make full use of keyboard functions.

The chording keys are ALT, SHIFT, and CTRL. These keys do not provide any functionality together or by themselves. In order for a chording key to have functionality, it must be used in conjunction with a non-chording key.

The following table lists additional key combinations you can use if your H/PC keyboard includes a Windows key: 

   Key              Result
   -------------------------------------------
   Windows          Open Start menu

   Windows+K        Open Keyboard tool

   Windows+I        Open Stylus tool

   Windows+C        Open Control Panel

   Windows+E        Explore My Handheld PC

   Windows+R        Display the Run dialog box

   Windows+H        Open Windows CE Help

   CTRL+Windows+A   Select all on desktop
Windows CE does not support the following keyboard keys:
  • INSERT
  • SCROLL LOCK
  • PAUSE
  • NUM LOCK
  • Function keys
  • PRINT SCREEN
