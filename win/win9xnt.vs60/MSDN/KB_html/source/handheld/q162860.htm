Find Opens Unmatched Contact Entries with Some Search Strings

 Last reviewed: April 14, 1997
Article ID: Q162860
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you use the Find tool in Contacts to search for some character strings, Find opens every entry for which the address field has been edited, even if the character string you search for does not seem to occur in that entry.

CAUSE

The Find tool may find a character string match in the Country box, which is included in the Address field of each entry.

If you edit the Address field for an entry, the Find tool includes the text in the Country box when searching for a character string. If you do not edit an entry's address field, the Find tool does not include the Country box when searching for a character string.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To view the Country box, tap the Address field, and then tap the button to the right of the Address field.
