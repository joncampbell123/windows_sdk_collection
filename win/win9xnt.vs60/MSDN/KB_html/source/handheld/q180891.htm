Using the Find Tool in Contacts May Hang Pocket Outlook

 Last reviewed: February 11, 1998
Article ID: Q180891
The information in this article applies to:
  • Microsoft Windows CE, versions 1.0, 2.0

SYMPTOMS

When you use the Find tool in Contacts, Microsoft Pocket Outlook may stop responding (hang).

CAUSE

This issue can occur if you open an e-mail message when you use the Find tool.

RESOLUTION

To quit the hung instance of Pocket Outlook, follow these steps:

  1. Press ALT+TAB.

  2. Tap Inbox, and then tap End Task.

To work around this issue, do not open an e-mail message when you use the Find tool.

MORE INFORMATION

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
