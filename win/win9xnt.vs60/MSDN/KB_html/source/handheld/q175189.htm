Formatting Hidden Rows Causes Them to Become Visible

 Last reviewed: October 15, 1997
Article ID: Q175189
The information in this article applies to:
  • Microsoft Windows CE, version 2.0
  • Microsoft Pocket Excel, version 2.0

SYMPTOMS

In Pocket Excel, if you select an area and apply formatting, any hidden rows in that selection will become visible.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Pocket Excel, version 2.0. Microsoft is researching this problem and will post new information here as it becomes available.

MORE INFORMATION

This problem occurs with any formatting function.
