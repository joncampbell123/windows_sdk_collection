Images Inverted After Transferring from H/PC to Desktop

 Last reviewed: April 14, 1997
Article ID: Q160033
The information in this article applies to:
  • Microsoft Windows CE version 1.0
  • Microsoft Pocket Word version 1.0

SYMPTOMS

When you copy a Pocket Word document from your Handheld PC (H/PC) to your desktop computer and then open the document using Microsoft Word, images may be displayed upside down.

CAUSE

This behavior can occur if the file conversion properties for the H/PC To Desktop converter are set to convert Pocket Word 1.0 documents to WordPerfect 5.x for Windows documents.

RESOLUTION

To resolve this issue, use either of the following methods:

  • Change the H/PC To Desktop file conversion setting for Pocket Word files using the following steps:

    1. In H/PC Explorer, click File Conversion on the Tools menu.

    2. Click the "H/PC -> Desktop" tab.

    3. Click Pocket Word 1.0 Document, and then click Edit.

    4. Change the Type setting from WordPerfect 5.x For Windows to the 

          appropriate version of Microsoft Word.
  • Open the document using WordPerfect so the conversion process finishes. Once the conversion process is finished, the document can be opened in Microsoft Word with the images displayed correctly.
