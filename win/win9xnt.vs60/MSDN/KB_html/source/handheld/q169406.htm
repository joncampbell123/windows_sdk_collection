MORE INFORMATION

*** Power Toys for Windows CE *** Note: The Power Toys for Windows CE are beta software. As such, they are not supported by Microsoft.

NEW -- Remote Control has been added BETTER -- Mute and Cascading Menus have been updated

Paint Program for Windows CE Create pictures and edit images with a pocket-sized version of the Windows 95 Paint program. Paint for Windows CE works just like Paint for Windows 95 on your desktop computer.

Cascading Menus Instant access to files and folders using cascading and scrolling menus. Places an icon on the Tray of your Taskbar on your Handheld PC.

NEW! Delay icon fetch speeds up performance 300%

To launch a file * On the Cascading menu, tap the file.

To browse subfolders * On the Cascading menu, tap the folder. The Cascading menu will display a submenu showing the contents of the folder.

To explore folders in Explorer view. * Hold CTRL while you tap the folder.

Fully customizable, just drag the files, folders, or shortcuts you'd like to see on the menu to the Windows\Start Menu folder.

Mute Turn off all sounds with a tap of a taskbar button.

The Mute program places a speaker icon in the Tray on the Taskbar. When sounds are off, the speaker icon is displayed with an X through it, and the volume is set to 0.

NEW! Better UI

To turn sounds on or off * Tap the speaker icon.

To quit the Mute program 1. Double-tap the speaker icon. 2. In the Mute Speaker dialog box, tap YES to exit.

Remote Control for Windows CE -- NEW! Control the HPC from your PC.

NEW! big keyboard data entry, screenshot your HPC and make presentations

PC keyboard and mouse affect the HPC HPC screen is visible on PC so you can make presentation and record screenshots Clipboard is available so you can copy data between the devices

Remote Control works closely with the Operating System and must be uninstalled differently. Uninstallation directions 

  1.Select Run from the Start Menu
  2.Type: rcreg unload
  3.You will be asked to reset the device 
  4.After you have reset the device you will be able to use the Remove
Programs Control 
  Panel to remove Remote Control.
Control Panel Annunciators Quickly open your favorite Control Panels from the taskbar.

To add control panel buttons to the taskbar 1. In Control Panel, select all the control panels you would like to appear as taskbar buttons. 2. ALT+TAP one of the selected control panels, and then tap Add Annunciator on the shortcut menu.

To remove control panel buttons from the taskbar 1. In Control Panel, select all the control panels you would like to remove. 2. ALT+TAP one of the selected control panels you selected, and tap Remove Annunciator on the shortcut menu.

Sound Schemes Copy one or more of the following custom sound schemes to your Handheld PC Use the Volume & Sounds control panel's Sound tab to activate them. The following schemes are available: Analog, Metallic, Organic.

Bitmap Tiles Copies 9 more bitmaps to your Handheld PC. Use the Display control panel to select a new image as the Windows CE wallpaper. The following bitmaps are available: Arcade, Argyle, Cars, Castle, Chitz, Concrete, Honey, Squares, Thatch, Weave
