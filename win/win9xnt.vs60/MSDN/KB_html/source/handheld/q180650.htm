Handheld PC Does Not Start After You Install Batteries

 Last reviewed: February 6, 1998
Article ID: Q180650
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you install new batteries in your Handheld PC (H/PC), the computer may not seem to start.

CAUSE

This behavior can occur for any of the following reasons:

  • You install the backup battery before the main batteries in your H/PC.
  • You do not lock the battery cover on your H/PC.
  • The display contrast is not adjusted correctly.

RESOLUTION

To resolve this behavior, use the following steps:

  1. Remove the backup and main batteries.

  2. Install the main batteries, and then install the backup batteries.

  3. Lock the battery cover.

  4. Start your H/PC.

  5. Adjust the display contrast.

For information about how to install batteries in your H/PC or how to adjust the display contrast, refer to the H/PC's documentation or manufacturer.
