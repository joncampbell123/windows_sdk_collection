Columns Do Not Appear in Tasks Program in Windows CE

 Last reviewed: May 23, 1997
Article ID: Q169027
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you open the Tasks program in Windows CE, you may experience one or more of the following symptoms:

  • No columns of information appear on the screen.
  • No tasks are displayed.

CAUSE

These behaviors can occur if the columns in the Tasks program have been set to a width of zero.

RESOLUTION

To resolve this issue, resize the columns using the stylus.
