Cannot Copy Pocket Excel Files to Your Desktop Computer

 Last reviewed: November 21, 1997
Article ID: Q177133
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

You may not be able to copy a Microsoft Pocket Excel file to your desktop computer.

CAUSE

This problem can occur if you create a Pocket Excel file with more than 240 characters in the file name.

RESOLUTION

To work around this problem, use a file name with 240 characters or less for each Pocket Excel file.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows CE version 2.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
