Keyboard Focus Lost After Changing Task Due Date to None

 Last reviewed: May 30, 1997
Article ID: Q169348
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you set the due date for a task in the Tasks program to None, you may not be able to use the TAB key to advance to the next field in the Task dialog box.

CAUSE

This behavior occurs when you set the Start field to a specific date, set the Due field to None, and then press the TAB key. At this point, the keyboard loses focus and you can no longer navigate the Task dialog box using the keyboard.

NOTE: This problem does not occur when you use the Edit Panel in the Task program to edit or create a task.

RESOLUTION

To restore focus, tap the screen on your Handheld PC (H/PC).
