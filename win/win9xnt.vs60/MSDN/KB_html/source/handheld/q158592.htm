Problems When Too Many Fonts Are Installed

 Last reviewed: April 14, 1997
Article ID: Q158592
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you have numerous fonts installed, you may experience any of the following symptoms:

  • The installed fonts are not available in the font list in Microsoft Pocket Word.
  • Icons disappear from the desktop and from My Handheld PC.
  • The command bars in your programs are distorted.
  • The Handheld PC (H/PC) may stop responding.

CAUSE

These problems can occur if you have too many fonts installed for the amount of available program memory.

Each TrueType font has several sizes, and the number of fonts installed may consume 1 MB of global windows environment (GWE) memory. Once the GWE heap reaches 1 MB, Windows CE may fail to properly allocate GWE memory.

RESOLUTION

To resolve these problems, reduce the number of fonts you have installed by deleting fonts from the Windows\Fonts folder.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
