Err Msg: Cannot Convert File. Cannot Locate Microsoft...

 Last reviewed: February 16, 1998
Article ID: Q163092
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you attempt to copy or move a Microsoft Word document from your desktop computer to your Handheld PC (H/PC), you may receive the following error message: 

   Cannot convert file. Cannot locate Microsoft Conversion Library
   (msconv32.dll).

CAUSE

The Mswrd632.cnv file is installed when you install Microsoft Word version 7.0 for Windows 95 and is required to convert Microsoft Word documents (.doc files) to Pocket Word documents (.pwd files.) Some programs may install a different version of the Mswrd632.cnv file that requires the Msconv32.dll file. If the Msconv32.dll file is missing, you may receive this error message.

RESOLUTION

To resolve this problem, use one of the following methods:

  • Restore the missing Msconv32.dll file. To do so, follow these steps:

    1. Reinstall the program that installed the version of the Mswrd632.cnv 

          file that requires the Msconv32.dll file.

    2. Reinstall H/PC Explorer over the existing installation.

    NOTE: Some programs may install a file named Msconv97.dll. In this case, substitute "Msconv97.dll" in place of "Msconv32.dll" in the above steps.

  • Reinstall the original version of the Mswrd632.cnv file from Microsoft Word version 7.0 for Windows 95. To do so, follow these steps:

    1. Click the Start button, point to Find, and then click Files Or 

          Folders.

    2. In the Named box, type "mswrd632.cnv" (without quotation marks), 

          make sure that the Look In box contains the root folder of the drive
      on which Microsoft Windows 95 is installed, and then click Find Now.

    3. When the file is located, use the right mouse button to click the 

          file, click Rename on the menu that appears, and then rename the
      file to Mswrd632.new.

    4. Reinstall Microsoft Word version 7.0 for Windows 95 over the 

          existing installation.

    5. Reinstall H/PC Explorer over the existing installation.
