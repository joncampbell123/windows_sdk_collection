Cannot Save a 7:00 A.M. Appointment in Calendar

 Last reviewed: February 9, 1998
Article ID: Q180735
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you try to save an appointment for 7:00 A.M. in Calendar, you may receive an error message.

CAUSE

This issue can occur if you tap New Appointment on the File menu or press CTRL+N on any of the following Handheld PC (H/PC) models:

  • NEC MobilePro 400
  • NEC MobilePro 700
  • Sharp 4100
  • Sharp 4500

RESOLUTION

To work around this issue, use any of the following methods:

  • Do not create the appointment for 7:00 A.M. Instead, create the appointment for 6:59 A.M. or 7:01 A.M.
  • Do not type a description in the Appointment Description box.
  • Use Day or Week view to add your appointment in Calendar. For information about how to add an appointment, please refer to Windows CE Help. To do so, tap Start, tap Help, tap Calendar, and then tap Adding Appointments.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
