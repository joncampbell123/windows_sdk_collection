Err Msg: RPC Initialization Failure

 Last reviewed: May 6, 1997
Article ID: Q168073
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you connect your desktop computer and your Handheld PC (H/PC), you may receive the following error message: 

   RPC Initialization Failure.

CAUSE

This behavior can occur when a Beta version of Microsoft Proxy Server is installed on your computer.

RESOLUTION

To resolve this problem, uninstall the Beta version of Microsoft Proxy Server. To do so, follow these steps:

  1. In Control Panel, double-click Add/Remove Programs, and then click the Install/Uninstall tab.

  2. Click Microsoft Proxy Server, click Add/Remove, and then follow the instructions to remove the Beta version of Microsoft Proxy Server.

MORE INFORMATION

For information about obtaining the released version of Microsoft Proxy Server, see the following Microsoft Internet site: 

   http://www.microsoft.com/proxy/
