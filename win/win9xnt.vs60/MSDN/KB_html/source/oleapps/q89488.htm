Graph: Misspelled Word in Tick-Mark Help Topic

 Last reviewed: September 10, 1996
Article ID: Q89488
The information in this article applies to:
  • Microsoft Graph, versions 3.0, 3.0a, and 3.0b
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, and 2.0b
  • Microsoft Windows operating system versions 3.0 and 3.1

SUMMARY

In Microsoft Graph, there is a documentation error in the "Tick-Mark Labels" online Help topic.

MORE INFORMATION

In the third paragraph, the fourth word from the end is misspelled as "adjusing." The correct spelling is "adjusting."

To view this documentation error:

  1. From the Help menu, choose Help Index. Choose the Search button.

  2. Select Tick-Mark Labels and choose the Show Topics button.

  3. Select the Tick-Mark Labels topic and choose the Go To button.
