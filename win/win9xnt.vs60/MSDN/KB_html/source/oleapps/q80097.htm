Equation Editor: Changing the Default Font

 Last reviewed: January 20, 1997
Article ID: Q80097
The information in this article applies to:
  • Microsoft Equation Editor for Windows, versions 1.0, 2.0, 2.0a, 2.1, 3.0

SUMMARY

In the Microsoft Equation Editor, you can change the primary (default) font for each of the eight styles listed in the Styles (Style Define in version 1.0) dialog box. To do this, choose Define from the Style menu.

In the "Microsoft Equation Editor User's Guide," there is no reference to the default font. The default font is always referred to as the "primary font."

REFERENCES

"Microsoft Equation Editor User's Guide," version 2.0, pages 66-68, 71-72
