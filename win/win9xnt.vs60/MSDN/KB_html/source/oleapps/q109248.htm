WordArt Object Isn't Updated with Quick Print Send

 Last reviewed: September 9, 1996
Article ID: Q109248
The information in this article applies to:
  • Microsoft WordArt, version 2.0

SYMPTOMS

In Word for Windows, if you send a print job immediately after inserting a WordArt object, the printed output may not be updated to include the correct WordArt text.

RESOLUTION

Allow the display to update before you send the print job.

MORE INFORMATION

Steps to Reproduce Problem

  1. Start Word for Windows.

  2. From Insert menu, choose Object.

  3. From the list, select Microsoft WordArt 2.0, then choose OK.

  4. In the WordArt text box, change the text from "Your Text Here" to some other text string.

  5. Very quickly press the following key combination: 

          ESC, CTRL+P, ENTER

This sends the print job without allowing the WordArt object to be updated. The printout contains a WordArt object with the text, "Your Text Here."
