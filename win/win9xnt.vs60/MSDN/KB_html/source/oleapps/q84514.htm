Equation Editor Fonts Don't Print on HP LaserJet Plus or 500

 Last reviewed: September 9, 1996
Article ID: Q84514
The information in this article applies to:
  • Microsoft Equation Editor version 1.0

SUMMARY

The following Microsoft Equation Editor PCL fonts do not print on the Hewlett-Packard (HP) LaserJet Plus or LaserJet 500 printers because an incompatible font type setting is located in the headers of the soft font files: 

   MT Symbol
   MT Extra
   Fences

MORE INFORMATION

The Equation Editor PCL fonts are installed by the Microsoft Word version 2.0 for Windows Setup program if you are running Word in Microsoft Windows version 3.0. In the headers of these soft font files, the font type byte of the font descriptor is set to 2. Font type 2 is not supported by the HP LaserJet Plus or the LaserJet 500; therefore, the Equation Editor PCL fonts do not print. The LaserJet Plus or 500 may substitute Courier for these fonts.

This problem does not occur if you are running Word in the Microsoft Windows operating system version 3.1. If you install the Equation Editor while running Windows version 3.1, the TrueType versions of the Symbol, MT Extra, and Fences fonts are installed instead of the PCL versions.
