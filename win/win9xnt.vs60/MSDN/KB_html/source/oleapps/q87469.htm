Graph 3.0: Data Labels Overlay Data Bar or Line

 Last reviewed: April 2, 1997
Article ID: Q87469
3.00 3.00b WINDOWS kbtool

The information in this article applies to:

  • Microsoft Graph, versions 3.0, 3.0b
  • Microsoft Word for Windows, version 2.0
  • Microsoft Windows operating system versions 3.0, 3.1

SUMMARY

Microsoft Graph version 3.0 overlays data labels on a data bar or line instead of positioning them at the top or end of the data bar or line. Microsoft has confirmed this to be a problem in Graph 3.0. This problem was corrected in Graph 3.0a. Microsoft Graph 3.0a correctly positions data labels at the top or end of the data bar or line.

MORE INFORMATION

To create data labels, choose Data Labels from the Chart menu.

Microsoft Graph version 3.0 is included with version 2.0 of Microsoft Word for Windows. Microsoft Graph version 3.0a ships with versions 2.0a and 2.0a- CD of Word for Windows. If you are experiencing problems with data label positioning in Graph 3.0, obtain the version 2.0a Word for Windows upgrade.
