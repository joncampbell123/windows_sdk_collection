Using the Equation Editor with ZenoGraphics SuperPrint

 Last reviewed: September 9, 1996
Article ID: Q80454
The information in this article applies to:
  • Microsoft Equation Editor version 1.0

SUMMARY

The Microsoft Equation Editor fonts are not installed correctly when you use Zenographics SuperPrint. This causes the following error message: 

   Equation Editor's fonts may not have been installed...

MORE INFORMATION

SuperPrint has a separate initialization (INI) file for storage of its font information; this file cannot be edited by the Equation Editor's Setup program. You must add the Equation Editor fonts manually using SuperPrint's SuperText utility. In addition, you must follow certain steps after installing the fonts to ensure that they are properly recognized by the Equation Editor.

To manually install the Equation Editor's fonts for use with SuperPrint, use the steps below:

  1. Start the SuperText utility. (Its icon is a large red letter S.)

  2. Select the Adobe Type 1 Fonts window, and the choose the Typefaces 

        button.

  3. In the Typeface dialog box, choose the Install button. For the 

        File Path, choose the path where Equation Editor is installed (usually
    C:\WINDOWS\MSAPPS\EQUATION).

  4. The four Equation Editor typefaces appear on the Files list. Select 

        all four, and choose the OK button. Close the Typeface dialog box.

  5. The four Equation Editor typefaces now appear in the Adobe Type 1 

        Fonts window. Select all four typefaces.

  6. Choose the Typeface button again. Choose the Next button until the 

        "MT-Extra:Medium" font appears as the selected font. Choose the Revise
    button.

  7. Edit the base font name by removing the dash (--). The revised 

        base font name should read "MT Extra." Choose the OK button.

  8. Repeat steps 6 and 7 for the "MT-Symbol:Medium" font.

  9. Repeat step 6 for the font that appears as "MT-Symbol-:Medium." 

        Note: This font differs from MT-Extra because it has an extra dash
    after the word "Symbol."

  10. Edit the base font name for this font so that it reads "MT Symbol 

        Italic." Select the Italic box, and choose the OK button.

  11. Close the Typeface dialog box. Close the SuperText utility.

  12. In the Windows Control Panel, double-click the Printers icon.

  13. Select the SuperPrint driver in the Installed Printers box, press 

        ALT+D to make it the default driver, and choose the OK button. At this
    point, you should receive a "Rebuilding font database" message.

The Equation Editor fonts should now be installed correctly for use with SuperPrint.
