Draw: Error in Help Section on Filled and Unfilled Objects

 Last reviewed: September 9, 1996
Article ID: Q79699
The information in this article applies to:
  • Microsoft Draw version 1.0

SUMMARY

In online help for Microsoft Draw version 1.0, the second set of drawings in the section for drawing filled and unfilled objects, is incorrect. To access this screen, choose Index from the Help menu, press the Search button, type fill, press the Show Topics button, select "Drawing filled and unfilled objects," and choose the Go To button. The second set of drawings should depict unframed but filled objects; however, the freeform drawing examples appear framed and unfilled.

Microsoft has confirmed this to be a problem in Microsoft Draw version 1.0. We are researching this problem and will post new information here as it becomes available.
