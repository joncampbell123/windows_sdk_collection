WordArt: No Large Buttons Available in WordArt

 Last reviewed: February 27, 1998
Article ID: Q137132
The information in this article applies to:
  • Microsoft WordArt for Windows 95, version 3.0
  • Microsoft WordArt for Windows, version 2.0

WordArt 2.0 and 3.0 do not contain large button images. If you are using large buttons in Publisher, Word, PowerPoint or Microsoft Excel, and if you edit a WordArt object, your toolbar will appear to shrink. Once you click outside the WordArt object, however, the toolbar will return to normal size.
