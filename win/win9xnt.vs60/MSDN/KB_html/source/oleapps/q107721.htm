OFF: Removing Line From 100% Pie Chart

 Last reviewed: February 3, 1998
Article ID: Q107721
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Graph 98 for Macintosh Edition
  • Microsoft Graph 97 for Windows
  • Microsoft Graph for Windows 95, version 5.0
  • Microsoft Graph for the Macintosh, versions 3.0 and 5.0
  • Microsoft Graph, versions 3.0, 3.0b

SUMMARY

If a three-dimensional pie chart is created in Microsoft Graph, and the data for the chart consists of a single number, that number will produce a one-piece pie chart with a line connecting the center point to the outside of the pie. There is no way to remove this central line using Microsoft Graph.

The following article describes a method to remove the line using Microsoft PowerPoint.

RESOLUTION

If you want this pie chart to appear in a PowerPoint slide without the single line (for example, you want to use a cylindrical shape in a system diagram), you can perform the following steps:

PowerPoint 95, 97, and 98

  1. Select the chart in Slide view.

  2. On the Draw menu, click Ungroup. WARNING: Ungrouping the Chart converts the chart to Microsoft Drawing objects, you are no longer able to edit the object with the Graph program. However, if needed you can undo the ungroup operation.

  3. Then you may see a message similar to the following: 

          This is an imported object not a group. If you convert it to a
      Microsoft Office drawing, embedded data or linking information will
      be lost. Do you want to convert the object?

    Click Yes to ungroup the object.

  4. Select the part of the chart that includes the line.

  5. On the Format menu. click Format AutoShape.

  6. Click the Colors and Lines Tab.

  7. Set the Line Color to No Line and click OK.

To ensure a consistant look and feel, remove the lines from the other parts of the chart.

PowerPoint 3.0 for Windows

NOTE: Do NOT do this with a two-dimensional pie chart in PowerPoint for Windows. It will result in the loss of the chart.

  1. Select the chart in Slide view.

  2. On the Arrange menu, click Ungroup. WARNING: This will break the link between the chart and the datasheet, and you will no longer be able to edit the chart in Microsoft Graph.

  3. Click away from the chart so that no objects are selected.

  4. Double-click the top portion of the chart (shaped like an ellipse). The shape will turn white and there should be a small dot in the center of the ellipse.

  5. Click the small dot in the center and drag it to the edge of the graph at the point where the line normally meets the edge. (If the invisible PowerPoint grid is on you may have to hold down the ALT key in order to drag precisely to the edge of the ellipse.)

  6. Release the mouse button.

  7. Click away from the chart.

PowerPoint 3.0 for the Macintosh

  1. Select the chart in Slide view.

  2. On the Arrange menu, click Ungroup. WARNING: This will break the link between the chart and the datasheet, and you will no longer be able to edit the chart in Microsoft Graph.

  3. Click away from the chart so that no objects are selected.

  4. Click the line.

  5. Press the DEL key.

  6. There may be two lines. If this is the case, repeat steps 4 and 5 to delete the remaining line.
