DRW97: Microsoft Draw 97 Available for Download

 Last reviewed: February 23, 1998
Article ID: Q181153
The information in this article applies to:
  • Microsoft Draw 97 for Windows

SUMMARY

The following file is available for download from the Microsoft Software Library. With Microsoft Draw 97, you can create 3-D text effects, diagrams, and so on. Draw 97 is an OLE server and can be used with any OLE-compliant program.
