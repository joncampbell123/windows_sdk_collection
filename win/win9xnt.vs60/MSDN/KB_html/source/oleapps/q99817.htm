Introduction

Microsoft ClipArt Gallery objects on PowerPC 603- or 604-based Power Macintosh computers may appear as gray hashed boxes instead of the selected ClipArt Gallery images. This problem may be related to the status of the Modern Memory Manager.

The ClipArt Gallery 1.1a Patch included with this Application Note allows Microsoft ClipArt Gallery to function with Modern Memory Manager active on a Power Macintosh with the PowerPC 603 chip and with Modern Memory Manager disabled on a Power Macintosh with the 604 chip.

For a more comprehensive description of Modern Memory Manager and the related ClipArt Gallery issues, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID:    Q135257
        TITLE:    OFF4: ClipArt Object Appears as a Gray Hashed Box
To determine if your Power Macintosh uses the PowerPC 603 or 604 processor, please consult your user's manual.

To install the ClipArt Gallery 1.1a Patch

Run the ClipArt Gallery 1.1a Patch program you received with this Application Note. After reading the directions, click the Patch button in the lower-right corner.

You will be prompted to find the current version of Microsoft ClipArt Gallery. ClipArt Gallery should be located in the Microsoft folder on your hard disk. Select the ClipArt Gallery program and click the Patch button.

The patch will update ClipArt Gallery from version 1.1 to 1.1a and will then inform you that the patch was successfully applied. After you quit the patch program, ClipArt Gallery will be updated.
