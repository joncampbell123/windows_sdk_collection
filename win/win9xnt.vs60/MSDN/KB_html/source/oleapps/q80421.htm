WordArt Gives "Too Large To Fit" Error on Original Image

 Last reviewed: September 10, 1996
Article ID: Q80421
The information in this article applies to:
  • Microsoft WordArt version 1.0

SUMMARY

The default size for Microsoft WordArt images is 2 inches by 2 inches. If an original WordArt image exceeds this size because of the length of the text or the point size of the font selected, the following error message displays: 

   Enlarge WordArt component in Document1?

   Your text or point size is too large to fit.
   Choose OK to have the object enlarged. Choose
   Cancel to continue edit in WordArt.

MORE INFORMATION

Microsoft WordArt is an OLE application that ships with Microsoft Word for Windows version 2.0. The above WordArt error message warns you that the current drawing has exceeded its original size because of changes you have made.

Microsoft has confirmed this to be a problem in Microsoft WordArt. We are researching this problem and will post new information here as it becomes available.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, pages 415-416
