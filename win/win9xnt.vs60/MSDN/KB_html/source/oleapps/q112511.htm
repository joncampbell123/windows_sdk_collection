Graph 5.0: Cannot Use First Row for x Data on XY Scatter Chart

 Last reviewed: September 10, 1996
Article ID: Q112511
The information in this article applies to:
  • Microsoft Graph for Windows, version 5.0

SYMPTOMS

In Microsoft Graph, when you import data on a data sheet for an xy (scatter) chart using the Paste Link command, you are unable to use the first row of data for the x data.

-or-

When you import data on a data sheet for an xy (scatter) chart using the Paste Link command, you are unable to use the first row of data for the first data series.

CAUSE

When you paste link data to a data sheet for an xy (scatter) chart in Microsoft Graph, the cell in which you paste the data determines whether the first row of your data is used for the x data or for the first data series. This is true even though the ChartWizard gives you the option to choose whether the first row of data you are linking is used for x data or the first data series.

WORKAROUND

To use the first row for the x data on an xy (scatter) chart, when you paste link data to the data sheet, select a cell in the X row on the data sheet before you choose Paste Link from the Edit menu.

To use the first row for the first data series on an xy (scatter) chart, when you paste link data to the data sheet for that chart, select a cell in a row below the X row on the data sheet before you choose the Paste Link command from the Edit menu.

MORE INFORMATION

When you import data on a data sheet in Microsoft Graph by using the Paste Link command, the ChartWizard dialog box appears and lets you choose how you want your data to be imported. When you link data on a data sheet for an xy (scatter) chart, the ChartWizard gives you the option to use the first row of your data either for the x data or for the first series data. However, this option depends on the cell selected on the data sheet when you choose the Paste Link command. The option you choose under Use First Row For in the ChartWizard does not affect this option.

REFERENCES

For more information about entering data for an xy (scatter) chart, choose the Search button in Microsoft Graph Help and type: 

   xy charts
