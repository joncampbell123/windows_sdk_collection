OFF4: ClipArt Object Appears as a Gray Hashed Box

 Last reviewed: February 3, 1998
Article ID: Q135257
The information in this article applies to:
  • Microsoft Office for the Power Macintosh, versions 4.2.1, 4.2.1a
  • Microsoft PowerPoint for the Power Macintosh, version 4.0
  • Microsoft Excel for the Power Macintosh, versions 5.0, 5.0a
  • Microsoft Word for the Power Macintosh, versions 6.0, 6.0.1
  • Microsoft Works for the Macintosh, versions 4.0, 4.0a, and 4.0b
  • Microsoft Works for the Power Macintosh, versions 4.0, 4.0a, and 4.0b
  • Microsoft Bookshelf for the Macintosh, 1994 edition
  • Microsoft Works & Bookshelf '95 for the Macintosh

SYMPTOMS

After you insert a ClipArt Gallery object into any of the above listed programs, the clip art may appear as a gray hashed box (in PowerPoint) or as an empty white object (in Microsoft Excel and Microsoft Word). Or you may see the following error message: 

   The server application, source file, or item cannot be
   found. Make sure the application is properly installed,
   and that it has not been deleted, moved, or renamed.
This problem may occur only on machines with the PowerPC 603 processor when the Modern Memory Manager is turned ON in the Memory control panel.

CAUSE

Microsoft ClipArt Gallery is designed to directly take advantage of the Motorola 68000-series of processors. ClipArt Gallery may not function properly on the PowerPC 603 processor with Modern Memory Manager enabled.

RESOLUTION

Microsoft has an Application Note available that includes a patch to correct this problem. For information about obtaining the patch, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q99817
   TITLE     : ClipArt Gallery 1.1a Patch Available for Power Macintosh
To work around this problem if you do not yet have the patch, open the Memory control Panel and turn off the Modern Memory Manager.

NOTE: Modern Memory Manager can only be turned off if you are running the Macintosh Operating System (OS) version 7.5.5 or earlier.

To turn off the Modern Memory Manager, follow these steps:

  1. On the Apple menu, click Control Panels.

  2. Double-click the Memory control panel.

  3. In the Modern Memory Manager section, click Off.

  4. Restart your computer for the change to take effect.

NOTE: This issue is unique to the PowerPC 603-series of Macintoshes. If you run ClipArt gallery on a 68000-series Macintosh with a PowerPC Upgrade Card, Modern Memory Manager can be turned ON.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

For additional information on the Modern Memory Manager, please contact Apple Technical Support at (800) SOS-APPL (767-2775).
