OFF4: PowerMac Graph 5.0 Requires Microsoft Dialog Library

 Last reviewed: February 3, 1998
Article ID: Q143016
The information in this article applies to:
  • Microsoft Graph for the Power Macintosh, version 5.0
  • Microsoft PowerPoint for Macintosh, version 4.0

SYMPTOMS

When you attempt to Start Microsoft Graph 5.0 for the Power Macintosh from the Finder, you may experience one of the following problems:

  • The zoom rectangle expands but the program will not start.
  • A message similar to the following appears: 

          The program "Microsoft Graph 5" could not be opened because
      "MicrosoftDialogLib" could not be found.

In addition, if you start Graph from any program that supports OLE, you may receive a message similar to the following: 

   The server program, source file, or item cannot be found.
   Make sure the program is properly installed and that it has
   not been deleted, moved, or renamed.

CAUSE

The Microsoft Dialog Library may be corrupted or missing. The Power Macintosh version of Graph requires this library to start.

At start time, Microsoft Graph searches the Extensions folder for the Microsoft Dialog Library. If the library is not found or is corrupted, the Power Macintosh version of Graph will not function.

RESOLUTION

Look in the Extensions folder for the Microsoft Dialog Library. The Extensions folder is located within your System Folder. If the Dialog Library is not found, search the computer. If you find the file, place the Microsoft Dialog Library into the Extensions folder.

Microsoft Graph should now Start correctly.

NOTE: You do not need to restart your computer for Graph to access the Microsoft Dialog Library: library extensions are loaded into RAM only when they are needed.

If the Microsoft Dialog Library is not found, install a "clean" copy of the library.

For additional information on installing an OLE component, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q139745
   TITLE     : How to Add an Extension with Office 4.2.1
NOTE: If you have an option 2 (users share the programs on the server) administrative installation of Office on a server, an uncompressed version of the Microsoft Dialog Library can be found within the OLE folder.

MORE INFORMATION

The "server program..." message you may receive when you start Graph from a container program is a generic message. This message may be displayed for other reasons. If the resolution listed above fails to correct the problem, please follow standard "server program..." troubleshooting steps.
