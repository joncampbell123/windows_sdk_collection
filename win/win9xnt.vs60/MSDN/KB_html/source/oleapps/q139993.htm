OFF4: ClipArt Image Does Not Appear with PCI Macintosh

 Last reviewed: March 13, 1998
Article ID: Q139993
The information in this article applies to:
  • Microsoft ClipArt Gallery for the Macintosh, version 1.1
  • Microsoft Office for the Power Macintosh, versions 4.2.1, 4.2.1a
  • Microsoft PowerPoint for the Power Macintosh, version 4.0
  • Microsoft Excel for the Power Macintosh, versions 5.0, 5.0a
  • Microsoft Word for the Power Macintosh, versions 6.0, 6.0.1
  • Microsoft Works for the Macintosh 4.0, 4.0a, and 4.0b
  • Multimedia Microsoft Works for the Macintosh, version 4.0b.

SYMPTOMS

When you attempt to insert a ClipArt Gallery image into a Microsoft Office program (Word, PowerPoint, or Excel), the ClipArt image may not appear. The exact behavior varies from program to program. After selecting the thumbnail from the gallery, any one of the following symptoms may occur.

Word

The image does not appear and no message is displayed.

Excel

A blank object is displayed for the placeholder.

PowerPoint

A gray hashed box is displayed for the ClipArt object placeholder.

CAUSE

With some Power Macintosh computers, ClipArt Gallery does not function correctly unless the Modern Memory Manager is enabled.

RESOLUTION

To resolve this problem, open the Memory control Panel and enable the Modern Memory Manager. To do this, follow these steps:

  1. Click Control Panels on the Apple menu.

  2. Double-click the Memory control panel.

  3. In the Modern Memory Manager section, click On.

  4. Restart your computer for the change to take effect.

ClipArt will now import correctly into your Office program.

NOTE: Power Macintosh computers with the 603 processor (Performa 5200 and 6200 series) have the opposite problem, ClipArt objects will not appear correctly unless the Modern Memory Manager is off. However, if you are running the Macintosh Operating System (OS) version 7.6 or later, Modern Memory Manager cannot be turned off. Therefore, to successfully insert ClipArt images please obtain the ClipArt Gallery 1.1a Patch.

Microsoft has an Application Note available that includes the ClipArt Gallery 1.1a patch to correct this problem. For information about obtaining the patch, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q99817
   TITLE     : ClipArt Gallery 1.1a Patch Available for Power Macintosh
