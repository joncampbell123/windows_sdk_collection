Draw: Hidden Lines Print on Plotters

 Last reviewed: September 9, 1996
Article ID: Q80295
The information in this article applies to:
  • Microsoft Draw version 1.0

SUMMARY

If you plot an embedded object from Microsoft Draw version 1.0, any hidden lines in the object are plotted.

Steps to Reproduce Problem

  1. From the Insert menu in Word for Windows version 2.0, choose Object, select Microsoft Drawing, and choose the OK button.

  2. Select the Ellipse tool, and draw two overlapping ellipses.

  3. From the File menu, choose Exit and Return to Document X to update the Word for Windows document to include the object.

  4. From the File menu, choose Print, and then choose the OK button to print the Word document using the Hewlett-Packard plotter driver. The lines hidden by the overlapping ellipses are plotted.

MORE INFORMATION

This problem occurs in all Windows applications when printing Microsoft Draw objects with hidden lines to a plotter. Microsoft is researching this problem and will post new information here as it becomes available.
