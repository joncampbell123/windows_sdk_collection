Publisher: Missing Text in WordArt 2.0 Object

 Last reviewed: February 13, 1998
Article ID: Q101819

The information in this article applies to:

  • Microsoft WordArt version 2.0
  • Microsoft Publisher version 2.0

SYMPTOMS

WordArt 2.0 objects in Microsoft Publisher do not display text. When you create a WordArt 2.0 object in Publisher 2.0, you see the text you type, both in the editing box and on the page. Once you click away from the WordArt 2.0 editing box and return to the document, the text vanishes.

CAUSE

This problem occurs if the OLE2.REG file is not properly registered during setup of Publisher 2.0.

WORKAROUND

To work around this problem, do the following:

  1. Quit Publisher 2.0.

  2. In Program Manager, choose Run from the File menu.

  3. Type "regedit /u ole2.reg" (without the quotation marks).

    The following message should be displayed: 

          Information in ole2.reg has successfully registered.

  4. Repeat step 3 replacing "regedit /u ole2.reg" with "regedit /u \windows\msapps\wordart\wordart2.reg"

Open Publisher 2.0 and create a new WordArt 2.0 object, the text should now appear in your document.
