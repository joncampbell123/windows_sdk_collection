CG1x: "Unable to Locate PICSTORE.DLL" Using ClipArt Gallery

 Last reviewed: February 27, 1998
Article ID: Q119387
The information in this article applies to:
  • Microsoft ClipArt Gallery version 1.1
  • Microsoft Publisher versions 2.0, 2.0cd, 2.0a, 2.0acd

SYMPTOMS

The ClipArt Gallery fails with the following error message: 

   Unable to locate PICSTORE.DLL.

CAUSE

The latest version of ClipArt Gallery, ARTGALRY.EXE version 1.1 dated 6/3/94, searches the system for PICSTORE.DLL. PICSTORE.DLL is a file that ships with PowerPoint for Windows versions 4.0x. If ClipArt Gallery cannot find this file, it generates the above error message.

The above error is displayed if Microsoft PowerPoint for Windows 4.0x had been installed, including ClipArt Gallery, and PICSTORE.DLL is no longer on the system.

WORKAROUND

  1. In the Windows Accessories group, double-click the Windows Notepad icon.

  2. Type a single character or number.

  3. From the File menu, choose Save.

  4. Save the file as PICSTORE.DLL to the Windows SYSTEM subdirectory.

  5. From the File menu, choose Exit.

Note: This is a workaround for the problem with Clipart Gallery and Publisher. It does not work if PowerPoint or Office are being used.
