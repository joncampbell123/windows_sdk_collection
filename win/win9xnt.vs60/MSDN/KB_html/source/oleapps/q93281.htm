General Protection Fault in Equation Editor

 Last reviewed: September 9, 1996
Article ID: Q93281
The information in this article applies to:
  • Microsoft Equation Editor version 1.0

SYMPTOMS

If you use Equation Editor with the view set to 100 percent, a general protection (GP) fault may occur when you select a tool to build an equation.

RESOLUTION

To avoid this problem, create equations with the view set to 200 percent. After creating your equations, you can switch back to a view of 100 percent.

STATUS

Microsoft has confirmed this to be a problem in Equation Editor, version 1.0. We are researching this problem and will post new information here as it becomes available.

REFERENCES

"Microsoft Equation Editor User's Guide," version 1.0, pages 11, 55- 56, 86-87
