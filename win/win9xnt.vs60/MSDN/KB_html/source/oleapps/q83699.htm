ANSI Characters That Are Not Supported by WordArt

 Last reviewed: September 9, 1996
Article ID: Q83699
The information in this article applies to:
  • Microsoft WordArt version 1.0

SUMMARY

Microsoft WordArt, an OLE server application included with Microsoft Word for Windows version 2.0, does not support the full range of ANSI characters. You can include unsupported ANSI characters in WordArt; however, they are not visible in the client Word for Windows document.

MORE INFORMATION

The following are the ANSI codes for characters NOT supported by WordArt:

ANSI code numbers 164, 166, 167, 171, 172, 173, 175, 176, 177, 178, 179, 181, 182, 183, 184, 185, 187, 188, 189, 190, 208, 215, 222, 240, 247, 254

The ANSI character set is displayed on page 568 of the "Microsoft Windows User's Guide" for Microsoft Windows version 3.0.

REFERENCES

"Microsoft Windows User's Guide," version 3.0, page 568
