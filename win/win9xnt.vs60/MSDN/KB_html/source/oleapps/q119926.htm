CG1x: "Cannot Find CTL3DV2.DLL" Error Starting ClipArt Gallery

 Last reviewed: February 27, 1998
Article ID: Q119926
The information in this article applies to:
  • Microsoft ClipArt Gallery version 1.1

SYMPTOMS

When you start ClipArt Gallery from any OLE container application in Windows, the following error message is displayed: 

   File Error
   Cannot find CTL3DV2.DLL.
The only option is to choose OK. When you choose OK, the following error message is displayed: 

   The server application cannot be found.
   Make sure that the application is properly installed, or exists in
   your DOS path, and that it has not been deleted, moved, or renamed.

CAUSE

This condition is caused by a damaged, deleted, or missing CTL3DV2.DLL file. This file should be in the Windows SYSTEM subdirectory and is required by ClipArt Gallery version 1.1.

RESOLUTION

  1. Search the hard disk for any instances of CTL3DV2.DLL.

  2. Delete all instances of CTL3DV2.DLL.

  3. Insert the first disk for the product and copy DECOMP.EXE into your C:\WINDOWS directory.

  4. Insert the disk containing CTL3DV2.DL_ into your disk drive. At the MS- DOS prompt, type the following command and press ENTER: 

          C:\WINDOWS\DECOMP.EXE A:\CTL3DV2.DL_ C:\WINDOWS\SYSTEM\CTL3DV2.DLL

    This command assumes that your floppy disk is drive A: and that Windows is installed in the C:\WINDOWS directory.

MORE INFORMATION

Microsoft ClipArt Gallery ships with Office versions 4.0, 4.2, and 4.3 for Windows and PowerPoint version for Windows 4.0. The latest version of ClipArt Gallery can also be obtained as an Application Note from Microsoft Product Support Services.
