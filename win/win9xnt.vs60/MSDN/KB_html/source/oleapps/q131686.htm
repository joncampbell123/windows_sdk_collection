OFF4: Clipart Gallery Hangs System on Upgraded Power Macs

 Last reviewed: February 3, 1998
Article ID: Q131686

The information in this article applies to:
  • Microsoft ClipArt Gallery for the Macintosh, version 1.1
  • Microsoft PowerPoint for the Macintosh, version 4.0
  • Microsoft Office for the Macintosh, version 4.2

SYMPTOMS

If you are using PowerPoint on a Quadra, Centris, or IIci Macintosh computer that has been upgraded with the PowerMac Upgrade card, Clipart Gallery may hang when you exit and return to PowerPoint. When this problem occurs, a gray hash box will appear on the slide in place of the Clipart graphic.

CAUSE

This error occurs if the Modern Memory Manager control panel device has been turned off.

RESOLUTION

Modern Memory Manager is required on Macintoshes that have been upgraded with the PowerMac Upgrade kit.

To turn on the Modern Memory Manager, follow these steps:

  1. Click the Apple menu icon and choose the Control Panel entry.

  2. From Control Panel, choose the Memory control panel device.

  3. On the Memory control panel, click the On button for Modern Memory Management.

Note, Modern Memory Manager will be active after you restart your Macintosh.
