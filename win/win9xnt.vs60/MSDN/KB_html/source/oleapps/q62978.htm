MS Graph: Editing Data Within Cells

 Last reviewed: January 20, 1997
Article ID: Q62978
The information in this article applies to:
  • PowerPoint Graph, version 1.0
  • Microsoft Graph for Windows, versions 3.0 and 5.0
  • Microsoft Graph for Windows 95, version 5.0
  • Microsoft Graph for the Macintosh, versions 3.0 and 5.0
  • Microsoft Graph 97 for Windows

SUMMARY

In Microsoft Graph, you can use either of the following methods to edit a cell:

Method 1: Click a cell and start typing. The data you type replaces all of 

          the existing data in that cell when you press ENTER.
Method 2: Double-click the cell. A dialog box appears with the contents of 
          the cell. You don't have to retype all the data. You can move the
          insertion point within this dialog box and enter or change data
          as necessary. Click OK or press ENTER when done.

MORE INFORMATION

For more information about this topic, see page 268 of your "Microsoft PowerPoint for Windows User's Guide," version 2.0.
