General Protection (GP) Fault in ORGCHART.EXE

 Last reviewed: January 21, 1997
Article ID: Q128951
The information in this article applies to:
  • Microsoft Organization Chart for Windows, version 1.0

SYMPTOMS

When you start Organization Chart for Windows either from within another application or as a standalone application, you may receive the following error message: 

   MSORGCHART caused a General Protection Fault in module
   ORGCHART.EXE @ ####:####

CAUSE

This problem may occur if either the MSOCLIP.FEN or the MSOPREFS.FEN file has been corrupted. Both files are located in the Windows directory.

RESOLUTION

To correct this problem, rename both the MSOCLIP.FEN and MSOPREFS.FEN files from the Windows directory, and then restart Organization Chart. Both files are re-created by Organization Chart if they are not already present when it is started.

MORE INFORMATION

Removing and reinstalling Organization Chart via the Setup program does not correct this problem, because the MSOCLIP.FEN and MSOPREFS.FEN files are located in the Windows directory; therefore, these files are not removed by the Remove modes of the PowerPoint or Office Setup programs.
