Graph: Import Data Command Unavailable on Graph File Menu

 Last reviewed: September 10, 1996
Article ID: Q79642
The information in this article applies to:
  • Microsoft Graph for Windows version 3.0

SUMMARY

In the Microsoft Graph application that ships with Microsoft Word for Windows version 2.0, unless the spreadsheet is active, the Import Data command appears gray (unavailable) on the File menu. To activate the Microsoft Graph spreadsheet, choose Data Sheet from the Window menu or position the insertion point in the spreadsheet.

Note: When you use the Import Data command, Word inserts the data starting at the selected cell of your spreadsheet; therefore, make sure you select the appropriate cell after you activate the spreadsheet and before you choose Import Data.
