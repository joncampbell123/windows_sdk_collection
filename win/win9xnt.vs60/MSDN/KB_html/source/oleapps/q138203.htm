OFF: Symbols in Graph Line Charts Cannot Be Resized

 Last reviewed: February 3, 1998
Article ID: Q138203

The information in this article applies to:
  • Microsoft Graph 97 for Windows
  • Microsoft Graph for Windows 95, version 5.0
  • Microsoft Graph for Windows, version 5.0
  • Microsoft Graph for Macintosh, version 5.0
  • Microsoft Office 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for the Macintosh, version 4.0
  • Microsoft PowerPoint for Windows, versions 4.0, 4.0a and 4.0c
  • Microsoft PowerPoint for Windows 95, version 7.0

SYMPTOMS

When a Microsoft Graph line chart is inserted into PowerPoint, the symbols on each data point are very small and cannot be resized.

RESOLUTION

To change the size of the symbols, use one of the following methods:

Method 1

IMPORTANT: The following steps instruct you to ungroup the chart. If you ungroup an embedded object, you can not edit it in the original source application.

  1. Make a backup copy of the presentation prior to following these steps.

  2. In PowerPoint, click the line chart to select it.

  3. On the Draw menu, click Ungroup.

    You will receive a message telling you that you will discard any embedded data by ungrouping the object. Click OK.

  4. Unselect everything by clicking a blank area of the slide.

  5. Press and hold down the SHIFT key. Click each of the data points that you want to resize.

  6. On the Draw menu, click Scale, and type a new value in the Scale To box.

  7. Click the Preview button to see how the change will look. When the size of the symbols is satisfactory, click OK.

The symbols in the chart will now be easier to see on the screen and when you print them.

Mehtod 2

You can also replace the data markers, or symbols, with pictures or drawn objects to make the data points easier to see. The advantage to replacing the data markers with pictures is that the graph object remains intact. Should your graph data change, you will not have to re-create a graph to reflect the new numbers.

To replace data markers with pictures:

  1. Switch to the application and the document containing the picture you want to use as a data marker.

  2. Copy the picture using the application's procedure for copying. (Shortcut: CTRL+C)

  3. Activate the chart in which you want to use the picture.

  4. Select the data series or individual marker you want to replace with the picture.

  5. Click the Paste button. (Shortcut: CTRL+V)

NOTE: If you replaced the data markers for a data series, the picture is also displayed as the legend key for the series.

This is not a problem in Graph 97 which allows you to format the size of the Marker from a minimum of 2 points to a maximum of 72 points.
