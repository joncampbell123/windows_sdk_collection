Microsoft Graph: Y-Axis Categories Appear in Reverse Order

 Last reviewed: September 10, 1996
Article ID: Q81036
The information in this article applies to:
  • Microsoft Graph for Windows version 3.0

SUMMARY

If you create a bar chart in Microsoft Graph version 3.0, which ships with Microsoft Word for Windows version 2.0, the chart displays the y-axis categories in reverse order compared with the data sheet.

Note: This behavior also occurs in Microsoft Excel for Windows versions prior to 3.0.

MORE INFORMATION

To display the y-axis categories in the same order as the data sheet, use the following steps:

  1. Select the y-axis. From the Format menu, choose Scale.

  2. Select the following three options:

    a. Value (Y) Axis Crosses Between Categories b. Categories in Reverse Order c. Value (Y) Axis Crosses at Maximum Categories

  3. Choose the OK button.

REFERENCES

"Microsoft Graph User's Guide," version 3.0, pages 87-90
