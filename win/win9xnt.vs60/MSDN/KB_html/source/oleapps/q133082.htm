OFF4: Selected Thumbnail Does Not Bring in Correct Image

 Last reviewed: February 3, 1998
Article ID: Q133082

The information in this article applies to:
  • Microsoft ClipArt Gallery, version 1.1
  • Microsoft PowerPoint for Windows, versions 4.0, 4.0a and 4.0c
  • Microsoft Office for Windows, versions 4.2c, 4.3 and 4.3c

SYMPTOMS

When you select a thumbnail image from the Clipart Gallery, you may not receive the correct clipart image.

CAUSE

This problem occurs as a result of corruption in the ClipArt Gallery Index file, Artgalry.idx.

RESOLUTION

To rebuild the ClipArt Gallery Index file, delete or rename the Artgalry.idx and Artgalry.thm files and reinstall the PowerPoint clipart.

By default, Artgalry.idx and Artgalry.thm are located in the Windows\MSApps\Artgalry directory.

To Reinstall the PowerPoint Clipart File (using PowerPoint Setup)

  1. Start the PowerPoint Setup program.

  2. Click Add/Remove.

  3. Clear (take the "x" out of) the box to the left of the Clip Art Files choice.

  4. Click Continue.

    Setup will delete the clipart files.

  5. Start the PowerPoint Setup program again.

  6. Click Add/Remove.

  7. Clear (take the "x" out of) the box to the left of the Clip Art Files choice, then click Change Options.

  8. Select (put an "x" in) the box to the left of each clipart category you would like to install. Click OK.

  9. Click Continue, and provide Setup with the appropriate disks.

To Reinstall the PowerPoint Clipart File (using Office Setup)

  1. Start the Office Setup program.

  2. Click Add/Remove.

  3. Click Shared Applications, and click Change Options.

  4. Clear the Clip Art check box, and click OK.

  5. Click Continue.

    Setup will delete the clipart files.

  6. Restart the Office setup program again.

  7. Click Add/Remove.

  8. Click the Shared Applications choice and click Change Options.

  9. Select (put an "x" in) the box to the left of the Clip Art choice and click Change Options.

  10. Select each clipart category you would like to install. Click OK.

  11. Click OK again, and then click Continue. Provide Setup with the appropriate disks.

The next time you run ClipArt Gallery, you will see the following message: 

   The ClipArt Gallery cannot find

   C:\WINDOWS\msapps\artgalry\artgalry.idx

   This file contains information about thumbnail pictures for the
   ClipArt Gallery. Choose Browse to locate the file. Choose New
   to create a new database of thumbnail pictures.
If you click New, ClipArt Gallery will build new Artgalry.idx and Artgalry.thm files and register the PowerPoint clipart images in them.

MORE INFORMATION

When ClipArt Gallery is run for the first time it reads the information under the [Import] section of the Artgalry.ini and merges the files referenced there into the appropriate Artgalry.idx and Artgalry.thm files.

This will create new, clean, ClipArt gallery files, but they will only contain the clipart that shipped with Office or PowerPoint. All other clipart files must be added in according to manufacturer's instructions.
