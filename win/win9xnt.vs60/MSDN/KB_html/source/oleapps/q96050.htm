Graph: Different Versions Ship with Different MS Applications

 Last reviewed: April 2, 1997
Article ID: Q96050
3.00 3.00a 3.00b WINDOWS kbtool

The information in this article applies to:

  • Microsoft Graph versions 3.0, 3.0a, 3.0b
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Windows operating system versions 3.0 and 3.1
  • Microsoft PowerPoint for Windows, version 3.0
  • Microsoft Project for Windows, version 3.0

SUMMARY

Different versions of Microsoft Graph ship with Microsoft Word versions 2.0 and 2.0a for Windows and other products. To determine which version of Graph you have, refer to the following chart: 

   Product and version               Microsoft Graph version included
   -------------------               --------------------------------

   Word 2.0 for Windows              Graph 3.0
   Word 2.0a for Windows             Graph 3.0a
   Word 2.0a-CD for Windows          Graph 3.0b
   Word 2.0b for Windows             Graph 3.0b
   Word 2.0c for Windows             Graph 3.0b
   Project 3.0 for Windows           Graph 3.0a
   PowerPoint 3.0 for Windows        Graph 3.0b
Microsoft Graph versions 3.0 and 3.0a do not support the Microsoft Excel 4.0 graph and spreadsheet file formats.

MORE INFORMATION

When you attempt to import a Microsoft Excel 4.0 spreadsheet into Graph 3.0 or 3.0a an extended ANSI character is inserted in the data sheet. Nothing occurs if you attempt to import a Microsoft Excel 4.0 chart.

Workaround

Do one of the following if you want to use a Microsoft Excel 4.0 chart or worksheet with Microsoft Graph:

  • Upgrade to Word version 2.0b or 2.0c for Windows. To obtain new or updated Microsoft products, call Microsoft Consumer Sales at (800) 426-9400. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

          http://www.microsoft.com/worldwide/default.htm

    -or-

  • Save the file in Microsoft Excel 3.0 format as follows:

    1. In Microsoft Excel version 4.0, open the chart or worksheet.

    2. From the File menu, choose Save As.

    3. In the Save File As Type box, select Excel 3.0.

    4. Choose the OK button.

REFERENCES

"Microsoft Graph User's Guide," version 3.0, pages 25-27
