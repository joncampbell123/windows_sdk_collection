Graph: Some Large Chart Labels Do Not Display

 Last reviewed: September 9, 1996
Article ID: Q97472
The information in this article applies to:
  • Microsoft Graph version 3.0b

SUMMARY

When creating a chart using Graph, you may notice that only every second data label is displayed.

The problem occurs when the size of the font chosen is too large. This generally occurs on the Y axis of the chart, but can also be seen on the X axis.

MORE INFORMATION

To display all labels:

  1. Select the affected axis.

  2. From the Format menu, choose Text.

  3. Choose Font and reduce the size of the selected font.

  4. Choose the OK button and return to the chart.

All labels should now be displayed on the chart.
