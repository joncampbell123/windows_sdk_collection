OFF: Error Message When Working with Large Organization Chart

 Last reviewed: February 3, 1998
Article ID: Q138640

The information in this article applies to:
  • Microsoft Organization Chart for Macintosh, version 2.0
  • Microsoft Organization Chart for Windows 95, version 2.0
  • Microsoft Organization Chart for Windows, version 1.0
  • Microsoft Organization Chart for Macintosh, version 1.0
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows, versions 4.0, 4.0a and 4.0c
  • Microsoft PowerPoint for Macintosh, version 4.0
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office for Windows, versions 4.2 and 4.3
  • Microsoft Office for Macintosh, versions 4.2 and 4.2.1

SYMPTOMS

When you are working with a large organization chart, you may receive one of the following error message: 

  - Your chart would exceed the MS Organization Chart size limit (22x17
    inches).

    -or-
  • This operation would put more than the maximum number of boxes (100) in the same group.

    CAUSE

    These messages are caused by the following two design limitations of Microsoft Organization Chart:

    • The chart cannot be any larger than 22 inches by 17 inches. When the chart exceeds this limit, you receive the first error.
    • A given group or level can have a maximum of 100 boxes. For example, only 100 people can report to a single manager. When this limit is exceeded, you will receive the second error.

    WORKAROUND

    To work around these limits, reduce the font size within the entire Organization Chart. You can also create two smaller Organization Charts side by side on a PowerPoint slide, and draw in the appropriate lines with the PowerPoint drawing tools.
