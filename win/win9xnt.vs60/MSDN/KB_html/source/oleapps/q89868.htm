Graph: How to Change the Default Font

 Last reviewed: September 9, 1996
Article ID: Q89868
The information in this article applies to:
  • Microsoft Graph, versions 3.0, 3.0a, 3.0b
  • Microsoft Windows operating system versions 3.0, 3.1

SUMMARY

To change the default font in Microsoft Graph, change the font in the current datasheet and chart, then save the changes to the Graph default datasheet and chart, DEFAULT.GRA, by choosing Set As Default Chart from the File menu.

Note: You can have different default fonts for different elements in your datasheet or chart. For example, the default font for the legend text can be different from the default font for the label text.

MORE INFORMATION

Use the following instructions to change the default datasheet and chart font:

  1. To launch Graph, choose Object from the Insert menu, select Microsoft Graph under Object Type, and choose the OK button. (You can also choose the Graph button on the toolbar to launch Graph.)

  2. Activate the Datasheet window. From the Edit menu, choose Select All.

  3. From the Format menu, choose Font. Select the desired font and size, then choose the OK button.

  4. Activate the Chart window and select the chart.

  5. From the Format menu, choose Font. (If the chart is not properly selected, the Font command is unavailable.) Select the desired font and size, then choose the OK button.

  6. From the File menu, choose Set As Default Chart. Choose the OK button when Graph prompts you to replace the existing DEFAULT.GRA.
