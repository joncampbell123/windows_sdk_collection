Draw: Arc Tool Creates Filled Pie Wedge Shape Rather Than Arc

 Last reviewed: September 9, 1996
Article ID: Q83206
The information in this article applies to:
  • Microsoft Draw version 1.0

SUMMARY

If you use the Arc tool in Microsoft Draw version 1.0, it may create a pie wedge shape rather than the expected arc. This behavior occurs because the default setting for the Arc tool is Filled. When you choose Filled on the Draw menu, the Arc tool creates a pie wedge shape. If you clear the Filled command on the Draw menu, the Arc tool creates an arc.

REFERENCES

"Microsoft Draw User's Guide," version 1.0, pages 25-27
