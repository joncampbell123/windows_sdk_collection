Err Msg: Graph Is Waiting for the Server to Release Object

 Last reviewed: September 10, 1996
Article ID: Q99160
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, and 2.0c
  • Microsoft Graph, versions 3.0, 3.0a, and 3.0b
  • Microsoft Windows operating system versions 3.0 and 3.1

SYMPTOMS

Microsoft Graph returns the error message "Graph is waiting for the Server to release object" when you update a dynamic data exchange (DDE) link if you do not press ENTER after modifying a cell in Microsoft Word for Windows. This error message persists until you press ENTER in Word for Windows.

WORKAROUND

To work around this problem:

  1. Press CTRL+ESC to open the Windows Task List.

  2. Select Word for Windows from the list of applications, and choose the Switch To button.

  3. In Word for Windows, press the ENTER key.

  4. Switch back to Graph.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, pages 682-690
