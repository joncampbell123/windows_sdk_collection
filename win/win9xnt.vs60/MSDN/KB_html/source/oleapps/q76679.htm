Certain ANSI Characters Are Not Supported by WordArt

 Last reviewed: September 9, 1996
Article ID: Q76679
The information in this article applies to:
  • Microsoft WordArt version 1.0

SUMMARY

The OLE server application WordArt, which is supplied with Microsoft Publisher for Windows, does not support the full range of ANSI characters. Characters that are unsupported will not be visible in the client document once WordArt is closed. This can become a particular problem when trying to implement specialized characters, or characters in foreign languages. For example, in WordArt the ANSI code 176 is unsupported; however, this character is an actual key on French keyboards, and is used in the creation of various foreign words.
