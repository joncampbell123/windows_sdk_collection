Media Clip Objects Not Played in Full Screen

 Last reviewed: January 20, 1997
Article ID: Q135750
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Media Player for Windows 95

SYMPTOMS

When you format a Media Clip object to be played in full screen, it will be played in a window if you insert it into a PowerPoint presentation, a Word document, or any other OLE container.

CAUSE

This problem is caused by an OLE limitation in the Windows 95 Media Player; this problem occurs only when you use Media Clips in client files.

RESOLUTION

To play your Media Clip in full screen, resize the Media Clip by selecting it and dragging the resize handles.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
