PPT: Cannot Import SYLK File into Graph

 Last reviewed: February 3, 1998
Article ID: Q171053
The information in this article applies to:
  • Microsoft Graph 98 for Macintosh
  • Microsoft Graph 97 for Windows
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Office 97 for Windows

SYMPTOMS

When you attempt to insert a Symbolic Link (SYLK) file into Microsoft Graph 97 or Graph 98, the process fails without any error message.

CAUSE

Microsoft Graph supports text files, Microsoft Excel worksheet files, and Lotus 1-2-3 spreadsheet files. Microsoft Graph cannot import SYLK formatted files.

WORKAROUND

Open the SYLK file in Microsoft Excel and save the file in the Excel Workbook (.xls) format. You can now insert the file into Microsoft Graph.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.

MORE INFORMATION

Microsoft Graph Help incorrectly states that you can import data from a .slk file into Graph.
