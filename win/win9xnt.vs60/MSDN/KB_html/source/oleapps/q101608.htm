OFF: Graph: Plotting Discontiguous Line and Scatter Graphs

 Last reviewed: February 3, 1998
Article ID: Q101608
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Graph 97 for Windows
  • Microsoft Graph for Windows 95, version 5.0
  • Microsoft Graph for the Macintosh, versions 3.0 and 5.0
  • Microsoft Graph for the Macintosh, version 3.0
  • Microsoft Graph for Windows, version 3.0

SYMPTOMS

When you create an xy (scatter) graph in Microsoft Graph, if your data is incomplete, the data points on your graph do not connect continuously.

RESOLUTION

To connect the data points, enter the following into the data sheet where data is missing: 

   #N/A
This forces the scatter points to connect even when there is no data available.

NOTE: In Microsoft Graph 5 and higher, you can leave the space blank (in the datasheet) and turn on empty cells plotted as interpolated using the following steps:

  1. On the Tools menu, click Preferences (Options in the Windows version).

  2. Click the Chart tab.

  3. In the Empty cells plotted as section, click the interpolated option.

Now the lines should connect.
