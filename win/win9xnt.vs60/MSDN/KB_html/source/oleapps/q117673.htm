OFF: Empty WordArt Frame with Insufficient Memory Allocated

 Last reviewed: February 3, 1998
Article ID: Q117673
The information in this article applies to:
  • Microsoft WordArt version 1.0
  • Microsoft Works for the Macintosh, version 4.0

SYMPTOMS

When you create or edit a WordArt object, you receive the following error message: 

   WordArt does not have enough memory to transfer the picture of
   the text you typed. Try typing less text, or close the WordArt
   application and use the Finder's Get Info menu command to
   increase the size of WordArt's memory partition.

CAUSE

If you enter more characters or add more formatting than WordArt can handle with the amount of memory you have allocated to it, this message appears. The resulting WordArt object will be blank.

RESOLUTION

Although the WordArt object appears to be empty, the information you entered is actually still contained in it. Use the Finder's Get Info menu command to allocate more memory to WordArt, then activate and close the WordArt object. When you close WordArt, the object should display correctly in Works.
