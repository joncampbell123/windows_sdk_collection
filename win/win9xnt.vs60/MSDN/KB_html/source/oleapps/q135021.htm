Earlier PowerPoint Objects Available After Office 7.0 Install

 Last reviewed: January 3, 1997
Article ID: Q135021
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows, versions 4.0, 5.0, 5.0c
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you install the Microsoft Office version 7.0 upgrade on a computer that contains an earlier version of Office, the following object types may still be available in the Object dialog box in Microsoft Excel versions 4.0 and 5.0 and Microsoft Word version 6.0: 

   MS PowerPoint 3.0 Presentation
   MS PowerPoint 3.0 Slide
   MS PowerPoint 4.0 Presentation
   MS PowerPoint 4.0 Slide

CAUSE

This behavior occurs when you install the Microsoft Office upgrade without first removing the earlier version of Office. This problem occurs whether you install Office version 7.0 to the folder that contains the earlier version of Office or to a different folder. Note that the object types shown above appear only in earlier versions of the Office applications, not the version 7.0 Office applications. The Microsoft PowerPoint version 7.0 objects (Microsoft PowerPoint Presentation and Microsoft PowerPoint Slide) appear correctly in the Object dialog box in all versions of the applications.

For example, if you install Microsoft Office version 7.0 to a different folder than the Office version 4.0 folder, when you run an Office version 7.0 application, the MS PowerPoint 3.0 Presentation and MS PowerPoint 3.0 Slide object types are no longer available. However, when you run Microsoft Excel version 5.0 or Microsoft Word version 6.0, the MS PowerPoint 3.0 Presentation and MS PowerPoint 3.0 Slide object types are available.

Note however that when you choose to insert one of the Microsoft PowerPoint 3.0 objects, a Microsoft PowerPoint 7.0 object is inserted instead.

This behavior does not affect the functionality of inserting a Microsoft PowerPoint object. When you install the Office 7.0 upgrade, the 7.0 version of the Office application becomes the OLE server and editor for that application. Even if an earlier version of the application is installed on your computer, all of the OLE objects for that application will be created and edited using the current version.
