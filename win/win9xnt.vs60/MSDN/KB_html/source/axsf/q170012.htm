ASP Error 0115 When Connecting to Sybase/Oracle Database

 Last reviewed: March 12, 1998
Article ID: Q170012

The information in this article applies to:

  • Microsoft Active Server Pages, versions 1.0 and 1.0b
  • Microsoft Internet Information Server version 3.0

SYMPTOMS

Active Server Pages (ASP) and ActiveX Data Objects (ADO), Internet Information and Microsoft Windows NT Server version 4.0, connections to an ORACLE/Sybase database on an UNIX system may fail with an ASP Error 0115.

CAUSE

The problem occurs when you use Intersolv/Sybase ODBC components along with Open Client from Sybase.

NOTE: The above error message may not occur as long as there is only one connection. When a second connection is established the following error message occurs: 

   ASP error 0115 Unexpected Error...<filename>.asp a trapable error
   occured in an external object the script cannot continue running.

RESOLUTION

Please contact Intersolv or Sybase for a fix for this problem.
