PRB: ASP Returns "No Current Record" Error on Conn.Open Line

 Last reviewed: December 11, 1997
Article ID: Q174639
The information in this article applies to:
  • Microsoft Active Server Pages, versions 1.0, 1.0b

SYMPTOMS

The following error occurs when attempting to open an ADODB.Connection in an .asp file: 

   ADODB.Connection error '800a0bcd'

   No current record.

CAUSE

There are two possible causes to this error:

  • There's a bad install of Active Server Pages (ASP)
  • You are using version 1.0.994 of ADO on a non-US install of Windows NT or Windows 95

RESOLUTION

If you have not installed a non-US version of Windows NT or Windows 95, reinstall Active Server Pages. If you have installed a non-US version of Windows NT or Windows 95, download the latest version of ADO from http://www.microsoft.com/data/.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
