TN3270 Admin Command Line Options

 Last reviewed: July 23, 1996
Article ID: Q149653
The information in this article applies to:
  • Microsoft SNA Server for Windows NT, versions 2.11 and 2.11 Service Pack 1

SUMMARY

TN3270 Server Admin supports the following two undocumented command line switches: 

   /c

   /no3d.

MORE INFORMATION

   /c - This will invoke the Config option when TN 3270 Admin is started.
/no3d - This causes the TN 3270 Admin to not use visual 3D effects 
        supported by the Windows NT Ctl3d32.dll module.
