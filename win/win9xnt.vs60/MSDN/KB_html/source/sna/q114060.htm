SUMMARY

The following is the readme file that accompanies SNA Server 2.0 patch 30 (sna20.p30):

Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #30 Status: Confirmed

PKZIP 2.04g format -- uncompress using the "-d" option.

Date: March 5, 1994

Modules updated: 

\SYSTEM\SNADLC.DLL   02/22/94  06:54p  74240
\SYSTEM\HWSETUP\SNADLC\SNADLC.DLL (same file)

Bugs fixed:

1985 SNA Server DLC timers not optimally configured.

Ensure that t1, t2, and ti timers configured for an SNA Server 802.2 connection defaults are configured optimally within the configuration file.

Update Procedure:

- Stop the SNA Server service

- Copy the new SNADLC.DLL to the above locations on the server.

*********************** End of Readme *******************************

SNA Server 2.0 patches are located on:

  1. Library #2 of the MSSNA Compuserve forum.

  2. The Internet server under BUSSYS\SNASRVR\SUP-ED\FIXES\ (supported patches) OR on BUSSYS\SNASRVR\UNSUP-ED\FIXES\ (unsupported patches).
