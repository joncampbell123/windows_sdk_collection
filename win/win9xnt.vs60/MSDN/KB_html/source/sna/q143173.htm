How to Obtain SNA Server 2.11 Service Pack 1

 Last reviewed: June 6, 1996
Article ID: Q143173
The information in this article applies to:
  • Microsoft SNA Server for Windows NT, version 2.11 Service Pack 1 

 ** THIS U.S. SERVICE PACK 1 IS FOR SNA SERVER VERSION 2.11 ONLY!! **
SNA Server 2.11 U.S. Service Pack 1 is a cumulative build of all fixes to SNA Server version 2.11.

To find out more information on bug fixes and enhancements which are included in this update, see the SNASP1.HLP file that accompanies the Service Pack.

AVAILABLE FORMATS FOR SNA SERVER 2.11 SERVICE PACK 1

SNA Server Service Pack 1 is available in self-extracting archive files for Intel (x86), MIPS, ALPHA, and Power PC-based computers, in addition to separate self-extracting files for SNA Server client software for Windows NT, Windows 95 and Windows 3.x/MS-DOS/OS2 computers.

The service pack is not available in 3.5-inch disk format.

INSTRUCTIONS FOR DOWNLOADING

Not all formats are available on all electronic forums. Because of their size, large archive files for the x86, MIPS, ALPHA, and Power PC versions of Service Pack 1 are only available on the Internet.

Service Pack 1 is available on the following electronic service: 

   Internet: ftp.microsoft.com (198.105.232.1)
      /bussys/winnt/sna-public/fixes/sna211/ussp1             (all formats)
      /winnt/i386/server/211sp1is.exe:  Server update         (Intel)
      /winnt/i386/client/211sp1ic.exe:  Windows NT client     (Intel)
      /clients/211sp1cl.exe:  Windows 3.x, MS-DOS, OS2 client (Intel)
      /win95/snawin95.exe:  Windows 95 client                 (Intel)

      NOTE: /winnt/mips, /winnt/alpha, and /winnt/ppc updates are also
      available.

ORDERING SNA SERVER 2.11 SERVICE PACK 1

If you do not have access to the electronic services listed above, you can order Service Pack 1 from the Microsoft Sales Information Center (MSIC) at 1-800-426-9400. The part number for the SNA Server 2.11 Service Pack 1 (U.S.) CD-ROM is 211-075-070. The cost for Service Pack 1 is $14.95.

INSTALLING SNA SERVER 2.11 SERVICE PACK 1

ALPHA, MIPS, Power PC, or x86 Network Upgrade:

Copy the self-extracting file to a directory on your hard drive and type the name of the self-extracting file with a "-d" switch after it. For example, to decompress the file, 211SP1IS.EXE, enter the following command: 

   211SP1IS -D
After the service pack has been decompressed, run the UPDATE.EXE program to install the updated files.

SNA Server symbol files are included in the self-extracting server updates for X86, MIPS, Alpha and Power PC. The symbol files are extracted to the \SYSTEM\SYMBOLS directory. The UPDATE program does not apply these symbols during service pack installation, though they can be applied manually by entering the following command: 

  XCOPY <extract-dir>\SYMBOLS\*.* <ntroot>\SYMBOLS /S /E /V
where <extract-dir> is the directory where the self-extracting file was extracted, and <ntroot> is the Windows NT root directory of the target machine (where the default is \WINNT35).
