Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #44 (includes patch #30) Status: Confirmed

PKZIP 2.04g format -- uncompress using the "-d" option.

Date: May 9, 1994

Modules updated: 

\SYSTEM\SNADLC.DLL   05/08/94  09:36p  74752
\SYSTEM\HWSETUP\SNADLC\SNADLC.DLL (same file)

Bugs fixed:

2058 SNALINK application exception in 802.2 link service.

Under stress, the SNA Server 802.2 link service may fail with an application exception. Looking at the drwtsn32.log, the failure occurred in routine "sbpa4s", caused by an uninitialized variable

1985 SNA Server DLC timers not optimally configured (Patch #30)

Ensure that t1, t2, and ti timers configured for an SNA Server 802.2 connection defaults are configured optimally within the configuration file.

Update Procedure:

- Stop the SNA Server service

- Copy the new SNADLC.DLL to the above locations on the server.
