3270 Single Sign-on Keywords MS$SAMEU & MS$SAMEP Not Documented

 Last reviewed: February 17, 1998
Article ID: Q180950
The information in this article applies to:
  • Microsoft SNA Server, versions 3.0, 3.0 Service Pack 1 (SP1), and 3.0 SP2

SYMPTOMS

The SNA Server 3.0 Planning Guide section "Configuring 3270 Single Sign-on" describes how the client logon script needs to work for 3270 Single Sign- on, but it does not mention the required keywords: MS$SAMEU for the user name and MS$SAMEP for the password.

STATUS

Microsoft has confirmed this to be a problem in SNA Server versions 3.0, 3.0 SP1, and 3.0 SP2. The documentation for the next update to SNA Server 3.0 will include this information in the SNA Server Administrators Guide. SNA Server 4.0 already includes this information.
