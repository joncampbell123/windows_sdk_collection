 \CLIENTS\OS2\SETUP.EXE  03/22/94  01:43p 142289
Bugs fixed:

2098 OS/2 client setup should not install 3270 icon

When installing the SNA Server OS/2 client software, the setup program incorrectly creates an icon for 3270 Emulation, though no OS/2 3270 emulator is included with SNA Server. While the OS/2 setup program does install all the SNA Server API support onto the OS/2 machine, the only application which should appear in the SNA Server program group is the setup program.

Update Procedure:

- Copy the updated setup program to the <snaroot>\CLIENTS\OS2 

  directory the SNA Server installation server, or to the
  OS/2 client floppy disk.
- Install the SNA Server OS/2 client software from the server 
  or from floppy disk.
