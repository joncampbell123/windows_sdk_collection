Maximum of Two DCA ISCA SDLC Adapters Allowed Per SNA Server

 Last reviewed: April 17, 1997
Article ID: Q151393

The information in this article applies to:
  • Microsoft SNA Server for Windows NT, versions 2.0, 2.1, 2.11, 2.11 Service Pack 1, and 3.0

SYMPTOMS

When you install a third instance of the DCA SDLC Link Service on a SNA Server, you will get the following message: 

   Setup Message

   The maximum of 2 DCA SDLC/X.25 boards are already configured.

CAUSE

The DCA SDLC Link Service for SNA Server has been written to allow a maximum of two DCA ISCA SDLC adapters per SNA Server.
