Sync Point Applications Now Supported on Windows NT & Windows 95

 Last reviewed: April 14, 1997
Article ID: Q165165
The information in this article applies to:
  • Microsoft SNA Server, version 3.0

SYMPTOMS

SNA Server does not have Sync Point Support for Windows NT or Windows 95.

RESOLUTION

To resolve this problem, obtain the hotfix mentioned below. With the hotfix, SNA clients running Windows NT or Windows 95 can now run Sync Point APPC applications.

NOTE: For more information on the Sync Point extension, see the SNA Server SDK documentation icon in SNA Server 3.0.

STATUS

Microsoft has confirmed this to be a problem in SNA Server version 3.0. This problem was corrected in the latest Microsoft SNA Server 3.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
