Bugs fixed:

2193 ASCII to EBCDIC conversion problem with "_" character

While IBM PC Support (and the SNA Server MS-DOS router) support the "_" character in an AS/400 username, the SNA Server Win16 and Win32 CSV convert verb would reject this character when converting using the AE conversion table. So, with this update applied, the 5250 applets will now support an AS/400 username which includes an underscore character.

1736 Win32 Allocate hangs due to CSV, Management verb conflict (Patch 9)

Depending on the order in which an application issues an APPC management verb and a common service verb, a subsequent APPC Allocate call may never return.

Update Procedure:

- Apply the appropriate DLL for the platform being used: 

   \SYSTEM\WINCSV.DLL  - Windows NT
   \CLIENTS\WIN3X\WINCSV.DLL - Windows 3.x (copy to the \windows\system
                         directory on the target client machine)
   \CLIENTS\ACSSVC.DLL - OS/2
