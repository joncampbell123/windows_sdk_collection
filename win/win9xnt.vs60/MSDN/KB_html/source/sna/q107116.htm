5250 Applet Keyboard Functions

 Last reviewed: April 18, 1997
Article ID: Q107116

The information in this article applies to:
  • Microsoft SNA Server for Windows NT, version 2.0

SUMMARY

The following is a list of 5250 keyboard functions of the 5250 Applet shipped with SNA Server: 

  NOTE: The 5250 Applet does not support keyboard remapping. This is an
  advanced feature supported by third-party vendors.
