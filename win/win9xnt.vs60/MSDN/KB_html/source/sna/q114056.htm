Bugs fixed:

1984 Application exception in RUI

Fix to protect against referencing the first LU within the RUI control block if there aren't any LU's (previously, this would leading to a gp-fault or application exception).

Update Procedure:

- For Windows NT, Windows 3.x, or OS/2: Copy the new LUA RUI DLL over the 

  previous version and restart the RUI application.
- For MS-DOS: Relink your MS-DOS RUI application with the new DOSACS.LIB.
