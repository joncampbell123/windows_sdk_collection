SNA Server NetWare Client Setup

 Last reviewed: April 17, 1997
Article ID: Q112635
The information in this article applies to:
  • Microsoft SNA Server for Windows NT, version 2.0

You can install SNA Server client files on a Novell NetWare workstation by doing one of the following:

  • Install from the SNA Server client disks.

    -or-

  • Install from a NetWare file server.

To set up the SNA Server client files on a Novell NetWare file server:

  1. Map a Novell client workstation drive on the NetWare server.

  2. Tree copy (XCOPY A:\*.* <net drive>:*.* /S) all the SNA Server client files from the SNA Server client disks to the mapped drive on the NetWare server.

Use this mapped drive for any further NetWare client installations.
