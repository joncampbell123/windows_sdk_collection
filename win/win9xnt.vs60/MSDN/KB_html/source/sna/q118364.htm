******************* Beginning of Readme *************************** Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #51 Status: Confirmed

Date: July 14, 1994

Modules updated: 

     \SYSTEM\IBMDFT.DLL           07/14/94  01:46p 70,144 (2.0.0.148)
     \SYSTEM\HWSETUP\IBMDFT\IBMDFT.DLL  (same file)
Bugs fixed:

2255 XPROG 752 using DFT link service when accessing CICSVS

This occurs with the DCA IRMA driver or IBM DFT driver, which both use the IBMDFT.DLL link service. Under certain circumstances, the number of data elements was being miscalculated by one byte, causing the driver/link to return extra (bogus) data to flow to the server and 3270 emulator, leading to an XPROG 752 error in the emulator.

Update Procedure:

- Stop the SNA Server Base (SNABASE) service on the server

- Copy the updated IBMDFT.DLL over the existing version in the server's <snaroot>\SYSTEM directory.

- Restart SNA Server

*********************** End of Readme *******************************
