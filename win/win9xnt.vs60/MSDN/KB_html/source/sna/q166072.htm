Unable to Install the Barr X.25 Link Service

 Last reviewed: April 14, 1997
Article ID: Q166072
The information in this article applies to:
  • Microsoft SNA Server version 3.0

SYMPTOMS

When attempting to install the Barr X.25 link service by using External Link Services on the Tools menu, you receive the following error message: 

   An error has occurred.
   Unable to open the file:
   <snaroot>\system\hwsetup\ibmx25.srl

CAUSE

The Ibmx25.srl file was inadvertently omitted from this directory.

RESOLUTION

A fix is available to correct this problem. To resolve this problem, obtain the hotfix mentioned below.

STATUS

Microsoft has confirmed this to be a problem in SNA Server version 3.0. This problem was corrected in the latest Microsoft SNA Server 3.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
