Microgate SDLC Link Service Must be Installed as External

 Last reviewed: December 3, 1997
Article ID: Q170671
The information in this article applies to:
  • Microsoft SNA Server, versions 3.0 and 3.0 SP1

SYMPTOMS

The Microgate SDLC link service cannot be installed on an Alpha computer with SNA Server version 3.0 and 3.0 SP1, except as an external link service. When installing the Microgate SDLC link service as internal, the Card Configuration tab and Dialer Settings tab do not contain information. When installing the link service as external, all of the information is entered in the correct fields.

WORKAROUND

Install the link service as an external link service.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SNA Server, versions 3.0 and 3.0 SP1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
