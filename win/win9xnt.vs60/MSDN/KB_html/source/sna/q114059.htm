Bugs fixed:

1987 IBM PC's shared folders hang when accessing through Windows

When running the SNA Server MS-DOS router over IPX, with IBM PC Support shared folders, a machine hang can occur when accessing an AS/400 drive from Windows file manager.

Update Procedure:

- Copy NWBASE.EXE over the previous version under the SNA Server MS-DOS 

  client directory, and re-load SNABASE.
