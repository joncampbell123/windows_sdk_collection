SNA: TN3270 Refresh Cycle Time Description

 Last reviewed: April 17, 1997
Article ID: Q137934

The information in this article applies to:
  • Microsoft SNA Server versions 2.11 and 3.0

The "Refresh Cycle Time" option under TN3270 Admin's Option Menu is used to modify the refresh rate of the information displayed in the Session Status dialog box. The valid range is 0-60 seconds; 0 means no refresh. If the refresh option is not set, you have to manually refresh the Session Status dialog box.
