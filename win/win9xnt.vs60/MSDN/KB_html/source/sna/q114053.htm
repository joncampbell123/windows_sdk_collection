Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #23 Status: Confirmed

PKZIP 2.04g format -- uncompress using the "-d" option.

Date: March 5, 1994

Modules updated: 

  All Windows 5250 applet files for the SNA Server Windows NT (32-bit)
  client.  (see below for a complete file list).
Bugs fixed:

1818 Windows 5250 applet problems with international keyboards

Keyboard translation problems with the 5250 applet have been observed on both German and Swiss keyboards.

Update Procedure:

- Expand the pkzip file from it's own directory, using the "-d" switch. 

  For example:

     md C:\patch23
     copy sna20.p23 C:\patch23
     cd C:\patch23
     PKUNZIP -d SNA20.P23

  This will create a subdirectory tree of SYSTEM, and expand all of the
  Windows 5250 components into the correct directory structure.
- XCOPY the above directory over your existing SNA Server Windows NT client 
  directory and server's \CLIENTS\WINNT directory, using the /S /E /V
  switches. For example:

     XCOPY c:\patch23\system\*.*  C:\SNA\SYSTEM /s /e /v

     XCOPY c:\patch23\system\*.* C:\SNA\CLIENTS\WINNT\SYSTEM\*.* /s /e /v
