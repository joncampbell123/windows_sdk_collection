GPF or Poor Performance with Interleaf Running in Windows 95

 Last reviewed: August 21, 1995
Article ID: Q132849
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you run any version of Interleaf prior to version 6.1.1, you may receive a general protection fault, or the program may perform poorly.

CAUSE

Interleaf version 6.1.0 is a beta version and does not run in Windows 95.

RESOLUTION

Contact Interleaf and request Interleaf version 6.1.1.
