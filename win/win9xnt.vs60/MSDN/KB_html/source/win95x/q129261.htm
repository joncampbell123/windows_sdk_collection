Cannot Install Windows 95 on Acer Portable Computer

 Last reviewed: September 13, 1995
Article ID: Q129261
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you run Windows 95 Setup on an Acer portable computer, Setup may not finish successfully.

CAUSE

The default advanced power management (APM) settings on Acer portable computers can cause interruptions in the Setup process.

RESOLUTION

To install Windows 95 successfully, you must first disable the APM features in the Acer computer's CMOS memory. For information about how to modify CMOS settings on your Acer computer, please consult the computer's documentation or manufacturer.
