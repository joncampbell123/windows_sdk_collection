Cannot Select 32-Bit Program Area with Intel ProShare 1.8

 Last reviewed: September 7, 1995
Article ID: Q129763
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you use the Screen Snapshot feature of Intel ProShare version 1.8 to select window areas, you cannot select 32-bit program areas.

CAUSE

You cannot directly select 32-bit program areas in ProShare 1.8.

RESOLUTION

To work around this behavior, start the selection process in a 16-bit client area and then drag it to a 32-bit area. For information about how to do this, please contact Intel.

MORE INFORMATION

Intel ProShare uses a digital camera board and an audio/ISDN board to provide video conferencing. You can select a section of the screen for a graphical snapshot.
