Updated Modem .inf Files for VoiceView-Compatible Modems

 Last reviewed: May 1, 1997
Article ID: Q142541
95 WINDOWS kbtool kbfile

The information in this article applies to:

  • Microsoft Windows 95

SUMMARY

Microsoft has released updated Windows 95 modem .inf files for VoiceView- compatible modems.

MORE INFORMATION

Installation

To install the updated modem .inf files, follow these steps:

  1. Download the INFCOPY.EXE file from one of the online services listed below to an empty folder.

  2. In My Computer or Windows Explorer, double-click the INFCOPY.EXE file.

  3. Follow the instructions on the screen.

  4. Read the Readme.txt file that is displayed on the screen.

    NOTE: The Windows 95 updated modem .inf files Readme.txt file is also available in the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q142278

          TITLE     : Windows 95 Updated Modem .inf Files Readme.txt File
