Garbled Display on Packard Bell Statesman 486/50 Computer

 Last reviewed: August 30, 1995
Article ID: Q132768
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

On a Packard Bell Statesman computer, the display may be garbled. The desktop may run off the right side of the screen, the colors may be wrong, and the text may be unreadable.

CAUSE

This problem can occur if you are using 256-color mode. The display adapter in this computer does not work in 256-color mode.

RESOLUTION

Change to 16-color mode using 640 x 480 resolution. 256-color mode is available, but you should not use it.
