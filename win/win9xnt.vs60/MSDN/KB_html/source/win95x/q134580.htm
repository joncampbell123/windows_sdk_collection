Cannot Install Fastback 6.0 with Install Program Wizard

 Last reviewed: August 23, 1995
Article ID: Q134580
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to install Fastback version 6.0 with the Install Program Wizard, you receive the following error message: 

   To install Fastback Plus, please type
   A:INSTALL if you have inserted the installation disk in drive A:, or
   B:INSTALL if you have inserted the installation disk in drive B:
When you do so, the Fastback Setup program cannot find its setup files and cannot continue.

CAUSE

When the Fastback Setup program runs from the Install Program Wizard, it receives an invalid drive letter option from the wizard.

RESOLUTION

Run the Setup program in MS-DOS mode. To do so, follow these steps:

  1. Click the Start button, click Shut Down, click the "Restart the computer in MS-DOS mode" option button, and then click Yes.

  2. Run the Setup program from the MS-DOS prompt.
