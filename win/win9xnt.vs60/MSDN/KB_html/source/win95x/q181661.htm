Files Included with the USB Supplement in OSR2.1 and OSR2.5

 Last reviewed: February 24, 1998
Article ID: Q181661
The information in this article applies to:
  • Microsoft Windows 95 OEM Service Release versions 2.1, 2.5

SUMMARY

This article lists the files included with the Universal Serial Bus (USB) Supplement on the Windows 95 OEM Service Release 2.1 and 2.5 CD-ROM.
