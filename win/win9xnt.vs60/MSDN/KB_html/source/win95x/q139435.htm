WMConfig Tool in MS Plus! Refers to Hover! in About Box

 Last reviewed: August 30, 1996
Article ID: Q139435
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! for Windows 95

SYMPTOMS

The About WMConfig dialog box in the WMConfig tool for 3D Pinball refers to: 

   Hover! Sound mixing configuration utility. Microsoft Hover!, Copyright
   1995.

MORE INFORMATION

The WMConfig tools included with Microsoft Plus! and the CD-ROM version of Windows 95 are identical. The WMConfig tool is used to troubleshoot problems with sound effects in 3D Pinball and Hover!.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Plus! for Windows 95, version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
