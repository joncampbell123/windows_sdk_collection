Cannot Print from Lotus Agenda in Windows 95

 Last reviewed: October 10, 1995
Article ID: Q137981
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to print a document from Lotus Agenda, the document may not be printed. If you print to a printer, no output is received. If you print to a file, a 0-byte file is generated.

CAUSE

Lotus Agenda is not completely compatible with Windows 95.

RESOLUTION

You may be able to resolve this problem by running Lotus Agenda in MS-DOS mode. For information about configuring programs to run in MS-DOS mode, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q134400
   TITLE     : General Tips for Using MS-DOS Mode
If running Lotus Agenda in MS-DOS mode does not correct the problem, you should contact the program's manufacturer for additional information. An updated version of Lotus Agenda that is completely compatible with Windows 95 is not available.
