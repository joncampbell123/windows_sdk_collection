Resource Kit Tools Included on the Windows 95 CD-ROM

 Last reviewed: April 24, 1996
Article ID: Q137725
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

Some of the tools included with the "Microsoft Windows 95 Resource Kit" are also included on the Windows 95 CD-ROM. These include tools for learning about Windows 95, upgrading your site to Windows 95, computer maintenance and troubleshooting, and customizing computers. This article lists the tools that are included on the Windows 95 CD-ROM, and describes how to use them.

MORE INFORMATION

Tools for Learning About Windows 95

You can run these tools directly from the CD-ROM. If you want to run these tools from your hard disk, copy the appropriate .hlp and .cnt files to a folder on your hard disk. To run a tool, double-click its .hlp file, or create a shortcut to the .hlp file.

These tools are located in the Admin\Reskit\Helpfile folder on the Windows 95 CD-ROM:

  • Windows 95 Resource Kit online (Win95rk.hlp, Win95rk.cnt), for finding information quickly.
  • Administrator's online tour (Tour4a.hlp, Tour4adm.cnt), for viewing the Windows 95 network and system management features described in the Resource Kit.
  • Information for Macintosh users (Macusers.hlp, Macusers.cnt), about the Windows 95 interface.

Tools for Upgrading Your Site to Windows 95

  • Batch Setup tool (Batch.exe) for creating Setup script batch files for installing Windows 95 on multiple computers. For more information about the Batch Setup tool (including instructions for installing and running the tool), click the Help menu in Batch Setup, or see the Batch.txt file or the related topic in Win95rk.hlp.

    This tool is located in the Admin\Nettools\Netsetup folder.

  • INF Installer tool (Infinst.exe) for adding software to the Windows 95 source files stored on a network server. For more information about the INF Installer tool (including instructions for installing and running the tool), see the Infinst.txt file or the related topic in Win95rk.hlp.

    This tool is located in the Admin\Nettools\Netsetup folder.

  • Sample Setup scripts for creating custom Setup scripts for Windows 95. For more information about Windows 95 Setup scripts, see the Readme.txt file in the Admin\Reskit\Samples\Scripts folder or the related topics in Win95rk.hlp.

    The following tools are located in the Admin\Reskit\Samples\Scripts folder: 

        - Automate.inf defines everything required for automatic setup except
      the user's name. It also installs Windows 95 Accessibility features,
      and can be used to customize other aspects of Windows 95 Setup.


        - Default.inf allows user input during Windows 95 Setup, but proposes
      a series of alternate defaults.


        - Iniprep.inf describes methods of removing and replacing settings in
      .inf files.


        - Minbatch.inf contains the minimum script required for automatic
      installation when upgrading from earlier versions of Windows.


        - Msnet.inf contains a script for installing and configuring
      components for Microsoft networking.


        - Nwnet.inf contains a script for installing and configuring the
      Microsoft Client for NetWare Networks and related networking
      components.


        - Regional.inf contains a script for configuring the locale, selected
      keyboard preferences, and multiple-language preferences.


        - Userprof.inf contains a script for installing and configuring user
      profiles, remote administration of the registry, and group policies.

Tools for Computer Maintenance and Troubleshooting

  • Configuration Backup (Cfgback.exe and Cfgback.hlp) for backing up the Windows 95 registry. To run this tool, copy the Cfgback.exe and Cfgback.hlp files to a folder on your hard disk. Double-click the Cfgback.exe file in Windows Explorer or My Computer, or create a shortcut to the Cfgback.exe file, and then double-click the shortcut. For more information about backing up the registry, see the related topic in Win95rk.hlp.

    This tool is located in the Other\Misc\Cfgback folder.

  • Extended Print Troubleshooter (Epts.exe) for fixing printer problems. For more information about the Extended Print Troubleshooter (including instructions for installing and running the tool), see the Readme.txt file in the Other\Misc\Epts folder.

    This tool is located in the Other\Misc\Epts folder.

  • Log Viewer (Logview.exe) for viewing Windows 95 log files, including Bootlog.txt, Detlog.txt, Netlog.txt, Setuplog.txt, Scandisk.log, and Ndislog.txt. This tool can be run directly from the CD-ROM. If you want to run this tool from your hard disk, copy the Logview.exe file to a folder on your hard disk. To run the tool, double-click the Logview.exe file in Windows Explorer or My Computer, or create a shortcut to the Logview.exe file, and then double-click the shortcut. For more information about the Windows 95 log files, see the related topic in Win95rk.hlp.

    This tool is located in the Other\Misc\Logview folder.

  • Shortcut tool (Shortcut.exe) for keeping track of Windows 95 shortcuts. For more information about the Shortcut tool (including instructions for installing and running the tool), see the Readme.txt file in the Admin\Apptools\Envvars folder.

    This tool is located in the Admin\Apptools\Envvars folder.

  • Sample policies for Windows 95 networks, for creating and customizing policy files for managing Windows 95 on client computers. For more information about system policies, see the Policies.txt file in the Admin\Reskit\Samples\Policies folder or the related topic in Win95rk.hlp.

    The following tools are located in the Admin\Reskit\Samples\Policies folder. 

        - Maximum.pol contains suggested policies for maximum network and
      desktop security.


        - Standard.pol contains suggested policies for moderate network and
      desktop security.

For Customizing Computers

  • Change Code Page tool (Changecp.exe) for changing the code page used by MS-DOS-based programs. For more information about the Change Code Page tool (including instructions for installing and running the tool), see the Changecp.txt file in the Other\Changep folder or the related topic in Win95rk.hlp.

    This tool is located in the Other\Changecp folder.

  • Minitel TTF files (Arialals.ttf and Arialalt.ttf) for installing fonts needed to emulate Minitel with HyperTerminal. Arial Alternative is the only font that contains the required graphic characters. To install the Arial Alternative font files, use the Fonts tool in Control Panel.

    This tool is located in the Other\Minitel folder.

CD-ROM Extras for Microsoft Windows 95 Upgrade

The CD-ROM Extras for Microsoft Windows 95 Upgrade includes files that are included in the CD-ROM version of Windows 95, but not in the floppy disk version. Some of the tools discussed in this article are included in the CD-ROM Extras for Microsoft Windows 95 Upgrade, and some are not. For more information about the CD-ROM Extras for Microsoft Windows 95 Upgrade, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q135315
   TITLE:      CD-ROM Extras for Microsoft Windows 95 Upgrade

Other Resource Kit Tools

The "Microsoft Windows 95 Resource Kit" includes additional tools that are not included on the Windows 95 CD-ROM. These tools include:

  • A description of Resource Kit tools (Overview.hlp, Overview.cnt).
  • Microsoft Project files for planning Windows 95 deployment. You can use this software to access a sample schedule file (Deploy.mpt) that summarizes the steps described in the Resource Kit Deployment Planning Guide.
  • Adapter Card Setup Help (Adapter.hlp) for configuring network, SCSI, and sound adapters.
  • Additional sample policies for Windows 95 networks, for creating and customizing policy files for managing Windows 95 on client computers. The Sample.adm file shows definitions of all custom controls that can be defined in .adm files.
  • Quick Resolution Changer (Quickres.exe) for changing the visible screen area without restarting Windows 95.
