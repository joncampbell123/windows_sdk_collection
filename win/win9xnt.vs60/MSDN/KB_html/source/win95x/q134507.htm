Cannot Add "NetBIOS Support for IPX/SPX-Compatible Protocol"

 Last reviewed: August 30, 1996
Article ID: Q134507
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

Although you can see the "NetBIOS support for IPX/SPX-compatible Protocol" component in Network properties, you cannot add it using the Add button.

CAUSE

If you select the "I want to enable NetBIOS over IPX/SPX" check box on the NetBIOS tab in IPX/SPX-Compatible Protocol properties, you see an entry for the "NetBIOS support for IPX/SPX-compatible Protocol" component in Network properties.

This component should not be displayed. You can install this component only by selecting the "I want to enable NetBIOS over IPX/SPX" check box. You cannot add this component using the Add button.

RESOLUTION

Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it become available.
