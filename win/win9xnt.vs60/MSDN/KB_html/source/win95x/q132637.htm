MORE Program Does Not Support Long Filenames

 Last reviewed: August 21, 1995
Article ID: Q132637
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you use the MORE program with a document that has a long filename, you may receive the following error message: 

   Invalid file name in command line

CAUSE

The MORE program does not support long filenames.

RESOLUTION

When you use the MORE program, either specify the short filename equivalent or use a redirection symbol to display the file. For example, at the MS-DOS command prompt, type "more < long filename" (without the quotation marks).
