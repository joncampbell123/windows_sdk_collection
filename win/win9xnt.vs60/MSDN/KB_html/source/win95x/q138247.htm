List of Documentation Errors in Windows 95 Readme.txt File

 Last reviewed: October 16, 1995
Article ID: Q138247
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

This article lists known documentation errors in the Microsoft Windows 95 Readme.txt file.

The following known documentation errors are described in this article:

  • If you purchased Windows 95 on floppy disks

MORE INFORMATION

If You Purchased Windows 95 on Floppy Disks

Step 1 of the "If you purchased Windows 95 on floppy disks" section of the Windows 95 Readme.txt file states: "Insert Disk 1 into drive A."

Step 1 should state: "Insert Disk 2 into drive A."
