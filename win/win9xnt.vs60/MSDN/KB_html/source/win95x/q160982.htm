Files Installed with Microsoft NetWare Directory Services

 Last reviewed: March 17, 1998
Article ID: Q160982
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows 95 OEM Service Release 2

SUMMARY

This article lists the files that are installed when you add the Microsoft NetWare Directory Services service to the installed network components.

MORE INFORMATION

The following files are installed:
