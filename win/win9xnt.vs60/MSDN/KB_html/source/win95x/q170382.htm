Files or Folders Not Deleted on Network Drive with DELTREE

 Last reviewed: June 19, 1997
Article ID: Q170382
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft MS-DOS operating system versions 6.0, 6.2, 6.21, 6.22

SYMPTOMS

When you delete files or folders using the DELTREE command (Deltree.exe), you may receive the following message instead of an error message when no files or folders have been deleted: 

   Deleting <file or folder>...

CAUSE

You have read-only permission to the network files or folder you are trying to delete.

RESOLUTION

You must have the appropriate permissions for the network folder to delete the files or folder. Change your permissions for the folder, or contact your network administrator.
