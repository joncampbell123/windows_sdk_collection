Closing a Folder Before Setup Is Complete Hangs Computer

 Last reviewed: August 21, 1995
Article ID: Q122497
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you are installing a software package, the computer stops responding (hangs).

CAUSE

Near the end of the installation process, most programs create a folder and icons. At times, this folder may be visible on the screen before the setup program is complete. If you close this folder before setup finishes, the system may hang.

RESOLUTION

To avoid this problem, do not close the folder before the installation is complete.
