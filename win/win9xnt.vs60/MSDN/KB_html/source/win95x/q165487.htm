Incorrect or Garbled Text on Tab Labels in Properties

 Last reviewed: October 16, 1997
Article ID: Q165487
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows 95 OEM Service Release version 1

SYMPTOMS

You may see incorrect or garbled text on tab labels when you are looking at an item's properties.

STATUS

This issue is resolved by the following updated file for Windows 95: 

   Comctl32.dll version 4.70 (dated 5/19/97) and later
To install this update, follow these steps:

  1. Download the Com32upd.exe file from the Microsoft Software Library to an empty folder.

  2. In My Computer or Windows Explorer, double-click the Com32upd.exe file you downloaded in step 1.

  3. Follow the instructions on the screen.

The following file is available for download from the Microsoft Software Library: 

 ~ Com32upd.exe (size: 332808 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
