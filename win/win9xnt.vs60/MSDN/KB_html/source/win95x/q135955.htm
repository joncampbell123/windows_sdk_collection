JPEG and GIF Files Open in Internet Explorer

 Last reviewed: December 19, 1996
Article ID: Q135955
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows version 3.1
  • Microsoft Plus! for Windows 95
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you double-click a JPEG (.jpg) or .gif file, the image opens in Internet Explorer.

CAUSE

When you install Internet Explorer, it replaces the file associations for these files.

RESOLUTION

To change the association for .jpg or .gif files, follow these steps:

  1. Double-click My Computer.

  2. On the View menu, click Options.

  3. On the File Types tab, click the file type you want to change in the Registered File Types box, and then click Remove. When you are prompted to confirm the change, click Yes.

  4. Click OK.

  5. Double-click a .jpg or .gif file in My Computer or Windows Explorer. When you are prompted for the program to use to open the file, select the program you want to use for the file.
