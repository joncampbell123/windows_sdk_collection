Loading HP ScanJet IIcx Driver Causes CD-ROM to Fail

 Last reviewed: October 5, 1995
Article ID: Q129416
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11
  • Microsoft Windows 95

SYMPTOMS

When you load the driver for the Hewlett-Packard (HP) ScanJet IIcx scanner attached to your computer, the computer's CD-ROM drive may stop working.

CAUSE

This problem has been reported to occur with version 2.1a of the HP ScanJet IIcx scanner driver.

RESOLUTION

Use version 2.0c or earlier of the HP ScanJet IIcx driver to correct this problem. For more information about the HP ScanJet IIcx drivers and how to obtain them, please contact Hewlett-Packard.

NOTE: The ScanJet IIcx drivers are real-mode drivers only.
