Configuration Backup Gives Confusing Warning on Restore

 Last reviewed: July 4, 1997
Article ID: Q135121
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you restore a previously backed up file with the Configuration Backup tool (Cfgback.exe), you may receive the following warning: 

   You are about to backup over a previous backup. Do you want to proceed?
Since you are performing a restore, and not a backup, this message is confusing.

RESOLUTION

This warning does not affect the operation of Configuration Backup. To get past the warning, click OK.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
