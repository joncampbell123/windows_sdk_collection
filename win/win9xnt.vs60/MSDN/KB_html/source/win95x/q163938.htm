Inserting Multifunction PC Card May Hang Computer

 Last reviewed: April 30, 1997
Article ID: Q163938
The information in this article applies to:
  • Microsoft Windows 95 OEM Service Release versions 2, 2.1

SYMPTOMS

When you insert certain multifunction PC Cards in a PC Card slot in some computers, the computer may stop responding (hang). When this occurs, you do not hear an audible insertion tone.

This behavior has been observed when a Xircom CreditCard Ethernet+Modem (CEM28) PC Card is inserted in one of the following computers:

  • Hewlett-Packard Omnibook 4000CT
  • Hewlett-Packard Omnibook 5500CT
  • Texas Instruments Extensia 570CDT

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows 95 OEM Service Release 2 (OSR2). A fix to this problem is in development, but has not been regression tested and may be destabilizing in production environments. Microsoft does not recommend implementing this fix at this time. Contact Microsoft Technical Support for more information about the availability of this fix.
