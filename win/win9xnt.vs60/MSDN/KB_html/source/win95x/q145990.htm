Windows 95 Update Information Tool (Qfecheck.exe)

 Last reviewed: October 20, 1997
Article ID: Q145990
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

The Update Information Tool (Qfecheck.exe) is added when you install the Windows 95 Service Pack 1 Update and has the following features:

  • System administrators and Microsoft Product Support Service Engineers can use this tool to find out which Windows 95 Service Pack 1 components have been installed.
  • You can use this tool to verify that the versions of the components installed match the versions listed in your computer's registry.

For additional information about the Windows 95 Service Pack 1 Update, please see the following articles in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q142794
   TITLE     : Availability of Microsoft Windows 95 Service Pack 1

   ARTICLE-ID: Q143003
   TITLE     : Description of Microsoft Windows 95 Service Pack 1 Updates

MORE INFORMATION

To run the Update Information Tool, click the Start button, point to Programs, point to Accessories, point to System Tools, and then click Update Information Tool.

For information about the Registered Updates and Updated Files Found tabs, click Help in the Update Information Tool.

Installing the Update Information Tool

To install the Update Information Tool, follow these steps:

  1. Download the QFECHKUP.EXE file from the Microsoft Software Library to an empty folder.

  2. Using Windows Explorer or My Computer, double-click the QFECHKUP.EXE file you downloaded in step 1.

  3. Follow the instructions on the screen.

The following file(s) are available for download from the Microsoft Software Library: 

 ~ Qfechkup.exe (size: 96408 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
