Microsoft Works 2.0 Setup Err Msg: Cannot Find Setup.ini

 Last reviewed: February 19, 1996
Article ID: Q135138
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Works for Windows, multimedia edition, version 2.0

SYMPTOMS

When you try to install Microsoft Works for Windows, multimedia edition, version 2.0 in Windows 95, you receive the following error message: 

   Cannot find Setup.ini

CAUSE

Microsoft Works for Windows, multimedia edition, version 2.0 Setup is not compatible with Windows 95.

RESOLUTION

Upgrade to Microsoft Works for Windows, multimedia edition, version 3.0 or later.
