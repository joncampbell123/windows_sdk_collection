MS Paint Text Tool Does Not Work with Large Size Zoom Option

 Last reviewed: August 22, 1995
Article ID: Q134481
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you are using the Large Size option on the Zoom menu in Microsoft Paint, the Text tool is disabled.

CAUSE

The Text tool types the exact text image directly into the bitmap, without any provision for scaling.

RESOLUTION

Do not use the Text tool in Microsoft Paint when you are using the Large Size option on the Zoom menu.
