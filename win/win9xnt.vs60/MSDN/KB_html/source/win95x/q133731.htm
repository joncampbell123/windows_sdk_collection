Error Message: Cannot Run This Version of Windows...

 Last reviewed: September 11, 1996
Article ID: Q133731
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to start Windows 95, you receive the following error message: 

   This version of Windows does not run on MS-DOS 6.x or earlier.

CAUSE

Win.com has been added to the Setver table with a version parameter of less than 7.00.

RESOLUTION

To resolve this problem, follow these steps:

  1. Restart your computer. When you see the "Starting Windows 95" message, press the F8 key, and then choose Command Prompt Only from the Startup menu.

  2. Type the following command, and then press ENTER: 

          setver win.com /d

  3. Restart your computer normally.
