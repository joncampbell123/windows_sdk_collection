Error Message: Cannot Find Ftcolor.exe

 Last reviewed: September 19, 1997
Article ID: Q174096
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to open a graphics file, the computer may stop responding (hang), or you may receive the following error message: 

   Cannot find Ftcolor.exe

CAUSE

This error message can occur for any of the following reasons:

  • The Ftcolor.ini file may be damaged.
  • The Ftcolor.exe file may be missing or damaged.

RESOLUTION

Ftcolor is a component included with the Logitech Fototouch Color software included with the Logitech FotoTouch Color Scanner. Contact Logitech for information about resolving this error message.
