Error Message: Rundll32 Has Performed an Illegal Operation...

 Last reviewed: March 11, 1998
Article ID: Q182293
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you install Dial-Up Networking using the Add/Remove Programs tool in Control Panel and then double-click Make New Connection in Dial-Up Networking, you may receive the following error message: 

   Rundll32 has performed an illegal operation and will be shut down.

CAUSE

This error message can occur if you are running the retail version of Windows 95, and you install Dial-Up Networking using a Microsoft Windows 95 OEM Service Release 2 (OSR2) CD-ROM.

RESOLUTION

To resolve this behavior, use one of the following methods:

Method 1

Obtain Dial-Up Networking Upgrade 1.2 from the following Microsoft Web site: 

   http://www.microsoft.com/windows95

Method 2

Uninstall Dial-Up Networking using the Add/Remove Programs tool in Control Panel, and then install Dial-Up Networking using a retail version of Windows 95.

NOTE: If you receive any file version conflict messages, click No to replace the existing files on the hard disk.
