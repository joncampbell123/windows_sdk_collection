List of Confirmed Bugs in Windows 95

 Last reviewed: December 15, 1997
Article ID: Q141602
95 WINDOWS kbref kbbuglist

The information in this article applies to:

  • Microsoft Windows 95

The following tables list confirmed problems in Windows 95. This is not a comprehensive list and is current as of 1/26/96. For more information about each issue, query in the Microsoft Knowledge Base on the article identifier (Article ID).

Summary List for Windows 95

Programs

   Article ID     Article Description


   Q136452        Context-sensitive Help may only appear briefly after
                  you press the F1 key in WordPad or Paint.
   Q137048        The Add/Remove Programs tool in Control Panel may
                  display incorrect available space.
   Q137112        After aborting the compression process, Compression
                  Agent may not give you the option to recompress a
                  different drive.
   Q137456        Centimeter measurements of a margin in WordPad may be
                  reported incorrectly (usually within one one-hundredth
                  of a centimeter).
   Q138051        The Undo command in Microsoft Paint does not restore
                  text.
   Q138392        WordPad Print Preview is inaccurate using the Generic
                  Text Only printer driver.
   Q138797        Adjusted free space may reflect a slight amount of extra
                  space (for example, 100 MB of free space is displayed as
                  100.14 MB).
   Q138827        ALT+F4 hangs Paint when in View Bitmap mode.
   Q139381        Copperplate Gothic Bold font disappears in WordPad.

Hardware

   Article ID     Article Description


   Q134709        Video performance may be slow with some single-speed
                  CD-ROM drives.
   Q136592        When many mouse events occur at the same time, the left
                  mouse button acts like the right mouse button.
   Q136835        PCMCIA hard disk may not be available in a non-IDE
                  computer unless it is reinserted.
   Q138667        New video card is not detected after replacing a Trident
                  card.

Microsoft Exchange

   Article ID     Article Description


   Q137455        When you send a mail message from Microsoft Exchange to
                  an empty distribution list from a CompuServe address
                  book, you may receive an error message.
   Q137593        Cannot insert object as icon in fax cover page editor.
   Q137641        If Microsoft Fax is not the program answering the phone,
                  a fax is not received.
   Q137933        "Port Already in Use" error may occur when starting
                  Microsoft Exchange if Microsoft Fax and Internet Mail
                  are installed in the same profile.

Microsoft Plus!

   Article ID     Article Description


   Q134712        The DIR /C command returns compression ratio for
                  uncompressed files.
   Q134842        The Leonardo da Vinci screen saver displays the image
                  of a tank briefly.
   Q135963        If Microsoft Plus! is installed (and DriveSpace 3 is in
                  use) and then Windows 95 is reinstalled, a DriveSpace
                  3-compatible startup disk cannot be created; however, a
                  work around is available.
   Q135968        You cannot change the destination folder during the
                  installation of Microsoft Plus!.
   Q135974        Smart URL expansion does not recognize port numbers
                  in Internet Explorer 1.0.
   Q136471        Long file names are not displayed correctly using FTP
                  with Internet Explorer.
   Q136837        If your compressed drive has not been upgraded from
                  DriveSpace 2 to DriveSpace 3, you may receive a
                  "Cannot Find the File Cmpagent.exe" error message.
   Q137531        Internet Explorer 1.0 does not display Wingdings
                  characters.
   Q137591        Recycle Bin icons may be reversed after you remove
                  Desktop themes.
   Q138836        Flying Fonts screen saver (from TrueType Font Pack 2.0)
                  may leave afterimages of the font on the screen if Font
                  Smoothing from Microsoft Plus! is enabled.
   Q138941        Installing Microsoft Plus! over a network or to a mapped
                  network drive is not supported.
   Q138990        Ball may not appear on the table in 3D Pinball if the
                  math coprocessor is faulty, or if an incompatibility
                  exists between the math coprocessor and the system
                  BIOS.
   Q139435        WMConfig tool in Microsoft Plus! refers to Hover! in
                  the About dialog box because it is used for multiple
                  applications.
   Q139436        Scheduled tasks in System Agent may be reported as
                  complete as soon as they are started if the scheduled
                  programs launch another program.

   Article ID     Article Description

   Q132517        The joystick port is not removed when the sound card is

   Q138082        Game port may not be redetected if the sound card is

Networking

   Article ID     Article Description


   Q123694        Canceling the Network Install Wizard causes an error
                  message when your computer restarts.
   Q125925        WFWG client may not be able to connect to a Windows 95
                  server if the Windows 95 server uses user-level access
                  control and the domain the server uses to validate
                  clients is different from the client's domain.
   Q127852        The total free space reported may be incorrect on
                  network drives larger than 2 GB.
   Q129234        Net Watcher continues to list disconnected client.
   Q131675        A Microsoft Windows NT client may be unable to connect
                  to a Windows 95 server in specific circumstances.
   Q131884        Cannot restore minimized NetWare logon script console
                  window.
   Q132806        Persistent connections are not preserved when you
                  upgrade from Microsoft LAN Manager 2.2 client.
   Q132815        Windows 95 may stop responding during a server-based
                  installation on an EISA computer using a NetBEUI
                  connection (this problem has no effect on the Setup
                  process).
   Q134485        After the dynamic loading and unloading of the
                  IPX/SPX-compatible protocol, your computer stops
                  acting as a server until it is restarted.
   Q134507        You cannot add the "NetBIOS Support for IPX/SPX-
                  Compatible Protocol" service with the Add button. To
                  add it, you must click the "I want to enable NetBIOS
                  over IPX/SPX" check box in the properties for the
                  IPX/SPX-compatible protocol.
   Q134713        Network printers are not visible in the Browse dialog
                  box if the printers are connected to computers running
                  third-party network software that cannot be accessed
                  using Universal Naming Convention (UNC) names.
   Q134845        A network shortcut to an MS-DOS-based program does not
                  remap the drive unless the Reconnect At Logon check box
                  is selected when you map the network drive.
   Q134936        Comments do not appear on the General tab in the
                  properties for a network printer that is installed
                  locally and then changed to a network printer.
   Q135175        Cannot use two Xircom PCMCIA network cards at once.
   Q135189        If you use a Hewlett-Packard HP27252 ISA Ethernet
                  network adapter with NDIS3 drivers, you can safely
                  ignore the "Your Network Adapter HP27252 Is Not
                  Configured" error message.
   Q135195        When Net Watcher is used to monitor connections to a
                  computer running the File and Printer Sharing for
                  NetWare Networks service, open files are not displayed
                  in the By Connection view. Click By Open Files to see
                  files that are open.
   Q135211        Users with the same user name on different servers or
                  domains appear to be the same user to Windows 95, and
                  use the same user profile.
   Q135279        Incomplete domain listing on very large networks.
   Q135819        You may receive an error message when you print to a
                  Novell NetWare printer with extended characters in the
                  name.
   Q137271        Group Policy resolution does not function with Novell's
                  real-mode NETX and VLM redirectors.
   Q137856        Windows 95 does not parse the LMHOSTS file if DNS
                  is enabled.
   Q137869        If the network disconnects during certain network
                  operations, you may receive a "System Error - Network
                  Error on Drive <X>" error message.
   Q138048        The server name may not be displayed in the login dialog
                  box with the Microsoft Service for NetWare Directory
                  Services (NDS) client.

Printing

   Article ID     Article Description


   Q132007        When you copy characters from the Wingdings font in
                  Character Map to WordPad, the characters may appear in
                  the wrong font.
   Q132603        Cannot view shared fonts folder on another computer.
   Q133079        When you preview a 1-page fax in Fax Viewer, or a
                  1-page document in WordPad, two pages appear. The
                  second page is blank.
   Q134773        The printer name does not appear during drag-and-drop
                  printing.
   Q135038        You cannot scroll through the list of printers with the

Setup

   Article ID     Article Description


   Q130757        Setup may not detect a Tseng 4000/W32 video adapter on
                  NEC Image P90E computers.
   Q131481        Settings for the Hewlett-Packard LaserJet 4Si are not
                  maintained during upgrade.
   Q133435        After an OEM driver is installed, Windows 95 cannot load
                  a Windows 95 driver over an OEM driver of the same name.
   Q134526        After installing Windows 95, you cannot access drive D

   Q134573        After you upgrade to Windows 95 with a Novell NE3200

Shell

   Article ID     Article Description


   Q136012        The Start button context menu is hidden by the

                  shortcut bar is positioned at the bottom of the screen.
   Q136451        Incorrect default Short Date style is used after you
                  change the style unless you restart Windows 95.
   Q136475        If horizontal spacing for icons is set too high, you may

   Q136954        Disk formatting is slow after failing on write-protected
