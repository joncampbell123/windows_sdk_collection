Ending DUN Connection Not the Same as Logging Off

 Last reviewed: July 22, 1997
Article ID: Q171696
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows 95 OEM Service Release version 2

SUMMARY

Ending a Dial-Up Networking connection is not the same as ending all network connections and then logging off the network.

MORE INFORMATION

To log off the network after ending a Dial-Up Networking connection, click Start, click Shut Down, click "Close all programs and log on as a different user," and then click Yes.
