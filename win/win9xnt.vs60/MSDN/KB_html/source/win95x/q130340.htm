Windows 95 Hangs Printing with Vertisoft Emulaser

 Last reviewed: August 28, 1995
Article ID: Q130340
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you print to an Emulaser port in Windows 95 your computer stops responding (hangs).

CAUSE

The printing architecture in Windows 95 is incompatible with Vertisoft Emulaser.

RESOLUTION

To print in Windows 95, use a Windows 95 printer driver instead of using Emulaser.

MORE INFORMATION

Emulaser is a software package designed to rasterize (or convert) PostScript print jobs to bitmap format. Emulaser lets Windows version 3.x print PostScript print jobs to non-PostScript printers (such as dot-matrix and PCL printers).
