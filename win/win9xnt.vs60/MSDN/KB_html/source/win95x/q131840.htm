WordPerfect Tutorial Hangs in Windows 95

 Last reviewed: September 7, 1995
Article ID: Q131840
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you run the tutorial in WordPerfect version 6.0, 6.0a, or 6.1, your computer stops responding (hangs) and the menu bar flashes when you click the Show Me button.

CAUSE

This behavior occurs when the mouse pointer moves automatically to an item on the menu bar that you can click.

RESOLUTION

You can close the WordPerfect components by pressing CTRL+ALT+DEL. If you try to start WordPerfect again, you receive the following error message: 

   ERROR: Cannot Initialize
To run WordPerfect again, restart your computer and do not run the tutorial.
