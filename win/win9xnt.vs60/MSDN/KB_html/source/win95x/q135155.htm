Unable to See MDI Window Frames in True Color Modes

 Last reviewed: August 21, 1995
Article ID: Q135155
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

You may be unable to see Multiple Document Interface (MDI) window frames while you are resizing child windows.

CAUSE

Normally, when you resize a child window, the new window size is indicated by a light gray outline.

In True Color modes, the color used for the child window outline is indistinguishable from the default window background color.

RESOLUTION

To make MDI window frames visible while you are dragging the window, use one of the following methods:

  • Change the window background to a different color
  • Change the display to use fewer colors.
