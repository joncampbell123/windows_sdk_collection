Paradox Err Msg: "The File WINTUTOR.EXE Could Not Be Found"

 Last reviewed: August 21, 1995
Article ID: Q124660
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to start the Getting Around In Paradox tutorial in Paradox version 5.0, you receive the following error message: 

   Interactive Tutors. The file Wintutor.exe could not be found.

CAUSE

The WINTUTOR.EXE tutorial application that is included with Microsoft Windows version 3.1 is not included with Microsoft Windows 95. This error should occur only on computers that have a clean installation of Windows 95.

RESOLUTION

Paradox will function correctly with the exception of the tutorial. If you want to run the tutorial, contact the program's vendor about receiving an upgrade to a newer version.
