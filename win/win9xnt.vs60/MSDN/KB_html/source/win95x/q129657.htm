Windows 95 Cannot Create 360K Floppy System Disk

 Last reviewed: September 13, 1995
Article ID: Q129657
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

Windows 95 cannot create a 360K floppy system disk.

MORE INFORMATION

When you create a floppy system disk, the system files require 422K of disk space. Therefore, the system files do not fit on 360K disks.

When you try to format a 360K floppy disk as a system disk using Windows Explorer, you receive the following error message: 

   Windows could not copy the system files to this disk. Do you want
   to continue formatting without copying system files?
If you click OK, Windows 95 formats the disk but does not copy the system files. If you click Cancel, Windows 95 stops formatting the disk.
