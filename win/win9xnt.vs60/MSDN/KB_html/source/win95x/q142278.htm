Windows 95(R) Updated Modem INF Files

INTRODUCTION

This package installs .inf files for these specific modems: 

   mdmdsi.inf    Creative Labs Phone Blaster
   mdmrock2.inf  Diamond Multimedia TeleCommander 2500
   mdmrock3.inf  Aztech Systems
   mdmzoom2.inf  Zoom Comstar
The next time you go to install a modem, Windows 95 will rebuild the modem manufacturer database automatically. If you need additional assistance with installing your modem under Windows 95, search on the word "modem" in Windows 95 help.

Older versions of these files have been backed up to the windows inf\ backup directory.

VOICEVIEW

If you have Windows 95 on disk and you have a VoiceView modem, please download the Microsoft File Transfer package. That package contains files necessary for a complete installation of some modems in the above- mentioned INF files.
