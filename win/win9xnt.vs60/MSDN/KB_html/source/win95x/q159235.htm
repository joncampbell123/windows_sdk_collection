APM Drive Spin Down Supported on IDE Devices Only

 Last reviewed: November 11, 1996
Article ID: Q159235
The information in this article applies to:
  • Microsoft Windows 95 OEM Service Release version 2

SYMPTOMS

With Windows 95 OEM Service Release 2 (OSR2), the drive spin down feature works only with IDE hard disks. It does not work, for example, with SCSI hard disk.

CAUSE

The drive spin down feature is only supported on IDE devices.

MORE INFORMATION

Although SCSI hard disks do support the drive spin down feature natively, Windows 95 does not support this feature with SCSI drives.

The IDE driver (Esdi_506.pdr) does support this feature.
