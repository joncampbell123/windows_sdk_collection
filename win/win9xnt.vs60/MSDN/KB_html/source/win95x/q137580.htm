Compaq Computers That May Require Updates for Windows 95

 Last reviewed: March 17, 1998
Article ID: Q137580
95 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Windows 95

SUMMARY

This article contains information provided by Compaq Computer Corporation regarding Compaq computers that may require additional Compaq drivers or supplements for Windows 95. These additional drivers can be obtained from Compaq's Softpaq library. For additional information about Compaq supplemental updates, contact Compaq Computer Corporation at 1-800-952- 7689. For a list of current Softpaqs, call the Compaq Fax Back Service at 1- 800-345-1518 and select the appropriate PAQFAX menu options to order PAQFAX Document #7, which contains a catalog of Windows 95 information.

MORE INFORMATION
