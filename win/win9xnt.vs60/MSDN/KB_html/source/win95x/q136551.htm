Accessibility Options Require Program Compatibility

 Last reviewed: July 8, 1997
Article ID: Q136551
95 WINDOWS kbui kb3rdparty

The information in this article applies to:

  • Microsoft Windows 95

SYMPTOMS

When you select either of the following Accessibility options, the option may not seem to have any effect:

  • Show Extra Keyboard Help in Programs
  • Use ShowSounds

CAUSE

The program you are using was not written to support these options.

RESOLUTION

Please contact the program's manufacturer for information about the compatibility of the program with these Windows 95 options.

MORE INFORMATION

In Windows 95, you can specify a keyboard preference option that informs programs that you prefer to use the keyboard rather than the mouse. Programs written to support this feature change their appearance to graphically display keyboard-related items.

The Use ShowSounds option instructs programs that normally convey information only with sounds to also provide information visually, such as by displaying text captions or informative icons.
