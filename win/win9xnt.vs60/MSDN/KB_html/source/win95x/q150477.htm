HP Omnibook PCMCIA Drivers in Drivers\Display\Ob600ss Folder

 Last reviewed: October 11, 1996
Article ID: Q150477
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows 95 OEM Service Release 1

Drivers that can be used to enable protected-mode socket services on Hewlett-Packard (HP) Omnibook 600 computers are located in the Drivers\Display\Ob600ss folder on the Windows 95 OEM Service Release 1 (OSR1) CD-ROM. The drivers in the Drivers\Display\Ob600ss folder are not display drivers, as indicated in the Windows Driver Library List contained in the Wdl_list.htm file.

For additional information about Windows 95 OSR1, please see the following article in the Microsoft Knowledge Base:
