Voice Modem Drivers Available in Unimodem V

 Last reviewed: August 11, 1997
Article ID: Q172573
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

The downloadable version of Unimodem V includes a number of voice modem drivers that are certified by Microsoft to take advantage of the features in Unimodem V. This article lists the modems drivers included with the downloadable version of Unimodem V.

For more information about the contents of Unimodem V, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140323
   TITLE     : Microsoft Unimodem V Readme.txt File
For information about obtaining Unimodem V, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q139383
   TITLE     : Unimodem V Adds Additional Support for Data/Fax/Voice 
               Modems
