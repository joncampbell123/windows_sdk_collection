Error Message: VGASWAP VDD Control Hook Recursion

 Last reviewed: August 21, 1995
Article ID: Q134530
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

You may receive the following error message in Windows 95: 

   386MAX ERROR: VGASWAP VDD Control Hook Recursion

CAUSE

The Qualitas 386Max Memory Manager VGASWAP feature is not completely compatible with Windows 95.

RESOLUTION

Disable the VGASWAP feature. For information about doing so, please consult the product documentation or call Qualitas Technical Support.
