ICW Starts Instead of Internet Explorer after OSR2 Reinstall

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows 95
  • Microsoft Windows 95 OEM Service Release versions 2, 2.1

SYMPTOMS

When you double-click the Internet icon on the desktop, the Internet Connection Wizard (ICW) may start instead of the version of Internet Explorer that is currently installed on your computer. If you are using Internet Explorer 4.0, the Internet Connection Wizard included with an earlier version of Internet Explorer starts. In addition, if you proceed through the ICW and choose to configure your own Internet settings, you may receive one or more of the following error messages:

  • The INETWIZ.EXE file is linked to missing export INETCFG.DLL:LaunchSignupWizard@8
  • A device attached to the system is not functioning.
  • The Internet Setup Wizard could not load the INETWIZ.EXE file.

CAUSE

This problem occurs if you reinstall OEM Service Release version 2 (OSR2) after you install Internet Explorer. When you install OSR2, the Internet icon is configured to start the Internet Connection Wizard. When you reinstall OSR2 after installing Internet Explorer 4.0, an earlier version of the Internet Connection Wizard is installed.

RESOLUTION

To resolve this problem, use the appropriate method.

Internet Explorer 3.x Is Currently Installed

To resolve this problem and retain your previous connection settings, follow these steps:

  1. Double-click the Internet icon to start the Internet Connection Wizard.

  2. Click Next, and then click Current.

  3. Click Next, and then click Finish.

Internet Explorer 4.0 Is Currently Installed

To work around this issue, delete the Internet icon on your desktop, and then create a new shortcut for Internet Explorer. To create a new shortcut for Internet Explorer, follow these steps:

  1. Click Start, point to Find, and then click Files Or Folders.

  2. Type "iexplore.exe" (without quotation marks) in the Named box.

  3. Click the hard disk on which Internet Explorer is installed in the Look In box, and then click Find Now.

  4. Right-click the Iexplore.exe file, and then click Copy.

  5. Right-click an open area on the desktop, and then click Paste Shortcut.
