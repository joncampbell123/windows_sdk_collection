Problems with Media Blitz 3.0 Setup

 Last reviewed: September 11, 1995
Article ID: Q132834
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

After you install Media Blitz version 3.0, when you try to start ClipMaker or ScoreMaker you receive an Open With dialog box.

CAUSE

Media Blitz 3.0 Setup adds associations for these files to the Win.ini file rather than to the registry.

RESOLUTION

Restart your computer to resolve the problem.
