MS Paint Color Eraser Tool Does Not Erase Bitmap

 Last reviewed: August 22, 1995
Article ID: Q134476
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to use the Color Eraser tool in Microsoft Paint, you find that you cannot make changes to a bitmap.

CAUSE

You are trying to edit a bitmap that originated as a 24-bit image.

RESOLUTION

After you make a backup copy of the image in its current format, save the bitmap in 256 (or fewer) color format, and then open it again.
