Error Message: The Search Could Not Be Started

 Last reviewed: February 23, 1996
Article ID: Q147345
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

When you try to use the Microsoft Exchange Find command on a public or shared folder on a Microsoft Mail 3.x post office or a Workgroups post office, you receive the following error message: 

   The search could not be started  [80040102]

MORE INFORMATION

These post offices do not support the Microsoft Exchange Find command on a shared or public folder.
