How to Install and Run Clipboard Viewer

 Last reviewed: December 15, 1995
Article ID: Q141310
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

Clipboard Viewer is not installed by default in Windows 95. This article describes how to install and run Clipboard Viewer.

MORE INFORMATION

To install Clipboard Viewer, follow these steps:

  1. In Control Panel, double-click Add/Remove Programs.

  2. Click the Windows Setup tab.

  3. Click Accessories, and then click Details.

  4. Click the Clipboard Viewer check box to select it, and then click OK.

  5. Click OK.

To run Clipboard Viewer after you install it, click the Start button, point to Programs, point to Accessories, and then click Clipboard Viewer. You can use Clipboard Viewer to view the contents of the Clipboard.
