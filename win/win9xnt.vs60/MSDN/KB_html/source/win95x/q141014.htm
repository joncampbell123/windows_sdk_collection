Emergency Recovery Utility Does Not Back Up Files

 Last reviewed: May 10, 1996
Article ID: Q141014
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you use the Emergency Recovery Utility (ERU) that is included on the Windows 95 CD-ROM to back up your configuration files, the utility may not work correctly. It may stop without any warning or error message.

CAUSE

This behavior can occur if one or more of the default configuration files is missing.

RESOLUTION

Verify that all the default configuration files exist on the hard disk.

MORE INFORMATION

The ERU does not function properly if any of the following files are missing: 

   Autoexec.bat
   Command.com
   Config.sys
   Io.sys
   Msdos.sys
   Protocol.ini
   System.dat
   System.ini
   User.dat
   Win.ini
