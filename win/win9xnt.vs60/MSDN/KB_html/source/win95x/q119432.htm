Windows 95 Setup Disables OS/2 Boot Manager

 Last reviewed: August 20, 1997
Article ID: Q119432
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you run Windows 95 Setup on a system that has the OS/2 Boot Manager partition active, the following message is displayed: 

   Setup has detected a boot manager partition on your system.
   Continuing Setup will disable Boot Manager.

   <Exit Setup> or <Continue>
Continuing Windows 95 Setup disables the Boot Manager.

CAUSE

Windows 95 Setup disables the Boot Manager to ensure that after installation the system will boot into Windows 95, not a different operating system. This allows Windows 95 to finish the installation procedure.

RESOLUTION

You can reactivate Boot Manager by running the FDISK tool that is shipped with Windows 95 or boot the system from the OS/2 boot disk and then run the Fdisk utility.

MORE INFORMATION

The Windows 95 Setup program does not run under OS/2. However, you can install Windows 95 on a computer running OS/2 if you boot MS-DOS and run Setup from the command prompt.
