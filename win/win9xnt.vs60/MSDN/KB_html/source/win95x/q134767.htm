Lens Screen Saver Hangs with Diamond Viper Video Drivers

 Last reviewed: August 28, 1995
Article ID: Q134767
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! For Windows 95

SYMPTOMS

When the Microsoft Plus! for Windows 95 Lens screen saver activates on a computer running 16-bit Diamond Viper video drivers in 1024 x 768 x 256 resolution, the desktop turns black and white and then the system stops responding (hangs).

CAUSE

The 16-bit (Windows 3.1) Diamond Viper video drivers are not compatible with Windows 95 and Microsoft Plus! in 1024 x 768 x 256 resolution.

RESOLUTION

Please contact Diamond to inquire about obtaining updated drivers for the video card.
