INCLUDE/DISPLAY Login Script Commands Do Not Accept UNC Paths

 Last reviewed: August 21, 1995
Article ID: Q135167
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

If you are using a NetWare login script, INCLUDE and DISPLAY commands in the login script that contain Universal Naming Convention (UNC) paths do not work. The named files are not run or displayed.

CAUSE

The login script processor cannot properly handle UNC paths.

RESOLUTION

Change the INCLUDE and DISPLAY commands to refer to local or mapped drives instead of UNC paths.
