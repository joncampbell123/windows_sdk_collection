Description of "Get ISDN for Microsoft Windows"

 Last reviewed: March 17, 1998
Article ID: Q151664
95 WINDOWS kbtool kbref

The information in this article applies to:

  • Microsoft Windows 95

SUMMARY

"Get ISDN for Microsoft Windows" is a software solution that simplifies the process of obtaining an ISDN (Integrated Services Digital Network) line for a Windows-based computer.

MORE INFORMATION

The "Get ISDN for Microsoft Windows" program includes information from major local telephone companies that offer ISDN service, including Ameritech, Bell Atlantic, Bell Canada, BellSouth, GTE, NYNEX, Pacific Bell, Southern New England Telephone, and US West Communications, MCI, and Sprint as providers of long-distance ISDN service. This software is also supported by leading online services and Internet service providers (ISPs) with broad ISDN access to their services, including MSN, The Microsoft Network, Netcom, PSINet, and UUNET.

The "Get ISDN for Microsoft Windows" program lets you request an ISDN line electronically by forwarding your request to the appropriate telephone company. "Get ISDN for Microsoft Windows" is available on Microsoft's World Wide Web Site on the Internet: 

   http://www.microsoft.com/windows/getisdn
For information about software for Windows 95 that lets you take advantage of an ISDN connection, please see the following article in the Microsoft Knowledge Base:
