APM Not Installed on Midwest Micro Elite Soundbook

 Last reviewed: August 21, 1995
Article ID: Q134300
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you install Windows 95 on a Midwest Micro Elite Soundbook computer, the Advanced Power Management (APM) features are not installed, and you cannot add Windows 95 APM support using the Add/Remove Programs tool.

CAUSE

The Windows 95 APM drivers do not work correctly with the BIOS in Midwest Micro Elite Soundbook computers. Therefore, APM support cannot be installed.

RESOLUTION

To use APM features in Windows 95, install the APM drivers that came with the computer. For information about installing these drivers, please refer to the computer documentation or manufacturer.
