GP Faults Running QuickBooks 2.0 in Windows 95

 Last reviewed: November 14, 1996
Article ID: Q130147
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you perform tasks in Intuit QuickBooks version 2.0 or 3.0, you receive various general protection (GP) fault error messages.

RESOLUTION

To correct this problem, use either of the following methods:

  • Upgrade to QuickBooks version 3.1 or higher.
  • Perform the following steps to run QuickBooks 2.0 or 3.0 in Windows 95:

    1. Use any text editor (such as WordPad) to open the WIN.INI file.

    2. Add the following line in the [Compatibility] section of the 

          WIN.INI file


             QBW=0x08000000

      where QBW is the module name for QuickBooks.

   3. Save and then close the WIN.INI file.

   4. Restart Windows 95.
NOTE: If you insert the "QBW=0x08000000" line in the WIN.INI file, you must remove the line if you later upgrade to QuickBooks 3.1 or higher. If you upgrade QuickBooks and do not remove this line, printing problems may occur.
