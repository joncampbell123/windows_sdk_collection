Err Msg: The Microsoft Dial-Up Adapter Is Not Installed...

 Last reviewed: January 30, 1998
Article ID: Q180143
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you attempt to connect to a remote computer after you install the Dial-Up Networking 1.2 Upgrade, you may receive the following error message: 

   The Microsoft Dial-Up Adapter is not installed or is not responding
   properly. In the Network section of the Control Panel, remove the
   driver and add it again.
When you click OK, you may receive one or more of the following error messages:
  • Rnaapp.dll caused an invalid page fault in Rnaapp.dll
  • Explorer caused an invalid page fault in Rnaui.dll
  • Rnaapp.dll caused an invalid page fault in Kernel32.dll

CAUSE

These error messages can occur if the Rasapi32.dll file is not updated properly when you install the Dial-Up Networking 1.2 Upgrade.

Note that the Rasapi32.dll file included in the Dial-Up Networking 1.2 Upgrade is version 4.00.1537.

RESOLUTION

To resolve this issue, use the appropriate method:

Method 1

Rename the Rasapi32.dll file located in the Windows\System folder, and then reinstall the Dial-Up Networking 1.2 Upgrade.

For information about how to rename a file, click Start, click Help, click the Index tab, type "renaming" (without quotation marks), and then double-click the "Renaming files" topic.

Method 2

Extract a new copy of the Rasapi32.dll file from the Dial-Up Networking 1.2 Upgrade. To do this, follow these steps:

  1. Rename the Rasapi32.dll file located in the Windows\System folder.

    For information about how to rename a file, click Start, click Help, Click the Index tab, type "renaming" (without quotation marks), and then double-click the "Renaming files" topic.

  2. Click Start, and then click Run.

  3. In the Open box, type the following command, and then press ENTER 

          <path1>\msdun12.exe /t:<path2> /c

    where <path1> is the path to the Msdun12.exe file, and <path2> is the folder to which you want to extract the Dial-Up Networking 1.2 Upgrade component files. For example: 

          c:\msdun12.exe /t:c:\temp /c

  4. Restart your computer. When you see the "Starting Windows 95" message, press the F8 key, and then choose Command Prompt Only from the Startup menu.

  5. At the command prompt, type the following command, and then press ENTER 

          extract <path2>\msdun.cab rasapi32.dll /l <windows>\system

    where <path2> is the path to the folder containing the Dial-Up Networking 1.2 Upgrade component files, and <windows> is the name of the folder in which Windows is installed. For example: 

          extract c:\temp\msdun.cab rasapi32.dll /l c:\windows\system

  6. Restart your computer.
