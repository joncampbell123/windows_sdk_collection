Double-Clicking Drive Icon in My Computer Runs Autorun.exe

 Last reviewed: February 12, 1997
Article ID: Q163505
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows 95 OEM Service Release version 2
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you open My Computer and double-click a drive icon, you may receive the following message: 

   Windows cannot find Autorun.exe

   This program is needed for opening files of type 'file'.
Additionally, if you use the right mouse button to click the drive icon, the default action is AutoPlay.

CAUSE

There is an Autorun.inf file located in the root folder of the drive.

RESOLUTION

Remove the Autorun.inf file from the root folder of the drive.

MORE INFORMATION

An Autorun.inf file is used to load a corresponding Autorun.exe file, usually on a CD-ROM. For example, the Windows 95 and Windows NT 4.0 CD-ROMs contain an Autorun.inf file. When you load one of these CD-ROMs, the Autorun.inf file is read and the Autorun.exe program is run. In most cases, this automatically starts a Setup program.
