How to Restart Windows 95 Without Restarting the Computer

 Last reviewed: January 13, 1998
Article ID: Q143272
The information in this article applies to:
  • Microsoft Windows 95

NOTE: This article contains information about using Windows 95 in a configuration that has not been tested and is not supported by Microsoft. If the steps described in this article do not function properly, do not use this method.

SUMMARY

This article describes how to restart Windows 95 without restarting your computer.

MORE INFORMATION

To restart Windows 95 without restarting your computer, follow these steps:

  1. Click the Start button, and then click Shut Down.

  2. Press and hold down the SHIFT key while you click Restart The Computer and then click Yes.

  3. As soon as the screen clears, release the SHIFT key.

NOTE: If you hold the SHIFT key too long, Windows 95 restarts in Safe mode.
