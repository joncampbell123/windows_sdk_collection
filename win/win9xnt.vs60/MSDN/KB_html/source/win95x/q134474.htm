Power Spanish Causes a GP Fault When You Quit Program

 Last reviewed: August 22, 1995
Article ID: Q134474
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you open a file in Power Spanish by Bayware, make a change, and then close the program without first saving the changes, a general protection fault occurs; however, your changes are successfully saved and there is no data corruption.

CAUSE

Power Spanish is incompatible with Windows 95.

WORKAROUND

To work around this behavior, manually save the file before you quit the program.

STATUS

This situation is most likely caused by design changes in Windows 95. Microsoft has confirmed that it is not caused by a problem in Windows 95. For more information about resolving this issue, contact Computer Associates International.
