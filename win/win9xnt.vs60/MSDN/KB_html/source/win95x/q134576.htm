PostScript Errors Printing with Bitstream Facelift 2.0

 Last reviewed: August 22, 1995
Article ID: Q134576
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you print to a PostScript printer with Bitstream Facelift version 2.0 enabled, the following error message is printed: 

   ERROR: undefined
   OFFENDING COMMAND: fpp
   STACK:

CAUSE

Facelift 2.0 uses internal PostScript commands that are defined by the Windows 3.1 PostScript driver. These commands are different in the Windows 95 PostScript driver.

STATUS

This situation is most likely caused by design changes in Windows 95. Microsoft has confirmed that it is not caused by a problem in Windows 95. For more information about resolving this issue, please contact Bitstream.
