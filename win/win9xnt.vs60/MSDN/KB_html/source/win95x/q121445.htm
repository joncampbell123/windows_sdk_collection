Installing LapLink Version 6.0b for Windows Causes GP Fault

 Last reviewed: November 9, 1995
Article ID: Q121445
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

After you install LapLink version 6.0b for Windows in Windows 95, you may receive a general protection (GP) fault error message when you restart your computer.

CAUSE

This problem occurs if you upgrade LapLink version 6.0 or 6.0a for Windows to version 6.0b. LapLink 6.0b for Windows removes the previous LapLink drivers and then prompts you to restart your computer and run Setup again. When you restart your computer, a GP fault occurs and you cannot continue.

RESOLUTION

To install LapLink 6.0b for Windows, follow these steps:

  1. Restart your computer in Safe mode. To do so, restart your computer, press the F8 key when you see the "Starting Windows 95" message, and then choose Safe Mode from the Startup menu.

  2. Run LapLink's Uninstall program. To do so, click the Start button, point to Programs, click LapLink For Windows, and then click Uninstall.

    NOTE: If you installed LapLink for Windows in a group named something other than LapLink For Windows, click that name instead.

  3. After the LapLink Uninstall program is finished, reselect your video adapter driver. To do so, follow these steps:

    a. In Control Panel, double-click the Display icon.

    b. On the Settings tab, click Change Display Type.

    c. Note the adapter type listed in the Adapter Type box, and then 

          click Change in the Adapter Type area.

    d. In the Models box, click the adapter type you noted in step C, 

          and then click OK.

    e. Click OK or Close until you return to Control Panel.

    f. When you are prompted to restart your computer, do so. Let 

          Windows 95 start normally.

  4. Install LapLink 6.0b for Windows.

NOTE: You cannot install LapLink for Windows if you are using a Number Nine Imagine 128 video adapter. For information about a possible work- around for this problem, please contact Number Nine.

MORE INFORMATION

To use animated cursors with LapLink installed in Windows 95, you must use LapLink 6.0b. Earlier versions of LapLink cause animated cursors to turn black and not animate.

For information about known issues that can occur after you install LapLink for Windows, please obtain document number 5521 from Traveling Software's FaxBack service.
