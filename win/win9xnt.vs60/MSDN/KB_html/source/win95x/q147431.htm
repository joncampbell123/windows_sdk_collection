Cannot Hide Host Drive for Compressed Volume on Removable Disk

 Last reviewed: February 26, 1996
Article ID: Q147431
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

When you try to hide the host drive for a compressed volume created on a removable drive, the Hide Host Drive check box is unavailable.

MORE INFORMATION

This behavior is by design. You cannot hide the host drive for compressed volumes created on removable drives. This includes floppy disk drives, Zip drives, Bernoulli drives, and external hard disk drives.
