Inf Generator Tool Available

 Last reviewed: March 17, 1998
Article ID: Q145664
95 WINDOWS kbnetwork kbtool

The information in this article applies to:

  • Microsoft Windows 95

NOTE: This article discusses an unsupported tool for Windows 95. Microsoft does not support this tool and provides this article as information only. Use this tool at your own risk.

SUMMARY

The Inf Generator tool (Ig.exe) can help you customize the automatic installation of Windows 95 in a corporate networking environment.

The following features are available in Inf Generator:

  • You can specify separate settings for the company, department, and user when Windows 95 is being installed. This is especially useful when static TCP/IP addresses are assigned.
  • You can configure a batch Setup that prevents icons for The Microsoft Network and the Inbox from being created on the desktop.

MORE INFORMATION

To install the Inf Generator tool, follow these steps:

  1. Download the Ig.exe file from one of the services listed below to an empty folder.

  2. Double-click the Ig.exe file to extract the files it contains.

  3. Double-click the Readme.txt file for installation instructions. 

    You can find Ig.exe (size: 89728 bytes) 
                   , a self-extracting file, on the following services:
  4. Internet (anonymous FTP): 

          ftp ftp.microsoft.com
      Change to the PEROPSYS/WIN_NEWS/FREESOFTWARE directory
      Get Ig.exe (size: 89728 bytes)
  5. Internet (World Wide Web): 

          http://www.microsoft.com/windows/software/servpak1/isprof.htm

    Note that the Inf Generator tool is not the same tool as the Inf Installer tool included on the Windows 95 CD-ROM.
