SSI Software's LASSIE Does Not Print Puzzle

 Last reviewed: August 21, 1995
Article ID: Q134564
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to print the puzzle in LASSIE, you may receive the following error message: 

   A printer is not connected

CAUSE

LASSIE does not currently support printing the puzzle in Windows 95.

WORKAROUND

Currently there is no workaround.

MORE INFORMATION

This problem is specific to the LASSIE program. It does not occur with other SSI programs, such as The Secret Garden.
