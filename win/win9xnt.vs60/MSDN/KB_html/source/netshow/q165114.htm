NetShow Did Not Set Up into the IIS Internet Service Manager

 Last reviewed: September 16, 1997
Article ID: Q165114
The information in this article applies to:
  • Microsoft NetShow version 2.0

SYMPTOMS

When I set up NetShow, it did not set up into the Microsoft Internet Information Server (IIS) version 3.0 Internet Service Manager.

CAUSE

Unlike NetShow version 1.0, NetShow version 2.0 has no dependency on IIS and will not show up in the Internet Service Manager.

WORKAROUN

All of the administration for NetShow 2.0 is performed via web based accessed from the NetShow program group. You select the NetShow administrator to access the HTML administration pages.
