How to Search for NetShow KB Articles by Topic

 Last reviewed: September 16, 1997
Article ID: Q164864
The information in this article applies to:
  • Microsoft NetShow versions 1.0 and 2.0

SUMMARY

You can use the following KBSubcategory keywords to make queries on the Microsoft NetShow Knowledge Base quicker and more precise. To achieve that speed and precision, the keywords contain the unique prefix (ns) to minimize the chance of finding a similar keyword in an unrelated article during a full-text query.

NOTE: In addition, you can use the more general KBCategory keywords (containing the "kb" prefix) that apply to all Knowledge Base articles. For a listing of KBCategory keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles

NetShow Knowledge Base Subcategories

Keyword       Subcategory


nssound       Sound Issues
nsclnt        Client Issues
nssrv         Server Issues
nstls         Tools Issues
nsgen         General Issues
nsmulti       Multicasting Issues

Additional query words: ns 1.00 kbkeyword
Keywords : kbkeyword kbref
Version : 1.00 2.00
Platform : WINDOWS
