DAO and Jet Drivers in Use Error Message During Setup

 Last reviewed: September 16, 1997
Article ID: Q171612
The information in this article applies to:
  • Microsoft NetShow version 2.0

SYMPTOMS

When you try to install the server components, and the Setup program prompts you for the username for the service, you can get an error message regarding DAO and Jet drivers being in use. You will get the option to Retry or Cancel.

NOTE: You can have the above problem even if you shut down all applications before you run the Setup program or stop all services that will not affect system performance (mail servers or web and ftp servers).

WORKAROUND

In a DOS prompt, go to the System32 directory. Rename the following files using the following commands as an example: 

   REN MSJT*.* *.old and REN MSJI*.*. *.old
Then try to set up NetShow Server.
