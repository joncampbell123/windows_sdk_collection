Regarding Publish to ASF from PowerPoint 97

 Last reviewed: January 23, 1998
Article ID: Q160805
The information in this article applies to:
  • Microsoft NetShow version 1.0
  • Microsoft PowerPoint 97
  • Microsoft Office 97

SYMPTOMS

The Help file with the Office 97 Value Pack (Valuepk8.hlp) has incorrect information regarding how to use the PowerPoint to ASF converter (Asfppt.exe) available on the Office 97 compact disc. The instructions state that the ActiveMovie player and filter must be installed to play the .asf files.

The instructions under the topic, "Microsoft Publish to ActiveMovie Stream Format", direct you to install the ActiveMovie player and filter to play the ASF files created with the PPT to ASF PowerPoint Add-On. However, to play ASF files, you must install the NetShow On-Demand player (available for download at no charge).

RESOLUTION

Download (at no charge) and install the NetShow On-Demand player from: 

   http://www.microsoft.com/netshow/

STATUS

Microsoft has confirmed this to be a problem with the Help file. We will post a modified Help file as it becomes available.
