Nsserver.exe Fails w/ Services Could Not Be Registered Error

 Last reviewed: September 16, 1997
Article ID: Q165276
The information in this article applies to: 

  - Microsoft NetShow version 2.0

SYMPTOMS

When you run Nsserver.exe, it fails during the process indicating the services could not be registered.

NOTE: This usually occurs during the Channel Manager service. However, it has also been seen with the Unicast service.

WORKAROUND

Change the suggested path C:\Program Files\Microsoft NetShow to something similar to C:\NetShow2.

This is currently under investigation. Any information including the File System, Server history, or other configuration reports or observations can be sent to nsreport@microsoft.com.
