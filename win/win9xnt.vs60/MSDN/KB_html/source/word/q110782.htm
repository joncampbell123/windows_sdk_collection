Foreign Language Proofing Tools Error Message During Setup

 Last reviewed: February 6, 1998
Article ID: Q110782
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

You may receive the following error message when you try to set up Word 6.0 for Windows if you have an older version of the Alki foreign language proofing tools: 

   Setup detected older foreign language proofing tools that are
   incompatible with Microsoft Word 6.0. To upgrade these tools,
   please call Microsoft Infromation Center or your local subsidiary.
   Choose Cancel to Exit Setup.

CAUSE

The foreign language proofing tools from Alki software are not compatible with Word 6.0 for Windows proofing tools files.

RESOLUTION

The above error message does not prevent Setup from continuing. To continue with Setup, choose OK.

MORE INFORMATION

Alki Software distributes foreign language proofing tools and dictionaries for Microsoft. Alki does not develop the proofing tools or the dictionary files; they are developed by Microsoft and distributed through Alki Software. Alki Software develops and ships the setup routines that are included with the foreign language dictionaries.
