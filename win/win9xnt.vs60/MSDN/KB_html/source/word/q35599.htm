WD: Err Msg "Not a Valid Action for Footnotes"

 Last reviewed: February 2, 1998
Article ID: Q35599
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 5.1a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Microsoft Word, if you edit a footnote, the following error message may appear: 

   Not a valid action for footnotes

CAUSE

This message usually results from trying to delete a paragraph mark that separates footnotes. You can format this mark, but you cannot delete it.

This message also appears if you try to delete the footnote number in the footnote window.

WORKAROUND

If you want to delete the footnote, delete the footnote reference number that appears in the text of your document.
