WD: Outline Numbered "Number Format" Wrong or Changes to Square

 Last reviewed: February 18, 1998
Article ID: Q176454
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you customize an Outline Numbered style, one or both of the following may occur.

Case 1

When you select a level and then click to select a Previous level number, the previous level number is added to the right of the Number format.

Case 2

When you move the insertion point in the Number format box to the left of the number format, and then click to select a previous level number, the numbering in the Number format box changes to a square.

WORKAROUND

In an Outline Numbered style, when you want to include the number from a previous level as part of the numbering scheme, follow these steps:

  1. On the Format menu, click Bullets And Numbering.

  2. Click to select the Outline Numbered tab.

  3. Click to select the numbering style you want to change, and then click Customize.

  4. Under Level, click to select the level you want to change. For example, click to select Level 2.

  5. Click the Number format box, and then move the insertion point to the left of the number format (the left-most position in the Number format box).

  6. In the Previous level number box, click to select the previous level. For example, click to select Level 1.

  7. Make any other changes you want to the Outline Numbered List, and then click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To reset a numbering style to its default, follow these steps:

  1. On the Format menu, click Bullets And Numbering.

  2. On either the Numbered or Outline Numbered tab, click to select the numbering style you want to reset, and then click Reset.

    When the following message is shown 

          Do you want to reset this gallery position to the default setting?

    click Yes.

For more information about how to customize outline numbering, click the Office Assistant, type "outline numbered," click Search, and then click to view "Modify outline numbering formats."
