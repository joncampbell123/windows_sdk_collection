WD: Page Number Incorrect in Header/Footer and Page View

 Last reviewed: January 30, 1998
Article ID: Q23699
The information in this article applies to:
  • Microsoft Word for the Macintosh versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1

SUMMARY

If the page number in the header or footer of a Microsoft Word for the Macintosh document does not appear to correspond with the actual page number of the displayed page, check the following things:

  1. In Show Paragraph mode (View in 5.0 and later or the Edit menu in prior versions), you should see a dotted box around the page number in the header or footer. If you do not, the page number was not created by clicking the page number icon and therefore will not update when your pagination changes. In this case, select the number, then click the page number icon.

  2. The page number displayed when you click the page number icon in the Open Header window corresponds to the page number of the first line in the main document window. You may be confusing the page number with the wrong page.

  3. The document should be repaginated to ensure that the page number displayed in the header or footer is correct. From the Tools (Document in versions prior to 5.0) menu, choose Repaginate Now (Repaginate in Word versions 3.0x).

  4. There is hidden text in the document, the Show Hidden Text option is selected in the Preferences dialog box, and the Print Hidden Text option is not selected in the Print dialog box. Word includes the hidden text in the pagination of the document and in Page Preview but not in the pagination of the printed version because it does not contain the hidden text. To work around this problem, cancel Show Hidden Text in the Preferences dialog box on the Tools (Edit in versions prior to 5.0) menu before previewing or printing the document. In Word 5.0, click on the View icon in the Preferences dialog box to display the check box for hidden text.

  5. In Word version 4.0 only, if you press TAB in front of a page number while in Page View, the wrong page number is displayed. Word will, however, print the correct page number. Additionally, if you scroll the header or footer off the screen and then scroll back, the page number is displayed incorrectly.
