WD98: Table Column Width Changes When Opened As HTML

 Last reviewed: March 16, 1998
Article ID: Q182468
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

Table column widths vary with the window size when you open an HTML document containing a table in Microsoft Word.

CAUSE

The default view for HTML documents in Word 98 Macintosh Edition is online layout view. Online layout view automatically sizes the column widths of tables to adjust for the window size.

WORKAROUND

To work around this behavior, view the document in normal or page layout view (click Normal or Page Layout on the View menu).
