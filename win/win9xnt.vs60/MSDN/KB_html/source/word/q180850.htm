WD98: What Is Stored in the Word Settings (8) File

 Last reviewed: February 12, 1998
Article ID: Q180850
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

The Word Settings (8) file contains the following Word for the Macintosh preference and option settings:

  1. Tools Preferences dialog box. The following table lists each tab in the Tools Preferences dialog box along with the settings for each tab that Word stores in the Word Settings (8) file: 

       Tab                   Settings Stored in Word Settings (8)
   ---                   ------------------------------------

   View                  Show, Window, Nonprinting Characters.

   General               General Options.

   Edit                  Editing Options.

   Print                 Printing Options, Include With Document.

   Track Changes         Inserted Text, Deleted Text, Revised Lines.

   User Info             Changes To Name, Initials, Mailing Address.

   Compatibility         NOTE: Options are saved only when you click
                         Default.

   File Locations        All.

   Save                  Save Options. File Sharing Options.

   Spelling & Grammar    Suggest, Ignore, Custom Dictionaries, Language,
                         Check Spelling, Show Readability.

   NOTE: The unlisted settings are not stored in the Word Settings (8)
   file.

  2. Tools AutoCorrect dialog box. The following table lists each tab in the Tools AutoCorrect dialog box along with the settings for each tab that Word stores in the Word Settings (8) file: 

       Tab                   Settings Stored in Word Settings (8)
   ---                   ------------------------------------

   AutoCorrect           Replace, Exceptions.

   AutoFormat As You
   Type                  Apply Headings, Borders, Automatic Lists, Replace.

   AutoText              Show Auto Complete tip, Show Toolbar.

   AutoFormat            Preserve, Apply Styles To, Adjust, Replace.

  3. "Tip of the Day" option (on or off).

  4. Location of the toolbars on screen (position on screen, docked or floating).

  5. Recently Used File list (MRU).

  6. Recently Used Font list (MRU).

  7. Table of authority categories.

  8. Tools Advanced Settings command options. (This command is not on the Word menus by default). Run the ToolsAdvancedSettings macro to open the Advanced Settings dialog box by following these steps:

    a. On the Tools menu, point to Macro, and click Macros.

    b. In the Macros In list, click Word Commands for a list of macro names.

    c. Select ToolsAdvancedSettings from the list, and then click Run.

    d. Select Startup Options, and press OK when finished.

MORE INFORMATION

NOTE: You cannot copy the Word Settings file between two computers, even if those computers have the same exact hard disk names and Word Setup parameters.

For more information about previous versions of Word Settings files and their interaction with Word 98, please see the following article in the Microsoft Knowledge Base:
