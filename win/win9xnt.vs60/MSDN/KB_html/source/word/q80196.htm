Word: Documents Modified by EndNote Plus Cause Hang

 Last reviewed: July 30, 1997
Article ID: Q80196
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SYMPTOMS

Adding a bibliography to a Microsoft Word for the Macintosh version 5.0 document using the stand-alone version of EndNote Plus version 1.0 causes Word to stop when it attempts to open the updated file.

Note: The EndNote Plus Plug-in Module (PIM) for Word 5.0 works correctly.

WORKAROUND

To open a file formatted by EndNote Plus 1.0:

  1. Start Microsoft Word for the Macintosh version 4.0.

  2. Open the file modified by EndNote Plus.

  3. From the File menu, choose Save As. Click the Save button.

  4. When prompted to "Replace existing...", click the Replace button.

  5. Quit Word 4.0.

  6. Start Word 5.0. From the File menu, choose Open and open the file.

MORE INFORMATION

EndNote Plus is an application that allows you to create bibliographies in a variety of Macintosh word processing applications, such as Microsoft Word.

With EndNote, you can add a bibliography to a document that contains EndNote references without opening the document in the word processor that was used to create it. The entire operation takes place within the EndNote application. EndNote compiles the bibliography from the EndNote references in the document and appends it to the selected file. It then saves the document in its original file format to a new name. However, there seems to be a problem in the way EndNote saves to Word's file format. If you try to open this file in Word 5.0, your program will stop.

Steps to Reproduce Problem

To create a bibliography using EndNote, do the following:

  1. From the Paper menu, choose Open Paper. Locate your file and click Open. A list of citations from that file will be presented.

  2. From the Style menu, choose the appropriate bibliographic style.

  3. From the Paper menu, choose Format. In the dialog box, enter a name for the new formatted file and click Save.

If you try to open this file in Word 5.0, Word will stop.

STATUS

The makers of EndNote Plus, Niles and Associates, Inc., are aware of this problem and will post new information as it becomes available.

Current versions of EndNote products are: EndNote 1.6.2; EndNote Plus, 1.3.2; EndNote PIM 1.3.
