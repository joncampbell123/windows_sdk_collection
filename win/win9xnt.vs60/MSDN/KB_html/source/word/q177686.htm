WD98: DocMaximize Does Not Appear to Maximize Document Window

 Last reviewed: February 17, 1998
Article ID: Q177686
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you run the DocMaximize command, the document is not resized to take up all of the available screen area.

CAUSE

By design, the document window maximizes to the "best" size (based on margins, and avoiding the hard disk and Trash icons).

WORKAROUND

Manually resize the document window to fit the entire screen. When the DocMaximize command is run, it will then toggle between normal window size and the document size you set manually.
