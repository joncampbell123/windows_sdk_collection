WD:Inserted Document Changes Section Formatting to Host Document

 Last reviewed: February 5, 1998
Article ID: Q132268
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

If you insert a document that is already open into another document (by clicking File on the Insert menu), the inserted document inherits the section formatting of the other document. This problem occurs even if you choose not to link the source document.

For example, If TEST.doc is open in Word and is set to landscape orientation and you open a new blank document (which has portrait orientation) and insert TEST.doc into the new document, TEST.doc now has portrait orientation.

WORKAROUND

To work around this problem, close the source document before you insert it into another document.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Word 97 for Windows and Word 98 Macintosh Edition.
