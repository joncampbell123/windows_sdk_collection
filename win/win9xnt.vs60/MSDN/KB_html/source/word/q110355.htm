Boxes Print Instead of Smart Quotes or Publishing Characters

 Last reviewed: February 5, 1998
Article ID: Q110355
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

Boxes may print instead of typesetters' (or "curly" or "smart quotes") quotation marks or other publishing characters when you use a nonscalable font such as Universe or CG Times. The publishing characters will display correctly on the screen as well as in print preview.

Note: If Smart Quotes display as black boxes when using a TrueType font, Draft Font has been selected from the Options menu by selecting the View tab.

CAUSE

The printer driver has a flag set to 0 (zero) for Publishing Characters, and therefore any publishing character will not print.

The printer driver that may cause this problem is: 

   Driver            Version
   ------            -------

   HPPCL5A.DRV           2.1

WORKAROUNDS

Method 1: Use a TrueType or ATM font.

These fonts have smart quotes, so they will display and print correctly.

Method 2: Use a printer heuristic to simulate publishing characters.

In the WINWORD6.INI file, add the following line: 

   fAvoidPubCharEsc = +16384

Method 3: Use the SetPrintFlags macro to set the printer heurisitic

Use the following procedure to run the SetPrintFlags macro:

  1. From the File menu, choose Open.

  2. From the List Files Of Type list, select Document Templates.

  3. From the Directories list, select the <WINWORD>\MACROS subdirectory.

  4. From the File Name list, select MACRO60.DOT and then choose OK.

  5. From the File menu, choose Macros.

  6. From the list of macros, select SetPrintFlags and then choose Run.

  7. Select "Simulate Smart Quotes and em dashes in non-scaled fonts" to turn the option on.

  8. Choose OK.

In methods 2 and 3, Word will not use the smart quote characters for this printer's built in fonts. Instead, Word will simulate smart quotes by sending two right/left facing single quotation marks, spacing them closely.

Note: Methods 2 and 3 may not work. The font may be missing right- and left-facing single quotation marks as well.
