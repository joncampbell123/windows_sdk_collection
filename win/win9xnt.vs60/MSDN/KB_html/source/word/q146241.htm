Custom Properties Not Saved with Document Protection Password

 Last reviewed: February 5, 1998
Article ID: Q146241
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you save a document with a protection password, settings added to the Custom tab on the Properties dialog box are lost.

STATUS

This problem was corrected in Word 97.

WORKAROUND

To work around this problem:

  • Save the document without a protection password.

    -or-

  • Save the document with a write reservation password instead.

MORE INFORMATION

For more information on different types of passwords, click Answer Wizard on the Help menu. In the first box, type "how do I protect a document," then click the Search button. Select the topic titled "Protect a document" and click the Display button.
