GP Fault Editing Document with Dell S3 Video Driver

 Last reviewed: February 5, 1998
Article ID: Q105313
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

A general protection (GP) fault error sometimes occurs while you edit a Word for Windows document when you use the Dell S3 video driver, which is supplied with some Dell computers, such as the Dell 466/ME.

CAUSE

Microsoft has reproduced this problem using the Dell S3 video driver version 3.1.3a (S3645K.DRV) in the 800 x 600, 256-color (512-kilobyte Small Fonts) mode. The problem frequently occurs when Word displays a line of text that contains a large gap. For example, if you type "a b" and then apply Justified paragraph alignment formatting, the GP fault error occurs.

The problem does not occur in Word 6.0 when you use version 3.1.32 or 3.1.4 of the Dell S3 driver. To obtain this updated version of the video driver, contact Dell Computer Systems at the following phone numbers: 

   Dell Technical Support: (800) 624-9896
   Dell Bulletin Board Service (BBS): (512) 728-8528

WORKAROUND

To work around this problem, use the standard VGA video driver supplied with Windows.
