WD: Delivery Point Barcode Option Is Unavailable

 Last reviewed: February 5, 1998
Article ID: Q164032
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When creating a new label, the Delivery Point Barcode check box is unavailable and you cannot add the barcode to the desired label.

CAUSE

Word will not allow the addition of the delivery point barcode if the field will not fit onto the selected label.

WORKAROUND

Method 1: Use a Different Label

Select and use a label large enough to allow Word to add the Delivery Point Barcode field to the label.

Method 2: Manually add the barcode field

Complete a new label document using the desired label size. Then add the barcode field in the desired location on each of the labels.

NOTE: Because Word detected that the label was too small to add the barcode field, you may have to modify the label layout and formatting to get the entire barcode onto the label. You still may not be able to fit the entire barcode onto the label. If this is the case, then you will have to use Method 1.

For information about how to do this Word 97, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q106012
   TITLE     : How to Add a Postal Barcode to Labels or Documents

MORE INFORMATION

To add a bar code to a label:

  1. On the Tools menu, click Envelopes And Labels.

  2. Click the Labels tab.

  3. Click the Options button, select the desired label product number, and then click OK.

  4. Type in the desired address.

  5. Click the Delivery Point Barcode check box (if available), and then click either Print or New Document.

Microsoft Word will allow you to add a Delivery Point Barcode to a label if the label size is sufficiently large enough to accommodate the barcode.
