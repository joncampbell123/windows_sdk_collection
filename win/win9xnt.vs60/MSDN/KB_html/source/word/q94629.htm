Some Toolbar Buttons Do Not Display on Toolbar

 Last reviewed: July 30, 1997
Article ID: Q94629
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, and 2.0c

SUMMARY

Although you can add up to 30 buttons to the toolbar in Word for Windows, only 26 buttons display if you use the standard VGA video driver supplied with the Microsoft Windows operating system. The four buttons on the right end of the toolbar do not display. All 30 toolbar buttons display if you use a Super VGA (SVGA) video driver that supports 1024 by 768 resolution.

MORE INFORMATION

You cannot resize the toolbar in Word for Windows, so you cannot access the undisplayed toolbar buttons. No error message notifies you of undisplayed buttons.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, pages 724-725
