Right (Secondary) Mouse Button Fails in Word with Pen Windows

 Last reviewed: February 5, 1998
Article ID: Q122234
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

Clicking the right (secondary) mouse or pen button in the document window area of Microsoft Word for Windows does not open a context-sensitive menu.

CAUSE

Word does not recognize right (secondary) button clicks when pen extensions are installed.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word versions 6.0, 6.0a, and 6.0c for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

If you are going to be using the mouse for an extended period of time, disable the pen extensions to restore the functionality of the right mouse button.
