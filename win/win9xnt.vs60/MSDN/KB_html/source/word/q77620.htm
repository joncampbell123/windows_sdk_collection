WD: Cannot Insert Nested Formula with Insert Field Command

 Last reviewed: February 2, 1998
Article ID: Q77620
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SUMMARY

The Insert Field command does not allow recursion or the nesting of multiple instructions or switches with the Options list box in Microsoft Word. Additional instructions or switches must be nested manually. If you use the Insert Field command and choose to nest instructions using the Options list box and the Add button, the instructions are not nested.

REFERENCES

"Microsoft Word for Windows User's Reference," pages 88-119
