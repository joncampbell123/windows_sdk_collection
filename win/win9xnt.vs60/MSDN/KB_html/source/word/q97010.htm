How to Print Merge Records that Start with a Certain Letter

 Last reviewed: February 5, 1998
Article ID: Q97010
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a

SUMMARY

When you perform a print merge operation, you cannot use wild card characters as record selection comparison criteria. You can, however, achieve similar results by selecting records that contain fields that start with a particular letter or group of letters. For example, to mail letters only to those clients whose last names start with the letter "S", set up the following two rules in Record Selection: 

   LastName is greater than or equal to S
   LastName is less than T
To find those records in which the last name starts with Z, use only one rule: 

   LastName is greater than or equal to Z
