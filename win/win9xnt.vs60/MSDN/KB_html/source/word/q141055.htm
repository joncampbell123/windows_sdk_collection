WD5X: "An Error Occurred While Installing..."

 Last reviewed: February 6, 1998
Article ID: Q141055
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a

SYMPTOMS

While the Installer is accessing the first installation disk for Word for the Macintosh 5.x, you may receive the following error message: 

   An error occurred while installing on the active startup disk
   <hard disk name>. Installation was not able to be completed,
   but your original software has been restored."

CAUSE

The Embedding Preferences file is damaged.

RESOLUTION

Use the following steps to successfully run the Installer:

  1. Remove the Embedding Preferences file. This file is located in the System:Preferences folder.

  2. Re-run the Installer.

The installer program will create a new copy of the Embedding Preferences file.

MORE INFORMATION

If the above error occurs after you insert another disk, this usually indicates the disk is damaged. To test the disk, copy the disk to the hard drive. If the disk is damaged, an error message will result.

To obtain a replacement disk set, call the Microsoft Order Desk at (800) 360-7561 and request part number 034-096-S39. If you are outside the United States, contact your local subsidiary. To locate your subsidiary, see the Microsoft World Wide Offices Web site at:
