Internet Assistant: Form Drop-Down Field Covers Check Box

 Last reviewed: February 5, 1998
Article ID: Q129069
The information in this article applies to:
  • Microsoft Internet Assistant versions 1.0, 1.0z, 2.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

When an HTML document has a Form Drop Down field immediately in front of a Check Box field, the selected drop down list will hide the check box in the browse mode of Internet Assistant. This behavior does not occur in other browsers, such as InternetWorks, Mosaic, or Netscape.

STATUS

Microsoft has confirmed this to be a problem in Internet Assistant version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
