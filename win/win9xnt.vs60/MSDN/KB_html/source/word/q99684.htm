Documentation Error on Page 201 of "Using WordBasic"

 Last reviewed: July 30, 1997
Article ID: Q99684
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

Line 13 on page 201 of "Using WordBasic" contains the following documentation error: 

   [.To=text,].PrintToFile=number,.Collate=number [,.FileName=text]
This line should read as follows: 

   [.To=text,].PrintToFile=number,.Collate=number [.FileName=text]
There should be no comma (,) before FileName.

MORE INFORMATION

If you type the syntax as shown in the book, you will receive the following error message: 

   WordBasic error 124, undefined sub or function will result if the
   argument ,.FileName=Text is typed.
