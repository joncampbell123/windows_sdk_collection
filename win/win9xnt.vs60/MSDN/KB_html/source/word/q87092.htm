PostScript: Second Page of Mixed-Bin Print Job is Distorted

 Last reviewed: July 30, 1997
Article ID: Q87092
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0 and 2.0a

SUMMARY

If you print a mixed-bin document to some PostScript printers, the page following the change of bins may print upside down, mirrored, and four times its original size.

Microsoft has confirmed this to be a problem with the PSCRIPT.DRV PostScript printer driver version 3.5, which ships with Microsoft Windows operating system version 3.1.

This problem was corrected in PSCRIPT.DRV version 3.53 and later versions. The latest PostScript driver is available in the Windows Drivers Library (WDL). For more information on the WDL, query on the following words in the Microsoft Knowledge Base:
