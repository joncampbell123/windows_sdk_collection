"Not a Valid Print Range" Error Printing from Word 6.0

 Last reviewed: February 5, 1998
Article ID: Q113054
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

When you print a single page of a document using the Print Range command in Word 6.0 for Windows, you get the error message "This is not a Valid Print Range."

CAUSE

The list separator box is blank in the Windows Control Panel International setting.

RESOLUTION

Change the List Separator setting to a comma by following these steps:

  1. In the Main group, choose the Windows Control Panel icon.

  2. In the Control Panel window, choose the International icon.

  3. In the List Separator box, type a comma.

  4. Choose OK.

The Print Range option in Word 6.0 will now work correctly.
