WD: Link to Contents Option Not Available in Open Dialog Box

 Last reviewed: February 5, 1998
Article ID: Q134897
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word 97 for Windows

SYMPTOMS

The Link To Contents option is not available when you view the Property sheet from the Open dialog box. Additionally, the icon beside any linked properties will appear as a broken chain link.

CAUSE

In Word documents, properties that are linked to document content use bookmarks as the source. Word must open documents to resolve bookmark contents. However, the Open dialog box does not always open the document to display previews and properties content. Word uses the thumbnail preview (if available) and the properties information contained in the document file header rather than opening the document. Because the document itself has not been opened, Word cannot determine whether the link s valid.

RESOLUTION

To edit document properties that are linked to document content:

  1. Open the document.

  2. Create bookmarks for the appropriate text if they do not already exist.

  3. On the File menu, click Properties.

  4. Click the Custom tab.

MORE INFORMATION

For more information about using document properties or bookmarks, click Answer Wizard on the Help menu, and search one of the following topics: 

   Set File Properties
   Link custom properties to content in an active file
   Add a bookmark
