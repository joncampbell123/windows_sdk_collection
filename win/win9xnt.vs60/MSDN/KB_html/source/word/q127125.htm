WD6: Internet Assistant, "Cannot Open Help File" Using Help in

 Last reviewed: February 5, 1998
Article ID: Q127125
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0

SYMPTOMS

You get the following error message when you press F1 to access Help or when you choose Index from the Help menu, after opening an FTP URL: 

   Windows Help: Cannot Open Help File

CAUSE

Help is not currently available in InternetWorks or while in an FTP URL. This feature is under review and will be considered for inclusion in a future release.

STATUS

Microsoft has confirmed this to be a problem in Internet Assistant for Word for Windows versions 1.0, and 1.0z. This problem was corrected in Internet Assistant version 2.0 for Word for Windows 95.
