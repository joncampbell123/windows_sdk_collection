WD6X: Fonts, Commands Not Available on the Font Menu

 Last reviewed: November 17, 1997
Article ID: Q153834
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When you add the Font menu to the menu bar using the 5.1 upgrade template, the following items are no longer available on the Font menu: 

   Font Sizes
   Up
   Down
   Other
   Default Font
   (approximately the first 19 fonts)
This problem does not affect the Font or Font Size lists on the Formatting toolbar.

CAUSE

This problem occurs when the following conditions are true:

  • You are using System 7.5.x and Adobe Type Reunion, version 1.2. 

        -and-
  • You changed the font at least once during the current Word session.

RESOLUTION

To work around this problem, use either of the following methods.

  • Turn off the Adobe Type Reunion extension using the Extension Manager. 

        -or-
  • Remove the Font menu and use the Font and Font size list boxes on the Formatting toolbar.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
