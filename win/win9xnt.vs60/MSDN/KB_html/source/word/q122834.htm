Duplicate Menu Items Converting Word 5.x Commands to Word 6.0

 Last reviewed: November 17, 1997
Article ID: Q122834
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

If you open your Word Settings (5) file in Word 6.0 for the Macintosh and save the settings as a new Word 6.0 template [named Word Settings (5) Template], some of your Word 6.0 menus may contain duplicate commands.

For example, the following duplicate commands may appear:

  • On the Window menu, the Document Maximize command appears twice.
  • The Header And Footer command appears three times on the View menu.
  • The Split command appears on both the View menu and the Window menu.
  • The Symbol command appears four times on the Insert menu.
  • On the Format menu:

    The Left Borders command appears six times. The Box Borders command appears twice. The Change Case command appears twice. The Insert Cells and Delete Cells commands appear twice.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for the Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

The duplicate commands will all function correctly. To remove them, choose Customize from the Tools menu, and select the Menus tab.

For more information about customizing menus, double-click the Help button on the Standard toolbar and type "customize" (without the quotation marks).
