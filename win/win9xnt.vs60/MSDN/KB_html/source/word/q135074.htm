Some Button Tooltips Don't Display Shortcut Keys

 Last reviewed: February 6, 1998
Article ID: Q135074
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SUMMARY

Some button ToolTips don't display the corresponding shortcut key next to the ToolTip. (To activate this option, click Toolbars on the View menu, and then click to select the With Shortcut Keys check box.)

Some buttons activate commands which do not have shortcut key assignments; therefore, Word cannot display a shortcut key.

MORE INFORMATION

If you assign a shortcut key to a button which currently doesn't have one, Word will display the shortcut key next to the button name.

The following toolbar buttons do not have corresponding shortcut keys.

Standard Toolbar

   New
   Print
   Insert Address
   Insert Table
   Insert Microsoft Excel Worksheet
   Columns
   Drawing
   Zoom Control
   TipWizard

Formatting Toolbar

   Highlight
   Numbering
   Bullets
   Increase Indent
   Decrease Indent
   Borders

Microsoft Toolbar

   all buttons

Borders Toolbar

   all buttons

Forms Toolbar

   all buttons

Database Toolbar

   Data Form
   Manage Fields
   Add New Record
   Delete Record
   Sort Ascending
   Sort Descending
   Insert Database
   Find Record
   Mail Merge Main Document

Drawing Toolbar

   all buttons

Word for Windows 2.0

   Numbering
   Bullets
