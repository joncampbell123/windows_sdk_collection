Microsoft Draw Disk and RAM Memory-Related Error Messages

 Last reviewed: July 30, 1997
Article ID: Q79876
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

If you have a version of the Atech Publisher's Powerpak earlier than 1.52 loaded on your computer, Microsoft Word for Windows cannot print. If the Publisher's Powerpak driver is selected in the Print Setup dialog box, the Print and Print Preview commands are dimmed (unavailable) on the File menu. The Create Envelope button on the tool bar is also unavailable.

Note: Atech Software is no longer doing business. No product support is available for Publisher's Powerpak.
