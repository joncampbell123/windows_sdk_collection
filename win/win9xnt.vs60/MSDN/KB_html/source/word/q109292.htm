WD6X: Grammar Readability Statistics Dialog Shown Improperly

 Last reviewed: February 5, 1998
Article ID: Q109292
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

The Word grammar checker readability statistics dialog incorrectly is displayed in a blank document, under certain conditions when it should not be displayed at all.

If you open a new document in Word for Windows and run the grammar checker with no text in the document, a dialog box is displayed that says: 

   Grammar check complete
If you add text to the document by copying and pasting (or by clicking File on the Insert menu), delete the text, and then run the grammar checker again, the readability statistics dialog box appears when it should not.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Word 6.0C for Windows and Word 6.0.1 for Macintosh
