WD97: Errors Starting Word After Installing EasyPhoto

 Last reviewed: January 2, 1998
Article ID: Q178732
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you start Word, you see one of the following error messages: 

   Word could not start this add-in program:
   <drive>:\<path>\Ezpwll.wll

   -or-

   Word could not start this add-in program:
   <drive>:\<path>\Ezpwll16.wll

   -or-

   Word could not start this add-in program:
   <drive>:\<path>\Ezpwll32.wll

   -or-

   Winword caused an Invalid Page fault in Module Ezphoto.wll at
   <<address>>.

   -or-

   Word could not load this add-in program.
   <drive>:\<path>\Ezphoto.wll
NOTE: These error messages may occur after you install EasyPhoto and then try to start Microsoft Word 97 for Windows.

CAUSE

The Word add-ins installed by EasyPhoto version 2.7 and earlier were designed to work with Microsoft Word for Windows 95, version 7.0, and earlier.

RESOLUTION

To correct this problem, please contact Storm Technology to determine the availability of an update to EasyPhoto to work with Microsoft Word 97 for Windows.

WORKAROUND

To work around this error, remove the EasyPhoto add-ins from the Word 97 Startup folder.

  1. Quit all Windows applications.

  2. Right-click the Windows Start button, and then click Explore on the shortcut menu.

  3. In Windows Explorer, go to the Office Startup folder.

    This folder is usually located at the following location: 

          C:\Program Files\Microsoft Office\Office\Startup

  4. If any of the following files are located in the Startup folder, move them to the Windows desktop or to the Ezphoto folder: 

          Ezphoto.wll
      Ezpwll.wll
      Ezpwll16.wll
      Ezpwll32.wll

Microsoft Word 97 for Windows should now start without any of the error messages listed in the "Symptoms" section of this article. EasyPhoto will still work correctly when you drag or paste photos onto a Word document.

STATUS

This problem occurs when you use Microsoft Word 97 for Windows and EasyPhoto.

MORE INFORMATION

Ezphoto.wll, Ezpwll.wll, Ezpwll16.wll and Ezpwll32.wll are add-in files designed to enhance the integration of EasyPhoto with Word version 7.0 and earlier. The purpose of the .wll add-in files are to put a button on the Word Standard toolbar to start EasyPhoto from Word.

NOTE: These add-in files have not been updated for Microsoft Word 97 for Windows.
