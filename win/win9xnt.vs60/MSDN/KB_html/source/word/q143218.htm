IA: InsertPicture Dialog Box Defaults to "All Files"

 Last reviewed: February 5, 1998
Article ID: Q143218
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 2.0z
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

By default, the Insert Picture dialog box allows you to select any graphic file format (to access this dialog box, click Backgrounds And Links on the Format menu).

In contrast, when you open the Insert Picture dialog box by selecting the Picture command on the Insert menu, the default graphics format is GIF.

WORKAROUND

Although you can select a picture in any graphics format, you should select only JPG or GIF images when you create HTML documents (most World Wide Web browsers support these two formats).

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Assistant for Word, version 2.0z. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
