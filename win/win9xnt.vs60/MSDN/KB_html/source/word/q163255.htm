WD: Unexpected Memory Problems with Word 97 and Outlook 97

 Last reviewed: February 4, 1998
Article ID: Q163255
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Outlook 97

SYMPTOMS

When you use Microsoft Word 97 and Outlook 97 for Windows, you may receive one or more of the following error messages or behaviors:

In Word

  • Can't find Winword, using Outlook editor instead.
  • There is insufficient memory or disk space. Word cannot display the requested font.
  • Your computer beeps or does not respond, or you receive an unexpected error in Microsoft Word 97.
  • This program has caused an illegal operation and will be shut down.

In Outlook

  • The message "Please wait while Microsoft Outlook exits" is not displayed when you quit Microsoft Outlook.
  • This program has caused an illegal operation and will be shut down.

CAUSE

This behavior occurs because Word 97 or Outlook processes remain in memory after you quit the programs. The application's server objects may not be closed when the application is exited.

WORKAROUND

To check if an application is still in memory, use press CTRL+ALT+DEL once to view the Close Program dialog box. If Word or Outlook is listed, then that application is still active in memory.

To quit the program remaining in memory, from the Close Program dialog box, click the application, and then click End Task. You may have to perform this step more than once to close both Word 97 and Outlook 97.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
