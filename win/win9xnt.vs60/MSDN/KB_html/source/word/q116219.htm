Main Text May Not Display in Header/Footer View

 Last reviewed: February 5, 1998
Article ID: Q116219
6.00 6.00a 6.00c WINDOWS kbusage buglist6.00 buglist6.00a buglist6.00c fixlist7.00

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you scroll through a document while your insertion point is in a header or footer, text in the main body of the document may not display or may display incorrectly.

This behavior appears to be inconsistent.

CAUSE

There is no problem with the document. The display is not updating correctly.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows versions 6.0, 6.0a, and 6.0c. This problem was corrected in Word version 7.0 for Windows 95.

RESOLUTION

If you are scrolling to go to the next or previous header or footer, use the Show Next or Show Previous Buttons on the Header/Footer toolbar.

If you are scrolling to move to another part of the document, close the Header/Footer view.
