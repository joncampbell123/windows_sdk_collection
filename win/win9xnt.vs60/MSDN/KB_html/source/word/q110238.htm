Gunning Fog Index Not Available in Word 5.1

 Last reviewed: July 30, 1997
Article ID: Q110238
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.1

SYMPTOM

In Word 5.1, after using the Grammar command to check a document for grammatical errors, Word does not display Gunning Fog Index information. This is a change in behavior from Word 5.0, which did display the Gunning Fog Index.

CAUSE

Word 5.1 does not display the information due to legal issues.

MORE INFORMATION

The Gunning Fog Index combines sentence length and the number of multi- syllable words per sentence to give a readability rating. Higher scores indicate more difficult text. The index uses the following algorithm: 

   Total= (Average number of words per sentence) + (Number of words of
          three syllables or more)

   Fog Index= Total x (0.4)
For more information on Gunning Fog Index, query on the following words in the Microsoft Knowledge Base: 

   grammar and gunning
