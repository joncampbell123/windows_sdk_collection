WD: Macintosh Gives Error When Printing to a PostScript Laser

 Last reviewed: February 5, 1998
Article ID: Q105149

The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0

SYMPTOMS

In Word, System Errors or a Bad F-Line Instruction error may occur when you are printing with version 8.0 of the Adobe PSPrinter, Apple LaserWriter, or Hewlett-Packard (HP) LaserJet driver:

This problem may also occur when you switch between the Global Fax TelePort driver and version 8.0 of these drivers.

CAUSE

The error may occur because of a conflict between Global Village's TelePort fax software, the Word Settings file, and the following printer drivers: 

   PSPrinter 8.0 from Adobe
   LaserWriter 8.0 from Apple
   LaserJet 8.0 from HP
All three drivers share the same basic technology.

If the Word Settings file has been irreparably corrupted, you must remove the file and allow Word to create a new settings file.

WORKAROUND

In the long term, the best solution to the problem is to upgrade to a later version of the printer software. In the meantime, either return to an earlier version of the printer driver or turn off the TelePort fax software.

STATUS

The problem does not occur with PSPrinter 8.1 and LaserWriter 8.1.1. HP has released version 8.1.1 of the LaserJet driver that correct this problem.
