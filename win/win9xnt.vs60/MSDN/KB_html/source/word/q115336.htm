CTRL+SHIFT+M Key Combination Does Not Work as Assigned

 Last reviewed: February 5, 1998
Article ID: Q115336
2.00 2.00a 2.00a-CD 2.00b 2.00c 6.00 6.00a 6.00c WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, version 2.0, 2.0a, 2.0b, 2.0c, 6.0, 6.0a
  • Microsoft Windows operating system, version 3.1

SYMPTOMS

If the After Dark screen saver is active in Windows, the CTRL+SHIFT+M key combination will not work inside Word or other Windows applications. The CTRL+SHIFT+M key combination is used as a default for the Unindent command in Word 6.0 for Windows.

CAUSE

This behavior occurs because the CTRL+SHIFT+M key combination is reserved by After Dark for muting the sound of the screen saver.

WORKAROUNDS

Method 1: Close After Dark.

Method 2: Reassign the Unindent (or other) function activated by 

          CTRL+SHIFT+M to another key combination.
