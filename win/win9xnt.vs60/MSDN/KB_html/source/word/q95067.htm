International, Localized Versions of Word for Windows

 Last reviewed: October 8, 1997
Article ID: Q95067
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0, 6.0.1

SUMMARY

Microsoft Word is available in a variety of international versions. Below is a list of the versions available and their status as of July 1996.
