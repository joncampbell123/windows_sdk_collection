WD6X: Word 6.0 for the Macintosh Operating Limitations

 Last reviewed: November 17, 1997
Article ID: Q142752
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

This article provides a list of the operating limitations for Word for the Macintosh, versions 6.0 and 6.0.1. 

Feature                                 Limitation
-------                                 ----------

Document size                           32 million characters

Number of open documents                50

Number of custom toolbars               limited by available memory

Number of custom tools                  limited by available memory

Number of fields per document           limited by available memory

Number of styles per document           4,093

Length of a style name                  253 characters

Number of fonts per document*           32,767

Number of characters per line           768

Font size                               1,637 points (22 inches)

Amount of space between characters      1,637 points
Distance text can be raised 
  or lowered                            1,637 points

Number of words in main dictionary      106,000

Number of custom dictionaries           10

Length of bookmark name                 40 characters

Number of bookmarks per document        32,767
Length of AutoText entry name 
  (including spaces)                    32 characters
Number of AutoText entries per 
  document template                     limited by available memory

Maximum macro size                      limited by available memory

Number of global templates              limited by available memory

Length of macro variable name           40 characters

Number of columns in a table            31

Number of newspaper style columns       45

Number of AutoCorrect entries           limited by available memory
Number of pages in multi-page view 
  (print preview)                       128 (depending on monitor)

Number of numbering levels              9

Number of outline levels                9
Size of master document 
  including subdocuments                32 million characters
Number of subdocuments per 
  master document**                     50

Number of levels of subdocuments        9

Number of colors                        256

Number of files in MRU file list        9

Number of template directories          2

Maximum size of a custom dictionary     32K

 * This number is limited by your System to a much lower number.
** Limit may be affected by available contiguous memory.
