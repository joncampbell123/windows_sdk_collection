WD: Default AutoCaption Label for Equation Uses Figure

 Last reviewed: February 18, 1998
Article ID: Q178175
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Word, when you use the AutoCaption feature to insert a caption for a Microsoft Equation 3.0 object (on the Insert Menu, click Caption, and then click AutoCaption), the Use Label default is Figure instead of Equation.

WORKAROUND

To work around this problem, click Equation in the Use Label list.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
