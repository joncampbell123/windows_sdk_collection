ToolsOptions Commands Unavailable Without an Open Document

 Last reviewed: February 5, 1998
Article ID: Q130570
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

If there is not at least one document open in Word for Windows NT or Word for the Macintosh, the ToolsOptions internal commands will not be available. This behavior is different from the 16-bit version of Word for Windows.

RESOLUTION

Open a document before trying to access the ToolsOptions internal commands.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The following is a list of all the commands that are unavailable when no document is open: 

   ToolsOptions
   ToolsOptionsAutoFormat
   ToolsOptionsCompatibility
   ToolsOptionsEdit
   ToolsOptionsFileLocations
   ToolsOptionsGeneral
   ToolsOptionsGrammar
   ToolsOptionsPrint
   ToolsOptionsRevisions
   ToolsOptionsSave
   ToolsOptionsSpelling
   ToolsOptionsUserInfo
   ToolsOptionsView
