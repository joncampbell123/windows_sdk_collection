WD6: BatchConversion Macro Fails to Convert To/From HTML

 Last reviewed: February 5, 1998
Article ID: Q128506
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

The BatchConversion macro fails to convert your documents to or from HTML format even though the conversion process finishes with a message indicating the conversion is successful.

When you convert multiple HTML files to Word documents using the BatchConversion macro, the documents are empty when opened. When you use this macro to convert Word documents to HTML format, no changes are made to your Word documents.

CAUSE

The BatchConversion Macro is not compatible with the import converter used by Internet Assistant. The macro expects one converter that provides both import and export capabilities, and Internet Assistant uses two separate converters--one for import, and one for export.

RESOLUTION

You must open each file individually and save it to the HTML format.
