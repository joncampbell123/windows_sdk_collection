WD: Can't Assign F14 and F15 Shortcut Keys in MacWord

 Last reviewed: February 5, 1998
Article ID: Q141048
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

You get a system beep or nothing happens when you try to assign commands to the following shortcut keys: 

   SHIFT+F14
   SHIFT+F15
   COMMAND+SHIFT+F14
   COMMAND+SHIFT+F15
   OPTION+SHIFT+F14
   OPTION+SHIFT+F15
After you merge the Word Settings(5) file into Word 6.0, the SHIFT+F14 shortcut key combination appears to be available; however, it does not open the Format Paragraph dialog box.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Use a shortcut key assignment other than those listed above.
