Envelope or Bin Selection Problems with the HP DeskJet 1200C

 Last reviewed: February 5, 1998
Article ID: Q113981
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

When you print a document from Word for Windows that is formatted to select envelopes or paper from different printer bins (or paper trays), the entire document will pull from the first bin (or paper tray) specified.

When you print a document that contains an envelope, the envelope will print correctly, but the rest of the document will not print. 

 This problem occurs when printing to the Hewlett-Packard (HP) DeskJet
1200C printer, using driver (HPPCL5C.DRV) version 2.20.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

To print correctly, you must set a print flag.

To Set a Print Flag

  1. From the File menu, choose Open.

  2. From the Word for Windows program directory, select the MACROS subdirectory.

  3. Under "List Files of Type," select Document Templates (*.dot).

  4. Choose the MACRO60.DOT template and choose OK.

  5. From the Tools menu, choose Macro.

  6. Select the SetPrintFlags macro and choose Run.

  7. Select the "Send Mixed Pages as Multiple Print Jobs" option and choose OK.

  8. Choose OK when the warning appears.

  9. Close the MACRO60.DOT file.

To Configure the Envelope Options to Correctly Print Envelopes

  1. From the Tools menu, choose Envelopes And Labels.

  2. Select the Envelopes tab.

  3. Choose the Options button.

  4. Select the Printing Options tab.

  5. Choose the Reset button to return the feed options to their default values.

  6. Choose the Feed Method icon that matches the way the printer accepts envelopes. For the 1200C, this is the third option from the right (right feed, landscape).

  7. Choose the Face Up button.

  8. From the Feed From text box, select Manual Feed.

  9. Choose OK.

If you send each change in paper source or orientation as a separate print job, the documents and envelopes will now print correctly.
