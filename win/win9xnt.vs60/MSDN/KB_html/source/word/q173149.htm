WD97: Avery Label Wizard Setup Gives Error, Hangs, or Crashes

 Last reviewed: January 23, 1998
Article ID: Q173149
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you run the Avery Label Wizard Setup program, you may experience either of the following symptoms:

- The Setup program hangs (stops responding) at the end of Setup. 

   -or-
  • You receive the following error message: 

          Can't quit at this time.

    In addition, depending on which components were installed during Office Setup, the following error message may be displayed: 

       Winword caused an invalid page fault in module <unknown>.

    CAUSE

    This behavior occurs when all of the following conditions are true:

    • Microsoft Office 97 has been installed.

      -and-

    • Microsoft Word 97 and the Office Assistant components are installed.

      -and-

    • Microsoft Word 97 has never been started.

    MORE INFORMATION

    The Avery products included here are manufactured by Avery Dennison, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.

    The Avery Label Wizard is a third-party add-on for Microsoft Word that is found in the ValuPack on the Word and Office compact disc.

    Once you get through the error messages, the Wizard will operate properly; however, you will not be able to uninstall the Wizard using the Add Remove Programs icon in Control Panel (to access Control Panel, click the Start menu, point to Settings, and click Control Panel).
