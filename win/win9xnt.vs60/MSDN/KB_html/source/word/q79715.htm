Can't Install to External Disk with File Sharing

 Last reviewed: July 30, 1997
Article ID: Q79715
The information in this article applies to:
  • Microsoft Word for the Macintosh versions 5.0, 5.1

SUMMARY

If you install Microsoft Word for the Macintosh version 5.0 from a Startup disk when running Apple System 7.0 with the File Sharing option turned on, the Installer cannot install Word to a hard drive other than the Startup disk. To install Word on a different disk, you must first deactivate the File Sharing option.

This is a limitation of Installer version 3.3, which is licensed from Apple Computer, Incorporated. Apple is aware of this problem.

MORE INFORMATION

To deactivate the File Sharing option, use the following steps:

  1. From the Apple menu, choose Control Panels.

  2. Double-click the Sharing Setup panel.

  3. In the File Sharing area, click the Stop button.

  4. When the prompt "How many minutes until File Sharing is disabled?" appears, enter the number of minutes you want to wait until you disable File Sharing, and press the OK button.

Note: You can select the File Sharing Monitor control panel to see if other users are sharing your computer, or just leave "0," and press the OK button to immediately disable File Sharing in step 4.
