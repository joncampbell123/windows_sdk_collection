Summary Info Page Says Document Consists of Only 1 Page

 Last reviewed: February 6, 1998
Article ID: Q129433
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0c

SYMPTOMS

When you print a document's fil\**

summary page and Background Printing is on, the number of pages shown on the summary page is always 1.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0c for Windows. This problem was corrected in Word version 7.0 for Windows 95.

WORKAROUNDS

Method 1: Disable background printing before printing the summary page: 

          1. From the Tools menu, choose Options.

          2. On the Print tab, clear the Background Printing check box.

          3. Choose OK.

          You can now print the summary page, and it will show the correct
          number of pages.
Method 2. Print the whole document and include the summary page: 

          1. From the Tools menu, choose Options.

          2. Select the Print tab. Under Include With Document, select the
             Summary Info check box.

          3. Choose OK.

          You can now print the document, and the summary page will show
          the correct number of pages.
