WD: WorldFont Foreign Language Fonts by Data-Cal Corporation

 Last reviewed: July 30, 1997
Article ID: Q99051
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

WorldFont for Windows is a font program from DataCal Corporation that provides Windows-compatible international fonts.

WorldFont for Windows also includes the KeyMapper, a Windows keyboard customizer utility program, and Adobe Type Manager.

MORE INFORMATION

WorldFonts for Windows is a high-quality font system that permits you to type and print international languages in any Windows application that uses the Windows font system. The KeyMapper works in conjunction with Windows 3.1. The package consists of three parts: first, the high quality international language fonts in both PostScript Type 1 format and TrueType format, Adobe Type Manager, and the WorldFont KeyMapper. Each of these parts makes using international fonts as easy as typing your favorite Windows application.

System Requirements

The system requirements for running WorldFont for Windows are as follows:

  • MS-DOS or PC-DOS version 3.1 or later
  • Windows 3.1
  • An IBM PC/AT or 100-percent compatible computer with an Intel 286, 386, or 486 processor
  • A Windows-compatible graphics card
  • A Windows-supported or Adobe-Type-Manager-supported matrix, ink jet, or laser printer
