Cell Border Extends Farther Left/Right Than Shading in Table

 Last reviewed: February 5, 1998
Article ID: Q106074
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

If a table contains a border and shading, the border may extend farther than the length of the cell.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed at the beginning of this article. This problem was corrected in Microsoft Word 97 for Windows.

WORKAROUND

Insert a column at the beginning of the shading that is .01-inch wide and has 0 inches of space between columns. Format this column for shading but not with a border. This will cause the shading and the border to be aligned.
