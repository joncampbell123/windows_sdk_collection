WD6X: Can't Choose Envelope With LaserWriter Select 300

 Last reviewed: February 5, 1998
Article ID: Q144888
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

You do not have the choice of selecting an envelope page size in the Page Setup dialog box when you are using the Apple LaserWriter Select 300 printer driver version 1.2. In contrast, the envelope sizes are available in the Finder Page Setup.

WORKAROUND

To work around this problem, use either of the following methods.

Method 1

Use the GX printer driver.

Method 2

Add an envelope to the document from the Envelope tab in the Envelopes and Labels dialog box, and then follow these steps:

  1. Click Page Setup on the File menu.

  2. In the Page Size drop down list, select US Letter and click Print.

  3. Select Manual in the Paper Source list.

  4. Click the Range button.

  5. Select Pages, type "0" (without the quotation marks) in the Pages box, and click OK.

The envelope will print correctly.
