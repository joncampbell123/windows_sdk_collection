IA: Duplicate Entries in the [No Proxy On] Section of IWIA.INI

 Last reviewed: February 5, 1998
Article ID: Q130767
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 1.0, 1.0z
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

Entries in the [No Proxy On] section of the IWIA.INI file are duplicated multiple times. This behavior occurs after you choose OK or Cancel in the Proxy Servers dialog box. (To locate this dialog box, start InternetWorks Internet Assistant. From the Options menu, choose Proxy Servers.)

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Using a text editor such as Notepad, edit the IWIA.INI file and remove duplicate entries in the [No Proxy On] section.
