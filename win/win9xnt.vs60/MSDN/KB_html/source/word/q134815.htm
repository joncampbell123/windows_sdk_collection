Scrap Object Drops into Wrong Table Cell

 Last reviewed: February 6, 1998
Article ID: Q134815
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

A scrap object that contains a table is moved to the cell to the right of the cell into which you are dropping the object.

Note that this behavior does not occur when you drop the object into the right-most cell of the table.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95, version 7.0. This problem has been corrected in Word 97 for Windows.

WORKAROUND

After you drop the object into the table, move the object from the incorrect location to the correct location. You can drag the object to the new location or you can cut the object from one cell and paste it to the other cell.
