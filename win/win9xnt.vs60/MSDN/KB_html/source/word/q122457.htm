Word 6.0c for Windows: Contents of Disk #5 (3.5-Inch DMF Set)

 Last reviewed: August 5, 1997
Article ID: Q122457
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0c

SUMMARY

This article contains the directory listing of Disk 5 for the 3.5-inch high- density (DMF) disk set for Word version 6.0c for Windows. The article includes the decompressed filename for each file, which you must provide when you use the EXTRACT.EXE utility to manually decompress a file from a cabinet file. In addition, this article identifies the location where Setup installs each file.
