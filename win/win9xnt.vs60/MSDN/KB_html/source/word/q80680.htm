WinWord: Print Preview Doesn't Show PostScript Scaling Factor

 Last reviewed: February 6, 1998
Article ID: Q80680
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows, if you select the PostScript driver that is supplied with Microsoft Windows and set the Scaling option to less than 100 percent, print preview mode does not display the scaling; however, Word for Windows does print the scaling correctly.

To illustrate this behavior, use the following steps:

  1. From the File menu, choose Open, select a document, and choose the OK button.

  2. From the File menu, choose Print Setup (Printer Setup in version 1.x).

  3. Select the PostScript driver, and choose the Setup button.

  4. Type "50%" (without the quotation marks) in the Scaling box. Choose the OK button twice.

  5. From the File menu, choose Print Preview. (Notice that print preview does not display the 50 percent scaling.)

  6. From the File menu, choose Print, then choose the OK button.

The printed document reflects the 50 percent scaling.

Word for Windows does not have a function that reflects the scaling set by the printer driver.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, pages 448-449
