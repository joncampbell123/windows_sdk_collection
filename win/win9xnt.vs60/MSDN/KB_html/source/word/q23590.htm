Math Symbols in Formulas Appear Incorrectly

 Last reviewed: November 17, 1997
Article ID: Q23590
The information in this article applies to:
  • Microsoft Word for the Macintosh versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1, 6.0

SYMPTOMS

In Microsoft Word for the Macintosh, math symbols (for example, summation signs or integrals) do not appear correctly on the screen or in the printed document.

CAUSE

Try reinstalling the Symbol font to your System folder.

The Symbol font was included with the System shipped with Word versions 3.0x. If you have Word version 4.0 or later, you must get the font from your System Tools disk. Use the Font/DA Mover to properly

install the font or drag it into the System suitcase under System 7.0.
