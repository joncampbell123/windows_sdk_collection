Custom Page Size Prints Incorrectly on Canon BubbleJet 200

 Last reviewed: February 5, 1998
Article ID: Q129418
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you print a document with a custom page size from Word for Windows to a Canon BubbleJet 200, the document may print too far to the right on the page.

CAUSE

Word uses information from the envelope feed method settings to determine how to print a custom-size document. The default setting for the Canon BubbleJet 200 printer driver versions 1.0 and 1.2 is incorrect.

RESOLUTION

To correct the default envelope feed method settings, follow these steps:

  1. From the Tools menu, choose Envelopes And Labels.

  2. Select the Envelope tab.

  3. Choose the Options button.

  4. Choose the Printing Options tab.

  5. Under Feed Method, select the first picture on left (the envelope is shown as landscape, with a left feed).

    Note: The default setting is second from the left (where the envelope is shown as landscape, with a center feed).

  6. Choose OK.

  7. Choose Cancel to close the Envelopes And Labels dialog box.

Your custom-size document will now print correctly. It is not necessary to change the envelope feed setting to print non-custom-size documents.

MORE INFORMATION

Although the "landscape, left feed" method may appear to be opposite of the way the printer handles custom paper sizes, it is correct. The printer feeds paper from the rear of the printer, but the icon indicates the feed method as if the paper were being fed into the front of the printer.

For more information on how the envelope feed method affects the printing of custom page sizes, query on the following words in the Microsoft Knowledge Base:
