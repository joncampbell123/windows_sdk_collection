INFO: List of Word 6.0 Files Modified by Word 6.0a Patch

 Last reviewed: February 6, 1998
Article ID: Q114006
6.00a WINDOWS 
 kbnetwork
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0a
  • Microsoft Windows operating system version 3.1

SUMMARY

This article contains a list of the Word 6.0 for Windows files that the Word 6.0a Patch modifies. When you update network workstation installations of Word 6.0, you must run the patch on each workstation because some of the files the patch modifies are shared Microsoft application files and Windows system files. You can update only the server if all your workstations share both Windows and Word for Windows.

MORE INFORMATION

The Word 6.0a Patch updates the following files: 

   CBTNORM.DOT         OLE 2PROX.DLL
   COMPOBJ.DLL         PCXIMP.FLT
   EQNEDIT.EXE         PLEADING.WIZ
   EQNEDIT.HLP         SHARERES.DLL
   EQNEDIT2.REG        STORAGE.DLL
   GIFIMP.FLT          TIFIMP.FLT
   IFFPCX.DLL          WDCBT.DEX
   MAPIVIM.DLL         WDREADME.HLP
   MSSPEL2.DLL         WINWORD.EXE
   MSWORD6.CNV         WORDCBT.DLL
   OLE 2.DLL            WORDPSS.HLP
   OLE 2.REG            WORDWIN2.CNV
   OLE 2CONV.DLL        WPFT5.CNV
   OLE 2DISP.DLL        WPTIMP.FLT
   OLE 2NLS.DLL         WRITWIN.CNV
                       WWINTL.DLL
