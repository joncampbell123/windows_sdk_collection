Word for Windows Paste Link with Excel 3.0 Chart

 Last reviewed: July 30, 1997
Article ID: Q71466
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

If you place a Microsoft Excel for Windows version 3.0 chart on a worksheet, you cannot paste link that chart into Word for Windows. To perform a paste link with a chart you have placed on a worksheet in Excel, follow these steps:

  1. Select the chart in Excel.

  2. Click the chart tool on the ribbon. This places a copy of the chart in its own window.

  3. Press SHIFT.

  4. From the Edit menu, choose Copy Picture.

  5. Select either "as shown on screen" or "as printed" and choose OK.

  6. Close the window that was opened for the chart by double-clicking the chart tool.

  7. Place the insertion point where you want the chart to appear in the Word for Windows document.

  8. From the Edit menu, choose Paste Special (Paste Link in version 2.x.)

For more information on paste linking with an Excel version 3.0 chart that is embedded in a worksheet, query on the words: 

   paste and link and chart
