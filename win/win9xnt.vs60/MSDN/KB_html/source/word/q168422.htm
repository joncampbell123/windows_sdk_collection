WD: Batch Conversion Wizard Cannot Convert Templates

 Last reviewed: November 17, 1997
Article ID: Q168422
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

The Word Batch Conversion Wizard will not convert templates or any file with a .dot extension.

CAUSE

By design, you cannot change the file format of a Word template.

WORKAROUND

To convert a Word template to another file format, use the following steps to create a new document from the template and then convert the document to the desired file type:

  1. On the File menu, click New.

  2. Click the template you want.

  3. Click the Document option under Create New.

  4. Click OK.

  5. On the File menu, click Save As.

  6. You can choose to save the document in the desired format, or you can save it as a Word document to be converted later using the Batch Conversion Wizard.
