WD97: Database Command Missing from the Insert Menu

 Last reviewed: February 4, 1998
Article ID: Q156486
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

The Database command is no longer on the Insert menu.

MORE INFORMATION

The Database menu selection has been removed from the Insert menu in Microsoft Word 97 for Windows. Databases are inserted into a Word document, by default, from the Database toolbar.

To activate the Database toolbar:

  1. On the View menu, click Toolbars.

  2. Select Database.

To add the Database selection back to the Insert menu:

  1. On the Tools menu, click Customize.

  2. Select the Commands tab.

  3. Under Categories, select All Commands.

  4. Under Commands, scroll down and select InsertDatabase.

  5. Using the mouse, point to InsertDatabase. Hold the mouse button down
and 
   drag to the Insert menu. The Insert menu will drop-down. Drag the
   pointer to the location on the Insert menu where you want the Database
   selection to be located and release the mouse button.

  • On the Customize dialog, click Close button.
