How to Export Data from Word for Windows to "PowerLeads!"

 Last reviewed: February 5, 1998
Article ID: Q117697

The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-cd, 2.0b, 2.0c, 6.0, 6.0a

SUMMARY

This article describes how to export data from a Microsoft Word data file to "PowerLeads!".

MORE INFORMATION

To export information from a Word for Windows table to PowerLeads!, you must save your information as an ASCII text file. If your data file is set up in a table format, you must first convert the table to text.

  1. In Word, select the table.

  2. From the Table menu, select Convert Table To Text.

  3. Word 2.x for Windows: In the Separate Text With box, select Commas and choose OK.

    Because the fields are now separated by a comma, followed by a space, you must strip out the spaces, or these will be converted as part of the fields. (Note: This extra space does not affect merge results when both the main document and data file are Word documents.) From the Edit menu, choose Replace and replace both the comma and space with a comma.

    Word 6.0 for Windows: In the Separate Text With box, select Other and use a comma to eliminate the extra space. This will eliminate the need to Edit/Replace the comma and space.

  4. Save the document as a text file.

  5. Open PowerLeads! and create a new database.

  6. From the Database menu, choose Import.

  7. In the Import dialog box, select ASCII and choose OK.

  8. In the Open Import File dialog box, select your file.

  9. Match the fields for your import data to the PowerLeads! fields, "Add" these fields to your database, and choose the Completed button when finished.

  10. When PowerLeads! prompts you with, "OK to skip duplicate entries?", choose the Yes button.

When finished, PowerLeads! will generate the message "Import process completed. Please review and update your new records."
