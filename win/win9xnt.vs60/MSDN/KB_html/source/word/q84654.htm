Updated Diamond Stealth Video Drivers Needed for Windows 3.1

 Last reviewed: October 10, 1997
Article ID: Q84654
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

If you run Microsoft Word for Windows in the Microsoft Windows operating system version 3.1 with the Diamond Stealth video driver that ships with Windows versions 3.0 and 3.1, your computer may hang when you save a document in Word for Windows. To correct this problem, you must upgrade the video driver.

WORKAROUND

To work around this problem, use the Windows version 3.1 standard VGA video driver until you receive the updated Diamond Stealth video driver.

MORE INFORMATION

Windows version 3.0 drivers for the Diamond Stealth VRAM display adapters may also cause problems if used with Windows version 3.1. The video driver for this third-party display adapter must be updated to display TrueType fonts correctly. Updated video drivers can be obtained from Diamond Computer Systems or from Microsoft by purchasing the Microsoft Windows Driver Library (WDL) version 3.1.

From the Windows Driver Library (WDL) S3.EXE dated 4-02-92, contains the updated drivers. (The 32K color driver is not yet available.)

You can download the WDL from CompuServe, GEnie, or the Microsoft Download Service at (425) 936-6735 (MSDL).

The Diamond Stealth products included here are manufactured by Diamond Computer Systems, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.

For more information on obtaining the Windows Driver Library, refer to pages 58-59 in the "Getting Started with Microsoft Windows" manual. To obtain updated drivers from Diamond Computer Systems, call Diamond technical support at (408) 736-2000. Your Diamond Computer Systems vendor may also have updated drivers available.

For more information on obtaining updated video drivers, query on the following words in the Microsoft Knowledge Base: 

   MSDL and WW0535 and appnote
