GP Fault in EXCEL.EXE Opening Word Document

 Last reviewed: February 5, 1998
Article ID: Q115864
6.00 6.00a 6.00c 7.00 WINDOWS 
 kb3rdparty
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

A general protection (GP) fault may occur when you open a Word for Windows document that contains a linked Microsoft Excel graph if Delrina WinFax Pro version 4.0 is installed on your computer.

CAUSE

The GP fault occurs if the Microsoft Excel 4.0 macrosheet, named MSEXCEL4.XLM, exists in your Microsoft Excel XLSTART subdirectory. MSEXCEL4.XLM is a file that WinFax Pro version 3.0 installs.

STATUS

Delrina has confirmed this to be a problem with WinFax Pro 4.0. For more information, contact Delrina WinFax Pro 4.0 product support at (416) 443-4390.

STATUS

Delrina recommends that you do the following to avoid the GP fault in Microsoft Excel:

  1. Remove the MSEXCEL4.XLM file from your Microsoft Excel XLSTART subdirectory.

  2. Copy the MSEXCEL5.XLS file from your WinFax Pro 4.0 program directory to your Microsoft Excel XLSTART subdirectory. MSEXCEL5.XLS appears in the WinFax Pro 4.0 program directory if you installed the Microsoft Excel macros when you installed WinFax Pro.
