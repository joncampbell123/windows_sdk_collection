WD: Bookmark in Pageref Field Truncated Exporting to WordPerfect

 Last reviewed: February 18, 1998
Article ID: Q145358
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a and 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows 95, version 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you save a Word document that contains a PageRef field in WordPerfect 5.1 for MS-DOS format, the bookmark in the PageRef field is truncated by one character. This does not cause an error or loss of functionality when you open the document in WordPerfect.

If you reopen the document in Word, the following error message appears in the field result: 

   Error! Bookmark not Defined.

WORKAROUND

If you plan to use the document in WordPerfect, you do not need to modify WordPerfect's Ref field. If you plan to use the document in Word, you need to correct the bookmark name inside the field code. To do this, select the field, press SHIFT+F9 to turn field codes on, type the missing character in the bookmark name, and then press SHIFT+F9 to turn the field codes back off.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

WordPerfect ignores the truncated bookmark name because it doesn't provide support for bookmarks. The export filter stores the page number information inside WordPerfect's Ref field, so that when you open the document in WordPerfect the page number reference is correct.

For Microsoft Word 98 Macintosh Edition, the WordPerfect 5.1x for DOS converter does not install with the Office or Word program. It can be copied from the Value Pack of the CD from the following folder: 

   Office (Word) CD:Valu Pack:Text Converters
It should be copied to the following folder (default): 

   Macintosh HD:Microsoft Office 98:Shared Applications:Text Converters
