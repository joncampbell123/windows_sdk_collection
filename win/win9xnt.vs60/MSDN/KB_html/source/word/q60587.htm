WD: Update Field Key (F9) Updates Only Selected Fields

 Last reviewed: February 23, 1998
Article ID: Q60587
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

In Word, the update field key (F9) operates only on the selected field or fields. Fields outside the selected area are not updated.

The "More Information" section of this article contains a WordBasic macro that you can use to update all fields in the active document.

   Sub MAIN
      ' This macro updates all fields excluding fields
      ' in Headers and Footers.
      EditSelectAll
      UpdateFields
   End Sub
NOTE: You can update the standard UpdateFields macro to use the code from this macro instead. Or, you can assign this macro to the F9 key to force all fields to be updated when you press F9.
