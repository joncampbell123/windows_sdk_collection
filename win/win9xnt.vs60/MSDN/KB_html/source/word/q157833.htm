WD97: "PAGE \#..." Displays in Screen Tip for Comments

 Last reviewed: July 31, 1997
Article ID: Q157833
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When View Field Codes is turned on, the ScreenTip for comments displays the following field code information in addition to the author's name and comment text: 

   PAGE \# "Page: '#'"
ScreenTips for footnotes, endnotes, or track changes do not display this field information.

MORE INFORMATION

This field code is inserted automatically by Word before each comment in the comment pane. If you choose to print comments when you print the document, the page number is printed with the comment to identify it's location in the document.

WORKAROUND

To work around this problem and hide the field code information, turn off View Field Codes.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
