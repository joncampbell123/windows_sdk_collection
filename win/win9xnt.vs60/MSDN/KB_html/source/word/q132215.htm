OPP: Word Hangs with Certain AutoCorrect Entry

 Last reviewed: February 6, 1998
Article ID: Q132215
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Windows 95

SYMPTOMS

When you follow the sequence of steps below, Word hangs (stops responding).

Steps to Reproduce Problem

  1. Close all other programs.

  2. Type ":-" (without the quotation marks).

  3. Turn on Revision marking.

  4. Type ")" (without the quotation marks) to activate AutoCorrect expansion.

Word hangs, instead of changing ":-)" to a smiley face.

STATUS

This is a known problem with the Office Preview Program release only. It will be fixed in the final version of Word for Windows 95.

WORKAROUND

Turn on Revision marks before typing ":-)"
