Word Err Msg: Formatting Too Complex

 Last reviewed: February 2, 1998
Article ID: Q49309
The information in this article applies to:
  • Microsoft Word for the Macintosh versions 4.0, 5.0, 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Microsoft Word for the Macintosh, you may receive the following error message: 

   Formatting too complex
NOTE: This error frequently occurs when tables containing very complex formatting are copied and pasted from one document to another.

CAUSE

This error message will be displayed when the following conditions are true:

  • Text in your document is formatted with a lot of paragraph, character, or section formatting. 

        -and-
  • The Fast Save option is enabled.

RESOLUTION

To prevent this error from occurring:

  1. On the File menu, click Save As, and click Options.

  2. Click to clear the Allow Fast Saves check box.

  3. Click OK.

  4. Close the document.

  5. Reopen the document.

For more information, query the following article in the Microsoft Knowledge Base:
