WD: MacWord Prints Excel Chart Hairlines as 1-Point Lines

 Last reviewed: November 17, 1997
Article ID: Q64006
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1, 6.0

SYMPTOMS

In Microsoft Word for the Macintosh, when you copy a Microsoft Excel for the Macintosh chart into Word, it is not printed properly if it contains hairlines. The hairlines are replaced with thicker 1-point lines.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel for the Macintosh. This problem was corrected in Excel version 5.0 for the Macintosh.
