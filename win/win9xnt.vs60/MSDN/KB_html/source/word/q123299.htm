WD: Suppress Endnotes Option Unavailable

 Last reviewed: February 17, 1998
Article ID: Q123299
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word 97 for Windows

SUMMARY

In Microsoft Word, the Suppress Endnotes option is not available until you designate endnotes to be printed at the end of each section. To access the Suppress Endnotes option, in Microsoft Word for Windows and Word for Windows NT choose Page Setup from the File menu, then select the Layout tab. In Word for the Macintosh, choose Document Layout from the File menu, then select the Layout tab.

MORE INFORMATION

The Suppress Endnotes option prevents endnotes from being printed at the end of a section; however, with this setting selected, the endnotes will print at the end of the document. To select the Suppress Endnotes option, follow these steps:

  1. On the Insert menu, click Footnote.

  2. In the Insert box, select the Endnote option button.

  3. Click Options and select the All Endnotes tab..

  4. In the Place At box, select End Of Section.

  5. Click OK.

  6. Click OK or Close if you do not want to insert a footnote or an endnote.

  7. In Word for Windows and Word for Windows NT, click Page Setup on the File menu, select the Layout tab, and then select the Suppress Endnotes check box.

    In Word for the Macintosh, click Document Layout on the File menu, select the Layout tab, and then select the Suppress Endnotes check box.

Note that default behavior for Word is to print endnotes at the end of the document.
