WD97: WordPerfect Help Topic Incomplete, Truncated

 Last reviewed: February 5, 1998
Article ID: Q157396
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

The last line in the WordPerfect Help topic is truncated. This problem appears when you access the topic by pressing the F4 key when the Help for WordPerfect Users checkbox is selected. (To locate this option, click Options on the Tools menu and then click the General tab.)

Text is missing from the last paragraph in the WordPerfect help topic for the F4 key (CTRL+T and CTRL+M). This problem is seen when you view the topic from the Help for WordPerfect Users dialog box. (To locate this topic, double-click WPH in the status bar. In the Command Keys list, click the Indent-> >> topic.)

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

To work around this problem, and view information about indentation, click the Index tab in Microsoft Word 97 Help, type the following text 

   indentation
and then double-click the selected text to display a list of topics.
