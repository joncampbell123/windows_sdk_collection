File, Send Unavailable Editing Document Routed by cc:Mail

 Last reviewed: August 5, 1997
Article ID: Q129629
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

When you open a Word for Windows document routed by cc:Mail, the Send and Add Routing Slip commands may not be available (dimmed) on the File menu.

RESOLUTION

To correct this problem:

  1. Close the Word document, saving changes if you want.

  2. In cc:Mail, choose User Setup from the Tools menu.

  3. In the Preferences list, choose Message.

  4. In the View/Run window, choose Run Double-Click.

When you double-click a Word document attachment in a cc:Mail message, the attachment will now open directly into Word, and the Send and Routing Slip options will be available on Word's File menu.

MORE INFORMATION

In cc:Mail's default configuration, double-clicking the icon for an attached Word document opens a view window in the mail message that displays the first page of the document as an OLE object. If you double- click the Word object in the view window, the object is opened in Word. When Word is acting as an OLE server, the Send and Add Routing Slip options are not available.
