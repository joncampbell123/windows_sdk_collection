WinWord: Epson Printers Need Tractor Setting to Print Envelope

 Last reviewed: February 5, 1998
Article ID: Q83786
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a

SUMMARY

In Microsoft Word for Windows, if you print envelopes with the Create Envelope command to one of the new Epson printers listed below and use the "gen 150" printer driver from Epson, you must set the paper source to Tractor. If you select any other setting, the envelopes do not print.

MORE INFORMATION

Affected Epson Printers

Epson DFX-5000 Epson L-750/1000 Epson LQ-1070 Epson LQ-1170 Epson LQ-570 Epson LQ-870 Epson LQ-850 Epson SQ-2000 Epson SQ-2500 Epson T-1000 Epson T-750
