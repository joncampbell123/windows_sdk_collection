WD98: Vertical Ruler Contains Black Boxes

 Last reviewed: February 3, 1998
Article ID: Q178711
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you view the vertical ruler, there are black boxes in place of numbers.

To view the vertical ruler, follow these steps:

  1. If you are not in Page Layout view, click Page Layout on the View menu.

  2. On the Tools menu, click Preferences.

  3. Click the View tab.

  4. Click the Vertical Ruler check box.

CAUSE

You may have installed SmoothType version 1.2 for the PPC.

RESOLUTION

Install SmoothType version 1.3 or later.

STATUS

Microsoft has confirmed this to be a problem in SmoothType version 1.2 for the PPC. This problem no longer occurs in SmoothType version 1.3 or later.
