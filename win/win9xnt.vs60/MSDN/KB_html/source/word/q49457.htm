Cannot Start Speller/Thesaurus/Converter When Memory Low

 Last reviewed: July 30, 1997
Article ID: Q49457
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

When you try to run the spelling checker, the thesaurus, the grammar checker or a file converter when conventional memory is low, one of the following error messages may result: 

   Cannot start Speller
   Cannot start Thesaurus
   Cannot start converter:<name>
   Cannot start WordPerfect Help
   Cannot start Grammar
This error message can be caused by low conventional memory or highly fragmented conventional memory. The spelling checker, the thesaurus, and the file converters are DLLs and must load in conventional memory. If you receive this error message, try to free some conventional memory using the following methods:

  1. Close some open documents.

  2. Change to draft view mode.

  3. Turn off background pagination.

  4. If the above do not work, try closing Word for Windows and restarting.
