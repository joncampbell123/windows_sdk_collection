Word 5.x Installation Hangs on a Power Macintosh

 Last reviewed: September 8, 1997
Article ID: Q112663
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1

SYMPTOMS

The Word 5 Installer hangs (stops responding) while installing the Equation Editor fonts on a Power Macintosh. 

 -or-
The following error message occurs: 

   An error occurred while installing on the active startup disk "<hard
   disk name>". Installation was not able to be completed, but your
   original software has been restored.

CAUSE

This problem occurs if you are using Word on a Power Macintosh. The Apple Installer that Word uses is incompatible with the Virtual Memory feature in some modes.

RESOLUTION

To force the Word Installer to quit so that you can try again:

  1. Hold down the OPTION+COMMAND keys and press ESC.

  2. When a dialog box appears asking you to confirm the force quit, choose the Force Quit button.

After you quit the Word Installer, use one of the following methods to successfully run the Installer:

Method 1:

Disable the Modern Memory Manager and the Virtual Memory option in the Memory control panel and run the Installer again.

NOTE: If you are running System 7.6 or later, the Modern Memory Manager is always on, and you cannot disable it.

Method 2:

Remove the Embedding Preferences file from the System:Preferences folder; remove the MT Extra Font from the System:Fonts folder.

Method 3:

Increase the memory assigned to the Installer program.

NOTE: Method 4 is suggested if you receive the error message.

Method 4:

Run the Installer again, but this time custom install Word and do not install any System Resources.

MORE INFORMATION

To disable the Virtual Memory feature on the Power Macintosh

  1. From the Apple menu, choose Control Panels.

  2. Double-click the Memory control panel.

  3. In the Virtual Memory section, select Off.

  4. Close the Memory control panel.

  5. Restart the computer.

To disable the Modern Memory Manager feature on the Power Macintosh

  1. From the Apple menu, choose Control Panels.

  2. Double-click the Memory control panel.

  3. In the Modern Memory Manager section, select Off.

  4. Close the Memory control panel.

  5. Restart the Power Mac.

To remove the Embedding Preferences and MT Extra font

  1. Drag the Embedding Preferences file to the Trash. This file is located in the Preferences folder (System 7) or in the System Folder (System 6)

  2. Drag the MT Extra font to the Trash. This file is located in the Fonts folder or in the System Suitcase in the System Folder.

To run the Installer again and set the memory

  1. From the Install disk in the Word 5.x disk set, select the Installer icon.

  2. From the File menu, choose Get Info.

  3. In the Preferred box, change the memory setting to 1300. Close the window.

  4. Start the Installer to install Word 5.x.

To run a Custom Installation

The Installer in Word 5.1 has a custom option to install all of Word's components without the System resources, called "Complete Install (No Sys. Resources)." This installation option does not interfere with the Virtual Memory feature and, therefore, installs properly with Virtual Memory on; however, the Equation Editor fonts and the capability to make voice annotations will be missing after such an installation.

After you complete the first phase of the Custom Installation, you will be prompted with a screen that asks if you want to continue with the installation or quit. Clicking Continue will return you to the Custom Installation screen. Clicking Quit will quit the Installer.

Scroll the list of Custom options and select "System Resources for System 6.0.6 and later." This phase of the operation will install the Equation Editor fonts and the Voice Annotations files. This phase combined with the first phase equals a complete Easy Install.

IMPORTANT: Do not forget to reset the Virtual Memory settings to their original configuration. When you use the Power Macintosh, programs will require twice as much memory if Virtual Memory is turned off.
