HP DeskJet "Scalable Font Initialization Error" with Word 6.0

 Last reviewed: February 5, 1998
Article ID: Q116108
6.00 6.00a 6.00c WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

When you print from Word for Windows to a Hewlett-Packard (HP) DeskJet using the DESKJETC.DRV version 3.1 printer driver, one of the following error message may occur: 

   Scalable Font Initialization Error

   Scaleable Font Initialization Error 7

   Scaleable Font Initialization Error b

CAUSE

The BulletPresent setting in the [HP DeskJet] section of your WIN.INI file is incorrect or missing.

RESOLUTION

  1. Open your WIN.INI file using Sysedit (System Configuration Editor [SYSEDIT.EXE]) or any text editor. (To start Sysedit, use the Run command on the File menu in Windows Program Manager. Type "sysedit" [without the quotation marks] in the Command Line box and choose OK.)

  2. Locate the [HP DeskJet] section.

  3. Look for the following line (where <n> is a number): 

          BulletPresent=<n>

    If the line exists, change the number <n> to 0 (zero). If the line does not exist, type it on a separate line in the [HP DeskJet] section, as follows: 

          BulletPresent=0

    NOTE: BulletPresent= may also be found in the [STATDLL] or the [DPSTAT] section of the WIN.INI.

  4. Save your WIN.INI file, then exit and restart Windows so the change can take effect.
