Documents Fax Incorrectly Using ZSoft UltraFax Software

 Last reviewed: July 30, 1997
Article ID: Q98501
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

When you fax a Word for Windows document using the ZSoft UltraFax Faxing software, the faxed document may appear to have a left margin of 7.5 inches. As a result, the faxed document contains only the text from the left side of the page, which appears on the far right side of the page. This problem occurs randomly; for example, a document may transmit correctly the first time and then display as described above the second time.

Workaround

ZSoft Technical Support suggests the following:

  • Repaginate the document (From the Tools menu, choose Repaginate Now).

    -or-

  • From the Format menu select Page Layout and re-enter any margin setting and choose OK. This will force Word to recognize the UltarFax driver.

For more information, contact ZSoft Technical Support at (404) 428-0008.
