WD: How to Delete All Auto-Numbered Footnotes

 Last reviewed: February 18, 1998
Article ID: Q176910
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SUMMARY

In Microsoft Word, you can use the Replace command to remove all auto- numbered footnote reference marks as well as the footnote text for each.

MORE INFORMATION

To delete all automatically numbered footnotes in a document, follow these steps:

  1. Position the insertion point at the beginning of your document.

  2. On the Edit menu, click Replace.

  3. If you see More, click it. Click Special, and then click Footnote Mark.

  4. Leave the Replace box empty.

  5. Click Replace All and click OK on the confirmation of how many replacements were made.

  6. Click Close.

This procedure will only remove footnotes and footnote reference marks that are automatically numbered. All other footnotes must be removed manually.
