WD: "Unknown File Format" Opening Word Files in WordPerfect

 Last reviewed: December 22, 1997
Article ID: Q145301
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

When you attempt to open a Word versions 6.0, 7.0, or 97 document into WordPerfect for Windows versions 6.x, you may receive the following error message: 

   Unknown File Format.

CAUSE

WordPerfect for Windows 6.x does not include a converter for Word versions 6.0, 7.0, or 97 files.

WORKAROUND

To work around this problem, open the document in Word and save it in WordPerfect 5.x for Windows format or Rich Text Format (RTF).

MORE INFORMATION

Contact Corel Corporation for information about the availability of a supplemental converter that imports and exports Word for Windows versions 6.x, 7.0, or 97 documents.

To find information about WordPerfect on the World Wide Web, connect to the following location:
