Word for Windows: Support for the Panasonic KX-P1191

 Last reviewed: July 30, 1997
Article ID: Q72570
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

The Panasonic KX-P1191 printer is not directly supported by Microsoft Windows or Microsoft Word for Windows.

MORE INFORMATION

According to Panasonic technical support, this printer emulates the Epson FX-86e/FX-800 printers in the default mode (all the DIP switches off). In this emulation mode:

  1. Choose Printer Setup from the File menu.

  2. Select the Epson 9-pin printer.

  3. Choose Setup and select Epson FX-86e/286e from the Printers list box.

    If the Epson driver does not appear in the Printer Setup dialog:

    a. Exit Word for Windows and from Program Manager, start the 

          Control Panel.

    b. Select Printers.

    c. Select Epson 9-pin printer from the Installed Printers list box. 

          Be sure Status is set to Active. If Epson 9-pin printer does
      appear in the list box, choose Add Printer and follow the
      instructions for installing a printer.

The Panasonic KX-P1191 printers can also emulate the IBM Proprinter II by changing the DIP switch settings. In this emulation mode, use the Windows IBM Proprinter driver configured for the Proprinter II.

Microsoft has not tested these emulations and therefore cannot guarantee their results.

For additional information about this printer, see the Panasonic "Operating Instructions Manual," pages 1-3 (under the "Specifications" heading) or call Panasonic technical support at (800) 222-0584.
