WD: How to Replace Smart Quotes with Straight Quotes

 Last reviewed: February 18, 1998
Article ID: Q176447
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SUMMARY

This article details how to search for curly quotes (smart quotes) so that you can replace them with standard quotation marks or to search for standard quotation marks and to replace them with smart quotation marks.

MORE INFORMATION

Replacing Existing Smart Quotes with Straight Quotes

Use the following method to replace existing smart quotation marks with straight quotation marks in a document.

  1. To Turn off smart quotes follow these steps: 

        a. On the Tools menu, click AutoCorrect and then select the AutoFormat
       As You Type tab.


        b. Under Replace As You Type, clear the Straight Quotes With Smart
       Quotes check box and then click OK.

  2. On the Edit menu, click Replace.

  3. In the Find What box, type a quotation mark.

  4. In the Replace With box, type a quotation mark.

  5. Click the Replace All button.

Replacing Existing Straight Quotes with Smart Quotes

Use the following method to replace existing smart quotation marks with straight quotation marks in a document.

  1. To turn on smart quotes, follow these steps: 

        a. On the Tools menu, click AutoCorrect and then select the AutoFormat
       As You Type tab.


        b. Under Replace As You Type, click to select the Straight Quotes With
       Smart Quotes check box and then click OK.

  2. On the Edit menu, click Replace.

  3. In the Find What box, type a quotation mark.

  4. In the Replace With box, type a quotation mark.

  5. Click the Replace All button.

REFERENCES

For more information about how to replace curly quotes with straight quotes, click the Office Assistant, type "smart quotes," click Search, and then click to view "Change curly quotes to straight quotes and vice versa."
