WD: Power Macintosh: Hard-Disk Space Needed to Install Word 6.0

 Last reviewed: February 6, 1998
Article ID: Q122373
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

This article lists the hard disk space you need for each installation option of Word for the Macintosh on both the standard Macintosh and the Power Macintosh computer.

MORE INFORMATION

Word 6.0 for the Macintosh

Minimum Installation           4.4 megabytes (MB)
Typical Installation          10.5 MB
Complete Installation         22.6 MB

Word 6.0 for the Power Macintosh

Minimum Installation           7.3 MB
Typical Installation          12.7 MB
Complete Installation         26.8 MB

Word 6.0 for the Macintosh and the Power Macintosh

Minimum Installation          10.9 MB
Typical Installation          16.3 MB
Complete Installation         31.7 MB
For information about the Word features that Setup installs with each installation option, double-click the Help button on the Standard toolbar and type "readme" (without the quotation marks).
