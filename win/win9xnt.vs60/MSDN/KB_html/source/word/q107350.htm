Installing Bookshelf 1993 with Word 6.0 for Windows

 Last reviewed: February 6, 1998
Article ID: Q107350
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SUMMARY

The Word for Windows integration features included with Microsoft Bookshelf 1993 will work correctly with Word 6.0 for Windows. However, because the Bookshelf 1993 Setup program is designed to work with Word 2.x for Windows only, Setup will not successfully install those features with Word 6.0.

The Word and Bookshelf integration features must be manually installed to function properly.

MORE INFORMATION

Use the following steps to install the Word and Bookshelf integration features for Word 6.0 for Windows to successfully communicate with Bookshelf 1993.

These steps do not install the Online Word User's Guide included with Bookshelf 1993. This guide is designed exclusively for use with Word 2.x for Windows.

Note: These steps assume Word 6.0 for Windows is already installed and functioning normally.

  1. Install Bookshelf 1993 according to its instructions. When given the option to install Word 2.0 for Windows Integration, do not. This option should not be checked. It will not function correctly with Word 6.0 installed.

  2. Copy the following files from the Bookshelf 1993 CD-ROM to the directories listed below (where "WINDOWS" is the name of your Windows program directory and "WINWORD" is the name of your Word 6.0 for Windows program directory): 

           File to be Copied from          Directory to Copy File to
       \VIEWER\BS93WORD Directory      on Hard Drive
       --------------------------      -------------------------

       BOOKSHLF.INI                    \WINDOWS
       BOOKSHLF.DLL                    \WINDOWS\SYSTEM
       WORDDDE.DAT                     \WINWORD
       DDEAPP.EXE                      \WINWORD

  3. Add the following text to the [Bookshelf] section of the WIN.INI file:

    Viewerpath=C:\WINDOWS\SYSTEM\MVIEWER2 Bookspath=D:\BOOKS\ Integration=2 SendToDlg=1 HelpFile=C:\WINWORD\WINWORD.HLP

    If your Windows directory is located somewhere other than C:\WINDOWS, modify the path given on the "Viewerpath" line above to reflect the correct path.

    If your CD-ROM drive uses a drive letter other than D:, modify the "Bookspath" line above to reflect the correct drive letter.

    If your Word 6.0 for Windows directory is located somewhere other than C:\WINWORD, modify the "HelpFile" line above to reflect the correct path.

  4. If Word 2.x for Windows was previously installed, check for the existence of the section [Microsoft Word 2.0] in the WIN.INI file. If found, make certain a "programdir" line exists specifying the correct path to the Word 6.0 for Windows program directory.

    If Word 2.x for Windows was not previously installed, or if the section [Microsoft Word 2.0] cannot be located in the WIN.INI file, create a new section with that name and add the following line of text to it:

    programdir=C:\WINWORD

    If the Word 6.0 for Windows program directory is located in a directory other than C:\WINWORD, modify the "programdir" line above to reflect the correct path.

  5. Start Word 6.0 for Windows.

  6. Using the Open command from the File menu, open the following document template:

    D:\VIEWER\BS93WORD\BSHELF93.DOT

  7. From the File menu, choose Save As.

  8. The following must be typed in the File Name box of the Save As window:

    C:\WINWORD\STARTUP\BSHELF93.DOT

    If the Word 6.0 for Windows program directory is located in a directory other than C:\WINWORD, modify the File Name line above to reflect the correct path.

  9. Exit and restart Word.

  10. From the Tools menu, choose Customize.

  11. Select the Menus tab.

  12. From the Categories list, select Macros.

  13. From the Macros list, select wabQuoteOfTheDay.

  14. From the Change What Menu list, select &View.

  15. From the Position on Menu list, select At Bottom.

  16. In the Name On Menu text box, type "&Quote Of The Day" (without the quotation marks).

  17. Choose the Add button.

  18. From the Macros list, select Separator.

  19. From the Change What Menu list, select &Tools.

  20. From the Position On Menu list, select At Bottom.

  21. Choose the Add button.

  22. From the Macros list, select wabBookshelfDlg.

  23. From the Change What Menu list, select &Tools.

  24. From the Position On Menu list, select At Bottom.

  25. In the Name On Menu text box, type "Bookshel&f" (without the quotation marks.

  26. Choose the Add button.

  27. From the Macros list, select Separator.

  28. From the Change What Menu list, select &Insert.

  29. From the Position On Menu list, select At Bottom.

  30. Choose the Add button.

  31. From the Macros list, select wabInsertVoiceAnno.

  32. From the Change What Menu list, select &Insert.

  33. From the Position On Menu list, select At Bottom.

  34. In the Name On Menu text box, type "&Voice Annotation" (without the quotation marks).

  35. Choose the Add button.

  36. From the Macros list, select wabInsertVideo.

  37. From the Change What Menu list, select &Insert.

  38. From the Position On Menu list, select At Bottom.

  39. In the Name On Menu text box, type "Media Pla&yer Clip" (without the quotation marks).

  40. Choose the Add button.

  41. Choose the Close button to close the Customize window.

Adding Bookshelf 1993 Integration to Word 6.0 for Windows is now complete.
