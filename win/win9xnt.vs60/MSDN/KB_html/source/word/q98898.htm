Printing ZIP Codes on Envelopes Using HP Bar Codes Cartridge

 Last reviewed: July 30, 1997
Article ID: Q98898
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

This article describes how you can use the Create Envelope command to print postal ZIP Codes on envelopes from Word for Windows to a printer using the Hewlett-Packard (HP) Bar Codes and More cartridge.

MORE INFORMATION

  1. From the Tools menu, choose Create Envelope.

  2. In the Addressed To box, type or insert the addressee information.

  3. After the address in the Addressed To box, press ENTER twice to insert a blank line. Type the ZIP Code again, omitting the hyphen between the fifth and sixth numbers. For example, instead of typing "98000-9999" (without the quotation marks), type "980009999". 

         Note: The HP Bar Codes and More cartridge ignores any hyphens
     you type in this string.

  4. Choose the Add To Document button. 

         Note: If you choose the Print Envelope button, Word formats the
     entire address, including the ZIP Code, with the font defined in
     the Envelope Address style. As a result, the second ZIP Code does
     not print in the bar code font.

  5. Select the second occurrence of the ZIP Code, which is two lines below the address.

  6. Format the numbers for the USPS ZIP font. On screen and in print preview, the ZIP Code does not appear as a bar code, but it usually appears as a sans serif font.

  7. From the File menu, choose Print.

  8. Select the Current Page option button.

    -or-

    Select the Pages option button and type "0" (zero, without the quotation marks) in the From and To boxes.

  9. Choose OK.
