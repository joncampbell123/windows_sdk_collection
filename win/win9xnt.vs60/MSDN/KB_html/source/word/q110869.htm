DocErr: Space Required on Network File Server

 Last reviewed: February 6, 1998
Article ID: Q110869
6.00 WINDOWS kbnetwork

The information in this article applies to:

  • Microsoft Word for Windows, version 6.0

SUMMARY

The Microsoft Word version 6.0 for Windows Resource Kit (page 316) and "Microsoft Word User's Guide" (page 769) incorrectly state: 

   The Word program requires 7 MB of available space
   on the file server or the shared directory.
The user's guide also indicates that 9.5 MB of available space is required for the shared components. These numbers are incorrect. The correct totals are 25 MB on the file server or the shared directory with the shared components requiring an additional 5 MB.

MORE INFORMATION

For more information on setting up Word 6.0 for Windows on a network, refer to Appendix B, pages 767-775, of the "Microsoft Word User’s Guide" and section 3 of Chapter 10, pages 314-316, in the Microsoft Word 6.0 for Windows Resource Kit.
