WD: How to Access the Microsoft Movie PIM from Online Services

 Last reviewed: February 5, 1998
Article ID: Q84896
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

The Microsoft Movie Plug-in Module (PIM) adds QuickTime movie support to Microsoft Word for the Macintosh version 5.0. Once installed in the Word Commands folder, the Movie PIM can be used to insert and play QuickTime movies within a Word document.

MORE INFORMATION

The following PIM file is available for download from the Microsoft Software Library: 

 ~ QTPIM.HQX (size: 94637 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
