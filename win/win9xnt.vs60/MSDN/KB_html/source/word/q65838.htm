Double-Clicking to Open Footnote Pane on Automatic References

 Last reviewed: July 30, 1997
Article ID: Q65838
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

Page 126 of the "Microsoft Word for Windows User's Reference" guide explains that double-clicking "an automatically-numbered footnote reference mark" in the document opens the footnote pane.

Because this feature works only when footnotes have been inserted with automatic reference marks checked, footnotes inserted with manual reference marks cannot be displayed with this method.

To display a footnote with a manual reference, do the following:

  1. Select the manual footnote reference.

  2. From the Insert menu, choose Footnote.

  3. Choose OK.

The footnote pane of the document window now opens with the requested footnote selected.
