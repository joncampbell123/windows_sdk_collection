HP DeskJet Drag-and-Drop Prints in Courier with Norton Desktop

 Last reviewed: February 5, 1998
Article ID: Q114430
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

When you drag and drop a Word 6.0 document onto the Hewlett-Packard (HP) DeskJet Printer icon to print it, the document prints in a monospaced (Courier) font. This problem occurs if you installed the HP DeskJet printer driver version 3.1 (DESKJETC.DRV), the universal printer driver version 3.1.4 (UNIDRV.DLL), and Norton Desktop for Windows version 2.2.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Print the document from within Word for Windows by opening the file, and then choosing Print from the File menu (or clicking the Print icon on the toolbar).
