How to Save FTP Downloaded Files in Internet Assistant

 Last reviewed: February 5, 1998
Article ID: Q129675
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 1.0, 1.0z
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SUMMARY

This article describes how to save a file that you downloaded using FTP. After you download the file, Word opens a new document and places an object icon in your document.

MORE INFORMATION

To save an FTP file that appears as a packager icon, do the following:

  1. With the right mouse button, click the embedded object. The shortcut menu will appear.

  2. From the shortcut menu, choose Edit Package Package.

  3. In the Object Packager, choose Save Contents from the File menu.

  4. Type a filename and choose a location for the file.

  5. Choose OK.

  6. Quit the Object Packager.

  7. Close the Word document that contains the package. (You do not need to save this document.)

Instead of seeing an object, you will see the following binary file information: 

  Binary file, size 635482 bytes.  ASCII portion follows
  mzk
To save a downloaded FTP file that is a binary file, do the following:

  1. With the right mouse button, click the FTP object. The shortcut menu will appear.

  2. From the Shortcut menu, choose Save As.

  3. Use the default name and file type that appears in the File Save As dialog box and choose OK.
