How to Install Missing Clip Art

 Last reviewed: February 6, 1998
Article ID: Q139867
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you click Picture on the Insert menu, no pictures appear in the Clipart folder.

CAUSE

When you installed Word, you chose to perform a Typical, Custom, or Compact installation. If you chose Custom, you may have cleared the Clip Art option. Clip Art is among the components not installed with a Typical or Compact installation.

RESOLUTION

To install Clip Art, use the following steps:

  1. Quit Word.

  2. Click the Windows Start button, point to Settings, and click Control Panel.

  3. Double-click Add/Remove Programs.

  4. Click the Install/Uninstall tab.

  5. In the list of installed software, select either Microsoft Word or Microsoft Office.

  6. Click Add/Remove.

  7. In the Setup box, click Add/Remove.

  8. If you are using Office Setup, select Office Tools and click Change Options.

    a. Select Clip Art and click OK.

    b. Click Continue.

    c. When prompted by Setup, insert the appropriate disk or compact disc.

    The new component is installed.

  9. If you are using Word Setup, select Word Clip Art and click Continue. When prompted by Setup, insert the appropriate disk or compact disc.

    The new component is installed.

WINDOWS NT

If you are running Word for Windows 95 on Windows NT, substitute the following steps for steps 1-5 above:

  1. Quit Word.

  2. From Program Manager, choose Run from the File menu.

  3. Type a:\setup. Click OK. Insert Disk 1 of the Word Setup disks. Continue with step 6 above.
