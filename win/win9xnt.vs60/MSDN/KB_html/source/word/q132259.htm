WD6X: Save Button Appears Completely Black on Mac PowerBook 140

 Last reviewed: February 5, 1998
Article ID: Q132259
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

In Microsoft Word for the Macintosh, version 6.x, the Save button on the Standard toolbar appears completely black. After using the button the first time in each Word session, the button is displayed properly.

CAUSE

This problem may occur when you use a PowerBook 140 if you have the Color Buttons option turned off in the Toolbars dialog box. (To locate this option, click Toolbars on the View menu.)

WORKAROUND

To work around this problem, turn the Color Buttons option on even though the PowerBook 140 has a black-and-white screen.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word listed at the beginning of this article.
