ViewEndnoteContNotice, ViewFootnoteContNotice

Syntax:

ViewEndnoteContNotice ViewFootnoteContNotice

Example:

This sample macro attempts to display both the Endnote Continue Notice and the Footnote Continue Notice. A message is displayed if they do not exist in the active document. 

   Sub MAIN
      On Error Goto Error1
      ViewFootnoteContNotice
   Error1:
      If Err <> 0 then
         MsgBox "No Footnote Continue Notice Found"
         Err - 0
      End If
      On Error Goto Error2
      ViewEndnoteContNotice
      Goto Bye
   Error2:
      MsgBox "No Endnote Continue Notice Found"
   Bye:
   End Sub
