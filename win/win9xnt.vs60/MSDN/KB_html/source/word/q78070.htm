How to Obtain the Word Binary File Format (BFF)

 Last reviewed: December 8, 1997
Article ID: Q78070
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

This article describes how to obtain a copy of the binary file format (BFF) for the Microsoft Word products listed above.

The file formats for earlier versions of Word, including Word for MS-DOS are no longer available. The file formats that are currently available are: 

   Word 6.0/7.0 "J" (Far East)
   Word 97 "U.S." and "J" (Far East)

MORE INFORMATION

Obtaining the Word 6.0/7.0 "J" (Far East) Binary File Format

To obtain the binary file format for Word 6.0/7.0 "J", send e-mail to wordbff@microsoft.com and include the following information: 

   Name
   Company Address
   Phone Number
NOTE: This e-mail alias is automated and will not be monitored for questions or comments; it is only for requesting the Word 6.0/7.0 "J" binary file format. 

   The documentation is provided "as-is" and no support will be provided
   by this alias.

Obtaining the Word 97 "U.S." and "J" (Far East) Binary File Format

NOTE: Word 97 "U.S." and "J" use the same binary file format.

The binary file format for Word 97 is available online. For more information, please see the MSDL Library Online at the following Microsoft Developer Network (MSDN) World Wide Web site: 

   http://www.microsoft.com/msdn
Once you subscribe to the MSDN, you can obtain an online copy of the Binary File Format. To do this, follow these steps:

  1. On the MSDN World Wide Web site, click MSDN Library Online.

  2. Under Member Area, click the Library Online tab.

  3. Double-click Microsoft Office Development.

  4. Double-click Office.

  5. Double-click Microsoft Office 97 Binary File Formats.

  6. Double-click Microsoft Word 97 Binary File Format.

  7. Double-click Microsoft Word 97 Binary File Format, (subtopic by the same name).

  8. Click to select the topic you want.
