WD6X: Error Message Starting Math Type or Equation Editor

 Last reviewed: February 6, 1998
Article ID: Q126088
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a
  • Microsoft Equation Editor version 2.01

SYMPTOMS

Math Type 3.0 does not function properly after you install Word version 6.0 for the Macintosh. When you try to start Math Type, you receive the following error message: 

   There has been an error this program will not be able to support
   object linking and embedding.
When you try to start Equation Editor 2.0, you receive the following error message: 

   The server application is not installed.
WORKAROUND

Use the Word for the Macintosh Setup program to remove Equation Editor, then reinstall MathType.

STATUS
