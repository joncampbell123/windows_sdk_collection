SGML: "The Stylesheet Was Empty..."

 Last reviewed: August 5, 1997
Article ID: Q125502
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

You receive the following error message when starting SGML Author for Word: 

   The stylesheet was empty or could not be read properly.

CAUSE

SGML Author tried to start Word for Windows version 2.0 instead of version 6.0. SGML Author requires Word version 6.0.

RESOLUTION

Close Word 2.0 for Windows and run Word 6.0 for Windows.
