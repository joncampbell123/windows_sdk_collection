WD6: Internet Assistant Repeat Find Replaces FTP Object

 Last reviewed: February 5, 1998
Article ID: Q129246
The information in this article applies to:
  • Microsoft Internet Assistant version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

If you use the Repeat Find command (SHIFT+F4) after opening an FTP URL, Word replaces the FTP object with the message "Click right mouse button in object to view options." This behavior occurs even after you find text within the FTP object.

CAUSE

Internet Assistant opens FTP sites as embedded objects. The Repeat Find command is a Word command, and it is not available inside an embedded object.

STATUS

Microsoft has confirmed this to be a problem in Internet Assistant version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

From the Edit menu, choose Find. Type in the search text and choose the Find Next button.
