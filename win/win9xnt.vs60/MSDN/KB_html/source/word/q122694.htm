WD: Can't Paste Link in MacWord 5.1 after Installing MacWord 6.0

 Last reviewed: February 6, 1998
Article ID: Q122694
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.1, 6.0

SYMPTOMS

If both Word 5.1 and 6.0 for the Macintosh are installed on your computer, the following error may occur in Word 5.1 when you paste link a selection: 

   Couldn't establish link to Microsoft Word for <filename>, link <N>.
   Common causes: Source file renamed or moved, program unavailable,
   not enough memory.
To paste link a selection from the Clipboard, click Paste Special on the Edit menu, select the Paste Link option, and then click OK.

CAUSE

When you install Word 6.0, Setup modifies the Embedding Preferences file, which disables the Paste Link functionality in Word 5.1.

RESOLUTION

To resolve this problem, follow these steps:

  1. Close both Word 5.1 and 6.0.

  2. In your hard disk folder, open the Preferences folder.

  3. Move the Embedding Preferences file to the trash.

The Paste Link feature should now work correctly in both Word 5.1 and 6.0.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for the Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

"Microsoft Word User's Guide," version 6.0. Document number WB60460-0794, pages 638-640
