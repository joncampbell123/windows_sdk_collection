WD: Use OPTION+RETURN to Insert Return in ASK or SET

 Last reviewed: February 5, 1998
Article ID: Q77774

The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 6.0, 6.0.1, 6.0.1a

SUMMARY

When using Microsoft Word for the Macintosh, the ASK and SET commands in a Print Merge allow you to enter custom information in a dialog box during the merge process. To enter multiple lines in this dialog box, press OPTION+RETURN at the end of the line. The resulting merged information has normal carriage returns at the end of each line.
