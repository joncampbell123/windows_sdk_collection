WD: Labels Print Page of "Error! Reference Source Not Found."

 Last reviewed: February 5, 1998
Article ID: Q140530
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you print a page of labels, every label prints the message: 

   Error! Reference source not found.

CAUSE

This problem occurs when you use REF fields in the document to display the information for the labels and you select the REF fields before clicking Envelopes And Labels on the Tools tab. Although the text displays correctly on the Labels tab, it is printed incorrectly when you click the Print button.

Envelopes created in this manner are printed correctly, however.

WORKAROUND

Unlink the fields in the Address box before clicking the Print button. To do this, follow these steps:

  1. Select the REF fields (the address) in the document.

  2. On the Tools menu, click Envelopes And Labels.

  3. Click the Labels tab.

    The text appears in the Address box.

  4. Select the text.

  5. Press CTRL+SHIFT+F9 (Windows) or COMMAND+SHIFT+F9 (Macintosh).

  6. Click Print.

The labels should now be printed correctly.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem, and we will post new information here in the Microsoft Knowledge Base as it becomes available.
