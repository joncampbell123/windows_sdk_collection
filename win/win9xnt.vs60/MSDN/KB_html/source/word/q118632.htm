WD: Cannot Sort Single-Column Framed Table

 Last reviewed: February 5, 1998
Article ID: Q118632
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you try to sort a single-column table contained in a frame, Microsoft Word does not sort the table.

CAUSE

This problem occurs when the frame is selected when you are trying to sort the table.

WORKAROUNDS

To work around this problem, use any of following methods:

  • Click inside one of the table cells or select the table instead of the frame, and then perform the sort. 

        -or-
  • Perform the sort before you frame the table. 

        -or-
  • Convert the table to text, perform the sort on the paragraphs, and convert the text back to a table.

STATUS

Microsoft has confirmed this to be a problem in the Word versions listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
