WD: Word Cannot Convert Password Protected Documents

 Last reviewed: March 4, 1998
Article ID: Q168098
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you try to open a password protected file into Word, one or more of the following error messages may be displayed:

  • Cannot convert password protected documents
  • Word cannot open the document.

CAUSE

Access to a file is restricted when the document has been saved with any type of password protection. All of the converters in Microsoft Word will not convert password protected files.

WORKAROUND

Open and re-save the document without password protection in the original program before you open the document in Word.

MORE INFORMATION

In Microsoft Word 6.0 or in Word for Windows 95, version 7.0, you cannot convert a password-protected file from Word 97.

Similarly, you cannot convert a password-protected file created in another word processor (such as Word Perfect, AmiPro, or Enable) from any version of Word.

NOTE: The password protection feature in Word requires you to type a case- sensitive password before you can open a document. Word converters do not prompt you for the password, and will not allow you to convert a password protected document.
