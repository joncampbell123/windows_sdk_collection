Cannot Make ODBC Connection After Installing 32-Bit Wbodbc.wll

 Last reviewed: February 5, 1998
Article ID: Q142334
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

You cannot make an ODBC connection, even though you have installed the 32-bit Wbodbc.wll supplied with the "Microsoft Word Developer's Kit, Second Edition."

CAUSE

You are using the 32-bit Wbodbc.wll file that was designed for use in Word for Windows NT. Microsoft does not support using this file with Word for Windows 95.

RESOLUTION

Upgrade to the Wbodbc.wll file that is shipped with the "Microsoft Word Developer's Kit, Third Edition." The Third Edition was shipped November 27, 1995.
