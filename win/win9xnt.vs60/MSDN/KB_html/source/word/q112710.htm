"Invalid Database" Error Attaching Mail Merge Data Document

 Last reviewed: February 5, 1998
Article ID: Q112710
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

In a mail merge main document, if you attach a data document that contains Script, Roman or Modern font formatting or have one of those fonts as your default font, Word for Windows displays the following error message: 

   Word could not open <FILENAME> because it did not
   contain a valid database.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Reformat your data document for a font other than Script, Roman or Modern.
