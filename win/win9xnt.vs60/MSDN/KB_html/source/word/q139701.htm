WD4: Type 3 Error Starting Word 4.0 on PCI-based Power Mac

 Last reviewed: November 17, 1997
Article ID: Q139701
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 4.0

SYMPTOMS

You receive a Type 3 error when you attempt to start Word 4.0 on the following PCI-based Power Macintosh computers: 

   5200
   6200
   7200
   7500
   8500
   9500
   Performa Power PC

CAUSE

This problem occurs because of an incompatibility between Word 4.0 and the PCI-based Power Macintosh computers that use the System 7.5.2 operating system software.

WORKAROUND

Upgrade to Word 5.1a or Word 6.0.1 for the Macintosh.

To obtain pre-sales information about new or updated Microsoft products, call the Microsoft Sales Information Center at (800) 426-9400. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm

MORE INFORMATION

Although this problem is similar to that experienced when you start Word 4.0 on a 68040-based Macintosh, turning on the More Compatible (Caches Disabled) option in the Cache Switch Control Panel does not allow you to run Word 4.0 on the machines listed in the "Symptoms" section above. The cache on these machines cannot be controlled in the same manner. In addition, you will not be able to run Word 4.0 if you disable the Modern Memory Manager or your extensions.
