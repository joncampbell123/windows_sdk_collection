WD6: Internet Assistant, What Happens When Setup Is Run fro

 Last reviewed: February 6, 1998
Article ID: Q128176
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

When you copy the file WORDIA.EXE to the <Winword>\INTERNET subdirectory (usually C:\WINWORD\INTERNET) and run Setup from that directory, certain files are not installed, and you receive error messages once you start Internet Assistant.

CAUSE

The file WORDIA.EXE and its expanded files are in the <Winword>\INTERNET directory. These files are designed to be run from a different directory.

RESOLUTION

Uninstall Internet Assistant. Move WORDIA.EXE to a new directory, for example, C:\WORDIA, run WORDIA.EXE, and run Setup again.

MORE INFORMATION

The following problems occur when WORDIA.EXE is run from the <Winword>\INTERNET subdirectory:

  • The following files are not installed 

          CREATING.DOC
      DEFAULT.DOC
      DEFAULT.HTM
      FAVORITE.DOC
      IWIA.EXE
      IWIA.REG
      LICENSE.TXT
      NAVIGAT.DOC
      README.DOC
      TIPS.DOC
      WORDHTML.HLP

    The following files are installed if you install from the \INTERNET directory but, are not installed if you install from the root or \WINWORD: 

          WORDHTML.WLL wordhtml.wll
      HTML.DOT html.dot
      WEBVIEW.DOT webview.dot
      HTMLCONV.CNV htmlconv.cnv
  • You get the following error when you choose Browse Web from the File menu. 

          Microsoft Word Err=1078
      This file could not be found. <path>\default.doc
  • You get the following error when you choose Open URL from the File menu: 

          Unable to Complete Download
      Error code: 800401f5
  • You get the following error when accessing Help: 

          Cannot Open Help File
  • Browse Web does not appear on the File menu.

These errors will occur even though setup finishes with a message indicating setup was successful.

After completing the setup Browse Web does not appear on the File menu.

After the setup is complete if you choose the Launch Word button, the following error occurs: 

   Word cannot open the document
   (C:\msoffice\winword\internet\readme.doc.)
For additional information, refer to the Installing Internet Assistant section of READMD.DOC
