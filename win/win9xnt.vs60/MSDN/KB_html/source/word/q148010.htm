Find File Advanced Search in Word 6 Ignores Word 5 Documents

 Last reviewed: February 5, 1998
Article ID: Q148010
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When you use Find File in 68K Word for the Macintosh version 6.x to search for Word for the Macintosh version 5.x documents that contain specific keywords, author, or text, Find File does not return any results.

This problem does not occur with the Power Macintosh version of Word.

WORKAROUND

To work around this problem and restore the keywords so that they are recognized by Find File, open and save the document in Word 6.0.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
