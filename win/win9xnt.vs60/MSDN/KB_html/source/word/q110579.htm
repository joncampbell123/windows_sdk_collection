Incorrect Dialog for ASK and FILLIN Fields in Help

 Last reviewed: February 5, 1998
Article ID: Q110579
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

In the Word 6.0 for Windows Help file, the diagram of the ASK and FILLIN dialog is incorrect. The actual dialog box does not contain the text "Press Shift+Enter to start a new line."

MORE INFORMATION

When you update an ASK or FILLIN field in Word 6.0 for Windows, a dialog box prompts you for information, but no "Press Shift+Enter to start a new line" instruction appears (as it does in Word 2.0).

The dialog boxes in Word 6.0 for Windows have been enhanced so that if you press ENTER to start a new line while the dialog box is displayed, Word returns text containing a new line character (same result as SHIFT+ENTER in Word 2.0) followed by a paragraph mark where the line breaks.

To obtain the Help topic for ASK or FILLIN:

  1. From the Help menu, choose Contents.
2. Select Reference Information. 3. Select Field Types and Switches. 4. Select Fields Types. 5. Choose ASK and FILLIN from the alphabetical list offered.
