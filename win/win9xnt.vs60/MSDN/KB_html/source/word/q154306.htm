Embedded PowerPoint Object Border Thin When Printed from Word

 Last reviewed: July 31, 1997
Article ID: Q154306
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

The lower and right borders on an embedded PowerPoint object are either not printed or are printed thinner than the top and left borders. This problem does not occur in print preview or page layout view.

CAUSE

This problem occurs with any PCL printer, such as the Hewlett-Packard (HP) LaserJet 4, 4si, or 4L, or the HP DeskJet 500c. This problem also occurs with some PostScript printers.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Word 97 for Windows.

WORKAROUND

To work around this problem, use the following steps:

  1. In Word, select the object, and then click Picture on the Format menu.

  2. Under Crop From, set any one of the cropping measurements (left, right, top, or bottom) to .01 inch, and then click OK.

The border will now be printed.
