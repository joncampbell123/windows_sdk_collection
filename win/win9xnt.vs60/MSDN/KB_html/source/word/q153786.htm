Revision Marks Don't Appear When You Edit Object

 Last reviewed: February 5, 1998
Article ID: Q153786
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you edit an object in a Word document and the Mark Revisions While Editing option is activated (on the Tools menu, click Revisions), the modified object is updated but revision marks are not displayed to indicate that the object has been modified. This problem does not affect revision marks applied to modified text.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
