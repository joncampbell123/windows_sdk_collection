WD6X: Same Graphic Prints/Displays in Multiple Places

 Last reviewed: December 5, 1997
Article ID: Q123618
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

One graphic is printed and displayed in place of other graphics in your document.

CAUSE

The first six letters of the file names are identical--for example, GRAPHIC1, GRAPHIC2, and GRAPHIC3.

WORKAROUND

Rename the graphics so that the first six characters of the filenames are not identical.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for the Macintosh. This problem was corrected in Word version 6.0.1 for the Macintosh.
