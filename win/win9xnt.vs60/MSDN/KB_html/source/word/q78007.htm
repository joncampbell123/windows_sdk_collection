WinWord 2.0: How to Suppress Blank Lines During Print Merge

 Last reviewed: July 30, 1997
Article ID: Q78007
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

If you have several documents open in Microsoft Word for Windows and you select Arrange All from the Window menu, the resulting window borders overlap slightly if you are using an EGA monitor.

Steps to Reproduce Problem

  1. From the Window menu, choose New Window until you have at least three windows open.

  2. From the Window menu, choose Arrange All.

The borders of the resulting windows overlap slightly.
