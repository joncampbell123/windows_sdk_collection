WD: Cannot Save File (Save Button Unavailable)

 Last reviewed: February 2, 1998
Article ID: Q42059
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1, 5.1a, 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Microsoft Word for the Macintosh, after you click Save or Save As on the File menu, the Save button sometimes remains dimmed even though a name has already been assigned to the document.

CAUSE

This problem can occur when you try to save to a floppy disk that is locked. Check to be sure that the disk is unlocked. This problem can also occur when you try to save to a folder on a server for which you don't have access privileges.

WORKAROUND

If you have a hard disk or a two-drive system, choose the Drive button (the Desktop button in System 7.0) in the Save or Save As dialog box. This procedure allows you to save to another disk, providing it is unlocked.
