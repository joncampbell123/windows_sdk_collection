WD: Send Command Sends Attachment as Word Document

 Last reviewed: January 15, 1998
Article ID: Q179313
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When you use the Send command on the File menu, the current document is attached as a Word document in the mail message even if you saved the file in a foreign format.

CAUSE

This behavior occurs when the Mail As Attachment option is selected on the General tab in the Preferences dialog box (on the Tools menu, click Preferences).

WORKAROUND

To mail a file in a foreign format, save the file in that format and then attach the file to your message from your mail program.

MORE INFORMATION

For additional information about this problem in later versions of Microsoft Word for Macintosh, please see the following article in the Microsoft Knowledge Base:
