WD: Password Protected MacWord Documents Missing Information

 Last reviewed: February 2, 1998
Article ID: Q107083
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Word for the Macintosh version 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

Incomplete File information and Summary information is displayed in the File Find dialog box when the document selected in the Listed Files box has been password protected.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you search for a file using the Word 6.0 Find File command from the File menu, the Find File dialog box offers three views. The Preview option displays a preview of the actual document, the Summary option shows the File Summary Information, and the File Info option shows the title, file size, author, and last saved date.

When a document is password protected, Word prompts for the password when you select the Preview option and displays a preview of the document if you enter the correct password. Word does not prompt for a password when you select either of the other views.

The Summary information option only correctly displays Creation date and Last Saved date. Other Summary statistics may have data but the information is incorrect; for example, Total Editing Time is zero. File Info view correctly displays the file size and the last saved date.
