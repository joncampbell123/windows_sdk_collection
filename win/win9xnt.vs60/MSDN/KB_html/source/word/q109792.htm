Error Message: "Cannot Display Macintosh Pictures..."

 Last reviewed: February 6, 1998
Article ID: Q109792
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you insert a Microsoft Word for the Macintosh document in a Word 6.0 for Windows document, the following error message may occur if your Macintosh document contains pictures: 

   Word cannot display Macintosh pictures without a graphics filter.

CAUSE

This message occurs if the Macintosh PICT graphics filter (PICTIMP.FLT) is not installed on your system. The error message does not specifically state which graphics filter is missing. The following error message wording is more descriptive: 

   Word cannot display Macintosh pictures without the PICT filter.
   Please install PICTIMP.FLT.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

Run the Word Setup program and install the Macintosh PICT filter.

REFERENCES

"Microsoft Word Quick Results," version 6.0, pages 10-11
