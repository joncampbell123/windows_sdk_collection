Word 6.0 Help for RD Field Doesn't Show Macintosh Syntax

 Last reviewed: February 3, 1998
Article ID: Q133411
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When you use an RD (referenced document) field with the syntax specified in Word Help, you may receive the following error: 

   ERROR! CANNOT OPEN FILE REFERENCED ON PAGE 1

CAUSE

The path syntax provided in Help is incorrect; it shows Windows syntax instead of Macintosh syntax. The Macintosh uses a colon (:) as the delimiter, where Windows uses a backslash (\).

RESOLUTION

Use the standard Macintosh path syntax as follows: 

   {RD "Macintosh HD:folder:filename"}
Notice also that there are quotation marks around the path string.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Word 98 Macintosh Edition.
