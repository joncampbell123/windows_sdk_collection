Comparison of Formatting Shortcut Keys Between Word 2.0 & 6.0

 Last reviewed: February 5, 1998
Article ID: Q118441
2.00 2.00a 2.00a-CD 2.00b 2.00c 6.00 6.00a 6.00c WINDOWS kbusage kbinterop

The information in this article applies to:

  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a

SUMMARY

This article summarizes the changes in formatting shortcut keys between Word for Windows versions 2.x and 6.0.

Paragraph Formatting

Action                 2.x                   6.0
--------------------   --------------------  -------------------

Center                 CTRL+E                CTRL+E
Right align            CTRL+R                CTRL+R
Left align             CTRL+L                CTRL+L
Justify                CTRL+J                CTRL+J
Increase indent        CTRL+N                CTRL+M
Decrease indent        CTRL+M                CTRL+SHIFT+M
Hanging indent         CTRL+T                CTRL+T
Remove hanging indent  CTRL+G                CTRL+SHIFT+T
Open/Close space       CTRL+0 (zero)         CTRL+0 (zero)
  before (Close space
  before in 2.0)
Single-space lines     CTRL+1                CTRL+1
Double-space lines     CTRL+2                CTRL+2
1.5-space lines        CTRL+5
Apply style            CTRL+S                CTRL+SHIFT+S
Reset to Style         CTRL+Q                CTRL+Q

Character Formatting

Action                 2.x                   6.0
--------------------   --------------------  -------------------

Bold                   CTRL+B                CTRL+B
Italic                 CTRL+I                CTRL+I
Continuous underline   CTRL+U                CTRL+U
Word underline         CTRL+W                CTRL+SHIFT+W
Double underline       CTRL+D                CTRL+SHIFT+D
Subscript              CTRL+=                CTRL+=
Superscript            CTRL+SHIFT+=          CTRL+SHIFT+=
All Caps               CTRL+A                CTRL+SHIFT+A
Small caps             CTRL+K                CTRL+SHIFT+K
Font                   CTRL+F                CTRL+D
Point size             CTRL+P                CTRL+SHIFT+P
Hidden                 CTRL+H                CTRL+SHIFT+H
Reset character        CTRL+SPACEBAR         CTRL+SHIFT+Z, or
                                             CTRL+SPACEBAR

Other Actions

Actions                2.x                   6.0
--------------------   --------------------  -------------------

Cut                    CTRL+X                CTRL+X
Copy                   CTRL+C                CTRL+C
Paste                  CTRL+V                CTRL+V
Repeat                 F4                    CTRL+Y or F4
Cancel                 ESC                   ESC
New line               SHIFT+ENTER           SHIFT+ENTER
New page               CTRL+ENTER            CTRL+ENTER
New column/split       CTRL+SHIFT+ENTER      CTRL+SHIFT+ENTER
  table
Optional hyphen        CTRL+HYPHEN           CTRL+HYPHEN
Nonbreaking hyphen     CTRL+SHIFT+HYPHEN     CTRL+SHIFT+HYPHEN
Nonbreaking space      CTRL+SHIFT+SPACEBAR   CTRL+SHIFT+SPACEBAR
Tab character (table)  CTRL+TAB              CTRL+TAB
Show nonprinting       CTRL+SHIFT+*          CTRL+SHIFT+*
  characters
