WD98: Programs Missing from Default Picture Editor List

 Last reviewed: February 19, 1998
Article ID: Q181437
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

Some graphics programs may not appear in the Picture Editor list; therefore, you cannot select them as the default program for editing imported graphics in Microsoft Word 98 Macintosh Edition. (To display the Picture Editor list, click Preferences on the Tools menu and click the Edit tab.)

CAUSE

Word 98 Macintosh Edition lists a graphics program in the Picture Editor list only if the registration information has been written to the Registration Database or the PPC Registration Database (for Power Macintosh computers).

These databases tell Word that the graphics program can edit metafiles or PICT files as OLE objects. Note that you cannot edit these files manually.

WORKAROUND

To work around this problem, try reinstalling the graphics program to register this information.

If you want to select an unlisted graphics program as the default picture editor in Word, contact the manufacturer of the graphics program for further instructions.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To change the default program for editing imported graphics, follow these steps:

  1. On the Tools menu, click Options, and then click the Edit tab.

  2. In the Picture editor box, click the program you want to use to modify imported graphics, and then OK.
