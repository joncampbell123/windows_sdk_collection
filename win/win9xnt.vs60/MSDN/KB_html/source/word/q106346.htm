Compare Versions Does Not Flag Header and Footer Changes

 Last reviewed: February 5, 1998
Article ID: Q106346
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

The Compare Versions feature does not flag header and footer differences between two Word for Windows documents if neither document contains text in the body of the document. (To locate the Compare Versions button, choose Revisions from the Tools menu.)

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

To compare information in the header or footer of a document, add some text (a space is sufficient) to the body of your document. Word correctly compares document versions when one contains text in the body.
