WD6: SGML: Setup Err Msg: "Word Is Not Installed, Install Word..

 Last reviewed: August 5, 1997
Article ID: Q139562
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

When you try to set up Microsoft SGML Author for Word, version 1.0, on a system with the original release of Word version 6.0 for Windows, you receive one of the following error messages: 

   Word for Windows 6.0 must be installed to continue Setup. Please exit
   Setup now, install Word, and then set up SGML Author for Word.

   -or-

   Word is not installed, install Word before running SGML Author Setup.

CAUSE

The SGML Author Setup utility cannot find where Word 6.0 for Windows is installed based on the Windows registry (REG.DAT). This problem does not occur with Word version 6.0a or 6.0c for Windows.

RESOLUTION

Install Word 6.0a or 6.0c for Windows before running Microsoft SGML Author 1.0 for Word Setup.

For additional information about obtaining support for Microsoft SGML Author for Word, please see the following article in the Microsoft Knowledge Base:

   TITLE     : SGML Author Support Policy
