INTRODUCTION

This Application Note contains instructions for creating custom mailing label templates using the Table feature in Word for the Macintosh. You can use these templates to generate mailing labels that print on an Apple(R) LaserWriter(R) or Apple ImageWriter(R) printer. Using these templates, you can enter the mailing label information directly, or you can create a print merge main document for merging to a data document.

Word versions 4.0, 5.0, and 5.1 supply several mailing label stationery documents you can use to print common mailing label sizes. If your labels match the dimensions of these common mailing label sizes, you do not need to use the instructions in this application note. You should use the instructions in this Application Note if your labels do not match the dimensions of the mailing label documents supplied with Word. For information on the mailing labels stationery, see the following resources:

WORD VERSIONS 5.0 AND 5.1

  • Mailing Labels ReadMe document, located in the Mailing Labels folder in the Sample Documents folder

WORD VERSION 4.0

  • Using Mailing Labels document, located in the Mailing Labels folder

MAILING LABEL TERMINOLOGY

 "Column Width":  The label width plus the horizontal gap, expressed in
                  inches

 "Horizontal Gap":  The distance in inches between two labels on the same
                    row.

 "Label Height":  The distance in inches between the top and bottom edges
                  of a label.

 "Label Width":  The distance in inches between the left and right edges of
                 a label.

 "Left Indent":  The distance in inches between the left edge of the label
                 and the beginning of each printed line.

 "Right Indent":  The distance in inches between the right edge of the
                  label and the point at which the printed line wraps.

 "Row Height":  The vertical distance in inches between the top edge of one
                label in a column and the label directly above it.

 "Side Margins":  The horizontal distance in inches between the edge of the
                  page and the edge of the label.

 "Top Margin":  The vertical distance in inches between the top edge of the
                page and the top edge of a label in the top row.

TO CREATE THE MAILING LABEL STATIONERY

Look up your labels in the Avery(R) Label Specifications table below. If your labels do not appear in this table, use the Mailing Label Terminology section above to determine the dimensions of your labels. 

  NOTE: In the steps below, the terms in parentheses correspond to the
  terms in the Avery Label Specifications table at the bottom of this
  document.

MAILING LABEL STATIONERY FOR THE

LASERWRITER, STYLEWRITER(R), AND DESKWRITER(R)

To create a full page of mailing labels:

  1. From the File menu, choose Page Setup. 

        a. Select the US Letter option button, select the Portrait (Tall)
       Orientation icon, and type "100" (without the quotation marks) in
       the Reduce Or Enlarge box.


           NOTE: If you are using a StyleWriter, DeskWriter, LaserWriter LS,
       or other non-PostScript(R) printer, choose the OK button and skip to
       step 2 below.


        b. Choose the Options button.


        c. Select the Larger Print Area check box and choose the OK button
       twice.


           NOTE: When you select Larger Print Area, the smallest margin you
       can set is .25 inch. When you don't select the Larger Print Area
       check box, the smallest margin you can set is .5 inch.

  2. From the Format menu, choose Document. 

        a. Type the indicated values for the following settings (see the Avery
       Label Specifications table at the bottom of this document):


            - Left and Right Margins (Side Margins)

        - Top Margin (Top Margin)

    b. In the Bottom Margin box, type "0" (zero) (without the quotation
       marks) and choose the OK button.

       NOTE: By setting the bottom margin to 0, you prevent a page break
       in the table.

  3. From the Insert menu, choose Table (in Word version 4.0, choose Insert Table from the Document menu). Type the indicated values for the following settings (see the Avery Label Specifications table at the bottom of this document) and then choose the Format button: 

        - Number of Columns (# of Columns)


        - Number of Rows (# of Rows)


        - Column Width

  4. In the Table Cells dialog box, type the indicated value for the following settings and then choose the OK button: 

        - In the Space Between Columns box, type "0" (zero) (without the
      quotation marks).


        - In the Row Height box, specify the height in points (to generate the
      row height, multiply the row height in inches by 72 to convert it to
      points). (In Word version 4.0, type a minus sign [-] before the Row
      Height value. For example, type "-144" (without the quotation
      marks).


        - From the Row Height list box, select Exactly (this step does not
      apply to Word version 4.0).

  5. Select the table. From the Format menu, choose Paragraph. Type the indicated values (see the Avery Label Specifications table at the bottom of this document) for the following settings and then choose the OK button: 

        - Left Indentation (Paragraph Left Indent): This setting determines the
      distance between the left edge of the label and the beginning of each
      line. For example, if you want your mailing label text to appear .25
      inch from the left edge of the label, type ".25 in" (without the
      quotation marks) in the Left Indentation box.


        - Right Indentation (Paragraph Right Indent): Add the Horizontal Gap to
      the desired right indent on the label (see Horizontal Gap in the
      Mailing Label Terminology section above).

  6. From the File menu, choose Save As. In the Save Current Document As box, type a name for the file.

  7. In the Save File As Type box, select Stationery, and then choose the Save button.

IMAGEWRITER MAILING LABELS STATIONERY

NOTE: If your printer is an ImageWriter LQ, you can use the instructions for either the ImageWriter or the LaserWriter.

Use the following steps to create a mailing label stationery (template) in which a page is one row long. The first inch on the left and right sides of tractor feed labels is not part of the page size.

  1. From the Tools menu, choose Preferences (in Word version 4.0, choose Preferences from the Edit menu). Choose the General icon. Type the indicated values for the following settings and then close the Preferences window: 

        - Custom Page Size Height (see Row Height in the Mailing Label
      Terminology section above)


        - In the Custom Page Width box, type "8.5 in" (without the quotation
      marks).


           NOTE: A setting other than 8.5 inches causes the ImageWriter to
       shift the text 0.125 inch to the right.

  2. From the File menu, choose Page Setup. 

        a. Select the Custom Page Size option button.


        b. Select the Portrait (Tall) Orientation icon.


        c. In the Reduce Or Enlarge box, type "100" (without the quotation
       marks).


        d. Select the No Gaps Between Pages check box and choose the OK button.

  3. From the Format menu, choose Document. Type the indicated values for the following settings: 

        - Left and Right Margins (Side Margins)


           NOTE: The minimum side margin setting is 0.25 inch on the
       ImageWriter II and ImageWriter LQ.


        - Top Margin (Top Margin)


           NOTE: You should usually set the top margin to 0 (zero) because you
       usually align the top of the label with the top of the paper bail,
       which leaves no top margin.


        - In the Bottom Margin box, type "0" (zero) (without the quotation
      marks) and choose the OK button.


           NOTE: By setting the bottom margin to 0, you prevent a page break
       in the table.

  4. From the Insert menu, choose Table (in Word version 4.0, choose Insert Table from the Document menu). Type the indicated values for the following settings and then choose the Format button: 

        - Number of Columns (# of Columns).


        - In the Number of Rows box, type 1.


        - Column Width

  5. In the Table Cells dialog box, type the indicated value for the following settings, and then choose the OK button: 

        - In the Space Between Columns box, type "0" (zero) (without the
      quotation marks).


        - Row Height in points (to generate the row height, multiply the
      vertical pitch in inches by 72 to convert it to points). (In Word
      version 4.0, type a minus sign [-] before the Row Height value. For
      example, type "-144" (without the quotation marks).)


        - From the Row Height list, select Exactly (this step does not apply
      to Word version 4.0).

  6. Select the table. From the Format menu, choose Paragraph. Type the indicated values for the following settings, and then choose the OK button: 

        - Left Indentation (Paragraph Left Indent): This setting determines the
      distance between the left edge of the label and the beginning of each
      line. For example, if you want your mailing label text to appear .25
      inch from the left edge of the label, type ".25 in" (without the
      quotation marks) in the Left Indentation box.


        - Right Indentation (Paragraph Right Indent): Add the Horizontal Gap to
      the desired right indent on the label (see Horizontal Gap in the
      Mailing Label Terminology section above).

  7. From the File menu, choose Save As. In the Save Current Document As box, type a name for the file.

  8. In the Save File As Type box, select Stationery, and then choose the Save button.

PRINT MERGE MAILING LABEL SETUP

You can create a print merge main document using your mailing label stationery. In your mailing label document, insert merge codes in place of the addressee information. Figure 2 is a sample print merge main document for mailing labels. For more information on using print merges to create mailing labels with Word versions 5.0 and 5.1, see "Creating the Main Document for Mailing Labels," which begins on page 628 of the "Microsoft Word User's Guide." If you are using Word version 4.0, see "Creating a Main Document," which begins on page 175 of "Reference to Microsoft Word".

To insert the merge codes, press OPTION+BACKSLASH (\) for «, and OPTION+SHIFT+BACKSLASH (\) for », or use the "print merge" Standard Glossary entry. Insert the DATA statement in the first cell. Replace the DATA statement with a NEXT statement in the remaining cells. For more information on the DATA and NEXT statements, see pages 598-615 of the "Microsoft Word User’s Guide" for Word versions 5.0 and 5.1. For Word version 4.0, see pages 294-297 of "Reference to Microsoft Word".

TROUBLESHOOTING

THE NAMES AND ADDRESSES DO NOT PRINT WITHIN THE MAILING LABEL WIDTH:

Increase or decrease the left and right indents.

THE LABELS START TOO HIGH OR TOO LOW ON THE PAGE:

Some sheets of labels have different top and bottom margins. Be sure you feed the correct end of the label sheet into the printer. In addition, check the Top Margin setting to be sure the setting is correct.

On an ImageWriter, make sure the top of the labels is aligned with the top of the paper bail. You may need to adjust the label up or down in the printer.

THE LABEL TEXT CONTINUES TO PRINT ON NEXT LABEL:

Make sure the Row Height setting is the same as the vertical pitch measurement. In the Table Cells dialog box, make sure the Exactly setting is selected in the Row Height list. (In Word version 4.0, make sure a minus sign [-] precedes the Row Height setting.)

SOME CELLS CONTAIN MERGE CODES

INSTEAD OF NAMES AND ADDRESSES WHEN THEY ARE MERGED OR PRINTED:

Word print merges one row at a time, beginning from the top of the document. If there are not enough entries to fill an entire row, Word fills the remaining cells in the row with the merge codes. You can discard these labels.

PRINT MERGE ERROR MESSAGES ARE GENERATED:

For more information on print merge error messages, see "Troubleshooting" on page 622 of the "Microsoft Word User's Guide" for Word versions 5.0 and 5.1. For Word version 4.0, refer to the "Merging Documents" Help topic. (From the Window menu, choose Help.)
