WinWord: GP Fault on Module MSPDPBF.DRV or MSPDFF.DRV

 Last reviewed: February 5, 1998
Article ID: Q95074
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SYMPTOMS

A general protection (GP) fault error message occurs when you select Options from the Tools menu in Word for Windows when you are using the video driver for the Diamond Scan Super VGA, installed for 256 colors. The module noted in the GP fault error message is MSPDPBF.DRV (big fonts) or MSPDFF.DRV (small fonts). This also occurs with the Diamond Speed Star 24X, installed for 256 colors version 2.02. The module noted in the GP fault error message is WSPDSF.DRV or WSPDPSF.DRV.

STATUS

Diamond Computer Systems has confirmed that the Diamond Scan Super VGA video display, installed for 256 colors, produces a GP fault in Word when you select Options from the Tools menu if the video driver file has a date earlier than January 1993. Video drivers dated January 1993 or later do not cause this problem. To obtain the updated video driver, call Diamond Computer Systems Technical Support at:

  • Technical support: (408) 325-7100
  • BBS support 2400 only: (408) 325-7080
  • BBS support 9600 only: (408) 325-7175

WORKAROUND

While waiting for your updated video driver to arrive, you can avoid this error by selecting the 16-color video display option using the Windows version 3.1 Setup program.
