Word: Err Msg: The Toolbar Preference File Is Locked...

 Last reviewed: July 30, 1997
Article ID: Q93408
The information in this article applies to:
  • Microsoft Word for Macintosh, version 5.1

SUMMARY

The following error message sometimes occurs in Word for the Macintosh when you choose the Addresses button in the Create Envelope dialog box: 

   The Toolbar Preferences file is locked, so your changes will not be
   saved. Do you want to use the Address editor anyway?
You can choose the OK button and print your envelope, but Word does not save your address (note that the Define button is unavailable).

MORE INFORMATION

To prevent the above error message, you must unlock the Word Toolbar Prefs file, as described below:

  1. In System 7.0, open the Preferences Folder, located in the System Folder, and click the Word Toolbar Prefs file once (in System 6.0x, the Word Toolbar Prefs file is located in the System Folder).

  2. From the File menu, choose Get Info.

  3. Clear the Locked check box.

This error may occur if the Word Toolbar Prefs file has become corrupted. If this occurs, Quit Word and remove the Word Toolbar Prefs file from the Preferences folder (System folder in System 6.x). Crashing while in the Create Envelope dialog box is a reason for corruption of this file.

The Word Toolbar Prefs file contains the addresses for the Create Envelope feature. Because this file is a binary file, it cannot be opened to retrieve the lost addresses. Periodic backups of this file are recommended.
