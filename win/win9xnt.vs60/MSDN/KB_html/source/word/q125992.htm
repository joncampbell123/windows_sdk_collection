SGML: Error Reading/Writing Building Association MAP File

 Last reviewed: February 5, 1998
Article ID: Q125992

The information in this article applies to:

  • Microsoft SGML Author for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

While building an association MAP file, you receive one or both of the following errors: 

   Error Reading or Writing

   -or-

   Insufficient space to complete the operation

CAUSE

There is insufficient space on the hard drive to build the temporary files needed during the build of the association file. The temporary swap files can grow as large as 50 megabytes (MB), while the mapping application can grow as large as 20 MB.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SGML Author for Word, version 1.0 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

To resolve this problem, free up a large amount of disk space.
