Word Hangs If You Apply Change Case Formatting in Outline View

 Last reviewed: February 5, 1998
Article ID: Q110940
6.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, version 6.0

SYMPTOMS

Word for Windows may hang (stop responding) if you use the Change Case command on the Format menu when in outline view.

CAUSE

Word hangs when you choose the Change Case command from the Format menu if all five of the following conditions are true:

  1. You are in outline view.

  2. The outline is collapsed so only Level 1 headings are visible. (To make only Level 1 visible, choose the 1 button on the Outlining toolbar.)

  3. The document contains at least one Heading 1 paragraph and at least one paragraph formatted with a style other than Heading 1.

  4. The selected text is formatted with the Heading 1 style and begins with a capital letter (or you previously used the Change Case command to apply Title Case formatting to the selection).

  5. The selection includes the Heading 1 paragraph mark.

Note: This problem does not occur if you do not select the Heading 1 paragraph mark.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows. This problem was corrected in versions 6.0a and later of Word for Windows.

WORKAROUNDS

Method 1 - Expand the outline to display at least the Level 2 headings.

Method 2 - Do not select the paragraph mark in your Heading 1 paragraph 

           when you use the Change Case command on the Format menu.
