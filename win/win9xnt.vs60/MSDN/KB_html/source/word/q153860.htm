WD: Versions of Internet Assistant Available for Download

 Last reviewed: February 3, 1998
Article ID: Q153860
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0.1
  • Microsoft Word for Windows, versions 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SUMMARY

There are several versions of Internet Assistant for Word available on the World Wide Web and other online services. The following table outlines these versions, describes which versions of Word are required for each version of Internet Assistant, and lists their respective file names and download locations.

NOTE: Microsoft Word 97 for Windows and Microsoft Word 98 Macintosh Edition have build-in web publishing features that replace the functionality of the Internet Assistant. Internet Assistant is not needed for use with these products and will generate an error if you attempt to install it.

For information about how to download these files, see the "To Download These Files" section of this article.

Version: 2.04z

Filename: Wdia204z.exe

Internet Location: http://www.microsoft.com/word/internet/ia/sysreq.htm

This version is for use with Word for Windows 7.0 (95) and Word 6.0 for Windows NT (32-bit versions of Word only). This version is compatible with Internet Explorer 3.0.

This version works with the following language versions of 32-bit Word: 

   Czech            French Canadian    Polish                  Swedish
   Danish           German             Portuguese (Brazilian)  Swiss German
   Dutch (Standard) Greek              Portuguese (Standard)   Turkish
   English          Hungarian          Russian
   Finnish          Italian            Slovenian
   French           Norwegian (Bokmal) Spanish (Traditional)

Version: 2.0

Filename: Macia.hqx

Internet Location: http://www.microsoft.com/word/internet/ia/sysreq.htm

This version is for use with Microsoft Word for Macintosh, version 6.0.1.

Version: 1.0z

Filename: Wordia.exe

Internet Location: http://www.microsoft.com/word/internet/ia/sysreq.htm

This version is for use with Word 6.0a and 6.0c for Windows (16-bit versions of Word only)

This version works with the following language versions of 16-bit Word: 

   English           French              German
