OPP: Show Me for Backspace Tip Is Blank

 Last reviewed: February 6, 1998
Article ID: Q132213
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Windows 95

SYMPTOMS

When you backspace several times (about 30) and the Tip Wizard button lights up, the Show Me button is blank or says "No Help topic is associated with this item."

STATUS

This is a known problem with the Office Preview Program release only. It will be fixed in the final version of Word for Windows 95.
