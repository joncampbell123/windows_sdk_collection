GP Fault Starting or Using Word 6.0 Under Windows NT 3.1

 Last reviewed: February 5, 1998
Article ID: Q122395
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows NT version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you start Microsoft Word version 6.0 for Windows under Windows NT version 3.1, you receive a General Protection (GP) fault.

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server version 3.1. This problem was corrected in the latest U.S. Service Pack for Windows NT and Windows NT Advanced Server version 3.1. For information about obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
The problem can also be corrected by upgrading to Windows NT Workstation version 3.5 or Windows NT Server version 3.5.
