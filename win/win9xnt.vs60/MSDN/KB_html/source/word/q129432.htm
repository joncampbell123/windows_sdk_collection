WD6: How to Speed Up Internet Assistant

 Last reviewed: February 5, 1998
Article ID: Q129432
The information in this article applies to:
  • Microsoft Internet Assistant, version 1.0, 1.0z
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SUMMARY

This article lists steps you can take to make Internet Assistant faster.

Cancel Import of Graphics

Internet Assistant can take a long time to import graphics in an Internet document. To speed up the process, import the text of the Internet document without the graphics. To do this, choose Load Images from the View menu. This setting will remain in effect until you turn it off.

Speed Up Internet Assistant

Other steps you can take to improve Internet Assistant's performance in Web Browse view:

  • Turn off Load Images (located on the View menu).
  • Choose Full Screen from the View menu.
  • Implement suggestions from SPEEDUP.DOC (located in the subdirectory where Word 6.0c for Windows is installed). SPEEDUP.DOC is also available through Microsoft Product Support Services.
