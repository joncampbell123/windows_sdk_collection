WD: Word Count Displays Incorrect Character Statistic

 Last reviewed: February 5, 1998
Article ID: Q108689
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

When you choose Word Count from the Tools menu in Word for Windows, the Characters statistic may change from the correct number to an incorrect number.

This error occurs with some documents after Word repaginates. Word displays the correct Character count prior to repagination. This problem has been noted in documents with the following characteristics:

  • Carriage Return/Line Feeds at the end of each line (Word 2.x documents).
  • Documents that contain hidden text.
  • Documents that contain fields.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
