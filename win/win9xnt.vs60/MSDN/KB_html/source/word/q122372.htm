WD: Cannot Paste Framed Graphic into Another Application

 Last reviewed: February 5, 1998
Article ID: Q122372
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a

SYMPTOMS

Sometimes you cannot paste a graphic you copied from Microsoft Word into another program (such as Microsoft Excel).

CAUSE

This problem occurs when frame formatting is applied to the graphic in Word.

WORKAROUND

To work around this problem, use any of the following methods.

Method 1

Remove the frame formatting from the graphic before you copy it.

Method 2

To paste the graphic, click Paste Special on the Edit menu and then select Picture in the As box.

Method 3 (Macintosh Only)

Clear the "Include Formatted Text in Clipboard" option before you copy the graphic. (To display the "Include Formatted Text in Clipboard" option, click Options on the Tools menu and then select the General tab.)

NOTE: When you disable this option, Word copies text to the Clipboard without formatting.

STATUS

Microsoft has confirmed this to be a problem in the products listed above. This problem was corrected in Word version 6.0 for Windows NT.
