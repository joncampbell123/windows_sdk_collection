WD: Word 5.x and 6.x Converts Word 98 Documents as Text Only

 Last reviewed: March 3, 1998
Article ID: Q181933
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a, 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

Microsoft Word 5.x and 6.x import Microsoft Word 98 Macintosh Edition documents as Text Only, even though the Word 97-98 Import Converter is in the Word Extensions folder (in Word 6.x), or the Word Commands folder (in Word 5.x).

CAUSE

The Word 97-98 Import Converter included with drag and drop installations on the Microsoft Office 98 CD-ROM is actually for use with Microsoft PowerPoint, not Microsoft Word.

RESOLUTION

To install the correct version of the converter for Word 5.x and 6.x, run the Word 97-98 Converter Installer.

You can find the Word 97-98 Converter Installer in the Office 98 CD-ROM in the Value Pack:Text Converters folder, or you can download it from the following Microsoft World Wide Web site: 

   http://www.microsoft.com/macoffice/prodinfo/office/coexist.htm

MORE INFORMATION

How to Identify the Correct Version

To determine whether you have the correct version of the converter, follow these steps:

  1. Click once to select the Word 97-98 Import Converter file.

  2. On the File menu, click Get Info.

  3. Check the Size, Created, and Modified dates.

    The incorrect version from the drag and drop install has the following attributes: 

          Size: 277,193 bytes
      Created: Fri, Oct 31, 1997, 9:49 AM
      Modified: Sat, Jan 17, 1998, 5:15 PM

    The version that is installed with the Word 97-98 Converter Installer has the following attributes: 

          Size: 503,579 bytes
      Created: Sat, Jan 17, 1998, 5:12 PM
      Modified: Sat, Jan 17, 1998, 4:50 PM

NOTE: You cannot make identification by version number alone, because the numbers are exactly the same.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
