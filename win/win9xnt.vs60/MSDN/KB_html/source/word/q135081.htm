Hang, Invalid Page Fault Inserting Document

 Last reviewed: August 5, 1997
Article ID: Q135081
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you create a document that contains only a text box and you insert that document into itself, your system hangs or you receive the following application error: 

   This program has performed an illegal operation and will be shut
   down. If the problem persists, contact the program vendor.

   Winword caused an invalid page fault in module Winword.exe at
   0137:5000fadc.
This behavior does not occur when the document does not contain a text box.

WORKAROUND

Create a copy of the document before you insert it into itself.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95, version 7.0. This problem was corrected in Microsoft Word 97 for Windows.

MORE INFORMATION

This behavior occurs when the original document contains only a text box and it is inserted into itself.
