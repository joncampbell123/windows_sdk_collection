WD6X: Adobe Type Reunion Not Functional in Word 6.0 Ribbon

 Last reviewed: November 17, 1997
Article ID: Q121378
The information in this article applies to:
  • Microsoft Word for Macintosh, version 6.0

SYMPTOM

When Adobe Type Reunion 1.1 or later is installed on your computer, it does not group the fonts displayed in the Font list on the Microsoft Word Formatting toolbar.

CAUSE

The MRU (most recently used) feature in the Word 6.0 font list prevents modification of the font list, and, therefore, Adobe Type Reunion is unable to grouping the fonts.

This behavior is part of the design of Word 6.0.

RESOLUTION

To resolve this problem, upgrade to Microsoft Word for the Macintosh version 6.0.1.

To temporarily work around this problem, use any of the following methods.

Method 1: Use an Advanced Setting

The Macintosh release of Word 6.0 includes an Advanced Setting that, when added, will allow Adobe Type Reunion to function correctly. To do this, follow these steps:

  1. On the Tools menu, click Customize.

  2. Select the Menus tab.

  3. From the Tools list, select Tools.

  4. From the Commands list, select ToolsAdvancedSettings.

  5. Click Add, and then click Close.

  6. On the Tools menu, click Advanced Settings.

  7. In the Option box, type "NoFontMRUList" (without the quotation marks).

  8. In the Setting box, type "Y" (without the quotation marks).

  9. Click the Set button and then click the OK button.

Method 2: Add Adobe Type Reunion to the Font Menu

Adobe Type Reunion also functions correctly from the Font menu when it is added to Word 6.0. TO do this, follow these steps:

  1. Open the 5.1 Upgrade template, which is located in the Macros folder.

  2. On the Tools menu, click Macro.

  3. Run the AddFontMenu macro.

  4. Click Yes to confirm the change to the menu.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word for Macintosh, version 6.0. This problem was corrected in Microsoft Word for Macintosh, version 6.0.1.

MORE INFORMATION

Adobe Type Reunion is a System Extension that sorts and groups fonts according to family name. All variations of a particular font appear with one name in the Font menu. Individual fonts appear in a submenu adjacent to the alphabetical list of family names.
