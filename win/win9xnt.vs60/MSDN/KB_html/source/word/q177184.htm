WD98: Character Formatting Keyboard Commands in MacWord

 Last reviewed: February 17, 1998
Article ID: Q177184
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

This article contains a list of keyboard shortcuts available in Microsoft Word 98 Macintosh Edition. This list is a compilation of the individual keyboard shortcut lists available in Word Help.

NOTE: Shortcut keys mentioned in the Help topics, menus, and dialog boxes refer to the U.S. keyboard layout. Keys on other layouts may not correspond exactly to the keys on a U.S. keyboard. If you are using keyboard layout different from the version of Word you are using, you may have to make adjustments when using shortcut keys.
