WD98: Undoing Grouped Linked Text Boxes Causes MacWord to Quit

 Last reviewed: February 17, 1998
Article ID: Q177229
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you use the Undo button to undo (all at once) the following three actions,

  • Group Objects 

        -and-
  • Insert Text Box 

        -and-
  • Text Box Linking

you may receive the following error: 

   The application "Microsoft Word" has unexpectedly quit, because an
   error of type 2 occurred. You should save your work in other open
   applications and restart the computer.
Clicking OK closes Microsoft Word without allowing you to save any work.

WORKAROUND

To work around this problem, use either of the following methods.

Method 1: Use the Undo Button

Click the Undo button three separate times, to undo each of the three actions one at a time.

Method 2: Use the Edit Menu

  1. On the Edit menu, click Undo Group Objects.

  2. On the Edit menu, click Undo Insert Text Box.

  3. On the Edit menu, click Undo Text Box Linking.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

This problem occurs when you perform these three actions in the following order with no other actions in between: 

   Group objects
   Insert Text Box
   Text Box Linking
If you perform any action in between one of the above actions, you can undo the actions all at once without an error.
