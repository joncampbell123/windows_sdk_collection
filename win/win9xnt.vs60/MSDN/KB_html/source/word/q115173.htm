WD6X: Create Labels Help Gives Incorrect Shortcut Keys

 Last reviewed: February 5, 1998
Article ID: Q115173
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

If you click Help in the Create Labels dialog box and select the "Using shortcut keys to format characters" topic, Help incorrectly states that you can use specific keystrokes to change the font size and other formatting in the Create Labels dialog box. The shortcut keys described in this topic do not function as described from within the Create Labels dialog box.

RESOLUTION

Use the appropriate method for your version of Word.

Word for Windows

To use a quick keystroke to format the text in the Create Labels dialog box, press CTRL+D to bring up a the font formatting menu.

Word for the Macintosh

Press COMMAND+D to open the Font formatting dialog menu. You can also create a custom keystroke and assign it for the paragraph formatting dialog box, the tab formatting dialog box, or other similar dialog boxes.

STATUS

Microsoft has confirmed this to be a documentation error in Microsoft Word. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
