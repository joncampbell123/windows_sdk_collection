WD97: Invalid Page Fault in PIMAB32.dll w/ Compaq Address Book

 Last reviewed: July 31, 1997
Article ID: Q165392
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Outlook 97

SYMPTOMS

If you are using the Compaq Address Book in Word 97, you may receive the following error message when you click Insert Address in the Envelopes and Labels dialog box (Tools menu) or when you run the InsertAddress macro. 

   Winword caused an Invalid Page Fault in PIMAB32.dll.
Note: You may also get this error message in Outlook 97

WORKAROUND

To workaround this problem, remove the Compaq Address Book from the user profile, using these steps:

  1. Close all applications.

  2. On the Windows Start menu, point to Settings, and then click Control Panel.

  3. Double-click the Mail or Mail And Fax Icon.

  4. On the Services tab, select Compaq Address Book from the list of services in the current profile and then click Remove.

  5. Click OK.

For more information on using the Address Book, please see the following article in the Microsoft Knowledge Base: 

     ARTICLE ID: Q134901
     TITLE     : How to Modify the Layout of an Address Book Entry
For more information about using the Outlook Contact List with Word, click the Office Assistant, type "contact list address," click Search, and then click to view "Insert an address from my personal address book."

NOTE: If the Assistant is hidden, click the Office Assistant button on the Standard toolbar. If Word Help is not installed on your computer, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q120802
   TITLE     : Office: How to Add/Remove a Single Office
               Program or Component
