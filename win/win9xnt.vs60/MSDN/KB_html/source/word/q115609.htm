How to Use "Take A Letter" with Word for Windows

 Last reviewed: February 5, 1998

The information in this article applies to:

  • Microsoft Word for Windows, version 2.0, 2.0a, 2.0b, 2.0c, 6.0, 6.0a
  • Microsoft Windows operating system, version 3.1

SUMMARY

This article describes how to use Take A Letter, version 1.0, from Centric Development, with Word for Windows.

MORE INFORMATION

System Requirements

   Microsoft Windows 3.1 or later
   At least 850K of hard disk space
   One 3.5-inch floppy disk drive

Installation

  1. Insert the Take A Letter installation disk in your floppy disk drive.

  2. From Program Manager File menu, choose Run.

  3. In the Command Line box, type "a:install" (without the quotation marks) (or "b:install," whichever is applicable) and choose OK.

Note: The setup program places all the system files in a directory on your hard disk called TAL unless you choose otherwise. During installation you will be prompted to select your word processor. Choose Word for Windows 2.0 or higher.

Starting and Using Take A Letter

Take A Letter allows you to browse through a selection of standard letters and guides you through how to fill in the blanks using suggestions and examples. To start Take A Letter, do one of the following:

  • To start Take A Letter from Windows, from Program Manager double-click the Take A Letter icon in the Take A Letter Group.
  • To start Take A Letter from Word for Windows, choose Take A Letter from the Tools menu.

Browse to find the letter of your choice and begin. Once you've completed the letter, choose the Send To Word Processor button, which will transfer control back to the word processor.

Uninstalling Take A Letter

To uninstall the program, from File Manager, run the uninstall file from the Take A Letter program directory (TAL).
