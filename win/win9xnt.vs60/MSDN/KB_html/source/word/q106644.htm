Find File Ignores Match Case for Title

 Last reviewed: February 5, 1998
Article ID: Q106644
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SYMPTOMS

In Word 2.x and 6.0 for Windows, Find File ignores the Match Case box for all fields except the Containing Text (version 6.0) or the Any Text (versions 2.x) field. Whether or not Match Case is selected, Word will find files that match what is typed in the Title, Subject, Author, Keywords, or Containing Text/Any Text fields, regardless of the case.

Match Case is used by the Containing Text/Any Text field.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To locate this dialog box in Word version 6.0:

  1. From the File menu, choose Find File.
2. Choose Search. 3. Choose Advanced Search. 4. Select the Summary tab.

To locate this dialog box in Word versions 2.x:

  1. From the File menu, choose Find File.
2. Choose Search.
