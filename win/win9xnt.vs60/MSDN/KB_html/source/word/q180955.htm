WD6X: Printing to a HP 4000 Printer from Microsoft Word

 Last reviewed: February 16, 1998
Article ID: Q180955

The information in this article applies to:

  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SUMMARY

The printing software for Macintosh that ships with the HP LaserJet 4000 family printer does not install a printer driver icon into the Chooser. This article identifies which icon should be highlighted to work with the printer description file (PPD) that was installed.

MORE INFORMATION

HP suggests that you use the system printing software already included on your Macintosh computer. Apple LaserWriter driver versions 8.3.4 and 8.4.1 are the recommended versions to work with the PPD provided by Hewlett- Packard. Both of these system printer drivers have been tested by Hewlett- Packard and are known to work with the HP LaserJet 4000 family printers when printing from a Macintosh computer.

Before printing successfully from HP 4000 using Word 6.x, the correct PPD (printer description file) must be selected using the Chooser.

Configuring the Printer Driver

In order to print correctly, you need version 8.3.4 or 8.4.1 or later of the Apple LaserWriter.

To configure the printer driver, follow these steps:

  1. On the Apple menu, click Chooser.

  2. Locate the printer driver for your LaserWriter and select it.

  3. Select the actual printer in the "Select a <Printer Driver Name> Printer" list.

  4. Click Setup.

  5. Click Auto Setup.

  6. Click OK.

    The PPD should be automatically selected for you.

    NOTE: If you receive a warning message that the correct printer description file (PPD) cannot be found, make sure that you have the PPD for your printer in the Printer Descriptions folder (either in the System folder or in the Extensions folder). If you ran the standard setup for the driver, this file should be properly installed.

  7. Click Configure.

  8. Choose the correct settings for your printer, and then click OK.

  9. Click OK twice.

    NOTE: If you do not know certain items of your printer's configuration, the Printer Info button in the Printer Setup dialog may provide the information. The dialog box will tell you whether or not certain options are installed including the types of networks supported and how much memory is installed in the printer.

  10. Close the Chooser.

NOTE: HP LaserJet 4000 series printers use PostScript Level 2 emulation. Hewlett-Packard has not licensed Adobe Postscript from Adobe for use with this family of printers. HP cannot and does not recommend use of Adobe printer drivers with the HP LaserJet 4000 printer family.
