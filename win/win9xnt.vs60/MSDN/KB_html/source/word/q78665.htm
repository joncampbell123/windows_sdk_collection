EndNote Plus Integrates with Word 5.1 for the Macintosh

 Last reviewed: July 30, 1997
Article ID: Q78665
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0, 6.0.1
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c,6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, version 7.0

SUMMARY

EndNote Plus Plug-in Module, EndNote Plus, and EndNote 2 Plus are software packages for building a bibliography and formatting the citations in a variety of bibliographic styles. This software works with Word to insert citations and references in Word documents.

As a bibliography maker, it builds lists of cited works automatically. EndNote Plus scans a document for text citations and compiles a bibliography using the information in your reference database. It then produces a copy of the document, reformats the text citations with a reference marker, and places the bibliography at the end of your document.

For example, the EndNote Plus Plug-in Module (for use with Word for the Macintosh) adds three commands to the Word menus:

  • The EndNote Plus Module command, which appears on the Tools menu, opens your library of references.
  • The Insert Citation and Format Citations commands, which appear on the Insert menu, allow you to point to a reference in the library and use the Insert Citation command to insert an embedded object into the text as you write. At any time, you can use the Format Citations command to create a formatted bibliography based on the embedded citations in the text. You can continue working on the same document, inserting more embedded citations, and formatting the bibliography as you proceed.

The EndNote Plus Plug-in Module is also flexible in how it formats footnotes: you can specify whether you want citations in footnotes to be formatted as citations or as full bibliographic references.
