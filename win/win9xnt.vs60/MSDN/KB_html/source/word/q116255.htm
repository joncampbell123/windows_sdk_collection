Fonts Display as Black Boxes on Radius Pivot Monitor

 Last reviewed: February 5, 1998
Article ID: Q116255
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Word Assistant, version 1.0
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

In Word for Windows, some fonts in the As Font dialog box display as black boxes if you are using a Radius Pivot display monitor. (To locate this dialog box, choose Font Selector from the Format menu, and select the As Font tab. The Font Selector command appears on the Format menu when you install Microsoft Word Assistant.)

CAUSE

In Word, the As Font dialog box displays only vector fonts (such as the Modern, Roman, or Script fonts supplied with Windows) on the Radius Pivot monitor. All other, non-vector fonts display as black boxes. This happens in either orientation (vertical or horizontal) on the Radius Pivot monitor.

This problem occurs only in the Word Assistant Font Selector dialog box. On the Radius Pivot monitor, non-vector fonts display correctly in Word's Format Font dialog box and in all the Microsoft Font Assistant dialog boxes.

By contrast, all non-vector fonts in the Font Selector dialog box display correctly on other display monitors.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
