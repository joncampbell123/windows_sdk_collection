DMM: Setup Fails to Find Word with Run-From-CD Install of Office

 Last reviewed: March 4, 1998
Article ID: Q179960
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SYMPTOMS

If you perform a Run-From-CD installation of Microsoft Office 97 and then attempt to install Microsoft Direct Mail Manager, you will receive the following error message: 

   Direct Mail Manager could not find Microsoft Word 97 on this computer.
   Direct Mail Manager can not operate without Word 97. Please install (or
   reinstall) Word 97 from the Microsoft Office 97 Small Business Edition
   CD-ROM.

CAUSE

Microsoft Direct Mail Manager was designed to be part of the Microsoft Office 97 Small Business Edition, which does not support a Run-From-CD installation option.

RESOLUTION

Run the Office 97 Small Business Edition Setup and install Microsoft Word 97.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
