Word Does Not Check Available Disk Space Before Saving File

 Last reviewed: February 5, 1998
Article ID: Q110447
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

If you have less disk space available than is necessary for Word to save the document, Word attempts to save the document anyway and displays an error message stating that the disk is full.

Word uses this design, rather then checking for sufficient disk space before saving the file, since there is no reliable way for Word to know how big a file will be, or whether there will be enough room for it with "on the fly" compression from products like MS-DOS version 6 and later, and Stac Electronics Stacker.
