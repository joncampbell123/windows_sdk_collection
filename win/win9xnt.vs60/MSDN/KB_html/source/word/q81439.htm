Word for Windows: Soft Type Fonts Not Displaying on the Ribbon

 Last reviewed: July 30, 1997
Article ID: Q81439
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

After installing Microsoft Word for Windows, ZSoft's Soft Type fonts may not all display on the ribbon even though the soft fonts do display in other applications.

MORE INFORMATION

This problem should occur only when using Soft Type version 1.0. According to ZSoft, version 1.0 has several stability problems running under Microsoft Windows version 3.0; therefore, you may also see this problem in other Windows applications (such as Microsoft Excel). ZSoft recommends upgrading to the latest version of Soft Type, version 1.1. To obtain this upgrade, call ZSoft Sales at (404) 428-0008.

Soft Type is manufactured by ZSoft, vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

REFERENCES

ZSoft technical support, (404) 428-0008
