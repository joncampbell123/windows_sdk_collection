WD: Word for Windows Returns Incorrect Number of Columns

 Last reviewed: February 5, 1998
Article ID: Q77133
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SYMPTOMS

In Microsoft Word for Windows, if you select the rows of a column and the end-of-row markers and choose Table from the Format menu, Word returns an incorrect result for the number of columns.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word for Windows listed above. This problem was corrected in Word version 6.0 for Windows.
