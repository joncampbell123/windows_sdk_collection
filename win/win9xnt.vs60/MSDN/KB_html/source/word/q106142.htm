Word 6.0 Uses ODBC to Open dBASE Files

 Last reviewed: February 5, 1998
Article ID: Q106142
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SUMMARY

Word for Windows does not supply a dBASE converter. Word opens dBASE files using the Open Database Connectivity (ODBC) driver supplied with Word.

MORE INFORMATION

When you open a dBASE file in Word 6.0 using the ODBC driver, Word does not display records in the file that have been marked for deletion but that have not been removed using the dBASE Pack command. This means you do not have to pack your dBASE file before using it in Word (this step is necessary if you open a dBASE file in Word version 2.x for Windows because the Word 2.x dBASE converter displays deleted records in unpacked dBASE files).

For more information on the Word 2.x dBASE converter, query on the following words in the Microsoft Knowledge Base: 

   dbase and pack and Word and flagged
