Setup Installs Converters and Filters in Unexpected Locations

 Last reviewed: February 6, 1998
Article ID: Q113742
6.00 6.00a 6.00c WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

In some cases, the Setup program for Word 6.0 for Windows updates existing text converters or graphic filters instead of installing the new filters in the MSAPPS subdirectory. This frequently happens when you upgrade over an existing installation of Word 2.x for Windows.

This only happens when you perform a standalone (non-network) installation of Word, from a server or using the program disks. It does not happen when you perform a network installation of Word.

CAUSE

Word updates existing text or graphics filters instead of installing them in the shared MSAPPS directory when both of the following are true:

  • There is not an existing copy of the converter or filter in the MSAPPS\TEXTCONV or MSAPPS\GRPHFLT subdirectory of the Windows program directory.

    -and-

  • The PATH statement in your AUTOEXEC.BAT file includes a directory that contains a file with the same name as the new text converter or graphics filter.

Furthermore, if you have Word 2.x on your computer and install Word 6.0 in a different directory, Word 6.0 Setup updates all the filters and converters in the Word 2.x directory that have the same filename as the Word 6.0 files.

If you have a separate Word 1.x for Windows directory, Word 6.0 Setup updates the PCX and EPS filters. In addition, Setup adds the IFFPCX.DLL, MS.EPS, and MS.PCX files to your Word 1.x directory. Word 6.0 Setup installs the rest of the converters and filters in the MSAPPS subdirectory, as expected.

WORKAROUND

Edit the PATH statement in your AUTOEXEC.BAT file to temporarily remove the directories that contain existing converters or filters. Restart your computer so the changes can take effect, then install Word for Windows. After you successfully install Word, you can add these directories back to your PATH statement. These entries do not affect the operation of Word 6.0.
