WordBASIC Macro to Run the Windows File Manager

 Last reviewed: July 30, 1997
Article ID: Q71342
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

If you need to do a file management task when working in Microsoft Word for Windows, such as moving files, you can run the following macro to bring up the Windows version 3.0 File Manager: 

   Sub Main
   Shell "winfile.exe"
   End Sub
After you complete your file management task, you can exit File Manager and resume Word for Windows again.

Note: You can assign this macro to a key or menu in Word for Windows for quick access to the File Manager.
