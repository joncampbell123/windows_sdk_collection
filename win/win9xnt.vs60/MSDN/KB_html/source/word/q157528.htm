WD97: "Outline Effect" Text Garbled or in All Caps When Printed

 Last reviewed: July 31, 1997
Article ID: Q157528
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you print a document, text formatted with the Outline font effect may be garbled or may appear in all capital letters.

NOTE: To format text with the Outline font effect, click Font on the Format menu, click the Font tab, and click to select the Outline check box under Effects.

CAUSE

This problem may occur when you print to one of the following printers:

  • Hewlett-Packard DeskJet 660C or Hewlett-Packard 1600 printers, using the legacy printer drivers.
  • Hewlett-Packard LaserJet 4 or 5 series PostScript printers
  • Hewlett-Packard Color LaserJet 5/5m PostScript printers
  • Hewlett-Packard Color LaserJet 5/5m non-PostScript (HPPCL) printers

    NOTE: Text formatted as Outline is printed as solid text.

  • Tektronix Phaser 550c
  • Lexmark Optra PS

WORKAROUND

To work around this problem, use a non-PostScript printer driver.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
