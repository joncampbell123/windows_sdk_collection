"File Permission Error" Saving Document with Leading Space

 Last reviewed: February 5, 1998
Article ID: Q140743
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows 95 operating system

SYMPTOMS

You get the following error message when you try to save a Word 6.0 document under Windows 95 

   Word cannot complete the save due to a file permission error
   <filename>.
where <filename> is the name of your document.

This problem does not occur with Word version 2.0 for Windows.

CAUSE

The first character of the file name is a space.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

Remove the leading space from the file name.
