Documents Print Garbage on DeskJet After Word 6.0a Upgrade

 Last reviewed: February 5, 1998
Article ID: Q118443
6.00a 6.00c WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

After you install Word 6.0a for Windows or upgrade Word 6.0 for Windows to 6.0a using the Upgrade Patch, documents may print to an HP DeskJet series printer with "garbage" (corrupt characters) or vertical bars.

CAUSE

Word 6.0a for Windows and the 6.0a Upgrade Patch update the Universal Printer Driver (UNIDRV.DLL) to version 3.1.4. This version of the Universal Printer Driver is incompatible with the DeskJet series driver (HPDSKJET.DRV) version 1.1.

The problem does not occur with later versions of the Universal Printer Driver, or with later versions of the DeskJet series driver (HPDSKJET.DRV version 2.0 or later, or DESKJETC.DRV versions 3.1 or later).

Versions 3.1 or later of the deskjet driver (DESKJETC.DRV) do not use the Universal Printer Driver, while the original deskjet driver (HPDSKJET.DRV version 1.1) does.

STATUS

Microsoft has confirmed that there is an incompatibility between the HPDSKJET.DRV version 1.1 and the UNIDRV.DLL version 3.1.4, which ships with Microsoft Word version 6.0a for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

Contact HP for an update to the DeskJet series driver (DESKJETC.DRV) version 3.1 or later by calling (303) 353-7650.

WORKAROUND

Set the Print resolution to 150 DPI.

  1. From the File menu, choose Print

  2. Choose the Printer button.

  3. Choose the Options button.

  4. Select Draft Quality, from the Print Quality list box.

For additional information, please see the following article(s) in the Microsoft Knowledge Base:

     Title:  TrueType Fonts Print Incorrectly on HP DeskJet 500
