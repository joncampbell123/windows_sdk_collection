WD98: Menus Do Not Work If TypeTamer Control Panel Installed

 Last reviewed: February 17, 1998
Article ID: Q178600
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you click a menu in Microsoft Word, you are unable to see the commands on the menu.

CAUSE

This problem can occur if you are using TypeTamer, version 1.0.6i, a control panel device made by Impossible Software.

WORKAROUND

To work around this problem, contact Impossible Software to find out if they have an updated version of the control panel. To contact Impossible Software, connect to the following Web site: 

   http://www.impossible.com/
