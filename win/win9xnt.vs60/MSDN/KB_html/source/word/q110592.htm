Word Assistant Help Doesn't Display If No Documents Are Open

 Last reviewed: February 5, 1998
Article ID: Q110592
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Word Assistant version 1.0

SYMPTOMS

In Word 6.0 for Windows, if you have not yet opened a document, File and Help are the only available menus. Word Assistant normally adds a command to the Help menu called Word Assistant Help; however, this command does not appear if you have not opened any documents.

MORE INFORMATION

Word Assistant uses the WASSTHLP.WLL add-in to modify the Help menu. Word- link libraries (WLLs) are similar to templates in that they need to be attached to a document to activate all settings. When a document is opened, the .WLL file is invoked and the Help menu is modified.
