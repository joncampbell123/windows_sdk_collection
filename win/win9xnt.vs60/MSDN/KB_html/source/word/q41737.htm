Position Formatting Ignored When Printing from Outline View

 Last reviewed: July 30, 1997
Article ID: Q41737
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0

SUMMARY

In Microsoft Word for Windows, if you print from outline view, position formatting is ignored. Absolutely positioned objects (APOs) are included in the document as normal text. To Print a Document with the APO's positioned properly, use normal or page layout view
