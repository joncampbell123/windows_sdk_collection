WD98: How to Turn Off Auto Features

 Last reviewed: February 3, 1998
Article ID: Q179225
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

This article describes how to turn off the auto features that run in the background in Microsoft Word 98 Macintosh Edition. It gives a very brief description of each feature and instructions for turning it off.

MORE INFORMATION

AutoCorrect

This feature watches as you type and corrects common errors automatically without your intervention. The AutoCorrect options are:

  • Correct TWo INitial CApitals
  • Capitalize first letter of sentences
  • Capitalize names of days
  • Replace text as you type

You can control all of these options by clicking AutoCorrect on the Tools menu and then clicking the AutoCorrect tab.

AutoFormat As You Type

This feature watches as you type and applies some types of formatting automatically without your intervention. The options are as follows.

Apply As You Type:

  • Headings
  • Borders
  • Tables
  • Automatic bulleted lists
  • Automatic numbered lists

Replace As You Type:
  • "Straight quotes" with "smart quotes"
  • Ordinals (1st) with superscript
  • Symbol characters (--) with symbols (-)
  • *Bold* and _italic_ with real formatting
  • Internet paths with hyperlinks

Automatically As You Type:
  • Format the beginning of list item like the one before it
  • Define styles based on your formatting

You can control all of these options by clicking AutoCorrect on the Tools menu and then clicking the "AutoFormat As You Type" tab.

AutoText

This feature watches as you type and shows you an AutoComplete tip for AutoText and dates automatically without your intervention. The AutoText options are:

  • Show AutoComplete tip for AutoText and dates

You can control all of these options by clicking AutoCorrect on the Tools menu and then clicking the AutoText tab.

Spelling and Grammar

This feature watches as you type and marks misspelled words with a red underline and grammar errors with a green underline on the fly with no operator intervention. The Spelling and Grammar options are as follows.

Spelling:

Check spelling as you type

Grammar:

Check grammar as you type

You can control all of these options by clicking Preferences on the Tools menu and then clicking the Spelling & Grammar tab.

Editing

These features do the following:

  • Replace any selected item with the next character typed
  • When using the mouse to select text selects entire words only
  • Automatically adjusts the spaces before and after a cut or paste
  • Watches as you type and changes the left paragraph indent under certain circumstances.

The Edit options are:
  • Typing replaces selection
  • When selecting, automatically select entire word
  • Use smart cut and paste
  • Tabs and backspace set left indent

You can control all of these options by clicking Preferences on the Tools menu and then clicking the Edit tab.

General

These features do the following:

  • Automatically update links opening a file
  • Periodically repaginates the document in normal view
  • Warns about macros in a document that is being opened.

The General options are:
  • Background repagination (only available in normal view)
  • Update automatic links at Open
  • Macro virus protection

You can control all of these options by clicking Preferences on the Tools menu and then clicking the General tab.

Print

This feature automatically updates fields and links when printing or print previewing. The Printing options are:

  • Update fields
  • Update links

You can control all of these options by clicking Preferences on the Tools menu and then clicking the Print tab.

Save

These features automatically do the following:

  • Save a backup copy of a file
  • Append changes to the end of the file rather than rewriting the entire file
  • Save the data only for forms
  • Determine the default file format to save in
  • Create a temporary back up file in case the computer stops responding

The Save options are:
  • Always create backup copy
  • Allow fast saves
  • Save data only for forms
  • Save Word files as:
  • Save AutoRecover info every <n> minutes

You can control all of these options by clicking Preferences on the Tools menu and then clicking the Save tab.
