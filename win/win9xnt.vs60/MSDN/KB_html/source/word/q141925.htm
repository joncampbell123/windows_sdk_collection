WD: How to Change Table Width After Changing Margins

 Last reviewed: February 3, 1998
Article ID: Q141925
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SUMMARY

When you first create a table in Word, the table width fits exactly between the margins. If you change the margins, the table does not automatically resize to fit the new settings.

This article explains how to make the table conform to the new margins.

MORE INFORMATION

After changing the margins, follow these steps to adjust the table:

  1. On the Table menu, click Select Table.

  2. On the Table menu, click Cell Height And Width.

  3. Click the Columns tab.

  4. In the Width Of Columns box, select Auto. Click OK.

The table now is as wide as the typing area.
