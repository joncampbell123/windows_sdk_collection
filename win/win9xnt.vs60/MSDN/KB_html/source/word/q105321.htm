SUMMARY

The following is the contents of the Word 6.0 revision of the Word for Windows Supplemental Offers Coupon. This coupon is included in each Word 6.0 retail package.

Microsoft Word for Windows Supplemental Offers

Congratulations on choosing Microsoft (R) Word version 6.0 for Windows (TM). Now you can order any of the five supplemental offers listed below to help you get even more out of Word for Windows. To order any of the supplemental offers, please complete the form on the other side of this page

  • 5.25" high-density (1.2 MB) Disks for Microsoft Word for Windows, free of charge
  • 3.5" low-density (720K) Disks for Microsoft Word for Windows, free of charge

- Supplemental File Conversion Disks, free of charge - Conversions 
  for most major file and graphic formats are included with your
  Microsoft Word version 6.0 for Windows package. But to work with
  the file and graphic formats listed below, you can order
  supplemental conversion disks [one 5.25" disk (1.2 MB) and one 3.5"
  disk (1.44 MB)] free of charge. File conversions include
  DisplayWrite (R) MultiMate (R) Microsoft Publisher 2.0, Microsoft
  Works for the MS-DOS (R) operating system, Microsoft Works for
  Windows, Microsoft Multiplan (R) WordStar (R) WordPerfect (R) 4.1
  and 4.2 for MS-DOS systems, and Lotus (R) 1-2-3 (R) WK1 and WK3 2.x
  and 3.x (read only). Graphics filters include AutoCAD plot file,
  Corel (R) DRAW 3.0, GIF, HPGL, Lotus 1-2-3 Graphics, Targa, and
  Photo-CD.
  • Switching Guide, free of charge - "Switching to Microsoft Word for Windows: For Users of DisplayWrite, MultiMate, WordPerfect, and Microsoft Word for MS-DOS (R) based PCs." Contains information to help users make the transition to Microsoft Word for Windows.
  • Microsoft Word 6.0 Legal Resource Kit, free of charge - Provides product tour of legal features, conversion information, Word 6.0 macros for direct access to Lexis (R) Nexis (R) services, and additional information on third-party legal products and solution providers.

    If you have any questions about this offer, please call the Microsoft Sales Information Center at (800) 426-9400. Deaf or hard-of-hearing customers can call Microsoft Text Telephone (TT/TDD) services at (800) 892-5234.

    Offer from Microsoft Press (R)

    Microsoft Press books are available wherever computer books are sold or directly from Microsoft Press. To place a credit card order, call toll-free (800) MSPRESS or (615) 793-5090. Please refer to reference code FW6 when you order.

    Microsoft Word for Windows Resource Kit

    ISBN 1-55615-720-7, $39.95 (plus freight and applicable sales tax). Provides technical information and tools to help your organization quickly move to Microsoft Word for Windows. Includes installation, conversion, and customization information, as well as training tools and product tips and tricks.

    Microsoft Word Developer's Kit

    ISBN 1-55615-630-8, $39.95 (plus freight and applicable sales tax). Includes a complete discussion on using WordBasic to customize Word 6.0 to work the way you do, streamline routine tasks, or build cross- application solutions. The kit also includes the WordBasic function reference, sample macros on a disk, as well as advanced programming topics on using the Word API, Microsoft Messaging API (MAPI), and Open Database Connectivity (ODBC) with Word 6.0.

    Developer Information

    If you are a consultant or software developer working with Microsoft Word for Windows and would like more information, please call the Microsoft Developer Resource Group at (800) 227-4679, ext. 11771.

    More information on this topic may be available from Microsoft Information Services. Microsoft Information Services are available on the World Wide Web by connecting to http://www.microsoft.com. The FTP site is located at ftp.microsoft.com.
