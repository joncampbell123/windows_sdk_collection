WD: GetText$() Changes Section Breaks to Page Breaks

 Last reviewed: February 2, 1998
Article ID: Q107878
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

Section breaks retrieved with the macro commands GetText$ or Selection$ become page breaks when inserted into a document.

WORKAROUND

Use the WordBasic EditCopy or EditCut and the EditPaste commands to retrieve and insert section breaks.

STATUS

Microsoft has confirmed this to be a problem with Selection$ and GetText$ in Word for Windows versions 6.x-7.x, Word versions 6.x for the Macintosh, and with Selection$ in Word 2.x for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
