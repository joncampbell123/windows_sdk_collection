WD: Copy Button Unavailable in Organizer Dialog Box

 Last reviewed: February 2, 1998
Article ID: Q105713
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Word for the Macintosh version 6.0
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

The Copy button is unavailable in the Organizer dialog box if any of the following conditions are true for the selected destination document or template:

  • It is protected (either for annotations or forms).
  • It is password protected.
  • It is read-only.
  • It is located on a network drive to which you do not have sufficient user privileges.

NOTE: The destination file is the document or template where you want to copy the style, macro, or AutoText.

WORKAROUND

Unprotect the protected document or remove the read-only file attribute. On a network, make sure the user has full rights to the document, or move the document to a different location.

MORE INFORMATION

To open the Organizer dialog box, click Organizer in the Format Style, File Template, or Tools Macro dialog boxes.
