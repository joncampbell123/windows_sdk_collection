Doc Err: ATI Driver Versions Incorrect in Word Readme Help

 Last reviewed: February 5, 1998
Article ID: Q110807
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

The Word 6.0 for Windows Readme Help file incorrectly reports that using the following driver versions will correct display problems associated with these video cards when in Word version 6.0: 

    Readme incorrectly states:                                Should say:
    -------------------------                                 -----------

    ATI MACH 32 video display driver version 2.1 or later     2.2
    ATI MACH 8 video display driver version 2.2 or later      2.0
To locate this incorrect information, open the Word Readme help file and choose Word for Windows Product Update. Select Screen Display and Video Drivers from the resulting list. The incorrect information can be found in the section titled "ATI display driver."
