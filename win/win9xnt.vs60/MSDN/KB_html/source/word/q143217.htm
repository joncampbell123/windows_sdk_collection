IA: Trailing Underline Added to Hyperlink Text

 Last reviewed: February 5, 1998
Article ID: Q143217
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 2.0z
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

When you create a hyperlink, Internet Assistant automatically underlines one space following the text. This problem occurs when you select some text and then click Hyperlink on the Insert menu.

WORKAROUND

To work around this problem, use one of the following methods.

Method 1: Use the following steps to type the text you want to be displayed 

          in the Hyperlink dialog box:

          a. Position the insertion point where you want to create the
             hyperlink, and then click Hyperlink on the Insert menu.

          b. In the Text To Display box, type the text you want to be
             displayed as your hyperlink, specify a file or URL, and then
             click OK.
Method 2: Use the following steps to delete the underlined space after you 
          mark a selection of text as a hyperlink:

          a. Click the HTML Hidden button to display the field codes.

          b. Delete the space immediately before the closing field code
             brace.

          c. Click the HTML Hidden button to turn off Field codes view.

          d. Add a space between the hyperlink field and any subsequent
             text, if necessary.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Assistant for Word, version 2.0z. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
