Removing Obsolete Word 2.0 Sections from the WIN.INI File

 Last reviewed: February 5, 1998
Article ID: Q110195
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SUMMARY

After installing Microsoft Word 6.0 for Windows, you may want to remove the sections in the WIN.INI file related to Word 2.0 for Windows. If you are only running Word 6.0 for Windows, you can delete the settings below the following headings: 

 [Microsoft Word 2.0]
 [MSWord Editable Sections]
 [DCAConv]
 [TextLytConv]
 [PCWordConv]
 [MacWordConv]
 [PCWorksConv]
 [MSWord Text Converters]
Delete the heading text that appears in square brackets and the settings below. For example, delete [Microsoft Word 2.0] and the settings below up until you reach the next heading (that is, [embedding]). 

  [Microsoft Word 2.0]
  INI-path=c:\winword
  programdir=D:\winword

  [embedding]
The majority of the Word 6.0 settings are now stored in either the WINWORD6.INI or MSTXTCNV.INI files located in your Windows directory (usually, "C:\WINDOWS").

Note: As always, make a backup copy of your WIN.INI file before making changes.
