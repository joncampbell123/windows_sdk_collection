Toolbars Disappear, No Graph After Inserting Graph 5 Object

 Last reviewed: February 6, 1998
Article ID: Q139776
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Graph 5.0

SYMPTOMS

When you try to insert a graph or chart object into a frame that contains another Word object, your toolbars may disappear, and the object may not be inserted, even after you update and/or exit Graph.

CAUSE

Graph 5.0 does not give other OLE applications the focus if they were started before you started Graph.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Graph version 5.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

Insert the Graph 5.0 object before you insert any other object.
