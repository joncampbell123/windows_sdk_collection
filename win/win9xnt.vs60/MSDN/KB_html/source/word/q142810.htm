WD: Gray or Colored Text Prints Black to Monochrome Printers

 Last reviewed: February 5, 1998
Article ID: Q142810
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a and 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

When you print to a monochrome printer under Windows 95, color or gray text will print as black, even though the same printer printed the text as gray under Windows 3.x.

CAUSE

This problem occurs when you print using the Unidrv.dll file supplied with Windows 95. All Windows 95 printer drivers, except PostScript drivers, use the Unidrv.dll file which maps all non-white and color text to black when you print to a black and white printer.

WORKAROUND

To print gray-scale text to a black and white printer, do either of the following:

  • Use WordArt to format your text.
  • If you are using a Hewlett-Packard LaserJet 4 printer, install the HP LaserJet 4/4M printer driver (Hppcl5.exe), version 31.1.18. This driver can be obtained either from the Microsoft Download Service at (206) 936-6735, or from Hewlett-Packard. This driver has the capability to print grayscale under Windows 3.x as well as Windows 95.
  • If your printer supports PostScript, use a PostScript printer driver.
