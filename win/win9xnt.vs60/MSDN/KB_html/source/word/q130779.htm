WD6: Internet Assistant, Insert Object Error If No Browse Option

 Last reviewed: September 25, 1997
Article ID: Q130779
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

When you choose Object from the Insert menu, you see a blank line. If you choose the blank line, you receive the following error message: 

   The server application, source file, or item cannot be found. Make sure
   the application is properly installed, and that it has not been deleted,
   moved, or renamed.

CAUSE

When you choose to include browse options during Internet Assistant Setup, "Internetworks Internet Assistant Document" is added to the registration database. It appears in the list of Object Types in the Insert Object dialog box. When you choose NOT to include browse options, a line is still added to the registration database, but it does not contain any text. Thus a blank line appears in the list of Object Types.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Remove the invalid entry in the Registration Database.

To remove the entry from the Windows Registration Database, use the following steps:

  1. Close any open programs. Only Program Manager should be running.

  2. In Program Manager, choose Run from the File menu.

  3. In the Command Line box, type 

          REGEDIT.EXE /V

  4. Choose OK.

  5. Search for the Key of IWIA.Document.

  6. Delete the Key.

  7. Save changes to the registry and exit.

  8. Restart Word and select Insert Object. The blank line is no longer present.

MORE INFORMATION

The following changes are made to the registry when Internet Assistant Setup is run: 

   Operating System             Changes made
   ----------------             ------------

   Windows NT 3.5               \HKEY_CLASSES_ROOT\IWIA.Document line added
                                to Registry

   Windows for Workgroups 3.11  IWIA.Document key added to Registration
                                Database

   Windows 3.11                 IWIA.Document key added to Registration
                                Database
