"Cannot Print" When Printer on Windows NT or Windows 95 Server

 Last reviewed: February 5, 1998
Article ID: Q138869
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you try to print from Word version 6.0, there appear to be two printers instead of one, and you receive the following error message: 

   Windows cannot print due to a problem with the current printer setup.

CAUSE

This problem occurs when the printer is on a Windows 95 server or a Windows NT 3.51 server, and there is a space in the server's name.

For example, if you have a Hewlett-Packard (HP) LaserJet IIISi printer shared as "\\Receptionist Printer\HP" (without the quotation marks), two names will appear on the list of available printers: 

     HP LaserJet IIISi on \\Receptionist
     HP LaserJet IIISi on Printer\HP

WORKAROUND

Remove the space from the server name, or upgrade to Word for Windows 95, version 7.0.
