Word: Accessing Word Files on Internet

 Last reviewed: July 30, 1997
Article ID: Q86894
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

Some files for version 5.0 of Microsoft Word for the Macintosh are available on the Internet at the Sumex-aim.stanford.edu FTP site, in the Info-mac archive.

To access Word for the Macintosh files on Internet:

  1. From a computer connected to the Internet, connect to Sumex- aim.stanford.edu using the command: 

          FTP sumex-aim.stanford.edu

    Note: In the event that the FTP command to connect to Sumex-aim using the name fails, try the direct Internet Protocol address 36.44.0.6 using the command "FTP 36.44.0.6".

  2. When prompted for a login name, enter: 

          anonymous

  3. Change directories to info-mac using the command: 

          cd info-mac

  4. Change directories to app using the command: 

          cd app

  5. Type the word "bget" (without quotation marks) followed by the appropriate name for the file that you wish to download. ("bget" is the unix command to retrieve a binary file.)

  6. Decode the .HQX file using an appropriate decoding utility such as Binhex, Stuffit, or Compact Pro.

  7. Decoding will result in a .SIT compressed file, which must be decompressed using an appropriate decompression utility such as Stuffit or Compact Pro.

MORE INFORMATION

The following files are available at the Sumex-aim FTP site: 

File Name                  Size     Description
---------                  ------   -------------------------------
msword-50a-updater.hqx      55492   Word 5.0a Updater (Font patcher)
word-50-spell-updater.hqx  357667   New Spelling and Grammar Modules
word-movie-plugins.hqx      48550   Word 5.0 Movie Plug-in Module
Sumex-aim at Stanford University is one of many FTP sites where software is accessible on the Internet. The Internet is a large network that is available to most universities and some major corporations. Most computers connected to the Internet use a version of the UNIX operating system.
