Err: "You Are Already Printing..." with LaserMaster WinPrinter

 Last reviewed: February 5, 1998
Article ID: Q140179
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you print to the LaserMaster WinPrinter printer, you may receive the following error message: 

   You are already printing in another application, close it or restart
   Windows.
When you click OK, the document is printed.

CAUSE

This error message occurs when you use a version of the LaserMaster WinPrinter driver earlier than version 1.70.

RESOLUTION

Use one of the following methods to print without the error message:

Method 1. Upgrade the printer driver to version 1.70 or later.

Method 2. Turn off Background Printing. To locate this option, click 

          Options on the Tools menu, and then select the Print tab.

STATUS

Microsoft has confirmed this to be a problem in the LaserMaster WinPrinter printer driver versions earlier than version 1.70. This problem was corrected in the LaserMaster WinPrinter printer driver versions 1.70 and later.
