WD97: Sample Macro Code to Select Table Columns and Rows

 Last reviewed: February 11, 1998
Article ID: Q170733
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

The Microsoft Word Visual Basic for Applications macro language provides commands for selecting columns or rows in Word for Windows tables.

The following Visual Basic for Applications examples demonstrate selecting columns or rows in a Microsoft Word for Windows table. 

   Selecting Columns
   -----------------

   This example selects the second column in the first table of a document.

      ActiveDocument.Tables(1).Columns(2).Select

   Selecting Rows
   --------------

   This example selects the second row in the first table of a document.

      ActiveDocument.Tables(1).Rows(2).Select
For more information about working with the Tables Collection object, while in the Visual Basic for Applications Editor, click the Office Assistant, type "Tables Collection" (without the quotation marks), click Search, and then click to view "Tables Collection Object."
