List of WINWORD6.INI Settings: Word Default Settings

 Last reviewed: February 6, 1998
Article ID: Q105593
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows operating system version 3.1

SUMMARY

This article contains a list of the sections and their corresponding settings in the WINWORD6.INI file, located in the Windows directory. These settings control the defaults and behavior of Word for Windows.

For a description of each setting and instructions on how to modify them, open the online README help file by doing the following:

  1. From the Help menu, choose "Search for Help on."

    -or-

    On the Standard toolbar, double-click the Help button.

  2. In the Search dialog box, type "readme" (without the quotation marks) and then choose the Show Topics button.

  3. Choose the Go To button.

  4. In the Word Readme Help Contents window, choose the "WINWORD6.INI Options" topic.

The following 2 settings are not documented in the Word readme Help file: 

   NoLongNetNames=yes/no    Disables UNC filename support.
                            The default is No.

   OLEDOT=<templatename>    Sets the template that Word will use on new
                            Word document objects.
