Installing Bookshelf 1992 with Word 6.0 for Windows

 Last reviewed: February 6, 1998
Article ID: Q108341
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SUMMARY

The Word for Windows integration features included with Microsoft Word for Windows & Bookshelf, Multimedia Edition, or Video and Sound Edition, will work with Word 6.0 for Windows; however, the Setup program will not successfully install those features with Word 6.0 since the Setup program is designed to work with Word 2.0 for Windows only.

MORE INFORMATION

To install the Word and Bookshelf integration features for Word 6.0 for Windows to successfully communicate with Bookshelf 1992 or 1994, use the following steps.

These steps do not install the Online Word User's Guide included with the Word & Bookshelf CD-ROM. This guide is designed exclusively for use with Word 2.0 for Windows.

Note: These steps assume Word 6.0 for Windows is already installed and functioning normally.

  1. Insert the Word for Windows & Bookshelf CD-ROM into the CD-ROM drive.

  2. In Program Manager, choose Run from the File menu.

  3. Type "d:\setup" (without the quotation marks) and choose OK.

    If the CD-ROM drive is represented by a drive letter other than D, make the appropriate change in the line above to reflect the correct drive letter.

  4. When the Setup program prompts for the Word directory (the "Install Word In:" option), be certain to specify the existing Word 6.0 for Windows directory already present on the hard drive.

  5. The Setup program will report that it has detected an existing version of Word for Windows. Choose the Continue button to proceed.

  6. When given the choice of installation type, choose the Custom Installation button.

  7. When the Setup Options are displayed, clear every option except for Microsoft Bookshelf.

    Note: If the Setup program being used is for the Video and Sound Edition of Word & Bookshelf, you can also select the "Video and Sounds" option. This option is optional and has no effect on the integration features between Word 6.0 and Bookshelf 1992.

  8. Choose the Setup button and continue with the rest of the Setup program normally.

  9. At the completion of the Setup program, Word 6.0 for Windows will be reporting a WordBasic error. Choose the OK button in the error window and restore the Word window from the minimized icon on the screen.

  10. From the File menu, choose Save As.

  11. In the File Name box, type "c:\winword\startup\bookshlf.dot" (without the quotation marks) and choose the OK button.

    If your Word directory is located somewhere other than C:\WINWORD, modify the path given in the File Name box above to reflect the correct path.

  12. From the File menu, choose Close.

  13. From the File menu, choose New then press the OK button to open a new document based on Normal.dot

  14. From the Tools menu, choose Macro.

  15. From the "Macros Available In" list, select "Normal.dot (Global Template)".

  16. For each of the following listed macros, select it in the Macro Name list and delete it by choosing the Delete button.

    AutoExec* Nul wabBookshelfDlg wabConst wabInsertVideo wabInsertVoiceAnno wabModifyToolbar wabReceive wabSetup wabSetupAll wabSetupUg

    *Note: If you are unsure whether the AutoExec macro is part of the Word & Bookshelf Edition or if it is a different macro needed for other purposes, you can rename it instead of deleting it using Word 6.0's Organizer feature.

  17. Choose the Close button in the Macro window.

  18. Quit and restart Word.

  19. From the Tools menu, choose Customize.

  20. Select the Menus tab.

  21. From the Categories list, select Macros.

  22. From the Macros list, select ------(Separator)------.

  23. From the Change What Menu list, select &View.

  24. From the Position on Menu list, select (At Bottom).

  25. Choose the Add button.

  26. From the Macros list, select wabQuoteOfTheDay.

  27. From the Change What Menu list, select &View.

  28. From the Position on Menu list, select (At Bottom).

  29. In the Name On Menu text box, type "&Quote Of The Day" (without the quotation marks).

  30. Choose the Add button.

  31. From the Macros list, select ------(Separator)------.

  32. From the Change What Menu list, select &Tools.

  33. From the Position on Menu list, select (At Bottom).

  34. Choose the Add button.

  35. From the Macros list, select wabBookshelfDlg.

  36. From the Change What Menu list, select &Tools.

  37. From the Position On Menu list, select (At Bottom).

  38. In the Name On Menu text box, type "Bookshel&f" (without the quotation marks).

  39. Choose the Add button.

  40. From the Macros list, select ------(Separator)------.

  41. From the Change What Menu list, select &Insert.

  42. From the Position On Menu list, select (At Bottom).

  43. Choose the Add button.

  44. From the Macros list, select wabInsertVoiceAnno.

  45. From the Change What Menu list, select &Insert.

  46. From the Position on Menu list, select (At Bottom).

  47. In the Name On Menu text box, type "&Voice Annotation" (without the quotation marks).

  48. Choose the Add button.

  49. From the Macros list, select wabInsertVideo.

  50. From the Change What Menu list, select &Insert.

  51. From the Position On Menu list, select (At Bottom).

  52. In the Name On Menu text box, type "Media Pla&yer Clip" (without the quotation marks).

  53. Choose the Add button.

  54. Choose the Close button to close the Customize window.

  55. In the [Microsoft Word 2.0] section of the WIN.INI file, add the following line:

    programdir=C:\WINWORD

    If your Word directory is located somewhere other than C:\WINWORD, modify the "programdir" line given above to reflect the correct path.

  56. Exit and restart Windows.

Adding Bookshelf 1992 Integration to Word 6.0 for Windows is now complete.
