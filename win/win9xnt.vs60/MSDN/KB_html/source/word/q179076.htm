WD97: Visio Object Background Turns Black When Doc Saved as HTML

 Last reviewed: January 12, 1998
Article ID: Q179076
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you save a Microsoft Word 97 document containing Visio, version 4, objects as HTML, the background of the objects will turn black.

MORE INFORMATION

Visio is a drawing package that lets you assemble illustrations and diagrams by dragging objects created from stencils, called SmartShapes. The Visio program installs icons on the Word and Excel toolbars so that you can toolbar.
