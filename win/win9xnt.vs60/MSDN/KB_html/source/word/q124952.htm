Microsoft Imager Available as Fulfillment Item

 Last reviewed: August 5, 1997
Article ID: Q124952
The information in this article applies to:

- Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

Microsoft Imager is now available as a fulfillment item for Word 6.0 and Office 4.2/4.3 users. Imager was previously only available with Microsoft Word & Bookshelf 1994 on CD-ROM, as well as Microsoft Office 4.2/4.3 and Bookshelf on CD-ROM.

To order Microsoft Imager, call the Microsoft Order Desk at (800) 360-7561 and request Microsoft Imager. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web page at: 

  http://www.microsoft.com/worldwide/default.htm
Description: FULFILL WIN/WORD 6.0 IMAGER KIT Part Number: 059-151-655 
Cost:         $5 shipping and handling charge
Availability: Available to anyone who owns a licensed copy of Word 6.0 
              for Windows or Office 4.x.

MORE INFORMATION

Microsoft Imager can be used as a stand-alone imaging application or as an OLE Server. Imager is designed to help Microsoft Word and Office users import, edit, and manipulate pictures easily within Microsoft Word or other Microsoft Office products. For example, you can adjust the colors in a image, clean up unwanted specks, create a negative image, and resize, crop, and rotate a picture. You can scan pictures into Imager using a TWAIN- compatible scanner, or you can open existing pictures.

In addition, Microsoft Imager includes a collection of ClipPix images from PhotoDisc, a provider of high quality photographic images. These ClipPix images are installed in the PHOTOS subdirectory of your Office directory (usually, C:\OFFICE). There is a PhotoDisc Readme and License Agreement document in this directory for more information about technical support for ClipPix.
