Cannot Create Dialog Boxes in a WLL Using the CAPI

 Last reviewed: February 5, 1998
Article ID: Q141217
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SUMMARY

Microsoft incorrectly left several identifiers in the WDCMDS.H and WDFID.H header files that might lead you to the conclusion that you can create custom WordBasic dialog boxes.

You cannot create your own WordBasic dialog boxes from the Word CAPI.

RESOLUTION

WLL developers can create much more robust dialog boxes using the Windows API.
