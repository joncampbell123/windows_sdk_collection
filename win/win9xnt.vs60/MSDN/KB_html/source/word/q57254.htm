Word: Word's Dictionary Reloads for Every Spell Check

 Last reviewed: July 30, 1997
Article ID: Q57254
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 3.0 , 3.01, 3.02, 4.0

SUMMARY

If the Dictionary in Microsoft Word reloads each time after using Word Finder, Word may have been allocated less memory than it needs under MultiFinder. Increasing the memory allocation in the Get Info box should prevent this problem from occurring.

To increase the memory allocated to Word, do the following:

  1. Quit out of Word and select the Word icon.

  2. From the Edit menu, choose Get Info.

  3. Type a larger number in the Application Memory Size box.

  4. Close the Get Info box and double-click the Word icon.
