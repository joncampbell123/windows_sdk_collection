WD97: Supplemental Text Converters Available for Word 97

 Last reviewed: February 3, 1998
Article ID: Q157092
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

The following supplemental converters are available for Microsoft Word 97:

  • Windows Write 3.0 or 3.1 (Write32.cnv)
  • RFT-DCA (Rftdca32.cnv)
  • Lotus Ami Pro 3.x (Ami332.cnv)
  • WordStar 3.3-7.0 for MS-DOS and WordStar for Windows 1.0-2.0 (import only) and WordStar 4.0 or 7.0 for MS-DOS (export only) (Wrdstr32.cnv)
  • Microsoft Word 3.x, 4.x, 5.x, and 6.x for MS-DOS (Doswrd32.cnv)

The following file is available for download from the Microsoft Software Library: 

 ~ Wdsupcnv.exe (size: 428768 bytes) 
                   This file includes the following converters and a
                   Readme file:

                   Ami332.cnv
                   Doswrd32.cnv
                   Rftdca32.cnv
                   Wrdstr32.cnv
                   Write32.cnv
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

How do I install the supplemental converters?

To install these converters, use the following steps:

  1. Close Microsoft Word.

  2. Save the Wdsupcnv.exe file to a temporary location on your hard disk.

  3. On the Windows 95 taskbar, click Start.

  4. Click Run.

  5. In the Open box type "Wdsupcnv.exe" (without the quotation marks).

  6. Click OK.

    The converters will be extracted from the .exe file and placed into the temporary location on your hard disk.

  7. Copy the extracted .cnv files into the following folder: 

          Program Files\Common Files\Microsoft Shared\Textconv

    NOTE: You may also copy the extracted .cnv files into the following folder: 

          Program Files\Microsoft Office\Office

    Word can find the supplemental converters in either location.

    If you are using a network installation of Microsoft Word and you are running Word from the network server, a network administrator having the required permissions on the network must copy the .cnv files to the following folder on the network: 

          MSapps\Textconv

NOTE: If you are running a network installation of Word on a workstation, you cannot install the supplemental file converters exclusively to the local drive of that workstation. The supplemental converters must be installed on the server; therefore, the converters will be available to all of the workstations using the network installation of Word. If you want to limit access to the converters such that they are available only to a single computer, you must perform a local installation of Microsoft Office or Microsoft Word on that computer.

  1. Start Microsoft Word. When you open a file that was created in one of the programs for which a converter is installed, Word will access the appropriate converter automatically and will open the document.

Obtaining the Converters If You Are Unable to Download Them

If you are unable to download these converters from the above sources, you can also access them from any of the following sources:

  • Obtain the Microsoft Office 97 Resource Kit, available from Microsoft Press. To purchase the Resource Kit in North America, call (800) MSPRESS (800) 677-7377. International purchases can be faxed to (425)936-7329. Do not call the Microsoft Sales Information Center (MSIC) to order the kit.

    NOTE: You can obtain this Microsoft Press title from any local computer bookstore or you can send in the Microsoft Office Supplemental Offers Coupon, which is included in your Microsoft Office retail package.
