600dpi Shading Prints at 300dpi with HP LaserJet 4

 Last reviewed: February 5, 1998
Article ID: Q117701
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

When you print a Word 6.0 document containing paragraph shading to a Hewlett-Packard (HP) LaserJet 4 (HPPCL5E.DRV version 1.30 or earlier), the paragraph shading prints at 300dpi even if the printer driver is set for 600dpi.

STATUS

Microsoft is researching this problem, and we will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

If you insert a figure into the drawing layer, such as a rectangle, the paragraph shading will now print at 600dpi when the printer driver is set for 600dpi. Formatting the border and fill as White or None does not allow you to print the shading at 600dpi.
