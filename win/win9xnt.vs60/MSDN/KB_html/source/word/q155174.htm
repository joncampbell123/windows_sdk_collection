Illegal Operation or Application Error Exiting Word

 Last reviewed: February 5, 1998
Article ID: Q155174
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SYMPTOMS

When you quit Word 6.0 for Windows NT or Word 7.0 for Windows 95, you may receive an Application Error or Illegal Operation error message.

CAUSE

This behavior occurs if the following conditions are true:

  • the Normal.dot file contains the SUBJECT field in the header

    -and-

  • the SUBJECT field contains the Title case switch in the Header

    -and-

  • prior to exiting Word, you have copied text to the clipboard.

NOTE: This may not be the only cause of this symptom.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Before exiting from Word, do either of the following:

  • Open the Normal.dot file and remove the Title Case switch from the SUBJECT field or remove the SUBJECT field from the Header.

    -or-

  • Create a new document and paste the information from the clipboard into the new document.
