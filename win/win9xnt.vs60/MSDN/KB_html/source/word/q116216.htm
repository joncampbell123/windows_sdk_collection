Incorrect Status Bar Message with No Annotations

 Last reviewed: February 5, 1998
Article ID: Q116216
6.00 6.00a 6.00c WINDOWS kbusage buglist6.00 buglist6.00a buglist6.00c fixlist7.00

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When the open document contains no annotations and you highlight the unavailable (dimmed) Annotations command on the View menu, the following message displays in the status bar: 

   Opens the annotation pane for reading annotations (toggle)
When a command is not available, the following message should be displayed in the status bar: 

     This command is not available because the document does
     not contain <commandname>

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. This problem was corrected in Word version 7.0 for Windows 95 and Word 6.0 for Windows NT.
