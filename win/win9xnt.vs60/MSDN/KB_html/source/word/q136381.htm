Object Lost When Dragged from PowerPoint to Word

 Last reviewed: February 6, 1998
Article ID: Q136381
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0

SYMPTOMS

When you drag and drop an object from PowerPoint to Word and then save, close, and reopen the Word document, the object appears as an empty box. If you double-click the box, you will get the following error message 

   There is a serious disk error on <filename>.
where <filename> represents the filename of the PowerPoint object.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUNDS

To preserve the object, do one of the following.

Method 1: After you drag and drop the object into Word, do the following: 

          a. Using the left mouse button, double-click the object. This
             opens the Picture editor.
          b. Click Close Picture.
          c. Save and close the document.
Method 2: Use copy and paste.
