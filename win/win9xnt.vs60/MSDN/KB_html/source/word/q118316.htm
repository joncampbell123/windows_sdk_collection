WD: All Body Text Prints with "Show First Line Only" Selected

 Last reviewed: February 5, 1998
Article ID: Q118316
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1,6.0.1
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you print from outline view or master document view with the Show First Line Only option selected, all body text will be printed, even though only the first line of each paragraph is displayed on the screen. If you display the document in print preview, all body text will be displayed. (To locate the Show First Line Only button, display the Outline toolbar. The button is between the All button and the Show Format button.)

The Word Help topic for "Printing an Outline" and the "Microsoft Word User's Guide" state, "When you print in outline view, Word prints only the headings and body text paragraphs displayed on the screen. You can control the amount of detail printed by expanding or collapsing the outline."

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In outline and master document views you can choose to view only the first line of body text in each paragraph by selecting the Show First Line Only option. This option does not affect the display of heading text.

Normally, when you print from outline or master document view, the printout resembles the text that appears on the screen. All of the other buttons on the outline toolbar, including the Show Heading Levels and Show Formatting, make changes on the screen that also affect printing.

Currently there is no way to print only the first line of the body text. You can easily prevent body text from printing by viewing the appropriate heading levels (1-9) instead of choosing All on the Outline Toolbar.

REFERENCES

"Microsoft Word User's Guide," version 6.0, Chapter 9, "Automatic Formatting and Styles"
