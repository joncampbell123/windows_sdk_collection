WD: "MacroRem Not Available" Error Message If Edited

 Last reviewed: November 17, 1997
Article ID: Q110327
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

If the macro command MacroREM is altered in any way, the following WordBasic Error 509 occurs: 

   The MacroREM command is not available because it is not a
   WordBasic macro command, but it can be assigned to a toolbar,
   menu, or shortcut key.

MORE INFORMATION

The MacroREM button is used to remark out lines in a macro in order to provide information and for debugging purposes. The button is available from the macro toolbar.
