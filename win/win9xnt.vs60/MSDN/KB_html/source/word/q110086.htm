Slow Printing or Hangs Using BubbleJet 600 Printer

 Last reviewed: February 5, 1998
Article ID: Q110086
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows for Workgroups version 3.1, 3.11

SYMPTOMS

When printing from Word 6.0 for Windows to a Canon BJC-600 printer using version 1.0 of the BJC printer driver (BJC600.DRV), Word may print very slow or hang during the printing of the document. Sometimes the tutorials will not run when this printer driver is selected.

CAUSE

Choosing the BJC-600 printer driver version 1.0 may cause the system RAM to become depleted when printing documents

STATUS

This problem is resolved by the v.2.0b driver which is available from Canon technical support. Canon Printer Technical Support can be reached at (800) 423-2366.

WORKAROUND

  1. Disabling Print Manager helps this situation.

  2. Disabling "Fast Printing to Port" helps this situation.

  3. Use the BJ-10 or BJ-200 printer driver to solve the resource consumption problem. Note: These printer drivers only allow for Black and White printing.

    If you need to be able to print in color, use the Epson LQ-860 driver available from the Microsoft Download Service (EP24.EXE) at 425-936- MSDL.

MORE INFORMATION

  • Setting the driver to "Screen Matching Colors" is much slower than "Color."
  • "Diffusion" is much slower than "Pattern."
  • Combinations of the above settings are very slow.
