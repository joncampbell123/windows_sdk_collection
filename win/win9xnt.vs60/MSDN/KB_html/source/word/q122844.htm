WD: "Bad Parameter" Error Using Resume Wizard

 Last reviewed: February 5, 1998
Article ID: Q122844
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

When running the Resume Wizard, you may receive the following error message, which causes the wizard to halt: 

   WordBasic Err=24
   Bad Parameter
Clicking OK cancels the wizard.

CAUSE

You have one line of text followed by an "empty" paragraph in the Mailing Address box on the User Info tab. To locate the User Info tab, choose Options from the Tools menu.

During Word setup, if you do not enter information in the Corporation text box in the Register Your New Software screen, Word Setup places a carriage return in the Mailing Address scroll box.

WORKAROUND

To eliminate this error when starting the Resume Wizard, remove the empty paragraph. To do this, follow these steps:

  1. On the Tools menu, click Options. Or, click Preferences in Word for Macintosh.

  2. Select the User Info tab.

  3. In the Mailing Address scroll box, use the arrow keys to move the cursor down until it stops.

  4. Press the BACKSPACE key to delete the paragraph 

        -or-

    add text to the second line of the Mailing Address.

    NOTE: If you do not want a Mailing Address, press the BACKSPACE key until there is nothing in the box.

STATUS

Microsoft has confirmed this to be a problem in the products mentioned at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
