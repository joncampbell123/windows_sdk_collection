Err Msg: "SPCLPPR.SPT Missing from Special Paper Directory"

 Last reviewed: February 5, 1998
Article ID: Q113107
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Word Assistant, version 1.0
  • Microsoft Windows operating system version 3.1

SUMMARY

The following Word Assistant error message may occur when you choose the Special Paper command from the Insert Menu: 

   SPCLPPR.SPT is missing from the Special Papers directory,
   so you cannot run Special Paper.

CAUSE

The above error occurs when the Paper Locations setting in the SPCLPPR.INI file is incorrect. This happens if you move the Special Paper files after you install Word Assistant but do not update the SPCLPPR.INI file to reflect the new location.

Note: When you run the Word Assistant Setup program, you cannot change the directories where the Special Paper files are located. This means you have to move them after you complete Setup.

WORKAROUND

Using a text editor such as Windows Notepad, open the SPCLPPR.INI file, located in the \WINDOWS directory, and correct the path of the Papers Location setting in the [Special Paper] section. The following example shows the default settings after Setup: 

   [SPECIAL PAPER]
   PAPERS LOCATION=C:\WINWORD\SPCLPPR
   HELP=C:\WINWORD\SPCLPPR.HLP
