WD: Spacing Problems Printing to HP DeskWriter

 Last reviewed: February 5, 1998
Article ID: Q119321
The information in this article applies to:
  • Microsoft Word for Macintosh, versions 4.0, 5.0, 5.1, 6.0

SYMPTOMS

When you print from a Hewlett-Packard (HP) DeskWriter using the DeskWriter version 5.0 driver, there may be spacing problems between characters and lines of text. Text may overlap or blank lines may be skipped.

CAUSE

These spacing problems can occur if the document is formatted using the New York or Geneva font, or if Fractional Widths is not activated.

WORKAROUND

Word 6.0

  1. Select the text.

  2. From the Format menu, choose Font and select Times or Helvetica from the Fonts list. Choose Ok.

  3. From the Tools menu, choose Options and select the Print tab.

  4. Select the Fractional Widths checkbox.

MORE INFORMATION

The DeskWriter 5.0 ReadMe document recommends not using the Geneva or New York fonts with Word. It also recommends activating Fractional Widths. The following is the text from the DeskWriter ReadMe document on MS Word: 

   Be sure to turn Fractional Widths ON in the Page Setup dialog box. This
   will help to ensure print quality and prevent gaps in underlining.

   For best results when printing, avoid using Geneva and New York fonts.
   Use Helvetica and Times instead.
To locate the above information, open the ReadMe (DW 5.0) file located on the HP DeskWriter disk 1. From the Contents menu, choose Software Notes and scroll down to Microsoft Word (products are listed alphabetically).

For more information, contact Hewlett-Packard's Driver Distribution Center at (800) 752-0900.
