WD: Using ChemDraw with Word for the Macintosh

 Last reviewed: February 6, 1998
Article ID: Q120245
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0

SUMMARY

Cambridge Scientific Computing, Inc., markets ChemDraw (including Chem3D and ChemFinder) as a chemist's drawing and modeling tool. Part of the feature set of ChemDraw is the capability to import these generated graphics into Word for the Macintosh.

You can start ChemDraw by double-clicking the graphic object in the Word document. This feature is called Edit Graphics Object (EGO) and is available only on Macintoshes running Apple System 7.

ChemDraw and its associated programs do not currently support the OLE features of version 6.0 of Word for the Macintosh.
