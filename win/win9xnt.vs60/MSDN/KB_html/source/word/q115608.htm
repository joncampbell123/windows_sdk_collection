Bar Codes Print Incorrectly on an Okidata Laser Printer

 Last reviewed: February 5, 1998
Article ID: Q115608
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

Bar codes may print incorrectly when you use an Okidata laser printer driver. This behavior occurs when the line of text beneath the bar code (normally the formal name) is shorter than the bar code and the ZIP Code is nine digits.

STATUS

Microsoft has confirmed this to be a problem with Okidata Laser Printers. The problem has been reported with the 800, 820, 830, and the Okidata Laser Line 6.0. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUNDS

Method 1: Use five-digit ZIP Codes with the Okidata printers.

Method 2: Add hidden or extra characters to the line of text directly below 

          the bar code.
Method 3: Use a Hewlett-Packard (HP) printer driver.
