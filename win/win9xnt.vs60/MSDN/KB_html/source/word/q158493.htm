WD97: Wizard Vanishes When Run from Banyan Vines Server

 Last reviewed: February 5, 1998
Article ID: Q158493
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you run one of the wizards in Word 97, the wizard vanishes. Most of the wizards do not display an error message; however, the Fax Wizard will display an error indicating that it cannot locate a file.

CAUSE

This problem occurs when the wizards and templates are installed to a Banyan Vines server instead of to your local computer.

WORKAROUND

To work around this problem, copy the wizards and the templates to your local Template directory.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
