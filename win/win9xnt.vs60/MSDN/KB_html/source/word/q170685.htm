WD6X: MacWord Prints Bullets Without Spaces to LaserWriter II SC

 Last reviewed: February 3, 1998
Article ID: Q170685
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you print a Microsoft Word 6.x for the Macintosh document with bullets inserted from the toolbar to either a LaserWriter II SC or a Personal LaserWriter SC printer, the text is printed right next to the bullet without any space between the text and bullet. However, the bullets and text appear correctly on the screen.

CAUSE

The Personal LaserWriter SC printer driver is incompatible with Macintosh System Software 7.5.2 and later. This printer driver is used for both the LaserWriter IISC and Personal LaserWriter SC printers. It is known to produce poor line layout under this configuration and therefore does not recognize the space between the text and the bullets.

The versions of the printer driver that have been tested and that produce These undesirable effects are 6.1, 7.0, and 7.0.1.

WORKAROUND

According to Apple, in order to print correctly to these printers while running 7.5.2 and later, you must install QuickDraw GX and use the LaserWriter IISC GX printer driver.

MORE INFORMATION

This problem is not specific to the Word program and can occur on any Macintosh computer running System 7.5.2 or later when printing to either the LaserWriter IISC or the Personal LaserWriter SC.

This condition continues to exist with Microsoft Word 98 Macintosh Edition because Apple is not developing standard drivers for this printer.

The Personal LaserWriter SC printer driver is manufactured by Apple Computer, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
