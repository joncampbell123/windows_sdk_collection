WD: "That That" and "Had Had" Not Flagged as Repeated Words

 Last reviewed: February 3, 1998
Article ID: Q122233
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

The Grammar and Spelling commands do not identify "that that" or "had had" as repeated words. This behavior is different from that of earlier versions of Microsoft Word.

CAUSE

This behavior is by design of Microsoft Word. The Grammar and Spelling commands recognize that these two words can be repeated within a sentence. Although such constructions are not grammatically correct in the strictest sense, they are generally accepted in colloquial English.
