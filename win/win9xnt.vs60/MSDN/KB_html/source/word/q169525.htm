WD: How to Change the Company Property Information

 Last reviewed: July 31, 1997
Article ID: Q169525
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SUMMARY

The "More Information" section of this article discusses various methods you can use to change the Company property in Microsoft Word.

This behavior is different from Microsoft Word 6.x because the Summary information in Microsoft Word for Windows 6.x did not include the company property field.

MORE INFORMATION

To change the Company property for a single document in Microsoft Word 97 or Microsoft Word 7.x:

  1. On the File menu, click Properties.

  2. Click the Summary tab.

  3. In the Company box, select the information, and delete it.

  4. Enter new information into the box or leave it blank.

  5. Click OK.

  6. Save the document.

NOTE: The property will be changed for that document only.

To change the property for any template in Microsoft Word 97 or Word 95 (7.0) (not including the Normal template in Word 97), use the following steps:

  1. On the File menu, click Open.

  2. Set the Files Of Type box to Document Template, and change the Look In box to the location of your template. Select the Template you want to change and click OK.

  3. On the File menu, click Properties.

  4. Click the Summary tab.

  5. Select the information in the Company Property field and delete it.

  6. Enter new information into the field or leave the field blank.

  7. Click OK.

  8. Save the document.

NOTE: Changing the company property field in the Normal template of Microsoft Word 97 and saving the change will not be reflected in new documents based on Normal.

To permanently change the company property in all user defined templates and the Normal template in Microsoft Word 97:

  1. Remove Microsoft Word 97.

  2. Use either of the following methods to reinstall Word 97: 

        - If you are reinstalling from a compact disc, place the correct
      information in the name dialog box.


        - If you are reinstalling from disks, you will need to reinstall from a
      new set of disks because company information is permanently written
      to the disks during installation.
