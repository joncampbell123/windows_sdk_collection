WD: Background Repagination Can't Be Turned Off in Word

 Last reviewed: February 3, 1998
Article ID: Q120106
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Microsoft Word, the Background Repagination option is selected and is unavailable (dimmed) for you to clear (turn off).

To locate this option, click Options on the Tools menu (or click Preferences in Word 98 Macintosh Edition), and select the General tab.

CAUSE

This behavior occurs because you are viewing the current document in page layout view. By design, page layout view requires that Background Repagination be turned on because Word must show the page breaks correctly.

RESOLUTION

To turn off Background Repagination, change the document view to normal, outline, or master document view.

MORE INFORMATION

Repagination in Word cannot be totally turned off. This is because Word has to repaginate the document in Page Layout and Print Preview in order to correctly display the document.
