Positioning Text and Graphics as One Object

 Last reviewed: November 17, 1997
Article ID: Q58421
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 5.1a

In Word use the Frame command (the Position command in Word 4.0) to overlap text and a graphic:

  1. Position the insertion point to the right of the graphic and press the RETURN key to make the graphic a separate paragraph from the text.

  2. Select the graphic and the text you want to include with it. (The text you want to include with the graphic should immediately follow the paragraph mark that occurs after the graphic.)

  3. From the Insert menu, choose Frame. With the frame selected, choose Frame from the Format menu to place the text in the desired position (for example, in the margin). (In Word 4.0 choose Frame from the Format menu).

  4. Choose the OK button.

  5. Place the cursor next to the paragraph mark at the end of the graphic, press the RETURN key, and then press COMMAND+SHIFT+N (COMMAND+SHIFT+P in Word 4.0 and 5.x). This inserts a normal paragraph mark between the graphic and the text, causing Word to overlap the positioned text and the graphic (Without the normal paragraph mark, Word will position the paragraph containing the text below the paragraph containing the graphic.)
