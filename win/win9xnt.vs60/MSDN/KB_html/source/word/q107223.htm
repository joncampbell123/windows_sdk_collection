WD: Foreign Language Dictionaries for Word

 Last reviewed: March 3, 1998
Article ID: Q107223
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 97 for Windows

SUMMARY

Alki Software Corporation has foreign language, legal, and medical dictionaries for Word. Several new languages have been added. In addition, Dutch, German, Portuguese, and Norwegian have combined dialects to be included on one disk.

MORE INFORMATION

Word proofing tools from Alki Software are available in many different languages. Some proofing tools are not available for all languages. Below is a table explaining the tools available for each language.

NOTE: You cannot use the proofing tools for Word for Windows versions 1.x or 2.x with Word for Windows version 6.0, and vice versa. You cannot use the proofing tools for Word for the Macintosh versions 5.x Word for the Macintosh 6.x, and vice versa. Every time that you upgrade to a newer version of Word for Windows, you should contact the Alki Software Corporation for the best way to proceed to upgrade your foreign language, legal, and medical dictionaries. Alki uses a different setup program for their products for each version of Word.
