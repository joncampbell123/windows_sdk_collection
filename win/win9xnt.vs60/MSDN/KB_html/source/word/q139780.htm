General Protection Fault in Module KRNL386.EXE

 Last reviewed: February 5, 1998
Article ID: Q139780
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, and 6.0c
  • Microsoft Windows operating system, versions 3.1, 3.11
  • Microsoft Windows for Workgroups, version 3.11

SYMPTOMS

When you choose Print To File in Word, you will get a general protection fault (GP fault) error in module KRNL386.EXE. This error occurs when you print to one of the following printers using the Hewlett-Packard (HP) HPPCL5E.DRV printer driver version 31.V1.30: 

   HP LaserJet 4 Plus/4M Plus
   HP LaserJet 4/4M
   HP LaserJet 4L
   HP LaserJet 4ML
   HP LaserJet 4P/4MP
   HP LaserJet 4Si/4Si MX

STATUS

Microsoft has confirmed this to be a problem in the HP printer driver HPPCL5E.DRV version 31.v1.30. This problem was corrected in the HPPCL5E.DRV printer driver versions 31.V1.35 and 31.V1.50.

RESOLUTION

Update your printer driver.
