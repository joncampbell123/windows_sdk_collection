Application Error When Spell Checking Annotations

 Last reviewed: February 5, 1998
Article ID: Q134729
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

Running the spelling checker on a Word document that is embedded in a Microsoft Excel workbook may result in the following error: 

   Winword.exe application error. "The instruction at  <addr> referenced
   memory at 0x00"
This error will occur during in-place editing of an embedded document that contains annotations. Canceling the spelling check while checking text in the Annotations pane will leave the document vulnerable to this error.

CAUSE

Normally, annotations are unavailable when you edit an embedded document with in-place editing; however, the spelling checker can still open the Annotations pane in this situation.

STATUS

Microsoft has confirmed this to be a problem in Word versions 6.x for Windows and Word version 7.0 for Windows 95. This problem was corrected in Microsoft Word 97.

WORKAROUND

Do not exit the spelling checker when in the Annotations pane.
