Repeated Warning Messages from Format Columns Dialog

 Last reviewed: February 5, 1998
Article ID: Q118348
6.00 6.00a 6.00c WINDOWS kbusage buglist6.00 buglist6.00a buglist6.00c fixlist7.00

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you enter an out-of-range value for the number of columns in the Format Columns dialog box, Word displays the following error message multiple times: 

   The number must be between 1 and 45.
The number of times the error message appears varies depending on whether you press ENTER on the keyboard or choose the OK button. In addition, if the mouse pointer rests over the OK button, the OK button will appear depressed. This behavior does not cause any problems with your Word document.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. This problem was corrected in Word version 7.0 for Windows 95.

WORKAROUND

Choose OK until the message disappears, then change the number of columns to a valid number between 1 and 45.
