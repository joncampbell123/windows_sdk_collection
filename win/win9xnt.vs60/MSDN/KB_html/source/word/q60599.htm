Mouse Pointer Disappears in Word for Windows Macro Record

 Last reviewed: July 30, 1997
Article ID: Q60599
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows, after you choose Macro Record, the mouse pointer will be lost while typing until the mouse is moved again.

To reproduce this behavior, do the following:

  1. Choose Macro Record.
2. Enter a macro name and choose the OK button. 3. Type a character. The mouse pointer disappears.

By design, the mouse pointer disappears so that it does not obstruct typing. The pointer returns as soon as the mouse is moved.
