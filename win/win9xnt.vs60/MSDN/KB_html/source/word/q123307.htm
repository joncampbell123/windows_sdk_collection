WD: Avery Label 4145 Does Not Print Correctly

 Last reviewed: February 3, 1998
Article ID: Q123307
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you print or perform a mail merge to a page of labels based on the Avery product #4145 for dot-matrix printers, your labels print incorrectly. Using Word for the Macintosh, the text is offset to the right approximately 2.5 inches, and the first two lines of the first label are truncated. Using Word for Windows, the first line of the first label is missing.

RESOLUTION

To resolve this problem, upgrade to the latest version of Microsoft Word.

To work around this problem, use the appropriate method for your version of Word.

Word for the Macintosh

After you create a document with the labels, make the following modifications before printing.

NOTE: If you are performing a mail merge, you should merge to a document and make the following changes before printing.

  1. On the File menu, click Page Setup.

  2. Click Custom.

  3. Change the Width from 4.5 inches to 8.5 inches.

  4. Click OK.

  5. Click OK again.

  6. On the Table menu, click Select Table.

  7. On the Table menu, click Cell Height And Width.

  8. Select the Row tab and set the Height Of Row to 68 points.

  9. Click OK.

  10. Deselect the table by clicking once anywhere in the document area.

  11. On the File menu, click Document Layout.

  12. On the Margins tab, set the Top margin to .30 inch.

    NOTE: If you receive the following message, click Ignore:

    One or more margins are set outside the printable area of the page. Choose the Fix button to increase the appropriate margins.

  13. Click OK.

Your labels should now print correctly.

Word for Windows

First create a document based on the 4145 labels, then make the following changes to correctly print your labels.

NOTE: If you are performing a mail merge, merge to a new document and make these changes before printing.

  1. Select the table.

  2. From the Table menu, choose Cell Height And Width.

  3. Select the Row tab and set the Height Of Row to 68 points.

  4. Click OK.

  5. Deselect the table by clicking once anywhere in the document window.

  6. From the File menu, choose Page Setup.

  7. Select the Margins tab and set the Top margin to .30 inch.

Your document should now print correctly.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word listed at the beginning of this article. This problem was corrected in Microsoft Word 97 for the Windows versions.
