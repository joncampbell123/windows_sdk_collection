WD6X: Word Quits When You Convert 5.1 Settings File Under GX

 Last reviewed: December 5, 1997
Article ID: Q124806
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

If you try to convert a Word 5.1 Settings file on an Apple Macintosh with QuickDraw GX installed, Word 6.0 unexpectedly quits.

WORKAROUND

Disable QuickDraw GX before converting the Settings file. To disable QuickDraw GX, remove the QuickDraw GX file from the Extensions Folder, and then restart the computer.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
