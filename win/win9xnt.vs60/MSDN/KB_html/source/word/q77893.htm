Hyphenation Dialog Options Automatically Reset

 Last reviewed: February 6, 1998
Article ID: Q77893
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word versions 1.x, for Windows, if you clear the Confirm option in the Hyphenate dialog box, the option is reset when you choose New or Open from the File menu to work on another document. This information also applies if you choose Hyphenation from the Tools menu in Word version 2.0 for Windows. Also, the Hot Zone in the Hyphenate (Hyphenation in 2.0) dialog box defaults to .25 inch when you choose New or Open.

In Word version 6.0for Windows the look of the dialog is a little different, but the default settings are the same. In version 2.x the "Confirm checkbox is checked by default. In Word version 6.0 there is an "Automatically Hyphenate Document" checkbox which is unchecked by default.

MORE INFORMATION

If you are using Word version 1.0, 1.1, or 1.1a, the information on page 155 of the "Microsoft Word for Windows User's Reference" may be confusing. Under "Inserting Hyphens Fully Automatically," the documentation does not specify that clearing the Confirm check box is a document-specific function rather than a global one.

For more information on clearing the Confirm option and changing the Hot Zone globally, query on the following words in the Microsoft Knowledge Base: 

   wordbasic and utilhyphenate and word for windows
Fields in the Hyphenation dialog box can also be changed globally by making the changes directly to Normal.dot.

  1. Load Normal.dot from the File Open dialog.
2. Select Tools Hyphenation from the menu bar. 3. Make the desired changes to the dialog. 4. Save and close the Normal.dot.

These changes will effect all subsequent new documents based on the Normal template.
