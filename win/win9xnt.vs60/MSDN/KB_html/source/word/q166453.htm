WD: Setting Newspaper Style Column Format Hides Table Columns

 Last reviewed: July 31, 1997
Article ID: Q166453
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

When you format a document that contains a table into newspaper style columns, some of the table columns may appear to be hidden.

CAUSE

The width of the table is larger than the width of the column. The table does not automatically readjust to fit the size of the column.

WORKAROUND

Method 1: Use AutoFit (Word 6.0 & later)

  1. Place the insertion point in the table.

  2. On the Table menu, click Select Table.

  3. On the Table menu, click Cell Height And Width.

  4. Click the Column tab.

  5. Click AutoFit.

Method 2: Adjust the Table Column Widths

Use this method if you inserted the table and then formatted the document for newspaper style column formatting.

  1. Place the insertion point in the table.

  2. On the Table Menu, click Select Table.

  3. On the Table Menu, click Cell Height And Width.

  4. Click the Column tab.

  5. Change the Width of the columns to a size that will allow the width of the entire table to fit within the newspaper column.

  6. Click OK.

Method 3: Set Newspaper Columns First

Format the document with newspaper style columns before you insert the table.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
