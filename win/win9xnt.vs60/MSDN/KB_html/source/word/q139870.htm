Word for Windows 95: ODBC Drivers Available

 Last reviewed: February 5, 1998
Article ID: Q139870
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Access versions 2.0, 7.0
  • Microsoft Excel for Windows 95
  • Microsoft Project for Windows 95
  • Microsoft Visual Basic version 4.0
  • Microsoft C++ version 4.0

SUMMARY

An Application Note containing the latest 32-bit Microsoft Open Database Connectivity (ODBC) drivers is available (GE1263, "Microsoft ODBC Drivers for 32-Bit Programs"). These drivers work with the products listed above.

NOTE: The Application Note is a duplicate of WX1220, a fulfillment item for coupons included in several packages, including Office for Windows 95 and Access for Windows 95. Because WX1220 appears to be specific only to Access, this duplicate Application Note was created to make the information more readily accessible to customers of all ODBC-compliant products.

For additional information about this Application Note and its availability, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140255
   TITLE     : GE1263: Microsoft ODBC Drivers for 32-Bit Programs

MORE INFORMATION

This Application Note contains the latest 32-bit Microsoft Open Database Connectivity (ODBC) drivers for the following data sources:

  • ORACLE versions 7.0 and 7.1
  • Microsoft SQL Server versions 4.21 and 6.0
  • Microsoft Jet database engine version 3.0, which supports the following formats: 

          Microsoft Access versions 2.0 and 7.0
      Microsoft FoxPro versions 2.0, 2.5, and 2.6
      dBASE III, dBASE IV, and dBASE for Windows
      Paradox versions 3.x, 4.x, and 5.0
      Microsoft Excel versions 3.x, 4.x, 5.x, and 7.0
      Text
