WD98: Supported Non-Word File Formats

 Last reviewed: February 17, 1998
Article ID: Q179607
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

Microsoft Word 98 supports conversion to and from a number of non-Word file formats. Word interprets some non-Word formats using built-in converters, and other formats through the use of external converter files.

MORE INFORMATION

Microsoft Word 98 can import the following text formats without a converter:

  • MS-DOS Text
  • MS-DOS Text with Line Breaks
  • Rich Text Format (RTF)
  • Text Only (TXT)
  • Text with Line Breaks
  • Unicode Text - (new to Macintosh)
  • Word 1.x - 5.1 for the Macintosh
  • Word 1.x - 6.0 for Windows
  • Word 95 for Windows

Unsupported Converters

Microsoft Word 98 does not support converters for exporting to WordPerfect versions 6, 7, or 8.0.

Location of Converters

Text converters are installed in the following default location: 

   Microsoft Office 98:Shared Applications:Text Converters
