GP Fault When Custom Toolbar Docked in Normal View

 Last reviewed: February 5, 1998
Article ID: Q134142
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

If you close a template that has a custom toolbar which is docked by itself, Word hangs (stops responding) or issues a general protection (GP) fault. This behavior occurs only when you are in Normal view.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. This problem was corrected in Word for Windows 95, version 7.0a.

RESOLUTION

Upgrade to Word for Windows 95, version 7.0a.

To obtain Word for Windows 95 version 7.0a, call the Microsoft Order Desk at (800) 360-7561. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web page at: 

  http://www.microsoft.com/worldwide/default.htm
