OutlineShowFormat

Syntax

OutlineShowFormat

Example

Toggles character formatting on and off in outline view: 

   Sub MAIN
      On Error Goto notoutlineview
      OutlineShowFormat
      Goto done
   notoutlineview:
      MsgBox "Switch to outline view to use the OutlineShowFormat \
      command."
   done:
   End Sub
