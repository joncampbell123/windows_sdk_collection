WD: Word 97 Linked Picture Causes Invalid Page Fault in Mswrd832

 Last reviewed: July 31, 1997
Article ID: Q167622
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

In Microsoft Word 7.x, if you open a Word 97 document that contains a linked picture, you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
When you click Details, the following message is displayed 

   WINWORD caused a divide error in module MSWRD832.CNV at 0137:<address>
where the value for <address> may vary.

CAUSE

This problem occurs when you do the following in Word 97 before you open the document in Word 7.x:

  • You insert a picture and click to select the Link To File check box.

    -and-

  • You click to clear the Save With Document check box.

    -and-

  • You click to select the Float Over Text check box.

NOTE: When you insert a picture in Word 97, these check boxes can be accessed in the Insert Picture dialog box. To display the Insert Picture dialog box, point to Picture on the Insert menu, and then click From File.

WORKAROUND

To work around this problem, use any of the following methods.

Method 1: Save As Word 6.0/95

From Word 97, save the file in the Word 6.0/95 file format.

Method 2: Click to Select the Save With Document Check Box

In Word 97, when you click the Link To File check box while inserting a picture, do not clear the Save With Document check box. Then, save the document in Word 97 format and open the file in Word 7.0.

Method 3: Clear the Float Over Text Check Box

In Word 97, when you insert a linked picture, clear the Float Over Text check box. Then, save the document in Word 97 format and open the file in Word 7.0.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Word for Windows 95, versions 7.0 and 7.0a, do not include a converter to open Word 97 documents. You must use the Word 97 import converter to open Word 97 documents in word 7.x. For additional information on obtaining the Word 97 file converter for Word 7.x (Word 95) and Word 6.x, please see the following article in the Microsoft Knowledge Base:
