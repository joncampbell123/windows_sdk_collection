WD97: Clipart Images Display Incorrect Format Picture Options

 Last reviewed: September 17, 1997
Article ID: Q173286
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

Only the Position tab is present in the Format Picture dialog box after you insert a ClipArt Image into a Web page and then click Format Picture.

RESOLUTION

Save the document as HTML, close, and then re-open the document. The ClipArt image will now have been converted to a GIF image and will display more option tabs in the Format Picture dialog box.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
