IA: Right Aligned Text in Table Displayed Incorrectly

 Last reviewed: February 5, 1998
Article ID: Q145635
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 2.0z
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

When you right-align text that is formatted with a hard carriage return (<P>) at the end of each line, all lines of text but the first are aligned with the left margin.

This problem shows up after you view the HTML source, save, close, and then reopen the HTML document, or when you browse a page that contains this formatting. In contrast, when your text ends with the Break (<BR>) element, all text is displayed aligned at the right margin. This is a display problem in Internet Assistant, and does not affect how other browsers display your page.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Assistant for Word, version 2.0z. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
