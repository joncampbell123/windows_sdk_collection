"Missing Picture" Error After Moving BatchConversion Macro

 Last reviewed: February 5, 1998
Article ID: Q117406
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

If you move the BatchConversion macro from the CONVERT.DOT template to a different template and then run the macro, the macro's dialog box may display three boxes with the words "missing picture" in them. This problem does not affect the operation or reliability of the macro.

CAUSE

This problem occurs because the pictures used in the dialog box for the macro are AutoText entries stored in the CONVERT.DOT template. AutoText entries are not moved automatically to the new template when you move a macro.

WORKAROUNDS

Method 1: If you move the BatchConversion macro to a new template, you must 

          also move the AutoText entries that contain the graphics. The
          AutoText entries are called "pic1" and "linepic." "Pic1" is a
          diagram using the Word logo and arrows, and "linepic" is a thick
          line.

          To move the AutoText entries from one template to another:

          a. From the File menu, choose Templates.

          b. Choose the Organizer button and select the tab for AutoText.

          c. In the AutoText Available In box on the left side of the
             Organizer, open the template that contains the AutoText entry
             you want to copy or move.

          d. In the AutoText Available In box on the right side of the
             Organizer, open the template to which you want to copy or
             move the AutoText entries.

          e. Select the AutoText entries you want to copy. To select
             individual AutoText entries, hold down the CTRL key and click
             each AutoText entry. To select a series, hold down the SHIFT
             key and click the first and last items in the series.

          f. Choose the Copy button and then choose the Close button.
Method 2: Instead of moving the macro to a new template, you can add the 
          CONVERT.DOT template as a global add-in that will make all its
          macros available. To do this, select Templates from the File menu
          and choose the Add button. Select the CONVERT.DOT template and
          choose OK. In the Templates And Add-ins dialog box, make sure
          that CONVERT.DOT is selected and choose OK. You can access the
          macro by choosing Macro from the Tools menu, then selecting
          CONVERT.DOT in the box below Macros Available In.
