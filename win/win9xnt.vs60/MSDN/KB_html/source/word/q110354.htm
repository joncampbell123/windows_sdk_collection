Embedded Font Displays as Substitution Font

 Last reviewed: February 5, 1998
Article ID: Q110354
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you open a document in Word with an embedded font and choose the Font Substitution button, the font embedded in the document will be shown as the default substitution for itself. (To locate the Font Substitution button, choose Options from the Tools menu and select the Compatibility tab.)

CAUSE

The substitute is obtained by asking Windows to load the font in question, then retrieving the font that Windows actually loaded as a substitute. Since the embedded font is available, Windows returns that value. There is no API to ask Windows for a font substitution when the font is available.

STATUS

This is not a problem in Word 6.0 for the Macintosh.
