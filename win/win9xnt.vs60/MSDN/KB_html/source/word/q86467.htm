WD: Bookmark Command Unavailable If End-Of-Cell Mark Is Selected

 Last reviewed: February 5, 1998
Article ID: Q86467
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SYMPTOMS

When you select a table cell (including the end-of-cell mark), the Bookmark command on the Insert menu is unavailable. The reserved bookmark \SEL is also not available.

Note: If you select an entire table or an entire row of the table, the Bookmark command will be available.

WORKAROUND

To work around this problem, do not select the end-of-cell mark. Select only the contents of the table cell.

NOTE: To see the end-of-cell marks, on the Tools menu, click Options. In the Options dialog box, under Category, click to select View, and then under non-printing characters, click to select the All check box, and then click OK.

REFERENCES
