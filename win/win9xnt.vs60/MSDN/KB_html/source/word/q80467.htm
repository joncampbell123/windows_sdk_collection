WD: Linking Color Charts Between Microsoft Excel and Word

 Last reviewed: February 2, 1998
Article ID: Q80467
The information in this article applies to:
  • Microsoft Word for Macintosh, versions 4.0, 5.0, 5.1, 5.1a
  • Microsoft Excel for Macintosh, versions 2.2, 3.0, 4.0
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

A Microsoft Excel for the Macintosh chart containing color that is linked to a Microsoft Word for the Macintosh document does not retain its color properties when the link is updated from Word.

WORKAROUND

To maintain the color properties of the chart, follow these steps:

  1. In Microsoft Excel, open the Chart file and click Print on the File menu.

  2. Click to select Print Using Color and Print Preview. Click Print.

  3. Click Close in version 3.0 or 4.0 or Cancel in 2.2 to exit Print Preview.

  4. On the Chart menu, click Select Chart (COMMAND+A).

  5. On the Edit menu, click Copy.

    NOTE: The above steps MUST be performed in the chart window. Copying a chart placed as an object on a spreadsheet using the Ribbon Chart Tool will not link to Word. If the chart is an object, double-click it to open it into the chart window.

  6. Start Microsoft Word.

  7. On the Edit menu, click Paste Special (Paste Link in Word 4.x).

  8. Click Paste Link in Word 5.0/5.1.

    NOTE: When you Paste Link a Microsoft Excel 4.0 chart into Word 5.x, the Paste option will be set to Picture by default. If you set the option to Excel Chart, Paste Link will be disabled. If you Paste Link as a picture, changes in the Microsoft Excel chart will be updated in the link to the Word document.

    To update the Microsoft Excel link in System 6.x or in System 7.x when the Link Options preference is set to Manually, follow these steps:

    a. Select the chart in Microsoft Word.

    b. On the Edit menu, click Link Options and the click Update Now. 

          (In Word 4.0, just click Update Link on the Edit menu).

Note 1

The chart automatically updates if the Link Option is set to Automatically. To set this option, click Link Options on the Edit menu and then select Automatically.

Note 2

Dynamic linking is not supported in Excel 2.2 or Word 4.0 since neither program supports AppleEvents under System 7.0. In addition, QuickSwitch, available in Microsoft Word 4.0 and System 6.x, is not supported in System 7.0.

Note 3

The update of the link is accomplished by following the steps for Word 5.0/5.1 listed above.

Note 4

Update the link by following the steps for Word 4.0 listed above.

Note 5

Word obtains the size of the chart; the updated link may be cropped in Word if the chart is enlarged in Microsoft Excel.
