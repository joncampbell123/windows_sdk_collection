WD97: Word for Alpha Crashes Converting Some WP Graphics

 Last reviewed: July 31, 1997
Article ID: Q168351
The information in this article applies to:
  • Microsoft Word 97 for Windows on the following platform: Alpha

SYMPTOMS

If you open a WordPerfect 5.1 for MS-DOS document in Microsoft Word 97 for Windows on the Alpha platform, you may experience an Invalid Page Fault (IPF) if the document contains some types of graphics.

CAUSE

The IPF occurs when the WordPerfect converter (WPFT532.CNV) calls the WPG export filter (WPGEXP32.FLT).

NOTE: Microsoft Word 97 running on the Intel platform does not generate an IPF; instead you get the following error message: 

   Unable to convert graphic. Continue with document conversion?

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
