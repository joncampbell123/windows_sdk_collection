Use WexTech Doc-To-Help to Create Windows Help Files

 Last reviewed: July 30, 1997
Article ID: Q80031
The information in this article applies to:
  • Microsoft Word for Windows versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

Doc-To-Help is a product from WexTech Systems that creates Microsoft Windows online Help files using Word for Windows document files.

MORE INFORMATION

The Doc-To-Help template ships with a tutorial that shows you how to embed graphics created in Microsoft PaintBrush or Microsoft Draw. The template contains one sample graphics file in Windows Metafile (WMF) format.

If your embedded graphics lose scaling or cropping formatting after you update the EMBED field, the \* mergeformat switch may be missing from the field. For more information on this problem, query on the following words in the Microsoft Knowledge Base: 

   updating and fields and reset and ole and scaling and winword
