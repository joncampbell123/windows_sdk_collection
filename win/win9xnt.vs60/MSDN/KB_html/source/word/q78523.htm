Word for Windows: Updating Fields in a Table

 Last reviewed: February 5, 1998
Article ID: Q78523
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows version 2.0, to update a field in a table, you must select one of the following: 

   The entire table (ALT+Keypad 5)

  -or-

   The field to be updated

  -or-

   The entire row of cells containing the field
If you make any other selections, Word produces a beep, and the following error message displays on the status bar: 

  Word cannot update fields in this selection.
In Word for Windows version 1.x, no error message displays; however, the field does not update unless your selection matches one of those listed above.
