WD6x: Word 6.0 for Windows Function Key Keyboard Commands

 Last reviewed: February 5, 1998
Article ID: Q118581
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

Microsoft has an Application Note available that contains a two-page list of all the keyboard shortcuts you can use in Word 6.0 for Windows. This list is a compilation of the individual keyboard shortcut lists available in Word Help.

To see the complete list of keyboard shortcuts, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119621
   TITLE     : Comparison of Shortcut Keys for MacWord 6 and WinWord 6
