Cannot Replace Border or Shading Formats

 Last reviewed: February 5, 1998
Article ID: Q105783
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

You cannot find or replace border or shading formatting using the Find and Replace commands on the Edit menu in Word for Windows. Word does not include the Borders And Shading command on the Format drop-down list in the Edit Replace and Edit Find dialog boxes.

WORKAROUND

Create a style with the border and shading formats you want to find or replace, then use that style as search criteria or replacement formatting in the Edit Replace dialog box, as described in the following instructions:

  1. From the Format menu, choose Style.

  2. Choose the New button.

  3. Type or select a name in the Name box.

  4. Choose the Format button and from the list that appears, select Border.

  5. Select the border and shading settings you want and then choose OK.

  6. If the style is for current document only, do not select the Add to Template option. Choose OK in the New Style dialog box.

  7. Choose the Close button in the Format Styles dialog box.

  8. From the Edit menu, choose Replace.

  9. To search for text based on the border and shading formats, position the insertion point in the Find What box and then choose the Format button. On the list that appears, select Style.

    -or-

    To replace with the border and shading formats, position the insertion point in the Replace With box and then choose the Format button. On the list that appears, select Style.

  10. Choose the Find Next, Replace, or Replace All button.
