WD6X: Mac Word File Doesn't Appear in QuarkXPress Get Text List

 Last reviewed: November 17, 1997
Article ID: Q137519
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When you try to open a Word for the Macintosh document in QuarkXPress 3.x for the Macintosh, the file does not appear in the Get Text dialog box.

CAUSE

The file does not appear because you don't have the correct QuarkXPress filter installed to recognize the Word 6.x file format.

WORKAROUND

Save the document in Word 4.x for the Macintosh format. Note that while saving the document in Word 5.x format allows QuarkXPress to recognize the file, the font applied to the text of the document may change to an unknown font, and a warning message may appear.

To save a document in Word 4.0 format from Word 6.x for the Macintosh, use these steps:

  1. On the File menu, click Save As.

  2. Change the name of the document, if desired.

  3. In the Save File As Type list, select Word 4.0 for the Macintosh.

  4. Click Save.

MORE INFORMATION

To open Word 6.x documents directly in QuarkXPress, you must obtain the QuarkXPress filter that recognizes and opens Word 6.x files. The current MS-Word filter is version 3.1, which allows you to import files into QuarkXPress from Microsoft Word 3.x to 6.x for Macintosh and Power Macintosh, and Microsoft Word 2.x to 6.x for Windows.
