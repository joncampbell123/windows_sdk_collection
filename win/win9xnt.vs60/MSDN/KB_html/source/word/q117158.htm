GP Fault with Word Assistant Under Windows NT

 Last reviewed: February 5, 1998
Article ID: Q117158
6.00 WINDOWS NT kbusage kbenv buglist6.00 fixlist6.00a

The information in this article applies to:

  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows NT version 3.1

SYMPTOMS

When you use Word Assistant's Font Selector command from the Format menu, you may receive a general protection (GP) fault in USER.EXE when you exit Word. This behavior occurs when you run Word 6.0 under Windows NT.

STATUS

Microsoft has confirmed this to be a problem in Word 6.0 for Windows. This problem was corrected in Word 6.0a for Windows.

RESOLUTION

To resolve this problem, upgrade to Word version 6.0a or later.

The most current version of Word 6.x for Windows is Word 6.0c. To obtain this update, contact the Microsoft Order Desk at (800) 360-7561 and request the Word 6.0c update. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web page at: 

  http://www.microsoft.com/worldwide/default.htm
