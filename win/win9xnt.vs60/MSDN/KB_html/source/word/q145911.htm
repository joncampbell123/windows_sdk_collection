Avery 2186 Label Printed Incorrectly with Canon BJC-70 Driver

 Last reviewed: February 5, 1998
Article ID: Q145911
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you print to an Avery 2186 label with the Canon BJC-70 printer driver that ships with Windows 95, only the first two lines of text are printed.

CAUSE

The Canon BJC-70 printer driver version 3.2 that ships with Windows 95 does not print correctly when you select a custom paper size with a height of 5 inches. The Avery 2186 label is 4.3 inches wide and 5 inches high.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Use one of the following methods to work around this problem:

Method 1: Change the page height to 5.1 inches or 4.9 inches.

Method 2: Install the Canon BJC-70 printer driver version 2.1 for 

          Windows 3.11. Printing will be slower, but you can specify a
          custom page size with a height of 5 inches.
