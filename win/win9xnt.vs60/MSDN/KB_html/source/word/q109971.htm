Errors If Share Is Loaded High

 Last reviewed: February 6, 1998
Article ID: Q109971
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

  • The following error message may occur in Word for Windows when you attempt to save a file: 

          Word cannot save or create this file. Make sure the disk is not
      write protected. <path><filename>
  • The following error messages may occur when you attempt to open a second or third file in Word 

          File is in use by another user. <path><filename>

    followed by: 

          Cannot open document.
  • One of the following errors message may occur when you open a file based on a template with a standalone (not a network ) installation of Word: 

          Word cannot open the document.


          Error: "<path>\filename is being used by another user.
      Do you want to make a copy?"

Word Version 2.x

  • The following error message may occur if you modify NORMAL.DOT and attempt to save the file: 

          You cannot save while the file is in use by another process.
      Try saving the file with a new name.

    The file cannot be saved, even to a new name.

CAUSE

If any LOADHIGH commands exist for SHARE, Word Setup will retain all those LOADHIGH commands when it writes the new SHARE line in the AUTOEXEC.BAT file. In doing so, the /L:500 /F:5100 switches may not load if there is not enough room in the upper memory area. No error will occur when the machine is started, but Word will give one of the errors listed above.

RESOLUTION

Method 1: Remove the LOADHIGH commands and rerun Optimize or MemMaker.

Method 2: If you are using Word 6.0a and operating Windows in the 386 

          Enhanced mode, VSHARE.386 replaces the need to load the MS-DOS
          SHARE utility.

MORE INFORMATION

Word uses the SHARE.EXE program for object linking and embedding (OLE ) version 2.0 operations. SHARE.EXE provides the range-locking functionality that Word uses in compound files.
