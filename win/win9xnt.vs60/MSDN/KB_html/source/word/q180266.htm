WD98: How to Install More Templates and Wizards

 Last reviewed: February 17, 1998
Article ID: Q180266
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

The text that appears in the "More Wizards and Templates" template contains incomplete information about using the Setup program, and incomplete information on how to install additional templates and wizards.

To find the More Wizards And Templates template, use the following steps:

  1. On the File menu, click New.

  2. Click the Other Documents tab, and then double-click the "More Templates and Wizards" icon.

RESOLUTION

Installing Additional Templates Using Office Setup

To install additional templates using Office Setup, follow these steps:

  1. Quit all Microsoft Office programs.

  2. Put the Office 98 Macintosh Edition CD-ROM into your CD-ROM drive, and double-click the Microsoft Office 98 Macintosh Edition icon on your Macintosh desktop.

  3. Double-click the Office Custom Install icon.

  4. Double-click the Microsoft Office Installer to start the Setup program. 

          The Setup program starts and the Microsoft Office Installer dialog
      box appears. The three choices that appear in the dialog box are Easy
      Install, Custom Install and Remove.

  5. Click to select Custom Install.

  6. Click to clear each of the check boxes.

    NOTE: Clearing each of the check boxes will NOT remove these applications.

  7. Expand the Microsoft Word option.

  8. Expand the Wizards and Templates option.

  9. to install any of the following items, click to select the check box next to it, and then click the Install button: 

           - Faxes
       - Memos
       - Reports
       - More Wizards
       - Resumes
       - Letters

The Microsoft Office Installer will now install the selected wizards and templates.

Installing Additional Templates from the Value Pack Folder

To install additional templates from the Value Pack folder, follow these steps:

  1. Quit all Microsoft Office programs and put the Office 98 Macintosh Edition CD-ROM into the CD-ROM drive.

  2. Double-click the Microsoft Office 98 Macintosh Edition icon on your Macintosh desktop.

  3. Double-click the Value Pack icon.

  4. Double-click the Value Pack Installer icon.

  5. Do one or both of the following: 

           - Click to select the Business Tools And Templates check box.
      

             -or-
      
       - Expand the Templates option and then click to select the
         Additional Word Templates check box.


  6. Click Install.

    

Installing Templates from the Microsoft Office Macintosh Edition Web Site

 


If you have a dial-up connection to the World Wide Web, use your Internet
browser to connect to the Microsoft Office Macintosh Edition Web Page to
manually check for the additional templates and wizards. To do this, follow
these steps:


  1. From your Internet browser connect to the following address on the World
   Wide Web:

    

          http://www.microsoft.com/macoffice

    

  2. Click Free Downloads.


  3. Click Microsoft Word Downloads and then follow the prompts to download.

    

MORE INFORMATION

 



Incomplete Text That Appears in the "More Wizards And Templates" Template

 


More Wizards and Templates:


Microsoft Word provides many wizards and templates to help you create great
documents. For example, the Letter Wizard helps you format a letter for
professional or personal use, and the Fax Wizard creates a cover page and
sends your document to someone else.


If you have not found the template or wizard you're looking for, just do
one or more of the following:


  • Run the Customized install application, and click the Custom button to
   install additional files.


  • Go to the Microsoft Office Web Site and download free templates and
   wizards. On the Help menu, point to Help on the Web, and then
   click Free Stuff in the resulting Web page. Or, click the hyperlink
   below.

    

          http://www.microsoft.com/macoffice/

    

  • If you installed Microsoft Word from a CD, insert the disk and
   double-click the Value Pack folder to see what other templates
   and wizards are available. Copy the files you want to the user template
   folder. The current user template folder can be found in Word when you
   click Preferences (on the Tools menu), and then click File Locations.
