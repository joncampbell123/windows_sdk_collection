WD: Error Msg: You Cannot Paste This Selection into a Table

 Last reviewed: February 5, 1998
Article ID: Q123702
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-cd 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for the Macintosh, version 6.0
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

When you try to paste a Microsoft Excel table into an existing table in Word, you will receive this error message: 

   You cannot paste this selection into a table.
NOTE: This error message is not generated in Word for the Macintosh, versions 5.1 and earlier.

WORKAROUNDS

To work around this problem, do either of the following:

  • Paste the selection directly above or below the table.

    -or-

  • Split the table into two tables to paste the selection "inside" the table, and remove any extra paragraph marks that appear between the tables.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
