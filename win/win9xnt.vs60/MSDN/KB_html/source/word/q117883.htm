The information in this article applies to:

- Microsoft Word for Windows, versions 6.0, 6.0a

SUMMARY

Microsoft has an Application Note available that contains instructions for modifying the InfoSoft International Main Dictionary, which is the spelling dictionary included with Microsoft Word 6.0 for Windows. The InfoSoft Main Dictionary Modifier utility is provided on the disk included with this Application Note. The instructions that accompany the disk explain how to install and use the utility to add, delete, and modify words in the Word 6.0 spelling dictionary.

This Application Note contains the "InfoSoft User's Guide for the Main Dictionary Modifier," with some modification by Microsoft. Chapter 1 contains the installation and usage instructions for the Main Dictionary Modifier. To modify a dictionary, which is a database, you must create an input file that tells the database which words to add, delete, or change. Chapter 2 provides instructions for creating this input file. Appendix A contains a list of extended ASCII characters and character mappings for InfoSoft International products.

This Application Note, WD0980, is titled "Main Spelling Dictionary Modifier" and can be obtained by calling the Microsoft Word for Windows support line at 462-9673. This Application Note is not available from online services.

TECHNICAL SUPPORT

The Main Dictionary Modifier is a product of the InfoSoft International Company, which is solely responsible for its support. Microsoft does not support this utility. If you need technical support or have any questions about the Main Dictionary Modifier, please contact the InfoSoft International Company at (617) 753-6500.
