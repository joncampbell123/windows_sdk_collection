WD6X: WordBasic Err=543 in 5.1 Upgrade Document

 Last reviewed: February 6, 1998
Article ID: Q122693
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0
  • Microsoft Office for the Macintosh, version 4.2

SYMPTOMS

If you installed Word for the Macintosh using the Microsoft Office 4.2 Setup program, the following error may occur when you open the 5.1 Upgrade document from the Word 6.0:Macro folder and choose the Convert 5.1 Addresses button: 

   WordBasic Err=543
   Unable to open specified library.
NOTE: The Convert 5.1 Addresses button is located on the 5.1 Upgrade toolbar, which you can display by choosing Toolbars from the View menu and selecting 5.1 Upgrade in the Toolbars list.

This error also occurs if you run the AddressBook macro from the Tools Macro dialog box.

This error does not occur if you installed Word using the Word 6.0 Setup program.

CAUSE

The macro cannot find the Address Book Converter file it needs. Office 4.2 Setup incorrectly installs the Address Book Converter file in the System:Preferences:Word Startup Folder (6) instead of the Word 6.0:Macros folder. This error also occurs if the Word icon is moved from the Word 6.0 folder.

RESOLUTION

Move the Address Book Converter file from the System:Preferences:Word Startup Folder (6) to the Word 6.0:Macros folder.

STATUS
