WD: Inserting Caption Changes Existing Caption Heading Style

 Last reviewed: February 3, 1998
Article ID: Q144082
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you insert a caption in a document that includes chapter numbering and that is associated with a heading style, all captions are reset to reflect the heading numbering associated with the last heading style you selected.

For example, when you define the first caption as Figure 1-1 using the Heading 1 style, and then create the second caption as Figure A-1 using the Heading 2 style, the second caption you create overwrites the first caption. You now have captions labeled Figure A-1 and Figure A-2.

CAUSE

When you select the Include Chapter Numbers and Chapter Starts With Style options, Word replaces the existing captions with new ones, modifying all captions to be based on the same heading style.

WORKAROUND

When you have finished adding all captions to your document, follow these steps to correct them:

  1. Press ALT+F9 (Windows) or OPTION+F9 (Macintosh) to view Field Codes. The caption should like something like this: 

          Figure {STYLEREF 2 \n}-{SEQ Figure \* ARABIC}

  2. For those captions you want to associate with the heading level 1 style, change the 2 to a 1, so that the caption field looks like this: 

          Figure {STYLEREF 1 \n}-{SEQ Figure \* ARABIC}

  3. Press F9 to update the field.

  4. Press ALT+F9 (Windows) or OPTION+F9 (Macintosh) to turn off Field Codes view.
