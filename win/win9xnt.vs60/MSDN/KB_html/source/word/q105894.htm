WD: List of Search Criteria for Next Citation

 Last reviewed: February 2, 1998
Article ID: Q105894
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a

SUMMARY

The following is a list of items that Next Citation searches for when marking citations for a table of authorities. Word finds any whole word or partial word instance of these items. 

   in re
   v.
   Id.
   , Supra
   Ibid
   , Infra
   Cong. (for Congress)
   Sess. (for Session)
   18xx) (to find dates in parentheses; x is any number)
   19xx) (to find dates in parentheses; x is any number)
   <section symbol>
   Any previously defined short form of an authority that is unmarked

MORE INFORMATION

To find Next Citation, click Index And Tables on the Insert menu, select the Table Of Authorities tab, click Mark Citation, and then click Next Citation.
