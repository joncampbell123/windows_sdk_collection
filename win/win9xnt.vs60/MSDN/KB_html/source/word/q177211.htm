WD5x: Table Box Border Converted to Partial Grid Border

 Last reviewed: December 2, 1997
Article ID: Q177211
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.1, 5.1a

SYMPTOMS

In Word 5.1 for the Macintosh, when you open a document created in a later version of Word, if the document contains a table bordered as a box, the table borders are not converted correctly and may be displayed with partial grid bordering.

RESOLUTION

Open the document in Word version 6.0 or later.

To work around this problem in Word 5.1, select your table and reapply the table border. To do this, follow these steps:

  1. Hold the OPTION key and then double-click the table to select it.

  2. On the Format menu, click Border.

  3. In the Border dialog box, under Presets, click to select Box and then click OK.

    NOTE: In the Border dialog box, you can change the type of table border and line thickness as desired.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been corrected in Microsoft Word 6.0 for the Macintosh.

MORE INFORMATION

For more information about working with tables in Word 5.1, click "Microsoft Word Help" on the Help menu, scroll down and select any of the Table topics, and then click the Help button.
