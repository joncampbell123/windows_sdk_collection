GP Fault in Internet Assistant Opening URL to Some Sites

 Last reviewed: February 5, 1998
Article ID: Q126169
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

You may receive the following general protection (GP) fault when attempting to connect to a server location: 

   Internet caused a General Protection Fault in module IWIA.EXE
   at 0006:23DA

CAUSE

  • The site is issuing an incorrect MIME (Multipurpose Internet Mail Extensions) type.

    -or-

  • The server's software is out of date.

RESOLUTION

  • Use a different browser to make a connection to the server. If the problem is a bad MIME type, using the other software may address the problem.

    -or-

  • When server software is the problem, try entering the IP address of the server, or try again later when the server is less busy.
