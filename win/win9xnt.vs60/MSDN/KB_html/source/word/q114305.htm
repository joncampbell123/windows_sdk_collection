Cannot Undo Because Undo List Is Empty

 Last reviewed: February 5, 1998
Article ID: Q114305
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

Sometimes you cannot reverse changes you make in a Word for Windows document because the Undo list is empty.

CAUSE

Below is a list of certain Word operations that clear the Undo list. When this happens, you can no longer undo any previous operations.

  • Word autosaves a Word version 2.x for Windows document.
  • You run the AutoFormat command, and then choose the Reject All button.
  • You change a normal Word document into a mail merge main document using the Mail Merge command on the Tools menu.

STATUS

Microsoft has confirmed this to be a problem in Word versions 6.0 and 6.0a for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
