WD: Large Text Buttons from Word 6.x Truncated in Word 7.0

 Last reviewed: February 5, 1998
Article ID: Q138963
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

When you open Word 6.0 for Windows templates in Word for Windows 95 or Word 97 for Windows and the templates contain customized toolbar settings from Word 6.0, text buttons that contain large amounts of text may appear truncated.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95 versions 7.0 and 7.0a and Word 97 for Windows. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

  1. Open the template that contains the customized toolbar.

  2. With the right mouse button, click the truncated button.

  3. Click Customize.

  4. Ignore the Customize dialog box. With the right mouse button, click the button on the toolbar again.

  5. Click Edit Button Image.

  6. Choose OK.

  7. Save the Template.
