WD97: Some Wizards Don't Show Finish On Progress Indicator

 Last reviewed: February 5, 1998
Article ID: Q157580
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When running one of the Word 97 wizards, the word Finish may not appear on the wizard progress indicator.

WORKAROUND

Change the Windows Appearance Colors. Use one of the following methods:

Method 1: To change the Selected Items color, use the following steps: 

          a.  From the Start menu, choose Settings, then click Control
              Panel.

          b.  From the Control Panel, double-click Display.

          c.  From the Display Properties dialog, select the Appearance
              tab. Then for "Item:", choose "Selected Items".

          d.  Under "Color:" for Font, select "White". Then click the OK
              button.
Method 2: To set the Windows Colors back to the Windows default, use the 
          following steps:

          a.  On the Start menu, choose Settings, then click Control
              Panel.

          b.  On the Control Panel, double-click Display.

          c.  In the Display Properties dialog, select the Appearance
              tab. Under "Scheme:", choose "Windows Standard", then click
              OK.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The appearance of the wizards for Microsoft Word 97 for Windows has changed. There are six Word wizards that provide a progress indicator on the left side of the wizard window. The top of the indicator has Start and the bottom of the indicator has Finish. The wizard indicator visually identifies the current step in the wizard process. It also indicates how many steps remain in the wizard process.

Certain font/color combinations can make the word Finish invisible on the wizard progress indicator.

There are six wizards provided in Word that use the progress indicator. They are: Agenda Wizard, Calendar Wizard, Fax Wizard, Newsletter Wizard, Pleading Wizard, and the Resume Wizard.

There are two wizards that exhibit this problem. They are the Agenda Wizard and the Resume Wizard. The other wizards display the word Finish correctly.
