Envelopes Print Blank Using Canon LBP-4 Printer

 Last reviewed: February 5, 1998
Article ID: Q110868
6.00 WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you try to print a custom size document or an envelope other than Com10 or #12 from Word 6.0 for Windows to a Canon LBP-4 printer (LBPIII.DRV version 3.09 and GENDRV.DLL version 1.65), the document or envelope will print out blank or missing text.

RESOLUTION

  1. Open a blank 8-1/2-by-11-inch letter document.

  2. From the File menu, choose Page Setup, select the Paper Size tab, and then select Portrait or Landscape under Orientation.

  3. Select the Margins tab and do one of the following: 

        - For landscape documents, increase the bottom and left margins so that
      the working area of the document equals the paper size you want to
      print.


        - For portrait documents, increase the left and top margins so that the
      working area of the document equals the paper you want to print.

  4. Select the Paper Source tab and select Manual Feed under First Page.

  5. Enter your text into the proper spot on the document and print it.

The document should print correctly. If the text prints in the wrong location, either move the text or change the margin settings to correct it.

MORE INFORMATION

Envelopes of size Com10 and #12 will print correctly using the Envelopes And Labels dialog box. All other envelope sizes or custom paper sizes will either print blank or print with text missing.
