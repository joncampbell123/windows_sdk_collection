WD6x: Message Box Buttons Appear in the Wrong Order

 Last reviewed: February 18, 1998
Article ID: Q123431
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1

SYMPTOMS

Buttons assigned to a message box appear in the opposite order than expected. This problem does not affect the default button that maintains its focus, or the results of choosing a specific button.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The following table shows how the buttons should and instead do appear in a message box. 

   Should appear as            But instead appear as
   ---------------------------------------------------

   Okay    Cancel               Cancel   Okay

   Abort   Retry   Ignore       Ignore   Retry   Abort

   Yes     No      Cancel       Cancel   No      Yes

   Yes     No                   No       Yes

   Retry   Cancel               Cancel   Retry

REFERENCES

"Microsoft Word Developer's Kit," version 6.0, page 600
