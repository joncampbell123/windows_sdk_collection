Can't Toggle Single Field Code in Page Layout View

 Last reviewed: February 5, 1998
Article ID: Q144022
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

In Page Layout View, holding the SHIFT key and pressing F9 to toggle a single field code will instead toggle all fields in the document.

WORKAROUND

Change to Normal View. (To change to Normal view, click Normal on the View menu.) Then press SHIFT+F9 to toggle a single field.

MORE INFORMATION

In Microsoft Word for Windows 2.0, pressing SHIFT+F9 toggled field codes. If the document was in Normal View, then just the field code at the insertion point would be toggled. If the document was in Page Layout View, then all the field codes in the document would be toggled.

In Microsoft Word for Windows 6.0, Microsoft Word added functionality with the ALT+F9 keys. This allowed the toggling of all field codes at once, regardless of the document view.

The backward compatibility was retained in Microsoft Word 6.0 and later versions to allow the use of the SHIFT+F9 keys. The functionality is the same as in Word 2.0.
