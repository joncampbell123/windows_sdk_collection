Files Opened from Mail and File Manager Not Added to MRU List

 Last reviewed: February 5, 1998
Article ID: Q106449
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you open a file from Microsoft Mail or File Manager and then close the file without saving, Word will not add the filename to the Most Recently Used (MRU) list at the bottom of the File menu.

CAUSE

Word displays the filenames of documents opened from Microsoft Mail or File Manager in the MRU list only if the document was saved before closing.

If the document was opened from Mail and saved, the path will be the TEMP directory.

For example, if your TEMP directory is C:\TEMP and the document opened from Microsoft Mail is FILE.DOC, the MRU list path will be C:\TEMP\FILE.DOC.

This behavior is by design.
