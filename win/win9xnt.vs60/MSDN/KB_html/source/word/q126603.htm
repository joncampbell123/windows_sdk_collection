SGML Author Compact Disk

Contents of this CD

The CD that is included in your package includes the following items.

Microsoft SGML Author for Word(TM). The version supplied on the CD is the same as the version on the floppy disks. The only difference is that files on the CD are not compressed. Running Setup from the CD is faster than running Setup from the floppy disks.

The Microsoft Office(TM) Developer's Kit. The Office development kit is a compilation of technical information that will help you build custom applications with the products included in Microsoft Office. See README.TXT in the OFFICEDK directory for more information.

SureSTYLE(TM), a product developed by Avalanche, Inc. For information about this utility, see the section called "About Avalanche SURESTYLE (Preview)," later in this document.

About SGML Author for Word

Microsoft SGML Author for Word allows you to save Word documents as SGML instances. SGML Author can be configured to create SGML that conforms to any DTD that describes documents that Word can create.

To install SGML Author

From the Windows Program Manager, select File/Run and type x:\Author\setup.exe (where x is the CD-ROM drive designation) then follow the instruction that appear on your screen.

About Avalanche SURESTYLE (Preview)

SURESTYLE(TM) is a utility developed by Avalanche Development Co. that helps you standardize the use of styles--both paragraph and character--in your Word documents. The sample version of Avalanche SURESTYLE, called SURESTYLE (Preview), which is included on the CD in your package, demonstrates the full functionality of the product. SURESTYLE (Preview) is also available direct from Avalanche on 3.5" disks; shipping charges apply. You can also obtain the full version of the product, SURESTYLE version 1.0, direct from Avalanche for $495 in the USA and Canada; shipping charges apply. Contact Avalanche Development Co., at the address below.

When you run SURESTYLE (Preview) on your document, the utility analyzes and processes the first seven pages and creates a new version of the document in which appropriate character and paragraph styles have been applied. The new document also includes annotations that give you an opportunity to review the changes SURESTYLE has made.

Just as the spell-checker compares the words it finds to a known list of words and reports back the discrepancies, SURESTYLE compares the objects it finds to a known set of object descriptions (the stylesheet). Using the stylesheet specified by the template attached to the document, SURESTYLE applies the paragraph and character styles that most closely match each object's formatting. SURESTYLE saves the results into a new file, which contains Word annotations that note the style differences between the original file and the new file. You can write documents in a way that is familiar and comfortable to you, then use SURESTYLE to bring the documents into conformity with a standard-- which in this case is defined by the stylesheet.

The power and configurability of SURESTYLE version 1.0 gives you the flexibility to direct and fine-tune the style-checking process by: 

  *  Enabling custom configuration through a visual interface

  *  Helping workgroups coordinate different authoring styles and templates

  *  Enforcing usage by "driving" styles into the correct template

  *  Normalizing different forms of legacy documents by transforming them
     into a single, preferred form

To install SureSTYLE

From the Windows Program Manager, select File/Run and type x:\surestyl\surestyl.exe (where x is the CD-ROM drive designation) then follow the instructions that appear on your screen.

Using SURESTYLE

After using Word to create a document, choose SureSTYLE from the Format menu, and then click OK in the initial dialog box. SureSTYLE will make a copy of your current document and use the annotation feature in Word to explain the modifications made. To see these annotations, click Annotations on the View menu. If you want to remove the annotations from your document, click Clear SureSTYLE Annotations on the Tools menu.
