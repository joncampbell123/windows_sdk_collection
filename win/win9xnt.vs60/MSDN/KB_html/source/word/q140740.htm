"Shortcut to Old Template.lnk Is Unavailable" Error

 Last reviewed: February 5, 1998
Article ID: Q140740
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

You get the following error message when you click New on the File menu. 

   The drive or network connection that the shortcut "Shortcut to Old
   Templates.lnk" refers to is unavailable. Make sure that the disk is
   properly inserted or the network resource is available and then try
   again.

CAUSE

The Shortcut to Old Templates.lnk points to an invalid folder. This link is created when you install Word 7.0 over Word 6.0 to provide a link to your Word 6.0 template folder. The error occurs if you remove or move this folder.

RESOLUTION

Remove the Shortcut to Old Templates.lnk. This file is typically located in the Msoffice\Templates folder.
