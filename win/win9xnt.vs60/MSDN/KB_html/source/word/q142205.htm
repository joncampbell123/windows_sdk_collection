Word Reports No Help Topic for DocProperty Field

 Last reviewed: August 4, 1997
Article ID: Q142205
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you select the DocProperty or Ref field in the Insert Field dialog box, and then press F1 to access the Help topic, you get the following error message (which is incorrect): 

   The topic does not exist. Contact your application vendor for an
   updated Help file. (129)

WORKAROUND

To view the Help topic for the DocProperty field, click Answer Wizard on the Help menu, select the Index tab, type "DocProperty" (without the quotation marks), then highlight the "DocProperty Field" topic and click Display.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95. version 7.0. This problem was corrected in Word 97.
