Text Disappears When Using the Fax Wizard

 Last reviewed: February 5, 1998
Article ID: Q115409
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

When you use the Fax Wizard, if your fax message exceeds the set cell height of 300 points, text that does not fit in the cell disappears.

CAUSE

This behavior is by design in the Fax Wizard. The message area of the Fax Wizard is actually the row of a table. The height of this row is set to exactly 300 points. Any text that exceeds the row height disappears below the bottom of the row. (To check the row height specified, place the insertion point in the message area and choose Cell Height And Width from the Table menu.)

WORKAROUNDS

Method 1: Change the cell height: 

          a. Open a new Fax Wizard document.
          b. Place the insertion point in the message area (the cell below
             "Remarks").
          c. From the Table menu, choose Cell Height And Width. Select the
             Row tab.
          d. Change the Height of Row 1 to Auto. Type in your message.
             When your text reaches the end of page 1, the border
             formatting will not be correct. There will be a border on the
             bottom of page 1 but no border on the top of page 2.
          e. To correct the border problem, choose Borders And Shading
             from the Format menu and set the appropriate border.
Method 2: Delete the cell and type the message outside of a table: 

          a. Open a new Fax Wizard document.
          b. Place the insertion point in the message area (the cell below
             "Remarks").
          c. From the Table menu, choose Select Row.
          d. From the Table menu, choose Delete Row. (At this point, you
             could save this file as a template to be used for faxes with
             large messages.)
          e. Type your text. Since the text is not in a table, it will
             flow correctly.
