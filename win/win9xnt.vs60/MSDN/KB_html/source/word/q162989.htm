WD97: Can't Save Versions of a Document from Office Binder

 Last reviewed: February 5, 1998
Article ID: Q162989
The information in this article applies to:
  • Microsoft Office Binder 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

When you use the Save Version command in the Save As dialog box to save a Word document from Office Binder, a version of the document is not saved. Office Binder does not provide a way for you to open a version of a document. (To locate this command, click Save As File on the Section menu.)

CAUSE

The Save Version button in the Save As dialog box can be selected even though this feature is not currently supported in Office Binder.

WORKAROUND

To use the version feature with documents that you include in Office Binder, save the Word section, close it, open it in Word and then do the following:

  1. On the File menu, click Versions.

  2. Click Save Now.

  3. Add any comments and click OK.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word listed above. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
