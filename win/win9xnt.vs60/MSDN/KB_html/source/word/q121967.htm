Shortcut Key Description Incorrect for Cursor Keys

 Last reviewed: February 5, 1998
Article ID: Q121967
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you press one of the following keys when your insertion point is in the Press New Shortcut Key box, this box displays a reference to one of the numeric keypad keys.

The following table lists how each key will be displayed in the Press New Shortcut Key box: 

   Key        Display
   ---        -------
   Up         Num 8
   Down       Num 2
   Left       Num 4
   Right      Num 6
   Home       Num 7
   End        Num 1
   Page Up    Num 9
   Page Down  Num 3

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word for Windows listed above. This problem has been corrected in Word version 7.0a for Windows 95.

RESOLUTION

Although the key description is shown incorrectly, the key assignment operates correctly.
