Doc Err: Help Incorrectly Refers to WINWORD.INI

 Last reviewed: February 5, 1998
Article ID: Q126595
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

The context-sensitive help in the Modify Location dialog box incorrectly names WINWORD.INI as the file that retains the locations of different file types. This file does not exist in Word 6.0. The correct filename is WINWORD6.INI.

To see the incorrect information, follow these steps:

  1. From the Tools menu, choose Options.

  2. Choose the File Locations tab.

  3. Choose the Modify button.

  4. Choose the Help button.

The help screen states "Modifies the location in your startup or WINWORD.INI file."
