Word Cannot Find the Word Forms File MSWDS_EN.LEX

 Last reviewed: February 5, 1998
Article ID: Q132047
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Windows 95

SYMPTOMS

When you use the Find command on the Edit menu and select the Find All Word Forms option, you may receive the error message: 

   Word cannot find the Word Forms File MSWDS_EN.LEX for English(US).

CAUSE

When you perform a Typical setup of Word or Office, the file Mswds_en.lex is not installed. This file is required for the Find All Word Forms feature to work.

You may also receive this error message if the file Mswds_en.lex is damaged or missing.

RESOLUTION

Rerun Office (or Word) setup in maintenance mode, and do the following:

  1. Click the Add/Remove button.

  2. Select Office Tools if you are running Office Setup (or Word Tools if you are running Word Setup).

  3. Click the Change Option button.

  4. Select the Find All Word Forms check box.

  5. Click OK.

  6. Click Continue
