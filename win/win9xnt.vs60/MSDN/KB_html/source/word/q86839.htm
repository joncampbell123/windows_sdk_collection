MasterWord Products Available from Alki Software

 Last reviewed: July 30, 1997
Article ID: Q86839
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0 and 2.0a
  • Microsoft Windows operating system, versions 3.0 and 3.1

SUMMARY

This article describes MasterWord products, made by Alki Software, for Microsoft Word for Windows.

The products included here are manufactured by Alki Software, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.

MORE INFORMATION

Products that are available from Alki Software for Word for Windows include the following:

MasterWord

MasterWord consists of Help, new tools, and 25 handy macros.

Help is the Word for Windows online help feature, enhanced with new topics, discussions, illustrations, examples, and easy-to-use new features. New tools include a floating CustomBar, which resembles the Word for Windows tool bar, with the added ability to design custom pictures for the tool bar buttons. The Nickname tool lets you assign shortcut keys to any or all of the Word for Windows macros and commands.

Comprehensive Proofing Tools

This product includes a thesaurus and spelling checker.

Comprehensive Foreign Proofing Tools

Alki offers proofing tools in English, French, Swedish, Norwegian, Finnish, Spanish, French, Danish, Portuguese (Brazilian), Portuguese, German, Italian, and Dutch. For more information on these proofing tools, query on the following words in the Microsoft Knowledge Base: 

   alki and foreign and proofing and tools
