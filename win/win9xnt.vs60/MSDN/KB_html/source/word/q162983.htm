WD97: Background Sound Continues to Play

 Last reviewed: February 5, 1998
Article ID: Q162983
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you open a Web page that contains a background sound object, the sound object continues to play until the sound file is finished, even though you may have jumped to another Web page.

RESOLUTION

To stop the sound object, return to the Web page that contains the background sound object, click Background Sound on the Insert menu, and then click Stop.

NOTE: You must be on the Web page that contains the background sound object. If you are on another Web page, clicking Stop has no effect.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word version 97. We are researching this problem and will post new information here as it becomes available.
