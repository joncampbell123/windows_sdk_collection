Association File Doesn't Show .DTD or .DOT Changes

 Last reviewed: February 5, 1998
Article ID: Q128052
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you make changes to a .DTD or .DOT file after creating its association file (*.MAP), those changes do not appear when you open the association file in Microsoft SGML Author for Word.

For example, if you add a new paragraph style to a template, that style does not appear on the list of Paragraph Descriptors.

RESOLUTION

To see the changes in either file, you must create a new association file.
