EndOfWindow, EndOfWindow()

Syntax

EndOfWindow [Select] EndOfWindow([Select])

Example

This macro checks to see if the insertion point is already at the end of the window and should select back to the beginning of the window, or start at the beginning of the window and select to the end of the window. 

   Sub MAIN
      If EndOfWindow() Then
         StartOfWindow 1
      Else
         MsgBox "Currently at end of window"
         StartOfWindow
         EndOfWindow 1
      End If
   End Sub
