DMM: Some Postal Codes Missing After Importing Address File

 Last reviewed: March 9, 1998
Article ID: Q181873
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SYMPTOMS

When you import an address list, the postal code may be missing from some addresses.

CAUSE

This functionality is by design. The postal code field must contain only numbers and must not contain a combination of letters and numbers (as in an international postal code).

WORKAROUND

Use a valid United States address and postal code.

MORE INFORMATION

For more information about importing addresses, on the Index tab in Microsoft Direct Mail Manager Help, type "import," click to select Import List and then click Display.

Additional informational is available through the Microsoft Direct Mail Manager Web site. You can access this Web site by clicking the "Learn More" button on the Welcome screen of Microsoft Direct Mail Manager.

REFERENCES

USPS Publication 25, Designing Business Letter Mail, August 1995
