WD: How to Print Multiple Copies of an Envelope

 Last reviewed: February 4, 1998
Article ID: Q155198
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

Word does not provide an option in the "Envelopes and Labels" dialog box (or the "Envelopes" dialog box) to print multiple copies of an envelope. This article provides two methods to accomplish this.

MORE INFORMATION

To print multiple copies of an envelope, choose one of the following methods.

Method 1: Add the Envelope to the Document and Then Print It

  1. On the Tools menu, click "Envelopes and Labels."

  2. Click the Envelopes tab, type the delivery and return addresses, and click "Add to Document."

  3. On the File menu, click Print.

  4. Select the number of copies you want to print, select Pages, type "0" (without the quotation marks) in the Pages box, and click OK.

Method 2: Change Number of Copies to Print from the Printers Control Panel

Before you print the envelope, use the method appropriate to your version of Windows.

If you are using Windows 95:

  1. Open Windows Control Panel, double-click the Printers icon.

  2. Right-click the printer icon, and then click Properties.

  3. Click the Paper tab, set the number of copies you want to print, click Apply, and click OK.

  4. In Word, on the Tools menu, click "Envelopes and Labels."

  5. Click the Envelopes tab, type the delivery and return addresses, and click "Add to Document."

  6. On the File menu, click Print, and then click OK.

If you are using Windows 3.1 or Windows for Workgroups 3.11

  1. Open Windows Control Panel, double-click the Printers icon.

  2. Select your printer, click the Setup button, specify the number of copies you want to print and click OK.

  3. Click Close.

  4. In Word, on the Tools menu, click "Envelopes and Labels."

  5. Click the Envelopes tab, type the delivery and return addresses, and click "Add to Document."

  6. On the File menu, click Print, and click OK.
