GP Fault Starting Equation Editor Under Windows NT

 Last reviewed: February 5, 1998
Article ID: Q108690
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Equation Editor, versions 1.0, 2.0
  • Microsoft Windows NT operating system version 3.1

SYMPTOMS

If you start Microsoft Equation Editor from Word for Windows running under Windows NT, a general protection (GP) fault error may occur in EQNEDIT.EXE. (To start Equation Editor in Word, choose Object from the Insert menu, select Microsoft Equation 1.0 or Microsoft Equation 2.0 in the Object Type box, and then choose OK.)

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Equation Editor version 2.0 is supplied with Word 6.0. Microsoft Equation Editor version 1.0 is supplied with Word 2.x. If Word 2.x is not installed on your computer, the Microsoft Equation 1.0 selection may not appear in the Insert Object dialog box.
