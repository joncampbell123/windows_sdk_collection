WinWord: No Confirmation When Defining a New Style by Example

 Last reviewed: February 6, 1998
Article ID: Q80074
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In the Microsoft Word for Windows versions listed above, when you define a new style based on a selected text, Word for Windows does not display the following message: 

   Define style (name) based on selection?
Word for Windows version 1.x does display the above message.

MORE INFORMATION

This prompt was eliminated by design in Word for Windows version 2.0. You ARE prompted to confirm redefinition of a style based on a selection, just as in version 1.x.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, pages 191-192, 198
