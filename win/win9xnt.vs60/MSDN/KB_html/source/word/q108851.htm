WinWord 6.0: Contents of Disk #9 (5.25/1.2 MB)

 Last reviewed: July 30, 1997
Article ID: Q108851
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SUMMARY

This article contains the directory of the 5.25-inch, high-density (1.2 MB) Disk #9 (disk 9 of 11) for version 6.0 of Word for Windows. The article includes the decompressed filename for each file, which you must provide when you use the DECOMP.EXE utility to manually decompress a file.
