WD: "Cannot Sort Within Bookmark" Error in MacWord When Sorting

 Last reviewed: November 17, 1997
Article ID: Q111351
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0 and 5.1

SYMPTOMS

If you sort text in a Microsoft Word for the Macintosh version 5.x document, you may receive the error: 

   Cannot Sort Inside a Bookmark
This error occurs when you sort text within a publisher, subscriber, link source, or link destination.

WORKAROUND

To sort text within a link source or publisher, follow these steps:

  1. Select the text to sort within the link source or publisher.

  2. Copy the text and paste it into a new document. If the text is a publisher, Word displays the error "There is another Publisher open to the Edition...". Click OK to close the message box.

  3. Unlink the link source or publisher. 

        - If the text is a publisher, click Publisher Options on the Edit
      menu and select Cancel Publisher. Click Yes when asked to cancel
      the publisher.


        - If the text is a source link, click Link Options on the Edit menu 
      and then select Unlink.

  4. Sort the copied text.

  5. Copy and paste the sorted text back over the original text copied from the link source or publisher.

To sort text within a link destination or subscriber, the link must be broken. To break the link, select the destination link or subscriber.
  • If the text is a subscriber, click Subscriber Options on the Edit menu and select Cancel Subscriber. Click Yes when asked to cancel the subscriber.
  • If the text is a destination link, click Link Options on the Edit menu and select Cancel Link.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For additional information on linking and using publish and subscribe, see pages 669-684 of "Microsoft Word User's Guide," version 5.0/5.1.

For additional information about sorting, see pages 286-289 of "Microsoft Word User's Guide," version 5.0/5.1.
