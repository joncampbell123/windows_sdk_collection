WD95: How to Reset Word 7.0 MRU Lists

 Last reviewed: February 5, 1998
Article ID: Q161630
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

IMPORTANT: This article contains information about editing the registry. Before you edit the registry, you should first make a backup copy of the registry files (System.dat and User.dat). Both are hidden files in the Windows folder.

SUMMARY

There are a number of Word dialog boxes that contain Most Recently Used (MRU) lists. The purpose of this article is to describe how to reset these lists. The MRU lists described below are accessed from the following Word dialog boxes: 

   Add Template             Insert Subdocument

   Answer Wizard            Merge Revisions

   Attach Template          Modify Location

   Browse                   Open

   Change Source            Open Data Source

   Insert File              Save As

   Insert Picture

MORE INFORMATION

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows 95. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.

For information about how to edit the registry, view the Changing Keys And Values online Help topic in Registry Editor (Regedit.exe). Note that you should make a backup copy of the registry files (System.dat and User.dat) before you edit the registry.

The information contained in the following MRU Lists cannot be reset using menu selections. The entries must be reset by modifying the Windows Registry.

Add Template Dialog Box

To locate this dialog box, click Templates on the File menu, and then click the Add button. This dialog box contains two MRU lists: File Name and Text Or Property.

To reset the File Name MRU list, delete the following registry key: 

   My Computer\HKEY_CURRENT_USER\Software\Microsoft\Open Find 95\
   Microsoft Word\Settings\Add Template\File Name MRU
To reset the Text Or Property MRU list, delete the following registry key: 

   My Computer\HKEY_CURRENT_USER\Software\Microsoft\Open Find 95\
   Microsoft Word\Settings\Add Template\Any Text MRU

Answer Wizard

To locate this dialog box, click Answer Wizard on the Help menu, and then click the Answer Wizard tab. This dialog box contains only the "Type your request, and then click Search" MRU list.

To reset the Answer Wizard MRU list, delete the following registry key: 

   My Computer\HKEY_CURRENT_USER\SOFTWARE\MICROSOFT\MICROSOFT OFFICE\95\
   IntelliSearch\Word 95
