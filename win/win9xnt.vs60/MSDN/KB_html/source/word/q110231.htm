WD: Heading Numbering Changes When a Cross-Reference Is Inserted

 Last reviewed: February 5, 1998
Article ID: Q110231
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

After you insert a cross-reference to a heading number in a Word document, only the last number of the heading number is displayed.

CAUSE

The Heading Numbering format has been modified.

When you apply the Heading Numbering format, the cross-reference for the heading number should display the same number as the heading that is being cross-referenced.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. This problem was corrected in Microsoft Word 97 for Windows.
