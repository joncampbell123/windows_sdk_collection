WD6X: Cannot Open List of Supplied Graphics Filters in Help

 Last reviewed: February 5, 1998
Article ID: Q122377
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

Case 1

In Word for the Macintosh, when you select the "Graphics Filters" jump topic in the "Importing an entire graphics file" Help topic, the following error message is displayed: 

   Microsoft Help - Cannot open Help file.
To open the "Importing an entire graphics file" Help topic, do the following:

  1. Double-click the Help button on the Standard toolbar and type "graphics filters" (without the quotation marks).

  2. Choose the Show Topics button.

  3. Select the "Importing an entire graphics file" topic and then choose the Go To button.

The "Graphics Filters" jump topic that generates the above error message is located in the second paragraph.

Case 2

The same error occurs if you select the "Information on Specific Filters" jump topic in the Word ReadMe Help "Graphics Filters" topic.

To open the "Graphics Filters" Help topic and display the "Information on Specific Filters" jump topic, do the following:

  1. Double-click the Help button on the Standard toolbar and type "readme" (without the quotation marks).

  2. Choose the Show Topics button, and then choose the Go To button.

  3. Select the "Graphics Filters" topic.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for the Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The supported graphics formats in Word 6.0 are:

  • Encapsulated Postscript Files (EPSF)
  • Tag Image File Format (TIFF)
  • Macintosh PICT files (PICT and PICT2)
  • Paint program bitmaps (PNTG)
  • Windows bitmaps (BMP)
  • Windows metafiles (WMF)
  • WordPerfect graphics files (WPG)
