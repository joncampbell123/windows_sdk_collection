Internet Assistant Saves Duplicate META Elements

 Last reviewed: February 5, 1998
Article ID: Q126248
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 1.0, 1.0z
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

When you save your HTML document, the META elements Build and Generator are duplicated in the markup. When you choose Summary Information from the File menu, this information will display multiple times until it is truncated because of the limitation on the number of characters in the scroll box.

Version information kept in the META Name element does not advance to a higher version number unless the document was converted from a Word document with a higher version number.

STATUS

Microsoft has confirmed this to be a problem in Internet Assistant for Word for Windows versions 1.0, and 1.0z. This problem was corrected in Internet Assistant version 2.0 for Word for Windows 95.

RESOLUTION

Open the HTML document as Text Only, delete the extra META elements and correct the version number.

MORE INFORMATION

The META elements appear within the Head element to specify information about the document, which can then be used by servers and clients for identification, indexing, and cataloging document information.
