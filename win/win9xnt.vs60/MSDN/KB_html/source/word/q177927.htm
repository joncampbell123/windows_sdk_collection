WD98: Proofing Tools Not on Menus and are Not Available

 Last reviewed: February 17, 1998
Article ID: Q177927
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you try to use one of the proofing tools within Microsoft Word, such as Spelling, Grammar, Thesaurus, or Hyphenation, the choices are not listed on the menus. If either Spelling or Grammar is not available on the menus, then the preferences for these files (click Preferences on the Tools menu) will be unavailable (dimmed).

CAUSE

The proofing tools are either not installed or their files have been moved from the following default location: 

   MACINTOSH HD:Microsoft Office 98:Shared Applications:Proofing Tools

WORKAROUND

Determine if the following proofing tool files were loaded during Microsoft Word Setup: 

   MS Spelling 2
   MS Grammar
   MS Thesaurus
   Hyphenation
To determine if the files were installed, follow these steps:

  1. Quit Microsoft Word.

  2. In the Finder, on the File menu, click Find.

  3. Set Find Items to your Macintosh HD.

  4. In the box beside Name Contains, type one of the above items, such as MS Spelling 2, and then click Find.

  5. On the results screen, under Name, look for the file with the name MS Spelling 2, select it, and identify its location.

    If the file is not already in the " MACINTOSH HD:Microsoft Office 98:Shared Applications:Proofing Tools" folder, move it so that it is.

    NOTE: This is the default folder for the installation of the Proofing Tools. If you installed them to another folder, move the file to the folder you specified when you ran the Setup program.

  6. Repeat steps 4 and 5 for each proofing tool.

If you cannot find the files, or if following these steps does not correct the problem, then reinstall Word.

Once the items appear on the menu, check their preferences to make sure they are turned on. To do this, follow these steps:

  1. On the Tools menu, click Preferences, and then click the Spelling And Grammar tab.

  2. Under Spelling, click to select the "Check spelling as you type" check box.

  3. Under Grammar, click to select the "Check grammar as you type" check box.

  4. Click OK.

MORE INFORMATION

Spelling and Grammar are proofing tool items found on the Tools menu; Thesaurus and Hyphenation are proofing tool items found by pointing to Language on the Tools menu.

If only the MS Spelling 2 file has not been installed or if it has been moved from its default location, then clicking the Spelling And Grammar button on the Standard toolbar will give you the following message: 

   The grammar check is complete.
Clicking OK closes the Spelling or Grammar dialog box. There is no indication that the spelling check was not completed.

If only the MS Grammar file has not been installed or if it has been moved from its default location, then clicking the Spelling And Grammar button on the Standard Toolbar will give you the following message: 

   The spelling check is complete.
Clicking OK closes the Spelling or Grammar dialog box. There is no indication that the grammar check was not completed.

The following proofing tools files are installed during a complete installation of Microsoft Word. By default, the files are installed to the following location: 

   MACINTOSH HD:Microsoft Office 98:Shared Applications:Proofing Tools:
The proofing tool files are installed: 

   Australian Dictionary
   Australian Hyphenation
   English (US) Word Forms
   English Dictionary
   English Hyphenation
   Hyphenation
   MS English Grammar Dictionary
   MS Grammar
   MS Spelling 2
   MS Thesaurus
   UK English Thesaurus
   US English Thesaurus
For additional information about proofing tools, please see the following article in the Microsoft Knowledge Base:

   TITLE     : WD: Some Menu Commands Unavailable (Document Protection)
