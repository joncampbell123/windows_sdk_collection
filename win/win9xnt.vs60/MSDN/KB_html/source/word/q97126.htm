General Information About WOPR Version 2.0

 Last reviewed: July 30, 1997
Article ID: Q97126
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

Woody's Office POWER Pack version 2.0 (WOPR) is an add-on package for Word for Windows. This package includes the following features:

  • A toolbar button editor called TBEdit
  • A macro to create Bar codes (Postnet) on envelopes
  • A macro to print duplex booklets (front and back)
  • A macro to print four pages on one duplexed page
  • A macro to close all open documents using one menu command
  • A macro to delete a file from within Word for Windows
  • A macro to view and print listings of filenames in different directories
  • A digital clock and optional date display on menu bar
  • A macro to count the number of words in a document or in selected text only
  • A macro library of helpful macro functions (includes WordBasic source code) for WordBasic programmers
