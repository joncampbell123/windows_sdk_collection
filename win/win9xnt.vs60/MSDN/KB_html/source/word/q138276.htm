WD: Margins Not Retained in Last Section of a Master Document

 Last reviewed: February 5, 1998
Article ID: Q138276
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows, versions 6.0, 6.0a and 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SYMPTOMS

In the last subdocument of a master document, the margins reflect Word's default margin settings, rather than the margins of the subdocument.

RESOLUTION

To resolve this problem, upgrade to the latest version of Microsoft Word.

To work around this problem, create an extra section at the end of the document so that the incorrect margin settings apply only to the extra section and don't affect the last subdocument. To do this, follow these steps:

  1. Place the insertion point at the end of the master document.

  2. On the Insert menu, click Break.

  3. In the Section Breaks section, select Next Page. Click OK.

    NOTE: Choosing Continuous for the last section of the document won't work; the last section of the document is always Next Page and will be set that way regardless of which section break type you choose. If you choose anything other than Next Page, Word resets the section break type the next time you open the document.

  4. Reset the margins of your document.

The next time you open your document, the margins of the document will be correct, and you will have an extra page at the end of the file. This extra page is necessary to correct the margin problem.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. This problem was corrected in Microsoft Word 97 for Windows and Word 98 Macintosh Edition.

MORE INFORMATION

With the Master Document feature of Word, the page margin settings can be changed for the entire document; use the Page Setup command from the File menu. If the page margins are applied to the Whole Document they are reflected that way in the current session of the Word document.

The default page margins are: 

   Top          1"
   Bottom       1"
   Left         1.25"
   Right        1.25"
