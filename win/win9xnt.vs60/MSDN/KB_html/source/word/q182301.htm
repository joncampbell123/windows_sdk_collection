DMM: General Information on Direct Mail Manager

 Last reviewed: March 16, 1998
Article ID: Q182301
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows
  • Microsoft Office Small Business Pack for Windows 95, version 2.0

SUMMARY

The purpose of this article is to describe what Direct Mail Manager (DMM) is, how to install it, and what files are installed during setup.

MORE INFORMATION

Direct Mail Manager is a supplemental product available on the Microsoft Office Small Business Edition version 2.0 CD-ROM.

Direct Mail Manager is a program which interacts with Word for Windows 97 to help you create merged documents. Direct Mail Manager can verify your address list (data source) for possible errors by connecting to an internet site (ZIP-Station) and it can also supply Word with templates and wizards used to create mail merge main documents.

How to Install Direct Mail Manager

To install Direct Mail Manager, follow these steps:

  1. Place disc 1 of the Small Business Edition into your CD-ROM drive.

  2. In the ValuPack folder, double-click the DMM folder.

  3. Double-click Setup.exe.

NOTE: By default, Direct Mail manager is installed to the following location: 

   C:\Program Files\Microsoft Office\Office\Direct Mail Manager
