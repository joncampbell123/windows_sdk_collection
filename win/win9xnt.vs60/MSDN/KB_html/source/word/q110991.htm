The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows operating system version 3.1

SUMMARY

This article is a list of Microsoft Knowledge Base articles and Application Notes (AppNotes) about Word version 6.0 for Windows that you can order from Microsoft FastTips. If you are using an online service, you can find these articles by querying on the article ID number or on words in the title and text.

How to Order Knowledge Base Offerings Through Microsoft FastTips

Microsoft FastTips is available 24 hours a day, 7 days a week, from a touch-tone telephone. To order items from this list, first select the items you want to receive, noting the four- to six-digit Item ID number for each item, and then:

  • Dial the toll-free FastTips number: (800) 936-4100.
  • When prompted, select the Microsoft product you are calling about.
  • Press 1 on your phone keypad to select Express Order Service.
  • When prompted, select your preferred delivery method, fax or mail.
  • When prompted, enter your three-digit area code and seven-digit fax number on your phone keypad, or leave your name and mailing address in the voice mailbox.
  • When prompted, enter the number of the Item ID and then press #, for up to five items (do not enter the letters in the Item ID).
  • When finished, simply hang up. If you have problems receiving a fax, please call (425) 635-3105.
