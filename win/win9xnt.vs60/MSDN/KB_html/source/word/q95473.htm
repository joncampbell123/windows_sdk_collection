  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SUMMARY

Foreign Language Fonts are available from Linguist's Software for use with all Microsoft Windows 95, Windows 3.1, and Macintosh applications that use fonts.

MORE INFORMATION

Linguist's Software produces professional-quality, hinted, scaleable foreign font packages in both TrueType and Type-1 format.

The foreign language font package also includes a Keyboard Switcher utility providing four characters per key in Windows (and the EuroScript utility providing four characters per key in Macintosh applications), a User's Manual, keyboard layout charts, ANSI Tables, and keycap sticker sheets.

System Requirements

The system requirements for running Linguist's Software are as follows:

IBM Compatible Systems:

  • MS-DOS or PC-DOS version 3.1 or later
  • Windows 3.1 or Windows 95
  • An IBM PC/AT or 100-percent compatible computer with an Intel 386 or greater processor
  • A Windows-compatible graphics card
  • A Windows-supported dot-matrix, ink jet, or laser printer

Macintosh Systems:
  • Macintosh System 6.0.7 or newer
  • Macintosh SE-30 or newer
  • A Macintosh-supported dot-matrix, ink jet, or laser printer
