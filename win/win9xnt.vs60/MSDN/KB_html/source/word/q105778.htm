Macro Recorder Turns Off After Tools Protect Document

 Last reviewed: July 30, 1997
Article ID: Q105778
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you turn on the macro recorder and then protect your document for forms using the following steps, Word for Windows automatically stops the macro recorder:

  1. From the Tools menu, choose Macro.

  2. Type a name in the Macro Name box and then choose the Record button.

  3. Choose OK in the Record Macro dialog box.

  4. From the Tools menu, choose Protect Document.

  5. Select the Forms option and then choose OK. Notice that the Macro Record toolbar disappears, and REC on the status bar turns off.

NOTE: Word does not turn off the macro recorder if you protect your document for revisions or annotations.

CAUSE

Word turns off the macro recorder because in a document protected for forms, the Macro command on the Tools menu is unavailable. No message appears to notify you that the macro recorder is no longer running.
