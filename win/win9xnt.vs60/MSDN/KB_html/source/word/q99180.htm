HP LaserJet II Printing Problems Corrected by Windows 3.1

 Last reviewed: July 30, 1997
Article ID: Q99180
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

This article lists the Hewlett-Packard (HP) LaserJet Series II printing problems corrected by the drivers included with Windows 3.1.

The problems listed below exist with Windows 3.0 printer drivers. To use the corrected Windows 3.1 printer drivers, you must upgrade to Windows 3.1. These drivers cannot be used with Windows 3.0.

MORE INFORMATION

  1. Letter Gothic Font May Be Truncated with ProCollection Cartridge ----------------------------------------------------------------

    Text prints truncated on the right when all three of the following circumstances exist: 

        - You are using the ProCollection font cartridge.


        - You are using the Letter Gothic font.


        - You are using version 3.0 of the Windows Hewlett-Packard (HP)
      LaserJet II or III printer drivers.

    If the text extends past the right margin indicator on the ruler in page or page layout view, the text will be truncated in print preview and on printouts.

  2. Scaled Picture Prints Cropped Instead -------------------------------------

    If you scale a picture in Word and print it to an HP LaserJet II, the picture will look correct in page view and print preview, but it may print cropped instead of scaled. In other words, only a portion of the picture prints, and it prints at the original size.

  3. Vector Fonts Don't Print in Landscape -------------------------------------

    Word may not print text that is formatted for the Windows vector fonts (Modern, Roman, and Script). This problem occurs on the HP LaserJet Series II printer when you use either version 3.3 or 3.4 of the HPPCL.DRV printer driver.

    This problem only occurs when all three of the following conditions exist: 

        - You are printing in landscape mode.


        - You select the Custom Size option in the Page Setup dialog box.


        - You use any Windows 3.0 vector font, in any size.

  4. Using 16 or More Soft Fonts Per Page Causes Error Message ---------------------------------------------------------

    You may receive the following error message when you print from Word 2.0 under Windows 3.0: 

          Soft Font Page Limit: Some fonts will be substituted

    This problem only occurs when all three of the following conditions exist: 

        - You are printing to a Hewlett-Packard (HP) LaserJet Series II
      printer.


        - You are printing multiple copies of a document.


        - The number of fonts on each page exceeds 16.

    This error message appears because 16 soft fonts is the physical limit on an HP LaserJet Series II. This error message should not appear if you print multiple copies of a document that contains fewer than 16 soft fonts per page.

    Workarounds -----------

    To work around this problem, use one of the following methods:

    a. Between print jobs, choose Print Setup from the File menu, 

          select the PCL/HP LaserJet driver again, and choose the OK
      button.

    -or-

    b. From the File menu, choose Print Setup, then choose the Setup 

          button. In the Printer box, select HP LaserJet IID, and choose
      the OK button twice to return to the document.
