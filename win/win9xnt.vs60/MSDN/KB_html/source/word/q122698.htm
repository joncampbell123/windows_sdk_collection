WD: Reverse Print Order Doesn't Work on HP DeskWriter in MacWord

 Last reviewed: February 5, 1998
Article ID: Q122698
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

Microsoft Word for the Macintosh may not print in reverse order on the Hewlett-Packard (HP) DeskWriter when the Reverse Print Order option is selected. (To locate the Reverse Print Order option, click Options on the Tools menu [in Word 98 click Preferences], and then click the Print tab.)

CAUSE

This problem occurs when Print Back To Front is also selected in the HP DeskWriter printer driver setup dialog box. That is, the Reverse Print Order option in Word turns on reverse printing, and then the HP DeskWriter Print Back To Front option turns it off again.

RESOLUTION

To print from Word in reverse order on an HP DeskWriter, select either the Print Back To Front option for the HP DeskWriter, or the Reverse Print Order option in Word, but do not select both at the same time.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for the Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
