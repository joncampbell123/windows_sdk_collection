Word 5.0 for the Macintosh IVR Fax: Printing

 Last reviewed: July 30, 1997
Article ID: Q84109
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

              Microsoft Word for the Macintosh, Version 5.0
               Automated Product Support Service Fax-Script
                      Questions and Answers: Printing
               --------------------------------------------

1. Q. How do I print envelopes without a return address on the
      LaserWriter II NT or II NTX?

   A. To print an envelope without a return address, follow these seven
      steps:

      a. From the File menu, choose Page Setup.
      b. Change Orientation to Wide, and select U.S. Letter for the
         paper size.
      c. Click the Document button and change the margins to the
         following:

            The top margin should be: 4.25 inches.
            The bottom margin should be: 2.25 inches.
            The left margin should be: 4.5 inches.
            The right margin should be: 2.0 inches.

      d. Type the mailing address.
      e. From the File menu, choose Print and select the manual feed
         option.
      f. Place the envelope face up with the flap facing the left side
         of the printer.
      g. Click the Print button.

      For more information on printing envelopes, please refer to pages
      103 through 108 of the "User's Guide to Microsoft Word."

2. Q. How do I print envelopes with a return address on the LaserWriter
      II NT or II NTX?

   A. To print an envelope with a return address, follow these 10 steps:

      a. From the File menu, choose Page Setup.
      b. Change Orientation to Wide, and select U.S. Letter for the paper
         size.
      c. Click the Document button and change the margins to the
         following:

            The top margin should be: 2.5 inches.
            The bottom margin should be: 2.25 inches.
            The left margin should be: 0.5 inches.
            The right margin should be: 2.0 inches.

      d. Type the return address.
      e. Press the RETURN key until you have a total of 12 returns,
         including the return address.
      f. Display the ruler and change the left indent to 4.5 on the
         ruler.
      g. Type the mailing address.
      h. From the File menu, choose Print and select the manual feed
         option.
      i. Place the envelope face up with the flap facing the left side
         of the printer.
      j. Click the Print button.

      For more information on printing envelopes, please refer to pages
      103 through 108 of the "User's Guide to Microsoft Word."

5. Q. How do I print envelopes without a return address on the Hewlett-
      Packard DeskWriter?

   A. To print an envelope without a return address, follow these seven
      steps:

      a. From the File menu, choose Page Setup.
      b. Select Envelope for the paper size.
      c. Click the Document button and change the margins to the
         following:

            The top margin should be: 2.5 inches.
            The bottom margin should be: 0.5 inches.
            The left margin should be: 4.5 inches.
            The right margin should be: 2.0 inches.

      d. Type the mailing address.
      e. Place the envelope in the printer face down and slide it
         forward until it stops against the print wheels.
      f. From the File menu, choose Print.
      g. Click the Print button.

      For more information on printing envelopes, please refer to pages
      103 through 108 in the "User's Guide to Microsoft Word."
