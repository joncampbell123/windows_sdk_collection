WD: Table Heading Rows Do Not Appear on Following Pages

 Last reviewed: February 18, 1998
Article ID: Q176855
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

If you have a table that extends over more than one page, and you specify several rows of the table as heading rows, none of them will appear on subsequent pages.

CAUSE

This problem will occur if you do any of the following:

  • Select more than one page of rows as heading rows.
  • Copy only the rows designated as heading rows to another part of the document or to a new document, and then make the table more than one page.
  • Create a manual page break or any type of section break within the table. (This will split the table into two different tables. This functionality is by design of Word.)

WORKAROUND

Method 1: If You Selected More Than One Page of Rows as Heading Rows

Make sure that no more than one page of contiguous rows are designated as heading rows. To do this, follow these steps:

  1. Select the entire table by clicking Select Table on the Table menu.

  2. On the Table menu, be sure Headings is cleared (so that it does not have a check mark beside it). If Headings has a check beside it, click it to remove the check mark.

  3. Select only the desired heading rows, and on the Table menu, click Headings.

Method 2: If you Copied the Heading Rows to a Different Location

When you copy the table headings, include at least one extra row beneath the rows designated as headings. Once you do this, you may need to follow the steps in method 1 to completely restore the table heading functionality.

Method 3: If Your Table Contains Manual Page Breaks or Section Breaks

You will need to look for manual page breaks and section breaks that may be splitting your table and delete them.

To find and delete a manual page break within the table, follow these steps:

  1. On the Edit menu, click Replace.

  2. If you see More, click it and then place your insertion point into the Find What box. Select and delete any entry that may be in that box.

  3. Click Special and click Manual Page Break.

    NOTE: The characters "^m" (without the quotation marks) will be placed in the Find What box.

  4. Click Find Next until you find a page break splitting your table. If you find one, click Replace. Click OK on the message that "Word has finished searching the document."

    NOTE: DO NOT click Replace All.

To find and delete Section breaks within the table, do the following:

  1. On the Edit menu, click Replace.

  2. If you see More, click it and then place the insertion point in the Find What box. Select and delete any entry that may be in that box.

  3. Click Special and click Section Break.

    NOTE: The text "^b" (without the quotation marks) will be placed in the Find What box.

  4. Click Find Next until you find a section break splitting your table. If you find one, click Replace. Click OK on the message that "Word has finished searching the document."

    NOTE: DO NOT click Replace All.

MORE INFORMATION

To be able to designate rows as heading rows in a table, you must select at least the first row of the table and any rows thereafter have to be contiguous. You cannot make non-contiguous rows heading rows. One you have highlighted the rows you want as headings to the table, on the Table menu, click Headings.
