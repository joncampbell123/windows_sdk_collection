WD5X: Can't Print Envelopes to the StyleWriter, StyleWriter II

 Last reviewed: October 3, 1997
Article ID: Q144374
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a

SYMPTOMS

When you print envelopes to the StyleWriter or StyleWriter II, you may have trouble using the Create Envelope feature. Specifically, when you print envelopes to a StyleWriter or StyleWriter II, the following problems may occur:

  • The printer feeds a blank envelope

    -and-

  • You receive the following error message: 

          Please Insert the next sheet of paper.

    -and-

  • If you feed another envelope through the printer, the printer's error light flashes.

CAUSE

When you use a StyleWriter, the envelopes must be fed in the portrait orientation; note that landscape is the predominant method for printing envelopes on most printers.

WORKAROUND

To print envelopes to the Stylewriter or StyleWriter II using the Create Envelope command in Word, follow these steps (note that you must feed the envelope twice):

  1. On the Apple menu, click Chooser.

  2. Select the StyleWriter or StyleWriter II printer icon. Turn off Background Printing and close the Chooser.

  3. In Word, click Create Envelope on the Tools menu.

  4. Type the Return and Delivery Addresses and click Print.

  5. In the Page Setup dialog box, select US Letter from the Page Size list. Select the portrait orientation icon. Select the Left option for Envelope Alignment. Select Size 10 in the Envelope Size list, and then click OK.

  6. In the Print dialog box, select Manual Feed from the Paper Source list.

  7. Insert an envelope in the manual feed slot in the back of the printer. The printer feeds the envelope through without printing, and Word displays the following message: 

          Please Insert the next sheet of paper. Click OK to Continue, or
      Cancel to Terminate.

  8. Insert the envelope face-up in the manual feed slot, and click OK in the message dialog box. When the printer's Error light blinks, press the Ready button on the printer. 

        The envelope should print correctly.


        NOTE: To stop the process click Cancel when you receive the following
    message:


           The paper is shorter than the size selected in Page Setup. Please
       Add the correct paper. Click OK to continue. Click Cancel to
       terminate.
