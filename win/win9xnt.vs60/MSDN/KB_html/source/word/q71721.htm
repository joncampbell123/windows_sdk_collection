Graphics/Pictures Not Printing from Word for Windows

 Last reviewed: July 30, 1997
Article ID: Q71721
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows, if graphics and pictures appear on the screen and in print preview but do not appear on printouts, verify that the printer is not in draft mode with the following steps:

  1. From the File menu, choose Print.

  2. Choose the Options button. Make sure the Draft Output check box (Draft check box in version 1.x) is not selected.

This problem can also occur if the printer driver is in draft mode. To change that setting, use the following steps:

  1. From the File menu, choose Printer Setup.

  2. Click the active printer.

  3. Choose the Setup button.

  4. If the printer has a Draft option, make sure it is not selected.

Graphics and pictures will not be printed from Word for Window if the Draft option is selected because this option places the printer in its native mode, which is not a graphical user interface (GUI).

REFERENCES

"Microsoft Word for Windows User's Reference," version 1.x, pages 277-278
