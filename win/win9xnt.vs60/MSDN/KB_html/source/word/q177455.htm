WD97: Problems with Document Based on Contemporary Memo Template

 Last reviewed: December 2, 1997
Article ID: Q177455
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you are working on a Microsoft Word document based on the Contemporary Memo template, the following things can occur:

  • The floating picture within the template disappears. 

        -or-
  • When you run the Undo Update Fields command, Microsoft Word stops responding.

CAUSE

This problem occurs when the Automatically Update Document Styles option has been turned on in the Templates And Add-ins dialog box.

WORKAROUND

To work around this problem, do not activate the setting to Automatically Update Document Styles for this document. You can find this setting by clicking Templates And Add-Ins on the Tools menu.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
