WD: ASK and SET Return No Error with Invalid Bookmark

 Last reviewed: February 2, 1998
Article ID: Q48920
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition
  • Microsoft Word 97 for Windows

SUMMARY

ASK and SET fields are not result fields, so if an invalid bookmark name or some other error causes the field to be improperly evaluated, the field will not display an error.

MORE INFORMATION

Errors are normally displayed as the last results of the field; however, because no result is displayed, no indication of the error is given.

An error can usually be detected in the field that uses the bookmark name that is defined in the SET or ASK statement. If the ASK or SET field has an error, the result field that references the bookmark will indicate that the bookmark is not defined.
