WD: International Characters Displayed as Squares

 Last reviewed: February 18, 1998
Article ID: Q178337
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you type international characters, such as any of the following 

   E, I, O, U        with an accent grave over the letter

   A, I, O, U        with an accent acute over the letter

   A, E, I, O, U     with a caret over the letter

   E, I              with a colon over the letter

   or an inverted exclamation mark
Word may show a square instead of the desired character.

NOTE: This problem occurs when you type text in any of the following locations:

  • An Address Book entry.
  • In an AutoCorrect entry.
  • On the Envelopes tab.
  • On the Envelopes tab (Word 6.x for the Macintosh only).
  • In the Fax, Letter, Memo, Newsletter and Resume wizards.
  • In the Fax, Letter, Mailing Label, Memo, Newsletter and Resume wizards.

CAUSE

Word uses the Geneva font to display the characters in a dialog box or wizard. The Geneva font does not contain these characters.

NOTE: The Geneva font is installed by the Macintosh Operating System. International versions of the Macintosh Operating System install a Geneva font that contains these characters.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Address Book

This problem occurs in Word 98 Macintosh Edition only. When you type in the address book to add a new, or change an existing contact, if you type any of the international characters listed in the Symptoms section of this article, Word will show a square instead of the desired character.

NOTE: The address book is available in the following wizards (click New on the File menu):

  • Letter Wizard (also available on the Tools menu)
  • Fax Wizard
  • Envelope Wizard
  • Mailing Label Wizard

NOTE: The address book is not available in Word 6.x for the Macintosh.

AutoCorrect

When you click AutoCorrect on the Tools menu, if you type any of the international characters listed in the "Symptoms" section of this article, Word will show a square instead of the desired character.

NOTE: When you click Add to add the AutoCorrect entry to the AutoCorrect List, Word will show the international character as a square. However, when an AutoCorrect entry is applied to a document, the desired character is inserted and shown correctly.

For more information about AutoCorrect, click the Office Assistant, type "AutoCorrect," click Search, and then click to view "Automatically correct text."

NOTE: If the Assistant is hidden, click the Office Assistant button on the Standard toolbar. If Microsoft Help is not installed on your computer, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q120802
   TITLE     : Office: How to Add/Remove a Single Office
               Program or Component

Envelopes Tab

When you click Envelopes And Labels on the Tools menu, if you type any of the international characters listed in the "Symptoms" section of this article on the Envelopes tab, Word will show a square instead of the desired character.

This problem occurs in Word 6.x only. When you click Envelopes and Labels on the Tools menu, if you type any of the international characters listed in the Symptoms section of this article on the Envelopes tab, Word will show a square instead of the desired character.

NOTE: This problem does not occur on the Labels tab.

Wizards

When you type any of the international characters in the "Symptoms" section of this article in the listed wizards, Word will show a square instead of the desired character.

To run a wizard, click New on the File menu, select the desired wizard, and then click OK. 

   Wizards That Exhibit
   This behavior             Location in wizard where problem occurs
   -----------------------------------------------------------------

   Fax Wizard                Letters & Faxes

   Letter Wizard             Letters & Faxes

   Mailing Label Wizard      Letters & Faxes

   Memo Wizard               Memos

   Newsletter Wizard         Other Documents

   Resume Wizard             Other Documents
