WD97: Templates and Wizards Not Included with SR-1 Patch

 Last reviewed: February 5, 1998
Article ID: Q172492
The information in this article applies to:
  • Microsoft Word 97 for Windows, Service Release 1 (SR-1)

SUMMARY

The following updated template files are included only with the Microsoft Office 97 Service Release 1 (SR-1) Enterprise Update or the Microsoft Word Service Release 1 (SR-1) Enterprise Update: 

   Pleading Wizard.wiz
   Fax Wizard.wiz
   Support8.dot
   Macros8.dot
Because these files are not available with the SR-1 Patch they are available for download from the Microsoft Software Library: 

 ~ Wdtlupd.exe (size: 664800 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

The following table lists the updated template files that are included in the Microsoft Office 97 Service Release 1 (SR-1) Enterprise Update. Because these versions of the templates are not available with the downloadable version of SR-1, Microsoft has made available a separate downloadable file (Wdtlupd.exe) that contains these templates. 

   Template             Default Location
   ---------------------------------------------------------------------

   Support8.dot         \Program Files\Microsoft Office\Macros
   Macros8.dot          \Program Files\Microsoft Office\Macros
   Pleading Wizard.wiz  \Program Files\Microsoft Office\Templates
                           \Legal Pleadings
   Fax Wizard.wiz       \Program Files\Microsoft Office\Templates
                           \Letters & Faxes
If you downloaded the SR-1 Patch from the World Wide Web, you may also want to download Wdtlupd.exe to obtain these template files. If you obtained the SR-1 CD, these files are located in the Templates or Macros folder on the CD.

To Use the Updated Templates

This procedure assumes that you have already downloaded and extracted the contents of Wdtlupd.exe.

  1. Locate each the existing templates on your hard disk and rename them.

    For example, locate the existing Pleading Wizard.wiz file, right-click it, and name it "Pleading Wizard.bak" (without the quotation marks).

  2. From the directory to which you downloaded and extracted Wdtlupd.exe, copy each updated template to the appropriate location. The default locations for these templates are listed earlier in this Readme.

    For example, copy the Pleading Wizard.wiz template to the \Program Files\Microsoft Office\Templates\Legal Pleadings folder.

For more information about renaming or copying files, see Windows online Help.
