WD: Bold Text in MacWord Address/Create Envelope Dialog Box

 Last reviewed: February 5, 1998
Article ID: Q91007
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.1, 6.0

SYMPTOMS

When you type characters in the Create Envelope and Addresses dialog boxes, characters may unexpectedly appear to be formatted in bold. However, the characters appear bold only on the screen; they appear correctly in the printout.

RESOLUTION

To prevent this problem from occurring, follow these steps:

  1. From the File menu, choose Page Setup.

  2. In the Page Setup dialog box, clear the Fractional Widths check box.

To work around this problem, remove the bold formatting. Follow the steps below to force Word to redraw the active dialog box:

  1. Click the title bar of the dialog box and drag it so that only the top of the dialog box is visible at the bottom of the screen. Release the mouse button.

  2. Drag the dialog box back to the center of the screen.

STATUS

Microsoft has confirmed this to be a problem in Word versions 5.1 and 6.0. We are researching this problem and will post new information here as it becomes available.

MORE INFORMATION

A similar bold appearance may occur if the characters in the Create Envelope or the Addresses dialog box are formatted in a small point size. This is unrelated to the above problem and is a result of the way the small characters are drawn on screen. In this case, forcing Word to redraw the dialog box will not remove the bold appearance. However, the characters will appear correctly in the printout.
