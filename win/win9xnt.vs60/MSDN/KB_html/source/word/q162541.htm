WD97: Can't Access Mail Merge Data Source on Banyan Network

 Last reviewed: February 5, 1998
Article ID: Q162541
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

A mail merge data source can't be edited from and will not be attached to a mail merge main document if the data source is saved on a Banyan network.

WORKAROUND

Create and save your mail merge data source on a local hard disk or a non-Banyan network drive.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When a mail merge data source is created and saved to a Banyan network drive, the file appears to be saved correctly. After the file is saved, the mail merge feature will not prompt you to edit the data source and the data source will not be attached to the mail merge main document.

The data source was saved to the Banyan network drive and you are able to open the file as a regular document in Word. However, you can't attach the file as a data source and no error message is presented.
