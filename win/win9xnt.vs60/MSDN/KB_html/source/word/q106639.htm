WD: Replace Within Selection Continues to End of Document

 Last reviewed: February 2, 1998
Article ID: Q106639
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, version 6.0
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Microsoft Word, if you are performing a find-and-replace operation on a selection and choose Replace All after having chosen Replace or Find Next, Word continues replacing past the selection to the end of the document.

STATUS

Microsoft has confirmed this to be a problem in Word. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

If you select a range of text, and then in the Replace dialog box, click Replace All or any combination of Replace and Find Next, replacing only takes place within the selection. You then see a dialog box asking if you want to continue replacing beyond the selection.

If however, you select a range of text and then click Replace or Find Next followed by Replace All in the Replace dialog box, Word does not stop after the selection but replaces to the end of the document.
