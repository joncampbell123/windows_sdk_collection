List of Files and File Locations Installed by Mac Word 6.0

 Last reviewed: February 6, 1998
Article ID: Q119439
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

The following is a list of files installed by Word 6.0 for the Macintosh and the files' locations and whether they are installed in the Minimum (M), Typical (T) or Complete (C) installation.

MORE INFORMATION

NOTE: "Word folder" refers to the folder the user specified during setup. "Microsoft folder" refers to the folder named Microsoft, which is installed on the root level of the hard drive.
