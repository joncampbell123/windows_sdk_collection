WD: Envelopes Print Wrong or Blank on HP DeskJet Printers

 Last reviewed: February 5, 1998
Article ID: Q137156
The information in this article applies to:
  • Microsoft Word for Windows 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

Envelopes printed from Word on Hewlett-Packard (HP) DeskJet printers 540, 550, and 560 may print incorrectly or blank.

CAUSE

Case 1: The envelope feed options in Word are set incorrectly. (The 

        documentation for the Hewlett-Packard printer incorrectly
        instructs you to turn off Clockwise rotation.)
Case 2: You are running a Beta version of Windows 95, Word version 7.0 
        for Windows 95 or Word 97 for Windows.

RESOLUTION

Case 1: An updated DeskJet printer driver is available with the Windows 95 

        Service Pack 1 that corrects this problem. For information on
        obtaining the Windows 95 Service Pack 1, please see the following
        article in the Microsoft Knowledge Base:

        ARTICLE ID: Q142794
        TITLE:WD: Availability of Microsoft Windows 95 Service Pack 1
If you are unable to obtain the Windows 95 Service Pack 1, you may workaround the problem using the following steps: 

        a. On the Tools menu, click Envelopes And Labels.

        b. Click the Envelope tab.

        c. Click Options.

        d. Click Printing Options.

        e. Click the feed method that is third from the right (face
           down, landscape, right feed). Although this option seems to
           be rotated 180 degrees from what the printer shows, it will
           force Word to print the envelope correctly.

        f. Select the Clockwise Rotation check box.

        g. Click OK.

        h. Click Print.
Case 2: Obtain and install the release version of Windows 95 and Word 
        version 7.0 for Windows 95 or Word 97 for Windows.
