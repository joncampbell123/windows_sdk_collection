WD6X: Word for Power Mac Fulfillment Package Ships Only 7 Disks

 Last reviewed: February 6, 1998
Article ID: Q123726
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

The Word 6.0 for the Power Macintosh fulfillment disks (or "Upgrade") correctly include only 7 of the 16 disks needed for installation. By design, only Disk 1 and Disks 11 through 16 are included. These disks are intended to be used with your original Word 6.0 for the Macintosh disk set.

The purpose of this article is to avoid the confusion that may arise when you receive this seemingly odd disk set.

MORE INFORMATION

The following list shows which disks are supplied with each disk set: 

   Word Package                         Disks Included in Disk Set
   ------------                         --------------------------

   Word 6.0 for the Macintosh           Disks 1 through 10
   Word 6.0 for the Power Macintosh
      fulfillment disks ("Upgrade")     Disk 1, Disks 11 through 16
   Word 6.0 for the Macintosh
      and the Power Macintosh (also
      referred to as "Combined")        Disks 1 through 13
The Upgrade disks are designed to be used with your existing Word 6.0 for the Macintosh disks.

To Install the Upgrade

  1. Use Disk 1 from the Upgrade disk set.

  2. Use Disks 2 through 10 from the original Word 6.0 for the Macintosh 10- disk set.

  3. Use Disks 11 through 16 from the Upgrade disk set.

This procedure is completely outlined in a cover letter that is included in the box. The following is an excerpt from that cover letter: 

   NOTE: This is not the entire contents of the cover letter.
To update your Word workstation or server installation, you will need your original Word 6.0 for the Macintosh disks (numbered Disk 1 through Disk 10) in addition to the Power Macintosh disks in this package (a new Disk 1 and Disks 11-16).

To install Word 6.0 for the Power Macintosh:

  1. Run Microsoft Word Setup from Install Disk 1 of the Power Macintosh disks, labeled "Install Disk 1."

  2. Double click on the Microsoft Word Setup icon.

  3. Run "Remove All" to remove previously installed components of Word.

  4. When the previous components are removed, select the installation options you desire.

  5. When prompted for disk 2, insert disk 2 from the Word 6.0 for the Macintosh disk set you originally purchased, labeled "Disk 2."

  6. Continue inserting Macintosh disks from the original set until you are prompted for disk #11, and then insert disk 11 from the Power Macintosh disks, labeled "Power Macintosh Disk 11."

  7. Disks 12 through 16 will also come from the Power Macintosh disk set. After disk 16, installation will be complete.

You will notice that the Power Macintosh disk set contains disks 11, 12, 13, 14, 15, and 16. The original Macintosh disk set ended at disk #10. This is correct. The supplemental disks contain additional files needed only by the Power Macintosh.
