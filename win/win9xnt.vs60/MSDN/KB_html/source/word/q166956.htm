WD97: Many Page Breaks With "First Page Footer" Causes Problems

 Last reviewed: July 31, 1997
Article ID: Q166956
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you work with a Word 97 document that contains many page breaks, the following may occur:

  • When you add a "Different First Page" footer to your document, some of the footer text may bleed into the first page footer.

    -and-

  • When you save your document as "Rich Text Format (*.rtf)," the following error message will be displayed: 

          This program has caused an illegal operation and will be shut down.


          If the problem persists, contact the program vendor.

    When you click Details, the following message is displayed: 

          WINWORD caused an invalid page fault in module WINWORD.EXE at
      0137:30080f6c.

NOTE: This problem has occurred with as few as 10 page breaks.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
