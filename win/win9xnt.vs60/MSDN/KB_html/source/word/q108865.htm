Envelope Address Style Is Double-Spaced

 Last reviewed: February 5, 1998
Article ID: Q108865
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows operating system version 3.1

SYMPTOMS

When you use a letter document created in Word 2.x for Windows in Word 6.0 for Windows and attach an envelope to the document, the Envelope Address Style becomes double spaced.

CAUSE

Envelopes attached to documents that were originally created using any of the Word 2.x letter templates will show and print as double spaced in Word 6.0 for Windows. The Envelope Address style contained in these letters converts with the paragraph formatting of 6 pt Spacing Before and 6 pt Spacing After. Additionally, if the address contained new line characters at the end of each line as a Word 2.x document, the new line characters will be replaced with normal carriage returns.

RESOLUTION

To correct this problem:

  1. Open any document created using a letter template.

  2. From the Format menu, choose Style.

  3. From the List box, select All Styles.

  4. From the Styles list, select Envelope Address, and choose the Modify button.

  5. From the Format menu, choose Paragraph.

  6. Change the settings for Spacing Before and After to 0 pt, and choose OK. Select the Add To Template check box, then choose OK.

  7. From the File menu, choose Save All. If prompted, choose Yes to save the template to the <Winword>\TEMPLATE directory, overwriting the file with the Microsoft Word for Windows 6.0 format.

To automatically update each document created from the template, you must perform the following steps:

  1. Open the previously created letter document.

  2. From the File menu, choose Templates. Choose the Attach button.

  3. Select the template that the letter was created from (from the above listed procedure) and choose OK.

  4. Select the Automatically Update Document Styles check box and choose OK. This will ensure that when a document created from the template is opened, that the style is automatically changed to reflect the changes to Spacing Before and Spacing After. Envelopes will now be created with the proper Envelope Address style.
