WD: Word Menu Commands Unavailable If Locked for Annotations

 Last reviewed: January 30, 1998
Article ID: Q49867

The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SUMMARY

When your Microsoft Word document is locked for annotations, certain Word commands are not available.

NOTE: Unavailable commands appear grayed or dimmed on the Word menus. Keyboard commands that edit or format text and graphics are also unavailable.

LIST OF UNAVAILABLE COMMANDS

Word for Windows 7.0

   Menu           Unavailable Commands
   ----           --------------------

   File           Templates, Page Setup

   Edit           All commands except Go To

   View           Outline, Master Document, Header and Footer,
                  Footnotes, Annotations

   Insert         All commands except Hyperlink

   Format         All commands

   Tools          All commands except Unprotect Document, Options

   Table          All commands except Gridlines

Word for Windows 6.0, Word 6.0 for Windows NT

   Menu           Unavailable Commands
   ----           --------------------

   File           Page Setup

   Edit           Undo, Cut, Paste, Paste Special, Clear AutoText,
                  Bookmark, Links and Object

   View           Footnotes, and Annotations

   Insert         All commands except Annotation

   Format         All commands

   Tools          All commands except AutoCorrect, (Un)Protect Document
                  Revisions, Macro, Customize, and Options

   Table          All commands except Gridlines

Word for the Macintosh 6.0, 6.0.1, 6.0.1a

   Menu           Unavailable Commands
   ----           --------------------

   File           Document Layout, Page Setup

   Edit           All commands except Select All, Find, Replace, Go To,
                  Publishing

   View           Footnotes, Annotations

   Insert         All commands except Annotations

   Format         All commands

   Tools          All commands except AutoCorrect, Unprotect Document,
                  Revisions, Macro, Customize, Options

   Table          All commands except Gridlines

Word for Windows 1.x and 2.0

   Menu           Unavailable Commands
   ----           --------------------

   File           Lock For Annotations option in the Save As dialog box
                  Summary Info Update**
                  Template**
                  Print Merge

   Edit           Undo, Repeat, Cut, Paste, Paste Link, Paste Special**
                  Replace, Summary Info, Header/Footer, Glossary, Table
                  Links** and Object**

   View           Footnotes, Annotations and Header/Footer**

   Insert         All commands except Annotation

   Format         All commands

   Tools**        All commands except Record Macro, Macro, and
                  Options

   Utilities      Customize

   Macro          Assign to Menu and Assign to Key have Template
                  context unavailable

   Table**        All commands except Gridlines

   NOTE: The ** denotes that these commands are Word for Windows version
   2.x commands only.

MORE INFORMATION

To lock a Word document for comments or annotations, follow the procedure appropriate for your version of Word.

Word 7.0 for Windows

On the Tools menu, click Protect Document, click Annotations, and then click OK.

Word 6.0 for Windows or Macintosh

On the Tools menu, click Protect Document, click Annotations, and then click OK.

Word 2.0 for Windows

On the File menu, click Save As, click the File Sharing button, click the Lock for Annotation check box, and then click OK.

Word 1.x for Windows

On the File menu, click Save As, click the Options button, click to select the Lock For Annotations check box, and then click OK.
