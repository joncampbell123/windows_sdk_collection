WD98: Memo Wizard Quits If Memo Title Is Over 255 Characters

 Last reviewed: February 17, 1998
Article ID: Q180269
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you type a title that is longer than 255 characters in the title box of the Memo Wizard, one of the following may occur.

Case 1

When you click Finish, you receive the following error message: 

   String is longer that 255 characters.
When you click OK in the error message dialog box, the wizard quits.

Case 2

Word quits and you receive the following error message: 

   The application "Microsoft Word" has unexpectedly quit, because an error
   of type 3 occurred.

   You should save your work in other open applications and restart the
   computer.

WORKAROUND

To work around this problem, either leave the title box blank or type a shorter title. When the wizard is finished, type the title in the document that the wizard creates.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 98 Macintosh Edition.
