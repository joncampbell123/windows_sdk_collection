Word 5.0 for the Macintosh IVR Fax: Installing

 Last reviewed: July 30, 1997
Article ID: Q84105
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

              Microsoft Word for the Macintosh, Version 5.0
               Automated Product Support Service Fax-Script
                     Questions and Answers: Installing
               --------------------------------------------

1. Q. What are the minimum system requirements for installing Mac Word
      version 5.0?

   A. The minimum system requirements for installing Mac Word 5.0 are:

      Hardware: Mac Word 5.0 will run on the Macintosh Plus, SE, Classic,
      Classic II, Portable or PowerBook series, Quada 700 and 900, or
      Macintosh II series with one floppy drive and a hard disk with at
      least 10 megabytes of free disk space.

      Memory: You need 1 megabyte of RAM free with Spelling and Thesaurus
      installed, or 512 kilobytes without Spelling and Thesaurus
      installed. You need 2 megabytes of RAM free if you are running
      MultiFinder, System 7.0, or the Grammar feature.

      System Software: Mac Word 5.0 requires System version 6.0.2 or
      later, including System 7.0.

      To install Mac Word 5.0, perform the following seven steps:

      a. Insert the Word Install disk and double-click the disk to
         open.
      b. Double-click the Installer.
      c. You will be reminded to register your copy of Microsoft Word.
         Click the OK button.
      d. You will be asked if your virus protection software is turned
         off, and be given the chance to continue or quit. If your virus
         protection is off, click the Continue button.
      e. The Easy Install dialog box will appear. Click the Install
         button to proceed with the installation. The Installer will
         check to see whether the hard drive has enough free space
         available.
      f. A dialog box will then require you to select or create a new
         folder for Word. To create a new folder, click the New Folder
         button. You will be prompted to name the folder and then click
         the Create button. If you want to install in an existing folder,
         select that folder and click the Open button.
      g. Click the Install button to continue the install process. The
         Installer will prompt you to insert the appropriate disks and
         will proceed with the installation of Word.

      After the installation is complete, you will be prompted to select
      a default font. After the default font has been selected, you will be
      prompted to restart the computer.
For more information on this subject, please turn to page 11, Chapter 2, in the "Microsoft Word Getting Started" manual.
