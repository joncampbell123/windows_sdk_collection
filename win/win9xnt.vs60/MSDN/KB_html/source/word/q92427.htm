Contents of Upgrade Package for 5.0 to 5.1 Upgrade

 Last reviewed: October 10, 1997
Article ID: Q92427
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.1

SUMMARY

If you purchase the individual upgrade from version 5.0 to version 5.1 of Word for the Macintosh, you receive a subset of the full Word version 5.1 package because some items are unchanged in Word version 5.1.

MORE INFORMATION

Below is a list of items you receive if you upgrade to Word version 5.1 from Word version 5.0:

  • Carton
  • Program disk set (six disks: Install, Program, Converters, Commands, More Commands, and Proofing Tools)
  • "What's New in Word 5.1" book
  • "Microsoft Graph User's Guide"
  • "Microsoft Word for the Macintosh Jump Start" (quick reference card)
  • Promotional materials and coupons

Below is the list of items you receive if you purchase the full Word version 5.1 package (not the upgrade package):
  • Carton
  • Program disk set (six disks: Install, Program, Converters, Commands, More Commands, and Proofing Tools)
  • "Microsoft Word User's Guide"
  • "Microsoft Word Getting Started"
  • "What's New in Word 5.1" book
  • "Microsoft Graph User's Guide"
  • "Microsoft Equation Editor User's Guide"
  • "Microsoft Word for the Macintosh Jump Start" (quick reference card)
  • License agreement
  • Registration card
  • Promotional materials and coupons

To obtain new or updated Microsoft products, call Microsoft Sales Information Center at (800) 426-9400. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, call Microsoft International Customer Service at (425) 936-8661.
