Bookmark Not Deleted When Internal Link Is Removed

 Last reviewed: February 6, 1998
Article ID: Q76456
The information in this article applies to:
  • Microsoft Word for Windows versions 2.0, 2.0a, 2.0a-CD, 2.0b, 

        2.0c, 6.0, 6.0a, 6.0c

SUMMARY

If you remove an internal link in a document in Microsoft Word for Windows, the bookmark is not automatically deleted.

MORE INFORMATION

Steps to Reproduce the Problem

  1. Select text in a Word for Windows document and choose Edit Copy.

  2. Position the cursor in another location in the same document and choose Edit Paste Special, Paste Link to create an internal link.

  3. Choose Edit Undo Link, Edit Links Cancel Link, or select the link and press CTRL+SHIFT+F9 to unlink the LINK field.

The link is deleted from the document, but the INTERN_LINK# bookmark still exists in the Insert Bookmark dialog box. As you create more internal links, the INTERN_LINK# will continue to grow and will not reset back to INTERN_LINK1 even if you manually delete the bookmarks with the Delete button in the Insert Bookmark dialog box.
