MORE INFORMATION

Cover Letter for Replacement Word 6.0a Patch Disk

March 16, 1994

Dear Word for Windows user:

Due to a disk duplication error, the original Word 6.0 for Windows maintenance disk you recently received will not install properly. Please use the enclosed maintenance disk (dated March 8,1994) to upgrade Word 6.0 for Windows. If you have already installed the original maintenance disk, please install the enclosed disk over it.

To install the maintenance disk please do the following:

  1. Close all applications, including Word.

  2. If Windows is not running, start Windows.

  3. Insert the enclosed disk in drive A.

  4. In the Program Manager window, choose Run from the File menu.

  5. Type A:\UPGRADE.EXE and press ENTER.

If you have any questions please contact Microsoft's Product Support Services at 425-462-9673.

We apologize for any inconvenience this may have caused you.

Chris James

Microsoft Customer Service
