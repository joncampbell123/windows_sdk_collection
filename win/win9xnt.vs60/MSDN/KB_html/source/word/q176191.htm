WD: Text Fails to Appear in Envelopes/Labels Delivery Box

 Last reviewed: November 11, 1997
Article ID: Q176191
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

Address text at the insertion point does not appear in the Delivery Address or the Address box of the Envelopes And Labels dialog box.

CAUSE

Word does not recognize the text as a valid address.

WORKAROUND

If any line of the address begins with a tab, the address will not appear when you display the Envelopes and Labels dialog box.

To work around this problem, do either of the following:

  • Select the address prior to choosing to display the Envelopes And Labels dialog box (on the Tools menu.)

    -or-

  • Remove the tabs from the beginning of the address lines in the document.

STATUS
