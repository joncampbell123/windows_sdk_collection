Automatic Word Selection Won't Allow Drag/Drop Between Quotes

 Last reviewed: February 5, 1998
Article ID: Q118520
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Windows operating system version 3.1

SYMPTOMS

You cannot drag and drop selected text between a pair of quotation marks when there is no space between the pair of quotation marks.

CAUSE

You have the Automatic Word Selection turned on. By design, when Automatic Word Selection is turned on, any string of characters not separated by a space is treated as a "word."

RESOLUTIONS

Method 1: Add a space between the quotation marks.

Method 2: Turn off Automatic Word Selection. (To change this option, from 

          the Tools menu, select Options. Select the Edit tab, and clear
          the Automatic Word Selection check box.)

REFERENCES

"Microsoft Word User's Guide," version 6.0, pages 36-37
