WD: Misspelled Words in Main Dictionary

 Last reviewed: February 3, 1998
Article ID: Q125803
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.01
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

The spelling checker does not indicate that the following words are misspelled: 

   histyory
     (Handled incorrectly in all the above versions of Word)

   ofd
     (Word 97 and 98 mark this word as incorrect)

   continuos
     (Continuos is the plural of continuo which is a musical term for a
      bass accompaniment. All versions of Word correctly accept this word.)

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of the article. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Create an exclude dictionary that contains the words that you want to be checked during a spelling check.

For additional information about how to create an exclude dictionary, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q142100
   How to Create an Exclude Dictionary in Word
