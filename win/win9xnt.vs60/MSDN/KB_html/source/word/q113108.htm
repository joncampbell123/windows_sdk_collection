ODBC: Installing the ODBC Administrator or ODBC Drivers

 Last reviewed: February 6, 1998
Article ID: Q113108
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SUMMARY

This article lists various applications that install open database connectivity (ODBC) drivers or the ODBC Administrator.

MORE INFORMATION

The ODBC Administrator is installed when you do any of the following:

  1. Perform a Complete Word 6.0 for Windows installation.

    -or-

    Perform a Custom Word 6.0 for Windows installation in which you select the Converters installation option, choose the Change Options button, and then select the Databases option.

  2. Run ODBC Setup from the Microsoft Access version 1.1 ODBC disk.

  3. Install Microsoft Visual Basic Professional Edition 3.0.

  4. Install the ODBC Drivers Pack.

  5. Install the ODBC SDK.

  6. Perform a Complete Microsoft Excel 5.0 for Windows installation.

    -or-

    Perform a Custom Microsoft Excel 5.0 for Windows installation in which you select the Data Access installation option, choose the Change Options button, and then select all the options.

Note: Depending on the application you use to install ODBC Administrator, a different subset of the ODBC drivers is installed.

Installing an ODBC Driver

  1. Run Windows Control Panel and choose the ODBC icon.

  2. Choose the Drivers button in the Data Sources Drivers list. The Installed Drivers dialog box appears.

  3. Choose the Add button.

  4. Select the location of the ODBC driver files in the Drives and Directories boxes and then choose OK.

  5. If ODBC Administrator finds an ODBC.INF file in the directory you specify, the Install Drivers dialog box displays.

  6. The Available ODBC Drivers list box displays all of the drivers you can install. Select the drivers you want to install and then choose OK.

  7. In the Installed ODBC Drivers dialog box, choose the Close button.

  8. Setup the appropriate data source or choose the Close button to exit ODBC Administrator.
