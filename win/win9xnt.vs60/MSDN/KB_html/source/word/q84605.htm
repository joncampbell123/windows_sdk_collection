Movie PIM Displays Clapboard Instead of Preview Picture

 Last reviewed: July 30, 1997
Article ID: Q84605
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

When using the Movie Plug-In Module (PIM) in Microsoft Word for Macintosh, version 5.0, a movie inserted into a document may display a clapboard image rather than the preview picture. The clapboard image is displayed when QuickTime fails to return a preview picture for Word to display. The clapboard image identifies the object as a QuickTime movie. Yet, the movie will play as expected.

MORE INFORMATION

This situation is a result of the QuickTime extension not returning a preview picture for Word to display. For further information on the QuickTime extension contact: 

   Apple Assistance Center
   Apple Computer, Inc.
   (800) 776-2333
The Clapboard looks similar to the icon for the Microsoft Movie Plug- In Module.
