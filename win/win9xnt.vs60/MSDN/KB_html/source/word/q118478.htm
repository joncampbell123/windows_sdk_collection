How to Merge from PowerLeads! to Word for Windows

 Last reviewed: February 5, 1998
Article ID: Q118478
2.00 2.00a 2.00a-CD 2.00c 6.00 6.00a 6.00c WINDOWS
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a

SUMMARY

When using PowerLeads! there are two ways to merge data to Microsoft Word for Windows. You can write a letter to a single customer, or you can use the PowerLeads! mail merge option.

MORE INFORMATION

Writing a Letter

  1. Start PowerLeads!

  2. Open the desired database.

  3. From the Record menu, choose Write Letter.

  4. When prompted with "OK to write a letter to the contact," choose Yes.

  5. Choose Yes when prompted "OK to use the default letter template?" PowerLeads! displays a message that it is "Sending letters to Microsoft Word."

Word opens with the mail merge main document displayed on screen. This document is attached to the template PWRWORD2.DOT (PWRWINWORD2.DOT in Word 6.0), supplied with PowerLeads! The data is attached as DATADOC.TXT and the header is attached as PWRFLDS.TXT. You are now ready to merge to a new document or directly to the printer.

Merging the Database to Microsoft Word

  1. Start PowerLeads!

  2. Open the desired database.

  3. From the Database Menu, choose Mailmerge.

  4. Select from the available options (select all the records, define a new query, or recall a query).

  5. PowerLeads! prompts you with the message, "Mailmerge requested is the entire database. Is this really what you want to do?" Choose Yes.

  6. Choose OK when prompted to use the default letter template.

Word opens with the mail merge main document displayed on screen. This document is attached to the template PWRWORD2.DOT (PWRWINWORD2.DOT in Word 6.0), supplied with PowerLeads! The data is attached as DATADOC.TXT and the header is attached as PWRFLDS.TXT. You are now ready to merge to a new document or directly to the printer.

Note: When first attempting to write a letter or merging from PowerLeads! to Word version 6.0, you will receive the following error: 

   Microsoft Word Err=1041
   This document template does not exist.
This happens because PWRWINWORD2.DOT does not actually ship with PowerLeads! version 1.2, even though PowerLeads is looking for this template. PowerLeads! ships with PWRWORD2.DOT which may be converted by opening it in Word version 6.0 and saving it as PWRWINWORD2.DOT.
