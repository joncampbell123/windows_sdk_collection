WD6X: Menu Items Available When Word 6.0 Is in Server Mode

 Last reviewed: February 6, 1998
Article ID: Q106282
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

The following menu commands are available when you edit a Word Document Object in place. The container program must support OLE version 2.0 and visual editing. The File and Window menus belong to the container program. 

   Edit Menu        View Menu         Insert Menu
   ---------        ---------         -----------

   Undo/Redo        Page Layout       Break
   Cut              Toolbars          Page Numbers
   Copy             Ruler             Date and Time
   Paste                              Field
   Paste Special                      Symbol
   Clear                              Caption
   Select All                         File
   Find/Replace                       Frame
   Go to                              Picture
   Bookmark                           Object

   Format Menu      Tools Menu        Table Menu
   -----------      ----------        ----------

   All Commands     Spelling          All Commands
                    Grammar
                    Hyphenation
                    Language
                    Revisions
