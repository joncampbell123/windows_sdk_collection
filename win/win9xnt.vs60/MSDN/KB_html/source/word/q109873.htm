WD: Envelope Settings Apply Only to Current Document

 Last reviewed: March 10, 1998
Article ID: Q109873
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

Some of the envelope formatting settings in the Envelope options menu apply only to the current document. (To locate the Envelope options menu, choose Envelopes And Labels from the Tools menu.)

Only the default font for the envelope delivery address and return address can be set from the Envelope Options dialog and be applied to all future envelopes that are created.

RESOLUTION

To make changes to the Delivery Address and Return Address formatting, you must modify the Envelope Address and Envelope Return styles one of the following methods:

Method 1

  1. On the Tools menu, click Envelopes And Labels.

  2. Click the Envelope tab.

  3. Click the Options button.

  4. Click the Font button for the desired address.

  5. Change the settings and click the Default button. Word will show one of the following messages: 

          Do you want to change the default envelope address font to
      <fontname>, <pointsize>?


          This change will affect all new documents based on the NORMAL
      template.


          -or-


          Do you want to change the default envelope return address font to
      <fontname>, <pointsize>?


          This change will affect all new documents based on the NORMAL
      template.

    When you click Yes, the font changes will apply to all new envelopes.

Method 2

  1. On the Format menu, click Style.

  2. In the Style dialog box, under List, select All Styles.

  3. In the Styles box, select either Envelope Address or Envelope Return.

  4. Click Modify.

  5. In the Modify Style dialog box, select the Add to Template check box.

  6. Click Format and then make changes to the style as desired.

  7. Click OK.

    NOTE: Be sure to click Yes to save your changes if Word shows the following message when you exit Word: 

          Changes have been made that affect the global template, Normal.dot.
      Do you want to save those changes?
