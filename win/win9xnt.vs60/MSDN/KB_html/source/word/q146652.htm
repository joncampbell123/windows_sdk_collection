WD6: LaserWriter Pro 630 8.3 Driver Doesn't Print Return Address

 Last reviewed: February 5, 1998
Article ID: Q146652
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When you use the envelope feeder of the LaserWriter Pro 630 to print an envelope, the return address may be missing.

CAUSE

This problem appears to be caused by version 8.3 of the printer driver.

WORKAROUND

To work around this problem, use any of the following methods.

Method 1

Use the version 8.2 of the LaserWriter printer driver rather than version 8.3 of the driver.

Method 2

Use the following steps to add the envelope to the document and print to the manual feed tray:

  1. Create a new document, and click Envelopes And Labels on the Tools menu.

  2. Select the Envelope tab.

  3. Type the delivery and return Address. Click to clear the Omit Return Address check box. Click Options.

  4. On the Envelope Options tab, be sure the correct envelope size is selected, and click OK.

  5. Click Add To Document.

  6. On the File menu, click Print.

  7. In the Paper Source area, select Manual Feed, and click Print.

  8. Click the Range button in the lower part of the Print dialog box.

  9. In the Pages box, type "0" (without the quotation marks), click OK, and then click Print.

You should now be able to print the return address on the envelope.
