WD97: Can't Print Multiple Files from Open Dialog Box

 Last reviewed: February 5, 1998
Article ID: Q160592
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you attempt to print multiple files from the Open dialog box (on the File menu, click Open), some files are not printed. In contrast, when you print multiple files from Windows Explorer, all files are printed.

WORKAROUND

To work around this problem, print fewer files from the Open dialog box. To print from the File Open dialog box, use these steps:

  1. On the File menu, click Open, and then select the documents that you want to print.

  2. Right-click the selection and then click Print.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
