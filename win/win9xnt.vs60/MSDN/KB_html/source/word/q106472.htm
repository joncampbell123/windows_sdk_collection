Underline Problems with HP LaserJet 4/4M

 Last reviewed: February 5, 1998
Article ID: Q106472
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you use the Hewlett-Packard (HP) LaserJet 4/4M printer driver 31.V1.18 or 31.V1.20 with Word 6.0 for Windows, the following underline problems occur:

  • Single underlined sentences stop underlining at the end of each sentence or when italicized text is used. Therefore, between the sentences there is a gap of white space as large as the number of spaces between the sentences.
  • Word for Windows has problems printing a dotted underline with small font point sizes (smaller than 8 point).

    The underline will print as dashes. The width of the dashes is out of proportion to the size of the text.

  • Small caps regular characters (not uppercase characters) affect the width of the underline.

The HP LaserJet 4M printer driver that is included with Windows 95 does not exhibit any of these problems.

RESOLUTION

The underlining problems can be corrected by obtaining an updated HP Laserjet 4/4M driver version 31.V1.26 (HPPCL5E.DRV).

To obtain this update contact the Hewlett Packard Technical Support at the following Web address
