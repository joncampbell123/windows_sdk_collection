Error Message: Not Enough Memory to Run Word

 Last reviewed: February 6, 1998
Article ID: Q81715
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows, you may receive the error message 

   Not enough memory to run word
for the following reasons:
  • The following line has been removed or does not exist in the REG.DAT file:

    Word version 6.0 ---------------- 

          .DOC = Word.Document.6

    Word verison 2.x ---------------- 

          .DOC = WordDocument

    To view the REG.DAT file, choose Run from the File menu in Windows Program Manager. In the Command Line box, type the following "REGEDIT /V" (without the quotation marks).

    If the line does not exist in the Registration Info Editor, choose Merge Registration from the File menu. Select WINWORD6.REG (in Word 6.0)or WW20.REG (in Word 2.x) from the \WINWORD subdirectory. Restart Word.

  • A third-party printer driver is causing problems. Check the printer configuration to ensure that a Microsoft-shipped Windows printer driver is installed and is the default printer. If you have any questions, refer to the Windows Compatibility List of printers.
