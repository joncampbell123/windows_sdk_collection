MacWord 6.0: Location of Files on Setup Disks (10-Disk Set)

 Last reviewed: February 6, 1998
Article ID: Q128852
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

The information in this article provides the location of each file on the Microsoft Word for the Macintosh Setup disks. Note that these files cannot be decompressed or extracted individually from the disks; they can only be installed using the Setup program.

This information may be useful when troubleshooting a Setup issue. For example, if Setup cannot read a particular disk, or the disk appears to be damaged, you can use the following information to determine which item in Setup to clear, so that Setup does not request the disk that is not being read.

The following information is located in the Microsoft Word Setup Data file, a text file located on Disk 1 of the Microsoft Word Setup disks.

(NOTE: In the following lists, "68K" refers to Microsoft Word for the Macintosh, and "PPC" refers to Microsoft Word for the Power Macintosh.)
