MMWinWord: How to Install Online Help Instead of User's Guide

 Last reviewed: July 30, 1997
Article ID: Q88403
The information in this article applies to:
  • Microsoft Word for Windows, version 2.0a-CD
  • Microsoft Windows operating system version 3.1

SUMMARY

Microsoft Word for Windows and Bookshelf, Multimedia Edition, includes the following two help files:

  • Word for Windows Online User's Guide
  • Word for Windows Online Help

The Word for Windows Setup program installs the online user's guide only; however, you can manually install the online help instead of the user's guide.

MORE INFORMATION

To install online help, use the following steps:

  1. Insert the Word for Windows and Bookshelf, Multimedia Edition, CD-ROM disk into the CD-ROM drive.

  2. Copy WINHELP.BAK from the WORD directory on the CD-ROM to the WINWORD program directory on the hard disk. To do this from an MS-DOS prompt, type the following: 

          COPY D:\WORD\WORDHELP.BAK C:\WINWORD

  3. Rename WORDHELP.BAK to WINWORD.HLP. To do this from an MS-DOS prompt, change to the WINWORD program directory, and type the following: 

          REN WORDHELP.BAK WINWORD.HLP

  4. Set the TOOLS-PATH option in the [Microsoft Word 2.0] section of the Windows WIN.INI file to the WINWORD program directory on the hard disk. To modify the WIN.INI, choose Options from the Tools menu, and select the WIN.INI category.

    The following is a sample WIN.INI entry: 

          [Microsoft Word 2.0]
      TOOLS-PATH=C:\WINWORD

  5. To activate online help, press F1, or choose Help Index or Word User's Guide from the Help menu. You can no longer access the user's guide using the Help menu.
