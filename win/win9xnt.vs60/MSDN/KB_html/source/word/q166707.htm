WD97: Resizing Graph Object After Pressing CTRL+Z Causes Error

 Last reviewed: July 31, 1997
Article ID: Q166707
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Graph, version 5.0
  • Microsoft Graph 97 for Windows

SYMPTOMS

If you insert a Microsoft Graph object into a Word document, press CTRL+Z on the keyboard to undo the edit while Microsoft Graph is active, and then attempt to resize the object, you may receive one of the following error messages:

  • Microsoft Word 97 with Microsoft Graph 97 Chart: 

          This program has performed an illegal operation and will be shut
      down. If the problem persists, contact the program vendor.

    If you click Details, you will receive the following error message: 

          GRAPH8 caused an invalid page fault in module GRAPH8.exe at
      <address>

    -or-

  • Microsoft Word 95 with Microsoft Graph 5.0: 

          The graph object will be inserted; however, it may not display
      completely.

    -or-

  • Microsoft Word 6.x with Microsoft Graph 5.0: 

          Graph5 caused General Protection Fault in module GRAPH5.exe at
      <address>.

WORKAROUND

To remove the Graph object from your file immediately after you've inserted it, follow these steps:

  1. Click the document to remove focus from the Graph object.

  2. On the Edit menu, click Undo.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

If you do not Undo the insertion of the Graph object and if you double- click the Microsoft Graph Object, you will receive one of the following errors:

  • Microsoft Word 97: 

          Word cannot edit the Microsoft Graph 97 Chart.
  • Microsoft Word 6.x: 

          Word cannot edit the Microsoft Graph 5.0.
