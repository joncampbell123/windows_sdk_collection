WD: Change to ToolsBulletsNumbers Not Reflected in Dialog Box

 Last reviewed: July 30, 1997
Article ID: Q94848
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SUMMARY

In Microsoft Word for Windows, when you modify the ToolsBulletsNumbers command to set a new default bullet, the change is not reflected in the Bullets And Numbering dialog box. However, when you click OK in the dialog box, the correct bullet is inserted in the document.

Note for Word 6.0 and 7.0: This statement is included for compatibility with the ealier versions of Word; the arguments for ToolsBulletsNumbers correspond to the options in the Bullets And Numbering dialog box (Tools menu) in Word version 2.x. Not every argument applies to each type of list.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word for Windows listed above. This problem was corrected in Microsoft Word 97 for Windows.

REFERENCES

"Microsoft Word for Windows User's Guide", pages 525-526. "Using WordBasic," by WexTech Systems and Microsoft, pages 304-305.
