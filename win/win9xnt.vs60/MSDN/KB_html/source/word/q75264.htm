WD: Summary Info Dialog Box Always Presented During MacWord Save

 Last reviewed: February 9, 1998
Article ID: Q75264
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0, 6.0.1, 6.0.1a

SUMMARY

In Microsoft Word for the Macintosh, the Summary Info dialog box is presented when you save a file if any information has changed in the document since the last save as long as the Prompt For Summary Info preference is selected.

To avoid receiving this dialog box, turn off the Prompt For Summary Info option.

MORE INFORMATION

To turn off the Prompt For Summary Info option, follow these steps:

  1. On the Tools menu, click Options.

  2. From the top, choose the Save tab (in Word 5.x, click the Open and Save category on the left).

  3. Clear the option for Prompt For Summary Info. Click OK.

Clearing this option prevents the Summary Info dialog box from appearing when you save a document; however, you can display the Summary Info dialog box for a particular file by choosing Summary Info from the File menu.
