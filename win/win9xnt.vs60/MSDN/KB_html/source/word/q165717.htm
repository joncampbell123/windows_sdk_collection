WD97: Errors After Running Avery Wizard Setup

 Last reviewed: January 13, 1998
Article ID: Q165717
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you install the Avery Wizard found in the ValuPack of Office 97 prior to running Word 97 for the first time, the following errors occur 

   Can't quit at this time
followed by: 

   This program has performed an illegal operation and will be shut down.
The details of the illegal operation show the following: 

   WINWORD caused an invalid page fault in module <unknown> at
   0000:6510165b.

CAUSE

Avery Wizard setup installs macros to make the Wizard run; it then closes Word before Visual Basic for Applications can complete the operation and close.

RESOLUTION

The Avery Wizard is installed successfully. There is no adverse effect caused by these errors.

STATUS

This problem occurs when you use Word 97 and the Avery Wizard version 1.0.

MORE INFORMATION

The Avery Wizard is used to format and print on Avery LaserJet and InkJet products.
