WD: Font Animation Lost After Saving to HTML Format

 Last reviewed: February 18, 1998
Article ID: Q155948
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

Font animation text effects are lost when you save a Microsoft Word document as Hypertext Markup Language (HTML) format.

WORKAROUND

To work around this problem and retain font animation, save the document in Word Document format and create a hyperlink from another Web page (HTML document) or Word document to this document.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

You can use Font Animation to create animated text effects in Word documents that will be read online. However, Word does not currently support this feature for authoring Web (HTML) documents.
