Common Defaults

 Last reviewed: July 30, 1997
Article ID: Q79173
The information in this article applies to:
  • Microsoft Word for the Macintosh version 5.0

SUMMARY

The following table lists all common defaults set for Microsoft Word for the Macintosh version 5.0:
