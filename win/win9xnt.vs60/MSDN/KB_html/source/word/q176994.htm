WD98: Style List Takes a Long Time to Be Displayed

 Last reviewed: February 17, 1998
Article ID: Q176994
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

The first time you click the Style box (on the Formatting toolbar), the list of styles may take a long time to appear.

CAUSE

By default, Word displays the list of styles in Fancy Font mode, which is a what-you-see-is-what-you-get (WYSIWYG) mode. This Fancy Font mode displays the style names in the format of the style. Being able to see the formats may make it easier for you to select a style.

WORKAROUND

To display the list faster, you must sacrificethe WYSIWYG view of the styles. To turn off the Fancy Font view of the Style list, follow these steps:

  1. On the Tools menu click Preferences.

  2. On the General tab click to clear the Fancy Font check box.

  3. Click OK.
