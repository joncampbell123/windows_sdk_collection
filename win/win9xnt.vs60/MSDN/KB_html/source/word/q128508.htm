WD6: Internet Assistant, General Protection Fault in MACX.DRV

 Last reviewed: February 5, 1998
Article ID: Q128508
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

The error message "general protection (GP) fault in Module MACX.DRV at 0003:005A" may appear when you do any of the following:

  • starting or using Internet Assistant
  • importing a GIF image into a normal Word document
  • selecting Find File from the File menu
  • clicking the Show/Hide button on the standard toolbar

CAUSE

You are using an ATI Mach 64 video driver with one of the following settings: 

   640 x 480 - 32K
   640 x 480 - 65K
   640 x 480 - 16.7M
   800 x 600 - 32K
   800 x 600 - 65K
   800 x 600 - 16.7M

WORKAROUND

Change your ATI Mach 64 video driver to one of the following video resolutions: 

   640 x 480 - 256
   800 x 600 - 256
   1024 x 768 - 256
   1024 x 768 - 32K
   1024 x 768 - 65K
   1280 x 1024 - 256
   1600 x 1200 - 256
