WD97: Printing File in File Open Hangs When File is Open

 Last reviewed: February 5, 1998
Article ID: Q163135
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

Word 97 will stop responding (hang) if you attempt to print a document from the Open dialog box (File menu), while the document is already open in Word.

This problem occurs if you select the Print command from the shortcut menu or the Commands and Settings menu in the Open dialog box.

WORKAROUND

To work around this problem, use either of the following methods:

  • Close the document and then print it from the Open dialog box.

-or-
  • Close the Open dialog box, switch to the open document (if not the active document), and then click Print on the File menu.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Turning off background printing or print spooling does not resolve this problem.
