WD97: Equation Editor Error: Elements That Are Not Available

 Last reviewed: July 31, 1997
Article ID: Q170877
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

In Microsoft Word 97 for Windows, when you double-click an equation that contains a horizontal brace character, you will receive the following error message: 

   This equation contains elements that are not available in Equation
   Editor.

WORKAROUND

To work around this problem, follow these steps to insert the Equation Editor object as an Enhanced Metafile:

  1. Open the document and click the Equation Editor Object.

  2. On the Edit menu, click Copy.

  3. On the Edit menu, click Paste Special, and then click Picture (Enhanced Metafile).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

For more information about the Equation Editor symbols and templates, do the following:

  1. Start the Equation Editor.

  2. On the Help menu, click Equation Editor Help Topics.

  3. On the Contents tab, double-click Reference Information.

  4. Click "Inserting mathematical templates in an equation" and then click "Inserting fence templates in an equation."

NOTE: If Equation Editor Help is not installed on your computer, please see the following article in the Microsoft Knowledge Base:
