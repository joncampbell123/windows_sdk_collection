Compare Versions, Merge Revisions Selects Current Document

 Last reviewed: February 5, 1998
Article ID: Q142753
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you compare two documents using the Compare Versions command, the active document instead of the original document is selected.

The same problem also occurs when you use the Merge Revisions command.

To locate these commands, click Revisions on the Tools menu.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUNDS

Use one of the following methods to work around this problem.

Method 1: Delete the file name in the File Name box, press F5 to update 

          the screen, and then select the document you are comparing (or
          merging).
Method 2: Delete the file name in the File Name box, click the Find Now 
          button, and then select the document you are comparing (or
          merging.
