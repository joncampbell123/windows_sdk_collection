Internet Assistant: How to Compare Versions of HTML Documents

 Last reviewed: February 5, 1998
Article ID: Q128503
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 1.0, 1.0z, 2.0z
  • Microsoft Word for Windows 95 version 7.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SUMMARY

This article describes how to compare two versions of an HTML document.

MORE INFORMATION

To compare two versions of your HTML document, do the following:

  1. Open the first document.

  2. From the File menu, choose Save As. In the Save File As Type list, choose Word Document. The filename now has a .DOC extension.

  3. Choose OK.

  4. Close the first document.

  5. Open the second document.

  6. From the File menu, choose Save As. In the Save File As Type list, choose Word Document. The filename now has a .DOC extension.

  7. From the File menu, choose Templates.

  8. Choose the Attach button.

  9. Highlight the NORMAL.DOT file in the Attach Template dialog box and choose OK.

  10. Choose OK.

  11. From the Tools menu, choose Revisions.

  12. Choose the Compare Versions button. Select the file you created in step 2 above, then choose OK.

hen you have finished comparing versions, reattach the HTML.DOT template to the second document and save the documents as HTML.
