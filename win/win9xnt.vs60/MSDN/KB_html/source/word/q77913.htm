WD: Names and Values for Business and Symbol Characters

 Last reviewed: February 6, 1998
Article ID: Q77913
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

The "Microsoft Word for Windows User's Reference Guide" does not list the names and values for business and symbol characters available with the symbol soft font set included with Word for Windows.

Below are the names and values of each symbol and business character available within the symbol soft font set. The ANSI equivalents are also shown if available:

Word for Windows versions 6.0, 7.0, 97

From the Insert menu, choose Symbol. A window will appear displaying tabs for Symbols and Special Characters. The Symbols tab allows you to select fonts and symbols from the installed fonts. The Special Characters tab displays certain special characters such as the copyright and trademark character. The Special Characters tab also displays the Shortcut Key assigned to these symbols. (Ex. Copyright symbol has the Alt+Crtl+C keystroke assigned to it.)

Word for Windows version 2.0

From the Insert menu, choose Symbol. A window displays that allows you to select the desired symbol in a variety of fonts.

For additional information, query on the following words: 

   word for windows and character and test and macro

Word for Windows version 1.x

From the File menu, choose Open; select KEYCAPS.DOC. This displays the Symbol characters that are accessible from the normal keyboard when the Symbol font is selected on the ribbon.

MORE INFORMATION

In Word 6.x-97, you can also quickly insert a symbol character by pressing the CTRL+SHIFT+Q keys and typing the letter that you wish to appear in the Symbol font.

In Word 6.x for the Macintosh the shortcut keys are COMMAND+SHIFT+Q.

For example to insert the pie symbol in Word for Windows, type CTRL+SHIFT+Q and the letter "p" (without the quotation marks). After the pie symbol appears, Word returns to the original font you were using.
