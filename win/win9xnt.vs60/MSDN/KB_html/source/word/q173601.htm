WD97: Cannot Color Table Border in HTML Document

 Last reviewed: September 10, 1997
Article ID: Q173601
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

In Microsoft Word 97, if you add a table to an HTML document, there is no option to color the border.

WORKAROUND

If you copy and paste an existing table to a new Word document, you can color the table border in the Word document. You can then copy and paste the newly colored table from the Word document to the existing HTML document.

REFERENCES

For additional information about features that are not available during Web authoring, please see the following article in the Microsoft Knowledge Base:
