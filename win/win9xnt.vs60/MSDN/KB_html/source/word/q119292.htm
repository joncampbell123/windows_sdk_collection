Word for Windows NT: Description of Features

 Last reviewed: August 5, 1997
Article ID: Q119292
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0

SUMMARY

Word 6.0 for Windows NT is an adaptation of Word 6.0 for Windows. All of the new features present in Word 6.0 for Windows NT were added specifically to take advantage of the following features, which are unique to Windows NT:

  • 32-bit architecture
  • Long filename support
  • Multithreaded printing
  • Registry support

MORE INFORMATION

Word for Windows NT offers the following features over its 16-bit Word for Windows counterpart:

  • 32-bit Architecture

    Word for Windows NT is written to the Win32 API specification and is, therefore, a full 32-bit application.

  • Long Filename Support

    Word for Windows NT supports up to 255 characters for document file names. All Word dialog boxes that involve filenames, such as File Open, are wider to accommodate longer filenames as well.

  • Multithreaded Printing

    The background printing feature in Word for Windows NT runs as a separate thread. This makes working on documents and background printing smoother.

  • Registry Support

    Word for Windows NT stores all of its settings in the Windows NT Registry. As a result, multiple configurations of Word can exist on the same computer for different users. Word will also support user profiles, which means a user can move from one machine to another and continue to use his or her same Word configuration.
