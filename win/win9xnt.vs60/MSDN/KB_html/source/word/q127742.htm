How to See Both WordArt And Graphic in Text Box

 Last reviewed: February 5, 1998
Article ID: Q127742
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you insert both a WordArt object and a graphic in a text box or frame, you can only see one of them at a time.

CAUSE

The "Don't blank the area behind metafile pictures" option is not selected.

RESOLUTION

  1. From the Tools menu, choose Options.

  2. Select the Compatibility tab.

  3. Select the "Don't blank the area behind metafile pictures" check box (the Recommended Options For box does not need to be changed).
