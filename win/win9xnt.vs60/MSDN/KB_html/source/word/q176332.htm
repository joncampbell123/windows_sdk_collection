WD: Registered Trade Mark Symbol Conversion Problems

 Last reviewed: February 18, 1998
Article ID: Q176332
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you save a Word document containing a registered trademark symbol (ASCII 0174) in a text file format (such as text only or text with line breaks) the registered trademark symbol may not be retained in the new format.

WORKAROUND

To ensure that a registered trademark symbol is retained when you save a file in text format, see the table in the "More Information" section of this article and use one of the text file formats that indicates "OK" under Conversion Results.

MORE INFORMATION

The following table presents the results of what happens to the registered trademark symbol in a Word document when you save the document in a certain text format. In the table OK means that the registered trademark symbol is retained. 

Text Type Format               6.x    7.x    97(Windows) 98(Macintosh)
---------------------------------   ----------Conversion Results-------

Text Only                           OK     (r)       (r)         (r)

Text only with Line Breaks          OK      OK        OK          OK

Ms-Dos Text                          r     (r)       (r)         (r)

MS-Dos Text with Line Breaks         r      r         r           OK

MS-Dos Text with Layout (Windows)    r      r         r           N/A

Text With Layout (Macintosh)        N/A     N/A       N/A         OK
