WD98: Dialog Display Problems With Greg's Power Windows

 Last reviewed: February 3, 1998
Article ID: Q178188
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Word, some dialog boxes, such the Break dialog box (on the Insert menu, click Break), become hard to see and use after you drag them around the screen.

CAUSE

This problem may occur if Greg's Power Windows version 1.2.1 has been installed your Macintosh.

RESOLUTION

To resolve this problem, obtain a more recent version of Greg's Power Windows.

To temporarily work around this problem, use the Extensions Manager to disable Greg's Power Windows. For Information about using the Extensions Manager in the Mac OS, follow these steps:

  1. On the Apple menu, point to Control Panels, and then click Extensions Manager.

  2. Click the ? button in OS 8 or click Help in OS 7.X.

  3. In OS 8 click Manage System Folder Extensions, and then click OK.

STATUS

This problem occurs when you use Microsoft Word 98 Macintosh Edition and Greg's Power Windows 1.2.1.
