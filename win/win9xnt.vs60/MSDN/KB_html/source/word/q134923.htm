WD: Tab As Indent Doesn't Work with Preset Tab Stops

 Last reviewed: February 5, 1998
Article ID: Q134923
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word 97 for Windows

SYMPTOMS

The Tab As Indent feature does not automatically indent your paragraphs if there are preset tab stops in the paragraph.

CAUSE

The Tab As Indent feature is designed to set a first line indent using the default tab stops. If other tab stop positions have been defined in the paragraph, the Tab As Indent feature is disabled.

WORKAROUND

To set a first line indent, click Paragraph on the Format menu.

MORE INFORMATION

To use the Tab As Indent feature, select the "Use Tab And Backspace Keys To Set Left Indent" option ("Tabs And Backspace Set Left Indent" in Word 97). (To locate this option, click Options on the Tools menu, then select the Edit tab.) This option is turned on by default.
