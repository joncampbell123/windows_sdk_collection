Foreign Dictionaries for Microsoft Word 5.x

 Last reviewed: July 30, 1997
Article ID: Q79570
The information in this article applies to:
  • Microsoft Word for the Macintosh versions 5.0, 5.1

SUMMARY

Foreign dictionaries for Microsoft Word for the Macintosh version 5.0 and 5.1 are available from Alki Software. These packages contain spelling, thesaurus, and hyphenation files for the following languages:

Note: These dictionaries do not work with Word for the Macintosh version 6.x.
