WD: Paper in Multiple-Section Document Feeds from Wrong Tray

 Last reviewed: February 5, 1998
Article ID: Q138417
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

When you print a document that has multiple sections and the first page is supposed to print from a different tray than the rest of the document, other pages also print from this tray.

CAUSE

When you chose Different First Page in the Page Setup dialog box, you also chose Whole Document in the Apply To list, but your document has more than one section.

If you choose Whole Document, the First Page setting will apply to the first page of EACH section, not to the first page of the entire document.

RESOLUTION

To have the first page print from a different tray than the rest of the whole document, use these steps:

  1. Place the insertion point in the first section.

  2. On the File menu, click Page Setup.

  3. Click the Paper Source tab.

  4. Choose the tray you want the first page to print from.

  5. In the Apply To list, choose This Section.

  6. Click OK.

If you have a document in which the first page of each section is printing from the wrong tray, use these steps to correct the problem:

  1. Place the insertion point in the first section.

  2. On the File menu, click Page Setup.

  3. Click the Paper Source tab.

  4. Choose the tray you want the first page to print from.

  5. In the Apply To list, choose This Section.

  6. Click OK.

  7. Select (highlight) the second and subsequent sections.

  8. On the File menu, click Page Setup.

  9. In the First Page list, choose the tray you want all the pages to print from (usually Default Tray).

  10. In the Other Pages list, make the same selection as you did in the First Page list.

  11. In the Apply To list, choose Selected Sections.

  12. Click OK.

The document will now print from the correct trays on the printer.
