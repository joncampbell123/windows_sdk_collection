Show All Option Displays Page and Section Breaks in WinWord

 Last reviewed: February 6, 1998
Article ID: Q78212
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows, page breaks and section breaks do not display in page or page layout view if the Show All option is selected. These breaks do not display in print preview or in printer output.

MORE INFORMATION

Section breaks inserted into a Word for Windows document display as double broken lines in normal, draft, and outline views. Page breaks display as single broken lines in normal, draft, and outline views.

Section breaks are also displayed as double broken lines in page layout (version 2.0) or page (version 1.x) view if you select the Show All option. Use the following instructions to activate the Show All option:

Word for Windows 6.0

  1. From the View menu, choose Page Layout.

  2. From the Tools menu, choose Options.

  3. Select the View tab. From the Non-printing Characters box, select the All option. Choose the OK button.
  • or

    Choose the Show/Hide Nonprinting Characters button on the toolbar to toggle the Show All option.

    Word for Windows 2.0

    1. From the View menu, choose Page Layout.

    2. From the Tools menu, choose Options.

    3. From the Category box, select View. From the Nonprinting Characters box, select the All option. Choose the OK button.

      or

    Choose the Show/Hide Nonprinting Characters button on the ruler to toggle the Show All option.

    Word for Windows 1.x

    1. From the View menu, verify that Page is selected.

    2. From the View menu, choose Preferences. Select the Show All option, and choose the OK button.

      or

    Choose the Show All button on the ruler to toggle the Show All option.
