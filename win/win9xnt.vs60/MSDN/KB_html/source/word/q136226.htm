List of Fixed Bugs in Word 7.0 for Windows 95

 Last reviewed: February 2, 1998
Article ID: Q136226
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SUMMARY

The following table lists problems in earlier versions of Word for Windows that were corrected in Word version 7.0 for Windows 95. This is not a comprehensive list and is current as of 1/8/96. For more information about each issue, query in the Microsoft Knowledge Base on the article identifier (ID).
