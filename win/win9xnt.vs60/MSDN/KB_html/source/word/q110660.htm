Pasting Paintbrush Picture: Error! Not a Valid Embedded Object

 Last reviewed: July 30, 1997
Article ID: Q110660
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you paste a Microsoft Paintbrush picture in Word for Windows, you may receive the following error message: 

   Error! Not a valid embedded object.

CAUSE

In Microsoft Paintbrush, you have the Omit Picture Format command turned on.

WORKAROUND

On the MS Paintbrush Options menu, clear (by choosing) the Omit Picture Format command.

MORE INFORMATION

To create embedded objects, a server application must make the Picture format available, along with the OwnerLink format. When you select the Omit Picture Format in Microsoft Paintbrush, and then copy the image to the Clipboard, the Picture (CF_Metafile) format is not included with the object in the Clipboard.
