WD6X: Word 6.0 for Macintosh Starts Slowly

 Last reviewed: November 17, 1997
Article ID: Q122392
The information in this article applies to:
  • Microsoft Word for Macintosh, version 6.0

SYMPTOMS

It takes a long time to start Word 6.0 for the Macintosh.

CAUSE

This problem is generally caused by a large number of fonts installed on a system. PostScript and bitmap/screen fonts are more likely to cause a longer startup time than are TrueType fonts.

Each time Word starts, Word lists and reads all the fonts available on the system. It uses this data to support kerning and other font functions that Word 5.x does not support.

RESOLUTION

To reduce the amount of time required to start Word, use one of the following methods:

  • Remove any fonts that are infrequently or never used.

    -or-

  • If you are using Suitcase II for the Macintosh, you may be able to disable some sets of fonts.

STATUS

This problem was corrected in Word version 6.0.1 for the Macintosh.
