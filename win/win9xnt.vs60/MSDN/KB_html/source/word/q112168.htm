Error Starting Word: "Invalid DC Index: 17"

 Last reviewed: February 5, 1998
Article ID: Q112168
6.00 6.00a 6.00c WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows operating system version 3.1
  • Microsoft Windows for Workgroups, versions 3.1, 3.11

SYMPTOMS

When you start Word for Windows, your computer may hang (stop responding) or the following error message may occur: 

   Invalid DC Index: 17

CAUSE

These symptoms may occur if either of the following Windows 3.0 printer drivers is installed and set as the default Windows printer:

  • Star Micronix XB-2410, LC-2420 (STARM24.DRV)
  • Star NX-2430
  • Brother M1824L (BROTH24.DRV version 1.00, dated 9/20/90)
  • Canon bj102 (CANON10E.DRV, dated 9/20/90)
  • Canon bj10E

When the "Invalid DC Index: 17" error message occurs, you can choose OK and continue working in Word.

In addition, Word may hang when you start the program if you have not installed a default Windows printer driver.

WORKAROUND

Microsoft does not recommend using Windows 3.0 printer drivers under Windows 3.1. You should remove such drivers under the Printers icon in Windows Control Panel.

There is no Windows 3.1 printer driver available for the Brother M1824L printer (this printer emulates the IBM Proprinter XL).

A Windows 3.1 printer driver for the Star Micronix XB-2410 and Star NX-2430 printers are available in the file "STAR.EXE" which you can download:
