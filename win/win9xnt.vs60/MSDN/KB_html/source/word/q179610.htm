WD6X: Not Enough Memory Error Using Agenda Wizard in MacWord

 Last reviewed: January 21, 1998
Article ID: Q179610
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When you create a new document in the Modern style using the Agenda Wizard, and that document contains a WordArt object, you may receive the following error: 

   Word cannot find the graphics filter, or there is not enough memory to
   load it.
Also, after you modify the WordArt object in the Agenda Wizard and click Apply, you may receive the following error: 

   WordArt does not have enough memory to transfer the picture of the text
   you typed.

CAUSE

This error message occurs if you have installed the Word 97-98 Import Converter, and Word does not have enough memory to load a graphics filter or run WordArt properly.

RESOLUTION

To work around this problem, try typing less text.

To resolve this problem, use the appropriate method for your situation.

Word Cannot Find the Graphics Filter

For the first error message 

   Word cannot find the graphics filter, or there is not enough memory to
   load it
remove the following files from the Word Extensions folder:
  • 68K JPEG Import
  • 68K Metafile Import
  • 68k PNG Import

This folder is normally in the following location: 

   <hard drive>:Microsoft Office:Microsoft Word 6:
NOTE: Removing these files does not affect your ability to import Word 97 or 98 documents into Word. However you will no longer be able to import pictures in JPEG, Windows Metafile, and PNG formats.

WordArt Does Not Have Enough Memory to Transfer the Picture of the Text

For the second error message 

   WordArt does not have enough memory to transfer the picture of the text
   you typed. Try typing less text, or close the WordArt application and
   use the finder's GetInfo command to increase the size of WordArt's
   memory partition.
increase the Preferred Memory Size for WordArt by following these steps:

  1. Locate the icon for the WordArt program, which is usually in the Microsoft folder on the hard disk, and click it.

  2. On the File menu, click Get Info.

  3. In the Preferred Size box change the memory size by adding an additional 1000 K (kilobytes).

  4. Close the window by clicking in the upper left corner.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
