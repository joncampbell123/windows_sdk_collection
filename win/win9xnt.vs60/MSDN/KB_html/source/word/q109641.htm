FileNameFromWindow$()

Syntax:

FileNameFromWindow$([WindowNumber])

Example:

The following sample macro gets the names of saved documents from the window menu, stores the names in an array and inserts the names in a new document. Open documents that have not been saved (for example, [Document1]) are extracted from the array and are not included in the document. 

   Sub MAIN
      FileNewDefault
      On Error Goto exit
      Number - (CountWindows() - 1)
      Dim Windo$(Number), holder$(Number)
      count - 0
      For I - 0 To number
         If FileNameFromWindow$(I + 1) <> "" Then
            holder$(count) - FileNameFromWindow$(I + 1)
            Count - Count + 1
         End If
      Next i
      count - count - 1
      Redim Windo$(Count)
      For x - 0 To count
         Windo$(x) - Holder$(x)
         Insert Holder$(x)
         InsertPara
      Next
   Exit:
   End Sub
