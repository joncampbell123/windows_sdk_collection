WD6X: Automatic, Manual Link Update Unavailable

 Last reviewed: February 6, 1998
Article ID: Q144878
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a and 6.0c
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

The Automatic or Manual options are unavailable when you attempt to update the link to a Canvas EPS graphic or another file. (To locate these options, select the object, and then click Link on the Edit menu.)

CAUSE

The object does not support dynamic data exchange (DDE).

WORKAROUND

To work around this problem, use either of the following methods.

Method 1

Select the object and then press F9 to update it.

Method 2

Create a macro that updates all fields in your document automatically when you open your document.
