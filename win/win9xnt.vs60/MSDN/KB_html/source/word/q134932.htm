WD: Memo Wizard Limits To and CC Lines If Address Book Selected

 Last reviewed: February 5, 1998
Article ID: Q134932
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

You are limited to 255 characters on the To: and cc: lines when you use the Memo Wizard and select addresses from the Address Book.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95 version 7.0. This problem has been corrected in Word 97 for Windows.

WORKAROUND

Method 1. If you have a distribution list that exceeds 255 characters, wait 

          until the the Memo Wizard is finished before typing the list.
          When the wizard finishes, type your list where you see "Click
          here and type names."
Method 2. Limit the number of names you select from the Address Book, then 
          type the remaining names after the Memo Wizard finishes.
NOTE: The 255 character limit includes punctuation and spaces.
