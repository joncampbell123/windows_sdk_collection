Blank Envelope Feeds Printing Envelopes to Canon BJ-200/230

 Last reviewed: February 5, 1998
Article ID: Q98902
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SYMPTOMS

Word Version 6.0

When you print envelopes to a Canon Bubble-Jet BJ-230 or BJ-200 printer using Manual Feed, the envelope may not print. This is true of both drivers, BJ-230.DRV and BJ200.DRV.

Word Versions 2.x

When you print envelopes, a blank envelope feeds and ejects, then an Insert Paper prompt appears (if you are using Windows Print Manager). After you insert a second envelope and choose OK, the envelope prints correctly.

Printing an envelope from Windows Write results in the same behavior.

NOTE: Use the Canon Bubble-Jet BJ-230 printer driver version 1.1 (BJ-230.DRV) or the Canon Bubble-Jet BJ-200 printer driver version 1.0 (BJ200.DRV) to print in Microsoft Windows version 3.1.

WORKAROUNDS

Word Version 6.0

  1. From the Tools menu, choose Envelopes And Labels. Select the Envelopes tab. Choose Options.

  2. Select the Printing Options tab. From the Feed From list, select Auto Sheet Feeder and choose OK.

  3. Choose the Print button.

The envelope should print correctly.

Word Versions 2.x

To avoid the first blank envelope, use one of the following workarounds:

  • Do not preload the envelope in the printer and set Windows Print Manager to "Always Warn." Leave the printer tray empty, print the envelope from Word for Windows and wait for the Insert Paper prompt. When you place the envelope in the tray and choose OK, the first envelope prints and no blank envelope feeds.

    -or-

  • Set one of the following printer heuristics in the [Microsoft Word 2.0] section of the Windows WIN.INI file: 

          BJ-230=257 (for the BJ-230 printer, using BJ-230.DRV)


          -or-


          BJ200=257 (for the BJ-200 printer, using BJ200.DRV)

After you set the heuristic, place the envelope(s) in the paper feed tray, but do not push them all the way into the paper rollers. When you print using the Create Envelope command, the envelopes feed one at a time into the printer and print correctly in landscape orientation. The Insert Paper prompt no longer appears.

The above heuristic is necessary to avoid the Insert Paper prompt. The drawback to using the 257 heuristic is that paper size information that you set in the Format Page Setup dialog box is lost. This problem is most evident when you format a document for legal-size paper.
