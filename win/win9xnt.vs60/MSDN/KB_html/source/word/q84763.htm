WD: Error "Clipboard Does't Contain Equation Data" (System 7.0)

 Last reviewed: November 17, 1997
Article ID: Q84763

The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0

SYMPTOMS

In Microsoft Word for the Macintosh, if an equation created with the Equation Editor is copied from a Word document and pasted into the Equation Editor, the following error message may appear: 

   Clipboard does not contain equation data.
This problem may occur when using System 7.0.

RESOLUTION

To avoid this problem, an equation should be edited by double-clicking it. When using System 7.0, this should automatically start the Equation Editor and place the equation in the active window.

STATUS

Microsoft is researching this problem and will post new information here as it becomes available.

MORE INFORMATION

Page 22 of Word 5.0's "Equation Editor User's Guide," correctly states that under System 6, the user should copy the data from Equation Editor into the Word 5.0 document using the Clipboard. It also correctly states that to edit the equation, you can copy from Word 5.0 back into Equation Editor.

However, it incorrectly states that "This method also is available under System 7."
