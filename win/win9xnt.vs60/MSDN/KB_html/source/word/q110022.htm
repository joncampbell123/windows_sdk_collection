WD6X: Font Formatting Does Not Display in Some Dialog Boxes

 Last reviewed: February 5, 1998
Article ID: Q110022
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

In the Tools Envelopes and Labels dialog box, under the Envelopes tab, Word does not display any font formatting applied to the Return or Delivery Address text. This happens regardless of which of the following methods you use to format the text:

  • Copy formatted text from your document and paste it in the Return Address or Delivery Address box. 

        -or-
  • Format the text by choosing the Options button in the Tools Envelopes and Labels dialog box and then choosing the Font button under Delivery Address or Return Address.

Even through font formatting is not displayed under the Envelopes tab, the formatting prints correctly. It also displays correctly if you add the envelope to your document.

In contrast, Word displays all the font formatting for the Address text under the Labels tab in the Tools Envelopes and Labels dialog box.

CAUSE

Word does not display font formatting in the text box of any dialog box that is not an editable Rich Text Edit (RTE) dialog box. For example, the Insert Caption, Mark Citation, Mark Index Entry dialog boxes, and the Envelopes tab of the Tools Envelopes and Labels dialog box are not editable RTE dialog boxes.

In contrast, Word displays all font formatting in an editable RTE dialog box (such as the Labels tab of the Tools Envelopes and Labels dialog box).

NOTE: In an editable RTE dialog box, you can apply multiple font formats and see the results of the formatting in the dialog box. For more information about RTE dialog boxes, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q106047
   TITLE     : Definition of Rich Text Edit (RTE) Dialog Boxes

STATUS

This is a limitation of RTE dialog boxes that are not editable.

MORE INFORMATION

Word displays the text in the Envelope dialog box according to the Envelope Address and Envelope Return styles. Any changes to the envelope styles will be reflected in the envelope dialog box. Direct formatting applied to the delivery address or return address text will not be displayed in the envelope dialog box.
