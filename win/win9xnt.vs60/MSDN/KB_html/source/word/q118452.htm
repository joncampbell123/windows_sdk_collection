What Is Stored in the Word Settings (6) File?

 Last reviewed: February 5, 1998
Article ID: Q118452

The information in this article applies to:

  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

The Word Settings (6) file contains the following Word for the Macintosh preference and option settings:

  1. Tools Options dialog box. The following table lists each tab in the Tools Options dialog box along with the settings for each tab that Word stores in the Word Settings (6) file: 

       Tab                   Settings Stored in Word Settings (6)
   ---                   ------------------------------------

   View                  Show, Window, Nonprinting Characters.

   General               General options.

   Edit                  Editing options.

   Print                 Printing options, include with document.

   Revisions             Inserted text, deleted text, revised lines.

   User Info             Changes to Name, Initials, Mailing Address.

   Compatibility         Note: Options are saved only when you choose
                         the Default button.

   File Locations        All.

   Save                  Save options.

   Spelling              Suggest, Ignore, custom dictionaries, Language.

   Grammar               Check Spelling, Show Readability.

   AutoFormat            Preserve, Apply Styles To, Adjust, Replace.

   Note: The unlisted settings are not stored in the Word Settings (6)
   file.

  2. "Tip of the Day" option (on or off).

  3. Location of the toolbars on screen (position on screen, docked or floating).

  4. Recently Used File list (MRU).

  5. Recently Used Font list (MRU).

  6. Table of authority categories.

  7. Tools Advanced Settings command options. (This command is not on the Word menus by default. Run the ToolsAdvancedSettings macro to open the Advanced Settings dialog box.)

Note: The Word Settings file is not transferable between two machines with the exact hard drive names and Word setup parameters.
