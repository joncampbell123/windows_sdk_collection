WD97: ActiveX Controls Lose Sizing After Previewing in Browser

 Last reviewed: February 6, 1998
Article ID: Q162598
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Internet Explorer version 3.01 for Windows 95

SYMPTOMS

When you view an HTML document in print preview that contains a sized ActiveX control in Microsoft Internet Explorer 3.01, the size information is lost and is reset to its original size.

CAUSE

This functionality occurs when you save the document as HTML in Word 97 and you view the HTML document in Microsoft Internet Explorer 3.01.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
