Hyphenation Not Available in Protected Documents

 Last reviewed: February 5, 1998
Article ID: Q118315
6.00 6.00a 6.00c WINDOWS kbusage buglist6.00 buglist6.00a buglist6.00c fixlist7.00

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

Hyphenation is not available when your document is protected for forms or annotations. If a form document contains multiple sections, protecting selected sections does not allow hyphenation in unprotected sections. This behavior does not occur when the document is protected for revisions.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows versions 6.0, 6.0a, and 6.0c. This problem was corrected in Word version 7.0 for Windows 95.

RESOLUTION

Turn off the protection for annotations or forms, hyphenate the document, then reprotect the document for annotations or forms.
