WD: ToolsEnvelopesAndLabels Command Doesn't Insert Label Address

 Last reviewed: February 3, 1998
Article ID: Q140746
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

A macro that contains the ToolsEnvelopesAndLabels command or that dimensions the ToolsEnvelopesAndLabels dialog box, does not place the address in the Address box on the Labels tab. (To locate this option, click Envelopes And Labels on the Tools menu.)

For example, the following macro copies the address to the Envelopes tab but not to the Labels tab. 

   Sub MAIN
      Dim dlg As ToolsEnvelopesAndLabels
      GetCurValues dlg
      Dialog dlg
      ToolsEnvelopesAndLabels dlg
   End Sub

Use the following macro to select your address text and place it in the Address box on the Labels tab. 

   Sub MAIN
      Dim dlg As ToolsEnvelopesAndLabels
      GetCurValues dlg
      dlg.ExtractAddress = 1
      If Dialog dlg Then ToolsEnvelopesAndLabels dlg
   End Sub
