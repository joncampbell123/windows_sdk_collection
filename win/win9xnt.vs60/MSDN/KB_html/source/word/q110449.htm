WD: Edit Redo May Misplace Empty Bookmarks

 Last reviewed: February 2, 1998
Article ID: Q110449
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows 95, versons 7.0, 7.0a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In Microsoft Word, using a combination of the multiple level Undo/Redo commands may result in an empty bookmark being "redone" at the wrong place.

This problem occurs only with empty bookmarks (bookmarks that contain no text). Bookmarks that contain text or graphics behave as expected.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Word 97 for Windows.
