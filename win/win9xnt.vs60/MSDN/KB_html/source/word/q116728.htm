Word for Windows 95: Files Installed During Setup

 Last reviewed: February 6, 1998
Article ID: Q116728
7.00 7.00a WINDOWS kbref

The information in this article applies to:

  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SUMMARY

This article contains the list of components and files installed by Word for Windows 95 during each of the Compact, Typical, and Complete Setup options.

A summary of the components installed is given first, followed by a complete list of files.

If you chose an installation type that did not install a component or file that you wanted, you can reinstall the missing component by running Setup again.
