Using Points as Unit of Measurement Increments Incorrectly

 Last reviewed: February 5, 1998
Article ID: Q121607
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-cd, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word 97 for Windows

SYMPTOMS

When you set the default unit of measurement to points, line spacing does not increment correctly. For example, if line spacing is set to Single, the first line of your document will indicate 72 points, but when you press ENTER, the status bar indicates 83.4 points. The same behavior occurs if you set line spacing to Exactly and specify a point size or if you choose one of the other Line Spacing options.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
