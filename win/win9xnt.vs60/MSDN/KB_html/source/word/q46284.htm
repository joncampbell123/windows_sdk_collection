WordBasic Error Code 54: Bad File Mode

 Last reviewed: July 30, 1997
Article ID: Q46284
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

In Microsoft Word for Windows, the WordBasic Error Code Number 54, "Bad file mode," indicates an attempt was made to execute an Open with a file mode other than INPUT, OUTPUT, or APPEND. This error can also occur if an attempt was made to read from a file opened for output or appending, or if a file hasn't been opened.

For more information on Open, please refer to the "Microsoft Word for Windows Technical Reference," page 71
