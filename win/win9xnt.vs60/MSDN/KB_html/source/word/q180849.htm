WD98: Keyboard Shortcut Changes Under 5.1 Interface

 Last reviewed: March 11, 1998
Article ID: Q180849
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

Microsoft Word 98 Macintosh Edition includes a User Interface for users who are more comfortable working with a Microsoft Word 5.1 for the Macintosh Interface (5.1 menus). To use the Word 5.1 menu items and their keyboard shortcuts, the Word 5.1 toolbar, Word 5.1 ruler, and the Word 5.1 ribbon, click Word 5.1 Menus on the View menu.

NOTE: Shortcut keys mentioned in the Help topics, menus, and dialog boxes refer to the U.S. keyboard layout. Keys on other layouts may not correspond exactly to the keys on a U.S. keyboard. If you are using keyboard layout different from the version of Word you are using, you may have to make adjustments when using shortcut keys.

The following Word 98 Commands have no key sequences assigned to them under the Word 5.1 Interface: 

   EditGoto
   OtherPane
   ToolsProofing
   ToolsThesaurus
   ExtendSelection
The following table lists Commands with key sequence changes between the Word 5.1 Interface and the Word 98 Interface. 

   Command       Word 98 key sequence    Word 5.1 key sequence
   ------------------------------------------------------------

   RightPara     COMMAND+R                COMMAND+SHIFT+R
   LeftPara      COMMAND+L                COMMAND+SHIFT+L
   Indent        COMMAND+M                CONTROL+M

For more information about using Word 5.1 key commands in Word 98, click Contents And Index on the Help menu (or on the Balloon Help menu if you are using a version of the Macintosh operating system earlier than 8.0), click the Index button in Word Help, type the following text 

   keyboard shortcuts, using
and then click Show Topics. Select the " Shortcut keys " topic, and click Go To. Under "What do you want to do?” click "Use keys for the Word 5.1 interface in Word 98." If you are unable to find the information you need, ask the Office Assistant.
