WD98: Creating New Template from Wizard Causes "Command Failed"

 Last reviewed: February 17, 1998
Article ID: Q176283
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you try to create a new template based on a wizard in Word, you may receive the following error message: 

   Command Failed
NOTE: This problem occurs when you try to create a new template based on any of the following wizards:
  • Agenda Wizard (from the ValuPack)
  • Fax Wizard
  • Memo Wizard
  • Resume Wizard

WORKAROUND

Create a new document based on the wizard and then save the new document as a template.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To create a new template based on a Wizard, follow these steps:

  1. On the File menu, click New.

  2. Click to select a Wizard.

  3. Under Create New, click to select Template, and then click OK.
