IncludeText Field Disappears from Converted SGML Doc

 Last reviewed: August 5, 1997
Article ID: Q125883
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you save a Word document as SGML, text that was the result of an IncludeText field disappears.

All other field code results are retained. Only the result of the IncludeText field is lost. No annotation is made to indicate that the text is gone.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SGML Author for Word. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Before converting the Word document, unlink the fields so that their contents are part of the document.

To unlink a field, use the following steps:

  1. In the Edit field, select Links.

  2. Select the Source Files you want to unlink.

  3. Click the Break Link button.
