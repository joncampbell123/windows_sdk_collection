WD6X: Doc Err: Converting Addresses from Word 5.1 to Word 6.0

 Last reviewed: February 5, 1998
Article ID: Q126893
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

In the "Microsoft Word User's Guide" for Word 6.0 (Document No. WB60460- 0794), there are two omissions on page 539 from the section titled "Creating and Printing Envelopes and Labels."

The user's guide explains the procedure for converting an existing address list from Word version 5.1 to AutoText entries or an address file in Word version 6.0. However, two important steps are not documented.

MORE INFORMATION

The paragraph headed "Converting the addresses" states 

     This template is installed in the Macros folder in the Word program
     folder when you perform a complete installation of Word.
If you did not perform a complete installation, you need to run Setup in Maintenance Mode and add the Macro Templates. Follow the steps on page 16 of the "Quick Results" book, in the section "To install or remove individual components of Word for the Macintosh." The Macro Templates are listed in the Wizards, Templates, and Letters category.

The same paragraph instructs you to "double-click the button in the template."

The button referred to by this instruction is named "Convert 5.1 Addresses" and is part of a custom toolbar found only in this template.

If this button does not appear, use the following steps to display the button:

  1. From the View menu, choose Toolbars

  2. Select (mark with an X) 5.1 Upgrade.

  3. click OK.

The toolbar containing the Convert 5.1 Address button now appears.
