WD: Selection with Tab Not Inserted in Delivery Address Box

 Last reviewed: February 17, 1998
Article ID: Q142343
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Word for Windows, version 6.0, 6.0a, 6.0c

SYMPTOMS

Text in the Delivery Address box of the Envelopes dialog box or the Labels Address box does not appear correctly.

Example 1

When you select the following text 

   <TAB> John Jones
   123 Main Street
   Anytown, USA
you get the following result in the Delivery Address box: 

   <blank line>
   123 Main Street
   Anytown, USA

Example 2

When you select this text 

   Confidential<tab>Susan Smith
you get the following result in the Delivery Address box: 

   Confidential

CAUSE

Microsoft Word does not recognize the text as a valid address.

WORKAROUND

   http://www.microsoft.com/supportnet/refguide/
Use the following sample macro to work around this problem: 

   Sub MAIN
      Dim dlg As ToolsCreateEnvelope
      GetCurValues dlg
      dlg.EnvAddress = Selection$()
      If Dialog(dlg) Then
         ToolsCreateEnvelope dlg
      EndIf
   End Sub

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been corrected in Word 97 for Windows and Word 98 Macintosh Edition.
