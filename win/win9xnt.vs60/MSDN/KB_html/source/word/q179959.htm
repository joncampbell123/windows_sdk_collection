DMM: Not All ZIP Codes Get a Hyphen Added to Them

 Last reviewed: March 4, 1998
Article ID: Q179959
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SUMMARY

After your address list is verified through the Dial-A-Zip Web site, not all of your ZIP codes may contain a hyphen (so that your ZIP codes are in the format xxxxx-xxxx). Instead, they may be in the format xxxxxxxxx.

MORE INFORMATION

When you type in a ZIP code, you can either use the format xxxxx-xxxx or the format without the hyphen xxxxxxxxx. Regardless of which format you use, Microsoft Direct Mail Manager will format the verified ZIP codes in the format xxxxx-xxxx. If Microsoft Direct Mail Manager is unable to verify an address, it will leave the ZIP code as you typed it.

REFERENCES

For more information about verifying addresses, on the Index tab in Microsoft Direct Mail Manager Help, type "verification," click to select Verification, and then click Display.

Additional information is available through the Microsoft Direct Mail Manager Web site. You can access this Web site by clicking the "Learn More" button on the Welcome screen of Microsoft Direct Mail Manager.
