WD97: Parentheses Don't Display in Result of a Form Field Numbe

 Last reviewed: August 22, 1997
Article ID: Q172962
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you modify the Number Format to include parentheses, within a Text Form Field Options dialog box, where the type of text form field is set to Number, the parentheses will not display in the form field result.

For example, if you want to format a form field to display a phone number formatted as 

   (206) 555-1234
using the number format of 

   (###) ###-####
the result will be 

   2065551234.00

WORKAROUND

Currently there is no workaround for this problem. The parentheses will need to be omitted.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Other symbols, such as the backslash (\), forward slash (/) or hyphen(-) will display in the form field result.
