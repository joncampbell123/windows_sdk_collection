WD97: Voice Dictation Systems Appear To Run Slowly With Word

 Last reviewed: July 31, 1997
Article ID: Q167877
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you use a voice dictation system with Word, the time spent generating and sending keystrokes to Word appears to be slow.

CAUSE

The following Word background options are turned on which causes the apparent slowness:

  • All AutoCorrect Options
  • AutoComplete (AutoText option)
  • All "AutoFormat As You Type" Options
  • Check Spelling As You Type
  • Check Grammar As You Type

WORKAROUND

To increase the speed of the voice dictation system in transferring keystrokes to Word, turn off the background options listed in the "Cause" section of this article.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
