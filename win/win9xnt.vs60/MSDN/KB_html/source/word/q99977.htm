WD: Word Does Produce Color Separations

 Last reviewed: February 2, 1998
Article ID: Q99977
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1 6.0.1a
  • Microsoft Word 98 Macintosh Edition
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

Word does not include a color separation printing feature.

This article contains a method you can use to create spot color separation in a document that contains text formatted for different colors. This method does not, however, provide color separation for color graphics, so color graphics are unaffected.

MORE INFORMATION

Color separation is the process of printing each of the colors in a document as a separate document so that the printer can print each separation using a different-colored ink. Many color printers use the cyan-magenta-yellow-black (CMYK) color model, so color separation often corresponds to this model.

Spot color separation is one type of color separation in which a single color of ink is specified and each page having elements in that color is printed as a separate print job. You can use spot color separation to create either positive or negative color separation.

When you print using a PostScript printer, you can alter the percentage of color used to print spot color separation by modifying halftone settings in the Advanced Options dialog box of the PostScript printer driver.

How to Create Color Separation for Text Only (Not for Graphics)

This procedure describes how to format all of the text in your multiple-color document for white except the text for which you are creating the color separation. Repeat these steps for each color for which you want to produce a color separation.

NOTE: While the file can be closed without saving after printing each color, it may be best to save the file with a new name before starting and use the copy of the file during this process. This will insure that the original file is maintained.

To format your text for color separation, follow these steps:

  1. On the Edit menu, click Replace.

  2. Click to clear the Find What check box. With the insertion point in the Find What box, click Format (in Word 97 or 98 click More to display the Format option), and then click Font.

  3. From the Color list, select one of the text colors for which you are not creating the current color separation. Click OK.

    For example, if you are creating a magenta color separation, select yellow, black, or cyan in the Color list.

  4. Click to clear the Replace With box, if necessary. With the insertion point in the Replace With box, Click Format click Format (in Word 97 or 98 click More to display the Format option), and then click Font.

  5. From the Color list, select White, and then click OK.

  6. Click Replace All and then click Close.

  7. Repeat steps 1 through 6 for each of the colors for which you are NOT creating the current color separation. For example, if you are creating the magenta color separation in a four-color document, repeat steps 1 through 6 once each for yellow, black, and cyan.

  8. Repeat steps 1 and 2. From the Color list, select the text color for which you are creating the current color separation. Click OK.

  9. Repeat step 4.

  10. From the Color list, select Black and then click OK.

  11. Repeat step 6.

  12. Print the document.

  13. Close the document without saving the changes

  14. Reopen the document and repeat steps 1 through 13 to create a color separation for each text color in your document.
