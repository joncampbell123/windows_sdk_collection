WD: User Information Mailing Address Limited to 125 Characters

 Last reviewed: February 18, 1998
Article ID: Q164031
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

After typing in your mailing address on the User Information tab, and then restarting Word, Word may truncate your mailing address to 125 characters.

WORKAROUND

Method 1: Limit the Length of Your Mailing Address

When typing your mailing address information on the User Information tab, limit the length of your mailing address to 125 or fewer characters. To access the User Information tab, on the Tools menu, click Options (Windows) or Preferences (Macintosh).

Method 2: Use an address book entry

Create a new listing in your address book for your return mailing address. This address can be selected in the Word features that may require a return address.

Note: You may want to ensure that the mailing address is deleted on the User Information tab. To access the User Information tab, on the Tools menu, click Options (Windows) or Preferences (Macintosh).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Word allows you to store your mailing address information as part of your default User Information. To access the User Information tab, on the Tools menu, click Options (Windows) or Preferences (Macintosh).

The mailing address information you enter may extend up to 255 characters and will allow Word to use that information as the default return address for envelopes and letters.

During the Word session when the mailing address was entered, Word will use the extended mailing address information, as usual. However, after restarting Word, Word will truncate any mailing address text that exceeds 125 characters.
