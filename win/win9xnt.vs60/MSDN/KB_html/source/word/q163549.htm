WD97: Templates and Wizards Available from Microsoft

 Last reviewed: February 6, 1998
Article ID: Q163549
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Office 97 for Windows

SUMMARY

This article describes the templates, wizards, and different Word Web Wizard free stuff that is available for download from the Microsoft Internet Web site.

Microsoft Word Free Stuff can be found on the Microsoft Web site at: 

   http://www.microsoft.com/officefreestuff/word/
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/

MORE INFORMATION

The following lists the templates, wizards, and other free stuff that is available for use in Microsoft Word 97. The items available from the Microsoft Internet site may differ from what is listed in this article as we are constantly updating and adding to our free offerings.

To access the Word Free Stuff, available on the Microsoft Web site, from Word, use the following steps:

  1. On the File menu, click New.

  2. Click the Web Pages (or Other Documents) tab.

  3. Click the "More Cool Stuff.dot" (or "More Templates and Wizards.dot") icon, and then click OK.

  4. On the "More Cool Stuff" (or "More Wizards and Templates") page, click the "Microsoft Word Web Site" link.

NOTE: Internet functionality may require Internet access and payment of a separate fee to a service provider.
