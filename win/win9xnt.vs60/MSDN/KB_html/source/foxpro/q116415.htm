The information in this article applies to:

  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6
  • Microsoft FoxPro for MS-DOS, versions 2.5, 2.5a, 2.5b, 2.6

Fpsqlcs.exe is a file that contains a five-part technical white paper entitled "FoxPro Client-Server Architecture for Enterprise Database Connectivity" that provides an overview of client-server application development using Microsoft FoxPro and SQL Server.
