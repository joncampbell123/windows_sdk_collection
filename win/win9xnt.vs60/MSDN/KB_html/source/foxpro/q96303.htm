PRB: "Sharing Buffers Exceeded"/"System Resources Exhausted"

 Last reviewed: April 29, 1996
Article ID: Q96303
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

When you are running FoxPro/LAN version 2.0 on a LANtastic network workstation, the following error messages may occur: 

   Sharing Buffers Exceeded

   -or-

   System Resources Exhausted

RESOLUTION

To resolve the problem, do the following:

  1. Increase the number of files in the CONFIG.SYS file to at least 100.

  2. Include a HIMEM.SYS command in the CONFIG.SYS file and add the /NumHandles=128 parameter to the HIMEM.SYS command.

  3. Add /F:5120 /L:500 to the SHARE.EXE command in the AUTOEXEC.BAT file.

  4. Contact Artisoft to make sure you are using the latest version of LANtastic.

  5. Turn off any disk-caching utilities, since these might cause FoxPro to lock up or corrupt files.
