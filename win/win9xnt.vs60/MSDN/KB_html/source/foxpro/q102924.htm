Default Setting for SYSMENU Is AUTOMATIC in FoxPro/Win

 Last reviewed: July 5, 1995
Article ID: Q102924
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a

SYSMENU is set to AUTOMATIC by default in FoxPro for Windows instead of ON as it is in FoxPro for MS-DOS. This design change was implemented because most Windows-based applications constantly have a menu available.

Additional reference words: FoxWin 2.50 2.50a 2.50b 2.60 2.60a
