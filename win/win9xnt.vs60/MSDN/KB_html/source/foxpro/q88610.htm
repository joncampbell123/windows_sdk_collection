FIX: Help Look Up Button Problem with Blank Line Highlighted

 Last reviewed: October 14, 1997
Article ID: Q88610
2.00 MS-DOS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

In a FoxPro 2.0 Help window, a problem may occur when a blank line is selected and you choose the Look Up button.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FoxPro version 2.0. This problem was corrected in FoxPro version 2.0 dated September 26, 1991.

MORE INFORMATION

The details level of the FoxPro Help window has a Look Up button and a See Also popup window. These controls provide links to related Help topics. For example, to see more information about a key word used in the Help details window, select the word and choose the Look Up button. The Look Up button is not enabled unless a word or a portion of the Help text is selected.
