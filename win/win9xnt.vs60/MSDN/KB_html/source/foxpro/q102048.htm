Documentation for DIMENSION ARRAY Doesn't List ALEN()

 Last reviewed: April 17, 1995
Article ID: Q102048
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

In the FoxPro 2.0 Help file and also in the "Commands & Functions" manual, ALEN() was incorrectly omitted from the "See Also" list of commands that are associated with the DIMENSION ARRAY command.

Additional reference words: FoxDos 2.00 ALEN array docerr
