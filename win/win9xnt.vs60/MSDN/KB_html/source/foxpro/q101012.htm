32-Bit Color Picker Displays Only 4 Colors on CGA

 Last reviewed: April 17, 1995
Article ID: Q101012
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

SYMPTOMS

In the 32-bit version of FoxPro 2.5 for MS-DOS, the Color Picker displays only black, white, gray, and blue in the color palette if the machine has a CGA monitor.

WORKAROUND

To work around this problem, you can use the Color Picker in the 16-bit version of FoxPro 2.5 for MS-DOS.

NOTE: This problem affects only the Color Picker. Commands that support color options are unaffected and behave normally.

Additional reference words: FoxDos 2.50 FAR grey 2.50a
