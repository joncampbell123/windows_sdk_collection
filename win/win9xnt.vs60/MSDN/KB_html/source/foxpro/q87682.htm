PRB: Must Change Options to Generate Screen

 Last reviewed: April 29, 1996
Article ID: Q87682
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5 and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

When regenerating a screen program in Microsoft FoxPro, the code options of the Generate window do not save the options from the previous generation of the screen.

RESOLUTION

To save the code options, include the screen in a project.

Additional reference words: FoxDos FoxWin 2.00 2.50 2.50a 2.x
