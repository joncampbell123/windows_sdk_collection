FIX: FoxDoc Does Not Cross-reference Keywords

 Last reviewed: September 22, 1997
Article ID: Q104351
2.50 WINDOWS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.5

SYMPTOMS

FoxDoc does not cross-reference keywords.

STATUS

Microsoft has confirmed this to be a problem in FoxPro 2.5 for Windows. This problem was corrected in FoxPro 2.5a for Windows.

MORE INFORMATION

Steps to Reproduce Problem

  1. Start FoxDoc by choosing FoxDoc from the Program menu.

  2. In the FoxDoc main dialog box, under Other Options, choose the Reports button.

  3. In the Report Options dialog box, choose the Options button next to Cross Reference.

  4. In the Cross Reference Options dialog box, under Cross-Reference Components, select FoxPro Keywords.

  5. Choose OK in the Cross Reference Options dialog box, then choose OK in the Report Options dialog box to return to the FoxDoc main dialog box.

  6. Choose Generate to create the FoxDoc report.

The keywords in the report will not be cross-referenced.
