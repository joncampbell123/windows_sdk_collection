List of Problems Solved by FoxPro 2.6a for MS-DOS

 Last reviewed: May 16, 1996
Article ID: Q128917
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.6a

SUMMARY

Below is a list of the problems in FoxPro version 2.6 for MS-DOS that were corrected in FoxPro version 2.6a for MS-DOS.

MORE INFORMATION

The following list contains the article identification numbers and the titles of Microsoft Knowledge Base articles that fully describe the FoxPro version 2.6 problems that were corrected in FoxPro 2.6a for MS-DOS. For more information about any of these problems, please see the full article.
