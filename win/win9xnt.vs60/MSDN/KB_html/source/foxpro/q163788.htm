INFO: FoxISAPI and Personal Web Server from FrontPage 97

 Last reviewed: March 19, 1997
Article ID: Q163788
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a

SUMMARY

Now that Personal Web Server supports ISAPI, the FoxISAPI sample will run on the Personal Web Server that comes with FrontPage 97.

MORE INFORMATION

The FoxISAPI sample is supported only with Internet Information Server (IIS) on NT 3.51, but will also work with NT 4.0 and FrontPage 97. It is strongly recommended that you download the updated version of the FoxISAPI sample in order to work with FrontPage 97 and NT 4.0.

You can currently download the updated version of the FoxISAPI sample from: 

   http://www.microsoft.com/vfoxpro
and choose the Free Downloads button. The file to download is FOXISAPI.ZIP (Other Samples and Tools).

Follow the steps stated in the first Knowledge Base article listed in the REFERENCES section below to set up FoxISAPI for IIS. For FoxISAPI to work with FrontPage 97, follow the same steps, but install the files to the C:\WEBSHARE\SCRIPTS directory instead of the C:\Inetsrv\Scripts directory for Internet Information Server (IIS).
