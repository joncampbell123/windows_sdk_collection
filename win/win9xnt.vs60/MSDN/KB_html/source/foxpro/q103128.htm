FIX: FoxDoc "Range Check Error" with Screen Summary Report

 Last reviewed: September 22, 1997
Article ID: Q103128
2.50 MS-DOS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.5

SYMPTOMS

If you run the MS-DOS-based version of FoxDoc on screens that contain Windows records (for example, screens that have been modified in FoxPro for Windows), you may occasionally receive a "Range Check Error" when FoxDoc outputs the screen summary.

RESOLUTION

Turn off the screen summary report or document these screens or projects using the Windows-based version of FoxDoc.

STATUS

Microsoft has confirmed this to be a problem in FoxPro 2.5 for MS-DOS. This problem was corrected in FoxPro 2.5a for MS-DOS.
