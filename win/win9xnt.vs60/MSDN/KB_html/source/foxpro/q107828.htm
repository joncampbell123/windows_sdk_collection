Disk Cache Recommendations for FoxPro for Macintosh

 Last reviewed: May 20, 1996
Article ID: Q107828
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b

FoxPro is compatible with the Macintosh operating system cache; however, it is recommended that a minimum amount of memory be allocated to system caching.

FoxPro for Macintosh uses its own internal disk-caching scheme for disk reads and writes. Allocating memory to a system cache duplicates functionality included in FoxPro. The memory used for the cache is not available to FoxPro or other applications currently running.

The optimal cache size may vary among different Macintosh configurations, but should probably be less than 512K. For example, a cache size of 128K seems to be optimal on a Macintosh IIci, but a cache size of 256K seems to be optimal on a Quadra 650.
