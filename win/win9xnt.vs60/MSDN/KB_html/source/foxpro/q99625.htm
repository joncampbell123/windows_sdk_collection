PRB: Report Writer Loses Duplex Printing Setting for HP IIISi

 Last reviewed: June 26, 1995
Article ID: Q99625
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

In the Report Writer, if you select a duplex printing setting of long edge or short edge for the Hewlett-Packard (HP) LaserJet IIISi printer, the setting appears to be saved with the .FRX file. However, when the file is closed and reopened, this setting will be None regardless of your earlier selection.

RESOLUTION

To work around this problem, use the RUN /N CONTROL PRINTERS command to invoke the Printer Driver dialog box, which will allow you to edit the printer driver.

MORE INFORMATION

To select a duplex printing setting in the Report Writer, choose Page Layout from the Report menu, choose Print Setup, choose Options, and then select None, Long Edge, or Short Edge under Duplex Printing.
