Index Buttons in Database Setup Dialog in Different Order

 Last reviewed: June 27, 1995
Article ID: Q101048
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a

The order of the index buttons in the Database Setup dialog box differs between FoxPro for versions 2.0, 2.5, and 2.5a for MS-DOS, and FoxPro versions 2.5 and 2.5a for Windows.

In FoxPro for MS-DOS, the buttons are in the following order: 

   Add
   Modify
   Remove
   No Order (or Set Order)
In FoxPro for Windows, the buttons are in the following order: 

   Add
   Modify
   No Order (or Set Order)
   Remove
