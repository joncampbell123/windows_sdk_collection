FoxPro for MS-DOS No Longer Supports Demand Paging

 Last reviewed: August 28, 1995
Article ID: Q121080
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5x, 2.6, 2.6a

SUMMARY

FoxPro for MS-DOS does not enable demand paging if there is not enough memory to start the Extended (32-bit) version of FoxPro.

MORE INFORMATION

Page 3-7 of the "Installation & Configuration Guide" states: 

   If necessary, the Extended version of FoxPro enables demand paging to
   allow it to run in as little as 1.5 MB.
This sentence is incorrect. Unlike earlier versions of FoxPro, the 2.5x, 2.6, and 2.6a versions of FoxPro for MS-DOS now use the Phar Lap DOS extender, which is DPMI compliant and does not support demand paging.

Additional reference words: FoxDos 2.50 2.50a 2.50b 2.60 2.60a foxprox.exe
