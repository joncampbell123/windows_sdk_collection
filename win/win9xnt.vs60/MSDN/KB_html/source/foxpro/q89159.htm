List of File Extensions Used in FoxPro 1.x for MS-DOS

 Last reviewed: April 17, 1995
Article ID: Q89159
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 1.0, 1.01, 1.02

The following table lists the extensions assigned to files used by FoxPro versions 1.x for MS-DOS. 

   File types                 Extension
   ------------------------------------

   Database                     .DBF
   Database memo file           .FPT
   Memo file backup             .TBK
   FoxBASE+ style memo          .DBT
   Single entry index file      .IDX
   Compound index file          .CDX
   Program                      .PRG
   Compiled program             .FXP
   Format file                  .FMT
   Compiled format              .PRX
   View file                    .VUE
   Text file                    .TXT
   File backup                  .BAK
   Report file                  .FRX
   Report memo                  .FRT
   Label file                   .LBX
   Label memo                   .LBT
   Screen file                  .SCX
   Screen memo                  .SCT
   Generated screen program     .SPR
   Compiled screen program      .SPX
   Menu file                    .MNX
   Menu memo                    .MNT
   Generated menu program       .MPR
   Compiled menu program        .MPX
   Generated query program      .QPR
   Compiled query program       .QPX
   Project file                 .PJX
   Project memo                 .PJT
   Generated application        .APP
   Executable program           .EXE
   Compilation error file       .ERR
   Memory variable save         .MEM
   Macro files                  .FKY
   Window file                  .WIN
   Resource files               FOXUSER.DBF
                                FOXUSER.FPT
   Help files                   FOXHELP.DBF
                                FOXHELP.FPT
   Configuration file           CONFIG.FP
   Temporary file               .TMP
   FoxDoc reports               .DOC
   FoxDoc action diagrams       .ACT
   FoxDoc supporting files      .FXD
   FoxDoc Help files            .HLP
   FoxDoc messages files        .MSG
   Libraries                    .PLB

Additional reference words: FoxDos 1.00 1.01 1.02 1.x
