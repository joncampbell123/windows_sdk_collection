FIX: TTOC Returns More Than 14 Characters

 Last reviewed: December 30, 1997
Article ID: Q135701
3.00 WINDOWS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, version 3.0

SYMPTOMS

TTOC({},1) returns more than 14 characters.

WORKAROUND

Use the following expression to provide a proper 14-character string: 

   IIF(<aDate> =={}, replicate("0",14), TTOC(<aDate>, 1))

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Visual FoxPro 3.0b for Windows.

MORE INFORMATION

Steps to Reproduce Problem

?TTOC({},1) &&"000000000478376201" 

?TTOC(CTOD('  /  /  '),1)   &&"000000000478376201"
