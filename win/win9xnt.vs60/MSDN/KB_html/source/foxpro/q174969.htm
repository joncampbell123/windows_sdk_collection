DOC: Btrieve Driver Does Not Ship with VFP 5.0

 Last reviewed: October 10, 1997
Article ID: Q174969
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a

SUMMARY

Page 5 of the Visual FoxPro 5.x Installation Guide & Master Index manual lists the ODBC drivers that are supplied with Visual FoxPro. The listing of the Btrieve driver is incorrect. Visual FoxPro does not supply a Btrieve ODBC driver. This driver must be obtained from a third-party supplier.

MORE INFORMATION

Below is the list of ODBC drivers from the Visual FoxPro Installation Guide & Master Index manual:

  • FoxPro version 2.x database files
  • Visual FoxPro databases and free tables
  • Microsoft Access database files
  • Paradox[ASCII 174] version 3.x and 4.x tables
  • dBASE III[ASCII 174] and dBASE IV[ASCII 174] files
  • Btrieve[ASCII 174] tables
  • SQL Server database files
  • Microsoft Excel spreadsheets
  • Fixed-width and delimited text files
  • Oracle[ASCII 174] 7 database files

The Btrieve driver is not provided and should not be listed.

REFERENCES

Visual FoxPro Installation Guide & Master Index, Page 5
