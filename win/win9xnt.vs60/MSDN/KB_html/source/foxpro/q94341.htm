SET PRINT ON Does Not Echo Typed Commands to Printer

 Last reviewed: June 27, 1995
Article ID: Q94341
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 1.0, 1.02, 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SUMMARY

Unlike its behavior in FoxBASE+ for MS-DOS, the SET PRINT ON command in FoxPro does not echo your typed command in the Command window to the printer. In FoxPro, only the results from the commands you enter in the Command window are sent to the printer.

MORE INFORMATION

The SET PRINT ON command in FoxBASE+ allows you to keep track of the commands being typed by echoing to the printer commands entered on the Command window.
