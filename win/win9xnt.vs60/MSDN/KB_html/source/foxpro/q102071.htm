PRB: Can't Select Build Application Button in Project Builder

 Last reviewed: June 27, 1995
Article ID: Q102071
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5
  • Microsoft FoxPro for MS-DOS, versions 2.0, and 2.5

SYMPTOMS

The Build Application radio button is disabled. Using the TAB key to move to it or clicking it with the mouse will not select the button.

CAUSE

The project file is probably corrupted. To confirm that this is the problem, open a different project file and choose Build. If the Build Application button is available, the problem relates only to the project file in question.

RESOLUTION

Create a new project file. The Build Application button should again be available.
