Convert To Static Is Not Like Paste Special

 Last reviewed: April 29, 1996
Article ID: Q95680
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

On page U3-9 of the "User's Guide," the last paragraph of the "Convert To Static" section incorrectly states that using the Convert To Static option to change an OLE object to a picture has the same effect as using Paste Special Picture format to embed or link the object.

After Convert To Static is chosen, the OLE object can no longer be edited. Double-clicking the object in the general field no longer invokes the source application to allow you to edit the object.

Double-clicking a Paste Special Picture-formatted embedded or linked object invokes the source application, allowing the embedded or linked object to be revised.
