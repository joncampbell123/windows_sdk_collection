Description of CATMAN = ON/OFF in CONFIG.FP(W)

 Last reviewed: April 30, 1996
Article ID: Q115091
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

SUMMARY

The information below describes the CATMAN = ON/OFF line in the configuration file (usually CONFIG.FPW in FoxPro for Windows and CONFIG.FP in FoxPro for MS-DOS).

This information is also found in FoxPro online Help in the "Configuring FoxPro" topic.

MORE INFORMATION

Starting the Catalog Manager

The CATMAN configuration item is used to specify whether the FoxPro Introductory Screen is displayed and the Catalog Manager launched when you start FoxPro.

Include the following line in your FoxPro CONFIG.FP(W) configuration file to suppress the FoxPro Introductory Screen and launch the Catalog Manager when FoxPro is started: 

   CATMAN = ON
Include the following line to suppress the FoxPro Introductory Screen and the Catalog Manager when FoxPro is started: 

   CATMAN = OFF
If your CONFIG.FP(W) doesn't contain the CATMAN configuration item, the FoxPro Introductory Screen is displayed when you start FoxPro.

Additional reference words: FoxDos FoxWin 2.60 foxhelp.dbf foxhelp.hlp
catman=on
catman=off
