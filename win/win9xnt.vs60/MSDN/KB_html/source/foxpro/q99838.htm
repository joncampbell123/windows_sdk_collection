Using FoxPro 2.0 Library Construction Kit with FoxPro 2.5

 Last reviewed: April 29, 1996
Article ID: Q99838
The information in this article applies to:
  • Microsoft FoxPro Library Construction Kit for MS-DOS, version 2.0
  • Microsoft FoxPro Library Construction Kit for Windows, version 2.5

SUMMARY

The FoxPro Library Construction Kit (LCK) version 2.0 for MS-DOS is a tool used to extend FoxPro functionality with C or assembly language. The FoxPro 2.0 LCK for MS-DOS allows you to access C or assembler code from within FoxPro. The FoxPro version 2.0 LCK for MS-DOS supports only the WATCOM C version 8.5 compiler.

FoxPro version 2.5 for Windows cannot use the .PLB files created with the FoxPro version 2.0 LCK for MS-DOS. The FoxPro version 2.5 LCK for Windows must be used to convert the .PLB files.

MORE INFORMATION

FoxPro version 2.5 for MS-DOS can use libraries (.PLB files) that were created with the FoxPro version 2.0 LCK for MS-DOS if these libraries are first relinked with the library files that are stored in the FOXPRO25\GOODIES\LCKUPDAT subdirectory. This process is described in the LCKUPDAT.TXT file in the LCKUPDAT subdirectory.
