BUG: Can't Use Keyboard to Change Vertical Size of Object

 Last reviewed: May 5, 1997
Article ID: Q135286
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 5.0a

SYMPTOMS

You cannot use the keyboard to change the vertical size of an object in the Report Writer when Snap to Grid is on.

WORKAROUND

Turn off Snap to Grid from the Format menu, or use the mouse to resize the object.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Create a new report.

  2. On the Format menu, click Snap to Grid On.

  3. Place a text (AB) object on the report.

  4. Attempt to resize the object vertically by pressing SHIFT+UP ARROW or SHIFT+DOWN ARROW.
