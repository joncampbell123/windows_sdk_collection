Proper Usage of Order Options Button in RQBE Order By

 Last reviewed: April 29, 1996
Article ID: Q100997
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a

SUMMARY

If the field(s) are not properly selected in the Order By dialog box, using the ORDER BY option in the Relational Query-By-Example (RQBE) window will not order the fields according to the selection (ascending or descending).

To change the order of a field in the Ordering Criteria list:

  1. Select the field to change in the Ordering Criteria list.
2. Select the desired Order radio button (Ascending or Descending).

To add a field to the Ordering Criteria list in a specific order:

  1. Select the field to add from the Selected Output list.
2. Select the desired Order radio button (Ascending or Descending). 3. Choose the Move push button.
