How to Reset FoxPro to Default Colors

 Last reviewed: February 15, 1995
Article ID: Q123582
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.6a

SUMMARY

At times, it may become necessary to set the FoxPro for MS-DOS colors back to their defaults after using some custom colors. Resetting colors to the FoxPro defaults is accomplished with the SET COLOR command.

MORE INFORMATION

How to Set the Colors to the Default

SET COLOR SET TO DEFAULT will use the default FoxPro colors. DEFAULT is one of several color sets that ships with FoxPro.

How to List the Color Sets

To list the color sets that ship with FoxPro for MS-DOS:

  1. Choose the Color ... option from the Window menu.

  2. Press the Load push button. Several color set names should be listed including COLOR, COMPQ_COLO, COMPQ_MONO, DEFAULT, MONOCHROME, T5200_COLO, T5200_MONO, and TOSH_3200.

How to Re-create the Color Sets

If the color sets are missing or the FOXUSER resource file is missing or damaged, it is possible to re-create the color sets. Listed below are three methods you can use to re-create the default color sets:

  1. Use the COLOR SET information listed below to re-create the resource
file 
   by using the COLOR PICKER. For example, to re-create the MONOCHROME
   color set, type the following in the COMMAND WINDOW:

   SET COLOR OF SCHEME 1 to ;
   "W/N,N+/W,W+/N,W+/N,W/N,B+/N,W+/N,N+/N,W+/N,W/N,-"

   SET COLOR OF SCHEME 2 to ;
   "W/N,W+/N,W+/N,W+/N,W/N,N+/W,B+/W,N+/N,W+/N,W/N,-"
   ...
   SET COLOR OF SCHEME 12 to ;
   "W/N,N+/W,W+/N,W+/N,W/N,B+/N,W+/N,N+/N,W+/N,W/N,-"

   Then go into the COLOR PICKER (Window menu, Color ...). Select SAVE, and
   type MONOCHROME as the color set name.

  • Use the COLOR SET information listed below to create several different CONFIG.FP files, one for each COLOR SET. For example, MONO.FP could contain this code:

    COLOR SET = "MONOCHROME" COLOR SCHEME(1) = "W/N,N+/W,W+/N,W+/N,W/N,B+/N,W+/N,N+/N,W+/N,W/N,-" COLOR SCHEME(2) = "W/N,W+/N,W+/N,W+/N,W/N,N+/W,B+/W,N+/N,W+/N,W/N,-" ... COLOR SCHEME(12) = "W/N,N+/W,W+/N,W+/N,W/N,B+/N,W+/N,N+/N,W+/N,W/N,-"

  • Use a program to allow interactive change of COLOR SETs. Be sure to start with a MONOCHROME color set, as all COLOR video cards and monitors can display MONOCHROME, but MONOCHROME video cards and monitors cannot display COLOR. This information could be displayed in a POPUP whose VALID could contain code similar to this:

    DO CASE 

          CASE user_color = 1  && User picked MONOCHROME

             SET COLOR OF SCHEME 1 to
            "W/N,N+/W,W+/N,W+/N,W/N,B+/N,W+/N,N+/N,W+/N,W/N,-"
         SET COLOR OF SCHEME 2 to
            "W/N,W+/N,W+/N,W+/N,W/N,N+/W,B+/W,N+/N,W+/N,W/N,-"
         ...
         SET COLOR OF SCHEME 12 to
            "W/N,N+/W,W+/N,W+/N,W/N,B+/N,W+/N,N+/N,W+/N,W/N,-"
      CASE user_color = 2 && User picked COLOR
       ...
      CASE user_color = 9
      ...
   ENDCASE
   CLEAR  && Need to issue CLEAR to paint the Desktop for the chosen colors
