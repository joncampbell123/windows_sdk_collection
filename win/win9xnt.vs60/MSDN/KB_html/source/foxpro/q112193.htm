PRB: Help Compiler Crashes with Word 6.0 for Windows RTF Files

 Last reviewed: April 30, 1996
Article ID: Q112193
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, and 2.5b

SYMPTOMS

When you are compiling Windows Help files created from Microsoft Word 6.0 for Windows rich-text format (RTF) files, general protection (GP) faults, violation of system integrity errors, or exception errors can occur, or the Help file may be compiled using only the Windows system font.

RESOLUTION

Use the updated Help compiler, version 3.10.505.

MORE INFORMATION

To obtain version 3.10.505 of the Help compiler, download VBHC505.EXE, a self-extracting file, from the Microsoft Software Library (MSL) on the following services:

  • Microsoft Download Service (MSDL) 

          Dial (206) 936-6735 to connect to MSDL
      Download VBHC505.EXE (size: 231170 bytes)
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the \SOFTLIB\MSLFILES directory
      Get VBHC505.EXE (size: 231170 bytes)

NOTE: This file was previously named HC505.EXE.

For more information about using the MSDL, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q104331
   TITLE     : How to Download FoxPro Files from the MSDL
