DOCERR:TeachText Replaced by SimpleText in FoxPro for Mac Docs

 Last reviewed: January 10, 1997
Article ID: Q155808
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SUMMARY

In the "Getting Started" manual for FoxPro 2.6 for the Macintosh, the manual instructs you to use TeachText, which comes with System 7, to add graphics to a table. In System 7.1.2, Apple introduced SimpleText as the text editor for the operating system.

MORE INFORMATION

The "Getting Started" manual for FoxPro for the Macintosh, Chapter 9 "Working with Your Own Table," instructs the user to use TeachText for inserting Graphics into a general field. As of version 7.1.2 of the operating system for the Macintosh, TeachText has been replaced by SimpleText. Instead of using TeachText to insert graphics as instructed in the "Adding Graphics to a Table" section, use SimpleText.

REFERENCES

Microsoft FoxPro Relational Database Management System for Apple Macintosh, version 2.6a, "Getting Started"
