PRB: Expression Builder Dialog Box Disappears in Windows 3.0

 Last reviewed: June 27, 1995
Article ID: Q100566
This article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

When FoxPro for Windows is used under Windows 3.0, the Expression Builder dialog box disappears when it is first accessed through the View window.

RESOLUTION

The Expression Builder dialog box behaves properly when it is accessed again.

MORE INFORMATION

Steps to Reproduce Problem

  1. Open the View window.

  2. Open one to three databases.

  3. Choose Setup.

  4. Choose Filter Data.

  5. Click the title bar of the Expression Builder dialog box to try to move it, or click any popup; the dialog box disappears behind the stack.

  6. Press ESC to reach the Setup dialog box.

  7. Choose Filter Data.

    The Expression Builder dialog box is now stable.
