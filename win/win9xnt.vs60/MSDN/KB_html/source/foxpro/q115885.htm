Incorrect Default Value of 256 Listed for MVCOUNT

 Last reviewed: June 27, 1995
Article ID: Q115885
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

SUMMARY

In the "Installation and Configuration" manual and in online Help, the incorrect default value of 256 is listed for MVCOUNT. The correct default value is 1024.

MORE INFORMATION

The default value for MVCOUNT was increased from 256 in FoxPro 2.5 to 1024 in FoxPro 2.6.

Locations of Documentation Errors

  • FoxPro for Windows "Installation and Configuration" manual, version 2.6, pages 2-13, 2-15
  • FoxPro for MS-DOS "Installation and Configuration" manual, version 2.6, pages 2-15, 2-18
  • FoxPro 2.6 for Windows (FOXHELP.DBF and FOXHELP.HLP) online help, "Configuring FoxPro" topic, under the "Special CONFIG.FPW Settings" section
  • FoxPro 2.6 for MS-DOS online help, "Configuring FoxPro" topic, under the "Special CONFIG.FP Items" section
