BUG: "Hide All Windows" Does Not Hide the Command Window

 Last reviewed: May 6, 1997
Article ID: Q129905
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 5.0a

SYMPTOMS

Holding down the SHIFT key as you choose Hide All from the Window menu does not hide the Command window or the Standard toolbar. However, holding down the SHIFT key as you choose Close All from the File menu does close all windows including the Command window.

RESOLUTION

To hide the Command window, press the CTRL+ALT+SHIFT keys, which hides all system windows. You can also choose Close form the File menu to close the selected window.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
