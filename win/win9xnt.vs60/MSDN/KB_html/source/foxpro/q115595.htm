List of Problems Solved by FoxPro 2.5c for Macintosh

 Last reviewed: May 16, 1996
Article ID: Q115595
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5c

SUMMARY

Below is a list of the problems in FoxPro version 2.5b for Macintosh that were corrected in FoxPro version 2.5c for Macintosh.

MORE INFORMATION

This list contains the numbers and titles of Microsoft Knowledge Base articles that fully describe the problems corrected in FoxPro 2.5c for Macintosh:

Knowledge Base Articles

 Q112520   PRFOX: Freeze or Crash When Printing to LaserWriter Select 300
 Q114964   PRFOX: Screen Prompts for Location of Push Button Picture
 Q115145   PRFOX: Insufficient Memory After Displaying List Boxes
 Q115290   PRFOX: After FLUSH, Loss of Power May Still Corrupt Table
 Q115332   PRFOX: "Type 1" Error While Printing to StyleWriter II
 Q115836   PRFOX: Screen Generated, "Error In SCX: Objtype=1 Not Found"

Additional reference words: FoxMac 2.50b 2.50c buglist2.50b fixlist2.50c
