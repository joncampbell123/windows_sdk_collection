PRB: Invalid Path Message During SETUP /A on Novell Network

 Last reviewed: June 27, 1995
Article ID: Q98684
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5

SYMPTOMS

When you are trying to install FoxPro version 2.5 for Windows on a Novell network using SETUP /A, the error message "The path you specified for FoxPro does not exist or does not begin with a drive letter" occurs.

This error message will occur even if a valid drive letter/directory designation has been entered on the location screen in the Setup program.

CAUSE

You do not have CREATE, READ, WRITE, and DELETE rights to the chosen installation location.

RESOLUTION

Have your network administrator perform the installation with the SETUP /A option. The administrator should have CREATE, READ, WRITE and DELETE rights to the directory needed for installation.

STATUS

This behavior is by design.
