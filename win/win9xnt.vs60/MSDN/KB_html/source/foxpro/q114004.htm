PRB: File GSCHAP4.QPR or GSCHAP5.QPR Does Not Exist

 Last reviewed: May 29, 1996
Article ID: Q114004
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b

SYMPTOMS

The file GSCHAP4.QPR, which page 38 of the "Getting Started" manual instructs you to open, does not exist.

-or-

The file GSCHAP5.QPR, which page 54 of the "Getting Started" manual instructs you to open, does not exist.

CAUSE

These files are not shipped with FoxPro. They are created by the user while doing the lessons in Chapter 4 (GSCHAP4.QPR) or Chapter 5 (GSCHAP5.QPR) of the "Getting Started" manual.

RESOLUTION

Follow the instructions in Chapter 4 to create GSCHAP4.QPR.

-or-

Follow the instructions in Chapter 5 to create GSCHAP5.QPR.

STATUS

This behavior is by design.
