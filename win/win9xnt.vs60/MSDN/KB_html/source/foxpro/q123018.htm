WindowShade Allows User-Defined Windows to Be Minimized

 Last reviewed: May 16, 1996
Article ID: Q123018
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SUMMARY

WindowShade, a Control-Panel utility available with Apple's System 7.5 operating system, allows FoxPro user-defined windows to be minimized.

NOTE: WindowShade is manufactured by a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

MORE INFORMATION

Page 369 of the Language Reference while describing the DEFINE WINDOW command states that user-defined windows cannot be minimized in FoxPro for the Macintosh because the MINIMIZE clause is ignored. However, because FoxPro implements true Macintosh windows, you can use Apple's WindowShade system-level utility to minimize any FoxPro window.

Clicking the window's title bar a certain number of times, as specified in WindowShade's options, causes any FoxPro window to roll up and display just a title bar. In effect, this minimizes the window. WindowShade is located in the Control Panels folder.
