DOC: Unbalanced Quotation Mark in APPEND GENERAL Help Topic

 Last reviewed: January 8, 1997
Article ID: Q136673
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.6, 2.6a

SUMMARY

The APPEND GENERAL example in the Help file contains a syntax error: 

   APPEND GENERAL chart FROM CHART1.XLC" CLASS EXCELCHART
This documentation error has been fixed in Visual FoxPro 5.0 for Windows.

MORE INFORMATION

The correct syntax is: 

   APPEND GENERAL chart FROM CHART1.XLC CLASS EXCELCHART
Note the absence of the unbalanced quotation mark.
