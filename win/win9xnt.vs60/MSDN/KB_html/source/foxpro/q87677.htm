PRB: Files with .TMP Extension Left on System

 Last reviewed: June 27, 1995
Article ID: Q87677
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5 and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

A number of files with the .TMP file extension remain on the system after Microsoft FoxPro has been run.

CAUSE

There are two causes for these files to remain on the system:

  • The machine was booted without first closing FoxPro.
  • The user does not have delete privileges in the directory in which FoxPro creates the temporary files.

RESOLUTION

Exit FoxPro and delete any .TMP files that remain.

MORE INFORMATION

Every time FoxPro loads, it creates a number of temporary files with the .TMP file extension. FoxPro automatically deletes these files during a normal shutdown.
