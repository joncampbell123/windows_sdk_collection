Using VM/386 with FoxPro 2.5 for MS-DOS

 Last reviewed: April 17, 1995
Article ID: Q99550
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

SUMMARY

IGC Software has confirmed that FoxPro versions 2.5 and later for MS-DOS will not run correctly with VM/386 version 2.01. If you are are using version 2.2 or earlier of VM/386, call IGC at (408) 441-0366 and request a product upgrade.

MORE INFORMATION

VM/386 is a multitasking environment that sets up virtual machines on the computer. These virtual machines are true MS-DOS sessions can have their own AUTOEXEC.BAT and CONFIG.SYS files. There are both single-user and multiuser versions of VM/386.

FoxPro version 2.0 for MS-DOS has no problems running in this environment; however, some changes have been made to FoxPro versions 2.5 and later that require the newest version of VM/386, which is 2.2.
