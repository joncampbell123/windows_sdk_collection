SAMPLE: Msgrp5cg.exe Graph Wizard Data Series Display Changes

 Last reviewed: February 19, 1998
Article ID: Q157225
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b
  • Microsoft FoxPro for Windows, version 2.6a

SUMMARY

Msgrp5cq.exe is a sample that shows how the default data representation changed in the Microsoft FoxPro Graph Wizard between Microsoft FoxPro 2.6x for Windows and Microsoft Visual FoxPro 3.0x for Windows.

In the sample, section A details the steps to create a sample graph in FoxPro 2.6x using the data from a FoxPro 2.6x sample table named Detail.dbf (the file is included in the SoftLib files available for download).

Then, in section B, the same graph is created in Visual FoxPro 3.0x using the same data to show the difference in display that is caused by a default choice in Series-in-Columns / Series-in-Rows, from MS Graph5's Data menu.

Finally a component of the default choice made within Microsoft Graph5, which is stored in a file named Default.gra that is saved in the default Windows subdirectory, also impacts the display process. Steps are shown that allow a new default to be saved.

The Grp5_beh.doc file is a Microsoft Word 95 document that details MS Graph5 behaviors and changes. This document contains many screen images.

The Detail.dbf and .cdx files provide the data upon which the queries and graphs are based.

The F26det.qpr and F30det.qpr files are the queries for FoxPro 2.6 and FoxPro 3.0, respectively.

The Fpw26gph.dbf and .fpt files contain the general field, which holds the FoxPro 2.6 graph.

The Vfpgraph.dbf and .fpt files contain the general field, which holds the FoxPro 3.0 graph.

Summary of File Use

In FoxPro 2.6 run the F26det.qpr file and follow the procedures in section A of Grp5_beh.doc. In Visual FoxPro 3.0 run the F30det.qpr file and follow documentation in section B.
