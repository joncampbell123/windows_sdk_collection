ON KEY LABEL RIGHTMOUSE Not Supported in FoxPro for Mac

 Last reviewed: June 3, 1996
Article ID: Q118847
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c

SUMMARY

The ON KEY LABEL RIGHTMOUSE command is not supported in FoxPro for Macintosh. This behavior is documented in the "FoxPro Readme" file in "Part 5: Notes About FoxPro for Macintosh" under the "Change to the ON KEY LABEL Command" section.

NOTE: ON KEY LABEL RIGHTMOUSE is supported in Visual FoxPro for Macintosh.

MORE INFORMATION

Page 20 of the "Installation and Macintosh Features Guide" and page 577 of the "Language Reference" manual for FoxPro version 2.5 for Macintosh incorrectly state that the ON KEY LABEL RIGHTMOUSE <command> can be accessed by holding down the CTRL key while clicking the mouse button (CTRL+click) or by holding down the COMMAND+OPTION keys while clicking the mouse button.

REFERENCES

Microsoft FoxPro for Macintosh "FoxPro Readme" file on Disk - 1 Setup or installed in the FoxPro folder
