SAMPLE: 1ToMany.exe Form for One-to-Many Order Entry App

 Last reviewed: February 20, 1998
Article ID: Q138041
3.00 5.00 WINDOWS kbprg kbfile kbwebcontent

The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, versions 3.0, 5.0

SUMMARY

1ToMany.exe is a sample that shows the minimum key components needed to create a customer lookup screen in Visual FoxPro. It is designed to help you understand components, properties, and event code.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library:

~ 1ToMany.exe

For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
There is a Readme.txt file included in the 1ToMany.exe sample that gives detailed information.

All the required files have been placed in a project file. You can use the sample project and form as a reference, or you can modify them and incorporate them into your own application.
