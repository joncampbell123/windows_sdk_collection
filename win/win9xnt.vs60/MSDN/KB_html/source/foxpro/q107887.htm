FIX: List of Problems Solved by FoxPro 2.5b for Windows

 Last reviewed: October 14, 1997
Article ID: Q107887
2.50b WINDOWS kbref kbfixlist

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.5b

SUMMARY

The following is a list of the problems in FoxPro versions 2.5 and 2.5a for Windows that were corrected in FoxPro version 2.5b for Windows.

MORE INFORMATION

This list contains the numbers and titles of Microsoft Knowledge Base articles that fully describe the problems corrected in FoxPro 2.5b for Windows.

Knowledge Base Articles

 Q107364 - COPY FILE Fails with Error "File Does Not Exist"
 Q107365 - Can't Delete Memo Field in Browse Window
 Q107395 - GPF with Zoom In/Out and SPACEBAR in Page Preview
 Q107396 - FoxPro Branded with "Engineering Prototype"
 Q107397 - GPF or Crash When Indexing a Long Numeric Calculation
 Q107399 - EVALUATE() Causes Error Message in Group Expression
 Q107514 - "Invalid Date" Error When Using a Between Operator
 Q107515 - Accented Characters in Field Names Garbled
 Q107516 - FoxPro Not Releasing System Resources or Memory
 Q107517 - SetupWizard Generates Extra Zeros in COMPRESS Command
 Q107518 - ON SHUTDOWN Does Not Work When a READ Is Active
 Q107520 - Memo Field Corruption Occurs with Code Page Differences
 Q107521 - WAIT WINDOW Forces FoxPro to Be Active Application
 Q107555 - Report Variables Return Incorrect Values
