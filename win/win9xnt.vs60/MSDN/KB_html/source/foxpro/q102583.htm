Summary Band in Report Can't Exceed One Page

 Last reviewed: June 27, 1995
Article ID: Q102583
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5

Page U14-8 in the FoxPro for Windows "User's Guide" states the following: 

   The Summary is one or more pages which appear after the main
   report.
This is incorrect; the Summary band can be only one page long.

NOTE: In FoxPro for MS-DOS, the Summary band can be more than one page long.
