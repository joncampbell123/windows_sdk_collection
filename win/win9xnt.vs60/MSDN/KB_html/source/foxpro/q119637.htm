KB Articles Included with FoxPro for Macintosh DK

 Last reviewed: May 17, 1996
Article ID: Q119637
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c

SUMMARY

KBVIEW.APP is a utility that provides access to 21 Microsoft Knowledge Base articles. It is provided with the Microsoft FoxPro Distribution Kit versions 2.5b and 2.5c for Macintosh and is installed in the root FoxPro for Macintosh installation folder.

MORE INFORMATION

The following is a list of the articles provided in KBVIEW.APP along with their corresponding Microsoft Knowledge Base article numbers (in order by article number):

   Q105600 Cannot Move Main FoxPro for Macintosh Screen
   Q105634 Records from Other Platforms Remain in .SCX/.FRX
   Q105837 How to Use XCMDs and XFCNs in FoxPro for Macintosh
   Q105838 FoxPro for Macintosh Does Not Use Templates for App
   Q107417 "Error Creating OLE Object" with APPEND GENERAL
   Q107824 Orientation and Paper Settings Not Transported
   Q107825 CAPSLOCK() Will Not Toggle Setting of CAPS LOCK Key
   Q107828 Disk Cache Recommendations for FoxPro for Macintosh
   Q107832 Maximum Number of Pixels on Report Grid Differs
   Q108094 Using AppleScript with FoxPro for Macintosh
   Q108097 Compilers Supported by FoxPro LCK for Macintosh
   Q108199 Changes Required After Migrating Screen from FoxBASE+/Mac
   Q108200 Requirements to Create External Library (.MLB) Files
