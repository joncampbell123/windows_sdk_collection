DOCERR: System Requirements for FoxPro Unclear

 Last reviewed: May 14, 1995
Article ID: Q129919
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.6

SUMMARY

Page 1-2 of the FoxPro for MS-DOS version 2.6 Installation and Configuration Guide, states that 640K of RAM is required to run the standard version. This number is the normal conventional RAM available on 8088 machines and higher.

This does not specify the amount of RAM that must be free in order to run FoxPro for MS-DOS version 2.6. However, in Chapter 3, "Optimizing Your System," page 3-2 under the conventional memory paragraph states that FoxPro requires the 512K of conventional memory and at least 420K after any TSR (terminate and stay resident) program is loaded.

MORE INFORMATION

On 8088 or higher machines, most MS-DOS programs require the use of the normal conventional memory of 640K. When network or other device drivers (such as mouse drivers) are loaded, the amount of free RAM is decreased. Therefore, Microsoft recommends that you have to have a minimum of 512K free conventional memory prior to starting FoxPro, especially on 8088 and 286 computers.
