PRB: Problem with MACH32 Video Driver from Gateway 2000

 Last reviewed: June 27, 1995
Article ID: Q95703
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5

Summary:

SYMPTOMS

The error message "General Protection Fault in module MACH32.DRV at 

   0003:0E22" occurs when the MACH32 video drivers from Gateway 2000
   are used.

CAUSE

This is caused by a conflict between the MACH32 video drivers and 

   FoxPro version 2.5 for Windows.

RESOLUTION

Contact Gateway 2000 for the most recent version of the MACH32.DRV 

   video driver. Until you receive the new driver, use the VGA.DRV or
   the 8514.DRV driver that ships with Windows. For more information
   on installing a Windows video driver, see the "Microsoft Windows
   User's Guide."
