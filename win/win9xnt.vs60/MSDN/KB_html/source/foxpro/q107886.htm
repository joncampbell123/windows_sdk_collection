FIX: List of Problems Solved by FoxPro 2.5b for MS-DOS

 Last reviewed: October 14, 1997
Article ID: Q107886
2.50b MS-DOS kbref kbfixlist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.5b

SUMMARY

The following is a list of the problems in FoxPro versions 2.5 and 2.5a for MS-DOS that were corrected in FoxPro version 2.5b for MS-DOS.

MORE INFORMATION

This list contains the numbers and titles of Microsoft Knowledge Base articles that fully describe the problems corrected in FoxPro 2.5b for MS-DOS.

Knowledge Base Articles

 Q88733  - "Invalid Date" Errors with Dates in FoxPro RQBE
 Q107365 - Can't Delete Memo Field in Browse Window
 Q107397 - GPF or Crash When Indexing a Long Numeric Calculation
 Q107398 - SET HELP TO Causes Browse Window to Close
 Q107419 - Russian Code Page Not Properly Mapped
 Q107511 - Corrupt Memo and General Fields in 16-Bit FoxPro
 Q107512 - SHIFT or CTRL Key Removes Second Line of Wait Window
 Q107513 - MODIFY MEMO Command Ignores Code Page
 Q107519 - SET REFRESH Incorrectly Moves Pointer in Child Browse
 Q107522 - Erroneous Messages Generated When Opening a .TXT File
