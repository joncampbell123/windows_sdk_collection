DOCERR: MousePointer Topic Missing Setting 99 (Custom)

 Last reviewed: October 29, 1996
Article ID: Q157190
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 5.0

SUMMARY

The Visual FoxPro 5.0 Help for the MouseIcon property (in the Remarks section) states the following: 

   To display the mouse icon, set the MousePointer property to
   99 (Custom).
However, this setting is not mentioned in the Help topic for the MousePointer property.

MORE INFORMATION

In the Settings section for ntype in the MousePointer property Help topic, the following setting should be added: 

   Setting     Description
   -----------------------

     99        Custom
