How to Suppress Screen Output from TYPE Command

 Last reviewed: June 27, 1995
Article ID: Q108089
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, and 2.5b
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, and 2.5b

The TYPE command in FoxPro echoes the output to the current active window, which is the FoxPro desktop window by default. For example, the TYPE C:\config.sys TO PRINTER command sends the CONFIG.SYS text file to the printer and simultaneously displays the contents of the CONFIG.SYS file on the screen.

To prevent the contents of the text file from being displayed on the desktop, use the following commands: 

   DEFINE WINDOW test FROM 1,1 to 10,10    && The window's coordinates are
   ACTIVATE WINDOW test NOSHOW             && not important.
   TYPE c:\config.sys TO PRINTER
   RELEASE WINDOW test
