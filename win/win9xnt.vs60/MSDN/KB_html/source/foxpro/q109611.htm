Error in Windows-Style Help for SET PRINTER Command

 Last reviewed: June 27, 1995
Article ID: Q109611
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b

SUMMARY

When you are using Windows-style Help in FoxPro for Windows, the help topic for the SET PRINTER command does not mention that the SET PRINTER TO \\<machine name>\<printername> = <dest> and SET PRINTER TO \\SPOOLER commands are for use only in FoxPro for MS-DOS.

However, if you are using MS-DOS-style help, these commands are correctly referenced as MS-DOS-only commands.

MORE INFORMATION

To find the documentation error, do the following:

  1. Start FoxPro for Windows.

  2. In the Command window, issue the following command: 

          SET HELP TO

  3. From the Help menu, choose Search For Help On.

  4. In the Search dialog box, type "set printer" (without the quotation marks) and choose OK, and then choose the Go To button to go to the "SET PRINTER Commands" topic.

  5. In the "SET PRINTER Commands" topic, scroll down to \\<machine name>\<printername> = <dest> and \\SPOOLER. The fact that these clauses are for use only with FoxPro for MS-DOS is not mentioned.

To see the correct documentation, do the following:

  1. Start FoxPro for Windows.

  2. In the Command window, issue the following command: 

          SET HELP TO C:\FOXPROW\FOXHELP.DBF

  3. In the Command window, issue the following command: 

          HELP SET PRINTER

  4. Scroll down to \\<machine name>\<printername> = <dest> and \\SPOOLER. The documentation for each clause states that the clause is for use with FoxPro for MS-DOS only.
