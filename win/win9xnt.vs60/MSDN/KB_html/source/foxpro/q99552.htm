Ensuring Query-Progress Thermometer Is Visible

 Last reviewed: April 29, 1996
Article ID: Q99552
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

A query (.QPR) executing in Microsoft FoxPro for Windows displays a progress thermometer. This thermometer is not visible unless TALK is set on.

The progress thermometer is a good visual indication of processing activity. Without it, the machine may appear to be locked up. To ensure that the thermometer is visible, add the following code to the SETUP code snippet of the screen: 

   #SECTION 2
   SET TALK ON

Additional reference words: FoxWin 2.50 2.50a rqbe hung
