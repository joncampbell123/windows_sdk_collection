PRB: System Hangs If "? CHR(7)" Command Issued or Bell Is Rung

 Last reviewed: April 17, 1995
Article ID: Q88287
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

In FoxPro, ringing the system bell or issuing the "? CHR(7)" command (without the quotation marks) causes the system to hang.

CAUSE

The CONFIG.SYS file contains the DNDEVICE.SYS device driver, that is provided with the Direct Access menu program which is created by the Fifth Generation Systems, Inc. This device driver disables the CTRL+C key combination.

RESOLUTION

For more information, contact Fifth Generation Systems, Inc.
