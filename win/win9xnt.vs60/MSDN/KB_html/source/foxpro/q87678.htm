PRB: "String Too Long to Fit" Error Importing Data

 Last reviewed: June 27, 1995
Article ID: Q87678
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

In the process of importing a data file into Microsoft FoxPro using the IMPORT or APPEND FROM commands, a "String too long to fit" error occurs.

RESOLUTION

Modify the data file to insert a carriage return/linefeed pair after each string of data.

Additional reference words: FoxDos FoxWin 2.00 2.50 2.50a line feed CR/LF
