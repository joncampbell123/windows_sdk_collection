No Close Box in Page Preview Dialog Box in Report Writer

 Last reviewed: June 3, 1996
Article ID: Q108322
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b

In FoxPro for Macintosh, the Page Preview dialog box in the Report Writer does not have a close box. (When present, the close box is located in the upper-left corner of a dialog box or window.) However, FoxPro for Windows and FoxPro for MS-DOS have a Control-menu box and a close box, respectively, in the Page Preview dialog box.

NOTE: This behavior does not occur in Visual FoxPro for Macintosh.

Additional reference words: FoxMac 2.50b
