SYS(1001) Is a Valid FoxPro for UNIX Function

 Last reviewed: August 28, 1995
Article ID: Q120955
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

The online Help file in FoxPro for UNIX incorrectly states the following about the SYS(1001) function: 

   Not applicable to UNIX. Returns 0.
The SYS(1001) function has been implemented in FoxPro for UNIX. This function returns the amount of memory available to FoxPro upon startup.

This number is affected by the setting of the MEMLIMIT command in the config.fpu file.

MORE INFORMATION

For more information about the MEMLIMIT command, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q120954
   TITLE     : MEMLIMIT Command Can Be Used in FoxPro/UNIX Config.fpu
