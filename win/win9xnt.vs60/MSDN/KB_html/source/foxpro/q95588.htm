Suppressing Window Title in MODIFY FILE or MODIFY COMMAND

 Last reviewed: June 27, 1995
Article ID: Q95588
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

The MODIFY COMMAND and MODIFY FILE commands open an editing window to allow changes to be made to a program or other text file. The name of the file being edited appears as the window title.

If you want to suppress the filename from appearing, define a window with no title, activate this window, and then modify the file in the new window.

Example

   DEFINE WINDOW test FROM 1,1 TO 10,75 TITLE ""
   ACTIVATE WINDOW TEST
   MODIFY COMMAND myprog WINDOW test
