BUG: Startup Screen Displays Incorrect Version Number for FPM

 Last reviewed: May 17, 1996
Article ID: Q120247
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

The screen that is displayed briefly as FoxPro starts reports that FoxPro is "Version 2.6" instead of "Version 2.6a."

STATUS

Microsoft has confirmed this to be a problem in FoxPro version 2.6a for Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Although the startup screen displays the wrong version number, the FoxPro logo displayed in the main FoxPro screen shows the correct version number.

Also, the correct version number is displayed when either of the following actions is performed:

  • Type "? VERSION(1)" (without the quotation marks) in the Command window.

    -or-

  • From the Apple menu, choose About Microsoft FoxPro.
