List of Visual FoxPro Files Installed Under Windows

 Last reviewed: October 18, 1996
Article ID: Q133741
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SUMMARY

This article lists the files installed by the Professional Edition of Visual FoxPro when Setup is run under Windows version 3.1 and Windows for Workgroups.

Visual FoxPro is a 32-bit application and requires a 32-bit operating environment. When the setup program is run under Windows or Windows for Workgroups, the WIN32S subsystem is installed.

MORE INFORMATION

Visual FoxPro must be able to add, modify, rename, and/or delete the files listed in this article. If the files already exist on the destination drive, the Setup program uses the following method to determine whether to replace the existing file.

  1. It compares version information. If the Visual FoxPro version of the file is newer, the Setup program replaces the existing file.

  2. If the existing file has no version stamp and the Visual FoxPro file does have version information, the Visual FoxPro file replaces the existing file.

  3. If the existing file has no version stamp and the Visual FoxPro file does not have version information, Visual FoxPro compares the file dates to determine if the file should be updated.

Directories Used by the Setup Program

Microsoft Visual FoxPro version 3.0 must have read, write, create, and delete access to these directories:

\WINDOWS \WINDOWS\SYSTEM \WINDOWS\SYSTEM\WIN32S \WINDOWS\MSAPPS\MSGRAPH5 \WINDOWS\MSAPPS\PROOF \VFP or wherever Visual FoxPro resides.
