@ ... EDIT Command Accesses Only First Array Element

 Last reviewed: June 27, 1995
Article ID: Q90552
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

When an application uses the @ <r,c> EDIT command with an array, only the first array element is available.

The following sample code demonstrates the @ <r,c> EDIT command: 

   CLEAR ALL
   CLOSE ALL
   SET TALK OFF
   DIMENSION info(4)
   info(1)="first"
   info(2)="second"
   info(3)="third"
   info(4)="fourth"
   @2,10 EDIT info SIZE 4,6
   READ

Additional reference words: FoxDos FoxWin 2.00 2.50 2.x 2.50a
