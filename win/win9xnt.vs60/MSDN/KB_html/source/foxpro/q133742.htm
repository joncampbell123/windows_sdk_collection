BUG: Toolbar Options Positioned Off the Screen

 Last reviewed: May 5, 1997
Article ID: Q133742
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 5.0a

SYMPTOMS

Toolbar options are inaccessible because they are positioned too far to the right and off the screen.

CAUSE

Visual FoxPro's toolbar engine is designed to check if any portion of a toolbar will fit on the current row, if not, it opens a new row. Visual FoxPro does not check to see if an entire toolbar will be visual to the user when docked.

WORKAROUND

Visual FoxPro toolbars are moveable and dockable. Position or dock the toolbar in another location on the screen, or change the video driver setting to a higher video resolution so that a larger portion of the toolbar will be visible when docked.

NOTE: The command 

   MOVE WINDOW "<toolbar name>" CENTER
will position the toolbar in the center of the FoxPro desktop only if the toolbar is not docked. It will not move the toolbar if it is docked.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Start Visual FoxPro.

  2. On the View menu, click Toolbars.

  3. Select these toolbars:

    Database Designer Form Designer Print Preview Layout

  4. Attempt to dock each toolbar at the top right corner of the screen. Notice that Visual FoxPro allows a toolbar to dock even though all of its options are not accessible.
