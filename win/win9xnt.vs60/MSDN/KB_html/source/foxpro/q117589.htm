ASLM Licensing Information

 Last reviewed: June 1, 1996
Article ID: Q117589
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c
  • Microsoft Visual FoxPro for Macintosh, version 3.0b

When you are distributing an application created with FoxPro for Macintosh, you may need to distribute the Apple Shared Library Manager (ASLM) with the application. For example, if the application calls Macintosh libraries created with the FoxPro for Macintosh Library Construction Kit (LCK), the Apple Shared Library Manager must be distributed with the application in order to take advantage of these features.

To legally distribute the ASLM with FoxPro for Macintosh applications, you must obtain a license from Apple Computer. To apply for a license, call the Apple Legal Department at (408) 974-4667, or send a request to the sw.license alias on AppleLink.
