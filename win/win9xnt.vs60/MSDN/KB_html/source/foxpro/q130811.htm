The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, version 3.0

Instructions

Microsoft FastTips is available 24 hours a day, 7 days a week, from a touch- tone telephone. To order items from this catalog, first select the items you want to receive, noting the five- to six-digit number of the Item ID shown below for each item, and then:

  • Dial the toll-free FastTips for Development Tools number (800) 936-4300.
  • Press six (6) on your phone keypad to select Microsoft FoxPro.
  • Press one (1) to select Microsoft Visual FoxPro and other FoxPro for Windows versions.
  • Press one (1) to select Microsoft Visual FoxPro.
  • Press one (1) to order an item from the FastTips Catalog.
  • Press one (1) to select fax as your delivery method.
  • When prompted, enter the number of the Item ID and press #, for up to five items.
  • When prompted, enter your three-digit area code and seven-digit fax number, and then press #.
  • To add personal ID (such as your name) to the fax sheet, press 1, otherwise, press *.

When finished, simply hang up. If you have problems receiving a fax, please call (425) 635-3199.
