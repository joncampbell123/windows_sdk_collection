FIX: "Insufficient Memory" When Selecting Fields in RQBE

 Last reviewed: September 22, 1997
Article ID: Q122360
2.60 2.60a MS-DOS kbtool kbbuglist kbfixlist kberrmsg

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, versions 2.60, 2.6a

SYMPTOMS

When running FoxPro for MS-DOS in standard or 16-bit mode, you receive the error message "Insufficient Memory" in RQBE after selecting the "Selected Fields" check box.

RESOLUTION

The error message only occurs in standard mode. To work around it, run FoxPro in extended mode or upgrade to FoxPro version 2.6a for MS-DOS.

STATUS

Microsoft has confirmed this to be a problem in FoxPro version 2.60 for MS-DOS. This problem was corrected in FoxPro version 2.6a for MS-DOS.
