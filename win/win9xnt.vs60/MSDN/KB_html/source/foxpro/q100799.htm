Behavior of ALT+Hot Keys in RQBE Window

 Last reviewed: April 29, 1996
Article ID: Q100799
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for MS-DOS, versions 2.5x, 2.6, 2.6a

All the RQBE menu popup commands are exact duplicates of the buttons and hot keys already available in the RQBE window itself.

Because FoxPro version 2.5 (and later versions) is now available on more than one platform, the hot keys were designed this way in order to be consistent between the different platform versions of FoxPro 2.5.

For example, the RQBE menu pad has a hot key of "Q" and so does the Do Query button in the RQBE window. To execute a query, you can press Q, ALT+Q, or CTRL+Q while in the query screen.
