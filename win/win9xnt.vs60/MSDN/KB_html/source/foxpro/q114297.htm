BUG: Can't Install Optional Components While in MS-DOS Window

 Last reviewed: April 18, 1995
Article ID: Q114297
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5b, 2.6

SYMPTOMS

You cannot install the optional components of FoxPro for MS-DOS when the installation program is executed in an MS-DOS window under Microsoft Windows. When you press any key to install the optional components, the installation program unexpectedly terminates.

RESOLUTION

To install the optional components of FoxPro for MS-DOS:

  1. Quit Windows.

  2. Run the installation program for FoxPro for MS-DOS; make sure Windows is not running.

To install the optional components of FoxPro for MS-DOS after the "basic portion" of FoxPro for MS-DOS has already been installed:

  1. Load FoxPro for MS-DOS.

  2. In the Command Window, type "DO INSTALL" (without the quotation marks).

    NOTE: The INSTALL.APP program is located in the default FoxPro for MS-DOS directory.

  3. Select and install the desired optional components.

STATUS

Microsoft has confirmed this to be a problem in FoxPro version 2.6 for MS- DOS. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The installation procedure for FoxPro for MS-DOS 2.6 involves two steps:

  • Installation of the "basic portion" of the product.
  • Installation of the "optional components" of the product.

After the "basic portion" of the product has been installed, the following message appears: 

   The basic portion of the product has been installed.
   Beginning optional component installation.
   If you do NOT want to install any optional components,
   you can press Escape (<ESC>) now.
   Press any key to continue.
If the installation program is being executed in MS-DOS (not under Windows), and the SPACEBAR or ENTER key is pressed, the optional components step of the installation procedure will be invoked.

If the installation program is being executed in an MS-DOS window under Microsoft Windows and the SPACEBAR or ENTER key is pressed, the installation program may terminate.
