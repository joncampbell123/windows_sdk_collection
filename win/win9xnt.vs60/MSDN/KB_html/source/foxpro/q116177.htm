REPORT...TO FILE Does Not Produce "Feature Not Available"

 Last reviewed: June 27, 1995
Article ID: Q116177
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6

SUMMARY

The "Language Reference" for FoxPro versions 2.5, 2.5a, 2.5b, and 2.6 for Windows states the following on page L3-823 in the "TO PRINTER [PROMPT] | TO FILE <file>" section of the documentation for the REPORT command: 

   In FoxPro for Windows, you cannot use the TO FILE option
   to send a graphics-based report created in FoxPro for
   Windows to a text file.  Including TO FILE generates a
   "Feature not Available" error message in FoxPro for Windows.
This paragraph is incorrect. The TO FILE clause of the REPORT command can indeed be used in FoxPro for Windows.

MORE INFORMATION

This paragraph is not found in the online Help file for FoxPro for Windows. For the most accurate information about the REPORT command, refer to the online Help file instead of the printed documentation.
