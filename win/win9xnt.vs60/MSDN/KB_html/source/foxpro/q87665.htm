The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, versions 1.02, 2.0, 2.5, 2.5a
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a
  • Microsoft FoxBASE+ for MS-DOS, version 2.1

SUMMARY

Third-party developers offer a number of freeware and shareware utility programs to recover damaged memo field data files. These utilities are supported by the third-party developers that wrote the utilities, not by Microsoft. The utilities on the list below have not been tested by Microsoft. The products included here are manufactured by vendors independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.

Files Available from Third-Party Vendors

Name:          FPTFIX.PRG
Title:         Memo File Fixer - Improved version
By:            Hank Costello
Description:   Improved version of FPTFIX with more options. Fixes
               both file pointer and blocksize bytes as required. If
               you order the source code for this utility, you also
               receive a program that works with FPTFIX2 to check all
               .FPT files and to fix damaged files.

Name:          CHKFPT.COM
Title:         Display information about .FPT and help diagnose
               corruption
By:            Pete Olympia
Description:   Length-damaged FoxPro memo field utility to display all
               information about .FPT file and help diagnose memo file
               data corruption. Includes block size, next available
               block, if the link is incorrect, and the correct link.
               For each memo field and database record, the utility
               shows memo field length and whether it is picture,
               text, or corrupted.

Name:          MEMOPL.INF
Title:         MemoPlus short product description
By:            Pete Olympia
Description:   Product description of MemoPlus (see below), a set of
               utilities for xBase memo file handling. Supports all
               xBase dialects, including FoxPro 2.0. Can diagnose
               and repair corrupt memo files automatically. Can import,
               export, pack memo field data in MS-DOS. You can also
               edit memo fields using your own text editor. Uploaded
               by Pete Olympia.
