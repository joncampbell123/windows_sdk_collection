PRB: VINES Network Message Corrupts Project File

 Last reviewed: December 18, 1997
Article ID: Q115587
2.50b WINDOWS kbprg kbprb

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.5b

WARNING: The information in this article has not been confirmed or tested by Microsoft. Some or all of the information in this article has been taken from unconfirmed customer reports. ANY USE BY YOU OF THE INFORMATION PROVIDED IN THIS ARTICLE IS AT YOUR OWN RISK. Microsoft provides this information "as is" without warranty of any kind, either expressed or implied, including but not limited to the implied warranties of merchantability and/or fitness for a particular purpose.

SYMPTOMS

Receipt of a network message on a Banyan VINES network while building an application from the Project Manager causes the project and/or screen files involved to become corrupted.

RESOLUTION

To avoid this problem, type "SET MESSAGE OFF" (without the quotation marks) at the MS-DOS command prompt prior to building projects in FoxPro for Windows.

Additional reference words: FoxWin 2.50b unconfirmed
