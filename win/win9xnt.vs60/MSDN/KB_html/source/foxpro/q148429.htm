PRB: Landscape Setting Under Quickdraw GX Not Respected

 Last reviewed: June 3, 1996
Article ID: Q148429
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

A report designed to print in landscape mode while QuickDraw GX printer drivers are loaded will print in portrait mode when using non QuickDraw GX printer driver for the same printer.

CAUSE

Whenever a different printer driver is used under the Macintosh operating system, landscape settings in the report frx table are ignored. Landscape settings are printer-specific on the Macintosh.

WORKAROUND

When designing reports, developers need to be aware of the printer drivers being used at the destination site. Select the printer driver that will be used at the destination site. If QuickDraw GX is not in use at the destination site, remove QuickDraw GX.

Landscape settings in a report will not be respected when a different printer driver is used even if you are using the same printing system either standard Macintosh printing or QuickDraw GX.

For more information on printer orientation under FoxPro for Macintosh, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q115773
   TITLE     : PRB: Printer Orientation & Settings Ignored by FoxPro
               for Mac

STATUS

Microsoft is researching this behavior and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

QuickDraw GX drivers come with System 7.5 and later versions.

Steps to Reproduce Behavior

  1. Load QuickDraw GX by clicking the installer for QuickDraw GX in the Install System Software folder on the Apple Macintosh CD-ROM. After restarting, a printer icon will appear on the desktop for the currently selected printer.

  2. Create a new report, and make it landscape by clicking the page setup button in the page layout dialog box. The page setup dialog box will display "GX Page Setup" and the name of the printer.

  3. Create a quick report using the customer table. The report will print and preview in landscape.

  4. Uninstall QuickDraw GX by clicking Install QuickDraw GX on the Apple Macintosh CD-ROM. Then select Custom Remove in the Install QuickDraw GX dialog box. Select all the check boxes, and then click Remove. After restarting, a black X will be appear on top of the QuickDraw GX printer on the desktop.

  5. Go to the chooser, and select the printer. Printers should no longer have GX after their names.

  6. Print the report. It prints in portrait mode.
