FIX: Transporter (Win) Won't Work w/ FoxPro/FoxBASE+ Reports

 Last reviewed: September 22, 1997
Article ID: Q103588
2.50 WINDOWS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.5

SYMPTOMS

The transporter program (TRANSPRT.PRG) has an error that precludes it from transporting FoxPro version 1.x and FoxBASE+ reports when it is run in FoxPro for Windows.

NOTE: These reports do not need to be transported in order to work with FoxPro 2.5 for MS-DOS.

RESOLUTION

To correct this problem, do the following:

  1. In the Command window, issue the following command: 

          modify command transprt.prg

  2. Find all occurrences of CONVERT102 and replace them with CVRT102.

  3. Find all occurrences of CONVERTFBP and replace them with CVRTFBP.

  4. Save the TRANSPRT.PRG file.

  5. Quit and restart FoxPro.

STATUS

Microsoft has confirmed this to be a problem in FoxPro 2.5 for Windows. This problem was corrected in FoxPro 2.5a for Windows.
