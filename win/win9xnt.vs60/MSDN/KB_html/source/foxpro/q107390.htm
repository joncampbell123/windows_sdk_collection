BUG: CTRL+END Key Combination Doesn't Solve Puzzle

 Last reviewed: April 18, 1995
Article ID: Q107390
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5a, 2.5b

SYMPTOMS

In FoxPro versions 2.5a and 2.5b for MS-DOS, the CTRL+END keyboard combination does not solve the puzzle.

This problem does not occur in FoxPro for Windows.

STATUS

Microsoft has confirmed this to be a problem in FoxPro versions 2.5a and 2.5b for MS-DOS. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. From the System menu, choose Puzzle.

  2. Press the letter S to shuffle the puzzle.

  3. Press CTRL+END.

The bell will ring, but the puzzle will remain shuffled.
