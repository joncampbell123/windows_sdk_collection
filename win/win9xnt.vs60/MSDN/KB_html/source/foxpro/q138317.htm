SUMMARY

Fw1180.exe is a file that contains an application note titled "Screen Issues in FoxPro for Windows" that explains how to address various screen and font issues encountered with FoxPro for Windows.

INTRODUCTION TO FW1180

This application note summarizes information about screen and font issues in FoxPro for Windows, emphasizing areas that are likely to impact application design. Information for this application note was taken from several Microsoft Knowledge Base (KB) articles, which are listed at the end of the application note.

Two main screen issues can seriously impact application design in FoxPro for Windows:

  • Fonts: Issues include font-related considerations when designing GUI applications in FoxPro.
  • Screen resolutions: Issues include handling multiple screen resolutions in an application. Methods of resolving these issues are suggested.
