FoxPro 1.02 Keyboard Shortcuts

 Last reviewed: April 17, 1995
Article ID: Q92355
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 1.02

The tables below list the keyboard shortcuts you can use to navigate in FoxPro and to execute commands and functions.

CTRL Key Shortcuts

   CTRL+A         Select all
   CTRL+C         Copy
   CTRL+D         Do
   CTRL+E         Replace and Find Again
   CTRL+F         Choose Find from the Edit menu
   CTRL+G         Choose Replace And Find Again from the Edit menu
   CTRL+K         Continue
   CTRL+M         Resume
   CTRL+O         Explicitly lock the current record
   CTRL+Q         Same as ESC
   CTRL+R         Redo
   CTRL+U         Undo
   CTRL+V         Paste
   CTRL+W         Exit current edit and save any modifications
   CTRL+X         Cut
   CTRL+F1        Cycle windows
   CTRL+F2        Display command window
   CTRL+F7        Move window
   CTRL+F8        Size window
   CTRL+F10       Zoom window
   CTRL+ALT+SHIFT Toggle display of windows on the desktop

Miscellaneous Shortcuts

   ESC            Exit current dialog box
   SHIFT+F10      Activate and deactivate the macro key
   ALT+254        Create bullet

Function Keys

   F1             Activate online help
   F2             Set
   F3             List
   F4             Directory
   F5             Display structure
   F6             Display status
   F7             Display memory
   F8             Display
   F9             Append
   F10            Activate or deactivate system menu bar

Key Combinations to Select Text

   SHIFT+LEFT ARROW          Select one character to the left
   SHIFT+RIGHT ARROW         Select one character to the right
   SHIFT+DOWN ARROW          Select one line down
   SHIFT+UP ARROW            Select one line up
   SHIFT+CTRL+LEFT ARROW     Select from the cursor position to the
                             end of the current word
   SHIFT+CTRL+RIGHT ARROW    Select from the cursor position to the
                             beginning of the current word
   SHIFT+CTRL+END            Select from the cursor position to the
                             end of the text
   SHIFT+CTRL+HOME           Select from the cursor position to the
                             beginning of the text
   CTRL+A                    Select the entire text

Cursor Movement Keys

   LEFT ARROW                Move cursor one character to the left
   RIGHT ARROW               Move cursor one character to the right
   DOWN ARROW                Move cursor down one line
   UP ARROW                  Move cursor up one line
   PAGE DOWN                 Move cursor down one window of text
   PAGE UP                   Move cursor up one window of text
   HOME                      Move cursor to the beginning of the line
   END                       Move cursor to end of the line
   CTRL+LEFT ARROW           Move cursor one word to the left
   CTRL+RIGHT ARROW          Move cursor one word to the right
   CTRL+HOME                 Move cursor to the beginning of the text
   CTRL+END                  Move cursor to the end of the text
