FoxApp Does Not Generate Multiuser Code

 Last reviewed: April 30, 1996
Article ID: Q108678
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, and 2.5b
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, and 2.5b

SUMMARY

The FOXAPP.APP application generator generates a single-user application. This application is not suitable for use in a multiuser environment.

To work around this limitation in the Macintosh environment, use the Screen Wizard. The Screen Wizard will create a screen suitable for use in a multiuser environment. This screen can be incorporated in an application created using the other power tools, such as the Menu Builder.

MORE INFORMATION

In a multiuser environment, special consideration must be given to handling record and file locking. A FoxApp-generated application does not include the necessary code to handle these situations.
