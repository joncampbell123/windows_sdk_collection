How to Check for a Defective Floppy Disk Using UNIX/XENIX

 Last reviewed: August 28, 1995
Article ID: Q118918
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

Periodically, disk errors may occur that may be caused by a defective floppy disk.

To check for a defective floppy disk using UNIX or XENIX, do the following:

  1. Place the floppy disk in the floppy disk drive.
2. At the system prompt, type: 

      dd if=/dev/install of=/dev/null
If a read error is returned, the floppy disk is defective.
