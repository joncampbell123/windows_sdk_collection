PRB: Table Wizard Fields Not Customizable

 Last reviewed: April 30, 1996
Article ID: Q130164
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SYMPTOMS

The Table Wizard's options are not extensive. What you see is what you get. You cannot modify or add to the number and names of fields and tables available in the Table Wizard.

CAUSE

The table and field names are hard coded in the WZTABLE.APP file, so you cannot change them.

WORKAROUND

The only workaround is to create your own Table Wizard.

STATUS

This behavior is by design.
