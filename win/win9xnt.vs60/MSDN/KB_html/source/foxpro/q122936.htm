DOCERR: 54 Megabytes Required for a Complete Installation

 Last reviewed: May 29, 1996
Article ID: Q122936
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

Page 2 of the "Installation and Macintosh Features Guide" for Microsoft Foxpro for Macintosh states incorrectly that a complete installation requires 26 megabytes. According to the "Readme Before Install-Pro" file found in the Foxpro folder, a complete installation requires 47 megabytes. According to the installation screens, a complete installation requires 47 megabytes.

All of these are incorrect. Actually, a complete installation requires 54 megabytes because an additional 7 megabytes of free diskspace is used during the installation process.

REFERENCES

Microsoft FoxPro for Macintosh "Installation and Macintosh Features Guide," version 2.6, page 2.
