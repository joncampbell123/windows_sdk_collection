How to Create FoxPro Macintosh Distribution Disks for 68K Only

 Last reviewed: May 24, 1996
Article ID: Q126965
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

You can, if necessary, create FoxPro for Macintosh distribution disks with only the 68K ESL file as opposed to creating disks with both ESL files.

By default, the setup wizard includes both the 68K and Power PC ESL support libraries on the distribution disks. The following steps modify the SETUP application program and eliminate all references to the Power PC ESL file.

These steps are not supported by Microsoft and you assume all risk for any modifications made to the SETUP program.

  1. Open DK Wizard:SETUP.PJX.
2. Browse the REQUIRED.DBF. 3. Delete the records for FOXPPC26.ESL. 4. Pack the DBF. 5. Rebuild the SETUP.APP, replacing the one in the FoxPro folder. 6. Use this new SETUP.APP to build your images.
