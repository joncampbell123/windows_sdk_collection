PRB: "Use Action Diagram Symbols" Unavailable in Beautify

 Last reviewed: April 29, 1996
Article ID: Q95318
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a

SYMPTOMS

When you are using the Beautify option in FoxPro for Windows, the check box for Use Action Diagram Symbols is dimmed (unavailable).

CAUSE

This option is available only for text that is displayed using FoxFont or FoxPrint.

RESOLUTION

From the Edit menu, choose Select All. From the Text menu, choose Font and change the font to FoxFont or FoxPrint. The Use Action Diagram Symbols check box in the Beautify dialog box will now be available.

When the Use Action Diagram Symbols option is selected, the Beautify application inserts box and line characters in the file that Beautify is running on. These characters are not available in most fonts; therefore, FoxFont or FoxPrint must be used.
