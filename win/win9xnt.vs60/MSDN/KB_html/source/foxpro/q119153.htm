FIX: Box or Line Drawn from Page Header to Footer Won't Print

 Last reviewed: October 20, 1997
Article ID: Q119153
2.5x MACINTOSH kbprint kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, versions 2.5x

SYMPTOMS

If you draw a box, rounded box, or line with the Rectangle, Rounded Rectangle, or Line tool from the Page Header band to the Page Footer band in the Report Writer, the line or box will not appear in Page Preview and will not be printed. If the line or box does not extend into the Page Header or Page Footer band, it will appear in Page Preview and will be printed.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft FoxPro version 2.6a for Macintosh.

MORE INFORMATION

This behavior does not occur in FoxPro for Windows.

Steps to Reproduce Problem

  1. From the File menu, choose New, select Report, and then choose OK.

  2. Using the appropriate tool, draw a line or a box from the Page Header band to the Page Footer band.

  3. From the Report menu, choose Page Preview.

The line or box will not appear in Page Preview. It also will not be printed if you use the REPORT FORM command to print the report.
