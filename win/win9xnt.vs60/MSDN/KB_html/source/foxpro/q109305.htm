Incorrect Compile Commands for Microsoft C/C++ in LCK Doc

 Last reviewed: January 5, 1995
Article ID: Q109305
The information in this article applies to:
  • Microsoft FoxPro Library Construction Kit for MS-DOS and Windows, version 2.5

On page 22 in the FoxPro Library Construction Kit (LCK) "Developer's Guide", the compile commands examples are reversed.

The correct example for an API library for FoxPro for MS-DOS is as follows: 

   cl /Gs /GW /ALw /FPc /c /Zp filename.c
The correct example for an API library for FoxPro for Windows is as follows: 

   cl /Gs /GD /ALw /FPc /c /Zp filename.c
