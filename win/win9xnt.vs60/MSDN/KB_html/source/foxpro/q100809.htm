Windows Control Panel Does Not Affect _dblclick

 Last reviewed: June 27, 1995
Article ID: Q100809
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SUMMARY

The _dblclick memory variable in FoxPro for Windows is not affected by the Windows Control Panel mouse settings.

MORE INFORMATION

The Windows Control Panel allow you to configure the time interval between double mouse clicks and the mouse tracking speed for Windows-based applications.

FoxPro for Windows does not use the Windows Control Panel settings to determine the interval between double clicks. To set this interval, store a new value to the _dblclick system memory variable.

This feature ensures cross-platform compatibility between FoxPro products running with different operating systems.
