COMMAND Key in FoxPro for Mac Is Treated Same as CTRL Key

 Last reviewed: June 1, 1996
Article ID: Q110251
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b
  • Microsoft Visual FoxPro for Macintosh, version 3.0b

In all areas of FoxPro for Macintosh, the COMMAND key is treated the same as the CTRL key in order to ensure cross-platform compatibility.
