Catalog Manager Cannot Pack Database Files

 Last reviewed: April 30, 1996
Article ID: Q113567
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

The Catalog Manager does not have the functionality to pack, or remove deleted records from, a table.

To pack a database that is part of a catalog being used in the Catalog Manager, do the following:

  1. Quit the Catalog Manager. Double-click the Control-menu box (close box) for the Catalog Manager window or choose Command Window from the File menu.

  2. Open the file that you want to pack. You can do this by issuing the USE <table> command in the Command window or by choosing Open from the File menu.

  3. Pack the table by issuing the PACK command or by choosing Pack from the Database menu.

  4. Close the table by choosing Close from the File menu or by typing "USE" (without the quotation marks) in the Command Window.

To return to the Catalog Manager, type "ASSIST" (without the quotation marks) in the Command window, or choose Catalog Manager from the File menu.
