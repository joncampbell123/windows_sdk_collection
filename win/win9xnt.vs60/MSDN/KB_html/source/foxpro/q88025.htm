FIX: MESSAGE(1) Returns Only Last Line of Command

 Last reviewed: October 14, 1997
Article ID: Q88025
2.00 MS-DOS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

In FoxPro version 2.0, the MESSAGE(1) function returns only the last line of a multiline command.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FoxPro version 2.0 for MS-DOS. This problem was corrected in Microsoft FoxPro version 2.0 for MS-DOS dated September 26, 1991.

MORE INFORMATION

In an ON ERROR routine, the MESSAGE(1) function returns the program source code that caused the error.
