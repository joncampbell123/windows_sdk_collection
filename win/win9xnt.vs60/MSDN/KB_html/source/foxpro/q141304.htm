This document provides an introduction to the concepts of Object Oriented Programming (OOP) and how they can be applied in Visual FoxPro. Its purpose is to provide you with a basic understanding of objects, classes, and the four key characteristics of an Object Oriented Programming Language: Abstraction, Encapsulation, Inheritance, and Polymorphism. This document can be read straight through or used as a mini-tutorial on the concepts of Object Oriented Programming. To use the program files included with this document you must have either the Standard or Professional version of Visual FoxPro.
