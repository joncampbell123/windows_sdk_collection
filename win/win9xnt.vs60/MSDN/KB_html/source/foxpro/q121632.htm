Location of System Capacities Chart in 2.5 or 2.6 Manuals

 Last reviewed: August 28, 1995
Article ID: Q121632
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6a

SUMMARY

In FoxPro versions 2.5 and later for MS-DOS, the "System Capacities" chart is located in the "Update" booklet. In FoxPro versions 2.5 and later for Windows and FoxPro versions 2.5 and later for Macintosh, it is in the "Developer's Guide." In previous versions of FoxPro, it was found in the "Quick Reference."

MORE INFORMATION

The locations of the "System Capacities" chart for FoxPro for MS-DOS, Windows, and Macintosh are listed below.

FoxPro 2.5x or 2.6 for MS-DOS: 

   "Update" booklet page 4-4
FoxPro 2.5x or 2.6 for Windows: 

   "Developer's Guide" page A-4
FoxPro 2.5x or 2.6a for Macintosh: 

   "Developer's Guide" pages 404-405
This chart is also located in the "System Capacities" topic in the online Help system.
