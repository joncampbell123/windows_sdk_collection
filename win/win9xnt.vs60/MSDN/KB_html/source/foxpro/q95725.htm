PRB: Print To File Not Prompting w/ TO PRINTER [PROMPT] Clause

 Last reviewed: April 29, 1996
Article ID: Q95725
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

Pages L3-450, 453, 456, 458, 460, and 462 of the "Language Reference" describe commands using the TO PRINTER [PROMPT] clause. Including the PROMPT clause displays a dialog box that provides the option of printing to a file. However, the Print To File check box never prompts for a filename after the OK button is chosen. The output is sent to the screen or active output window, as well as to the printer, but not to a file.

To work around this problem, use the TO FILE <filename> option.

Additional reference words: FoxWin 2.50 2.50a report docerr
