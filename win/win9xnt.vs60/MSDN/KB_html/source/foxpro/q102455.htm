BUG: "Working Directory Specified Is Invalid" Error Message

 Last reviewed: April 29, 1996
Article ID: Q102455
The information in this article applies to:
  • Microsoft FoxPro Distribution Kit for Windows, versions 2.5, 2.5a, 2.5b, and 2.6

SYMPTOMS

An .EXE file created by the FoxPro for Windows Distribution Kit and installed by a SETUP.EXE file created by SETUP.APP (the SetupWizard) fails with the following error message: 

   The working directory specified is invalid

CAUSE

The Program Manager Icon Description specified in the SetupWizard contains a comma.

RESOLUTION

Run SETUP.APP again and remove the comma from the Program Manager Icon Description.

STATUS

Microsoft has confirmed this to be a problem in FoxPro Distribution Kit 2.5, 2.5a, 2.5b, and 2.6 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
