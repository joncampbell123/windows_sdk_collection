Upgrade Process for FoxPro for UNIX

 Last reviewed: August 28, 1995
Article ID: Q118912
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

When installing a new version of FoxPro for UNIX, you must remove the current version before installing a new version of the product.

MORE INFORMATION

In order to remove the old version of the product and install the new one, you must be logged on to the UNIX or XENIX system as "root". You can then use sysadmsh, the menu-driven system administration utility, to remove the current version of FoxPro for UNIX and install the newest version of FoxPro for UNIX.
