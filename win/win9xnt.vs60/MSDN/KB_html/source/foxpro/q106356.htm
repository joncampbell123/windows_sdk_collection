LCK Manual Incorrectly Refers to FPFI as FPDI Data Type

 Last reviewed: August 28, 1995
Article ID: Q106356
The information in this article applies to:
  • Microsoft FoxPro Library Construction Kit for MS-DOS, version 2.5
  • Microsoft FoxPro Library Construction Kit for Windows, version 2.5

Page 6 of the "Developer's Guide" for the FoxPro Library Construction Kit incorrectly lists the FPFI data type as FPDI. FPFI is a FAR pointer to a function returning an integer.
