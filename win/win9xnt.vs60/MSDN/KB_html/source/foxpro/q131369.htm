FIX: Specified Page Range Ignored with the TYPE Command

 Last reviewed: October 20, 1997
Article ID: Q131369
2.50c MACINTOSH kbprint kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.5c

SYMPTOMS

When you enter a range of pages to be printed in the Macintosh Print Setup dialog, FoxPro ignores the page range setting and prints the wrong page or the entire document.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in in version 2.60a of FoxPro for the Macintosh.

MORE INFORMATION

Steps to Reproduce Problem

  1. In the Command window, type:

    TYPE "FoxPro Readme" TO PRINTER PROMPT

  2. In the prompt dialog, specify from page 3 to page 3.

Depending on the printer being used, a single page different from what you asked for may print, or all the pages may print.
