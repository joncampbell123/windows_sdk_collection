PRB: Setup Wizard Error: DKSETUP.SPR Does Not Exist

 Last reviewed: June 27, 1995
Article ID: Q124289
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.6 and 2.6a

SYMPTOMS

Attempting to run DKSETUP.PRG results in the following error: 

   Microsoft FoxPro SetupWizard Error: File DKSCRN.SPR Does Not Exist

CAUSE

An attempt to run DKSETUP.PRG was made.

RESOLUTION

DKSETUP.PRG is only a piece of the Setup Wizard application. It is not intended to be run stand-alone. To run the Distribution Kit Wizard, select SETUP.APP as the application to run.

STATUS

This behavior is by design.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Choose Application or Program from the Run menu.

  2. Move to the FOXPRO\DKWIZARD subdirectory.

  3. Select DKSETUP.PRG as the application to run.
