Q: What is Visual FoxPro 5.0a?

A: Visual FoxPro 5.0a is a maintenance release of Visual FoxPro 5.0. Version 5.0a was released to address some specific problems in Visual FoxPro, as well as to help coordinate with the Visual Studio 97 release.
