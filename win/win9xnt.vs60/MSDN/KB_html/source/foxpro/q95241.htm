Using the Show Position Option in FoxPro Screen Builder

 Last reviewed: April 29, 1996
Article ID: Q95241
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

Show Position is a new feature in Microsoft FoxPro for Windows that displays the vertical and horizontal coordinates of the mouse pointer or the dimensions of selected objects in the Windows-style status bar.

In the Screen Builder, if the Show Position option is selected and there is no highlighted object, FoxPro activates the status bar and continuously displays the position of the mouse pointer.

If an object is selected, its location and dimensions are displayed in the status bar instead. The Show Position unit is the same as the unit being used in the ruler in the Ruler/Grid dialog box.
