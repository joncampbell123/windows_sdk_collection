How to Run .EXE Without FOXW2500.ESL in Same Directory

 Last reviewed: April 29, 1996
Article ID: Q102705
The information in this article applies to:
  • Microsoft FoxPro Distribution Kit for Windows, version 2.5

FoxPro for Windows executable files will not run if they cannot locate the FOXW2500.ESL file. Page 20 of the "User's Guide" for the FoxPro Distribution Kit for Windows states that this file should be located in the same directory as the executable file.

However, you can place FOXW2500.ESL in a different directory if the AUTOEXEC.BAT file contains a PATH enviroment variable that lists this directory. Because the FOXW2500.ESL file requires 2.9 MB of disk space, this option is useful when you have executable files in different directories.

Additional reference words: FoxWin DK 2.50 "Cannot locate
support library"
