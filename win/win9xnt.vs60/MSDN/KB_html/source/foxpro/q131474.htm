PRB: PRINT Option Disabled in Modified Screen Code

 Last reviewed: April 30, 1996
Article ID: Q131474
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

When a screen is modified through the Catalog Manager and code snippets are opened, the PRINT option is disabled on the File menu.

WORKAROUND

To print the code snippets for a screen, open it by using the MODIFY SCREEN command or by choosing Open from the File menu.

STATUS

This behavior is by design.

MORE INFORMATION

Steps to Reproduce Behavior

If a screen is opened in FoxPro through the Screen Builder and not through the Catalog Manager, code snippets can be printed by choosing Print from the File menu once the snippets are opened.

However, if a screen is modified through the Catalog Manager, the Print option on the File menu is disabled (grayed out) when snippets are opened with the intent of printing.
