PRB: "Cannot Start Microsoft Word" Error When Doing Mail Merge

 Last reviewed: February 28, 1997
Article ID: Q113667
2.60 3.00 WINDOWS kbtool kbinterop kbprg kberrmsg kbprb

The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, version 3.0
  • Microsoft FoxPro for Windows, version 2.6

SYMPTOMS

After you have filled out all the selections and chosen the Finish button to run the Mail Merge Wizard in FoxPro version 2.6 for Windows, the error message "Cannot start Microsoft Word" is displayed.

CAUSE

The Mail Merge Wizard directs Microsoft Word to open the selected database using a 1-Tier FoxPro ODBC driver. If the FoxPro 1-Tier ODBC driver is not installed or configured properly, the above error message is displayed.

NOTE: The ODBC icon will not be not available in Control Panel unless you have installed ODBC drivers from either the FoxPro Connectivity Kit or another ODBC source.

RESOLUTION

Install the FoxPro 1-Tier ODBC driver with the correct configuration information, as described below:

  1. From the Program Manager in Windows, open the Control Panel.

  2. In the Control Panel, double-click the ODBC icon.

  3. In the Data Sources dialog box, choose the Add button.

  4. Select FoxPro Files (*.dbf), and then choose OK.

    If the FoxPro Files (*.dbf) option is not available, the Desktop Drivers have not been installed. To obtain the Desktop Drivers, call the Microsoft Sales Information Center at (800) 426-9400. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, go to the Microsoft Web site http://www.microsoft.com/worldwide/default.htm

    NOTE: The desktop drivers for FoxPro are also supplied with Word 6.0 for Windows. They can be installed by double-clicking the Word for Windows Setup icon which starts the maintenance-mode installation program. Select the box for Converts, Filters and Data Access.

  5. In the ODBC FoxPro Setup dialog box, enter the following information: 

          Data Source Name: FoxPro Files
      Description: <Type whatever you like.>

  6. For Version, select FoxPro version 2.5.

  7. Select the Use Current Directory check box.

The Mail Merge Wizard should now work.
