Tips & Tricks Topic Lists Wrong Cross-Platform Jump Text

 Last reviewed: May 29, 1996
Article ID: Q118492
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c

SUMMARY

In the Windows-style graphical Help file (FOXHELP.HLP), the "Tips and Tricks" topic displays the "Cross Platform Development" jump text instead of the "Cross Platform Tips" jump text.

To display the "Cross Platform Tips" topic, do the following:

  1. Choose the Search button.
2. Type "Cross-Platform Tips" (without the quotation marks). 3. Choose Show Topics, and then choose Go To.

MORE INFORMATION

NOTE: There is also a slight problem with the search text. In step 2 above, you must type "Cross-Platform Tips" (with the hyphen) to display the correct topic. However, the title of the topic is actually "Cross Platform Tips" (without the hyphen).

In the .DBF-style Help file (FOXHELP.DBF), the correct cross-platform text is listed under "Tips and Tricks."
