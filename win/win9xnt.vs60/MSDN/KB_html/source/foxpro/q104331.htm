How to Download FoxPro Files from the MSDL

 Last reviewed: April 29, 1996
Article ID: Q104331
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, and 2.5b
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, and 2.5b

SUMMARY

Support files for Microsoft FoxPro for Windows and MS-DOS are available from the Microsoft Download Service (MSDL) 24 hours a day, 7 days a week.

There is no connect-time charge for using this service. However, standard connect-time fees and long-distance telephone charges, if any, do apply during downloading.

The phone number is (206) 936-MSDL (936-6735). The communications settings are 8 data bits, no parity, and 1 stop bit. Baud rates are 1200, 2400, 9600, or 14,400 baud (V.32).

MORE INFORMATION

To download files for FoxPro:

  1. Using your communications software, call the MSDL at (206) 936-6735 between 1200 and 14,400 baud. Set the data bits to 8, the parity to None, and the stop bits to 1.

  2. When prompted, type your full name, and then press ENTER.

  3. When prompted, type your city and state, and then press ENTER.

  4. When prompted, press Y for Yes or N for No, indicating whether you have entered your name, city, and state correctly.

  5. Several screens of new user information may be displayed. Clear each screen of new user information by pressing ENTER until you reach the Main Menu.

  6. At the Main Menu, press F for File Index.

  7. In the File Index menu, press 7 for FoxPro, Access, and Money.

  8. In the FoxPro, Access, and Money Files menu, press 1 for FoxPro.

  9. In the FoxPro Files menu, press the menu number for the appropriate FoxPro category.

  10. Locate the file you want to download.

  11. At the next prompt, press D for Download, and then press ENTER.

  12. At the File Name prompt, type the filename to be downloaded, and then press ENTER.

  13. Choose the transfer protocol used by your communications software. If you are unsure, press X for Xmodem.

  14. Start the download feature of your communications software. (In Microsoft Windows Terminal, for example, choose Receive Binary File from the Transfers menu).

  15. When your communications software has finished downloading the file, press ENTER.

  16. To leave the MSDL, press E to Exit, then use the hang-up function of your communications software.
