FIX: Open File Dialog Box Displays in an Application

 Last reviewed: April 17, 1995
Article ID: Q88642
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

In FoxPro 2.0 for MS-DOS, the Open File dialog box appears when you are running an application that includes a report form and a database called by the report.

CAUSE

If the application is moved from the directory it was originally created in, the Open File dialog box will incorrectly appear.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FoxPro version 2.0 for MS-DOS. This problem was corrected in Microsoft FoxPro version 2.0 for MS-DOS dated March 25, 1992.
