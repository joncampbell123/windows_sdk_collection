Border for GETS Check Box Not Included in Documentation

 Last reviewed: April 29, 1996
Article ID: Q102587
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5

On page U11-80 in the "User's Guide," the picture that shows the Generate Screen dialog box does not show the Border For GETS check box.

Although the explanation for the check box is not included in the "User's Guide," you can find information in the online Help system pertaining to borders for editing regions created with @ ... GET by typing "help set readborder" (without the quotation marks) in the Command window.
