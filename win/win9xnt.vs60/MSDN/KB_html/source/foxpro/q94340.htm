Defining Menu at a Specific Row with FoxPro Menu Builder

 Last reviewed: April 17, 1995
Article ID: Q94340
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a

Use the following steps to define a menu at any row on the screen using the Menu Builder:

  1. When you are in the Menu Builder editing a menu, choose General Options from the Menu menu.

  2. In the General Options dialog box, select the Setup check box and then choose OK.

  3. Insert the following line of code in the setup code procedure 

          DEFINE MENU _msysmenu BAR AT LINE <expN>

    where <expN> is any valid row number on the screen. When the menu is run, FoxPro will display the menu on the row that you specified in <expN>.
