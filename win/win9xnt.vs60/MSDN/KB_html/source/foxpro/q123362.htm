HOWTO: Search for FoxPro Articles Using KB-wide Keywords

 Last reviewed: March 10, 1998
Article ID: Q123362
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 5.0
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6x
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5x, 2.6x
  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6a
  • Microsoft Visual FoxPro for Macintosh, version 3.0b
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

The Microsoft Knowledge Base (KB) is categorized and subcategorized by using keywords referred to as the KB-wide and product-specific KBKeywords. This article lists the KB-wide keyword associated with each product-specific KBKeyword specific to Microsoft FoxPro articles.

NOTE: For intuitive keywords to limit searches to specific platforms (for example, "VFoxWin" limits searches to Visual FoxPro for Windows articles), please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q122791
   TITLE     : Query Keywords Specific to the FoxPro Knowledge Base

MORE INFORMATION

Each article in the FoxPro collection may contain a product-specific keyword (called a KBkeyword) that places the article in an appropriate FoxPro category. Each FoxPro-specific KBKeyword is associated with a KB- wide keyword. The KB-wide keywords are standard throughout the Microsoft Knowledge Base, appearing in all KB articles, regardless of product.

This article maps each FoxPro-specific KBKeyword to its associated KB-wide keyword. Although each KBKeyword has one KB-wide keyword assigned to it, an article can also include additional KB-wide keywords.

For more information about the general KB-wide keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
For more information about using KB-wide and FoxPro KBKeywords in queries, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q122923
   TITLE     : How to Search for FoxPro Articles by KBSubcategory
