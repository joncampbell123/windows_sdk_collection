Help Incorrectly Lists Unsupported SQL Field Functions

 Last reviewed: June 28, 1995
Article ID: Q114479
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

SUMMARY

According to the "Switching from dBASE" topic in FoxPro for Windows and FoxPro for MS-DOS online Help, FoxPro does not support the Structured Query Language (SQL) field functions AVG(), COUNT(), MAX(), MIN(), and SUM().

This statement is incorrect; these field functions are supported in FoxPro's SQL-SELECT statement.

MORE INFORMATION

To go directly to this documentation error, choose dBASE Help from the Help menu. Then do one of the following, depending on which online Help file is in use:

  • FOXHELP.HLP: Choose "SQL Differences in dBASE IV and FoxPro."
  • FOXHELP.DBF: Scroll to the "SQL Differences" heading, located at the bottom of the topic.
