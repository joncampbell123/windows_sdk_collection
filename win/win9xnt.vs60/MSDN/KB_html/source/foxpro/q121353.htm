Unreadable Text in ON KEY LABEL Help Topic

 Last reviewed: May 29, 1996
Article ID: Q121353
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SUMMARY

The description in the graphical-style online Help file (FOXHELP.HLP) of the ON KEY LABEL command contains lines of text that are so small they are unreadable.

This problem does not occur in the .DBF-style online Help file (FOXHELP.DBF).

MORE INFORMATION

Location of Unreadable Text

In the ON KEY LABEL help topic, locate the following line of text near the end of the topic: 

   Also, ON KEY LABEL DEL is equivalent to ON KEY LABEL BACKSPACE.
Note that the next line is unreadable. It says: 

   The ALT key in FoxPro for MS-DOS and FoxPro for Windows is emulated in
   FoxPro for Macintosh with CTRL+OPTION.
