FIX: Arrange Button Increments Negative Window Coordinates

 Last reviewed: September 22, 1997
Article ID: Q104341
2.50 WINDOWS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.5

SYMPTOMS

Choosing the Arrange button in the Generate Screen dialog box increments negative screen positions by one pixel.

STATUS

Microsoft has confirmed this to be a problem in FoxPro 2.5 for Windows. This problem was corrected in FoxPro 2.5a for Windows.

MORE INFORMATION

Steps to Reproduce Problem

  1. Issue the following command in the Command window in order to start the Screen Builder: 

          CREATE SCREEN

  2. From the Program menu, choose Generate.

  3. Choose the Arrange button.

  4. Drag the window definition for a screen panel to the left and up, until part of it is off the arrange area. At least one of the Horizontal or Vertical Position values should be negative.

  5. Choose OK.

  6. Choose the Arrange button again.

The negative position(s) are incremented by one pixel.
