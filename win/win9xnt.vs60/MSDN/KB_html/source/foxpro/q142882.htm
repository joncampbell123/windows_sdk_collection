FoxPro 2.6a for Windows - Availability and Support Options

 Last reviewed: May 19, 1997
Article ID: Q142882
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a
  • Microsoft FoxPro for Macintosh, version 2.6a
  • Microsoft FoxPro for MS-DOS, version 2.6a
  • Microsoft FoxPro for Unix, version 2.6a

SUMMARY

For the time being, Microsoft is continuing to sell all versions of FoxPro 2.6a to assist developers creating cross-platform applications. Contact the Microsoft Sales Information Center (MSIC) for information on ordering this older product. The U.S. phone number is 1-800-426-9400. To contact the Canadian sales office call 1-800-563-9048.

MORE INFORMATION

Microsoft Technical Support policies include a provision for "Expiration of Standard Support." According to this policy, Standard Support for a Developer Support product expires 120 days after the newest version of that product becomes available.

All versions of FoxPro 2.6a are an exception to this policy at this time. Standard Support, as described in the current Microsoft support policies, is still available for the following:

  • FoxPro for Windows 2.6a
  • FoxPro for MS-DOS 2.6a
  • FoxPro for Macintosh 2.6a
  • FoxPro for UNIX 2.6a

This policy may change without notice and is subject to the Microsoft support policies in place at the time you contact us.
