DOCERR: DK Manual Unclear About Image Copying

 Last reviewed: May 23, 1996
Article ID: Q124883
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SUMMARY

On page 26 of the FoxPro Distribution Kit for Macintosh "User's Guide," the sentence in the paragraph which reads: 

   Copy the disk image folders and their contents to 1.44 floppy disks.
is incorrect and unclear.

MORE INFORMATION

The steps listed on page 26 of the Distribution Kit for Macintosh "User's Guide" indicate that the folder and all files in the folder that contain the distribution files should be copied to the floppy disk. If these steps are followed, attempting to run the install program from the floppy disk\DISK1 folder results in the following error message: 

   The file <filename> appears to be missing from the installation disk.
   You may either continue installation with the remaining files, or stop
   Setup.
The sentence should read as follows: 

   Copy the disk image folder contents to 1.44 floppy disks.
The contents of the DISK<n> folders should be copied to the floppy disk; the actual folder should not be copied.

Additional reference words: FoxMac 2.60a appears to be missing DK errmsg
