PRB: VFP ODBC Drivers Will Not Install without a Valid Program

 Last reviewed: July 3, 1997
Article ID: Q154063
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SYMPTOMS

When trying to install the Visual FoxPro ODBC driver from the VFPODBC.EXE file provided on the Internet or on MSDN, the following error will occur if one of the listed Microsoft products is not already installed on the computer: 

   Beginning of error message:

   You are not allowed to install these drivers (message in
   title bar).

   You must have one of the Microsoft products listed below in
   order to install the drivers.

      Office
      Access
      Excel
      MS Project
      Visual C++
      Visual C++ 2.0
      Word
      Fox
      Visual FoxPro
      Visual Basic 4.0
      Developer

   End of error message.

CAUSE

There are no valid Microsoft products installed on the computer.

RESOLUTION

Install one or more of the valid Microsoft products listed in the error message, or call FoxPro Product Support for further information.

STATUS

This behavior is by design.

MORE INFORMATION

The computer must have one of the listed products installed before the Visual FoxPro ODBC driver can be installed. An EXE built with Visual FoxPro does not qualify as a Microsoft product. Therefore, these drivers cannot be distributed unless there is a Microsoft product from the above list installed on the computer.

Please refer to the following article in the Microsoft Knowledge Base for the location of the Visual FoxPro ODBC driver: 

   ARTICLE ID: Q149840
   TITLE     : Visual FoxPro ODBC Driver 1.0 Available in MS Software
               Library        [odbc]
