MAIN.PRG File Does Not Exist in Organizer Application

 Last reviewed: June 27, 1995
Article ID: Q103468
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

An incorrect file is documented as defining the user-defined function (UDF) LOCATEDB() called by the Organizer application. On page D2-41 of the FoxPro for Windows "Developer's Guide" and on page D2-34 of the FoxPro for MS-DOS "Developer's Guide" under the heading "Setup Code Example 3," the text incorrectly claims that the LOCATEDB() UDF is found in MAIN.PRG.

The text associated with the code diagrams on these pages states: 

   Call UDF LOCATEDB() to Locate .DBF and the associated .FPT.
   LOCATEDB() is defined in MAIN.PRG.
The Organizer application does not contain a file named MAIN.PRG. The LOCATEDB() UDF is defined in the UTILITY.PRG file in the SAMPLE\PRGS subdirectory.
