Where to Find Info on SQL pass-through Versus Remote Views

 Last reviewed: September 17, 1995
Article ID: Q136853
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SUMMARY

For a comparison of the Remote View and SQL Pass-Through technologies, see the Visual FoxPro "Professional Features Guide," pages 46 and 47. These pages describe the differences between the two technologies, as well as explain the advantages and disadvantages of using SQL pass-through queries.

MORE INFORMATION

You can find additional information that explains how to improve update and delete performance by optimizing view and connection properties on page 80 of the "Professional Features Guide."

The Help file contains information about Optimizing Data Fetching. Click Contents on the Help menu. Click Using Visual FoxPro, and then click "Changes to Printed Documentation." Then click Professional Features Guide. Scroll down to "Chapter 4: Optimizing Client/Server Performance."
