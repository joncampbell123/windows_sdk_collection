BUG: Scroll Bars Not Visible for Edit Region

 Last reviewed: May 20, 1996
Article ID: Q125966
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

When you use the Screen Builder to create an edit region that has four or fewer lines in height, the scroll bars do not appear in the generated screen even though they do appear in the .SCX file.

WORKAROUND

To have the scroll bars appear, make the edit region taller.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Open the Salesman database in the tutorial directory.

  2. Type CREATE SCREEN JUNK.SCX in the Command window, and press ENTER.

  3. Choose the option for edit region from the toolbox on the screen and select the Scroll Bar as the edit region option.

  4. Choose the memo field salesman.notes from the fields list, and size the edit region so that it is there lines in height.

  5. From the Program menu pad, select Generate to generate the screen.

  6. Type DO JUNK.SPR in the Command window, and note that the scroll bars are not present on the edit region.
