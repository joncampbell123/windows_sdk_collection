PRB: "Microsoft FoxPro Graph Wizard Error" with UPDATEGRPH

 Last reviewed: April 30, 1996
Article ID: Q118291
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6

SYMPTOMS

When you are using the UPDATEGRPH procedure from GENGRAPH.APP on a graph created with the Graph Wizard, the following error messages will appear: 

   Microsoft FoxPro Graph Wizard Error
   I could not find any numeric data to graph
   Press any key to continue

   Unrecognized Command Verb
   <Cancel> <Suspend> <Ignore>

CAUSE

The database or query that the graph was created from is not the current selected database. Most likely, the database that has the graph information in it, such as QRYGRAPH.DBF, is the current database selected.

RESOLUTION

Select the database or query that the graph was created from.
