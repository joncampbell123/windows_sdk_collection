FAQ: What's New in Visual FoxPro 5.0a

 Last reviewed: September 18, 1997
Article ID: Q173768
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 5.0a

Q: What's new in Visual FoxPro 5.0a?

A. No additional features have been added to the product. The changes that were made were minor bug fixes and synchronization with other Visual Studio 97 products. Visual FoxPro 5.0a includes an updated version of the ActiveX controls and the ODBC 3.5 desktop drivers. Visual FoxPro 5.0a also includes Service Pack 2 for Microsoft Windows NT 4.0, Service Pack 5 for Microsoft Windows NT 3.51, and Microsoft Internet Explorer 3.01.
