PRB: MIGRATE.APP Does Not Work with Microsoft Windows 3.0

 Last reviewed: June 27, 1995
Article ID: Q101804
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

MIGRATE.APP, the Migration Kit application, does not work when FoxPro for Windows is running under Windows 3.0.

After the DO MIGRATE.APP command is executed, the message "The Migration Tools menu pad is now installed. Press any key..." appears in the upper- right corner of the screen. After you press a key, the Migration Tools menu pad appears to the right of the Help menu pad. When you select the Migration Tools menu pad, a popup of menu commands appears. When you choose any of these commands, such as New Analysis or Open Analysis, with the mouse or the hot key, nothing happens. No error messages appear. The only way to quit MIGRATE.APP is to choose Cancel from the Program menu.

RESOLUTION

To resolve this problem, you must upgrade to Windows 3.1.
