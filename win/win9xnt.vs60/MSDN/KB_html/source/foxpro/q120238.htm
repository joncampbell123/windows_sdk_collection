MAIN.PRG Should Be UTILITY.PRG on Page 42 of Dev. Guide

 Last reviewed: May 29, 1996
Article ID: Q120238
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6a

Page 42 of the FoxPro for Macintosh "Developer's Guide" incorrectly refers to a MAIN.PRG file, which is supposed to contain the LOCATEDB() function. However, this file does not exist in the CREDIT project. Instead of MAIN.PRG, page 42 should refer to UTILITY.PRG, which does contain the LOCATEDB() function.

Additional reference words: FoxMac 2.50b 2.50c 2.60a docerr
