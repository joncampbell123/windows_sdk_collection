How to Add Searchable Type When Using HotSpot Editor

 Last reviewed: June 26, 1995
Article ID: Q100803
The information in this article applies to:
  • Microsoft FoxPro for Windows Distribution Kit, versions 2.5 and 2.5a

The Microsoft FoxPro for Windows HotSpot Editor that is installed with the Distribution Kit can have up to four different types in the Binding section of an object's attributes.

The four types available to the HotSpot Editor are:

  • Popup
  • Jump
  • Macro
  • Searchable

The Searchable option will be available only if the following lines are added to the WIN.INI file: 

   [HotSpot Editor]
   searchable = 1

Additional reference words: FoxWin 2.50 2.50a hot spot
