Lang Ref for DEFINE WINDOW NONE and TITLE Is Incomplete

 Last reviewed: April 29, 1996
Article ID: Q97641
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

Page L3-432 of the "Language Reference" manual describes the different types of border clauses of the DEFINE WINDOW command. The description of the NONE clause on page L3-432 is not complete; therefore, this clause may act differently than expected.

Page L3-432 should state the following: 

   When the NONE clause is selected in conjunction with the TITLE
   clause in FoxPro for MS-DOS, a border or a title may not appear.
   However, in FoxPro for Windows, a half-height title and no
   border will be displayed.

Additional reference words: FoxDos FoxWin 2.50 2.50a documentation error
