Cancel Option Available on Program Pad While in READ

 Last reviewed: July 5, 1995
Article ID: Q102922
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a

The Cancel menu option is available on the Program menu while in a READ in FoxPro for Windows (READ MODAL is an exception). This feature is an enhancement to FoxPro for Windows and is unavailable in FoxPro for MS-DOS when in a READ.

To disable this feature, use one of the following commands: 

   SET DEVELOPMENT OFF

   -or-

   SET SKIP OF BAR _mpr_cancl OF _mprog .T.

Additional reference words: FoxWin 2.50 2.50a 2.50b 2.60 2.60a cancel
