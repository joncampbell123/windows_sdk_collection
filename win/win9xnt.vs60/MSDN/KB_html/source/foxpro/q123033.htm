List of CTRL and Corresponding COMMAND Key Shortcuts

 Last reviewed: August 28, 1995
Article ID: Q123033
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6, 2.6a
  • Microsoft FoxPro for MS-DOS, versions 2.5, 2.5a, 2.5b, 2.6, 2.6a
  • Microsoft FoxPro for Macintosh, version 2.5b, 2.5c, 2.6, 2.6a

SUMMARY

The information below lists the shortcuts assigned to the CTRL (Control) key in FoxPro for Windows and FoxPro for MS-DOS, and the COMMAND key in FoxPro for Macintosh.

This information is also found in the following locations in online Help:

  • FoxPro version 2.6a for Macintosh FOXHELP.HLP online Help file: Choose the Search button, then type "CONTROL KEY" (without the quotation marks). Choose Show Topics, then choose Go To.
  • FoxPro version 2.6a for Macintosh FOXHELP.DBF online Help file, "Control Key Shortcuts" topic, under the "General Topics" section.

MORE INFORMATION

Control Key Shortcuts

Control key shortcuts enable you to choose certain menu options without displaying the menu. All you need to do is press the Control key combination that is listed next to the desired option on the menu. In most cases, these Control key shortcuts choose menu options that affect the frontmost window.
