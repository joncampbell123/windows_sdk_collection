PRB: Word 5.1 Won't Print with MXText and Reserved.MLB

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c

SYMPTOMS

Microsoft Word version 5.1 for the Macintosh will not print with both Reserved.MLB and MXText in the Extensions folder.

RESOLUTION

If you are running FoxPro at the same time you are running Word, remove MXText from the Extensions folder while you are using FoxPro. MXText is a file supplied by Fontographer 3.2.
