How to Print Icons in a Report

 Last reviewed: June 26, 1995
Article ID: Q105598
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5 and 2.5a

SUMMARY

You can print an icon inside a FoxPro for Windows report as long as the icon is imported from an OLE server. For example, you can use the following procedure to print an icon:

  1. Open Object Packager.

  2. Choose Insert Icon.

  3. Select an icon.

  4. From the Edit menu, choose Copy.

  5. In FoxPro, open the general field and choose Paste from the Edit menu.

  6. Place the general field on the report (using the Picture tool in the toolbox), and print the report.

MORE INFORMATION

The Object Packager is located in the Accessories group in the Windows Program Manager. You can use it do the following:

  • Insert a complete document as a package in a destination document.
  • Insert part of a document as a package in a destination document.
  • Insert an MS-DOS command line to be carried out in a destination document.
  • Customize the icon that represents a packaged object.
  • Change the label of an icon.
