FIX: "Printer Not Ready" When Label or Report Wizard Invoked

 Last reviewed: October 20, 1997
Article ID: Q114272
2.6x WINDOWS kbenv kbprint kbfixlist kbbuglist kberrmsg

The information in this article applies to:

  • Microsoft FoxPro for Windows, versions 2.6x

SYMPTOMS

A window with the title "FoxPro Error" is displayed when the Label Wizard or the Report Wizard is invoked. The window contains the following information: 

   Line Number: 83
   Program:     GETWINWID
   Error:       Printer not ready
   Source:      ...

CAUSE

This problem occurs when no printer drivers are installed.

RESOLUTION

Install a printer driver, as follows:

  1. Open the Windows Control Panel.
2. Double-click the Printers icon. 3. Install at least one printer. 4. Choose the Close button in the Printers dialog box.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Visual FoxPro 3.0 for Windows.
