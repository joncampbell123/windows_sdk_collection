PRB: Product Sizes Listed in Installation Program Misleading

 Last reviewed: April 18, 1995
Article ID: Q116031
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, 2.5b, 2.6

SYMPTOMS

The installation program, INSTALL.EXE, lists each product, its description, and size as follows: 

   Product         Description                              Size
   --------------------------------------------------------------

   FoxPro          Standard (16-bit) version of FoxPro      1881K
   Help file       FoxPro Help file                         2608K
   FoxPro (X)      Extended (32-bit) version of FoxPro      1823K
The sizes listed are for those specific files of each product only. Because each product requires additional support files, the size listed does not reflect the total amount of disk space needed if you choose Y to install that product.

NOTE: The sizes listed above are for FoxPro version 2.6. These sizes will vary for different versions of FoxPro.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. At the MS-DOS prompt, type the command, "DIR" (without the quotation marks). Note the last line of the display, which is the amount of bytes free on the hard disk.

  2. Run INSTALL.EXE.

  3. Proceed to the fourth screen, which lists each FoxPro product and its size. Note the size listed for the Standard version of FoxPro.

  4. Choose Y to install FoxPro.

  5. Choose N when prompted to install the Help file.

  6. Choose N when prompted to install FoxPro (X).

  7. Choose N when prompted to change the PATH.

  8. When prompted to install optional components, press ESC to quit the installation program.

  9. At the MS-DOS prompt, type the command, "DIR" (without the quotation marks). Note the last line of the display, which is the amount of bytes free on the hard disk.

  10. Subtract the amount of free disk space after installation, noted in step 9, from the amount of free disk space before installation, noted in step 1. Divide this amount by 1024 to get the size in kilobytes.

Note that the number of kilobytes used is much greater than the size listed to install the Standard version of FoxPro, noted in step 3 above.
