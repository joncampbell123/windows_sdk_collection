Last 2 "Save..." Options Listed Twice in SQL Wizard Help

 Last reviewed: April 18, 1995
Article ID: Q117720
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.6

SUMMARY

In online Help, the description of the SQL Query Wizard's last screen lists the Save And Run Query and Save And Create An AutoReport radio buttons twice. Furthermore, the two descriptions of Save And Create An AutoReport differ.

MORE INFORMATION

Location of Documentation Errors

These errors are located at the bottom of the "SQL Query Wizard" Help topic, under "Step 6 of 6: Previewing, saving, and browsing your query."

NOTE: The "SQL Query Wizard" topic is located near the bottom of Help, under "The FoxPro Interface" section. This section is located below "The FoxPro Language" section, which contains commands, functions, and system memory variables.

Save And Run Query Error

The following text is listed twice: 

   - Save and run query
It is listed after Save Query For Later Use and again after the first Save And Create An AutoReport option.

Save And Create An AutoReport Error

The following text is listed twice: 

   - Save and create an AutoReport
It is listed after each Save And Run Query option. The first listing contains the following description: 

   Creates and runs an automatic report based on the fields in the
   query.
The second listing contains the following description, which is more complete: 

   Saves the query and creates and runs an automatic report based on
   the fields in the query.
