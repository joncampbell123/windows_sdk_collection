dBASE NOORGANIZE Clause Ignored in FoxPro

 Last reviewed: June 28, 1995
Article ID: Q115282
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

SUMMARY

The dBASE NOORGANIZE clause of the BROWSE, CHANGE, INSERT, and APPEND commands is ignored in FoxPro.

MORE INFORMATION

In dBASE IV 1.5, the NOORGANIZE clause suppresses the Organize menu pad in the menu bar.

Steps to Reproduce Behavior

NOTE: These steps assume that drive C contains a main FoxPro directory called FPW26.

Type the following commands in the Command window: 

   USE C:\FPW26\TUTORIAL\CUSTOMER.DBF
   BROWSE NOORGANIZE
Note that the BROWSE NOORGANIZE command behaves identically to the BROWSE command.
