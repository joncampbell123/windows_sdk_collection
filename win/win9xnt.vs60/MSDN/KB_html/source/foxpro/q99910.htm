Color Table by Color Pair Information Is Missing

 Last reviewed: April 29, 1996
Article ID: Q99910
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SUMMARY

The FoxPro for Windows documentation does not include the color pair listings for each color scheme. This information is necessary to control the color of each screen element in FoxPro for Windows. Documentation is available by choosing the topic "COLORS" (under the "MISCELLANEOUS" topic) in FoxPro for Windows Help.

MORE INFORMATION

   Color                 Code
   --------------------------

   Black                  N
   Blank                  X
   Blue                   B
   Brown                  GR
   Cyan                   BG
   Green                  G
   Magenta                RB
   Red                    R
   White                  W
   Yellow                 GR+
On monochrome monitors, only four "colors" are available: white (W), black (N), underlined (U) and inverse video (I).

A plus sign (+) denotes high intensity for the foreground color.
