BUG: "Unable to Create Item in Program Manager Group" Error

 Last reviewed: April 29, 1996
Article ID: Q102200
The information in this article applies to:
  • Microsoft FoxPro Distribution Kit for Windows, versions 2.5, 2.5a, 2.5b, 2.6
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6

SYMPTOMS

The Setup program (SETUP.EXE) created by SETUP.APP (the FoxPro Distribution Kit for Windows Setup Wizard) fails with the following error message when you are trying to create a program item in Program Manager: 

   Unable to create item in Program Manager group: '(program item name)'
NOTE: This article does not apply to Visual FoxPro. In Visual FoxPro, the Setup Wizard file is WZSETUP.APP, and SETUP.EXE is not created by the Setup Wizard.

CAUSE

The Program Manager Icon Description specified in SETUP.APP contains one of the following characters (a parenthesis or a bracket): 

   ( ) [ ]

RESOLUTION

Run SETUP.APP again and remove the parenthesis or bracket from the Program Manager Icon Description.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
