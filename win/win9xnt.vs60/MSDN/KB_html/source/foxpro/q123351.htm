BUG: Error When Using FoxDoc: Error 22 Opening error log file

 Last reviewed: May 24, 1996
Article ID: Q123351
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

When generating documentation in FoxDoc the following error occurs: 

   Error 22 Opening error log file. Bad Directory?

CAUSE

This error occurs if an existing file named ERROR.DOC exists in the output folder specified in FoxDoc.

RESOLUTION

Select an empty folder or a folder that does not contain a file named ERROR.DOC when selecting an Output Folder in FoxDoc.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Start FoxDoc by selecting it from the Program menu, or by typing FoxDoc in the Command window.

  2. Select a Main File to document, a Program Folder, and a Data Folder.

  3. Select an Output Folder that already contains an ERROR.DOC file.

  4. Generate using FoxDoc.
