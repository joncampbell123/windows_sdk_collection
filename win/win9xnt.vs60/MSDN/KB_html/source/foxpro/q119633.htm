Files That Can Be Distributed with a CK Application

 Last reviewed: April 30, 1996
Article ID: Q119633
The information in this article applies to:
  • Microsoft FoxPro Connectivity Kit, versions 2.5x, 2.6
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6
  • Microsoft FoxPro for MS-DOS, versions 2.5x, 2.6

SUMMARY

A licensed user of both the FoxPro Distribution Kit and the Connectivity Kit can distribute the FPSQL.FLL file and the ODBC Setup disk that comes with the Connectivity Kit.

MORE INFORMATION

The CKREADME.TXT file states that a licensed user of both the FoxPro Distribution Kit and the Connectivity Kit can freely distribute the FPSQL.PLB and/or FPSQL.FLL library file(s) with his or her distributed application(s).

However, an application developed with the Connectivity Kit requires that the components of ODBC be installed. A user will have these components only if he or she has installed an application that installs the ODBC components, such as Microsoft Word 6.0, Microsoft Excel 5.0, or Microsoft Visual Basic 3.0 Professional Edition. To work around this, the ODBC Setup Disk can be distributed also; the ODBC Setup Disk can be used to install the necessary ODBC components and allow the user to install the SQL Server or Oracle ODBC driver.
