PRB: Wyse 995 Multiple-Port Serial Card Problems Under Xenix

 Last reviewed: May 22, 1995
Article ID: Q128392
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

NOTE: Products mentioned in this article are manufactured by vendors independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.

SYMPTOMS

FoxPro for UNIX, when run on XENIX with a Wyse 995 multiple port serial I/O card, has problems with keyboard input from terminals connected to the serial card.

For example, when you start FoxPro on a terminal, it might just show a blank screen or you may lose keyboard input.

CAUSE

Wyse no longer supports this board but has confirmed this problem with FoxPro and FoxBase+.

RESOLUTION

Because Wyse no longer supports this board, there will not be any updates for this board or the firmware and drivers on this board. This can affect terminals and the XENIX console. You need to use a different multiple-port serial I/O card to resolve this problem.
