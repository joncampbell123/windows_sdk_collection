SAMPLE: Cksamp.exe Contains FoxPro 2.6 Cksamp.app Source Code

 Last reviewed: February 20, 1998
Article ID: Q117775 
2.60    | 2.60
WINDOWS | MS-DOS kbprg kbcode kbfile

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

Cksamp.exe is a sample that provides source code for the sample application Cksamp.app shipped with previous versions of FoxPro.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library:

~ Cksamp.exe

For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

Service-Specific Instructions

To obtain Cksamp.app, download Cksamp.exe, which is a compressed, self- extracting file. After you download Cksamp.exe, run it to extract the file it contains.
