How to Retain Memo Editing Window Preferences

 Last reviewed: April 5, 1995
Article ID: Q128515
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5, 2.5a, 2.5b, 2.6, 2.6a

SUMMARY

FoxPro for MS-DOS will not save Memo Editing position and size preferences until you tell FoxPro to "Save these preferences as default for Memos" and "Save Preference" at least once. Follow the instructions in this article to save the Memo Editing windows position and size.

MORE INFORMATION

Step-by-Step Procedure

  1. Make sure Set Resource is set to ON.

  2. Open a table (for example, Salesman) that contains a Memo field.

  3. Type Browse in the Command window.

  4. Open the Memo Editing window by pressing CTRL+PAGE DOWN or by double-clicking the Memo field.

  5. After opening a Memo Editing window, choose Preferences from Edit Menu, and select both "Use these preferences as default for Memos" and "Save Preference."

At this point, FoxPro saves the Memo Editing preference to the Resource file (usually FOXUSER.DBF or .FPT) with an id of WINDMEMO.

After completing all five steps, all new Memo Editing windows opened will absorb the position and size preference that was previously saved. Any previously opened table that had a Memo Editing window position and size preference saved will not be affected.
