Error in Description of USE ... NOUPDATE Clause

 Last reviewed: June 27, 1995
Article ID: Q110126
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, and 2.5b
  • Microsoft FoxPro for MS-DOS, versions 2.5, 2.5a, and 2.5b

SUMMARY

In the "Language Reference" and in the online help system, the description of the USE command's NOUPDATE clause is missing the word "and." The incorrect description is: 

   Including NOUPDATE prevents changes to the table/.DBF its
   structure.
The description should be: 

   Including NOUPDATE prevents changes to the table/.DBF and
   its structure.

MORE INFORMATION

Locations of Documentation Errors

  • FoxPro for MS-DOS and Windows "Language Reference," versions 2.5, 2.5a, and 2.5b, page L3-1112
  • FoxPro 2.5, 2.5a, and 2.5b for Windows and MS-DOS online help (FOXHELP.DBF), "USE Command" topic

NOTE: The FoxPro 2.5, 2.5a, and 2.5b for Windows FOXHELP.HLP online Help file is correct.
