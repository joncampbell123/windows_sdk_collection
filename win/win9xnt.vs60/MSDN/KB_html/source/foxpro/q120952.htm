System Requirements for FoxPro 2.6 Runtime for UNIX

 Last reviewed: March 23, 1995
Article ID: Q120952
The information in this article applies to:
  • Microsoft FoxPro Runtime for UNIX, version 2.6

The FoxPro Runtime for UNIX enables FoxPro for UNIX developers to build executables, applications, and other FoxPro compiled programs. Listed below are the system requirements for installing and running the FoxPro for UNIX Runtime.

Hardware Requirements

  • 386SX processor or higher
  • 4 MB of RAM for first user, plus 1 MB per additional user
  • 2 MB of free disk space
  • One 5.25-inch high-density (1.2 MB) or one 3.5-inch high-density (1.44 MB) floppy disk drive

Software Requirements

  • Microsoft FoxPro 2.6 for UNIX
  • SCO UNIX System V release 3.2.2 or higher -or- SCO XENIX System V release 2.3.4 or higher

Additional reference words: FoxUnix 2.60 run-time system requirements
