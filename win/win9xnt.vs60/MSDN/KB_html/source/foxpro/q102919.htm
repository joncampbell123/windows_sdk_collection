How to Insert Messages Through the Menu Builder Interface

 Last reviewed: April 17, 1995
Article ID: Q102919
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5x, 2.6, 2.6a

The Menu Builder interface of FoxPro for MS-DOS does not have built-in functionality for inserting messages.

To display messages using the Menu Builder, use the following workaround:

  1. In the Prompt field, type the menu prompt.

  2. On the same line, type the MESSAGE clause and enclose it in quotation marks.

  3. On the same line (after the closing quotation mark), add the text of the message.

For example, type the following in a menu prompt: 

   Test Pad" Message "Awesome!
The following code will be generated by GENMENU: 

   DEFINE PAD whatever OF _MSYSMENU PROMPT "Test Pad" MESSAGE "Awesome!"

Additional reference words: FoxDos 2.00 2.50 2.50a 2.50b 2.60 2.60a
