FIX: Changes to Windows Menu Font not Reflected in FoxPro

 Last reviewed: October 29, 1997
Article ID: Q139596
2.5x 2.6x WINDOWS kbenv kbui kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Windows, versions 2.5x, 2.6x

SYMPTOMS

In Windows 95, changes made to the Windows menu font while FoxPro version 2.x for Windows is running do not show up in FoxPro.

CAUSE

FoxPro for Windows does not check for changes made to the Windows menu font.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Visual FoxPro 3.0 for Windows.s

MORE INFORMATION

Steps to Reproduce Problem

  1. Start, and then minimize FoxPro.

  2. Right-click the Windows 95 desktop, click Properties, and then click the Appearance tab.

  3. In the Item list, click Menu, and then select a different font and size. Click OK.

  4. Switch to FoxPro, and then click the Edit menu.

Note that the Edit menu does not use the new font.
