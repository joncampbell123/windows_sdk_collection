PRB: "Cannot Update the Cursor" Error When Running Browser.app

 Last reviewed: November 20, 1995
Article ID: Q139864
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SYMPTOMS

When attempting to run Browser.app, you receive this error message: 

   Cannot Update the Cursor

CAUSE

Browser.app is being run from a network, and the user doesn't have write privileges.

RESOLUTION

Move Browser.app to the workstation, or give the user write privileges to the directory were it resides on the network.

STATUS

This behavior is by design.

MORE INFORMATION

Browser.app creates a .dbf file and an .fpt file that hold information and preferences for each class that you examine.
