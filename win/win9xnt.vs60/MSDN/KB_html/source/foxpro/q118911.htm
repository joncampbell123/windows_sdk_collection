FoxPro for UNIX Configuration File Is Case Sensitive

 Last reviewed: August 28, 1995
Article ID: Q118911
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

In FoxPro for UNIX, the configuration file must be named "config.fpu" (all lowercase). If you use CONFIG.FPU instead and then choose About from the System menu, FoxPro will show no configuration file in use. SYS(2019) will also show no evidence of a configuration file being used.
