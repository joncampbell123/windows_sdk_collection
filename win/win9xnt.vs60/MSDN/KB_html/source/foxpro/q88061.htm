FIX: Problem Copying Directory Using Filer

 Last reviewed: October 14, 1997
Article ID: Q88061
2.00 MS-DOS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

Using the Filer in tree mode to copy a directory that contains hidden system files (for example, IO.SYS), FoxPro displays the message "<filename> has system and hidden attributes." Choosing the "Skip this file" option for files with the system and hidden attributes copies the directory structure, but does not copy the files.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FoxPro version 2.0 for MS-DOS. This problem was corrected in Microsoft FoxPro version 2.0 for MS-DOS released March 25, 1992.
