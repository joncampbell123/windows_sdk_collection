FoxDoc Limits for FoxPro for MS-DOS & FoxPro for Windows

 Last reviewed: April 29, 1996
Article ID: Q104253
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

The limits for FoxDoc can be found by bringing up FoxDoc under FoxPro for MS-DOS and pressing F1 three times when the initial screen appears. The limits will then be listed. This feature is not available under FoxDoc running under FoxPro for Windows. The limits listed are also the same for FoxDoc running under FoxPro for Windows.

The limits of FoxDoc running under FoxPro for MS-DOS and FoxPro for Windows are as follows: 

   Maximum program files:               1024
   Maximum of each other file type:     255
   Maximum tokens:                      Unlimited
   Maximum length of a token:           20
   Maximum number of keywords:          1600
   Maximum length of keyword:           15
   Maximum levels of program nesting:   20
   Maximum number of database fields:   1500
   Maximum length of filename:          79
   Maximum number of macro definitions: 100
   Maximum program/data directories:    25

Additional reference words: FoxDos FoxWin 2.50 2.50a capacities limitations
