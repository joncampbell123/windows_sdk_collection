FILE: Files the Setup Wizard Installs

 Last reviewed: January 2, 1998
Article ID: Q174186
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a

SUMMARY

The files for this article provide information on the components installed by the Visual FoxPro 5.0 and 5.0a Setup Wizard under various options chosen by the user.

The following file is available for download from the Microsoft Software Library: 

 ~ Swfiles.exe (size: 32109 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

MORE INFORMATION

Setupwiz.xls and Setwiz5a.xls are Microsoft Excel 97 and 5.0/95 workbooks that contain a list of files installed by the Visual FoxPro Setup Wizard. This is an accurate list of files for the most current version of Visual FoxPro. However, if a newer version of the Setup Wizard for Visual FoxPro 5.0 becomes available, these files may change. The columns listed across the top the spreadsheet indicate the various options you might choose. The specific files are listed along the right-hand side of the spreadsheet. The asterisks indicate the installation of a file under that particular option.
