How to Troubleshoot "Please Call Technical Support" Error

 Last reviewed: April 29, 1996
Article ID: Q95249
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

Any error message in the online Help file or in the manual that states "Please call technical support" indicates a memory-related issue.

To troubleshoot this error, check for the following:

  • Terminate-and-stay-resident (TSR) programs (clean boot)
  • Incompatible or incorrectly set memory manager (clean boot)
  • Corrupt database file
  • Bad memory board or chips
  • EMSNET or XMSNET drivers instead of NETx on Novell networks
  • Early version of product (upgrade or patch)

NOTE: If, after exhausting the above, this error can be reproduced each time, break the code down to the smallest portion that can reproduce the problem so that it can be tested by Testing.
