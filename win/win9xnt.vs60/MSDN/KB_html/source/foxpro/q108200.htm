Requirements to Create External Library (.MLB) Files

 Last reviewed: May 17, 1996
Article ID: Q108200
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b

An .MLB file is an external library file that has been compiled for use within FoxPro for Macintosh through the SET LIBRARY TO command. The software requirements for compiling .MLB files are:

  • Version 3.3 or later of the Macintosh Programmer's Workbench Development System (MPW) compiler
  • The Apple Shared Library Manager (ASLM)
  • System 7.0.1 or later
  • 8 MB of RAM recommended
