BUG: Menu Still Accessible with SKIP Set to .T.

 Last reviewed: June 2, 1995
Article ID: Q119760
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5x, 2.6

SYMPTOMS

After disabling the menu bar, you can use the ALT key to activate and access options in the rightmost menu pad.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Start FoxPro.

  2. In the Command window, type: 

          SET SKIP OF MENU _MSYSMENU .T.

  3. Press and release the ALT key, and then press and release any other key.

  4. Note that the Run menu has now appeared, and options within the menu are completely accessible.
