BUG: Encryption Causes "Warning: Too Many Help Topics"

 Last reviewed: June 2, 1995
Article ID: Q114461
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.5

SYMPTOMS

If you have created a custom Help file that is included in your project and you have selected Encrypt in the Build Options for the project, you may get the message "Warning: Too Many Help Topics" when you try to invoke Help. The Help topics are still displayed and can be selected, however.

RESOLUTION

To work around this problem, turn encryption off, or exclude the help database from the project.

STATUS

Microsoft has confirmed this to be a problem in FoxPro version 2.5 for MS-DOS. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
