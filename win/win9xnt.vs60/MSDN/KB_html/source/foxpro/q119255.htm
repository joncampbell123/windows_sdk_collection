Help Incorrectly Says MEMLIMIT Can Be Used in FoxPro/Mac

 Last reviewed: May 29, 1996
Article ID: Q119255
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c

SUMMARY

The "Performance Tips" online Help topic incorrectly states that you can use the MEMLIMIT command to control the amount of memory used by FoxPro.

However, FoxPro for Macintosh ignores the MEMLIMIT command, which is used only by FoxPro for Windows or FoxPro for MS-DOS with a DPMI-compliant memory manager.

MORE INFORMATION

The amount of memory available to FoxPro for Macintosh is controlled by the Minimum Size and Preferred Size text boxes in the Get Info dialog box for the "Microsoft FoxPro" file.

For more information about configuring the amount of memory available to FoxPro for Macintosh, please see the following article(s) in the Microsoft Knowledge Base:

     TITLE     : FastTips for FoxPro for Mac: Installation &
