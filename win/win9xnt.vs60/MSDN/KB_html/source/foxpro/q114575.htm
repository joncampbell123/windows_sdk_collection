PRB: Now Utilities Conflicts with FoxPro for Macintosh

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.5b

SYMPTOMS

The Now Utilities program by Now Software contains several individual utilities that may produce anomalous behavior in FoxPro for Macintosh.

For example, NowMenus allows the user to define and redefine shortcut keys for any menu. This feature does not work in FoxPro for Macintosh.

MORE INFORMATION
