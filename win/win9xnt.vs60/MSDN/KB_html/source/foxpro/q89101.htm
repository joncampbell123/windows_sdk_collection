Using SET RELATION Command in FoxPro 1.x

 Last reviewed: April 17, 1995
Article ID: Q89101
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 1.0, 1.01, and 1.02

In FoxPro versions 1.x, only one-to-one relationships can be set. With the SET RELATION TO command, the parent database will point to deleted records in the child database. This will occur even if SET DELETED is ON.

This command has been enhanced in FoxPro version 2.0, and the deleted status of records in related files is respected.

Additional reference words: FoxDos 1.00 1.01 1.02 1.x
