PRB: FoxPro RQBE Report Repeats One Record

 Last reviewed: December 11, 1996
Article ID: Q87681
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

The report generated by the Microsoft FoxPro Relational Query By Example (RQBE) displays and prints the same record continuously.

CAUSE

The report definition was based on the alias fields of the database and not the query fields.

RESOLUTION

Modify the field definitions in the report to refer only to the field and not to a database alias, or create a new report with the work area of the query selected.
