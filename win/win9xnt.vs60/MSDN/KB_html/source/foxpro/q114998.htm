BUG: Next Button Does Not Prompt for Permission to Discard

 Last reviewed: June 27, 1995
Article ID: Q114998
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.6, 2.6a
  • Microsoft FoxPro for MS-DOS, versions 2.6, 2.6a

SYMPTOMS

In the SQL Query Wizard, you are not asked if the expression should be discarded when you select the Next button.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Load FoxPro 2.6 for Windows or the Extended (32-bit) version of FoxPro 2.6 for MS-DOS (FOXPROX.EXE).

  2. Start the SQL Query Wizard by choosing Wizard from the Run menu and then choosing Query.

  3. Proceed to step 5 in the Query Wizard (limiting conditions).

  4. Type something in the Value box.

  5. Choose the Next button.

    A prompt will appear, asking if you want to discard the expression.

  6. Choose the No button.

  7. Choose the Next button.

This time the expression is discarded. No prompt is displayed asking if the expression should be discarded.
