Error in Lang Ref: BROWSE NOCLEAR Is Not Ignored

 Last reviewed: April 29, 1996
Article ID: Q98365
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for MS-DOS, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6, 2.6a

According to the "Language Reference" manual, the BROWSE NOCLEAR command is ignored in FoxPro for Windows. This is incorrect. The BROWSE NOCLEAR command acts as documented.

Location of Documentation Errors

Page L3-244 of the FoxPro for Windows and MS-DOS "Language Reference" Page 216 of the FoxPro for Macintosh "Language Reference"
