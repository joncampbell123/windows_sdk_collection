Location of Foxtools.fll Information

 Last reviewed: December 2, 1996
Article ID: Q136945
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

Information about the FOXTOOLS API library is included in the Foxtools.hlp Help file located in the Vfp\ToolS directory.

NOTE: The FOXTOOLS functions are not supported by Microsoft Technical Support either electronically or by telephone. Many of these functions are included for backward compatibility with FoxPro version 2.6. Some functions behave differently depending on the platform for which they are used.
