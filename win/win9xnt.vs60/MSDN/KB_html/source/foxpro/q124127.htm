PRB: File Error: Cannot find <filename>.DRV

 Last reviewed: April 30, 1996
Article ID: Q124127
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a

SYMPTOMS

When opening a report, you see the following message: 

   " File Error
   Cannot find <filename>.DRV"

CAUSE

The report was created on a different computer, with the Specific Printer option selected under Print Setup, and the computer opening the report does not have the driver for that printer installed.

RESOLUTION

Press the Close button on the above message. When the message reappears, press the close button again. The report will then open. From the Report menu, choose Page Layout. The error will appear again; press the Close button to clear it. From the Page Layout dialog, press the Print Setup push button. You can now choose either the default printer or a specific printer that uses a driver that has been installed on this computer.

NOTE: This behavior does not occur under Windows 95.

STATUS

This behavior is by design.
