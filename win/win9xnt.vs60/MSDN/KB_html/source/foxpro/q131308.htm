BUG: ERROR: Cannot Open OLE Server and OLE Link Failure

 Last reviewed: December 18, 1995
Article ID: Q131308
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a

SYMPTOMS

After installing Microsoft Office version 4.x and attempting to create a graph in Microsoft FoxPro for Windows, you may receive this error message: 

   Cannot open OLE Server - OLE link failure.
This problem can also appear when you try to use UPDATEGRPH to update a graph title in FoxPro for Windows after Microsoft Office version 4.x has been installed. The title of the graph won't change.

CAUSE

Microsoft FoxPro for Windows is an OLE 1.0-compliant application. Microsoft Graph 3.0, which ships with FoxPro for Windows, is OLE 1.0-compliant. However, Microsoft Graph 5.0, which ships with Microsoft Office, is OLE 2.0- compliant.

RESOLUTION

In order for FoxPro for Windows to use the correct version of Microsoft Graph, FoxPro for Windows must be reinstalled.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
