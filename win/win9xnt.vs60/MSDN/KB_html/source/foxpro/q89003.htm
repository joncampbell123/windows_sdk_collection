FoxPro 2.0 for MS-DOS Basic Installation Files

 Last reviewed: April 17, 1995
Article ID: Q89003
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SUMMARY

This article includes a list of the files installed during basic installation. File sizes and descriptions are included. 

Installation                                Disk space requirements


Basic installation, standard LAN product    Approximately 4.6 MB

Basic installation, extended product        Approximately 4.6 MB

Installation of both products               Approximately 6.4 MB

MORE INFORMATION

Filename          Size      Description


ADDUSER.APP       20197     Used to configure workstations on a
                            network. Basic Installation with the LAN
                            version.

DEMO.APP           2129     Self-running demo application.

DEMO.BAT              8     Batch file to start the demo.

FOX.EXE            8582     Loader for any version of FoxPro.

FOXAPP.APP       112749     Application generator.

FOXHELP.DBF       51065     Help file.

FOXHELP.FPT     1678744     Help memo file.

FOXL.EXE           8070     Loader for the multiuser version.

FOXPROL.EXE      480786     Main executable file: multiuser

FOXPROL.OVL     1300608     Overlay file for the multiuser version.

FOXPROLX.EXE    1747582     Executable file for Extended version.

FOXSWAP.COM        5971     Memory manager utility used to make more
                            memory available when using a RUN command.

FOXUNPAK.EXE      30241     Used to "unpack" files from the
                            distribution floppy disks.

GENMENU.PRG       53870     Program used to generate a menu when using
                            the Menu Builder.

GENPD.APP        297868     Printer drivers application.

GENSCRN.PRG      138868     Program used to generate a screen when
                            using the Screen Builder.

GENXTAB.PRG       26440     Program used to generate cross-tab
                            queries.

INSTALL.APP       41309     Application that installs the optional
                            applications.

INSTALL.DBF        4994     Database used to run the install
                            application.

INSTALL.FPT        4047     Memo file for INSTALL.DBF

PROAPI16.EXE      11834     Used to go from 32-bit mode to 16-bit
                            mode. Used by the printer driver.

PROCOLOR.APP     131085     Application used to customize colors.

SNIPFMT.APP       12695     Allows users to format code snippets in
                            the Screen or Menu Builder.
