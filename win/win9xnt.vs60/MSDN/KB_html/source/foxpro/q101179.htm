How to Suppress the Banner Page from Printing

 Last reviewed: April 17, 1995
Article ID: Q101179
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

SUMMARY

The online Help file incorrectly states that the following command can be used to suppress the banner page: 

   SET PRINTER TO [\\SPOOLER[\N]]
The correct command to suppress printing of the banner page is as follows: 

   SET PRINTER TO [\\SPOOLER[\NB]]
NOTE: The "Language Reference" manual has the correct no banner option on page L3-973.

Additional reference words: FoxDos 2.50 2.50a docerr
