BUG: SHARED Keyword Not Capitalized by BEAUTIFY

 Last reviewed: April 30, 1996
Article ID: Q127920
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

The SHARED keyword is not capitalized when BEAUTIFY is run on a program that contains the word SHARED.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Create a program by using this command:

    use customer shared

  2. Choose BEAUTIFY from the Program menu.

  3. Leave the default choices in effect (uppercase for keywords, lowercase for variables). The resulting command is:

    USE customer shared
