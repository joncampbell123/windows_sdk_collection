Scancode Mode Requires UNIX Release 3.2.4

 Last reviewed: August 28, 1995
Article ID: Q121408
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

Page 16 of the FoxPro for UNIX "Installation and Configuration Guide" states that UNIX System V 3.4 or later is needed for scancode mode support. This is incorrect. In fact, UNIX System V 3.2.4 or later is needed for scancode mode support.

MORE INFORMATION

Page 1 of the "Installation and Configuration Guide" correctly states that UNIX System V 3.2.4 or later is needed for scancode mode support.

REFERENCES

FoxPro for UNIX "Installation and Configuration Guide."
