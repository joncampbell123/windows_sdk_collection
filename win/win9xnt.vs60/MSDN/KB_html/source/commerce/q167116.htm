Is There an OPP Component to Send E-mail to a Shopper?

 Last reviewed: February 25, 1998
Article ID: Q167116
The information in this article applies to:
  • Microsoft Commerce Server, version 2.0

Microsoft Commerce Server version 2.0 does not have an e-mail Order Processing Pipeline (OPP) component. However, the following is a list of alternatives:

  • Use the stage independent ExecuteProcess component in the Accept stage to run a process that can send e-mail. For example, you can use blat, http://gepasi.dbs.aber.ac.uk/softw/Blat.html, to send e-mail messages from batch files.

    The third-party product discussed here is manufactured by a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

  • Write your own Scriptor OPP component. Take a look at the following Microsoft web site, for Active Server sample applications: 

        http://www.microsoft.com/ithome/resource/exchange/active/default.htm
  • Use the Microsoft SQL Server extended stored procedure xp_sendmail.
