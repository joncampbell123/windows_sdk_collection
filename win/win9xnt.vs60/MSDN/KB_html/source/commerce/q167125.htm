Error Running Setup or Host Admin: "Could Not Secure Store"

 Last reviewed: September 10, 1997
Article ID: Q167125
The information in this article applies to:
  • Microsoft Commerce Server, version 2.0

SYMPTOMS

You may receive the following error during Commerce Server Setup or while while running the host administrator: 

   Could not secure store.

WORKAROUND

This error may occur if you are running Commerce Server Setup on a backup domain controller (BDC). Commerce Server Setup only supports installation on a non-domain controller or on a primary domain controller (PDC). It is recommended that you do not install Commerce Server on a BDC. It is generally not a good idea to install any Internet-related software on a PDC or a BDC. The Readme.txt file included with Commerce Server 2.0 has a possible workaround, but it has not been tested and is not supported.

If you must install Commerce Server on a BDC, please refer to the following Knowledge Base article for step by step instructions: 

   ARTICLE-ID: Q173154
   TITLE     : INF: Installing Commerce Server on a Backup Domain Controller
