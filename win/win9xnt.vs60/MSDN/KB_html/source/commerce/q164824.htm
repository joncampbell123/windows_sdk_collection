Commerce 2.0 OPP Components Not Supported in Transaction Server

 Last reviewed: July 16, 1997
Article ID: Q164824
The information in this article applies to:
  • Microsoft Commerce Server, version 2.0

Microsoft Commerce Server 2.0 Order Processing Pipeline(OPP) components are not supported with Microsoft Transaction Server.

Theoretically, any COM component can be installed in Transaction Server, but because Commerce Server 2.0 OPP components have not been tested with it, they are not supported.
