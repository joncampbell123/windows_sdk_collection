Error: "Cannot Execute Script" Running OPP

 Last reviewed: July 16, 1997
Article ID: Q154130
The information in this article applies to:
  • Microsoft Commerce Server, version 2.0

Q. I am getting a "Cannot Execute Script" error while running the Order 

   Processing Pipeline (OPP). What should I do?
A. First, check the OPP log to determine the stage where the script is 
   failing. Then, check the script for any changes you may have made. It is
   possible to have multiple Scriptor OPP components in a stage, so check
   all the scripts used in a stage. Also, there may be an incompatible
   scripting engine on the server. If you have a beta version of a browser
   that may have installed a different scripting engine, remove that
   browser and try running the OPP again.
