Passing Linker Options from the Compiler Command Line

The information in this article applies to:

  • The Microsoft C/C++ Compiler (CL.EXE) included with: 

        - Microsoft C for MS-DOS, versions 6.0, 6.0a, and 6.0ax
    - Microsoft C for OS/2, versions 6.0, and 6.0a
    - Microsoft C/C++ for MS-DOS, version 7.0
    - Microsoft Visual C++ for Windows, versions 1.0 and 1.5

There are compiler command line options that will pass link options to the linker, for example the /Fm compiler switch will pass the /MAP option to the linker. In addition, there is a /link compiler option which directly passes the /NOD switch or library names to the linker. For more information see the Enviornment and Tools book or the online help.
