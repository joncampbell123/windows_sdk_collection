INFO: List of .DLL and .VBX Files in 16-bit Development Products

 Last reviewed: July 22, 1997
Article ID: Q124791
The information in this article applies to:
  • Microsoft Visual C++ for Windows, versions 1.0, 1.5, 1.51, 1.52
  • Professional Edition of Microsoft Visual Basic for Windows, version 3.0
  • Microsoft Visual Control Pack
  • Microsoft ODBC SDK 2.01a

SUMMARY

This article contains the list of the .DLL and .VBX files that ship with the Microsoft development products listed at the beginning of this article. You can use this list to confirm which versions are installed.

NOTE: Not all of these files will be installed during setup; this is the list of files that ship with the products.

To check the version installed or to find these files on a CD-ROM disk, do the following:

  1. Run File Manager.

  2. Select the directory containing the installed product or the CD-ROM.

  3. Choose Search from the File menu.

  4. Enter *.DLL or *.VBX in the Search For edit box.

  5. To find the version, select (highlight) the file and choose Properties from the File menu.

To determine which version is installed, compare the file size and version numbers with those given below. The date and time stamps are provided primarily as a rough estimate, because it is possible for these values to change after installation.

NOTE: There may be duplicate files in different directories. For exact duplicates, only one listing for that file is given. However, there are two situations where confusion could arise. The first situation occurs when two slightly different versions are shipped with a product. For example: CTL3D.DLL is added to one of the samples in a slightly newer version than the one normally installed. The second situation appears when two files with the same name and version are present with only the file size differing. The larger of the two files is a debug version of the file; to help avoid confusion, the larger debug versions are marked with an asterisk (*) at the beginning of the line.
