Exit Codes for the Microsoft C Optimizing Compiler

kbprg

The information in this article applies to:

  • The Microsoft C/C++ Compiler (CL.EXE) included with: 

        - Microsoft C for MS-DOS, versions 6.0, 6.0a, and 6.0ax
    - Microsoft C for OS/2, versions 6.0 and 6.0a
    - Microsoft C/C++ for MS-DOS, versions 7.0
    - Microsoft Visual C++ for Windows, versions 1.0 and 1.5

The compiler returns the following exit codes: 

   Code       Meaning
   ----       -------

    0         No error
    2         Command line and source code errors.
    4         System level error (such as out of disk space or
              compiler internal error), or a detected control-C.
