DOCERR: /J Not Documented in Visual Workbench User's Guide

 Last reviewed: July 22, 1997
Article ID: Q118680
1.00 1.50 WINDOWS kbref kbdocerr

The information in this article applies to:

  • Microsoft Visual C++ for Windows, versions 1.0 and 1.5

The documentation for the "Visual Workbench Users Guide" incorrectly omits the /J option in its "Alphabetic Guide to Compiler Options." This listing is found on page 52 of the version 1.0 guide and page 32 of the version 1.5 guide.

The /J option is still valid for the Visual C++ for Windows compiler, versions 8.0 and 8.0c. The option is documented correctly in other locations of the printed documentation and in the online Help.
