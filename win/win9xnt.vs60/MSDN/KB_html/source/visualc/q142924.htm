INFO: What's New in Visual C++ 1.52C

 Last reviewed: August 26, 1997
Article ID: Q142924
The information in this article applies to:
  • Microsoft Visual C++ for Windows, version 1.52C

SUMMARY

Visual C++ 4.0 ships with 16-bit Visual C++ 1.52C for Windows. This is a minor upgrade from Visual C++ 1.52B. Visual C++ 1.52C is only available when you purchase Visual C++ 4.0. New purchases of Visual C++ 1.5 for Windows will be version 1.52B, which does not include the CDK (Control Development Kit).

MORE INFORMATION

What's new to Visual C++ 1.52C:

  • N2D.bat now prints a warning before installing the 16-bit debug kernels.
  • Sysincl.dat now lists four more headers (Afxctl.h, Afxctl.inl, Olectl.h, and Olectlid.h).
  • Various copies of Ctl3d.dll and Ctl3dv2.dll were updated from 2.29 to 2.31.

Visual C++ 1.52C includes the 16-bit CDK with the following changes:
  • 32-bit specific files have been removed.
  • Mscuistf.dl_, Setup.in_, Setup.lst and Setupcdk.pc_ are changed.
  • Stdctl.h has been added.
  • Sysincl.cdk was removed (same four headers were added to Sysincl.dat).
