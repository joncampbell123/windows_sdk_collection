INFO: How Visual Studio Service Pack 1 Affects Visual FoxPro5.0

 Last reviewed: October 20, 1997
Article ID: Q173696
The information in this article applies to:
  • Microsoft Visual Studio 97sp1
  • Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a

SUMMARY

This article explains how Visual Studio Service Pack 1 affects Visual FoxPro versions 5.0x and the files that they use. However, installing Service Pack 1 will update some of the Visual Studio shared components that Visual FoxPro uses.

MORE INFORMATION

Visual Studio Service Pack 1 does not include any bug fixes for Visual FoxPro versions 5.0x.

To install Visual Studio Service Pack 1, run the Setup.exe program. The setup routine automatically detects the Visual Studio components that you have installed on your system and displays a list of files that need to be updated. Once confirmed, the files are updated.

After updating your installation of any of the Visual Studio applications, Visual Studio Service Pack 1 appends any actions to the VS97SP1.log file, which is stored in the root directory of the drive where Windows is installed.

When applied, Visual Studio Service Pack 1 updates the following files that are used by Visual FoxPro: 

   COMCTL32.OCX
   COMCTL32.DEP
   MFC42.DLL
   MSVCRT.DLL
   ODBCCR32.DLL
