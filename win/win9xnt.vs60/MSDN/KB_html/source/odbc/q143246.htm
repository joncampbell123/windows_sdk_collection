PRB: Error: This Version of Msvcrt20.dll Is Not Compatible

 Last reviewed: July 14, 1997
Article ID: Q143246
The information in this article applies to:
  • Microsoft Visual FoxPro ODBC Driver, version 1.0

SYMPTOMS

When installing the Visual FoxPro ODBC Driver in Windows or Windows for Workgroups 3.11, you receive the following error message: 

   Error:  This version of MSVCRT20.DLL is not compatible with Win32s.
After you click the OK button on the error message dialog box, Windows stops responding.

CAUSE

The Visual FoxPro ODBC Driver is 32-bit, so it will run only in Windows 95 or Windows NT 3.5x and above. The Visual FoxPro ODBC Driver's documentation contains this information.

STATUS

This behavior is by design.
