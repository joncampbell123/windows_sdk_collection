WX1350: ODBC Desktop Database Drivers 3.5 for 32-Bit Programs

 Last reviewed: January 30, 1998
Article ID: Q159674
The information in this article applies to:
  • Microsoft SQL Server, versions 4.21a, 6.0, and 6.5
  • Microsoft Access version 7.0 and Microsoft Access 97
  • Microsoft FoxPro versions 2.0, 2.5, and 2.6
  • dBASE III, dBASE IV, and dBASE 5 for Windows
  • Paradox versions 3.x, 4.x, and 5.0
  • Microsoft Excel, versions 3.x, 4.x, 5.0/7.0, and Microsoft Excel 97
  • Microsoft Office 97
  • Microsoft Visual Basic 5.0
  • Microsoft Visual C++ 5.0
  • Microsoft Visual Studio 97

This Application Note contains the latest Microsoft Open Database Connectivity (ODBC) drivers for the following data sources:

  • Microsoft SQL Server versions 4.21a, 6.0 and 6.5
  • Microsoft Jet database engine version 3.5, which supports the following formats: 

        - Microsoft Access versions 2.0, 7.0, and Microsoft Access 97


        - Microsoft FoxPro versions 2.0, 2.5, and 2.6


        - dBASE III, dBASE IV, and dBASE 5 for Windows


        - Paradox versions 3.x, 4.x, and 5.0


        - Microsoft Excel versions 3.x, 4.x, 5.0/7.0, and Microsoft Excel 97


        - Text

The ODBC drivers are 32-bit only, and are designed for use by ODBC- compliant, 32-bit programs running under Microsoft Windows 95 or Microsoft Windows NT 3.51 or 4.0.

Installation

To install the Microsoft ODBC drivers, do the following:

This procedure assumes that you have already downloaded and extracted the compressed file to a folder on drive C named Wx1350.

  1. If you are running Windows 95, click the Start button, and then click Run. (If you are running Windows NT 3.51, click Run from the File menu in either Program Manager or File Manager.)

  2. In the Open box (or in the Command Line box of Windows NT 3.51), type the following command, and then click OK: 

          c:\wx1350\dataacc

  3. Follow the Setup instructions on the screen. Note that the Setup program also runs the ODBC Administrator (Odbcad32.exe), which enables you to set up data sources for the Microsoft ODBC drivers.

Help Files for ODBC Drivers

This Application Note includes the following two driver-specific Help files:

  • The ODBC SQL Server Help file (Drvssrvr.hlp) contains information about adding, modifying, and connecting to Microsoft SQL Server data sources, connection string attributes, and ODBC conformance levels.
  • The ODBC Desktop Database Drivers Help file (Odbcjet.hlp) contains information about adding and modifying Microsoft Jet database engine data sources, connecting to and deleting data sources, and levels of ODBC conformance. The Odbcjet.hlp file contains a "Product Support" topic with details about ODBC support offerings from Microsoft. The Odbcjtnw.hlp file is a subset of the Odbcjet.hlp file and describes new features of the ODBC drivers.
