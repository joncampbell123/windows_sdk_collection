  • Microsoft Visual FoxPro ODBC Driver, version 1.0

SUMMARY

Vfpodbcx.exe is a file containing the Microsoft Visual FoxPro ODBC Driver, version 1.0 for Windows 95 or Windows NT 3.50 (or later).

The Visual FoxPro ODBC Driver provides extremely fast access to Visual FoxPro 3.0 and FoxPro 2.x tables, as well as Visual FoxPro 3.0 databases. It supports all database rules, triggers, and default values and conforms with the ODBC level 1 API. It does not work with Crystal Reports.

DESCRIPTION

The Vfpodbc.exe file is the released version of the Visual FoxPro ODBC driver. The Visual FoxPro ODBC driver allows an ODBC-aware application, such as Microsoft Access, Microsoft Excel, Visual Basic, or Word for Windows, to access data from the following types of tables:

FoxBASE+ and FoxPro Tables

The Visual FoxPro ODBC Driver gives you access to FoxBASE+, FoxPro 1.x, FoxPro 2.x, and Visual FoxPro 3.0 tables. It also provides access to tables that are associated with a Visual FoxPro database container (.dbc).

NOTE: The Visual FoxPro ODBC Driver allows transactions only on tables within a Visual FoxPro database. You cannot create transactions on free tables.

dBASE(C) IV Tables

The Visual FoxPro ODBC Driver gives you access to dBASE IV and earlier version tables. The dBASE files are supported in the following manner: 

 * dBASE versions 2.x and 3.x: The files are read without conversion.

 * dBASE versions 4.x and 5.x: If the files do not have memo fields, and
   tables have fewer than 255 columns, the files are read.

 * dBASE versions 4.x and 5.x tables with memo fields or dBASE indexes:
   These files require conversion in Visual FoxPro for Windows before they
   can be used with the Visual FoxPro ODBC Driver. The driver itself will
   not read or convert these files. If you attempt to connect to a dBASE
   file that contains a memo field or index, an error occurs.

System Requirements

To install the Visual FoxPro ODBC driver, you need: 

 * Windows 95 or Windows NT 3.50 (or later)
 * 2 MB of hard disk space
IMPORTANT: Visual FoxPro ODBC Driver 1.0 is a 32-bit driver only. Windows 3.1 and Windows for Workgroups platforms are not supported. If you attempt to install on these platforms, you will receive the following error message: 

   This version of Msvcrt20.dll is not compatible with Win32s
Also, note that clicking the OK button on this error dialog box may cause the system to stop responding.

How to Install the Visual FoxPro ODBC Driver

  1. Copy the Vfpodbc.exe self-extracting file into a temporary directory.

  2. Run Vfpodbc.exe.

The VFPODBC file will extract files and automatically start the Setup.exe program. Follow the setup program's instructions.

NOTE: The Vfpodbc.txt file contains late-breaking or other information that supplements the Microsoft Visual FoxPro ODBC Driver Help file. To view this file, double-click Vfpodbc.txt, located in the Windows 95 \System, or Windows NT \System32 subdirectory.
