Keystroke Sounds Are Not Available in Windowed MS-DOS Prompt

 Last reviewed: October 25, 1996
Article ID: Q120805
The information in this article applies to:
  • Microsoft Natural Keyboard with IntelliType software version 1.0

When you are running an MS-DOS Prompt in a window, keyboard sounds associated with keystrokes are not available even though these sounds are available in Microsoft Windows and Windows-based applications.

This product limitation is by design. The message center from within Windows controls this output, not the keyboard interrupt; therefore, the sounds are available in Windows-based applications only.
