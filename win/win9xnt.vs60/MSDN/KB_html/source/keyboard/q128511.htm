IntelliType Manager Options Unavailable on IBM Thinkpad

 Last reviewed: October 25, 1996
Article ID: Q128511
The information in this article applies to:
  • Microsoft Natural Keyboard with IntelliType software version 1.0

SYMPTOMS

When you use the Microsoft Natural Keyboard with the IBM Thinkpad docking station, some of the options in the IntelliType Manager (such as Sounds and Key Locks) may not be available.

CAUSE

If the Thinkpad laptop computer is attached to its docking station while you install the IntelliType software, the IntelliType Manager does not recognize the Microsoft Natural Keyboard.

RESOLUTION

To make the dimmed IntelliType Manager options available, run the IntelliType software Setup program again using the "nonwinkbd" switch. The Setup command should be: 

   setup.exe nonwinkbd
