Cannot Edit or Add Sounds for IntelliType

 Last reviewed: October 25, 1996
Article ID: Q120468
The information in this article applies to:
  • Microsoft Natural Keyboard with IntelliType software version 1.0

You cannot add additional sounds to the IntelliType Manager, nor can you edit the sounds included with the IntelliType software.

To see the keyboard sound options in the IntelliType Manager, choose the Keyboard Options tab and look under Sounds. These sounds are actually embedded within an internal .DLL file included with the IntelliType software. For this reason, you cannot edit the existing default sounds (Click, Drip, Electric Typewriter, Manual Typewriter, Maracas, Scissors, and System Speaker), and you cannot add any additional sounds.
