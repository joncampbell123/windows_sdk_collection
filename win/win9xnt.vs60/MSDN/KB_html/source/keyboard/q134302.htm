Online Documentation Contains Information About MouseKeys

 Last reviewed: July 8, 1997
Article ID: Q134302
The information in this article applies to:
  • Microsoft IntelliType Software version 1.1 for Windows 95

SYMPTOMS

The online documentation for the IntelliType software version 1.1 contains information about the MouseKeys feature. However, there is no setting in the Windows 95 properties for the keyboard to enable this feature.

CAUSE

The MouseKeys feature was removed from the IntelliType software version 1.1 for Windows 95 because the feature is included in Windows 95.

RESOLUTION

For information about enabling the MouseKeys feature in Windows 95, please see pages 78-79 in the Microsoft Windows 95 "User's Guide."

MORE INFORMATION

The MouseKeys feature is an accessibility feature that you can use to control the mouse pointer using the ARROW keys on the numeric keypad.
