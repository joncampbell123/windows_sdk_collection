IntelliType Setup Under Windows NT Produces Error Message

 Last reviewed: October 25, 1996
Article ID: Q126818
The information in this article applies to:
  • Microsoft Natural Keyboard with IntelliType software versions 1.0, 2.0

SYMPTOMS

When you run the IntelliType software Setup program on a computer running Windows NT on a non-Intel processor, you receive the following error message: 

   Script or DLL has been corrupted. Unable to read dialog: MSCUISTF.DLL

CAUSE

The IntelliType software is designed to run on Intel 80x86 and compatible processors. The IntelliType software and Setup program will fail to run correctly on other processor types.

MORE INFORMATION

MIPS and DEC Alpha systems running Windows NT or NT Advanced Server are known to produce this error.
