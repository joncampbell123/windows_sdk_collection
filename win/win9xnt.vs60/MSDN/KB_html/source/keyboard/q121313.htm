KBDCPL /S Option Unclear Under KBDCPL /?

 Last reviewed: October 25, 1996
Article ID: Q121313
The information in this article applies to:
  • Microsoft Natural Keyboard with IntelliType software version 1.0

SUMMARY

The KBDCPL /S command-line option for the Microsoft Natural Keyboard with IntelliType software sets the values found in the MSINPUT.INI file for the CAPS LOCK, NUM LOCK, SCROLL LOCK, repeat rate, and repeat delay.

MORE INFORMATION

If you type "KBDCPL /?" (without the quotation marks), the /S choice reads as follows: 

   Sets apps lock, num lock, scroll lock, repeat rate and repeat delay with
   values in MSINPUT.INI.
This can easily be misunderstand as actually setting the option ON (that is, CAPS LOCK and NUM LOCK activated) and setting the newly set values in the MSINPUT.INI file. These options can only be changed through the IntelliType Manager. KBDCPL /S only initiates the settings found in the saved MSINPUT.INI file.
