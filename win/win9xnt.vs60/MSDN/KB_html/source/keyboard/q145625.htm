Microsoft Plus! Screen Saver "Hot Corners" No Longer Work

 Last reviewed: November 18, 1996
Article ID: Q145625
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! for Windows 95
  • Microsoft IntelliType software version 1.1

SYMPTOMS

After you install the Microsoft IntelliType software in Windows 95, the "hot corners" you have defined for your screen savers may no longer work.

RESOLUTION

To work around this problem, reinstall Microsoft Plus!.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
