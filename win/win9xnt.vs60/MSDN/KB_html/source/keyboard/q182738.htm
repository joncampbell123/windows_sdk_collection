No Software Is Available for the Natural Keyboard Elite

 Last reviewed: March 19, 1998
Article ID: Q182738
The information in this article applies to:
  • Microsoft Natural Keyboard Elite

SUMMARY

No device software is included with the Microsoft Natural Keyboard Elite. The device drivers needed to use the Natural Keyboard Elite are provided by Microsoft Windows 95 or later.

MORE INFORMATION

You can change the settings for the Natural Keyboard Elite, by following these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Keyboard.

  3. In the Keyboard Properties dialog box, make the changes you want, and then click OK.
