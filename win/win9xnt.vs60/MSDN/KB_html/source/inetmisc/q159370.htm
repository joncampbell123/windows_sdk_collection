Microsoft Sponsored Internet Mailing Lists

 Last reviewed: April 16, 1997
Article ID: Q159370

SUMMARY

There are many mailing lists that cover the broad range of Microsoft Internet technologies and related tools.

For a current listing of these mailing lists go to: 

   http://www.microsoft.com/sitebuilder/resource/mail.asp

MORE INFORMATION

Microsoft does not offer any formal support in these mailing lists. Instead, Microsoft sponsors this service as a platform for participating in the global community of Microsoft developers and product experts. Microsoft monitors the mailing lists to ensure accuracy of information.
