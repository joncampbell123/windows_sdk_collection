WINS Server Event 4206 Log Entries May Not be Valid

 Last reviewed: November 10, 1997
Article ID: Q129113
3.50 WINDOWS kbnetwork kbbug3.50

The information in this article applies to:

  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

Event 4206 is a stop message that indicates a serious error occurred when a push thread was requested by a Windows Internet Name Service (WINS) server. This message can appear in the server system log when the WINS databases are valid. They should continue to function properly.

Microsoft has confirmed this to be a problem in Windows NT version 3.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available. This problem was corrected in Windows NT version 3.51.
