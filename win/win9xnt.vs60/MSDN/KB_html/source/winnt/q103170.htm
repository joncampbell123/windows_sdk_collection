Extra Drivers Included on Windows NT and NTAS CD-ROM

 Last reviewed: May 8, 1997
Article ID: Q103170
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The CD-ROM disk shipped with Windows NT and Windows NT Advanced server contains several extra drivers that are not on the floppy disks. These drivers can be found in the DRVLIB directory on the CD-ROM disk.

Drivers for the following hardware are included: 

Hardware                   Location                         Platform

IBM 2.0GB 4mm tape drive   \DRVLIB\STORAGE\IBM4MM\X86       Intel x86
Panasonic CR-52x CD-ROM    \DRVLIB\STORAGE\MKECR5XX\X86     Intel x86
Panasonic CR-56x CD-ROM    \DRVLIB\STORAGE\MKECR5XX\X86     Intel x86
Creative Labs 
    Sound Blaster CD-ROM   \DRVLIB\STORAGE\MKECR5XX\X86     Intel x86
Sony CDU 31a CD-ROM        \DRVLIB\STORAGE\SONY_31A\X86     Intel x86
                           \DRVLIB\STORAGE\SONY_31A\MIPS    MIPS
Trantor T348 Miniscsi Plus \DRVLIB\STORAGE\T348\X86         Intel x86
Compaq AVGA Video Driver   \DRVLIB\VIDEO\CPQ_AVGA\X86       Intel x86
The instructions for installing most of these drivers are included in each driver's directory in README.TXT files, with the exception of the Compaq AVGA Video driver, which can be installed by the following procedure:

  1. Copy all the files in \DRVLIB\VIDEO\CPQ_AVGA\X86 to a blank floppy disk.

  2. Run Windows NT Setup from the Main group in Program Manager.

  3. Choose Change System Settings from the Options menu.

  4. In the list of display settings, select Other (requires disk from a hardware manufacturer).

  5. When prompted for the disk, insert the disk you made in step 1.

  6. Select the correct display mode for your monitor.

  7. Choose OK twice, exit Windows NT Setup, and restart the system.
