Forced Shutdown from a Remote System Not Implemented

 Last reviewed: May 8, 1997
Article ID: Q103043

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The option for a forced shutdown from a remote system is not available. However, this option is listed in the Policies, User Rights section of the User Manager. If this right is assigned, it will have no effect in the current version of Windows NT.

This feature is under consideration for future releases of Windows NT. Please note that if this right is granted to users, it is possible that in a future release of Windows NT, those users may be able to perform a remote shutdown.
