Zenith Z-Note 425Ln and i82593 Motherboard LAN Module

 Last reviewed: May 13, 1997
Article ID: Q107851
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Zenith Z-Note 425Ln portable computer is listed on the Windows NT Hardware Compatibility list. However, the on-board internal Ethernet adapter is not recognized. The internal NIC is the Intel 82593 motherboard LAN Module. It is currently not supported under Windows NT.

This NIC is supported under LAN Manager and Windows for Workgroups using the I82593.DOS NDIS 2 driver. This driver does not work under Windows NT.
