Windows NT 3.5 Hangs on Ambra DP60 with Genoa Systems 8900

 Last reviewed: April 24, 1995
Article ID: Q129267
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

When you tranfer files between an IDE hard disk and a SCSI hard disk on an Ambra DP60 PCI computer with a Genoa Systems 8900 PCI video adapter, Windows NT version 3.5 stops responding (hangs).

RESOLUTION

To work around this problem,

  • Replace the Genoa Systems 8900 PCI video adapter with a supported video adapter. For more information, refer to the latest Windows NT version 3.5 Hardware Compatibility List (HCL).

    -or-

  • Contact AMBRA Technical Support at (800) 465-2227.
