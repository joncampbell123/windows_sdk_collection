Cannot Install OmniForm 2.0 in Windows NT 4.0

 Last reviewed: March 28, 1997
Article ID: Q158333
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you try to install OmniForm 2.0 in Windows NT 4.0, you receive the following error message and OmniForm is not installed: 

   Severe
   The setup disks appear to be invalid

CAUSE

OmniForm cannot be installed because it interprets the Windows NT 4.0 version number incorrectly.

RESOLUTION

Contact the program's manufacturer, Caere Corporation, for information about a possible upgrade.
