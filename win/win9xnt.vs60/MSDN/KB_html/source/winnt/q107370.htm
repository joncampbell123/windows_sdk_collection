SCSI Bus Reset Problems with Ultrastor 14f Controller

 Last reviewed: May 13, 1997
Article ID: Q107370
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Backup generates the following error message when doing a back up with an Ultrastor 14f controller that has both a SCSI tape drive and a SCSI hard disk attached to it: 

   Fatal error on drive 1

CAUSE

This problem appears to be caused by a reset of the SCSI bus. The reset is caused by a timeout waiting for the hard disk to respond. This problem typically occurs on machines with slow hard disks.

WORKAROUND

To work around this problem, do one of the following:

  • Use a different model SCSI controller

    -or-

  • Isolate the tape drive and the hard disk drive to two different Ultrastor 14f controllers.
