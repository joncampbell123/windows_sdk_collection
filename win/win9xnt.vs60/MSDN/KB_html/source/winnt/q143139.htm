Screen Captures From ProComm Plus

 Last reviewed: February 15, 1996
Article ID: Q143139
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

SYMPTOMS

When you send screen content information to the printer by pressing ALT+N in ProComm Plus for Windows on a Windows NT 3.51 Service Pack 3 computer, the printed output is garbled.

RESOLUTION

To work around this problem, replace WINSRV.DLL with the file in Windows NT Service Pack 2. Shut down and restart Windows NT.
