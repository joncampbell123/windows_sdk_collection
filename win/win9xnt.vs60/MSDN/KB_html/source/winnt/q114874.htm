Setup on Intel Xpress May Fail

 Last reviewed: May 15, 1997
Article ID: Q114874
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Although the Intel Xpress is listed on the Microsoft Windows NT Hardware Compatibility List, Setup may fail when the SCSI adapter detection takes place. The installation program may stop responding (hang), or a STOP message may appear, indicating a page fault.

This problem may occur on system boards that contain an integrated Adaptec 2740 SCSI controller.

RESOLUTION

To avoid this problem, install the latest BIOS version for this system board. Because the Intel Xpress employs Flash BIOS, you can download an update from Intel. Please note that along with any change to the BIOS revision, the matching EISA Configuration Utility (ECU) must also be installed.

At this time, the latest BIOS revision of the Phoenix BIOS E486 used by Intel is 1.00.16.AM0. The matching ECU is ECU2_9.EXE. To verify the latest version available from Intel, and for instructions on obtaining these, please call Intel technical support at (800) 843-4481.
