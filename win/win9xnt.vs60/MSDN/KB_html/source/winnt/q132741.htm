Sun Exabyte 8500 Tape Device Requires Firmware Upgrade

 Last reviewed: April 11, 1997
Article ID: Q132741
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

After you backup data on a Sun Exabyte Model 8500 tape device and attempt to catalog tape information using Windows NT Backup (NTBACKUP.EXE), the following error message appears: 

   Tape drive error detected
If more than one volume set is backed up on a tape, only the first volume set is visible during the catalog process.

RESOLUTION

To correct this problem, obtain the following from Exabyte Technical Support at (800) 445-7736 and upgrade the firmware: 

   - Code Load Tape software (part number 310059-000)
   - Firmware Upgrade - 85C06X0
