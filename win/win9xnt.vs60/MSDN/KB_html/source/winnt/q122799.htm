Err Msg: Unable to Find Seed Router w/COPS LTI Appletalk NIC

 Last reviewed: November 14, 1994
Article ID: Q122799
The information in this article applies to:
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

After you configure Windows NT Server version 3.5 with a COPS LTI Appletalk network card in a single subnet, the following error message may appear: 

   Unable to find Seed Router.

RESOLUTION

To correct this problem, make sure that the routing option on the COPS LTI Appletalk network card is disabled. For more information, call COPS at (404) 840-0810.
