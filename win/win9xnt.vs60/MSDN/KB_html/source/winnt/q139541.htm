How WINS Static Entries Replicate to Partners

 Last reviewed: April 4, 1997
Article ID: Q139541
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.5 and 3.51
  • Microsoft Windows NT Server version 4.0

Static entries from a WINS Server do not show up in the Mappings\Static Mappings of its replication partners.

Static entries in a WINS database are entered within Mappings\Static Mappings. After the entry is added to the local database of the owner, it replicates to its partners as a database entry. A WINS Static entry that replicates to a partner does not show up in the Mapping\Static Mappings box of its partners.
