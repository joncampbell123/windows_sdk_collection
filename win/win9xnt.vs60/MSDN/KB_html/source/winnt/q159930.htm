Event ID 534 In The Security Log

 Last reviewed: November 17, 1997
Article ID: Q159930
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.5, 3.51, and 4.0

SYMPTOMS

Event Viewer yields Event ID 534 in the Security Log. "The user has not been granted the requested logon type at this machine."

-or-

No one can logon unless they are a member of the Administrators group.

CAUSE

The user right to "Access this computer from the network" has been removed for the Everyone group in User Manager for Domains.

RESOLUTION

To resolve this problem, perform the following steps:

  1. Start User Manager for Domains.

  2. Select "User Rights" from the "Policy" option on the menu bar.

  3. Re-add the Everyone group to the user right, "Access this computer from the network."
