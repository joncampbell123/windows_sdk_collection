List of Confirmed Bugs in Windows NT Version 3.51

The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

For more information about each issue, query in the Microsoft Knowledge Base on the article identifier (ID). For a current list of confirmed problems in Windows NT version 3.51, query on KBBUG3.51 in the Microsoft Knowledge Base.
