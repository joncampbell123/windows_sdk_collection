MediaVision Hardware Support in Windows NT

 Last reviewed: May 12, 1997
Article ID: Q105236

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Windows NT includes drivers to support many of the multimedia hardware components manufactured by MediaVision. Some components may not be able to take advantage of some capabilities of the driver.

The following is a list of functionality supported by the Windows NT drivers for each device: 

Product                  CD Interface            Audio

Thunder board            n/a                     Yes
Thunder & Light          n/a                     Yes
ProAudio Spectrum (PAS)  Yes                     No
PAS Plus                 Yes                     No
PAS Sony                 No                      No
PAS16                    Yes                     Yes
PAS MM                   Yes                     Yes
PAS Studio               Yes                     Yes
Fusion CD                Yes                     No
Fusion16                 No                      Yes
Fusion 16E               Yes                     Yes
Fusion Double CD         Yes                     Yes
CDPC                     No                      No
CDPCXL                   Yes                     Yes
The current drivers provided with Windows NT support audio if a MediaVision sound card has the MV101 mixer and the version 508 or later chip.

If a MediaVision sound card has a SCSI interface, the CD-ROM interface is supported by the drivers provided with Windows NT. If the MediaVision sound card contains a proprietary CD-ROM interface, then the CD-ROM interface is not supported by the drivers currently available.

Any SCSI device listed in the Windows NT Hardware Compatibility List should function properly with the MediaVision/Trantor SCSI interface. Currently, the Sony and Philips proprietary interfaces found on some of the MediaVision sound cards are not supported.
