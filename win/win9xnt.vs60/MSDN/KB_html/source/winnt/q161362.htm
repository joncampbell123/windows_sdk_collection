Windows NT 4.0 Driver for SideWinder Now Available

 Last reviewed: September 24, 1997
Article ID: Q161362
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 4.0 with Service Pack 2
  • Microsoft Windows NT Server versions 4.0 with Service Pack 2
  • Microsoft SideWinder 3D Pro version 1.0

SUMMARY

The CD-ROM version of Windows NT 4.0 Service Pack 2 contains an updated driver for the Microsoft SideWinder joystick. This driver is available in the Drvlib\Joystick\Sidewind folder of the CD-ROM and is an update to the driver included with Windows NT 4.0.

MORE INFORMATION

Perform the following steps to install the SideWinder driver:

  1. Log on as Administrator.

  2. Click the Start button, point to Settings, click Control Panel, and then double-click Multimedia.

  3. Click the Devices tab and then click Add.

  4. Click Unlisted or Updated Driver and then click OK.

  5. Insert the path to install the SideWinder driver. For example, for Intel-based computers, the path is: 

          <Drive Letter>:\Drvlib\Joystick\Sidewind\I386

  6. Click OK, click Microsoft SideWinder 3D Pro Joystick, and then click OK.

  7. Click OK to close the Microsoft SideWinder Joystick Configuration dialog box.

  8. Restart your computer.
