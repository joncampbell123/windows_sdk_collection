Early Versions of InocuLAN May Cause Problems with Windows NT

 Last reviewed: August 28, 1996
Article ID: Q153761
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

Excessive disk activity and slow disk I/O may occur when you run Cheyenne's InocuLAN anti-virus software.

CAUSE

InocuLAN's anti-virus services do not properly release resources after initializing.

RESOLUTION

Upgrade to InocuLAN for Windows NT version 1.01 build 41 or higher.
