Unable to Start Windows NT 3.51 MCP Assessments Exams

 Last reviewed: May 1, 1996
Article ID: Q138225
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

SYMPTOMS

When you attempt to start the sample Windows NT Workstation or Server 3.51 exam on the Microsoft Certified Professional (MCP) Assessment Exams version 4.0 or below, the following error message appears: 

   There are no tests defined. You must install some
   test definitions before you can take a test.

RESOLUTION

MCP Assessment Exams version 4.1 or above include updated drivers that allow the Windows NT 3.51 exams to start properly. In addition, this version includes the following new sample exams: 

   - Programming in Microsoft Visual FoxPro 3.0
   - Networking Basics
   - Implementing and Supporting Microsoft Windows NT 3.51 Workstation
   - Implementing and Supporting Microsoft Windows NT 3.51 Server
To correct this problem, do one of the following:
  • Upgrade your system to Windows NT 3.51 Service Pack 2.

    -or-

  • Upgrade to the MCP Assessment Exams version 4.1 or above. You can download the software upgrade files ASM#1.ZIP, ASM#2.ZIP and ASM#3.ZIP from the following electronic services locations: 

        - CompuServe
      GO MECFORUM
      In Library #3, search for files named ASM*.ZIP
      Display results and download


        - Internet (anonymous FTP)
      ftp ftp.microsoft.com
      Change to the SERVICES/MSEDCERT directory
      Get ASM#1.ZIP
      Get ASM#2.ZIP
      Get ASM#3.ZIP
