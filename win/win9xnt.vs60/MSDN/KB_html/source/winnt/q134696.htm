Adaptec APA-1460 Support in Windows NT 3.51

 Last reviewed: April 1, 1996
Article ID: Q134696
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

The Adaptec APA-1460 SlimSCSI host adapter for PCMCIA portable computers is currently not on the Windows NT 3.51 Hardware Compatibility List (HCL).

The driver for the Adaptec APA-1460 SlimSCSI host adapter is available for download. To obtain the driver, download 1460WNT.EXE (self-extracting file) from one of the following electronic services:

  • Adaptec Bulletin Board Service (BBS) at (408) 945-7727
  • FTP: ftp.adaptec.com
  • World Wide Web site: www.adaptec.com

For more information, contact Adaptec Technical support at (800) 959-7274 or (408) 934-7274.
