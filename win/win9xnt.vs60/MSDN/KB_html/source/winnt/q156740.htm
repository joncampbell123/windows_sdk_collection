Window Frame Blinks When Dragged

 Last reviewed: March 27, 1997
Article ID: Q156740
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you use the Matrox Millennium display adapter at 1600 x 1200 resolution in True Color (24-bit) or True Color (32-bit) mode, a window's frame may blink or flash when you drag the window across the screen.

CAUSE

This problem occurs when the Show Window Contents While Dragging option is disabled on the Plus! tab in Display properties.

RESOLUTION

Enable the Show Window Contents While Dragging option. To do so, follow these steps:

  1. In Control Panel, double-click Display.

  2. Click the Plus! tab.

  3. Click the Show Window Contents While Dragging check box to select it, and then click OK.
