PAS16: DMA, IRQ, SCSI IRQ Settings Reset to Default

 Last reviewed: May 8, 1997
Article ID: Q103082

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

The values for DMA, IRQ, and SCSI IRQ will be reset to the default values of DMA=5, IRQ=7, SCSI IRQ=15, and the previous settings will be lost without any warning if the following is true:

  • The Pro Audio Spectrum 16 (PAS16) sound card driver is installed.
  • You enter the Setup dialog box in the Drivers option in Control Panel to view or change the current DMA Channel, Interrupt, or SCSI Interrupt settings for the PAS16 driver, and choose Cancel to exit the Setup screen.

CAUSE

The current defined values are not being restored properly when Cancel is chosen and the default values for the driver are being inserted.

MORE INFORMATION

Microsoft has confirmed this to be a problem in Windows NT version 3.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
