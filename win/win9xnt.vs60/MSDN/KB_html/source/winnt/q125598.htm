Ensoniq Soundscape Support in Windows NT

 Last reviewed: July 24, 1996
Article ID: Q125598
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

The Ensoniq line of sound cards are not currently on the Hardware Compatibility List (HCL). However, Ensoniq does provide Windows NT drivers for the following cards on their Web page (WWW.ENSONIQ.COM) and Bulletin Board System (BBS) at (610) 647-3195: 

   Soundscape
   Soundscape Elite
   Soundscape (OPUS based)
   Soundscape VIVO
The download web page for the drivers is: 

   http://www/ensoniq.com/mmd/ssosupgr.htm
For more information, contact Ensoniq Technical Support at (800) 942-0096.

Soundscape is a 16-bit, 32-voice wavetable synthesizer designed for maximum compatibility with all PC audio standards, with an emphasis on use with games and multimedia applications. The sound card is also featured in most Gateway 2000 multimedia systems.
