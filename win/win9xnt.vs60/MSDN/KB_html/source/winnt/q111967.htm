Disappearing Check Boxes and Option Buttons

 Last reviewed: May 14, 1997
Article ID: Q111967
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

If you have a Diamond Stealth 24 VLB video card, you may experience disappearing check boxes and option buttons when you are using high resolution S3 drivers. Standard VGA and the 800x600x16 VESA driver do not display this problem.

RESOLUTION

To avoid this problem, run the setup program that comes with the Diamond Stealth 24 VLB and disable the fast-memory access and fast enhanced memory access options. Some of these options are fast write buffer, fast write enhanced buffer, and address latching. According to Diamond Technical Support, disabling these features should not significantly affect performance. For more information, call Diamond Technical Support.
