Racal ES3210 NIC Support in Windows NT 3.51

 Last reviewed: February 8, 1996
Article ID: Q142861
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

SYMPTOMS

After you install the Racal ES3210 Ethernet network interface card (NIC) driver in Windows NT, it cannot communicate with other computers in the network. No error messages or event logs appear regarding the problem.

CAUSE

This problem occurs when you are using the Racal ES3210 NIC with Revision AA firmware. To check the firmware revision, look at the component side of the NIC.

RESOLUTION

To correct this problem, do one of the following: 

   - Using the EISA Configuration utility of the computer, disable DMA
     and Memory Map for the NIC. Disabling the memory map of the NIC may
     slow down the performance of the NIC.

   -or-

   - Contact Racal Technical Support at (800) 722-2555.
