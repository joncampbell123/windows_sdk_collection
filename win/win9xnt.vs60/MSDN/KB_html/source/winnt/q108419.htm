Setup Problem on Intel 486 Express

 Last reviewed: May 13, 1997
Article ID: Q108419
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

You may receive the following error message during Windows NT setup on an Intel 486 Express MX rev. B: 

   An error occurred reading from or determining the configuration of a
   hard disk drive attached to your computer. This may indicate a hardware
   or configuration problem with a disk drive, disk controller, or SCSI
   adapter.

CAUSE

This error message may be caused by one of the following:

  • Your SCSI cable may be too short (less than one inch between the drive and the SCSI card).

    -or-

  • You have an incorrect jumper setting for video, such as Video Memory set to 512K.

RESOLUTION

To work around this problem, do one of the following:

  • If your SCSI cable is too short, move it to another slot on the ribbon (50-pin connector).

    -or-

  • If you have an incorrect jumper setting for video, change Video Memory to 1 MB.
