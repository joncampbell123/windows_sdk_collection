RPLCMD ENUM Utility Does Not Pause Between Screens of Data

 Last reviewed: August 7, 1995
Article ID: Q134325
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.5 and 3.51

The RPLCMD ENUM Utility does not have the option to pause between screens of data.

To be able to view the output of RPLCMD:

  1. Open the MS-DOS Command Prompt (CMD.EXE).

  2. From the control bar menu, select Properties.

  3. In the "Command Prompt" Properties dialog box, select the Screen Size and Position tab.

  4. Increase the Screen Buffer Size Height value to 9999 and choose OK.
