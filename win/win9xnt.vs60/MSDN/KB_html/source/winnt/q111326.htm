Keyboard Errors with Cybex Commander & Dell Computers

 Last reviewed: May 14, 1997
Article ID: Q111326

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Intermittent keyboard error messages appear when running Windows NT or Windows NT Advanced Server on a Dell Computer Corporation computer with a Cybex Autoboot Commander keyboard switch box.

CAUSE

There is a problem with the BIOS on Dell computers that does not allow the necessary 10 milliseconds for the Autoboot Commander to properly respond to commands.

STATUS

Dell Computer Corporation is aware of the problem and is making corrections to its BIOS. Below is a list of affected Dell computers and tentative BIOS revisions that Dell expects will fix this problem: 

   Dell Computer               Tentative BIOS Revision
   ---------------------------------------------------
   LM/LT                       A15
   ME/XE                       A12
   L/LV/MX/MXV                 A06
   Omniplex 560/566 (Pentium)  A02 or A03
Dell technical support can be contacted at (800) 624-9896.
