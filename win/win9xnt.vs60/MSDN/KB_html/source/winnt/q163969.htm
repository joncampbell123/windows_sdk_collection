Event 552: DNS Was Unable to Serve a Client Request

 Last reviewed: May 19, 1997
Article ID: Q163969
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

On a computer running Windows NT 4.0 and the Domain Name System (DNS) Server, the following message may appear in the Event log: 

   Event 552: DNS was unable to serve a client request due to a shortage
   of available memory.

RESOLUTION

To resolve this problem, obtain the hotfix mentioned below.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
