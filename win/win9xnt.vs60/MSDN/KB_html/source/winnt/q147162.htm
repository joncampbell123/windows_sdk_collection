SMS File AUDIT.RUL Updated

 Last reviewed: May 1, 1996
Article ID: Q147162
The information in this article applies to:
  • Microsoft Systems Management Server version 1.1

SUMMARY

The file, AUDIT.RUL, is used by the Systems Management Server (SMS) Audit program to audit the software inventory of SMS clients. The AUDIT.RUL file that was delivered with SMS 1.1 has been updated with a pre-release version of AUDIT.RUL for the next version of SMS.

This file has been updated with new file sizes and additional rules for over 5300 software packages.

MORE INFORMATION

This file is available for download in the following location: 

   FTP://ftp.microsoft.com/bussys/winnt/sms-public/msaudit.zip
