Show Domain Members Only Option has Dimmed Icons

 Last reviewed: May 5, 1997
Article ID: Q110348
The information in this article applies to:
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft NT Workstation 3.5 and 3.51
  • Microsoft NT Server 3.5 and 3.51

SYMPTOMS

Page 440 of the Windows NT Advanced Server "System Guide" incorrectly states: 

   "If a computer does not support remote administration, or if it is
   unavailable to the network (for example, if it is turned off), its icon
   is dimmed, and the version number and description are absent."
When you select the "View/Show Domain Members Only" option in Server Manager For Domains, all machines shown are dimmed.

CAUSE

This behavior is by design.

REFERENCES

Windows NT Advanced Server "System Guide," pages 440, 443
