Gateway Telepath Modem Doesn't Work Correctly Under RAS

 Last reviewed: May 14, 1997
Article ID: Q111254
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

If your Gateway Telepath modem is one of the new OEM (original equipment manufacturer) versions of a U.S. Robotics Sportster 14400 and is configured as a Telepath under Remote Access Service (RAS), it may not answer or call correctly.

CAUSE

This problem occurs because Gateway changed suppliers of their modems and are selling U.S. Robotics OEM modems.

RESOLUTION

To avoid this problem, set up the modem for a U.S. Robotics Sportster 14400 if the Telepath settings don't work.

To determine if the modem is an original Telepath modem, see if there is a daughter board attached. If there is, it is an original Telepath modem.
