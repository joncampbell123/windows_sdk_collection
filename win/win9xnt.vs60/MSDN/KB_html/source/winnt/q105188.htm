KODAK Photo CD System CD-ROM Drive Compatibility Guidelines

 Last reviewed: May 9, 1997
Article ID: Q105188

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

Kodak Multisession Photo CD support has been designed into the Microsoft Windows NT operating system. The requirements for using Kodak Multisession Photo CD in the Windows NT environment depends on how the manufacturer of the CD-ROM drive has implemented Photo CD in their firmware.

MORE INFORMATION

In an effort to help facilitate the compatibility of a Kodak Multisession Photo CD with Windows NT and future Windows operating systems, Microsoft is distributing a document to CD-ROM drive manufacturers (READTOC.DOC) that will help outline the necessary details to enable this compatibility through the Read TOC command.

If a drive's firmware implements the Read TOC command as outlined in this document and the drive is listed in the Windows NT Hardware Compatibility List, then no additional drivers will be needed to access Kodak Multisession Photo CD in the Windows NT environment. Also, any SCSI interface adapter listed in the Windows NT Hardware Compatibility List should support this process.

If a drive's firmware does not or cannot implement the Read TOC Command in the manner described in this document, then a filter driver will need to be written to enable CD-ROM drives to be multisession-capable in Microsoft Windows NT.

The following CD-ROM drives are known to be compatible with the Kodak Multisession Photo CD support designed in Windows NT: 

   Sony CDU-561, CDU-31a
   Texel DM-5024 or Plextor DM-5024
   NEC CDR-74-1
The following CD-ROM drive is known to not be compatible: 

   Toshiba 3401
If other manufacturers have used the same Kodak Multisession Photo CD implementation in their firmware as that used by Sony, then their drives would also be compatible.

The manufacturer of the CD-ROM drive should be contacted to determine the compatibility of a specific drive and availability of a filter driver for Windows NT if needed.

For the contents of READTOC.DOC, query on the following words in the Microsoft Knowledge Base: 

   readtoc and kodak
