Client Add Pack Disks Have Incorrect Label

 Last reviewed: March 5, 1998
Article ID: Q180107
The information in this article applies to:
  • Microsoft BackOffice Small Business Server International English version 4.0

SUMMARY

The Client Add Pack Disks for the International English version of BackOffice Small Business Server may be incorrectly labeled. On January 8, 1998, it was discovered that the Client Add Pack disks for use on the Intel- based computer platform were incorrectly labeled as Alpha-based computer disks.

Although the disks were incorrectly labeled, they do have the correct code for the Intel-based computer platform, and will work as expected.

MORE INFORMATION

Microsoft has corrected the disk labels for the Client Add Pack disks.
