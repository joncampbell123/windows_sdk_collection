8-Bit Future Domain Adapters Don't Work with IBM Optical Drive

 Last reviewed: May 5, 1997
Article ID: Q102599
The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

The 8-bit series Future Domain adapters (Fd8xx) do not work in combination with the IBM 5.25-inch magneto optical (MO) read/write drive in Windows NT 3.1.

There is no correction available at this time.
