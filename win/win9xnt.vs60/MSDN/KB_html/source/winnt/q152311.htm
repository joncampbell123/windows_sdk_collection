INF: How to Get Text with Colored Background on HP Color Print

 Last reviewed: July 29, 1996
Article ID: Q152311
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

When you perform the following actions:

  1. Create an Excel 95 sheet
2. Mark a number of cells, press the right mouse button, and select "Format 
   cells"
3. Format this range to display background color 4. Enter text into the colored area

then what is displayed is only a square in the color of your choice, but no text.

RESOLUTION

To resolve this problem, click the "Print Text as Graphics" button. The button is accessed as follows:

  1. In the File menu, click Print, then select the correct printer.
2. In Properties, select Options. 3. In the dialog box called Advanced Document, select "Print text as 
   Graphics."

MORE INFORMATION

Documents overlapping device fonts or downloaded TrueType fonts and graphics may print erroneously due to the design of the raster printer driver Rasdd.dll.
