Cannot Access Floppies While Using DC2000 Class Tape Drives

 Last reviewed: May 13, 1997
Article ID: Q107392

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

You are unable to access your floppy disk drives while the tape driver for your DC 2000 tape drive is running, that is, while Backup is running. DC 2000 class tape drives are all QIC floppy controller-based tape drives. Because most drivers for this type of tape drive take full control of the floppy controller, other processes are not able to access the floppy disk drives.

To regain access to the floppy drives, close Backup.
