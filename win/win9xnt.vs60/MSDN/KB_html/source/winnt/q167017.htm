Lexmark Optra L and L Plus Printers Incorrectly Show Middle Tray

 Last reviewed: May 20, 1997
Article ID: Q167017
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you set up the tray options for the Lexmark Optra L and L Plus Series printers under Windows NT version 4.0, you are presented with a Middle Tray option. A middle tray does not exist on these two models of the Optra. A middle tray does exist on the R Series Optra printer.

STATUS

Support for middle tray has been removed for the L and L Plus series Optra printers that use Optra.dll driver is included in the latest Windows NT version 4.0 U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces):
