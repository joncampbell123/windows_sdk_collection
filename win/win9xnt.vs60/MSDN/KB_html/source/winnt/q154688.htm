DPT PCI-SCSI Fails on PPC if IRQ Greater Than 15

 Last reviewed: September 7, 1996
Article ID: Q154688
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

DPT PCI-SCSI controllers may fail to operate on IBM Power PC computers that support Hardware Interrupts (IRQ) 17 - 31 if the IRQ of the adapter is set above 15.

STATUS

This problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
