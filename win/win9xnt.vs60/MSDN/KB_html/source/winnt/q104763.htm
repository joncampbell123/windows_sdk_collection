Cannot Print from AutoCAD 12 for Windows

 Last reviewed: May 9, 1997
Article ID: Q104763

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

If Windows NT is installed to either a new directory or over an existing version of Windows and AutoCAD 12 for Windows is already installed on the system, AutoCAD 12 for Windows may not be able to print. The following error message appears: 

   Cannot Initialize Any Configured Plot Device
To enable AutoCAD 12 for Windows to print under Windows NT, you must install it from the Windows NT operating system. During installation, choose option 14 for System Device (the default printer driver in Windows NT Print Manager).
