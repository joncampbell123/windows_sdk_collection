Xircom IIps Credit Card Adapter Fails in Windows NT 3.51

 Last reviewed: June 4, 1996
Article ID: Q140471
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server versions 3.51

SUMMARY

Installing the Xircom IIps Credit card adapter using the file provided in the DRVLIB on the 3.51 Server/Workstation CD-ROM may not work with newer Xircom IIps cards.

MORE INFORMATION

The driver in the DRVLIB\NETCARD\X86\XCE10 is version 1.17. Xircom has updated this driver to version 2.25 which can be found on CompuServe, Xircom's BBS, or their FTP Site. This driver supports Windows for Workgroups, and Windows NT versions 3.1, 3.5, and 3.51.

The file name to download is CE2DRV.EXE with a date of 4/8/96.
