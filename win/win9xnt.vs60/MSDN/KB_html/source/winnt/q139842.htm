Windows NT 3.51 SP2 Cannot Create Mac Volume for CD-ROM Drives

 Last reviewed: April 11, 1997
Article ID: Q139842

The information in this article applies to:

- Microsoft Windows NT Server version 3.51

SUMMARY

After you install Windows NT 3.51 Service Pack 2, CD-ROM volumes for Macintosh clients will no longer be available. When users attempt to create a replacement volume, the following error message appears: 

   File Service for Macintosh only supports NTFS partitions.
   Please choose a directory on an NTFS partition.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.51 Service Pack 2. A supported fix is now available, but is not fully regression-tested and should be applied only to systems experiencing this specific problem. Unless you are severely impacted by this specific problem, Microsoft recommends that you wait for the service pack release containing this fix. Contact Microsoft Product Support Services for more information.
