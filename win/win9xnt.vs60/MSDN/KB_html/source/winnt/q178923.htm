ServerProtect Antivirus Software for Windows NT

 Last reviewed: March 16, 1998
Article ID: Q178923
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you install the Trend Micro ServerProtect antivirus software for Windows NT 3.51 on a server running Windows NT 4.0, the following blue screen STOP error messages may be displayed:

  • Stop: 0x00000024 (0x001901h9, 0xf7cdfbf0, 0xf700cdfa2c, 0x8015f003)
  • Stop: 0x00000024 (0x001901f9, 0xfcc88bf0, 0xfcc88a2c, 0x8015fb03)
  • Stop: 0x0000001e (0xc0000005, 0x00000000, 0x00000000, 0x00000000)
  • Stop: 0x0000007f (0x0000000d, 0x00000000, 0x00000000, 0x00000000)

RESOLUTION

To resolve this problem, uninstall the ServerProtect antivirus software.
