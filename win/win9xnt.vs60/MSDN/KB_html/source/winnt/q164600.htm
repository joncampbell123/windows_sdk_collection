4 mm DAT Driver Reports DEC TZ9L Supports Setmarks

 Last reviewed: May 20, 1997
Article ID: Q164600
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 4.0
  • Microsoft Windows NT Server versions 4.0 -------------------------------------------------------------

SYMPTOMS

The 4 mm DAT Driver reports that the DEC TZ9L supports setmarks in GetDriveParameters, but the device does not actually support setmarks. This may cause certain applications to attempt to use setmarks on the device.

CAUSE

After support for this device was added in Windows NT 4.0 Service Pack 2, it was determined the device does not support setmarks.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
