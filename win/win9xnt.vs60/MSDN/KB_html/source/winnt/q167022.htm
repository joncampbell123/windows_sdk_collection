Forms Missing for Epson EPL-9000 Printer

 Last reviewed: May 20, 1997
Article ID: Q167022
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you view the Form to Tray Assignment properties of the Epson EPL-9000 printer under Device Settings, the following forms are not available: 

   A3
   B4(JIS)
   B5(JIS)
   Tabloid

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K

Additional query words: missing unavailable paper size paper tray gone
