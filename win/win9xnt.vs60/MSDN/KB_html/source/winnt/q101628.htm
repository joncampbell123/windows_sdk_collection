Windows NT Supports Only One Sound Adapter

 Last reviewed: May 7, 1997
Article ID: Q101628

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

Windows NT and Windows NT Advanced Server support only one sound adapter installed on a machine. More than one installed sound adapter can cause problems even if only one device driver that supports only one sound adapter is installed. 

   NOTE: Some internal or built-in sound devices on a machine are the
   equivalent of a sound adapter.
