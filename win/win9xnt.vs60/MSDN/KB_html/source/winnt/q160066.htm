The Second Processor's Speed is Reported Incorrectly to WinNT

 Last reviewed: November 17, 1997
Article ID: Q160066
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SUMMARY

The second processor speed is displayed incorrectly in Windows NT Diagnostics.

Swapping the processors has no effect on the second processor speed being displayed.

CAUSE

This problem is because of a hardware problem with the motherboard.

RESOLUTION

Customers experiencing this problem should contact Digital Equipment Corp. Digital revised the motherboard to report the processor speed information correctly to Windows NT Diagnostics.

DEC's Technical Support Center phone number is 800-354-9000. You can also go to Digital's Web site at http://www.digital.com.

MORE INFORMATION

Stepping is a way to label a CPU's manufacturing revision. The first version of a new processor is the A-0 step. Functional (bug) fixes or manufacturing revisions will increase the stepping number.
