LMHOSTS Keywords Must Be Uppercase

 Last reviewed: March 2, 1998
Article ID: Q110976
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation, versions 3.5, 3.51, and 4.0
  • Microsoft Windows NT Server, version 3.5, 3.51, and 4.0
  • Microsoft Windows 95

Keyword entries in the LMHOSTS files of Microsoft Windows NT systems must be entered in uppercase to be recognized. For example: 

   #INCLUDE \\<servername>\<sharename>\<filename> #PRE
If the keywords #INCLUDE and #PRE are not uppercase, they are considered to be comments.
