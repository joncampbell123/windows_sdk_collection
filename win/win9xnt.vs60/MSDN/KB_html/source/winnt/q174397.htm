Hewlett Packard NetRAID Card May Require Updated Firmware

 Last reviewed: March 20, 1998
Article ID: Q174397

The information in this article applies to:
  • Microsoft Windows NT Server, Enterprise Edition version 4.0 - Microsoft Cluster Server

SUMMARY

If you use a Hewlett Packard (HP) NetRAID adapter card with Microsoft Cluster Server, the card may need a later firmware revision for it to operate correctly.

MORE INFORMATION

HP clusters using the HP NetRAID card will not correctly run Microsoft Cluster Server unless the NetRAID driver and firmware have been upgraded (version A.03.16 or higher).
