Idlewild Blanks the Screen But No Screen Saver Starts

 Last reviewed: May 7, 1997
Article ID: Q101357
The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

After a period of inactivity, Idlewild blanks the screen but it does not start a screen saver.

CAUSE

The Windows NT Control Panel does not specify the fully-qualified path to the screen saver files.

RESOLUTION

Choose the System icon in the Windows NT Control Panel and specify the Idlewild file path.
