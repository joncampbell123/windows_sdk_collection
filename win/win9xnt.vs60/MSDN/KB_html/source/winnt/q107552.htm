Problems Trying to Add Digiboard PC/16E Drivers

 Last reviewed: May 13, 1997
Article ID: Q107552

The information in this article applies to:
  • U.S. Service Pack 1 for Windows NT and Windows NT Advanced Server version 3.1

SYMPTOMS

When you try to install the Digiboard PC/16E drivers that are available with U.S. Service Pack 1, an error message appears informing you that part of the OEMNADDS.INF information file is missing.

CAUSE

No reference to the Digiboard PC/16E drivers was added to the OEMNADDS.INF file.

WORKAROUND

Add the following section to the OEMNADDS.INF file: 

   [Files-DigiBoard16e]
   2, NTXALL.SYS, SIZE=999
   [Files-DigiBoard16i]
   2, NTXALL.SYS, SIZE=999
   [Files-DigiBoard4i]
   2, NTXALL.SYS, SIZE=999

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server version 3.1. This problem was corrected in the latest U.S. Service Pack for Windows NT and Windows NT Advanced Server version 3.1. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
