XTree Gold for DOS Slows Down Windows NT Performance

 Last reviewed: October 5, 1995
Article ID: Q137721
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

XTree Gold for DOS slows down the overall system performance under Windows NT. For example, when you start XTree Gold for DOS, the processor utilization increased to 100 percent, causing other applications to run very slow.

To correct this problem, run XTree Gold for DOS at a lower priority by starting it with the /LOW command. For example: 

   START /LOW XTGOLD.COM
For more information, call XTree Technical Support (Executive Systems) at (800) 964-2490.

XTree Gold for DOS is manufactured by the XTree Company, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
