How to Order GA0650; Instructions in NT Guide Incorrect

 Last reviewed: November 7, 1997
Article ID: Q103812
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

Page xxiv of the "Microsoft Windows NT System Guide" and the "Microsoft Windows NT Advanced Server System Guide" states that the Application Note GA0650 can be ordered on disk by calling (425) 637-7098. This phone number is the technical support phone number for Windows 3.1 and is not the correct number to order the Application Note GA0650.

MORE INFORMATION

Customers wanting to order Application Note GA0650, which contains two keyboard layouts for people who type with only one hand, should call one of the following Windows NT support numbers:

  • (425) 635-7018, 30-day free warranty line
  • (900) 555-1230, single telephone incident for $150 billed to phone bill
  • (425) 635-7022, single telephone incident for $150 billed to credit card
