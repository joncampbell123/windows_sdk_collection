Err Msg: QuicKeys -- Unable to Open VIEWER.INI

 Last reviewed: May 14, 1997
Article ID: Q111691
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

After you install the 1992 edition of Microsoft Bookshelf, the following error message appears when you log on to Windows NT: 

   QuicKeys
   Unable to open VIEWER.INI

CAUSE

This error message appears because QuicKeys is not compatible with Windows NT.

WORKAROUND

To avoid receiving this error message, remove QuicKeys from the Common Startup group.
