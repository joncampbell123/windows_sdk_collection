Chinese Code Converter Saves Incorrect Code Page Format

 Last reviewed: August 5, 1997
Article ID: Q166432
The information in this article applies to:
  • Pan Chinese Windows NT Workstation version 4.0

SYMPTOMS

The Chinese Code Converter in Pan Chinese Windows NT Workstation version 4.0 saves a converted file in the wrong code page if you view another code page and click Save As. This is true even if you choose the original code page.

RESOLUTION

After converting a file using Convert Now, click Save As button to save the converted content before viewing another file.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q166430
   TITLE     : PCNTW: Error Viewing Text File Using Chinese Code
               Converter.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
