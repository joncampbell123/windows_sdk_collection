VCPI Not Supported in Windows NT

 Last reviewed: May 7, 1997
Article ID: Q101780
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

VCPI (Virtual Control Program Interface) is not supported in Windows NT. In order to do so on a 386 or 486-based computers, control of the system page tables would have to be given to the application violating system security. On RISC-based computers, only 286 emulation is available, which precludes the existence of system page tables.
