Information on Adaptec 2940 Controller Models

 Last reviewed: January 8, 1998
Article ID: Q158316
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5, 3.51, and 4.0
  • Microsoft Windows NT Server versions 3.5, 3.51, and 4.0

SUMMARY

The AHA-2940 series of SCSI host adapters is manufactured by Adaptec. Some models require specific adapter BIOS versions for proper operation and functionality. Some models may be upgraded by means of software (flash update); others require chip replacement on the adapter.

MORE INFORMATION

The following is a list of AHA-2940 models and the proper BIOS versions for each card. In some cases, the BIOS version varies depending on the date of manufacture, or if the adapter was produced by an OEM.

Items marked with * indicate that the BIOS version varies based upon the existing BIOS version on the card. 1.1x versions will require version 1.16. 1.2x versions will require 1.23.

If your existing adapter BIOS ends in the letter S, the BIOS version was likely produced by an OEM. Contact the manufacturer of the adapter for more information. 

ADAPTER NAME       FLASH/CHIP      VER     DESCRIPTION

AHA-2940           CHIP            1.16*   Base model AHA-2940
                                   1.23*
AHA-2940W          CHIP            1.16*   2940 with wide support
                                   1.23*
AHA-2940U          FLASH           OEM     OEM Version with Ultra
                                           OEM makes BIOS. Call OEM.
AHA-2940UW         FLASH           1.23    2940 with ultra and wide
AHA-2940AU         CHIP            1.30    Replaces original 2940
                                           adds Ultra support
AHA-3940           CHIP            1.14    Base model AHA-3940
AHA-3940UW         CHIP            1.24    3940 with ultra wide
