Mail Menu Not Added to File Manager Menu Bar

 Last reviewed: May 8, 1997
Article ID: Q102399

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

When you install Microsoft Mail (the 16-bit version), the Mail menu is not added to the File Manager menu bar.

CAUSE

Only members of the Power Users and Administrators groups can make changes that are visible to other users. In order to add the Mail menu to File Manager, you must have these privileges.

RESOLUTION

Use an administrative account to install Microsoft Mail.
