Media Player Err Msg: There Is No Driver Installed...

 Last reviewed: May 9, 1997
Article ID: Q103359

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Windows NT supports only the Microsoft Video 1 compression standard for Audio Video Interleaved (AVI) files.

If you attempt to play AVI files using a different compression method, such as the Intel Indeo compression standard, the following error message may be displayed: 

   Media Player - There is no driver installed on your system.
AVI files created with applications which may use other compression methods, such as Gryphon Software's Morph, will not play under Windows NT as well.
