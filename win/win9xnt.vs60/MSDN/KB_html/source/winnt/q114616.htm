Sony MCI VISCA Driver Available

 Last reviewed: May 15, 1997
Article ID: Q114616
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

Sony Corp. of America is distributing a Windows NT MCI Video System Control Architecture (VISCA) driver that provides video cassette recorder (VCR) services within Windows NT.

MORE INFORMATION

You can obtain the Sony MCI VISCA driver through the Sony BBS at (408) 955-5107.

To download the driver, select option (J) at the main menu to join a conference. When you are prompted to enter which conference you want to join, select (1) to join the VISCA conference. Within the VISCA conference, select (D) to download a file and download the file VISCAMCI.EXE. This is a self-extracting executable file which contains the VISCA MCI drivers for both Windows NT and Windows 3.1.

VISCAMCI.EXE extracts the following files: 

   INSTALL.TXT
   MCIVIS32.DLL
   MCIVISCA.DRV
   MSMCIVCR.EXE
   OEMSETUP.INF
   README.TXT
   VISCASON.DOC
For installation instructions, read the file INSTALL.TXT.

The following information is taken from the VISCASON.DOC file that accompanies the driver: 

   "The MCI VISCA driver controls Sony VISCA-compatible videotape
   recorders, such as the CVD-1000 VDeck. The VISCA driver controls the
   parts of a VCR: the tape transport, channel tuners, and VCR input and
   output channels. To play the content of a videotape on your PC, you must
   connect the video and audio outputs from your VCR to the video capture
   or overlay card of your PC, and to the audio inputs of your PC. You
   might also need an MCI digital-video device driver installed on your
   PC."
