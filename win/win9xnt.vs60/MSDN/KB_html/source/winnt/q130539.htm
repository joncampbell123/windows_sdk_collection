Third-Party Defrag Utility for Windows NT

 Last reviewed: June 23, 1995
Article ID: Q130539
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

Diskeeper for Windows NT is an on-line disk defragmenter for the Windows NT operating system. Diskeeper requires Windows NT version 3.5 or later to run.

Diskeeper is available in Server and Workstation versions. Diskeeper runs on the following Windows NT computer platforms:

  • Intel x86
  • Digital Alpha AXP
  • MIPS PC
  • PowerPC
