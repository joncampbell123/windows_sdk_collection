"Not Enough Resources" Error Running WINS Administrator

 Last reviewed: August 13, 1997
Article ID: Q155924
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

If you run the Windows NT 3.51 WINS administrator on a Windows NT 4.0- based computer to administer a Windows NT 3.51 server over the network, you receive the following error message when you try to select the WINS server: 

   Not enough resources available to complete this operation.

CAUSE

The Windows NT 3.51 WINS administrator does not run correctly in Windows NT 4.0.

RESOLUTION

Use the Windows NT 4.0 WINS administrator (Winsadmn.exe) to administer both Windows NT 3.51 and 4.0 WINS servers over the network.
