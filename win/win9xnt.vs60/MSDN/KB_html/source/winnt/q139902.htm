Doc Err: Invalid IP Address in NETWORKS File

 Last reviewed: March 21, 1997
Article ID: Q139902
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

Page 83 of Update Information for Version 3.51 states the following: 

   The following additional entry is required in the NETWORKS file:

   campus  284.122.107.0

   The following entry, as shown in the examples in the NETWORKS
   file, is incorrect:

   campus  284.122.107
This is incorrect. The first octet of the IP address (284) is an invalid value. The first octet value can be 184.

NOTE: The NETWORKS file is in the %SystemRoot%\SYSTEM32\DRIVERS\ETC directory.
