Dell Machines Recognize Only 16 MB of Memory

 Last reviewed: May 6, 1997
Article ID: Q93500
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Some DELL computers such as the 486D/50, 433DE EISA and 25/50 MHz ISA may not be able to recognize more than 16 MB of RAM when running Windows NT.

CAUSE

This is a known problem with the BIOS of these Dell machines.

RESOLUTION

Dell's BIOS upgrade version A08 corrects this problem. If your computer incorrectly recognizes only 16 MB of memory, contact Dell to receive information about this upgrade.
