Err Msg: The Driver Detected a Controller Error...

 Last reviewed: May 9, 1997
Article ID: Q104560
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

The following error message may be displayed in the Event Viewer: 

   Date    Time       Source   Category  Event  User
   1/1/93  5:32:04AM  Aha154x  None      11     N/A
Event Details shows: 

    The driver detected a controller error on \Device\ScsiPort0

CAUSE

This error occurs when the Adaptec 1740 SCSI controller is configured for standard mode (154x compatible mode).

RESOLUTION

This error can be ignored and the card will work properly in 154x mode or the 174x driver can be installed.

To install the 174x driver, do the following:

  1. From the Main group, run Windows NT Setup.

  2. Select Options and Add/Remove SCSI Adapter.

  3. Choose Add.

  4. Choose OK on the Setup Message, and then select Adaptec 174x.

  5. Choose Install and then follow the instructions for inserting disks or the CD-ROM.

Configure the Adaptec 1740 SCSI Controller for enhanced mode. Please refer to your Adaptec 1740 documentation for steps to change the card to enhanced mode.

After the Adaptec card has been reconfigured for enhanced mode operation you will need to remove the 154x driver.

To remove the 154x driver, do the following:

  1. From the Main group, run Windows NT Setup.

  2. Select Options and Add/Remove SCSI Adapter.

  3. Select the Adaptec 154x driver.

  4. Choose Remove.
