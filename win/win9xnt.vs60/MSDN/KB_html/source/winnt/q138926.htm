Print Jobs Not Clearing for Compaq Pagemarq Printers

 Last reviewed: March 26, 1997
Article ID: Q138926

The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you print to a Windows NT 3.51 server connected to a Compaq Pagemarq 15 or Pagemarq 20 network laser printer, print jobs are not deleted from the print queue.

RESOLUTION

Check the firmware revision of your printer. Your Pagemarq printer should have a firmware revision of 1.13 or later. To obtain the latest firmware contact Compaq Technical Support at (800) 652-6672.
