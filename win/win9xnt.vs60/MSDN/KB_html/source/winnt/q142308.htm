WINS Proxy Button is Unavailable

 Last reviewed: September 6, 1996
Article ID: Q142308
The information in this article applies to:

- Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

It is not possible to make a Windows NT Server act as a WINS Proxy agent. The WINS Proxy Agent button is grayed out.

WORKAROUND

Remove the WINS Server service from the Windows NT Server.
