How to Obtain Updated Printer Drivers for Windows NT

 Last reviewed: September 17, 1997
Article ID: Q161361
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51, 4.0

SUMMARY

This article describes how to obtain printer drivers for Windows NT 3.51 and 4.0.

MORE INFORMATION

There have been several printer driver releases for Windows NT 3.51 and 4.0. Information about these drivers is available in the following articles in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q142643
   TITLE     : Windows NT 4.0 Driver Library

   ARTICLE-ID: Q100654
   TITLE     : Windows NT 3.51 Driver Library
You can query the Microsoft Knowledge Base on the World Wide Web by connecting to the following site: 

   http://www.microsoft.com
Click the Support button, and then click Knowledge Base.

You may also be able to obtain a driver for your printer from the printer's manufacturer. Contact your printer's manufacturer for more information.

All of these files need to be extracted in it’s own directory with the –d Command
