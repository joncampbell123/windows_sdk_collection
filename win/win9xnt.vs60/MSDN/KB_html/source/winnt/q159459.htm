NWCONV Does Not Transfer Queue Operator Group

 Last reviewed: November 11, 1997
Article ID: Q159459
The information in this article applies to:

- Microsoft Windows NT Server versions 3.51 and 4.0

SYMPTOMS

When you use NWCONV to migrate Novell servers to Windows NT, the Queue Operator group does not automatically transferred to Windows NT.

MORE INFO

According to the "Services for NetWare Networks" manual, "The Migration Tool creates a new user group, called Queue Operators, and adds NetWare Print Queue Operators to that group." But this group is not being created after migration is done.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.51 and 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
