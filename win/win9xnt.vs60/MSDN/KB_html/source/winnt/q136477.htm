How to Install MingLiU Fonts for BIG5 Applications

 Last reviewed: September 5, 1997
Article ID: Q136477
The information in this article applies to:
  • Pan Chinese Windows NT Workstation version 4.0

SUMMARY

This article lists the steps necessary to install the MingLiU font for BIG5 applications in Pan Chinese Windows NT Workstation 4.0 Taiwan locale.

MORE INFORMATION

  1. Log on as Administrator or as a user with Administrator rights.

  2. Go to the Langpack folder on the CD-ROM.

  3. Right-click the Tchinese.inf file and then click Install.

  4. Shut down and restart your computer.

For additional information on language support in Pan Chinese Windows NT Workstation, please refer to the Readme.txt file in the Langpack folder on the Pan Chinese Windows NT Workstation 4.0 CD-ROM.
