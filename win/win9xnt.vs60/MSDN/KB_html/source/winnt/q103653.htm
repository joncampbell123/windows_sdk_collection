Windows NT Dr. Watson Report Does Not Show User Domain Name

 Last reviewed: May 9, 1997
Article ID: Q103653

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

In Windows NT, Dr. Watson reports do not show the Domain name of the user logged in during report generation. This may cause confusion when the same user name is logged into more than one domain.

Dr. Watson generates reports when an application error occurs. Much of the information generated for the report can be useful when attempting to isolate application errors. In addition to error information, System Information is also logged to the file. The User's domain is not reported in System Information section in the log.

To determine the domain the User is logged into, you can use the Network section of Control Panel.
