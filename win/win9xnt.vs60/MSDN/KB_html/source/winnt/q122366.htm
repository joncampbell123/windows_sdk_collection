SP2 File Dates Different on CD-ROM and Floppy Media

 Last reviewed: April 29, 1996
Article ID: Q122366
The information in this article applies to:
  • Microsoft U.S. Service Pack 2 for Windows NT version 3.1
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

U.S. Service Pack 2 file dates are not the same on the CD-ROM and the download files from the electronic services. However, the executable files are the same.

MORE INFORMATION

The file date stamp for files on the SP2 CD-ROM distribution media are 2/1/94 with a time stamp of 5:28a. The date stamp for files on floppy media created from downloaded electronic services files is 1/31/94 with a time stamp of 5:28a.

The file sizes are the same between the two media types.
