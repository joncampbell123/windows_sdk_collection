Windows NT Driver Available for the AIR SCSI Controller Board

 Last reviewed: May 13, 1997
Article ID: Q109084
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Advanced Integrated Research makes a SCSI controller board which is not supported by Windows NT and Windows NT will not install correctly with this board by default.

However, there is a Windows NT driver available for this board from Advanced Integrated Research. To obtain this driver, contact AIR at (408) 428-0800. This driver can also be obtained by calling their BBS number at 1- 408-428-1735. The file to download is SCSINT.ZIP.

For technical support or additional information on this driver, contact Advanced Integrated Research at (408) 428-0800.

The AIR SCSI Controller is manufactured by Advanced Integrated Research, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this products' performance or reliability.
