Cannot Change Duplex Printing Mode In Excel (16-Bit)

 Last reviewed: March 25, 1997
Article ID: Q131771

The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5 and 3.51
  • Microsoft Windows NT Server version 3.5 and 3.51
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

When you print from a 16-bit version of Excel, and the printer driver has the duplex option set, Excel uses the duplex setting from Print Manager. You cannot change the duplexing option without exiting and restarting Excel. If you choose Print Setup from the File menu, change the duplex option, and print, the change does not affect the printed output.

To change the duplexing option in Excel, choose Page Setup from the File menu, and then choose Options.
