Pacific Data PostScript Cartridge is Limited Under Windows NT

 Last reviewed: May 13, 1997
Article ID: Q108244
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation version 3.5 and 3.51
  • Microsoft Windows NT Server version 3.5 and 3.51

SUMMARY

The Pacific Data PostScript Cartridge doesn’t support its hardware-based auto-switching functionality under Windows NT. Currently, there are no Windows NT drivers for this cartridge.

MORE INFORMATION

The Pacific Data PostScript Cartridge does not switch between PCL and PostScript under Windows NT. The printer works under Windows NT with the Apple Laser Writer II NTX PostScript driver; this option allows you to print but it may not support all the different tray options.

The Pacific Data PostScript Cartridge is manufactured by a vendor independent of Microsoft and; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
