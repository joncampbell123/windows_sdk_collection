Err Msg: Cannot Find MIDIMAP.CFG or Temporary File

 Last reviewed: May 14, 1997
Article ID: Q111689

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

When trying to access the Midi Mapper in Control Panel the following error may appear: 

   Cannot find the MIDIMAP.CFG or temporary file. These files are
   required to start and quit MIDI Mapper. Copy the original file
   to your Windows SYSTEM directory, and try again. CAUTION: if you
   are quitting MIDI Mapper you will lose any changes.
In order to correct this error make sure that the TMP variable is set properly in the System section of Control Panel. The TMP variable should be pointing to a valid path and directory and have sufficient free space to store the MIDIMAP.CFG file.

MORE INFORMATION

This error message can be somewhat misleading. There is probably nothing wrong with the MIDIMAP.CFG file because if the MIDIMAP.CFG file is found to be corrupt or is not located in the SYSTEM32 directory, the following error would appear: 

   MIDI Mapper could not find a valid MIDIMAP.CFG file in your Windows
   SYSTEM directory. If cannot run without this file. Do you want to
   create and initialize a new MIDIMAP.CFG file?
