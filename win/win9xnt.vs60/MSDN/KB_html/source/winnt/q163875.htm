Group Policies Not Applied If DC Name Is More Than 13 Characters

 Last reviewed: May 19, 1997
Article ID: Q163875
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When using policies in a domain with a file Ntconfig.pol on a server whose computer name is longer than 13 characters, policies for users belonging to global groups are not applied; however, policies for specific users for the same Ntconfig.pol file will work.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
