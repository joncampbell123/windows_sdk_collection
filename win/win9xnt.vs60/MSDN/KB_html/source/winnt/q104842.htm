Changes to Windows NT Driver Library: Network Adapters

 Last reviewed: May 9, 1997
Article ID: Q104842

The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

- THESE DRIVERS ARE TESTED AND CERTIFIED WITH WINDOWS NT VERSION 3.5 ONLY -

SUMMARY

Below is a summary of the changes to the network card adapter drivers in the Microsoft Windows NT Driver Library (WNTDL). The WNTDL offers support for several printers, displays, and network adapter cards not provided with Windows NT version 3.5.

To see WNTDL.TXT, which contains a complete listing of the WNTDL's contents and instructions for obtaining the WNTDL, query on "Q100209" (without the quotation marks) in the Microsoft Knowledge Base.
