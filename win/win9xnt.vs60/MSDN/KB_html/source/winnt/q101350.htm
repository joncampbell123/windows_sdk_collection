Compaq QVision Card Requires Windows NT QVision Drivers

 Last reviewed: May 7, 1997
Article ID: Q101350
The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

To install Microsoft Window NT on a computer that has a Compaq QVision controller installed (for example, a QVision 1024/E), use the Compaq QVision video drivers provided with Windows NT.

The QVision cards are not VGA-compatible. You may experience video-related problems if you attempt to use the standard VGA (640 x 480) video driver.

Problems reported to date relate to running an MS-DOS-based application such as the Introduction to Windows NT (INTRO.EXE) and changing the application display from full-screen to windowed mode.
