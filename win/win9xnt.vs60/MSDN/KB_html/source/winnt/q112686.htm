SQLStor Backup Software for Windows NT

 Last reviewed: May 28, 1997
Article ID: Q112686
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

SQLStor is a high-end tape management system for SQL Server on Windows NT and Windows NT Advanced Server systems. SQLStor includes the following features:

  • Multiple tape drive support, allowing task initiation to more than one tape drive concurrently.
  • Reduces failure due to dirty drives. SQLStor can assign a cleaning cartridge to a tape slot and schedule tape drive cleaning.
  • Database Server specific backups. SQLStor allows for reliable backups and restores while users are still on-line.
  • Works with database dump and load commands. Adheres to the standard database format allowing tapes to be read by standard backup utilities.
  • Client-server based implementation. Clients are an MDI (multiple document interface) application. The client software can manage backup schedules, retention cycles, view catalogs, initiate catalog verification, view database resources, and manage event queues as well as print reports.

MORE INFORMATION

The client software is a Windows 3.1 application licensed for one or more clients and supports one or more SQLStor systems.

The server software is used to execute scheduled events, perform the actual backups and restores, manage and balance the load for available tape drive and autochanger resources, track event histories, and report on SQLStor errors.
