Registry Keys Added by Remote Access Service

 Last reviewed: December 22, 1997
Article ID: Q177118
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SUMMARY

When you install the Remote Access Service (RAS), several registry keys are added or updated. This article provides a list of those keys for reference. This list may be helpful in a situation where uninstallation or reinstallation of the service is not completely successful.

MORE INFORMATION

These are the registry keys that are added or updated during installation of RAS. The registry parameters added by RAS will vary, depending on the installed services, bindings, and software. 

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\AsyncMac

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\AsyncMac3

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\NdisTapi

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\NdisWan

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\NdisWan(x)

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\NetBT\Adapters
   \NdisWan(x)

   NOTE: The above registry key is one path; it has been wrapped for
   readability.

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RasAcd

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RasArp

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RasAuto

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RasMan

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RASPPTPE

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RASPPTPF

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RASPPTPM

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RASPPTPM(x)

   HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\RemoteAccess

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\AsyncMac

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\NdisWan

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\RAS

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\RasAuto

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\RasMan

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\RASPPTP

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\RASPPTPE

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\RASPPTPM

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\RemoteAccess

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\TAPI DEVICES

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion
   \NetworkCards\(x)

   NOTE: The above registry key is one path; it has been wrapped for
   readability.
