Performance Monitor Does Not Monitor Process File Operations

 Last reviewed: January 8, 1996
Article ID: Q125384
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

The following Process Object counters, available in Windows NT version 3.1, are not available in Windows NT version 3.5 because they adversely affect performance: 

   File Control Bytes/sec
   File Control Operations/sec
   File Read Bytes/sec
   File Read Operations/sec
   File Write Bytes/sec
   File Write Operations/sec for the Process Object
