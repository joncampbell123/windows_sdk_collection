HP DeskJet 550C Prints Envelopes Upside-Down

 Last reviewed: May 7, 1997
Article ID: Q101506

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

An attempt to print envelopes with a Hewlett-Packard (HP) DeskJet 550C printer prints envelopes upside-down relative to the diagram on the printer.

CAUSE

This is a limitation in the information the Windows NT printer driver users to characterize the printer.

RESOLUTION

Insert envelopes upside-down relative to the diagram on the printer.

STATUS

Microsoft has confirmed this to be a limitation of the HP DeskJet 550C driver provided with Windows NT. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
