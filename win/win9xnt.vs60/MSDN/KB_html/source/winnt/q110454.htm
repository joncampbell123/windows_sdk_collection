HP LaserJet 4si Printer Driver Supports Only One Cartridge

 Last reviewed: May 14, 1997
Article ID: Q110454
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

The Hewlett-Packard (HP) LaserJet 4si driver in Microsoft Windows NT and Microsoft Windows NT Advanced Server only allows you to choose one font cartridge.

CAUSE

The HP LaserJet 4si printer supports two font cartridges. The HP LaserJet 4si printer driver in Windows NT and Windows NT Advanced Server was ported over from the HP LaserJet 4, which only supports one font cartridge.

WORKAROUND

You can use two font cartridges if you use the HP LaserJet 3si driver, but you will lose the ability to print 600 dots per inch (dpi).
