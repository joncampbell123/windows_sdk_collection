Non Critical Error Installing Software in Windows NT 3.51

 Last reviewed: February 22, 1996
Article ID: Q146123
The information in this article applies to
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

When you attempt to install additional software in Windows NT 3.51 from the source media (compact disc), the following error message appears: 

   Non critical error. Can't open file source: <path>\<file name>.CNT

CAUSE

This problem occurs if you attempt to add software from a Windows NT 3.5 compact disc for Windows NT 3.51. Windows NT 3.51 installs *.CNT files for the new Help files.

RESOLUTION

To correct this problem, use the Windows NT 3.51 source media to install any additional software.
