GP Fault in Peachtree Accounting for Windows 3.0

 Last reviewed: June 23, 1995
Article ID: Q123672
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

When you try to design a form or a financial statement in Peachtree Accounting for Windows version 3.0 under Windows NT version 3.5, the following error message appears: 

   General Protection Fault in module PEACHW.EXE

RESOLUTION

According to Peachtree for Windows Technical Support, upgrading to Batch 03 should correct this problem. For additional information, contact Peachtree Technical Support at (404) 279-2099.
