File Corruption Problems with Chinon CD-ROM Drives

 Last reviewed: May 13, 1997
Article ID: Q109783
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Chinon CD-ROM Drives display a "file system corrupted" error message when you access the CD-ROM from within File Manager.

This error message occurs with earlier Chinon CD-ROM Drives such as the CDS- 431, CDX-431, CDS-435, and CDX-435 models.

CAUSE

This error message occurs because the earlier firmware does not support Windows NT or Windows NT Advanced Server properly.

RESOLUTION

To avoid this error message, obtain a firmware upgrade for your CD-ROM Drive by calling Chinon Technical Support at (310) 533-0274.

NOTE: The Chinon CDS-435 and CDX-435 drives are not on the supported hardware list, but function properly with the upgraded firmware.
