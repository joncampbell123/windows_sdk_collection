E21xx Cabletron Network Cards Do Not Work Properly in MPS

 Last reviewed: December 19, 1997
Article ID: Q164878
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

The Cabletron E21xx series network adapters do not work in multiprocessor computers, including multiprocessor capable computers.

In the event log, you will get the entry: 

   Event ID: 7000
      Source: Service control manager
      Type: Error
      Description:  A Device attached to the system is not functioning.

CAUSE

The driver does not work properly with a multiprocessor capable computer as stated in the Help file on the Windows NT Workstation or Server CD-ROM in the Drvlib\Netcard\X86\Cte21 folder.

RESOLUTION

Use a network card that will work in a multiprocessor capable computer. Or contact Cabletron for a possible fix.
