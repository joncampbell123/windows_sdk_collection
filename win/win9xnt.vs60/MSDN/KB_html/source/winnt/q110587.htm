Number Nine GXE Level 14 Not Supported with S3 Drivers

 Last reviewed: May 14, 1997
Article ID: Q110587
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Number Nine GXE Level 14 card is a 4 megabyte (MB) Vesa local bus card that utilizes the S3 928 chip set. According to Vesa, this card is not supported with any of the S3 drivers that ship with Windows NT 3.1. These drivers don’t work because of changes made to the card’s BIOS and DAC (digital to audio conversion).

This card should not be confused with other Number Nine GXE cards with similar names (such as the Number Nine GXE Level 12) that are supported under Windows NT.
