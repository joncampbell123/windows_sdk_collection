Windows NT Update for IBM ThinkPad 750 Computers

 Last reviewed: November 7, 1997
Article ID: Q98727
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

The file TP750.EXE is a self-extracting archive containing an update to Windows NT for the IBM ThinkPad 750 series of computers.

MORE INFORMATION

This patch has been tested on the ThinkPad 750 and 750C models. It is, however, expected to work on all 750 models including the 750p and 750cs models.

TP750.EXE can be downloaded from a self-extracting file. You can find TP750.EXE , a self-extracting file, on the following services:

  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select the product.
      Enter kbfile TP750.EXE , and click GO!
      Open the article, and click the button to download the file.
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder.
      Get TP750.EXE
  • The Microsoft Network 

          On the Edit menu, click Go To, and then click Other Location.
      Type "mssupport" (without the quotation marks).
      Double-click the MS Software Library icon.
      Find the appropriate product area.
      Locate and Download TP750.EXE .
  • Microsoft Download Service (MSDL) 

          Dial (425) 936-6735 to connect to MSDL
      Download TP750.EXE

For additional information about downloading, please see the following article in the Microsoft Knowledge Base:

To extract the update, copy TP750.EXE to an empty directory, change to that directory and then type the command: 

   TP750 <ENTER>
After the files have been extracted, follow the directions below to update your Windows NT installation floppy disks.

Note: The following information is also included in the README.TXT that is part of the TP750.EXE self-extracting file.

Installing Windows NT on the IBM ThinkPad 750

The Microsoft Windows NT product group has recently provided support for the IBM ThinkPad 750 system. The ThinkPad 750 has passed all required Hardware Compatibility Tests and will be listed on the next update of the Windows NT Hardware Compatibility List. This update was tested on ThinkPad models 750 and 750c. We anticipate successful results on other models such as the 750cs and 750p as well.

At this time, we recommend that a FAT file system be used on the ThinkPad 750. We have experienced some difficulties with the NTFS file system on this system which will be addressed in a future Service Pack update.

In order to run Windows NT Setup on this system, you must first obtain the updated files and recreate Disks #1 and #8 from the Windows NT floppy install disk set. The identified files are: 

   NTDETECT.COM
   NTLDR
   SETUPLDR
   FLOPPY.SYS
   UPDAT750.BAT
Please follow the instructions below to successfully install Windows NT on the IBM ThinkPad 750.

After installing Windows NT on the ThinkPad, do not apply Service Packs 1 or 2 on the system as the Service Pack may overwrite the above referenced files. In the near future, we will release a Service Pack update which may be used on the ThinkPad 750.

To Install Windows NT 3.1 on the IBM ThinkPad 750 _________________________________________________

  1. Use diskcopy or other preferred copy utility to make a new copy of disks #1 and #8 from the Windows NT 3.1 3.5" install disk set.

  2. Copy the updated files:

    NTDETECT.COM NTLDR SETUPLDR FLOPPY.SYS UPDAT750.BAT

    to a directory on the hard drive on the ThinkPad 750.

  3. Make the directory containing the updated files the current directory.

  4. Execute the batch file UPDAT750.BAT, specifying the floppy drive to use. For example:

    UPDAT750 A:

  5. Insert the recreated copies of Disks #1 and #8 in the floppy drive of the ThinkPad 750 as directed.

  6. You may now execute Windows NT Setup on the ThinkPad 750 using the modified disks. Do not use them to install Windows NT onto any other system other than the ThinkPad 750.

Another option is to install U.S. Service Pack 2b for Windows NT version 3.1. The service pack includes the NTDETECT and NTLDR files needed to successfully update the IBM ThinkPad 750.
