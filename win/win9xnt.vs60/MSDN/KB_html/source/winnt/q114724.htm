Digicom Connection 14.4+Fax Modem Does Not Work in Windows NT

 Last reviewed: May 15, 1997
Article ID: Q114724
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

The Digicom Connection 14.4+Fax modem does not work in Windows NT. The Connection 14.4+Fax is totally software-configurable and requires MS-DOS- based drivers to initialize it. Because the software attempts to download configuration information directly to the RAM on the modem, these drivers fail to load under Windows NT.

Currently, the only options are:

  • Start MS-DOS and load the drivers to initialize the modem. Then start Windows NT. The modem will now be operational until the computer is turned off. Repeat this procedure each time you turn your computer on after you shut it down.

    -or-

  • Use another modem that is not totally software-configurable.

MORE INFORMATION

Digicom supplies Smart Modem software for the Connection 14.4+Fax modem. It generally puts a SMODEM directory in the path and you will have one of the following which initializes the modem.

  • A SMLOAD.EXE line in the AUTOEXEC.BAT.

    -or-

  • DL <FILENAME>.DSI in the AUTOEXEC.BAT or other batch file.

    -or-

  • A Smart Modem Manager which loads in Windows and a replacement for the Windows 3.x communication driver COMM.DRV.

For additional information, contact Digicom technical support at (408) 262- 1277.
