Unable to Print after Installing French Service Pack 2

 Last reviewed: August 27, 1996
Article ID: Q141297
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

SYMPTOMS

After installing the French Service Pack 2 on a French Microsoft Windows NT 3.51 computer connected to a Hewlett-Packard printer, you may be unable to print and receive the following error message: 

   Impossible to access the printer.

   Make sure that your printer is properly connected and use the Control
   Panel to verify that the printer is properly set up.

RESOLUTION

To resolve the problem:

  • Install French Service Pack 3 for French Windows NT.

    -or-

  • Expand the HPPCL5MS.DL_ file from the Windows NT 3.51 compact disc to the \SystemRoot\SYSTEM32\SPOOL\DRIVERS\W32X86\1\HPPCL5MS.DLL.

STATUS

Microsoft has confirmed this to be a problem in Service Pack 2 for the French Windows NT. This problem was corrected in the latest Windows NT French Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
