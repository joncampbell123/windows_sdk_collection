Logitech Mouse Support in Windows NT

 Last reviewed: November 14, 1997
Article ID: Q129668
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation version 3.5 and 3.51
  • Microsoft Windows NT Server version 3.5 and 3.51
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

For more information on Logitech Mouse support under Windows NT, contact Logitech FaxBack service and request the following documents: 

   - Document #1213, Using Logitech Pointing Devices Under Windows NT
     Using Windows NT provided drivers to support any Logitech mouse in
     Windows NT.

   - Document #1340, Logitech Pointing Devices Setup Under Windows NT

   - Document #1, Logitech Technical Information Catalog
The Logitech Bulletin Board System (BBS) may also include the information above. The phone number are as follows: 

   Logitech FaxBack Service, (800) 245-0000
   Logitech BBS, (510) 795-0408
The Logitech Mouse is manufactured by Logitech, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
