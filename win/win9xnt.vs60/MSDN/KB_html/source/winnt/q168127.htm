"Install New Modem" and Modem List is Blank.

 Last reviewed: January 13, 1998
Article ID: Q168127
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

The list of modem manufacturers and models in the Install New Modem wizard may be blank when you click "Don't detect my modem, I will select it from a list." You should be able to view the available modems even if you do not have one installed.

CAUSE

The list of modems do not appear if the hidden folder \%SYSTEMROOT%\Inf is empty or missing.

RESOLUTION

To resolve this problem, expand the .In_ files from the Windows NT CD to the \%SYSTEMROOT%\Inf folder or copy them from another Windows NT computer to this folder.

MORE INFORMATION

The Modem.inf file contained in the \<WinntRoot>\System32\Ras folder is used only when installing RAS. It is not used by the Modem.cpl application to provide the list of available modems. This list is built by the modems available in the \%SYSTEMROOT%\Inf folder.
