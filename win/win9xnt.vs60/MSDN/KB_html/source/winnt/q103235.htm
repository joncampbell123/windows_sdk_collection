Clarion 3 May Cause Erratic Mouse Behavior in a VDM

 Last reviewed: May 9, 1997
Article ID: Q103235

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

When running Clarion 3 in a virtual MS-DOS machine (VDM) under Windows NT, you may encounter erratic mouse behavior.

RESOLUTION

To correct this problem use the text-mode mouse pointer. To change the mouse pointer, edit the CLARION3.INI file and add the following line: 

   SmoothMouse=Off
