Requirements for Using RAS to Connect to NetWare Connect

 Last reviewed: July 14, 1997
Article ID: Q171340
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.51, 4.0
  • Microsoft Windows NT Workstation versions 3.51, 4.0

SUMMARY

This article lists the server and client requirements for using Windows NT Remote Access Service (RAS) and the IPX/SPX-compatible protocol to connect to Novell NetWare Connect.

MORE INFORMATION

Server Requirements

  • Novell NetWare Connect version 2.0.28 or later
  • IPX protocol

Client Requirements

  • One of the following NetWare network clients: 

        - Microsoft Client Service for NetWare
    - Novell Client for Windows NT
  • Remote Access Service (RAS)
  • IPX/SPX-compatible protocol
