Windows NT Drivers for Yamaha CDR-100 CD-ROM Drive

 Last reviewed: February 15, 1996
Article ID: Q142390
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

The Yamaha CDR-100 CD-ROM drive does not work with any of the CD-ROM drivers included in Windows NT. The CD-ROM drive is not on the Windows NT Hardware Compatibility List.

To obtain the CD-ROM driver, contact Elektroson Technical Support at (610)617-0850. The driver is included in a software package called Gear.
