Error Reading Directory with Gateway Services for NetWare

 Last reviewed: March 27, 1997
Article ID: Q154964
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51 and 4.0
  • Microsoft Windows NT Server versions 3.51 and 4.0

SYMPTOMS

When you set up Gateway Services for NetWare (GSNW) for shares with multiple GSNW account IDs, the user connecting to the first GSNW share will receive the following error message: 

   Error Reading Directory - The specified user does not exist.

CAUSE

This is by design. Only one account for Gateway Services for NetWare is allowed.
