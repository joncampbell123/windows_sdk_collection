Windows NT Backup Does Not Recognize Windows 95 Backup Tapes

 Last reviewed: March 27, 1997
Article ID: Q155979
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

If you attempt to use the Backup tool in Windows NT 4.0 to restore data from a backup tape created with Microsoft Windows 95, you receive the following error message: 

   Foreign Tape
   The tape in the drive must be erased before it can be used.

CAUSE

The Backup tool in Windows NT uses a different format for tapes than the Backup tool in Windows 95. The two formats are not interchangeable.

RESOLUTION

You must restore the data from the tape using the Backup tool on a computer running Windows 95.
