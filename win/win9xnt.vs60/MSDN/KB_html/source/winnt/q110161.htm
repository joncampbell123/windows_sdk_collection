Digitizer Drivers Available from Wacom

 Last reviewed: November 7, 1997
Article ID: Q110161
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1 -------------------------------------------------------------------------

The Wacom Digitizer (puck or stylus) is not a supported pointing device in Windows NT. However, Wacom Corp. currently has Windows NT drivers for this device available on the Wacom BBS at (408) 982-2737. The filename is WIN_NT.ZIP. Once this file is expanded (unzipped), there should be directories for the three different hardware platforms currently supported by Windows NT: Alpha, MIPS, and I386. The drivers have been written to allow the Wacom Digitizer to work with a Microsoft Mouse installed, or as the only pointing device installed.
