Incorrect Error Message with a B1 Stepping 386 DX CPU

 Last reviewed: May 9, 1997
Article ID: Q103398

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

When you install Windows NT on a B1 stepping of the Intel 386 DX CPU, you may receive the following error message: 

   Your system may be using an early version of the Intel 386(TM) DX
   CPU which is not supported in this beta version of Windows NT.
   Please contact Intel at 1-800-228-4561, in Europe at 44-793-431144,
   or 1-503-629-7354 to determine if you need to acquire an Intel 386
   CPU upgrade.
This error message should not contain the word beta, it should read as follows: 

   Your system may be using an early version of the Intel 386(TM) DX
   CPU which is not supported in this version of Windows NT. Please
   contact Intel at 1-800-228-4561, in Europe at 44-793-431144, or
   1-503-629-7354 to determine if you need to acquire an Intel 386 CPU
   upgrade.
