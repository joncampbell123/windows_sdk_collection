NWLink OnLine Help Documentation Error

 Last reviewed: May 8, 1997
Article ID: Q102832

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

When you view online Help in the NWLink Configuration dialog box, the pop-up text for Network Number incorrectly states that the Network Number should be in decimal form. The Network Number should be in hexadecimal form.

MORE INFORMATION

When you install NWLink, one of the fields in the NWLink Configuration dialog box is Network Number. The Network Number should be a hexadecimal value that specifies the IPX network number to assign to the binding of NWLink and the network adapter card. The online Help pop-up panel for Network Number incorrectly states: 

   Enter the network number in decimal form...
This statement should read: 

   Enter the network number in hexadecimal form...
