Bluescreen When Building to Novell Server on MIPS Computer

 Last reviewed: March 24, 1997
Article ID: Q147911
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51
  • Microsoft Windows NT Server versions 3.51

SYMPTOMS

A Bluescreen problem occurs while doing a build to a Novell server using either a single- or dual-processor Mips computer. The Bluescreen will happen using either CSNW or GWSN.

CAUSE

The problem case reports that the Novell server the user is trying to build to has 70 to 80 NLMs on it. When they build to a plain Novell server, it works fine. The problem happens on Windows NT Service Packs 1, 2 and 3.
