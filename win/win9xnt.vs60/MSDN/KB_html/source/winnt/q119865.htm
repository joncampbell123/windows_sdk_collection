Err Msg: System Error 1376 Has Occurred

 Last reviewed: September 1, 1996
Article ID: Q119865
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you issue the command "Net localgroup <groupname> /Domain" to display local groups in the domain, and the domain name is identical to the local group name, the following error message may appear even though the local group does exist: 

   System error 1376 has occurred.
   The specified local group does not exist.

WORKAROUND

When you set up local groups, avoid using the same name for the local group name and the domain name.

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server version 3.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
