Error Message: Bad or Missing Ktvga.fon

 Last reviewed: September 13, 1996
Article ID: Q154119
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

When you start Attachmate's KEA version 4.2 terminal emulation software, you may encounter the following error under Microsoft Windows NT version 4.0: 

   Bad or missing Ktvga.fon
even though the file exists, is viewable in Control Panel\Fonts, and can be printed.

CAUSE

The "Show only True Type fonts in the programs on my computer" check box is enabled in Control Panel\Fonts under the View\Options\True Type fonts tab.

RESOLUTION

Clear the check box "Show only True Type fonts in the programs on my computer" in Control Panel\Fonts under the View\Options\True Type fonts tab.
