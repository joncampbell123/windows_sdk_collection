License Wizard Displays Unreadable Enterprise Details

 Last reviewed: March 28, 1997
Article ID: Q158359
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you click the enterprise name in License Manager to view its details, one or more names may contain unreadable characters.

CAUSE

The computers with unreadable names may be using a multi-byte character set.

RESOLUTION

Use a single-byte character set on the computers whose name is unreadable.

MORE INFORMATION

Examples of single-byte character sets include English and Greek. Examples of multi-byte character sets include Chinese and Japanese.
