Server Manager Displays Garbage in List of Services (LM/UNIX)

 Last reviewed: May 8, 1997
Article ID: Q102573

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

When Server Manager is focused on a Microsoft UNIX LAN Manager/X server and the you choose Services from the Computer menu, the list of services available on the UNIX LAN Manager/X system contains random garbage characters.

CAUSE

Some original equipment manufacturer (OEM) versions of UNIX LAN Manager/X (especially 2.0), do not correctly query the list of services.

RESOLUTION

Depending on the OEM, this may or may not be corrected in a later release. Contact your OEM for details.
