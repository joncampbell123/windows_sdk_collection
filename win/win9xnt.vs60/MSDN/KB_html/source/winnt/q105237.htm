Calendar Creator Plus May Not Print 132 Columns in Windows NT

 Last reviewed: May 12, 1997
Article ID: Q105237

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you use Power Up! Software's Calendar Creator Plus under Windows NT, you may not be able to print in 132 column mode. Other functions of Calendar Creator Plus should work properly under Windows NT.

RESOLUTION

There is currently no workaround other than to print in 80 column mode in Windows NT.

For additional information, call Power Up! Software technical support at (415) 345-0551.
