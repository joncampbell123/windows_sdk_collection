Unattended Installation of Service Pack 4 After NT Installed

 Last reviewed: September 3, 1996
Article ID: Q153968
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51
  • Microsoft Windows NT Server versions 3.51

Windows NT 3.51 Service Pack 4 can be installed unattended on an existing installation of Windows NT, by means of the /U parameter.

From a booted copy of Windows NT, choose File Run in Program Manager. Type the path to the Service Pack directory for your computer type, followed by Update.exe. Add the /U parameter to the end of the line.

Example: 

   e:\i386\update.exe /u
