Stripe Sets Store Data in 64K Blocks

 Last reviewed: May 9, 1997
Article ID: Q104231
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

When you use a stripe set (with or without parity), 64K blocks of data are stored on each of the drives in turn. 64K was selected because external research suggested that optimum block size is between 64K and 128K, depending on the drive and the system. The block size is not adjustable.
