LinkSys Windows NT Drivers for PCMCIA Ethernet Cards

 Last reviewed: May 15, 1997
Article ID: Q114614
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

LinkSys has written a Windows NT driver for their PCMCIA Ethernet adapters. This driver is only compatible with computers that use the Intel 82365SL PCMCIA chip set. The name of the driver that is compatible with Windows NT version 3.1 is ECARD2.EXE and is available on the LinkSys BBS. The number for the LinkSys BBS is (714) 222-5111.

MORE INFORMATION

Download Instructions

After you log on to the LinkSys BBS, the Main menu appears. From this menu, select (L) to go to the library of files. From the File Library menu, select (D) to download a file. When you are prompted for the filename, enter ECARDNT.EXE.

After you enter the filename, a short list of files appear. Type the number corresponding to ECARDNT.EXE, and select an appropriate transfer protocol. At this time, initiate the download process from your communications application.

Installation Instructions

ECARDNT.EXE is a self-extracting executable file. Run the command ECARDNT from a command prompt, File Manager, or the File-Run menu option. The following files are extracted: 

   DISK1
   DISK2
   ECARD.SYS
   OEMSETNT.INF
ECARDNT.EXE does not include a README file that provides installation instructions. To install the ECARD.SYS driver, choose the Network icon in Control Panel, and then choose the Add Adapter button. From the list of drivers, select <Other>. When you are prompted, enter the location of the LinkSys ECARD adapter driver.

A configuration screen for the ECARD driver appears next, which asks for IRQ Level and I/O Address. After you choose the correct settings, you will be prompted to shutdown and restart Windows NT.
