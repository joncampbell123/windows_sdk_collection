Err Msg. Unable to Create The Directory \C:

 Last reviewed: June 16, 1997
Article ID: Q151563
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 4.0
  • Microsoft Windows NT Server versions 4.0

SYMPTOMS

During an unattended setup of Windows NT using a text file created with the Windows NT 4.0 Resource Kit utility Windows NT Setup Manager, the following error message may appear: 

   Unable to create the directory \C:

CAUSE

This problem occurs if you specify a drive letter in the installation path (for example, C:\Winnt).

RESOLUTION

To resolve this problem, do not include a drive letter in the installation path. Only include the folder name (for example, Winnt).
