No Error During Attempt to Remove Common Group

 Last reviewed: May 6, 1997
Article ID: Q101222
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

If you attempt to remove a common group from Program Manager from an account that has less than Power User privileges, no error message is displayed, but the group is not removed. This is a user notification problem. The functionality is correct; you are not able to remove the group because of your account privileges.
