UPS Problems on HP Net Server

 Last reviewed: May 15, 1997
Article ID: Q115507
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you use an American Power Conversion Uninterruptible Power Supply (UPS) on a Hewlett-Packard (HP) Net Server 4D/66LM, starting the UPS service fails and the following message appears: 

   Could not start the UPS service. Check the connections and the interface
   signals.

RESOLUTION

Make sure the BIOS on the Net Server is revision 1.0016AM0 or higher. Also, if the system board is part number D2182-69001 or D2182-60001, it may be necessary to obtain an upgrade for the system board from Hewlett-Packard. A board with serial number 34005 or higher should not experience this problem.
