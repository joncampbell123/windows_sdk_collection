Name Spaces Supported Under CSNW

 Last reviewed: September 16, 1994
Article ID: Q118899
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

The name space supported under the Microsoft Client Service for NetWare is OS/2 long filenames. To activate this option for the NetWare server, type the following at the command line of the NetWare server: 

   ADD NAME SPACE <name space name> [TO [VOLUME]] <volume_name>

MORE INFORMATION

Using the command above allows non-MS-DOS files such as Macintosh and OS/2 long filenames to be stored on a NetWare volume. Make sure that the previous name space module is replaced with the new module, such as OS2.NAM or MAC.NAM.

NOTE: In order to reverse the operation of the command above, you must run the NetWare VREPAIR utility.
