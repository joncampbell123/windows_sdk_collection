Photoshop 2.5 Causes a GPF When Opening an EPS

 Last reviewed: February 7, 1995
Article ID: Q120431
The information in this article applies to:
  • Microsoft Windows NT workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

If you try to open an EPS in Adobe Photoshop 2.5, a GPF occurs in PHOTOSO2.DLL at 0007:42FD.

CAUSE

The DLL shipped with Photoshop 2.5 is incompatible with Windows NT 3.5.

RESOLUTION

Contact Adobe for a new DLL or upgrade to Photoshop 2.5.1.

STATUS

This is a design incompatibility corrected in the updated release.
