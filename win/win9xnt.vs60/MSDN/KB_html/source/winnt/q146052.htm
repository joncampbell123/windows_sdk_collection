<1Ch> Entries Can Be Removed From Static Mappings Dialog Box

 Last reviewed: April 30, 1997
Article ID: Q146052
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.5 and 3.51
  • Microsoft Windows NT Server version 4.0

When you add a static <1Ch> entry to the WINS database, the entry is added to the <1Ch> group with other dynamic and static <1Ch> entries for the specified domain. The group is then designated as a static group in the WINS database screen of WINS Manager.

In the Static Mappings dialog box, you can remove any one of the dynamic or static <1Ch> entries for the domain. If you remove a <1Ch> static entry, the next entry in the group moves to the static mapping position. The <1Ch> entry is then designated as a static entry.
