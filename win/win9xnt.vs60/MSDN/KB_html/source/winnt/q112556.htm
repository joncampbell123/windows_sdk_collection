SCSI Adapter Drivers for DTC3x50, DTC3x70 and DTC3x80

 Last reviewed: May 28, 1997
Article ID: Q112556
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The DTC SCSI host adapter drivers for models 3x50, 3x70 and 3x80 are available on DTC’s BBS at (408) 942-4010. The filename is 3270NT.ZIP. For additional information, call DTC Technical Support at (408) 262-7700.

The 3270NT.ZIP driver package for Windows NT includes the following files: 

   README
   OEMSCSI.SYS
   OEMSETUP.INF
   TXTSETUP.OEM
   OEM_SCSI
For driver installation instructions, refer to the README file using a text editor.
