Inconsistent Print Results Using HP LaserJet 5L

 Last reviewed: March 21, 1997
Article ID: Q138257
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51
  • Microsoft Windows NT Server versions 3.51

SYMPTOMS

Incorrect or inconsistent print results may occur when using the Microsoft supplied printer driver for the Hewlett-Packard LaserJet 5L; for example printing may be blurred, or extra characters appear in a corner of the page.

WORKAROUND

Change the memory settings for the printer from default of 1,024K to 2,048K as follows:

  1. In Print Manager, select Properites.

  2. Select Setup, and change Printer Memory to 2048K.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.51. This problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces):
