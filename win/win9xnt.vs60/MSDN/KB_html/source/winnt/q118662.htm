Alpha Fails After Firmware Version Change

 Last reviewed: May 16, 1997
Article ID: Q118662
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Alpha AXP computers by Digital Equipment Corp. (DEC) fail when you install a firmware update with a disk from an earlier firmware version.

CAUSE

Ensure that the firmware version you are using for an update is a later version than the one currently installed on the computer. When you install new firmware, there are no warnings or messages to prevent "downgrading" to an earlier version. If an incorrect, or earlier, version of the firmware is installed, the computer fails immediately after the system power-up tests.

STATUS

At this point, the computer does not respond to input. At this time, the only way to restore the computer is to replace the motherboard. This requires a service call from a Digital Customer Service Engineer.
