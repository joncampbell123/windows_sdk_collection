World Tour Tennis Fails to Run on Windows NT

 Last reviewed: May 8, 1997
Article ID: Q102056

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

World Tour Tennis (an MS-DOS - based game) may fail to run on Windows NT.

CAUSE

The World Tour Tennis sound generation algorithm increases the timer hardware tic rate by 11kHz. The overhead required for Windows NT virtual DOS machine (VDM) to generate and service the timer interrupts (to emulate this very high tic rate) causes World Tour Tennis to constantly service the timer tics. As a result, World Tour Tennis appears to stop responding (hang).

RESOLUTION

To work around this problem:

  • Turn off sound in World Tour Tennis.

    or

  • Create a .PIF file for World Tour Tennis which has Compatible Timer Hardware Emulation enabled.
