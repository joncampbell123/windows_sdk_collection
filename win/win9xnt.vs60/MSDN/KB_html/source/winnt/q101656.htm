List of Network Services Information Files & Descriptions

 Last reviewed: May 7, 1997
Article ID: Q101656

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

The text below describes each of the information files (*.INF) that support network services. Each filename has the following format: 

   OEMXXXYY.INF
XXX is one of the following: 

   NAD - network adapter
   NXP - transport
   NSV - service
YY is a two letter abbreviation representative of the component.
