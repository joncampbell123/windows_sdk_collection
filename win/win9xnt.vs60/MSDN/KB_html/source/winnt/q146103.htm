Unable To Print To Apple LaserWriter Pro 810 on Network

 Last reviewed: May 1, 1997
Article ID: Q146103
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

After you install an Apple LaserWriter Pro 810 printer on the network, you are not able to print from Windows NT using the TCP/IP protocol or Appletalk.

CAUSE

This problem occurs if the Apple Laserwriter Pro 810 ethernet network adapter does not have a PrintServer ROM version 1.2 or later.

RESOLUTION

To obtain the PrintServer ROM version 1.2 or later for your ethernet network adapter, contact Apple Technical Support at (800) 767-2775.
