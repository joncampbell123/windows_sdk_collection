Keyboard Hangs During Setup on Tandon Computers

 Last reviewed: May 6, 1997
Article ID: Q99985
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1

SYMPTOMS

Your Tandon keyboard may stop responding (hang) during Windows NT installation if you have a Tandon Options Pro 486/33. Keyboard hangs have been reported after disk 9 on floppy disk installations and shortly after Setup enters graphics mode on CD installations.

CAUSE

This problem is due to an older keyboard controller BIOS. If you encounter this problem, you likely have a keyboard BIOS revision earlier than version 1.7.

RESOLUTION

The only way to avoid this problem is to obtain a later revision of the keyboard controller BIOS. K & M Factory provides the necessary upgrade and can be reached at: (805) 522-6844.
