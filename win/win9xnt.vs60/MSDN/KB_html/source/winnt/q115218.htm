MSDN Insert Incorrectly States Service Pack 3 on CD

 Last reviewed: May 16, 1997
Article ID: Q115218
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The insert for the Microsoft Developer Network CD (MSDN) for April 1994 incorrectly states that Windows NT Service Pack 3 is on the CD. This is not true. At the time of the MSDN release, Service Pack 3 has not been released. The Service Pack included with this MSDN release is Service Pack 2, located on CD number 4 under the SRVPACK2 subdirectory.
