Err Msg: The System Cannot Find the File Specified

 Last reviewed: April 4, 1997
Article ID: Q101866

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

You may receive the following message when you connect to a Windows NT print server from an Windows NT workstation: 

   The system cannot find the file specified.

CAUSE

This message occurs when someone has installed the printer drivers on the server, but has since deleted the printer driver files manually. When the workstation connects to the server, it tries to copy down the printer driver files. When this fails, you receive the above error message.

RESOLUTION

To resolve this situation, reinstall the printer driver files on the Windows NT print server.
