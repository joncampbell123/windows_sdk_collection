WINS Partner Registry Settings May be Deleted

 Last reviewed: September 7, 1996
Article ID: Q151306
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

User-defined WINS Partner registry settings are deleted after double- clicking the OK button in the WINS Replication Partners dialog box.

CAUSE

The HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Wins\Partners key has two subkeys, \Pull and \Push, under which are subkeys for the IP addresses of all push and pull partners, respectively, of the WINS server.

When you double-click OK, CReplicationPartners::Store flushes the WINS cache and calls RegDeleteKeyEx for all IP Addresses of the WINS server's push and pull partners.

The keys are immediately recreated and only default values are reset.

STATUS

Microsoft has confirmed this to be a problem in Windows NT versions 3.5 and 3.51. This problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
