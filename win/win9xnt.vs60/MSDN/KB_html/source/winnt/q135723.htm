Migrating Macintosh Volumes from NetWare to Windows NT

 Last reviewed: March 25, 1997
Article ID: Q135723
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

The NWCONV.EXE Utility corrupts files when it migrates a NetWare Macintosh volume to a Windows NT Macintosh volume.

CAUSE

The NWCONV.EXE utility provided with Windows NT 3.5 and 3.51 does not support the migration of Macintosh files from a NetWare Macintosh Volume to a Windows NT Macintosh Volume.

WORKAROUND

To transfer directories and files from a NetWare server to a Windows NT server:

  1. From a Macintosh client, mount both the NetWare and the Windows NT volume.

  2. Select the NetWare directory structure and drag it onto the Windows NT directory structure.
