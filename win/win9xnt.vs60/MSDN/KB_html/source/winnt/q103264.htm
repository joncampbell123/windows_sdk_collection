Windows NT README.WRI Erroneous Reference to AD-LIB Driver

 Last reviewed: May 9, 1997
Article ID: Q103264
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The README.WRI file that ships with Windows NT states the following in section 10.1: 

   If you can play .WAV files but not .MID files, then you need to
   install the AD-LIB MIDI driver. You can install this driver from the
   Drivers application in Control Panel.
There is no AD-LIB MIDI driver provided with Windows NT. The driver that should be installed is the MIDI Synthesizer driver.
