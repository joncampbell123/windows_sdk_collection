4MM DAT DDS2 Drives Supported on Intel Platform Only

 Last reviewed: May 5, 1997
Article ID: Q120056
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The May 1994 Windows NT hardware compatibility list includes the 4MM DAT DDS2 tape drives now supported in Windows NT, but does not make clear that they are supported only on the Intel x86 platform.

The drives require an MSDL driver (4MMS2.EXE) that does not work on MIPS or ALPHA systems, only on the x86 platform. This is not stated in the HCL or in the README file downloaded with 4MMS2.EXE.
