Limited AppleTalk Stack Included in Windows NT Workstation

 Last reviewed: March 25, 1997
Article ID: Q116343

The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5 and 3.51
  • Microsoft Windows NT Server version 4.0

SUMMARY

Windows NT Workstation includes a limited version of the Microsoft AppleTalk network protocol stack that allows software developers to write to the stack using Windows Sockets. Developers can test their programs with this stack without using Windows NT Server.

MORE INFORMATION

Windows NT Workstation does NOT support MacFile or MacPrint. That is, it does not install the MacFile or MacPrint services on your local machine.

Windows NT Server does include MacFile and Print services as well as the AppleTalk Stack.
