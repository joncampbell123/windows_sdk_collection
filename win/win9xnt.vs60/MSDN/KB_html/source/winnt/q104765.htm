Err Msg: Noncritical Error... Installing a Driver from Floppy

 Last reviewed: May 9, 1997
Article ID: Q104765

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

You try to install a driver using Windows NT Setup and the following error message appears: 

   Noncritical Error
   An error has occurred.

   Unable to open the file A:\xxxxx.xxx

CAUSE

The Floppy Device service has been stopped or disabled to increase system performance. Since Windows NT cannot read from the floppy drive, an error is generated.

RESOLUTION

Either restart the Floppy Device service or use an alternate path which does not rely on a floppy drive, such as a CD-ROM drive. To restart the Floppy Device services, use the Devices portion of Control Panel.
