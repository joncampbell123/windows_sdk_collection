Doc Err: No PROFILE.TXT file in Windows NT Resource Kit 3.5

 Last reviewed: April 10, 1997
Article ID: Q129491
The information in this article applies to:
  • Microsoft Windows NT Resource Kit for Windows NT version 3.5, Windows NT Resource Guide Volume 1

Page 103 of the Windows NT Resource Guide (Volume 1) for Windows NT version 3.5 incorrectly states the following: 

   For information about editing PROFILE.INI, see the PROFILE.TXT
   file in the \I386\CPS directory on the Windows NT resource kit
   CD-ROM.

   -and-

   With the Windows NT resource kit CD-ROM in the CD-ROM drive of
   the master system computer, copy the \I386\CPS directory from
   the CD-ROM to the directory created in Step 1. For example, if
   your CD-ROM drive is D:, type the following:

      xcopy d:\i386\cps
The two statements above are incorrect. The \I386\CPS directory and the PROFILE.TXT file do not exist in the Windows NT resource kit CD-ROM. When you install the Windows NT Resource Kit for Windows NT version 3.5, Resource Kit Setup creates a \RESKIT35\CPS directory. The PROFILE.TXT file is not in this directory.
