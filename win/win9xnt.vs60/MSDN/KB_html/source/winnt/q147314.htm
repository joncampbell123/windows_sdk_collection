The information in this article applies to:

  • Microsoft Windows NT Workstation versions 3.51
  • Microsoft Windows NT Server versions 3.51

These values can be set with the Registry Editor (REGEDT32.EXE) in the HKEY_LOCAL_MACHINE subtree in the following key: 

   \SYSTEM\CurrentControlSet\Control\Session Manager\

      GlobalFlag     REG_DWORD
