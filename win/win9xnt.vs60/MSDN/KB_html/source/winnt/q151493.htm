ThinkPad 760 Audio Requires IBM ESS Audio Driver

 Last reviewed: July 19, 1996
Article ID: Q151493
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

SUMMARY

Microsoft Windows NT does not contain drivers that support the WAVE/MIDI audio functions of the IBM ThinkPad 760 series.

MORE INFORMATION

In order to use the WAVE/MIDI features of the IBM ThinkPad 760 series, you must install the IBM ESS Audio Driver that is available from IBM. The following files may be needed: 

   AFTPFNT1.EXE      IBM ESS Audio Driver
   UTTP1WNT.EXE      ThinkPad NT Utility Diskette
   TPWINNT1.TXT      ThinkPad General Instructions for NT
These files may be obtained from IBM's web site. The last known URL for obtaining more information about these files is: 

   http://www.pc.ibm.com/support/tps/winnt.html
Files may also be available from IBM's BBS at (919) 517-0001.

For best results, obtain the Tpwinnt1.txt file and follow the instructions provided.

Some models of the IBM ThinkPad 760 may also require a BIOS upgrade. For more information on the BIOS upgrade, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q141735
   TITLE     : Windows NT Setup Fails on an IBM ThinkPad 760C P120
or contact IBM for more information.
