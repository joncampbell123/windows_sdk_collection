Peachtree Accounting for MS-DOS Installation May Cause Errors

 Last reviewed: May 9, 1997
Article ID: Q104923

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you install Peachtree Accounting for MS-DOS version 7.0 under Windows NT, if you attempt to restart the computer when prompted by Peachtree's setup program, you may receive one of the following error messages: 

   The NTVDM CPU has encountered an illegal instruction.

   -or-

   ERROR Writing to LPT1: for Document Remote Downlevel Document:
   The device is not ready

   Do you wish to retry or cancel the job?

RESOLUTION

Windows NT will not allow Peachtree to restart the computer. Always choose No when prompted during the installation. To workaround the above error messages, close Peachtree and restart it.
