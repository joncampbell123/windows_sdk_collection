Unable to Print Multiple Quattro Pro Spreadsheets

 Last reviewed: August 30, 1996
Article ID: Q120420
The information in this article applies to:
  • Microsoft Windows NT workstation version 3.5 and 3.51
  • Microsoft Windows NT Server version 3.5 and 3.51

SYMPTOMS

Only one copy of a spreadsheet prints when you request multiple copies in Quattro Pro's Print dialog box.

CAUSE

The escape sequence Quattro Pro uses to request multiple copies is not supported by Windows NT.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.5. We are researching this problem and will post new information in the Microsoft Knowledge Base as it becomes available.
