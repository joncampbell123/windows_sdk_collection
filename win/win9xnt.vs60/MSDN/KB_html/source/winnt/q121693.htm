The information in this article applies to:

  • Microsoft Windows NT 3.5 Workstation and Server U.S. Service Pack 3 

 ** THIS U.S. SERVICE PACK 3 IS FOR WINDOWS NT VERSION 3.5 ONLY!! **
The Windows NT 3.5 Workstation and Server U.S. Service Pack 3 is a cumulative build of all fixes to Windows NT Workstation and Windows NT Server version 3.5.

To find out which bugs are fixed by this update, see the README.TXT file that accompanies the Service Pack or query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K

AVAILABLE FORMATS AND FILES FOR WINDOWS NT 3.5 SERVICE PACK 3

Service Pack 3 is available as single self-extracting archive files for Intel (x86), MIPS, and ALPHA-based computers. For x86-based computers, 9 self-extracting files are available to create a 3.5-inch floppy disk upgrade set.

New symbol files for debugging Windows NT 3.5 with Service Pack 3 on x86, MIPS, and ALPHA-based computers are also available on the Internet.

The Remote Access Service (RAS) software compression device drivers for Windows NT version 3.1 and Windows for Workgroups version 3.11 clients are included in Windows NT 3.5 Service Pack 3.

INSTRUCTIONS FOR DOWNLOADING

Not all formats are available on all electronic forums. Because of their size, the single large archive files for the x86, MIPS, and ALPHA-based versions of Service Pack 3 are only available on the Internet.

Service Pack 3 is available on the following electronic services: 

   Internet: ftp.microsoft.com (198.105.232.1)
   /bussys/winnt/winnt-public/fixes/usa/nt35/ussp3    (all formats)

   Microsoft Download Service (425)936-6735       (x86 3.5 disk set)

INSTALLING WINDOWS NT 3.5 SERVICE PACK 3

ALPHA, MIPS, or x86 Single Disk Upgrade

Copy the self-extracting file to a directory on your hard drive and type the name of the self-extracting file with a "-d" switch after it. For example, to decompress the file, SP3_35I.EXE, enter the following command: 

   SP3_35I.EXE -d
After the service pack has been decompressed, run the UPDATE.EXE program to install the updated files.

x86 3.5-inch Disk Set Upgrade

  1. Copy the following files to a directory on your hard disk drive: 

            35SP3D1.EXE
        35SP3D2.EXE
        35SP3D3.EXE
        35SP3D4.EXE
        35SP3D5.EXE
        35SP3D6.EXE
        35SP3D7.EXE
        35SP3D8.EXE
        NT35SP3.EXE

  2. Run the NT35SP3.EXE program.

  3. Run the MK35SP3.BAT batch file to create the update disks.

    NOTE: You need eight formatted 1.44 MB 3.5" disks for this procedure.

  4. Read the README.TXT file and proceed as directed.

RAS Software Compression Drivers Included with Service Pack 3

Windows NT 3.5 Service Pack 3 includes RAS software compression drivers for Windows NT version 3.1 and Windows for Workgroups version 3.11 clients.

ASYNCMAC.SYS supports software compression over RAS on Windows NT and Windows NT Advanced Server version 3.1. ASYNCMAC.SYS should be installed in the %SystemRoot%\SYSTEM32\DRIVERS directory.

NOTE: Microsoft has not tested this ASYNCMAC.SYS file on computers running Windows NT 3.1 with Service Pack 2 or Service Pack 3, and makes no guarantees as to its performance.

RASMAC.386 supports software compression over RAS on Windows for Workgroups 3.11. RASMAC.386 should be installed in the %Windowsroot%\SYSTEM directory.

To install these drivers, back up your current files with the same name or rename them and copy the corresponding new file in its place. Restart your computer.
