Compaq laptops Restarts Instead of Powering Down

 Last reviewed: July 3, 1997
Article ID: Q155373
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

Some Compaq computers do not power down after shutdown after you set the registry value: 

   HKEY_LOCAL_MACHINE\software\Microsoft\WindowsNT\CurrentVersion\
   winlogon\
PowerdownAfterShutdown:REG_SZ:1

when you are using the Microsoft HAL. This value enables a Shut Down And Power Off option on the Shut Down menu, but if you use this option, the computer restarts instead of shutting down.

CAUSE

This feature does not work correctly with the HAL included with Windows NT 4.0.

RESOLUTION

To use this feature, you must obtain the HAL that supports this feature from the computer's manufacturer.

Download file SP2991 V1.10 Rev-A from Compaq for updated hal.dll

Supported machines: Armada 1510, Armada 1520, Armada 1550, Armada 4110, Armada 4120, Armada 4125, Armada 4130, Armada 7710, Armada 7730, Armada 7750, LTE 5000, LTE 5100, LTE 5150, LTE 5200, LTE 5250, LTE 5280, LTE 5300, LTE 5380, and the LTE 5400.

www.compaq.com/support/files/portables/softpaqs/pages/sp2338.html

Download SP3229 V2.00 Rev-A from Compaq for updated hal.dll

Supported machines: Armada 1510, Armada 1520, and the Armada 1550.

www.compaq.com/support/files/portables/softpaqs/pages /Ar1500SupportNT40.html
