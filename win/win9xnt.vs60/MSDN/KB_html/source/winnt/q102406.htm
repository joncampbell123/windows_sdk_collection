Fastback Runs Very Slowly with Windows NT

 Last reviewed: May 8, 1997
Article ID: Q102406

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Fifth Generation Systems Fastback and Fastback Plus run very slowly in Windows NT.

CAUSE

This problem occurs because Fastback directly programs the floppy disk controller (FDC) to increase the speed of floppy disk reads and writes.

Unfortunately, Windows NT has to trapping all input and output in the virtual DOS machine (VDM), which causes the floppy reads and writes to be slower instead of faster.
