Toshiba Photo CD Class Driver for Windows NT Version 3.10

 Last reviewed: April 29, 1996
Article ID: Q124652
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

The Toshiba 3401 CD-ROM drive is not compatible with the Windows NT Kodak Multisession Photo CD support. There is, however, a third-party driver available that allows the Toshiba 3401 to access Kodak Multisession Photo CDs under Windows NT version 3.1.
