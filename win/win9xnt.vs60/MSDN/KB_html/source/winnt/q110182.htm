Err Msg: ATIOEM Accelerator Not Found

 Last reviewed: May 13, 1997
Article ID: Q110182
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Windows NT defaults to standard VGA and the following Event Viewer error message appears when you use an ATI display driver on an AST Premmia series computer: 

   ATIOEM accelerator not found.

CAUSE

Windows NT does not properly recognize the ATI display adapter on these systems.

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server version 3.1. This problem was corrected in Windows NT version 3.5.

For more information about this problem, see the README.WRI file in Windows NT.
