Automatic Dialing Does Not Dial with SLIP Accounts

 Last reviewed: July 22, 1997
Article ID: Q168676
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you configure Dial-Up Networking (DUN) in Windows NT 4.0 to use the automatic dialing feature to access the Internet, it may not automatically dial.

CAUSE

You may be using a SLIP account for your dial-up access.

RESOLUTION

Automatic dialing does not work with a SLIP account. If your Internet service provider (ISP) also offers a PPP account, you can configure automatic dialing to work with the PPP account.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
