RAS STOP Msg: 0x00000000a (6194ea98,00000002,00000001,80115534)

 Last reviewed: October 17, 1997
Article ID: Q139714
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

SYMPTOMS

On your Windows NT computer running Remote Access Service the following STOP message appears: 

   STOP 0x0000000a (6194ea98,00000002,00000001,80115534)

RESOLUTION

Microsoft has modified the file, ASYNCMAC.SYS, to correct this problem. Upgrade to the latest Windows NT 3.51 U.S. Service Pack.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.51. This problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
