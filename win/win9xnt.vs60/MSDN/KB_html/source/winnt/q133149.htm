Cannot Use Manual Feed with Schedule Plus Junior / Pocket Forms

 Last reviewed: August 26, 1996
Article ID: Q133149
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

When you attempt to print a page from Schedule Plus and use any option other than standard for the paper format, the printer does not pull the paper from the manual feed tray, even if a custom form is set and associated with manual feed.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
