NEC CDR-500 and CDR-600 May Require Firmware Upgrade

 Last reviewed: January 14, 1995
Article ID: Q122319
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

When you install or access an NEC CDR-500 (3Xi) or NEC CDR-600 (3Xe) CD-ROM in Windows NT version 3.5, problems may occur.

CAUSE

Problems may occur if the NEC SCSI CD-ROM is configured as a SCSI-2 device.

RESOLUTION

To correct this problem, do one of the following:

  • Configure the NEC SCSI CD-ROM as a SCSI-1 device.

    -or-

  • Contact NEC Technical Support for version 2.5 or later of the CD-ROM firmware to make it work as a SCSI-2 device. The number for NEC Technical Support is (800) 388-8888.
