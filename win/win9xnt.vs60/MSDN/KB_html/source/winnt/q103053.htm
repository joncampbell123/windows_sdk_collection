Compaq Netflex Driver Fails with STOP Message

 Last reviewed: May 8, 1997
Article ID: Q103053

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Compaq Netflex token ring device driver fails with a STOP message in some situations where hardware conflicts are present on the system (interrupt or I/O address conflicts for example). If you have difficulty with this adapter, check your system for hardware conflicts.
