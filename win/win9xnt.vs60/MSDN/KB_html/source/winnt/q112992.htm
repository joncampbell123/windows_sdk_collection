Windows NT Directories on MSDN Level II CDs

 Last reviewed: May 14, 1997
Article ID: Q112992
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Microsoft Developer Network (MSDN) Level II contains several versions of Windows NT on three CDs. Following is a chart listing each version, location on the MSDN CDs, and build number of the version. 

Platform          CD       Directory             Build

ALPHA ( US )      2        WINNT\USALL\Alpha     528(CSD002)
INTEL  ( US )     2        WINNT\USALL\i386      528(CSD002)
NT MIPS ( US )    2        WINNT\USALL\Mips      528(CSD002)
French ( INTEL )  1        WINNT\X86\French      511
French ( MIPS )   1        WINNT\MIPS\French     511
German  ( MIPS )  1        WINNT\MIPS\German     511
German ( INTEL )  1        WINNT\X86\German      511
Japanese ( INTEL )SDK1     I386                  511
Spanish ( INTEL ) 1        WINNT\X86\Spanish     511
Spanish ( MIPS )  1        WINNT\MIPS\Spanish    511
Update ( INTEL )  2        WINNT\CSD\i386        CSD 002
Update ( MIPS )   2        WINNT\CSD\MIPS        CSD 002
The telephone support number for the MSDN is (800) 936-5800.

MSDN Level II is not a replacement of MSDN Fall 93. MSDN Level II includes the MSDN Fall 93 CD, along with CDs that contain all Microsoft operating systems and all Microsoft Software Development Kits.
