Access Is Denied When Setting Up a Trust

 Last reviewed: March 5, 1998
Article ID: Q177040
The information in this article applies to:
  • Microsoft BackOffice Small Business Server Version 4.0

SYMPTOMS

When you try to set up a trust relationship to either a trusted or trusting domain, you receive the following error message: 

   User Manager for Domains

   Access is denied.

CAUSE

This error can be caused if you try to set up a trust from a computer running Small Business Server. Small Business Server does not support trusts.

RESOLUTION

To determine if you have the Small Business Server version of Windows NT 4.0, in Control Panel, double-click Licensing. If you have the SBS version, you will receive the following error message about licensing: 

   Not available in Microsoft BackOffice Small Business Server.
