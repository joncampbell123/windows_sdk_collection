Resizing Telnet Window with Full Drag Enabled Is Jerky

 Last reviewed: March 3, 1995
Article ID: Q122367
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

When you enable the Full Drag option in the Desktop portion of Control Panel and resize the Telnet window, the window bounces back and forth until you finish dragging it and its screen refresh is not as smooth as when you resize other windows.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
