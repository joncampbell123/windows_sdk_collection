Unable to Change Font Cartridge Selection

 Last reviewed: May 20, 1997
Article ID: Q165950

The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you select an installed font cartridge in the device settings of a raster printer (PCL) driver and then later select a different cartridge, the change is saved.

RESOLUTION

  1. Click the Start button, point to Settings, click Printers, and then double-click the printer you wish to configure.

  2. Click Properties on the Printer menu and then click the Device Settings tab.

  3. For Installed Font Cartridge(s) click Not Available and then click OK.

  4. Repeat steps 1 and 2.

  5. Click the font cartridge that is installed in your printer.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
