Telebit T2500 Modem Fails to Dial Out

 Last reviewed: May 9, 1997
Article ID: Q103498
The information in this article applies to:
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

The Telebit T2500 often fails to dial out on a port that is configured for Dial-out and Receive Calls when used with Windows NT Remote Access Service (RAS).

CAUSE

This problem is due to the modem dropping the data set ready (DSR or modem ready) signal when it is being reset for dial out by the RAS server. Other modems do not exhibit this behavior, like the Hayes Ultra 9600.
