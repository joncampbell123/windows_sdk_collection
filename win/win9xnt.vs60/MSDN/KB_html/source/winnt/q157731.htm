Philips CD-ROM Drive Drivers Available for Windows NT

 Last reviewed: November 7, 1997
Article ID: Q157731
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

Philips LMS manufactures low-cost CD-ROM drives that operate on non-SCSI host adapters. These drives are not currently on the Windows NT Hardware Compatibility List (HCL).

Beta version drivers are available for the LMS205, LMS206, and LMS207 CD-ROM drives. The driver can be obtained by either calling the LMS bulletin board system (BBS) at (719) 593-4081 or accessing the Philips Sound & Vision web page: 

   http://www-eu.philips.com/sv/pcaddon/cdrom/
The following list is the drivers on the LMS BBS: 

   206_NT31.EXE    Beta 1 NT 3.1 for 205MS/206
   206_NT.EXE      Beta 1 NT 3.5 for 205MS/206
   206_NT2.EXE     Beta 2 NT 3.5 for 205MS/206
   207_NT2.EXE     Beta 2 NT 3.5 driver for 207
The following list is the drivers on the Philips Sound & Vision web page: 

   206_NT.EXE      CM206 Beta Driver for Windows NT
   206_NT2.EXE     CM206 Beta Driver for Windows NT
   206_NT31.EXE    CM206 Beta Driver for Windows NT 3.1
   207_NT2.EXE     CM207 driver for Windows NT
The drivers also work with both CM250 and CM260 host adapter cards.

The Philips products discussed here are manufactured by Philips, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.
