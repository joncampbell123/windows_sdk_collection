CSNW Supports NetWare 4.x Bindery Emulation

 Last reviewed: May 16, 1997
Article ID: Q117395

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

Microsoft Client Service for Netware supports NetWare 4.x bindery emulation.

MORE INFORMATION

To configure the Client Service for Netware 4.x bindery emulation, you must make sure that the bindery context you have set for the Netware 4.x server directory includes the users you want to have access to the servers. To view and set the bindery context on NetWare 4.x servers, follow these steps:

  1. Load the SERVMAN NetWare Loadable Module (NLM).

  2. View and set the SET BINDERY CONTEXT parameter.

    NOTE: Type SET at the command line to view available SET parameters.
