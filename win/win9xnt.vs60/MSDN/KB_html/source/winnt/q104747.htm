MediaVision SCSI Adapter Supports Devices on Windows NT HCL

 Last reviewed: May 9, 1997
Article ID: Q104747

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

MediaVision, which manufactures the ProAudio Spectrum 16 (PAS16) sound card, distributes a list of CD-ROM drives that have been tested and certified as compatible with the ProAudio Spectrum's on-board Trantor SCSI controller. This list is specific to the MS-DOS Trantor SCSI driver (TSLCDR.SYS) distributed by MediaVision.

According to Trantor Systems, the manufacturer of the controller on the PAS16, if you use the Trantor MediaVision SCSI driver provided with Windows NT (TMV1.SYS), the PAS16 SCSI controller should support any SCSI device that is on the Windows NT Hardware Compatibility List (HCL).

NOTE: A CD-ROM device that is listed by MediaVision as tested and compatible with the PAS16 does not necessarily imply that it will work under Windows NT.

MORE INFORMATION

The PAS16 automatically provides termination for its end of the SCSI bus. However, it may be necessary to configure the CD-ROM drive (or the last device on the bus) to terminate the other end.

The PAS16 reserves SCSI ID #7 by default. (This cannot be changed.) Any other SCSI ID number (0-6) can be used with the CD-ROM drive or other SCSI devices.
