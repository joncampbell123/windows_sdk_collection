DHCP.MIB and WINS.MIB Not Included in Resource Kit

 Last reviewed: November 7, 1997
Article ID: Q142259
3.51 WINDOWS kbnetwork

The information in this article applies to:

  • Microsoft Windows NT Resource Kit version 3.51

The DHCP.MIB and WINS.MIB SNMP MIB files are not included in the Windows NT Resource Kit 3.51. They are as follows: 

   - LMMIB2.MIB (LAN Manager 2.0)
   - MIB_II.MIB (RFC1213-MIB)
   - SMI.MIB    (RFC1155)
The DHCP.MIB and WINS.MIB MIB files are available in the following locations: 

You can find NEWMIB.EXE (size: 21255 bytes) 
                       , a self-extracting file, on the following
services:
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select the product.

          Enter kbfile NEWMIB.EXE (size: 21255 bytes) 
                             , and click GO!
      Open the article, and click the button to download the file.


  • Internet (anonymous FTP)

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder.
      Get NEWMIB.EXE (size: 21255 bytes) 

    

  • The Microsoft Network

          On the Edit menu, click Go To, and then click Other Location.
      Type "mssupport" (without the quotation marks).
      Double-click the MS Software Library icon.
      Find the appropriate product area.
      Locate and Download NEWMIB.EXE.

    

  • Microsoft Download Service (MSDL)

          Dial (425) 936-6735 to connect to MSDL
      Download NEWMIB.EXE (size: 21255 bytes) 

    

For additional information about downloading, please see the following
article in the Microsoft Knowledge Base:
