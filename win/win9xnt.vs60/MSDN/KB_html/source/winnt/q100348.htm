Err Msg: Image Can't Be Relocated No Fixup Information

 Last reviewed: May 6, 1997
Article ID: Q100348
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1

SYMPTOMS

You may receive the following error message when installing Windows NT on a Compaq Prosignia or SystemPro/XL: 

   OS Loader 2.10
   Loading file
   multi(0)disk(0)rdisk(0)partition(1)\winnt\system\ntoskrnl.exe
   :Image can't be relocated no fixup information
   The system did not load because it cannot find the following file:
   WINNT\system\ntoskrnl.exe
   Please re-install a copy of the above file.

CAUSE

This error message is caused by the way memory is configured on Compaq SystemPro/XL and Compaq Prosignia computers. The factory default is "640K Linear." To successfully install Windows NT, you need to change this to "640K Compaq Compatible."

RESOLUTION

To make the necessary change, follow these steps:

  1. Boot your computer with the Compaq System Configuration disk.

  2. At the first screen, press CTRL+A for Advanced Configuration.

  3. Press ENTER.

  4. Choose "Configure computer" from the main menu.

  5. From EISA Configuration Utility, choose "View or edit details."

  6. Select Compaq Memory.

  7. Select Base Memory and press ENTER.

  8. Select 640 Kbytes, Compaq Compatible.

  9. Press ENTER.

  10. If asked to reconfigure memory addresses press OK.

  11. Press F10.

  12. From EISA Configuration Utility choose "Save and Exit."

  13. Save the configuration and press OK to reboot.
