Windows NT Installable File System Kit (IFS) Description

 Last reviewed: November 21, 1997
Article ID: Q168513
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

The Microsoft(R) Windows NT(R) Installable File System (IFS) Kit is a developer's kit for the kernel mode file system and file system filter driver models.

For the most current information about the Windows NT Installable File System Kit (IFS), see the following Microsoft Web site: 

   http://www.microsoft.com/hwdev/ntifskit/default.htm
