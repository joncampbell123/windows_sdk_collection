HP ScanJet IICX SCSI Controller Driver Availability

 Last reviewed: May 14, 1997
Article ID: Q110455
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Hewlett-Packard (HP) ScanJet IIcx SCSI interface card (C2502-66500 for ISA and C2503-66500 for MCA) is not a currently supported SCSI controller under Windows NT.

HP does have Windows NT drivers available for this card so that you can use the HP ScanJet IICX, IIP, and IIC scanners connected to the HP ScanJet IICX SCSI interface card. The driver is available on the HP CompuServe forum (Go HPPER), in library 7. The filename is DS20E.EXE. This file needs to be expanded to a high density floppy disk by using the following command after you download the file 

   ds20e.exe -d c:
where c: is the destination disk drive.

For more information or if you do not have a CompuServe account, contact Hewlett-Packard at (208) 323-2551.

NOTE: A Windows NT NCR driver may work for the HP ScanJet IIcx. Also, if you continue to have difficulties with either of these drivers, make sure the controller is the last in the SCSI chain and is terminated properly.
