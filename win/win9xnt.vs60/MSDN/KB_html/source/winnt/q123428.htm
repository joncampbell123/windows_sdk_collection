RASPHONE.HLP Err Msg: Help Is Not Available for this Request

 Last reviewed: August 22, 1996
Article ID: Q123428
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

In the Remote Access Help file (RASPHONE.HLP), when you select the "Getting Started" topic under "Using Remote Access as Client", the following error message appears: 

   Help is not available for this request.
Choosing OK returns the same error message. Choosing OK for the second time removes the error message.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
