Add Network Adapter Dialog Box Shows No Adapters

 Last reviewed: May 16, 1997
Article ID: Q117368
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

From Control Panel, when you choose the Network icon and then select the Add Adapter button to install a network adapter, the only available selection in the Network Adapter Card list is "<Other> Requires disk from manufacturer."

CAUSE

The only available selection in the Network Adapter Card list is "<Other> Requires disk from manufacturer" when the UTILITY.INF file is missing or damaged, or is not in the SYSTEM32 directory.

RESOLUTION

To restore the Network Adapter Card list, expand the UTILITY.INF file from CD-ROM or floppy disk and place it in the SYSTEM32 directory. UTILITY.INF is in the install directory on CD-ROM and on disk 7 of the disk set.
