NetWare Preferred Server Password Requested at Each Logon

 Last reviewed: May 16, 1997
Article ID: Q117398

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Client Service for Netware prompts you for a password for your NetWare preferred server after you log on to Windows NT.

This happens when your NetWare preferred server password differs from your Windows NT logon password. If you do not want to be prompted for your NetWare preferred server password, make your NetWare preferred server and Windows NT logon passwords the same. You may change your Netware preferred server password using the Netware setpass command at the command prompt.
