DNS Manager Shows Identical Entries In-addr.arpa zone

 Last reviewed: December 19, 1997
Article ID: Q166420
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0

Windows NT 4.0 Domain Name Service (DNS) Manager shows 2 identical entries for the x.x.in-addr.arpa zone.

DNS Manager will show a double entry in x.x.in-addr.arpa zone, if more than 29 PTR records are defined. This entry appears only once in the x.x.in- addr.arpa.dns file.

Functionality is not affected.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.00. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
