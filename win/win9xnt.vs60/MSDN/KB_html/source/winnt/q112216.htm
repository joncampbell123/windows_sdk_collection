Mitsumi CD-ROM Support Information

 Last reviewed: April 11, 1997
Article ID: Q112216
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

Mitsumi Electronics manufactures low-cost CD-ROM drives that operate on non- SCSI host adapters. These drives are not currently on the Windows NT Hardware Compatibility List.

MORE INFORMATION

There are three sources for Mitsumi CD-ROM drivers for Windows NT:

  • Mitsumi Electronics has released a driver which supports the LU005S, FX001, and FX001D CD-ROM drives. This driver can be obtained from Library 4 of the CDROM forum on CompuServe. This driver is supported by Mitsumi Technical Support at (800) 801-7927.
  • A shareware driver written for the Mitsumi LU002 and LU005 CD-ROM drives is available on CompuServe. This driver works only for 30 days after installation unless you send the author a registration fee of $20. This unsupported driver can be obtained from the Third Party section of Microsoft's MSWIN32 forum on CompuServe. Please note that this driver is not supported by Microsoft or Mitsumi.
  • Media Synergy has written a driver that supports the Mitsumi LU005S, LU002, and FX001D CD-ROM drives. This driver is included in a package titled CD-ROM NOW! Call Media Synergy at (416) 593-0647 for information and support

The products included here are manufactured by vendors independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.
