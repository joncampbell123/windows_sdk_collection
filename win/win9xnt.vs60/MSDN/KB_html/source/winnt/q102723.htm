Toshiba MK538FB Drive Repeats Next to Last Byte in Sector

 Last reviewed: May 8, 1997
Article ID: Q102723
The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

The Toshiba MK538FB 1.2-gigabyte (GB) drive sometimes returns bad data. The second to last byte of a sector is repeated in the last byte of the sector. This problem occurs using several different Small Computer System Interface (SCSI) host adapters with firmware versions 6030 and earlier.

CAUSE

There is a firmware problem in the Toshiba drive.

RESOLUTION

Contact Toshiba America, Inc. for a firmware update.
