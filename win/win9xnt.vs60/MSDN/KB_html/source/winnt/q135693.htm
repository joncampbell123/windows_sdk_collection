Andrew ISA II Adapter 1 TR Does Not Function Properly

 Last reviewed: August 25, 1995
Article ID: Q135693
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

SYMPTOMS

After you install the Andrew ISA IIA Adapter 1 TR network interface card (NIC) driver in Windows NT 3.51, the following error message appears: 

   ANDTOK1: has determined that the adapter is not functioning
   properly.

WORKAROUND

To work around this problem, use another supported NIC. Refer to the Windows NT 3.51 Hardware Compatibility List (HCL).

STATUS

Andrew Corporation is aware of this problem and is currently working on a fix. For more information, contact Andrew Corporation at (800) 826-3739.
