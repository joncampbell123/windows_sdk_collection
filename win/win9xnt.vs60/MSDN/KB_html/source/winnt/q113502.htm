Patch Restores COMPAQ Windows NT Files Deleted By Service Pack

 Last reviewed: May 14, 1997
Article ID: Q113502

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

COMPAQ's SP0703.ZIP patch contains OEM Windows NT files originally installed with Windows NT on COMPAQ machines but overwritten when Service Pack 2 for Windows NT is applied. SP0703.ZIP is available on CompuServe in the Software Solutions section of the COMPAQ forum (GO COMPAQ) and from the COMPAQ BBS at (713) 378-1418.

SP0703.ZIP is dated 3/25/94 and has a filesize of 98026 bytes. It is self extracting, and unzips to create SP0703.EXE (158914 bytes, 2/25/94 03:09p). It contains these files for the update process: 

02/25/94  03:08p                 18421 OEMNADDS.IN_
02/25/94  03:08p                  6647 OEMNADLM.IN_
02/25/94  03:08p                  7356 OEMNADLT.IN_
02/25/94  03:08p                 21930 OEMNXPSM.IN_
02/25/94  03:08p                 22804 PATCHDLL.DLL
02/25/94  03:08p                 35328 UPDATE.EXE
02/25/94  03:06p                 26643 UPDATE.INF
02/25/94  03:08p                  2056 UPDATE.RDM (instructions for
                                                   applying the update)
