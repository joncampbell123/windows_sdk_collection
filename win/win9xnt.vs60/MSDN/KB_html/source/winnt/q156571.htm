How to Perform Scandisk in Windows NT 4.0

 Last reviewed: March 27, 1997
Article ID: Q156571
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 4.0
  • Microsoft Windows NT Server versions 4.0

After you install Windows NT 4.0, there is no scandisk command in the System tools folder.

To check the hard disk drive integrity in Windows NT 4.0, follow the following steps: 

   1. Double-click My Computer and select the hard disk drive you want to
      check.

   2. Use the mouse to point to the selected hard drive and click the right
      mouse button.

   3. Select Properties and click Tools.

   4. In Error-checking, click Check Now.
