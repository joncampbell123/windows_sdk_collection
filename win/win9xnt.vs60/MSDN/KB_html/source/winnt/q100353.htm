True Type Fonts Supported by Windows NT

 Last reviewed: May 6, 1997
Article ID: Q100353
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1, 3.5, and 3.51
  • Microsoft Windows NT Advanced Server version 3.1, 3.5, and 3.51

True Type fonts designed for Microsoft Windows operating system version 3.1 are supported by Windows NT. Any installed True Type fonts will be available to both 16-bit and 32-bit Windows-based applications. They will not be available in the character-based subsystems, such as MS-DOS, OS/2, and POSIX.

Replacement font managers designed for Windows 3.1, such as Adobe Type Manager and Bitstream FaceLift, are not supported and will not work under Windows NT. These products require updates to work under Windows NT. Contact your font manager vendor for more information.
