UltraStor Controller Stop Message Inaccessible Boot Device

 Last reviewed: May 12, 1997
Article ID: Q106375
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

If you attempt to set up Windows NT 3.1 on an ESDI hard disk with the UltraStor 12f ESDI controller, you may receive the following STOP message: 

    Stop (hex dump)
    Inaccessible_boot_device

WORKAROUND

According to UltraStor, you must upgrade the firmware to version 28001-012 and the BIOS to version 38001-010.

To order these upgrades, call UltraStor Technical Support at (510) 623-8955. If you have an EPROM burner, you can also download the micro code images from the UltraStor BBS at (510) 623-9091.
