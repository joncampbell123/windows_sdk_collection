Err Msg: Unexpected DOS Error 11 Using Server Tools

 Last reviewed: March 18, 1997
Article ID: Q140642
3.50 3.51 WINDOWS kbtools

The information in this article applies to:

  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

When starting Windows NT Remote Server Tools 3.51, such as User Manager for Domains, from a Windows NT 3.5 Workstation, you will receive the following error message: 

   Unexpected DOS Error 11
This error message will also appear if you run the Windows NT Remote Server Tools 3.5 from a Windows NT 3.51 Workstation.

RESOLUTION

The Windows NT Remote Server Tools are version specific. You must run the Windows NT Remote Server Tools from the same version as the version of Windows NT Workstation that is running the Server Tools.
