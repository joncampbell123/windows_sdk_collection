Sample SMSVIEW Reports and Queries Not Included in SMS

 Last reviewed: April 23, 1997
Article ID: Q131699
The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

Page 665 of the Systems Management Server Administrator's Guide incorrectly states the following: 

   SMS provides sample reports and queries for use with Microsoft Word
   for Windows version 6.0, Microsoft Excel version 5.0, and Microsoft
   Access version 2.0
The Systems Management Server 1.0 compact disc does not include the sample reports and queries.
