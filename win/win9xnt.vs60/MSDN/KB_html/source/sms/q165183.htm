SMS: "Could Not Spawn Initmain.exe" Error Message

 Last reviewed: March 25, 1997
Article ID: Q165183
The information in this article applies to:
  • Microsoft Systems Management Server, version 1.2

Systems Management Server clients may encounter the following error message during the execution of the Smsls.bat file: 

   Could Not Spawn Initmain.exe
This error message may occur when inventorying a computer by means of a logon script, and the computer is low on executable memory (possibly because Emm386 is not loaded).
