SMS Inventory Processor: Compilation Error on Dell Computers

 Last reviewed: April 22, 1997
Article ID: Q126027

The information in this article applies to:
  • Microsoft Systems Management Server version 1.0

SYMPTOMS

Some Dell computers are reporting the BIOS date in European format instead of United States (American) format, causing the SMS Inventory Processor to report the following error when processing a RAW file from one of the computers: 

   Compilation Error
NOTE: All errors are posted in the INVPROC.LOG file.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.0. This problem was corrected in Systems Management Server version 1.1.
