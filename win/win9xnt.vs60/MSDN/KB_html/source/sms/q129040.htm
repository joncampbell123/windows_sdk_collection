Cannot Remote Control a 256 Color 1024x768 SVGA Machine

 Last reviewed: April 23, 1997
Article ID: Q129040

The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

SYMPTOMS

When you attempt to remote control a client which has its video resolution set at 1024x768 with 256 colors, you connect to the client but the Quick Window Viewer is completely gray.

CAUSE

Remote Control is not able to view the desktop at 1024x768 with 256 colors.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.0 and 1.1. This problem was corrected in the latest US Service Pack for Systems Management Server. For information on obtaining this update, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
