Help Desk Options Incorrectly Displays Error Message

 Last reviewed: April 23, 1997
Article ID: Q136546

The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

If you run Help Desk Options on a computer that does not have USERTSR or USERIPX loaded, the following incomplete error appears: 

   Current setting not available. Remote control agent
The error should display: 

   Current setting not available. Remote control agent not loaded.
Microsoft has confirmed this to be a problem in Systems Management Server version 1.0 and 1.1. This problem was corrected in the latest US Service Pack for Systems Management Server. For information on obtaining this update, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K

Additional query words: prodsms sms help desk remote
