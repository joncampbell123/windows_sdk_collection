SMS 1.1 Does Not Detect IP Configuration Parameters

 Last reviewed: March 4, 1998
Article ID: Q141585
The information in this article applies to:
  • Microsoft Systems Management Server version 1.1

SYMPTOMS

Systems Management Server 1.1 does not detect IP configuration parameters on localized versions of Windows 95.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.1. This problem has been corrected in Systems Management Server version 1.2.
