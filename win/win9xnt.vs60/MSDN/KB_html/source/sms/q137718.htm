APPCTL /Delete Command Line Option Does Not Work

 Last reviewed: April 23, 1997
Article ID: Q137718
The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

On page 534, the Systems Management Server "Administrator's Guide" incorrectly states that /DELETE is a valid command line option for APPCTL: 

   /delete
      Removes all SMS program groups from the Program Manager and from the
      local registry.
The correct command line option is the following: 

   /deinstall
The SMS Help file (SMS.HLP) lists the correct command line option for APPCTL.
