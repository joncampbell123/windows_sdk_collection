SMS: PGC Does Not Work in a Trusted Domain With Windows 95

 Last reviewed: May 21, 1997
Article ID: Q157589

The information in this article applies to:
  • Microsoft Systems Management Server version 1.2

SYMPTOMS

When you use Program Group Control (PGC) and Windows 95 in a trusted domain, Windows 95 client computers do not receive shared applications. This symptom occurs if the Windows 95 client computer is validated in a master user domain and Systems Management Server is located in a resource domain. Windows 95 client computers can receive PGC applications if it is validated in the resource domain.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.2. A fix to this problem is in development, but has not been regression-tested and may be destabilizing in production environments. Microsoft does not recommend implementing this fix at this time. Contact Microsoft Product Support Services for more information on the availability of this fix. receive
