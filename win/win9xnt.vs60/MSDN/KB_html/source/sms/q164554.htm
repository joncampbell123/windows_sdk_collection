SMS: Different Site/Domain Message Is Always in English

 Last reviewed: May 21, 1997
Article ID: Q164554
The information in this article applies to:
  • Microsoft Systems Management Server, version 1.2

SYMPTOMS

When you join a different site and/or domain that is localized (for example, French or German) by connecting to the SMS_SHR share and running the Runsms.bat batch file, the text in the installation window is not localized. The installation window shows the following message: 

   You are logged on to a different Site/Domain Combination.
This text should appear in the site's default language, which is not always English.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.2. This problem was corrected in the latest Microsoft Systems Management Server version 1.2 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
