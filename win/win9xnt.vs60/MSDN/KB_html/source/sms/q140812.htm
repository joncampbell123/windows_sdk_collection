File Date Stamps in AUDIT.RUL Do Not Match Actual Date Stamps

 Last reviewed: April 23, 1997
Article ID: Q140812
The information in this article applies to:
  • Microsoft Systems Management Server version 1.1

SYMPTOMS

The date stamps of the Microsoft software products listed in the Systems Management Server AUDIT.RUL file do not match the date stamps on the files from the actual software products.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.1. This problem has been corrected in Systems Management Server version 1.2.
