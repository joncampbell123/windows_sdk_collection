Microsoft Systems Management Server Help File, November 1995

 Last reviewed: November 7, 1997
Article ID: Q128534

The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

The Microsoft Product Support Services (PSS) Knowledge Base group regularly compiles the articles from the Microsoft Systems Management Server database and creates a Windows Help file that is available in the Microsoft Software Library on the Microsoft Network, the Microsoft Download Service (MSDL), Microsoft OnLine, and the Internet.

The Help file for Microsoft Systems Management Server is in a self- extracting file named SMSKB.EXE and contains technical articles written by PSS Engineers and Knowledge Base writers published prior to December 5, 1995. These articles contain:

  • References to files available in the Microsoft Software Library
  • Bug lists and fix lists for Systems Management Server
  • Additional references
  • Questions and answers
  • Recent product information not addressed in the user documentation
  • Helpful tips

NOTE: This Help file can only be run from Windows 95 or Windows NT version 3.51. If you have Windows NT 3.5 or earlier, you must upgrade to version 3.51 to view this Help file.

SMSKB.EXE contains the following files: 

   SMSKB.HLP   The KB Help file
   SMSKB.CNT   The Contents file

You can find SMSKB.EXE (size: 212039 bytes) 
                      , a self-extracting file, on the following
services:
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select the product.

          Enter kbfile SMSKB.EXE (size: 212039 bytes) 
                            , and click GO!
      Open the article, and click the button to download the file.


  • Internet (anonymous FTP)

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder.
      Get SMSKB.EXE (size: 212039 bytes) 

    

  • The Microsoft Network

          On the Edit menu, click Go To, and then click Other Location.
      Type "mssupport" (without the quotation marks).
      Double-click the MS Software Library icon.
      Find the appropriate product area.
      Locate and Download SMSKB.EXE.

    

  • Microsoft Download Service (MSDL)

          Dial (425) 936-6735 to connect to MSDL
      Download SMSKB.EXE (size: 212039 bytes)
