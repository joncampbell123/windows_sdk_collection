German PDF File for Office 4.2a Detects Office 4.2c

 Last reviewed: April 23, 1997
Article ID: Q140808
The information in this article applies to:
  • Microsoft Systems Management Server version 1.1

SYMPTOMS

German PDF file, OFF42A.PDF, for Office 4.2a detects Office 4.2c. Since the first version of Microsoft Office released in Germany was version 4.2c, the file name OFF42A.PDF indicates the wrong version.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.1. This problem has been corrected in Systems Management Server version 1.2.
