QuickViewer Does Not Pass Through ALT+Print Screen

 Last reviewed: April 23, 1997
Article ID: Q140053

The information in this article applies to: - Microsoft Systems Management Server versions 1.0 and 1.1

SYMPTOMS

When you remotely control a workstation, QuickViewer does not pass through the "ALT+Print Screen" key sequence regardless of how the "system key pass through" option is set.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server versions 1.0 and 1.1. This problem has been corrected in Systems Management Server version 1.2.
