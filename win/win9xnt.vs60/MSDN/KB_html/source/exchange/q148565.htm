XFOR: IMC Fails to Start with 2140, Event IDs 4014 4032 4057

 Last reviewed: March 16, 1998
Article ID: Q148565
The information in this article applies to:
  • Microsoft Exchange Server, versions 4.0, 5.0, 5.5

An Exchange administrator can define the way messages will be delivered based on the domain name of the destination. This is configured through the Internet Mail Connector (IMC) or Internet Mail Service (IMS) Connections property page. If one or more per domain routing is set up incorrectly, the MSExchangeIMC or MSExchangeIMS service will fail to start with an Error 2140, as well as logging Event IDs 4032, 4057, and 4014 in the Windows NT Event Viewer Application log. Resolution requires correcting or removing the erroneous per domain routing entry, which is specified in the previously listed in the Event Viewer entries. This is by design.
