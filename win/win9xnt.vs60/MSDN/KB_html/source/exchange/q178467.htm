XADM: CD Key for Exchange 5.5 Missing in MCSP Package

 Last reviewed: February 27, 1998
Article ID: Q178467
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5
  • Microsoft Outlook Windows 95 client, version 8.03

SYMPTOMS

The package sent to Microsoft Certified Solution Providers (MCSPs) beginning in December 1997 that contains the Exchange Server 5.5 CD-ROM and Outlook 8.03 CD-ROM is missing the CD key.

WORKAROUND

MCSPs can use the same CD key that is contained in their Microsoft Developer Network (MSDN) package for their Exchange and Outlook CDs.
