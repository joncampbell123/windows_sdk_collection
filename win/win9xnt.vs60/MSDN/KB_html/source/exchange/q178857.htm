XADM: Setup.exe Not Found On Premier Select CD-ROM

 Last reviewed: February 12, 1998
Article ID: Q178857
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

The Setup.exe file is not found on the Premier Select CD-ROM.

CAUSE

Select CD-ROMs include both the Standard and Enterprise versions of Exchange Server.

WORKAROUND

To avoid including two Setup.exe files, you must run the version specific executables.

To install or upgrade to Exchange Server 5.5 Standard Edition, use the Srvmin.exe file. To install or upgrade to Exchange Server 5.5 Enterprise Edition, use the Srvmax.exe file.

MORE INFORMATION

The actual differences between the Standard and Enterprise versions of Exchange Server are changes made to the heuristics bit in the RAW property of the server. SRVMIN and SERVMAX are simply a graphical interface to allow the user to add/remove options; the same application files are used.
