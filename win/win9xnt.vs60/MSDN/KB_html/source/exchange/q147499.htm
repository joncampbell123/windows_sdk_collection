XADM: Service Did Not Start Due to Logon Failure

 Last reviewed: April 21, 1997
Article ID: Q147499
The Information in this article applies to:

- Microsoft Exchange Server, version 4.0

SUMMARY

When installing the Exchange Server the user may encounter the "STOP" message: 

  While attempting to start the Microsoft Exchange System Attendant
  service, the following error was encountered:

    The service did not start due to a logon failure.

  Microsoft Windows NT
  ID no: 0xc002042

MORE INFORMATION

This error is related to the account that was used when prompted for "Choose the services account and type the Password" at the beginning of the Exchange Server setup.

RESOLUTION

Enter the correct password for the chosen account. If the password cannot be remembered, run User Manager for Domains, provided with Microsoft Windows NT Server, and reset the account's password.
