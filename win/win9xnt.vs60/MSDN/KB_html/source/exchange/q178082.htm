XFOR: BCC Recipient Appears in To: Field When Sending to PROFS

 Last reviewed: February 13, 1998
Article ID: Q178082
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

When you send a message that has a recipient on the BCC: line to a PROFS user over the Internet, the e-mail message arrives with the BCC: recipient appearing in the To: line. The BCC: recipient should be hidden and should never be displayed in the header information except to the originator of the message.

STATUS

Microsoft has confirmed this to be a problem Microsoft Exchange Server version 5.5. A supported fix is now available, but has not been fully regression-tested and should be applied only to systems experiencing this specific problem. Unless you are severely impacted by this specific problem, Microsoft recommends that you wait for the next Service Pack that contains this fix. Contact Microsoft Technical Support for more information.
