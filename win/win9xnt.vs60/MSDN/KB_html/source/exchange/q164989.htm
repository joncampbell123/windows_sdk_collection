XADM: Error -1003 When You Run EDBUTIL

 Last reviewed: November 25, 1997
Article ID: Q164989
The information in this article applies to:
  • Microsoft Exchange Server version 4.0, 5.0

SYMPTOMS

When you run EDBUTIL you may receive an error -1003.

CAUSE

This error occurs because you have an incorrect switch in the command line.

MORE INFORMATION

When you use EDBUTIL, certain switches are required to specify which operations should be run. When the switches are improperly specified, a help session is usually enabled. In rare circumstances, an improper switch will cause the -1003 error.
