XFOR: Migration Wizard Contains Inaccurate Help File Option

 Last reviewed: December 1, 1997
Article ID: Q175103
The information in this article applies to:
  • Microsoft Exchange Server version 5.5

SYMPTOMS

In the Migration Wizard, if you set the focus on the Help button and then press F1, the following note is displayed: 

   Click this to display an overview of this dialog box.
   For Help on an item, click ? at the top of the dialog box, and then
   click the item.
The Migration Wizard dialog boxes do not have the Help (?) button in the upper-right corner of the dialog box.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 5.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
