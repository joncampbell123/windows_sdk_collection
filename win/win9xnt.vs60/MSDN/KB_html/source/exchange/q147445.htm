XCLN: Cannot Toggle Header When Sending Message

 Last reviewed: April 2, 1997
Article ID: Q147445
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SYMPTOMS

When using Microsoft Exchange Windows clients, you are not able to toggle the message headers on or off when composing a message.

This may make composing messages difficult on machines that are using standard VGA 640x480 resolution. In this configuration the message header may take up a large portion of the screen, especially in headers with many recipient names.

CAUSE

This behavior is by product design. It is not possible to turn off the header information which contains the To, From, Sent and Subject information when composing a message.

WORKAROUND

To work around this problem, toggle off all other options under the View menu to improve the viewing and composing of the text in your message.
