XCON: Two Error Messages When Installing Notes Connector

 Last reviewed: March 6, 1998
Article ID: Q181915
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

When you install the Exchange Notes Connector (for Lotus Notes), the following error message may appear: 

   Read only file, [...], was found while attempting to copy files to the
   destination location. To overwrite the file, click the Yes button,
   otherwise click the No button.
After you click Yes, another error message appears: 

   Could not update the e-mail address generator [...]. The address
   generator is probably in use by the Microsoft Exchange Server System
   Attendant. Choose 'Yes' to retry the update or 'No' to exit setup.
Clicking Yes will continue the Setup process.

CAUSE

Both of these error messages mean that the file being copied is held open by another process. Both error messages mean the same thing, the second one being more descriptive.

WORKAROUND

Clicking Yes will continue with Setup; however, it is recommended that you click No and try to determine the cause of the files being open.

Click No to exit the Setup program. Next, stop and restart all the Exchange Services, or restart the server. This should clear up any file locking problems. After restarting, start the Exchange Notes Connector Setup again.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server, version 5.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
