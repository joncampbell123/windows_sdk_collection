XCLN: Empty the 'Deleted Items' Folder Option doesn't Work

 Last reviewed: April 21, 1997
Article ID: Q155510
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, version 4.0

SYMPTOMS

When you enable the Empty the 'Deleted Items' Folder Upon Exiting option using the Options command on the Tools menu of a Microsoft Exchange client, the Deleted Items folder will not be emptied.

MORE INFORMATION

If you minimize the Microsoft Exchange client for Windows 95 and select either Close All Programs and Log on as a Different User or Restart the Computer, the Deleted Items folder will not be emptied.

STATUS

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft Exchange. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
