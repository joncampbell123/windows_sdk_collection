XADM: Server Monitor Cannot Restart Services On Cluster Server

 Last reviewed: January 23, 1998
Article ID: Q177764
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

When you set up a Server Monitor for a Microsoft Exchange Cluster Server, you will notice that all the fields on the Action tab for the Server Monitor properties are disabled.

This is by product design.

The Action tab contains all the service and server restart settings. A Server Monitor cannot cycle the services on a Microsoft Exchange Cluster Server because it would undermine the Cluster Environment.
