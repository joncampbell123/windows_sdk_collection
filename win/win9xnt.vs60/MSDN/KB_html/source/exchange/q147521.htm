XADM: MTA Crashes with a Large Recipient List

 Last reviewed: April 21, 1997
Article ID: Q147521
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOMS

When a user attempts to send a message to a large number of recipients, either selected from the Address Book or manually typed, the Microsoft Exchange Message Transfer Agent (MTA) fails to convert the P2 message and faults in the error path. This causes the MTA to crash.

WORKAROUND

Use a Microsoft Exchange Distribution List.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange version 4.0. This problem was corrected in the latest Microsoft Exchange Server U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
