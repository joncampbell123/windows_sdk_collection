XGEN: How to Obtain Exchange Server 4.0 Service Pack 4

 Last reviewed: April 9, 1997
Article ID: Q158693
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

WARNING: Due to U.S. State Dept. export regulations, all SP4 client software on these online services contains EXPORTABLE (40-bit) security only. U.S. & Canadian customers should NOT install these, or else they will no longer be able to read existing 64-bit encrypted mail. A North American- only version of SP4 is available to US and Canadian customers (for $14.95 plus shipping and handling) by calling Microsoft at 1-800-360-7561.

VERY IMPORTANT: Please note that while SP4 contains the fixes included in SP3, it does not contain the fixes included in SP2. You MUST install SP2 before installing SP4.

SUMMARY

This article lists the files available for SP4 and where they can be found. For information on how to install SP4, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q158699
   TITLE     : XGEN: README.TXT: Microsoft Exchange 4.0 Service Pack 4

AVAILABLE FORMATS FOR MICROSOFT EXCHANGE 4.0 SERVICE PACK 4

Service Pack 4 is available as single self-extracting archive files for Intel (x86), MIPS, PPC, ALPHA, and Macintosh-based computers.

ORDERING MICROSOFT EXCHANGE 4.0 SERVICE PACK 4

If you do not have access to the electronic services listed below, you can order Service Pack 4 (CD-ROM) from the Microsoft at 1-800-360-7561.

INSTRUCTIONS FOR DOWNLOADING

You can download any of these self-extracting files from the following services:

  • Internet (anonymous FTP)

  1. Ftp to ftp://ftp.microsoft.com/softlib/mslfiles or ftp://198.105.232.1/softlib/mslfiles. 

         2. Click the appropriate file's hyperlink.
  • Microsoft's World Wide Web Site on the Internet

    1. On the www.microsoft.com home page, click Support on the toolbar.
    2. Click Microsoft Technical Support. 3. Select Exchange in the dropdown listbox and click Go. 4. Click the Microsoft Exchange Service Packs hyperlink. 5. Click the appropriate Library File.
    • Microsoft's Download Service

    1. Connect to MSDL.
    2. Type [A] Download File. 3. Type the appropriate file name and press Return.

    For additional information about downloading from Online Services, please see the following article in the Microsoft Knowledge Base:
