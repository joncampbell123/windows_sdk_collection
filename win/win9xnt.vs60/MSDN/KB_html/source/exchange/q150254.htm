XGEN: How to Obtain Exchange Server 4.0 U.S. Service Pack 1

 Last reviewed: January 6, 1998
Article ID: Q150254
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0 

 ** THIS U.S. SERVICE PACK 1 IS FOR MICROSOFT EXCHANGE SERVER 4.0 ONLY!! **
The Microsoft Exchange Server U.S. Service Pack 1 is a cumulative build of all fixes to Microsoft Exchange Server version 4.00.

To find out which bugs are fixed by this update, see the README.TXT file that accompanies the Service Pack or query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K

AVAILABLE FORMATS FOR MICROSOFT EXCHANGE SERVER 4.0 SERVICE PACK 1

Service Pack 1 is available as single self-extracting archive files for Intel (x86), MIPS, and ALPHA-based computers.

ORDERING MICROSOFT EXCHANGE SERVER 4.0 SERVICE PACK 1

If you do not have access to the electronic services listed below, you can order Service Pack 1 (CD-ROM or Disk Set) from the Microsoft Sales Information Center (MSIC) at (800) 426-9400.

INSTRUCTIONS FOR DOWNLOADING

Because of their large size, the self-extracting files for Microsoft Exchange Service Pack 1 are only available on the Internet and The Microsoft Network. You can obtain these self-extracting files on:

INSTALLING MICROSOFT EXCHANGE SERVER SERVICE PACK 1

To install the Service Pack from compressed files:

  1. Create two subdirectories, SETUP and SUPPORT.

  2. Download the compressed setup file appropriate for your hardware 

        platform into the SETUP directory.


        Select the appropriate file as indicated by the following list:


           Alpha AXP(TM):  SP1_400A.EXE

           Intel(R):       SP1_400I.EXE
       MIPS(R):        SP1_400M.EXE
       PowerPC(TM):    SP1_400P.EXE


  3.  Download the compressed support file appropriate for your hardware

        platform into the SUPPORT directory.

    

        Select the appropriate file as indicated by the following list:

    

           Alpha AXP(TM):  SP1S400A.EXE

           Intel(R):       SP1S400I.EXE
       MIPS(R):        SP1S400M.EXE
       PowerPC(TM):    SP1S400P.EXE


  4.  At the Windows NT command prompt, go to the SETUP directory and type

        the filename (downloaded in Step 2) followed by a -d to ensure that the
    correct directories are created. For example, to decompress the file
    SP1_400A.EXE, type the following command:

    

           SP1_400A.EXE -d

    

  5.  At the Windows NT command prompt, go to the SUPPORT directory and type

        the filename (downloaded in Step 3) followed by a -d to ensure that the
    correct directories are created. For example, to decompress the file
    SP1S400A.EXE, type the following command:

    

           SP1S400A.EXE -d

    

After the service pack has been decompressed, run the UPDATE.EXE program
from the SETUP directory to install the updated files.
