XADM: How to Change Your Name in Microsoft Exchange Server

 Last reviewed: April 3, 1997
Article ID: Q158559

The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

If you change your name (for example, because you get married) and you want to change your name accordingly in Microsoft Exchange, follow the procedure in this article.

MORE INFORMATION

To change your alias and mailbox name:

  1. Select the appropriate recipients container, select your mailbox and select File-Properties.

  2. On the General property page change the First, Last, Display, and or Alias fields as appropriate.

  3. Make the appropriate changes to the E-mail Addresses tab for X400, MS, and SMTP addresses. The mailbox name for the client profile will also need to be changed to reflect any changes made to the display name.
