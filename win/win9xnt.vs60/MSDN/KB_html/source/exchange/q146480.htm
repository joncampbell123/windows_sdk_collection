XGEN: Additional Addresses Not Included with Resent Message

 Last reviewed: January 6, 1998
Article ID: Q146480
The Information in this article applies to:
  • Microsoft Exchange Server, version 4.0

When you resend an Exchange 4.0 message that has failed with an NDR and it includes additional user names in the TO or CC line, the additional user names do not appear on the recipients' message. Only the first user name address appears on the TO line.

This is by design.
