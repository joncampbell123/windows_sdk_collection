XFOR: No Help File in Exchange Notes Connector

 Last reviewed: January 27, 1998
Article ID: Q177416
The information in this article applies to:
  • Microsoft Exchange Server version 5.5

SYMPTOMS

When you run the Microsoft Exchange Administrator program and attempt to view the Help file for the Exchange Notes Connector, the Help file does not appear, even though the Help button is selectable. To locate the Help button, open the Exchange Notes Connector properties page, click the Queues tab, and then click the Delete button.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 5.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
