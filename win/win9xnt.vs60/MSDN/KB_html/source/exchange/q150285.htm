XCLN: StarNine Mail Subject and Body Text Truncated

 Last reviewed: March 31, 1997
Article ID: Q150285
The information in this article applies to:
  • Microsoft Exchange Windows client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • StarNine Mail version 3.1d
  • Microsoft Mail for PC Networks version 3.x clients

SUMMARY

Users and administrators of Microsoft Exchange Server might notice that messages inbound from StarNine mail will have a subject line that will not exceed 40 characters and the body text will be wrapped at 79 characters.

This is by design. This will also occur for other gateways which can handle more than 40 characters in the subject and 79 characters in the body.
