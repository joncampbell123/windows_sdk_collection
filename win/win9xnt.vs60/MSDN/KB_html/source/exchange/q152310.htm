XADM: Event id 5007 - Check Free Disk Space

 Last reviewed: January 5, 1998
Article ID: Q152310
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

You might receive the following error in the application log of the Windows NT Event Viewer: 

   Event id=5007  Source= MSEXchange SA
   An error occurred during the message tracking decode operation.

MORE INFORMATION

This error indicates that the disk drive where the tracking logs are stored is full. The error message should indicate that there is not enough free disk space.

STATUS

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft Exchange. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
