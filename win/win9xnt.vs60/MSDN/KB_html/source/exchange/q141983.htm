XCLN: Msg: Do You Want to Save Changes to This Item?

 Last reviewed: April 3, 1997
Article ID: Q141983
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange MS-DOS client, versions 4.0 and 5.0

SUMMARY

When you exit or close the Microsoft Exchange client, you may get the following message: 

   Do you want to save changes to this item? (Yes/No/Cancel)

MORE INFORMATION

Any message that has been edited and left open or minimized in the icon form will generate this message. The subject of the current message is not indicated with this dialog box, but the messages that you have been working on will become maximized and brought to the front along with the dialog box. The dialog box will appear for each message until all open messages have been saved, discarded, or sent.

This feature allows you to save, discard or send any message that you may currently have open.
