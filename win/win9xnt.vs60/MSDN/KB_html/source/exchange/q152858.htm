XGEN: How to Obtain Exchange Server 4.0 U.S. Service Pack 2

 Last reviewed: January 6, 1998
Article ID: Q152858
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

WARNING: Due to U.S. State Dept. export regulations, all SP2 client software on these online services contains EXPORTABLE (40-bit) security only. U.S. & Canadian customers should NOT install these, or else they will no longer be able to read existing 64-bit encrypted mail. A North American- only version of SP2 is available to US and Canadian customers (for shipping and handling) by calling Microsoft at 1-800-360-7561.

SUMMARY

This article lists the files available for Microsoft Exchange Service Pack 2. It describes where they can be found and how to install them.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q153655
   TITLE     : README.TXT: Microsoft Exchange 4.0 U.S Service Pack 2

MORE INFORMATION

The Microsoft Exchange Server U.S. Service Pack 2 is a cumulative build of all fixes to Microsoft Exchange Server version 4.0.

AVAILABLE FORMATS FOR MICROSOFT EXCHANGE 4.0 SERVICE PACK 2

Service Pack 2 is available as single self-extracting archive files for Intel (x86), MIPS, PPC, and ALPHA-based computers.

ORDERING MICROSOFT EXCHANGE 4.0 SERVICE PACK 2

If you do not have access to the electronic services listed below, you can order Service Pack 2 (CD-ROM) from the Microsoft at 1-800-360-7561.

INSTRUCTIONS FOR DOWNLOADING

You can download any of these self-extracting files from the following services:

- Internet (anonymous FTP) 

     1. Ftp to ftp://ftp.microsoft.com/softlib/mslfiles or
        ftp://198.105.232.1/softlib/mslfiles.
     2. Click the appropriate file's hyperlink.
  • Microsoft's World Wide Web Site on the Internet 

         1. On the www.microsoft.com home page, click Support on the toolbar.
     2. Select Microsoft Exchange in the Product listbox and click Next.
     3. Click the Read the Feature Articles button.
     4. Click the Microsoft Exchange Service Pack 2 hyperlink.
     5. Click the Click This Button to Retrieve the File List button.
     6. Click the appropriate Library File.
  • Microsoft's Download Service 

         1. Connect to MSDL.
     2. Enter [A] Download File.
     3. Enter the appropriate file name and hit return.

    INSTALLING MICROSOFT EXCHANGE SERVER SERVICE PACK 2

    To install the Service Pack for Microsoft Exchange clients from compressed files:

    1. Create a Client sub-directory.
    2. Download the compressed setup file appropriate for the client for 
       your hardware platform directory.
    3. At a command prompt, go to the Client directory and type the filename 
       (downloaded in Step 2) followed by a -d to ensure that the correct
   directories are created. Sp2_40na -d, for example.

  • To update a client go to the Client and platform directory, and type Setup. Follow the instructions displayed on the screen.

    To install the Service Pack for Microsoft Exchange Servers from compressed files:

    1. Create two subdirectories, Setup and Support.
    2. Download the compressed setup file appropriate for your hardware 
        platform into the Setup directory.
    3. Download the compressed support file appropriate for your hardware 
        platform into the Support directory.
    4. At the Windows NT command prompt, go to the Setup directory and type 
        the filename (downloaded in Step 2) followed by a -d to ensure that
    the correct directories are created. Sp2_400a.exe -d, for example.
    5. At the Windows NT command prompt, go to the Support directory and 
        type the filename (downloaded in Step 3) followed by a -d to ensure
    that the correct directories are created. Sp2s400a.exe -d, for
    example.
    After the service pack has been decompressed, run the Update.exe program from the Setup directory to install the updated files.
