XCLN: Extended Characters do not Appear in the File Name

 Last reviewed: April 15, 1997
Article ID: Q157690
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, version 4.0

SYMPTOMS

When you attempt to mail a Microsoft Word for Windows 95 document, and it has extended characters in it's file name, using the Send command from the File menu, the extended characters will not show up correctly. This only occurs when the computer is running Windows 95, the Microsoft Exchange client for Windows 95, and Microsoft Word for Windows 95.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 4.0. This problem was corrected in the latest Microsoft Exchange Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
