XFOR: MSMail Migration Wizard Stops in Progress

 Last reviewed: July 15, 1997
Article ID: Q166593

The information in this article applies to:

  • Microsoft Exchange Server version 5.0

SYMPTOMS

The Microsoft Mail for PC Networks Migration stops responding during the migration process.

This happens on a multi-processor computer, when the Publishing Editor Access: Includes Create Folders option is selected.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 5.0.
