XADM: Performance Optimizer Display Control Settings

 Last reviewed: March 2, 1998
Article ID: Q181922
The information in this article applies to:
  • Microsoft Exchange Server, versions 4.0, 5.0, 5.5

SUMMARY

Display settings for the Performance Optimizer options are maintained in the following registry key: 

   HKEY_CURRENT_USER\SOFTWARE\MICROSOFT\PERFWIZ\QUESTION STATE
The information is stored as a REG_BINARY entry.

MORE INFORMATION

The following is a breakdown of all options and how they map to this key value. The complete entry is a 32-digit string.

Estimated # of Users: Variable is read from the first two values in the 32-digit string. 

1-25        = 00
26-50       = 01
51-100      = 02
101-250     = 03
251-500     = 04
More than 500 = 05

Type of Server: Variable is read from the 18th value in the 32-digit string. 

Nothing Selected                                         = 0
Public                                                   = 1
Private                                                  = 2
Private,Public                                           = 3
Connector/Directory Import                               = 4
Connector/Directory Import,Public                        = 5
Connector/Directory Import,Private                       = 6
Connector/Directory Import,Private,Public                = 7
Multi-server                                             = 8
Private,Multi-server                                     = A
Private,Public,Multi-server                              = B
Connector/Directory Import,Multi-server                  = C
Public,Multi-server,Connector/Directory Import           = D
Private,Multi-server,Connector/Directory Import          = E
Private,Public,Multi-server,Connector/Directory Import   = F
# in organization: Variable is read from the 10th value in the 32-digit string. 

Less than 100   = 0
100-999         = 1
1000-9999       = 2
10000-99999     = 3
100000 or more = 4

Memory Usage: Variable is read from the 25th and 26th values in the 32-digit string. 

No limit    = 00
24 Megs     = 18
32 Megs     = 20
48 Megs     = 30
64 Megs     = 40
128 Megs    = 80
