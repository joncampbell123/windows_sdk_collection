XADM: Cannot Connect to Stand-Alone Server

 Last reviewed: December 19, 1997
Article ID: Q147622
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

When you attempt to install a Standard Edition of Microsoft Exchange Server and connect it to another site, the following error appears: 

   "Server Name" is licensed for stand-alone use only. Install
   Microsoft Exchange Server Connector on "Server Name" if you want
   to join its site.

MORE INFORMATION

Microsoft sells a Standard Edition and an Enterprise Edition version of Microsoft Exchange Server. The Standard Edition will not be able to connect to another server. There is an Add-On package that will upgrade the Standard Edition version with a connector to connect to an existing site or another server in the same site.

NOTE: This problem could also occur if there is corruption on the server. Please contact Microsoft Product Support Services for help in troubleshooting this error if this is not a Standard Edition version.
