XADM: Collabra Migration Tool Does Not Handle OLE Embedded Docs

 Last reviewed: February 12, 1998
Article ID: Q178820
The information in this article applies to:
  • Microsoft Exchange Server, version 5.0

SYMPTOMS

When you try to migrate Collabra forums that contain embedded Word documents into Exchange, the Migration Wizard will stop responding indefinitely.

CAUSE

Fixes were originally provided in Microsoft Exchange Server 5.0 SP1 (detailed in Q166612) that solved many issues with Collabra forums migration; however, this is a missed case.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 5.0.

This problem has been corrected in the latest U.S. Service Pack for Microsoft Exchange Server version 5.0. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
