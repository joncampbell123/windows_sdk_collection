XCLN: Cannot Send Folder as Attachment with Windows Explorer

 Last reviewed: January 5, 1998
Article ID: Q145776
The information in this article applies to:

- Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

SYMPTOMS

When I try to send a folder as an attachment to a Microsoft Exchange e-mail message using the Microsoft Windows 95 Explorer's File menu Send To command, I receive an error message stating: 

   The item could not be displayed. The file could not be opened.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft Windows 95 Explorer. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
