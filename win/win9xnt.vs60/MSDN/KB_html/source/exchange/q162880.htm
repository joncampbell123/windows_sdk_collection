XADM: Event ID-1305 Jet_errObjectNotFound

 Last reviewed: December 18, 1997
Article ID: Q162880
The information in this article applies to:
  • Microsoft Exchange Server version 4.0

SYMPTOMS

When you attempt to run Edbutil /d /r to repair a corrupted EDB file, you may see the following error returned from the Edbutil.exe application: 

   -1305 Jet_errObjectNotFound

CAUSE

This error can occur if you run the Edbutil utility with the /d and /r options. During the repair process one or more datatables is discarded due to corruption. To verify this is the case, check the Application event log for the following Event ID: 

   Event ID: 44
   Source:   EDB
   Type:     Warning

   Description: The database engine lost one table called <datatablename>
This error means that Edbutil discarded this particular table during the repair process. If this table is one of the required tables, such as Deleted Messages, Mailbox, or Folder Tombstone, the store may fail to start and the “-1305 Jet_errObjectNotFound” error message will be generated.

WORKAROUND

Because “-1305 Jet_errObjectNotFound” is an unrecoverable error, you must restore from a valid backup to recover.
