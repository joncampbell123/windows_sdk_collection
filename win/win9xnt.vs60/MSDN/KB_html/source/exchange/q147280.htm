XCLN: Err: Detected Incompatible Version of CTL3DV2.DLL

 Last reviewed: April 2, 1997
Article ID: Q147280
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SYMPTOMS

When you start the Microsoft Exchange client, you may get the following error message: 

   Exchange has detected an incompatible version of CTL3DV2.DLL.
   Some things may not work correctly.

CAUSE

The correct CTL3DV2.DLL file located in the Windows\System directory should be 27,632 bytes and dated 11/30/94. The version number of this file as displayed by Microsoft Diagnostics (MSD) is 2.31.

MORE INFORMATION

Some third party products might install an older version of CTL3DV2.DLL with a modified date. The file installed is CTL3DV2.DLL dated 2/17/95 size 21,648 bytes and version 2.05; actually this is an old version (1993) with a modified date of 1995.
