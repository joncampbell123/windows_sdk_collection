XFOR: NNTP Authentication Denied with Space in NT Password

 Last reviewed: July 15, 1997
Article ID: Q166537

The information in this article applies to:

  • Microsoft Exchange Server, version 5.0

SYMPTOMS

When you try to access NNTP newsgroups, authentication may fail. A 501 (authentication failure) error message appears in the application log.

CAUSE

The NNTP authentication algorithm cannot handle a space in a user's password.

WORKAROUND

Use passwords without spaces.

RESOLUTION

Apply the fix described below. The NNTP authentication algorithm was fixed to properly handle a password that contains spaces.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 5.0. This problem was corrected in the latest Microsoft Exchange Server 5.0 U.S. Service Pack.
