XGEN: How to Obtain Exchange Server 5.0 U.S. Service Pack 2

 Last reviewed: February 13, 1998
Article ID: Q168858
The information in this article applies to:
  • Microsoft Exchange Server, version 5.0

SUMMARY

** THIS U.S. SERVICE PACK 2 IS FOR MICROSOFT EXCHANGE SERVER 5.0 ONLY!! **

The Microsoft Exchange Server version 5.0 U.S. Service Pack 2 is a cumulative build of all fixes to Microsoft Exchange Server version 5.0.

To find out which bugs are fixed by this update and how to install this Service Pack, see the Readme.txt file that accompanies the Service Pack or query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K

Available Formats for Microsoft Exchange Server 5.0 Service Pack 2

Service Pack 2 is available as single self-extracting archive files for Intel (x86), ALPHA, and Macintosh-based computers.

Ordering Microsoft Exchange Server 5.0 Service Pack 2

If you do not have access to the electronic services listed below, you can order Service Pack 2 (CD-ROM or Disk Set) from the Microsoft Sales Information Center (MSIC) at (800) 426-9400.

Instructions for Downloading

Because of their large size, the self-extracting files for Microsoft Exchange version 5.0 Service Pack 2 are only available on the Internet. To obtain these self-extracting files using anonymous FTP on the Internet, go to: 

   ftp://ftp.microsoft.com
-or- 

   ftp://198.105.232.1
and then go to: 

   /bussys/exchange/exchange-public/fixes/eng/exchg5.0/sp2
-or- 

   /Softlib/MSLFILES
The files are also available on Microsoft's World Wide Web site on the Internet. To find them:

  1. Go to http://support.microsoft.com/support.

  2. Click Downloads. Note that you may need to click Show Long Menu before you click Downloads.

  3. Click BackOffice.

  4. Click Exchange Server.

  5. Click the name of the file you want to download.

For additional information about downloading, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
