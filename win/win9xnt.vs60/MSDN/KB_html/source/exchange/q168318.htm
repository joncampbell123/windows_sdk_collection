XADM: Guidgen.exe Available Only for Intel Platforms

 Last reviewed: December 31, 1997
Article ID: Q168318
The information in this article applies to:
  • Microsoft Exchange Server versions 4.0 and 5.0

SUMMARY

The Guidgen.exe utility used in the rebuilding of Site Folders is available for the Intel platform only. However, this utility can be run on any Intel computer to generate a new globally unique identifier (GUID) for an Exchange Server running on Intel or non-Intel platform.

MORE INFORMATION

The GUIDGEN.EXE utility is used to randomly generate a GUID that will be used to rebuild the site folders. This utility is available in the Microsoft Developers Network under the product samples.

For additional information on rebuilding site folders or running Guidgen, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q152960
   TITLE     : XADM: Rebuilding the Site Folders in a Site
