XADM: Exchange Server Must Be Installed in a Domain

 Last reviewed: September 23, 1997
Article ID: Q147709
The Information in this article applies to:
  • Microsoft Exchange Server, version 4.0

Microsoft Exchange Server version 4.0 uses the security features of Microsoft Windows NT Server, including authentication, access control, and auditing, to prevent unauthorized use of the system. As a result, Microsoft Exchange Server must be installed on a Windows NT Server computer that is part of a domain, rather than a workgroup.
