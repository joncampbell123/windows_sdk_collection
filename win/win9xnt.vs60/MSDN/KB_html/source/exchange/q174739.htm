XWEB: Empty Rectangles Appear in Outlook Web Access

 Last reviewed: December 3, 1997
Article ID: Q174739
The information in this article applies to:
  • Microsoft Outlook Web Access version 5.5

SYMPTOMS

When you are accessing an Exchange Server computer by using Outlook Web Access and Netscape Navigator version 4.0, empty rectangles appear in various areas of the screen.

STATUS

This problem occurs when you use Microsoft Outlook Web Access version 5.5 and Netscape Navigator 4.0. This problem will not occur if you use Netscape Navigator 4.01 or later.

MORE INFORMATION

These rectangles appear in various areas of the screen but have no detrimental effect on the Outlook Web Access client.
