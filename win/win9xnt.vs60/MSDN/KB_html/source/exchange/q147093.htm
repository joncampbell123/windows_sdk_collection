XCON: One Invalid X.400 Recp. Causes NDR For All X.400 Recps.

 Last reviewed: April 3, 1997
Article ID: Q147093
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOMS

When there is a invalid X.400 address in the To: field of a message, the Microsoft Exchange Message Transfer Agent (MTA) will not deliver the message to any valid X.400 addresses that might also be present in the To: field.

CAUSE

The O/R Address is generated according to the Oneoff Address and the message is submitted to the MTA. The MTA detects an invalid recipient and generates Non-Delivery Receipts (NDRs) for all addresses. This is by product design.
