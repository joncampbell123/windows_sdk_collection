XWEB: Outlook View Does Not Appear in OWA View List

 Last reviewed: December 1, 1997
Article ID: Q176034
The information in this article applies to:
  • Microsoft Outlook Web Access version 5.5

SYMPTOMS

Folder views defined with Microsoft Outlook may not appear in the list of available views when you are using Microsoft Outlook Web Access (OWA).

WORKAROUND

To allow views created in Microsoft Outlook to be seen in OWA, perform the following steps:

  1. In Microsoft Outlook, select the folder for which you want to create a view.

  2. From the File menu, select Folder, and then Properties for "<folder>".

  3. On the General tab, select the Automatically Generate Microsoft Exchange Views option. Press OK.

  4. Create the view by selecting View and then selecting Define Views.

The view should now be visible from Outlook Web Access. Alternatively, you can create views using Microsoft Exchange.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook Web Access version 5.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
