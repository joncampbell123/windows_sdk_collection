XCLN: Page Fault in Mprexe.exe When Starting Client

 Last reviewed: December 29, 1997
Article ID: Q151708
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Outlook 97

SYMPTOMS
The following error message may appear when you start the Microsoft Exchange client for Windows 95 or Microsoft Outlook 97: 

   MPREXE caused an invalid page fault in Kernel32.dll
In certain cases, the option to remember passwords, "Save this password in your password list", may be grayed out or unavailable.

CAUSE

This is due to a corrupt password list (PWL) file. A user's PWL may become corrupt after upgrading their Windows 95 operating system to the Service Pack 1 (SP1).

WORKAROUND

Locate and delete or rename your PWL file. The file is named <Username>.pwl, where <Username> is the name you use to log on. It should be located in the Windows folder. After you delete or rename the file, restart Windows 95.
