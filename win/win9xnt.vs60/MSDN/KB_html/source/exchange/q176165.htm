XADM: Setup Error Message: This version of setup is older...

 Last reviewed: December 18, 1997
Article ID: Q176165
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

When you run Setup to add or remove components from an existing Microsoft Exchange 5.5 Server computer, you may receive the following error message: 

   This version of Exchange Setup is older than the one you previously
   used. Use a newer version of the Server Setup program.

CAUSE

You are running the Setup program from Microsoft Exchange Server version 5.0.

RESOLUTION

Use the Microsoft Exchange Server version 5.5 Setup program located on the Microsoft Exchange Server version 5.5 CD-ROM.
