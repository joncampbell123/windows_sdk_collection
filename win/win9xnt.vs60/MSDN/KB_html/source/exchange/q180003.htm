XADM: Restoring Public Folder on Offline Recovery Server

 Last reviewed: February 3, 1998
Article ID: Q180003
The information in this article applies to:
  • Microsoft Exchange Server, versions 4.0, 5.0, and 5.5

In certain situations, you may have to restore a public folder from backup to a recovery server. The procedure is similar to restoring a single mailbox from the backup on the recovery server (see below). If the recovery server name is different, the administrator may not be able to open up the property page of a public folder. To be able to open up the property page of a public folder, all the available public folders must be included under Folders on this Information Store on the Instances tab of the <server-name> property.

For additional information on restoring a single mailbox from the backup on the recovery server, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q163713
   TITLE     : Recovering a Single Mailbox from the Online Backup
