XGEN: Build Numbers for Exchange

 Last reviewed: January 6, 1998
Article ID: Q158530
4.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Exchange Server, version 4.0

Each version of Microsoft Exchange Server and subsequent service packs have different build numbers. Listed below are the build numbers for each release: 

Microsoft Exchange Server 4.0   Build 837.10
SP1                             Build 838.15
SP2                             Build 993.4
SP3                             Build 994.64
SP4                             Build 995.54
You can determine the build number by viewing properties of the server.
