XCLN: Same Msg Appears Twice When Opening MS Excel Freedoc

 Last reviewed: April 9, 1997
Article ID: Q147498
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SYMPTOMS

When you open a Microsoft Excel Freedoc that was previously saved with the Read-Only Recommended option, you are prompted twice with the following message: 

   <filename>.XLS should be opened read-only unless you need to save
   changes to it. Open as read-only?

                 (Yes) (No) (Cancel)
No matter whether you choose Yes or No, this message appears again. However, the choice you make during the second appearance is the one Microsoft Excel uses.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To apply the Read-Only Recommended option to a Microsoft Excel document, choose Save As from the Microsoft Excel File menu. Then click Options and click Read-Only Recommended.
