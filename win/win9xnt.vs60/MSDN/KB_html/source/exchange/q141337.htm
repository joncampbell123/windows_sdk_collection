XCLN: Err Msg: The Item Could Not Be Displayed

 Last reviewed: April 21, 1997
Article ID: Q141337
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOMS

In the Microsoft Exchange version 4.0 Windows clients, the following message may appear when you try to open a typical message (IPM.Note): 

   The item could not be displayed. The form could not be loaded.
   Launching default form instead.

CAUSE

This error can occur if the FRMCACHE.DAT and FRMCACHE.BAK are missing from the WINDIR\FORMS directory. WINDIR is the Windows subdirectory, and FRMCACHE.DAT is the Microsoft Exchange Electronic Forms Designer cache that is used to pass information to Microsoft Exchange regarding message item types. FRMCACHE.BAK is a back up file to FRMCACHE.DAT that is updated and always contains electronic forms cache information prior to the last update.

RESOLUTION

Replace FRMCACHE.DAT by restoring the file to the WINDIR\FORMS directory.
