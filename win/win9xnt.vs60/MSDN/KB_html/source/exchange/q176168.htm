XADM: Schema Changes for Exchange Server 5.0 and 5.5

 Last reviewed: December 1, 1997
Article ID: Q176168
The information in this article applies to:
  • Microsoft Exchange Server versions 5.0 and 5.5

SUMMARY

New schema attributes have been added to the Microsoft Exchange Server 5.5 directory.

MORE INFORMATION
