XADM: Admins Can Edit DL Members Even When Hidden from AB

 Last reviewed: December 18, 1997
Article ID: Q176145
The information in this article applies to:
  • Microsoft Exchange Server, versions 4.0, 5.0, and 5.5

SYMPTOMS

If you create a distribution list (DL) on an Exchange Server computer and choose the Hide Membership from Address Book option or the Hide from Address Book option on the Advanced tab, anyone who is allowed to edit the membership (including the owner and administrators) will still be able to view the DL membership.

CAUSE

This is by design.

MORE INFORMATION

Users who are allowed to edit the membership can use the Microsoft Exchange Client or Exchange Administrator program to edit the membership.
