XCON: Can't Mix Cluster & Non-cluster Servers in Server Monitor

 Last reviewed: March 17, 1998
Article ID: Q177999
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

When you are setting up a Server Monitor and adding a Microsoft Exchange Cluster Server to the monitor, the following dialog box may appear: 

   The server you are attempting to add to the Monitored Servers list is a
   Microsoft Cluster Server. The servers currently in the Monitored Servers
   list are not Microsoft Cluster Servers. You are not allowed to mix both
   types of servers in the same Monitored Servers list.

   Microsoft Exchange Administrator
   ID no: c1031ab6

RESOLUTION

Adding the Microsoft Exchange Cluster Server to a new Server Monitor group will work.

CAUSE

Due to the differences between Exchange Cluster Server and non-cluster servers, it would be potentially destabilizing to have these servers controlled by the same Server Monitor.
