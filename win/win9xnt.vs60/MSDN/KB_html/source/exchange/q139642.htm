XCLN: Err Msg: TCP/IP Is Required to Run Internet Mail

 Last reviewed: April 2, 1997
Article ID: Q139642

The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

SYMPTOMS

When you use Internet Mail from a Microsoft Exchange, version 4.0, Windows 95 client the following error message appears: 

   TCP/IP is required to run Internet Mail and is not installed on your
   computer. To install TCP/IP, click the Network icon in the Control
   Panel. Click Add, then click Protocol. Click Microsoft, then install
   TCP/IP.

CAUSE

The TCP/IP protocol is not installed on your computer. Internet Mail depends on TCP/IP to connect to other Internet Mail hosts.

RESOLUTION

Follow the directions given in the error message to install TCP/IP.
