XFOR: Changing InOutThreads does not Affect IMS Performance

 Last reviewed: January 5, 1998
Article ID: Q169845
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

Administrators of the Microsoft Exchange Server Internet Mail Service (IMS) will find that when they increase the registry value InOutThreads, they will not see measurable increases in IMS throughput.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server, version 5.5. We are currently researching this issue and will post more information here in the Microsoft Knowledge Base as it becomes available. and addressed

MORE INFORMATION

This registry key is located under:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\MSExchangeIMC\Paramete rs
