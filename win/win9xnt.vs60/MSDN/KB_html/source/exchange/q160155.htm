XGEN: DocErr: No Purchase Order Form Sample Program

 Last reviewed: January 6, 1998
Article ID: Q160155
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOMS

Page 278 of the "Application Designer's Guide," in the section on "Typical Visual Basic Programming Scenarios," states that 

   For most of the examples in this section, a Purchase Order form is used.
No such Purchase Order form sample program comes with Microsoft Exchange Server 4.0. This is a documentation error.

STATUS

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft Exchange. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
