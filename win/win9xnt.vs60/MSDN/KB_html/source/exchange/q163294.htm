XCON: Exchange MTA Build Numbers

 Last reviewed: October 3, 1997
Article ID: Q163294
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0
  • Microsoft Exchange Server, version 5.0

SUMMARY

The build numbers for the Microsoft Exchange Server Message Transfer Agent (MTA) are as follows: 

Version 4.0 RTM    Build 837.9
Service Pack 1     Build 838.10
Service Pack 2     Build 993.6
Version 4.0a       Build 993.6
Service Pack 3     Build 994.54
Service Pack 4     Build 995.54
Version 5.0 RTM    Build 1457.8
Service Pack 1     Build 1458.56

MORE INFORMATION

You can verify the MTA build on a Microsoft Exchange Server by opening MSExchangeMTA X.400 Service events 9298 or 9299 in the Windows NT Event Viewer. These events are logged at MTA startup and shutdown respectively.
