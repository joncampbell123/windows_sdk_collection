XADM: Names and Permissions of Exchange Server Shares

 Last reviewed: April 3, 1997
Article ID: Q147376
The Information in this article applies to:

- Microsoft Exchange Server, version 4.0

SUMMARY

This article lists the names and permissions of the default Microsoft Exchange Server shares.

MORE INFORMATION

Created and shared when the Microsoft Exchange Server is installed.

Share Name: Add-ins 

Path:        \exchsrvr\ADD-INS
Comment:     Access to EDK objects
Permissions: 
  Everyone:                   Read
  Local Administrators Group: Full Control
  Exchange Service Account:   Full Control
Created and shared when the Microsoft Exchange Server is installed.

Share Name: Address 

Path:        \exchsrvr\ADDRESS
Comment:     Access to address objects
Permissions: 
  Everyone:                   Read
  Local Administrators Group: Full Control
  Exchange Service Account:   Full Control
Created and shared when the Microsoft Exchange Server is installed.

Share Name: connect$ 

Path:        \exchsrvr\CONNECT
Comment:     Access to gateway connectors
Permissions: 
  Everyone:                   Read
  Local Administrators Group: Full Control
  Exchange Service Account:   Full Control
Created and shared when a Exchange Client install point is installed.

Share Name: Exchange 

Path:        \client
Comment: Permissions: 
  Everyone:                   Read
  Local Administrators Group: Full Control
Created and shared if the MS Mail Connector is installed.

Share Name: maildat$ 

Path:        \exchsrvr\CONNECT\MSMCON\MAILDATA
Comment:     Access for MSMI postoffice
Permissions: 
  Everyone:                   Full Control
  Local Administrators Group: Full Control
  Exchange Service Account:   Full Control
Created and shared when the Microsoft Exchange Server is installed.

Share Name: Resources 

Path:        \exchsrvr\res
Comment:     Event logging files
Permissions: 
  Everyone:                   Read
  Local Administrators Group: Full Control
  Exchange Service Account:   Full Control
Created and shared if the Sample Applications are installed.

Share Name: Samples 

Path:        \exchsrvr\SAMPAPPS\CLIENT
Comment:     Exchange public folder sample apps
Permissions: 
  Everyone:                   Full Control
  Local Administrators Group: Full Control
  Exchange Service Account:   Full Control
Created and shared when the Microsoft Exchange Server is installed.

Share Name: tracking.log 

Path:        \exchsrvr\tracking.log
Comment:     Exchange message tracking logs
Permissions: 
  Everyone:                   Read
  Local Administrators Group: Full Control
  Exchange Service Account:   Full Control
