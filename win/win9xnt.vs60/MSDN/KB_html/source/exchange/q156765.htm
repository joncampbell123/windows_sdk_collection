XADM: Multiple Private IS Logons for Each User

 Last reviewed: February 5, 1998
Article ID: Q156765

The information in this article applies to:

  • Microsoft Exchange Server, version 4.0

In the Microsoft Exchange Server Administrator program, when you choose SERVERS\<servername>\Private Information Store\Logons, you will see multiple entries for people connected to the servers.

This is by design. The client logs on once for the private store and once for the public store; the spooler logs on once for the private store.

Version           : 4.0
