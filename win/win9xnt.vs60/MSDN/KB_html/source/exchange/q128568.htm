XADM : Managing Distribution Lists from Windows Clients

 Last reviewed: April 3, 1997
Article ID: Q128568
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

A Microsoft Exchange administrator can configure distribution lists so selected users can manage list membership using one of the Microsoft Exchange Windows clients. Before a user can add and remove addresses from a list, he or she must first have the appropriate permissions to that list.

Granting Permissions to a List Manager

To grant the permissions needed by a user who will perform distribution list management, do the following:

  1. Start the Microsoft Exchange Administrator program.

  2. Select the distribution list name in the Recipients container.

  3. Click the Modify button in the Owner section and select a user from the
list box to be an owner of this distribution list.

Managing a Distribution List from a Windows Client

To manage a distribution list (DL) from a Microsoft Exchange Windows client, do the following:

  1. From the Tools menu, choose Address Book.

  2. From the Global Address List or the Recipients container for the site containing the DL, select the DL to be modified.

  3. Bring up the properties for the DL, by double-clicking the selected item or by choosing Properties from the File menu.

  4. Click the Modify Members button to bring up the members of the DL.

  5. If you have the correct permissions on the DL, you should be able to modify the DL.
