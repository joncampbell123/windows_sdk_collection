XCON: Exchange Content Type Encoding

 Last reviewed: October 17, 1997
Article ID: Q162430
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

The following is a list of Exchange Server content types and their corresponding object identifier encoding. These object identifiers are used in the Microsoft Exchange Server Administrator program as well as the Windows NT Event Logs and can be used to determine what type of content conversion was being attempted. 

   Content Type      Object Identifier Encoding
   --------------------------------------------
   undefined         0x56030300
   P2-84             0x56010A00
   P2-88             0x56010A01
   EDI               0x56070A00
   Voice             0x56080800
   MDBEF             0x2A864886F7140501
   TNEF              0x2A864886F7140502
