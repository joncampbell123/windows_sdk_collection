XCON: Cannot Create RAS Connector With More Than 10 entries

 Last reviewed: January 5, 1998
Article ID: Q152591
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOMS

When you attempt install a RAS Connector with more than 10 entries in the RAS Phone book, you will receive the following error: 

   The System Attendant on server <server name> could not read the Remote
   Access Service(RAS) Phone Book entries.
   The error '317' occurred while formatting Windows NT error '603'.

   Microsoft Windows NT
   ID NO: 0xc002025b

STATUS

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft Exchange. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
