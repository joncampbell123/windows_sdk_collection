XCLN: Autoforwarded E-form Does Not Prefix Subject Line With FW:

 Last reviewed: February 4, 1998
Article ID: Q180060
The information in this article applies to:
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Outlook Windows NT client, versions 8.00, 8.01, 8.02,
8.03
  • Microsoft Outlook Windows 95 client, versions 8.00, 8.01, 8.02,
8.03

SYMPTOMS

When you AutoForward an Electronic Form using an Inbox Assistant rule, the subject line does not begin with “FW:”

WORKAROUND

To work around this problem, use Outlook Forms Designer. When you AutoForward forms using Outlook Forms Designer, the subject line is prefixed with “FW:”

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
