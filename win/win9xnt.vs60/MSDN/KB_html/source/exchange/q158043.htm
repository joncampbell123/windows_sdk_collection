XCLN: Err Msg: There was a Problem Deleting the Form

 Last reviewed: July 18, 1997
Article ID: Q158043

The information in this article applies to:
  • Microsoft Exchange Windows 3.x client, version 4.0
  • Microsoft Exchange Windows 95 client, version 4.0
  • Microsoft Exchange Windows NT client, version 4.0

SUMMARY

If you have a lot of forms installed to your forms cache (they don't need to be running, just installed) and you try to delete a form that you do not have permissions to, you may receive the following error: 

   There was a problem deleting the form. You need more memory or system
   resources. Close some windows and try again.
instead of an error stating that you do not have privileges to perform the operation.

STATUS

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft Exchange. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
