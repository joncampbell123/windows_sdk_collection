XCLN: Offline Address Book Does Not Contain All GAL Addresses

 Last reviewed: September 23, 1997
Article ID: Q151204
The information in this article applies to:

- Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0 - Microsoft Exchange Windows NT client, versions 4.0 and 5.0 - Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

SUMMARY

By default, the Microsoft Exchange Offline Address Book contains only the addresses in the default recipients container. The Offline Address Book will not contain all addresses in the Global Address List (GAL) unless it is configured to do so.

MORE INFORMATION

To generate an Offline Address Book that contains all addresses in the GAL:

  1. Start the Microsoft Exchange Administrator program.
2. Double-click the DS Site Configuration in the Configuration container. 3. Click on the Offline Address Book tab. 4. Click on the Modify button. 5. Click on the Global Address List. 6. Click OK 7. Click on the Regenerate Offline Address Book Now button.

The users will now need to download the new Offline Address Book.
