XCLN: Options Not Supported by WordMail

 Last reviewed: September 23, 1997
Article ID: Q146969
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

The following options that are available in the default editor of Microsoft Exchange version 4.0 clients are not supported by the Office 95 WordMail editor:

  • Include original text when replying (Read)
  • Indent original text when replying (Read)
  • Set Replying Font (Read)
  • Set Sending Font (Send)
  • Show ToolTips (General)
  • Autoselect entire words (General)
  • All Spelling options (Spelling)

In addition to the above options, the following are also not supported:
  • Use of the FROM line in the header
  • Use of the BCC line in the header
  • Use of the Autosignature option

For additional information regarding WordMail, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q139285
   TITLE     : Specifying Microsoft Word as E-Mail Editor
This behavior is by product design.
