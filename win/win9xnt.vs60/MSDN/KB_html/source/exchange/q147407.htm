XCLN: Cannot Get Details from One Address for Distribution List

 Last reviewed: March 19, 1997
Article ID: Q147407
4.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Exchange Windows 95 client, version 4.0
  • Microsoft Exchange Windows 3.x client, version 4.0
  • Microsoft Exchange Windows NT client, version 4.0
  • Microsoft Exchange MS-DOS client, version 4.0

SYMPTOMS

When sending a mail message to the One-Off X.400 address of a distribution list (DL), you are not able to view details and obtain the membership list.

If you send a message to the One-Off X.400 address of a DL, you cannot double-click and view the details. The receiver sees what the sender typed in.

CAUSE

This is by design.
