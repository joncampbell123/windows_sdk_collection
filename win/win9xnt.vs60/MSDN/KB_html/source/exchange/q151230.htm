XCLN: OLE App Required to Send Attachments Using OLE

 Last reviewed: January 6, 1998
Article ID: Q151230
The information in this article applies to:
  • Microsoft Exchange Windows 3.x client, version 4.0
  • Microsoft Exchange Windows NT client, version 4.0
  • Microsoft Exchange Windows 95 client, version 4.0

SUMMARY

The following error might appear when you attempt to attach a file to a message and you are using the ReadFromFile method for Microsoft Object Linking and Embedding (OLE) Messaging. 

   Runtime error in OLE. Error OLE Messaging [8004010F]

MORE INFORMATION

The ReadFromFile method requires that the OLE application, such as Microsoft Word or Excel, be installed on the computer when you send an attachment of that document type using the OLE Messaging ReadFromFile method.
