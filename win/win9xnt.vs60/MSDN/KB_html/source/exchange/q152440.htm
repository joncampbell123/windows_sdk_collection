XADM: Import Error Message: One Warning Event Was Encountered

 Last reviewed: April 3, 1997
Article ID: Q152440
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

When you perform a Microsoft Exchange directory import, you might receive the following error: 

   The directory import from file C:\exchsrvr\bin\<filename> is complete.
   One warning event was encountered but not logged.
   No errors were encountered.

MORE INFORMATION

If you encounter errors during the directory import, a file will be created with the same name as your import file with an extension of .err. If subsequent imports report errors using the same file, the extension will be e01, e02, and so forth. Import will record the lines with which it had problems in this text file. It can be edited and then used as your import file.
