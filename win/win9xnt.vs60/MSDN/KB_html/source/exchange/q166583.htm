XADM: Folders Are Sorted When Non-English Settings Are Used

 Last reviewed: August 28, 1997
Article ID: Q166583

The information in this article applies to:

  • Microsoft Exchange Server, version 5.0

SYMPTOMS

When Microsoft Outlook users choose regional settings other than English, they may find that their folders are sorted on the last item modification date.

CAUSE

When a non-English regional setting is used, indexing within a specific Jet database table is handled differently than a case where the English regional setting is used.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 5.0. This problem was corrected in the latest Microsoft Exchange Server 5.0 U.S. Service Pack.
