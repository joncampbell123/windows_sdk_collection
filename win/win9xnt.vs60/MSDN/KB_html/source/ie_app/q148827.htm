Contents of the Msie20.exe File

 Last reviewed: January 16, 1997
Article ID: Q148827
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

Microsoft Internet Explorer is available for download as a self-extracting file named Msie20.exe. The extracted files are temporarily placed in the Windows\Temp folder and are deleted after installation. This article lists the contents of the Msie20.exe file and the two cabinet files it contains.

For information about obtaining the Msie20.exe file, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140649
   TITLE     : Microsoft Internet Explorer 2.0 for Windows 95 Available
