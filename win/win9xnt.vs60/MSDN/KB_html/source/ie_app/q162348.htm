Blind Carbon Copy (Bcc) Not Available In Internet Mail

 Last reviewed: September 5, 1997
Article ID: Q162348
1.00 WINDOWS kbui

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.0 for Windows 95

SUMMARY

The Blind Carbon Copy (Bcc) feature is not included with Internet Mail 1.0.

MORE INFORMATION

The Blind Carbon Copy feature allows you to send a copy of a message to someone without any of the other message recipients knowing you are sending a copy to that person.

To use the Blind Carbon Copy feature, use Microsoft Exchange (Windows Messaging) with the Internet Mail information service installed. For additional information about how to setup Microsoft Exchange with the Internet Mail information service, please see the following article in the Microsoft Knowledge Base:
