"Internet Extras!" and "About This CD" Do Not Open

 Last reviewed: August 29, 1997
Article ID: Q149602
2.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Internet Starter Kit version 2.0
  • Microsoft Internet Explorer version 2.0 for Windows 95

SYMPTOMS

When you click Internet Extras! or About This CD inside the Microsoft Internet Starter Kit main window, the page may not open.

CAUSE

This behavior may occur if you have multiple instances of the Microsoft Internet Explorer 2.0 demonstration running.

RESOLUTION

To view the page, close the multiple instances of the Internet Explorer demonstration, and then click Internet Extras! or About This CD again.

MORE INFORMATION

You can open a separate window in Internet Explorer by clicking New Window on the File menu, or by pressing and holding down the SHIFT key when you click a Uniform Resource Locator (URL).
