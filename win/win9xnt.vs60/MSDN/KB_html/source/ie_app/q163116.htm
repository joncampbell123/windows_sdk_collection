Contents Remain in Preview Pane After Sending a Message

 Last reviewed: August 19, 1997
Article ID: Q163116
1.00 1.10 MACINTOSH kbui

The information in this article applies to:

  • Microsoft Internet Mail and News versions 1.0, 1.1 for Macintosh

SYMPTOMS

After you send a message previously marked as Finish Later in Internet Mail and News, the contents of the message may remain in the Preview pane.

STATUS

Microsoft has confirmed this to be a problem in Internet Mail and News versions 1.0 and 1.1 for Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Although the contents are still visible in the Preview pane, the message does successfully leave the Outbox. This behavior does not affect the functionality of the product.

Finish Later is a feature for placing a message in the Outbox, but not sending it until it is opened and the Send/Post Message command is used.
