Formatting Carriage Return to Change Format of Bullet

 Last reviewed: September 2, 1997
Article ID: Q153913
The information in this article applies to:
  • Microsoft Pocket Word version 1.0
  • Microsoft Windows 95
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SUMMARY

You can change the formatting (color, underline, and so on) of the bullet character on a bulleted line in WordPad or Microsoft Exchange in Windows 95 or Pocket Word in Windows CE by changing the formatting of the carriage return (<CR>) at the end of that line.

This also applies to Microsoft Internet Mail and News version 1.0 when you format a message using HTML.

NOTE: The <CR> is not a visible character.

MORE INFORMATION

Use the following steps to change the formatting of the bullet character on a bulleted line:

  1. Select the last space on the line (beyond the last visible character). This space is the <CR>.

  2. Apply the formatting you want for the bullet. Note that the bullet changes as you apply formats.

If you want to change the formatting of the text on a bulleted line with- out changing the formatting of the bullet, do not select the <CR> at the end of the line when you select the other characters.
