Availability of Microsoft Internet Mail for Windows 3.1

 Last reviewed: September 10, 1997
Article ID: Q159680
The information in this article applies to:
  • Microsoft Internet Mail version 1.0 for Windows 3.1

SUMMARY

Microsoft has made Microsoft Internet Mail for Windows 3.1 available.

MORE INFORMATION

Microsoft Internet Mail is an Simple Mail Transfer Protocol (SMTP)/Post Office Protocol 3 (POP3) mail client that enables you to work offline, sort and store mail any way you want, and view messages quickly in the preview pane.

You can obtain Internet Mail for Windows 3.1 from the following Microsoft Web site: 

   http://www.microsoft.com/ie/
