Favorites Bar Not Displayed Properly in Internet Explorer

 Last reviewed: September 10, 1997
Article ID: Q166870
The information in this article applies to:
  • Microsoft Internet Explorer version 3.0 for Macintosh

SYMPTOMS

The Favorites bar may not be displayed, or may be displayed improperly on the toolbar.

CAUSE

This problem can be caused by any of the following situations:

  • The Favorites bar has been disabled.
  • The System Folder:Preferences:Explorer:Favorites.html file has been renamed, modified, moved, deleted, or damaged.
  • The Toolbar Favorites folder has been renamed, modified, moved, deleted, or damaged.

RESOLUTION

Favorites Bar Is Missing

If the Favorites bar is missing from the toolbar, it may have been disabled, or the System Folder:Preferences:Explorer:Favorites.html file may have been moved, renamed, deleted, or damaged.

To add the Favorites bar to the toolbar, use either of the following methods:

  • If the Favorites bar has been disabled, click Preferences on the Edit menu, click the Show Favorites Bar check box to select it, and then click OK.
  • If the Favorites.html file has been moved, you can search for it by clicking Find File on the Apple menu, typing "Favorites.html" (without the quotation marks), and then clicking Find. If the Favorites.html file is on your hard disk, move it to the System Folder:Preferences:Explorer folder, then quit and restart Internet Explorer.
  • If the Favorites.html file has been renamed, rename the file back to Favorites.html.
  • If the Favorites.html file has been deleted or damaged, reinstall Internet Explorer.

Toolbar Favorites Links Are Missing

If Toolbar Favorites links are missing from the Favorites bar, the Toolbar Favorites folder may have been renamed, moved, deleted, or damaged. To rebuild the Toolbar Favorites folder, click Open Favorites on the Favorites menu, and then use either of the following methods:

  • If the Toolbar Favorites folder has been renamed, rename the folder back to Toolbar Favorites, and then click Close on the File menu.
  • If the Toolbar Favorites folder is not in the Favorites window, click New Folder on the Favorites menu, name the new folder Toolbar Favorites, and then click Close on the File menu.
  • If the Toolbar Favorites folder already exists in the Favorites window, it may be damaged. To create a new Toolbar Favorites folder, drag the existing folder to the trash can, click New Folder on the Favorites menu, name the new folder Toolbar Favorites, and then click Close on the File menu.

MORE INFORMATION

The links on the Favorites bar are contained in the Toolbar Favorites folder in the System Folder:Preferences:Explorer:Favorites.html file. Internet Explorer does not create a Toolbar Favorites folder on the hard disk.
