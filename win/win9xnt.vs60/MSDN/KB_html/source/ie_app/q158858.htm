Cannot Rename Users and Groups in Personal Web Server

 Last reviewed: August 27, 1997
Article ID: Q158858
1.00 1.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Personal Web Server versions 1.0, 1.0a for Windows 95

SYMPTOMS

You cannot rename users and groups, or you may be unable to create groups, using Microsoft Personal Web Server version 1.0 for Windows 95.

CAUSE

Personal Web Server does not support the renaming of users and groups, or the nesting of groups (the creation of one group inside of another).

RESOLUTION

To resolve this issue, create a new user or group account and then delete the old one.
