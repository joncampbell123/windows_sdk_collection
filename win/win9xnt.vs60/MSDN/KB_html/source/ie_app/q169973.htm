How to Uninstall Internet Explorer version 3.01a for Macintosh

 Last reviewed: September 10, 1997
Article ID: Q169973
The information in this article applies to:
  • Microsoft Internet Explorer version 3.01a for Macintosh

SUMMARY

This article describes how to uninstall (remove) Internet Explorer 3.01a for Macintosh.

MORE INFORMATION

Internet Explorer 3.01a does not have an uninstall feature. To uninstall Internet Explorer, you must manually remove the files and folders created by Setup. For a complete list of files and folders created by Internet Explorer Setup, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q168516
   TITLE     : Files Installed by Internet Explorer Version 3.01a for
               Macintosh
For more information about deleting files on a Macintosh computer, see your Macintosh operating system documentation.
