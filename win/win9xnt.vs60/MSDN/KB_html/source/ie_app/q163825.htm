Making Changes to Properties May Not Enable Apply Button

 Last reviewed: September 4, 1997
Article ID: Q163825
3.00 3.01 WINDOWS kbui

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02 for Windows NT 4.0

SYMPTOMS

When you click Options on the View menu in Internet Explorer and then make any change on the Connection, Navigation, or Program tab, the Apply button may not be enabled.

RESOLUTION

After making changes to the settings on the Connection, Navigation, or Program tab, click OK to apply the changes.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Apply button and the OK button function alike in that they both save any changes that you have made. However, clicking the Apply keeps the dialog box open after you have made changes, whereas clicking OK closes it.
