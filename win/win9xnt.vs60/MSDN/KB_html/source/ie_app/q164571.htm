Microsoft Personal Web Server 1.0 for Macintosh Available

 Last reviewed: March 5, 1997
Article ID: Q164571
The information in this article applies to:
  • Microsoft Personal Web Server version 1.0 for Macintosh

SUMMARY

Microsoft has released Microsoft Personal Web Server version 1.0 for Macintosh.

MORE INFORMATION

Microsoft Personal Web Server is a full-featured Web server that makes it easy to publish documents on the Internet or an intranet. The Microsoft Personal Web Server control panel that is included with Personal Web Server makes it easy to set up and monitor the server. It also lets you customize a home page with text, graphics, and links without using HTML.

For additional information about Microsoft Personal Web Server version 1.0 for Macintosh, visit the following Microsoft Web site: 

   http://www.microsoft.com/ie/mac/features/pws.htm
