Internet Explorer Does Not Start with Extensions Off

 Last reviewed: September 4, 1997
Article ID: Q165285
The information in this article applies to:
  • Microsoft Internet Explorer version 3.01 for Macintosh

SYMPTOMS

When you start Internet Explorer 3.01 for Macintosh, you may receive one of the following error messages:

  • The application "Internet Explorer 3.01" can not be opened because an unexpected error occurred
  • This program can not be run on this computer 
       Characteristic       Your Machine      Minimum Required
   Internet Config      Not Present       Required
When you click OK or Quit, the error message disappears and Internet Explorer 3.01 does not start.

CAUSE

This behavior can occur if you start your computer without loading the Internet Config extension, or without loading any extensions.

RESOLUTION

To resolve this issue, use the appropriate method:

  • If you started your computer without loading any extensions, restart your computer and let the extensions load.
  • If you started your computer without loading the Internet Config extension, restart your computer and let this extension load.
