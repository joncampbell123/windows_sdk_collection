The information in this article applies to:

  • Microsoft Internet Explorer version 2.1 for Windows 3.1

SUMMARY

Microsoft Internet Explorer version 2.1 for Windows 3.1 is available as a self-extracting file named Dlmini21.exe. The Dlmini21.exe file contains only the Internet Explorer 2.1 for Windows 3.1 Internet browser.

This article lists the contents of the Dlmini21.exe file and the cabinet file it contains.

For information about obtaining the Dlmini21.exe file, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q148875
   TITLE     : Microsoft Internet Explorer 2.1 for Windows 3.1 Available
