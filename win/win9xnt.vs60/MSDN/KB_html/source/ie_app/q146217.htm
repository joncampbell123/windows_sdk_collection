International Extensions for Internet Explorer 2.0 Available

 Last reviewed: September 2, 1997
Article ID: Q146217
95 WINDOWS kbfile kbtool

The information in this article applies to:

  • Microsoft Internet Explorer version 2.0 for Windows 95

SUMMARY

Microsoft has released International Extensions for Microsoft Internet Explorer 2.0 in Internet Explorer 2.0i. International Extensions for Internet Explorer 2.0 lets you view HyperText Markup Language (HTML) documents that were created in different languages in any language version of Windows 95. The currently supported language groups are Cyrillic, Eastern European, English, Greek, Japanese, Korean, and Western European.

To use International Extensions for Internet Explorer 2.0, you need any language version of Windows 95 and any language version of Internet Explorer 2.0. Please note that after you install International Extensions for Internet Explorer 2.0, you will have an English language user interface for Internet Explorer.

International Extensions for Internet Explorer 2.0 does not run with beta versions of Internet Explorer 2.0. For information about the availability of Internet Explorer 2.0, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140649
   TITLE     : Microsoft Internet Explorer 2.0 for Windows 95 Available

MORE INFORMATION

Installation

To install International Extensions for Internet Explorer 2.0, follow these steps:

  1. Download the MSIE20I.EXE file from one of the online services listed below to an empty folder.

  2. In My Computer or Windows Explorer, double-click the MSIE20.EXE file you downloaded in step 1.

  3. Follow the instructions on your screen.

If you want to add Japanese or Korean support to International Extensions for Internet Explorer 2.0, follow these steps:

  1. Install International Extensions for Internet Explorer 2.0.

    NOTE: If you have already installed International Extensions for Internet Explorer 2.0, skip this step.

  2. On the View menu in Internet Explorer, click Options, and then click the Languages tab.

  3. In the Scripts section, click Add.

    NOTE: This takes you to a Microsoft Web page where you can download Japanese and Korean support.

  4. Follow the instructions to install Japanese or Korean support.

NOTE: If you use Traditional Chinese Windows 95, do not install Japanese or Korean support. For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q147440
   TITLE     : Troubleshooting Internet Explorer 2.0i

User Interface Differences Between Internet Explorer 2.0 and 2.0i

International Extensions for Internet Explorer 2.0 offers the following two visual differences from Internet Explorer 2.0:

  • International Extensions for Internet Explorer 2.0 adds a character set or script box next to the Address box. You can use this setting to change the script or encoding of the current Web page.
  • International Extensions for Internet Explorer 2.0 adds a Languages tab to the Options screen in Internet Explorer. To see this change, click Options on the View menu.
