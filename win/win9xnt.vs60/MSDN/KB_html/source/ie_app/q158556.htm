The information in this article applies to:

  • Microsoft Internet Explorer version 2.1 for Windows 3.1

SUMMARY

Microsoft Internet Explorer version 2.1 for Windows 3.1 is available as a self-extracting file named Dlfull21.exe. The Dlfull21.exe file contains the Internet Explorer 2.1 for Windows 3.1 Internet browser, an Internet mail program, a Transmission Control Protocol/Internet Protocol (TCP/IP) stack, and a dialer tool.

This article lists the contents of the Dlfull21.exe file and the cabinet files it contains.

For information about obtaining the Dlfull21.exe file, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q148875
   TITLE     : Microsoft Internet Explorer 2.1 for Windows 3.1 Available
