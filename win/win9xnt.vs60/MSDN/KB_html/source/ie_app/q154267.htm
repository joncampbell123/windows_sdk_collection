Select All in Internet News Selects Only Visible Messages

 Last reviewed: September 2, 1997
Article ID: Q154267
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SUMMARY

When you use the Select All command in Internet News, only the visible messages are selected. Threaded messages that are not expanded are not selected.

MORE INFORMATION

The Select All command differentiates between what is hidden and what is visible. If an item is in the list, it is selected. If a message thread is collapsed, the responses to that message are not in the list, and are not selected.
