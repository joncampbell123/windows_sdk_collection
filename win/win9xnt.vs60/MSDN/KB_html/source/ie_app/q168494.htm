Uninstalling Web Publishing Wizard Does Not Delete Folder

 Last reviewed: August 27, 1997
Article ID: Q168494
The information in this article applies to:
  • Microsoft Web Publishing Wizard versions 1.1, 1.5 for Windows 95
  • Microsoft Web Publishing Wizard versions 1.1, 1.5 for Windows NT 4.0

SYMPTOMS

When you uninstall the Microsoft Web Publishing Wizard with the Add/Remove Programs tool in Control Panel, the Program Files\Web Publish folder is not removed.

CAUSE

The Microsoft Web Publishing Wizard version 1.1 .inf file does not remove folders.

RESOLUTION

Remove the "Web Publish" folder manually using the following steps:

  1. Click Start, point to Programs, and then click Windows Explorer.

  2. Double-click the Program Files folder.

  3. Click the Web Publish folder.

  4. On the File menu, click Delete.

  5. Close Windows Explorer.
