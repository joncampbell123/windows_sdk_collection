Error Message: The Internet Connection Wizard Is Unable...

 Last reviewed: September 9, 1997
Article ID: Q162802
3.00 MACINTOSH kberrmsg kbtool

The information in this article applies to:

  • Microsoft Internet Explorer version 3.0 for Macintosh

SYMPTOMS

When you attempt to run the Internet Connection Wizard (ICW) for Internet Explorer 3.0 for Macintosh, the following error message may be displayed: 

   The Internet Connection Wizard is unable to run because the installed
   revision of Open transport (0.0) is not supported. Install Open
   Transport revision 1.1 to 1.1.1 and try again.

CAUSE

This problem occurs when you are using a version of the Open Transport Transmission Control Protocol/Internet Protocol (TCP/IP) earlier than version 1.1 or later than version 1.1.1. The Internet Connection Wizard only supports Open Transport versions 1.1 to 1.1.1.

RESOLUTION

To work around this problem, install a supported version of Open Transport. If Open Transport is not included with your Macintosh System software, contact Apple for information about obtaining Open Transport.
