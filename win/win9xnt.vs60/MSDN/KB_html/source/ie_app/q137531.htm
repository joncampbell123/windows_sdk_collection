Internet Explorer 1.0 Does Not Display Wingdings Characters

 Last reviewed: August 29, 1997
Article ID: Q137531
The information in this article applies to:
  • Microsoft Plus! for Windows 95
  • Microsoft Windows 95

SYMPTOMS

Internet Explorer 1.0 does not display the Wingdings characters specified in the <FONT FACE=Wingdings> Hypertext Markup Language (HTML) extension of a Web page. Wingdings characters are replaced with corresponding characters from the default font.

CAUSE

When Internet Explorer 1.0 checks for valid font information, it uses the American National Standards Institute (ANSI) character set. However, the Wingdings font is not in this character set.

STATUS

Microsoft has confirmed this to be a problem in Internet Explorer version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
