Internet Mail and News Opens Many Untitled Message Windows

 Last reviewed: March 16, 1998
Article ID: Q173016
The information in this article applies to:
  • Microsoft Internet Mail and News version 3.0a for Macintosh
  • Microsoft Internet Explorer version 3.01a for Macintosh

SYMPTOMS

If you are using Apple Macintosh System Software version 8.0 (Mac OS 8) and you click Mail on the Internet Explorer toolbar or click a "mailto:" link (a link that sends a mail message) on a Web page, untitled mail message windows may be opened until all available memory is used.

CAUSE

Macintosh System Software version 8.0 changes the way Apple events are handled.

RESOLUTION

To stop the creation of untitled messages, quit Internet Mail and News by pressing Command+Q.

To resolve this issue, obtain the latest version of Internet Mail and News (version 3.0c) for Macintosh from the following Microsoft Web site: 

   http://www.microsoft.com/ie/download
Download the Imn30c.hqx file.

STATUS

This problem no longer occurs in Internet Mail and News version 3.0c for Macintosh.
