Err Msg: This Program Does Not Support the Protocol...

 Last reviewed: May 19, 1997
Article ID: Q166205
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02a for Windows 3.1

SYMPTOMS

When you click the Home, Search or any of the link buttons, you may receive the following error message: 

   This program does not support the protocol for accessing
   '<web site address>'
When this occurs, the corresponding Web page is not loaded.

CAUSE

There is not a leading "http://" in the page's address.

RESOLUTION

To resolve this issue, follow these steps:

  1. On the View menu, click Options.

  2. Click the Navigation tab.

  3. In the Customize section, click the link that is not working in the Page box.

  4. Add a leading "http://" (without quotation marks) to the page's address.

  5. Click OK.
