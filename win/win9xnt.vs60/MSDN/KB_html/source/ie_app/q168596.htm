Error Message: Access to Specified Device, Path, or File Denied

 Last reviewed: August 28, 1997
Article ID: Q168596
The information in this article applies to:
  • Microsoft Web Publishing Wizard version 1.5 for Windows 95
  • Microsoft Web Publishing Wizard version 1.5 for Windows NT 4.0

SYMPTOMS

When you select multiple files, click the files with the right mouse button, point to Send To, and then click Web Publishing Wizard, you may receive the following error message: 

   Access to the specified device, path, or file is denied.

RESOLUTION

This behavior may occur when you attempt to publish from 10-13 files. To work around this behavior, publish fewer files at a time.

MORE INFORMATION

This behavior occurs with the following programs in Microsoft Windows 95:

  • Microsoft Exchange
  • Microsoft Outlook 97
