How to Clear History Items or Prevent History Creation

 Last reviewed: September 11, 1997
Article ID: Q173698
The information in this article applies to:
  • Microsoft Internet Explorer version 3.01a for Macintosh

SUMMARY

This article describes two ways to manage history information:

  • How to clear history items
  • How to prevent history creation

MORE INFORMATION

How to Clear History Items

To clear history items, follow these steps:

  1. On the Go menu, click Open History.

  2. On the Edit menu, click Select All.

  3. Drag the item(s) to the Trash.

  4. Close the History window.

Note that history is stored in the following file, which you can move or delete: 

   System Folder:Preferences:Explorer:History.html

How to Prevent History Creation

To prevent history creation, follow these steps:

  1. On the Edit menu, click Preferences.

  2. Under Web Browser, click Advanced.

  3. In the "Remember the last <nnn> places visited" box, type 0 (zero). The default value is 500. Setting the value to 0 clears the existing history and prevents additional history from being created.

  4. Click OK.
