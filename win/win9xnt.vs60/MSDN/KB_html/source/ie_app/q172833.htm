Stationery Is Attached to Plain Text Message

 Last reviewed: September 10, 1997
Article ID: Q172833
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you compose a new message using stationery and then change the format of the message to Plain Text, the stationery's background image is attached as a graphics file when you send the message.

CAUSE

Outlook Express does not remove graphics files displayed in HTML format when you switch to Plain Text format. Instead, graphics files are sent as attachments to the Plain Text-format message.

RESOLUTION

To work around this issue, use either of the following methods:

  • Compose a new message without using stationery by clicking New Message on the Compose menu, and then clicking Plain Text on the Format menu in the New Message window.
  • Set Plain Text as the default sending format by clicking Options on the Tools menu, clicking the Send tab, clicking Plain Text under Mail Sending Format, and then clicking OK.
