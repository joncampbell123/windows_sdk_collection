Error Message: Some of the Messages Could Not Be Moved...

 Last reviewed: September 4, 1997
Article ID: Q154282
1.00 WINDOWS kbenv

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

When you try to move a mail message to another folder by clicking the message, pointing to Move To on the Mail menu, and then clicking the folder you want to move the message to, you may receive the following error message: 

   Some of the messages could not be moved. An error has occurred.

CAUSE

This error message can occur if the folder you try to move the message to is configured as read-only.

RESOLUTION

Remove the read-only attribute from the appropriate file. To do so, follow these steps:

  1. Close Internet Mail by clicking Close on the File menu.

  2. Click Start, point to Find, and then click Files Or Folders.

  3. In the Named box, type the file name of the mail folder you tried to move the message to, and then click Find Now. For example, if you tried to move the message to the Test mail folder, type "Test.mbx" (without quotation marks).

    NOTE: If your computer is configured for User Profiles, be sure to use 

       the correct Program Files\Internet Mail and News\   
   <Username>\Mail\<Folder.mbx> file, where <Username> is the name
   you use to log into Windows 95, and <Folder.mbx> is the file name of
   the mail folder you tried to move the message to.
4. Right-click the appropriate .mbx file, and then click Properties on the 
   menu that appears.

  • Click the Read-Only check box to clear it, and then click OK.

  • Start Internet Mail, and then try to move the message to the folder again.
