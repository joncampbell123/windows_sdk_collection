Default Settings for Internet Mail Plain Text

 Last reviewed: September 2, 1997
Article ID: Q153982
1.00 WINDOWS kbref kbtool

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.0 for Windows 95

SUMMARY

Microsoft Internet Mail contains a dialog box that you can use to select settings for sending messages. This article outlines the default settings for the Plain Text mail sending format.

MORE INFORMATION

To access the Plain Text Settings, use the following steps:

  1. Click Start, point to Programs, and then click Internet Mail.

  2. On the Mail menu, click Options.

  3. Click the Send tab.

  4. Under Mail Sending Format, click Plain Text, and then click Settings.

The following settings are the default Plain Text Settings:
  • Message Format: MIME (Multipurpose Internet Mail Extensions)
  • Encode Text Using: None
  • Allow 8-Bit Characters In Headers check box is not selected.
  • Automatically wrap text at 76 characters, when sending.
