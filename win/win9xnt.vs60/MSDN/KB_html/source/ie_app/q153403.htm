MS Empowerment Pack for Mac Err Msg: This Program Does Not...

 Last reviewed: September 2, 1997
Article ID: Q153403
The information in this article applies to:
  • Microsoft Empowerment Pack for the Mac version 1.0

SYMPTOMS

When you attempt to install any of the programs or updates included on the Microsoft Empowerment Pack for the Mac CD, you may receive the following error message: 

   This program does not support the protocol for accessing
   launch:///Microsoft Empowerment Pack/<path to application>
Note that this error message occurs only when you attempt to install files that use an installation program. The error message does not occur with Internet Assistant for Microsoft Excel. Internet Assistant for Microsoft Excel is a template file that has been compressed into a self-extracting archive and does not require an installation program.

CAUSE

The error message occurs when you try to install the files on the Empowerment Pack CD by using the released version of Microsoft Internet Explorer for Macintosh that is currently available from online services. The version of Internet Explorer 2.0 in the root folder of the Empowerment Pack CD-ROM is a modified version of Internet Explorer that has been rewritten to allow it to start the installation programs on the CD.

RESOLUTION

To resolve this issue, use either of the following methods:

  • Double-click the Empowerment Pack CD icon on the desktop, and then double-click the file named "Microsoft Empowerment Pack." This starts the correct version of Internet Explorer and allows the installation to proceed normally.
  • Quit the current version of Internet Explorer, open the folder of the component you want to install, and then double-click the Setup/Installer icon to start the installation.
