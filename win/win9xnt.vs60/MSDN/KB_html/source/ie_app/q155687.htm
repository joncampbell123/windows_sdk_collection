Multiple Selections in Boxes Return Incorrect Information

 Last reviewed: September 5, 1997
Article ID: Q155687
The information in this article applies to:
  • Microsoft Internet Explorer versions 2.0, 2.01, 2.1 for Windows 3.1

SYMPTOMS

When you click more than one item in a list box on a Web page, you may receive incorrect information.

CAUSE

Multiple-select list box functionality is not implemented in Internet Explorer versions 2.0, 2.01, and 2.1 for Windows 3.1.

RESOLUTION

To resolve this issue, click only one item in the list box.
