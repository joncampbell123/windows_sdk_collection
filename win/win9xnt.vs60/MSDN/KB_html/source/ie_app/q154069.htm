Save Attachments Command Not Available in Internet Mail

 Last reviewed: September 3, 1997
Article ID: Q154069
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

When you are viewing a message with an attachment in the preview window, the Save Attachments command on the Internet Mail File menu is not available.

CAUSE

Attachments to a mail message can be saved only when the message is opened, not when it is being previewed.

RESOLUTION

To save attachments in a mail message, use one of the following methods:

  • Double-click the mail message and then click Save Attachments on the File menu.
  • Press down and hold the CTRL key while you click the paper clip icon in the top right corner of the preview pane.
