Known Issues for Microsoft Web Publishing Wizard

 Last reviewed: August 27, 1997
Article ID: Q162217
1.00 1.00a 1.10 WINDOWS kbprb

The information in this article applies to:

  • Microsoft Web Publishing Wizard versions 1.0, 1.0a, 1.1 for Windows 95
  • Microsoft Web Publishing Wizard versions 1.0, 1.0a, 1.1 for Windows NT 4.0

SUMMARY

This article summarizes the currently known issues for Microsoft Web Publishing Wizard version 1.0.

MORE INFORMATION

  • Posting directory trees is not supported on America Online and GNN Web sites.
  • Web Publishing Wizard version 1.0 for Windows NT 4.0 is not yet supported on non-Intel platforms such as PowerPC, MIPS, or Alpha.
  • AOL and GNN do not support the Windows 95 or Windows NT Dial-Up Networking feature. Access to AOL or GNN Web sites is accomplished using the AOL or GNN software dialer. Before you use Web Publishing Wizard, make sure you have installed the AOL software version 2.5 or later or the GNN software version 1.30 or later.
  • The MSN, The Microsoft Network, Sign-In dialog box may appear unexpectedly when you run Web Publishing Wizard. If this occurs, click Cancel to continue posting.
  • The Program Files\Web Publish folder is not removed after you uninstall Web Publishing Wizard.
  • The Windows\Wplog.txt file is not removed after you uninstall Web Publishing Wizard.
  • Several registry keys are not removed after you uninstall Web Publishing Wizard.

For Corporate Users

  • Web Publishing Wizard does not support posting through a proxy server to AOL, GNN, CompuServe, or SpryNet Web sites.
