Personal Web Server Cannot Be Used as a Gopher Server

 Last reviewed: August 26, 1997
Article ID: Q158832
The information in this article applies to:
  • Microsoft Personal Web Server versions 1.0, 1.0a for Windows 95

SUMMARY

Microsoft Personal Web Server cannot be used as a Gopher server.

MORE INFORMATION

The Gopher protocol is a menu-based interface for opening resources on the Internet. The menus have the same function as hyperlinks on the Web.

Personal Web Server supports only the Hypertext Transfer Protocol (HTTP) and File Transfer Protocol (FTP). There is no support for the Gopher protocol.
