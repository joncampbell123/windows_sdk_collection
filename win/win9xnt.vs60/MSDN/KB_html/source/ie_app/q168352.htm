Err Msg: Importing the Movie Failed. Error -50

 Last reviewed: September 10, 1997
Article ID: Q168352
The information in this article applies to:
  • Microsoft Internet Explorer version 3.01 for Macintosh

SYMPTOMS

When you attempt to play a video (.avi) file on a Web page using Microsoft Internet Explorer version 3.01 for Macintosh, the following error message may occur: 

   Importing the movie failed.
   Error -50

CAUSE

This error message occurs when the .avi file has been encoded with a unsupported compression/decompression software (codec). Some .avi files can be played correctly while others, using an unsupported codec, may produce error messages.
