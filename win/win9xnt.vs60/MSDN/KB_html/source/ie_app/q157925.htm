Internet Explorer Error Message: No Dial-In Driver Loaded

 Last reviewed: September 2, 1997
Article ID: Q157925
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02a for Windows 3.1

SYMPTOMS

After you install Internet Explorer for Windows 3.1 with the Dialer tool, you may receive the following error message when you try to connect to your Internet service provider (ISP): 

   No dial-in driver loaded.

CAUSE

This behavior can occur if either of the following conditions exists:

  • You have not yet restarted your computer after installing Internet Explorer.
  • You installed Internet Explorer by manually copying the installation files from one computer to another instead of running Internet Explorer Setup.

RESOLUTION

To resolve this issue, use one of the following methods:

  • If you have not yet restarted your computer after installing Internet Explorer, do so now. Internet Explorer Setup makes changes to your system files, and you must restart your computer to invoke these changes.
  • If you copied files manually from one computer to another to install Internet Explorer, you will need to run Setup to install Internet Explorer correctly. If you do not have the Setup files for your version of Internet Explorer, download and install the latest version of Internet Explorer for Windows 3.1.
