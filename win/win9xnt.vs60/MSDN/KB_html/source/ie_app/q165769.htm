Messages Received with Internet Mail Cannot Be Edited

 Last reviewed: September 9, 1997
Article ID: Q165769
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95
  • Microsoft Internet Mail and News version 1.0 for Windows NT 4.0

SYMPTOMS

Once a message is received using Internet Mail 1.0, the text of the message cannot be edited.

CAUSE

Microsoft Internet Mail 1.0 does not allow the editing of text in received messages.

RESOLUTION

To modify the text of the message, use any of the following methods:

  • Forward the message to yourself. Once you click Forward, you can edit the message.
  • Reply to the message and modify the text.
  • Save the message as a text file and modify it with any text editor.
  • Use Microsoft Windows Messaging to edit the message.
