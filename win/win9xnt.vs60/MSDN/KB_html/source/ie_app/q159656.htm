Error Message: Error Opening File Attachment

 Last reviewed: September 2, 1997
Article ID: Q159656
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 3.1

SYMPTOMS

When you attempt to view a .jpg or .gif file in Internet Mail or Internet News for Windows 3.1, you may receive the following error message: 

   Error opening file attachment

CAUSE

This error may occur if Internet Explorer is already running when you attempt to view the file. Internet Mail and Internet News cannot open a second instance of Internet Explorer.

RESOLUTION

Close Internet Explorer before you attempt to view attached files from Internet Mail or Internet News.

STATUS

Microsoft is researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.
