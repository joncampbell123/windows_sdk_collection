New Message Window Position Is Not Retained

 Last reviewed: September 3, 1997
Article ID: Q154351
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SUMMARY

When you reposition the New Message window, subsequent New Message windows are not placed in the same position on the screen.

MORE INFORMATION

Because Internet Mail and Internet News are programs based on Windows Explorer, the New Message window position is determined by Windows Explorer. Internet Mail and Internet News only "remember" the width and height of the window.
