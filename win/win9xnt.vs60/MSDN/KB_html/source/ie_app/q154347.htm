Folder View Options Apply to All Folders of the Same Type

 Last reviewed: March 16, 1998
Article ID: Q154347
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows 95
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you click Columns or Sort By on the View menu in Internet Mail and customize these views, changes you make affect more than just the current message folder.

CAUSE

Folder view settings apply to all folders of the same type.

MORE INFORMATION

Message folders in Internet Mail are categorized by whether they contain primarily inbound or outbound messages. Folder view settings apply to all folders in the same category. For example, changing the Columns or Sort By settings for the Outbox also changes the settings for the Sent Items folder.

Inbound message folders include:

  • Inbox
  • Deleted Items
  • User-created folders

Outbound message folders include:
  • Outbox
  • Sent Items

News folders in Internet News constitute a third folder category. Therefore, changing the view settings for one news folder changes the settings for all news folders.
