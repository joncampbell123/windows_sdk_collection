Check Names Dialog Box Appears When Checking Names

 Last reviewed: September 4, 1997
Article ID: Q155276
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

When you use the Check Names command or attempt to send a message before using the Check Names command in Microsoft Internet Mail and News version 1.0, the Check Names dialog box may appear, prompting you to create a new contact or select an existing contact from the address book. This behavior may occur even if the name you typed in the To box very closely resembles a name in the address book, or if you were attempting to send the message by specifying the Internet address of the recipient instead of specifying a name that appears in the address book.

CAUSE

This behavior can be caused by either of the following situations:

  • If you attempt to send a message by specifying a name in the address book, the Check Names dialog box appears unless you type the desired name exactly. If you use a different spelling or only type a portion of the name, the Check Names dialog box appears, even if no other name in the address book closely resembles the name or portion of the name you typed.
  • If you attempt to send a message by specifying the Internet address of the recipient, the Check Names dialog box appears unless you type the recipient's full Internet address. The Internet address you type must use the "user@domain" format.

RESOLUTION

To work around this issue, use the appropriate method:

  • If you are attempting to send a message by specifying a name in the address book, you must type the recipient's name exactly.

    NOTE: If you are attempting to send a message by specifying a name in the address book, you can also select the recipient directly from the address book. To do so, click Address Book on the File menu in the message window, and then double-click the recipient.

  • If you are attempting to send a message by specifying the Internet address of the recipient, you must type the full Internet address.
