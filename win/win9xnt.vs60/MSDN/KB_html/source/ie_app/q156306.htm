Err Msg: This Version of Microsoft NetMeeting Doesn't Run...

 Last reviewed: January 15, 1997
Article ID: Q156306
The information in this article applies to:
  • Microsoft Internet Explorer version 3.0 for Windows NT 4.0
  • Microsoft NetMeeting version 1.0 for Windows 95

SYMPTOMS

When you try to install the full version of Microsoft Internet Explorer 3.0 for Windows NT 4.0, you receive the following error message: 

   This version of Microsoft NetMeeting doesn't run on Windows NT.

CAUSE

The full version of Internet Explorer includes NetMeeting. NetMeeting is not currently compatible with Windows NT 4.0.

RESOLUTION

Click OK to acknowledge the error message. Installation then continues normally. To prevent the error message from occurring, click the option to choose which components are installed during Setup, and choose not to install NetMeeting.

STATUS

Microsoft is aware of this situation and will post more information here in the Microsoft Knowledge Base when it becomes available.
