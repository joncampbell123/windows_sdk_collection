Internet Explorer Quits During Startup

 Last reviewed: September 9, 1997
Article ID: Q165284
The information in this article applies to:
  • Microsoft Internet Explorer version 3.01 for Macintosh

SYMPTOMS

When you start Internet Explorer 3.01 on a Macintosh computer with a Motorola 68020, 68030, or 68040 processor, the program may quit unexpectedly, or you may receive the following error message: 

   The application "Internet Explorer 3.01" can not be opened because an
   unexpected error occured.

CAUSE

This behavior occurs when the 32-Bit Addressing memory option in Control Panels is not selected.

RESOLUTION

To correct this issue, follow these steps:

  1. In Control Panels (click Control Panels on the Apple menu) and double- click the Memory Control Panels device.

  2. Click the 32-Bit Addressing check box to select it.

  3. Close the Memory dialog box, and then restart your computer.
