Err Msg: Error 407 - Unauthorized Proxy Authentication Failed

 Last reviewed: September 9, 1997
Article ID: Q162061
2.10 WINDOWS kbenv kbnetwork kberrmsg

The information in this article applies to:

  • Microsoft Internet Explorer version 2.1 for Windows 3.1

SYMPTOMS

When you use Microsoft Internet Explorer version 2.1 for Windows 3.1 to access the Internet, you may receive the following error message: 

   Error 407 - Unauthorized - proxy authentication failed

CAUSE

You are using a proxy server (a server on a local area network that lets you connect to the Internet without using a modem) configured for user authentication. Proxy authentication is not supported in this version of Internet Explorer.

STATUS

Proxy authentication is supported in later versions of Internet Explorer. For information about how to obtain the latest version of Internet Explorer for Windows 3.1, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q159584
   TITLE     : Microsoft Internet Explorer 3.02a for Windows 3.1 Available
