Internet Mail Window Loses Focus After Incorrect Login

 Last reviewed: September 2, 1997
Article ID: Q154133
1.00 WINDOWS kbenv kbui

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.0 for Windows 95
  • Microsoft Internet Mail and News version 1.0 for Windows NT 4.0

SYMPTOMS

When you are attempting to send and receive mail, if you type an incorrect password, click Cancel when you receive the Authentication Failed error message, and then log in correctly, the Internet Mail window is no longer selected. When this occurs, no window is selected.

RESOLUTION

Click the Internet Mail window to select it.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
