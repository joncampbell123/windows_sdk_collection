Error Message: This Folder Has Been Damaged...

 Last reviewed: February 3, 1998
Article ID: Q162316
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95
  • Microsoft Internet Mail and News version 1.0 for Windows NT 4.0

SYMPTOMS

When you try to delete a message from the Deleted Items folder in Internet Mail 1.0, you may receive the following error message: 

   This folder has been damaged.
   It will now be repaired.
This error message may be followed by the following error message: 

   The selected message could not be deleted.
   An error has occurred.
NOTE: You may also receive these error messages if the "Empty messages from the Deleted Items folder" option is enabled.

RESOLUTION

To resolve this issue, follow these steps:

  1. Compact the Deleted Items folder. To do so, follow these steps in Internet Mail:

    a. On the File menu, point to Folder, point to Compact, and then click 

          Deleted Items.

    b. Quit and restart Internet Mail.

    Try to delete a message from the Deleted Items folder. If you still receive error messages, continue with step 2.

  2. Quit Internet Mail by clicking Close on the File menu.

  3. Click Start, point to Find, and then click Files Or Folders.

  4. In the Named box, type "Deleted Items.mbx" and "Deleted Items.idx" (including quotation marks), and then click Find Now.

  5. Right-click the "Deleted Items.mbx" file, and then click Rename on the menu that appears.

  6. Type a new name for the file, and then press the ENTER key.

  7. Repeat steps 5 and 6 for the "Deleted Items.idx" file.

    NOTE: If your computer is configured for User Profiles, be sure to use the correct "Deleted Items.mbx" and "Deleted Items.idx" files located in the Program Files\Internet Mail and News\<Username>\Mail folder, where <Username> is the name you use to log into Windows 95.

  8. Restart Internet Mail. The "Deleted Items.mbx" and "Deleted Items.idx" files will be automatically re-created.

MORE INFORMATION

The symptoms in this article may also occur when you try to delete a message from the Outbox. To resolve the issue, apply the resolution above to the Outbox. For example, instead of renaming the "Deleted Items.mbx" and "Deleted Items.idx" files, you rename the Outbox.mbx and Outbox.idx files.
