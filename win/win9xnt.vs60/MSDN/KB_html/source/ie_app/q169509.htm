Internet News Contains a Blank Entry in News Server List

 Last reviewed: September 9, 1997
Article ID: Q169509
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.1 for Macintosh

SUMMARY

Using Microsoft Internet Mail and News version 1.1 for Macintosh, it is possible to insert a blank entry into the News Server list.

MORE INFORMATION

When you attempt to add a news server by clicking Preferences on the Edit menu, and then clicking News Server, the Add button is available even when no text is present in the Server Address box. This allows a single, blank entry to be inserted. Once inserted into the list, you can select and delete the blank entry.
