Microsoft Internet Mail Service Available for Download

 Last reviewed: May 1, 1997
Article ID: Q148228
95 WINDOWS kbfile kbtool

The information in this article applies to:

  • Microsoft Windows 95

SUMMARY

Microsoft Internet Mail Service for Windows 95 lets you use Microsoft Exchange to send and receive mail over the Internet. With Internet Mail Service you can receive mail from POP3 (Post Office Protocol) servers that send mail using SMTP (Simple Mail Transport Protocol).

NOTE: Internet Mail Service for Windows 95 is included in Microsoft Internet Explorer version 1.0 and later.

MORE INFORMATION

To install Internet Mail Service, follow these steps:

  1. Download the INETMAIL.EXE file from one of the online services listed below to an empty folder.

  2. In My Computer or Windows Explorer, double-click the INETMAIL.EXE file you downloaded in step 1.

  3. Follow the instructions on the screen. 

    You can find INETMAIL.EXE (size: 240640 bytes) 
                         , a self-extracting file, on the following
services:
