Cannot Remove Microsoft Virtual Reality Modeling Language Add-In

 Last reviewed: September 3, 1997
Article ID: Q145738
The information in this article applies to:
  • Microsoft Internet Explorer version 2.0 for Windows 95

SYMPTOMS

When you attempt to uninstall the Microsoft Virtual Reality Modeling Language (VRML) Add-In for Microsoft Internet Explorer 2.0, you may receive the following error message: 

   ThunkConnect32 Failure!!

CAUSE

This error message may occur if you uninstall Internet Explorer 2.0 before you uninstall VRML Add-In. When Internet Explorer 2.0 is uninstalled, files that are needed to uninstall VRML Add-In are removed.

RESOLUTION

Upgrading Internet Explorer

The latest version of Microsoft Internet Explorer offers more product features, greater reliability and better performance than Internet Explorer 2.0. To download the latest version of Internet Explorer, please see the following Microsoft World Wide Web site 

   http://www.microsoft.com/ie/
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/

WORKAROUND

If you cannot upgrade to the latest version of Microsoft Internet Explorer, use the following workaround.

NOTE: This article only applies to Internet Explorer version 2.0. If you have Internet Explorer version 3.0 or later, do not use the workaround described in this article.

To remove VRML Add-In, you must first reinstall Internet Explorer 2.0. You can then uninstall VRML Add-In, and then uninstall Internet Explorer 2.0.

Reinstalling Internet Explorer 2.0

Reinstall Internet Explorer 2.0 from the Internet Starter Kit version 2.0. Or, if you installed Internet Explorer from an online source (such as the Microsoft World Wide Web site or The Microsoft Network), contact Microsoft Technical Support to request that a copy of Internet Explorer 2.0 be mailed to you. Internet Explorer 2.0 is no longer available for download.

Uninstalling the VRML Add-In for Internet Explorer 2.0

  1. Click the Start button, point to Settings, and then click Control Panel.

  2. In Control Panel, double-click the Add-Remove Programs icon.

  3. In the Add/Remove Programs Properties dialog box, click on the Install/Uninstall tab.

  4. In the software list, click to select the VRML Add-In for Internet Explorer.

  5. Click Add/Remove to begin uninstalling the VRML Add-In. When the process is complete, VRML Add-In for Internet Explorer 2.0 should no longer appear in the list of installed programs.

Uninstalling Internet Explorer 2.0

To uninstall Internet Explorer 2.0, follow the steps described above for uninstalling the VRML Add-In, but in step 4 click to select Microsoft Internet Explorer 2.0, instead of selecting the VRML Add-In.
