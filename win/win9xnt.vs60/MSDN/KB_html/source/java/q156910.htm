FIX: ClassView Allows Duplicate Variables and Methods

 Last reviewed: January 22, 1998
Article ID: Q156910
The information in this article applies to:
  • Microsoft Visual J++ 1.0

SYMPTOMS

The right mouse button in the ClassView window allows you to create new variables and methods. Using this option to create duplicate variables or methods should be illegal, but at this time you can create duplicate variables or methods without any warnings.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been fixed in Visual J++ 1.1.
