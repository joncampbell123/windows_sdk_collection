INFO: Historical List of Shipping Vehicles for Java VM

 Last reviewed: January 29, 1998
Article ID: Q169803
The information in this article applies to:
  • Microsoft Win32 Virtual Machine for Java on the following platform: - x86
  • Visual J++ 1.0, 1.1
  • Internet Explorer 4.0
  • SDK for Java 2.0

SUMMARY

The Microsoft Virtual Machine for Java has been included with many Microsoft products and available for download as an upgrade to an existing Virtual Machine. The following section of this article lists the products and build numbers of the Virtual Machines that have shipped.

MORE INFORMATION

Also note, that it is important to use the tools from the products above with the corresponding build of the Virtual Machine you have installed.
