Cannot ALT+ESC from Works Under Windows 3.0 in Standard Mode

 Last reviewed: November 23, 1994
Article ID: Q70207
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

You cannot switch out of Microsoft Works when you are running from a WORKS.PIF file under Windows 3.0 in standard mode, and you are using ALT+ESC, if one or more of the Directly Modifies options in the PIF file settings is checked.

This problem does not occur when Microsoft Windows 3.0 is running in enhanced mode.

WORKAROUND

Cancel the Directly Modifies options selections in the WORKS.PIF file for COM1, COM2, COM3, and COM4.
