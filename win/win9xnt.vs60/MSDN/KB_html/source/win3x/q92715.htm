WFWG: Minimum System Requirements for Workgroup Connections

 Last reviewed: November 21, 1994
Article ID: Q92715
The information in this article applies to:
  • Microsoft Workgroup Connections version 1.0

SUMMARY

The following is a list of the minimum system requirements for Workgroup Connections for MS-DOS.

The client installation of Workgroup Connections for MS-DOS requires:

  • MS-DOS version 3.3 or later
  • An 8088/8086 or later processor
  • 640K or more random-access memory (RAM) (320K of free conventional memory)
  • 900K of free disk space (1.5 MB with Mail services)
  • Floppy or hard disk.
  • NDIS-certified network adapter

MORE INFORMATION

There is no server installation of Workgroup Connections for MS-DOS. You must have at least one computer with Windows for Workgroups acting as a server before Workgroup Connections can be used as a client.
