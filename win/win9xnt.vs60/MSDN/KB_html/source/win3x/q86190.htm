Norton Anti-Virus 2.0 Requires Updated Driver

 Last reviewed: November 23, 1994
Article ID: Q86190
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

Using Symantec's Norton Anti-Virus version 2.0 (dated 12/21/91) with Microsoft Windows operating system version 3.1 causes the entire system to restart when you press CTRL+ALT+DEL from an MS-DOS command prompt or from MS-DOS-based applications in 386 enhanced mode.

RESOLUTION

Obtaining an updated VNAVD.386 file from Symantec corrects this problem.

MORE INFORMATION

VNAVD.386 is a virtual device driver located in the Windows SYSTEM subdirectory.
