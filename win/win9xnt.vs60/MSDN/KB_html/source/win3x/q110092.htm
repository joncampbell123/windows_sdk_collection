How to Flush 32-Bit File Access Cache (VCACHE)

 Last reviewed: November 21, 1994
Article ID: Q110092
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

You can commit the contents of the 32-bit file access cache (VCACHE) to disk by running MS-DOS Prompt from the Main group and the pressing CTRL+C or CTRL+BREAK.

CTRL+C and CTRL+BREAK cause MS-DOS to issue a disk reset, which causes 32-bit file access to flush, but not invalidate. the cache.
