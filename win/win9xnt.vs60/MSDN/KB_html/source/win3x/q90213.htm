AT&T Mouse Problems with Windows for Workgroups

 Last reviewed: November 22, 1994
Article ID: Q90213
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.1

SYMPTOMS

If you are using a three-button AT&T mouse with Windows for Workgroups and you have the Windows Logitech mouse driver installed, your NUM LOCK, CAPS LOCK, and SCROLL LOCK indicator lights may blink.

WORKAROUND

To correct this problem, run Windows for Workgroups Setup and change the mouse driver to "Microsoft or IBM PS/2."
