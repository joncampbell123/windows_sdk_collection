LANtastic Server Causes Communications Slow Down

 Last reviewed: November 21, 1994
Article ID: Q88296
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

When running Microsoft Windows version 3.1 from a LANtastic version 4.1 server, serial data communication speeds may be degraded due to the network software. Bit rates of 9600 bits per second (BPS) or higher may not run reliably.

Contact LANtastic technical support for possible solutions to this problem.

WORKAROUND

To work around this problem, run your communication software at speeds of 2400 BPS.
