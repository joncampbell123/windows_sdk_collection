Disconnect Network Printer Only in Print Manager

 Last reviewed: November 21, 1994
Article ID: Q89299
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

In Windows for Workgroups version 3.1, the Print Manager must be used to disconnect network printers.

You cannot disconnect network printers through the Control Panel in Windows for Workgroups. In earlier versions of Windows, this feature has been available in the Printers option of Control Panel.

NOTE: You can still connect a network printer through the Control Panel Printers option.

MORE INFORMATION

Steps to Disconnect a Network Printer in Windows for Workgroups

  1. Open the Print Manager.

  2. From the Printer menu, choose Disconnect Net Printer.

  3. Select the network printer you want to disconnect, then choose the OK button.
