Supplemental Driver Library Printer Driver Enhancements

 Last reviewed: February 28, 1997
Article ID: Q67269
3.00 3.00a WINDOWS

The information in this article applies to:

  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

All the drivers in the Supplemental Driver Library (SDL) are new, with the exceptions of the PostScript, HPPCL, and HPPCL5A drivers, which are modifications of existing drivers. Below you will find the additions and changes.

MORE INFORMATION

PostScript (PSCRIPT.DRV Version 3.4)

  1. Supports Adobe Document Structuring Conventions

  2. Corrects a problem with multi-bin PostScript printers and Word for Windows (Word for Windows 1.10 required)

HPPCL (HPPCL.DRV Version 3.4)

  1. Performance enhancements and half-toning capabilities

HPPCL5A (HPPCL5A.DRV Version 3.61)

  1. Performance enhancements

  2. Half-toning capabilities

  3. Support for the Hewlett-Packard (HP) LaserJet IIID

  4. Scalable font support; internal scalable fonts (CGTimes and Univers) have always been supported

  5. Support for Type Director scalable type and two new scalable font cartridges, as follows: 

          Brilliant Presentations I/Compelling Publications II
      Distinctive Documents I/Compelling Publications I

The performance enhancements to both the HPPCL and HPPCL5A drivers is the addition of Variable Banding Support. Bands are not limited to 16K, as in previous LaserJet drivers. This was changed in both the HPPCL and HPPCL5A drivers. Band sizes are dependent on available memory.

NOTE: The Windows 3.00 Supplemental Driver Library can be obtained from Microsoft Sales and Service by calling (800) 426-9400. International customers should contact the nearest Microsoft subsidiary.
