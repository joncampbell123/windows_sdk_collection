PageMaker Versions Compatible with Windows 3.0 and 3.1

 Last reviewed: November 30, 1994
Article ID: Q62787
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

The Microsoft Windows version 3.0 Applications Reference List states that Aldus PageMaker version 3.0 is compatible with Windows version 3.0. You must have PageMaker version 3.01 to run with either standard mode or 386 enhanced mode Windows version 3.0.

PageMaker 3.01 or later is required to run with Windows versions 3.1 and 3.11.

MORE INFORMATION

The most current release of Aldus PageMaker is 5.0. For more information about PageMaker, call Aldus technical support.
