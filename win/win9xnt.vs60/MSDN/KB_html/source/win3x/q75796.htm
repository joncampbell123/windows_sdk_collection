UAE When Running X-Vision 4.0

 Last reviewed: November 23, 1994
Article ID: Q75796
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

You may receive an Unrecoverable Application Error (UAE) when using X-Vision 4.0 in Microsoft Windows version 3.0.

CAUSE

The XHOST.TXT file contains blank lines.

WORKAROUND

You may be able to work around the problem by deleting the XHOST.TXT file or filling the blank lines with text.

MORE INFORMATION

For more information, contact Unipress Software Distributors technical support.
