Windows 3.1 Resource Kit Files Available

 Last reviewed: May 1, 1997
Article ID: Q124435
3.1 WINDOWS kbfile

The information in this article applies to:

  • Microsoft Windows for Workgroups version 3.1

SUMMARY

The file RESKIT.EXE contains the Windows 3.1 Resource Kit. For information about obtaining RESKIT.EXE, refer to the MORE INFORMATION section below. RESKIT.EXE contains the following files:

MORE INFORMATION

You can find RESKIT.EXE (size: 1785913 bytes) 
                       , a self-extracting file, on the following
services:
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select the product.

          Enter kbfile RESKIT.EXE (size: 1785913 bytes) 
                             , and click GO!
      Open the article, and click the button to download the file.


  • Internet (anonymous FTP)

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder.
      Get RESKIT.EXE (size: 1785913 bytes) 

    

  • Microsoft Download Service (MSDL)

          Dial (425) 936-6735 to connect to MSDL
      Download RESKIT.EXE (size: 1785913 bytes)
