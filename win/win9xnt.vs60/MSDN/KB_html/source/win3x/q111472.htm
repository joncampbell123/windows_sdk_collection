WFWG 3.11 and Sun PC-NFS 5.0 Err Msg: Invalid COMMAND.COM

 Last reviewed: July 19, 1995
Article ID: Q111472
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

If you have Sun Microsystems PC-NFS 5.0 installed, when you quit Microsoft Windows for Workgroups 3.11, you may receive the following error message: 

   Invalid COMMAND.COM

CAUSE

This error message occurs when the following three conditions exist:

  • Windows for Workgroups 3.11 is running.
  • Sun PC-NFS 5.0 is installed as a secondary network.
  • 32-Bit File Access is enabled.

WORKAROUND

To work around this problem disable 32-Bit File Access.
