Wordstar Err Msg: Application Has Violated System Integrity

 Last reviewed: November 23, 1994
Article ID: Q80859
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

When you use WordStar 2000 version 3.5 Revision A, B, or C within Windows, the following error may occur: 

   This Application Has Violated System Integrity

CAUSE

This error occurs when your system attempts to use expanded memory for WordStar within Windows.

RESOLUTION

To resolve the problem, you need to obtain a patch from WordStar 2000 technical support. The patch disables WordStar's use of expanded memory.

NOTE: Constructing a program information file (PIF) that does not allocate expanded (EMS) memory does not prevent the error.
