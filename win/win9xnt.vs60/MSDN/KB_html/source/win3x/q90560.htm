Hearts Requires EGA (or Above) and Mouse

 Last reviewed: January 3, 1995
Article ID: Q90560
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

The Microsoft Hearts Network card game (MSHEARTS.EX) included with Windows for Workgroups requires a minimum display resolution of EGA and a Windows- compatible mouse.

With the Trace Access Pack installed you can use the numeric keypad to move the mouse. For more information about the Trace Access Pack for Windows, query on the following words in the Microsoft knowledge base: 

   trace and access and pack and windows
