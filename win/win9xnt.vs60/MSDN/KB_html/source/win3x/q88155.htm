Notepad Print Setup May Not Appear

 Last reviewed: November 23, 1994
Article ID: Q88155
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

When using the Notepad and Calendar applications included with Windows 3.1, the Print Setup dialog box may not appear after changing the default printer in the Control Panel while these applications are running.

WORKAROUND

Save the current file, exit, and restart the application.
