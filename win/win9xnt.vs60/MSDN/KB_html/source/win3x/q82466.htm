Cognose Powerplay's Files Box Is Not Refreshed Correctly

 Last reviewed: November 23, 1994
Article ID: Q82466
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

In the application Cognos Powerplay, the Files box is incorrectly refreshed when a new directory is selected in the Directories box.

When you display a new directory, if it has fewer qualifying files than the first directory, some of the files from the first directory are still displayed.

If there are fewer qualifying files in the second directory, then the remaining files from the first directory are still displayed along with the new files.
