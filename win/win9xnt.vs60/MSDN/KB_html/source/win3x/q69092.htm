Paintbrush Prints Blank Sheet on HP Printers

 Last reviewed: November 23, 1994
Article ID: Q69092
The information in this article applies to:
  • Microsoft Windows operating system version 3.0, 3.0a

When you print an image from Windows Paintbrush on a Hewlett-Packard (HP) LaserJet series printer, be sure the Quality section is set to "Proof" in the File menu's Print dialog box.

If "Draft" is selected, the printer will print a blank sheet of paper. This is also true when using the Deskjet family of printers.
