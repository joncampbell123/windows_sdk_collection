KEAterm 1.1.0 Does Not Print TrueType Fonts to an HP III

 Last reviewed: November 21, 1994
Article ID: Q82426
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

KEAterm version 1.1.0, a terminal emulation software package, prints garbage characters when you try to print TrueType fonts to a Hewlett- Packard (HP) LaserJet III printer using version 3.89 of the HPPCL5A.DRV driver. TrueType fonts must be turned off to print with this application to an HP III printer.

STATUS

This problem does not occur with later version of KEAterm.
