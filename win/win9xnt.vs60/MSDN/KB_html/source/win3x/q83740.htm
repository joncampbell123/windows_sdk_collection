Cannot Use Certain Shortcut Keys in Program Item Properties

 Last reviewed: March 27, 1995
Article ID: Q83740
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

Under Microsoft Windows operating system version 3.1, when you set up shortcut key combinations for an icon by choosing Properties from the File menu Program Manager, certain key combinations are not permitted. The key combinations that aren't allowed conflict with the shortcut key combinations used for keyboard selection of items. The combinations in question are: 

   ALT+B   ALT+C
   ALT+D   ALT+H
   ALT+I   ALT+R
   ALT+S   ALT+W
