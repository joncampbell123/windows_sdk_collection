Cannot See Writing with Pen Pointing Device and Super VGA

 Last reviewed: November 22, 1994
Article ID: Q111569
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11
  • Microsoft Windows for Pen Computing version 1.0

SYMPTOMS

If you use the Super VGA video driver (SVGA256.DRV) included with Microsoft Windows for Workgroups 3.11 with Microsoft Windows for Pen Computing 1.0, you cannot see the "ink" as you write with the pen pointing device.

CAUSE

The Windows for Workgroups Super VGA video driver (SVGA256.DRV) does not support the use of pen pointing devices.

WORKAROUND

To work around this problem use the VGA video driver, or a video driver supplied by your video card vendor.

STATUS

Microsoft has confirmed this to be a problem in Windows for Workgroups version 3.11. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
