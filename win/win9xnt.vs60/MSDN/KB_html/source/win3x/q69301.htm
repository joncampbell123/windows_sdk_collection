Windows Demonstration Won't Run Full-Featured Applications

 Last reviewed: November 23, 1994
Article ID: Q69301
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

When you run the Windows demonstration package, the message 

   Application Execution Error
   The application you are trying to run is a full-featured
   application. This demonstration version of Windows runs only
   demonstration applications. To run the application, see your
   Windows dealer to purchase the full version of Windows.
will occur if you try to start any Windows application designed for the retail package of Windows. As the message indicates, the demonstration version of Windows will only run the demonstration applications that come with the demonstration package.

MORE INFORMATION

For additional information on which applications work with the demonstration version of Windows, query on the words: 

   windows and applications and functional and working and model
