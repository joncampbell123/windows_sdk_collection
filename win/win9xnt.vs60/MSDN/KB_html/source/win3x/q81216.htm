PIF Files (from G to L) Created by Windows 3.0 Setup Program

 Last reviewed: November 21, 1994
Article ID: Q81216
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The following are the default parameters for PIF files (in alphabetical order from G to L) created by the Microsoft Windows version 3.0 Setup program. These defaults may be modified by loading the PIF file into the Windows 3.0 PIF Editor (PIFEDIT.EXE).
