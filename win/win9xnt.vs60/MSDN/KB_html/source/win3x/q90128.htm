Leading Technology PC Partner Requires VGA (Version 3.0)

 Last reviewed: November 22, 1994
Article ID: Q90128
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

When you use the Leading Technology PC Partner computer with Microsoft Windows with the standard VGA driver installed, text is garbled after exiting Windows.

The problem exists in both standard and enhanced mode. The screen continually displays binary characters or the video display is garbled after exiting Windows. The system then hangs and the only way to continue is to restart the computer.

WORKAROUND

Using the VGA (version 3.0) video driver supplied with Windows 3.1 corrects this problem.

For additional information, call Leading Technology technical support.
