Cannot Disconnect from Network with Norton Desktop 2.0

 Last reviewed: November 21, 1994
Article ID: Q88077
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft LAN Manager version 2.1
  • Microsoft Windows for Workgroups version 3.1

SUMMARY

Symantec technical support has confirmed that when you use version 2.0 of Norton Desktop for Windows with Windows 3.1 and a Microsoft LAN Manager 2.1 Enhanced network, you may not be able to disconnect from network drives.

MORE INFORMATION

If you are using Desktop for Windows 2.0 as the shell for Windows 3.1 and have installed the LAN Manager 2.1 Enhanced network driver, you may not be able to disconnect from network drives when choosing Disconnect Net Drive from the Disk Menu of Desktop for Windows 2.0.

Page 4-7 of the "Norton Desktop for Windows User's Guide" for version 2.0 states that when you choose this option: 

   Either the Disconnect Network Drive dialog box or the Network
   Connections dialog box appears, depending on your network and
   which version of Windows you are running."
When using Windows 3.1 with the LAN Manager 2.1 Enhanced network driver installed, the Disconnect Network Drive dialog box should appear. Instead, the Connect Network Drive dialog box appears.

Symantec has confirmed that this to be a problem with Desktop for Windows 2.0. Symantec/Norton Desktop for Windows developers are currently studying the problem and plan to have a correction or update available soon.

REFERENCES
