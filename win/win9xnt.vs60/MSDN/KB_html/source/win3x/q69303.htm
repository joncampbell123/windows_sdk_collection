Using the Western Digital Ethernet Plus Card and Windows 3.0

 Last reviewed: November 22, 1994
Article ID: Q69303
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

If you are using a Western Digital Ethernet Plus network card, the card must be version 3.06 or higher to run Microsoft Windows 3.0 in 386 enhanced mode.

MORE INFORMATION

To find out which driver version you are using, type the following at the MS-DOS prompt and then press ENTER: 

   IPX I
The following problems may occur if you are using a version of the Western Digital Ethernet Plus network card earlier than 3.06:
  • Machine stops responding (hangs) when going into enhanced mode.
  • Machine drops back to MS-DOS when entering enhanced mode.

For more information on the latest version of the Ethernet Plus network card, contact Western Digital technical support.
