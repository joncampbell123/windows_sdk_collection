APPS.HLP Incorrectly Lists Print Setup Option (Word 1.1)

 Last reviewed: November 21, 1994
Article ID: Q84168
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Word for Windows, version 1.1

SUMMARY

The APPS.HLP file provided with Windows 3.1 incorrectly refers to "Print Setup" as a command available from the File menu of Microsoft Word for Windows, version 1.1.

The correct option under the File menu is "Printer Setup."

MORE INFORMATION

The APPS.HLP file included with Windows 3.1 discusses compatibility issues that may arise after upgrading from an earlier version of Microsoft Windows.

The section dedicated to Microsoft Word for Windows, version 1.1, outlines a technique to "refresh" the font selection to allow TrueType fonts to be used.

This technique may be needed if TrueType fonts do not appear after upgrading to Windows 3.1, but the information is only valid if a printer driver that is compatible with the TrueType font technology is selected.

The following is a corrected, modified version of the excerpt from APPS.HLP.

Version 1.1 of Microsoft Word for Windows

When you upgrade Windows from version 3.0 to version 3.1, the new fonts that are included with version 3.1 may not appear in the Word for Windows list of available fonts.

To correct this:

  1. From the File menu, choose Printer Setup from the File menu in Word.

  2. When the Printer Setup dialog box appears, choose the OK button.

To use Word for Windows with Windows version 3.1, Microsoft recommends that you upgrade to Word for Windows version 2.0.

For information about obtaining Microsoft Word for Windows version 2.0, contact your software vendor or call Microsoft Sales Information Center (MSIC) at (800) 426-9400. Outside the United States, contact your local software vendor or Microsoft subsidiary.
