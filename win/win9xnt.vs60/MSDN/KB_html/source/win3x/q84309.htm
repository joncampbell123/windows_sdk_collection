Summary of WRI Files Included with Windows 3.1 Resource Kit

 Last reviewed: November 21, 1994
Article ID: Q84309
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

This article contains a brief summary of the WRI files included with the Windows Resource Kit for Windows operating system version 3.1. The files included are : 

   31README.WRI
   3270.WRI
   COMMQA.WRI
   LAYOUTS.WRI
   MOUSEQA.WRI
   NETWORKS.WRI
   PRINTERS.WRI
   README.WRI
   SYSINI.WRI
   WININI.WRI
   T2HINTS.WRI
   VIDEOQA.WRI
   IPX.WRI

MORE INFORMATION

31README.WRI

Duplicate of Windows 3.1 README.WRI file

3270.WRI

Describes some specific problems you might encounter when running non-Windows 3270-emulation applications with Windows version 3.1

COMMQA.WRI

Common questions and answers about Windows 3.1 and serial communication

LAYOUTS.WRI

Windows 3.1 disk layouts by file, in the following form: 

  Filename  1.44Mb  1.2MB  720Kb  360Kb  Expanded Size  FileDescription
MOUSEQA.WRI

Common questions and answers about using Windows 3.1 with a mouse

NETWORKS.WRI

Duplicate of the Windows 3.1 NETWORKS.WRI file

PRINTERS.WRI

Duplicate of the Windows 3.1 PRINTERS.WRI file

README.WRI

The README.WRI covers the following topics:

  • Adding third-party drivers to a shared copy of Windows
  • Background on Stacks = 9,256
  • Setting Up Windows over an asynchronous connection
  • RAMDrive Suggestions
  • Documentation Notes
  • Utilities
  • Windows 3.1 Disk Layouts

SYSINI.WRI

Explanation of the format of the Windows 3.1 SYSTEM.INI file

This includes the SYSTEM.INI file and a complete listing of all entries in the file. The SYSINI.WRI that is included with Windows 3.1 is only a partial listing.

WININI.WRI

Explanation of the format of the Windows 3.1 WIN.INI file

This includes the WIN.INI file and a complete listing of all entries in the file. The WININI.WRI that is included with Windows 3.1 is only a partial listing.

T2HINTS.WRI

TrueType hints

VIDEOQA.WRI

Common questions and answers about Windows 3.1 and video support

IPX.WRI

The Microsoft IPX Upgrade Utility (NEWIPX.EXE) is designed to help network administrators who need to update the version of Novell NetWare on their client workstations in order to run Windows 3.1.
