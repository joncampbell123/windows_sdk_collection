Windows Err Msg: Control Panel-Not Enough Memory for this Task

 Last reviewed: November 23, 1994
Article ID: Q84167
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

When you attempt to set up a bitmap as the wallpaper in Windows operating system version 3.1, the following error message may display: 

                   Control Panel
   Not enough memory available for this task.
   Quit one or more applications to increase available
   memory, and then try again.

MORE INFORMATION

This error occurs when the file being chosen is not in a BMP format recognized by Windows. To verify that the file is in the BMP format, open it in Windows Paintbrush. If it will not open properly in Paintbrush, the file is not in a BMP format.

Reference(s): "Microsoft Windows User's Guide," version 3.1, page 151
