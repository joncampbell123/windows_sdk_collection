Everex EGA Card Hangs When Running MS-DOS Apps

 Last reviewed: November 22, 1994
Article ID: Q73225
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

Your system may stop responding (hang) while running MS-DOS applications inside or outside of Microsoft Windows version 3.0 if you have an Everex EGA card model EV659.

CAUSE

Everex EGA card models EV659 with ROM versions earlier than 1.56 cause this problem.

RESOLUTION

Everex offers upgrade ROMS for this card. The new ROMs are available from Everex Technical Support.

MORE INFORMATION

Contact Everex for more information.
