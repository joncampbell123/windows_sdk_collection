Inactive DOS Window After Exiting Non-Windows Application

 Last reviewed: November 21, 1994
Article ID: Q91675
The information in this article applies to: 

  - Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

If the Close Window On Exit check box is not selected in a non-Windows- based application's PIF (program information file), exiting the application generates an MS-DOS window with "(Inactive [program description])" on the title bar.

RESOLUTION

To prevent this from happening, make sure that box is selected.

To close the MS-DOS window, open the Control menu (-) and choose Close.
