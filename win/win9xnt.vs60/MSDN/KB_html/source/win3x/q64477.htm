Windows 3.0: Microsoft MusicBox Windows Version Won't Run

 Last reviewed: November 22, 1994
Article ID: Q64477
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

The Windows version of Microsoft MusicBox does not run in Microsoft Windows version 3.0 standard or enhanced mode. If you try to run MusicBox from standard or enhanced mode, the following error message is displayed: 

   CD-ROM Drive 1 Not Connected or MSCDEX Not Started
   Check Installation of MSCDEX

STATUS

Microsoft is researching this problem and will post new information here as it becomes available.

MORE INFORMATION

Microsoft MusicBox was shipped in both an MS-DOS version and a Windows version. The MS-DOS version can be run with a standard PIF.
