Write Err Msg: There Are No Fonts Installed...

 Last reviewed: November 21, 1994
Article ID: Q83421
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

In the Microsoft Windows version 3.1, Microsoft Windows Write may display the following error message if you choose Fonts from the Character menu: 

   There are no fonts installed; Run Control Panel to install fonts.

CAUSE

This message appears if the following two conditions are true:

  • The default printer is set to Generic/Text Only.

    -and-

  • Both the Enable TrueType Fonts and the Show Only TrueType Fonts In Applications check boxes are marked in the TrueType dialog box of the Control Panel.
