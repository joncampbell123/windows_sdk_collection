Archive's Qicstream Tape Backup Hangs Windows

 Last reviewed: November 23, 1994
Article ID: Q68171
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

You may be unable to run Microsoft Windows version 3.0 after doing a tape backup using Archive's Qicstream tape backup software.

WORKAROUND

To work around the problem, do one of the following:

  • Restore the Windows directory from the tape backup.

    -or-

  • Reinstall Microsoft Windows.

MORE INFORMATION

Maynard Electronics is currently researching a solution to this problem. No upgrade is currently available.

For more information, contact Maynard Electronics Technical Support.
