Customer Letter Included with TrueType Font Pack Version 1.0a

 Last reviewed: May 1, 1997
Article ID: Q117730
3.10 3.11 WINDOWS kbprint

The information in this article applies to:

  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

In June 1992, Microsoft released an upgrade to the original TrueType Font Pack version 1.0 (files dated 3/9/92). This upgrade is version 1.00a (files dated 5/5/92). For information about obtaining version 1.00a, call the Microsoft Sales Information Center (MSIC) at (800) 426-9400 or Microsoft Product Support Services at (425) 637-7098. This article contains the letter that is included with this upgrade package.

MORE INFORMATION

Dear Customer:

This Customer Service Disk Set contains version 1.00a of the Microsoft TrueType Font Pack for Windows. We've localized the font style names (such as bold and italic) for this version and will include localized font style names in all future versions of the TrueType Font Pack. Since we've made no other changes from version 1.0, you only need to install version 1.00a if you wish to see the font style names in a language other than English.

The font style names will now appear in whatever language you've chosen in the Windows Control Panel International dialog box. If you use Windows in English, the font style names will still appear in English. If you choose a different language in the International dialog box, the font style names will appear in that language. However, the localized font style names will appear on your screen only if you change your language setting before you install fonts from TrueType Font Pack version 1.00a. Chapter 5 of the "Microsoft Windows User's Guide" contains instructions on changing your language setting.

Before installing the localized font files, you must remove the fonts you installed from version 1.0 of the TrueType Font Pack. Remove only the fonts listed below; do not remove any other fonts. Chapter 5 in the "Microsoft Windows User's Guide" contains instructions on adding and removing fonts.

Follow these steps to install Microsoft Windows TrueType Font Pack version 1.00a:

  1. Remove the TrueType Font Pack version 1.0 fonts from your hard disk (see the "User's Guide").

  2. Change your language setting in the Control Panel International dialog box (see the "User's Guide").

  3. Restart your system.

  4. Insert Customer Service Disk 1 in drive A.

  5. Type a:\setup and press ENTER.

  6. Follow the on-screen directions.
