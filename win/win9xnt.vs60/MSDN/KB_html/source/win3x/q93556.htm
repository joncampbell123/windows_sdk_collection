Calendar Does Not Display New Date

 Last reviewed: November 23, 1994
Article ID: Q93556
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

If you run Windows Calendar and leave the computer on for 24 hours continuously, the date displayed on the status line does not advance to the next day, causing the date to appear incorrectly.

WORKAROUND

To change the date displayed by Calendar, do one of the following:

  • Quit Calendar, then restart it. The current date is then displayed on the status line.

    -or-

  • Manually advance the date by clicking the right arrow in the Calendar status line. (If you are using the keyboard, press CTRL+Page Down.)
