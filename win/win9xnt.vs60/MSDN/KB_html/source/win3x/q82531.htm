WordPerfect Office Cannot Be Started With File Run

 Last reviewed: November 23, 1994
Article ID: Q82531
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

You cannot start WordPerfect Office with the File Run command in Microsoft Windows 3.1.

CAUSE

The name of the main WordPerfect Office file is SHELL.EXE. When you choose the File Run command, Windows version 3.1 sees that SHELL.DLL is loaded and fails to run WordPerfect Office.

WORKAROUND

To work around this problem, start WordPerfect Office from the MS-DOS Command Prompt.

MORE INFORMATION

For more information about WordPerfect Office, contact WordPerfect Corporation.

NOTE: The WordPerfect Office Notify option does not work correctly with Microsoft Windows version 3.0.
