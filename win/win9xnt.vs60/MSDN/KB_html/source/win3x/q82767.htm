Relay Gold 5.0 May Cause a GP Fault with Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q82767
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

In the Microsoft Windows operating system version 3.1, Relay Gold version 5.0 causes a general protection (GP) fault if you attempt to connect a second session through the Session Manager option. The following error message may display: 

   GP Fault in KRNL386.EXE at 0001:575D
The GP fault is caused by Relay Gold, not Windows 3.1.

For more information, contact Relay Gold.
