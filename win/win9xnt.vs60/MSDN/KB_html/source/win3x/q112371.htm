WFWG Err Msg: Error: Cannot Find CORE.DLL

 Last reviewed: November 21, 1994
Article ID: Q112371
The information in this article applies to:
  • Microsoft Windows operating system version 3.11
  • Microsoft Windows for Workgroups version 3.11
  • Microsoft PowerPoint for Windows, version 4.0

SYMPTOMS

When you run Windows or Windows for Workgroups with software that initializes the printer driver, such as Microsoft Write or Microsoft Word for Windows, you may receive the following error message: 

   Error - cannot find CORE.DLL
Launching Microsoft PowerPoint for Windows 4.0 may cause a GPF in HPPCL5E3.DLL. Deleting HPPCL5E3.DLL and replacing it will give the CORE.DLL error message the next time PowerPoint is launched.

CAUSE

The Hewlett-Packard (HP) supplied printer driver for the HP 4/4M printer fails to update correctly when you upgrade from either Windows 3.1 or Windows for Workgroups 3.1 to Windows for Workgroups 3.11. As a result, when an application attempts to initialize the printer, it cannot do so, and the error message is generated.

RESOLUTION

To correct this problem:

  1. Delete the HP LaserJet 4 driver (HPPCL5E.DRV) from the Windows SYSTEM directory.

  2. Remove the HP LaserJet 4/4M driver from the Printers dialog box in Control Panel.

    NOTE: You receive a Windows error message stating that the printer driver cannot be found and that you should reinstall it. This is an expected message, and it informs you that you have deleted the correct printer driver.

  3. Reinstall the HP LaserJet 4/4M in the Printers dialog box.

    If you receive and error message stating that the file HPPCL5E3.DLL is different than the currently installed driver, choose to replace the existing file.
