Formatting 2.8-MB Floppy Disks from File Manager

 Last reviewed: November 21, 1994
Article ID: Q98946
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

Windows 3.0 and 3.0a File Manager does not support formatting 2.8-megabyte (MB) floppy disks. To format a 2.8-MB floppy disk, you must use the MS-DOS FORMAT command.

Windows 3.1 File Manager does support the formatting of 2.8-MB floppy disks. When the File Manager detects a 2.8-MB disk, File Manager displays a 2.8-MB option in the Format Disk dialog box.

MORE INFORMATION

Some IBM PS/2 computers have 2.8-MB floppy disk drives and come with instruction sheets recommending that if you use Windows 3.0 or 3.0a, you should quit Windows to format 2.8-MB floppy disks. The following IBM PS/2 models have 2.8-MB floppy disk drives:

  • 56 SX
  • 56 SLC
  • 57 SX
  • 57 SLC
  • 90 OLD
  • 90 OLF
  • 95 OLF
  • 95 OMF
