HPPCL5A Scalable Font Support in Supplemental Driver Library

 Last reviewed: November 23, 1994
Article ID: Q67270
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

The Microsoft Windows version 3.0 Supplemental Driver Library (SDL) contains the HPPCL5A.DRV version 3.61, which supports Type Director scalable fonts and also two new scalable cartridges: 

   Brilliant Presentation I/Compelling Publications II
   Distinctive Documents I/Compelling Publications I
The Hewlett-Packard (HP) LaserJet IIIs internal scalable fonts (CGTimes and Univers) have always been supported.
