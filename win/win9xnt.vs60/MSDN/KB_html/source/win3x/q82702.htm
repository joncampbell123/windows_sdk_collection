LanFax Redirector Appends Extra Extension to Filename

 Last reviewed: November 21, 1994
Article ID: Q82702
The information in this article applies to:
  • Microsoft Windows operating system versions 3,0, 3.00a, 3.1, 3.11

SUMMARY

In the Microsoft Windows operating system versions 3.0 and 3.1, Lan Fax Redirector for Windows appends an extra extension to the filename if you retry to save a file to nonexistent path. For example, after trying to save FILE.TXT a few times, the filename is listed as FILE.TXTTXTTXT

To correct this problem, double-check the filename before saving the file.

This problem occurs in both Windows 3.0 and 3.1.
