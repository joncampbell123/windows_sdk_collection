Toshiba T3100/10 /20 BIOS Required to Work with Windows 3.0

 Last reviewed: November 21, 1994
Article ID: Q81297
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The Toshiba T3100/10 (with a 10-MB hard drive) and the Toshiba T3100/20 (with a 20-MB hard drive) run correctly with Microsoft Windows version 3.0 with the BIOS versions indicated below.

MORE INFORMATION

The T3100/10 requires BIOS version 2.01 or earlier. BIOS versions greater than 2.01, such as 4.20, should not be used even if the hard drive has been upgraded to 20 megabytes.

The T3100/20 works with Windows 3.0 with the following BIOS versions and displays: 

   BIOS            Display
   ----            -------

   2.01            CGA

   4.20            Olivetti/AT&T Monochrome or PVC Display
NOTE: Both of these models (and the T3100e) do not display a highlight bar during Setup. There is currently no workaround available for this problem.
