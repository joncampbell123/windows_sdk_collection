Error when Installing Fastback for Windows on Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q83587
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

When running the installation program for Fifth Generation Systems' Fastback for Windows, the installation program will hang during the process. Moreover, running the FB.EXE file directly (from File Manager), you will receive the following error: 

   Cannot run on anything except Windows V3.0

MORE INFORMATION

Fifth Generation Systems' Fastback for Windows is designed specifically for Windows 3.0 or 3.0a. Apparently, supported Window's version numbers are written into Fastback, and thus will not run on Windows 3.1.
