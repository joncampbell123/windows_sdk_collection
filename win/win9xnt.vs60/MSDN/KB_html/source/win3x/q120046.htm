Media Player Err Msg: The MCI Device Required to Play This...

 Last reviewed: November 22, 1994
Article ID: Q120046
The following information applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11
  • Microsoft Windows Sound System version 2.0

SYMPTOMS

You receive the following error message when you try to select CD Audio from the Device menu of Media Player: 

   The MCI device required to play this file is currently not installed
   on your system. Media Player cannot play this file unless the device
   is installed.
-or-

You receive the following error message when you try to start the Music Box application from Windows Sound System: 

   The CD-Audio device driver is not installed.
In both cases, the CD-Audio device driver may have already been installed through Control Panel.

CAUSE

These errors occur if either of the following conditions exists:

  • The CD-Audio device driver is not installed.

    -or-

  • There is no audio CD in the CD-ROM drive.

RESOLUTION

To ensure that the CD-Audio driver is installed, run Control Panel and choose the Drivers icon. Check the list of Installed Drivers in the Drivers dialog box. If the [MCI] CD-Audio driver is not listed, choose the Add button and install the driver. If the driver does appear in the list, choose the Setup button to make sure that the CD-ROM drive is being detected.

If the CD-Audio driver appears to be properly installed, make sure that there is an audio CD in the CD-ROM drive.
