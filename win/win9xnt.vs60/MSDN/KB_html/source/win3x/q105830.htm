GP Fault in MS Mail "Work Offline" Dialog When Exiting WFWG

 Last reviewed: November 21, 1994
Article ID: Q105830
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

If the "Would you like to work offline?" dialog box is open when you exit Windows for Workgroups, Microsoft Mail causes a general protection (GP) fault.

STATUS

Microsoft has confirmed this to be a problem in version 3.1 and version 3.11 of Microsoft Windows for Workgroups Mail. We are researching this problem and will post new information here as it becomes available.
