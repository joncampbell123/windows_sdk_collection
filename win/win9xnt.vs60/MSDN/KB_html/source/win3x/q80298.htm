AST Premium Exec Notebook May Need Reverse Video

 Last reviewed: November 22, 1994
Article ID: Q80298
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

When you use the AST Premium Exec Notebook 386/25 and Microsoft Windows, the LCD display is easier to read if it is set to reverse video.

To set reverse video on the AST Premium Exec Notebook, hold down the FUNCTION key on the lower-left side of the keyboard and press the R key.

MORE INFORMATION
