TOSHIBA.DRV Causes GP Fault

 Last reviewed: February 8, 1995
Article ID: Q94558
The following information applies to:

- Microsoft Windows operating system versions 3.1, 3.11 - Microsoft Windows for Workgroups versions 3.1

SUMMARY

The Toshiba P1351 printer driver (TOSHIBA.DRV), which is included in the TOSH24.EXE file included in the Windows Driver Library (WDL), may cause a general protection (GP) fault in COMMDLG.DLL when you first run or print from the Windows Write application.

To correct this problem, use the TOSHIBA.DRV file included with Microsoft Windows and Windows for Workgroups (WFWG) version 3.1.

MORE INFORMATION

The following is a list of TOSHIBA.DRV versions and dates: 

Filename                      Version   Date


TOSHIBA.DRV (WDL)             3.0       10-31-90
TOSHIBA.DRV (Windows 3.1)     1.1       3-10-92
TOSHIBA.DRV (WFWG)            1.1       10-1-92
