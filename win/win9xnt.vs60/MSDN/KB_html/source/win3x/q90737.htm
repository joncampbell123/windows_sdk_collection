Changes to Windows Driver Library - Miscellaneous

 Last reviewed: April 17, 1995
Article ID: Q90737
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

Below is a summary of the changes to the miscellaneous drivers in the Microsoft Windows Driver Library (WDL). The WDL offers support for many printers, displays, and sound cards not provided with Microsoft Windows operating system version 3.1.

To see WDL.TXT, which contains a complete listing of the WDL's contents and instructions for obtaining the WDL, query on "Q82006" (without the quotation marks) in the Microsoft Knowledge Base. 

                 WINDOWS DRIVER LIBRARY CHANGE LOG
                 =================================
********************************************************************** 
*                          MISCELLANEOUS                             *
**********************************************************************

Access Pack for Microsoft Windows -

Software for Persons with Disabilities (ACCP.EXE)

03/12/93 - This file has been updated on the WDL to remove unnecessary 

           files.
02/09/93 - This file has been updated on the WDL to correctly rename 
           ACCESS.TXT to INSTALL.TXT. The correct ACCESS.TXT file
           has also been added to this file.
07/10/92 - THE DRIVERS HAVE NOT CHANGED. A new license agreement is 
           included.

Arches Wallpaper (ARCHES.EXE)

10/27/92 - ARCHES.EXE is new to the WDL. This file contains the Arches 

           wallpaper that ships with Windows 3.1. It is being included
           in the WDL because it did not ship with Windows for
           Workgroups 3.1.

C & D Printer Fonts (CDFONT.EXE)

07/10/92 - THE FONTS HAVE NOT CHANGED. A new license agreement is 

           included.

Chitz Wallpaper (CHITZ.EXE)

10/27/92 - CHITZ.EXE is new to the WDL. This file contains the Chitz 

           wallpaper that ships with Windows 3.1. It is being included
           in the WDL because it did not ship with Windows for
           Workgroups 3.1.

Chord Sound File (CHORD.EXE)

10/27/92 - CHORD.EXE is new to the WDL. This file contains the Chord 

           sound file that ships with Windows 3.1. It is being
           included in the WDL because it did not ship with Windows
           for Workgroups 3.1.

GlidePoint Trackpad 9.00a (GLIDE.EXE)

04/17/95 - GLIDE.EXE has been added to the WDL. This driver supports the 

           GlidePoint TrackPad pointing device.

Marble Wallpaper (MARBLE.EXE)

10/27/92 - MARBLE.EXE is new to the WDL. This file contains the Marble 

           wallpaper that ships with Windows 3.1. It is being included
           in the WDL because it did not ship with Windows for
           Workgroups 3.1.

Microsoft CD-ROM Extensions (CDEXT.EXE formerly MMEXT.EXE)

07/10/92 - THE CD-ROM EXTENSIONS WERE NOT UPDATED. A new license 

           agreement is included.
04/20/92 - This file corrects random problems that users were 
           encountering with the file that was originally uploaded on
           the WDL on 4/6/92. Customers should use the CDEXT.EXE file
           that was uploaded on 4/20/92 or later. The earlier MSCDEX
           file that was distributed was corrupted, even though it had
           the same date and size as the correct file. (They did not
           binary compare).
04/16/92 - This file was renamed from MMEXT.EXE to CDEXT.EXE to reduce 
           the confusion about its contents (CD-ROM Extensions). NO
           CHANGE WAS MADE TO THE CONTENTS.

Microsoft Mail Help (MMHELP.EXE)

10/27/92 - MMHELP.EXE is new to the WDL. This file contains a more 

           comprehensive Microsoft Mail Help file than the one that was
           shipped with Windows for Workgroups version 3.1.

Microsoft Schedule+ Help (SPHELP.EXE)

10/27/92 - SPHELP.EXE is new to the WDL. This file contains a more 

           comprehensive Microsoft Schedule+ Help file than the one
           that was shipped with Windows for Workgroups version 3.1.

Windows for Workgroups Accessory Files (WFWADD.EXE)

05/20/93 - WFWEXT.EXE is new to the WDL. This file contains the same 

           accessory files included in previous versions of Windows
           but not included in Windows for Workgroups. The following
           files are contained in WFWADD.EXE:

              ARCHES.BMP, CHITZ.BMP, CHORD.WAV, MARBLE.BMP, MSDOS.EXE,
              REVERSI.EXE, REVERSI.HLP, SSMYST.SCR, TADA.WAV, TARTAN.BMP

MONOUMB.386 (MONO.EXE)

07/10/92 - MONOUMB.386 WAS NOT UPDATED. A new license agreement is 

           included.

MS-DOS Executive (MSEXE.EXE)

08/12/92 - The version that is included in the MSEXE.EXE dated 8/12/92 

           is the same as the version that was shipped with Windows
           3.0. This version is provided for backward compatibility
           only. The version previously distributed on the WDL was
           corrupt.
07/10/92 - THE MS-DOS EXECUTIVE WAS NOT UPDATED. A new license 
           agreement is included.

Mystify Screen Saver (SSMYST.EXE)

10/27/92 - SSMYST.EXE is new to the WDL. This file contains the 

           Mystify screen saver that ships with Windows 3.1. It is
           being included in the WDL because it did not ship with
           Windows for Workgroups 3.1.

Novell NetWare Upgrade (NOVELL.EXE)

07/10/92 - NOVELL NETWARE WAS NOT UPDATED. A new license agreement is 

           included.

Reversi (REVERS.EXE)

07/10/92 - REVERSI WAS NOT UPDATED. A new license agreement is 

           included.

SYSINI.WRI AND WININI.WRI (WRI.EXE)

07/10/92 - SYSINI.WRI AND WININI.WRI WERE NOT UPDATED. A new license 

           agreement is included.

Tada Sound File (TADA.EXE)

10/27/92 - TADA.EXE is new to the WDL. This file contains the Tada 

           sound file that ships with Windows 3.1. It is being
           included in the WDL because it did not ship with Windows
           for Workgroups 3.1.

Tartan Wallpaper (TARTAN.EXE)

10/27/92 - TARTAN.EXE is new to the WDL. This file contains the Tartan 

           wallpaper that ships with Windows 3.1. It is being included
           in the WDL because it did not ship with Windows for
           Workgroups 3.1.

Visionetics MPEG Master 3.1 (VISVIDEO.EXE)

04/17/95 - VISVIDEO.EXE has been added to the WDL. This driver supports 

           the Visionetics MPEG Master video playback device.

VPD.386 (VPD386.EXE)

07/10/92 - VPD.386 WAS NOT UPDATED. A new license agreement is 

           included.

Wacom ArtPad Tablet 1.0 (ARTPAD.EXE)

12/05/94 - ARTPAD.EXE has been added to the WDL. This driver supports

Wacom ArtZ Tablet 1.0 (ARTZ.EXE)

           the Wacom ArtZ digitizer tablets.
