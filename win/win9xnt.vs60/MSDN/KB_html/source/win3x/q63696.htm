Hardware Options Offered When Installing Windows

 Last reviewed: November 21, 1994
Article ID: Q63696
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

This article lists the hardware options offered during the installation of Microsoft Windows version 3.0 or 3.0a. If the Setup program offered any other choices for hardware, it is not Microsoft Windows but an OEM (original equipment manufacturer) version of Windows; additional support must be obtained from the manufacturer.

MORE INFORMATION

The following choices are listed during the Setup process.
