Display Problems Solved by the Latest WDL Drivers

 Last reviewed: November 9, 1995
Article ID: Q87082
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11 

                                  SUMMARY
                                  =======
Below is a summary of the problems solved by the latest drivers available on the Microsoft Windows Driver Library (WDL). The WDL provides many drivers for printers, displays, sound cards, and network card adapters not provided with Windows 3.1.

For a complete listing of the WDL's contents and instructions for obtaining the WDL, query on "Q82006" (without the quotation marks) in the Microsoft Knowledge Base. 

                             MORE INFORMATION
                             ================

DISPLAY PROBLEMS SOLVED BY WDL DISPLAY DRIVERS DATED 5/6/92 or 6/15/92

Symptom                             Known Problem On
-------                             ----------------

Incorrect TrueType font drawing.    ATI Ultra, DGIS, S3, RGDI

Setup crashes when switching to     ATI Ultra, S3, RGDI
the Windows portion during Express or Custom Install or Upgrade. 

Windows will not run at the         ATI Ultra, S3, RGDI
desired video resolution. 

Cannot run MS-DOS-based             ATI Ultra, S3, RGDI
applications in 386 enhanced mode. 

No mouse support for                ATI Ultra, DGIS, S3, ET4000, RGDI
MS-DOS-based application windows. 

No font selection for MS-DOS        ATI Ultra, DGIS, S3, ET4000, RGDI
application windows.
