Ground Hog Graphics VGA Card Screen Capture Problem

 Last reviewed: November 22, 1994
Article ID: Q68457
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

You cannot copy graphics from MS-DOS applications to the Clipboard when you use the Ground Hog Graphics 1 MB Shadow VGA card with Microsoft Windows version 3.0. Pressing either ALT+PRINT SCREEN or PRINT SCREEN does NOT copy VGA, EGA, or CGA graphics to the Clipboard. You can capture text to the Clipboard.

This problem occurs with all Ground Hog drivers, regardless of the video mode.

CAUSE

This problem occurs because of a limitation of the driver.

WORKAROUND

Use the standard Microsoft VGA driver.

STATUS

Graphics is researching this limitation and is working on solutions to this problem. Any questions regarding updates to the drivers should be made to Ground Hog Graphics.
