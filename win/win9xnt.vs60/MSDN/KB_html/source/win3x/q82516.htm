DEC Pathworks System Error with Windows 3.1

 Last reviewed: November 21, 1994
Article ID: Q82516
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

If you are using Microsoft Windows operating system version 3.1 on workstation A, and workstation B boots up to the Digital Equipment Corporation (DEC) Pathworks network with the same node name, a system error occurs on workstation A.

The error dialog box allows only the Cancel button to be used. When Cancel is chosen, a general protection (GP) fault occurs in the USER.EXE file.

Workstation B does not connect to the network.
