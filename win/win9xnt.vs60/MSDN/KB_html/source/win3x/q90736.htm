Changes to Windows Driver Library - Audio

 Last reviewed: November 22, 1994
Article ID: Q90736
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

Below is a summary of the changes to the audio drivers in the Microsoft Windows Driver Library (WDL). The WDL offers support for many printers, displays, and sound cards not provided with Microsoft Windows operating system version 3.1.

To see WDL.TXT, which contains a complete listing of the WDL's contents and instructions for obtaining the WDL, query on "Q82006" (without the quotation marks) in the Microsoft Knowledge Base. 

                 WINDOWS DRIVER LIBRARY CHANGE LOG
                 =================================
********************************************************************** 
*                              AUDIO                                 *
**********************************************************************

Artisoft Sounding Board Driver (ARTIS.EXE)

06/19/92 - The Artisoft Sounding Board driver has been updated to 

           provide improved performance and quality. It is recommended
           that you upgrade your driver if you are using previously
           released versions of this driver.

           A message was added to the self-extracting .EXE file to
           remind customers NOT to run the .EXE file in the Windows
           directory or the Windows SYSTEM subdirectory. A new license
           agreement is also included.

Creative Sound Blaster Pro/Pro Basic (CREAT.EXE)

08/19/93 - CREAT.EXE has been deleted from the WDL.

06/19/92 - THE DRIVERS HAVE NOT BEEN CHANGED. 

           A message was added to the self-extracting .EXE file to
           remind customers NOT to run the .EXE file in the Windows
           directory or the Windows SYSTEM subdirectory. A new license
           agreement is also included.
06/02/92 - No changes were made to any of the Creative Sound Blaster 
           Pro drivers. The only change occurred in CREATIVE.TXT.
05/28/92 - The drivers in CREAT.EXE have been updated to be more 
           stable. The main changes are to MIDI. The Mixer has also
           been improved from the earlier WDL version of this driver. All
           buttons and features, such as Save Settings, now function
           correctly. CREATIVE.TXT includes more information on
           installing and using these drivers.

IBM M-Audio Adapter (IBM.EXE)

06/19/92 - THE DRIVERS HAVE NOT BEEN CHANGED. 

           A message was added to the self-extracting .EXE file to
           remind customers NOT to run the .EXE file in the Windows
           directory or the Windows SYSTEM subdirectory. A new license
           agreement is also included.

PC Speaker Driver (SPEAK.EXE)

06/19/92 - THE DRIVERS HAVE NOT BEEN CHANGED. 

           A message was added to the self-extracting .EXE file to
           remind customers NOT to run the .EXE file in the Windows
           directory or the Windows SYSTEM subdirectory. A new license
           agreement is also included.
06/02/92 - This version of SPEAKER.DRV contains a workaround for 
           machines (specifically AST, but there may be others) that
           implement speaker ports in a nonstandard way. Specifically, both
           PPI bits are now toggled on and off, rather than just the
           speaker bit.
05/06/92 - No change was made to the driver or supporting files. 

           This file was replaced because a .TXT file needed to be
           changed.

Pro AudioSpectrum Audio Adapters (PROAUD.EXE)

05/14/92 - PROAUD.EXE has been removed from the Windows Driver 

           Library. These drivers have been found to hang systems
           when running multimedia-intensive tasks (for example,
           animation, video, and sound) at the same time.

Thunderboard (THUNDR.EXE)

06/19/92 - THE DRIVERS HAVE NOT BEEN CHANGED. 

           A message was added to the self-extracting .EXE file to
           remind customers NOT to run the .EXE file in the Windows
           directory or the Windows SYSTEM subdirectory. A new license
           agreement is also included.
