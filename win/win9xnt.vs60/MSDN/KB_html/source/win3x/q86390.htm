Aldus PageMaker 4.0 Stack Fault w/ Aldus Clipart Import Filter

 Last reviewed: December 9, 1994
Article ID: Q86390
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

You may receive the following error message if you use Aldus PageMaker version 4.0 with Microsoft Windows version 3.1: 

   PM4 has caused a Stack Fault in
   module ARTIMP.FLT at 0005:000E.

CAUSE

The Clipart Import filter that comes with PageMaker 4.0 causes this problem.

RESOLUTION

Aldus Technical support has confirmed that the import filter (ARTIMP.FLT, size 130432, date 2/8/91) needs to be upgraded to work with Windows 3.1.

An updated version of this filter is available from Aldus. Ask for Filter Pack #5.

MORE INFORMATION

Once this error has occurred, you should exit Windows because the system resources will drop dramatically.
