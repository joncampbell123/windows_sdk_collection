WFWG 3.11: Hardware Compatibility List - Modems for RAS

 Last reviewed: November 22, 1994
Article ID: Q107374
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SUMMARY

This article contains a list of modems that have been tested and certified by Microsoft to be compatible with Windows for Workgroups 3.11 when using the Remote Access Services. Microsoft will add additional information to this article as it becomes available.

NOTE: This list is independent of the Microsoft At Work PC Fax supported modems list.
