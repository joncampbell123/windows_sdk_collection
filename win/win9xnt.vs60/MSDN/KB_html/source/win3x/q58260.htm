Mouse Lost Under IBM DisplayWrite IV

 Last reviewed: November 23, 1994
Article ID: Q58260
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

If you run IBM DisplayWrite IV under Microsoft Windows 3.0, your mouse may not work when you return to Windows.

CAUSE

This is a known problem and is due to DisplayWrite IV reprogramming the mouse.

WORKAROUND

No workaround is available at this time.
