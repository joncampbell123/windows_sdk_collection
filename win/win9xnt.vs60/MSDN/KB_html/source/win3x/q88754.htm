RM/COBOL-85 Is Incompatible with SMARTDrive 4.0

 Last reviewed: November 23, 1994
Article ID: Q88754
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

According to Liant Software, SMARTDrive (SMARTDRV.EXE) version 4.0 provided with Microsoft Windows 3.1 is not compatible with RM/COBOL-85's file system and should not be used with it.

WORKAROUND

To work around this problem, use the SMARTDrive version packaged with Windows 3.0 or MS-DOS 5.0.

MORE INFORMATION

Liant Software, who has purchased the rights to Ryan McFarland's RM/COBOL-85 program development software, has confirmed that using SMARTDrive with the RM/COBOL file system may cause data loss.

For more information, contact Liant Software.
