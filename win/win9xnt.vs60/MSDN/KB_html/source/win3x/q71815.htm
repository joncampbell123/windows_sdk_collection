Setting PSCRIPT.DRV to Print to an EPS File Isn't Saved

 Last reviewed: November 23, 1994
Article ID: Q71815
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

Setting the Microsoft Windows 3.0 PostScript drivers (versions 3.3 and 3.4) to print to an encapsulated PostScript (EPS) file lasts only for that instance of Windows. When you exit Windows, the driver is reset to the original printer setting.

STATUS

Microsoft has confirmed this to be a problem in Windows version 3.0.
