Color Support for NEC Pinwriter Printers in Windows 3.0

 Last reviewed: November 23, 1994
Article ID: Q63664
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

NEC has a Microsoft Windows version 3.0 driver available for its Pinwriter printers that supports color as well as more internal fonts. The current driver that is packaged with Windows 3.0 supports a limited number of fonts and no color.

The new driver can be downloaded from NEC's bulletin board service.
