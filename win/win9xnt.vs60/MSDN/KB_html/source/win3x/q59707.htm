PIF Editor Advanced Button Not Working

 Last reviewed: November 23, 1994
Article ID: Q59707
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

If you run Microsoft Windows version 3.0 PIF Editor, choose the Advanced button, and nothing happens, you may not have enough memory to run the application. The Advanced button is disabled at this point.

The Advanced button does not become available automatically by freeing memory. You must close the PIF Editor, free memory by closing other applications, and rerun the PIF Editor.
