Windows for Workgroups Fails If Installed on DoubleDisk

 Last reviewed: November 23, 1994
Article ID: Q90211
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.1
  • Microsoft Workgroup Connections version 1.0

SYMPTOMS

If you attempt to use Windows for Workgroups or Workgroup Connections with Vertisoft System's DoubleDisk disk compression program, you receive bad or missing file error messages when you restart the computer after installation.

CAUSE

Windows for Workgroups and Workgroup Connections are not compatible with the DoubleDisk versions 1.x and 2.x.

WORKAROUND

There is no workaround at this time. You must install Windows for Workgroups on an uncompressed drive.
