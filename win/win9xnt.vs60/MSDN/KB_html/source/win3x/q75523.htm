Norton Desktop and Novell NetWare Require 9/20/90 or Later

 Last reviewed: May 1, 1997
Article ID: Q75523
3.00 3.00a WINDOWS kbnetwork kb3rdparty

The information in this article applies to:

  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

Norton Desktop for Windows (NDW) requires that if you are running on a Novell Network, you must have VIPX.386, VNETWARE.386, and NWPOPUP.EXE with a file date of 9/20/90 or later.

MORE INFORMATION

The file date of 9/20/90 is from the Microsoft Windows Supplemental Driver Library (SDL). If you do not have the SDL, you will need to obtain these files by ordering Windows version 3.0a or the Microsoft Windows Supplemental Driver Library (SDL). To order one of these products, call Microsoft at (425) 637-7098.
