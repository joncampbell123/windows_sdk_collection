No Menu Item Highlights Visible

 Last reviewed: November 23, 1994
Article ID: Q59682
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

If you use the Control Panel's Color facility to set your menus to white text on a black background, no highlight is visible when you select a menu item.

This problem occurs because the highlight color is always white text with a black background. You need to change the HighlightText color settings in the WIN.INI file.

MORE INFORMATION

Control Panel does not support the highlight color setting. You must manually change the setting in the WIN.INI. This procedure and the settings themselves are documented in the WININI.TXT readme text file.
