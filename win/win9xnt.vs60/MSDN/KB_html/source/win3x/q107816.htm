WFWG 3.11 Setup Installs Older Logitech Mouse Driver

 Last reviewed: November 1, 1995
Article ID: Q107816
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

Your Logitech three-button mouse loses some functionality after you install Windows for Workgroups version 3.11.

CAUSE

The Logitech Mouse driver supplied with Windows for Workgroups 3.11 is version 6.02. Windows for Workgroups 3.11 Setup overwrites newer versions of this driver with version 6.02.

RESOLUTION

To correct this situation, reinstall the latest version of the Logitech Mouse software.
