Identifying Third-Party Mouse by FCC ID Number

 Last reviewed: November 30, 1994
Article ID: Q93266
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11
  • Microsoft MS-DOS operating system versions 3.x, 4.x, 5.x, 6.x
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

The following is a list of FCC ID (first three digits) numbers and the mice that use those FCC ID numbers. This can help in determining which Windows or MS-DOS mouse driver to use.

The FCC ID number can usually be found on the bottom of the mouse.
