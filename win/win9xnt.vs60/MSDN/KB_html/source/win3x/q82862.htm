Poor Print Quality in Taxview 1991

 Last reviewed: November 21, 1994
Article ID: Q82862
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

Under Microsoft Windows version 3.1, the printing quality when using Taxview for Windows (retail and professional versions) is very poor if the files are printed at in high resolution.

Changing the printer resolution to the lowest possible setting may correct this problem.

MORE INFORMATION

If you are printing with a Hewlett-Packard (HP) LaserJet series printer, be sure to install the HP soft fonts when installing Turbo Tax. Then add the soft fonts to Windows by starting the Control Panel and choosing the Printers icon.

For HP LaserJet printers, the lowest resolution is 75 dots-per-inch (dpi).
