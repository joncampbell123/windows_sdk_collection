Creating Printer Drivers for Windows 3.0

 Last reviewed: May 30, 1996
Article ID: Q64527
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

The following companies can write printer drivers for Windows 3.0 on a contract basis: 

   GDT Softworks Inc. at (800) 663-6222 (Ask for PC Development)

   Paradise Software at (800) 367-5600 (Ask for PC Development)
Note that you will need both the Windows SDK (Software Development Kit) and the Windows DDK (Device Development Kit) to write printer drivers. A retail version of Windows 3.0 is also required.
