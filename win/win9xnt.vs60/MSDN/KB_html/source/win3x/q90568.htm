Using Calcomp Drawing Board/Pad Series II with Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q90568
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

Calcomp Drawing Pad or Drawing Board Series II Setup program may produce the following error when you try to install drivers in Windows 3.1: 

   Cannot find SETUP.INF.

CAUSE

Calcomp technical support has confirmed that the Calcomp Drawing Pad or Drawing Board Series II graphics tablets require driver setup version 1.5 or later to work properly with Windows 3.1.

The error message indicates that the version of the installation program is earlier than 1.5 and was designed for Windows 3.0. According to Calcomp technical support, version 1.5 allows the successful installation of the drivers in Windows 3.1. The version number can be found on the Calcomp driver installation disk label.

WORKAROUND

To work around this problem contact for an updated driver installation disk.
