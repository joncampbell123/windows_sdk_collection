Windows: A File Cannot Be Automatically Loaded into Clipboard

 Last reviewed: November 23, 1994
Article ID: Q75190
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

A file cannot be automatically loaded into Clipboard when you run the Clipboard icon.

MORE INFORMATION

Step 6 on page 91 of the "Microsoft Windows 3.0 User's Guide" explains how include a document with a program item. However, this information does not apply to the Clipboard.
