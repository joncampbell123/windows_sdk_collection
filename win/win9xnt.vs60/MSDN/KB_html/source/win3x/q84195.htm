Hardware Options When Installing Windows 3.1

 Last reviewed: November 21, 1994
Article ID: Q84195
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

This article lists the hardware options offered during the installation of Microsoft Windows operating system version 3.1. If the Setup program offered any other choices for hardware, then you have a OEMx.INF file located in your WINDOWS\SYSTEM directory, or you are not using Microsoft Windows but an OEM (original equipment manufacturer) version of Windows. Additional support must be obtained from the manufacturer.
