List of TrueType Fonts from Microsoft Retail Font Packages

 Last reviewed: November 21, 1994
Article ID: Q103755
This information in this article applies to:
  • Microsoft TrueType Font Pack for Windows
  • Microsoft TrueType Font Pack 2 for Windows
  • Microsoft Hewlett-Packard Font Set
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

This article contains a list of the TrueType fonts supplied with each of the following Microsoft TrueType font packages: 

   Microsoft TrueType Font Pack for Windows

   Microsoft TrueType Font Pack 2

   Microsoft Hewlett-Packard Font Set
