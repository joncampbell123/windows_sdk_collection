Previous Button Dimmed in Connect Net Drive Dialog Box

 Last reviewed: November 21, 1994
Article ID: Q71207
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

The Previous button in the Connect Network Drive dialog box in File Manager is used to quickly access network drive connections (made in prior sessions from File Manager).

This button may be dimmed and unavailable if the WINFILE.INI is corrupted or damaged. There may not be any other sign that this corruption has taken place.

WORKAROUND

Delete the WINFILE.INI and exit File Manager. The WINFILE.INI will automatically be rewritten.
