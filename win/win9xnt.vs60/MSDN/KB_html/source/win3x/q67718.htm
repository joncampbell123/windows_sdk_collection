Fatal Error When Running DayBook

 Last reviewed: December 4, 1995
Article ID: Q67718
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

If you experience a fatal error when you run DayBook, it may have been caused by a corruption of the DAYBOOK.TBK file's script. If you have data in the DAYBOOK.TBK file that you would like to recover, use the following procedure:

  1. Rename DAYBOOK.TBK to OLDBOOK.TBK.

  2. Reinstall ToolBook.

  3. From the File menu, choose Merge DayBook Data

  4. Type OLDBOOK.TBK
