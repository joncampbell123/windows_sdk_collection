Calendar: Starting Time Not Displayed in Upper-Left Corner

 Last reviewed: November 23, 1994
Article ID: Q105293
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

On page 428, the Windows 3.1 "User's Guide" states that the Starting Time set in the Day Settings window will specify the time that appears at the upper left of the Calendar window when you first view an appointment day.

This applies only when the Starting Time is less than or equal to one of the following:

8:30 PM (Interval = 15, Hour Format = 12)

5:00 PM (Interval = 30, Hour Format = 12)

10:00 AM (Interval = 60, Hour Format = 12)

20:30 (Interval = 15, Hour Format = 24)

17:00 (Interval = 30, Hour Format = 24)

10:00 (Interval = 60, Hour Format = 24)
