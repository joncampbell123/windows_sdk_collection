Keytronics 122-Key Keyboard Compatible with Windows

 Last reviewed: November 22, 1994
Article ID: Q73604
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

The Keytronics 122-key keyboard is designed for 3270 emulation. It is compatible with Microsoft Windows 3.0 or 3.1 if it is installed as an "Enhanced 101 or 102 key US and Non US keyboards". However, you cannot use the additional keys when running Windows.

MORE INFORMATION

For more information, call Keytronics Technical Support.
