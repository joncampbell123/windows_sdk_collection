Simply Accounting Needs Update to Work with Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q81984
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

Using CA Simply Accounting may cause your system to stop responding (hang) when you attempt to post a transaction in the payroll journal.

CAUSE

Initial testing of Microsoft Windows version 3.1 has shown that CA Simply Accounting for Windows version 1.0 may not work reliably with Windows 3.1.

RESOLUTION

To successfully use CA Simply Accounting with Windows 3.1, use CA Simply Accounting version 1.0f.

MORE INFORMATION

For more information, please contact CA Simply Accounting.
