Banyan Err Msg: Sorry, There Is No Descriptive Text for This..

 Last reviewed: November 21, 1994
Article ID: Q86366
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

When running Microsoft Windows version 3.1 on a Banyan Vines network, you may receive the following error message: 

   VINES error 84.

   Sorry, there is no descriptive text for this error.

CAUSE

This error is caused by trying to connect to a network resource to which you are all ready connected.
