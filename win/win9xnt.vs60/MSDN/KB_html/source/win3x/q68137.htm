Calculator Err Msg: Result Is Too Large for Display

 Last reviewed: November 23, 1994
Article ID: Q68137
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

In Microsoft Windows version 3.0, the following Calculator error message appears differently in Calculator's standard mode and Calculator's scientific mode: 

   Result is too large for display
In Calculator's standard mode, the above message is truncated to read as follows: 

   Result is too large for

CAUSE

The error message is too large for Calculator's display.

STATUS

Microsoft has confirmed this to be a problem in version 3.0. This problem does not occur in later versions of Windows.
