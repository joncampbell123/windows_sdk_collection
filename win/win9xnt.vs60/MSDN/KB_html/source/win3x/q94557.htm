Write Restores Subscripts to Incorrect Font Size

 Last reviewed: November 23, 1994
Article ID: Q94557
The following information applies to:
  • Microsoft Windows Operating System versions 3.0, 3.0a, 3.1, 3.11
  • Microsoft Windows For Workgroups versions 3.1, 3.11

SYMPTOMS

When Microsoft Windows or Windows for Workgroups Write, the font point size may be set incorrectly when restoring from a subscript mode to the original state.

WORKAROUND

To work around this problem:

  1. Select the incorrectly sized text.

  2. From the Character menu in Write, choose Font to manually reset the font to its proper size.

MORE INFORMATION

This problem applies to all font types (TrueType, system, and so on) set at font point sizes 4-127, with the exception of point sizes 6, 8, 10, 12, 14, 24, 26, 36, 72, and 127.

STATUS

Microsoft has confirmed this to be a problem in Write. We are researching this problem and will post new information here as it becomes available.
