Data World NB325 Cannot Run MS-DOS Prompt in Enhanced Mode

 Last reviewed: November 23, 1994
Article ID: Q85067
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

You may experience problems running MS-DOS Prompt under Microsoft Windows version 3.1 in 386 enhance mode on a Data World NB325 computer. This problem does not occur in Windows 3.0, or Windows 3.1 in standard mode.

CAUSE

Data World has confirmed this to be a problem with the Data World NB325. For more information, please contact Data World.
