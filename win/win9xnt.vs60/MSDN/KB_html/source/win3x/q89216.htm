MicroRegister Can Cause Windows 3.1 to Stop

 Last reviewed: November 23, 1994
Article ID: Q89216
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

If you run a version of Micro Methods' MicroRegister earlier than 7.54 before starting Microsoft Windows 3.1, Windows may stop responding (hang).

CAUSE

According to Micro Methods, this is a confirmed problem and is corrected in versions 7.54 and later.

WORKAROUND

Reset or restart your computer before you start Windows.

MORE INFORMATION

MicroRegister is a point-of-sale software package that tracks inventory and produce accounts receivable reports. It can be used in place of a cash register, allowing you to print receipts for customers.
