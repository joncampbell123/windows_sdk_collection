Using MS-DOS Applications on HP Vectra in Standard Mode

 Last reviewed: November 22, 1994
Article ID: Q66357
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

If your Hewlett-Packard (HP) Vectra is running in standard mode in Windows 3.0, you cannot use a hot key to move to an MS-DOS application. If you attempt to do this, the screen becomes blank and your system stops responding (hangs).

STATUS

Microsoft has confirmed this to be a problem in Windows version 3.0. We are researching this problem and will post new information here as it becomes available.
