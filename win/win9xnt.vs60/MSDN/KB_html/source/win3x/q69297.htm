Windows Err Msgs: Paintbrush Message List

 Last reviewed: November 23, 1994
Article ID: Q69297
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

Below is a list of the possible error messages you may receive from the Paintbrush application (PBRUSH.EXE) included with Microsoft Windows version 3.00.

NOTE: The "(null)" and "0%" symbols in a message indicate a variable string used by the application.

MORE INFORMATION

  1. (null) already exists; choose another name using File Save As.

  2. (null) is not a valid filename; check to ensure the filename has no more than 8 characters followed by a period and a 3-letter extension.

  3. (null) was created using a previous version of Windows; Paintbrush cannot load this file.

  4. Cannot copy image; select smaller image or increase available memory by closing one or more Windows applications.

  5. Cannot copy to or paste from the Clipboard because another application is using the Clipboard; wait and try again.

  6. Cannot find file (null); check to ensure the path and filename are correct.

  7. Cannot paste; run the Clipboard application to ensure that the Clipboard contains a bitmap.

  8. Cannot print; check to ensure your printer is connected and properly set up or close Windows applications to increase available memory.

  9. Image requested is too large; Paintbrush is now reducing the image height.

  10. Insufficient disk space to edit image; delete one or more files from drive (null) and try again.

  11. Insufficient memory for this operation

  12. Insufficient memory for this operation; close one or more Windows applications to increase available memory.

  13. Insufficient memory to complete operation; close one or more Windows applications to increase available memory and try again.

  14. Insufficient memory to edit image; close one or more Windows applications to increase available memory and try again.

  15. Insufficient memory to perform this operation; close one or more Windows applications to increase available memory and try again.

  16. Insufficient memory to print; close one or more Windows applications to increase available memory and try again.

  17. Insufficient memory to run program; close one or more Windows applications to increase available memory and try again.

  18. Invalid height; check to ensure the entry is a positive number.

  19. Invalid number of copies; check to ensure the entry is a positive number.

  20. Invalid number; check to ensure the margin value is a positive number.

  21. Invalid scaling; check to ensure the entry is at least 0% and at most 0%.

  22. Invalid width; check to ensure the entry is a positive number.

  23. Overwrite existing (null)?

  24. Pasting from MSP files is not supported; load MSP file and save as a BMP file.

  25. Save current image (null)?

  26. The printer that was selected for printing in Paintbrush has been changed or deleted; the current default printer will be selected.

  27. This is an old format; translate to BMP format?

  28. Unable to open (null) because the format is unrecognized; use a backup copy of (null) or recreate the file.

  29. Unable to open (null); check to ensure the path and filename are correct.

  30. Unable to save file (null); close one or more files in another Windows application and try again.

  31. Unable to save file (null); delete one or more files to increase available disk space and try again.

  32. Unable to save to a read-only file; choose a different filename.

  33. Unsupported file format.

  34. Warning: The image you are about to paste will replace the image you are editing; do you want to continue?

  35. Your margin settings leave no room for the image; adjust margins to increase available space.
