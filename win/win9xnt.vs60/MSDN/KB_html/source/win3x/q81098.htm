Word for MS-DOS 5.0 PIF Settings Under Windows 2.x, 3.0

 Last reviewed: November 23, 1994
Article ID: Q81098
The information in this article applies to:
  • Microsoft Word for MS-DOS, version 5.0
  • Microsoft Windows operating system versions 2.x, 3.0, 3.0a

SUMMARY

The following are the recommended program information file (PIF) settings to run Microsoft Word 5.0 in text mode under Windows/286, Windows/386, and Windows 3.0. Include the full path and executable filename for the Program Name.

MORE INFORMATION

It may be necessary to run Word 5.0 in text mode to reduce Word to an icon when returning to Windows. To switch to text mode while in Word, select Options and set Display Mode to 1. 

   Field                           Setting
   -----                           -------

   Program Name:                   C:\WORD\WORD.EXE
   Program Title:                  Microsoft Word 5.0
   Program Parameters:             None
   Initial Directory:              C:\Word
   KB Required                     384
   KB Desired                      450

  *Directly Modifies               Screen
  *Program Switch                  Graphics/ Multiple Text

 **Usage Controls                  Full Screen
 **Program Switch                  N/A

   Screen Exchange                 Graphics/Text
   Close Window On Exit            Optional (see note below)

    * Windows/286 only
   ** Windows/386 only
NOTE: In Windows/286, if an MS-DOS-based application is run from a PIF file without the Close Window On Exit option selected, when the MS-DOS-based application is closed, an inactive application window remains open until it is closed. To close an inactive application window, press ALT+SPACEBAR and choose Close from the Control menu.
