Control Panel Timer Not Accurate to the Second

 Last reviewed: November 23, 1994
Article ID: Q50831
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

The Control Panel sets up a timer to check the current time and date periodically. When the timer activates, it reads the system clock and updates its display. There is no guarantee that the timer will activate in 1-second increments or that it will be synchronized with the system clock. The Control Panel display is accurate plus or minus a few seconds.
