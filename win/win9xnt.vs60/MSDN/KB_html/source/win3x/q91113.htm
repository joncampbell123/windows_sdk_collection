Changes to Windows Driver Library - Network Card Adapters

 Last reviewed: November 21, 1994
Article ID: Q91113
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

Below is a summary of the changes to the network card adapter drivers in the Microsoft Windows Driver Library (WDL). The WDL offers support for many printers, displays, and sound cards not provided with Windows version 3.1 or Windows for Workgroups.

To see WDL.TXT, which contains a complete listing of the WDL's contents and instructions for obtaining the WDL, query on "Q82006" (without the quotation marks) in the Microsoft Knowledge Base.
