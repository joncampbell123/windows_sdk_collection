Madison Avenue Requires DMABufferSize=64 Setting

 Last reviewed: November 22, 1994
Article ID: Q83584
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

If you run Madison Avenue by Eclipse Technologies within Microsoft Windows version 3.0, you must use the DMABufferSize=64 setting in the [386Enh] section of the SYSTEM.INI file.

MORE INFORMATION

Windows may generate an error message indicating that you should increase the DMABufferSize from 16 (the default setting) to 32; however, DMABufferSize actually needs to be increased to 64. Madison Avenue is a Multimedia MS-DOS application that can be run from within Windows.

For more information on Madison Avenue, contact Eclipse Technologies technical support.
