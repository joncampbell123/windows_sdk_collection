Cardfile Err Msg: No Modem Connected

 Last reviewed: November 23, 1994
Article ID: Q80174
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

When you attempt to use the Autodial feature in the Cardfile program included with Microsoft Windows versions 3.0 and 3.0a, the following error message may occur: 

   No Modem Connected

CAUSE

This message can indicate one of the following:

  • There is no modem connected.
  • A modem has not been installed correctly.
  • The modem that is connected is NOT Hayes-compatible.

MORE INFORMATION

Cardfile does not have the ability to automatically dial ("autodial") to a modem that is not Hayes-compatible. Cardfile has been hard coded to accept only Hayes-compatible strings.
