Control Panel Missing with Faxit for Windows

 Last reviewed: November 30, 1994
Article ID: Q48733
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

When you try to run the Microsoft Windows Control Panel when using "Faxit for Windows" by Alien Computing, you get the Faxit Program Manager. The Control Panel is not present.

CAUSE

When the Faxit program is installed, it renames CONTROL.EXE to WINCRTL.EXE, and FAXIT.EXE to CONTROL.EXE.

WORKAROUND

To access Windows Control Panel, run the WINCRTL.EXE program.

MORE INFORMATION

Refer to the Faxit user's guide for more information.
