Battery Watch Pro 3.0 Requires Upgrade for Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q85378
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

Traveling Software has confirmed that Battery Watch Pro version 3.0 is not compatible with Microsoft Windows version 3.1. Upgrade to Battery Watch Pro version 4.0 for use with Windows 3.1 running in 386 enhanced mode.

MORE INFORMATION

Battery Watch Pro (BW.COM) is a reverse time counter program used on laptop computers to keep track of battery life.

To determine the version of Battery Watch Pro, look at the BW.COM file. BW.COM version 3.0 is 35,904 bytes and dated 4/12/91.

To obtain Battery Watch Pro version 4.0, contact Traveling Software.
