Word Wrap in Notepad Causes UAE

 Last reviewed: November 23, 1994
Article ID: Q71942
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

Using the Word Wrap option, in Notepad for Windows version 3.0 may cause an Unrecoverable Application Error (UAE).

NOTE: This problem does not occur in later versions of Windows.

MORE INFORMATION

Steps to Reproduce Problem

  1. Open Notepad.

  2. From the Edit menu, choose Word Wrap.

  3. Type text of any length.

  4. Maximize Notepad.

  5. Press any key on the keyboard.

This problem occurs in both standard and enhanced modes. In real mode, Windows may not display the error message but your computer may stop responding (hang or lock up).
