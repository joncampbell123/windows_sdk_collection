Using the Sleep Button on COMPAQ Portables with Windows

 Last reviewed: November 22, 1994
Article ID: Q68209
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

Some COMPAQ portable computers have a Sleep button. Pressing this button temporarily suspends computer operations to save battery life. If you press this button while Microsoft Windows version 3.0 is running in the foreground, you may not get out of "sleep" mode without rebooting the computer. However, if you open an MS-DOS Prompt window, then press the Sleep button, the computer "sleeps" (and "wakes up"), just as the manual says it should.
