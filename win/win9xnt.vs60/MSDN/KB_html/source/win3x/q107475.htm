Total Number of Workgroups That Can Be Displayed

 Last reviewed: November 21, 1994
Article ID: Q107475
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

Microsoft Windows for Workgroups version 3.11 is limited to displaying 1,400 different workgroups simultaneously.
