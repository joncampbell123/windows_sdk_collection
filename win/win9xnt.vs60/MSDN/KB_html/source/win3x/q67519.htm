UAEs When Cutting and Pasting with Micrografx Designer 3.01

 Last reviewed: November 23, 1994
Article ID: Q67519
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

Intermittent UAEs (Unrecoverable Application Errors) and crashes may be experienced when you cut and paste with Micrografx Designer version 3.01 running with Windows 3.0.

MORE INFORMATION

According to Micrografx, this is a documented problem caused by metafile transfer. Micrografx has developed Designer version 3.02 to correct the problem. You can get Designer 3.02 as a free upgrade by calling Micrografx technical assistance.
