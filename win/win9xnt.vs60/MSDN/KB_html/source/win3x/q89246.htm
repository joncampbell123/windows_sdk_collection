Windows Err Msg: Unable to Connect to a Network Drive

 Last reviewed: November 21, 1994
Article ID: Q89246
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

You may get the error message: "Unable to connect to a Network drive" when running Microsoft Windows on 3Com 3+ Open network. This error occurs in File Manager because File Manager is unable to connect to shared drives outside of the current user's domain.

WORKAROUND

  • Connect to these drives before you start Windows.
  • Upgrade your 3Com network to LAN Manager 2.1.
