Distorted Mouse Pointer with 8515 Monitor and 8514/A

 Last reviewed: November 23, 1994
Article ID: Q68284
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

Your mouse pointer may appear slightly distorted when you use an 8515 monitor and 8514/A adapter with Microsoft Windows version 3.0.

WORKAROUND

Installing a mouse driver in your CONFIG.SYS or AUTOEXEC.BAT file can sometimes resolve this problem.

Windows uses the MOUSE.DRV (or similar .DRV) file to provide a mouse to Windows and Windows applications. Installing a mouse driver at the MS-DOS level can sometimes provide a more stable configuration.
