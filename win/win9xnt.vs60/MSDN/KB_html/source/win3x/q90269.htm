NetWare Printing in Windows for Workgroups

 Last reviewed: November 21, 1994
Article ID: Q90269
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

In general, Microsoft Windows for Workgroups Print Manager extends the printing capabilities from those in Microsoft Windows version 3.1. In the case of NetWare printers, the capabilities are much the same as in Windows 3.1. Only the jobs submitted by the user are shown, and jobs may be paused, resumed, or deleted. These actions can be done from the document menu or the appropriate buttons on the toolbar.
