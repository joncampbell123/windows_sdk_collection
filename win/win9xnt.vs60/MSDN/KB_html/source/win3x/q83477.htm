New Key Combinations for Edit Menu Commands in Windows 3.1

 Last reviewed: August 7, 1996
Article ID: Q83477
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11
  • Microsoft Windows 95
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

The Edit menu keyboard commands in the Microsoft Windows operating system version 3.1 differ from those available in Windows 3.0. These new key combinations were implemented to standardize keyboard commands in Windows-based applications.

Note: The Windows 3.0 keystrokes are still available in Windows 3.1 and work correctly.

MORE INFORMATION

The new key combinations correspond to four of the Edit menu commands. The old and new key combinations for each of these commands are listed in the following chart: 

   Command        Windows 3.0      Windows 3.1
   -------        -----------      -----------

   Undo           ALT+BACKSPACE     CTRL+Z
   Cut            SHIFT+DELETE      CTRL+X
   Copy           CTRL+INSERT       CTRL+C
   Paste          SHIFT+INSERT      CTRL+V
