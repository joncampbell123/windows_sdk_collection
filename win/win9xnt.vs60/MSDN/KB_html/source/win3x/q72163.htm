Use of Help in Windows File Manager

 Last reviewed: November 23, 1994
Article ID: Q72163
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

In the File Manager, if you display a directory of files and use Help when you return to the directory window, you cannot use the Print, Search, Move, Copy, Delete, Rename, or Change Attributes commands from the File menu.

WORKAROUND

Close the directory window, then open it again.

MORE INFORMATION

The following are the error messages that occur.

Move, Copy, Delete, and Rename

  • Cannot find file; check to ensure the path and filename are correct.

Print

  • Cannot Print
  • The file to be printed cannot be found.

Change Attributes

  • File System Error
  • Unable to change attributes

Search

  • No matching files were found
