Terminal Err Msg: Cannot Communicate with Modem

 Last reviewed: November 21, 1994
Article ID: Q61712
The information in this article applies to:
  • Microsoft Windows operating system versions 1.x, 2.x, 3.0, 3.0a, 3.1, 3.11

With certain Hayes-compatible modems (for example, Practical Peripherals), Microsoft Terminal connects but is unable to communicate. If Terminal does not release the communication port when you try to close Terminal, the following error message occurs: 

   Cannot Communicate with Modem
This prevents other communication packages (for example, DynaComm or CrossTalk) from connecting.

To end the connection, it must be physically broken. You should physically disconnect the phone and then reconnect.
