Kraft Trackball Mouse and Windows 3.0

 Last reviewed: November 22, 1994
Article ID: Q78167
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

According to Kraft Systems, the Kraft Trackball mouse is compatible with Windows 3.0.

For the mouse to work properly, adhere to the following guidelines:

  • Use only COM1 or COM2.
  • Set up the mouse as a Microsoft or IBM PS/2 mouse.
  • Use Microsoft mouse driver version 7.04 or 8.00.

MORE INFORMATION

You may experience a frozen mouse pointer when using the Kraft Trackball mouse with Windows 3.0. According to Kraft Systems technical support, this usually happens with machines that are equipped with a bus card. The only known workaround is to disable the bus card through a dip switch or jumper, or physically removing the card from the machine.
