Orchid Fahrenheit Drivers Cause GP Faults

 Last reviewed: November 23, 1994
Article ID: Q95297
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

If you use the Orchid Fahrenheit 1280 VGA card with Orchid's video drivers dated 8/10/92, the following problems may occur: 

  - Random general protection (GP) faults

  - Garbled output when exiting Windows

  - System stops (hangs) when entering Windows

RESOLUTION

Orchid has confirmed that these drivers may cause problems with all versions of Windows. They recommend that you upgrade to the latest versions of their BIOS and driver, which are available from Orchid's bulletin board service (BBS).

The new BIOS is version 3.0 and the corresponding driver is version 7.0.
