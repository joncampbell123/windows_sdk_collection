Setting Up a Dvorak Keyboard for Windows Version 3.0

 Last reviewed: July 8, 1997
Article ID: Q63739
3.00 3.00a WINDOWS

The information in this article applies to:

  • Microsoft Windows operating system versions 3.0, 3.0a

Do the following to set up a Dvorak keyboard with Microsoft Windows version 3.0:

  1. From the Program Manager Main group, select the Control Panel Icon. (The keyboard command is ALT, W. Choose the Main group; use the arrow keys to select the Control Panel; and press the ENTER key.)

  2. Select the International Icon. (The keyboard command is ALT+S, I.)

  3. Under the Keyboard Layout section, choose US-Dvorak. (The keyboard command is ALT+K. Press the DOWN ARROW key until US-Dvorak appears.)

  4. Click OK or press the ENTER key.
