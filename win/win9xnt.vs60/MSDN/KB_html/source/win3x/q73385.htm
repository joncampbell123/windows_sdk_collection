Notebook 386SX Screen Saver Reboots in Enhanced Mode

 Last reviewed: November 22, 1994
Article ID: Q73385
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The Notebook 386SX computer by Associates Mega Subsystems has a built-in screen saver. If Windows 3.0 is running in enhanced mode, the system reboots 4 minutes after the screen saver is activated. Standard and real modes are not affected.

MORE INFORMATION

For more information, contact Associates Mega Subsystems.
