Text Is Clipped in Micrografx Designer 3.1

 Last reviewed: November 21, 1994
Article ID: Q82570
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

TrueType fonts in Microsoft Windows operating system version 3.1 are clipped in Micrografx Designer 3.1.

The problem is caused by Designer using the printer width, rather than the screen width. TrueType is not scaling well from the printer to screen with certain drivers. The HPPCL4 driver works correctly.

Two solutions are:

  • Use the fonts that ship with Designer 3.1 and not the TrueType fonts.

-or-
  • Choose a different printer for output.
