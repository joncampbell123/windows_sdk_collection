HP DeskJet 500 Scalable Font Initialization Error

 Last reviewed: February 2, 1996
Article ID: Q73427
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

Trying to print two jobs at once using the Hewlett-Packard (HP) DeskJet 500 driver you get the following errors: 

   Scalable font Initialization Error E: 800

   -then-

   Scalable font Initialization Error B: (this has a STOP icon with it)

   -then possibly-

   Unrecoverable Application Error (UAE)

RESOLUTION

To work around this problem:

  • Wait until the first print job has finished before starting another.

    -or-

  • Use the HP DeskJet Family printer driver supplied with Windows 3.0.
