Workgroup Connections Setup Restarts System in Windows 3.0

 Last reviewed: November 21, 1994
Article ID: Q91094
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a
  • Microsoft Workgroup Connections version 1.0
  • Microsoft Workgroup Add-On for MS-DOS, version 3.11

SYMPTOMS

If you run Workgroup Connections or Microsoft Workgroup Add-On for MS-DOS Setup under Microsoft Windows version 3.0, Setup gives you the option to restart the computer at the end of the installation. If other applications are running when the system is restarted, you will lose any data that you have not saved.

CAUSE

Workgroup Connections Setup does not know that it is running under Windows 3.0. Workgroup Connections Setup is only aware of Windows when it runs under Windows 3.1. Under Windows 3.1, Setup does not give you the option to restart the system.

WORKAROUND

To work around this problem, either run Setup outside Windows, or exit all applications before running Setup.
