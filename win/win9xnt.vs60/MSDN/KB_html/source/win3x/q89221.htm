Minimized Clock Truncates Digits in Digital Mode

 Last reviewed: July 20, 1995
Article ID: Q89221
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

When the System, Fixedsys, or Courier fonts are used with the Microsoft Windows Clock version 3.1 in Digital mode, the display's last digit or two may not appear when Windows Clock is minimized.

CAUSE

The System, Fixedsys, and Courier fonts are fixed space fonts; they do not resize when Windows Clock is minimized.

WORKAROUND

To work around this problem, use the MS Sans Serif or a TrueType font with Windows Clock.
