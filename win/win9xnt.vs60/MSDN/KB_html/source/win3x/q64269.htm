Windows Accessories Do Not Save Page Settings

 Last reviewed: September 18, 1996
Article ID: Q64269
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

If you choose Page Setup from the File menu in Notepad, Cardfile, Calendar, or Paintbrush, the settings are not saved when the file is saved.

CAUSE

All accessory applications use a flat-file format and the formatting information is not saved with the file; therefore, it is not shown with its last settings when the file is opened.

MORE INFORMATION

Note that this information applies to the version of Notepad included with Microsoft Windows 95 also.
