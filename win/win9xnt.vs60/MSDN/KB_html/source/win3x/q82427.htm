Alt+F6 Doesn't Cycle Through Visual Basic Windows Properly

 Last reviewed: November 23, 1994
Article ID: Q82427
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Visual Basic for Windows, version 1.0

Using the Alt+F6 key combination to cycle through VB (Visual Basic) windows works differently in Microsoft Windows version 3.1 or 3.11 than in Windows 3.0.

Microsoft has confirmed this to be a problem in Visual Basic 1.0. We are researching this problem and will post new information here as it becomes available.
