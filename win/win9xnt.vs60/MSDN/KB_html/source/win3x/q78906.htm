PRINT SCREEN to Clipboard Appears as Black-and-White Image

 Last reviewed: November 23, 1994
Article ID: Q78906
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

If you are running Microsoft Windows 3.0 in real mode and you press PRINT SCREEN or ALT+PRINT SCREEN, the image may be placed in the Clipboard as a black-and-white image.

CAUSE

On some computers, if Windows is in real mode the system may not have enough system resources or available memory to display the image in color. The image may also be distorted and unreadable.

RESOLUTION

Run Windows in standard or enhanced mode if possible.
