WFWG: Chat Text Is Pasted in the Wrong Window

 Last reviewed: November 21, 1994
Article ID: Q112368
This information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

When you use Windows for Workgroups Chat to communicate with several other computers, information pasted on the originating computer may be received in the wrong window on the other computers. This problem occurs if the user at the originating computer pastes in information before typing any text.

WORKAROUND

To avoid this problem, type any character on the originating computer before you do the paste operation.

STATUS

Microsoft has confirmed this to be a problem in Windows for Workgroups version 3.11. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
