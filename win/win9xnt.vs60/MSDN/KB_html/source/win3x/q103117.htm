WACOM PENWIN.DLL Generates "Out of Memory" Message with Write

 Last reviewed: June 12, 1995
Article ID: Q103117
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

If you use the WACOM pen tablet PENWIN.DLL driver or ACECAD 5" x 5" graphics digitizing tablet PENWIN.DLL driver with Microsoft Windows or Windows for Workgroups, you may receive an Out of Memory error when you use Microsoft Windows Write (WRITE.EXE). The error message may be similar to the following: 

   Not enough memory for Write to complete this operation.
   Quit one or more applications to increase available memory,
   and then try again.

MORE INFORMATION

WACOM Technical Support has confirmed this to be a problem. Currently, no workaround is available. To avoid the error, use Microsoft Windows Notepad (NOTEPAD.EXE) or another word processor.
