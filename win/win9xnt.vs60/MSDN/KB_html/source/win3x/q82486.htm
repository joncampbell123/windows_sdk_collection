Picture Publisher Stack Fault Changing Scanner Option

 Last reviewed: November 23, 1994
Article ID: Q82486
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

Picture Publisher version 3.0 displays the following error message when you use the mouse to try to change scanner options under Microsoft Windows: 

   PP caused a Stack Fault in module PP.EXE at 0048:034C

WORKAROUND

Use the keyboard to change the scanner options, not the mouse.
