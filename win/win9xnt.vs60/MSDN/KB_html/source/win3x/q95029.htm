CSS Laboratories 386-20 May Require BIOS Upgrade

 Last reviewed: November 22, 1994
Article ID: Q95029
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

Microsoft Windows stops responding (hangs) in 386 enhanced mode on your CSS Laboratories 386/20 system.

CAUSE

According to CSS Laboratories, if you have a CSS Laboratories 386-20 with an Award BIOS version earlier than 3.11 or a keyboard BIOS earlier than version 3.5, Windows may hang in enhanced mode.

RESOLUTION

To solve the problem, you need to obtain a BIOS upgrade from CSS Laboratories.

MORE INFORMATION

For additional information, contact CSS Laboratories.
