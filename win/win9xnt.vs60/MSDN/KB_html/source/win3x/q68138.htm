Calculations that Require Calculator to Be Reinitialized

 Last reviewed: November 21, 1994
Article ID: Q68138
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

There are two situations in Microsoft Windows Calculator that require the use of the Clear Entry (CE) or Clear All (C) buttons:

  • Dividing by zero.
  • Calculating a number that is beyond the range of the current number system.

Under these two situations, selecting other keys or menu items results in a warning beep if the warning beep is enabled.

MORE INFORMATION

Calculator displays two error messages that indicate this situation:

  • Cannot divide by zero
  • Result is too large for display

    -or-

    Result is too large for (in Windows 3.0 standard view only)

The above error message has been changed in Microsoft Windows 3.1 Calculator to read: Result is too large to store.

REFERENCES

For valid number systems for calculator and their range limits, refer to page 363 of the "Microsoft Windows User's Guide" for version 3.0 and page 415 of the "Microsoft Windows 3.1 User's Guide" for version 3.1.
