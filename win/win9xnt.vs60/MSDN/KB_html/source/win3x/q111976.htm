Help Files on Current Windows for Workgroups 3.11 Issues

 Last reviewed: May 1, 1997
Article ID: Q111976
3.11 WINDOWS kbfile

The information in this article applies to:

  • Microsoft Windows for Workgroups version 3.11

SUMMARY

The Microsoft PSS Knowledge Base group regularly extracts the most current articles from the Windows for Workgroups database and creates a Windows Help file that we then post to the appropriate forum and the Microsoft Software Library. We also update these files as new information becomes available.

The files listed below are available to download from the Microsoft Software Library (MSL). For information about downloading these files, please see the "Instructions for Downloading" section at the end of this article.

FILES AVAILABLE FROM ONLINE SERVICES

For Windows for Workgroups issues, there are two files available. WFWG.EXE contains a large collection of articles about setting up Windows for Workgroups and networking issues. WFW12.EXE contains a collection of 100 of the articles most frequently used by PSS Support Engineers as of January 1995. 

Item ID      Filename           Description

S14589       WFWG.EXE          Windows for Workgroups articles
S14766       WFW12.EXE         100 Top articles for WFWG

INSTRUCTIONS FOR DOWNLOADING

You can download any of these self-extracting files from the following services:

  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select the product.
      Enter kbfile <FILENAME>.EXE, and click GO!
      Open the article, and click the button to download the file.
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder.
      Get <FILENAME>.EXE
  • Microsoft Download Service (MSDL) 

          Dial (425) 936-6735 to connect to MSDL
      Download <FILENAME>.EXE
