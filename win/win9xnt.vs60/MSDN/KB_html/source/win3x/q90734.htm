Changes to Windows Driver Library - Displays

 Last reviewed: March 30, 1995
Article ID: Q90734
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

Below is a summary of the changes to the Microsoft Windows Driver Library (WDL). The WDL offers support for many printers, displays, and sound cards not provided with Microsoft Windows version 3.1.

For a complete listing of the WDL's contents and instructions for obtaining the WDL, query on "Q82006" (without the quotation marks) in the Microsoft Knowledge Base. 

                 WINDOWS DRIVER LIBRARY CHANGE LOG
                 =================================

*                            DISPLAYS                                *
**********************************************************************

Appian RGDI and Compatible (RGDI.EXE)

06/15/92 - THE DRIVERS HAVE NOT BEEN CHANGED. All the compressed files 

           were renamed using the .XX_ convention. A message that
           displays when the .EXE file is run was also added. These
           changes are designed to reduce the problems that occur when
           the .EXE file is run from the Windows directory or Windows
           SYSTEM subdirectory.

           In addition, the LICENSE.TXT file was replaced with the
           current revision.
05/06/92 - No change was made to the driver or supporting files. 

           WIN.CNF was added to RGDI.EXE to work around a maintenance
           mode setup problem (users may be prompted for a Windows
           disk to access WIN.CNF).

ATI DRIVERS (ATI.EXE)

06/25/92 - New file. Replaced ATIULT.EXE. Contains updated drivers for 

           the ATI 8514 Ultra, ATI Graphics Vantage, and the ATI
           Graphics Ultra. These updated drivers offer improved
           performance and quality. Also included in ATI.EXE are
           drivers for the ATI VGA Wonder cards.

Super VGA 256-Color Driver (SVGA.EXE)

06/24/93 - SVGA.EXE has been updated in the WDL to make additions to 

           the README.TXT file.
04/30/93 - SVGA.EXE is new to the WDL. It contains a new high-speed 
           256-color display driver for Windows 3.1 that supports
           most non-accelerated Super VGA display types. You can
           install this driver by using Windows Setup to select one
           of the following display types:

              Super VGA  640x480 256 colors
              Super VGA  800x600 256 Large; Large fonts
              Super VGA  800x600 256 Small; Small fonts
              Super VGA 1024x768 256 Large; Large fonts
              Super VGA 1024x768 256 Small; Small fonts

           Displays supported by these selections include the following:

              ATI VGA series, including:

                 ATI VGA Wonder
                 ATI VGA Wonder Plus
                 ATI VGA Wonder 24XL

              Cirrus Logic VGA (6420, 5420 series)

              Oak Technology VGA (077 series)

              Paradise VGA, including:

                 Paradise VGA 1024
                 Paradise VGA Professional

              Trident VGA (8900C series), including:

                 Trident Impact

              Tseng VGA (ET4000 series), including:

                 Boca VGA
                 Cardinal VGA
                 Diamond Speedstar series
                 Everex Viewpoint series
                 Metheus Premier VGA
                 Orchid Prodesigner II
                 Sigma VGA Legend
                 STB Powergraph

              Video Seven VGA, including:

                 Video Seven FastWrite
                 Video Seven 1024i
                 Video Seven VRAM
                 Video Seven VRAM II

              Western Digital VGA

       NOTE: MS-DOS-based applications run on Trident adapters may not
       be restored correctly when switched back to full-screen mode
       from Windows.

       NOTE: MS-DOS-based applications run on some Tseng ET4000 adapters
       may not be restored correctly when switched back to full-screen
       mode from Windows. If this happens, use Windows Setup to select
       one of the ET4000 options. However, this may slow the operation
       of MS-DOS-based applications running in Windows.

Western Digital Paradise 90C30 Drivers (WD9030.EXE)

02/03/93 - These drivers are new to the WDL. They support the 

           following resolutions and colors:

             - 640 x 480 256 colors
             - 640 x 480 32K colors
             - 640 x 480 32K colors (Ventura Publisher, Aldus PageMaker
                                    only)
             - 640 x 480 64K colors
             - 800 x 600 256 colors (512K)
             - 800 x 600 256 colors (1 MB)
             - 800 x 600 32K colors
             - 800 x 600 32K colors (Ventura Publisher, Aldus PageMaker
                                    only)
             - 800 x 600 64K colors
             - 1024 x 768 16 colors
             - 1024 x 768 256 colors (1 MB)
             - 1280 x 960 16 colors
             - 1280 x 1024 16 colors

            NOTE: The 32K color 640 x 480 and 800 x 600 resolution
            drivers both have a specific driver for use with Ventura
            Publisher and Aldus PageMaker. Neither of these
            applications functions with 32K colors. The 32K color
            drivers provided for Ventura Publisher and Aldus PageMaker
            (P32K480X.DRV and P32K600X.DRV) do not have the capability
            of the full 32K colors.
