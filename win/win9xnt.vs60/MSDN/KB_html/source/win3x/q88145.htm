SMARTDrive Shows Status of Nonexistent Floppy Drives

 Last reviewed: November 23, 1994
Article ID: Q88145
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

SMARTDrive (SMARTDRV.EXE) version 4.0 from Windows 3.1 shows the status of floppy drives A and B, even if no floppy disk drives are installed.

WORKAROUND

To work around this problem, specifically instruct SMARTDrive not to cache the missing floppy drives. For example, type the following at the MS-DOS command prompt: 

   c:\windows\smartdrv.exe a- b-
