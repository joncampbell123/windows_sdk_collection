IIS Version 1.0 Does Not Support Byte-Range Header

 Last reviewed: April 22, 1997
Article ID: Q142716
The information in this article applies to:
  • Microsoft Internet Information Server version 1.0

Microsoft Internet Information Server version 1.0 does not support the byte-range header.

For more information on the byte-range header, search for byte-range on your favorite World Wide Web (WWW) search engine.
