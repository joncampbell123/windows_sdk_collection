WebTV Does Not Support 128-bit SSL

 Last reviewed: February 3, 1998
Article ID: Q170869

The information in this article applies to:

  • Microsoft Internet Information Server version 3.0

SYMPTOMS

When you use a WebTV set top device to connect to the Internet Information Server (IIS) Web Service, you may get an error that the service cannot connect.

WebTV uses 120-bit Secure Sockets Layer (SSL) encryption, and the domestic version of IIS supports 128-bit SSL encryption.

WORKAROUND

Use a 40-bit SSL key on the IIS computer.

MORE INFORMATION

You can contact WebTV Technical Support at (800) 469-3288.
