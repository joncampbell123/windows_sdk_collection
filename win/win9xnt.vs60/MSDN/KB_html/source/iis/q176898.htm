FTP "Resume" Functionality Not Supported

 Last reviewed: January 22, 1998
Article ID: Q176898
The information in this article applies to:
  • Microsoft Internet Information Server versions 1.0, 2.0, 3.0
  • Microsoft Peer Web Server version 1.0 for Windows NT 4.0

Some FTP servers support the capability to "resume" an interrupted download.

Microsoft Internet Information Server and Microsoft Peer Web Server do not support this capability.
