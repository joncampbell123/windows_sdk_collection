How to Search for IIS, Index Server, & IE KB Articles by Topic

 Last reviewed: February 19, 1998
Article ID: Q142858
1.00 WINDOWS kbkeyword kbref

The information in this article applies to:

  • Microsoft Internet Information Server version 1.0
  • Microsoft Index Server version 1.0
  • Microsoft Internet Explorer version 1.0

SUMMARY

You can use the following KBSubcategory keywords to make queries on the Microsoft Internet Information Server (IIS), Microsoft Index Server, and Microsoft Internet Explorer (IE) Knowledge Base quicker and more precise. To achieve that speed and precision, the keywords contain the unique prefix (for example, "iis", "indx", or "ie") to minimize the chance of finding a similar keyword in an unrelated article during a full-text query.

NOTE: In addition, you can use the more general KBCategory keywords (containing the "kb" prefix) that apply to all Knowledge Base articles. For a listing of KBCategory keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
