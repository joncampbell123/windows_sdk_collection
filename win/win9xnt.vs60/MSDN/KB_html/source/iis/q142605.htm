Unicode Code Page Not Supported in Internet Information Server

 Last reviewed: April 22, 1997
Article ID: Q142605
The information in this article applies to:
  • Microsoft Internet Information Server version 1.0

SYMPTOMS

UNICODE is not currently supported on the Microsoft Internet Information Server (IIS).

WORKAROUND

Be sure to select Save as, Save File as Type, Text Files (*.txt), when you edit or save any HTM file for use on the Microsoft Internet Information Server from Notepad.

MORE INFORMATION

If you save an HTM file from Notepad or any other editor that supports Unicode you may see strange characters on your browser when accessing that specific page.
