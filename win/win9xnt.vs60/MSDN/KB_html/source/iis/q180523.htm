Ifilttst.exe & Ifilttst.ini Files Missing from IFilter Document

 Last reviewed: February 4, 1998
Article ID: Q180523

The information in this article applies to:

  • Microsoft Internet Information Server version 4.0

SYMPTOMS

The January 1998 MSDN Library and Platform Software Development Kit (SDK) contains documentation on programming with the IFilter interface. The documentation mentions a tool that will verify an IFilter implementation. The tool consists of two files: Ifilttst.exe and Ifilttst.ini. These files were not included with the documentation.

RESOLUTION

You can get the missing files by downloading the IFilter SDK from the BackOffice Download and Trial Center at the following location: 

   http://www.microsoft.com/backoffice/promo/downtrial/default.htm.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft Internet Information Server version 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Documentation on the IFilter interface is located in the Platform SDK documentation at the following location:

Internet/Internet/Extranet Services Index Server Programmer's Guide Customizing Index Server
