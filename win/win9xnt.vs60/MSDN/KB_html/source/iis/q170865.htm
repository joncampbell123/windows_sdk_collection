ErrMsg: Reentrant Call to Interupt Window on Netscape 2.0 (UNIX)

 Last reviewed: November 13, 1997
Article ID: Q170865

The information in this article applies to:

  • Microsoft Internet Information Server versions 2.0 and 3.0

SYMPTOMS

When you use Netscape Navigator version 2.0 on UNIX, you may see the following error message when you browse a site that has been created by Microsoft FrontPage. 

   Reentrant Call to Interupt Window.

CAUSE

The FrontPage editor inserts the following META tag into all new documents: 

   <META HTTP-EQUIV="Content-Type" CONTENT="text/html; 
   charset=iso-8859-1">

WORKAROUND

Remove the META tag or upgrade your UNIX browser to Netscape version 3.0.
