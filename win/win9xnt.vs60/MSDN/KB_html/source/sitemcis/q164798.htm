AV When You Try to Open Workspace

 Last reviewed: February 3, 1998
Article ID: Q164798

The information in this article applies to:

  • Microsoft MCIS, version 1.0

SYMPTOMS

If you open Performance Monitor, and you open counters on a remote Microsoft Personalization System (MPS) server, and then save the workspace, quit Performance Monitor, open Performance Monitor, and try to load the workspace, workspace will experience an access violation (AV).

STATUS

Microsoft has confirmed this to be a problem in Microsoft Commercial Internet System version 1.0.

MORE INFORMATION

Two variables were incorrectly set in the code. The two files that need to be updated, Smail.dll, and Ssovote.dll, have been corrected and are in Service Pack 1 (SP1).
