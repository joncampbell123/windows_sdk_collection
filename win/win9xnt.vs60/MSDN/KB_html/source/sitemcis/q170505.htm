Webpost DBCS Names Not Displayed Properly

 Last reviewed: March 20, 1998
Article ID: Q170505
The information in this article applies to:
  • Microsoft Commercial Internet System version 1.0 - Microsoft Membership Server

SYMPTOMS

When you try to enter a double-byte character (DBCS), used in languages such as Korean, Chinese, or Japanese, for a file name or folder name in Webpost, the name is not displayed correctly.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Commercial Internet System version 1.0. This problem has been corrected in the latest U.S. Service Pack for Microsoft Commercial Internet System version 1.0. For information about obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
