Mail Client Won't Start & Reports One Or More Components Busy

 Last reviewed: February 3, 1998
Article ID: Q161699
The information in this article applies to:
  • Microsoft Commercial Internet System, version 1.0

SYMPTOMS

The Microsoft Commercial Internet System (MCIS) Mail client stops working and reports the following error message: 

   One or more components may be busy; try again later.

CAUSE

The problem is that the <username>.wab file (the Windows Address Book) is failing to load, because it is not initializing properly.

RESOLUTION

To resolve this problem, delete the <username>.wab file.
