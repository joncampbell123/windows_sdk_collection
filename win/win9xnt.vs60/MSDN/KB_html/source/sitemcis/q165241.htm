How to Find KB Articles for Site Server and Related Components

 Last reviewed: February 9, 1998
Article ID: Q165241
The information in this article applies to:
  • Microsoft Site Server version 2.0
  • Microsoft Site Server, Enterprise Edition version 2.0
  • Microsoft Commerce Server version 2.0
  • Microsoft Visual InterDev version 1.0
  • Microsoft Personalization System

SUMMARY

This article discusses how to find Microsoft Knowledge Base articles that pertain to the components of Microsoft Site Server and Site Server, Enterprise Edition.

MORE INFORMATION

Site Server includes the following components: 

  - Content Replication System 2.0 
  - Content Replication System SDK 2.0 
  - Internet Locator Server 2.0 
  - Personalization System 2.0 
  - Microsoft Site Analyst 2.0 
  - Microsoft Usage Analyst for Access 2.0 
  - Microsoft Usage Analyst for SQL Server  2.0 
  - Microsoft Web Publishing Wizard 1.5 
  - Microsoft Posting Acceptor 1.0 (NT version) 
  - Customizable Start Sites aka Weblications (see *1 below) n/a 
  - Visual InterDev
Site Server, Enterprise Edition includes the following components: 

  - Content Replication System 2.0 
  - Content Replication System SDK 2.0 
  - Internet Locator Server 2.0 
  - Personalization System 2.0 
  - Microsoft Site Analyst 2.0 
  - Microsoft Usage Analyst for Access Enterprise Edition 2.0 
  - Microsoft Usage Analyst for SQL Server Enterprise Edition 2.0 
  - Microsoft Web Publishing Wizard 1.5 
  - Microsoft Posting Acceptor 1.0 (NT version) 
  - Customizable Start Sites aka Weblications (see *1 below) n/a 
  - Visual InterDev
  - Commerce Server 2.0
Below are the URL's that can be used to search for knowledge base articles and other information on the Support Online web site:

Commerce Server: 

   http://support.microsoft.com/support/c.asp?PR=COM
Visual InterDev: 

   http://support.microsoft.com/support/c.asp?PR=VID
Web Publishing Wizard and Microsoft Posting Acceptor: 

   http://support.microsoft.com/support/c.asp?PR=IEAU
Personalization System and all other components: 

   http://support.microsoft.com/support/c.asp?PR=CIS
