Location of Membership Broker SDK

 Last reviewed: February 3, 1998
Article ID: Q160880
The information in this article applies to:
  • Microsoft Commercial Internet System, version 1.0

SUMMARY

On the Microsoft Commercial Internet System beta 2 compact disc, the software developer kit (SDK) for Membership Broker was in its own subdirectory. I cannot find it on later versions of the compact discs. Where is the SDK?

MORE INFORMATION

The SDK is included as part of the Membership Broker Setup routine. If you need to install the SDK, select Custom instead of Production when installing the Membership Broker, and ensure the SDK check box is selected.

NOTE: There is a bug in the RC1 and RC2 install.If you select the SDK option by itself, the install will terminate without error message and without installing the SDK. The install will work fine so long as any other component of the install is checked.
