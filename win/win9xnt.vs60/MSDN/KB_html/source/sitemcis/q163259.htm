Location of Currencies for Pricing File

 Last reviewed: January 27, 1998
Article ID: Q163259
The information in this article applies to:
  • Microsoft Commercial Internet System, version 1.0

SUMMARY

The version 1.0 documents for Membership Services (specifically, the Web Authoring documents and the Software Development Kit) indicate the need for useable currency values when working with pricing files. However, the version 1.0 documents do not contain this information.

MORE INFORMATION

For the correct information on usable currency values for pricing files, see the Getting Started document that comes with Microsoft Commercial Internet System 1.0 Service Pack 1. If that document is unavailable, the following is a list of values you can use: 

   unknown currency
   Australian dollars
   Austrian schillings
   Belgian francs
   British pounds
   Canadian dollars
   Danish kroner
   Finnish markkaa
   French francs
   German mark
   Italian lire
   Japanese yen
   Dutch gulden
   New Zealand dollars
   Norwegian kroner
   Portuguese escudos
   Spanish pesetas
   Swedish kronor
   Swiss francs
   United States dollars
