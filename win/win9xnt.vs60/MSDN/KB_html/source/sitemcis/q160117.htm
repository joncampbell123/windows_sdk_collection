IMS: Domain Literals Not Accepted for Recipients

 Last reviewed: February 3, 1998
Article ID: Q160117
The information in this article applies to:
  • Microsoft Commercial Internet System, version 1.0

SYMPTOMS

Without the capability to use IP addresses in domain literals, such as rick@[172.28.201.5], some UNIX SMTP servers cannot forward mail or receive mail in an Internet configuration where the computers do not have Domain Name Service (DNS).

STATUS

Microsoft has confirmed this to be a problem in Microsoft Commercial Internet System version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
