Authprox 16 Install Fails Due to Missing Network Libraries

 Last reviewed: February 2, 1998
Article ID: Q173029
The information in this article applies to:
  • Microsoft Commercial Internet System version 1.0

SYMPTOMS

The following message box warning may appear during installation of the 16- bit Microsoft Authentication Proxy for Netscape on a Microsoft Windows 3.x system running the Microsoft Lanman 2.2c network client. 

   Unable to locate the required network libraries. Microsoft
   Authentication Proxy for Netscape cannot be installed.
The installation will terminate after this error.

CAUSE

Winsock.dll cannot be located in certain directories. The installation process currently can only find Winsock.dll if it is located in any of the following locations: 

   - %windir% directory
   - %windir%\system directory
   - same directory as netscape.exe
Type the “set” command in a command prompt shell under Windows 3.x to determine the value for %windir%.

RESOLUTION

Copy Winsock.dll to either of the locations mentioned above. The Lanman 2.2c version of Winsock.dll is located in the lanman.dos\netprog directory.
