Err Msg: HTTP Filter DLL C:\MEMBER~1\prxyfltr.dll Failed

 Last reviewed: March 12, 1998
Article ID: Q172021

The information in this article applies to:

  • Microsoft Commercial Internet System version 1.0
  • Microsoft Membership Server

SYMPTOMS

A STOP error is reported in the Event Log. The error message reads: 

   The HTTP Filter DLL C:\MEMBER~1\prxyfltr.dll failed to load.
The data is the error. No error message (such as, a service failed to start) comes up.

CAUSE

The problem has to do exclusively with Microsoft Commercial Internet System Service Pack 1 (SP1) code. During the installation of the broker (MBS), a change is made to the following registry key: 

   HKEY_LOCAL_MACHINE\System\CCS\Services\W3SVC\Parameters\Filter DLLs
Prxyfltr.dll is added to the list. However, a comma is used for a delimiter, which causes the STOP error to be reported in the Event Log.

WORKAROUND

Nothing needs to be done to fix this issue. If you want to, you can change the delimiter from a comma to a semicolon, so that the error message no longer appears.
