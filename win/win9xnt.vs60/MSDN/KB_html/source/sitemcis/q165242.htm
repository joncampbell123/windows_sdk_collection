What is Site Server?

 Last reviewed: February 9, 1998
Article ID: Q165242
The information in this article applies to:
  • Microsoft Site Server version 2.0

SUMMARY

The following is a overview of Microsoft Site Server. To see additional information, please see the following Internet site: 

   http://backoffice.microsoft.com/products/siteserver/

MORE INFORMATION

Microsoft Site Server is a comprehensive Web-site environment for enhancing, deploying, and managing intranet sites using the Microsoft Windows NT Server network operating system and Microsoft Internet Information Server.

Personalize and Enhance Your Site

Microsoft Site Server enables you to provide targeted business information to groups of employees and create relationships with individuals on the Internet through the delivery of customized content. Microsoft Site Server helps you increase the efficiency of business processes by enabling delivery of information that is relevant to users. You can extend and enhance your site through the easy creation of dynamic Web applications.

Personalization System 

   Works with Active Server Pages to dynamically generate Web
   pages based on stored user preferences. Personalization
   System enables you to deliver targeted content.
Microsoft Visual InterDev 
   Included as a promotion. Visual InterDev is an integrated
   development system for building dynamic Web applications.
   It includes an integrated development environment, seamless
   connectivity to any ODBC-based databases, prebuilt features,
   wizards, and content creation tools. The Visual InterDev Web
   development system provides everything a developer needs to
   quickly develop and enhance Web sites for business-critical
   applications.

Deploy Content Robustly

Microsoft Site Server enables you to publish Web content easily and securely to one or multiple Web servers inside the corporate firewall or across the Internet. Microsoft Site Server provides you with the ability to stage and test mission-critical content before global deployment.

Content Replication System* 

   Enables the implementation of site staging and mirror servers
   and the connection of departmental Web sites into a corporate
   backbone. Content Replication System provides a highly
   reliable, secure, and efficient way to move content within
   a corporation or across the Internet. Content Replication
   System automates the replication process.
Web Publishing Wizard and Posting Acceptor 
   Enables users to easily post Web pages to your Web server.
   Web Publishing Wizard automates the process of copying files
   from your computer to the Web server. Posting Acceptor enables
   Microsoft Internet Information Server to accept Web content
   from Web Publishing Wizard and other clients via HTTP Post.
   Posting Acceptor also integrates with Content Replication
   System to automate the posting of Web content to multiple servers.
* To replicate content with Content Replication System, you must acquire at least one additional license to install Content Replication System on a second server. For Microsoft Site Server pricing and licensing information visit the Microsoft World Wide Web site.

Manage and Analyze Your Site

Microsoft Site Server enables you to actively manage Web content, regardless of the mix of Web servers utilized, and to learn about how your site is being used. Microsoft Site Server enables you to visualize the layout of your site, measure the site's effectiveness and continually improve user experiences.

Usage Analyst 

   Imports Web server logs and provides a comprehensive suite of
   usage analysis reports. Choose from the variety of summary, detail,
   and comprehensive analysis reports so you can get the appropriate
   level of information about your site activity to those who need it.
   The standard reports can reveal information such as request trends and
   visit trends, navigational patterns, and server load patterns.
   Supports 28 Web server log file formats on Windows NT, UNIX, and
   Macintosh operating systems, including those from Microsoft, Netscape,
   Apache, and O'Reilly.
Site Analyst 
   Provides comprehensive site visualization, content analysis,
   link management, and reporting capabilities for managing Web sites.
   Easily visualize how your site is laid out, recognize repetitive
   content, and identify broken links. Site Analyst lets you create
   comprehensive HTML reports that detail site content and consistency.
   Site Analyst enables you to manage Web content efficiently and
   effectively in order to provide a positive user experience.
