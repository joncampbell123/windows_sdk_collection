CRS Webadmin Dialog Boxes Too Small in IE 4.0

 Last reviewed: March 12, 1998
Article ID: Q177739
The information in this article applies to:
  • Microsoft Commercial Internet System 1.0 - Content Replication System
  • Microsoft Site Server 2.0 - Content Replication System

SYMPTOMS

Content Replication System (CRS) Webadmin dialog boxes are sized incorrectly when viewed with Internet Explorer (IE) 4.0. The OK, Cancel, and Help buttons are not visible and the dialog box window cannot be resized.

WORKAROUND

To work around this problem:

  • To click a button, use the Tab key to scroll the button into view.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Commercial Internet System 1.0 and Site Server 2.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
