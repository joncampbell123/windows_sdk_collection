DOCERR: CVTHPPFM.EXE Cannot Convert .PFM Format to UNITOOL

 Last reviewed: February 24, 1995
Article ID: Q92412
The information in this article applies to:
  • Microsoft Windows Device Driver Kit (DDK) for Windows version 3.1

The CVTHPPFM.EXE utility that comes with the Windows version 3.1 DDK cannot convert earlier Hewlett-Packard (HP) .PFM files to UNITOOL .PFM files. The utility allows only UNITOOL .PFM files to be converted to the earlier HP .PFM file format.

Page 34 of the Windows version 3.1 DDK's "Printer and Fonts Kit" manual incorrectly states that the CVTHPPFM.EXE utility does both conversions.
