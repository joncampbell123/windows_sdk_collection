Display Driver Compatibility Testing Appnote Available

 Last reviewed: February 15, 1996
Article ID: Q87005
The information in this article applies to:
  • Microsoft Windows Device Driver Kit (DDK) for Windows versions 3.1 and 3.11

An application note that contains information regarding the display driver compatibility testing program offered by Microsoft Product Support Services is now available. Application note PW0563 includes an overview of the display driver testing program, participation details, and a list of the tests involved. It also includes an update of a test application and of a test document included with version 3.1 of the Microsoft Windows Device Driver Kit (DDK) and two additional optional display driver tests.

Before submitting any display driver to Microsoft for testing, obtain the latest copy of this application note, which will always contain the most current information regarding this program.

Application note PW0563 is available in the Software/Data Library or from Microsoft. In the United States, application notes can be obtained by calling Microsoft Product Support Services at (206) 454- 2030. Outside the United States, contact the Microsoft subsidiary or distributor that serves your country.
