Printer Driver Does Not Realize Pens or Brushes

 Last reviewed: February 16, 1995
Article ID: Q43037
The information in this article applies to:
  • Microsoft Windows Device Development Kit (DDK) for Windows versions 3.0 and 3.1

In the Microsoft Windows graphical environment, a printer driver can indicate that it does not realize its own pens or brushes by specifying -1 as the value for the dpNumBrushes or dbNumPens members of the GDIINFO data structure.
