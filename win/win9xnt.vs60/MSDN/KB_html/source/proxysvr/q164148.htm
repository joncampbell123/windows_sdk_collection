Proxy Server Doc Err in LAT Setting for Caching Internet Objects

 Last reviewed: January 22, 1998
Article ID: Q164148

The information in this article applies to:

  • Microsoft Proxy Server version 1.0

SUMMARY

Chapter 6 of the Proxy Server documentation has a section describing how to set up the Proxy Server to cache Internet Objects. The Local Address Table (LAT) settings in this section are incorrect.

MORE INFORMATION

The sub-section labeled "Cache-Only Implementation" has the following text: 

   To configure this scenario, the Local Address Table (LAT) for
   Microsoft Proxy Server must include all IP addresses and subnets
   that will be allowed to use the Web Proxy service cache. If
   inbound client requests are permitted, then the LAT should
   have the address range from 0.0.0.0 to 255.255.255.255.
The actual IP range that you must use is 0.0.0.1 to 255.255.255.254. If you try to enter the other IP address, you will get the following error message: 

   Invalid IP Address 0.0.0.0
