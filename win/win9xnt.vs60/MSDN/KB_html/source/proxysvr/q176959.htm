Err Msg w/ Proxy Server or IIS: Requested Header Not Found

 Last reviewed: December 17, 1997
Article ID: Q176959

The information in this article applies to:

  • Microsoft Proxy Server versions 1.0 and 2.0
  • Microsoft Internet Information Server versions 1.0, 2.0, and 3.0

SYMPTOMS

When you use the Microsoft Proxy Server service, a client request through the proxy may return the following error message: 

   Requested Header Not Found.
You may get the same error message when you request a web page from an Internet Information Server (IIS).

CAUSE

An authentication method under the WWW Service in the Internet Service Manager is not enabled, and the Proxy and IIS server needs credentials for the request.

RESOLUTION

Enable at least one of the authentication methods in the Internet Service Manager WWW service properties:

Basic (Clear Text)

-or-

Windows NT Challenge/Response

This will allow the Proxy or IIS server to request credentials from the client web browser.
