Proxy Client: Fail to Fetch Server Network Address

 Last reviewed: December 30, 1997
Article ID: Q175177

The information in this article applies to:

  • Microsoft Proxy Server version 2.0

SYMPTOMS

When you try to refresh the Winsock Proxy client configuration using the Winsock Proxy client Control Panel application, you may get the following error message: 

   Fail to fetch server network address.
WORKAROUND

Verify that a Novell NetWare server is present on the network or at least one Microsoft Windows NT Server on the network has File and Print Services for NetWare (FPNW) installed. FPNW and NetWare server have the capability of resolving IPX names to IPX addresses.

MORE INFORMATION

Winsock applications may be able to use the Proxy Server successfully, and the CHKWSP32 /F may report that the client and server are functioning normally.

The Control Panel application as well as CHKWSP32 /S must be able to resolve the IPX name to an IPX address.
