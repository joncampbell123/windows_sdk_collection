How to Search for Microsoft Proxy Server KB Articles by Topic

 Last reviewed: October 6, 1997
Article ID: Q140965
The information in this article applies to:
  • Microsoft Proxy Server versions 1.0 and 2.0

SUMMARY

You can use the following keywords to make queries on the Microsoft Proxy Server Knowledge Base quicker and more precise. To achieve that speed and precision, the keywords contain a unique prefix (for example, "pxs") to minimize the chance of finding a similar keyword in an unrelated article during a full-text query.

NOTE: In addition, you can use the more general keywords (containing the "kb" prefix) that apply to all Knowledge Base articles. For a listing of these general keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles

Proxy Server Knowledge Base Subcategories

Keyword        Subcategory


pxsclient      Clients: Internet Explorer, Netscape, etc.
pxsconfig      Configuration and Tuning
pxsdocerr      For errors in Help and Documentation
pxsgeneral     General, Other
pxshowto       How To
pxsinterop     Interoperability (Windows for Workgroups, Windows 95,
               Netscape, etc.)
pxsmsn         MSN access component
pxsodbc        Access/SQL ODBC logging/connectivity
pxsperm        FTP, Gopher, WWW proxy permissions/issues
pxssetup       Setup
pxsui          User Interface
pxsregistry    Registry
pxsrpc         RPC issues
pxssecurity    Security issues

Additional query words: prodiis 1.00 proxykeyword
