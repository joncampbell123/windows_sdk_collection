Pegasus Mail Client Stops Responding When Used w/ Winsock Proxy

 Last reviewed: December 30, 1997
Article ID: Q170539

The information in this article applies to:

  • Microsoft Proxy Server version 1.0
  • Microsoft Windows for Workgroups, version 3.11

SYMPTOMS

When you run 16-bit Pegasus mail on a Microsoft Windows for Workgroups version 3.11 computer with Winsock Proxy client, everything works fine to the point where it finishes downloading pop mail. At that point, the client stops responding. The only way to recover is to use Ctrl+Alt+Del to force Windows for Workgroups to end the Pegasus mail process.

CAUSE

The Winsock proxy client is returning the incorrect return code.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Proxy Server version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
