Err Msg: Internet Server Manager Not Installed on this Machine

 Last reviewed: December 30, 1997
Article ID: Q164074

The information in this article applies to:

  • Microsoft Proxy Server version 1.0

SYMPTOMS

When you try to setup Microsoft Proxy Server on a Microsoft Windows NT Server computer with Microsoft Internet Information Server (IIS) version 1.0, you will get the following error message: 

   Internet Server Manager is not installed on this machine.

RESOLUTION

To install the Proxy Server, you must have the following software already installed on the server computer:

  • Microsoft Windows NT Server version 4.0
  • Microsoft Internet Information Server version 2.0
  • The Windows NT Server 4.0 Service Pack (provided on the Microsoft Proxy Server compact disc).

MORE INFORMATION

For additional information, please refer to Chapter 2 of the Proxy Server documentation.
