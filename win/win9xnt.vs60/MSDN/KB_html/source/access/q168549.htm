ACC97: Choosing the Appropriate DB White Paper Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q168549
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

The Choose97.exe file contains a document called "Choosing the Appropriate Database Development Tool," which discusses the relative strengths of Microsoft's database products: Microsoft Access, Microsoft Visual Basic, Microsoft Visual FoxPro, and Microsoft SQL Server. Mention is also made of Microsoft Visual C++ and Microsoft Visual J++. 

 ~ Choose97.exe (size: 132769 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following files:

   Choosing.doc    The "Choosing the Appropriate Database Development
                   Tool" white paper.
   Readme.txt      This information in text format.
Choosing.doc is in Microsoft Word 97 format. To view this document in an earlier version of Microsoft Word, download and install the Microsoft Word 97 Converter from http://support.microsoft.com/support/word/.

For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
