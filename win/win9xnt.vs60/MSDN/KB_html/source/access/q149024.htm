ACC95: "Paradox Database Driver" Help Index Spelling Error

 Last reviewed: April 3, 1997
Article ID: Q149024
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

The Help Index in Microsoft Access for Windows 95 contains a misspelling in the following topic: 

   Paradox Datbase Driver

MORE INFORMATION

This topic should read as follows: 

   Paradox Database Driver

REFERENCES

For more information about Paradox database drivers, search for "Paradox," and then "Paradox Database Driver" using the Microsoft Access for Windows 95 Help Index.
