ACC97: Access 97 Does Not Include Microsoft Personal Web Server

 Last reviewed: October 13, 1997
Article ID: Q162402
The information in this article applies to:
  • Microsoft Access 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

Some Microsoft Access 97 CD-ROM liners state the following: 

   The Microsoft Access 97 CD-ROM includes the Microsoft Access 97
   ValuPack, which contains additional value-added components, such as:
   Microsoft Internet Explorer 3.0, sounds, additional drivers, Microsoft
   Personal Web Server, the Office Upgrade Wizard, and Microsoft Camcorder.
This information is incorrect.

MORE INFORMATION

The Microsoft Access 97 ValuPack does not contain Microsoft Personal Web Server. However, you can download the Microsoft Personal Web Server for Windows 95 free (other than the cost of connect time, if applicable) from Microsoft's Web site at http://www.microsoft.com/ie/download/.

Microsoft Personal Web Server (PWS) for Windows 95 enables your Windows 95 computer to act as a Web server, and enables easy publication of personal Web pages without the system requirements of a full Web server, such as Microsoft Internet Information Server (IIS).
