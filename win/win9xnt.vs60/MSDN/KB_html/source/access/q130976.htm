ACC2: Service Pack Not Available on Microsoft Select CD-ROM

 Last reviewed: April 25, 1997
Article ID: Q130976
The information in this article applies to:
  • Microsoft Access version 2.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

Microsoft Select is a flexible program for licensing and upgrading Microsoft software in volume. The Select program offers tiered pricing, flexible usage models, purchase monitoring, and product distribution on CD-ROM.

The Microsoft Access 2.0 Service Pack is not available as part of Microsoft Select. However, you can download the Service Pack from several electronic services or order it by mail. For more information about how to obtain the Service Pack, please see the following article in the Microsoft Knowledge Base:
