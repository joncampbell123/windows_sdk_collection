ACC95: MS Jet 3.0 Performance Overview Available on MSL

 Last reviewed: October 21, 1997

SUMMARY

Advanced: Requires expert coding, interoperability, and multiuser skills.

The V3PERF.EXE file contains an overview of Microsoft Jet Database Engine version 3.0. The overview discusses the primary goal of Microsoft Jet Database Engine 3.0, benchmark timings, and details on performance enhancements.
