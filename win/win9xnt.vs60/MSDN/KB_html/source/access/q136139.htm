ACC95: Supported File Types for Import, Export, and Link

 Last reviewed: January 30, 1998
Article ID: Q136139
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article lists the supported file types for importing, exporting, and linking data using the various ISAM drivers installed by Microsoft Access for Windows 95 version 7.0.

For information about supported file types in Microsoft Access for Windows version 8.0, search the Help Index for "supported file types," or ask the Microsoft Access 97 Office Assistant.

MORE INFORMATION

REFERENCES

For more information about importing, exporting, or linking data, search for "Supported File Types" using the Answer Wizard from the Microsoft Access for Windows 95 Help menu.
