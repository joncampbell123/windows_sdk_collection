ACC95: Microsoft Access Design Goals Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q139371
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

The Desgoals.exe files contains a document that discusses database market trends, the history of Microsoft Access, and the design goals of Microsoft Access for Windows 95. 

 ~ Desgoals.exe (size: 40117 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following file:

   Desgoals.doc    Microsoft Access for Windows 95: Design Goals:
                   Developer Edition
