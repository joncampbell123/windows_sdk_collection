ACC95: Summary of Files Upgraded by MS Access 7.0a

 Last reviewed: June 10, 1997
Article ID: Q151163
The information in this article applies to:
  • Microsoft Access version 7.0a

SUMMARY

The following is a list of files that were upgraded by Microsoft Access for Windows 95 Version 7.0a. No additional files are installed by the upgrade.

NOTE: When you click About Microsoft Access on the Help menu, the version number still reflects Version 7.00. To make sure that you have Version 7.00a installed, check the information on one of the individual files.
