ACC97: Internet ActiveX Sample Database Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q167268
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

Inetsamp.exe contains a sample Microsoft Access database application that demonstrates the use of the Winsock and the Msinet ActiveX controls. These controls are available with the Microsoft Office Developer Edition. You must have Microsoft Access 97 installed on your computer in order to use this sample database.
