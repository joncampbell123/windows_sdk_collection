ACC97: Can't Paste Special Link as Icon into Word or Excel

 Last reviewed: June 6, 1997
Article ID: Q167060
The information in this article applies to:
  • Microsoft Access 97

SYMPTOMS

Moderate: Requires basic macro, coding, and interoperability skills.

When pasting a Microsoft Access object into Microsoft Excel or Microsoft Word using the Paste Link command with the Display As Icon option, the image that is pasted onto the spreadsheet or document will have the following behavior.

In Microsoft Word

   The table will be pasted onto the document and you will not be able to
   activate the object by double-clicking it.

In Microsoft Excel

   A blank rectangle will be pasted onto the spreadsheet, and you will
   not be able to active the object by double-clicking it.

CAUSE

You cannot paste a Microsoft Access object into Microsoft Word or Microsoft Excel and display it as an icon.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Access 97. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Open Microsoft Excel or Microsoft Word.

  2. Open Microsoft Access and the sample database Northwind.mdb.

  3. In the Database window, select the Categories table and on the Edit menu, click Copy.

  4. Switch to Microsoft Excel or Microsoft Word and on the Edit menu, click Paste Special. In the Paste Special box, select Paste Link, click Paste As Icon, and then click OK.

    Note that the icon is pasted as a table in Microsoft Word and as a blank rectangle in Microsoft Excel.

REFERENCES

For more information about linking table data from Microsoft Access, search the Help Index for "linked tables, overview," or ask the Microsoft Access 97 Office Assistant.
