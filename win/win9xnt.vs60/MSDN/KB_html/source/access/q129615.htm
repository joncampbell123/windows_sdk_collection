ACC2: Tech*Ed 95: Sessions on Accessing External Data

 Last reviewed: October 21, 1997
Article ID: Q129615
The information in this article applies to:
  • Microsoft Access version 2.0

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The Microsoft Tech*Ed conference is an annual event for computer professionals who design, develop, integrate, or support Microsoft Office, Microsoft BackOffice, and Microsoft Windows-based solutions. The conference offers hundreds of technical detailed sessions about product development, integration, management, and future product directions.

The TE9517.EXE file contains documentation from a Tech*Ed 95 session about Microsoft Access version 2.0 strategies and techniques for working with external data sources such as Microsoft FoxPro, dBASE, Paradox, and Btrieve. Topics include discussions of recordsets versus snapshots, updateable views, and the Microsoft Access Upsizing Tools.
