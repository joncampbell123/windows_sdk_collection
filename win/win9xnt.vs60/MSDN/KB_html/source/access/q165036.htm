ACC97: Setup Program Prompts for Disk 7 When Using DMF Disks

 Last reviewed: May 5, 1997
Article ID: Q165036
The information in this article applies to:
  • Microsoft Access 97

SYMPTOMS

When you run the Microsoft Access 97 Setup program in maintenance mode on a computer on which you installed Microsoft Access using DMF floppy disks, and you select the Add/Remove or Remove All option, you are prompted for Disk 7 when the Setup program starts removing the files from your hard disk.

CAUSE

If you installed the Data Access component when you ran the Setup program, you will be prompted for Disk 7 when you click Add/Remove or Remove All.

RESOLUTION

To successfully remove all of the files from your hard disk, insert Disk 7 into the floppy drive when you are prompted.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Run Setup to do a complete install of Microsoft Access 97 from the DMF floppy disks.

  2. After Setup is finished successfully, start maintenance mode setup.

  3. Click Add/Remove or Remove All. If you choose Add/Remove, remove the Data Access component.

  4. Note that as the Setup program continues, you are prompted to insert Disk 7.

REFERENCES

For more information about removing Microsoft Access 97 components from your hard disk, search the Help Index for "Setup, adding/removing Microsoft Access components," or ask the Microsoft Access 97 Office Assistant.
