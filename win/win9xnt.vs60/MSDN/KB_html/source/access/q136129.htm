ACC95: Options Command Differences Between Versions 2.0 and 95

 Last reviewed: May 21, 1997
Article ID: Q136129
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

When you upgrade to Microsoft Access for Windows 95 version 7.0, you may notice that the Options command has moved to the Tools menu and many of its settings have been renamed or reorganized. This article lists the differences in the Option command settings between Microsoft Access version 2.0 and Microsoft Access 7.0.
