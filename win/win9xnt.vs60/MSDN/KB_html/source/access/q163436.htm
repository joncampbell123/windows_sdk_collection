ACC97: Removing Sample Databases Deletes Most Recently Used List

 Last reviewed: April 30, 1997
Article ID: Q163436
The information in this article applies to:
  • Microsoft Access 97
  • Microsoft Office 97 Professional Edition

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

After you run Setup to remove the sample databases for Microsoft Access 97, your most recently used file list is missing from the File menu.

For information about how to deliberately remove the most recently used list, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q162819
   TITLE     : ACC97: How to Remove the Most Recently Used (MRU) Files List

CAUSE

When you install the sample databases originally, the Setup program adds a selection for the Northwind sample database to the bottom of the File menu. When you run Setup to remove the sample databases, it removes the most recently used file list as part of removing the sample databases.

RESOLUTION

Run Setup again and reinstall the sample databases. Setup places the Northwind sample database at the top of the most recently used file list and moves all other databases on the list down one position. If the list is full, the last entry is removed from the list when Setup inserts Northwind at the top.

You can remove the sample databases from your computer manually by deleting the following files from the \Program Files\Microsoft Office\Office\Samples folder: 

   Northwind.mdb
   Orders.mdb
   Solutions.mdb

MORE INFORMATION

Steps to Reproduce Behavior

  1. Start Microsoft Access 97.

  2. Click the File menu and notice the list of most recently used databases at the bottom.

  3. Quit Microsoft Access 97.

  4. Run the Setup program for Microsoft Access.

  5. In the Installation Maintenance dialog box, click Add/Remove.

  6. If you have Microsoft Access 97 as a stand-alone product, click to clear the Sample Databases check box, and then click Continue.

    If you have Microsoft Office 97 Professional Edition, select Microsoft Access, and then click Change Option. Click to clear the Sample Databases check box, click OK, and then click Continue.

  7. When prompted to confirm that you want to remove three components, click Yes.

  8. After Setup is finished, start Microsoft Access 97.

  9. Click the File menu and notice that the databases you opened recently are no longer listed.

If you want to restore the most recently used file list, repeat steps 1-9 but click to select the Sample Databases check box instead of clearing it. Note that the list of most recently used databases is restored in Microsoft Access 97.
