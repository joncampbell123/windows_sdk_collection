ACC1x: Error in Object Permissions Table

 Last reviewed: April 2, 1997
Article ID: Q108437
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

SUMMARY

There is a table in the Microsoft Access "User's Guide" that summarizes the permissions available for objects in a database. This table contains an error. In version 1.0, the table is on page 621. In version 1.1, the table is on page 627.

The third and fourth lines in the table incorrectly show that Read Data and Modify Data permissions apply to form objects. These two permissions do not apply to form objects because forms display data from an underlying dynaset; they do not actually contain data. Therefore, your ability to modify data through a form is actually determined by the Read Data and Modify Data permissions on the form's underlying table or query.

STATUS

This problem no longer occurs in Microsoft Access version 2.0.

REFERENCES

Microsoft Access "User's Guide," version 1.0, pages 620-621

Microsoft Access "User's Guide," version 1.1, pages 626-627
