ACC1x: How to Remove Relationships

 Last reviewed: August 6, 1997
Article ID: Q92649
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

SUMMARY

This article explains how to delete a relationship in Microsoft Access after it has been created.

MORE INFORMATION

The sample database NWIND.MDB contains a relationship between the Employees and the Orders tables. The following steps demonstrate how to delete this relationship:

  1. Open sample database NWIND.MDB.

  2. From the Edit menu, choose Relationships.

  3. In the Primary Table box, select Employees.

  4. In the Related Table box, select Orders. Because these tables already have a relationship, Microsoft Access displays Employee ID in the Select Matching Fields box, and makes the Add button unavailable.

  5. Choose the Delete button to remove the existing relationship.

  6. Choose Close to exit the Relationships dialog box.

Note that you can also view existing relationships by creating a query. Include all the tables in your database in the query, and Microsoft Access will display join lines to indicate existing relationships.
