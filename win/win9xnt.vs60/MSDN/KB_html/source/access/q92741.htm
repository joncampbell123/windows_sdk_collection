ACC: Choosing a New Printer Resets Some Printer Settings

 Last reviewed: May 28, 1997
Article ID: Q92741
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0, 7.0, 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

When you change printers in Microsoft Access, some of the printer settings for your forms and reports change.

CAUSE

This functionality is controlled by the Windows Common Dialog dynamic-link library (DLL). The same behavior occurs in Microsoft Write.

STATUS

This behavior is by design.

MORE INFORMATION

Microsoft Access retains certain printer settings with form and report objects.

Some of these settings will not be transferable between printer drivers. This being the case, when the printer driver is changed from within the report, some settings will revert to the default settings for that printer.

This happens when the printer is explicitly changed in the File Print Setup option within the report, or when the printer is implicitly changed in the report by changing the default printer in Microsoft Windows.

This behavior can also occur when the printer port changes.
