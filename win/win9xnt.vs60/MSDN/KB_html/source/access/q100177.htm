ACC: Importing, Exporting, and Attaching FoxPro 2.0, 2.5 Data

 Last reviewed: April 2, 1997
Article ID: Q100177
The information in this article applies to:
  • Microsoft Access versions 1.1, 2.0, 7.0, 97

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

In Microsoft Access 1.1, 2.0, 7.0, and 97, you can import from, export to, and link (attach) data in FoxPro versions 2.0 and 2.5.

MORE INFORMATION

If you point to Get External Data on the File menu, and then click either Import or Link Tables, you can choose FoxPro 2.0 or FoxPro 2.5 from the Data Source list box.

NOTE: In Microsoft Access 1.1 or 2.0, click either Import or Attach Table on the File menu.

When you link (attach) a FoxPro file, you can also associate the appropriate FoxPro index (.idx or .cdx file). The procedure to import, export, and link files is the same for FoxPro files as it is for dBASE and Paradox files.
