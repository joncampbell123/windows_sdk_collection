ACC: Default Width in Report Design Does Not Reflect Margins

 Last reviewed: May 28, 1997
Article ID: Q92670
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0, 7.0, 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

The default grid area is independent of how a report or form is printed. The controls you place in the document determine the size.

In the United States, the default width of the grid area is set to 5 inches, regardless of the print settings (paper size, orientation, and margins). There is no relationship between the default grid area size and the default paper size or print area.

STATUS

This behavior is by design.

REFERENCES

For more information about how to change the default layout for forms or reports, search for "templates, forms and reports," and then "Specify a new template for forms and reports" using the Microsoft Access Help Index.
