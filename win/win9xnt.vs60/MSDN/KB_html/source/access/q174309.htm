ACC97: MS Office 97 SR-1 Patch Fails to Patch MS Jet Files

 Last reviewed: September 25, 1997
Article ID: Q174309
The information in this article applies to:
  • Microsoft Access 97
  • Microsoft Office 97 Professional Edition

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

The Microsoft Office 97 Service Release 1 (SR-1) Patch may fail to upgrade the Microsoft Jet database engine components.

CAUSE

The version of the Jet database engine on your computer is not the same as that which shipped with Microsoft Access 97. For example, if you have installed Microsoft Visual Basic 5.0, the version of the Jet database engine on your computer is more recent than the version installed by Microsoft Access; therefore, it is not detected when you run the patch to update Microsoft Office 97.

RESOLUTION

Install the updated version of Microsoft Jet 3.5; this version of the Jet database engine is more recent than that included in the Microsoft Office 97 Service Release 1 (SR-1) Patch.

For information about how to obtain this update, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q172733
   TITLE     : ACC97: Updated Version of Microsoft Jet 3.5 Available on
               MSL

MORE INFORMATION

Steps to Reproduce Problem

  1. Install Microsoft Office 97 Professional Edition.

  2. Install Microsoft Visual Basic version 5.0.

  3. Open Windows Explorer and note the size and date of Msjet35.dll in your Windows\system folder (or your Windows\System32 folder if you are using Microsoft Windows NT).

  4. Run the Microsoft Office 97 Service Release 1 (SR-1) Patch.

    Note that the size and date of Msjet35.dll have not changed; this indicates that the file has not been updated.
