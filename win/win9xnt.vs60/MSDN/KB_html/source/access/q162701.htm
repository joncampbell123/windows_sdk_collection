ACC97: MS Jet 3.5 Performance White Paper Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q162701
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

Advanced: Requires expert coding, interoperability, and multiuser skills.

The V35perf.exe file contains an overview of Microsoft Jet Database Engine version 3.5. The overview discusses the primary goal of Microsoft Jet Database Engine 3.5, benchmark timings, details on performance enhancements, and configurable performance settings. 

 ~ V35perf.exe (size: 712940 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following files:

   V35perf.doc     The Jet 3.5 Performance White Paper
   Readme.txt      Information on converting from Word 97
                   format
