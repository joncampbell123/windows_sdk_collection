ACC97: Access 97 Replica to Regular DB Wizard Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q168398
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The Wzrplrgl.mde file is an add-in for Microsoft Access 97 that enables you to convert the Design Master database from a replica set to its original, nonreplicated state. 

 ~ Wzrprg97.exe (size: 90021 bytes) 

   file is available for download from the Microsoft Software Library. It
   contains the following files:

   Wzrplrgl.mde    Replica to Regular Database Wizard
   Readme.txt      Information and installation instructions
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
