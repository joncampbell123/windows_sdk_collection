ACC95: Summary List of New Features: MS Access & ADT 95

 Last reviewed: April 25, 1997
Article ID: Q140211
The information in this article applies to:
  • Microsoft Access version 7.0
  • Microsoft Access Developer's Toolkit 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article provides a summary list of the features of Microsoft Access for Windows 95 version 7.0 and Microsoft Access Developer's Toolkit for Windows 95 version 7.0.
