ACC97: Paradox, Lotus, & MS Exchange/Outlook ISAMs in ValuPack

 Last reviewed: February 4, 1998
Article ID: Q159322
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The Paradox, Lotus 1-2-3, and Microsoft Exchange/Outlook indexed sequential access method (ISAM) drivers are not installed with the Microsoft Access 97 or Microsoft Office 97 Professional Setup; however, they are included with the Office 97 ValuPack software.

MORE INFORMATION

The Office 97 ValuPack contains additional software such as drivers, Help files, online books, and other useful programs and files. The Paradox, Lotus and Microsoft Exchange/Outlook ISAM drivers are in a file called DataAcc.exe, provided with the Office 97 ValuPack.

If you have Microsoft Access 97 or Microsoft Office 97 Professional on compact disc, the disc contains the Office 97 ValuPack in addition to the normal Setup program. Open the \ValuPack\DataAcc folder on the compact disc, and then double-click DataAcc.exe to run Setup for the ISAM drivers.

REFERENCES

For more information about features of the Office 97 ValuPack, search the Help Index for "ValuPack," or ask the Microsoft Access 97 Office Assistant.

For more information about which features are installed with Microsoft Access 97 Setup, search the Help Index for "installed components in Microsoft Access," or ask the Microsoft Access 97 Office Assistant.
