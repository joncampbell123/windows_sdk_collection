ACC95: Microsoft Jet Replication White Paper Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q138828
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

The Replicat.exe file contains the Microsoft Jet Database Replication white paper. This 42-page document discusses the Microsoft Jet Replication Architecture as well as how to implement replication, create replicas, synchronize data, resolve conflicts, and other topics. 

 ~ Replicat.exe (size: 77999 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following file:

   Replicat.doc    The Microsoft Jet Database Replication white paper.
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
