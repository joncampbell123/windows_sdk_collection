ACC: Microsoft Access Knowledge Base Help Files Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q167120
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0, 7.0, 97
  • Microsoft Access Distribution Kit version 1.1
  • Microsoft Access Developer's Toolkit versions 2.0, 7.0
  • Microsoft Office 97 Developer Edition Tools

SUMMARY

Each Knowledge Base (KB) Help file contains all the current KB articles for a particular version of Microsoft Access, the Microsoft Access Distribution Kit, the Microsoft Access Developer's Toolkit, and the Microsoft Office Developer Edition Tools. As with any Windows Help file, each file has a Contents list and can be searched using the Index tab and the Find tab.

NOTE: These help files are compatible with Windows 95 or Windows NT only.

Each of the exe files listed in this article includes the following:

  • A Help file. For example, Acc97kb.hlp.
  • A supporting Content file. For example, Acc97kb.cnt.
  • A Readme.txt file with information and installation instructions.

The following files are available for download from the Microsoft Software Library: 

 ~ Acc1xkb.exe (size: 1302934 bytes) 
                 Microsoft Access 1.x KB Help file.

 ~ Acc2kb.exe (size: 1605527 bytes) 
                 Microsoft Access 2.0 KB Help file.

 ~ Acc7kb.exe (size: 1479653 bytes) 
                 Microsoft Access 7.0 KB Help file.

 ~ Acc97kb.exe (size: 1328718 bytes) 
                 Microsoft Access 97 KB Help file.

 ~ Adk1xkb.exe (size: 72926 bytes) 
                 Microsoft Access Distribution Kit 1.1 KB Help file.

 ~ Adt2kb.exe (size: 103145 bytes) 
                 Microsoft Access Developer's Toolkit 2.0 KB Help file.

 ~ Adt7kb.exe (size: 87851 bytes) 
                 Microsoft Access Developer's Toolkit 7.0 KB Help file.

 ~ Ode97kb.exe (size: 130743 bytes) 
                 Microsoft Office 97 Developer Edition Tools KB Help file.
