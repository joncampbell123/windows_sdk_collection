ACC1x: Btrieve 6.0 Data Type Does Not Work with Microsoft Access

 Last reviewed: April 2, 1997
Article ID: Q93687
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

SUMMARY

Microsoft Access was not tested with Btrieve version 6.0. Therefore, new data types introduced with this version of Btrieve may not be compatible with Microsoft Access.

MORE INFORMATION

Btrieve 6.0 introduced the new data type Negative Sign Trailing Numeric Field. Microsoft Access does not explicitly translate this data type.

REFERENCES

Microsoft Access "User's Guide," versions 1.0 and 1.1, Chapter 4, "Importing, Exporting, and Attaching," page 66
