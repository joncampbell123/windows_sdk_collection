ACC95: Setup Folder Structure for Single-User Installation

 Last reviewed: April 25, 1997
Article ID: Q136063
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article lists the default folder structure used by the Setup program when you perform a single-user installation of Microsoft Access version 7.0.

MORE INFORMATION

c:\Msoffice\Access                     Bitmaps

REFERENCES

For more information about installing Microsoft Access components, search or "setup," and then "adding/removing Access components" using the Microsoft Access for Windows 95 Help Index.
