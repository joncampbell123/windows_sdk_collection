ACC97: Microsoft Access 97 Data Access Pack File Contents

 Last reviewed: June 24, 1997
Article ID: Q162752
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article lists the file contents of the Data Access Pack, Dataacc.exe, included with the Microsoft Office 97 ValuPack. The Data Access Pack contains additional drivers for Paradox, Lotus 1-2-3, and Microsoft Exchange/Outlook. It is located in the \ValuPack\Dataacc folder on the Microsoft Access 97 and Microsoft Office 97 Professional Edition CD-ROM.

NOTE: If you are using the Setup Wizard from Microsoft Office 97 Developer Edition for distributing custom run-time applications, you are not licensed to redistribute the Data Access Pack (Dataacc.exe). However, you are licensed to redistribute the individual drivers contained in the Data Access Pack, which are available as optional components for distribution in the Setup Wizard.
