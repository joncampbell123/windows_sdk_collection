ACC97: Error Converting Database To Design Master

 Last reviewed: September 12, 1997
Article ID: Q173775
The information in this article applies to:
  • Microsoft Access 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

When you attempt to replicate a database and convert it to a Design Master, you may receive the following error message: 

   Microsoft Access can't convert the database to a Design Master.

   Cannot make the <tablename> object in Tables container replicable.
When you click OK, you return to the Database window; the database has not been replicated.

CAUSE

You have installed the updated version of the Microsoft Jet Database Engine (version 3.51 of Msjet35.dll), but you have not installed the updated version of the replication dynamic-link library (Msrepl35.dll). For example, this problem occurs after you install Microsoft Money 98, which includes Microsoft Jet version 3.51, but not the updated version of Msrepl35.dll.

RESOLUTION

Obtain Jet35upd.exe from the Microsoft Web site; this file contains the updated version of Msrepl35.dll, as well as version 3.51 of the Microsoft Jet Database Engine.

For information on how to obtain Microsoft Jet 3.51, please see the the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q172733
   TITLE     : ACC97: Updated Version of Microsoft Jet 3.5 Available on MSL

MORE INFORMATION

To determine if you have installed Microsoft Jet version 3.51, open Windows Explorer and check the version of Msjet35.dll, which is located in the Windows\System folder (or the Windows\System32 folder if you are using Microsoft Windows NT). The updated Msjet35.dll has a version number of 3.51.0623.4. The version number of the updated Msrepl35.dll is 3.51.0623.0.
