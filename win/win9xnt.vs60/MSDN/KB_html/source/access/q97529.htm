ACC1x: GPF Using OpenForm Action to Hide Modal Pop-Up Form

 Last reviewed: May 14, 1997
Article ID: Q97529
The information in this article applies to:
  • Microsoft Access version 1.0

SYMPTOMS

Using the OpenForm action to hide a modal pop-up form may cause a general protection (GP) fault in Microsoft Access.

RESOLUTION

Instead of using the OpenForm action to hide a modal pop-up form, use the SetValue action to set the form's Visible property to False.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Access version 1.0. This problem no longer occurs in Microsoft Access version 1.1.
