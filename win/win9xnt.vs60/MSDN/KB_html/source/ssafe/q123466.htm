PATCH: FIXFRE.EXE Fixes Database Errors Reported by ANALYZE.EXE

 Last reviewed: March 12, 1997
Article ID: Q123466 
3.0x   | 3.0x | 3.0x    | 3.0x       | 3.0x
MS-DOS | UNIX | WINDOWS | WINDOWS NT | MACINTOSH kbtool kbfile

The information in this article applies to:

  • SourceSafe for MS-DOS, versions 3.0x
  • SourceSafe for Windows, versions 3.0x
  • SourceSafe for Macintosh, versions 3.0x
  • SourceSafe for Windows NT, versions 3.0x
  • SourceSafe for UNIX, versions 3.0x

SUMMARY

Fixfre.exe fixes a variety of database errors reported when running ANALYZE.EXE, such as:

  • Invalid extended header structure.
  • Error in extended header.
  • Error processing checkout free chain.
  • Checkout free chain contains a loop.
  • Checkout chain contains a loop.

The following file is available for download from the Microsoft Software Library: 

 ~ Fixfre.exe (size: 38631 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
MORE INFORMATION

Once you have Fixfre.exe, run it in an empty directory to obtain Fixfre.exe. Then use the following command to run FIXFRE.EXE: 

   FIXFRE <path to data dir>
Fixfre.exe is compiled for MS-DOS.

NOTE: If more than one user has checked out a file, all but the first user will be unchecked out of the SourceSafe database. To avoid this problem. Have all users Uncheckout or Update their files before running FIXFRE.
