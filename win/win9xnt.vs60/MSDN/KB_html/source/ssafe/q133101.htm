INFO: Visual SourceSafe Setup Uses ACME 95

 Last reviewed: February 28, 1997
Article ID: Q133101
4.00 5.00 | 4.00 5.00 
WINDOWS   | WINDOWS NT
kbsetup

The information in this article applies to:

  • Microsoft Visual SourceSafe for Windows NT and Windows 95, versions 4.0, 5.0

SUMMARY

Visual SourceSafe uses both the ACME 95 setup and a 16-bit setup. Visual SourceSafe installs on all 16-bit operating systems as it did before. However, the ACME 95 32-bit setup portion only works on certain 32-bit environments. The operating systems that have been tested successfully with Visual SourceSafe are:

  • Microsoft Windows 95
  • Microsoft Windows NT 3.51
  • Windows NT 4.0

To run Visual SourceSafe on any other Windows operating system:

  1. Perform a server install from one of the operating systems listed above that are supported by ACME 95 on the desired network server.

  2. From your machine, run Visual SourceSafe from the server.
