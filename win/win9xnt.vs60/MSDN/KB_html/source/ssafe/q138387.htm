Visual SourceSafe Viewer Accelerator Keys

 Last reviewed: May 30, 1996
Article ID: Q138387
The information in this article applies to:
  • Microsoft Visual SourceSafe for Windows, versions 4.0
  • Microsoft Visual SourceSafe for Windows NT, versions 4.0
  • Microsoft Visual SourceSafe for Windows 95, versions 4.0

The following accelerator keys are for the visual difference, UNIX and SourceSafe difference, and text windows. 

Key                 Description

CTRL+C              Copy text
CTRL+F              Find text
CTRL+G              Go to line #
SHIFT+CTRL+G        Go to line # - If a selection is active, the
                       accelerator key maintains the selection state and
                       modifies the selection
CTRL+N              Go to next difference (visual difference only)
SHIFT+CTRL+N        Go to next difference (visual difference only) - If
                       a selection is active, the accelerator key
                       maintains the selection state and modifies the
                       selection
SHIFT+CTRL+O        Select different text (visual difference only)
CTRL+P              Go to previous difference (visual difference only)
SHIFT+CTRL+P        Go to previous difference (visual difference only)
                       - If a selection is active, the accelerator key
                       maintains the selection state and modifies the
                       selection
ALT+BACKSPACE       Edit undo
SHIFT+BACKSPACE     Cut text
CTRL+INSERT         Copy text
HOME                Go to the beginning of the line
SHIFT+HOME          Go to the beginning of the line - If  a selection
                       is active, the accelerator key maintains the
                       selection state and modifies the selection
CTRL+HOME           Go to the beginning of the file
SHIFT+CTRL+HOME     Go to the beginning of the file - If  a selection
                       is active, the accelerator key maintains the
                       selection state and modifies the selection
PAGE UP             Scroll the display up one page
SHIFT+PAGE UP       Scroll the display up one page - If  a selection is
                        active, the accelerator key maintains the selection
                        state and modifies the selection
END                 Go to the end of the line
SHIFT+END           Go to the end of the line - If  a selection is
                       active, the accelerator key maintains the selection
                       state and modifies the selection
CTRL+END            Go to the end of the file
SHIFT+CTRL+END      Go to the end of the file -
PAGE DOWN           Scroll the display down one page
SHIFT+PAGE DOWN     Scroll the display down one page - If  a selection
                       is active, the accelerator key maintains the
                       selection state and modifies the selection
UP                  Scroll the display up one line
SHIFT+UP            Scroll the display up one line - If  a selection is
                       active, the accelerator key maintains the selection
                       state and modifies the selection
DOWN                Scroll the display down one line
SHIFT+DOWN          Scroll the display down one line - If  a selection
                       is active, the accelerator key maintains the
                       selection state and modifies the selection
LEFT                Scroll the display left one character
SHIFT+LEFT          Scroll the display left one character - If  a
                       selection is active, the accelerator key maintains
                       the selection state and modifies the selection
CTRL+LEFT           Scroll the display left one word
SHIFT+CTRL+LEFT     Scroll the display left one word - If  a selection
                       is active, the accelerator key maintains the
                       selection state and modifies the selection
RIGHT               Scroll the display right one character
SHIFT+RIGHT         Scroll the display right one character - If  a
                       selection is active, the accelerator key maintains
                       the selection state and modifies the selection
CTRL+RIGHT          Scroll the display right one word
SHIFT+CTRL+RIGHT    Scroll the display right one word - If  a selection
                       is active, the accelerator key maintains the
                       selection state and modifies the selection
F1                  Context sensitive help
F3                  Find next string
ALT+F3              Find text string
SHIFT+F4            Find next string
F5                  Refresh file list
