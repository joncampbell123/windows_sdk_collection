INFO: Cannot Set EOL Option from the Visual SourceSafe 5.0 GUI

 Last reviewed: November 10, 1997
Article ID: Q169535
The information in this article applies to:
  • Microsoft Visual SourceSafe, 32-bit, for Windows, version 5.0

SUMMARY

Unlike Visual SourceSafe 4.x, you cannot set the EOL option cannot within the Visual SourceSafe 5.0 Advance dialog box. In Visual SourceSafe 5.0, the EOL option has to be manually set in the Srcsafe.ini file. This is by design.
