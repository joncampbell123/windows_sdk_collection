SAMPLE: FIXPRNT Fixes Database Errors Reported by ANALYZE.EXE

 Last reviewed: May 23, 1997
Article ID: Q130178
The information in this article applies to:
  • SourceSafe for MS-DOS, version 3.0x
  • SourceSafe for Windows, version 3.0x
  • SourceSafe for Macintosh, version 3.0x
  • SourceSafe for Windows NT, version 3.0x
  • SourceSafe for UNIX, version 3.0x

SUMMARY

You can run Fixprnt.exe to repair various database errors reported by Analyze.exe such as: 

    - Error in Parent Chain.
    - Parent not found.
FIXPRNT will remove references to projects that no longer exist.

The following file is available for download from the Microsoft Software Library: 

 ~ Fixprnt.exe (size: 43701 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
After downloading Fixprnt.exe, run it in an empty directory to get Fixprnt.exe. Once you have Fixprnt.exe, use it as follows: 

      FIXPRNT <path to data directory>
NOTE: FIXPRNT modifies data files, so Microsoft strongly recommends that you back up your data before running FIXPRNT.
