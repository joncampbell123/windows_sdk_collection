HOWTO: Search for SourceSafe Articles Using KBKeywords

 Last reviewed: March 10, 1998
Article ID: Q133377
The information in this article applies to:
  • Microsoft Visual SourceSafe, 32-bit, for Windows, versions 4.0, 4.0a, 5.0
  • Microsoft Visual SourceSafe, 16-bit and 32-bit, for Windows, versions 4.0, 4.0a
  • Microsoft Visual SourceSafe, 16-bit, for Windows, version 4.0a
  • Microsoft SourceSafe for Macintosh, versions 3.1, 4.0
  • Microsoft SourceSafe for Windows, version 3.1
  • Microsoft SourceSafe for MS-DOS, version 3.x

SUMMARY

The Microsoft Knowledge Base (KB) is categorized by using keywords. This article lists the keywords specific to Microsoft SourceSafe and Microsoft Visual SourceSafe articles.

MORE INFORMATION

Microsoft SourceSafe KBKeywords

Each KB article in the SourceSafe collection may contain a product-specific keyword (called a KBKeyword) that places the article in an appropriate SourceSafe category.

Here are the topics and the corresponding KBKeywords. 

   Topic                                               KBKeyword
   -------------------------------------------------------------

   Setup, Configuration, INI files                     ssconfig
   Upgrading to SourceSafe (ddconv)                    ssconv
   Database issues (Analyze)                           ssdb
   SourceSafe integration with Access                  ssacc
   SourceSafe integration with FoxPro                  ssfox
   SourceSafe integration with FrontPage               ssfp
   SourceSafe integration with Visual Basic            ssvb
   SourceSafe integration with Visual C++              ssvc
   SourceSafe integration with Visual InterDev         ssidev
   SourceSafe integration (in general)                 ssint
   SourceSafe Command Line                             sscl
   SourceSafe Explorer and Admin                       ssui
   Using SourceSafe but not covering
   the areas above                                     ssusage
   Other SourceSafe issues                             ssother

Product-Specific Keywords

You can use the KBKeywords to organize SourceSafe articles or to search for specific groups of SourceSafe articles. For information about KBKeywords for other Microsoft developer products, please query the Microsoft Knowledge Base using these keywords: 

   dskbguide and kbkeyword

KB-wide Keywords

Each article in the SourceSafe collection also contains at least one generic, KB-wide keyword. The KB-wide keywords are standard throughout the Microsoft Knowledge Base, appearing in all KB articles, regardless of product. You can use the KB-wide keywords to organize all KB articles or to search for articles across several Microsoft products. For more information about these KB-wide keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
