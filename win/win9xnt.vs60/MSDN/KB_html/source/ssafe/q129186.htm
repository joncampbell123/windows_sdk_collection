PRB: Delta Labels Are Not Converted with DELTA_SS.EXE

 Last reviewed: February 28, 1997
Article ID: Q129186
3.10 4.0 | 3.10 4.00 5.00 | 3.10 4.00 5.00 
MS-DOS   | WINDOWS        | WINDOWS NT
kbtool kbprb

The information in this article applies to:

  • Microsoft Visual SourceSafe, versions 4.0, 5.0
  • SourceSafe for MS-DOS, version 3.1
  • SourceSafe for Windows, version 3.1
  • SourceSafe for Windows NT, version 3.1

SYMPTOMS

Delta Labels are not converted to SourceSafe after running Delta_SS.EXE.

STATUS

This behavior is by design.
