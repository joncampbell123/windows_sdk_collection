INFO: Common Speller API (CSAPI) SDK on the MSDN

 Last reviewed: January 12, 1998
Article ID: Q124053
The information in this article applies to:
  • Microsoft Developer Network (MSDN) Development Library
  • Microsoft Developer Network (MSDN) Level I CD

SUMMARY

The MSDN Development Library CD contains technical reference material that can help you develop Windows-based software. It includes technical articles, sample source code, books, journal articles, Microsoft technical white papers and specifications, as well as the Microsoft Knowledge Base and documentation for SDKs, DDKs, and tools such as Visual C++ and Visual Basic.

The Development Library differs from the Development Platform in that the Development Platform contains software development kits (SDKs) and device driver kits (DDKs) along with current versions of all Microsoft operating systems.

Beginning with the October release of the Development Platform, the Common Speller API is in \Specifications\Common Speller API (CSAPI) Specification. In the October release, it is documented in the Development Library in the Product Documentation, SDKs, Common Speller Application Programming Interface area of the CD.
