MSDN Setup Err Msg: Insufficient Disk Space

 Last reviewed: August 4, 1995
Article ID: Q108694
The information in this article applies to:
  • Microsoft Development Network (MSDN) Disc 4

SYMPTOMS

When you select the Minimum Install option during setup of MSDN Disc 4, the MSDN Setup program displays an error stating that there is insufficient disk space to install MSDN.

WORKAROUND

The easiest way to work around this is to:

  1. Select one of the other Setup options.

    -or-

  2. Run Microsoft Multimedia Viewer (MVIEWER2.EXE) directly from the CD- ROM and open the MSDNCD4.MVB file.

To use the minimal installation you must copy the Setup files to a floppy or to your hard drive and run Setup from that location. The files are:
