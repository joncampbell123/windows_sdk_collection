MSDN 8: Manual Installation Instructions

 Last reviewed: August 4, 1995
Article ID: Q119324
The information in this article applies to:
  • Microsoft Developer Network, Disk 8

If you encounter problems using Setup to install the Microsoft Developer Network (MSDN) Disk 8, you can use the directions in the MANSETUP.TXT file on the CD-ROM to perform a manual installation of the program. This file is located in the root directory of the MSDN disk.
