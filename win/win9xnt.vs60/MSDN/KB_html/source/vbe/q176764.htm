VBE: Not All Shortcuts Listed on Visual Basic Editor Menus

 Last reviewed: February 3, 1998
Article ID: Q176764
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SUMMARY

When you use the Visual Basic Editor in any of the applications listed above, some keyboard shortcuts do not appear on the menus. This article lists the keyboard shortcuts that are not displayed on the menus in the Visual Basic Editor.

MORE INFORMATION

The following table contains a list of keyboard shortcuts that do not appear on the menus in the Visual Basic Editor. 

Menu     Menu Command             Shortcut Key

Edit     Clear                    DELETE or DEL
Edit     Indent                   TAB
Edit     Outdent                  SHIFT+TAB
Debug    Step Into                F8
Debug    Step Over                SHIFT+F8
Debug    Step Out                 COMMAND+SHIFT+F8
Debug    Clear All Breakpoints    COMMAND+SHIFT+F9
View     Object                   SHIFT+F7
View     <Program Name>           OPTION+F11

REFERENCES

For more information about using shortcut keys in the Visual Basic Editor, click the Office Assistant, type "shortcut keys," click Search, and then click to view "Code Window Keyboard Shortcuts."

NOTE: If the Assistant is hidden, click the Office Assistant button on the Standard toolbar. If the product Help or Visual Basic Help are not installed on your computer, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q120802
   TITLE     : Office: How to Add/Remove a Single Office
               Program or Component
