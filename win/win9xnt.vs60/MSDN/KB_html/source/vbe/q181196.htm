OFF98: Cannot Customize Toolbars or Menu Bar in VB Editor

 Last reviewed: February 18, 1998
Article ID: Q181196
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SYMPTOMS

In the programs listed at the beginning of this article, the following problems occur in the Visual Basic Editor:

  • It is not possible to customize any of the following toolbars: 

          Standard
      Edit
      Debug
  • It is not possible to customize the menu bar or any of the shortcut menus.
  • You cannot undock the toolbars from the top of the screen.
  • When you start the Visual Basic Editor, the Standard toolbar appears. The Edit and Debug toolbars do not appear, even if you displayed them previously.

STATUS

This behavior is by design of the Visual Basic Editor in the programs listed at the beginning of this article.

MORE INFORMATION

In the programs included in Microsoft Office 97 for Windows, it is possible to customize the toolbars and the menu bar in the Visual Basic Editor by clicking the View menu, pointing to Toolbars, and clicking Customize. This functionality is not supported in the programs included in Microsoft Office 98 Macintosh Edition.

In the programs included in Microsoft Office 97 for Windows, it is also possible to undock toolbars, and when you display either the Edit toolbar or the Debug toolbar, the toolbar automatically appears the next time you start the Visual Basic Editor. These features are not available in the programs included in Microsoft Office 98 Macintosh Edition.
