XL: Properties Window Is Blank When Embedded Chart Selected

 Last reviewed: February 12, 1998
Article ID: Q180855
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In the versions of Microsoft Excel listed above, when you view the Visual Basic Properties window, the window may appear blank.

CAUSE

This problem may occur if the currently-selected object is an embedded chart contained in a worksheet.

Note that this problem does not occur if the active sheet is a chart sheet.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In Microsoft Excel 97 for Windows and Microsoft Excel 98 Macintosh Edition, you can view the Properties window by following these steps:

  1. On the Tools menu, point to Macro, and then click Visual Basic Editor.

  2. On the View menu, click Properties Window.

You can also view the Properties window in Microsoft Excel 97 for Windows by following these steps:

  1. On the View menu, point to Toolbars, and then click Control Toolbox.

  2. On the Control Toolbox toolbar, click the Properties button.

When you view the Properties window, it displays the properties of the selected sheet, chart sheet, module, UserForm, or control. Charts that are embedded do not have any properties to display in the window; therefore, the Properties window is blank when an embedded chart is selected.
