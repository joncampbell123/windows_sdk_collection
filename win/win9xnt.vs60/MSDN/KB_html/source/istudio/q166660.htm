INFO: Link View Does Not Prepend GOPHER:// To URLs

 Last reviewed: December 9, 1997
Article ID: Q166660
The information in this article applies to:
  • Microsoft Visual InterDev, version 1.0

SUMMARY

When you invoke "View Links on WWW" from the Tools menu and type in a URL without an explicit scheme or protocol, Link View assumes that the URL should be accessed via http://. For example, if you type in gopher.unc.edu, Link View assumes that you want to View Links on http://gopher.unc.edu, instead of gopher://gopher.unc.edu.

MORE INFORMATION

This is by design. The "View Links on WWW" does not have all the richness of the Internet Explorer address field in its ability to take incomplete URLs and convert them into fully qualified URLs.
