BUG: Uninstall Removes "New DataBase Wizard" from VC++

 Last reviewed: December 11, 1997
Article ID: Q166879
The information in this article applies to:
  • Microsoft Visual InterDev, version 1.0

SYMPTOMS

After uninstalling Visual InterDev 1.0 on a machine with Visual C++ 5.0 installed, "New DataBase Wizard" creation icons no longer appear in Visual C++'s File\New menus (Projects tab).

This issue applies to those who have just uninstalled Visual InterDev and who are now experiencing loss of functionality in Visual C++ 5.0.

CAUSE

Uninstalling Visual InterDev 1.0 removes registry keys that Visual C++ 5.0 depends on for providing use of some extended functionality.

RESOLUTION

Re-install Visual InterDev or Visual C++ for complete functionality.

STATUS

Microsoft has confirmed this to be a bug in the Microsoft products listed at the beginning of this article. We are researching this bug and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Install Visual InterDev and Visual C++ from the Visual Studio installation CD.

  2. Remove Visual InterDev via the Add/Remove icon in Control Panel.

  3. Launch Visual C++, and click New from the File menu (Projects tab).

The New DataBase Wizard icon no longer appears.
