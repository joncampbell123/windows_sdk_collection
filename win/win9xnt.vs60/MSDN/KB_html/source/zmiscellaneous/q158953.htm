Updated Display Drivers Included with DirectX

 Last reviewed: February 26, 1998
Article ID: Q158953
The information in this article applies to:
  • Microsoft Close Combat for Windows, version 1.0, 1.0a
  • Microsoft Deadly Tide for Windows, version 1.0
  • Microsoft Flight Simulator for Windows 95, version 6.0
  • Microsoft Gex version 1.0
  • Microsoft Hellbender for Windows, version 1.0
  • Microsoft Monster Truck Madness, version 1.0
  • Microsoft NBA Full Court Press for Windows, version 1.0
  • Microsoft Soccer version 1.0
  • DreamWorks Interactive, The Neverhood for Windows, version 1.0

SUMMARY

Following is a list of updated display drivers included with DirectX. These drivers are newer than those on the Windows 95 CD-ROM and are all certified for use with DirectX DirectDraw.

A DirectX-certified display driver is not required for these games. However, performance will be better with a certified driver.

NOTE: There may be newer drivers available from your hardware vendor.

MORE INFORMATION
