Can I Use Outlook Express Without Upgrading to MSN 2.5?

 Last reviewed: February 19, 1998
Article ID: Q174810
The information in this article applies to:
  • The Microsoft Network, versions 1.2, 1.3, 2.0

SUMMARY

Microsoft Outlook Express does not work with MSN, The Microsoft Network, version 2.0 or earlier. To use Outlook Express with MSN, you need to install MSN 2.5.

MORE INFORMATION

If you recently downloaded the new Internet Explorer, version 4.0 and are attempting to use Microsoft Outlook Express with MSN, you may be experiencing difficulty. This is because Outlook Express does not work with MSN, version 2.0 or earlier. To use Outlook Express with MSN, you need to install MSN, version 2.5.

NOTE: Internet Explorer 4.0 works with both MSN 2.0 and 2.5.

MSN 2.5, which includes both IE 4.0 and Outlook Express, has just been released and new compact discs are being issued to all of our current members. Once you receive your CD, you can install it and use MSN, IE 4.0 and Outlook Express together.

Until your new CD arrives, you will still be able to send and receive e- mail using Microsoft Outlook or Windows Messaging/Exchange (the Inbox).

To switch your default mail program back to Microsoft Outlook or Windows Messaging/Exchange, follow these steps:

  1. Using the right mouse button, click the Internet Explorer icon, and then click Properties.

  2. Click the Programs tab.

  3. In the Messaging area, click the down arrow next to the Mail box.

  4. Click either Microsoft Outlook or Windows Messaging.

  5. Click OK.
