MacWord 6.x Supplemental File Conversion Disk Available

 Last reviewed: October 14, 1997
Article ID: Q119859
6.00 6.0.1 MACINTOSH kbinterop kb3rdparty kbfile

The information in this article applies to:

  • Microsoft Word for the Macintosh version 6.0, 6.0.1

SUMMARY

A Supplemental File Conversion Disk is available for Word 6.x for the Macintosh. Converters supplied on this disk enable conversion to and/or from the following file formats:

  • MacWrite II version 1.x
  • Microsoft Word for MS DOS, versions 3.x through 6.0
  • Microsoft Word for Windows, versions 1.x (only required for export)
  • Microsoft Works for the Macintosh, versions 2.x and 3.x
  • WordPerfect for the Macintosh, versions 2.x and 3.0

TO OBTAIN THE SUPPLEMENTAL FILE CONVERSION DISK

You can find MW6SUP.HQX (size: 824319 bytes) 
                       , a self-extracting file, on the following
services:
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select the product.

          Enter kbfile MW6SUP.HQX (size: 824319 bytes) 
                             , and click GO!
      Open the article, and click the button to download the file.


  • Internet (anonymous FTP)

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder.
      Get MW6SUP.HQX (size: 824319 bytes) 

    

  • Microsoft Download Service (MSDL)

          Dial (425) 936-6735 to connect to MSDL
      Download MW6SUP.HQX (size: 824319 bytes)
