Introduction

The WV1225 disk included with this Application Note contains an updated Video for Windows Screen Capture tool that you can use in Windows 95. Note that this update is intended only for licensed users of Video for Windows version 1.1 who run Video for Windows in Windows 95.

Installing the Updated Screen Capture Tool

To install the updated Screen Capture tool, follow these steps:

  1. Insert the WV1225 disk in the appropriate floppy disk drive.

  2. Type the following commands at an MS-DOS prompt, pressing ENTER after each line: 

          cd \<windows>\system
      ren scrncap.drv scrncap.old
      cd \<winvideo>
      ren capscrn.hlp capscrn1.old
      ren capscrn.exe capscrn2.old
      copy <fdrive>:\scrncap.drv <hdisk>:\<windows>\system\scrncap.drv
      copy <fdrive>:\capscrn.hlp <hdisk>:\<winvideo>\capscrn.hlp
      copy <fdrive>:\capscrn.exe <hdisk>:\<winvideo>\capscrn.exe

    When you type these commands, substitute the name of your Windows folder for <windows>, the name of your Video for Windows folder for <winvideo>, the drive letter of the floppy disk drive containing the WV1225 disk for <fdrive>, and the drive letter of the appropriate hard disk for <hdisk>.

    For example, if the VW1225 disk is in drive A, your Windows folder is called Windows, your Video for Windows folder is called Winvideo, and the Windows and Winvideo folders are located on drive C, type the following commands: 

          cd \windows\system
      ren scrncap.drv scrncap.old
      cd \winvideo
      ren capscrn.hlp capscrn1.old
      ren capscrn.exe capscrn2.old
      copy a:\scrncap.drv c:\windows\system\scrncap.drv
      copy a:\capscrn.hlp c:\winvideo\capscrn.hlp
      copy a:\capscrn.exe c:\winvideo\capscrn.exe

  3. Use any text editor (such as Notepad) to edit the System.ini file. Add the following line to the [drivers] section of the file: 

          msvideo8=scrncap.drv

  4. Save and then close the System.ini file.

  5. Shut down Windows 95, and then restart your computer.

When you run Capscrn.exe, the Scrncap.drv device driver is loaded into memory. Msvideo.dll, the file responsible for loading this device driver, was changed in Windows 95 and requires the line you add in step 3 to load the device driver correctly.
