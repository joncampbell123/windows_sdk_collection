Microsoft Excel for Windows and Macintosh

NOTE: AppNotes that begin with the prefix XE apply to Excel for both Windows and Macintosh. 

XE0124     Regression Analysis and Best Fit Lines
           [NOTE: This replaces WE0124 and WE0126]
           No Disk Included
           Applies to Microsoft Excel 3.0, 4.0, 4.0a
           Software/Data Library: Not Available
           Last Update: 8/93
           MSDL: XE0124.EXE (Windows); XE0124.SEA (Macintosh)
           FastTips: Not available

   This Application Note discusses how to use Microsoft Excel
   functions to perform simple, multiple, and polynomial regression
   analysis. It contains examples of how to use LINEST(), LOGEST(),
   TREND(), and GROWTH() to describe a best fit line or curve and to
   make predictions about your data. It also outlines some of the new
   statistical functions and tools available with Microsoft Excel
   version 4.0.

XE0125     Cell Referencing from a Macro
           [NOTE: AppNote WE0125 was deleted and replaced with XE0125]
           No Disk Included
           Applies to Microsoft Excel versions 3.0, 4.0
           Software/Data Library: Not Available
           Last Update: 8/93
           MSDL: XE0125.TXT (Windows); XE0125.SEA (Macintosh)
           FastTips: Not Available

   This Application Note outlines how to reference cells from a macro
   and discusses the A1 and R1C1 reference styles. It includes tables
   that define terms such as "absolute reference" and "relative
   reference" and contains descriptions of specific macro functions
   and the types of cell referencing they will accept.

XE0127     Using Dates and Times
              [NOTE: AppNote WE0127 was deleted and replaced by XE0127]
           No Disk Included
           Applies to Microsoft Excel versions 1.0, 1.03, 1.04, 1.06,
           1.5, 2.2, 3.0, and 4.0, for the Macintosh; and to Microsoft
           Excel versions 2.0, 2.1, 2.1c, 2.1d, 3.0, and 4.0, for
           Windows
           Software/Data Library: Not Available
           Last Update: 6/93
           MSDL: XE0127.TXT
           FastTips: (800) 936-4100

   This application note describes how to use new and existing date
   and time functions in Microsoft Excel 4.0 and earlier versions, for
   both Windows and Macintosh. It also describes how dates and times
   are stored.

XE0183     Tips for Manipulating the Data in a Chart
           (Replaces ME0183)
           No Disk Included
           Applies to Microsoft Excel versions 3.0, 4.0, and 4.0a for

           Last Update: 11/93

   This Application Note provides and overview of how values are

XE0210      Creating and Using Tables
               [NOTE: AppNote number ME0210 was deleted and replaced by
               XE0210]
            No Disk Included
            Applies to Microsoft Excel versions 2.x, 3.0, 4.0, and 4.0a
            for Windows and Microsoft Excel versions 1.0, 1.03, 1.04,

            Last Update: 5/93

XE0449     Using Dynamic Dialog Boxes
           No Disk Included
           Applies to Microsoft Excel  versions 3.0 and 4.0 for Windows,
           and 3.0 and 4.0 for the Macintosh
           Software/Data Library: Not Available
           Last Update: 12/93
           MSDL:  XE0449_W.EXE(MS-DOS/Windows) or XE0449_M.SEA (Macintosh)

   This Application Note discusses how to create a dynamic dialog box and
   provides a sample dialog box definition table and sample macro code
   you can use to activate the dialog box. It also provides an example of

           Software/Data Library: Pending

           Disk Included

           Software/Data Library: S14205
           Last Update: 7/93
           MSDL: N/A
           FastTips: (800) 936-4100

   formats numbers and alphanumeric characters in text files. The
   ME0802 disk included with this Application Note contains the Text
