WD: File and Creator Types for All Versions

 Last reviewed: February 2, 1998
Article ID: Q63304

The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 1.0, 1.05, 3.0, 3.01, 3.02, 4.0, 5.0, 6.0, 6.0.1
  • Microsoft Word 98 Macintosh Edition

SUMMARY

This article lists the file types and creators for Microsoft Word versions 1.x, 3.x, 4.0, 5.0, 6.0, and 98.

Programmers and file-recovery programs wanting to access a particular file type sometimes require file type and creator information.

NOTE: The file format is specified in the Save As dialog box (from the File menu, choose Save As).

NOTE: Although Word versions 3.x and 4.0 have the same file types and creators, the file format for Word 4.0 is different. Consequently, Word 4.0 can read Word 3.x documents; however, Word 4.0 documents must be saved using the Word 3.0/Microsoft Write format for Word 3.x to read them correctly.
