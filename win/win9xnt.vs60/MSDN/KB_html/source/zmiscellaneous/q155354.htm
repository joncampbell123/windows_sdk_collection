Cannot Run Internet Explorer with Trumpet WinSock PPP

 Last reviewed: December 2, 1997
Article ID: Q155354
The information in this article applies to:
  • Microsoft Internet Explorer version 3.0, 3.01, 3.02, 4.0, 4.01 for Windows 95

SYMPTOMS

You may be unable to run Microsoft Internet Explorer version 3.0 for Windows 95 when the Trumpet WinSock Point-to-Point Protocol (PPP) is installed on your computer.

CAUSE

Trumpet WinSock uses a proprietary Windows Sockets interface that does not support all the Windows Sockets Application Programming Interface (API) calls used by Internet Explorer.

RESOLUTION

To allow Internet Explorer to function properly, you must remove Trumpet Winsock PPP. To remove Trumpet Winsock PPP from your computer, consult the documentation that came with the Trumpet Winsock software, or consult Trumpet Software International Pty Ltd, support.
