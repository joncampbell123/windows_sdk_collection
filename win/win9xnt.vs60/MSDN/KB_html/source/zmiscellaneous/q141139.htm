Organization Chart Not Installed with Typical or Compact Setup

 Last reviewed: March 11, 1998
Article ID: Q141139
The information in this article applies to:
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Office for Windows 95, version 7.0

SUMMARY

If you do a Typical or Compact installation of PowerPoint or Office, Microsoft Organization Chart is not installed.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft PowerPoint for Windows 95, version 7.0b

NOTE: PowerPoint 7.0b was released only as part of Office 7.0b and the stand-alone International versions of PowerPoint.
