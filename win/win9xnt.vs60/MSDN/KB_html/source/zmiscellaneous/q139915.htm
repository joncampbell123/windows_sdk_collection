PPT4: Contents of PowerPoint 7.0 Translator Disk (for Macintosh)

 Last reviewed: March 20, 1998
Article ID: Q139915
The information in this article applies to:
  • Microsoft PowerPoint for the Macintosh, version 4.0
  • Microsoft PowerPoint 7.0 Translator, version 1.0

SUMMARY

The following is a list of the contents of the PowerPoint 7.0 Translator Disk. This converters on this disk translate PowerPoint for Windows 95, version 7.0, files to PowerPoint for the Macintosh, version 4.0. 

   License Agreement (Text File)

   PP7Trans for Mac 68k (Folder)
      Readme (Text File)
      Pp7trans (68k version of the translator)

   PP7Trans for PowerMac (Folder)
      Readme (Text File)
      PP7TRANS (PowerMac version of the translator)

MORE INFORMATION

To install, drag the appropriate translator to your PowerPoint folder. The PowerPoint 7 Translator will now be available from the "List Files of Type" list.
