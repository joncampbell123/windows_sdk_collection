Availability of Internet Explorer 3.02 for Windows 95 and NT 4.0

 Last reviewed: November 24, 1997
Article ID: Q164475
The information in this article applies to:
  • Microsoft Internet Explorer version 3.02 for Windows 95
  • Microsoft Internet Explorer version 3.02 for Windows NT 4.0

SUMMARY

Microsoft has released Internet Explorer version 3.02 for Windows 95 and Windows NT version 4.0.

MORE INFORMATION

Internet Explorer version 3.02 includes the following new features:

  • Security Update 3.0b/3.01b for Internet Explorer

    For additional information about the Security Update, see the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q165163

          TITLE:      Description of Internet Explorer 3.0b/3.01b Security
                  Update


  • Automatic Proxy Configuration

    
   This feature allows the browser to automatically select a proxy server
   according to a specified configuration. Using INS files, administrators
   can dynamically change features within the browser. They can also use
   a combination of INS files and JavaScript to create browser
   configurations and advanced proxy configurations.


  • Updated User Agent String

    
   The User Agent string allows Web sites to detect the type of browser
   being used to access the site. After installing Internet Explorer
   version 3.02, the user agent string reported is one of the following,
   depending on the version installed:

    

        - Mozilla/2.0 (compatible; MSIE 3.02; Windows 95)
    - Mozilla/2.0 (compatible; MSIE 3.02; Windows NT)

    

  • Authenticode 2.0

    
   Authenticode is a technology that allows software publishers to sign
   their products, helping to ensure that content and files you download
   from the Internet are authentic and safe. Authenticode consists of two
   parts:

    

        - The certificate in the software being downloaded.

    

        - The verification code in Internet Explorer. If a software
      publisher's certificate is invalid, Internet Explorer prompts you
      and offers to cancel the download.

    
   Authenticode 2.0 includes the following new features:

    

        - Time-stamping support: This feature ensures that the software you
      download from the Internet is signed while the publisher's
      certificate is valid.

    

        - Support for new code-signing formats.

    

        - Renewal of internal certificates that expired on June 29, 1997.

    

In addition, Internet Explorer version 3.02 includes all the features
available in Internet Explorer versions 3.0 and 3.01.
