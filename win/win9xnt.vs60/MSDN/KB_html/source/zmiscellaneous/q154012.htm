How to Remove and Reinstall IE 3.0 for Windows 95

 Last reviewed: September 29, 1997
Article ID: Q154012
The information in this article applies to:
  • Microsoft Internet Explorer version 3.0 for Windows 95

SUMMARY

This article describes how to remove and reinstall Internet Explorer 3.0 for Windows 95.

MORE INFORMATION

Use the following steps to remove and reinstall Internet Explorer 3.0:

  1. Download a copy of the released version of Internet Explorer 3.0 from the following World Wide Web site. Make sure to save the file to your hard disk when you are prompted: 

          http://www.microsoft.com/ie/

  2. Uninstall Internet Explorer 3.0. To do so, follow these steps: 

        a. Click the Start button, point to Settings, and then click Control
       Panel.


        b. In Control Panel, double-click Add/Remove Programs.


        c. In the list of installed programs, click Internet Explorer 3.0.


        d. Click Add/Remove.


        e. Click Yes.

  3. Use My Computer or Windows Explorer to remove the Microsoft Internet and Windows\Temporary Internet Files folders. To do so, select the folders one at a time, and then on the File menu, click Delete.

    NOTE: If you first installed Internet Explorer by installing the Internet Jumpstart Kit included with Microsoft Plus! for Windows 95, the Microsoft Internet folder is most likely located in the Program Files/Plus! folder. If you first installed Internet Explorer from an online source (such as the Microsoft World Wide Web site or The Microsoft Network), the Microsoft Internet folder is located in the folder in which you chose to install Internet Explorer. Use the following steps to locate the Microsoft Internet folder: 

        a. Click the Start button, point to Find, and then click Files Or
       Folders.


        b. In the Named box, type "microsoft internet" (without quotation
       marks), and then click Find Now.

  4. Install Internet Explorer 3.0 again by double-clicking the Internet Explorer Setup file you downloaded in step 1.
