WD6X:WinWord 6.0 Text Converter Locations and Their .INI Entries

 Last reviewed: February 2, 1998
Article ID: Q106257
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a, 6.0c

SUMMARY

The following is a list of file locations and .INI entries for all the text converters that come with Microsoft Word version 6.0 for Windows and its supplemental text converters.

MORE INFORMATION

Text converters supplied with Word 6.0 for Windows are either shared with other programs or not (that is, used only by Word 6.0). If Word installs a converter in the C:\Windows\Msapps\Textconv directory (where C:\Windows is the location of your Windows program directory), it is a shared converter. If Word installs a converter in the Word 6.0 program directory, it is a non- shared converter. All shared converters are registered in the Win.ini file under the [MS Text Converters] section. All non-shared converters are registered in the Winword6.ini file under the [MSWord Text Converters] section.

Word 6.0 for Windows Text Converters and Their Associated Files

Shared Word 6.0 Converters (placed in the Textconv directory)

   Word for Windows 6.0                  (Allows Word for Windows 2.0
                                         to read Word for Windows 6.0
                                         documents)
   Word for Windows 2.0                  (Only required for export)
   Word for MS-DOS 3.x-6.0
   Word for the Macintosh 4.0, 5.0, 5.1  (Only required for export)
   Windows Write
   Microsoft Excel 2.x, 3.x, 4.x, 5.0    (Import only)
   WordPerfect 5.0 and 5.1 for MS-DOS
      and 5.1 and 5.2 for Windows
   RFT-DCA

Non-shared Word 6.0 Converters (placed in the Word program directory)

   Text with Layout/MS-DOS Text with Layout

Word 6.0 Font Mapping Files (placed in the Textconv directory)

   rtf_dca.txt  (RFT-DCA font mapping file, only used for export)
   rtf_mw5.txt  (Word for the Macintosh 5.0 font mapping file, only
                   used for export)
   rtf_pcw.txt  (Word for MS-DOS font mapping file, only used for
                   export)
   rtf_wp5.txt  (WordPerfect 5.1 for MS-DOS, and 5.1 and 5.2 for
                   Windows font mapping file, only used for export)

   Text Only/MS-DOS Text Only
   Text with Line Breaks/MS-DOS Text with Line Breaks
   Rich Text Format (RTF)

   Word for Windows 1.x   (Imports internally; exports using a converter)
   Word for Windows 2.0   (Imports internally; exports using a converter)

   Word for the Macintosh 4.x (Imports internally; exports using a
   converter)

   Word for the Macintosh 5.x (Imports internally; exports using a
   converter)

Supplemental Text Converters and Their Associated Files

Shared Supplemental Converters (placed in the Textconv directory)

   Word for Windows versions 1.x (only required for export)
   Microsoft Publisher for Windows version 2.0
   Microsoft Works for MS-DOS 1.0 and 2.0
   Microsoft Works for Windows 2.0
   Microsoft Works for Windows 3.0

Non-shared Supplemental Converters (placed in the Word program directory)

   DisplayWrite versions 4v1, 4v2, and 5.0 formats (Import only)
   Lotus 1-2-3 versions 2.x and 3.x formats        (Import only)
   Multiplan versions 3.x and 4.2 formats          (Import only)
   MultiMate versions 3.3, MultiMate Advantage, MultiMate Advantage II,
      and 4.0 formats
   WordPerfect 4.1 and 4.2 formats
   WordStar versions 3.3, 3.45, 4.0, 5.0, 5.5, 6.0, and 7.0 formats

Supplemental Font Mapping Files (placed in the Textconv directory)

   rtf_wwp.txt   (Works for MS-DOS font mapping file, only used for
                    export)
Supplemental Word for MS-DOS Glossary Files

(placed in the C:\Winword\Template directory)

   50convrt.gly  (Import only)
   55convrt.gly  (Import only)
   60convrt.gly  (Import only)
   msword.dot    (Template used to import Word for MS-DOS glossary files)
