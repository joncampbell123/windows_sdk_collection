WD: Graphics Lost Saving from WinWord to MacWord 5.x Format

 Last reviewed: February 3, 1998
Article ID: Q111372
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Macintosh, versions 4.0, 5.0, 5.0a, 5.1, 5.1a

SYMPTOM

When a Word for Windows or Word for Windows NTdocument containing pictures (graphics) in either the text or drawing layer is saved in a Microsoft Word for the Macintosh 5.0 or 5.1 format, the pictures disappear when you open the document in Word for the Macintosh.

CAUSE

The versions of Word listed at the beginning of this article do convert of graphics when you save a document in Word 5.x for the Macintosh format.

WORKAROUND

To retain graphics originally created in the text layer, save your Word for Windows or Word for Windows NT document in its native format, transfer the document file to the Macintosh, and then open the file in Word 5.x for the Macintosh using the Word 6.x/7.0 converter. For more information on how to obtain this converter, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q105319
   TITLE     : WD: How to Obtain a Word 6.x/7.0 Converter for MacWord
               5.x Users
To retain graphics created in the Word 6.x/7.0 drawing layer you must first move them to the text layer: the drawing layer is supported in Word 5.x for the Macintosh.

If you save the document from Word 97 for Windows, save it in Word 6.0 or 7.0 format or in Rich Text Format, because there will not be a Word 97 converter available for Word 5.x for the Macintosh.

NOTE: If you complete these steps, and if pictures in the text layer are still not retained, run Word for the Macintosh Setup and install the Windows Metafile converter.

STATUS

A Word 6.x/7.0 Converter Kit (for Word 5.x for the Macintosh users) is now available. The converter supplied with this kit allows Word for the Macintosh users to open Word 6.0 for Windows documents and retain graphics during conversion.
