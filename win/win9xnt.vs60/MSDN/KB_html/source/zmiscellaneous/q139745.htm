OFF4: How to Add an Extension with Office 4.2.1

 Last reviewed: November 17, 1997
Article ID: Q139745
The information in this article applies to:
  • Microsoft PowerPoint for the Macintosh, version 4.0
  • Microsoft Office for the Macintosh, version 4.2.1

SUMMARY

During the installation of Microsoft Office, the extensions that support the individual Office programs (such as PowerPoint, Microsoft Excel, and Word) are placed in your system's Extensions folder. Some of these extensions are critical for proper operation of the program. If a particular extension is damaged or missing, you can use the Office Setup program to restore it.

MORE INFORMATION

This process adds any missing extension(s) that PowerPoint installs.

NOTE: Should you need to use this method with Microsoft Excel or Word, you do not need to remove the PowerPoint Settings (4) file or the Spelling component from Office.

How to Add an Extension with PowerPoint

  1. Drag the extension you want to replace from the Extensions folder to the desktop. If the extension is missing, go to Step 2.

  2. Run Microsoft Office Setup from Install Disk 1 or your compact disc.

  3. Click Add/Remove when the Microsoft Office Setup (Maintenance Mode) dialog box appears.

  4. Under Installed Components, click the option triangle for Microsoft PowerPoint.

  5. All options will be selected by default. Leave the Microsoft PowerPoint option on the second level selected. Clear the remaining items.

  6. Under Installed Components, click the option triangle for Tools.

  7. Leave the Spelling option selected. Clear the remaining items.

  8. Click Continue to remove PowerPoint and Spelling. After the removal is complete, click OK.

  9. Drag the PowerPoint Settings (4) file from the Preferences folder to the trash.

  10. Run Microsoft Office Setup again and click Add/Remove.

  11. Under Components Not Yet Installed, click the option triangle for Microsoft PowerPoint.

  12. All listed options will be selected by default. Leave the Microsoft PowerPoint option on the second level selected. Clear the remaining items.

  13. Under Components Not Yet Installed, click the option triangle for Tools.

  14. Leave the Spelling option selected. Clear the remaining items.

  15. Click Continue to add PowerPoint and Spelling. After the installation is complete, click OK.

Which Program Installs What Extensions

Use the following list to determine which Office program you need to remove to obtain the missing or damaged extension. The list below was collected by installing just the single program without any of the components (templates, samples, and so on). Microsoft Office Setup installs the extensions listed below only if they do not exist within the Extensions folder. If the files do exist, the older versions will be retained.

PowerPoint 4.0

   Microsoft Dialog Library
   Microsoft Find File Library
   Microsoft Mail Library
   Microsoft OLE Extension
   Microsoft OLE Library
   Shared Code (Folder)
      OLE Client Dll
      OLE Reg Dll
      OLE Server Dll
   Shared Code Manager
Note: If you need to remove PowerPoint, you should also remove Spelling from the Tools section, along with PowerPoint.

Microsoft Excel 5.0

   Microsoft Dialog Library
   Microsoft Find File Library
   Microsoft Mail Library
   Microsoft OLE Automation
   Microsoft OLE library
   Microsoft Tool Editor Library
   ObjectSupportLib
   Type Libraries (Folder)
      Standard OLE Type (PowerMac)
      Vba En Olb (PowerMac)
   Visual Basic for Applications

Word 6.0.1

   Microsoft Dialog Library
   Microsoft Find File Library
   Microsoft Mail Library
   Microsoft OLE Extension
   Microsoft OLE Library
   Microsoft Tool Editor Library
