Internet Explorer Security Breach Patch Available

 Last reviewed: September 29, 1997
Article ID: Q157923
The information in this article applies to:
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01 for Windows 95
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01 for Windows NT 4.0

SUMMARY

Microsoft has released a patch for an Internet Explorer problem that allows security to be compromised on computers on which Internet Explorer for Windows 95 or Windows NT 4.0 is used to access Web pages.

If you are running Internet Explorer 3.0 for Windows 95 or Windows NT 4.0, you should download the Ie30icw.exe file from the Microsoft FTP site at: 

   ftp://ftp2.microsoft.com/msdownload/ieinstall/ie30icw.exe
If you are running Internet Explorer 3.01 for Windows 95 or Windows NT 4.0, you should download the Ie301icw.exe file from the Microsoft FTP site at: 

   ftp://ftp2.microsoft.com/msdownload/ieinstall/ie301icw.exe
If you are running Internet Explorer 2.0 for Windows 95, you should upgrade to Internet Explorer 3.02.

For information about obtaining Internet Explorer 3.02, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q164475
   TITLE     : Availability of Internet Explorer 3.02 for Windows 95 and
               NT 4.0

MORE INFORMATION

Microsoft recently became aware of a problem in Internet Explorer for Windows 95 and Windows NT 4.0. The problem allows Web site administrators to create Web pages that run programs on your computer and that prevent Internet Explorer from prompting you before the associated content is processed. Although there are very few Internet sites that are known to expose this problem, anyone using any of the versions of Internet Explorer listed above to access Web pages is at risk.

For additional information about this problem and the patch that resolves the problem, please see the following Microsoft Web site: 

   http://www.microsoft.com/ie/security/update.htm
For additional information, see the following article in the Microsoft Knowledge Base: 

   ARTICLE ID: Q165163
   TITLE:      Description of Internet Explorer 3.0b/3.01b Security Update
