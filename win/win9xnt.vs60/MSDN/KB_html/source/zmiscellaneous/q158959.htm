Files Installed by DirectX 1.0

 Last reviewed: February 26, 1998
Article ID: Q158959
The information in this article applies to:
  • Microsoft Close Combat for Windows, version 1.0
  • Microsoft Return of Arcade for Windows, version 1.0

SUMMARY

The following is a list of the DirectX system files installed to your hard drive by DirectX 1.0.

MORE INFORMATION

All files are marked as version 4.02.0095.

DirectDraw

  • Ddhelp.exe
  • Ddraw.dll
  • Ddraw16.dll

DirectPlay

  • Dplay.dll
  • Dpserial.dll
  • Dpwsock.dll

DirectSound

  • Dsound.dll
  • Dsound.vxd
