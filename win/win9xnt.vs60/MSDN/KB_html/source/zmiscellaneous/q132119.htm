How to Perform Net Setup of WordPerfect Conversions Kit

 Last reviewed: October 14, 1997
Article ID: Q132119
6.00 6.00a 6.00c WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

This article describes how to install the Supplemental WordPerfect Conversions Kit on a network so that users can set up the converters to use on their workstations.

MORE INFORMATION

Network installation is a two-part process. First, the Network Administrator performs an Administrative Setup, then each user at a workstation performs a Workstation Setup.

Part 1: Administrative Setup

If users on the network are running shared Windows from the server, the person performing the Network install must be running the same shared Windows, and must have all directory privileges to the shared Windows directory.

  1. If you are installing the Conversion Kit from a floppy disk, insert the disk in the appropriate floppy disk drive.

    If you downloaded WPCNV.EXE from an online service, you'll need to expand the files so they can be installed. Make sure the WPCNV.EXE file has been placed in a new directory before expanding the files it contains.

    IMPORTANT: Don't expand the files into the Word directory, as the Setup program will fail when run.

    To expand the files contained in the WPCNV.EXE file so that you can proceed to the next step:

    a. In File Manager, locate WPCNV.EXE and double-click the filename.

    b. Choose OK.

  2. In File Manager, click Run on the File menu.

  3. Type "<source>:\setup /A" (without the quotation marks) and press ENTER, where "<source>" is the directory containing the extracted contents of WPCNV.EXE, or if the kit was ordered on disk, the drive letter where the disk was inserted.

    For example, type 

           c:\download\setup /A

    -or- 

           a:\setup /A

  4. Specify the destination directory for the setup files. The default directory is WPKIT at the root level of the server.

  5. Specify the shared files directory. This directory is where the kit's shared components are stored, and is typically the MSAPPS directory on the server.

  6. Choose the location where shared files will be accessed by users who do subsequent Workstation installations from the server. The choices are: Server, Local Hard Drive, and User's Choice.

    The WordPerfect Kit Administrative Setup File Organization section below illustrates where the files are placed on the server.

Part 2: Workstation Setup

Each user at a Workstation will then need to run SETUP.EXE from the network server location. Each user performing a Workstation install must specify his or her own Word directory as the destination directory when prompted by Setup. The user's Word directory can reside on the local workstation, or on the server.

  1. Exit Microsoft Word if it is running.

  2. In Program Manager, click Run on the File menu.

  3. Type "<source>:\setup" (without the quotation marks) and press ENTER, where "<source>" is the path to the directory located on the server as specified by the Network Administrator in Step 4 in the Administrative Setup. (The default directory is WPKIT located at the root level.)

  4. At the screen where the user is prompted to choose the desired type of installation (Complete, Custom, or Workstation), the user must specify the user's Word directory.

    NOTE: The shared Word directory on the server may appear as the default destination directory in the dialog screen. However, the shared Word directory must only be selected during the Administrative Setup of the kit.

  5. Specify where shared files will be installed. The user has the choice of installing shared items to the local <Windows>\MSAPPS directory or the shared items directory on the server.

The WordPerfect Kit Workstation Setup File Organization section below illustrates where the files are placed.

WORDPERFECT KIT WORKSTATION SETUP FILE ORGANIZATION

The items in brackets represent directories on the local workstation.

<WINWORD> 

   WPCONV.DOC
   WPKSETUP.STF
   <TEMPLATE>
      WPDOS.DOT
      WPWIN.DOT
<WINDOWS> 
   <MSAPPS>
      <GRPHFLT>
         WPGEXP.FLT*+
         WPGIMP.FLT*+
      <TEXTCONV>
         WPFT5.CNV*+
         WPFT5.EQU*+
         WPFT6.CNV*+
   <SYSTEM>
      GREKSYMB.TTF+
      ICONSYEX.TTF+
      MATHEXT.TTF+
      MULTIEXT.TTF+
      TYPOEXT.TTF+

 * These items can be stored on the server if the user so specifies. The
   user will be able to specify where the starred items are stored if the
   Network Administrator chooses "User's Choice" in step 7 of
   Administrative Installation.

 + In a shared Windows environment, these items will be stored in the user
   Windows directory either on the server or on the local drive. Note that
   under shared Windows, a <Windows>\SYSTEM subdirectory does not exist.
   All files that would typically be installed in the <Windows>\SYSTEM
   subdirectory of a local install of Windows would be installed in the
   Windows directory.

WORDPERFECT KIT FILES INSTALLED

WPFT5.CNV      WordPerfect 5.x import/export converter
WPFT6.CNV      WordPerfect 6.x import only converter
WPFT5.EQU      WordPerfect Equations converter
WPGIMP.FLT     WordPerfect graphic import filter
WPGEXP.FLT     WordPerfect graphic export filter
WPCONV.DOC     WordPerfect Kit Documentation
WPWIN.DOT      Template: WordPerfect for Windows users
WPDOS.DOT      Template: WordPerfect for DOS users
WordPerfect Fonts: 

GREKSYMB.TTF   Greek Symbols
ICONSYEX.TTF   Iconic Symbols
MATHEXT.TTF    Math Ext
MULTIEXT.TTF   Multinational Ext
TYPOEXT.TTF    Typographic Ext
IMPORTANT NOTE: Before opening WordPerfect documents in Word 6.0 for Windows, make sure the Confirm Conversions check box is selected in the File Open dialog box. When opening a WordPerfect 6.0 or 6.1 document, you will need to manually select the WordPerfect 6.0 or 6.1 file type in the Convert File dialog box.

MORE INFORMATION

For information about the two templates and the features that are converted by the WordPerfect converters, open and read the WPCONV.DOC file that is installed in the Word for Windows program directory.

If you have questions about installing or using the converters supplied on this disk, in the United States call Product Support Services at (425) 462- 9673. Outside the United States, contact your local Microsoft subsidiary.
