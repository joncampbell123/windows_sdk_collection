PC WSPlus: Support for Timex Datalink and Schedule+ Interface

 Last reviewed: March 11, 1997
Article ID: Q121796
1.00 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Schedule+ for Windows, version 1.0

Timex began shipping their Data Link watches October 1, 1994. They have written software that allows you to interface with Schedule+. You can download data to the watch or extract data from Schedule+. Data Link is supported solely by Timex; the Data Link application will ship in their box.

Timex's toll-free technical support line is (800) 328-2677.

Data Link is manufactured by Timex, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
