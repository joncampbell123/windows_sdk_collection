IntelliPoint Setup Does Not Modify Autoexec.bat on Floppy Disk

 Last reviewed: September 12, 1996
Article ID: Q137999
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.1
  • Microsoft IntelliType software version 1.1

SYMPTOMS

When you set up the IntelliPoint or IntelliType software on a computer that starts from a floppy disk, the Autoexec.bat file on the hard disk is modified by Setup, rather than the Autoexec.bat file on the floppy disk.

CAUSE

Setup assumes that drive C is the boot disk, and modifies the startup files on that drive.

RESOLUTION

Transfer the changes made by Setup to the floppy disk. To do so, copy the modified lines from the Autoexec.bat file on the hard disk to the Autoexec.bat file on the floppy disk.

For example, copy the following lines to the Autoexec.bat file on the floppy disk: 

   c:\msinput\itype.exe
   set msinput=c:\msinput

STATUS
