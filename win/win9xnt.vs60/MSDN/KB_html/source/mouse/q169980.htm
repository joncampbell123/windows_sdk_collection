IntelliMouse Clicks Not Recognized on Packard Bell Computer

 Last reviewed: February 18, 1998
Article ID: Q169980
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

After you install a Microsoft IntelliMouse and the Microsoft IntelliPoint software version 2.0 on a Packard Bell computer, the computer may not respond to mouse clicks.

RESOLUTION

To resolve this issue, use either of the following methods:

Method 1

  1. Restart the computer. When you see the "Starting Windows 95" message, press the F8 key, and then choose Safe Mode Command Prompt Only from the Startup menu.

  2. Type the following lines, pressing ENTER after each line: 

          cd windows\system
      ren msmouse.vxd msmouse.bak
      ren msmouse.old msmouse.vxd

    NOTE: In Windows 95, the Msmouse.old file should be dated 7/11/95. If you are using Windows 95 OEM Service Release 2 (OSR2), the Msmouse.old file should be dated 8/24/96.

  3. Restart your computer normally.

Method 2

  1. In Control Panel, double-click Mouse.

  2. On the General tab, click Change.

  3. Click Show All Devices.

  4. In the Manufacturers box, click Microsoft.

  5. In the Models box, click Microsoft PS/2 Port Mouse.

  6. Click OK, and then click Close.

  7. When you are prompted to restart your computer, do so.

MORE INFORMATION

When you use one of the methods listed above, IntelliMouse wheel functionality will be disabled on your Packard Bell computer. For information about obtaining IntelliMouse wheel functionality, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q169511
   TITLE     : IntelliMouse Wheel May Not Function on Packard Bell
               Computer
