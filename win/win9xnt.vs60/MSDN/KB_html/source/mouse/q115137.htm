Right Mouse Button Doesn't Work in Links Golf

 Last reviewed: September 16, 1996
Article ID: Q115137
The information in this article applies to:
  • Microsoft Mouse driver for MS-DOS, version 9.0

SYMPTOMS

When you use the Microsoft Mouse driver version 9.0 with Links Challenge or Links 386 Pro Golf games, the right mouse button does not work.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Mouse driver version 9.00. This problem was corrected in Microsoft Mouse driver version 9.01.

RESOLUTION

To correct this problem, upgrade to version 9.01 of the Microsoft Mouse driver.
