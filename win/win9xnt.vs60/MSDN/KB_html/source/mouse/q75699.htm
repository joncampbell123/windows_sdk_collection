Installing a Microsoft Mouse During OS/2 1.3 Setup

 Last reviewed: September 16, 1996
Article ID: Q75699

SUMMARY

During setup for OS/2 version 1.3, several options are given for the Microsoft Mouse.

This information applies to all versions of the Microsoft Mouse and BallPoint Mouse.

MORE INFORMATION

During the installation of OS/2 version 1.3 (standard or extended edition), a menu of choices for mouse type is given. For more information on determining the exact type of Microsoft Mouse you have, query on the following words: 

   Mouse FCC ID
The OS/2 Setup menu for mice is as follows: 

OS/2 Screen Menu                              Type of Microsoft Mouse
----------------                              -----------------------

Microsoft Mouse, Bus Version (037-099)        Green button bus mouse
Microsoft Mouse, Bus Version (037-199)        Gray button bus mouse
Microsoft Mouse, Bus Version (056-X99)        None
Microsoft Mouse, InPort Version               400 ppi InPort(bus) mouse
Microsoft Mouse, Serial Version (039-099)     Green button serial mouse
Microsoft Mouse, Serial Version (039-199)     Gray button serial mouse &
                                              400 ppi Serial mouse
Microsoft Mouse, Serial Version (056-X99)     None
Microsoft Mouse, Micro Channel Version        Any MS Mouse on PS/2 port
PC Mouse Systems Mouse Visi-On Mouse None of the above

Installing a Mouse on a PS/2 Style Port

If you are attaching a Microsoft Mouse to a PS/2 style mouse port, choose Microsoft Mouse, Micro Channel Version. This choice will have the Setup program check for the type of machine you have, and will install the appropriate device driver.

Installing the BallPoint Mouse

If the BallPoint Mouse will be attached to serial port COM1 or COM2, make the same choice you would for a 400 ppi serial mouse. If the BallPoint Mouse will be on a PS/2 Style Port, then make the same choice as you would for any Microsoft Mouse on a PS/2 style port. Under all these choices, the BallPoint Mouse will work with the default settings (right handed). For more information on installing a BallPoint Mouse in OS/2, query on the words "Mouse BallPoint OS/2" in this Knowledge Base.

A choice to install a Microsoft Mouse during OS/2 1.3 installation will result in ONLY the Microsoft Mouse device drivers for that choice to be unpacked into the C:\OS2 directory. Choosing "None of the above" results in no drivers being unpacked, and no CONFIG.SYS lines added for the Mouse.
