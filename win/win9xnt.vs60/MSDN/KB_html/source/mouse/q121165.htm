Combining Special Effects from Standard and Home Mouse

 Last reviewed: September 12, 1996
Article ID: Q121165
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.0

SUMMARY

The special effects found in the Microsoft Home Mouse with IntelliPoint software 1.0 can be included in the Microsoft Standard Mouse with IntelliPoint software 1.0.

MORE INFORMATION

Both the Home and Standard Mouse versions of IntelliPoint include their own set of special effects. These special effects are loaded from the MSINPUT.INI file located in the MSINPUT directory. The special effects for the Home Mouse can be combined with the Standard Mouse by editing the MSINPUT.INI file to reflect entries for both and then copying the accompanying files to the correct directory.

To combine the Home and Standard Mouse special effects:

  1. Copy all the .WAV and .CUR files found with Home Mouse to the Standard Mouse directory. For example: 

          copy c:\home\*.wav c:\msinput\mouse\effects
      copy c:\home\*.cur c:\msinput\mouse\effects

    The special effects that ship with the Home Mouse with IntelliPoint version 1.0 are Unassigned, Magic, Nautical, Party Time, Pop-art and Sci-fi.

    The Special Effects that ship with the Standard Mouse with IntelliPoint version 1.0 are Unassigned, Comix, Construction, Fright Night, Jungle and Lazy Afternoon.
