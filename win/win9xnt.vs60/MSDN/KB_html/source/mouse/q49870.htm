Microsoft Paintbrush 4.00 Requires Mouse Driver 7.00

 Last reviewed: September 16, 1996
Article ID: Q49870

SUMMARY

Microsoft Paintbrush Version 4.00 requires Microsoft Mouse driver Version 7.00 or later.

If version of the Microsoft Mouse driver prior to 7.00 is installed, or if a non-Microsoft mouse driver is used, Paintbrush gives a "No mouse installed" error message. To correct the problem, run MSETUP from the Version 4.00 Microsoft Mouse Setup disk. If MOUSE.SYS was used previously, remove that line from your CONFIG.SYS file.
