Err Msg: IntelliPoint Has Detected That Another Microsoft...

 Last reviewed: February 18, 1998
Article ID: Q168938
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

When you change an IntelliPoint setting using the Mouse tool in Control Panel, you may receive the following error message: 

   IntelliPoint has detected that another Microsoft hardware application
   is currently open. For example, Microsoft IntelliType. Close the other
   Microsoft hardware application and then start IntelliPoint again.

CAUSE

When you enable an IntelliPoint feature and then restart your computer, Point32 is loaded in memory. The behavior described above can occur when the Pointer.exe file is also loaded in memory.

RESOLUTION

To resolve this problem, remove any reference to Pointer.exe from the "load=" or "run=" line in the Win.ini file, and then restart Windows 95.
