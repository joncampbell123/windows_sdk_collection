Operation of InPort Chip Versions 1.00 and 1.10 Versus 1.20

 Last reviewed: September 16, 1996
Article ID: Q49003
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.x
  • Microsoft InPort Mouse, version 1.x

SUMMARY

Versions 1.00 and 1.10 of the Microsoft Mouse InPort Chip operates in the timer interrupt mode (mouse service routine interrupt occurs at a fixed rate).

Version 1.20 of the InPort Chip operates in the data interrupt mode (mouse service routine interrupt occurs whenever mouse data is present).

This information applies to the Microsoft InPort Mouse, versions 1.00, 1.10, and 1.20.
