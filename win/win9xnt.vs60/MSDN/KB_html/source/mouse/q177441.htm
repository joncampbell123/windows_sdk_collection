IntelliPoint Features Do Not Work Using Multiple Input Devices

 Last reviewed: December 1, 1997
Article ID: Q177441
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.1
  • Microsoft IntelliMouse version 1.0

SYMPTOMS

Any of the following IntelliPoint features may not work:

  • Orientation
  • ClickLock
  • Pointer Speed

CAUSE

This problem can occur if the following conditions exist:

  • You have multiple input devices connected to your computer.
  • One (or more) of the input devices has a wheel (such as the IntelliMouse).
  • One (or more) of the input devices does not have a wheel.

STATUS

Microsoft has confirmed this to be a problem in Microsoft IntelliPoint version 2.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
