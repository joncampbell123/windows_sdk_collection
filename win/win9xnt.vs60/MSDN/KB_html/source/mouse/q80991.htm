Microsoft Mouse Driver 8.1 Update Kit Contents

 Last reviewed: September 16, 1996
Article ID: Q80991
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 8.1

SUMMARY

Below is a description of the contents of the Microsoft Mouse Driver Version 8.1 Update Kit. The contents vary depending on which version of the Microsoft Mouse driver you are upgrading from.

MORE INFORMATION

On every disk, there is a README.EXE file that provides an easy way to view the README.TXT contents.

Mouse

Mouse 5.25-inch and 3.5-inch disks (both low density)

If you are upgrading from a Microsoft Mouse driver version earlier than 8.0, you will receive the following:

  • Microsoft Mouse Setup disk.
  • "Microsoft Mouse User's Guide" that covers through versions 8.0.
  • A sheet of paper with installation instructions and a reference to the README.TXT file. This file contains information about the new features in Microsoft Mouse driver 8.1.
  • A copy of the Microsoft License Agreement.

If you are upgrading from version 8.0, you will not receive the "Microsoft Mouse User's Guide," but will get all other items mentioned above.

BallPoint

In this case, you will already have at least Microsoft Mouse driver version 8.0.

3.5-inch disk (only in low density)

  • BallPoint 1.0 Setup disk. (Note that this is version 1.0 of the BallPoint, not version 1.0 of the driver.)
  • A sheet of paper with installation instructions and a reference to the README.TXT file. This file contains information about the new features in Microsoft Mouse driver version 8.1.
  • A brief written description of the 8.1 features.
  • A copy of the Microsoft License Agreement.

5.25-inch disk (only in high density)

  • BallPoint 1.0 Setup disk. (Note that this is version 1.0 of the BallPoint, not version 1.0 of the driver.)
  • A copy of the Microsoft License Agreement.
