Symphony Mouse Menu Works in Text Mode Only

 Last reviewed: September 16, 1996
Article ID: Q28425
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.0

SUMMARY

When using the Lotus Symphony mouse menu, make sure Symphony is set up for "text and graphics separate." Microsoft Mouse Menus work in text mode only and do not support combined configurations (text with graphics).

MORE INFORMATION

If you are running Symphony with an EGA monitor, Symphony appears in green for "text and graphics separate" configurations, or in red for combined (text with graphics) configurations. To ensure that your Microsoft Mouse menus work correctly, verify that Symphony displays in green.
