Microsoft Mouse Choices During OS/2 1.2x Installation

 Last reviewed: September 16, 1996
Article ID: Q75698

SUMMARY

There are several choices for the Microsoft Mouse given during installation of OS/2. This information applies to all versions of the Microsoft Mouse and BallPoint Mouse.

MORE INFORMATION

During the installation of OS/2 version 1.2x, either Standard or Extended Edition, this menu of choices for Mouse type is given. To the right of each menu choice is the actual Microsoft Mouse it refers too. For more information on determining the exact type of Microsoft Mouse, query on the words: 

   Mouse and FCC and ID

   OS/2 Screen Menu                           Type of Microsoft Mouse
   ------------------                         -------------------------

   Microsoft Mouse, Bus Version (037-099)     Green Button Bus Mouse
   Microsoft Mouse, Bus Version (037-199)     Gray Button Bus Mouse
   Microsoft Mouse, Bus Version (056-X99)     None
   Microsoft Mouse, InPort Version            400ppi InPort(Bus) Mouse
   Microsoft Mouse, Serial Version (039-099)  Green Button Serial Mouse
   Microsoft Mouse, Serial Version (039-199)  Gray Button Serial Mouse
   Microsoft Mouse, Serial Version (056-X99)  400ppi Serial Mouse
   PC Mouse Systems Mouse
   Visi-On Mouse
   None of the above

What to Do for a PS/2-Style Mouse Port

If someone wants to install a Microsoft Serial Mouse on any PS/2 style port, they should choose the "Microsoft Mouse, Serial Version (056-X99)." This WILL NOT WORK, but will result in having to make the fewest changes. For more information on installing a Microsoft Serial Mouse on a PS/2 Style Mouse Port, query on the words: 

    Mouse and OS/2 and 1.2x and install

What to Do for the Microsoft BallPoint Mouse

If the BallPoint Mouse will be attached to either Serial Port COM1, or COM2 choose the same as you would for a 400ppi Serial Mouse. The BallPoint Mouse will work with the default settings (right handed). If it will be on a PS/2 Style Port make the same 400ppi Serial Mouse choice. As in the case of the 400ppi Serial Mouse, it will not work,but the changes to be made will be minimized with this choice.

Note: Making any choice for a Microsoft Mouse during installation of OS/2 1.2x will result in all the Microsoft Mouse Device Drivers being unpacked into the C:\OS2 directory.
