IntelliPoint Cut, Copy, and Paste Features Not Functioning

 Last reviewed: September 12, 1996
Article ID: Q129950
The information in this article applies to:
  • Microsoft IntelliPoint software versions 1.0, 1.0a

SYMPTOMS

When you use the cut, copy, and paste shortcut features that are included with the Microsoft IntelliPoint software, the selected text is not cut, copied, or pasted, although the feature is activated.

CAUSE

Cut, copy, and paste features that are provided by applications do not work with IntelliPoint's cut, copy, and paste features. If you activate the Cut, Copy, or Paste shortcut feature in the IntelliPoint Tools And Controls Manager, that functionality will not work in applications that provide that feature.

RESOLUTION

If the program you are using provides cut, copy, and paste functionality, disable this feature in Tools And Controls Manager.

STATUS

Microsoft has confirmed this to be a problem in Microsoft IntelliPoint versions 1.0 and 1.0a. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
