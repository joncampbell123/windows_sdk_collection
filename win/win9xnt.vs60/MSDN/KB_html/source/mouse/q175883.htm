The EasyBall Flip Cursor Feature Does Not Work

 Last reviewed: October 29, 1997
Article ID: Q175883
The information in this article applies to:
  • Microsoft EasyBall version 1.0
  • Microsoft IntelliPoint software version 2.1

SYMPTOMS

When you install the IntelliPoint software version 2.1, the EasyBall Flip Cursor feature may not work.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The EasyBall Flip Cursor feature reverses the direction your mouse pointer faces when you reach the edge of a document.
