Windows Setup Shows Mouse Systems Mouse After BallPoint Setup

 Last reviewed: September 16, 1996
Article ID: Q70177

SUMMARY

If you have a Mouse Systems mouse previously installed on a machine and you run the Microsoft BallPoint Mouse version 8.0 Setup program, BallPoint will install properly. However, when you bring up the Windows Setup choices again, it will indicate that you have a Mouse Systems mouse on the machine. The BallPoint Mouse will function normally, even though the Setup program displays the wrong mouse.

This problem was corrected in the Microsoft BallPoint Mouse version 8.1 Setup program.

This information applies to the Microsoft Ballpoint Mouse version 1.0 for Windows 3.0.
