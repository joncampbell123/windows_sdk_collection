SHIFT+Wheel Roll Closes Online User's Guide Topics

 Last reviewed: April 7, 1997
Article ID: Q165917
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliPoint software version 2.0
  • Microsoft Windows NT Server version 3.51
  • Microsoft Windows NT Workstation version 3.51

SYMPTOMS

When you press and hold down the SHIFT key while you roll the IntelliMouse wheel up, topics in the IntelliPoint Online User's Guide may close instead of opening.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
