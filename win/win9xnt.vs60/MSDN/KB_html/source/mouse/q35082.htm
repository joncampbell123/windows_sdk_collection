Mouse Versions Prior to New Mouse 1.00 Used MOUSE.SYS

 Last reviewed: September 16, 1996
Article ID: Q35082
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.0

SUMMARY

Although there is no difference in functionality between the MOUSE.SYS and MOUSE.COM drivers, versions of the Microsoft Mouse prior to the 400 series used MOUSE.SYS loaded from the CONFIG.SYS, rather than MOUSE.COM used via the AUTOEXEC.BAT file.

The MOUSE.COM driver provides greater flexibility in that it is able to deallocate from memory and is executable from MS-DOS.

This information applies to Microsoft Mouse driver versions 8.00 and earlier for MS-DOS systems.
