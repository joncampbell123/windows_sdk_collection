Serial Mouse May Fail with an Older AT&T 6300

 Last reviewed: September 16, 1996
Article ID: Q28681
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.0

SYMPTOMS

The mouse driver may not install correctly on an older AT&T 6300 system with a serial mouse.

RESOLUTION

Using the "/C1" or "/C2" switch may solve the problem.

If the problem continues, check the ROM date and upgrade the ROM if you are using an older one. AT&T recommends Version 1.43 ROM or later. Contact AT&T for ROM upgrade costs.

MORE INFORMATION

This information applies to the Microsoft Mouse driver versions 8.00 and earlier for MS-DOS systems.
