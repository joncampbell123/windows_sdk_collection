IntelliPoint 1.x Settings Not Saved After Upgrade to Version 2.1

 Last reviewed: October 28, 1997
Article ID: Q175774
The information in this article applies to:
  • Microsoft IntelliPoint software versions 1.0, 1.1, 2.1

SYMPTOMS

When you upgrade the IntelliPoint software from version 1.0 or 1.1 to version 2.1, your IntelliPoint 1.x settings may not be saved.

RESOLUTION

To correct this problem, select your preferred IntelliPoint software settings again. To do this, follow these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Mouse.

  3. Click each tab in the Mouse Properties dialog box and select the settings you want.

  4. Click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft IntelliPoint software version 2.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
