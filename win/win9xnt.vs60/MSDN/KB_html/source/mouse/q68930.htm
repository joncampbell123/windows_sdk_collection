CADKEY Requires Microsoft Mouse Driver Version 7.04 or Later

 Last reviewed: September 16, 1996
Article ID: Q68930
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, 8.x and 9.0

The CAD software packages CADKEY and CADKEY Light require Microsoft Mouse driver version 7.04 or later to run properly. Versions of the driver earlier than version 7.04 do not allow you to operate the mouse correctly with CADKEY. In one instance, such as when using driver version 7.0, the cursor is confined to a small area of the upper-left corner of the screen.
