No Scrolling and Panning Ability After Installing IntelliType

 Last reviewed: November 14, 1997
Article ID: Q165750
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall version 1.0
  • Microsoft IntelliType software version 1.1
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

In Microsoft Windows NT version 3.51, if you install the IntelliType software version 1.1 after installing the IntelliPoint software version 2.0, you can no longer use the IntelliMouse wheel and wheel button to scroll and pan.

RESOLUTION

To resolve this problem, reinstall the IntelliPoint software version 2.0.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The IntelliPoint software version 2.0 provides scrolling and panning functionality in some programs using the IntelliMouse wheel and wheel button. In a program that supports IntelliMouse wheel functions, you can scroll up and down by rolling the IntelliMouse wheel and you can pan around the program window by clicking and holding the IntelliMouse wheel button.
