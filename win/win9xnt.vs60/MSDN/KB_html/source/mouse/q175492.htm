The Pointing Device Stops Responding When You Click SmartSpeed

 Last reviewed: October 22, 1997
Article ID: Q175492
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

When you click the IntelliPoint SmartSpeed feature in the Mouse tool in Control Panel, the pointing device may stop responding (hang) or become very slow.

CAUSE

This behavior can occur if you have more than one copy of the IntelliPoint Point32.exe program running.

RESOLUTION

To resolve this issue, follow these steps:

  1. Press CTRL+ALT+DELETE to open the Close Program dialog box.

  2. Press the DOWN ARROW key to select Point32.exe, and then press ENTER.

  3. Press CTRL+ALT+DELETE to open the Close Program dialog box again.

  4. Press the DOWN ARROW key to select Mouse Properties, press ENTER, and then press ENTER again.

  5. Press CTRL+ESC to open the Start menu.

  6. Press the UP ARROW key to select Settings.

  7. Press the RIGHT ARROW key to select Control Panel, and then press ENTER.

  8. Use the ARROW keys to select Mouse, and then press ENTER. The pointing device should begin working correctly.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft product listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
