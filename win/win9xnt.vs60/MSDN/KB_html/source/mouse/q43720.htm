Microsoft Bus Mouse with NEC PowerMate

 Last reviewed: September 16, 1996
Article ID: Q43720
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 7.0

SUMMARY

When using the NEC PowerMate and the Microsoft Mouse with the bus interface, you must place the J3 mouse jumper on the secondary setting. The PowerMate can use the primary addressing for its screen I/O.

However, the computer hangs if you load mouse driver version 7.0 and use the J3 jumper on the bus mouse in the secondary setting. You must use mouse driver version 7.04 or later to make the mouse installation succeed.

This information applies to Microsoft Mouse driver versions 7.0 and 7.04 for MS-DOS systems.
