Microsoft Mouse Driver 9.0 Changes to the SYSTEM.INI File

 Last reviewed: September 16, 1996
Article ID: Q98237
The information in this article applies to:
  • Microsoft Mouse Driver for Windows, version 9.0
  • Microsoft Mouse Driver for MS-DOS, version 9.0

The following table describes the additions that the Microsoft Mouse driver version 9.0 adds to the Microsoft Windows SYSTEM.INI file. Affected section names are identified in brackets. 

   [Section] and Line                Explanation
   --------------------------------------------------------------

   [boot]

   Mouse.drv=c:\<dir>\mouse.drv      Forces Windows to use the
                                     driver in the path

   [boot.description]

   mouse.drv=Microsoft Mouse         Identifies Microsoft mouse
             version 9.00            driver version installed

   [386Enh]

   keyboard=c:\mouse\mousevkd.386    Specifies the virtual mouse
                                     driver path

   mouse=*vmd                        Force Windows to use the
                                     default virtual mouse driver
