Rolling the IntelliMouse Wheel Scrolls the Entire Web Page

 Last reviewed: January 26, 1998
Article ID: Q179724
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliPoint software versions 2.0, 2.1
  • Microsoft Internet Explorer version 4.01 for Windows 95
  • Microsoft Internet Explorer version 4.01 for Windows NT 4.0

SYMPTOMS

When you roll the IntelliMouse wheel in Internet Explorer, you may scroll the entire Web page.

CAUSE

This issue can occur if you use Internet Explorer 4.01 and you configure the IntelliMouse wheel to scroll one screen at a time.

RESOLUTION

To work around this issue, follow these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Mouse, and then click the Wheel tab.

  3. Under Wheel, click Settings, and then click "Scroll <n> lines at a time," where <n> is a number from -1 to 32767.

  4. Click OK, click OK, and then close Control Panel.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
