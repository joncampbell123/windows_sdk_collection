IntelliMouse Wheel Does Not Scroll in Windows Explorer

 Last reviewed: November 14, 1997
Article ID: Q169906
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall version 1.0
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

After you install the Microsoft IntelliMouse and the IntelliPoint version 2.0 software in Windows 95, you may not be able to scroll in Windows Explorer. When this behavior occurs, the wheel pans, but does not scroll.

CAUSE

This behavior occurs if you have Hi-Jaak Pro or Hi-Jaak 95 Graphics Suite installed. These products load a device driver named Runner that disables the ability to scroll when you use the IntelliMouse wheel.

RESOLUTION

To work around this problem, use the following steps:

  1. Press CTRL+ALT+DELETE to open the Close Program dialog box.

  2. Click Runner in the list of programs, and then click End Task.

If this resolves the problem, consult the Hi-Jaak product documentation or the program manufacturer for more assistance with this issue.
