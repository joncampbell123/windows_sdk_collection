TUTORIAL.EXE Error Message: "Undefined Variable at Line 21"

 Last reviewed: September 16, 1996
Article ID: Q96008
The information in this article applies to:
  • Microsoft Ballpoint Mouse, versions 1.0 and 2.0
  • Microsoft Mouse Driver for MS-DOS, versions 8.2 and 9.0

If you run TUTORIAL.EXE from the MS-DOS command line, the following error occurs: 

   Undefined Variable at line 21
   Press any key to continue.
TUTORIAL.EXE was not designed to be used as a stand-alone utility. To use the tutorial, run ORIENT.EXE from the MS-DOS command line and choose TUTORIAL from the main menu.

The Microsoft BallPoint tutorial is included with the Microsoft BallPoint Mouse installation disk. It is designed to introduce the features of the Microsoft BallPoint Mouse, Control Panel, and basic BallPoint techniques.
