Using the Microsoft Mouse with Open Access III

 Last reviewed: September 16, 1996
Article ID: Q68931
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.x

SUMMARY

Open Access III is an integrated function application that combines a database, a word processor, and a spreadsheet into one application. The product is developed and marketed by Software Products International.

You can use the Microsoft Mouse with Open Access III if you perform the following steps:

  1. Copy the Open Access III files MMENU.DEF and MMENU.MNU to the MOUSE1 subdirectory.

  2. Copy the file MENU.COM to the MOUSE1 subdirectory.

  3. Load the Microsoft Mouse driver.

  4. Go to the MOUSE1 subdirectory and type: 

          MENU MMENU

    You should get a "Menu Installed" message.

  5. Go to the Open Access III subdirectory and start the Open Access III program.

MORE INFORMATION

If this sequence does not work, you may have problems with memory resident software. In which case, perform the above steps after doing a "clean boot" with a copy of the original MS-DOS floppy disk.

You can obtain the MENU.COM program from Microsoft Product Support.
