IntelliPoint Mouse Driver Not Recognized at DOS on PS/2 Port

 Last reviewed: December 31, 1996
Article ID: Q125042
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.0
  • Microsoft Mouse driver for MS-DOS, version 10.0

SYMPTOMS

When you try to load the Microsoft Mouse version 10.0 driver, the following error message may appear: 

   Microsoft mouse not found: Try another 9 or 25 pin serial port
   or check for interrupt conflicts.
You can still use the mouse in Microsoft Windows.

CAUSE

This problem can occur on systems that have a hidden extended BIOS for PS/2 devices.

RESOLUTION

To install Microsoft Mouse driver version 10.0 for MS-DOS on such systems, add the following line to the [Mouse] section of the Msinput.ini file: 

   FailIfNoExtendedBIOS=FALSE
