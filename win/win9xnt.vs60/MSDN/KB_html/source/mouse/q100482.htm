Mouse Driver 9.0 Group Is Not Created During Setup

 Last reviewed: September 12, 1996
Article ID: Q100482
The information in this article applies to:
  • Microsoft Mouse Driver for Windows, version 9.0
  • Microsoft Windows operating system version 3.1
  • Microsoft IntelliPoint software version 1.0

If you have more than 40 groups in the PROGMAN.INI file, the Mouse group may not be created during Mouse 9.0 Setup, and you may not receive any error messages.

Since Windows 3.1 is limited to 40 groups in Program Manager, you must remove groups until you have fewer than 40, then run Mouse Setup again; or, add the group and icons to Program Manager manually.
