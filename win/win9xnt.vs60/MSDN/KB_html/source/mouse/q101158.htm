Mouse Driver 9.0 & Toolworks Animals for MS-DOS Hangs Machine

 Last reviewed: September 16, 1996
Article ID: Q101158
The information in this article applies to:
  • Microsoft Mouse driver for MS-DOS, version 9.0

SYMPTOMS

If you use the 9.0 Microsoft Mouse driver and the Software Toolworks multimedia program called Animals for DOS, your machine stops responding (hangs) or restarts when you play movies.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Mouse driver version 9.00. This problem was corrected in Microsoft Mouse driver version 9.01.

RESOLUTION

To correct this problem, upgrade to version 9.01 of the Microsoft Mouse driver.

MORE INFORMATION

For more information about Animals for DOS, contact Toolworks.
