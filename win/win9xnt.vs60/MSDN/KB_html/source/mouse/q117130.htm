BallPoint Mouse Is Erratic on Canon Notejet After Sleep Mode

 Last reviewed: September 16, 1996
Article ID: Q117130
The information in this article applies to:
  • Microsoft Windows operating system version 3.1
  • Microsoft Windows for Workgroups version 3.11
  • Microsoft BallPoint Mouse versions 1.0, 2.0
  • Microsoft Mouse driver versions 8.2, 9.0, 9.01

SYMPTOMS

When you use the Microsoft BallPoint Mouse on the PS/2 port of the Canon Notejet computer, the BallPoint Mouse behaves erratically when the computer resumes activity after being in sleep mode. The mouse pointer may jump to the left side of the screen or exhibit other unexpected behavior. The symptoms are specific to Microsoft Windows in 386 enhanced mode and do not occur in standard mode or in MS-DOS-based applications.

NOTE: The Microsoft BallPoint Mouse version 2.0 supports the PS/2 interface only; you cannot use this mouse on the serial port.

RESOLUTION

To correct this problem, obtain the updated BNPMC.SYS file from Canon. For more information, call Canon Technical Support.
