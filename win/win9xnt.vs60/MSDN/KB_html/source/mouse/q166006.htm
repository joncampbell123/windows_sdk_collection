ALT+TAB Does Not Switch Programs with Focus Feature Enabled

 Last reviewed: April 7, 1997
Article ID: Q166006
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

When you press ALT+TAB to switch from one program to another, the active window may not change.

CAUSE

If the Focus feature in the IntelliPoint software version 2.0 is enabled and you place the mouse pointer on the title bar of the active window, another program momentarily becomes the active window when you press ALT+TAB, but then the Focus feature returns focus to the window with the mouse pointer on its title bar.

RESOLUTION

To prevent this behavior, use one of the following methods:

  • Move the mouse pointer so that it does not rest on the title bar of the active window, and then press ALT+TAB.
  • Disable the Focus feature in the IntelliPoint software.
