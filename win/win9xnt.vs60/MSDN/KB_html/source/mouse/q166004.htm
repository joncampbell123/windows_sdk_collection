No Uninstall Option in IntelliPoint Setup Program

 Last reviewed: December 4, 1997
Article ID: Q166004
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.1

SYMPTOMS

After you install the Microsoft IntelliPoint software, you are unable to uninstall the software by running the IntelliPoint Setup program again.

CAUSE

The IntelliPoint Setup program does not have an option to uninstall the IntelliPoint software.

RESOLUTION

To uninstall the IntelliPoint software, follow these steps:

Windows NT Version 3.51

  1. Locate and open the Microsoft Input Devices program group in Program Manager.

  2. Double-click IntelliPoint Uninstall, and then click Finish.

Windows 95

  1. In Control Panel, double-click Add/Remove Programs, and then click the Install/Uninstall tab.

  2. In the list of installed programs, click Microsoft IntelliPoint 2.0, click Add/Remove, and then click Finish.
