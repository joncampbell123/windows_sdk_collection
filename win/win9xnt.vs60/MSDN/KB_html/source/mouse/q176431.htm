Text Selected or Window Scrolls When You Press the Wheel Button

 Last reviewed: December 3, 1997
Article ID: Q176431
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.1
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall, version 1.0

SYMPTOMS

When you press the IntelliMouse wheel button, you may experience one of the following problems:

  • Text is selected when you move the mouse. In addition, you cannot copy and paste the selected text by pressing CTRL+C and CRTL+V.
  • The window scrolls when you move the pointer outside of the document window.
  • The window scrolls when you move the pointer over the scroll arrows.

CAUSE

These problems can occur if the IntelliMouse wheel button assignment is set to Double-Click in the Mouse tool in Control Panel.

RESOLUTION

This problem is fixed in the latest version of the Point32.exe and Point32.dll files. You can obtain these files by downloading the IntelliPoint 21p32upd.exe update file. For information about obtaining this file, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q177274
   TITLE     : IntelliPoint Software 2.1: Point32 Update Available

MORE INFORMATION

When you press the IntelliMouse wheel button with the button assignment set to Double-Click, the mouse behaves as though the left mouse button is being held down. When you click another mouse button, this behavior stops.

You can change the Button Assignment setting by following these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Mouse.

  3. Click the Wheel tab.

  4. In the Button Assignment box, click a different option.

  5. Click OK.
