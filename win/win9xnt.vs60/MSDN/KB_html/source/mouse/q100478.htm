Setup Bitmaps Do Not Display on Hercules Monochrome Monitor

 Last reviewed: September 16, 1996
Article ID: Q100478
The information in this article applies to:
  • Microsoft Mouse Driver for Windows, version 9.0

When you install the Microsoft Mouse driver 9.0 with a Hercules monochrome monitor, some setup bitmaps and features may not display correctly. During Setup, Meet the Mouse and Orientation bitmaps may not display or may appear distorted. The Compass section of Orientation may also be affected.

Currently no workaround for this problem is available.

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

Note: The Setup program does install all the necessary files to set up and run the mouse.

The Hercules monochrome monitor is manufactured by Hercules Computer Technology, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
