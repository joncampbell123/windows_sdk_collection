Internal Processing Error and OS2MOUSE.V12 with Mouse

 Last reviewed: May 1, 1997
Article ID: Q63226
1.20 OS/2 kbsetup kbhw

The information in this article applies to:

  • Microsoft Mouse Driver for OS/2, version 1.2

SUMMARY

After you install the Microsoft Mouse Driver for OS/2, version 1.2 (OS2MOUSE.V12) and select the communications manager, you may receive an internal processing error and your system may stop (hang). This problem occurs because the driver interferes with the communications manager.

OS2MOUSE.V12 has now been fixed so that there is no interference with the communications manager. To obtain the updated version, contact Microsoft Product Support Services at (425) 454-2030.
