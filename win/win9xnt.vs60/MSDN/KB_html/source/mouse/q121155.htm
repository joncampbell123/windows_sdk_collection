Double-Clicking the Mouse Does Not Work in Windows

 Last reviewed: September 12, 1996
Article ID: Q121155
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.0
  • Microsoft Mouse driver versions 8.0, 9.0

SYMPTOMS

Inside Microsoft Windows, double-clicking your mouse does not work correctly, although the single click does work as expected. Adjusting the double-click speed has no effect.

CAUSE

This problem occurs when the following lines are set to either zero (0) or a letter (for example, x): 

   [Windows]
   doubleclickheight=
   doubleclickwidth=

RESOLUTION

  1. Open the WIN.INI file and search for the following lines: 

          doubleclickheight=
      doubleclickwidth=

  2. Change the values to a valid number (for example, 5 or 10).
