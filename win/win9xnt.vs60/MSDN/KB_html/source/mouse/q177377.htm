Computer Hangs Starting 16-Bit Lotus Notes

 Last reviewed: November 26, 1997
Article ID: Q177377
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.0a
  • Microsoft IntelliMouse version 1.0

SYMPTOMS

When you start the 16-bit version of Lotus Notes, your computer may stop responding (hang).

CAUSE

This behavior can occur if you are using Microsoft IntelliPoint version 2.0 when you start the 16-bit version of Lotus Notes.

RESOLUTION

To resolve this behavior, install IntelliPoint version 2.1 or later. To obtain the latest version of IntelliPoint, please contact Microsoft Sales at (800) 360-7561. Make sure to have your product identification number (PID) ready when you call.
