Command-Line Switches Are Allowed with MOUSE.SYS

 Last reviewed: September 16, 1996
Article ID: Q73065
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 8.0

SUMMARY

Command-line switches are allowed with the MOUSE.SYS version of the Microsoft Mouse driver version 8.0. The documentation on page 22 of the "Microsoft Mouse User's Guide" is incorrect. This information applies to the Microsoft Mouse driver version 8.0.

Older versions of MOUSE.SYS also support command line switches.

Reference(s):

"Microsoft Mouse User's Guide" version 8.0, document number: SY20990-0291.
