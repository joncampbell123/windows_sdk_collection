IntelliMouse Wheel Roll Does Not Close Context Menu

 Last reviewed: November 14, 1997
Article ID: Q165799
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall version 1.0
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

When a context menu is displayed on the screen in Windows 95, the context menu remains on the screen when you roll the IntelliMouse wheel.

CAUSE

The operating system controls the opening and closing of context menus based on various mouse or keyboard actions, but does not recognize rolling the IntelliMouse wheel as an action that should close the context menu.

RESOLUTION

To close the context menu, use one of the following methods:

  • Press the ESC key.
  • Use the left mouse button to click something other than the context menu.
