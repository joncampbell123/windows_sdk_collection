SnapTo Feature May Not Work in Mouse Orientation Tool

 Last reviewed: April 7, 1997
Article ID: Q166199
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

If the SnapTo feature is enabled when you use the Mouse tool in Control Panel to set the orientation of your mouse, the mouse pointer does not automatically move to the default button.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
