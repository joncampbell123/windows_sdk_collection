Point32 and Mouse Icon Stay Loaded with All Features Disabled

 Last reviewed: April 7, 1997
Article ID: Q165752
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

When you disable all IntelliPoint features and then restart your computer, Point32 is loaded into memory and then unloaded, and the IntelliPoint icon is not displayed on the taskbar. However, if you run the Mouse tool in Control Panel, Point32 remains loaded in memory and the IntelliPoint icon is displayed even after you close the Mouse tool.

CAUSE

When you start your computer, Point32 is loaded into memory. If no IntelliPoint features are enabled, Point32 is unloaded from memory. However, if you start the Mouse tool in Control Panel, Point32 is not unloaded when you close the Mouse tool.

RESOLUTION

To remove Point32 from memory, ensure that all IntelliPoint features are disabled, and then restart your computer.
