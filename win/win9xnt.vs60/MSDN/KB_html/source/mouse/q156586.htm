Command-Line Switches for Lzari.exe

 Last reviewed: September 26, 1996
Article ID: Q156586
The information in this article applies to:
  • Microsoft EasyBall version 1.0

SUMMARY

The files on the disk included with the EasyBall are compressed with a proprietary tool called Lzari.exe. This article describes the command- line switches for the Lzari.exe tool.

MORE INFORMATION

To obtain a list of the command-line switches for Lzari.exe, follow these steps:

  1. Insert disk 1 on the appropriate floppy disk drive.

  2. Type the following line at a command prompt 

          <drive>:\lzari /?

    Where <drive> is the floppy disk drive containing disk 1. For example, if you inserted the disk in drive A, type the following line: 

          a:\lzari /?

When you follow these steps, the following information is displayed on the screen: 

   Usage: [C|E] srcfile destfile
      C    compress srcfile to destfile
      E    expand srcfile to destfile
For example, to extract the Point32.exe file from the disk in drive A to the Msinput folder on drive C, type the following line: 

   a:lzari e point32.ex_ c:\msinput\point32.exe
