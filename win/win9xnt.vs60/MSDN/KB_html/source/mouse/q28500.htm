Use MOUSE.SYS with Flight Simulator

 Last reviewed: September 16, 1996
Article ID: Q28500
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 6.11

SUMMARY

If you have Microsoft Flight Simulator and the Microsoft Mouse driver version 6.11 installed, you should use the MOUSE.SYS file in the CONFIG.SYS file rather than the MOUSE.COM in the AUTOEXEC.BAT file that is installed by the Mouse Setup program.

MORE INFORMATION

The Microsoft Mouse driver version 6.11 (MOUSE.COM only) may cause Flight Simulator to continuously ask if you have a mouse. Using the MOUSE.SYS file solves the problem.
