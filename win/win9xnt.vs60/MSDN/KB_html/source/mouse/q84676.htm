8.2 POINTER.EXE Requires MOUSE.DRV 8.2 or Later

 Last reviewed: September 16, 1996
Article ID: Q84676
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 8.2
  • Microsoft Windows operating system versions 3.0, 3.00a, 3.1, 3.11

SYMPTOMS

When you try to run POINTER.EXE version 8.2 to change the size of the mouse cursor, the following message is displayed: 

   This Version of Pointer Options requires MOUSE.DRV version 8.2 or
   later.

CAUSE

The MOUSE.DRV= line in the SYSTEM.INI file may be pointing to a MOUSE.DRV file that is earlier than version 8.2.

Attempting to use POINTER.EXE with a third party mouse driver (such as Logitech's LMOUSE.DRV) also causes the above message.
