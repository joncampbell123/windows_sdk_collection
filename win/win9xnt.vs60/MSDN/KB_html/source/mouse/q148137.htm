Point32 Causes GP Fault in Visual Basic 4.0 16-Bit Menu Editor

 Last reviewed: October 18, 1996
Article ID: Q148137
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.1
  • Standard, Professional, and Enterprise editions of Microsoft Visual Basic, 16-bit only, for Windows, version 4.0

SYMPTOMS

If you have Microsoft IntelliPoint version 1.1 installed, you may receive a general protection (GP) fault in User.exe when you use the Visual Basic Menu Editor function in 16-bit mode.

CAUSE

Point32.exe, a tool included with the Microsoft IntelliPoint software version 1.1, and the 16-bit version of Microsoft Visual Basic 4.0 for Windows are not compatible.

RESOLUTION

Microsoft has released updated versions of the Point32.exe and Point32.dll files that correct this problem. For information about obtaining these updates, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q146216
   TITLE     : Updated Pointer Files Available
