Mouse 9.0 Locate Feature Disables File Manager File Selection

 Last reviewed: September 16, 1996
Article ID: Q109498
The information in this article applies to:
  • Microsoft Mouse driver for MS-DOS, version 9.0
  • Microsoft Mouse driver for Windows, version 9.0

SYMPTOMS

If the Locate feature of Mouse Manager is set to CTRL+Primary, the ability to randomly select more than one file in File Manager is disabled.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Mouse driver version 9.00. This problem was corrected in Microsoft Mouse driver version 9.01.

RESOLUTION

To reenable the ability to randomly select files, the Locate feature must be used with a key sequence other than the CTRL+Primary.
