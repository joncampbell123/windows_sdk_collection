Laplink from PC Tools Causes Mouse to Not Work

 Last reviewed: September 16, 1996
Article ID: Q72840
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 7.x and 8.x

SUMMARY

When laplink from PC Tools, version 6, is loaded, it causes the Microsoft Mouse to not work in PC Tools' PCShell and Desktop if either of these two programs is loaded as a TSR (terminate-and-stay resident program) by using the /r switch.

To enable the Microsoft Mouse in PCShell or Desktop, you must either load PCShell and Desktop without the /r switch, or you cannot have laplink running.
