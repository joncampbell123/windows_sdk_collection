"Microsoft Mouse Driver Not Found" Error Message

 Last reviewed: September 16, 1996
Article ID: Q32263
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.x

SUMMARY

The MOUSE.COM or MOUSE.SYS driver must be loaded before you attempt to load a mouse menu. The "Microsoft Mouse driver not found" error message is generated when a mouse expert menu tries to load without the mouse driver already loaded.

This error message should not be confused with the message "Driver not installed-- Microsoft Mouse not found", which is a mouse driver not loading message.
