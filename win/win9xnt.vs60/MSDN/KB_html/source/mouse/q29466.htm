Leading Edge Model D Error "Unexpected SW Interrupt 33"

 Last reviewed: September 16, 1996
Article ID: Q29466
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.0

SUMMARY

The error message "Unexpected SW Interrupt 33" can occur on some Leading Edge Model D computers when you install a mouse driver. You usually can continue after this error message has appeared by typing C to continue. For a patch, call Leading Edge technical support at (900) 370-4800.

This article applies to Microsoft Mouse driver versions 8.00 and earlier for MS-DOS systems.
