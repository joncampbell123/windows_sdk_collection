Explanation of Animated Cursor Availability in IntelliPoint

 Last reviewed: September 12, 1996
Article ID: Q120563
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.0
  • Microsoft IntelliPoint Software Home Edition version 1.0

SUMMARY

When you choose the Special Effects tab in IntelliPoint Tools and Controls and then choose the Browse Animation button, be aware that not all of the cursor files listed in the Browse dialog box are truly animated.

MORE INFORMATION

Only the specific cursor "grouping" files have animation ability. These grouping files can be selected only for the actions listed when you choose the Special Effects tab. (These actions include Single-click, Double-Click, Drag Object, Drop Object, Moving Over An Object, Both Button Click, and Secondary Click.) These same grouping files do NOT provide animation for the active cursors listed when you choose the Appearance tab.

A total of 46 animated cursors are included with both the Microsoft Home and standard editions of IntelliPoint software. These animated cursors are listed below. The grouping files are usually designated with a "01," "101," "201," or "11" numerical series with the same beginning filename (the filename extension remains .CUR). When you click the first grouping file in the series (that is, <FILE>01.CUR),Selecting other grouping files (for example <FILE>02.CUR), you see a preview of the cursor animation.
