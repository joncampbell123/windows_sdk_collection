IntelliPoint Software Displays Incorrect Product ID Number

 Last reviewed: December 4, 1997
Article ID: Q177584
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall, version 1.0
  • Microsoft IntelliPoint software versions 2.0, 2.1

SYMPTOMS

When you attempt to obtain product information for the Microsoft IntelliMouse using the Microsoft IntelliPoint software, the following Product Identification Number (PID) may be displayed: 

   11111-111-1111111-11111

CAUSE

Newer models of the Microsoft IntelliMouse have the PID stamped on the bottom of the pointing device. If the IntelliPoint software displays a PID of 11111-111-1111111-11111, examine the bottom of the pointing device to obtain the correct PID.

STATUS

This behavior is by design.

MORE INFORMATION

For information about how to locate the PID in the IntelliPoint software, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q174440
   TITLE     : IntelliPoint 2.0: Where to find Product Information
