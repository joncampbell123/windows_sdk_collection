Tests with the Compaq 386/20

 Last reviewed: September 16, 1996
Article ID: Q28495
The information in this article applies to:
  • Microsoft Bus Mouse, version 1.x
  • Microsoft PS/2 Serial Mouse, version 1.x
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.x

SUMMARY

Microsoft Bus and PS/2 serial mice work properly with the Compaq 386/20 computer.

MORE INFORMATION

The bus mouse and revision G of the bus interface board worked properly with the Compaq 386/20 with the version 6.11 and later of the mouse driver.

The white-button and gray-button serial mice tested successfully with the Compaq 386/20's serial port.
