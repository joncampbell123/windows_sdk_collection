Mouse Pointer Disappears Performing Search in File Manager

 Last reviewed: September 12, 1996
Article ID: Q145624
The information in this article applies to:
  • Microsoft IntelliPoint software versions 1.0, 1.0a, 1.1
  • Microsoft IntelliType software versions 1.0, 1.1
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

When you search for a file using File Manager in Windows or Windows for Workgroups version 3.1 or 3.11, the mouse pointer may disappear. When you move the mouse after the search is completed, the mouse pointer may not reappear.

CAUSE

This problem can occur when the following conditions are met:

  • The IntelliPoint or IntelliType Vanish feature is enabled.
  • The mouse pointer is located in the Search For or Start From box when you start the search.
  • You press ENTER to start the search instead of clicking OK.

RESOLUTION

To cause the mouse pointer to reappear, you must restart Windows. To prevent the problem from occurring again, disable the Vanish feature, make sure the mouse pointer is not located in the Search For or Start From box when you start the search, or click OK instead of pressing ENTER to start the search.

To disable the IntelliPoint Vanish feature, follow these steps:

  1. In Control Panel, double-click the Mouse icon.

  2. Click the Visibility tab.

  3. Click the "Make the pointer disappear when you begin typing" check box to clear it.

  4. Click OK.

To disable the IntelliType Vanish feature, follow these steps:

  1. In Control Panel, double-click the Keyboard icon.

  2. Click the Pointer Activity tab.

  3. Click the "Make the pointer disappear when you begin typing" check box to clear it.

  4. Click OK.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Vanish feature causes the mouse pointer to disappear when you begin typing in a Windows-based program. When you move the mouse after typing text in a Windows-based program, the pointer reappears. This feature is useful if you frequently move the mouse pointer out of the way before you begin typing.
