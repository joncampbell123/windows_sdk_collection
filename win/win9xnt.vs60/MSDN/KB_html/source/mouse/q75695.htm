How to Determine OS/2 Version Before Installing Mouse

 Last reviewed: September 16, 1996
Article ID: Q75695
The information in this article applies to:
  • Microsoft Serial-PS/2 Mouse, versions 1.0 and 2.0
  • Microsoft BallPoint Mouse, versions 1.0 and 2.0

SUMMARY

To properly install a Microsoft Mouse or Microsoft BallPoint Mouse for an OS/2 system, you must first determine what version of OS/2 you are running. This information applies to all versions of the Microsoft Mouse and Microsoft BallPoint Mouse.

MORE INFORMATION

Change to an OS/2 command line (full screen or window). At the OS/2 command prompt, type "SYSLEVEL" (without the quotation marks). After a moment, the version and type (extended or standard edition) will be displayed.

If the command is not recognized by the OS/2 system, try changing to the DOS compatibility box and type "SYSLEVEL" to obtain the system information.
