Cursor Moves Behind Mouse Manager with Genius Video Card

 Last reviewed: September 16, 1996
Article ID: Q101409
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 9.0

If you use the Microsoft Mouse driver version 9.0 with a Genius video card in text mode, the cursor appears to move behind the Mouse Manager dialog box at the MS-DOS level. The cursor does, however, still work.

This does not occur in graphics mode.
