Features and Benefits of Version 8.0-Series Mouse Drivers

 Last reviewed: September 16, 1996
Article ID: Q71663
The information in this article applies to:
  • Microsoft BallPoint Mouse, version 1.0
  • Microsoft Mouse Driver for MS-DOS, version 8.0

SUMMARY

The Microsoft Mouse Driver version 8.0 for the BallPoint mouse was released on March 11, 1991.

This driver offers several performance enhancements, detailed in the following:

MOUSE.COM

  • Supports BallPoint, Bus(InPort), and Serial-PS/2 mouse in DOS.
  • Reads settings from MOUSE.INI text file.
  • Automatically senses what type of mouse is connected.
  • Can be loaded into High Memory Area or Expanded Memory.
  • Supports IBM Extended Graphics Array (XGA) video mode.

MOUSE.DRV

  • Supports BallPoint, Bus(InPort), and Serial-PS/2 mouse in Windows 3.0.
  • Reads settings from MOUSE.INI text file.
  • Automatically senses what type of mouse is connected.

MOUSE.INI

  • Uses English-like parameters similar to Windows 3.0.
  • Referred to by both DOS and Windows; settings affect both environments.
  • Includes default acceleration profiles.
  • Editable ASCII text file.

CPANEL.COM

  • Allows separate control of horizontal and vertical sensitivity.
  • Non-TSR capability gives it the ability to save changes permanently.

POINT.EXE

  • A new Windows 3.0 Control Panel.
  • More settings than standard Windows 3.0 Control Panel.
  • Combines DOS Control Panel and BallPoint Compass capabilities.
  • Automatically senses mouse type and displays correct setting choices.
  • Includes Help (POINT.HLP) displayed in WinHelp 3.0.

COMPASS.EXE

  • Allows you to orient "which way is up" for BallPoint mouse.
  • Online tutorial covering the BallPoint mouse (TUTORIAL.EXE/.GL/.TXT).
  • Provides DOS Control Panel for BallPoint mouse.
  • Includes a README viewer.

SETUP.EXE

  • Install software for both DOS and Windows 3.0.
  • Includes both Quick Install and Custom Install options.
  • Includes Help for each step of the installation process.
  • Installs a MOUSE group in Windows 3.0 at Program Manager.
  • Includes a README viewer.
