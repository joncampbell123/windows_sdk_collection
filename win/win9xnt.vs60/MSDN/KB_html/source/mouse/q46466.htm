Mouse Driver Menu Error Messages in Appendix C, Not B

 Last reviewed: September 16, 1996
Article ID: Q46466

SUMMARY

On Page 44 of the "Mouse Programmer's Reference" from Microsoft Press, the incorrect appendix is specified for error messages pertaining to mouse menus.

The book incorrectly specifies Appendix B for mouse driver error messages. The correct appendix for menu error messages is Appendix C.
