Identifying the Version Number of a Logitech Mouse Driver

 Last reviewed: September 16, 1996
Article ID: Q81549

SUMMARY

To obtain the Logitech Mouse driver version number, type "mouse" (without quotation marks) at the command prompt, reboot the computer, and read the version of the mouse driver displayed on the screen as it is started, or use the Microsoft Diagnostics (MSD) program if it is available.

MSD displays the Microsoft Mouse version that Logitech tries to emulate. For example, the Logitech Mouse driver version 5.0 will display as emulated mouse version 6.02 in MSD. The current version of the Logitech driver is 6.0; MSD displays this version as the emulation Microsoft Mouse driver version 6.36.

If you need to update your driver so that your mouse will perform properly with your application, call Logitech at (415) 795-0801.
