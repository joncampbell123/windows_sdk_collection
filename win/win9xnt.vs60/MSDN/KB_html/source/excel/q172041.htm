XL97: Blank Workbook Remains Open Using Custom (Book.xlt)

 Last reviewed: January 7, 1998
Article ID: Q172041
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you open a workbook that you previously saved in Microsoft Excel 97, the default blank workbook is not automatically closed.

CAUSE

This problem occurs if you create a custom workbook template (Book.xlt) and replace the default workbook template in Microsoft Excel 97.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

If this problem occurs you must manually close the blank workbook.

REFERENCES

For more information about creating templates, click the index tab in Microsoft Excel 97 Help, type the following text: 

   templates, customizing workbook defaults
and then double-click the selected text to go to the "Customize the defaults for a workbook or workshet by using a template" topic.
