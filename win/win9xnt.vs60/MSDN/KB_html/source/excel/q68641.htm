Marquee Must Surround Entire Object(s) to Select

 Last reviewed: November 3, 1994
Article ID: Q68641
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 5.0

SUMMARY

In order to select an object or group of objects using the selection tool from the tool bar, the object(s) must lie completely within the marquee (selection rectangle).

MORE INFORMATION

To select an embedded chart and an oval that you have drawn, do the following:

  1. Click the selection tool (the button in the tool bar that has a dotted square on it).

  2. Click and drag a square around the two objects. The outline square must completely surround both objects for them to be selected.
