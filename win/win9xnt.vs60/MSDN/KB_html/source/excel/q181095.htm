XL98: Web Form Wizard Add-in Is Not Available

 Last reviewed: February 13, 1998
Article ID: Q181095
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

Microsoft Excel 98 Macintosh Edition does not support using the Web Form Wizard add-in. Because this add-in uses functionality that is not available on the Macintosh platform, this feature is not available in Excel 98.

CAUSE

The Web Form Wizard allows you to request and gather information over the Internet by using forms. Because you can use a Microsoft Excel worksheet to simulate a form, you can use Microsoft Excel 97 for Windows as an engine to generate a form for the Web. The Web Form Wizard add-in uses either the Microsoft Internet Information Server or Common Gateway Interface (CGI) to send information to an internet site. The wizard saves the data in the Microsoft Access database (.mdb) file by using Data Access Objects (DAO). However, Data Access Objects are not available on the Macintosh platform.

STATUS

This behavior is by design of Microsoft Excel 98 Macintosh Edition.
