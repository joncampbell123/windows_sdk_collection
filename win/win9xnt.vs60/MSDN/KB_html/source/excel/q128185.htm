XL: Macrofun.exe File Available on Online Services

 Last reviewed: September 3, 1997
Article ID: Q128185
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SUMMARY

To display help topics for Microsoft Excel 4.0 macro functions, you must first install the complete Macro Function help file. You can obtain the complete Macro Function help file (Macrofun.hlp), from various online services.

The following file(s) are available for download from the Microsoft Software Library: 

 ~ MACROFUN.EXE (size: 696544 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
