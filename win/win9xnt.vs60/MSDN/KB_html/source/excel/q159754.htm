XL97: Page Numbers in "Now Printing" Dialog May Skip Numbers

 Last reviewed: January 15, 1998
Article ID: Q159754
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you print a large number of pages in Microsoft Excel 97, the page numbers in the "Now Printing" dialog box may skip some numbers.

CAUSE

To speed up printing, the Now Printing dialog box does not update for each printed page. This may cause some page numbers to appear to be skipped, but does not affect the actual number of pages printed.

STATUS

This behavior is by design of Microsoft Excel 97.

REFERENCES

For more information about printing workbooks, click Contents and Index on the Help menu. Click the Index tab, type the following text 

   printing, workbooks
and then double-click the selected text to go to the "Print the active sheets, a selected range, or an entire workbook" topic.
