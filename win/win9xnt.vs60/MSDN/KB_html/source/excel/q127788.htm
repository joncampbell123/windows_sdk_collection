MXL5: "Number must be between 1 and 32767" in Print Preview

 Last reviewed: November 14, 1997
Article ID: Q127788
5.00 MACINTOSH kbprint kberrmsg xlmac

The information in this article applies to:

  • Microsoft Excel for the Macintosh, version 5.0

SYMPTOMS

In Microsoft Excel, if you are using an Apple Personal LaserWriter SC printer driver, or an Apple LaserWriter II SC printer driver, you receive the following error message when you view your document in print preview: 

   Number must be between 1 and 32767

CAUSE

This problem occurs because when you view your document in print preview, if you are using one of the printer drivers listed above, Microsoft Excel reads an incorrect 'number of copies' value from the printer driver. This behavior occurs if you choose Print Preview from the File menu or if you choose the Print Preview button from the Standard toolbar.

WORKAROUND

To avoid receiving this error message when you view your document in print preview when you are using an Apple Personal LaserWriter SC printer driver or an Apple LaserWriter II SC printer driver, choose the Print Preview button in the Print dialog box, instead of choosing Print Preview from the File menu.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel version 5.0a for the Macintosh.
