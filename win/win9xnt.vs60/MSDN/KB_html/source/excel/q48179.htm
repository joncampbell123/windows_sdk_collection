Excel Err Msg: "Can't Add More Formats" or "Too Many Formats"

 Last reviewed: July 27, 1995
Article ID: Q48179

The information in this article applies to:

  • Microsoft Excel for Windows, versions 1.x, 2.x, 3.x, 4.0, 5.0, 5.0c
  • Microsoft Excel for the Macintosh, version 1.x, 2.x, 3.x, 4.0, 5.0, 5.0a

SYMPTOMS

In Microsoft Excel, when you attempt to add a custom number format, you may receive the following error message: 

   Cannot add any more custom formats

CAUSE

In Microsoft Excel, there is a limit to the number of custom number formats that you can create. The following table lists these limits. 

   For this version         Approximate number
   of Microsoft Excel       of custom number formats
   -------------------------------------------------

   5.0, 5.0a, 5.0c             Limited by available memory
   4.0                         214
   3.x                         233
   2.x and earlier              43

WORKAROUND

To work around this situation, do the following to remove some custom number formats:

  1. On the Format menu, click Cells.

  2. Click the Number tab, and select the Custom category.

  3. Select the custom format you want to delete, and then click Delete.

NOTE: You cannot delete the built-in number formats in Microsoft Excel.

MORE INFORMATION

For more information about the number of custom number formats you can use in Microsoft Excel, click Contents on the Help menu, and then click Reference Information. Click Microsoft Excel Specifications, and then click Worksheet and Workbook specifications, and scroll to the Formatting section.
