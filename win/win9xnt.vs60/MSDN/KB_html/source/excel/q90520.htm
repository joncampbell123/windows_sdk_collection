Excel: Links Command not on Shortcut Menu

 Last reviewed: November 29, 1994
Article ID: Q90520
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

Page 25 of the "Microsoft Excel User's Guide 1" contains a documentation error in the display of the Worksheet Shortcut Menu. The third menu item shows the Links command as unavailable (grayed). This should instead be the Paste command. Whether or not it is available depends on if there is anything on the Clipboard.

The Links command is not accessible from the Worksheet Shortcut Menu, which is brought up by clicking the right mouse button on a cell in a worksheet or macro sheet. The Links command is, instead, accessible from the File menu on the built-in menu bars.

REFERENCES

"Microsoft Excel User's Guide 1," version 4.0, pages 24-25
