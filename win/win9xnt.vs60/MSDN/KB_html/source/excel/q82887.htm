Q+E and SQL Server Common .DLL Files

 Last reviewed: November 22, 1994
Article ID: Q82887

The information in this article applies to:

  • Microsoft Excel for Windows, version 3.0
  • SQL Server version 4.2

SYMPTOMS

SQL Server version 4.2, includes two .DLL files, DBNMP3.DLL and W3DBLIB.DLL, that are also included with Q+E. The version of these DLLs included in the Microsoft Excel 3.0 package are earlier versions of the files.

If Microsoft Excel and Q+E are installed after SQL Server, then any application that depends on this version of these .DLLs may not operate correctly.

WORKAROUNDS

To work around this problem:

  • Delete the versions of the .DLLs installed by Microsoft Excel

    -or-

  • Remove Microsoft Excel from the Path statement in the AUTOEXEC.BAT

    -or-

  • Move Microsoft Excel so that is appears after SQL Server on the Path statement
