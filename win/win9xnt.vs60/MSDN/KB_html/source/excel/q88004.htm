Excel: COMMAND+O Highlights Save Button in Save As Dialog Box

 Last reviewed: November 18, 1994
Article ID: Q88004
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0

Pressing COMMAND+O in the Microsoft Excel for the Macintosh Save As dialog box highlights the Save button before displaying the Options dialog box. The Save button is not activated.

The Save button in the Save As dialog box changes to Open if a folder is highlighted. The Macintosh system software traps the COMMAND+O, highlights the Save button, and then passes the COMMAND+O onto Excel because COMMAND+O is not the keystroke to activate the Save button. Excel recognizes the COMMAND+O and activates the Options button.

This is a limitation of the Macintosh system software.
