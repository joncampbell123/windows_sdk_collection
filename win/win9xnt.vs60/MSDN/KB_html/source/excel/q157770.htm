XL97: Error When Dragging Chart Into Microsoft Word

 Last reviewed: January 15, 1998
Article ID: Q157770
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

In Microsoft Excel 97, if you drag an embedded chart to a Microsoft Word 97 document, you may receive the following error message: 

   Microsoft Map
   An error has occurred while retrieving data from Microsoft Excel.
When you click OK, you may also receive the following error message: 

   WINWORD caused an invalid page fault in module GDI32.DLL at
   014f:bff3501d.
and Microsoft Word 97 stops responding.

CAUSE

This error may occur if the following conditions are true:

  • You are dragging an embedded chart from a Microsoft Excel 97 worksheet to a Microsoft Word 97 document. 

        -and-
  • The Microsoft Excel 97 worksheet also contains an embedded Microsoft Map.

WORKAROUND

If this problem occurs, use the following steps to copy your chart from Microsoft Excel into Microsoft Word:

  1. In Microsoft Excel, select the chart.

  2. On the Edit menu, click Copy.

  3. Activate Microsoft Word, and select the location in your document where you want to paste the chart.

  4. On the Edit menu, click Paste.

The chart pastes correctly into Microsoft Word.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Office programs, such as Microsoft Excel and Microsoft Word, support drag-and-drop functionality between programs. For instance, you can select a range of cells in a Microsoft Excel worksheet and drag the range to a Microsoft Word document. When you release the mouse button, the dragged object appears in the document.

Under normal circumstances, if you drag a chart embedded in a Microsoft Excel worksheet to a Microsoft Word document, a copy of the chart will appear in the Microsoft Word document. However, this will not work if the Microsoft Excel worksheet contains an embedded Microsoft Map.
