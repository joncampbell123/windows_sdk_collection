MXL5: Complete Installation File Locations for Macintosh

 Last reviewed: February 2, 1998
Article ID: Q129569
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 5.0

SUMMARY

The following list shows the filename and folder locations for all of the files that are installed on a Macintosh during a Complete/Custom setup of Microsoft Excel version 5.0.

NOTE: "Microsoft Excel 5" refers to the folder you specified during setup. "Microsoft" refers to the folder named Microsoft, which is installed at the same level on the hard disk drive as the "Microsoft Excel 5" folder.
