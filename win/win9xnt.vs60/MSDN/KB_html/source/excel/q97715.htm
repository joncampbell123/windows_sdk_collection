Excel: Name of Version 4.0 Settings File Is EXCEL4.INI

 Last reviewed: September 12, 1996
Article ID: Q97715
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0 and 4.0a

SUMMARY

Page 253 of the "Microsoft Excel Function Reference," version 4.0 manual, may contain the following incorrect definition for Syntax 3 of the Command argument of of the LINE.PRINT function: 

   Global settings (saved in EXCEL.INI)
The correct name of the settings file for Microsoft Excel version 4.0 is EXCEL4.INI. The corrected statement is: 

   Global settings (saved in EXCEL4.INI)
In addition, page 254 of the "Microsoft Excel Function Reference," version 4.0, states the following for the Update argument of Syntax 3 of the LINE.PRINT function: 

   If TRUE, the settings are saved in the EXCEL.INI file...
This statement should read: 

   If TRUE, the settings are saved in the EXCEL4.INI file...

REFERENCES

"Microsoft Excel Function Reference," version 4.0, pages 253-254 "Switching to Microsoft Excel from Lotus 1-2-3," version 4.0, page 39
