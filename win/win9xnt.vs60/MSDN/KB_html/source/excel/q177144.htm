XL98: Error Clicking Create New Query

 Last reviewed: February 5, 1998
Article ID: Q177144
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you click Get External Data on the Data menu, and then click Create New Query, the following error message appears: 

   Microsoft Excel could not start Microsoft Query because it is not
   installed. To install it, run the Value Pack installer and select the
   Microsoft Query option in the Data Access category. If Microsoft Query
   is already installed, run it standalone to force it to reregister with
   the system.

WORKAROUND

To resolve this problem, install Microsoft Query. To do this, follow these steps:

  1. Ensure that no programs are running and insert the Microsoft Office 98 Macintosh Edition compact disc.

  2. Double click the CD icon on the desktop to open it. Double-click the the Value Pack folder on the compact disc, and then double-click the Value Pack Installer icon.

  3. In the Value Pack Setup program, click to select the Data Access check box. Click Install.

After you install the Data Access components, you no longer receive the error message when you click Create New Query.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
