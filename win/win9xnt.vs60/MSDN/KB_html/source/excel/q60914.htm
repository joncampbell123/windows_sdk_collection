SUMMARY

Crystal Ball, by Market Engineering Corporation, is a powerful forecasting and risk-management program that helps take the mystery out of decision making. To use Crystal Ball, you set up a model in Microsoft Excel, save it as a SYLK file, and then open it in Crystal Ball. Each uncertain variable in a model is assigned a probability distribution by describing how that variable behaves. Using a technique known as "Monte Carlo simulation," the program predicts the results most likely to occur in a given scenario.

MORE INFORMATION

Crystal Ball's features include:

  1. The ability to process up to 250 variables and 30 forecasts per problem

  2. Descriptive information on forecasts

  3. Sensitivity analysis

  4. Preference settings

  5. Background operation under MultiFinder

Any application that involves assessing risk in a model (financial or otherwise) could benefit from a product like Crystal Ball.
