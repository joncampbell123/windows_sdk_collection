XL97: ALT+DOWN ARROW Doesn't Invoke AutoComplete

 Last reviewed: January 15, 1998
Article ID: Q171020
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you try to use the AutoComplete shortcut key, ALT+DOWN ARROW, on a worksheet with embedded objects, the shortcut key fails to work.

WORKAROUND

To view the AutoComplete list, press F2 first, and then press ALT+DOWN ARROW.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about shortcut keys, click the Index tab in Microsoft Excel 97 Help, type the following text 

   keys, shortcuts
and then double-click the selected text to go to the "Enter data by using shortcut keys" topic.
