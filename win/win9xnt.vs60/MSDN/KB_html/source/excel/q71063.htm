No Option to Change Directory in Q+E Save As Dialog Box

 Last reviewed: November 2, 1994
Article ID: Q71063

SUMMARY

If you choose Save As from the File menu in Q+E for Microsoft Excel, you will not be given an option to change the directory in which your query will be saved. The current directory is used by default.

MORE INFORMATION

Unlike most Windows 3.00 applications, the Save As dialog box in Q+E does not give you the option to browse your directories. The current directory is indicated in the Directory Is field.

Solution

To save your query to a directory other than the default, you have two options:

  1. Type the specific path followed by the filename in the File Name field.

  2. Browse your disk or server by choosing the Open option from the File menu. When the Directory Is field in this dialog box shows the path you desire, choose Cancel. This will change the Q+E default directory. Your selection will now appear by default in the Save As dialog box.

REFERENCES

"Q+E for Microsoft Excel" User's Guide, pages 27-31.
