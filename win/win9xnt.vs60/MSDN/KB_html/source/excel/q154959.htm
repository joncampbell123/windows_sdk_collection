XL97: Chart Embedded on Chart Sheet Does Not Preview

 Last reviewed: February 18, 1998
Article ID: Q154959
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

If you have a chart embedded on a chart sheet, you may not be able to view a sample of the embedded chart in the Chart Type dialog box.

CAUSE

If you have a chart sheet with a second chart embedded on it, you can preview only one of the charts in the Chart Type dialog box. The chart that is currently selected is the one that appears when you click "Press and hold to view sample" in the Chart Type dialog box.

RESOLUTION

To be able to view a sample (in the Chart Type dialog box) of either chart on the chart sheet, you must first select the desired chart.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Excel 97 includes a new feature that allows you to preview a sample of your chart while you make changes to the chart. If you already have a chart created, you can select the chart and then click Chart Type on the Chart menu. In the Chart Type dialog box, you can change the type of chart you have created and then click "Press and hold to view sample" to see what your chart will look like with the changes you have made. If you like the changes, click OK to accept them. If you do not like the changes, you can either make different changes, or you can click Cancel to revert to your original chart.

REFERENCES

For more information about Creating Charts click the Index tab in Microsoft Excel Help, type the following text 

   creating charts in Excel
and then double-click the selected text to go to the "select a different chart type" topic.
