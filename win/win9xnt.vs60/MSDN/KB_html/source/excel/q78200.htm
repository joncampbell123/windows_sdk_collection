Excel: Solver's Options Dialog Box Will Not Close

 Last reviewed: November 30, 1994
Article ID: Q78200
The information in this article applies to:
  • Microsoft Excel for Windows versions 3.0 and 4.0
  • Microsoft Excel for the Macintosh version 3.0
  • Microsoft Excel for OS/2 version 3.0

SUMMARY

The Solver Options dialog box will not close if Iterations is set to 32768 or higher.

MORE INFORMATION

Page 25 of the "Microsoft Excel Solver User's Guide" for version 3.0 states that only a very exceptional problem would require more than 1000 iterations. If the number in the Iterations box is set to a number larger than 32767, the dialog box will not close when choosing OK. No error message will be displayed.

Choosing Cancel or lowering the entry for iterations will allow you to choose OK and close the dialog box.

REFERENCES

"User's Guide 2," version 4.0, pages 99-100 "Microsoft Excel Solver User's Guide," for Windows, version 3.0, page 25
