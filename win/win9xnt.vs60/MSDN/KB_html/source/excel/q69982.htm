Copying Entire Rows and Columns Copies Formatting in Excel

 Last reviewed: November 3, 1994
Article ID: Q69982
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0

SUMMARY

When copying or sorting information in Microsoft Excel, if you select the entire column or row, the formatting associated with the selection is copied also.

This includes cell formatting, font formatting, column width (if column is selected), row height (if row is selected), and number formatting.

MORE INFORMATION

If you do not select the entire column or the entire row when you select a range to copy or sort, the column width and the row height are not preserved. However, if the copy or sort data contains a cell with a text string as the contents and the text is wrapped and aligned, when that record copies or sorts to its new position, the taller row height is carried to the new position.
