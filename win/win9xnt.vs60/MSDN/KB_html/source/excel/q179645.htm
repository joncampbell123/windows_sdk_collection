XL98: Cannot Use Keyboard to Select Items in Special List

 Last reviewed: February 2, 1998
Article ID: Q179645
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, it is not possible to use the keyboard to select items in the Special list in the Summary tab in the Advanced Search dialog box. You must use the mouse pointer to select items in this box.

WORKAROUND

When you select an item in the Special list, a corresponding code appears in the Containing Text box. If you cannot select an item in the list, you can type the appropriate code in the Containing Text box. To do this, follow these steps:

  1. Press COMMAND+N to move the insertion point to the Containing Text box.

  2. Enter one or more of the codes in the following table. 

          List item             Code
      ---------------------------

      Any Character          ?
      Character Range        [-]
      Beginning of Word      <
      End of Word            >
      Expression             ()
      Not                    [!]
      Num Occurrences        {,}
      Previous 1 or More     @
      0 or More Characters   *

  3. Make any other necessary changes in the box. When you are finished, press RETURN.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Excel 98 Macintosh Edition includes a number of accessibility features that make it possible to use most features of the program even if you cannot use a mouse.

Example

To see an example of the problem described in this article, press the following keys, in order, in Microsoft Excel 98 Macintosh Edition. 

   Keystroke   Function
   ----------------------------------------------------

   COMMAND+O   Displays Open dialog box.
   COMMAND+F   Displays Search dialog box.
   COMMAND+A   Displays Advanced Search dialog box.
   COMMAND+S   Selects Summary tab.
   COMMAND+R   Selects Use Pattern Matching check box.
Note that the Special list becomes available for use. However, it is not possible to select items in the list box by using only the keyboard.
