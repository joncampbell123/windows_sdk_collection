XL97: Chart Data Is Not Printed Under NT with Some HP Drivers

 Last reviewed: March 3, 1998
Article ID: Q181958
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you print a Microsoft Excel 97 for Windows chart that contains a large number of data points, some of the data points may not be printed if you are using Microsoft Windows NT version 4.0.

CAUSE

This behavior may occur if the following conditions are true:

  • You are printing a chart that contains a large number of data points (approximately 2,000 or more). 

        -and-
  • You are using Microsoft Windows NT version 4.0. 

        -and-
  • You have randomly placed blank values in the data series. 

        -and-
  • You are using the Hewlett-Packard (HP) print drivers that are listed in the following table. 

          Driver       Version
      --------------------

      HP LJ IIP       3.02
      HP LJ IIIP      3.37
      HP LJ 5P        3.52

RESOLUTION

To resolve this problem, decrease the number of data points in the chart series or print the chart from a computer running Microsoft Windows 95.

You can also try installing the latest version of the printer driver from Hewlett Packard to see if the updated driver resolves the issue.

You may be able to improve the printed results, by changing the resolution settings for the printer. To change the resolution, follow these steps:

  1. On the File menu, click Page Setup.

  2. Click Options.

  3. On the Advanced tab, click the plus sign to the right of Graphic to view the settings for Graphic.

  4. Click Resolution under Graphic and click to select a different graphic resolution setting for the printer.

  5. Click OK, and then click OK again.

  6. Print the chart.

  7. After printing the chart, repeat steps 1 through 5 and change the resolution setting for the printer back to the original setting.
