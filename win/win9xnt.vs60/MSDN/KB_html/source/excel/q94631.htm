XL: Regression Tool Error If More Than 16 X Ranges

 Last reviewed: February 2, 1998
Article ID: Q94631

The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for the Macintosh, versions 4.0, 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel, the Regression analysis tool, included in the Analysis ToolPak add-in macro, has an X-range limit of 16. If you attempt to specify more than 16 X ranges, you will receive one of the following errors:

Excel 5.0, 7.0, Excel 97, and Microsoft Excel 98 Macintosh Edition

   Regression - Input X Range cannot contain more than
   16 variables (columns).

Excel 4.0 or 4.0a

   a #REF! error value in the Regression output

MORE INFORMATION

The Regression analysis tool uses the LINEST function to perform linear regression calculations. Because LINEST has an X-range limit of 16, the Regression analysis tool has the same limitation.
