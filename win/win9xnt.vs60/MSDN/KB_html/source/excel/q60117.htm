Excel: Unable to Invoke Macro with CAPS LOCK Key Pressed Down

 Last reviewed: November 29, 1994
Article ID: Q60117
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SUMMARY

If a Microsoft Excel macro has been defined using a capital letter such as an "A," the macro cannot be invoked by having the CAPS LOCK key pressed down while pressing the COMMAND+OPTION keys.

To run a macro that was defined using a capital letter, hold down the SHIFT key along with the COMMAND+OPTION keys to invoke the macro (for example, COMMAND+OPTION+SHIFT+A).
