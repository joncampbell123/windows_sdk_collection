XL4: Can't Find Dialog Box Definitions in "User's Guide 1"

 Last reviewed: February 2, 1998
Article ID: Q104356
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a
  • Microsoft Excel for the Macintosh, version 4.0

SYMPTOMS

On page 113 of the "Function Reference" version 4.0, under the Example section for the DIALOG.BOX function, the example refers to Chapter 8 of the "User's Guide 1." This reference is incorrect. The correct reference is to Chapter 8 of "User's Guide 2."

The documentation error applies to the following document numbers: AB26298-0392, AB26296-0492, and AB26297-0492.

STATUS

This problem has been corrected in manuals whose document numbers end with -0692 and later. To find the document number on your User's Guide, look on the back of the title page.
