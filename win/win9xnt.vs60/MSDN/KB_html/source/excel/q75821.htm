Excel: IF Function Returns 0 Instead of TRUE or FALSE

 Last reviewed: November 2, 1994
Article ID: Q75821

SUMMARY

Page 125 of the "Microsoft Excel Function Reference" version 3.0 manual incorrectly states that "If logical_test is TRUE and value_if_true is omitted, TRUE is returned" and "If logical_test is FALSE and value_if_false is omitted, FALSE is returned."

If value_if_true is not included in the formula "=IF(logical_test,,value_if_false)", then 0 (zero) will be returned if logical_test is TRUE. If value_if_false is not included in the formula "=IF(logical_test,value_if_true,)", then a 0 will be returned if logical_test is FALSE. However, if value_if_false is left off in the formula "=IF(logical test,value_if_true)", then FALSE will be returned when logical_test is FALSE.

MORE INFORMATION

To get the IF function to return either TRUE or FALSE depending on the value of logical_test, value_if_true and value_if_false should have TRUE and FALSE entered for their respective arguments. For example, the following formula 

   =IF(logical_test,TRUE,FALSE)
will return TRUE if logical_test is TRUE and FALSE if logical_test is FALSE.
