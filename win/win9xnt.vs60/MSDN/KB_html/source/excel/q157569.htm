XL97: Can't Select Cells When Using Netscape Navigator

 Last reviewed: January 15, 1998
Article ID: Q157569
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When viewing a Microsoft Excel 97 worksheet in Netscape Navigator 3.0, you are unable to select cells within the worksheet.

CAUSE

This may occur if the following conditions are all true:

  • You are using Netscape Navigator 3.0, and you have the NCompass ScriptActive plug-in installed. 

        -and-
  • You are viewing an embedded Microsoft Excel 97 workbook using Netscape Navigator 3.0. 

        -and-
  • You have activated any other window besides Netscape Navigator's window, or you clicked the About button for NCompass ScriptActive.

WORKAROUND

If this problem occurs, click the Reload button on the toolbar in Netscape Navigator 3.0. When the page has completed reloading, you should be able to select and edit cells within the worksheet.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

NOTE: This problem does not occur when using Microsoft Internet Explorer version 3.0.

MORE INFORMATION

The NCompass ScriptActive plug-in for Netscape Navigator 3.0 allows you to view Microsoft Excel workbooks (and other types of documents) directly within Netscape Navigator.
