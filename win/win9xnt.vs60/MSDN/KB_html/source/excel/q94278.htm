MXL: Page Setup Info Lost When File Opened in Later Version

 Last reviewed: February 2, 1998
Article ID: Q94278
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.5, 2.2, 3.0, 4.0

SYMPTOMS

If you are using version 3.0 or 4.0 of Microsoft Excel for the Macintosh, and you open a document that was created in an earlier version of Microsoft Excel for the Macintosh, some page setup information may be lost from your document.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected Microsoft Excel versions 5.0 and later.

MORE INFORMATION

Microsoft Excel Version 3.0 Opened in Microsoft Excel Version 4.0

If you open a file in version 4.0 that was saved from version 3.0 with Landscape orientation, the orientation shown in the version 4.0 Page Setup dialog box will be Portrait but the file will still be printed in landscape orientation.

Microsoft Excel Version 2.2 Opened in Microsoft Excel Version 4.0

When you open a version 2.2 Microsoft Excel for the Macintosh document in version 4.0 of Microsoft Excel, the setting for paper orientation in the Page Setup dialog box is lost (if you selected the Landscape option for the page orientation in version 2.2 of Microsoft Excel, the document will revert to the Portrait option). (Portrait is the default setting for Microsoft Excel.)

NOTE: This information is not lost if you open your document in version 3.0 of Microsoft Excel.

Microsoft Excel Version 1.5 Opened in Microsoft Excel Version 3.0

When you open a version 1.5 Microsoft Excel document in version 3.0 of Microsoft Excel, the following changes occur:

  • The Paper Size option is set to US Letter.
  • The Reduce or Enlarge option is changed to 100-percent.
  • The Page Orientation option is set to Portrait.
  • Printer Effects options that were previously cleared are selected.
  • The Print Gridlines option is selected.
  • LaserWriter options that were selected are cleared.

Microsoft Excel retains the following Page Setup settings:
  • Margins
  • Headers and footers
  • Print Row and Column Heading options
