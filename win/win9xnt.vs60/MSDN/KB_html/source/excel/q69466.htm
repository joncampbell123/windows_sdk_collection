UAE Using Canon LBP-8III/LBP-4 Driver

 Last reviewed: November 1, 1994
Article ID: Q69466

SUMMARY

Using the CANONIII.DRV driver dated 5/1/90 may cause an Unrecoverable Application Error (UAE) when printing from Microsoft Excel version 3.00.

Generally, the UAE error will occur when printing charts or worksheets that contain heavy graphics (borders, charts, buttons, etc.). Worksheets that are not formatted with an exact matching screen and printer font also may cause a UAE error.

MORE INFORMATION

The above problem is corrected by using the LBPIII.DRV driver dated 9/20/90 along with the associated GENDRV.EXE and CAN_ADF.EXE files as found on the Supplemental Driver Library disks.

If the LBPIII.DRV driver is used, then text will bleed through any graphic object that has been placed on top of worksheet data. However, the object will block the gridlines. The print preview appears correctly.

For more information, query on the following words: 

   Contents and Cells and Show and Objects.
