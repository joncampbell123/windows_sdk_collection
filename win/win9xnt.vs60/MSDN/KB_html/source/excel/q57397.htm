Excel Err Msg: Unable to Paste Link

 Last reviewed: November 1, 1994
Article ID: Q57397

SUMMARY

When you copy data in Microsoft Excel version 2.20 and then choose Paste Link from the Edit menu to link that data to another worksheet, the error message "Unable to Paste Link" will be displayed if any folder or disk name in the pathname of that link has an apostrophe.

For example, if the hard drive is named "John's Hard Drive" or a folder is named "Microsoft's Applications," this error message will be displayed.

To work around this problem, either manually type the formula to link the two worksheets or rename the folder or disk to a name without an apostrophe.
