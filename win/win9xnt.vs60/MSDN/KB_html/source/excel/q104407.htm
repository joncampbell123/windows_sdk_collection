XL5: Find File Command Different from Find File in Word

 Last reviewed: September 12, 1996
Article ID: Q104407
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

In Microsoft Excel version 5.0, you can use the Find File command to preview documents from the list of found files prior to opening them or performing some other action. This preview screen differs from that available in Microsoft Word for Windows versions 2.0 and later. The preview you see in Microsoft Word for Windows is a zoomed version of the original file. As a result, you can scroll through this file to see different parts of it. In Microsoft Excel, the preview is a snapshot of the upper left corner of the sheet that was active when you last saved the workbook; it is not scrollable.
