Version History of Microsoft Excel for Windows

 Last reviewed: July 29, 1997
Article ID: Q66913

The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SUMMARY

The following is a version history of Microsoft Excel for Windows, beginning with the most recent version: 

Version  Date     System Requirements

 8.0     1/15/97   1. 8 MB of memory for use on Windows 95; 16 MB of memory
                      for use on Windows NT workstation
                   2. Hard disk drive
                   3. Microsoft Windows 95, Microsoft Windows NT
                      Workstation 3.51 Service Pack 5 or later, or
                      Microsoft Windows NT Workstation 4.0
                   4. CD-ROM drive
                   5. VGA or higher-resolution video adapter
                   6. Personal computer with a 486 or higher processor
                   7. Hard disk space required:
                         Compact installation: approximately 22 MB
                         Typical installation: approximately 36 MB
                         Custom(complete) installation: approximately 64 MB
                   Enhancements:
                      1. Text can be rotated and indented
                      2. Hyperlinks can be inserted into cells
                      3. Multiple levels of undo
                      4. Formula AutoCorrect
                      5. Page Break Preview
                      6. Improvements made to Shared Workbooks
                      7. Save workbooks in HTML

 7.0     7/27/95   1. 6 MB of memory for use on Windows 95; 12 MB of memory
                      for use on Windows NT workstation
                   2. Hard disk drive
                   3. Microsoft Windows 95 or Windows NT version 3.51
                      or later
                   4. 3.5-inch high density disk drive
                   5. VGA or higher-resolution video adapter
                   6. Personal computer with a 386DX or higher
                      microprocessor
                   7. Hard disk space required:
                         Compact installation: approximately 8 MB
                         Typical installation: approximately 18 MB
                         Custom (maximum) installation: up to 38 MB
                   Changes:
                      1. File Find functionality now built into the File
                         Open dialog box, can search for workbooks even if
                         you don't know the file name you're looking for.
                      2. Custom AutoFill series.
                      3. Shared lists allow multiple users to make changes
                         to workbooks simultaneously.
                      4. Template Wizard with Data Tracking.
                      5. DataMap allows you to create geographical charts.
                      6. Files now have more properties.

 5.0     12/14/93  1. 4 MB of memory
                   2. Hard disk
                   3. MS-DOS version 3.1 or later
                      -and-
                      any of the following:
                      Microsoft Windows version 3.1 or later
                      Microsoft Windows for Workgroups version 3.1 or later
                      Microsoft Windows for Pen Computing
                      -or-
                      Microsoft Windows NT(TM) version 3.1 or later
                   4. 3.5-inch or 5.25-inch high density disk drive
                   5. EGA or higher-resolution monitor compatible
                      with Microsoft Windows 3.1 or later
                   6. Personal computer with a 286 or higher microprocessor
                   7. Hard disk space required:
                      Laptop (minimum) installation: approximately 6 MB
                      Typical installation: approximately 15 MB
                      Custom installation: up to 22 MB
                   Changes:
                      1.  New workbook file format: contains as many sheets
                          as you like, with workbook tabs to navigate.
                      2.  Inclusion of PivotTable Wizard to build a summary
                          table of your data
                      3.  Inclusion of Microsoft Query: application that
                          supports ODBC and can be used to import data to
                          Microsoft Excel
                      4.  New TextWizard feature to import text files and
                          choose the way you want them to be displayed
                      5.  New common macro language, Visual Basic
                          Programming System, Applications Edition
                      6.  Support for directly entering and editing
                          information in cells
                      7.  Support for combination charts: different chart
                          type can be applied to each series
                      8.  Tip Wizard feature that teaches you shortcuts.
                      9.  Display ToolTips for displaying button names.
                      10. All files are now workbooks.
                      11. Support for directly entering and editing
                          information in cells.
                      12. Support for creating named ranges in the Name box
                          on the Formula bar.
                      13. New Function Wizard that creates formulas with
                          worksheet functions.
                      14. 3-D formulas and 3-D names for workbook models.

 4.0a    11/1/92  1. 2 MB of random-access memory (RAM)
                  2. Hard disk
                  3. MS-DOS version 3.1 or later
                  4. Microsoft Windows version 3.0 or later running
                     in standard or 386 enhanced mode
                  5. 1.2 MB 5.25-inch or 1.44 MB 3.5 inch high
                     density drive
                  Changes: Maintenance Release
                     1. Improved support for Lotus 1-2-3 file
                        formatting, including Allways support
                     2. Support for Crosstab reports from network
                        server read-only files
                     3. Minor improvements based on feedback from
                        registered users of Microsoft Excel for
                        Windows version 4.0

 4.0     4/1/92   1. 2 MB of memory (extended)
                  2. Hard disk
                  3. MS-DOS version 3.1 or later
                  4. Microsoft Windows 3.0 or later
                  5. 1.2 MB 5.25-inch or 1.44 MB 3.5 inch
                     high density drive
                  Changes:
                     1. Inclusion of set of statistical, engineering,
                        and financial tools, known as Analysis Tools.
                     2. Custom tool bar features
                     3. Extended charting features, including Chart
                        Wizard
                     4. Support print scaling  (reduce/enlarge)
                     5. Workbook file feature: save group of files as
                        a bound or unbound file.

 3.0    12/9/90   1. 1 MB memory (extended)
                  2. Hard disk
                  3. MS-DOS version 3.1 or higher
                  4. Microsoft Windows 3.0 or later
                  5. 1.2 MB 5.25 inch or 1.44 MB 3.5 inch
                     high density drive
                  Changes:
                     1. Requires Windows 3.0 or later to install and
                        run
                     2. Graphic tool bar feature
                     3. Add-in macro feature (*.XLA macro files)
                     4. Inclusion of Microsoft Excel Solver: an
                        application designed to work with Microsoft
                        Excel for equation solving and optimization.
                     5. Inclusion of Q+E for Microsoft Excel: a
                        front-end relational database application for
                        database file updating and manipulation.
                     6. 3-D charting feature
                     7. Spreadsheet support of graphic images

2.1d     7/13/90  1. 640K of memory
                  2. Hard disk
                  3. MS-DOS version 3.0 or higher
                     (MS-DOS 3.1 or higher with Windows 3.0)
                  4. 1.2 MB 5.25-inch high-density drive or
                     360K 5.25-inch low-density disk drive
                     720K 3.5-inch drive
                  Changes:    Maintenance Release

2.1c    12/14/89  1. 640K of memory
                  2. Hard disk
                  3. MS-DOS version 3.0 or higher
                     (MS-DOS 3.1 or higher with Windows 3.0)
                  4. 1.2 MB 5.25-inch high-density drive or
                     360K 5.25-inch low-density disk drive
                     720K 3.5-inch drive
                  Changes:     Windows 3.0 Compatibility

2.1b    6/30/89   1. 640K of memory
                  2. Hard disk
                  3. MS-DOS version 3.0 or higher
                  4. 1.2 MB 5.25-inch high-density drive or
                     360K 5.25-inch low-density disk drive
                     720K 3.5-inch drive
                  Changes:    Includes run-time Windows 2.11


2.1    7/31/88    1. 640K of memory
                  2. Hard disk
                  3. MS-DOS version 3.0 or higher
                  4. 1.2 MB 5.25-inch high-density drive or
                     720K 3.5-inch drive
                  Changes:
                     1. Inclusion of a dialog editor
                     2. Addition of a Multiplan macro translator
                     3. Technical documentation on hardware and how to
                        use it optimally with Microsoft Excel
                     4. Inclusion of Windows version 2.10 features,
                        including improved setup routine, MEMSET.EXE,
                        HIMEM.SYS, and new device drivers
                        such as the HP PaintJet
                     5. Faster recalculations by 5 percent over
                        earlier versions
                     6. Reduced memory requirements for booting
                        (possible to start Excel with only 400K
                        instead of the 440K required in earlier
                        versions)

2.01  2/29/88     1. 640K of memory
                  2. Hard disk
                  3. MS-DOS version 3.0 or higher
                  4. 1.2 MB 5.25-inch high-density drive or
                     720K 3.5-inch drive
                  Changes:
                     1. International keyboard drivers added
                     2. SYLK enhanced to handle color formats
                     3. Inclusion of PS2EMM.SYS on 3.5-inch disks
                     4. Addition of BLOCK flag to WIN.INI file

2.0   10/31/87    1. 640K of memory
                  2. Hard disk
                  3. MS-DOS version 3.0 or higher
                  4. 1.2 MB 5.25-inch high-density drive or
                     720K 3.5-inch drive
