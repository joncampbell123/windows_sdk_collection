Underline Appears Under Character in User Information Dialog

 Last reviewed: November 2, 1994
Article ID: Q68628

SUMMARY

The first time that the Excel 3.00 setup program is used, you will be asked to enter your name and company name into the User Information dialog. After you have entered the requested information and have chosen OK, a confirmation dialog will appear. If you entered an ampersand character (&) in either box, you will see an underline in the confirmation dialog under the character that appeared after the "&" character.

Workaround

To workaround this, you need to enter two "&" characters in a row for the text to display correctly.

For example, if you entered M&S Baking for the company, MS Baking would appear with an underscore beneath the "S". Entering M&&S Baking will result in M&S Baking being displayed.

Note: Once you have accepted the name in the confirmation dialog, it is not possible to change the name.
