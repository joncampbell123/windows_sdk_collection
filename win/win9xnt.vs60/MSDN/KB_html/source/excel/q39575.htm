Symphony Macros Are Not Translated Correctly

 Last reviewed: November 2, 1994
Article ID: Q39575
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 5.0

The Lotus Symphony macro and Command Language will not translate correctly in many cases using the Excel Macro Translator. The Macro Translator is specifically designed for the macro language of Lotus 1-2-3. The Symphony macro language differs in many respects from the 1-2-3 macro language; therefore, translation will not be correct.

Simple macros can translate correctly, but Microsoft does not guarantee predictable results.
