Symphony UK 2.2 with ATM Displays Huge Row Numbers in Excel

 Last reviewed: November 2, 1994
Article ID: Q74989

SUMMARY

When using Adobe Type Manager (ATM) version 1.0 and importing Symphony UK version 2.2 files into Microsoft Excel 3.0, the row numbers may fill the entire workspace.

MORE INFORMATION

The large row numbers are the only items displayed on the screen. However, the contents of the cells are visible in the formula bar and can be edited. The file can be viewed normally in print preview. Saving the file in the Excel Normal format does not correct the problem. By disabling ATM, the file will display as normal. Updating to ATM version 1.1 may also correct this problem.
