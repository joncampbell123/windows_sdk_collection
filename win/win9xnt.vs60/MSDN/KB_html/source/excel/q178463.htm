XL98: Unable to Change Fill Color for Walls on 3-D Chart

 Last reviewed: February 2, 1998
Article ID: Q178463
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

If you click the Fill Color toolbar button to change the fill color on the walls or floor of a three-dimensional (3-D) chart in Microsoft Excel 98 Macintosh Edition, the fill color may not change.

CAUSE

This problem may occur if all of the following conditions are true:

  • You create a 3-D chart.

    -and-

  • You paste object onto the walls or floor of the chart.

    -and-

  • You click the Fill Color toolbar button on the Formatting toolbar to change the fill color of the walls or floor with the object.

WORKAROUND

To work around this problem, follow these steps:

  1. Select the walls (or floor) of the chart.

  2. On the Format menu, click Selected Walls (or Selected Floor).

  3. On the Patterns tab of the Format Walls (or Format Floor) dialog box, click a color under Area, and then click OK.

NOTE: After you change the fill color by using these steps, the object on the walls (or floor) no longer is visible. You must paste the object onto the walls (or floor) again.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about pasting objects onto a 3-D chart, click the Index tab in Microsoft Excel Help, type the following text 

   3-d charts, formatting
and then double-click the selected text to go to the "Change colors, patterns, lines, fills, and borders" topic.
