Excel Error Msg: Extract Range Is Not Valid

 Last reviewed: November 2, 1994
Article ID: Q28514
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0
  • Microsoft Excel for OS/2, versions 2.2, 3.0

A valid extract range consists of a range of cells containing only field names from the database, or a range of cells containing field names and the cells that will contain the extracted data. If the Data Extract command is executed with any other range of cells selected, the error "Extract Range is not Valid" will occur.

This error will also occur if the field names from the database contain values that are not text values. For example, if a field name from the database contains a number, a logical value, an error value, a blank cell, or a formula, choosing Data Extract will produce this error even if the selected range is otherwise valid.
