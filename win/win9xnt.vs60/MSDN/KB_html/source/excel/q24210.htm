Excel: Reordering Series Names in Legend

 Last reviewed: October 31, 1994
Article ID: Q24210

SUMMARY

In Microsoft Excel, there are two methods for changing the order of the series names in a chart legend. In Excel version 3.0, either method 1 or method 2 (below) will work. In versions of Excel before 3.0, method 2 must be used.

Method 1

  1. From the Chart menu, choose Edit Series.

  2. Change the plot order number of the series by selecting the series to change the plot order for and reducing the plot order number if the series name is to appear higher in the legend, or increasing it if the name is to appear lower.

Method 2

  1. Select the series you want to change by clicking a bar, column, line, or data point in that series. The SERIES formula is now displayed in the formula bar.

  2. Change the plot order number, which is the fourth argument in the SERIES formula. The plot order number should be reduced if the series name is to appear higher in the legend, or increased if it is to appear lower.

For example, to change the third series to be the first, click a plotted value for the third series and change the "3" to a "1" in the SERIES formula.

For more information about editing the SERIES formula, see pages 441-444 of the "Microsoft Excel User's Guide" version 3.0 manual. If you are using version 2.2, see pages 547-580 of the "Microsoft Excel Reference" manual.
