XL97: Cannot Undo Background in Text Box on a Chart Object

 Last reviewed: January 7, 1998
Article ID: Q172684
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel, when you try to restore the background color of a text box by using the Undo command, the background color is not restored to the original color.

CAUSE

This problem occurs if the following conditions are true:

  • You are working with a text box that is on top of a chart object. 

        -and-
  • You format the background of the text box to a color other than the original setting. 

        -and-
  • You attempt to use the Undo command to change the background color back to the original setting.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel 97. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.
