Canon BJ-130 Does Not Display Correct Paper Size in Page Setup

 Last reviewed: November 1, 1994
Article ID: Q76673

SUMMARY

In Microsoft Excel 3.0 the Canon Bubble-Jet 130 driver may display incorrect paper sizes in the Page Setup dialog box, but will print the paper size that has been selected in the Printer Setup dialog box. (To access these dialog boxes, choose Page Setup or Printer Setup from the File menu.)

MORE INFORMATION

If you use the Canon Bubble-Jet 130 driver version 1.0 (from the Supplemental Driver Library [SDL]) with Excel 3.0, the Paper option in the Page Setup dialog box will display ASCII characters and "&Select" (without the quotation marks) as the available paper size options.

This option is not available as a per document attribute with this printer driver and should be dimmed. If you choose Setup from the Print Preview dialog box, the Paper Size text box is unavailable and dimmed.
