Excel: Help Files Difficult to Read on Macintosh Color Screen

 Last reviewed: October 31, 1994
Article ID: Q50452

SUMMARY

The Excel Help files can often be difficult to read on a color monitor. If blue text is selected, the highlight color is black, creating a low contrast, difficult-to-read screen.

There is currently no way to control these colors through Excel. The current version of a third-party start-up document called "KOLOR" is also unable to affect these colors. Microsoft is researching this problem and will post new information here as it becomes available.
