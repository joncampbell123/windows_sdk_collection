XL7: No Shortcut Menu for Template in New Dialog Box

 Last reviewed: March 27, 1997
Article ID: Q132125
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, when you click the right mouse button on a file in the New dialog box to display details about the file, your computer may beep instead of displaying a menu for the file.

CAUSE

This behavior occurs if you save a Microsoft Excel template file to the Xlstart folder. The Microsoft Excel Xlstart folder is available for compatibility with Microsoft Excel version 5.0. In Microsoft Excel version 7.0, you can still create a new workbook from a template stored in this folder. However, when you click the right mouse button on the template file in the New dialog box, your computer may beep instead of displaying a shortcut menu for the file.

All new templates that you create in Microsoft Excel version 7.0 should be saved to either of the following locations 

   MSOffice\Template

   -or-

   MSOffice\Template\Excel
where MSOffice is the folder to which you installed Microsoft Office. When you save a file as a template in Microsoft Excel, the default file location is the Template folder in the Microsoft Office folder.

Normally, the shortcut menu for a template in the New dialog box in Microsoft Excel contains the following commands: 

   New
   Open
   Cut
   Copy
   Create Shorcut
   Delete
   Properties

RESOLUTION

To display the shortcut menu for a template file in the New dialog box, save the file to either the default template folder (that is, the Template folder in the Microsoft Office folder) or the Excel folder in the Template folder, instead of to the Xlstart folder.
