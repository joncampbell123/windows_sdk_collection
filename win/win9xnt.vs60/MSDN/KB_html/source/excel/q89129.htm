Excel: Disk Space Requirements for Excel 4.0

 Last reviewed: November 16, 1994
Article ID: Q89129
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 4.0

Summary:

The disk space required for the installation of Excel 4.0 varies according to which portions of Excel are selected for installation in the Setup program.

More Information:

The following are the disk space requirements for Excel 4.0. These numbers may vary slighty from machine to machine: 

File/Folder              Space Required

Microsoft Excel                   1892K
Spelling Checker                   422K
Macro Library                     1008K
Microsoft Excel Help              1662K
Microsoft Excel Tutorial          1554K
Dialog Editor                       44K
Example Files                      126K
Microsoft Analysis ToolPak         952K
Solver                             304K
Crosstab ReportWizard              632K
Data Access                        174K
  AutoStart                          4K
Space required for complete installation: 8774K Space required for minimum installation: 5296K

If Excel 4.0 is installed over an earlier version, it may not be necessary to free the total amount of disk space required, as earlier files of the same name are replaced by version 4.0.

Note: For a full list of files which are/are not replaced by installing Excel version 4.0 over version 3.0, open the README file on the Excel 4.0 Disk 1-Setup disk and read the section titled: Installing Microsoft Excel Version 4.0 over Version 3.0.
