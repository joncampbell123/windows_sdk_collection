How to Chart Multiple Noncontiguous Areas on an XY Chart

 Last reviewed: November 30, 1994
Article ID: Q32783
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2, version 3.0
  • Microsoft Excel for the Macintosh, version 3.0, 4.0

SUMMARY

When using Microsoft Excel, you can create an XY (scatter) chart with multiple, noncontiguous ranges of Y values.

MORE INFORMATION

Version 5.0

  1. Enter the following into a worksheet 

          A1:      B1:  y   C1:   D1:  y   E1:   F1:  y
      A2:  2   B2:  3   C2:   D2:  1   E2:   F2:  5
      A3:  4   B3:  5   C3:   D3:  3   E3:   F3:  6
      A4:  5   B4:  6   C4:   D4:  7   E4:   F4:  9

  2. Select cells A1:B4. Please note that the Y labels in Row 1 cannot be left out and that cell A1 is left blank.

  3. Holding down CTRL, select cells D1:D4 and F1:F4.

  4. From the Edit menu, choose Copy.

  5. Select cell A1 (or any blank cell).

  6. From the Insert menu, choose Chart, then select As New Sheet.

  7. From the Edit menu, choose Paste Special. Select Values Y in Columns, Categories (X Labels) in First Column, and choose OK.

  8. In the Format menu, select Chart Type, then select XY (Scatter), and choose OK.

Version 3.0, 4.0

  1. Enter the following into a worksheet 

          A1:      B1:  y   C1:   D1:  y   E1:   F1:  y
      A2:  2   B2:  3   C2:   D2:  1   E2:   F2:  5
      A3:  4   B3:  5   C3:   D3:  3   E3:   F3:  6
      A4:  5   B4:  6   C4:   D4:  7   E4:   F4:  9

  2. Select cells A1:B4. Please note that the Y labels in Row 1 cannot be left out and that cell A1 is left blank.

  3. Holding down CTRL, select cells D1:D4 and F1:F4.

  4. From the Edit menu, choose Copy.

  5. Select cell A1 (or any blank cell).

  6. From the File menu, choose New and select Chart.

  7. From the Edit menu, choose Paste Special. Select Values Y in Columns, Categories (X Labels) in First Column, and choose OK.

  8. In the Gallery menu, select the desired XY chart form.

This allows all three Y plots to correspond with the X coordinates.
