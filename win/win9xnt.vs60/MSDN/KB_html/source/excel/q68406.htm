Determining Page Breaks with Objects in Excel

 Last reviewed: November 4, 1994
Article ID: Q68406
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0

Excel breaks objects onto pages according to where the page- break lines cross the object. If an object doesn't fit entirely on a page, Excel clips the part of the object that doesn't fit and places that piece on the next page. You must move and/or size the object to fit within the boundaries of the page-break lines if you want the entire object to fit on the page.
