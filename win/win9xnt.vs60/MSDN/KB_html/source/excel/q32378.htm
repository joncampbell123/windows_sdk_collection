XL: Rounding to a Number That Is Not a Multiple of 10

 Last reviewed: February 19, 1998
Article ID: Q32378
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for the Macintosh, version 5.0
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

The MROUND worksheet function returns a number that is rounded to a specific multiple. However, MROUND requires the Analysis ToolPak add-in to be installed before the function is available. If you do not wish to install the Analysis ToolPak, you can use the ROUND worksheet function instead.

MORE INFORMATION

To round a number to the nearest multiple of "x" by using ROUND in Microsoft Excel, use the following syntax: 

   =ROUND(<value or reference>/x,0)*x
For example, to round a number to the nearest multiple of 5, enter the following into a worksheet: 

   A1:  17     B1:  =ROUND(A1/5,0)*5
The resulting function returns 15.

REFERENCES

For more information about the ROUND function, click the Index tab in Microsoft Excel Help, type the following text 

   ROUND worksheet function
and then double-click the selected text to go to the "ROUND" topic.
