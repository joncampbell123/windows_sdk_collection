Excel DAM: Copying and Pasting in the Query Editor

 Last reviewed: November 2, 1994
Article ID: Q61842

SUMMARY

When accessing external database information from Microsoft Excel using the Data Access Macro (DAM), copy and paste operations can be performed within the Query Editor.

Even though the Copy and Paste commands cannot be chosen from the Edit menu, pressing COMMAND+C executes a copy in the Query Editor and pressing COMMAND+V pastes information from the Clipboard. These key combinations can be used to move information within the Query Editor, as well as to copy information in and out of the dialog box.
