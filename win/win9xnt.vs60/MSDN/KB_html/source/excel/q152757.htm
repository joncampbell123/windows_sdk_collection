SBFM: Wrong Version of Peachtree for Windows Listed in Help

 Last reviewed: October 16, 1996
Article ID: Q152757
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Small Business Financial Manager for Microsoft Excel, version 1.0

SYMPTOMS

In the online help for Microsoft Small Business Financial Manager for Microsoft Excel (SBFM), the version of Peachtree for Windows that is supported by SBFM is incorrectly listed.

STATUS

The driver for Peachtree version 3.5 is available as of 9/1/96.

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q103005
   TITLE     : SBFM: Contents of the README.TXT Peachtree Additional
               Driver

MORE INFORMATION

In the online help for SBFM in the section about "Types of accounting data you can import", there is a table listing the different accounting packages that are supported by SBFM. In this table, on the item for Peachtree for Windows, the supported version listed is 3.x. This is incorrect as the version of Peachtree for Windows that is supported by the driver that originally ships with SBFM is 3.0. The correct version, though, is listed on the back of the box in which SBFM ships.

Note, in the help file topic for Peachtree for Windows, there is information that says the following: 

   The import filters and other drivers you need to import data from
   Peachtree for Windows Version 3.5 will be available within 30 days of
   your purchase. To order this driver directly from Microsoft, please call
   (800) 426-9400 and it will be shipped to you free of charge. You can
   access the driver electronically via the following online locations:

   On CompuServe: Type GO MSEXCEL

   On MSN: Go to the "Free Stuff" area of Microsoft Excel.

REFERENCES

For more information about Peachtree for Windows in Microsoft Small Business Financial Manager for Microsoft Excel, click the Index tab in Microsoft Small Business Financial Manager Help and type: 

   Peachtree for Windows
