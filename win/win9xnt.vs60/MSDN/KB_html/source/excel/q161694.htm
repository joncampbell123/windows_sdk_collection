XL97: Control Toolbox Toolbar Disappears

 Last reviewed: January 15, 1998
Article ID: Q161694
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

When you click Properties on the Control Toolbox toolbar, the Control Toolbox may disappear.

CAUSE

This problem may occur when both of the following conditions are true:

  • The Microsoft Excel workbook is embedded in a Microsoft Word document. 

        -and-
  • The Control Toolbox toolbar is floating.

NOTE: If the Properties window is already visible and you click the Properties window to switch to it, the Control Toolbox toolbar may disappear.

RESOLUTION

To resolve this problem, dock the Control Toolbox toolbar before you click Properties.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

If you activate an embedded Microsoft Excel workbook that is in a Microsoft Word document, display the Control Toolbox toolbar, deactivate the workbook by clicking the Word document, and then reactivate the embedded Microsoft Excel workbook; the toolbar may not be visible. If the toolbar does not reappear, point to Toolbars on the View menu, and then click Control Toolbox.

REFERENCES

For more information about Docking Toolbars, click the Office Assistant, type "docking," click Search, and then click to view "Move a toolbar."

NOTE: If the Assistant is hidden, click the Office Assistant button on the Standard toolbar. If Microsoft Excel Help is not installed on your computer, please see the following article in the Microsoft Knowledge Base:
