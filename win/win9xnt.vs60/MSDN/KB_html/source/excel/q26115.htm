Finding a Range of Dates in a Single-Data Field in Excel

 Last reviewed: November 2, 1994
Article ID: Q26115
The information in this article applies to:
  • Microsoft Excel for Windows versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2 versions 2.2, 2.21, and 3.0

SUMMARY

To perform a logical AND on a set of criteria that apply to the same field name, place the criteria side by side. The field name must also be duplicated in the criteria range. For example, the following criteria will extract records with a Date between 8/1/91 AND 8/10/91 (not inclusive) with a Type of 4.5. 

   Date      Date      Amount   Type
   >8/1/91   <8/10/91           =4.5

MORE INFORMATION

A logical OR of criteria may be accomplished by entering the criteria in different rows. The following is an example of a logical OR: 

   Date      Amount   Type
   ----      ------   ----

   <8/1/91            =4.5
   >8/10/91           =4.5
If an extract was performed on the above example, the records with a Date less than 8/1/91 and a Type of 4.5 OR records with a Date greater than 8/10/91 and a Type of 4.5 would be extracted.
