Data Sort Does Not Keep Border Format

 Last reviewed: November 2, 1994
Article ID: Q61580
Question:

When I do a Data Sort in Excel 2.x, I am able to sort correctly, but my shading does not move. What am I doing wrong?

Response:

When doing a Data Sort, the formatting of the border will not move with the cell data. As a result, the cells that are shaded or outlined will not move with the sort. Any formatting done under Format Number, Alignment, Font, or Cell Protection will still be associated with the cell data, but formatting under Border will not. This situation occurs because the Border format is specific to the actual cell, not the cell contents.

In Excel 3.00, the Format Border settings will be retained with the data after sorting.
