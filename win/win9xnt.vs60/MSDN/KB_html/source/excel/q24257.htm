Excel: Extending a Macro Without Redefining the Macro Range

 Last reviewed: November 29, 1994
Article ID: Q24257
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SUMMARY

To define a Microsoft Excel macro so that it is not necessary to redefine it after having added new steps, define the macro to be just the first cell in the range.

It is not necessary to define the macro to be the entire range. If the macro starts in cell A1 on the macro sheet, define the macro name to refer to cell A1 only. Excel will run the macro that begins at that cell until it runs into a RETURN().
