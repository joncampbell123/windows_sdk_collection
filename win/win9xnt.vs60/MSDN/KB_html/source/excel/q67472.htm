Excel: Outline Gray in Border Dialog Box

 Last reviewed: November 29, 1994
Article ID: Q67472
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0

SUMMARY

When the Border dialog box is displayed when using Format Styles, the Outline border type is gray, and not selectable. To use the Outline border, Border must be selected directly from the Format menu.

Styles apply to single cells; therefore, the format of a style should affect only single cells. Outline is used to draw a border around a group of cells. Therefore, Outline is not a valid border format when defining a cell style.

REFERENCES

For more information on defining styles, see pages 200-205 in the "Microsoft Excel User's Guide," version 3.0. If you are using version 4.0, see pages 241-245 in the "Microsoft Excel User's Guide 1," version 4.0.
