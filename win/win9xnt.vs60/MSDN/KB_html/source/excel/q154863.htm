MS Excel Virus Search Add-In Is Not Needed on the Macintosh

 Last reviewed: February 2, 1998
Article ID: Q154863
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

Microsoft has developed add-ins for Microsoft Excel for Windows, called Xlscan.xla and Xlscan97.xla, that search for and clean workbooks infected with the ExcelMacro/Laroux virus. These add-ins do not work in the Macintosh environment. However, the ExcelMacro/Laroux virus is not propagated among workbooks in the Macintosh environment, and you do not need either of these add-ins to detect the virus.

MORE INFORMATION

If you install Xlscan.xla or Xlscan97.xla in the "Excel: Macro Library" folder on your Macintosh computer and try to load the add-in, you receive the following error message: 

   This Microsoft Excel Virus Search Add-In will not function in the
   Mac environment
Your only option is to click OK.

NOTE: You cannot load the add-in, but it will appear in the Add-Ins dialog box as though installed. To remove the add-in from the list of installed add-ins, follow these steps:

  1. On the Tools menu, click Add-Ins.

  2. In the Add-Ins Available list, clear the check box next to the Microsoft Excel Virus Search, and then click OK.

You can obtain the Microsoft Excel Virus Search add-In from the following sources:
  • Microsoft's World Wide Web Site on the Internet
  • Microsoft Product Support Services
