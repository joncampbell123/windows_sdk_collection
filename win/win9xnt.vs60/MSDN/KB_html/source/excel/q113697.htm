XL5: Examples And Demos Not Compatible with 80286 Processor

 Last reviewed: September 12, 1996
Article ID: Q113697
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

The Examples And Demos tutorial will not run correctly on any 80286-based computer. However, the Quick Preview will run correctly.

Choosing Examples And Demos from the Help menu and selecting a topic and then a subtopic will result in an alert message: 

   Cannot Run CBT. Run the Excel Setup program
   and reinstall CBT files

CAUSE

The Examples And Demos will not run on any 80286 computer. This is a function of the CBT help file (XLHELP.DLL). However, Quick Preview will run on an 80286 because this tutorial is not started from a help file.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
