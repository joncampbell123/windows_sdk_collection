XL98: All Views Listed in Custom Views and Report Manager

 Last reviewed: February 2, 1998
Article ID: Q178871
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In the Custom Views and the Report Manager in Microsoft Excel 98 Macintosh Edition, views that are not defined in the active worksheet are listed in the Views list box and the View drop-down list box.

CAUSE

This occurs because, in Microsoft Excel 98 Macintosh Edition, all views contained within the active workbook are listed in the Views list box and the View drop-down list box, not just views defined within the active worksheet.

STATUS

This behavior is by design of Microsoft Excel 98 Macintosh Edition.

MORE INFORMATION

In earlier versions of Microsoft Excel, when you define views in a workbook and then display the View Manager (now called "Custom Views") or the Report Manager, only views defined within the active worksheet are listed in the Views list box and the View drop-down list box.

In Microsoft Excel 98 Macintosh Edition, all views defined within the active workbook are listed in the Views list box and the View drop-down list box, regardless of which worksheet is currently active.
