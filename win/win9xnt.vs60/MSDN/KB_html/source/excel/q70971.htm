Excel Err Msg: Document Not Saved/Previous Copy Deleted

 Last reviewed: November 29, 1994
Article ID: Q70971
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.5, 2.2, 3.0, 4.0

SUMMARY

In Microsoft Excel, when a document is saved and the save is canceled, the error message "Document not saved, any previously saved copy will have been deleted" is returned. To avoid permanent loss of data, you must save the document again.

MORE INFORMATION

Microsoft Excel saves files by deleting the old file from the hard drive and saving the new document to the same location. If Microsoft Excel is unable to complete the save, it may have deleted the old file during the aborted save process, and no copy of the document is available.
