Changing the Order of Printed Notes in Excel Is Not Possible

 Last reviewed: November 30, 1994
Article ID: Q33358
The information in this article applies to:
  • Microsoft Excel for Windows, version 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2 versions 2.2, 3.0
  • Microsoft Excel for the Macintosh version 2.x, 3.0, 4.0

SUMMARY

If you print cells from Microsoft Excel that contain notes in the spreadsheet, Excel prints notes in the following order: 

   Cell: A1   Cell: B1   Cell: C1
   NOTE:      NOTE:
Excel does not provide an option to change the order of how notes are plotted. Excel prints notes across rows on the worksheet, instead of down columns.
