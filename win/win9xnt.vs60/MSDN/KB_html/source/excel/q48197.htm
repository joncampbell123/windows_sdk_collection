Previous Button Dimmed in Page Preview if Notes Are Displayed

 Last reviewed: October 31, 1994
Article ID: Q48197

SUMMARY

When the Both option is selected in the Print dialog box (on the File menu) in Microsoft Excel version 2.20 (meaning that both the worksheet and the cell notes are to be printed), the Previous button in Page Preview is unavailable (dimmed) when the first page of cell notes is viewed.

Excel cannot go back to the document information once the first page of notes has been displayed in Page Preview.

To view the worksheet pages after the first page of notes has been viewed, click Cancel, then reenter Page Preview.
