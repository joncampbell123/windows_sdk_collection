Off97: Menu Disappears with Kensington TrackBall

 Last reviewed: February 12, 1998
Article ID: Q163269
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Access 97 for Windows
  • Microsoft Outlook 97 for Windows

SYMPTOMS

When you click a menu in any of the programs listed at the beginning of this article, the menu disappears before you click any of the menu commands.

CAUSE

This problem occurs if an older version of the Kensington Expert Mouse (Trackball) driver is installed on your computer. The problem is corrected in versions 5.00b and later of the Kensington software.

RESOLUTION

To resolve this problem, obtain and install the latest version of the Kensington Expert Mouse (Trackball) driver software.

MORE INFORMATION

Normally, you can select a menu command by clicking a menu and then clicking the menu command. However, if an older version of the Kensington Expert Mouse (Trackball) driver software is installed, use the following steps:

  1. Click a menu and hold down the mouse button.

  2. Drag the cursor to the menu command you want to use.

  3. When you select the menu command, release the mouse button.
