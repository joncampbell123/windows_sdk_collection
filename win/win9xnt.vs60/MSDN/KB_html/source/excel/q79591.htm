Excel: Wildcard Characters Do Not Modify OPEN?() File List

 Last reviewed: November 1, 1994
Article ID: Q79591

SUMMARY

The "Microsoft Excel Function Reference" version 3.0 manual, page 169, incorrectly states that the <File_text> parameter for the OPEN?() macro function can be given wildcard characters (asterisk and question mark) to modify the list of available files shown in the Open dialog.

<File_text> only modifies the list in Excel for Windows. In Excel for the Macintosh, all files that can be opened by Excel will be listed regardless of what is specified in the <file_text> parameter.

Reference(s):

"Microsoft Excel Function Reference," version 3.0, pages 169-170
