XL5 Err Msg: Object Library is Corrupted or Missing

 Last reviewed: January 19, 1998
Article ID: Q118401

The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SYMPTOMS

When you start Microsoft Excel for Windows versions 5.0 and 5.0c, you may receive the following error message: 

   Object Library is Corrupted or Missing

CAUSE

This error message may be the result of a corrupted Reg.dat file.

MORE INFORMATION

This error message may be corrected by recreating the Reg.dat file and then registering the individual applications.

For the steps to rebuild the Reg.dat file and to register the applications, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q80393
   TITLE:      How to Rebuild the Default Windows REG.DAT File
