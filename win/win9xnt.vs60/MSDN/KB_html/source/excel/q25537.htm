Excel: Referencing the Active Worksheet in a Macro

 Last reviewed: November 2, 1994
Article ID: Q25537

SUMMARY

In Microsoft Excel, to reference cells on the active worksheet in a macro, enter an exclamation mark (!) directly in front of the cell reference.

For example, to select cell A1 on the active worksheet, use the following command: 

   =SELECT(!A1)
