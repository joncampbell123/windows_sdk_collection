Excel 2.2: F10 Key Causes Hang with Radius Monitor

 Last reviewed: October 31, 1994
Article ID: Q50494
The information in this article applies to:
  • Excel for the Macintosh, version 2.2

SUMMARY

If you are using Microsoft Excel version 2.20 on a machine with a Radius two-page display, Excel may hang when you activate the menu bar by pressing F10 or by pressing the alternate menu key (forward slash [/] by default). This problem occurs with older versions of the Radius INIT if Radius' Tear Off Menus option is enabled in the Control Panel.

To correct this problem, select the Radius INIT in the Control Panel and deselect the option for Tear Off Menus.

To receive the latest INIT, which is fully compatible with Excel 2.20, call Radius at (408) 434-1012.

Note: This problem does not occur with later versions of Microsoft Excel for the Macintosh.
