Excel: White Text Prints As Black on LaserWriter LS

 Last reviewed: November 4, 1994
Article ID: Q80501
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0

SUMMARY

If a range of cells formatted with a white font color is printed on an Apple Personal LaserWriter LS, the first cell in the range will print correctly, but the other cells in this range will print as if the font is black.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.

Steps to Reproduce Problem

  1. Open the Chooser and make sure the LaserWriter LS is the active printer.

  2. Create a new worksheet by choosing New from the File menu, selecting Worksheet, and choosing the OK button.

  3. Highlight cells A1:A10, type "abc" (without quotation marks), then press OPTION+ENTER. This will fill the range of cells with this word.

  4. With the range still highlighted, choose Font from the Format menu, select White for the color, then choose the OK button. The text should now be invisible.

  5. From the File menu, choose Print. Choose the OK button.

The text in the first cell prints white and is not visible, but all remaining cells print black text.
