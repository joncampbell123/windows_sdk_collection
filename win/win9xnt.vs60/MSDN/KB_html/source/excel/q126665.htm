XL5: Contents of Lotus 1-2-3 WK4 File Converter Disk

 Last reviewed: September 12, 1996
Article ID: Q126665
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

The table in the "More Information" section of this article lists the files contained on the Lotus 1-2-3 WK4 File Converter disk.

This converter is available in the "Lotus 1-2-3 WK4 File Converter (WE1130)" Application Note. For information about how to obtain this Application Note, please see the following article in the Microsoft Knowledge Base: 

   TITLE: XL5: AppNote: Lotus 1-2-3 WK4 File Converter (WE1130)
   ARTICLE ID: Q122583

MORE INFORMATION

When you run the Setup program, the Setup program copies the file XLCONVW4.DLL to your Microsoft Excel directory. This is the only file that is copied to the computer's hard drive. The rest of the files contained on the disk are the Setup program files.

NOTE: Filenames with underscores (_)(such as ACMSETUP.EX_) indicate compressed files.
