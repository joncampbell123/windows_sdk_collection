Excel: Creating Blank Lines in User-Defined Help File

 Last reviewed: October 31, 1994
Article ID: Q49263

SUMMARY

When you create a user-defined help file in Microsoft Excel version 3.00 or 2.20, blank lines in the text Help file do not always appear as blank lines in the Help dialog box.

To have the blank lines appear in the Help dialog box, a character, such as a space, must be placed on each blank line in the text file.

For more information about using Custom Help, query on the following words: 

   using and custom and help and Excel and 2.20 and 3.00
