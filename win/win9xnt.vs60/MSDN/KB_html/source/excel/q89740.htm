Excel: GP Fault in Data Form When Linking Database to Word 2.0

 Last reviewed: November 8, 1994
Article ID: Q89740
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0

SYMPTOMS

When linking a database from Microsoft Excel 3.0 or 4.0 to version 2.0 of Microsoft Word for Windows as formatted or unformatted text, Excel causes a Gereral Protection Fault when you return to Excel and use the Data Form to add a new record to the database.

WORKAROUNDS

  • Add records by inserting rows in the database.

    -or-

  • Close the Word document while adding records. Once all the records have been added, open the Word document and update links to add the new data to the table in Word.
