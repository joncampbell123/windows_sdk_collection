Files Installed with the Data Access Options

 Last reviewed: September 13, 1996
Article ID: Q139725
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office for Windows 95, version 7.0

This article contains two sections. The first section lists the components and files that are installed when you choose the Data Access option during setup for any of the programs listed at the beginning of this article. The second section lists all files that are removed, changed, or left on the computer after you remove the Data Access components.

NOTE: This article assumes that you have previously performed a Custom installation and selected all available options except for the Data Access option.

FILES THAT ARE INSTALLED WITH DATA ACCESS

The following Data Access components are installed only during a Custom or maintenance mode setup: 

   Microsoft Query
   Microsoft Query Help and Cue Cards
   Data Access Objects for Visual Basic
   Online Help for Data Access Objects
   Microsoft Access Driver
   dBASE and Microsoft Foxpro Drivers
   Microsoft Excel Driver
   Paradox Driver
   Microsoft SQL Server Driver
   Text Driver
The tables below show the name, location, description, version, and size of each file installed with the Data Access options. Each table lists the files by the folder that they are located in.

FILES THAT ARE REMOVED, CHANGED, OR LEFT ON YOUR

COMPUTER AFTER REMOVING THE DATA ACCESS COMPONENTS

NOTE: The Odbc.ini and Odbcinst.ini files are not removed, but sections pertaining to any drivers being removed are deleted from the files. Also, all files located in the Windows\System directory (listed in the "Files That Are Installed With Data Access" section of this article) are left on your computer.
