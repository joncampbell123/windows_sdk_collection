Cannot Close Window with ALT+HYPHEN+C in Q+E

 Last reviewed: October 31, 1994
Article ID: Q45885
Summary:

With Q+E version 2.10, you cannot press ALT+HYPHEN+C to close a window from the keyboard as you can in Excel. To close a window from the keyboard, press CTRL+F4.

The inability to use ALT+HYPHEN+C also affects the windows displayed by choosing Show Order from the Sort menu, Show Conditions from the Select menu, Show Joins from the Select menu, or Show Memo from the Edit menu. Once a topic from Help is selected, the information is also displayed in a window.

Microsoft has confirmed this to be a problem in Q+E versions 2.10 and 2.10c. This problem has been corrected in Q+E version 2.50.
