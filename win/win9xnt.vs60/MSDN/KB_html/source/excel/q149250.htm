Accounting Format and Wrap Text Display Erroneous Characters

 Last reviewed: September 12, 1996
Article ID: Q149250
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0c

SYMPTOMS

In Microsoft Excel version 5.0c, if you turn the Wrap Text option on in adjacent cells in at least two columns that are formatted with any of the accounting formats, some erroneous characters may be displayed. The characters appear in the second and successive columns, but not in the first.

WORKAROUND

To work around this problem, do either of the following:

  • Turn off the Wrap Text option. (To locate the Wrap Text check box, on the Format menu, click Cells, and then click the Alignment tab.)

    -or-

  • Use a regular number format instead of an accounting number format.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about the Wrap Text option, click the Index tab in Microsoft Excel Help, type the following text 

   wrap
and then double-click the selected text to go to the "Wrapping text in cells" topic.
