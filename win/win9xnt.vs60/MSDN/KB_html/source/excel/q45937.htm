Explanation of the All Radio Button in File Open Dialog

 Last reviewed: October 31, 1994
Article ID: Q45937

SUMMARY

When the All radio button is selected from the File Open dialog box, only files with .DBF or .QEF extensions appear in the Files list box. Since Q+E can open only these two types of files, there is no need to display any other types.

If a file such as TEST.XLS is typed into the File Name Edit box and OK is selected, the dialog refreshes and TEST.??F appears in the File Name Edit box.

To open index files, use the File Open Index command.
