XL97: Cannot Use Form on FrontPage Personal Web Server

 Last reviewed: February 12, 1998
Article ID: Q162024
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft FrontPage 97 for Windows
  • Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

If you create a form by using the Microsoft Excel 97 Web Form Wizard, you may not be able to use the form or the other files that are created by the Web Form Wizard when you click "Microsoft Internet Information Server with the Internet Database Connector" in step 3 of the wizard.

CAUSE

This problem occurs when your Web is located on a FrontPage Personal Web Server. The FrontPage Personal Web Server does not support the Internet Database Connector interface.

RESOLUTION

To resolve this problem, move the files that are created by the Web Form Wizard to another Web server, such as an Internet Information Server or a Microsoft Personal Web Server. Then, rerun the Web Form Wizard so that the name of the new server is correctly entered in all the files that are created by the wizard.

NOTE: Microsoft FrontPage 97 includes both the FrontPage Personal Web Server and the Microsoft Personal Web Server.

STATUS

This behavior is by design of Microsoft Excel and the Microsoft FrontPage Personal Web Server.

MORE INFORMATION

When you click "Microsoft Internet Information Server with the Internet Database Connector" in the Web Form Wizard, four files are created. The following table lists the four files (the default names are listed) that are created. 

   File         Description
   ------------------------------------------------------------------------

   Myform.xls   Imported into your web; is the form used to collect
                data

   Myform.mdb   Data from your form is written to this Microsoft Access
                database. A System data source must be created on the
                Web server for this database.

   Myform.idc   Must be copied to the Scripts folder on the Web server;
                contains the Structured Query Language (SQL) query and
                database connection information

   Myform.htx   Must be copied into the Scripts folder on the Web server;
                contains the format for the query results

REFERENCES

For more information about the Web Form Wizard, click the Index tab in Microsoft Excel Help, type the following text 

   web form wizard
and then double-click the selected text to go to the "About setting up your data gathering system" topic.
