XL97: Background Turns Black Pasting Object in 3-D Chart Floor

 Last reviewed: January 15, 1998
Article ID: Q157826
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you paste an object into the floor of a three-dimensional (3-D) chart, the background color of the floor may turn black.

CAUSE

This problem occurs when you set the Area pattern of the floor to None before you paste the object in the chart.

NOTE: If you select the floor of the chart and press DELETE, you set the Area pattern to None by default.

WORKAROUND

To paste an object into the floor of a 3-D chart and prevent the background floor color from turning black, do not set the Area pattern of the floor to None before you paste the object.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about adding pictures to charts, click the Index tab in Microsoft Excel 97 Help, type the following text 

   chart formatting, adding and removing pictures
and then double-click the selected text to go to the "Add a picture to a chart item" topic.
