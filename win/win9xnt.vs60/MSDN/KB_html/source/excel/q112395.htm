XL5: Not All Files Removed in Upgrade to 5.0 from 3.0

 Last reviewed: September 12, 1996
Article ID: Q112395
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0 and 5.0

SUMMARY

When you upgrade from Microsoft Excel version 3.0 to Microsoft Excel version 5.0, not all of the version 3.0 program files are removed. These files are not removed because the setup program for Microsoft Excel version 5.0 was designed to remove only files associated with the earlier version of Microsoft Excel (that is, versions 4.0 and 4.0a). These remaining files will not affect the performance of the new version.

MORE INFORMATION

For the best installation of Microsoft Excel version 5.0, install Microsoft Excel to a different directory than the one where 3.0 is located, and then manually delete the Microsoft Excel version 3.0 files from the old directory.

NOTE: Be certain that you do not delete any data files that you may have saved in the Microsoft Excel version 3.0 directory.

A list of those files that remain are provided below for informational purposes. There are about 2 megabytes (MB) worth of files that are not removed when you update from 3.0 to 5.0.
