Excel: Faster Option Unavailable in StyleWriter Print Dialog

 Last reviewed: November 4, 1994
Article ID: Q71337
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0

Summary:

When you print from Microsoft Excel to the Apple StyleWriter using Apple's StyleWriter driver version 7.0, the Faster option is unavailable in the Print dialog box (on the File menu).

Excel prints at the highest resolution for the printer, in this case 360 dots per inch (dpi). At this resolution, the StyleWriter does not support the Faster option. The StyleWriter prints at a lower resolution when printing Faster.
