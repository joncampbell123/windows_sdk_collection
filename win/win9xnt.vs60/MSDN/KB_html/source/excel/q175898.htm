XL98: Toolbars Disappear When UserForm Is Visible

 Last reviewed: February 2, 1998
Article ID: Q175898
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you close a custom dialog box (a UserForm), the Microsoft Excel toolbars do not appear. When you try to quit Excel, the program appears to stop responding (hang).

CAUSE

This problem occurs when all of the following conditions are true:

  • You create a UserForm and display it in the Visual Basic Editor.

    -and-

  • You close the Visual Basic Editor while the UserForm is visible.

    -and-

  • You close the UserForm, and then attempt to quit the program.

WORKAROUND

To work around this problem, switch to the Finder from the Application menu (in the upper-right corner of the desktop). Then switch back to the Excel program.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
