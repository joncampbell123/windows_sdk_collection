<ChartType> Group Commands Missing in Embedded Chart Object

 Last reviewed: September 12, 1996
Article ID: Q115132
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SYMPTOMS

When you edit a Microsoft Excel 5.0 Chart object in a client application, the <ChartType> Group commands (where <ChartType> is the name of the chart type groups in the activated chart object) are not listed on the Format menu.

CAUSE

The <ChartType> Group commands are displayed on the Format menu when a chart is the active sheet in a workbook. However, when you activate a Microsoft Excel 5.0 Chart object, these commands are not listed until you select any of the objects on the chart.

Note that this behavior does not occur under the following conditions:

  • Microsoft Excel is running when you edit the chart object.

    -and-

  • You have created a chart in the current session of Microsoft Excel.

WORKAROUND

To make the <ChartType> Group commands appear on the Format menu, do either of the following:

  • Select any item in the chart, such as the legend.

    -or-

  • Switch to another application by pressing ALT+TAB, and then switching back to the client application by pressing ALT+TAB again.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you edit a Microsoft Excel chart, a command appears on the Format menu for each of the chart type groups in the active chart. For example, if the active chart contains a column chart type, and an area chart type, the following commands are added to the bottom of the Format menu: 

   1 Column Group
   2 Area Group
You can use these commands to edit chart type groups in your chart separately.

REFERENCES

For more information about the Subtype Tab [chart type] Group Command (Format Menu), choose the Search button in Help and type: 

    subtypes, chart
