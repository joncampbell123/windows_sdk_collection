Excel: File Transfers Between 4th Dimension and Excel

 Last reviewed: November 2, 1994
Article ID: Q50491

SUMMARY

To import 4th Dimension documents into Microsoft Excel, do the following:

  1. Open the file in 4th Dimension.

  2. From the Environment menu, choose User. In 4th Dimension version 2.1, choose Export Data from the Fill menu.

  3. Select the Text option.

  4. Set the Field delimiter to 9 and the Record delimiter to 13.

  5. Rename the file and click Save.

This file can be opened in Excel; however, all formatting will be lost. 4th Dimension cannot read or write an Excel-compatible SYLK file; therefore, text is the only method of file transfer.

To export documents from Excel to 4th Dimension, do the following:

  1. Open the Excel worksheet you want to transfer.

  2. Make sure the worksheet consists only of the database. If the worksheet contains any other defined names or information, delete them from the worksheet.

  3. From the File menu, choose Save As.

  4. In Excel version 2.20, click the Options button, select the Text option, and click OK. In Excel versions 1.50 and earlier, select the Text option in the Save As dialog box.

  5. Type in a new name for the worksheet and click Save.

When you open the file in 4th Dimension, make sure a file with the appropriate fields has been predefined. Documents can only be imported after you choose User from the Environment menu.
