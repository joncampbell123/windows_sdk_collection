Excel Err Msg: Change Disk Cannot Find <Filename.dll>...

 Last reviewed: November 3, 1994
Article ID: Q68418
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0

Versions of Microsoft Excel earlier than version 3.00 require dynamic link library files to have the extension .EXE. If you attempt to use these existing libraries with Excel 3.00, you will receive the error, "Change Disk cannot find <filename>.dll please insert into drive a:."

To correct this problem you must rename your library from <FILENAME>.EXE to <FILENAME>.DLL.
