XL98: Type 2 Error Appears After Viewing "About Microsoft Excel"

 Last reviewed: February 2, 1998
Article ID: Q176248
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, if you display the About Microsoft Excel dialog box while you rename a sheet tab, you may receive the following error message after you dismiss the dialog box: 

   The application "unknown" has unexpectedly quit, because an error of
   type 2 occurred.
When this problem occurs, your computer may stop responding.

CAUSE

This problem occurs when you do the following:

  1. Double-click a sheet tab, or click the Format menu, point to Sheet, and then click Rename.

  2. On the Apple menu, click About Microsoft Excel.

  3. Press the TAB key, the RETURN key, or the ENTER key. Then, click OK in the About Microsoft Excel dialog box.

  4. Press any key on the keyboard.

RESOLUTION

To resolve this problem, do not display the About Microsoft Excel dialog box when you are renaming a sheet.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
