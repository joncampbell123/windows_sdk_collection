Excel: Batch Printing of Files Without a Macro

 Last reviewed: November 30, 1994
Article ID: Q87335

The information in this article applies to:

  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0
  • Microsoft Excel for OS/2, version 3.0

SUMMARY

Microsoft Excel allows worksheets and macro sheets to be batch printed using the Workgroup feature in Microsoft Excel version 3.0 and the Group Edit feature in Microsoft Excel version 4.0.

MORE INFORMATION

With the files that you want to batch print open in Microsoft Excel version 4.0, choose Group Edit from the Options menu. In Microsoft Excel version 3.0, choose Workgroup from the Window menu. From the dialog box, select the files to be printed. You cannot include a separate chart file in a Group Edit in Microsoft Excel version 4.0 or a Workgroup in Microsoft Excel version 3.0, but you can embed the chart on a worksheet which is included in the group.

The group association will be indicated in Microsoft Excel version 4.0 by "[Group]" and by "[Workgroup]" in Microsoft Excel version 3.0.

Choose Print from the File menu to print each document in the group.
