Excel Err Msg: Excel.inf Is Missing the checksum...

 Last reviewed: June 12, 1995
Article ID: Q69615
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0

When you attempt to run the Microsoft Excel Setup program, you may encounter the error message 

  "Excel.inf is missing the checksum section. Setup cannot proceed."
This error may occur if you have write-protected your disk. Remove the protection and try again.
