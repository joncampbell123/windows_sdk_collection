XL5: CELL() Function Does Not Return the Worksheet Name

 Last reviewed: September 12, 1996
Article ID: Q132282
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel version 5.0 for Windows, the CELL() function may not return the name of the worksheet for the reference when you use "filename" for the Info_Type argument. When you specify "filename" with the CELL() function, the function should return the filename (including full path) of the file that contains the reference.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been corrected in Microsoft Excel for Windows, version 5.0c.
