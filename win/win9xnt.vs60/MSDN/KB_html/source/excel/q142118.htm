How to Obtain Missing Disks or Manuals

 Last reviewed: February 2, 1998
Article ID: Q142118
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, versions 7.0, 7.0b
  • Microsoft Office for Windows, versions 4.2, 4.2c, 4.3, 4.3c
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Office for the Macintosh, versions 4.2, 4.21, 4.21a
  • Microsoft Office for Windows NT, version 4.x

SUMMARY

You can obtain missing disks or manuals from the Microsoft Order Desk.

MORE INFORMATION

If the product that is missing a disk or manual was purchased from a local retailer, please return to the retailer for an immediate exchange. If the product was purchased elsewhere, or if the retailer will not exchange the product or has no more copies, contact the Microsoft Order Desk at (800) 360-7561. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm
