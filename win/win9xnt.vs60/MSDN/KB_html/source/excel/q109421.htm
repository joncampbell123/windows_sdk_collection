XL5: Can't Preview Word for Windows Files in Find File

 Last reviewed: September 12, 1996
Article ID: Q109421
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Word for Windows, versions 2.x, 6.0
  • Microsoft Office for Windows, version 4.2

SUMMARY

You cannot use the Find File feature in Microsoft Excel version 5.0 for Windows to display a preview of files saved in Microsoft Word version 2.x or 6.0 for Windows.

MORE INFORMATION

You cannot use the Microsoft Excel Find File feature to preview some Word for Windows files because versions 2.x and 6.0 Microsoft Word for Windows do not save object linking and embedding (OLE) information to files. The Find File feature in Microsoft Excel needs this OLE information to generate a preview of a file: because this information is not available in the Word file, the Find File feature cannot show a preview of Word files.

Note that you CAN use the Find File feature in Microsoft Word version 6.0 for Windows to display a preview of any version Microsoft Excel for Windows file. The Microsoft Word Find File feature includes a file converter that can read a Microsoft Excel file and display a preview of it (it does not depend on the presence of OLE information to display a preview).
