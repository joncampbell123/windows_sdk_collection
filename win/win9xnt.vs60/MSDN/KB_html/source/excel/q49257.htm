Excel: Custom Menus Not Appearing When Macro Is Run

 Last reviewed: October 31, 1994
Article ID: Q49257

SUMMARY

If one of Microsoft Excel's built-in menu bars is modified, those modifications will not appear if that particular menu bar is not displayed. If a custom menu does not appear when a macro is run, check to see if the correct menu bar is being viewed.

For example, if you redefine menu bar 1 (full menus for worksheets or macro sheets), that menu bar will not be displayed while Excel is in short menus (menu bar 5 for worksheets and macro sheets.)

To find out the menu ID of the active menu bar, use the following macro command: 

   =GET.BAR()

MORE INFORMATION

There are six built-in menu bars in Excel version 3.00 and 2.20. They are as follows: 

   Menu        Bar
   ----        ---

   1           Full menu macro sheet and worksheet bars
   2           Full menu chart bar
   3           Null menu bar
   4           Info menu bar
   5           Short menu macro sheet and worksheet bars
   6           Short menu chart bar
