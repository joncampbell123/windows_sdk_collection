XL: Cannot Use Cut, Copy, or Paste in Dialog Boxes

 Last reviewed: February 5, 1998
Article ID: Q175995
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you use Microsoft Excel for the Macintosh, you cannot copy data from the worksheet and paste it into a dialog box. Likewise, you cannot copy data from dialog box and paste it into a worksheet.

Also, you cannot copy and paste data within a dialog box.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The dialog boxes in which the Cut, Copy, and Paste commands do not work are the following: 

   Find
   Replace
   Go To
   Summary Info

REFERENCES

For more information about the Copy and Paste commands, click the Index tab in Microsoft Excel Help, type the following text 

   copying cell
and then double-click the selected text to go to the "Move or copy cell data" topic.
