SBFM97: Quickbooks Version 5.0 Filter Is Available

 Last reviewed: January 7, 1998
Article ID: Q171341
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Small Business Financial Manager 97 for Microsoft Excel 97

SUMMARY

A Quickbooks version 5.0 filter for the Small Business Financial Manager 97 is available. You can download this update free of charge. The self- extracting file contains two files: Readme.txt and Aapqbw50.dll.

NOTE: This filter has been tested with releases 1.0 through 4.0 of Quickbooks 5.0.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Sbfmqb5.exe (size: 301056 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
