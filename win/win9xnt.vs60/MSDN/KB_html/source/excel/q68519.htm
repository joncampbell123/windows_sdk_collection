Can't Apply Names If the Name Refers to Formula or Constant

 Last reviewed: November 3, 1994
Article ID: Q68519
The information in this article applies to: - Microsoft Excel for Windows versions 2.x, 3.0, 4.0 - Microsoft Excel for OS/2 versions 2.2, 2.21, and 3.0

SUMMARY

If you define a name to refer to a formula or a constant, you cannot use the Apply Names command to replace every occurrence of that formula or constant with that name. You can only use this command if the name refers to a reference.

Workaround

Use the Formula Replace command. Replace the formula or constant with the name that represents it.
