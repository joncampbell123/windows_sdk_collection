XL98: Cannot AutoFit Rows/Columns That Contain Merged Cells

 Last reviewed: February 2, 1998
Article ID: Q179045
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, you may be unable to AutoFit columns or rows that contain merged cells.

CAUSE

In Microsoft Excel 98, you cannot AutoFit a column that contains a cell merged with cells in other columns. Likewise, you cannot AutoFit a row that contains a cell merged with cells in other rows.

STATUS

This is by design in Microsoft Excel 98 Macintosh Edition.

REFERENCES

For more information about adjusting row widths and column heights, click the Index tab in Microsoft Excel Help, type the following text 

   columns, width
click the Display button and then double-click the topic "Change column width and row height."
