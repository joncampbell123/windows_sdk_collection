Excel: Single-Ply WK3 Files Open as Workbooks

 Last reviewed: November 29, 1994
Article ID: Q85973
The information in this article applies to:

- Microsoft Excel for Windows, version 4.0 - Microsoft Excel for the Macintosh, version 4.0

SUMMARY

Page 27 of "Switching to Microsoft Excel from Lotus 1-2-3" states: "If the WK3 file is a single-sheet file, it will open as a WK3 file and will not convert to a workbook file." This is a documentation error.

In Microsoft Excel, all WK3 files, regardless of whether they are single-sheet or multiple-sheet, are converted to workbooks. The workbook will assume the name originally given to the WK3 file and the single document will be named "A" in keeping with the Microsoft Excel standard of naming WK3 plys.
