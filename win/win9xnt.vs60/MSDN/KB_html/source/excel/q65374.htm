Excel: MacroMaker (Prior to 1.0.2) Macros Run Indefinitely

 Last reviewed: November 2, 1994
Article ID: Q65374

SUMMARY

Versions of MacroMaker prior to version 1.0.2 run indefinitely when used with Microsoft Excel version 2.20. Once a macro is invoked, it will run correctly, but when it is finished, the MacroMaker icon remains highlighted in the menu bar, and the only way to end the macro is to click the mouse button.

Apple has corrected this problem in MacroMaker version 1.0.2, which is shipped with System version 6.0.5.

MORE INFORMATION

For information on installing and using MacroMaker, see Chapter 6 of the "Macintosh Utilities User's Guide."
