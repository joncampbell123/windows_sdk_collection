XL7: Data Map Colors Lost When File Exported to Macintosh

 Last reviewed: February 2, 1998
Article ID: Q134261
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for the Macintosh, version 5.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you export a file that contains a Data Map object to Microsoft Excel 5.0 for the Macintosh or Microsoft Excel 98 Macintosh Edition, the map may become dithered and may lose most of its color.

CAUSE

The color is lost because of the differences between the Microsoft Windows color scheme and the Macintosh color scheme.
