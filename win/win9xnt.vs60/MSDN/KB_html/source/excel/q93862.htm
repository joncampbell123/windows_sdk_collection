Excel: Columns Half Width or Invisible in Column Chart Overlay

 Last reviewed: September 28, 1995
Article ID: Q93862
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SYMPTOMS

In Microsoft Excel, if you use a column chart overlay with a line chart, the first and last columns in the column chart overlay may be displayed at half the width of the other columns or they may not be visible at all.

WORKAROUND

To display the chart so that the columns are all displayed and are all the same width, use the appropriate method below.

Method 1: If you are using Microsoft Excel version 5.0 or later: 

          1. Select the chart and click AutoFormat on the Format menu.

          2. Under Galleries, click Combination, and click the first option
             in the Formats box.
Method 2: If you are using Microsoft Excel version 4.0a or earlier: 

          Use the following steps to use the Combination Chart 3 in the
          Combination dialog box (from the Gallery menu, choose
          Combination), and remove the y-axis value overlay.

          1. From the Gallery Menu, choose Combination.

          2. In the Chart Gallery dialog box, select option 3.

          3. Choose the OK button.

          4. From the Format menu, choose Overlay.

          5. In the Format Chart dialog box, select Column from the Overlay
             Chart Type list.

          6. Choose the OK button.

          7. From the Chart menu, choose Axes.

          8. In the Axes dialog box, clear the Value (Y) Axis option under
             Overlay.

          9. Choose the OK button.
