XL97: Nothing Is in Printout of Embedded MS Excel Chart

 Last reviewed: January 15, 1998
Article ID: Q160099
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

When you print an embedded Microsoft Excel 97 chart from a PowerPoint presentation, nothing may be printed on the page.

CAUSE

This behavior may occur when all of the following conditions are true:

  • In Microsoft PowerPoint, you click Options on the Tools menu, click the Print tab, and clear the "Print inserted objects at printer resolution" check box. 

        -and-
  • You create an embedded Microsoft Excel chart in a PowerPoint presentation. 

        -and-
  • You activate the embedded chart.

WORKAROUND

To work around this problem, use the following steps:

  1. If the embedded Microsoft Excel chart is activated, click the PowerPoint slide that contains the chart to deactivate it.

  2. On the Tools menu, click Options, click the Print tab, and select the "Print inserted objects at printer resolution" check box.

  3. Click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about embedding objects, click the Index tab in Microsoft PowerPoint Help, type the following text 

   embedded objects, charts
and then double-click the selected text to go to the "Add a chart to a presentation" topic.
