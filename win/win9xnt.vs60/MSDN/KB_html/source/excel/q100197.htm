Excel: Macro Interpreter Won't Run International 1-2-3 Macro

 Last reviewed: September 12, 1996
Article ID: Q100197
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0

SUMMARY

In Microsoft Excel versions listed above, the Macro Interpreter for Lotus 1- 2-3 Users will not run a macro created in a foreign version of Lotus 1-2-3. Likewise, a United States (US) version of a Lotus 1-2-3 macro will not run in an international version of Microsoft Excel. The Lotus 1-2-3 macro must be translated to the same language as your Microsoft Excel version.

MORE INFORMATION

Unlike Microsoft Excel, Lotus 1-2-3 does not support international compatibility with its macros. As a result, a US Lotus 1-2-3 macro can only be run in US Lotus 1-2-3 and, therefore, US Microsoft Excel.

Similar to the Macro Interpreter, the Macro Translator only works on 1-2-3 macros in the same language as your Microsoft Excel version.

REFERENCES

"User's Guide," version 5.0, chapter 3 "Switching to Microsoft Excel from Lotus 1-2-3," page 33
