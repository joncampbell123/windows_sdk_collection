Order of Files Saved in Excel Workspace

 Last reviewed: November 2, 1994
Article ID: Q29528
The information in this article applies to:
  • Microsoft Excel for Windows versions 2.x, 3.0, 4.0
  • Microsoft Excel for OS/2 versions 2.2, 2.21, and 3.0

SUMMARY

When opening a workspace file that contains a dependent sheet with external references to other supporting worksheets, the message "Update external references?" appears when opening the workspace file.

MORE INFORMATION

When using the Save Workspace command, the worksheets are saved in alphabetical order, the same way they are listed under the Windows command. The worksheets also will be opened in the same order.

If a worksheet with external references comes before a supporting worksheet in the array returned by the WINDOWS function, the message will appear. To prevent the message from appearing, the dependent sheet must follow all of its supporting worksheets in the array.
