Excel: SYLK File Format Additional P Record Fields

 Last reviewed: November 2, 1994
Article ID: Q64338

SUMMARY

The SYLK file format P record fields are as follows: 

   ;Pnum
   ;Ffontname
   ;Efontname

MORE INFORMATION

The first four font definitions are stored in ;F fields. The remaining definitions are stored in ;E fields. ;F stands for "Font" and ;E stands for "Extended Font" or "Extended Definition."

The subfields of the ;E and ;F fields are as follows: 

   Subfield     Description
   --------     -----------

   ;Mnum        Size of font * 20
   ;Sstyle      Style of the font (bold, italic, etc.)
   ;Lnum        Color of the font
For example, the line 

   P;FGeneva;M200;SB;L4
would be Geneva 10pt, bold, green.

For more information on the SYLK file format, query on the following words: 

   SYLK and file and format
