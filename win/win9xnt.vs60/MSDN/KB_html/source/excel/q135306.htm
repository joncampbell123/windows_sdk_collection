XL7: Strange Cursor Behavior When Control Menu is Displayed

 Last reviewed: September 13, 1996
Article ID: Q135306
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

When you use Microsoft Excel for Windows 95, version 7.0, the insertion point may not behave as expected when you click the right mouse button.

CAUSE

This behavior will occur if both of the following conditions are true:

  • Your document window is in a restored state (not maximized).

    -and-

  • Your document control menu is activated with the primary mouse button.

Normally, when a menu or dialog box is displayed, clicking the right mouse button will not perform any action. However, when the conditions above are met and you click anywhere in the application window, the insertion point will behave strangely causing the active cell to change erratically.
