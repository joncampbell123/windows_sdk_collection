XL: Supported Graphic File Formats for Import

 Last reviewed: June 3, 1997
Article ID: Q118374
5.00 5.00c 7.00 97 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0

SUMMARY

In Microsoft Excel, you can import a number of different graphics file formats. The following is a list of the supported graphic file formats that you can insert on a chart or worksheet by clicking Picture on the Insert menu.

MORE INFORMATION

Microsoft Excel 97

The following table lists the graphic file formats that Microsoft Excel can import. If you are unable to insert a picture that is in one of these formats, you may not have the correct graphic filter installed. Note that you do not need a filter to insert Windows Metafile (.wmf), Windows Bitmap (.bmp, .rle, .dib), Enhanced Metafile (.emf), Joint Photographic Experts Group (.jpg), or Portable Network Graphics (.png) images.

NOTE: Filters for the AutoCAD 2-D, Computer Graphics Metafile, Enhanced Metafile, Kodak Photo CD, Targa, Windows Bitmap, Windows Metafile, and WordPerfect Graphics file formats are available only if you installed Microsoft Excel through the Microsoft Office Setup program.

All of these filters are installed in the \Program Files\Common Files\Microsoft Shared\Grphflt folder. 

   Graphics Filter                     File Name
   -----------------------------------------------

   AutoCAD Format 2-D (.dxf)           Dxfimp32.flt
   Computer Graphics Metafile (.cgm)   Cgmimp32.flt
   Encapsulated PostScript (.eps)      Epsimp32.flt
   Enhanced Metafile (.emf)            Emfimp32.flt
   Graphics Interchange Format (.gif)  Gifimp32.flt
   JPEG File Interchange Format (.jpg) Jpegim32.flt
   Kodak Photo CD (.pcd)               Pcdimp32.flt
   Portable Network Graphics (.png)    Png32.flt
   Micrografx Designer/Draw (.drw)     Drwimp32.flt
   Targa (.tga)                        Tgaimp32.flt
   Tagged Image File Format (.tif)     Tiffim32.flt
   WordPerfect Graphics (.wpg)         Wpgimp32.flt
   PC Paintbrush (.pcx)                Pcximp32.flt
   CorelDRAW 3.0 (.cdr)                Cdrimp32.flt
   The Macintosh PICT (.pct)           Pictim32.flt
   Windows Metafile (.wmf)             Wmfimp32.flt
   Windows Bitmap (.bmp, .rle, .dib)   Bmpimp32.flt

Microsoft Excel version 7.0

The following table lists the graphic file formats that Microsoft Excel can import. If you are unable to insert a picture that is in one of these formats, you may not have the correct graphic filter installed. Note, you do not need a filter to insert a Windows Metafile (.wmf) or a Windows Bitmap (.bmp) image.

All of these filters are installed in the \Program Files\Common Files\Microsoft Shared\Grphflt folder. 

   Graphics Filter                    File Name
   -----------------------------------------------

   Computer Graphics Metafile (.cgm)  Cgmimp32.flt
   Encapsulated PostScript (.eps)     Epsimp32.flt
   HP Graphics Language (HPGL)        Hpglim32.flt
   Micrografx Designer/Draw (.drw)    Drwimp32.flt
   Tagged Image File Format (.tif)    Tiffim32.flt
   WordPerfect Graphics (.wpg)        Wpgimp32.flt
   PC Paintbrush (.pcx)               Pcximp32.flt
   CorelDRAW 3.0 (.cdr)               Cdrimp32.flt
   The Macintosh PICT (.pct)          Pictim32.flt
   Windows Metafile (.wmf)            Wmfimp32.flt
   Windows Bitmap (.bmp, .rle, .dib)  Bmpimp32.flt

Microsoft Excel version 5.0

The Microsoft Excel Setup program gives you the option to install the following graphic file filters under the Graphics Filters option: 

   Windows Metafile (WMFIMP.FLT)
   HP Graphic Language (HPGLIMP.FLT)
   Computer Graphics Metafile (CGMIMP.FLT)
   Encapsulated PostScript (EPSIMP.FLT)
   Tagged Image Format (TIFFIMP.FLT, IFFTIFF.DLL)
   PC Paintbrush (PCXIMP.FLT, IFFPCX.DLL)
These file filters are located in the \Windows\Msapps\Grphflt folder, along with any other Microsoft shared graphic filters. If you have installed other Microsoft applications for Windows, such as Microsoft Word for Windows, you may be able to import more graphic file formats than are listed above.
