XL97: Some Objects Not Saved in Custom Chart

 Last reviewed: February 12, 1998
Article ID: Q165930
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you save a text box, bitmap, or floating text in a user-defined chart, the objects fail to appear in the user-defined chart the next time you use it.

WORKAROUND

To work around this behavior, wait to save text boxes, bitmaps, or floating text in a user-defined chart. Add these objects to the chart after you finish creating the chart.

STATUS

This behavior is by design in Microsoft Excel 97

REFERENCES

For more information about creating charts, click the Index tab in Microsoft Excel 97 Help, type the following text 

   charts, creating
and then double-click the selected text to go to the "Create a chart (Microsoft Excel)" topic.
