XL: Zip Codes in Data Map Must Be Formatted with Zip Code Style

 Last reviewed: June 3, 1997
Article ID: Q131929
7.00 97 WINDOWS kbole kbtool

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SYMPTOMS

If you select a column of unformatted zip codes and you attempt to create a Data Map object from this information, you will receive the following error message in Microsoft Excel version 7.0: 

   A column of data in your selection must contain geographical data.
In Microsoft Excel 97, no error message displays, but no data populates the chart.

CAUSE

When you create a Data Map object based on zip codes, the zip codes must be formatted with the special Zip Code number format.

WORKAROUND

To format a range of zip codes, use the following steps:

  1. On the Format menu, click Cells.

  2. Click the Number tab.

  3. In the Category list, click Special.

  4. In the Type list, click the Zip Code or the Zip Code + 4 format.

STATUS

This behavior is by design to prevent the possible misinterpretation of numeric data as zip codes.
