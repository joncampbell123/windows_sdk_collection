XL Err Msg: "Cannot Use That Command on Multiple Selections"

 Last reviewed: February 18, 1998
Article ID: Q141763
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel for the Power Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

In Microsoft Excel, if you select a range that includes nonadjacent cells and attempt to perform a command on those cells, you may receive one of the following error messages: 

   Cannot use that command on multiple selections

   -or-

   The command you chose cannot be performed with multiple selections.
   Select a single range and click the command again.

MORE INFORMATION

In Microsoft Excel, the error message appears when you use one of the following commands with a nonadjacent cell range.

Microsoft Excel 97

   AutoFormat
   Sort
   AutoFilter
   Form
   SubTotals
   Text to Columns
   Group and Outline

Microsoft Excel 5.0 and 7.0

   Delete Cells
   Insert Cells
   AutoFormat
   Sort
   AutoFilter
   Form
   SubTotals
   Text to Columns
   Group and Outline
