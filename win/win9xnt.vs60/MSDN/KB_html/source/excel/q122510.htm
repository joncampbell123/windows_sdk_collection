Query Err Msg: Wrong Number of Parameters

 Last reviewed: September 12, 1996
Article ID: Q122510
The information in this article applies to:
  • Microsoft Query for Windows, version 1.0
  • Microsoft Access for Windows, version 2.0

SUMMARY

In Microsoft Query version 1.0, you may receive the following error message when you use a field that contains a question mark (?) from a table in a Microsoft Access 2.0 database: 

     Wrong Number of Parameters
If the field name from a Microsoft Access version 2.0 database table contains a question mark, you will receive this error message while you attempt to use this field in a query. If you attempt to create a table in a Microsoft Access 2.0 database where a field name in that table contains a question mark, you will also receive this error message.

MORE INFORMATION

To use this field in Microsoft Query, change the field name so that it does not contain a question mark. This error message does not occur with any other of the special characters that Microsoft Access 2.0 allows for its field naming conventions.
