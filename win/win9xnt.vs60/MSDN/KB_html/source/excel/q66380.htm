Excel: ImageWriter Can't Print Borders and Gridlines in Draft

 Last reviewed: November 29, 1994
Article ID: Q66380
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SUMMARY

The Apple ImageWriter cannot print graphics in draft mode. Microsoft Excel gridlines and borders are graphics and won't display in Print Preview or print if Draft quality is selected in the Print dialog box.

MORE INFORMATION

To print gridlines and/or borders to an ImageWriter, choose Faster or Best as the print quality in the Print dialog box.
