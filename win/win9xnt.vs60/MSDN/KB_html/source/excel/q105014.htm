Excel: CTRL+U Activates Windows Sound System Proofreader

 Last reviewed: September 12, 1996
Article ID: Q105014
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0, 4.0a, 5.0

SUMMARY

In Microsoft Excel, you cannot use the CTRL+U key sequence to underline text when the Windows Sound System is installed: when this system is installed, CTRL+U activates the proofreading tool. CTRL+N is the designated key sequence to enable the Sound System proofreader.

WORKAROUND

To work around this situation, add a button for the underline tool to the standard toolbar and use this button instead of the CTRL+U key sequence.
