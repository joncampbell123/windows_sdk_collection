Excel: Clicking Scroll Bar in Print Preview Causes System Fail

 Last reviewed: October 7, 1997
Article ID: Q78781
3.00 MACINTOSH kbprint

SUMMARY

Clicking one of the worksheet scroll bars in print preview mode may cause Microsoft Excel for the Macintosh version 3.0 to unexpectedly quit or may cause the system to fail. This problem can only occur if Excel version 3.0 is running on a Macintosh with two monitors and print preview is displayed on one monitor while, the worksheet appears on the other monitor.

For a solution to this problem, please call Microsoft Product Support Services at (425) 635-7080.
