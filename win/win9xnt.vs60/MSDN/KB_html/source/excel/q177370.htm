SBFM97: "Setup Could Not Find Microsoft Excel" Error Message

 Last reviewed: January 9, 1998
Article ID: Q177370
The information in this article applies to:
  • Microsoft Small Business Financial Manager 97 for Microsoft Excel 97

SYMPTOMS

When you run the Setup program for the Microsoft Small Business Financial Manager 97 for Microsoft Excel 97 (SBFM), the following error message appears: 

   Setup could not find Microsoft Excel 97 installed
   on your system. Please install Microsoft Excel 97
   before installing this application.
When you click OK, the Setup program does not continue.

CAUSE

This problem occurs if one of the following conditions is true:

  • You are running Microsoft Excel from a network server. 

        -or-
  • You are running Microsoft Excel from a compact disc (CD).

WORKAROUND

To work around this behavior, install Microsoft Excel on your computer, and then install the Microsoft Small Business Financial Manager 97. To do this, follow these steps:

  1. Remove the Run From Network Server or Run From CD installation of Microsoft Excel. To do this, follow these steps: 

        a. On the Start menu, point to Settings, and then click Control Panel.


        b. Double-click Add/Remove Programs. In the list of installed programs,
       click Microsoft Office 97, and then click Add/Remove. If you
       installed Microsoft Office from a CD, you are prompted to insert the
       first CD.


          If a dialog box notifies you that programs are running, quit the
      programs, and then click OK.


        c. In the Microsoft Office 97 Setup dialog box, click Remove All.

  2. Perform a new local installation of Microsoft Excel. To do this, run the Setup program again and choose Typical or Custom as the installation type.

  3. Run the Setup program for Microsoft Small Business Financial Manager.

STATUS

This behavior is by design.

MORE INFORMATION

You cannot install the Microsoft Small Business Financial Manager 97 for Microsoft Excel 97 if you are running Microsoft Excel from the network or from a CD. Microsoft Excel must be installed locally on your computer before you can install Microsoft Small Business Financial Manager 97.
