XL5: Macro Error with AutoSave and Embedded Chart

 Last reviewed: September 12, 1996
Article ID: Q121828
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel, if the AutoSave add-in is loaded, and if it attempts to save a workbook while you are editing an embedded chart in the workbook, you receive the following error message: 

   Macro Error In Cell [AUTOSAVE.XLA]AutoSave!$B$130

WORKAROUND

When you receive this error message, you can choose the Halt button to close the error message dialog box. The workbook is not saved, but the AutoSave feature will attempt to save it again at the specified time interval.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel for Windows version 5.0c.
