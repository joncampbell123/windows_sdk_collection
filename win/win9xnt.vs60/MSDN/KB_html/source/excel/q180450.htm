XL98: Template Wizard Allows You to Save in Only One Format

 Last reviewed: February 4, 1998
Article ID: Q180450
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you run the Template Wizard in Microsoft Excel 98 Macintosh Edition, you can create only a database in a Microsoft Excel Workbook. Other file formats are not supported.

STATUS

This behavior is by design of Microsoft Excel 98 Macintosh Edition.

MORE INFORMATION

To run the Template Wizard in Microsoft Excel, click Template Wizard on the Data menu. In step 2 of the wizard, a list box allows you to select the kind of database you want the wizard to create.

In Microsoft Excel 98 Macintosh Edition, the list box contains only the following file format: 

   Microsoft Excel Workbook
In Microsoft Excel 97 for Windows, the list box contains additional formats: 

   Microsoft Excel Workbook
   Access Database
   dBASE III Database
   dBASE IV Database
   FoxPro 2.0 Database
   FoxPro 2.5 Database
It is not possible for the Template Wizard to use these additional formats in Microsoft Excel 98 Macintosh Edition.
