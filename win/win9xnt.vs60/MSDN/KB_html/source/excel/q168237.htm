XL97: Line Chart with Smoothed Line Selected May Cause Error

 Last reviewed: August 15, 1997
Article ID: Q168237
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, if you move the pointer around a chart sheet or a worksheet that contains an embedded chart, you may receive an error message similar to either of the following: 

   EXCEL caused an invalid page fault in module MSO97.DLL at
   014f:306c349b.

   -or-

   EXCEL caused an invalid page fault in module MSO97.DLL at
   0137:306c34b9.

CAUSE

The error message appears if all of the following conditions are true:

  • You use one or more data series that are the line chart type.

    -and-

  • The Smoothed Line feature is checked for one or more of the lines.

    -and-

  • The data series that are lines have source data that are unequal in range size; they have a different number of data points.

RESOLUTION

To correct this problem, install Microsoft Excel 97 Service Release 1 (SR-1).

To work around this problem, avoid checking the Smoothed Line check box for all lines in the chart, or verify that all the data series of the lines in the chart have the same number of data points.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel 97 for Windows. This problem was corrected in Microsoft Excel 97 SR-1.

For additional information about SR-1, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q172475
   TITLE     : OFF97: How to Obtain and Install MS Office 97 SR-1

MORE INFORMATION

In Microsoft Excel, the Smoothed Line feature smoothes angularity or jagged edges in the lines of a line chart or an xy (scatter) chart. This smoothing feature does not affect data.

Clearing the Smoothed Line Check Box

To clear this check box, do the following:

  1. Click the line in the chart to select the appropriate data series.

  2. On the Format menu, click Selected Data Series.

    NOTE: If Selected Data Series does not appear on the Format menu, you did not select the data series.

  3. Click the Patterns tab.

  4. Click to clear the Smoothed Line check box.

REFERENCES

For more information about line chart formatting, click the Index tab in Microsoft Excel Help, type the following text 

      line charts, formatting
and then double-click the selected text to go to the "Smooth the angles of line charts" topic.
