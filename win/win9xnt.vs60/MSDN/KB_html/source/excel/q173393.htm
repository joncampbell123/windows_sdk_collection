XL97 Err Msg: Solver: An Unexpected Internal Error Occurred

 Last reviewed: January 7, 1998
Article ID: Q173393
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel, if you use the Solver add-in to generate the Answer report, the Limits report, or the Sensitivity report, you may receive the following error message: 

     Solver: An unexpected internal error occurred, or available
     memory was exhausted.

CAUSE

This error message occurs when the following conditions are true:

  • You select one or more reports in the Solver Results dialog box. 

        -and-
  • A sheet in the workbook has the same name as the report that Solver creates, for example "Answer Report 1." 

        -or-
  • The worksheet contains merged cells.

WORKAROUND

To work around this problem, rename or delete any sheets with names that Solver uses to create reports. The following table lists these names. 

   Sheet Names of Solver Reports
   -----------------------------

   Answer Report <number>
   Sensitivity Report <number>
   Limits Report <number>
The variable <number> increments each time you generate Solver reports.

If your worksheet contains merged cells, remove the merged cells by performing the following steps:

  1. Select the merged cell.

  2. On the Edit menu, point to Clear, and click Formats.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about Solver reports, click the Index tab in Microsoft Excel Help, type the following text 

   solver, reports
and then double-click the selected text to go to the "Create a report by using Solver" topic.
