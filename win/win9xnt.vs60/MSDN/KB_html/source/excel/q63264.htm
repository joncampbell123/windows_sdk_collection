Excel: Mathematical Operation and Formatting in Paste Special

 Last reviewed: November 29, 1994
Article ID: Q63264
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.5, 2.2, 3.0, 4.0

SUMMARY

In Microsoft Excel, when a mathematical operation is performed from the Paste Special dialog box (on the Edit menu), the attribute chosen under Paste will transfer to the paste area also.

For example, if the Paste attribute is All and the Operation is Multiply, all number and style formatting of the copy area will be transferred to the paste area.

To retain both the style and number formatting of the paste area, choose either the Values or Formulas option along with the mathematical function.

For more information on the Edit Paste Special command, see pages 317-319 in the "Microsoft Excel User's Guide 1," version 4.0 manual. If you are using version 3.00, see pages 150-151 in the "Microsoft Excel User's Guide" version 3.0 manual. If you are using Excel 2.20, see pages 218-221 in the "Microsoft Excel Reference" version 2.2 manual.
