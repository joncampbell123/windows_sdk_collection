XL7: Map Control Panel Not Hidden During Print Preview

 Last reviewed: June 3, 1997
Article ID: Q135507
7.00 97 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97

SYMPTOMS

The Data Map Control dialog box, which is displayed when a map object is inserted into worksheet, is not hidden when you view a worksheet in print preview. (Note, however, that this dialog box does not appear on the printout when you print your worksheet.)

WORKAROUND

To work around this problem, close the Data Map Control dialog box before you view the worksheet in print preview. To close the dialog box, click the Show/Hide Data Map Control button on the Data Map toolbar.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you insert a map object into a Microsoft Excel worksheet, all customized formatting components of the map can be controlled through the Data Map Control dialog box.
