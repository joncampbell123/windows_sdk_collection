MXL5: Excel for the Power Macintosh Installed on Workstation

 Last reviewed: February 2, 1998
Article ID: Q124810
The information in this article applies to:
  • Microsoft Excel for the Power Macintosh, version 5.0

SYMPTOMS

When you run the Microsoft Excel Setup program from a network server, Microsoft Excel for the Power Macintosh may be installed, even if your computer is not a Power Macintosh.

Note that when this occurs, you receive a system error -192 error message when you start Microsoft Excel.

CAUSE

When you run the Microsoft Excel 5.0 Administrative Setup program, you must select an installation type based on the workstations that Microsoft Excel will be installed on. If you select the Power Macintosh option, Microsoft Excel for the Power Macintosh is installed on any workstation that this server installation is run from, even if the workstation is a non-Power Macintosh computer. You do not receive any type of warning when you run the Setup program from a non-Power Macintosh computer, even though this Microsoft Excel installation type will not run on your computer.

WORKAROUNDS

To avoid this behavior, when you run the Microsoft Excel 5.0 Administrative Setup program, select the Macintosh and Power Macintosh installation type if Microsoft Excel will be installed from the network server to both Power Macintosh workstations and Macintosh workstations.

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q120256
   TITLE     : System Error -192 Starting Microsoft Excel
