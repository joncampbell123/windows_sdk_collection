XL97: Visio 4 Drawing Object May Not Be Displayed Correctly

 Last reviewed: January 9, 1998
Article ID: Q168949
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you insert a Visio 4 Drawing object into a worksheet in Microsoft Excel 97, the object is only displayed correctly as long as you are editing it. When you deactivate the object, it is displayed as a blank rectangle.

CAUSE

This problem occurs because of a minor incompatibility between Microsoft Excel 97 and Visio 4.

NOTE: This problem does not occur in Microsoft Word 97, Microsoft PowerPoint 97, or earlier versions of Microsoft Excel, Microsoft Word, or Microsoft PowerPoint.

WORKAROUND

To force the Visio 4 Drawing object to be displayed correctly, follow these steps:

  1. Activate the worksheet that contains the Visio 4 Drawing object.

  2. On the File menu, click Print Preview.

  3. Quit Print Preview by clicking Close.

The Visio 4 Drawing object now appears correctly. Note that you must perform these steps each time you make any changes to the Visio 4 Drawing object.

MORE INFORMATION

If you are experiencing this problem when using Visio 4 Drawing objects in Microsoft Excel 97, you may want to check Visio Corporation's web site and see if a newer version of Visio is available. Use your Web browser to go to the following address: 

   http://www.visio.com/
