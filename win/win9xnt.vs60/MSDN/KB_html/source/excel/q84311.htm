Excel: No Keyboard Selection for Print Quality in File Print

 Last reviewed: November 4, 1994
Article ID: Q84311
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SYMPTOMS

The Print Quality list box in the File Print dialog box in Microsoft Excel 4.0 cannot be selected from the keyboard.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. This problem was corrected in version 5.0 of Microsoft Excel for Windows.

MORE INFORMATION

After choosing Print from the File menu, you should be able to select the Print Quality list box from the keyboard by using ALT-Q or the TAB key. However, these keys select the Copies edit box instead. The Print Quality list box can be changed with a mouse or by using the PRINT() macro function to print your documents.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. This problem was corrected in version 5.0 of Microsoft Excel for Windows.
