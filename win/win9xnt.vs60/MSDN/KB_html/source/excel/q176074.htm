XL: Workbook Is Recalculated Many Times When You Save as HTML

 Last reviewed: February 2, 1998
Article ID: Q176074
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Excel 97 for Windows

SYMPTOMS

In the versions of Microsoft Excel listed at the beginning of this article, the following problems may occur when you use the Internet Assistant Wizard (Html.xla) to save files in the Hypertext Markup Language (HTML) format:

  • The first time you click Save As HTML on the File menu, all open workbooks are recalculated two times.
  • When you click Finish in step 4 of the Internet Assistant Wizard, all open workbooks are recalculated 14 times.

This may be a problem when any open workbooks take a long time to be recalculated, when worksheets in open workbooks use circular references or iteration, or when formulas in open workbooks use volatile functions, such as the RAND and NOW functions.

RESOLUTION

To resolve this problem, set the calculation mode to manual calculation before you click Save As HTML on the File menu. To do this, follow these steps:

  1. On the Tools menu, click Options (or Preferences).

  2. Click the Calculation tab.

  3. Click Manual and click OK.

After you have saved the HTML file, you can select Automatic or Automatic Except Tables under Calculation.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
