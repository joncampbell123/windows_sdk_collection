Printing Text in Color with Adobe Type Manager (ATM)

 Last reviewed: November 4, 1994
Article ID: Q72581
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0

SUMMARY

Adobe Type Manager (ATM) will print text in color only if you are printing to a color PostScript printer, a Hewlett-Packard (HP) PaintJet (using driver version 1.01), the Epson LQ860, or Fujitsu DL series. This is a problem with ATM and calls should be directed to Adobe.
