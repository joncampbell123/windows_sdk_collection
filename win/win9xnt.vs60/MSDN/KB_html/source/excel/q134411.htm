MSQUERY: Syntax Error When Field Name Starts with a Number

 Last reviewed: March 27, 1997
Article ID: Q134411
1.00 2.00 WINDOWS kbtool xlquery

The information in this article applies to:

  • Microsoft Query for Windows, version 1.0
  • Microsoft Query for Windows 95, version 2.0

SYMPTOMS

In Microsoft Query, if a field name starts with a number, math operations involving the field may result in the following error message: 

   Syntax Error

WORKAROUND

To work around this problem, change the field name so that it does not start with a number.
