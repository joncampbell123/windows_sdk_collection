XL: Error When Copying Chart Objects to Other Programs

 Last reviewed: February 2, 1998
Article ID: Q165569
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you copy a chart from one the versions of Microsoft Excel listed above, you may receive an error message in the program to which you are pasting. The errors may include the following.

Microsoft PowerPoint

   The server application, source file, or item cannot be found. You
   should reinstall the server application.

Microsoft Word

   There is insufficient memory. Save the document now.

CAUSE

You will experience this problem if you are copying a chart from a workbook that has multiple windows.

RESOLUTION

To resolve this problem, use one of the following methods.

Method 1

Close any extras windows that are open for the workbook from which you are copying the chart. You can identify these windows by looking at the Microsoft Excel title bar for the following: <workbook name>:1, <workbook name>:2, and so on. To close the window <workbook name>:2, do the following:

  1. Activate the window named <workbook name>:2.

  2. Click Close on the window's Control menu.

Repeat this procedure until there is only one window left open.

NOTE: You will need to save your workbook after completing this procedure.

Method 2

You can paste the chart as a picture to avoid the error messages mentioned in the "Symptoms" section. To do this, do the following:

  1. Activate the chart in Microsoft Excel.

  2. Click Copy on the Edit menu.

  3. Open the program into which you want to paste the chart.

  4. Click Paste Special on the Edit menu.

  5. Select Picture, and then click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about closing workbook windows, click the Index tab in Microsoft Excel Help, type the following text 

   windows, closing
and then double-click the selected text to go to the "Close workbook windows" topic.
