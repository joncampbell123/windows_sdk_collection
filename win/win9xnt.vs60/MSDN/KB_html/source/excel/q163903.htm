Revision Number and Total Edit Time Boxes Are Empty

 Last reviewed: February 12, 1998
Article ID: Q163903
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In the programs listed at the beginning of this article, in the Statistics tab of the Properties dialog box (click Properties on the File menu) the Revision Number and Total Edit Time boxes are blank.

STATUS

This behavior is by design of Microsoft Excel. The Properties dialog box is a shared by Microsoft Office programs. The Revision Number and Total Edit Time boxes are used by some Microsoft Office programs. However, Microsoft Excel does not use either of these boxes.

REFERENCES

For more information about file properties, click the Index tab in Microsoft Excel Help, type the following text 

   properties, setting
and then double-click the selected text to go to the "Set file properties for the active workbook" topic.
