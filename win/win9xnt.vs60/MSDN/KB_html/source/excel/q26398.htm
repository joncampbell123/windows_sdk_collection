Excel: Positioning a Chart Legend

 Last reviewed: October 31, 1994
Article ID: Q26398

SUMMARY

In Microsoft Excel version 3.00 it is possible to drag the legend to any position in the chart.

In versions of Microsoft Excel previous to 3.00 the legend in a chart can only be positioned in one of four fixed positions. To set these positions (bottom, corner, top, and vertical), do the following:

  1. From the Chart menu, choose Add Legend.

  2. Select the legend (which, by default, is located on the right side of the chart and is of vertical type).

  3. From the Format menu, choose Legend.

  4. Select the type of legend desired (Bottom, Corner, Top, or Vertical) and click OK.
