XL: Data from Query Not Returned to the Worksheet

 Last reviewed: February 2, 1998
Article ID: Q115430
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Query for Windows 95, version 2.0
  • Microsoft Query for Windows, version 1.0
  • Microsoft Query for the Macintosh, version 1.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In the versions of Microsoft Excel mentioned above, if you click Get External Data on the Data menu and then create a query in Microsoft Query, you may be unable to return the data to Microsoft Excel. After you click "Return Data to Microsoft Excel" on the File menu, you see the normal prompts, and an area on the sheet is selected (this area is equivalent to the number of records and fields that the query returns). However, the actual data does not appear in the Microsoft Excel sheet.

CAUSE

In Microsoft Excel, you can choose to ignore requests from other dynamic data exchange (DDE) programs by selecting the Ignore Other Applications check box (this check box is located on the General tab in the Options dialog box or the General tab in the Preferences dialog box). If this option is selected, you experience the symptoms described in this article.

RESOLUTION

To re-enable the connection between Microsoft Excel and Microsoft Query, clear the Ignore Other Applications check box. To do this, follow these steps:

  1. Click Options on the Tools menu or click Preferences on the Tools menu (in Microsoft Excel 98 Macintosh Edition.)

  2. Click the General tab.

  3. On the General tab, clear the Ignore Other Applications check box.

  4. Click OK.
