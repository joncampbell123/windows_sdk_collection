Excel: Can't Dock Toolbar on Left or Right Side of Window

 Last reviewed: November 30, 1994
Article ID: Q84017
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 4.0, 5.0

Toolbars that contain drop-down lists will not function correctly if they are docked on the left or right side of the screen. Therefore, these toolbars cannot be docked along the left or right sides of the application Window.

Tools that contains drop-down lists include the following: 

   Style
   Size
   Font
Toolbars that contains these tools include the following: 

   Standard
   Microsoft Excel 3.0
   Formatting
Note that these toolbars can be docked on the top or the bottom of the screen and can also be used as floating tool palettes anywhere on the screen.

REFERENCES

"User's Guide," version 5.0, page 633-634 "User's Guide 1," version 4.0, pages 161-162
