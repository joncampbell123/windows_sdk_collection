XL97: Error Printing Microsoft Excel Section in a Binder

 Last reviewed: January 15, 1998
Article ID: Q160342
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Office 97 for Windows

SYMPTOMS

When you print a Microsoft Excel section in a Binder you may receive the following error: 

   This action cannot be completed because the "Microsoft Excel" program is
   busy. Click the appropriate button on the task bar to activate the busy
   program and correct the problem.
If you then click Cancel, you will receive the following error message: 

   An error has occurred printing section.
   <Section name>
   Do you want to continue printing the binder document?
where <Section name> is the name of the section you are printing.

If you click No, you will receive the following error message: 

   An error occurred while printing the binder. There may not be enough
   memory available on your computer. Close other programs and then try
   printing again.
Your only option in this last error message is to click OK.

CAUSE

These errors may occur if the Microsoft Excel section is in edit mode when you print the Binder section. Edit mode is activated when you use in-cell editing (which allows you to edit the cell contents directly on the worksheet) or when the cursor is located in the formula bar. You cannot print a Microsoft Excel document while it is in edit mode.

RESOLUTION

To resolve this problem, press ENTER or ESC to deactivate edit mode, and then print the section.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

You may also receive an error when you activate edit mode and print a worksheet from within Microsoft Excel.

REFERENCES

For more information about edit mode, click the Index tab in Microsoft Excel Help, type the following text 

   Edit mode
and then double-click the selected text to go to the "How do I: Edit Cell Contents" topic.

For more information about printing a Microsoft Excel section, click the Index tab in Binder Help, type the following text 

   Printing
and then double-click the selected text to go to the "Sections" topic.
