Excel: Text File Opens Partially or Data Missing

 Last reviewed: November 30, 1994
Article ID: Q75072
Summary:

When Microsoft Excel version 1.0, 1.03, 1.04, 1.06, 1.5, 2.2, or 3.0 for the Macintosh opens text files that contain mismatched quotation marks, the text will not be parsed correctly.

When Excel encounters a quotation mark, everything between it and the closing quotation mark is entered in the same cell. If there are more than 255 characters between the quotation marks, the information will be truncated.

Note: This information is also valid for Microsoft Excel for Windows versions 2.1, and 3.0, and Microsoft Excel for OS/2 versions 2.2, 2.21, and 3.0.
