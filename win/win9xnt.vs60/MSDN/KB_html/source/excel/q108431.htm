XL: Default Borders in Pie Chart Do Not Print to Plotter

 Last reviewed: September 13, 1996
Article ID: Q108431
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a

In the versions of Microsoft Excel listed above, default borders in pie charts do not print when printed to a plotter. However, borders can be printed to plotters on other types of charts. The borders for pie charts must be set to the lightest weight when printing to a plotter. None of the other border weights will print to a plotter.

Unlike Microsoft Excel 4.0, which uses the hairline weight as the default borders, Microsoft Excel versions 5.0 and 7.0 use the next heaviest weight as the default weight.
