XL7: Insert/Delete Animation Inactive with Sheet Background

 Last reviewed: March 27, 1997
Article ID: Q137525
7.00 WINDOWS kbother

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, the animated insert/delete feature may become inactive.

NOTE: This problem does not affect your ability to insert or delete information; only the animation of these tasks is affected. Note also that the Animate Insertion And Deletion check box is a new feature in Microsoft Excel version 7.0.

CAUSE

This problem occurs if you add a background to your worksheet.

WORKAROUND

To avoid this behavior, use the following steps to remove the background from your worksheet:

  1. On the Format menu, point to Sheet, and then click Background.

  2. In the Sheet Background dialog box, click None.

Once you have removed the sheet background, you are able to view the animated insertion and deletion of rows and columns.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel for Windows 95, version 7.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

If you click Options on the Tools menu, the Animate Insertion And Deletion check box enables animated insertion and deletion. When this check box is selected, inserted worksheet rows and columns slowly appear, and deleted worksheet rows and columns slowly disappear.
