XL5: Utility That Allows Microsoft Excel to Use .VBX Files

 Last reviewed: September 12, 1996
Article ID: Q119095
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

InterGroup Technologies offers a product, Outreach, that allows some products, including Microsoft Excel, to use custom controls (.VBX files).

InterGroup Technologies is offering OutReach at an introductory price of $199.00. For more information about Outreach, contact InterGroup at (206) 643-8089 or Tom McKenna on CompuServe at 74602,3121.

MORE INFORMATION

Visual Basic, Applications Edition, which ships with Microsoft Excel 5.0, currently does not support .VBX files. Visual Basic 3.0 does support .VBX files for additional add-on custom files.
