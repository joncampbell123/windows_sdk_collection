Excel: Printing Chart Fit to Page Doesn't Enlarge Text

 Last reviewed: November 29, 1994
Article ID: Q43823
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0

SUMMARY

When printing a chart from Microsoft Excel version 2.2, 3.0 or 4.0 with the Fit to Page option selected, Excel will not scale the text in the chart. The size of any text on the chart is absolutely controlled by Format Font. To change the size of text on a chart, select the text item and choose Font from the Format menu.

Versions of Excel earlier than 2.20 do not size text absolutely, but instead scale text with other chart objects when printing.
