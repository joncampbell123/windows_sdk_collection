XL5: Advanced Filter Extracts All Records

 Last reviewed: September 12, 1996
Article ID: Q117753
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel version 5.0, the Advanced Filter may return all records to the "Copy To" range, even though the criteria to return all records is not specified.

CAUSE

This problem occurs when a Print Area is set on the worksheet and you do either of the following:

  • You insert a row or column in the Print Area region.

    -or-

  • You insert a row above the list range.

When this problem occurs, the Advanced Filter will not function properly on any worksheet until you follow one of the workarounds below.

WORKAROUNDS

To work around this problem, do either of the following:

  • Exit and restart Microsoft Excel.

    -or-

  • Save the worksheet in SYLK file format, close the worksheet, and reopen it.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel for Windows version 5.0c.
