Excel: Auto Save Add-in Macro Does Not Save Preferences

 Last reviewed: July 16, 1997
Article ID: Q92696
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 4.0

Summary:

In version 4.0 of Microsoft Excel for the Macintosh, the Auto Save add-in macro does not save changes you make to preferences when you exit Microsoft Excel.

To save the changes you make to preferences, follow the steps below to modify the Auto Save add-in macro:

  1. From the File menu, choose Open.

  2. In the Open Document dialog box, open the Macro Library folder, and select the Auto Save macro.

  3. Hold down the SHIFT key and choose Open from the File menu.

  4. If the "Revert to Saved" alert appears:

    a. Hold down the SHIFT key and choose the OK button.

    b. From the Window menu, choose Unhide.

    c. In the Unhide dialog box, select Auto Save and choose the OK 

          button.

  5. From the Formula menu, choose Define Name.

  6. In the Define Name dialog box, under Name, type "Auto_Close" (without the quotation marks).

  7. Under Refers To, type "=$B$34" (without the quotation marks).

  8. Choose the OK button.

  9. From the File menu, choose Save.

More Information:

For more information about the Auto Save macro, search on Auto Save in Microsoft Excel Help.
