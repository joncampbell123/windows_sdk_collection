Excel: No Working Models Available for Excel 4.0

 Last reviewed: November 29, 1994
Article ID: Q89742
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SUMMARY

A working model of Microsoft Excel 4.0 for Windows is not available.

MORE INFORMATION

A Working Model is a "crippled" version of the product it demonstrates. Working models of Microsoft Excel for Windows, versions 2.1 and 3.0 were produced, but no working model is available for version 4.0 of Microsoft Excel for Windows.
