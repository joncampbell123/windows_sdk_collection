Excel Err Msg: Data Form Is Not Correct

 Last reviewed: November 2, 1994
Article ID: Q46507

SUMMARY

The following two types of dialog box items are allowed in custom data forms in Microsoft Excel versions 2.2 and 3.0:

  • Static text (item 5)
  • Edit boxes (items 6, 7, 8, 9, and 10)

All other items cause two error messages to appear when Form is chosen from the Data menu. The first dialog box displays the cell reference in the dialog box in which the error occurred, and the second displays the message "Data Form is not Correct."

A custom data form must also contain seven columns of information, as with any custom dialog box. If less than seven columns are included in the custom data form dialog box definition table, the message "Data Form is not Correct" will be displayed.

For more information on data forms, see pages 365-368 of the "Microsoft Excel User's Guide" version 3.0 manual or pages 157-166 of the "Microsoft Excel Reference" version 2.2 manual.

For more information on custom dialog boxes, see pages 620-628 of the "Microsoft Excel User's Guide" version 3.0 manual or pages 164-178 of the "Microsoft Excel Functions and Macros" version 2.2 manual.
