Buttons Print with Rounded Corners in Excel

 Last reviewed: November 30, 1994
Article ID: Q74399
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2 version 3.0
  • Microsoft Excel for the Macintosh version 3.0

SYMPTOMS

If you create a button object with the button tool located on the Microsoft Excel 3.0 tool bar, the button object may print with rounded corners even though it appears rectangular on the screen. In this case, it will also appear with rounded corners in print preview.

WORKAROUND

Use the text box tool from the tool bar in place of the button tool. The text box will print with rectangular corners.

MORE INFORMATION

This design feature was to differentiate buttons from other objects when printed.

For more information about printing macro buttons, query on the following words: Excel and Macro and Buttons and Print and Object.

Macintosh Only

Buttons will appear rounded on a monochrome display on the Macintosh if you are using system 6, and if your color settings don't support the three-dimensional appearance.
