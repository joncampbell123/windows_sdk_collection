XL98: Nothing Happens When You Click Outside Spelling Dialog Bo

 Last reviewed: February 2, 1998
Article ID: Q176864
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

If you click outside the Spelling dialog box in Microsoft Excel 98 Macintosh Edition, the computer beeps twice but nothing else happens.

CAUSE

This problem occurs if the following conditions are true:

  • You click Spelling on the Tools menu to start the spelling checker, and the spelling checker finds a word that is not in the dictionary.

    -and-

  • You attempt to click outside the Spelling dialog box.

WORKAROUND

To work around this problem, click Cancel to stop the spelling checker, perform the actions you want, and then return to the spelling checker. To return to the spelling checker, click Spelling on the Tools menu.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about the spelling checker, click the Index tab in Microsoft Excel Help, type the following text 

   spell checking, spell
and then double-click the selected text to go to the "Check spelling or automatically correct spelling" topic.
