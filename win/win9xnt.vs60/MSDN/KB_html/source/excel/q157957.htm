Query97: Redraw Problems in Choose Columns Dialog Box

 Last reviewed: January 15, 1998
Article ID: Q157957
The information in this article applies to:
  • Microsoft Query 97 for Windows

SYMPTOMS

When you view the "Choose Columns" dialog box in the Query Wizard in Microsoft Query 97, items in the Available Tables and Columns list box are redrawn incorrectly.

For example, if the list contains the following two items: 

   Description
   Picture
you may see an item that looks like this: 

   Picturetion
Also, if you click the Remove (<) or Remove All (<<) button to remove an item from the Columns in Your Query list box, an item may appear twice in the Available Tables and Columns list box.

CAUSE

The first problem may occur when you select an item in the Available Tables and Columns list box and then add it to the Columns in Your Query list box, either by double-clicking the item or by clicking the Add (>) button.

The second problem may occur whenever you click the Remove or Remove All button to remove an item from the Columns in Your Query list box.

WORKAROUND

To eliminate these redraw problems, perform the following steps:

  1. Click the minus sign (-) next to the table name that contains improperly drawn or duplicate items. For example, if you see the following in the Available Tables and Columns list box: 

          (-) Categories

                 CategoryID
             CategoryName
             Picturetion
             Picture

   Click the minus sign next to "Categories". Then, click the plus
   sign to re-expand the list.


  2. If duplicate items still appear in the list, click each duplicate
   item once. This should cause the item to redraw with the correct
   name.

    

STATUS

 


Microsoft has confirmed this to be a problem in the Microsoft products
listed at the beginning of this article. We are researching this problem
and will post new information here in the Microsoft Knowledge Base as it
becomes available.


The problems described above only occur when certain versions of the file
Comctl32.dll are installed on your computer. When a version of Comctl32.dll
that corrects this problem becomes available, this article will be updated
to include information on how to obtain the new file.



MORE INFORMATION

 


The "Choose Columns" dialog box in the Query Wizard helps you decide what
tables and fields to include in a query.


The file Comctl32.dll, located in your \Windows\System folder, is used when
displaying the "Choose Columns" dialog box. Certain versions of the
Comctl32.dll file will cause the problems described above. The Comctl32.dll
file included with Microsoft Internet Explorer version 3.0 is known to
cause the problem.
