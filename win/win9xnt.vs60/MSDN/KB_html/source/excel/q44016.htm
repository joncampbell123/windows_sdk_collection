Excel 2.20: Scroll2 Version 1.06 Crashes While Scrolling

 Last reviewed: October 31, 1994
Article ID: Q44016

SUMMARY

Microsoft Excel version 2.20 may crash when you attempt to scroll while the shareware Control Panel device (CDEV) Scroll2 version 1.06 is installed in the System.

This problem has been corrected with Scroll2 version 1.10.
