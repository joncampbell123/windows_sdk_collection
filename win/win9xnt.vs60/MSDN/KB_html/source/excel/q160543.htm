XL97: Error Evaluating Natural Language Formula

 Last reviewed: January 15, 1998
Article ID: Q160543
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you attempt to evaluate part of a natural language formula in the formula bar (by pressing F2, and then pressing F9), the follow message may appear: 

   The formula you typed contains an error.

CAUSE

This behavior occurs because Microsoft cannot resolve the label as an address, and Microsoft Excel fails to evaluate the formula. Note that a label is not a cell address; if you try to evaluate a label in the formula bar, the error message appears.

STATUS

This behavior is by design of Microsoft Excel 97 for Windows.
