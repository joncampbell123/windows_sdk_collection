XL: List of XLM Macro Functions Available

 Last reviewed: February 2, 1998
Article ID: Q103751

The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 4.0, 5.0

SUMMARY

Following are the 386 macro functions available in Microsoft Excel. This list also includes the functions that are available with the CROSSTAB.XLL, SLIDES.XLA, and Solver add-in macros.
