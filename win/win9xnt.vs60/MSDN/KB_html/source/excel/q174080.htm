XL97: Wrong Printer Is Used After Changing Printer Properties

 Last reviewed: January 7, 1998
Article ID: Q174080
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Windows 95

SYMPTOMS

In Microsoft Excel 97, when you print a worksheet, a chart, or an entire workbook, Microsoft Excel 97 may print to the default printer even though you selected a different printer in the Print dialog box.

CAUSE

This problem may occur when the following conditions are true:

  • In the Print dialog box, you select a printer other than the default printer in the Name box. 

        -and-
  • The printer you selected is the following: 

          HP LaserJet 5Si Mopier


        -and-
  • You click Properties, change any property of the selected printer, and then click OK. 

        -and-
  • You click OK to print the selected document.

WORKAROUND

To work around this problem, select a different printer in the Name box, and then select the printer you actually want to use. When you click OK, Microsoft Excel 97 prints to the correct printer.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
