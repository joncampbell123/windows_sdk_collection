Options on Edit Menu Are Dimmed in Q+E

 Last reviewed: October 24, 1995
Article ID: Q71749

The information in this article applies to:

  • Microsoft Excel for Windows, versions 3.0, 4.0

SYMPTOMS

In Q+E for Microsoft Excel, several options on the Edit menu may be unavailable (dimmed). This will occur if a Microsoft Excel database or a text file database has been loaded into Q+E. Examples of unavailable options include Allow Editing, Add Record, Delete Records, Paste Append, and Update All.

CAUSE

These options are unavailalbe due to a limitation of the Excel and TextFile drivers.

MORE INFORMATION

Editing Excel Databases

Page 143 of the "Q+E for Microsoft Excel User's Guide," version 4.0, and page of 157 "Q+E for Microsoft Excel User's Guide," version 3.0 states: 

   You cannot use Q+E to edit or define Microsoft Excel Worksheet
   files, although you can use the File Save As command to save query
   results to a new worksheet file.
Editing a Microsoft Excel database can only be done in Microsoft Excel.

Editing Text File Databases

Page 137 of the "Q+E for Microsoft Excel User's Guide," version 4.0, and page 135 of the "Q+E for Microsoft Excel User's Guide" states: 

   You cannot use Q+E to edit or define text files, although you can
   use the File Save As command to save query results to a new text
   file.
Editing Text databases can be done in any text editor or word processor.
