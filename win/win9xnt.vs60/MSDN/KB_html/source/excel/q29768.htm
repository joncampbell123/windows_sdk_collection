Excel: Adding Attached and Unattached Text to a Chart

 Last reviewed: November 2, 1994
Article ID: Q29768

SUMMARY

Text can be attached to a Microsoft Excel chart by using attached or unattached text.

To add attached text to a chart, choose Attach Text from the Chart menu and specify one of the following areas of the chart to attach text to:

  1. The chart title
2. The category axis 3. The value axis 4. Any series or data point

Unattached text is used when you want to add text to your chart, but do not want the text associated with a particular part of the chart.

To add unattached text to a chart, do the following:

  1. Make sure no object in the chart is selected.

  2. Type the text you want.

  3. Press the ENTER key or click the check mark in the formula bar to enter the text.

  4. Position the text on the chart by first clicking the text, then dragging the text box.
