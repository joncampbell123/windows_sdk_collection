Excel: AVI Files Are Not Played in Excel Client Window

 Last reviewed: September 12, 1996
Article ID: Q94354
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0, 4.0a

SUMMARY

An audio video interleaved (AVI) file that is embedded in a Microsoft Excel worksheet will not play in a Microsoft Excel window. When you select the AVI file, it will be played in a Media Player window instead.

MORE INFORMATION

When you select an embedded AVI file in a client application that supports object linking and embedding (OLE), the AVI file should play inside the client application. However, selecting the embedded AVI object in Microsoft Excel brings up a separate window that is similar to the Media Player window. After the AVI file is played it will disappear.

Microsoft has confirmed this problem. It is corrected in Microsoft Video for Windows, version 1.1.

REFERENCES

"Microsoft Excel User's Guide 2," version 4.0, pages 142-146
