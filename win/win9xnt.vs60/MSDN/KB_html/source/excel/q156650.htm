XL97: Cannot Drag Drawing Objects Between Worksheets

 Last reviewed: February 18, 1998
Article ID: Q156650
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, you cannot use the ALT key to drag a drawing object or an AutoShape from one worksheet to another in the same workbook.

WORKAROUND

To move a drawing object or an AutoShape from one worksheet to another in the same workbook, use one of the following methods:

Method 1

  1. Select the drawing object or AutoShape.

  2. On the Edit menu, click Cut.

  3. Activate the sheet on which you want to place the object.

  4. On the Edit menu, click Paste.

Method 2

  1. On the Window menu, click New Window.

    A new window with a copy of the workbook appears. For example, if the active workbook in step 1 is called Book1.xls, Microsoft Excel creates two windows called Book1.xls:1 and Book1.xls:2.

  2. On the Window menu, click Arrange.

  3. In the Arrange Windows dialog box, click Tiled, and then click OK.

  4. In the second window, activate the sheet on which you want to place the drawing object.

  5. Activate the first window.

  6. In the first window, press and hold the ALT key, click the drawing object, drag it to the second window, and release the ALT key.

MORE INFORMATION

In Microsoft Excel 97, you can drag a range of cells from one worksheet to another in the same workbook by pressing the ALT key while you drag the cells. For example, if you want to drag cells A1:A3 from Sheet1 to Sheet2, use the following steps:

  1. Select cells A1:A3 on Sheet1.

  2. While pressing the ALT key, drag the range of cells on top of the tab for Sheet2.

  3. After Sheet2 is active (its tab changes color to indicate that it is active), release the ALT key. Then, drag the cells to the desired location on Sheet2.

  4. Release the mouse button to paste the cells.

This functionality does not apply when trying to drag drawing objects between worksheets.

REFERENCES

For more information about moving cells between sheets in a workbook, click the Index tab in Microsoft Excel Help, type the following text 

   drag-and-drop
and then double-click the selected text to go to the "Move or copy whole cells" topic.
