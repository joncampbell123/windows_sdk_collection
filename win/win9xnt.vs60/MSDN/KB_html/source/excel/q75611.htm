Error Message: This Value Cannot Be Changed

 Last reviewed: November 2, 1994
Article ID: Q75611

SUMMARY

The error message "This value cannot be changed" will occur when you try to edit a computed column in a query. This is correct behavior.

Because a computed column is the result of a calculation, changing an element can affect the entire column; therefore, it is not allowed.

MORE INFORMATION

Steps to Reproduce Error Message

  1. Start Q+E.

  2. Load the example file EMP.DBF from the QE subdirectory.

  3. Highlight the SALARY column.

  4. From the Layout menu, choose Define Column.

  5. Type "TEST" (without the quotation marks) in the Heading box.

  6. Under Expression, type "salary+1000" (without the quotation marks).

  7. Choose Add and then press ENTER to insert the new column.

  8. From the Edit menu, choose Allow Editing.

  9. Select one field in the column TEST and enter a number.
