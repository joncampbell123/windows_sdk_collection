XL97: Errors in Web Form Wizard Saving Form to FrontPage

 Last reviewed: January 15, 1998
Article ID: Q161898
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft FrontPage 97 for Windows
  • Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

When you use the Microsoft Excel 97 Web Form Wizard to save a form for a FrontPage Web, you may receive one of the following error messages: 

   You need to open a Web by using the FrontPage Explorer.

   -or-

   FrontPage cannot be found on your machine. Please make sure that it is
   installed and working properly.

CAUSE

The first error message may appear if you do not open a Web in the FrontPage Explorer before you save the form. If a Web is open, and you select "Add the result to your FrontPage Web" in the "Web Form Wizard - Step 4 of 6" dialog box, the wizard saves all the files it creates in the open FrontPage Web.

The second error message may appear if FrontPage is not installed on the computer. In order to use the "Add the result to your FrontPage Web" option in the "Web Form Wizard - Step 4 of 6" dialog box, FrontPage must be installed (and a FrontPage Web must be open in the FrontPage Explorer).

RESOLUTION

To resolve this behavior, use the following appropriate method.

Method 1

To prevent the first error message from appearing, open a Web in the FrontPage Explorer before you click "Add the result to your FrontPage Web" in the "Web Form Wizard - Step 4 of 6" dialog box. After you open a Web, the Web Form Wizard saves the files it creates to the open Web.

Method 2

To prevent the second error message from appearing, install Microsoft FrontPage on the computer on which you are running the Microsoft Excel Web Form Wizard.

STATUS

This behavior is by design of Microsoft Excel 97.

REFERENCES

For more information about the Web Form Wizard, click the Index tab in Microsoft Excel Help, type the following text 

   web form wizard
and then double-click the selected text to go to the "About setting up your data gathering system" topic.
