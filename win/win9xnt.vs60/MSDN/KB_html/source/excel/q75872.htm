Excel Err Msg: Cannot Access Document

 Last reviewed: November 7, 1994
Article ID: Q75872

The information in this article applies to:

  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0

SUMMARY

If you attempt to load files into Microsoft Excel on a workstation configured with MS-DOS 5.0 and IBM PC-LAN 1.3, you may receive the error "Cannot Access Document," even though other applications can access files from the server.

MORE INFORMATION

PC-LAN version 1.3 does not properly handle requests from an application to lock files. When Microsoft Excel attempts to lock data files before reading them, the network returns an error indicating that the document is being used by another person on the network, and the "Cannot Access Document" message is generated.

Obtain CSD 3030 of PC-LAN version 1.34 to correct the problem.

(Note: CSD is the acronym for Corrective Service Diskette. To obtain the CSD, contact the network supplier.)
