Excel: GET.WINDOW(4) Documentation Error

 Last reviewed: November 30, 1994
Article ID: Q88977
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

Page 207 of the "Microsoft Excel Function Reference Guide" contains a documentation error in the description of TYPE_NUM 4 for the GET.WINDOW function. TYPE_NUM 4 should read "Y position, measured in points from the bottom edge of the formula bar to the top edge of the window," instead of "Y position, measured in points from the top edge of the workspace (in Microsoft Excel for Windows) or screen (in Microsoft Excel for the Macintosh) to the top edge of the window."

ref "Microsoft Excel Function Reference," version 4.0, page 207
