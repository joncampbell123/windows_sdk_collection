Excel: SHOW.BAR Doesn't Restore Built-in Menu Bar

 Last reviewed: November 2, 1994
Article ID: Q63101

SUMMARY

If one of Microsoft Excel's built-in menu bars has been modified so that it no longer contains a particular default menu for that menu bar, such as the Formula menu, the bar cannot be restored to its original state by using the SHOW.BAR(#) command.

To return to the original state of Excel's built-in menu bar, you must quit and restart Microsoft Excel.
