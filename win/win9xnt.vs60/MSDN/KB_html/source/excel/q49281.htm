Excel: Undo Command in Custom Menu Has No Effect

 Last reviewed: October 31, 1994
Article ID: Q49281

SUMMARY

In Microsoft Excel versions 1.50, 2.20 and 3.00, when you create a customized menu that uses the UNDO function, choosing the Undo command from that menu appears to have no effect. This behavior occurs because the last action performed was the call to the macro that contains the UNDO function. This is the action that is undone, and therefore it appears as if nothing happens. There is no way to make the UNDO function work in a macro the way it does when it is chosen from the built-in menu.
