XL5: "Could Not Open the File..." Running WK4 Converter Setup

 Last reviewed: September 12, 1996
Article ID: Q123259
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SYMPTOMS

When you run the WK4 File Converter for Microsoft Excel Setup program, you may receive the following error message 

   Could not open the file named: '<drive>\DISK1\_MSTEST.EX_'
where <drive> is the letter of the drive from which you are running the Setup program.

After you receive this error message, the following error messages appears: 

   Setup is not yet complete. If you exit now, this software will not be
   correctly installed for use. Exit Setup?

   The WK4 File Converter has not been properly installed.

CAUSE

This problem occurs when you run the WK4 File Converter Setup program from your computer's hard drive, or a network drive, if the name of the directory that contains the Setup program is not Disk1. The Setup program must be run from either the Setup disk, or a Disk1 directory on your computer or a network drive. The following are examples of valid directories to run the Setup program from: 

   a:
   c:\disk1

   c:\files\disk1

   x:\wk4conv\disk1

WORKAROUND

To avoid receiving this error message, do either of the following:

  • Run the Setup program from the Lotus 1-2-3 WK4 Setup floppy disk.

    -or-

  • Copy the files to a Disk1 directory on your hard drive, or on a network server drive, and run the Setup program from this directory.
