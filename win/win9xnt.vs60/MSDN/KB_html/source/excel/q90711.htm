Orientation Setting Lost Transferring From Windows to Mac

 Last reviewed: November 30, 1994
Article ID: Q90711
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0 and 4.0
  • Microsoft Excel for the Macintosh, versions 3.0 and 4.0

SUMMARY

When you transfer a worksheet from Microsoft Excel for Windows to Microsoft Excel for the Macintosh, the Landscape orientation setting is lost. After you transfer your worksheet to Microsoft Excel for the Macintosh, the Landscape option is still selected in the Page Setup dialog box, however, your worksheet previews and prints in portrait mode.

MORE INFORMATION

Transferring files between Microsoft Excel for Windows and Microsoft Excel for the Macintosh requires a file transfer utility program such as Apple FileExchange.

To print your worksheet in Landscape mode after transferring it from Microsoft Excel for Windows to Microsoft Excel for the Macintosh, select the Landscape option in the Page Setup dialog box.

To select Landscape orientation in Microsoft Excel:

  1. From the File menu, choose Page Setup.

  2. In the page setup dialog box, select the Landscape option.

  3. In the page setup dialog box, choose the OK button.
