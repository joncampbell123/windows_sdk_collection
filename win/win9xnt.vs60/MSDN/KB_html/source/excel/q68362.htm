Screen Blanks While Using Split Window Command in Excel

 Last reviewed: November 2, 1994
Article ID: Q68362

SUMMARY

While using the IdleWild screen saver (included in the Windows Entertainment Pack) with Excel, it is possible to corrupt the video memory used by Excel.

Microsoft is researching this problem and will post new information here as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Start IdleWild.

  2. Start Excel.

  3. Open the Window Control Bar in Excel.

  4. Choose SPLIT.

  5. Move to the far upper-right side of the screen. (This activates Idlewild.)

  6. Move the mouse to bring Excel back. (The spreadsheet is now blank.)

  7. Click the mouse anywhere to set the SPLIT.

If you force IdleWild on again (by moving to the upper-right side of the screen, you will receive the message: 

   Not enough memory to display completely
The video memory is now completely corrupted for this instance of Excel.

Data is NOT corrupted, it just isn't displayed correctly. You should save the files that are currently open and then close Excel immediately.
