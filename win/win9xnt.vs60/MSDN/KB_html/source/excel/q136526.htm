XL: Using 16-Bit and 32-Bit ODBC Drivers

 Last reviewed: February 2, 1998
Article ID: Q136526
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Query for Windows, version 1.0
  • Microsoft Query for Windows 95, version 2.0
  • Microsoft Query 97 for Windows

SUMMARY

With Microsoft Excel and Microsoft Query, the Open Database Connectivity (ODBC) drivers that you may use depend on the operating system you are using and whether the program is the 16-bit or 32-bit version.

Microsoft Windows NT

If you are running Windows NT, 16-bit programs can use either 16-bit or 32- bit ODBC drivers. 32-bit programs can use only 32-bit ODBC drivers.

Microsoft Windows 95

If you are running Windows 95, 16-bit programs can use only 16-bit ODBC drivers and 32-bit programs can only use 32-bit ODBC drivers.
