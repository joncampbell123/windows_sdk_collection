Not Showing Certain Options as Available in Page Setup

 Last reviewed: November 1, 1994
Article ID: Q68407

SUMMARY

Excel is marketed as being able to save certain printing attributes with the document (that is, Orientation, Paper, Reduce Or Enlarge, or Fit To Page). This ability is printer-driver dependent. Some printer drivers, such as the current Epson printer drivers, do not support the per document printer attributes. Also, there are some printer drivers that only allow part of these attributes to be saved with the document. For example, the Hewlett-Packard (HP) LaserJet driver will allow Orientation, but not Paper (size), Reduce Or Enlarge, or Fit To Page.

MORE INFORMATION

Windows 3.00 supports extended attribute settings, which allows certain printing attributes to be saved with the document. If the driver does not support extended attribute settings, the Excel program will not be able to save the attributes with the document. To save these attributes with the document, use a driver that supports this.
