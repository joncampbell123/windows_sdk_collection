Excel: Selecting a Tool to Draw Multiple Objects

 Last reviewed: November 30, 1994
Article ID: Q89608
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0 and 4.0
  • Microsoft Excel for the Macintosh, versions 3.0 and 4.0

Summary:

To draw multiple graphic objects of the same type in version 4.0 of Microsoft Excel, double-click the tool for the object on the toolbar. The button will stay depressed, allowing the drawing of multiple objects.

In version 3.0 of Microsoft Excel, to select a tool to draw several objects, hold the COMMAND (CTRL in Excel for Windows) key down while clicking on the tool.

To cancel the drawing tool, press the same tool again, press ESC, or click in another part of the spreadsheet.
