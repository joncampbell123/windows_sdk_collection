XL5: Components Installed with Each Installation Option

 Last reviewed: September 12, 1996
Article ID: Q119132
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

The following information describes the components that are installed with each option in a Complete/Custom setup of Microsoft Excel, version 5.0. The table lists which options are installed with the typical, minimum, and complete installations, and it also lists the file size for each of the components.

MORE INFORMATION

When you perform a Custom/Complete setup of Microsoft Excel version 5.0, a dialog box appears with the following options: 

   Microsoft Excel
   Online Help and Lessons
   Data Access
   Graphics Filters
   Add-ins Tools
In this list, you can clear the options that you do not want to install. Each option above, except the first, has several components that you can clear to avoid installing.

If certain components are missing in Microsoft Excel, you may not have performed a Complete installation of Microsoft Excel. This list below shows the components for each category.

NOTE: The components are displayed when you choose the Change Option button in the Complete/Custom setup screen.

Microsoft Excel 

   No suboptions
Online Help and Lessons 
   Help for Microsoft Excel
   Help for Visual Basic for Microsoft Excel
   Help for Microsoft Excel 4.0 Macros
   Product Support Information
   Examples and Demos
   Sample Files
Data Access 
   Microsoft Query
   Help for Microsoft Query
   Microsoft Query Cue Cards
   Access Driver
   dBASE Driver
   FoxPro Driver
   Paradox Driver
   SQL Server Driver
Graphics Filters 
   Windows Metafile filter
   HP Graphic Language filter
   Computer Graphics Metafile filter
   Encapsulated PostScript filter
   Tagged Image Format filter
   PC Paintbrush filter
Add-ins 
   Autosave
   Analysis ToolPak

   Slide Show
   Solver
   View Manager
Tools
