Computed Criteria Example on Page 64 Results in Blank Extract

 Last reviewed: November 2, 1994
Article ID: Q74239

SUMMARY

The first example on page 64 of the "Q+E for Microsoft Windows User's Guide" is incorrect.

If you set up your criteria exactly as the manual demonstrates, you will not receive any records. Adding an "=" sign will allow the example to work.

MORE INFORMATION

The correct criteria range should read: 

   Date_Hired
   =HIRE_DATE<NOW()-30

REFERENCES

"Q+E for Microsoft Excel User's Guide," page 64
