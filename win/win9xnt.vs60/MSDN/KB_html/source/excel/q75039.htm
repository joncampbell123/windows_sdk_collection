Excel: Color Window Not Displayed in 16 Gray Mode in System 7

 Last reviewed: November 1, 1994
Article ID: Q75039

SUMMARY

Microsoft Excel versions 2.2 and 3.0 will not display the 3-D look of the title bar when running under System 7.0 in 16 gray scale mode. Launching Excel may also disable the color windows for other applications. To work around this problem, use the 256 colors or grays options in the Monitors Control Panel device (CDEV).

MORE INFORMATION

Microsoft is researching this problem and will post new information here as it becomes available.
