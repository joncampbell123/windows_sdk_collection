Excel: Template Name Not Showing Up in File New

 Last reviewed: November 2, 1994
Article ID: Q67478

SUMMARY

For a template to appear in the File New dialog box for Microsoft Excel 3.00, three conditions must be met:

  1. The document must have been saved as a Template, as follows:

    a. From the File menu, choose Save As. b. Select the Options button. c. Select Template from the File Format list box. d. Select OK. e. Select Save.

  2. The document must be in a folder named "Excel Startup Folder," and this folder must be in the System Folder. Or, the document must be in the alternate startup folder, which was set using the "Alternate Startup Folder" macro.

  3. Excel must be restarted before the name of the template appears. Excel obtains the names in File New when it starts.

For more information on Template files, see pages 551-553 in the "Microsoft Excel User's Guide" for version 3.0.
