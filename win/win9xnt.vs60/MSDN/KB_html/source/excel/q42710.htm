Excel: Words in Category Names Wrap in Column Chart

 Last reviewed: November 2, 1994
Article ID: Q42710
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2, versions 2.2, 3.0

SYMPTOM

In Microsoft Excel, the category names of a column chart may wrap to two lines such that the wrap occurs in the middle of a word.

WORKAROUND

To work around this wrapping problem, do any of the following:

  • If you are using Microsoft Excel 5.0, decrease the font size can so that you can fit more text on a single line.

    -or-

  • Increase the size of the chart. (The size to which you will need to increase you chart will depend on the number of category names in the chart.)

    -or-

  • Do the following to reduce the number of category labels that appear on your chart:

    1. Select the category axis.

    2. In Microsoft Excel 5.0, choose Selected Axis from the Format menu, 

          and select the Scale tab. In versions earlier than 5.0, choose Scale
      from the Format menu.

    3. In the Number Of Categories Between Tick Marks box, increase the 

          number (you may have to experiment with this to find number of tick
      marks to use in order to display your chart the way you want).

    This procedure reduces the number of category labels used so the remaining ones can all fit on one line. The default value is 1. Changing this value to a 2 in the Number Categories Between Tick Labels displays every-other label.
