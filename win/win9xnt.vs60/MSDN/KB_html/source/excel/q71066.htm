Excel: Reduction Buttons Dimmed with Personal LaserWriter LS

 Last reviewed: October 7, 1997
Article ID: Q71066
2.20 3.00 4.00 MACINTOSH kbprint

The information in this article applies to:

  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, and 4.0

SUMMARY

If you print from Microsoft Excel for the Macintosh version 2.2 or 3.0 to the Personal LaserWriter LS using Apple's Personal LaserWriter LS driver 1.0, the following options are dimmed (unavailable) in the Page Setup dialog box on the File menu:

  • 75% size
  • 50% size
  • Precision Bitmap Alignment (4% reduction)

MORE INFORMATION

Microsoft Excel prints at 300 dpi. The LaserWriter LS does not support print reduction at 300 dpi, therefore the print reduction options are dimmed in the Page Setup dialog box. For a solution to this problem, please contact Microsoft Product Support Services at (425) 635-7080.
