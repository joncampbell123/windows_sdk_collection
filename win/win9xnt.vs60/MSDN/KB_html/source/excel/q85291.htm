Excel: Chart Legend Cannot be Sized Manually

 Last reviewed: November 4, 1994
Article ID: Q85291
The information in this article applies to:
  • Excel for Windows, version 3.0
  • Excel for the Macintosh, version 3.0

Summary:

The "Microsoft Excel User's Guide," version 3.0, incorrectly states the following: 

   You can display the legend in an automatic position, or you can
   position and size the legend manually.
More Information:

The documentation should read as follows:

You can display the legend in an automatic position, or you can position the legend manually.

Reference(s):

"Microsoft Excel User's Guide" for the Macintosh, version 3.0, page 451

"Microsoft Excel User's Guide" for Windows, version 3.0, page 453
