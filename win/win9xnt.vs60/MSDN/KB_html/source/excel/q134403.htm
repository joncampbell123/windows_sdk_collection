XL7: Map Object Cut Off on Printout

 Last reviewed: April 15, 1997
Article ID: Q134403
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, when you print a worksheet that contains a map object, the map object may be cut off on the printout. Specifically, the bottom portion of the map may not be printed.

CAUSE

This behavior occurs if your printer does not have enough memory to print the entire map. In general, it requires more memory to print a map object than other objects in Microsoft Excel: for example, a chart object.

RESOLUTION

To work around this problem, follow these steps:

  1. Check the amount of memory on your printer, and make sure that the memory setting in the Properties dialog box for this printer is correct. To display or modify this setting, follow these steps:

    a. Double-click My Computer, and then double-click Printers.

    b. Right-click the printer you are using, and then click Properties.

    c. Click the Device Options tab.

    d. Note the value in Printer Memory. If this value is not correct, 

          enter the correct value, and then click OK.

  2. To reduce the resolution setting follow these steps:

    a. In Microsoft Excel, click Print on the File menu.

    b. Click Properties.

    c. Click the Graphics tab. In the Resolution list, click a lower 

          resolution setting if there is one available. For example, click
      300dpi instead of 600dpi.
