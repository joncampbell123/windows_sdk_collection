Excel: Page Preview Does Not Show or it Disappears

 Last reviewed: November 2, 1994
Article ID: Q59247

SUMMARY

If you choose Page Preview in the File Print dialog box, when there is no data in the worksheet, Excel 2.20 displays Page Preview momentarily and then returns to the worksheet. In Excel 1.50, Page Preview does not display even momentarily if the document is blank.

To view the header and/or footer of a blank Excel document, the worksheet must contain at least one space in a cell.
