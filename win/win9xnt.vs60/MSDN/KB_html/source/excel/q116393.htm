XL5: "Cannot Open Help File" Running Examples And Demos

 Last reviewed: September 12, 1996
Article ID: Q116393
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel, when you choose Examples And Demos from the Help menu, the Microsoft Help window appears, and you receive the following error message: 

   Cannot open Help file.

CAUSE

This error message appears when Microsoft Excel cannot locate the MAINXL.HLP file. The Microsoft Excel Help file, MAINXL.HLP, is required to run the Examples And Demos. However, if you install the Examples And Demos option during Setup, and you do not install the Help For Microsoft Excel option, you cannot run the Examples And Demos.

WORKAROUND

To run the Examples And Demos in Microsoft Excel, install the Microsoft Excel Help file using the following steps:

  1. Run the Microsoft Excel Setup program.

  2. From the Options list in the Microsoft Excel 5.0 Maintenance Mode dialog box, select the Online Help And Lessons check box.

  3. Choose the Change Option button. From the Options list, select the Help For Microsoft Excel check box. Choose OK.

  4. Continue with the Setup program.

REFERENCES

For more information about Examples And Demos, choose the Search button in Help and type: 

   examples
