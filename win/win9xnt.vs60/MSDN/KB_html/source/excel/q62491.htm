Excel: Screen Refreshes Repeatedly When Adding to Database

 Last reviewed: November 30, 1994
Article ID: Q62491
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0

SUMMARY

In Microsoft Excel, when a record is added to a database using the Form command on the Data menu, the screen refreshes once for every field that has formatting information (other than borders), such as a font other than the standard font. This behavior occurs when you use either the standard or a custom form.

For example, if there are 10 fields and all are in a font other than the standard font, the screen refreshes 10 times when a record is added.

For every field that is not in the standard font, Microsoft Excel pastes down the formatting information to the new record and refreshes the screen.
