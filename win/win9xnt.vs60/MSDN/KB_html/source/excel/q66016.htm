SUMMARY

The Microsoft Excel Small Business Consultant is a package offered by Microsoft Press. The following is from the mail advertisement sent out by Microsoft press:

"17 Timesaving Templates for Small Businesses!"

Now you no longer need to spend hours to create powerful financial- modeling tools. This valuable, timesaving toolkit is for any small- business owner or manager, accountant, or financial analyst using Microsoft Excel for the IBM PC, the PS/2, or a compatible. The 17 ready-to-use disk-based templates address the critical tasks of financial analysis, budgeting, and forecasting. Templates cover depreciation analysis, debt amortization, future-value analysis, cost-center summary, sales forecasting and analysis, and more. A fact-filled companion book includes instructions for loading the templates, customizing them, and interpreting the results.

System requirements: Microsoft Excel version 2.10 or later Package contains: 314-page book and one 5.25-inch disk By Stephen L. Nelson
