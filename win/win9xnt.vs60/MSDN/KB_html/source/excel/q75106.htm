Cannot Convert Lotus PrintGraph (PIC) Files into Excel

 Last reviewed: November 4, 1994
Article ID: Q75106
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0
  • Microsoft Excel for OS/2, versions 2.2, 2.21, 3.0

A Lotus PrintGraph (*.PIC) file cannot be converted into a Microsoft Excel chart. A PrintGraph file contains an image of a graph saved in a Lotus 1-2-3 spreadsheet.

NOTE: In Microsoft Excel version 5.0, you can import a Lotus 1-2-3 graphics (*.PIC) file as a picture by choosing the Picture command from the Insert menu.

REFERENCES

"Microsoft Excel User's Guide, Book 1." Version 4.0, pages 55-57.

"Microsoft Excel User's Guide." Version 3.0, pages 56-57.
