Excel: Objects in Dialog Editor Appear Distorted

 Last reviewed: September 12, 1996
Article ID: Q104839
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a

In Microsoft Excel, certain third-party fonts may distort objects in the Dialog Editor and cause them to appear compressed or "stretched." For example, if small system fonts such as the System (Set #3) font are added to Windows Control Panel, objects in the Dialog Editor may appear distorted: menus may appear shorter or longer than normal.

This problem may not affect applications other than Microsoft Excel.

Workaround

To work around this problem, delete the small system font from Control Panel and use a custom Setup to reinstall the Dialog Editor. If the problem persists, check for the presence of small system fonts or third-party high-resolution video drivers that may install a system font.
