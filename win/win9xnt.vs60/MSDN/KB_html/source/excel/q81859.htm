Excel: Unbound Closed Files Remain Listed in Workbooks

 Last reviewed: November 30, 1994
Article ID: Q81859
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

In Microsoft Excel version 4.0, an unbound document that is listed in a workbook will not be removed from the workbook document list, even if you close the workbook without saving changes.

MORE INFORMATION

Example

  1. From the File menu, choose New. Select Worksheet, and choose OK.

  2. From the File menu, choose New. Select Workbook and choose OK.

  3. Choose the ADD button.

  4. Select a worksheet from list of open documents and choose OK.

  5. Unbind the document by clicking on the icon to the right of the filename.

  6. Choose the Options button. Enter a new filename and choose OK.

  7. Activate the document by double-clicking the document name.

  8. From the File menu, choose Close, and choose No when prompted to save the changes.

The workbook should now be the visible document and the worksheet name is still listed in the workbook. When you attempt to activate this document or rename the document, you will receive the "Can't Find <filename>" alert message. Note that if you did not rename the document as shown in step 6, Microsoft Excel will open a new document using the name listed in the workbook.

To remove the document from the workbook, click the document name in the workbook list and choose the Remove button. Excel will display the "Can't Find <filename>" alert message. Choose OK and the document will be removed from the workbook document list.
