XL5: Exiting Windows Leaves Files in TEMP Directory

 Last reviewed: September 12, 1996
Article ID: Q121120
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

When you are using Microsoft Excel for Windows version 5.0, if you close Microsoft Windows while Microsoft Excel is still running, there will be several files with the .TMP filename extension left in the temporary directory.

The exact number of these files will vary depending on what files are loaded in the currently running session of Microsoft Excel.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Normally when you quit Microsoft Windows, all temporary files (that is, those that have a .TMP filename extension) are automatically deleted from the TEMP directory (this directory is specified in the "SET TEMP =" line of your AUTOEXEC.BAT file).
