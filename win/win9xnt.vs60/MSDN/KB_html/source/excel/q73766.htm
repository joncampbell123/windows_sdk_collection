Next and Previous Buttons Are Grayed in Custom Help

 Last reviewed: November 2, 1994
Article ID: Q73766

SUMMARY

Microsoft Excel 3.0's Custom Help Conversion Utility allows you to create customized Online Help. Custom Help is accessed by using Excel's macro language. Each custom Help feature can access only one custom Help topic (for example, =HELP("C:\EXCEL\HELP.TXT!101")). Because only one topic can be accessed at a time, the Next and Previous buttons are grayed. There is essentially no Next or Previous Help topic.

MORE INFORMATION

Page 651 of the "Microsoft Excel User's Guide" for version 3.0 states that "When using Custom Help, the Next and Previous buttons jump to the custom Help topic with the next highest or lowest topic number." However, these buttons are not available.

You must create full-featured custom Help to access the Previous and Next buttons. Chapters 16-18 of the "Software Development Kit -- Tools" document, which is included with the Software Development Kit (SDK) version 3.0, explains how to create Help files with HC.EXE.

REFERENCES

"Microsoft Excel User's Guide." Version 3.0, pages 650-651.
