XL: Macro Translation Assistant Not Installed

 Last reviewed: September 12, 1996
Article ID: Q118956
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0c
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you install the versions of Microsoft Excel listed at the beginning of this article, the Macro Translation Assistant file, TRANS.EXE, is not installed on your computer.

CAUSE

This is by design. The Macro Translation Assistant is not included in these versions of Microsoft Excel. When you run the Setup program, the Macro Translation Assistant is not an available option.

For more information about the Macro Translation Assistant, query on the following words in the Microsoft Knowledge Base: 

   macro and translation and assistant and 5.0
