XL5: List of File Formats Supported in Microsoft Excel 5.0

 Last reviewed: March 12, 1997
Article ID: Q110761
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

Microsoft Excel 5.0 supports opening and saving files in many different file formats. The following is a list of these file formats, including version numbers (where applicable).

Items marked above with an asterisk do not actually appear in the list of file formats in the Open dialog box, but they can still be opened.

Note that although the above file formats are supported, certain features may not be available when opening or saving files in those formats: for example, some file formats do not support embedded charts or formatted text.
