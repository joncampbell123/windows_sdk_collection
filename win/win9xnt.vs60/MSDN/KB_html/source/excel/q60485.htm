Excel 2.20: Changing the Markers on a Hi-Lo Chart

 Last reviewed: November 2, 1994
Article ID: Q60485

SUMMARY

To change the markers of a hi-lo or hi-lo-close chart, tab through the chart and series items using the ARROW keys on the keyboard. If your keyboard doesn't have ARROW keys, use the 2, 8, 4, and 6 keys (for bottom, top, left, and right, respectively) on the numeric keypad with NUM LOCK turned off. Once the markers on the series you want to change are highlighted, choose Patterns from the Format menu. Then select the desired line marker pattern and click OK.

To change the markers in earlier versions of Excel, double-click the marker (series) that needs to be changed and the Pattern screen will appear.

For more information, see pages 36-94 in the "Microsoft Excel Reference" manual for Excel version 2.2.
