No EXCEL Function Exists in Microsoft Excel

 Last reviewed: November 3, 1994
Article ID: Q68813

SUMMARY

Page 342 of the "Microsoft Excel User's Guide," version 3.0, incorrectly states that "The EXCEL function returns the task identification of an application it starts."

The sentence should read: "The EXEC function returns the task identification of an application it starts."

REFERENCES

"Microsoft Excel User's Guide," version 3.0, page 342
