Using Micrologic MoreFonts with Excel

 Last reviewed: November 3, 1994
Article ID: Q71265

SUMMARY

MoreFonts by MicroLogic Software is a soft-font package that can create and install soft fonts for the Windows environment. According to Micrologic technical support, MoreFonts versions earlier than 2.04 will not work with Microsoft Excel 2.1c, 2.1d, or 3.0. Fonts may not print and you may receive an Unrecoverable Application Error (UAE) while printing. To determine which version you have, run MF.EXE from the directory in which MoreFonts is installed.

MORE INFORMATION

If you are a registered user of MoreFonts, you are eligible for a free upgrade to version 2.04 by calling Micrologic at (415) 652-5464 or write MicroLogic Software, 6400 Hollis Street, Suite 9, Emeryville, CA, 94608.

Micrologic has upgraded its product to version 3.0. For pricing information and upgrade costs, please contact Micrologic at (415) 652-5464.
