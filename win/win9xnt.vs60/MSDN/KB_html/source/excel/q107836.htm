XL: PhoneticAccelerator Property Added to Recorded Macro

 Last reviewed: June 13, 1997
Article ID: Q107836
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

When you change the property of an object while you are recording a macro, the recorded macro may contain the PhoneticAccelerator property.

PhoneticAccelerator returns or sets the phonetic keyboard accelerator key character for the control (this property is available only in Far East Microsoft Excel). The phonetic accelerator is used when the system accelerator mode is switched to phonetic characters (as opposed to roman characters, which use the Accelerator property).

REFERENCES

For more information about the PhoneticAccelerator, in Visual Basic Help, choose the Search button in Help and type: 

    PhoneticAccelerator
