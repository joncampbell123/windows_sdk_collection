Excel: PASTE.SPECIAL Arguments for Skip_Blanks, Transpose

 Last reviewed: October 31, 1994
Article ID: Q45247

SUMMARY

The description of the PASTE.SPECIAL function on page 258 of the "Microsoft Excel Functions and Macros" guide for version 2.20 does not have a description for the arguments to skip_blanks or transpose. The following information should have been included:

  • Skip_blanks and transpose are logical values (TRUE or FALSE).
  • Skip_blanks does not paste values copied from empty cells if TRUE.
  • Transpose will transpose the values before pasting if TRUE.
