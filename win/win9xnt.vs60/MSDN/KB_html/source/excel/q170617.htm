XL97: "am/pm" Changed to "AM/PM" in Custom Number Format

 Last reviewed: January 9, 1998
Article ID: Q170617
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

If you open a workbook created in an earlier version of Microsoft Excel, and the workbook contains time data, the formatting for time data in cells or in charts may be changed "am/pm" to "AM/PM." For example, if you create the time "1:00 am" (without the quotation marks) in cell A1 and you open the workbook in Microsoft Excel 97, cell A1 displays "1:00 AM" (without the quotation marks).

CAUSE

Microsoft Excel 97 does not support custom number formats in which "AM/PM" is lowercase. For example, the following format is not supported: 

   h:mm am/pm

RESOLUTION

To display "am/pm" in lowercase, change the number formatting to a format similar to the following 

   h:mm a/p"m"
including the quotation marks.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about custom number formats, click the Index tab in Microsoft Excel Help, type the following text 

   number formats, in cells
and then double-click the selected text to go to the "Custom number, date, and time format codes" topic.
