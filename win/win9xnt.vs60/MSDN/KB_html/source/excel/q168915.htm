XL97: Row and Column Submenus Missing in Excel Viewer

 Last reviewed: June 19, 1997
Article ID: Q168915
The information in this article applies to:
  • Microsoft Excel 97 Viewer

SUMMARY

In the Microsoft Excel 97 Viewer, the Column and Row submenus do not exist on the View menu. This is by design of the Microsoft Excel 97 Viewer.

NOTE: These submenus do exist in the Microsoft Excel 95 Viewer.

MORE INFORMATION

In the Microsoft Excel 95 Viewer, the Column and Row submenus provide the functionality to change the column width and row heights of the selection in the active worksheet. There is no equivalent menu command to provide this functionality in the Microsoft Excel 97 Viewer.

For information on how to change column widths and row heights in the Microsoft Excel 97 Viewer, follow these steps:

  1. Start the Microsoft Excel 97 Viewer.

  2. Open a file to view.

  3. On the Help menu, click Contents.

  4. Double-click "Moving Around in a Workbook and Finding Data."

  5. Select Change Column Width or Change Row Height, and then click Display.
