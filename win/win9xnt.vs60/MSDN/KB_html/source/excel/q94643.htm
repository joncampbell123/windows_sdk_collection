Row Headings Displayed Incorrectly with Zoom Command

 Last reviewed: September 12, 1996
Article ID: Q94643
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SUMMARY

In version 4.0 of Microsoft Excel, row headings in your document may be displayed incorrectly when you choose the Zoom command from the Windows menu. For example, if you change the zoom setting to 50 percent, garbage characters will be attached to the row headings in your document.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. We are researching this problem, and will post more information here in the Microsoft Knowledge Base as it becomes available.
