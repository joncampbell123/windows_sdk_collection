Recently Opened Worksheet List Not Updated in Excel

 Last reviewed: November 3, 1994
Article ID: Q72619
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2 version 3.0

SUMMARY

In Microsoft Excel, the File menu displays a list of the four most recently opened files at the bottom of the menu. This list will not update if a file is opened from a macro sheet with the OPEN statement.

This also is true if you use the FILELIST.XLM macro that is included with the Microsoft Excel Solution Kit.

The file list does not update is if you exit Microsoft Excel abnormally through a reboot or Unrecoverable Application Error (UAE).
