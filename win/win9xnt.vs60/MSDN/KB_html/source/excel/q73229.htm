Linked Cell Picture Does Not Update Automatically

 Last reviewed: November 3, 1994
Article ID: Q73229
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2, version 3.0

SUMMARY

The camera tool on the Microsoft Excel Toolbar allows you to copy a picture of cells. This picture is linked and should update automatically whenever you update the source cells. However, if the linked picture is placed directly on top of the cells from which it was copied, it will not update when the source information is updated.

MORE INFORMATION

WORKAROUND

Choose an alternate location to paste the linked picture.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES
