XL7: "Repeat Pick From List" Command Unavailable on Edit Menu

 Last reviewed: March 27, 1997
Article ID: Q140544
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

When you use Pick From List on the shortcut menu (select a cell an click the right mouse button), the Repeat command on the Edit menu is not available. Note, however, that the Undo Entry command is available to allow you to undo an entry.

MORE INFORMATION

The Pick From List feature is new to Microsoft Excel version 7.0 for Windows. This feature is part of the AutoComplete functionality. It allows you to choose a text entry from a drop-down list that is acquired from previous entries in the current column.

REFERENCES

For more information about Pick From List, click Microsoft Excel Help Topics on the Help menu. Click the Index tab and type: 

   AutoComplete
Press the ENTER key and then click Entering And Analyzing Data Quickly. To see the information, click Display.
