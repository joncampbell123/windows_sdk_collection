XL5: Find File: Pressing Letter Doesn't Select First Item

 Last reviewed: September 12, 1996
Article ID: Q112989
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

In Microsoft Excel version 5.0, when you use the Copy command in the Find File dialog box, if you press a letter on the keyboard, the first item in the list starting with that letter in the Directories list is not highlighted.

Normally, when you are in a dialog box (such as the Open dialog box) that contains a list, you can press a letter on the keyboard and the first item in the list that starts with that letter will be selected. This functionality is not available when you use the Copy dialog box in Find File in Microsoft Excel version 5.0.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel version 5.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
