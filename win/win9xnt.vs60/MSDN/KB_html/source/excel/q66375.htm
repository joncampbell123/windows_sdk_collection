SUMMARY

Microsoft Excel Class-in-a-Box is a training kit designed to teach Microsoft Excel basics in 10 distinct modules.

Microsoft Excel Class-in-a-Box contains the following items:

  1. An Instructor's Guide
2. A Student Workbook 3. A 3.5-inch Practice Files disk

The minimum hardware requirements are a Macintosh Plus with a hard drive (or two 800K disk drives) and 1 MB of RAM.

The software requirement is Microsoft Excel 3.00 or 2.20 and the Microsoft Excel Class-in-a-Box Practice Folder disk.

MORE INFORMATION

Microsoft Excel Class-in-a-Box can be purchased by calling Microsoft End User Sales and Service at (800) 426-9400.
