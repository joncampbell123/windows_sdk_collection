Excel: Selecting SHIFT Menu Options Using Alternate Menu Key

 Last reviewed: November 2, 1994
Article ID: Q42069

SUMMARY

In Microsoft Excel version 2.20 or 3.00, when you are using alternate (keyboard interface) menus, to select extra menu items (that is, those items that appear only when you click the menu name if the SHIFT key is held down), hold down the SHIFT key when typing the appropriate underlined character.

For example, do the following to choose Copy Picture from the alternate Edit menu:

  1. Activate the formula bar by pressing the Alternate Menu key. (This key is set in the Options Workspace dialog box.)

  2. Hold down the SHIFT key and type "E" (for Edit), then type "C" (for Copy Picture). Note: You do not need to hold down the SHIFT key when you type the "C".
