Excel: Black Mountain Software's Commando

 Last reviewed: August 22, 1996
Article ID: Q78092

SUMMARY

Black Mountain Software has written a program called Commando, which is an Excel 3.0 utility for a floating tool palette. The icons on this tool palette represent frequently-used Excel functions and cannot be customized. The product is distributed by Baarnes Consulting Group and Black Mountain Software.

MORE INFORMATION

Below is an explanation by column of which Excel menu commands the icons in Commando's floating tool palette represent:

Column 1

   File Open
   File Save
   File Close
   File Print
   File Print Preview

Column 2

   Edit Clear All
   Edit Undo
   Edit Cut
   Edit Copy
   Edit Paste

Column 3

   Edit Paste Special
   Insert Row
   Insert Column
   Edit Fill Right
   Edit Fill Down

Column 4

   Formula Find
   Formula Note
   Formula Reference
   Formula Paste Function
   Formula Define Name

Column 5

   Format Number
   Format Alignment
   Format Font
   Format Border
   Format Patterns

Column 6

   Data Sort
   Data Series
   Format Row Height
   Format Column Width
   Format Style

Column 7

   Gallery Area
   Gallery Bar
   Gallery Column
   Gallery Line
   Gallery Pie

Column 8

   Chart Add (Delete) Legend
   Gallery 3-D Pie
   Gallery 3-D Column
   Format 3-D View
   Gallery XY (Scatter)
NOTE: You must create a chart before you can utilize the tools in the last two columns (7 and 8).
