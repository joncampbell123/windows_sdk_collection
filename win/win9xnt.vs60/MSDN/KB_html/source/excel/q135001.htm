XL: Rich Text Format Canceled by Accounting Formats

 Last reviewed: February 2, 1998
Article ID: Q135001
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Excel for Windows NT, version 5.0

SYMPTOMS

When you format a portion of a text string in a cell of a worksheet, and then apply accounting formats or a custom number format, rich text formatting may be lost.

STATUS

This behavior is by design of Microsoft Excel.
