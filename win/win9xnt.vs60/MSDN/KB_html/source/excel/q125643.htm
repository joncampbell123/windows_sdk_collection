XL5: Embedded Chart Links Not Updated When Copying Sheet

 Last reviewed: September 12, 1996
Article ID: Q125643
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel 5.0, if you copy a sheet containing an embedded chart to another workbook, and the chart has links to cells on that sheet, the new chart will still refer to the sheet in the original workbook. This behavior is incorrect: the new chart should refer to the cells on the new sheet.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel version 5.0c for Windows.

REFERENCES

For more information about copying worksheets, choose the Search button in Help and type the following:
