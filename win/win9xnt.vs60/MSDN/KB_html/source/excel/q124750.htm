MSQUERY: "Error While Starting Cue Cards" After MS Excel Setup

 Last reviewed: September 12, 1996
Article ID: Q124750
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0a
  • Microsoft Query for Windows, version 1.0

SYMPTOMS

If you run the Microsoft Excel version 5.0 Setup program in maintenance mode, and you use the Microsoft Excel version 5.0a Setup disks to reinstall the Microsoft Excel files, you may receive the following error message when you run Microsoft Query: 

   Error While Starting CueCards

CAUSE

This behavior occurs if you run the Microsoft Excel version 5.0 Setup program in maintenance mode, and you choose the Reinstall button, and you insert the Microsoft Excel version 5.0a Setup disks instead of the version 5.0 Setup disks.

The problem occurs because the version 5.0 Setup program copies files from the Microsoft Excel version 5.0a Setup disks, and the application is upgraded to the correct version. However, the Microsoft Query Cue Card file, CCARD200.EXE, is not installed to the Microsoft Windows directory if you perform a maintenance mode installation of Microsoft Excel version 5.0, using the version 5.0a Setup disks. Because this file is not installed, you receive the error message the first time you run Microsoft Query, and additionally when you choose Cue Cards from the Help menu in Microsoft Query.

WORKAROUND

To avoid receiving this error message when you run the Cue Cards in Microsoft Query, do not use the Microsoft Excel version 5.0a Setup disks with the Microsoft Excel version 5.0 Setup program.

Remove Microsoft Excel version 5.0 before installing Microsoft Excel version 5.0a. You can do this by using the following steps:

  1. Run the Microsoft Excel version 5.0 Setup program in maintenance mode.

  2. Choose the Remove All button.

  3. Run the Microsoft Excel version 5.0a Setup program from the version 5.0a Setup disks.

REFERENCES

For more information about the Cue Cards Command (Help Menu), choose the Search button in Help and type: 

   Cue Cards
