Starting Q+E Message in Status Bar Is Not Removed

 Last reviewed: November 2, 1994
Article ID: Q69984

SUMMARY

After selecting Activate Q+E from the Data menu in Excel 3.00, the message "Starting Q+E" is displayed in the status bar at the bottom of Excel. Even after closing Q+E, this message is still displayed.

MORE INFORMATION

This is a problem with the QESTART.XLA add-in macro. Upon starting Q+E, the message function is called to display the "Starting Q+E" message but the message function is never called again to erase the message. To get rid of the message, close Excel and start it again, choose another option on the Data menu that is supported by the add-in, or execute the macro command "=MESSAGE(FALSE)".

The following steps will clear the status bar:

  1. Choose SQL Query from the Data menu.

  2. Select Close.

The following macro will clear the status bar: 

   A1: =MESSAGE(FALSE)
   A2: =RETURN()
