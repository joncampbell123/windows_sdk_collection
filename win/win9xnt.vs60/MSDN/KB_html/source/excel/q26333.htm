Excel: Menu Options Available on a Protected Document

 Last reviewed: November 2, 1994
Article ID: Q26333

SUMMARY

Many of the menu commands will be unavailable (dimmed) if one of the following is true:

  • You are in Edit mode (the formula bar is active). Either cancel the current entry by clicking the X in the formula bar or by pressing the Escape key (if one exists on your keyboard), or enter the contents by pressing the ENTER or RETURN keys.

    -or-

  • A protected document is active. To enable the commands, choose Unprotect Document from the Options menu.

The following table lists the menus and commands that are or are not available while a protected document is active. On the File, Window, and Help menus, ALL commands are available while a document is protected. On the Format menu, NONE of the commands are available.

Several commands in Microsoft Excel are enabled when a certain condition is true. For example, the Undo command on the Edit menu is only available when you've just completed an action that can be undone. If you see a command listed under Available that is dimmed on your display, ensure that you have executed the actions that enable that command. Commands that are listed as Not Available but appear selectable on the menu will result in an error if you choose them (for example, the Cut and Paste commands).

The following table assumes that your active document is a worksheet and that you have protected it with the default options selected. In Microsoft Excel 3.0 and 4.0, the cell contents and objects are selected. In Microsoft Excel 2.x, the cell contents are selected. If you vary these options, different menu commands will be available. For example, if you turn off the cell contents option when you protect your document, all of the commands on the Format menu will be available.
