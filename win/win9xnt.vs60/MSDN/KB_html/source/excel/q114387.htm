XL5: List of Files Installed with Each Setup Method

 Last reviewed: September 12, 1996
Article ID: Q114387
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

The following is a list of the files and file sizes of the files that are installed with each Setup method (Local/Floppy, Administrative, Workstation) available for Microsoft Excel 5.0.

MORE INFORMATION

The information in this article is copied from the FILELIST.TXT file located on the Microsoft Excel version 5.0, Setup Disk 1. FILELIST.TXT is an uncompressed text file.

The files are grouped by the directory in which each is installed, as indicated by capital letters preceded with a backslash (for example, EXCEL\LIBRARY\SOLVER). Also included is the list of files that are installed to the Windows directory when Microsoft Excel is installed.

The total size of files, in bytes, listed as shipped with Microsoft Excel 5.0, dated 12/14/93, is: 

   Complete = 23,201,585
   Typical  = 16,810,633
   Minimum  = 7,840,998
The total size of files, in bytes, listed as shipped with Microsoft Excel 5.0a, dated 3/2/94, is: 

   Complete = 23,467,719
   Typical  = 17,122,179
   Minimum  = 7,893,533
