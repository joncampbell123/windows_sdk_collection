Excel: Cut, Paste Link Acts the Same as Copy, Paste Link

 Last reviewed: November 30, 1994
Article ID: Q78635
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2, versions 2.2, 2.21, and 3.0

SYMPTOMS

If you choose Paste Link from the Edit menu in Excel after using the Cut command, Microsoft Excel will not cut the source information. Similar behavior also occurs when you choose Copy and then choose Paste Link; Excel will not copy the source information.

MORE INFORMATION

Page 311 of the "Microsoft Excel User's Guide" describes how to link a range of cells by choosing Copy from the Edit menu, followed by choosing Paste Link from the Edit menu. Although it is not documented, choosing Cut from the Edit menu to place the cell reference information into the Clipboard will work the same way. Internally, Excel converts the cut operation to a copy operation because the origin (on the supportive sheet) needs to stay in place.

REFERENCES

"Microsoft Excel User's Guide," version 3.0, pages 311-312
