XL97: Redo Toolbar Button Does Not Repeat Formatting

 Last reviewed: February 12, 1998
Article ID: Q165240
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, you cannot use the Redo command on the Standard toolbar to repeat the last formatting change that you made in a cell on a worksheet.

CAUSE

The Redo command on the toolbar becomes available only after you use the Undo command.

This behavior is different from the behavior of the Repeat command on the Standard toolbar in earlier versions of Microsoft Excel.

WORKAROUND

To repeat formatting that you applied to a cell, do either of the following:

  • On the Edit menu, click Repeat Format Cells. 

        -or-
  • Press CTRL+Y.

REFERENCES

For more information about repeating actions, click the Index tab in Microsoft Excel Help, type the following text 

   repeating actions
and then double-click the selected text to go to the "Repeat the last action" topic.
