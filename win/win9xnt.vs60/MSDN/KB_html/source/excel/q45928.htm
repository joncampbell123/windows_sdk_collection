"Cannot Open <Filename>.DBF" When Opening Query File in Q+E

 Last reviewed: October 31, 1994
Article ID: Q45928
Problem:

When I try to open a query file (.QEF) into Q+E, I get the following error message: 

   Cannot open <filename>.DBF
Response:

The error occurs because Q+E cannot find the database (.DBF) file that is associated with the query file.

A query file is a series of instructions that have been performed on a .DBF file. The data itself is not stored in the query file, and therefore, both the .QEF and .DBF must be accessible to Q+E.
