SBFM: Small Business Financial Manager 1.0 Requires Office 7.0

 Last reviewed: February 12, 1998
Article ID: Q163682
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office Small Business Pack for Windows 95, version 1.0

SUMMARY

The Microsoft Small Business Financial Manager (SBFM), version 1.0, requires Microsoft Excel 7.0. Version 1.0 of the SBFM is not compatible with Microsoft Excel 97 for Windows.

MORE INFORMATION

Changes in Visual Basic for Applications procedures prevent version 1.0 of the SBFM from working with Microsoft Excel 97.

For additional information, please see the following article in the Microsoft Knowledge Base:
