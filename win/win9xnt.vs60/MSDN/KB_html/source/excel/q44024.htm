Excel: Order of Searching Menus for Keyboard Commands

 Last reviewed: November 2, 1994
Article ID: Q44024

SUMMARY

Microsoft Excel version 2.20 and 3.00 searches menus from right to left, top to bottom, for keyboard shortcuts. If a user-defined menu command has a keyboard shortcut associated with it that duplicates an existing built-in keystroke, the user-defined command will be executed only if it appears sequentially before the built-in command because the menus are searched in this order.

In Excel version 1.50, the above is true except for the fact that menus are searched from left to right, rather than from right to left.
