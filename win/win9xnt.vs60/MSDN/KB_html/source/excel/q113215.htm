Excel: Function Wizard Pastes Function Incorrectly in Formula

 Last reviewed: September 12, 1996
Article ID: Q113215
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel, when you use the Function Wizard, it is possible to insert a function in an inappropriate position in a formula. For example, if you are working with a =SUM() formula, and you position the insertion point between the "U" and the "M," and you then choose Function from the Insert menu, the Function Wizard will allow you to insert a function in this location.

CAUSE

This behavior is by design.

If you choose the Function Wizard while you are editing a function, it behaves the way the Microsoft Excel 4.0 Paste Name and Paste Function commands behave; that is, it prompts you for functions and arguments that need to be inserted and inserts the resulting formula into the original formula (at the position you selected when you started the Function Wizard).

RESOLUTION

To make sure that functions are inserted where you want them, verify that the mouse pointer is correctly positioned before you start the Function Wizard.

If you want to use the Function Wizard to edit an existing function, select the cell that contains the function (but be careful not to select the function itself) and then choose Function from the Insert menu.

REFERENCES

"User's Guide," version 5.0, Chapter 10, "Using the Function Wizard"

For more information about using the Function Wizard to edit formulas, choose the Search button in Help and type: 

   function wizard
