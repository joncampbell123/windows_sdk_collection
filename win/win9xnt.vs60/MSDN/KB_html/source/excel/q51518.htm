Dates Appear As Serial Values from Excel DBF 2 File

 Last reviewed: November 2, 1994
Article ID: Q51518
Summary:

Dates in files saved from Excel in the DBF 2 format are brought into Q+E as serial values.

To correct this problem, save the file from Excel in DBF 3 file format. Note that all dates, regardless of formatting in Excel, will have the mm/dd/yy format when saved to a file in the DBF 3 format (and, hence, when loaded into Q+E).
