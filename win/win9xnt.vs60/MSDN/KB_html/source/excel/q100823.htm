Excel: Crosstab Report May Not Sort Records Correctly

 Last reviewed: September 12, 1996
Article ID: Q100823

The information in this article applies to:

  • Microsoft Excel for Windows, versions 4.0a

SUMMARY

In Microsoft Excel, the Crosstab ReportWizard creates a report based on the current database, criteria, and selected options from the ReportWizard. The records within the report are sorted in ascending order. However, if you are using the Crosstab Report Wizard in Microsoft Excel version 4.0a, the records may not be sorted correctly.

NOTE: In Microsoft Excel version 4.0, the Crosstab ReportWizard sorts records in the correct order.

Workaround

To work around this problem, do one of the following:

  • Sort the database manually before you generate the Crosstab Report.

    -or-

  • Manually cut and paste the records in the crosstab report that are not sorted properly.

REFERENCES

"User's Guide 1", version 4.0, pages 346-356.
