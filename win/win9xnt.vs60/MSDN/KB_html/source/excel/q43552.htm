Function Macro to Determine Even or Odd Number

 Last reviewed: October 31, 1994
Article ID: Q43552
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0

SUMMARY

The following macro function determines if a given cell reference contains an odd or even number. Please note that the macro returns a #VALUE! error if the cell referred to does not contain a numeric entry and returns FALSE for all nonintegers (for example, 4.224). 

   A1: ISEVEN
   A2: =ARGUMENT("number",1)
   A3: =RETURN(number/2=INT(number/2))
