XL7: Cannot Insert Template Using Shortcut Menu

 Last reviewed: March 27, 1997
Article ID: Q134619
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you insert a template using the shortcut menu on a sheet tab, the template toolbar is displayed, but the template is not inserted into the workbook.

WORKAROUND

  1. On the File menu, click New.

  2. On the Spreadsheet Solutions tab, click any template, and then click OK.

  3. Insert a new worksheet into the template workbook. (You will need least one visible sheet in the workbook when you are done with the steps below.)

  4. The template will be opened in its own workbook. Activate the first sheet in the template, and choose Move or Copy Sheet from the Edit menu.

  5. From To Book, choose another workbook to move the sheet into.

  6. From Before Sheet, specify the position that the sheet will be inserted into, and then choose OK.

  7. To move all the sheets from the template to the workbook of your choice, repeat steps 4-6 as necessary.

NOTE: When you are done moving all the sheets from the template workbook into another workbook, the template toolbar will no longer be displayed.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

All the templates in the Spreadsheet Solutions tab contain a hidden dialog sheet. The behavior described above is a problem with all templates that contain hidden dialog sheets. Any other template that contains a hidden dialog sheet will produce the same behavior.

The templates in the Spreadsheet Solutions tab also have an attached toolbar. When you attempt to insert the template, the toolbar is displayed because it is attached to the sheet. If the template you are trying to insert does not have an attached toolbar, a toolbar will not be displayed.
