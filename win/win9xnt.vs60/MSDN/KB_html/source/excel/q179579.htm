XL98: The Macro ''<Path>'!<Name>' Cannot Be Found

 Last reviewed: February 2, 1998
Article ID: Q179579
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When using Microsoft Excel 98 Macintosh Edition, you may receive an error message similar to one of the following: 

   The macro ''My Macintosh's HD:Microsoft Office 98:Office:
   Excel Add-Ins:Internet Assistant'!DoHTMLCommand' cannot be found.

   -or-

   The macro ''My Macintosh's HD:Microsoft Office 98:Office:
   Excel Add-Ins:Solver'!Stub' cannot be found.

   -or-

   The macro ''My Macintosh's HD:Microsoft Office 98:Office:
   Excel Add-Ins:Lookup Wizard'!DoLookupCommand' cannot be found.

   -or-

   The macro ''My Macintosh's HD:Microsoft Office 98:Office:
   Excel Add-Ins:Report Manager'!Mcp01.PrintReport' cannot be found.

   -or-

   The macro ''My Macintosh's HD:Microsoft Office 98:Office:
   Excel Add-Ins:Conditional Sum Wizard'!DoSumIfCommand' cannot be found.

   -or-

   The macro ''My Macintosh's HD:Microsoft Office 98:Office:
   Excel Add-Ins:File Conversion Wizard'!DoConvertCommand' cannot be found.

   -or-

   The macro ''My Macintosh's HD:Microsoft Office 98:Office:
   Excel Add-Ins:Template Wizard'!StartWizard' cannot be found.

CAUSE

This problem may occur if the following conditions are true:

  • There is an apostrophe (') in the path for an add-in.

    -and-

  • You attempt use a command that calls a demand-loaded add-in.

WORKAROUND

To work around this problem, rename the hard disk or folder. To rename the hard disk, follow these steps:

  1. Select the hard disk icon on the desktop.

  2. Select the entire text of the hard disk name. Be sure to select the entire text of the hard disk name so that every character in the hard disk name is replaced.

  3. Type the name "test" (without the quotation marks), and press RETURN.

STATUS

This behavior is by design of Microsoft Excel.

MORE INFORMATION

The following commands call demand-loaded add-ins: 

   Save as HTML
   Report Manager
   Solver
   Conditional Sum Wizard
   File Conversion Wizard
   Lookup Wizard
   Template Wizard

Before You Rename the Hard Disk

Before you can rename the hard disk, you must turn File Sharing off. To turn off File Sharing, follow these steps:

  1. On the Apple menu, point to Control Panels, and then click File Sharing.

  2. Click the Start/Stop tab, and then click Stop under File Sharing on.

  3. Type 0 (zero) in the box. Click OK.

  4. Close the File Sharing dialog box.

You can select and rename the name of the hard disk.

After You Rename the Hard Disk

To turn File Sharing back on, follow these steps:

  1. On the Apple menu, point to Control Panels, and then click File Sharing.

  2. Click the Start/Stop tab, and then click Start under File Sharing on.

  3. Close the File Sharing dialog box.
