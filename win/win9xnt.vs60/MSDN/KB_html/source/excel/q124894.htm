XL5: Current User Settings Remain After Remove All Under NT

 Last reviewed: September 12, 1996
Article ID: Q124894
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Windows NT

SYMPTOMS

When you run Microsoft Excel Setup in maintenance mode from a Microsoft Windows NT workstation, and you choose Remove All, the Excel\5.0 settings remain in HKEY_CURRENT_USER.

Note that the Excel\5.0 settings are removed from HKEY_LOCAL_MACHINE as expected.

CAUSE

This problem occurs because the security system in Windows NT prevents Microsoft Excel from affecting other user's registry entries. That is, the system is designed to prevent one user from removing the Excel\5.0 settings for all users.

STATUS

This is expected behavior if you are running Microsoft Excel under Windows NT. In Windows NT, only the user or the administrator can modify the user's environment.
