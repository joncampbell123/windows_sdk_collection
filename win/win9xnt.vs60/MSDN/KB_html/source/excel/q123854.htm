XL5: "User's Guide" Documentation Changes

 Last reviewed: September 12, 1996
Article ID: Q123854
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Office for Windows, versions 4.2, 4.3

SUMMARY

The Microsoft Excel "User's Guide" that ships with Microsoft Office versions 4.2 and 4.3, contains several changes from the original "User's Guide" that shipped with the stand-alone version of Microsoft Excel 5.0. The changes to the "User's Guide" include corrections to the original text as well as some additional information.
