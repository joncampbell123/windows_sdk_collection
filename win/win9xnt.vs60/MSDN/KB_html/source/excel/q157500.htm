Orientation Does Not Change with LaserWriter 8.4 Driver

 Last reviewed: February 2, 1998
Article ID: Q157500
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Project for the Macintosh, version 4.0

SYMPTOMS

In the Microsoft Excel or Microsoft Project Page Setup dialog box, when you change the paper orientation from Landscape to Portrait, or Portrait to Landscape, you cannot see the change in Print Preview. However, when you print the document, it is printed with the correct orientation.

CAUSE

This problem may occur when all of the following conditions are true:

  • You use the Apple LaserWriter, version 8.4, printer driver. 

        -and-
  • You click Options in the LaserWriter Page Setup dialog box and change the paper source. For example, in the Paper list, you change US Letter Small to US Letter. 

        -and-
  • You click the Page tab in the Microsoft Excel or Microsoft Project Page Setup dialog box, and click a setting in Orientation list. For example, you change Portrait to Landscape.

NOTE: This problem occurs when you use version 8.4 of the LaserWriter printer driver.

RESOLUTION

To resolve this problem use either of the following methods:

Method 1

Use an earlier version of the LaserWriter printer driver.

Method 2

In the Apple LaserWriter Page Setup dialog box, click Options and change the paper source to something other than the current setting.

When you switch to Print Preview, the orientation will match the setting in the Microsoft Excel Page Setup dialog box.

NOTE: If you change the setting in the Orientation list again, the problem reoccurs. If this happens, use Method 2 to display the orientation correctly in Print Preview.
