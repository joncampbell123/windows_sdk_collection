Packrat Includes Excel Macro to Exchange Data

 Last reviewed: November 4, 1994
Article ID: Q73802
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a

SUMMARY

Packrat 3.0 is an Information Management Windows-based application by Polaris. Packrat includes a Microsoft Excel macro that allows data from Microsoft Excel to be transferred to Packrat. The macro name is STARTRAT.XLM and is usually found in the PACKRAT directory after a successful Packrat installation. The STARTRAT.XLM macro sheet also includes an Auto_Open macro. Any questions regarding this macro should be directed to Polaris.

Packrat 4.0 puts the file PIL4.XLA in the XLSTART directory that loads Packrat menus in Microsoft Excel.

Packrat version 5.0 puts a file called PIL4.XLA in the XLSTART directory.
