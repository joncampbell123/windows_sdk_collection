Excel: Printer Options Settings via Page Setup Don't Work

 Last reviewed: November 4, 1994
Article ID: Q89243
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0 4.0a

SUMMARY

In Microsoft Excel for Windows, the printer options settings accessed by selecting Options in the Page Setup dialog box don't work. Changes you make in the Options dialog box are not used and won't be retained when you close the Page Setup dialog box.

MORE INFORMATION

When you choose the Options button on the Page Setup dialog box, Microsoft Excel displays the Options dialog box that contains settings for the currently selected printer (actual options vary with printer drivers). The options displayed are the same options available by choosing Setup in the Printer Setup dialog box and then selecting the Options button.

Note: Some printer drivers don't provide additional options. In this case, the Options button is unavailable or nothing happens when you select it.

Workaround

The only workaround is to make the changes you want in the Options dialog box available through Printer Setup.

To change options settings:

  1. From the File menu, choose Page Setup.

  2. In the Page Setup dialog box, choose the Printer Setup button.

  3. In the Printer Setup dialog box, choose the Setup button.

  4. In the printer dialog box, choose Options.

  5. In the Options dialog box, make the changes you want, then choose OK.

  6. Choose OK on all the dialog boxes to return to Microsoft Excel.

The Options settings made this way affect printing from other Windows applications using this printer. Also, if these options settings are changed in another application, those changes affect printing with Microsoft Excel.

STATUS

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft Excel for Windows. We are researching this problem and will post new information here as it becomes available.
