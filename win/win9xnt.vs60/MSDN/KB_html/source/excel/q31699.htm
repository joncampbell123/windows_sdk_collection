Excel: Save Chart as PICT Graphic to Copy to Other Application

 Last reviewed: November 29, 1994
Article ID: Q31699
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SUMMARY

To save a Microsoft Excel chart as a PICT graphic, hold the SHIFT key down and select Copy Picture from the Edit menu. If you are using Microsoft Excel version 1.50 or earlier, choose Copy Chart from the Edit menu.

The chart can then be pasted into the Scrapbook or can be copied directly into another application.

In Microsoft Excel versions 2.20 and later, the Copy Picture command brings up a dialog box that offers options as to how the picture is to be copied. By selecting different options, the appearance and/or size of a chart can be copied as it appears on the screen or on the printer.
