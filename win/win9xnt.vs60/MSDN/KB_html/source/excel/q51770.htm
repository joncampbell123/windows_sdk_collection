Macro Halts and Returns Error at SIZE() Command

 Last reviewed: November 29, 1994
Article ID: Q51770
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2, versions 2.2, 3.0

SUMMARY

A macro error occurs if the SIZE() function is used and the active sheet is currently maximized. To prevent this error, the SIZE() function should be preceded by a command that restores the window to its previous size.

For example:

Microsoft Excel 2.x through 4.0

   =FULL(FALSE)         Restores the window to its previous size
   =SIZE(264,264)

Microsoft Excel 4.0 and 5.0

   =WINDOW.RESTORE()     Restores the window to its previous size
   =WINDOW.SIZE(264,264)
Note: the information above also applies to version 5.0 only when using version 4.0 macro language.
