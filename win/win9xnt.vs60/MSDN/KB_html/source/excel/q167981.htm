XL97: Error Viewing Properties of Visual Components Control

 Last reviewed: July 30, 1997
Article ID: Q167981
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you insert a VisualSpeller control or a First Impression Chart control into a worksheet in Microsoft Excel 97, you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you click Details, you may receive one or more error messages similar to any of the following: 

   EXCEL caused an invalid page fault in module VBA332.DLL at
   014f:65200039.

   EXCEL executed an invalid instruction in module <unknown> at
   0000:00400039.

   EXCEL caused an invalid page fault in module OC30.DLL at
   014f:5fb532ae.

   EXCEL caused an invalid page fault in module KERNEL32.DLL at
   014f:bff798fc.

   EXCEL caused an invalid page fault in module MFCANS32.DLL at
   014f:5f107633.

CAUSE

This problem may occur if either of the following conditions is true:

  • You attempt to display the Properties window for a VisualSpeller control or for a First Impression Chart control that is on a worksheet.

    -or-

  • You click a VisualSpeller control or a First Impression Chart control when the Properties window is visible.

Note that this problem does not occur when the VisualSpeller control or First Impression Chart control is on a UserForm.

RESOLUTION

To prevent this problem from occurring, do not display the Properties window when a VisualSpeller control or a First Impression Chart control is active in a worksheet.

MORE INFORMATION

In Microsoft Excel 97, you can insert controls into a worksheet by using the Control Toolbox toolbar. To insert a control other than the default controls, click More Controls, and then click the tool you want to insert from the list of controls.

You can display the Properties window for a control by right-clicking the control, and then clicking Properties on the shortcut menu. The Properties window remains visible until you close it. If you display the Properties window and click a VisualSpeller control, the problem described in this article occurs.

Contact Visual Components to see whether newer versions of the VisualSpeller control and/or the First Impression Chart control are available. It's possible that newer versions do not cause this problem. To visit the Visual Components Web site on the Internet, use the following address: 

   http://www.visualcomp.com/
You can also contact Visual Components technical support by calling (913) 599-6500, or by sending e-mail to the following address: 

   support@visualcomp.com
