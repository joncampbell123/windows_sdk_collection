XL: Lotus 1-2-3 WK4 File Macro Support

 Last reviewed: December 17, 1996
Article ID: Q123265
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

In Microsoft Excel, the following restrictions apply to running macros contained in a Lotus 1-2-3 WK4 file:

  • The commands in the macro must be backward compatible to Lotus 1-2-3 Release 2 (note that Release 2 lacks many of the macro commands available in Release 4).

    -and-

  • When you run the macro, the worksheet that contains the macro code must be active.

    NOTE: This restriction means that if you have a separate worksheet that contains the macro code, you cannot use those macros with data that is contained on a different worksheet.

You will receive a macro error when you run a macro contained in a Lotus 1- 2-3 WK4 file in Microsoft Excel if either of the above conditions is not true.

MORE INFORMATION

Microsoft Excel versions 7.0 and 97 can open Lotus 1-2-3 WK4 files without the need for an additional converter.

To open Lotus 1-2-3 WK4 files in Microsoft Excel version 5.x, you must obtain the Lotus 1-2-3 WK4 File Converter. The Lotus 1-2-3 WK4 File Converter is available in application note WE1130 "Lotus 1-2-3 WK4 File Converter."

You can obtain this Application Note from the following sources:

  • Microsoft's World Wide Web Site on the Internet
  • The Internet (Microsoft anonymous ftp server)
  • The Microsoft Network (MSN)
  • Microsoft Download Service (MSDL)
  • Microsoft Product Support Services

For more information about obtaining the Lotus 1-2-3 WK4 File Converter, please see the following article in the Microsoft Knowledge Base:
