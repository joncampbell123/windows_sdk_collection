Excel: Changing Column Width Causes Number to Round

 Last reviewed: November 30, 1994
Article ID: Q63266
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.1, 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0
  • Microsoft Excel for OS/2, versions 2.2 and 3.0

SUMMARY

When a number with values to the right of the decimal is entered into a cell in Microsoft Excel and the entire decimal portion of the number does not fit, the number rounds to the displayed digit. For example:

  1. Enter 10.17453 into cell A1 and set the column width to display the entire number.

  2. Decrease the column width so that only 10.174 can be displayed.

10.175 appears in cell A1, because the last digit (4) has been rounded up.

NOTE: Number signs (#) are displayed in a cell only if the integer portion of the number cannot fit within the current width of the cell.
