Excel: ME0185: Using Custom Help

 Last reviewed: February 2, 1998
Article ID: Q97579

The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, and 4.0

SUMMARY

The Application Note "Using Custom Help," is now available. This Application Note discusses custom help for Microsoft Excel versions 2.2, 3.0, and 4.0 for the Macintosh. With custom help, you can create and display your own help files. You can call these help files directly with a macro statement, or you can access them from custom menus and custom dialog boxes. In Microsoft Excel version 4.0, you can also access custom help files from message alerts and input windows.

MORE INFORMATION
