Excel err msg:...Selection Already Contains Only Visible Cells

 Last reviewed: November 30, 1994
Article ID: Q80481
Summary:

Normally, if a worksheet contains an outline with collapsed levels and a single cell is selected, clicking on the Select Visible Cells button will highlight the whole worksheet minus the collapsed cells.

If rows at the top of a worksheet or columns at the left edge of a worksheet are demoted and collapsed, clicking on the Select Visible Cells button will cause Excel to display the following error message: 

  The current selection already contains only visible cells.
Microsoft is researching this problem and will post new information here as it becomes available.

Reference(s):

"Microsoft Excel User's Guide," for the Macintosh, version 3.0, page 231
