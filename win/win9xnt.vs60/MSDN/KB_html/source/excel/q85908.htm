Excel: Views Do Not Save Row Heights and Column Widths

 Last reviewed: November 30, 1994
Article ID: Q85908
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0. 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 4.0, 5.0

SUMMARY

In Microsoft Excel, Views do not save the row height or column width of cells.

MORE INFORMATION

Page 255 of the "User's Guide 1" states that a view includes the settings of the current row height and column width. This is incorrect. The view will only save information on which rows and columns are hidden.

Note: in version 5.0, when you try to save a view with hidden columns or rows, you receive the following error: 

   Macro error at cell:
   [VIEWS.XLA]Views.XLM!B860

REFERENCES

"User's Guide 1," version 4.0, page 255
