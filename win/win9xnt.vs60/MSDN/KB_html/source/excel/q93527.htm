Excel: Pasting Embedded Chart Changes Source of Link

 Last reviewed: November 30, 1994
Article ID: Q93527
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 3.0
  • Microsoft Excel for Windows, version 3.0

Summary:

In version 3.0 of Microsoft Excel, you can copy an embedded chart that was created on one worksheet to another worksheet by selecting the chart, choosing Copy from the Edit menu, and pasting the chart in the second worksheet.

When a chart is copied with this method, the chart references source data from the second worksheet not the from the original worksheet. To copy the embedded chart to another worksheet and have it retain the reference to the original source data:

  1. Double-click the embedded chart to open the chart in a Chart window.

  2. From the Chart menu, choose Select Chart.

  3. From the Edit menu, choose Copy.

  4. Switch to the destination worksheet.

  5. From the Edit menu, choose Paste.

More Information:

This undocumented feature of Microsoft Excel version 3.0 was changed in Microsoft Excel version 4.0. In Microsoft Excel version 4.0, when you copy an embedded chart it always retains the links to the original source data.
