Excel: Shadow on Unfilled Rounded Objects Don't Print

 Last reviewed: November 29, 1994
Article ID: Q85588
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 4.0

Summary:

In Microsoft Excel for the Macintosh, an unfilled rounded object (an oval or a rectangle with rounded corners), formatted to show a shadow, prints without the shadow to an Apple LaserWriter printer. The shadow will display correctly on the screen but will not preview or print.

More Information:

There is currently a problem in the LaserWriter printer driver that results in the shadow being printed as a solid filled object on top of the unfilled object. Microsoft Excel doesn't print the shadow because of this problem.

Reference(s):

"Microsoft Excel User's Guide 1," version 4.0, pages 491-497, 508
