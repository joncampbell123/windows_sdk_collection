Support for Excel Developers

 Last reviewed: November 29, 1994
Article ID: Q86163
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

Microsoft has implemented an improved system for supporting Microsoft Excel developers. Due to the changes in the services we offer, the Microsoft Excel Developers Program is no longer accepting new memberships. Current members will continue to receive membership services.

The following materials are available for Microsoft Excel developers:

Microsoft Excel Macros Step by Step

  • 200 page macro tutorial provides complete coverage of Microsoft Excel macros.
  • Includes samples for Windows and Macintosh systems.
  • Available from Microsoft Press.

Microsoft Excel Software Development Kit (SDK)

  • 400 page reference including disks for system-level developers.
  • Documentation and tools for using the new Microsoft Excel 4.0 API.
  • File format documentation for XLS, XLC and XLW files for Microsoft Excel versions 2.0 to 4.0.
  • Apple Events reference.
  • Information on Dynamic Data Exchange (DDE).
  • Microsoft Excel startup files.
  • Cross-platform Help files (for Windows and Macintosh systems).
  • Includes disks for both Microsoft Excel for Windows and Microsoft Excel for the Macintosh.
  • Now available.

NOTE: The preliminary copy of the Microsoft Excel SDK (which was available from May 20 to October 1, 1992) contained no macro information. For more information or to order call Microsoft Developer Services at (800) 227- 4679, product number: 065-000-020.
