XL97: IntelliMouse Cannot Zoom Beyond 100%

 Last reviewed: March 13, 1998
Article ID: Q157750
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you use the zoom feature with the Microsoft IntelliMouse pointing device to magnify or reduce the screen display, 100% is the maximum zoom percentage allowed.

CAUSE

Although Microsoft Excel is capable of zooming out to 400%, the Microsoft IntelliMouse zooms out to 100%.

WORKAROUND

To zoom out over 100%, zoom out manually. To manually zoom in Microsoft Excel, use either of the following methods:

  • On the View menu, click Zoom, select the Zoom percentage you want, and click OK. 

        -or-
  • Click the zoom percentage you want on the Zoom box on the standard toolbar.

MORE INFORMATION

The Microsoft IntelliMouse is a mouse pointing device that also contains a wheel between the primary and secondary mouse buttons. You can use this wheel to perform different actions within programs that are written to recognize the new mouse driver. One such action is zooming by moving the mouse wheel. To zoom in or out on a worksheet using the Microsoft IntelliMouse press and hold the CTRL key on the keyboard, and then rotate the IntelliMouse wheel forward (to zoom in) or backward (to zoom out).

Microsoft Excel provides an option in the General tab of the Options dialog box called "Zoom on roll with IntelliMouse." By selecting this option, you can zoom without holding CTRL.

REFERENCES

For more information about zooming, click the Index tab in Microsoft Excel Help, type the following text 

   Zooming
and then double-click the selected text to go to the "Scroll and zoom on a Microsoft Excel sheet by using the Microsoft IntelliMouse pointing device" topic.
