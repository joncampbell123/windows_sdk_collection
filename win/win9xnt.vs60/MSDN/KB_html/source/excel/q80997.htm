Excel: Maximum Number of Custom Menu Items Limited to 255

 Last reviewed: November 30, 1994
Article ID: Q80997
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 5.0

The maximum number of custom menu items that can be added to Excel is 255. This occurs whether the commands are added on one menu or spread across several menus. Creating more than 255 custom menu items results in a macro error on the step on which the menu item is created.
