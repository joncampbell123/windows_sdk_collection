Excel: Chart Page Number Incorrect in User's Guide 1 Index

 Last reviewed: November 29, 1994
Article ID: Q90897
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

Page 638 of the User's Guide 1 Index for Microsoft Excel version 4.0 refers to Volume/Hi-Lo-Close charts on pages 399 and 412. This is a documentation error; the correct pages are 399 and 410.

REFERENCES

"Microsoft Excel User's Guide 1," version 4.0, pages 399, 410
