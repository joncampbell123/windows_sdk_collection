XL5: Can't Drag Information from One Workbook to Another

 Last reviewed: September 12, 1996
Article ID: Q105099
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

In Microsoft Excel version 5.0, if you want to drag information from one workbook and place it in another workbook, you need to run a second instance of Microsoft Excel.

MORE INFORMATION

One of the features of object linking and embedding (OLE) 2.0 is the ability to take data from one application and drag it to another application without using the Copy and Paste commands. In Microsoft Excel 5.0, you can use this feature to drag information from Microsoft Excel to other applications that support OLE (including other instances of Microsoft Excel). You can also use this feature to drag a worksheet from one workbook to a different workbook.

However, you cannot drag information from one Microsoft Excel workbook to another in the same instance of Microsoft Excel. That is, you cannot drag information from one workbook to another workbook unless the second workbook is running in another instance of Microsoft Excel. In addition, you cannot drag data from one worksheet to another within the same workbook. This feature is by design.
