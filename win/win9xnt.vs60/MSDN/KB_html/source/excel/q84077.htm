Excel: Tutorial Doesn't Run Under Pen Windows

 Last reviewed: December 7, 1994
Article ID: Q84077
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SYMPTOMS

If you are using Microsoft Excel version 4.0 for Windows and the Pen Windows Computing extensions are installed on your system, you may not be able to run the online tutorial (also known as computer-based training or CBT).

CAUSE

This problem occurs because of a conflict between the CBT engine and the Pen Windows Computing extensions.

On a non-Pen Windows system, the CBT intercepts all messages that are sent to the Microsoft Excel window. However, the recognizer in the Pen Windows extensions also wants to receive messages going to the Microsoft Excel window (so that it can process the link); the result of this conflict is that the CBT will stop when you attempt to issue certain commands.

REFERENCES

"User's Guide 1," version 4.0, page 12
