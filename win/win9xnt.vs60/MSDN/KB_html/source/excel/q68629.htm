Cannot Change Information Entered in User Information Dialog

 Last reviewed: November 3, 1994
Article ID: Q68629
The information in this article applies to:
  • Microsoft Excel for Windows versions 3.0 and 4.0

Summary:

In Microsoft Excel, the first time that the Setup program is started, you will be asked to enter your name and company name into the User Information dialog. After you have entered the requested information and have chosen OK, a confirmation dialog will appear. Choosing OK in the confirmation dialog will cause the setup program to write this information to Excel. After you have left the confirmation dialog, you cannot change any of this information.

The information that has been entered will appear as part of the logo screen that appears every time Excel is invoked; it will also appear in the Help About dialog.
