XL: How to Find Resources on Microsoft Excel Solver

 Last reviewed: February 2, 1998
Article ID: Q146606
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

You can find information about Microsoft Excel Solver in the following resources:

  • For detailed information about the algorithm Microsoft Excel Solver uses, please see the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q82890

          TITLE     : XL: Solver Uses Generalized Reduced Gradient Algorithm


  • For more information about Microsoft Excel Solver, click the Index
   tab in Microsoft Excel Help, and type the following text:

    

          solver

    

  • For more information about the internal solution process of Microsoft
   Excel Solver, contact the following:

    

          Frontline Systems
      P.O. Box 4288
      Incline Village, Nevada  89450-4288
      (702) 831-0300

    

  • Information can also be found on the following Web site:

    
   http://www.frontsys.com/
