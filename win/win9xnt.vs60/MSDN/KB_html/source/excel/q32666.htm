Excel 1.50: Item Height and Width in Wrong Position

 Last reviewed: October 31, 1994
Article ID: Q32666

SUMMARY

The illustration on pages 182 and 183 of the "Microsoft Excel Arrays, Functions and Macros" version 1.5 manual is incorrect.

In the screen dump of the dialog box definition table on pages 182 and 183, the positions of the Item Height and the Item Width columns are reversed. In a dialog-box definition table, the Item Width should be in the fourth column and the Item Height should be in the fifth column.
