Excel: Radius Pivot Monitor May Hang Excel Version 4.0

 Last reviewed: November 16, 1994
Article ID: Q93446
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

The Radius Pivot variable-orientation monitor system may cause Microsoft Excel version 4.0 for the Macintosh to hang. According to Radius Technical Support, changing some of the software settings or upgrading the Radius Pivot ROM may correct this problem.

Workaround 1

To prevent this problem from occurring, change following settings in the Radius Pivot software:

  • On Page One of the Pivot software, clear (turn off) the Enlarged Menu Fonts option.

    -and-

  • On Page Three of the Pivot software, clear (turn off) the Auto Centering, Auto Lowering, and Big Boundaries options.

If changing the above settings does not correct the problem, try changing the following setting in addition to the above:
  • On Page Two of the Pivot software, clear (turn off) the Dynamic Desktop option.

If you have changed all of the above settings and Microsoft Excel continues to hang, contact Radius Technical Support. A ROM upgrade of the Pivot hardware or an upgrade of the Pivot software may be necessary to correct this problem.

Workaround 2

Disable the Pivot software.

More Information:

As of December 1992, the latest version of the Radius Pivot software is version 2.1.1.
