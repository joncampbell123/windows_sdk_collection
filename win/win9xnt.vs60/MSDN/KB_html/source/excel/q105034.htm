MSQuery: Can't Print from Microsoft Query

 Last reviewed: September 12, 1996
Article ID: Q105034
The information in this article applies to:
  • Microsoft Query for Windows, version 1.0
  • Microsoft Query for Windows 95, version 2.0

SUMMARY

You cannot print from Microsoft Query.

MORE INFORMATION

Microsoft Query is an application included with Microsoft Excel versions 5.0 and 7.0, and may be included in other Windows applications in the future. It can be used to retrieve, view, and organize data from a variety of sources.

Because Microsoft Query is not a reporting tool--it is a tool you can use to transfer data to reporting tools (for example, to Microsoft Excel or Microsoft Word)--the ability to print is not available in Microsoft Query. To print the information brought into Microsoft Query, the information must be saved to a file and opened in another application such as Microsoft Excel, Word, Notepad, or Write.
