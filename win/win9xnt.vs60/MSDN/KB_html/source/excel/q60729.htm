Excel: DIRECTORY Returns Error When Activating Root Level

 Last reviewed: November 2, 1994
Article ID: Q60729

SUMMARY

To activate the root directory from a macro, the path_text argument must be followed by a colon (:).

For example, to activate the directory "Hard Disk," use the following formula: 

   =DIRECTORY("Hard Disk:")
For more information on the DIRECTORY function, see page 58 of the "Microsoft Excel Function Reference" version 3.0 manual. If you are using Excel 2.20, see page 213 of the "Microsoft Excel Functions and Macros" version 2.2 manual.
