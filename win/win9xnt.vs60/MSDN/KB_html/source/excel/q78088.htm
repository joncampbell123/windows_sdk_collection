OPEN Macro Command Switches Don't Work in Q+E

 Last reviewed: November 3, 1994
Article ID: Q78088
Summary:

Q+E's OPEN macro command has four versions. Including any optional parameter other than "source" will cause the query to fail.

More Information:

The four OPEN commands are for the following formats: 

   dBasefile
   Excelfile
   Textfile
   SQL Server
Each of these formats has one required parameter (the filename) and varying numbers of optional parameters, depending on the source file. None of the optional parameters, except for the second one (source) will work with any OPEN version.

Reference(s):

"Q+E for Microsoft Excel User's Guide," version 3.0, page 101
