Excel: Quick Keys for Top and Bottom Borders Disabled

 Last reviewed: November 2, 1994
Article ID: Q71088

SUMMARY

When Microsoft Excel is used with the CloseView CDEV (Control Panel device) installed in the System Folder, the quick keys for adding top and bottom borders (COMMAND+OPTION+UP ARROW, COMMAND+OPTION+DOWN ARROW) may not function.

If CloseView is causing the quick keys in Excel to not function properly, you must do one of the following:

Method 1

Remove CloseView from the System Folder and then restart the Macintosh.

Method 2

Turn off the Keyboard Shortcuts option for CloseView from the Control Panel, and then restart the Macintosh.

MORE INFORMATION

CloseView uses COMMAND+OPTION+UP ARROW and COMMAND+OPTION+DOWN ARROW to change the screen resolution when CloseView is activated.

For more information on CloseView, see the "Macintosh Utilities User's Guide."
