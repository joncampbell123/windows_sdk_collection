Excel: DATE Function with Dates After December 31, 1999

 Last reviewed: November 2, 1994
Article ID: Q26450

SUMMARY

Note: Microsoft Excel version 3.00 allows dates up until 12/31/2078 to be entered using the normal date-entry method. After the year 2078, use the method described below.

To enter a date in Microsoft Excel that falls after December 31, 1999, use a year number of 100 or greater. For example, to generate a serial date value for February 20, 2002, enter the following: 

   =DATE(102,2,20)
