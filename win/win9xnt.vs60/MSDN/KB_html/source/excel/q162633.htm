XL97: Visual Basic Help File Is Available for Microsoft Map

 Last reviewed: February 12, 1998
Article ID: Q162633
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SUMMARY

The Visual Basic for Applications Help file for Microsoft Map is available for Microsoft Excel 97. This Help file is available to download from the Microsoft Software Library (MSL). The file is the same file that is included in the ValuPack folder on the Microsoft Office 97 compact disc.

The following file is available for download from the Microsoft Software Library: 

 ~ Vbamap8.EXE (size: 55995 bytes) 
               55KB
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
