MXL5: Menus May Become Detached or Hard to Read

 Last reviewed: February 2, 1998
Article ID: Q174047
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SYMPTOMS

Menus in Microsoft Excel may become hard to read and use. Submenus may become separated from the parent menu or command, and you cannot select a menu command on a submenu.

CAUSE

This behavior occurs if all of the following conditions are true:

  • You are using Mac OS 8. 

        -and-
  • You are running Microsoft Excel for the Macintosh, versions 5.0 or 5.0a.

    This problem does not occur in Microsoft Excel 98 Macintosh Edition. 

        -and-
  • You select "System-wide platinum appearance" in the Appearance control panel device in Control Panel. 

        -and-
  • In Microsoft Excel, you click Options on the Tools menu, click the General tab, and click On under Command Underline.

NOTE: If you click Off or Automatic under Command Underline, the behavior occurs when you use the keyboard to navigate menus in Microsoft Excel.

WORKAROUND

To workaround this behavior, use either of the following methods.

Method 1: Set the Command Underline Option to Automatic or Off

To disable Command Underline, follow these steps:

  1. On the Tools menu, click Options.

  2. Click the General tab and click Automatic or Off under Command Underline.

  3. Click OK.

NOTE: Even if you click Automatic or Off under Command Underline, you cannot navigate menus by using the keyboard. If you want to navigate by using the keyboard, use Method 2.

Method 2: Turn off "System-wide platinum appearance"

To disable "System-wide platinum appearance," follow these steps:

  1. On the Apple menu, click Control Panels.

  2. In the Control Panels window, double-click Appearance.

  3. Click Options, and then clear the "System-wide platinum appearance" check box.

  4. Close the Appearance window.

  5. Restart your computer.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. The problem is corrected in Microsoft Excel 98 Macintosh Edition.

MORE INFORMATION

The definitions for the Command Underline settings in Microsoft Excel are described as follows:

  • On: Underlines the letter that activates the menu in each menu title. 

           Also underlines letters that activate commands in dialog boxes.
  • Off: Turns off underlining of letters in menu titles and dialog boxes. 
            The letters continue to work, but the underlines are not
        displayed.
  • Automatic: Turns off underlining of letters in menu titles and dialog 
                  boxes until you press F10 or the / key (or the key that is
              specified in the Transition tab of the Options dialog box).
    To activate the menu bar and move around Microsoft Excel menus by using your keyboard, press F10. Depending on the setting you choose under Command Underline, an underlined letter may appear on each menu or menu item.
