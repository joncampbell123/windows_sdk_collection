XL: Drawing Selection Tool Mouse Pointer Retains Arrow Shape

 Last reviewed: March 27, 1997
Article ID: Q112322
5.00 5.00c 7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

In Microsoft Excel versions 5.x and 7.0, when you choose the Drawing Selection button, the mouse pointer changes to an arrow, not a thin plus sign.

This is expected behavior.

NOTE: This behavior differs from the behavior of other drawing tool buttons. When you choose other drawing tool buttons, the mouse pointer changes to a thin plus sign.

REFERENCES

"User's Guide," version 5.0, pages 232-233

For more information about the Drawing Selection Button, choose the Search button in Help and type: 

   drawing
