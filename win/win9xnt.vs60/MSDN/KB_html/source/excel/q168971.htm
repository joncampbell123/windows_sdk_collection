XL97: Error Sending Workbook to Mail Recipient

 Last reviewed: January 9, 1998
Article ID: Q168971
The information in this article applies to:
  • Microsoft Excel 97 for Windows

WARNING: The information in this article has not been confirmed or tested by Microsoft. Some or all of the information in this article has been taken from unconfirmed customer reports. IMPORTANT: This article contains information about editing the registry. Before you edit the registry, you should first make a backup copy of the registry files (System.dat and User.dat). Both are hidden files in the Windows folder.

SYMPTOMS

If you point to Send To on the File menu, and then click Mail Recipient, you may receive the following error message: 

   Cannot load the mail service. Check your mail installation.

CAUSE

You may receive this error message if you had the "Lotus VIM Mail Support" setup option selected when you ran the Office Setup program.

NOTE: The "Lotus VIM Mail Support" setup option is a sub-component of the "Office Tools" option.

The Microsoft Office 97 Setup program is incorrectly writing a key to the Windows registry and Microsoft Excel 97 is supposed to ignore this key if it exists.

RESOLUTION

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows 95. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.

For information about how to edit the registry, view the Changing Keys And Values online Help topic in Registry Editor (Regedit.exe). Note that you should make a backup copy of the registry files (System.dat and User.dat) before you edit the registry.

To prevent this error from occurring, remove the following key from the Windows registry: 

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\8.0\Common\MapiDLL

MORE INFORMATION

To send or route workbooks through e-mail, you need one of the following:

  • Microsoft Exchange (or other mail system compatible with the Messaging Application Programming Interface [MAPI]).
  • Lotus cc:Mail (or another mail system compatible with Vendor Independent Messaging [VIM]).

REFERENCES

For more information about Mailing Workbooks, click the Office Assistant, type "mail," click Search, and then click to view "Distribute workbooks to other people."
