Excel: Lines in Line Chart Not Printed to HP LaserJet III

 Last reviewed: October 7, 1997
Article ID: Q102952
4.00 4.00a WINDOWS kbprint

The information in this article applies to:

- Microsoft Excel for Windows, versions 4.0, 4.0a

SYMPTOMS

The Hewlett-Packard (HP) Laserjet III driver, HPPCL5A.DRV, version 31.3.89, dated 3/10/92, that shipped with Windows 3.1, will not print the lines on a line chart when the line colors are a light color such as light green, light gray, cyan, violet, or yellow, and the line weight is not the default lightest weight.

STATUS

This problem was corrected in the HP Laserjet III driver version 2.0 (HPPCL5MS.DRV).

MORE INFORMATION

HPPCL5MS.DRV is available for downloading through the Microsoft Download Service (MSDL). HPPCL5MS.DRV is a compressed, self-extracting .EXE file. After you download HPPCL5MS.EXE, run it to extract the file(s) it contains.

The number for the MSDL is (425) 936-6735 (MSDL). The highest download speed available is 9600 bits per second (bps). Set your communications software to 8 data bits, 1 stop bit, and no parity.
