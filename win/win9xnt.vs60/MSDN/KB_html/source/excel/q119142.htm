XL5: DialogFrame Object Does Not Have BottomRightCell Property

 Last reviewed: September 12, 1996
Article ID: Q119142
The information in this article applies to:

- Microsoft Excel for Windows, version 5.0

SUMMARY

In Microsoft Excel 5.0, in Visual Basic online help, when you select the "Dialog Frames: Underlying Cells" topic and choose the Show Topics button, the BottomRightCell property topic is displayed. This reference is incorrect: the BottomRightCell property does not apply to the DialogFrame object. Only worksheet objects can have the BottomRightCell property.

MORE INFORMATION

To locate the error in the help file:

  1. In Microsoft Excel, choose Contents from the Help menu.

  2. In the Contents dialog box, choose Programming With Visual Basic.

  3. Choose the Search button.

  4. In the search box, type "dialog frames: underlying cells" (without the quotation marks), and choose the Show Topics button.

Note that the BottomRightCell property is displayed as a topic.

This error is corrected in the Visual Basic online help included with Microsoft Excel, version 5.0c for Windows and version 7.0 for Windows 95.
