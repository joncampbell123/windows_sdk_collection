XL: Can't Run Lotus Help Demos

 Last reviewed: March 27, 1997
Article ID: Q106360
5.00 7.00 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Excel for Windows, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

In Microsoft Excel for Windows, the Lotus Help demos will be unavailable if either of the following conditions is true: 

   The AutoFilter feature is enabled in the active worksheet.

   -or-

   The active worksheet is an embedded object.

MORE INFORMATION

Lotus Help is a feature of Microsoft Excel that helps you learn Microsoft Excel equivalents to Lotus 1-2-3 menu items.

To use Lotus Help demos or instructions in Microsoft Excel, you must have an unprotected worksheet active. The Help for Lotus 1-2-3 Users dialog box will display a message explaining that these conditions must be true before you can run a demo or display instructions.

Example using Lotus Help in Microsoft Excel

  1. From the Help menu, choose Lotus 1-2-3.

  2. In the Help for Lotus 1-2-3 Users dialog box, under Help Options, select the Demo button.

  3. From the Menu list, select Range and choose OK.

  4. From the Menu list, select Format and choose OK.

  5. From the Menu list, select Fixed and choose OK.

  6. In the Decimal Places field, type "2" (without the quotation marks) and choose OK.

A demo will show you how to format the current cell to a two-decimal fixed number format in Microsoft Excel.

When the AutoFilter feature is enabled in the current worksheet, or the current worksheet is an embedded object, you will be unable to run a Lotus Help demo and there will be no message explaining why. The Lotus Help instructions will still work under these conditions.
