Q+E's File Open Command Is Disabled When 11 Files Are Open

 Last reviewed: October 31, 1994
Article ID: Q45888

SUMMARY

Q+E can have a total of 11 files open at one time. As soon as this limit is reached, the File Open command is grayed (disabled). If a file or index is closed, File Open is enabled.

Each .DBF, .NDX, and .MDX file counts as one file. If a dBASE IV .DBF file is opened and there is a corresponding index file for it, both files open, using two files. With dBASE III, the index files have to be opened separately. If opened dBASE III indexes that are not in use are closed, each of them returns one file. This is because each index is in a separate file in dBASE III, while all indexes are in one file in dBASE IV.
