XL5: Err Msg: Running Auto_Open from Shortcut Key

 Last reviewed: September 12, 1996
Article ID: Q124331
The information in this article applies to:

Microsoft Excel for Windows, version 5.0

SYMPTOMS

When you use Microsoft Excel version 5.0 for Windows, you may receive the following error message 

   Cannot find macro 'WORKBOOK_NAME![MODULE_NAME].Auto_Open'
where WORKBOOK_NAME is the name of the workbook , MODULE_NAME is the name of the module sheet, and Auto_Open is the name of the sub routine.

CAUSE

This error message is displayed if all of the following conditions are true:

  • The Auto_Open subroutine is a Visual Basic, Applications Edition, macro

    -and-

  • The Auto_Open macro has been assigned a shortcut key

    -and-

  • The Auto_Open module is being run by using the shortcut key

WORKAROUND

To avoid this error message, run the Auto_Open module by choosing Macro from the Tools menu, or assign it to a menu item or toolbar button.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel version 5.0c for Windows .
