Excel: White Out Format Does Not Display on 3-D Column Chart

 Last reviewed: October 7, 1997
Article ID: Q79589
3.00 MACINTOSH kbother

SUMMARY

In a 3-D column chart in Microsoft Excel for the Macintosh version 3.0, text attached to the last data series in the chart may occasionally fail to display the White Out format, if it has been selected. This only applies to labels in 3-D column charts and only to the last series in the plot order.

Please contact Microsoft Product Support Services at (425) 635-7080 for a solution to this problem.
