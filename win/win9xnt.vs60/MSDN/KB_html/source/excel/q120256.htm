MXL5: System Error -192 Starting MS Excel for Power Macintosh

 Last reviewed: February 2, 1998
Article ID: Q120256
The information in this article applies to:
  • Microsoft Excel for the Power Macintosh, version 5.0

SYMPTOMS

When you start Microsoft Excel 5.0 for the Power Macintosh, you receive a System Error -192 error message.

CAUSE

This problem may occur for either of the following reasons:

  • You are using Microsoft Excel for the Power Macintosh on a non-Power Macintosh computer. 

        -or-
  • You are using Microsoft Excel for the Power Macintosh on a Macintosh with a Power Macintosh upgrade card, and the "Power Macintosh Card" Control Panel device is set to Off.

WORKAROUND

To work around this problem, use the appropriate method below.

Method 1

If you chose the incorrect platform during setup, run the Setup program again from Disk 1 of your Microsoft Excel Setup disks, and in the Maintenance Mode dialog box, choose Remove All. Then run the Setup program again to install Microsoft Excel for the correct platform.

Method 2

To set the "Power Macintosh Card" Control Panel device to On, go to your System:Control Panels folder and double-click the Power Macintosh Card control panel device. Click ON, and restart your computer.

MORE INFORMATION

To check to see which version of Microsoft Excel 5.0 you installed (Macintosh Only, Macintosh and Power Macintosh, or Power Macintosh Only), do the following:

  1. Select the icon for the Microsoft Excel application.

  2. From the File menu, choose Get Info.

  3. Compare the Size entry with the values in the following table: 

             Size       Platform
         ----------------------------------------

         5209549    Macintosh ONLY
         7365598    Power Macintosh ONLY
         11904215   Macintosh and Power Macintosh
You must install Microsoft Excel version 5.0 for Macintosh ONLY or for Macintosh and Power Macintosh in order to run the application on a non- Power Macintosh computer.
