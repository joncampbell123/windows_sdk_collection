Excel: EXCEL4.INI Not Installed in Directory During Setup

 Last reviewed: September 12, 1996
Article ID: Q104937
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a

The NETWORK.TXT file that ships with Microsoft Excel versions 4.0 and 4.0a contains the following information: 

   ...your own copy of the Microsoft Excel settings file, EXCEL4.INI,
   is stored in the directory you specified in step 7...
where step 7 refers to the step-by-step instructions in the "To install so Microsoft Excel runs from a network server" section of the file; this information is incorrect.

The EXCEL4.INI file is not installed by the Setup program. It is created the first time you run Microsoft Excel and is always saved to the WINDOWS directory (as is the EXCEL.XLB toolbar file). When you install Microsoft Excel by following the instructions specified in the NETWORK.TXT file, files are not copied to the directory you specify during Setup.

REFERENCES

NETWORK.TXT, Included with Microsoft Excel version 4.0 NETWORK.TXT, Included with Microsoft Excel version 4.0a
