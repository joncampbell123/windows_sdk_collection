Excel: Page Setup Information Not Retained When Saved as SYLK

 Last reviewed: November 2, 1994
Article ID: Q47576

SUMMARY

When a Microsoft Excel file is saved in SYLK format, closed, and then reopened, settings recorded in the Page Setup dialog box of the original file are not retained. Instead, the Page Setup dialog box of the SYLK file displays Excel's default settings.

This situation occurs because the SYLK format does not contain driver-specific information, as it would not be transferable to other environments.

MORE INFORMATION

The Page Setup settings include paper size and orientation, space for header and footer codes, margin measurements, and options for display of gridlines and row and column headings. In addition, the dialog box for an ImageWriter includes check boxes for Tall Adjusted, 50% Reduction, and No Gaps Between Pages.

The LaserWriter Page Setup box includes check boxes for Font Substitution, Smoothing, and Faster Bitmap Printing; the Options button allows access to further LaserWriter options.
