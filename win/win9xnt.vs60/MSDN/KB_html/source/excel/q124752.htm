Excel: Unable to Select Nonadjacent Cells with MS Mouse 9.01

 Last reviewed: September 12, 1996
Article ID: Q124752
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.x, 4.0, 5.0, 5.0c

SYMPTOMS

If you are using the Microsoft Mouse Driver version 9.01, you may be unable to select nonadjacent cells (other functionality may be limited as well). This unexpected behavior may be caused by Microsoft Mouse settings.

WORKAROUND

To reset the Mouse to its default settings, do the following:

  1. From the Main Group in Program Manager, start Control Panel.

  2. Double-click the Mouse icon to start Microsoft Mouse Manager.

  3. Choose the Reset button to return the Mouse to the default settings

    Note, if you do not want to reset all the settings to their defaults, you can clear the Magnify And Locate options in the Extra Points box. (If you do not want to turn off these two functions, then re-assign the options so you do not use the CTRL+PRIMARY or the SHFT+PRIMARY options. These options are used frequently in Microsoft Excel.)

  4. Choose OK from the Mouse Manager dialog box and return to Microsoft 

        Excel.

NOTE: These settings are global settings in Windows that will have precedence over other application settings.

REFERENCES

Microsoft Mouse User's Guide.
