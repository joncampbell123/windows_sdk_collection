Excel: Setup Program Requests Ejected Disk and Then Fails

 Last reviewed: October 6, 1997
Article ID: Q78648
3.00 MACINTOSH kbsetup

SUMMARY

The Microsoft Excel for the Macintosh version 3.0 Setup program fails if one or more of the files to be installed is missing on one of the installation disks. When the Setup program tries to install the missing file, it ejects the current disk and displays the following message: 

   Please insert: <Current Disk Name>
When the current disk is reinserted, the message remains on the screen, but the disk is not ejected. The only option is to click the Cancel button. This brings up the following alert: 

   Install Interrupted. Some Options May Be Only Partially Installed
The Setup program returns to the initial setup screen.
