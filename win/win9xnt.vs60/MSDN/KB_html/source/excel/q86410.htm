Excel: PAGE DOWN Gives Error in Format Patterns Dialog Box

 Last reviewed: November 30, 1994
Article ID: Q86410
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SYMPTOMS

In version 4.0 of Microsoft Excel for Windows, when you press the PAGE DOWN or PAGE UP keys in the Patterns dialog box, you may receive a general protection (GP fault) fault in Microsoft Windows 3.1 or Unrecoverable Application Error (UAE) in Microsoft Windows 3.0.

NOTE: Earlier versions of Microsoft Excel do not allow the use of the PAGE DOWN or PAGE UP keys in the Patterns dialog box.

WORKAROUND

To avoid receiving these error messages, use the UP ARROW and DOWN ARROW keys or the mouse to move between the different choices in the dialog box.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel for Windows version 4.0. This problem was corrected in Microsoft Excel for Windows version 4.0a.

REFERENCES

"User's Guide 1," version 4.0, page 508
