XL5: Formatting Dialog Frame When Sheet Protected Hangs Excel

 Last reviewed: September 12, 1996
Article ID: Q109422
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

In Microsoft Excel version 5.0 for Windows, if you choose Object from the Format menu and if the following three conditions are true

  • The dialog frame is not locked,

    -and-

  • The dialog frame is the currently selected object,

    -and-

  • The dialog sheet is protected,

Microsoft Excel will hang and you will receive the following error message: 

   Application Error
   integer divide by 0

WORKAROUND

If your dialog sheet is protected, and if the dialog frame is the currently selected object, do not choose Object from the Format menu.

Note that you can format other objects in the dialog sheet even if the dialog sheet is protected.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel for Windows 95, version 7.0.
