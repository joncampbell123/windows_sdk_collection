XL97: Floating Toolbars Disable Animation Feedback

 Last reviewed: January 12, 1998
Article ID: Q173979
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you scroll through a Microsoft Excel 97 worksheet, the slow motion or animation effect that slows scrolling does not work even though the "Provide feedback with Animation" check box is selected in the Edit tab of the Options dialog box (click Options on the Tools menu).

CAUSE

This problem occurs if any of the following conditions is true:

  • You display a toolbar so that it is floating (not docked to one side of the window) over the worksheet. 

        -or-
  • You display the Office Assistant so that it is floating over the worksheet.

WORKAROUND

To work around this problem, dock any floating toolbars and move the Office Assistant off the worksheet.

To dock a floating toolbar, use the following steps:

  1. Click the title bar on the floating toolbar.

  2. Drag the toolbar to a new location. If you drag the toolbar to the edge of the program window, it becomes a docked toolbar.

To move the Office Assistant, drag it to any location on the screen that is not on the spreadsheet.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about docking a toolbar, click the Index tab in Microsoft Excel Help, type the following text 

   toolbars
and then double-click the selected text to go to the "moving" topic.

For more information about moving the Office Assistant, click the Index tab in Microsoft Excel Help, type the following text 

   Office Assistant
and then double-click the selected text to go to the "moving" topic.
