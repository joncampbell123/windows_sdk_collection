Excel: "Can't Extend Database" Error with Data Form

 Last reviewed: November 29, 1994
Article ID: Q59614
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0

SYMPTOMS

When you attempt to add records to a Microsoft Excel database using the Data Form command, the error message "Can't Extend Database" may be displayed.

CAUSE

This error occurs if there is previously entered data located in cells directly below the defined range of the database. Excel gives this error to prevent the loss of data. The Data Form command does not insert rows and therefore previously entered information in the cells below a database would be lost if Excel didn't give this error message and prevent this action from occurring.

WORKAROUND

Inserting some blank rows or clearing the data from the cells below the database allows the form to operate correctly.
