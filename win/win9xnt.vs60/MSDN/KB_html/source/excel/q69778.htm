HP NewWave Error Messages in Excel

 Last reviewed: November 2, 1994
Article ID: Q69778

SUMMARY

When using Hewlett-Packard (HP) NewWave Office and Microsoft Excel 3.00 and an error message such as the following occurs 

   Internal Excel error, XL###
where the # signs are numbers, this is not an Excel error. The error comes from NewWave.

MORE INFORMATION

For more information about the error message, contact Hewlett-Packard Post Sales Technical Support at (208) 323-2551.

REFERENCES

"Microsoft Excel User's Guide." Version 3.00. Appendix D, "Using Microsoft Excel with HP NewWave," pages 705-714.
