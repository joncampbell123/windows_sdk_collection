XL97: Problems Using Office Assistant in Netscape Navigator

 Last reviewed: January 7, 1998
Article ID: Q173613
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you insert an object into a Microsoft Excel workbook that is open in a Web browser, you may be unable to access the Office Assistant, either by pressing the F1 key or by clicking Microsoft Excel Help on the Help menu. As a result, you may be unable to access Help in Microsoft Excel 97.

CAUSE

This problem may occur if all of the following conditions are true:

  • You are using Netscape Navigator 3.0, and you have the NCompass ScriptActive plug-in installed. 

        -and-
  • You are viewing an embedded Microsoft Excel workbook using Netscape Navigator 3.0. 

        -and-
  • You inserted an object, such as a hyperlink, a map, or another document, into the workbook.

WORKAROUND

If this problem occurs, it is possible to redisplay the Office Assistant by activating another program that also uses the Office Assistant: 

   Microsoft Word 97
   Microsoft Access 97
   Microsoft PowerPoint 97
   Microsoft Outlook 97
If you activate another program that uses the Office Assistant and then switch back to Netscape Navigator 3.0, the Office Assistant will be available for use in the Microsoft Excel workbook.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

This problem does not occur when using Microsoft Internet Explorer version 3.x.

MORE INFORMATION

The NCompass ScriptActive plug-in for Netscape Navigator 3.0 enables you to view Microsoft Excel workbooks (and other types of documents) directly within Netscape Navigator.
