Excel: Camera Picture Prints Gridlines Despite Page Setup

 Last reviewed: November 4, 1994
Article ID: Q80999
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0, 5.0

SUMMARY

Images created with the camera tool in Microsoft Excel display and print gridlines according to the Gridlines setting in the Options Display dialog box. Page Setup options for Gridlines have no effect on images created with the camera tool.

MORE INFORMATION

The camera tool takes a picture of a selected area on the worksheet, creating a duplicate image that reflects the selected cells contents and formatting, including the screen display of gridlines.

To not print gridlines in a camera image, do the following:

  1. From the Options menu, select Display.

  2. Cancel the Gridlines box selection and choose OK.

REFERENCES

"Microsoft Excel User's Guide" for the Macintosh, version 3.0, pages 66, 520
