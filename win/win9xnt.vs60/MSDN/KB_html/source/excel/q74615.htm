Excel: Links Don't Update During Manual Document Calculation

 Last reviewed: November 1, 1994
Article ID: Q74615

SUMMARY

In Microsoft Excel versions 2.2 and 3.0, if manual calculation is turned on, holding down the SHIFT key and choosing Calculate Document from the Options menu will not update external references.

One workaround is to use the Links command on the File menu to redefine the link. When a link is redefined, the linked values are automatically updated. To update the links in this manner, do the following:

  1. Activate the dependent document (the one containing the link formula to be updated).

  2. From the File menu, choose Links.

  3. Select the name of the supporting document and click the Change button.

  4. Select the name of the same file again, then click the Change button.

Links can also be redefined with the following macro command: 

   =CHANGE.LINKS("filename","filename")
Note that the old filename and the new one are the same. This command assumes that the dependent document is active.

MORE INFORMATION

Excel determines whether links need to be updated by checking a flag associated with each supporting document. The flag is normally set if changes have been made. If manual calculation is turned on, the flag is not updated, and therefore Excel cannot determine that an update is necessary.
