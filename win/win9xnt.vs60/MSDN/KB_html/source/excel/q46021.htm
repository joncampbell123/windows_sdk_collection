TRIM() and RTRIM() SQL Functions Work the Same

 Last reviewed: November 2, 1994
Article ID: Q46021
Summary:

There is no difference in the functionality of the TRIM() and RTRIM() dBASE IV SQL functions. They are both included to retain dBASE IV SQL compatibility.

If you are working with a long SQL statement where minimizing the number of characters becomes essential, using TRIM() saves an extra character.
