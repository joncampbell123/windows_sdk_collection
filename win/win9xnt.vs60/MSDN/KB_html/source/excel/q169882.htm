XL97: How to Rename a Chart Using Name Box

 Last reviewed: January 9, 1998
Article ID: Q169882
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SUMMARY

In earlier versions of Microsoft Excel, you can rename an embedded chart by clicking the chart once, clicking the Name box, and then entering the new name for the chart. This process changed slightly in Microsoft Excel 97 because clicking a chart in Microsoft Excel 97 activates the embedded chart; and you cannot rename a chart while it is active.

MORE INFORMATION

To rename an embedded chart in Microsoft Excel 97, follow these steps:

  1. Select any cell in the worksheet, press CTRL, and then click the embedded chart once.

    A series of white handles (with a black outline) appear around the chart. These handles are different from the handles that appear around the chart when it is active. Handles around an active chart have a solid black color.

  2. Click in the Name box.

    This selects the default name for the chart (for example, Chart 1).

  3. Type the new name for the chart, and then press ENTER.
