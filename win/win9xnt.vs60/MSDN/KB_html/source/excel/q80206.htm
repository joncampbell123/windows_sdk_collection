Excel: Error in Instructions with Excel Help for Lotus 1-2-3

 Last reviewed: November 30, 1994
Article ID: Q80206

SUMMARY

Choosing Lotus 1-2-3 from the Help menu in Microsoft Excel and selecting Instructions for the File Import Numbers or File Import Text command provides incorrect command instructions. The following workaround gives the proper instructions.

MORE INFORMATION

Workaround

Ignore step 5 of the Lotus 1-2-3 instructions in Help for Excel.

Steps to Reproduce Problem

  1. From the Options menu, choose Workspace. Choose "/" (without the quotation marks) as the Alternate Menu or Help key.

  2. From the Help menu, choose Lotus 1-2-3. Select Instructions.

  3. Type "/FIN" (without the quotation marks).

Instructions will appear in a text box on the active document. Steps 5 and 6 of these instructions state: 

   5. Choose File Close.

   6. From Edit choose Paste Special.
Choosing Close from the File menu (step 5) will make the Paste Special command unavailable from the Edit menu (step 6). It therefore is not possible to complete the operation properly.

REFERENCES

"Microsoft Excel User's Guide," for Windows and OS/2, version 3.0, pages 45-72
