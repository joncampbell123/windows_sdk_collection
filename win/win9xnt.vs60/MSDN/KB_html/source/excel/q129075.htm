XL5 Doc Error: Single Custom List Entry Character Limit

 Last reviewed: February 2, 1998
Article ID: Q129075
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel for Windows NT, version 5.0

SUMMARY

Page 122 of the "Microsoft Excel Users Guide," under the Guidelines for Creating Custom Lists, states that there is a limit of 80 characters per entry in a custom list. This information is incorrect. The limit is actually 255 characters.

MORE INFORMATION

In the versions of Microsoft Excel listed above, you have the option of creating a custom AutoFill list that is saved in the toolbar file. Each entry (or each line) of a custom list is capable of containing up to 255 characters. The entire custom list is limited to a total of 2000 characters.
