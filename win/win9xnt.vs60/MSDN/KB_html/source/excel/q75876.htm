"External Reference Is Not Valid" Error on an Excel Chart

 Last reviewed: November 2, 1994
Article ID: Q75876

SUMMARY

In Microsoft Excel 3.0, the error message "External Reference Is Not Valid" will occur in both embedded charts and independent charts if a series formula for the chart returns a #REF! error. This will occur if you have cut information from another part of your sheet and pasted it over the data on which your series formula is based.

This does not occur in Excel 2.x for Windows nor in Excel 2.x for OS/2.

Steps to Reproduce Problem

  1. Type the following into a blank sheet: 

          A1:     1
      A2:     2
      A3:     3
      A4:     4

  2. Highlight cells A1:A4 and copy to cells C1:C4.

  3. Highlight cells A1:A4 and create a chart.

  4. Highlight cells C1:C4 and choose Cut from the Edit menu.

  5. Select cell A1 and choose Paste from the Edit menu. You will receive the error message "External Reference is Invalid." The series formula on the chart will refer to the correct worksheet, but the cell references in the formula will be replaced by #REF!.
