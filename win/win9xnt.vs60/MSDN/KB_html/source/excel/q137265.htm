XL7: Text of CellTip Truncated

 Last reviewed: September 13, 1996
Article ID: Q137265
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, the text of a CellTip (a cell note) may be cut off because the text may not be wrapped in the cell note pop-up window, even if there is room for another line in the CellTip pop-up window.

CAUSE

The CellTip window has a limited width, and it uses a proportional spaced font to set the width. The line will appear to be truncated if a string of characters that does not include a space or carriage return exceeds that width.

The number of characters allowed on one line depends on which characters are used. For example, you may be able to use 65 'i' characters but only 20 

'w' characters.

RESOLUTION

To work around this problem, insert a space or carriage return in the string of characters. The text will be wrapped where the space or carriage return occurs.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel for Windows 95, version 7.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
