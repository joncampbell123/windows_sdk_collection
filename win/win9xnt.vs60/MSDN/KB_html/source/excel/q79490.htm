Excel: Data Access Macro Does Not Function with System 7.0.1

 Last reviewed: October 6, 1997
Article ID: Q79490
3.00 MACINTOSH kbusage

The information in this article applies to:

  • Microsoft Excel for the Macintosh, version 3.0
  • Macintosh system software version 7.0.1

Summary:

The Data Access Macro (DAM) from Microsoft Excel for the Macintosh version 3.0 will not function properly if you are using System 7.0.1. When the macro attempts to retrieve a list of servers, you will receive the following error message: 

   No valid hosts in DAL driver
There may be other problems associated with running DAM under System 7.0.1 as well.

Microsoft has confirmed this to be a problem in Microsoft Excel for the Macintosh version 3.0. This problem has been corrected in Microsoft Excel for the Macintosh version 4.0.

More Information:

The DAM checks for the System software version to determine where to look for server information and which Data Access Language calls to make. If it encounters System 7.0, it retrieves host information from the DAL Preferences file in the Preferences folder in the System folder. If it encounters a System version other than 7.0, it looks for the Hosts.cl1 file in the System folder. System 7.0.1 is not recognized as a version of System 7.0 and is therefore is treated as System version 6.x.
