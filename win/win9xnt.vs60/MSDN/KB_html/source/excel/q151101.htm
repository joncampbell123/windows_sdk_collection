Freeze Panes and Split Work Differently with View Mgr

 Last reviewed: March 31, 1997
Article ID: Q151101
5.00 7.00 WINDOWS kbenv kbtool

The information in this article applies to:

  • Microsoft Excel for Windows, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you use Freeze Panes with View Manager, you can click Remove Split or you can click Unfreeze Panes on the Window menu. If you click Unfreeze Panes, a Split is added. If you click Remove Split, the Freeze Panes are removed.

View Manager is an add-in macro that saves the current window as a view and enables you apply your saved views to see your worksheet data in different formats.

WORKAROUND

When you use Freeze Pane with View Manager, you must click Remove Split to remove the Freeze Pane.

MORE INFORMATION

For more information about View and Freeze panes, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q107600
