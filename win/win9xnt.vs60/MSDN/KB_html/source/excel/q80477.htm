Excel: Worksheet1 Created with Document Opened from Finder

 Last reviewed: November 4, 1994
Article ID: Q80477
Summary:

When an Excel document is double-clicked to launch Excel from the Finder, Excel does not create a blank worksheet along with opening the document; however, if the Add-In Manager is copied or moved to the Excel Startup Folder, it will load when Excel is launched and will open a blank worksheet when Excel is launched from a document.

More Information:

Other macros will have this same effect if they execute the NEW() function in an auto-open macro and are stored in the Excel Startup folder.
