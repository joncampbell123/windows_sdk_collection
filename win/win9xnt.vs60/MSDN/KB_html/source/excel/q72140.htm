Rotated Text Prints Black Boxes if No Vector Fonts Installed

 Last reviewed: November 1, 1994
Article ID: Q72140

SUMMARY

If rotated text is placed on a chart or a worksheet in Microsoft Excel version 3.00, and the Modern, Roman, and Script vector fonts are not installed in Windows, the rotated text will print as black boxes.

MORE INFORMATION

If your rotated text is printing incorrectly (as black boxes), check to see whether the vector fonts are installed by going into the Control Panel and selecting Fonts. Look for the following listings in the list box: 

   Modern (All Res)
   Roman  (All Res)
   Script (All Res)
If you do not see these fonts listed, choose Add. In the Font Filename box that appears, type the correct path to your Windows SYSTEM subdirectory. All of the font files in that directory will be listed in the list box. Choose MODERN.FON, ROMAN.FON, and SCRIPT.FON, and choose OK after each.
