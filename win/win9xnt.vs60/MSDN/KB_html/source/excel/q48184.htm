Excel: Charting Multiple Selection Yields Incorrect Results

 Last reviewed: October 31, 1994
Article ID: Q48184

SUMMARY

When you create a chart in Microsoft Excel from a multiple selection in a worksheet, be sure to select the entire first range BEFORE holding down the COMMAND key.

If the COMMAND key is held down when the first row or column of data is selected, Excel plots the cell(s) that were highlighted before beginning to select the first row or column of data as the first series.
