Excel: Aligning Regular Number Formats with Dollar Formats

 Last reviewed: November 2, 1994
Article ID: Q23378

SUMMARY

In Microsoft Excel, to align regular number formats with dollar formats, do the following:

  1. Select the cells with the regular number format.

  2. From the Format menu, choose Number and select the dollar format.

  3. In the Format box in the Number dialog box, delete the dollar signs. This procedure actually creates a new format that now will be listed in the Number dialog box for that document.

  4. Click OK or Apply.

This new number format can be applied to all numbers to be lined up with dollar formatted cells.
