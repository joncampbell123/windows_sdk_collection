Excel: Cannot Use Attached Text to Add Text to Overlay Axis

 Last reviewed: November 29, 1994
Article ID: Q63313
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SUMMARY

The following is for versions of Microsoft Excel previous to version 3.0. It is possible to attach text to the overlay axis in Microsoft Excel versions 3.0 and 4.0 using the Attach Text command.

In Microsoft Excel, the Attach Text command on the Chart menu cannot be used to add text to an overlay axis.

To add text to an overlay axis, use unattached text. To apply unattached text to an overlay axis, do the following:

  1. Make sure that no object in the chart is selected.

  2. Type the desired text.

  3. Press the ENTER key or click the check box in the formula bar to enter the text.

  4. Position the text on the chart by first clicking the text, and then dragging it to the desired position.
