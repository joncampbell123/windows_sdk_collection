XL5: Chart Sheet Tab Contains Filename Extension When Opened

 Last reviewed: September 12, 1996
Article ID: Q112032
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel, when you open a Microsoft Excel version 4.0, 3.0, or 2.1 chart file, the chart sheet tab contains the filename extension as well as the filename.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel for Windows version 5.0c.

MORE INFORMATION

When you open a file in Microsoft Excel, the file appears with at least one sheet tab. Each sheet tab contains a name such as "Sheet1", "Chart1", "Module1", and so on. Sheet names normally do not contain a filename extension. However, when you open a chart file, (*.XLC), the entire chart filename, including the filename extension, appears on the chart tab.

Even if you rename a chart sheet in Microsoft Excel, to a name without a filename extension, when you save the file to the Microsoft Excel version 4.0, 3.0, or 2.1 Chart file format, and open the file again, the entire chart filename appears on the chart sheet tab again.

Note that if you open a Microsoft Excel version 4.0 Workbook that contains a chart, the chart tab does not contain a filename extension.
