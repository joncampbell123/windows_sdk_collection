XL: QPW Default Column Width Not Converted Properly

 Last reviewed: January 10, 1997
Article ID: Q136223
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you open a Quattro Pro for Windows version 5.0 file in Microsoft Excel, the default column width will be different than the original document.

CAUSE

There is no direct conversion from Quattro Pro for Windows version 5.0 points (or twips) to the Microsoft Excel character-width measuring system.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

For additional information on column widths, please see the following article in the Microsoft Knowledge Base:
