XL98: Button Caption Text May Use a Bold Font

 Last reviewed: February 2, 1998
Article ID: Q177827
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you open a workbook that contains a worksheet that contain buttons in Microsoft Excel 98 Macintosh Edition, some of the button captions may be formatted in a bold font.

CAUSE

This problem may occur when you create the workbook in Microsoft Excel for Windows and add the buttons to the worksheet in Microsoft Excel for Windows.

NOTE: This problem does not occur when you open a workbook in Microsoft Excel for Windows. This problem also does not occur if you create the buttons in Microsoft Excel for the Macintosh or if you create the workbook in Microsoft Excel for the Macintosh.

RESOLUTION

To resolve this problem, manually reformat the button captions as required. To select a button, hold down the OPTION key and click the button. You can then select the caption text and either click the Bold tool on the Formatting toolbar or click Control on the Format menu.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
