XL7: Can't Create Charts in File Saved with Multi-User Editing

 Last reviewed: March 27, 1997
Article ID: Q133053
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel for Windows 95, version 7.0, when a file is saved with the Allow Multi-User Editing option turned on, you cannot create a chart in that file.

WORKAROUND

The only way to be able to insert the chart into the file is to resave the file with the Allow Multi-User Editing option turned off.

WARNING: Resaving the file with the Allow Multi-User Editing option turned off will prevent other users from being able to save any changes they have made to the file.

MORE INFORMATION

The Allow Multi-User Editing feature can be turned on to make the file a shared list. This will allow multiple users to access and make changes to the file concurrently.

REFERENCES

For more information about shared lists, Select the Find tab in Help and type: 

   Create a Shared List
