XL5: Find File Command or Button Dimmed

 Last reviewed: September 12, 1996
Article ID: Q104406
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

The Find File feature in Microsoft Excel version 5.0 allows you to search for documents, preview version 5.0 files, and open, print, move, and delete multiple files simultaneously.

To use this feature, choose the Find File command from the File menu or choose the Find File button in the Open, Insert Object, or Insert Picture dialog box. The Find File command and the Find File buttons may be unavailable (dimmed) when your machine has little available memory. To enable the command and buttons, free up memory by closing applications and files that you are not currently using.
