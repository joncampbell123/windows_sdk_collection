Excel: EXCEL4.INI Settings for Size of Toolbar Tools

 Last reviewed: September 12, 1996
Article ID: Q101844
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a

In Microsoft Excel version 4.0 for Windows, you can change toolbar sizes by changing the button style (BtnStyle) setting in the EXCEL4.INI file.

The button style (BtnStyle) statement in the EXCEL4.INI can take five settings. This statement should be added under the [Microsoft Excel] section, as in the following example: 

   [Microsoft Excel]
   BtnStyle=4
The five settings available for this statement are as follows: 

   0     For Standard tools, default except for 9-inch Macintosh
         screens.
   1     For Small tools used for 9-inch Macintosh screens (not 3
         dimensional).
   2     For Minimal three-dimensional tools.
   3     For Minimal tool face only.
   4     For Small versions of the Standard tools.
As a general guide, setting 3 will give you the smallest toolbar, setting 2 is the next smallest, setting 1 is the next smallest after that, and setting 4 is the next smallest. To set tools back to the standard size, use setting 0 or remove the BtnStyle= line from the EXCEL4.INI file.
