Excel: Can't Insert Page Break with Row or Column Selected

 Last reviewed: November 30, 1994
Article ID: Q81273
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 2.x, 3.0, 4.0, 5.0

SUMMARY

If the active cell within a selected column is below a horizontal page break, or if the active cell within a selected row is to the right of a vertical page break, then the Page Break command on the Options menu is Remove Page Break instead of Set Page Break.

To set a page break under these conditions, press TAB to move the active cell within the selection away from the existing page break, then choose the Set Page Break command from the Options menu.

REFERENCES

"Microsoft Excel User's Guide," for the Macintosh, version 3.0, pages 529-530
