SBFM: Contents of the Additional Drivers Disk

 Last reviewed: October 9, 1996
Article ID: Q154662
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Small Business Financial Manager for Microsoft Excel, version 1.0

SUMMARY

This article lists the files on the Additional Driver disk for the Microsoft Small Business Financial Manager for Microsoft Excel (SBFM).

MORE INFORMATION

The drivers on the disk allow Microsoft Excel to import data from the following third-party programs: 

   Businessworks for MS-DOS and Windows, version 9.0
   M A S 90 Evolution/2 for MS-DOS, version 1.5
   Platinum Series for MS-DOS and Windows, version 4.1*
   Quickbooks for Windows, version 3.1
*The Platinum Series driver does not support importing data from Platinum for MS-DOS, version 4.1a, an update to version 4.02 for MS-DOS. In addition, the driver does not support importing data from Platinum for MS- DOS and Windows, version 4.3, an update to version 4.1 for MS-DOS and Windows.

The Additional Drivers disk contains the following files: 

   Aapbww.dll
   Aapmas90.dll
   Aapplat.dll
   Aapqbw.dll
   Msaap.dll
   Qbdrivhk.dll
   Readme.txt
For information about how to install these drivers, please read Readme.txt on the disk.

NOTE: The Peachtree 3.5 for Windows filter is also available online. This file did not ship with the Additional Drivers disk.
