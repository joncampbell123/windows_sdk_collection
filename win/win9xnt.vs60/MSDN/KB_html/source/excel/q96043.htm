Excel: Incorrect Margin Settings in Lotus WK3,FM3 File

 Last reviewed: September 12, 1996
Article ID: Q96043
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0a

SYMPTOMS

The left, right, top, and/or bottom margins shown in the File Page Setup dialog box may be incorrect when a file in Lotus WK3,FM3 format is opened in Microsoft Excel. In fact, the margins may be highly irregular, ranging from 0 to 65.535 inches. As a result of the irregular sizes, the error message "Margins do not fit page size" will display when you choose Print or Print Preview from the File menu. After you choose the OK button in response to this message, Print Preview or the printed output will show multiple blank pages.

WORKAROUND

To reset the margins to Microsoft Excel's defaults, do the following:

  1. From the File menu, choose Page Setup.
2. In the Left and Right Margin boxes, type .75. 3. In the Top and Bottom Margin boxes, type 1. 4. Choose the OK button.

You must save the file in Microsoft Excel's Normal file format in order for the correct margin settings to be retained. To do so:

  1. From the File menu, choose Save As.
2. In the Save File As Type box, select Normal. 3. Choose the OK button.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. We are researching this problem, and will post more information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

This problem does not occur in Microsoft Excel version 4.0 and will occur in version 4.0a only when an Impress FM3 formatting file is loaded along with the Lotus WK3 worksheet file.

REFERENCES
