Excel: Add-in Files and Their Associated Functions

 Last reviewed: September 12, 1996
Article ID: Q106464
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0, 4.0a

SUMMARY

If you receive the #NAME? error value after you enter a function, you should verify that the correct add-in file is loaded. To do this, use the following list to locate and reopen the associated file.

The following is a complete list of all of the Microsoft Excel version 4.0 and 4.0a add-in functions and command macros, and their associated filenames and default locations.
