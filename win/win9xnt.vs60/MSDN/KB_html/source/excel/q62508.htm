Excel: Printing Range of Pages with Altered Page Numbers

 Last reviewed: November 2, 1994
Article ID: Q62508

SUMMARY

In Microsoft Excel version 2.20, page numbering can begin at any number. This can be specified in the header or footer in the Page Setup dialog box (on the File menu) by adding the increased amount to &P. For example, to begin page numbers at 100, the header or footer would contain &P+99.

To print a range of these records, the actual number of pages in the document must be referenced rather than the numbers related to the header/footer settings.

For example, consider a situation where the page numbering is set to start at 100 (&P+99) and the document is five pages long. To print only the third page (page 103) of the document, From 3 To 3 must be specified in the Print dialog box (on the File menu). In this example, the third page of the document prints and the page number reads "103".

This also is true when a print area has been set and a range of pages within the print area is to be printed. To print the second page of the print area, From 2 To 2 must be entered in the Print dialog box.

For more information on alternative page numbering, see page 532 of the "Microsoft Excel User's Guide" version 3.0 manual. If you are using Excel 2.20, see page 390 of the "Microsoft Excel Reference" version 2.2 manual.
