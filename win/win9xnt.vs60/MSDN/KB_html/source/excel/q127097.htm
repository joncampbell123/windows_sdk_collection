Microsoft Excel 5.0c Disk Contents or .CAB Files

 Last reviewed: September 12, 1996
Article ID: Q127097
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0c

SUMMARY

The following table lists the disk contents for the cabinet files of the 3.5-inch version of Microsoft Excel 5.0c for Windows. On each disk from disk 2 through 7, there is only one cabinet file. Disk 1 contains the files needed to install Microsoft Excel.
