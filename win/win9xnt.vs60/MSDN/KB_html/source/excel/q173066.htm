XL: Screen Refresh Problem After Inserting Multiple Page Breaks

 Last reviewed: August 26, 1997
Article ID: Q173066
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SYMPTOMS

After inserting multiple page breaks into a Microsoft Excel worksheet, the lines that indicate the vertical page breaks may disappear.

CAUSE

This problem may occur when both of the following conditions are true:

  • You insert multiple vertical page breaks in the same window of a worksheet.

    -and-

  • You insert a page break in the rightmost column, causing the worksheet to scroll to the right.

WORKAROUND

To work around this problem, click Page Setup on the File menu, and then click OK in the Page Setup dialog box. When the display refreshes, the vertical page break lines should reappear.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel 97.
