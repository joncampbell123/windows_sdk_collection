XL: Formats Not Copied From Microsoft Query

 Last reviewed: March 27, 1997
Article ID: Q105551
5.00 7.00 WINDOWS kbother kbprb kbtool

The information in this article applies to:

  • Microsoft Excel for Windows, version 5.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Query for Windows, version 1.0
  • Microsoft Query for Windows 95, version 2.0

Any formatting you apply to data in Microsoft Query will be lost when you copy this data to the Clipboard.

This is by design.

You can use Microsoft Query, an application included with Microsoft Excel, to retrieve, view, and organize data from a variety of sources. Microsoft Query is not a database application; it is a database "front-end" used to view data from various sources. If you want to bring data from Microsoft Query into Microsoft Excel, you will have to reformat the retrieved data from within Microsoft Excel.

REFERENCES

"User's Guide," version 5.0, Chapter 23, "Retrieving Data from Lists and External Databases Using Microsoft Query," page 449
