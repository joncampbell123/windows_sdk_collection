Excel: Iteration Setting Not Saved in Workspace File

 Last reviewed: November 30, 1994
Article ID: Q79543
Summary:

The "Microsoft Excel User's Guide," version 3.0, page 548, incorrectly lists Iteration as a setting that is saved in a Workspace file.

Although the Automatic/Manual calculation setting is saved, the Iteration setting is not saved.

Reference(s):

"Microsoft Excel User's Guide," version 3.0, page 548
