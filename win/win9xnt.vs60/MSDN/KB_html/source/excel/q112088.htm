XL5: Cannot Select More Than 23 Files in Open Dialog Box

 Last reviewed: June 19, 1997
Article ID: Q112088
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

In Microsoft Excel 5.0 for Windows, in the Open dialog box, you can select several files to open simultaneously. The maximum number of files that can be selected at one time is 23.

MORE INFORMATION

To select more than one file to open, choose Open from the File menu and do either of the following:

  1. Select the first file

  2. Press the SHIFT key and select the last file.

The files listed between the first and last files will be highlighted (in addition to the first and last files). If you attempt to select more than 23 files, the selection will be reduced to just the first file.

-or-

  1. Select the first file.

  2. Press and hold down the CTRL key and click the other files to be opened.

This method allows you to select nonadjacent files from the file list. If you attempt to select more than 23 files, only the first 23 files will remain selected
