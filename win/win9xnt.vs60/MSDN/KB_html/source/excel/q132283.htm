XL: Not All Database Sheets Listed in Template Wizard

 Last reviewed: June 3, 1997
Article ID: Q132283
7.00 97 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel, when you use the Template Wizard to create and link a template to an existing database file, the Sheet list in the Template Wizard - Step 4 of 5 dialog box may not contain all of the worksheets in your database file. Because of this, you cannot link cells in your template file to every sheet in the existing database file.

CAUSE

This behavior occurs because you can only link data in a template file to the first 25 worksheets in an existing database file. However, when you create a template using the Template Wizard, and you link the template to a database file that contains more than 25 worksheets, you do not receive an error message indicating that this limit exists and that not all of the worksheets in your database will appear in the Sheet list.

WORKAROUND

To work around this behavior, make sure that the worksheets that you want to use in the database are the first 25 worksheets in the database file. If your database contains more than 25 worksheets, you must move the additional worksheets to a new database file in order to link these to a template file using the Template Wizard.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about how to create a template using the Template Wizard, click the Index tab in Microsoft Excel Help, type the following text 

   Template
double-click the selected text, and then double-click "Create a template to export worksheet data to a database" to go to the "Create A Template To Export Worksheet Data To A Database" topic.
