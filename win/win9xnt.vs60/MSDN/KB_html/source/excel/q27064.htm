Excel: Using Exponentiation

 Last reviewed: November 30, 1994
Article ID: Q27064
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 1.x, 2.x, 3.0, 4.0
  • Microsoft Excel for OS/2, versions 2.2 and 3.0

SUMMARY

In Microsoft Excel, the symbol for exponentiation is a caret (^), which is created by pressing SHIFT+6 on the keyboard.

For example, to enter the formula =10*10, you could type =10^2.
