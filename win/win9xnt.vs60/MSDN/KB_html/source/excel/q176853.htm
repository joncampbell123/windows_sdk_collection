XL7: Cannot Print a Bold Chart Y-Axis Under Microsoft Windows NT

 Last reviewed: December 2, 1997
Article ID: Q176853
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Windows NT Workstation versions 3.51, 4.0

SYMPTOMS

When you print a chart in Microsoft Excel 7.0, the y-axis may not be printed in bold.

CAUSE

This problem may occur if the following conditions are true:

  • You are using Microsoft Windows NT

    -and-

  • The chart is on a chart sheet.

    -and-

  • You format the y-axis with a bold font style.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The y-axis is not printed with the bold font style even though the font style looks bold in the chart, appears as a bold font style in the Font tab of the Format Axis dialog box, and appears in print preview.
