XL: 4.0 Menus Different From Actual 4.0 Menus

 Last reviewed: September 13, 1996
Article ID: Q107975
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a

SUMMARY

There are minor differences between the Microsoft Excel 4.0 menus used in Microsoft Excel versions 5.0 and 7.0, and the actual 4.0 menus (the ones that are used in Microsoft Excel version 4.0).

The differences between the menus are as follows: 

   MENU
      Command      Version 4.0 menus in 5.0, 7.0  Version 4.0 menus
   ----------------------------------------------------------------
   FILE
      Save         No hot key                     SHIFT+F12
      Save As      No hot key                     F12

   EDIT
      Clear        No hot key                     DEL

   OPTIONS
      Group Edit   Not present                    Present
      New Menus    Present                        Not present

   APPLICATION (upper-left corner of Microsoft Excel window)
      Run          Not present                    Present

   DOCUMENT (upper-left corner of document window)
      Next Window  CTRL+TAB                       CTRL+F6
      Split        Not present                    Present
Note that this table does not include menu items that are added by add-in macros, such as Print Report, Views, and Crosstab.

Note also that you have the same functionality regardless of which menu you use: it is still possible to perform all of the same tasks using the Microsoft Excel 4.0 menus, but you may have to use a different method to perform the task.

For example, to use Group Edit in Microsoft Excel versions 5.0 and 7.0, you select two or more sheet tabs at the bottom of the screen, rather than using a dialog box as in Microsoft Excel version 4.0.

MORE INFORMATION

In Microsoft Excel versions 5.0 and 7.0, you can use Microsoft Excel version 4.0 menus. To do switch to the 4.0 menus, do the following:

  1. From the Tools menu, choose Options.

  2. Select the General tab.

  3. Select the Microsoft Excel 4.0 menus check box.

  4. Choose OK to accept the change.

To return to the standard Microsoft Excel menus, choose New Menus from the Options menu.
