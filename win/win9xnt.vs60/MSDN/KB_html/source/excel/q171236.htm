XL97: Opening Workbook in Internet Explorer Does Not Open Viewer

 Last reviewed: January 9, 1998
Article ID: Q171236
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 97 Viewer for Windows

SYMPTOMS

If Microsoft Excel 97 and Microsoft Excel 97 Viewer are installed on the same computer, when you click a hyperlink to open a workbook in Microsoft Internet Explorer, Microsoft Excel 97 is opened in Internet Explorer instead of in Microsoft Excel 97 Viewer.

This behavior is different from the behavior of Microsoft Excel Viewer for Windows 95. When you click a hyperlink, Microsoft Excel Viewer for Windows 95 is opened in Microsoft Internet Explorer.

CAUSE

Microsoft Excel 97 Viewer is not registered as the server program; Microsoft Excel Viewer for Windows 95 is registered as the server program.

WORKAROUND

To open Microsoft Excel 97 Viewer in Internet Explorer, start the viewer before you click any hyperlinks.

STATUS

This behavior is by design of Microsoft Excel 97 Viewer.
