SBFM: Quickbooks Driver Currently Supports Version 3.1 Only

 Last reviewed: March 19, 1997
Article ID: Q155458
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Small Business Financial Manager for Microsoft Excel, version 1.0

SUMMARY

The box for Microsoft Small Business Financial Manager version 1.0 for Microsoft Excel (SBFM 1.0), lists Intuit Quickbooks for Windows 3.x as a supported accounting system. This is incorrect. SBFM 1.0 supports only Quickbooks for Windows version 3.1.

MORE INFORMATION

The latest version of Intuit Quickbooks for Windows is version 5.0. At this time, the SBFM 1.0 driver for Quickbooks for Windows supports only version 3.1 of Quickbooks for Windows. A Quickbooks 4.0 driver is available for use with SBFM 97.
