Internal Excel Error, XL with HP NewWave's Edit Share Command

 Last reviewed: November 2, 1994
Article ID: Q73365

SUMMARY

If you are using Hewlett-Packard (HP) NewWave Office and Microsoft Excel 3.0, and you select an embedded chart and then choose Share from the Edit menu, the following error message appears 

   Internal Excel error, XL###
where the # signs are numbers. This is not an Excel error. The error comes from HP NewWave.

MORE INFORMATION

The Edit Share command is available in Excel only when you are using HP NewWave. This command copies to the Clipboard a shared view of what you have selected. However, this command does not work properly with an embedded chart; it must be a separate chart for the command to work correctly. To create the separate chart, double-click the embedded chart with your mouse. You can now choose Share Chart from the Edit menu.

For more information regarding this error message, contact Hewlett-Packard Post Sales Technical Support at (800) 752-0900.

REFERENCES

"Microsoft Excel User's Guide." Version 3.0, Appendix D, "Using Microsoft Excel with HP NewWave," pages 705-714.
