Excel: Tutorial Does Not Launch Properly with Hypercard 2.0

 Last reviewed: January 23, 1995
Article ID: Q78220
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0, 5.0

SUMMARY

On a Macintosh with Hypercard version 2.0 installed, if you double-click the Excel Tutorial stack in Microsoft Excel for the Macintosh, Hypercard launches, but the tutorial does not.

This problem is specific to Hypercard version 2.0. You can run the tutorial by first launching Hypercard and then opening the tutorial with the Open Stack command from the Hypercard File menu.

MORE INFORMATION

With version 2.0 or other versions of Hypercard, even if the program exists on the disk, double-clicking the tutorial stack may produce the following message: 

   Application Hypercard is Busy/Missing...
In this case, rebuild the desktop on the drive by holding down COMMAND+OPTION while you restart the Macintosh. The tutorial should launch if you have a version Hypercard other than 2.0.
