Module Font Reset to "Fixedsys" When You Quit Microsoft Excel

 Last reviewed: September 12, 1996
Article ID: Q129510
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0

SYMPTOMS

If you change the Module Font in the Options dialog box (from the Tools menu, choose Options), the font may be changed to "FixedSys" when you exit and restart Microsoft Excel. This behavior may occur with, but is not limited to, the MS Line Draw and LinePrinter fonts.

WORKAROUND

To work around this problem, do either of the following when you set the Module Font in the Module Format tab of the Options dialog box:

  • Use a font that is being reset. To do this, change the Module Font in the Options dialog box each time you start Microsoft Excel.

    -or-

  • Use a Module Font that is not replaced by "FixedSys."

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
