XL: Microsoft Excel Viewer Does Not Display Formulas

 Last reviewed: February 2, 1998
Article ID: Q143347

The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, version 2.x, 3.0, 4.0, 5.0, 5.0c
  • Microsoft Excel for the Macintosh, version 4.0, 5.0, 5.0a

SUMMARY

When you use Microsoft Excel Viewer, formulas are not displayed. Only values are displayed.

MORE INFORMATION

Microsoft Excel Viewer gives you the ability to view Microsoft Excel spreadsheets even if you don't have Microsoft Excel installed on your computer. The Microsoft Excel Viewer is not designed to display formulas; it can only show their corresponding values.
