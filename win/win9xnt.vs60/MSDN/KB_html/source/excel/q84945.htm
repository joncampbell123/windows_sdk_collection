Corrupt EXCEL.XLB File May Cause General Protection Fault

 Last reviewed: November 22, 1994
Article ID: Q84945

The information in this article applies to:

  • Microsoft Excel for Windows, version 4.0

SYMPTOMS

When you attempt to start Microsoft Excel for Windows under Microsoft Windows operating system version 3.1, the following error message may be displayed if the EXCEL.XLB file is corrupt: 

   Application Error

   EXCEL caused a General Protection Fault in module EXCEL.EXE at
   0015:XXXX.
If the EXCEL.XLB file is corrupt and Microsoft Excel is started under Windows version 3.0, the error message is: 

   UNRECOVERABLE APPLICATION ERROR

   Terminating current application.

RESOLUTION

To correct this problem, do the following:

  1. Quit Windows.

  2. Rename or delete the EXCEL.XLB file in the WINDOWS directory. All custom settings for the Microsoft Excel tool bar will be lost.

  3. Restart Windows, and then restart Microsoft Excel.

  4. Quit Microsoft Excel. The EXCEL.XLB file will be recreated with the default settings.

MORE INFORMATION

The layout of the toolbar in Microsoft Excel is saved in the EXCEL.XLB file, which is located in the WINDOWS directory. If this file becomes corrupt or damaged, one of the above error messages may be displayed.
