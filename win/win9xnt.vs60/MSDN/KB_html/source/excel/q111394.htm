Excel: Opening File with Reference to Add-In Gives Error

 Last reviewed: September 12, 1996
Article ID: Q111394
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0

SUMMARY

When you open a worksheet in Microsoft Excel, you may receive the error message 

   Update references to unopened documents?
even when the worksheet does not appear to be linked to another worksheet.

MORE INFORMATION

In Microsoft Excel, it is possible to link worksheets so that data in one worksheet (or macro sheet) will appear in another worksheet. When a function provided by an add-in is referenced by a worksheet, a link to the add-in document is created in the worksheet. If the worksheet is opened and the add-in providing the function is not available, Microsoft Excel displays the error message "Update references to unopened documents?"

To work around this problem, either open the add-in document before opening the worksheet or add the add-in document to the startup list by choosing Add-ins from the Options menu and selecting the appropriate add-in macro.
