XL: Chart Text Cut Off When Printed or Viewed in Print Preview

 Last reviewed: February 5, 1998
Article ID: Q179806
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In the versions of Microsoft Excel listed at the beginning of this article, if you print or view a chart in print preview, and the chart is embedded in a worksheet, some of the text in the chart may be cut off, even though it appears correctly on the screen.

CAUSE

This problem may occur if the chart contains any text boxes.

NOTE: This problem does not occur when you print or view a chart in a chart sheet in print preview.

RESOLUTION

To resolve this problem, repeat the following steps for each text box in the chart:

  1. Select the text box by clicking its border.

    If the blinking cursor appears in the text box, press ESCAPE once.

  2. On the Format menu, click Text Box. Click the Properties tab.

  3. Click Size With Chart. Click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
