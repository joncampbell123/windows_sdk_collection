Excel Err Msg: 'REGRESSION: Ranges Cannot Overlap'

 Last reviewed: November 30, 1994
Article ID: Q89504
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 4.0, 5.0

SUMMARY

Using the Regression Tool, you will receive the message "REGRESSION: Ranges Cannot Overlap" if the ranges specified for the Summary, Residual, or Probability Output tables do not have the proper amount of column space necessary for the tables to be written.

MORE INFORMATION

Page 599 in the "User's Guide" for version 5.0 suggests the following guidelines:

  • Arrange output ranges side by side on your worksheet because output tables generally vary in length rather than width.
  • 2 columns for the probability data output table
  • 4 columns for the residuals output table

Page 42 of the "User's Guide 2" for version 4.0 specifies the following space recommendations for each output table:
  • 7 columns for the summary output table
  • 2 columns for the probability data output table
  • 4 columns for the residuals output table

REFERENCES

"User's Guide," version 5.0, page 599 "User's Guide 2," version 4.0, page 42
