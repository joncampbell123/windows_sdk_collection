Excel: Number Appears at the Top of a Cell with Text Wrap

 Last reviewed: November 3, 1994
Article ID: Q79485
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 3.0

Summary:

With Microsoft Excel for the Macintosh, text and numbers in cells formatted with Wrap Text incorrectly appear at the top of cells. However, if a formula that calculates a number as its result is in a cell formatted with Wrap Text, the number may correctly appear at the bottom of the cell.

This problem becomes apparent when the height of the row containing the formula is increased and the value of the formula is changed several times.

Microsoft has confirmed this to be a problem in Microsoft Excel for the Macintosh version 3.0. This problem was corrected in Microsoft Excel version 4.0.
