Excel: How to Resize Data Markers on Charts

 Last reviewed: February 2, 1998
Article ID: Q97589

The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0, 5.0
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

Data markers on xy (scatter), line, and radar charts have a fixed size in all versions of Microsoft Excel (although alternate patterns and styles are available). However, there is a technique you can use to replace the current data markers with objects of a different size.

MORE INFORMATION

To replace the existing data markers with different objects, follow these steps:

  1. Create a chart.

  2. Create the object you want to use as your data marker on your chart. You can use the drawing tools on the Drawing toolbar to do this.

  3. Click the object and click Copy on the Edit menu.

  4. If the chart is embedded on the worksheet, activate the chart by double-clicking it. If the chart is in a chart file, activate it by clicking the name of the chart on the Window menu.

  5. Select the series you want to change by selecting the data marker on the chart. On the Edit menu, click Paste.

The selected data marker and series are replaced with the copied object.

NOTE: The objects you create can be any size or pattern, including text edit boxes. The chart legend is automatically updated to show the new data marker. Note also that the above method does not work for line or radar charts that do not have data markers.
