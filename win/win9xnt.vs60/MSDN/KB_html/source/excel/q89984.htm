Excel: All Views Have the Same Paper Size

 Last reviewed: November 29, 1994
Article ID: Q89984
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

When printing views of a spreadsheet using Print Reports, all of the views print on the same size paper, regardless of the size of the paper selected when the view was created.

MORE INFORMATION

All of the views in a document have the paper size of the last view added in the View Manager. This results because Microsoft Excel add-ins cannot get or set paper size. However, other options set in Page Setup (margins, orientation, scaling, and page order) are saved with each view.

REFERENCES

"Microsoft Excel User's Guide 1," version 4.0, pages 255-256
