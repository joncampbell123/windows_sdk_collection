History of Windows ODBC Drivers Available from Microsoft

 Last reviewed: July 21, 1997
Article ID: Q143349
The information in this article applies to:
  • Microsoft Open Database Connectivity, versions 1.0, 2.0, 3.0, 3.5
  • Microsoft Access, version 2.0
  • Microsoft Access for Windows 95, version 7.0
  • Microsoft Access 97 for Windows
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Office for Windows, versions 4.2, 4.2c
  • Microsoft Office Professional for Windows, versions 4.3, 4.3c
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office Professional for Windows 95, version 7.0
  • Microsoft Office 97 for Windows, Professional Edition
  • Microsoft Office 97 for Windows, Standard Edition

SUMMARY

This article lists the various versions of the Microsoft Open Database Connectivity (ODBC) drivers, the products they are shipped with, and how to obtain them.

ODBC Drivers Shipped with Microsoft Excel 5.0

The following drivers are shipped with Microsoft Excel, versions 5.0 and 5.0c for Windows: 

   Driver Name     Supported Database Versions   Driver Version
   -------------   ---------------------------   --------------
   Access Data     1.0, 1.1                      1.01.2115
   dBASE Files     3.0, 4.0                      1.01.2115
   FoxPro Files    2.0, 2.5                      1.01.2115
   Paradox Files   3.0, 3.5                      1.01.2115
   SQL Server      1.11 to 4.21a, Sybase 4.2     1.02.3231
The Microsoft SQL Server driver shipped with Microsoft Excel, versions 5.0 and 5.0c, uses the SQLSRVR.DLL file. All other ODBC 1.0 drivers listed above use the SIMBA.DLL database engine.

A 32-bit version of the Microsoft SQL Server driver (SQLSRV32.DLL, version 2.00.1912) is shipped with Microsoft Excel 5.0 for Windows NT.

The Microsoft Access 2.0 Driver

The following version of the Microsoft Access 2.0 driver (ODBCJT16.DLL) is shipped with Microsoft Access, version 2.0: 

   Driver Name                Supported Database Versions   Driver Version
   ------------------------   ---------------------------   --------------
   Access 2.0 for MS Office   1.0, 1.1, 2.0                   1.00.1521
This driver can only be used if Microsoft Access is installed on the user's computer. See the "References" section at the end of this article for more details. Another version of this driver is available in the ODBC 2.0 Drivers Kit, which is discussed later in this article.

The ODBC 1.0 Drivers Kit

NOTE: The ODBC 1.0 Kit is no longer available from Microsoft. Instead, you should order the ODBC 2.0 Desktop Database Drivers Kit mentioned below.

The Microsoft ODBC Desktop Database Drivers Kit, version 1.0 includes the drivers mentioned above, as well as the following additional drivers: 

   Driver Name    Supported Database Versions   Driver Version
   ------------   ---------------------------   --------------
   Btrieve Data   5.1                           1.01.2115
   Excel Files    3.0, 4.0                      1.01.2115
   Oracle         6.0.34 or later               1.00.3112
   Text Files     n/a                           1.01.2115
The ORACLE driver uses the SQORA.DLL file. All other ODBC 1.0 drivers listed above use the SIMBA.DLL database engine.

The ODBC 2.0 Drivers Kit

The Microsoft ODBC Desktop Database Drivers Kit, version 2.0, may be obtained by contacting Microsoft at (800)-360-7561. The part number for this kit is 273-054-030 and includes the following 16-bit and 32-bit ODBC drivers: 

   Driver Name                Supported Database Versions   Driver Version
   ------------------------   ---------------------------   --------------
   Microsoft Access Driver     1.0, 1.1, 2.0                 2.00.2317
   Microsoft dBASE Driver      3.0, 4.0                      2.00.2317
   Microsoft Excel Driver      3.0, 4.0, 5.0                 2.00.2317
   Microsoft FoxPro Driver     2.0, 2.5, 2.6                 2.00.2317
   Microsoft Paradox Driver    3.5, 4.0, 4.5                 2.00.2317
   Microsoft Text Driver       n/a                           2.00.2317
   SQL Server                  1.2 to 4.21a, Sybase 4.2      2.00.1912
The Microsoft SQL Server driver listed above uses SQLSRVR.DLL (16-bit) or SQLSRV32.DLL (32-bit). All other ODBC 2.0 drivers listed above use the ODBCJT16.DLL (16-bit Jet engine) file or the ODBCJT32.DLL (32-bit Jet engine) file.

The ODBC 2.0 Kit also includes 16-bit only versions of the following drivers: 

   Driver Name                Supported Database Versions   Driver Version
   ------------------------   ---------------------------   --------------
   Microsoft Btrieve Driver    5.1, 6.x                      2.00.2317
   Oracle                      6.0.34 or later               1.00.3112
   Oracle7                     7.0                           1.11.0000
The Btrieve driver uses the ODBCJT16.DLL file and also requires the WBTRCALL.DLL file, which must be obtained from Btrieve Technologies, Inc.

The ORACLE driver uses the SQORA.DLL file and the ORACLE7 driver uses the SQORA7.DLL file.

ODBC 3.0 Drivers

The Microsoft ODBC 3.0 drivers are 32-bit only and are designed for use by ODBC-compliant, 32-bit applications running under Microsoft Windows 95 or Microsoft Windows NT 3.51.

NOTE: Sybase SQL Server is not supported by Microsoft SQL Server driver 2.50 and later.

The following ODBC 3.0 drivers shipped with Microsoft Office Standard for Windows 95, version 7.0 and Microsoft Excel for Windows 95, version 7.0. These drivers use version 2.99 of the Microsoft Jet database engine (ODBCJT32.DLL). Btrieve and ORACLE drivers are not included with this release: 

   Driver Name                Supported Database Versions   Driver Version
   ------------------------   ---------------------------   --------------
   Microsoft Access Driver     1.0, 1.1, 2.0, 7.0            2.99.1731
   Microsoft dBASE Driver      3.0, 4.0, 5.0                 2.99.1731
   Microsoft Excel Driver      3.0, 4.0, 5.0                 2.99.1731
   Microsoft FoxPro Driver     2.0, 2.5, 2.6                 2.99.1731
   Microsoft Paradox Driver    3.5, 4.0, 4.5, 5.0            2.99.1731
   Microsoft Text Driver       n/a                           2.99.1731
   SQL Server                  4.21a or later                2.50.0121
The following drivers are updated versions of the ODBC 3.0 drivers, which are shipped with Microsoft Office Standard and Office Professional for Windows 95, version 7.0a and 7.0b, and Microsoft Access for Windows 95, version 7.0a and 7.0b: 

   Driver Name                Supported Database Versions   Driver Version
   ------------------------   ---------------------------   --------------

   Microsoft Access Driver     1.0, 1.1, 2.0 7.03.40.2505
   Microsoft dBASE Driver      3.0, 4.0 5.03.40.2505
   Microsoft Excel Driver      3.0, 4.0, 5.0 7.03.40.2505
   Microsoft FoxPro Driver     2.0, 2.5 2.63.40.2505
   Microsoft Paradox Driver    3.x, 4.x 5.03.40.2505
   Microsoft Text Driver       n/a                           3.40.2505
   SQL Server                  4.21a or Later                2.50.0126

ODBC 3.0 Drivers Application Note

In addition, a 32-bit ORACLE 7.0 driver is available through an application note named "Microsoft ODBC Drivers for 32-Bit Programs" (GE1263), which includes the following updated ODBC 3.0 drivers: 

   Driver Name                Supported Database Versions   Driver Version
   ------------------------   ---------------------------   --------------
   Microsoft Access Driver     1.0, 1.1, 2.0, 7.0            3.40.2204
   Microsoft dBASE Driver      3.0, 4.0, 5.0                 3.40.2204
   Microsoft Excel Driver      3.0, 4.0, 5.0, 7.0            3.40.2204
   Microsoft FoxPro Driver     2.0, 2.5, 2.6                 3.40.2204
   Microsoft Paradox Driver    3.5, 4.0, 4.5, 5.0            3.40.2204
   Microsoft Text Driver       n/a                           3.40.2204
   Visigenic 32-bit Oracle     7.x                           1.03.0009
   Driver
   SQL Server                  4.21a or later                2.50.0126
The Visigenic ORACLE driver uses the file VSORAC32.DLL. The SQL Server driver uses SQLSRV32.DLL.

ODBC 3.5 Drivers

The Microsoft ODBC 3.5 drivers are 32-bit only. These drivers are designed for use on Microsoft Windows 95 or later, or Windows NT 3.51 or later. Only 32-bit applications are supported on Windows 95 or later; 16-bit and 32-bit applications are supported on Windows NT 3.51 or later.

NOTE: Sybase SQL Server is not supported by Microsoft SQL Server driver 2.50 and later.

The following ODBC 3.5 drivers shipped with Microsoft Office 97, Standard and Professional editions. These drivers use version 3.50 of the Microsoft Jet database engine(ODBCJT32.DLL). Btrieve and ORACLE drivers are not included with this release: 

   Driver Name                Supported Database Versions   Driver Version
   ------------------------   ---------------------------   --------------
   Microsoft Access Driver     1.0, 1.1, 2.0, 7.0, 97        3.50.3402
   Microsoft dBASE Driver      3.0, 4.0, 5.0                 3.50.3402
   Microsoft Excel Driver      3.0, 4.0, 5.0, 97             3.50.3402
   Microsoft FoxPro Driver     2.0, 2.5, 2.6                 3.50.3402
   Microsoft Paradox           3.5, 4.0, 4.5, 5.x            3.50.3402
   Microsoft Text Driver       n/a                           3.50.3402
   SQL Server                  4.21a or later                2.65.0213
NOTE: New SQL Server drivers are available in the SQL Server 6.5 Service Pack 3. This Service Pack includes version 2.65.0252 of both the Sqlsvr.dll and Sqlsvr32.dll files. If you already have version 2.65.x of these two files, you can copy over them with the newer versions contained in the Service Pack. If you have versions of these 2 files earlier than 2.65.x, you must first upgrade to 2.65.x, and then copy over them with the versions contained in the Service Pack.
