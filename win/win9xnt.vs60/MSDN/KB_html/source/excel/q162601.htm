ActiveX Image Control on a Worksheet Cannot Be Edited

 Last reviewed: February 12, 1998
Article ID: Q162601
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

You cannot edit an ActiveX Image control on a worksheet. The context menu for Image Object is "(Empty)" and cannot be selected.

CAUSE

You cannot select and edit an ActiveX Image control from a worksheet.

MORE INFORMATION

In Microsoft Excel 97, you can add an ActiveX control to a worksheet. The ActiveX controls that you can add to a worksheet are on the Control Toolbox toolbar. To display the Control Toolbox toolbar, point to Toolbars on the View menu, and then click Control Toolbox.

The ActiveX Image control displays a graphical image from a bitmap, icon, or metafile on a form. An image that is displayed in an Image control is only decorative. You cannot edit this image from the worksheet.

You can edit many of the ActiveX controls on the worksheet. For example, you can edit the TextBox control on the worksheet. You can edit the text in the TextBox control from the worksheet by right-clicking the TextBox control, pointing to TextBox Object and clicking Edit.

REFERENCES

For more information about ActiveX controls, click the Index tab in Microsoft Excel Visual Basic Editor Help, type the following text 

   toolbox
and then double-click the selected text to go to the "Toolbox" topic.
