Excel UAEs in Print Preview with Attachmate Extra for DOS

 Last reviewed: November 1, 1994
Article ID: Q71473

SUMMARY

Microsoft Excel version 3.00 may generate an Unrecoverable Application Error (UAE) in print preview if Attachmate Extra for DOS version 1.4x is running at the same time.

MORE INFORMATION

According to Attachmate Technical Support, this problem is solved by modifying a DOS environment variable that is placed in your AUTOEXEC.BAT file by the Extra Setup program.

Open the AUTOEXEC.BAT file in a text editor. There will be an entry that reads "SET ACPINT=". Modify this line to read "SET ACPINT=62" (without the quotation marks.)

ACPINT (Attachmate control program interrupt) is a variable that specifies which interrupt level to operate.

For further information on the ACPINT variable, please contact Attachmate Technical Support.
