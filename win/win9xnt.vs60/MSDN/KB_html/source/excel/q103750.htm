XL: List of Available Worksheet Functions

 Last reviewed: February 2, 1998
Article ID: Q103750

The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 4.0, 5.0

SUMMARY

The following are the 307 worksheet functions available in Microsoft Excel. This list also includes the functions that are added when you load the add- in macros ADDINFNS.XLA, ANALYSF.XLA, and CROSSFNC.XLA.
