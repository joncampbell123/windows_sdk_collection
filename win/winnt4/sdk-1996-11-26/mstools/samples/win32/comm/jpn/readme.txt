サンプル: 通信 API 関数のデモ

要約:

COMM は、Win32 通信 API 関数の基本的な使い方を示すサンプル 
アプリケーションです。

詳細:

COMM プログラムは、Windows の関数である OpenFile、ReadFile、
SetCommState、SetCommMask、WaitCommEvent、WriteFile および 
CloseFile を使用して通信を行ないます。

このサンプルは、COMM 受信イベントを監視して、通知メッセージを
メインの端末ウィンドウに送るために、バックグラウンド スレッ
ドを作成します。フォアグラウンドの文字処理は通信ポートに
書き込まれます。

単純な TTY 文字変換が行なわれ、スクリーンバッファは、
I/O ウィンドウを使うかのように実現されています。

オーバーラップド ファイル I/O の技法も示されています。

使用法:

ボー レート、データ ビット、ストップ ビット、パリティ、ポート、
RTS/CTS によるハンドシェイク、DTR/DSR によるハンドシェイク、
および XON/XOFF によるハンドシェイクが設定メニューの項目として
変更できます。

一度通信設定が行なわれると、動作メニュー項目で TTY プログラム
との接続や接続断を選択できるようになります。
