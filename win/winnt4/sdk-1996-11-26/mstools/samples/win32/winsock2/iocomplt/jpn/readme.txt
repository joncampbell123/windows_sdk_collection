I/O 完了ポートを使用したオーバーラップ I/O


このソケット サンプルには、Windows ソケットを使用する簡単な Windows NT のサーバーが含まれています｡
ReadFile の完了通知を受け取るために I/O 完了ポートを使用するオーバーラップ I/O のデモンストレーションをします｡
I/O 完了ポートはオーバーラップ I/O のもっとも効率的な形式であり、Windows NT 3.5 で導入されました。

このサンプルにふくまれているサーバーの中で使用している CreateIoCompletionPort 関数は Windows NT でのみサポートされています。
