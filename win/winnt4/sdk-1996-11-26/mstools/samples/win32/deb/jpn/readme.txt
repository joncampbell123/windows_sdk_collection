サンプル: Win32 デバッグ API のデモ

要約:

デバッグ イベント ブラウザ (DEB) Win32 SDK サンプルは、新しい 
Win32 デバッグ API のデモを行います。このサンプルは、新しく
作成されたデバッグ対象プロセスのデバッガとして、または現在
アクティブなプロセスに付加されたデバッガとして動作します。

DEB は、旧来の感覚のデバッガではありません。これは文字通り
ブラウザです。DEB はデバッグ イベントと関連する情報を、
発生したままに表示し、デバッグ対象プロセスまたはシステムに
よって与えられるデフォルト ハンドラを呼び出します。
デバッグ イベントの表示など、最小限のデバッグ イベント処理
だけが行なわれ、デバッグ対象プロセスは通常どおりの実行を
続けます。

このサンプル は Intel 80x86、MIPS R4x00、および DEC Alpha AXP の 
Windows NT プラットフォームに対してソース コード互換です。
