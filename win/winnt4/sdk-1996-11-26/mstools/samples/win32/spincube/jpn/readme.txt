サンプル:　標準的な DLL と カスタム コントロール クラスの作成例

概要:

SPINCUBE は汎用の Windows NT ダイナミック リンク ライブラリ テンプレート
を提供するサンプルで、DLL のエントリ ポイント、エクスポート変数、DLL での 
C ランタイムの使用等の使い方についてデモを行います。

また、このサンプルはカスタム コントロール クラスの作り方について機能上
の例を提供しますが、これはダイアログ エディタと同様、アプリケーション 
(例: SPINTEST.EXE) で使うことができます。


詳細:

SPINCUBE.DLL には コントロール ペイント ルーチン (PAINT.C参照) と同様、
ダイアログ エディタ（SPINCUBE.C 参照) に必須のコントロール ウィンドウ 
プロシージャとインターフェース関数が入っています。SPINTEST.EXE は小さ
なテスト プログラムで SPINCUBE.DLL をロードしカスタム コントロールを
いくつか作成します。

ダイアログ エディタを使って SPINCUBE をテストする方法:

    1.	エディタを開始します。[ファイル] メニューの [カスタム コント
	ロールを開く］を選択します。

    2.	SPINCUBE.DLL のパス名とファイル名を入力します。

    3.	新しいダイアログ ボックスを作り、コントロール パレットの [カスタム 
	コントロール] ボタン (右下角にあります) を選択します。

    4.	ダイアログ ボックスをクリックして SPINCUBE コントロールを作成します。

    5.	ダイアログ ボックス テンプレートを保存します。

    6.	作成された .DLG ファイルを確認します。
