この PRINTER サンプルは以下を行います。

    -	プリンタのデバイス コンテキスト ハンドル (HDC) を獲得する 
	CreateDC() とPrinterDlg() の 2 つのメソッドを使用して、NT 上で
	印刷を行う方法を示します。また完全なデバイス フォント セット
	だけでなく、異なるグラフィカル オブジェクトを印刷することが
	できます。また [中止] ダイアログも用意しています。

    -	すべてのプリンタとディスプレイに関するデバイスの能力に関する
	情報を提供します。

    -	EnumPrinters から返される情報 (レベル 1 と 2) を提供します。

    -	特定のデバイス コンテキストのためのフォントの列挙の方法を
	紹介します。

    -	さまざまなマッピング モードの違いを示します。

    -	GDI 関数のデモを行います。



メイン アプリケーション ウィンドウにはメニューとツールバーがあります。
サブメニューは以下の通りです。

  - オプション

        GetDCによる印刷        ツールバーのコンボボックスで選択された
                               プリンタのデバイス コンテキストを CreateDC 関数
                               により獲得し、この DC に現在のグラフィック 
                               オプションで印刷を行います。

        PrintDlg               プリンタのデバイス コンテキストを PrintDlg 関数
                               により獲得し、この DC に現在のグラフィック 
                               オプションで印刷します。

        GetDeviceCaps          ツールバーのコンボボックスで現在選択されている
                               デバイスに関するデバイス固有の情報を取得し、
                               ダイアログ ボックスに表示します。

        EnumPrinters           EnumPrinters 関数によって得られるレベル 1 と 
                               2 の情報を取得し、ダイアログ ボックスに
                               表示します。

        GetPrinterDriver       現在選択されているプリンタのレベル 1 と 2 の
                               情報を GetPrinterDriver 関数により獲得し、
                               ダイアログ ボックスに表示します。

        EnumPrinterDrivers     EnumPrinterDrivers 関数によって得られる
                               レベル 1 と 2 の情報を取得し、ダイアログ 
                               ボックスに表示します。

        リフレッシュ           ツールバーのコンボボックスの内容を最新の表示
                               にします。

        バージョン情報         アプリケーションに関する情報のダイアログです。


  - マッピング モード          マッピング モードを選択します。

  - グラフィック               表示したいプリミティブを選択します。

  - ペン                       描画用のペンのサイズ、色、およびスタイルを
                               設定します。

  - ブラシ                     描画用のブラシのサイズ、色、およびスタイルを
                               設定します。

  - 文字色...                  文字を描画するときの色を設定します。
