サンプル: 動的ダイアログの作成

概要:

一般に、アプリケーションはダイアログ テンプレートを RC または DLG 
ファイル内に定義しており、ここから実行時にダイアログを作成します。
しかし、実行時にダイアログ テンプレートを作成することも可能です。
これを Win32 でどのように行うかを、このサンプルで示します。

詳細:

Win32 のダイアログ テンプレートは、概念的には Windows 3.1 バー
ジョンのものと似ていますが、違いがあるため、移行には特別な作業
が必要となります。例えば、すべての文字列は Unicode で保管されて
います。ダイアログ テンプレートの形式は、関連するヘルプ ファイル
に記述があります。

実行時に必要なもの:

spincube.dll ファイル内で定義されていると思われるカスタム 
コントロールを使っているダイアログが 1 つ必要です。dyndlg 
サンプルは、実行時にこの DLL を次の位置から読み込もうとします。
    "..\spincube\spincube.dll"
つまり、同じレベルのディレクトリで、名前が spincube のディレクトリ
からです。これはサンプル コードを調べれば、簡単に変更して再コンパ
イルできます。spincube は Win32 SDK で配布される、別のサンプルです。
