サンプル: コンソール API 関数のデモ

要約:

この CONSOLE サンプルは、ほとんどの Win32 コンソール API 関数の
使用法を例示します。プログラムはパラメータを取りません。開始する
には CONSOLE.EXE を実行するだけです。サンプルを開始したら
スクリーン上の関数をマウスでクリックすると、その関数のデモが
行われます。関数のデモ中は、コンソール ウィンドウのタイトルが、
その関数を含んでいるソース ファイル名を示します。したがって、
興味のある関数が、どのサンプル コードに含まれているかが
簡単に分ります。

一部のデモは、複数の API の例示を含んでいるため、メニューで選ぶ
項目の一部に、同じデモを実行するものがあります。
