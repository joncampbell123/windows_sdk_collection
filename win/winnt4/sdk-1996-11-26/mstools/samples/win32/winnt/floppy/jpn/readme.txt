モジュール名:

    mfmt.c

概要:

    このプログラムは、Win32 API セットを使って、フロッピー ディスクの
    物理ドライブにアクセスする方法を示します。

    このプログラムは 2 つの主な機能を持っています。

        - ディスクのジオメトリ (セクタ情報) を表示できます。

            mfmt -g a:

        - ディスク イメージを生成することができます。また、ディスク 
          イメージをフロッピー ディスクに書くことができます。

            mfmt -c a: bootdisk         - a: のディスク イメージを生成する

            mfmt -c bootdisk a:         - a: を bootdisk イメージと同じにする

    このプログラムは非常に単純なものです。最小限のエラー検査しかしていません。
    以下の項目の例を提供するために作成されています。

        - 物理ディスクをオープンする

        - ディスクのジオメトリを読み取る

        - 低レベルのフォーマット操作を行う

        - 物理セクタを読み書きする
