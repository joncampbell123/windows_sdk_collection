                       FILER.EXE - ファイル I/O サンプル
                       =================================

要約:

Filer.exe はファイルマネージャやノートン コマンダーに似た、基本的な
ファイル管理アプレットです。これは、多数の新しいファイル I/O API と、
ドライブの列挙、.EXE のバージョン情報取得、およびディレクトリ ツリーの
拡張などの関係するアルゴリズムをデモします。また、子ウィンドウ管理、
サブクラス、同期、およびコントロール ウィンドウ管理などのユーザーに
関連するアルゴリズムのデモも行います。


説明:

ファイラはユーザーに設定可能な 2 つの子ウィンドウを示します。
ユーザーはそれぞれをシステム上で利用可能なドライブに
関係づけます。

[ドライブ] ツールバーはシステム上で利用可能なドライブを示します。ユーザーは
ツールバーまたは [ドライブ] メニューからドライブを選択します。また、機能ツール
バーは [ファイル] メニューの中のファイル I/O 機能に対応しています。アプリケー
ションの下のコマンド ライン ウィンドウは、入力したコマンドでコマンド シェルを
起動します。指定したコマンドが完了したときにコマンド ウィンドウを保持するか
破棄するかを指示するオプションもコマンドと一緒に指定します。

それぞれの [ドライブ] 子ウィンドウは、[ディレクトリ] リスト ボックスと 
[ファイル] リスト ボックスを含んでいて、これにより選択したドライブ上の
ファイルを見ることができます。

ファイラを使うと、ファイルのオープン、複写、削除、移動、名前の変更、
さらにはディレクトリの作成、削除などができます。Win32 ファイルに
埋め込まれたバージョン情報を表示することもできます。アクティブな 
[ドライブ] 子ウィンドウがファイル I/O 操作の元となり、アクティブでない 
[ドライブ] 子ウィンドウがデフォルトの操作先となります。

また、2 つの [ドライブ] 子ウィンドウの位置は、左右または上下に設定でき、
[ファイル] と [ディレクトリ] のリスト ボックスの位置も入れ換えることが
できます。ディレクトリ ツリーをいっぱいに引き伸ばすこともできます。

ファイラのすべての機能は、マウス、キーボード、またはメニュー項目から
選択できます。
