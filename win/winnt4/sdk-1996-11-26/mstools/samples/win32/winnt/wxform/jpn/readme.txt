サンプル： ワールド座標変換

要約：

WXFORM サンプル プログラムは新しいワールド座標変換
のデモ プログラムです。このサンプルは世界座標の
矩形領域と変換値を持ったマトリックスを表示します。
ユーザーは領域を直接操作して、変換値を変える
ことができます、また、変換値を設定して領域を
操作することもできます。


詳細：

プログラムは、最初にクライアント領域の中央にビュー
ポートの原点を設定します。その後、世界座標の(0,0)
から(100,100)までの領域に矩形を描画します。ユーザー
はマウスの左右のボタンを使って、矩形領域を直接操作
することができます。動作の詳細は [操作ヘルプ] 
ダイアログ ボックスに、より詳しく表示されます。

"ワールド座標変換" ダイアログ ボックスには、
GetWorldTransform 関数を実行して得た、WXFORM 
構造体の eM11、eM12、eM21、eM22、eDx および 
eDy フィールドの値が表示されます。このダイアログ 
ボックスのボタンを選択することによって、ユーザーは
SetWorldTransform 関数をプログラム中で実行すること
ができます。

このサンプル プログラムには 3 種類の座標系を
持っています。1 番目は世界座標系で、これは 
Win32 で新たに追加されたものです。2 番目の
座標系はデバイス座標系で、座標は、ウィンドウ
に表示される前に、最終的にこの座標系にマップ
されます。三番目は、スクリーン座標系で、
マウス ポインタとの相互動作に用いられます。

[マウスの位置] ダイアログ ボックスはマウス 
カーソルの位置を 3 種類の座標系の座標で表示
します。ただし、デバイス座標はクライアント領域の
左上角を原点としており、ビューポートの原点とは
関連していません。

その他の参照情報:	ModifyWorldTransform
