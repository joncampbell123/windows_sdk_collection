
Mazelords は 3 次元の迷路ゲームです。ゲームの目的は、他のプレーヤを倒したり、
迷路内をうろついている自動ドローンを壊すことで、高い得点を得ることです。
他にプレーヤがいなくても、ドローンを壊すだけでもおもしろいゲームです。

Mazelords は、起動時の指定により、ネットワーク環境でも使用できます。どちらの
バージョンでも、プレーヤは宮殿の聖なる部屋におかれます。この部屋の中では、
プレーヤは撃つことができませんが、ドローンがプレーヤを傷つけることもありま
せん。しかし一度、この 3x3 の範囲から出ると、制約が外されます。他のプレーヤや
ドローンを見つけ出して、倒してください。倒されたプレーヤは、テレポートで
聖域に連れ戻されます。ドローンは、迷路内のランダムな位置にテレポートします。

ドローンは、灰色のロボットです。ドローンの数とスピードは [オプション] から
設定できます。ドローンの数 20 とスピード 2 が、ゲームを楽しめる設定です。

Mazelords の制御は簡単です。

  上方向キー:   1 区画前進します。
  左方向キー:   同じ区画内で左を向きます。
  右方向キー:   同じ区画内で右を向きます。
  下方向キー:   1 区画後退します。
  スペース:     視線にあるものを何でも撃ちます。
  X: ゲームを終了します。
  W: 他のプレーヤにささやくことができます。自分のいる位置から、指定した数の
     区画内にいるプレーヤに密談メッセージを送ることができます。
  S: 叫ぶこともできます。Whisper (ささやき) と似ていますが、デフォルトの距離内
     のプレーヤにメッセージが送られる点が違います。

ゲームを楽しむには、マシンにメモリを拡張し、サウンド機能を使用できるようにする
ことが理想的ですが、サウンド機能がなくても、ゲームを行うことはできます。また、
重要なサウンドはスピーカのビープ音に変換されます。


迷路を起動するとき、以下のオプションを選ぶことができます。
    ファイル->新規ゲーム: 新しいゲームを始めます。デフォルトの設定を変えたり、
                     ネットワークを使う場合は、[OPTIONS] メニューでゲームの
                     設定を変えてください。
    ファイル->ゲームの一時停止: すべてのドローンの動きを止めます。しかし自分の
                     動きも止まるため、他にプレーヤがいれば、いいカモになって
                     しまいます。この機能は、変更されるかもしれません。
    ファイル->ゲームの停止: ゲームを停止してリセットします。
    ファイル->終了:  ゲームを終了します。

    オプション->ネットワーク/ローカル: ネットワーク ドメイン内の他のユーザーと
                     ゲームを行うかどうか指定します。NETWORK で実行して他に
                     プレーヤがいなくてもゲームは行えます。
                     デフォルトは LOCAL です。
    オプション->プレーヤの設定: 2 つのリスト ボックスのあるダイアログを表示し
                     ます。1 つめは、自分のキャラクタを表現するビットマップを
                     選ぶためのリストを表示します。現在は、魚とスマイル フェイ
                     スだけですが、選択項目は増えるでしょう。家の迷路も選ぶこ
                     とができますが、現在は使用できません。しかし、すぐに、既存
                     のランダム迷路の一部として追加されるでしょう。この場合、
                     そこにテレポートできるようになります。
    オプション->ドローンの設定: ドローン設定ダイアログを表示します。望みのドロー
                     ンの数とドローンの移動速度を選択してください。速度は、1 秒
                     間に移動する区画数で指定します。移動アルゴリズムは、現在
                     無視され、同じランダム アルゴリズムですべてのドローンが移動
                     します。



表示は、以下で構成されます。

o   迷路の 3 次元表示:  正確性の点から見ると、いくつかの描画バグがありますが、
    これを見ていたいのでなければ、特に問題はありません。壁があるかどうか疑問
    に思ったばあいは、壁に向かって移動することで確認できます。

o   上から見た俯瞰図:  これは、プレーヤのいる場所から各方向に 2 区画分の表示
    です。俯瞰図では、他のプレーヤを見ることはできませんが、自分の向いている
    方向を表す、小さな三角形が中央に表示されます。このウィンドウは絶対座標で、
    プレーヤが迷路内でどちらを向いているかにかかわらず、このウィンドウの上は
    常に北を指します。

o   テキスト ウィンドウ:  ゲーム中で起きていることの情報を示します。倒した
    相手や、やっつけられた相手、さらにプレーヤがゲームに参加したことや抜けた
    ことが通知されます。他のプレーヤが近くにいる場合は、ささやきや叫びのメッ
    セージも表示されます。

o   得点ウィンドウ:  ネットワークでゲームを行っている場合は、ドメイン内の
    アクティブなプレーヤをすべて表示します。この表示は、プレーヤが使っている
    ビットマップ、得点、およびそのプレーヤがいる区画に行くために進むべき方向
    が絶対座標で示されます。表示されている値は、変更の度に更新されます。


ゲームを楽しんでください。

MazeLords は、現在も開発中です。Windows 3.1 用にクロス コンパイルすることも
できますが、まだ Windows 3.1 では実行できません。
