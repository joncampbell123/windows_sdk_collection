****************************************************************

			マルチメディア サンプル

****************************************************************

サンプル	説明
---------       ------------------------------------------------
ACMAPP          ウェーブ ファイル フォーマットの表示や、ウエーブ
		ファイルの再生、レコード、変換を行います。

AVIEDIT		AVIFile の編集 API を使用した簡単な AVI	編集アプ
		リケーションです。

AVIVIEW		AVIFile の入出力 API を使用した簡単な AVI ビューア
		アプリケーションです。

CAPTEST		AVICap キャプチャー ウィンドウ クラスを使用する
		簡単なキャプチャー アプリケーションです。

DSEQFILE	DIB シーケンスの AVIFile ファイル ハンドラです。
		C++ でインプリメントされています。インストールする
		には、dseqfile.reg を実行して dll をシステム ディレ
		クトリにコピーしてください。AVIEDIT や AVIVIEW と
		いっしょに使用することができます。

ICMAPP		ICM API の用途を示します。

ICWALK		現在インストールされているすべての ICM コンプレッサ
		を表示するサンプル アプリケーションです。ICM API を
		使用します。

IDFEDIT         MIDI IDF (Instrument Definition File) エディタ。

JOYTOY		ジョイスティック サンプル アプリケーション。

LANGPLAY	マルチ オーディオ ストリーム AVI ファイルの再生を行
		います。CD に MULTILNG.AVI というサンプル ファイルが
		用意されています。

LOWPASS		ウェーブフォーム オーディオのローパス フィルタを実行
		します。

MCIAPP		MCI のテスト アプリケーションです。

MCIPLAY		簡単なビデオ再生アプリケーションです。
		MCIWnd を使用します。


MCIPUZZL	ビデオ再生の「15」パズルを実装するアプリケーションです。
		このサンプルは、インストール可能な描画プロシージャの
		使い方を示します。

MIDIMON		MIDI 入力メッセージの情報を表示するサンプルです。

MIDIPLYR        MIDI プレーヤ サンプル アプリケーション。

MIXAPP		ミキサ アプリケーションのサンプルです。

MMCAPS		ハードウェアのマルチメディア性能を示すサンプルです。

MOVPLAY		MCI を使用して映像を再生する簡単なアプリケーションです。
		このアプリケーションには、2 つのバージョンが含まれており、
		１つは mciSendCommand を使い、もうひとつは mciSendString
		を使います。

MPLAY		簡単な映像再生用アプリケーションです。メディア  プレーヤ
		の機能の一部分を含んでおり、MCIWnd ウィンドウ クラスを
		使用しています。

PALMAP		ビデオ ストリームを異なるビット デプスのストリームに変換
		します。例えば、24ビットから8ビットに変換することができ
		ます。dll をシステム ディレクトリにコピーして、AVIVIEW 
		の [Find Optimal Palette] メニュー アイテムで使用します。

REVERSE		ウェーブフォームを逆向きに再生する簡単なアプリケーション
		です。

SOUNDAPP	SndPlaySound API を示す簡単なアプリケーションです。

TXTOUT		AVI ファイル内のテキスト データの為のテキスト ストリーム
		を描画する dll です。Win95 にインストールするには、dll を
		システム ディレクトリにコピーして下記に示す行を system.ini
		の [drivers32] セクションに追加してください。
		    txts.draw=txtout32.dll
		NT の場合は、コントロール パネルのドライバから、サンプル
		ディレクトリを指定してインストールします。
		テキスト ストリームの例は、WRITEAVI によって作成される
		OUTPUT.AVI があります。

VIDCAP		フル使用のビデオ キャプチャ アプリケーションです。

WAVEFILE	ウェーブフォーム オーディオ ファイルの 簡単な AVIFile
		ファイル ハンドラです。インストールするには、dseqfile.reg
		を実行して dll をシステム ディレクトリにコピーしてください。
		AVIEDIT や AVIVIEW といっしょに使用することができます。

WRITEAVI	AVIView などのアプリケーションで開くことができるようなファ
		イルを出力するための AVIFILE インタフェースの使い方を示す
		サンプル アプリケーションです。
