サンプル:  Windiff

概要:

Windiff はディレクトリやファイルを比較し、グラフィックで比較結果を表示します。

コマンド ライン構文:

	WINDIFF [paths] [saveoption]

paths

   path
	パスの内容とカレント ディレクトリの内容を比較します。

   path1 path2
	パス1 の内容とパス2 の内容を比較します。

   注:	パスは相対パスでも絶対パスでも、ネットワーク上でもローカルでも、
	ファイルでもディレクトリでもかまいません。

saveoption = -slrd savefile

   ここで、slrd はこれらの 4 つの文字の任意の組み合わせで、該当する
   ファイル名を書き出すために使います。

       s - 両方のパスにあり同一内容のもの
       l - 左側のパスのみにあるもの
       r - 右側のパスのみにあるもの
       d - 両方のパスにあるが、内容が違うもの

詳細:

    色
	赤色の背景は左のファイルからのテキストを表します。
	黄色の背景は右のファイルからのテキストを表します。
	青色のテキストは移動された行を表します。
	黒色のテキストは上記以外のテキストを表します。


    [ファイル] メニュー
	[ファイルの比較...]
		比較を行う 2 つのファイルの各々に [ファイルを開く] 
		ダイアログを表示します。
	[ディレクトリの比較]
		比較する 2 つのディレクトリの名前を入力するための
		ダイアログを開きます。
	[閉じる]
		現在作業中のファイルを閉じます。
	[中止]
		操作が進行中でないときは灰色表示になっています。
		選択すると操作が完了する前に強制終了させます。
	[ファイルリストの保存]
		保存するファイル (同一内容のもの、異なるもの、左側パス
		のみにあるもの、または右側パスのみにあるもの) をリストします。
	[ファイルのコピー...]
		ファイルをディスクに書き込むためのダイアログを表示します。
	[印刷]
		比較結果を印刷します。
	[WinDiffの終了]
		WinDiff を終了します。

[表示] メニュー
	[概要表示]
		ファイルのリストを表示します。
	[詳細表示]
		選択されたファイルの比較結果を表示します。
	[ピクチャ]
		選択されたファイルと一緒にピクチャを表示します。
	[変更の検索＜上＞ (F7)]
		ファイル中の、前の相違点までスキップします。
	[変更の検索＜下＞ (F8)]
		ファイル中の、次の相違点までスキップします。

[詳細表示] メニュー
	[左側のファイルのみ]
		左側のファイルの行のみを表示します (ただし、行が変更された
		ことを強調するため色が付けられます)。
	[右側のファイルのみ]
		右側のファイルの行のみを表示します (ただし、行が変更された
		ことを強調するため色が付けられます)。
	[両方のファイル] (デフォルト)
		両方のファイルをマージして表示します。左側ファイルの行は
		すべて、ファイル中にあるのと同じ順序で表示されます。右側
		ファイルについても同様です。左側ファイルのみにある行は赤色
		で表示されます。右側ファイルのみにある行は黄色で表示されます。
	[左の行番号]
		左側のファイルの行番号を表示します。
	[右の行番号]
		右側のファイルの行番号を表示します。
	[行番号なし]
		行番号を表示しません。

[オプション] メニュー
	[空白文字を無視]
		詳細表示モードでは空白が無視され、空白部分のみが異なる行は
		同一内容として表示されます。
	[同じファイルを表示]
		一覧表示モードで各々のパスで同一内容のファイルを表示します。
	[左側にのみ存在するファイル]
		一覧表示モードで左側のパスのみにあるファイルを表示します。
	[右側にのみ存在するファイル]
		一覧表示モードで右側のパスのみにあるファイルを表示します。
	[異なるファイルの表示]
		一覧表示モードで両方のパスにあるが、内容の異なる
		ファイルが含まれます。

[ヘルプ] メニュー
	[バージョン情報]
		WinDiff に関するバージョン情報を表示します。
