サンプル: DlgEdit

概要:

DlgEdit は、リソース スクリプト内でダイアログ ステートメントを定義するかわり
に、ディスプレイ画面上でダイアログ ボックスの設計とテストを行うことができる
ツールです。DlgEdit を使って、新しいダイアログ ボックスを作成したり、既存の
ダイアログ ボックスのコントロールを追加したり削除したりできます。その後、この
リソース ファイルをリソース コンパイラでコンパイルします。リソース コンパイラ
からのバイナリ出力は、アプリケーションの実行可能ファイルにリンクされます。

DlgEdit は以下の技法のデモを行います。

1. ダイアログ ボックスの作成
2. リソース ファイルの読み書き
3. コントロール ウィンドウの作成
4. ヘルプ ファイルの作成
5. カスタム コントロールの作成
6. コントロールのグループの作成
7. インクルード ファイルでの作業
