Sample: Event Logging

Summary:

  This sample shows how to do event logging. The sample demonstrates
  how to set up the proper registry entries to register your message & 
  category DLLs, how to make the actual event log entries, how to pull 
  event-log data from the event log, how to extract the event messages 
  from the message DLL, and how to set up a notification routine so that
  the application is notified when the event log changes.
