	  Windows Telephony Information - SPI Programmer's Guide

This directory contains the Windows Telephony Service Provider Programmer's 
Guide, which consists of the following files:

TITLE.DOC       Title
TOC.DOC         Table of Contents
INTRO.DOC       Introduction
OVERVIEW.DOC    Chapter 1, Overview of the Telephony SPI
MODEL.DOC       Chapter 2, The Telephony Programming Model
START.DOC       Chapter 3, Programming Telephony Service Providers: 
			   Getting Started
LINEDEV.DOC     Chapter 4, Concepts of Line Devices
PHONDEV.DOC     Chapter 5, Concepts of Phone Devices
SPFUNC.DOC      Chapter 6, Service-Provider Functions
SPLINEFN.DOC    Chapter 7, TSPI Line-Related Functions
SPLINEMS.DOC    Chapter 8, TSPI Line-Related Messages
SPDATA.DOC      Chapter 9, TSPI Data Types
SPLINEDA.DOC    Chapter 10, TSPI Line-Device Data Structures
SPLINCON.DOC    Chapter 11, TSPI Line-Device Constants
SPPHONFN.DOC    Chapter 12, TSPI Phone-Related Functions
SPPHONMS.DOC    Chapter 13, TSPI Phone-Related Messages
SPPHONDA.DOC    Chapter 14, TSPI Phone-Related Data Structures
SPPHONCO.DOC    Chapter 15, TSPI Phone-Related Constants
TELEINI.DOC     Appendix A, Configuration Information and TELEPHON.INI
CALLSTAT.DOC    Appendix B, Call States and Events
GLOSSARY.DOC    Appendix C, Glossary of Telephony Terminology

NOTES:
1.  Please read the Introduction for more information about the organization 
    of the Service Provider Programmer's Guide.
2.  These files of the Service Provider Programmer's Guide can be found in 
    the documentation directories of the Windows Telephony SDK. They can 
    be opened and printed using Microsoft Word for Windows 2.0.
3.  Source code for the Sample Service Provider is included with the 
    Telephony SDK distribution.
4.  The online help files distributed with Telephony may be slightly 
    more up to date than the printed documentation.
5.  The line and phone device reference files (Chapters 6-15, files
    SPFUNC.DOC, SPLINEFN.DOC, SPLINEMS.DOC, SPDATA.DOC, SPLINEDA.DOC,
    SPLINCON.DOC, SPPHONFN.DOC, SPPHONMS.DOC, SPPHONDA.DOC, SPPHONCO.DOC)
    are located in the ..\REF directory.
