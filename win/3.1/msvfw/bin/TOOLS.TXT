Tools for working with VfW:

riffwalk.exe:
	Tool for dumping RIFF files, with special support for AVI and
	WAV files.  Use "riffwalk -f <filename>" to dump AVI headers.
	Execute "riffwalk" without a file for full parameters.


ddtest.exe:
	Tool for testing DrawDib functions, display drivers, and
	installable video codecs.  Can test quite a few things, and
	definitely needs better documentation.


uuidgen.exe:
	OLE 2 tool for generating unique identifiers.
