Capwndx Test Using Visual Basic
===============================

CAPTURE.MAK     Makefile for VB project.
CAPTURE.FRM     Form containing a CAPWNDX control and 
                buttons for controlling properties.
VBCAPTST.EXE    The compiled VB program, located in the BIN directory
VBCAPTST.TXT    This file.


Release Notes:
==============
This sample app creates and displays a capture window using
CAPWNDX.VBX and a playback window using MCIWNDX.VBX.
