             INSTALLING THE MISCROSOFT WINDOWS 3.1 DCT/PCT

  The Microsoft Windows 3.1 Display Driver Compatibility Test (DCT) and
Printer Driver Compatibility Test (PCT) are collections of tests to assist
in testing drivers for Windows.  The tests include both specialized test
applications and retail ISV applications.  The included applications are
provided with the permission of their authors, and are included for testing
purposes only.  ANY OTHER USE OF THESE PROGRAMS IS A VIOLATION OF THE
LICENSING AGREEMENT AND INTERNATIONAL COPYRIGHT LAW.

  The tests are installed by running DCTSETUP.EXE for the DCT, or
PCTSETUP.EXE for the PCT. The setup program must be run from within
Windows. Either installation will require approximately 17 MB of disk
space. There is no option for a partial installation, as the included
automation tools are designed for a specific directory structure.

  Below is a complete list of the files included in the DCT/PCT. These
files are not compressed.
