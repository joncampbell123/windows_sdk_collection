	 Windows Telephony Information - API Programmer's Guide

This directory contains the Microsoft Windows Telephony Programmer's Guide,
which consists of the following files:

TITLE.DOC       Title
TOC.DOC         Table of Contents
INTRO.DOC       Introduction
OVERVIEW.DOC    Chapter 1, Overview of Telephony
MODEL.DOC       Chapter 2, The Telephony Programming Model
START.DOC       Chapter 3, Programming Telephony Applications: Getting Started
ASSISTED.DOC    Chapter 4, Programming with Assisted Telephony
LINEDEV.DOC     Chapter 5, Concepts of Line Devices
PROGGUID.DOC    Chapter 6, Programming a TAPI Application
SUPPLINE.DOC    Chapter 7, Supplementary Line Functions
MULTIAPP.DOC    Chapter 8, Multiple-Application Programming Considerations
PHONEDEV.DOC    Chapter 9, Programming with Phone Devices
LINEFUNC.DOC    Chapter 10, Line Device Functions
LINEMSG.DOC     Chapter 11, Line Device Messages
LINEDATA.DOC    Chapter 12, Line Device Data Structures
LINCONST.DOC    Chapter 13, Line Device Constants
PHONHELP.DOC    Chapter 14, Phone Device Functions
		Chapter 15, Phone Device Messages
		Chapter 16, Phone Device Data Structures
		Chapter 17, Phone Device Constants
FUNCREF.DOC     Appendix A, TAPI Quick Function Reference
TELEINI.DOC     Appendix B, Windows Telephony Configuration Parameters
CALLSTAT.DOC    Appendix C, Call States and Events
GLOSSARY.DOC    Appendix D, Glossary of Telephony Terminology
INDEX.DOC       Index

NOTES:
1.  Please read the Introduction for more information about the organization 
    of the Telephony Programmer's Guide.
2.  These files of the Telephony Programmer's Guide can be found in the 
    documentation directories of the Windows Telephony SDK. They can be 
    opened and printed using Microsoft Word for Windows 2.0.
3.  Source code for the Dialer application is included with the Telephony 
    SDK distribution.
4.  The online help files distributed with Telephony may be slightly more 
    up to date than the printed documentation.
5.  The line and phone device reference files (Chapters 10-17, files
    LINEFUNC.DOC, LINEMSG.DOC, LINEDATA.DOC, LINCONST.DOC, and 
    PHONHELP.DOC) are located in the ..\REF directory.
