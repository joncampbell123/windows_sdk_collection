============================================================================
The Microsoft Win32 Software Development Kit is provided on CD-Only.  Printed
documentation is available as a stand-alone package.
The Win32 SDK CD-ROM includes the Postscript source files for all printed
documents, as well as on-line help format.

NOTE: The WinHelp documentation should always be considered the most
      accurate information other than the README.TXT found on the disk.  
      We encourage the customer to use those files as their major source of
      development information wherever possible.


Below is a list of the contents of each documentation file included on the 
Win32 SDK CD-ROM, along with its location.  Most of these files are comp-
ressed on the CD.  In order to view and/or print them, you must first run
expand -r filename.  For the Postscript files, you can then just copy them
to a standard Postscript Printer. 
============================================================================
